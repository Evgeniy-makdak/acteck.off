Imaginary property getters in OOP in JavaScript

With the help of accessors, you can create imaginary properties that actually do not exist in the object. Let's look at an example. Let's say we have a class with two public properties:

class User { constructor(name, surn) { this.name = name; this.surn = surn; } }

Let's make a third property that will contain the first and last name. Let's create an accessor property for this:

class User { constructor(name, surn) { this.name = name; this.surn = surn; } get full() { return this.name + ' ' + this.surn; } }

Let's check. Let's create a class object, passing the first and last name as parameters:

let user = new User('john', 'smit');

Let's display the values ​​of public properties:

console.log(user.name); // 'john' console.log(user.surn); // 'smit'

Now let’s print the value of our imaginary property:

console.log(user.full); // 'john smit'
