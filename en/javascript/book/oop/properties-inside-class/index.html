Accessing properties inside classes in OOP in JavaScript

Inside class methodsthiswill point to an object of this class:

class User { show() { console.log(this); // объект } }

This means that we can access the properties of an object viathis. Let's try it. Let our object have a propertyname. Let's display this property in our method:

class User { show() { console.log(this.name); } }

Let's now create an object of our class:

let user = new User;

Let's write down the property we need:

user.name = 'john';

Let's now call the method, thereby displaying the value of the property:

user.show(); // выведет 'john'
