Property getters in OOP in JavaScript

Let's say we have the following class with private properties:

class User { #name; #surn; constructor(name, surn) { this.#name = name; this.#surn = surn; } }

As you can see, these properties are set once when the object is created. Now, however, these properties cannot be read, because they are private and there are no corresponding methods for this.

Let's make special methods for our properties that allow us to read these properties. Such methods (they are calledgetters) must begin with the wordget, followed by the name of the property being read.

Let's make our property getters:

class User { #name; #surn; constructor(name, surn) { this.#name = name; this.#surn = surn; } getName() { return this.#name; } getSurn() { return this.#surn; } }

Let's check their work. Let's create an object by passing user data as a parameter:

let user = new User('john', 'smit');

Let's read this data using getters:

console.log(user.getName()); console.log(user.getSurn());
