Calling methods inside classes in OOP in JavaScript

Some methods can be called inside others viathis. Let's look at an example. Let us be given a class with a user and a method that returns a property:

class User { show() { return this.name; } }

Let us also have a methodcape, which converts the first letter of the string to capital:

class User { show() { return this.name; } cape(str) { return str[0].toUpperCase() + str.slice(1); } }

Let's use the methodcapeinside a methodshow:

class User { show() { return this.cape(this.name); } cape(str) { return str[0].toUpperCase() + str.slice(1); } }
