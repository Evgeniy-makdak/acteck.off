Descendant methods in OOP in JavaScript

A descendant class can have its own methods. For example, let’s add a getter and setter for the year of study to our student:

class Student extends User { setYear(year) { this.year = year; } getYear() { return this.year; } }

In the descendant class, both its personal methods and inherited ones will be available. Let's check it out. Let's create a class object:

let student = new Student;

Let's set its name using the inherited method, and the year of study using our own method:

student.setName('john'); student.setYear(1);

Let's read his name and year of study:

let name = student.getName(); let year = student.getYear(); console.log(name, year);
