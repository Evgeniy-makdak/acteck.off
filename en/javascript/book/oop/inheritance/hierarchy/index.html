Class Inheritance Hierarchy in OOP in JavaScript

You can inherit from a class that is itself a descendant. Let's look at an example. Let us have the following parent class:

class User { setName(name) { this._name = name; } getName() { return this._name; } }

The following class inherits from this class:

class Student extends User { setYear(year) { this._year = year; } getYear() { return this._year; } }

And from this class another class inherits:

class StudentProgrammer extends Student { setSkill(skill) { this._skill = skill; } getSkill() { return this._skill; } }

Make a classEmployee, which will inherit from the classUser.

Make a classProgrammer, which will inherit from the classEmployee.

Make classesDesigner, which will inherit from the classEmployee.
