Manipulating objects in classes in OOP in JavaScript

Method parameter classes can accept objects of other classes and manipulate those objects. Let's look at an example. Let us have the following class:

class User { #name; constructor(name) { this.#name = name; } getName() { return this.#name; } }

Let's say we decided to make a class that will manipulate a set of objects with users:

class UsersCollection { }

We will store objects with users as an array in a private property:

class UsersCollection { #users; constructor() { this.#users = []; } }

Let's create a method for adding a new user to the array:

class UsersCollection { #users; constructor() { this.#users = []; } add(user) { this.#users.push(user); } }

Now let’s create a method that will display the names of all users in the console:

class UsersCollection { #users; constructor() { this.#users = []; } add(user) { this.#users.push(user); } show() { for (let user of this.#users) { console.log(user.getName()); } } }

Let's take a look at our class's work. First, let's create its object:

let uc = new UsersCollection;

Now let's add several users to our collection:

uc.add(new User('john')); uc.add(new User('eric')); uc.add(new User('kyle'));

Now let’s call a method that will display the names of all users in the console:

uc.show();
