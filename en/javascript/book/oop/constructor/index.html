Constructor in OOP in JavaScript

There is a special method that will be called when a new class object is created. This method is calledconstructor. Let's do it in our class:

class User { constructor() { console.log('+++'); } }

Let's now create a class object and make sure that the method is called:

new User; // выведет '+++'
