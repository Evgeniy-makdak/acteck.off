Method source code in OOP in JavaScript

You can get the source code of a class method by accessing it without the parentheses. Let's try it. Let us have the following class:

class User { #name; constructor(name) { this.#name = name; } getName() { return this.#name; } }

Let's create an object of this class:

let user = new User('john');

Let's display the source code of the method:

console.log(user.getName);
