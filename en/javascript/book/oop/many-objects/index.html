Несколько объектов одного класса в ООП в JavaScript

Можно создавать несколько объектов одного класса. К примеру, создадим двух юзеров:

let user1 = new User; let user2 = new User;

Дадим им имена:

user1.name = 'john'; user2.name = 'eric';

Выведем эти имена:

console.log(user1.name); console.log(user2.name);
