Introduction to Classes and Objects in JavaScript

Now we will study OOP in JavaScript. Let's look at a real-life example and then transfer it to JavaScript.

Let's take a car as an example. It has wheels, color, body style, engine size, and so on. In addition, the driver can give him commands: drive, stop, turn right, left, etc.

It can be said that there is someClasscars that have common properties (they all have wheels and commands can be given to all of them).

A particular car standing on the street is a representative of this class, or, in other words,objectthis class. All objects of this class haveproperties: number of wheels, color, body type andmethods: drive, stop, turn right, left.

In other words, the class itself is a drawing according to which cars are made at the factory. The object is the machine itself, made according to these drawings.

In JavaScript, a class is created using a keywordclassfollowed by the name of that class. Let's make a classCar:

class Car { // тут код, то есть JavaScript-чертеж автомобиля }

Let us now indicate in our drawing that any car created according to this drawing will have a property for color and a property for the amount of fuel.

To do this, we will write a property inside the classcolorand propertyfuel:

class Car { color; // цвет автомобиля fuel; // количество топлива }

Let's now make the methods of our class. In JavaScript, methods are similar to regular functions, only they are declared without a keyword.function.

As mentioned above, our car can drive, can turn, and can stop. Let's create the corresponding methods in our class:

class Car { color; // цвет автомобиля fuel; // количество топлива // Команда ехать: go() { // какой-то JavaScript код } // Команда поворачивать: turn() { // какой-то JavaScript код } // Команда остановиться: stop() { // какой-то JavaScript код } }

We made a drawing of our car. Now you need to go to the factory and make an object of this class (that is, a specific car).

In JavaScript this is done using a keywordnewfollowed by the class name:

new Car; // командуем заводу сделать автомобиль

However, if you simply create an object of a class, it will lead to nothing (it’s the same as, for example, declaring an array and not writing it anywhere). We need a variable to store this object.

Let this variable be calledmyCar- write the object we created into it:

let myCar = new Car;

After creating a car, you can access its properties. They are addressed through a dot. Let's set the properties of our object:

let myCar = new Car; // командуем заводу сделать автомобиль myCar.color = 'red'; // красим в красный цвет myCar.fuel = 50; // заливаем топливо

That's it, our car is created, painted and fueled. Now we can give it commands through the methods of this car.

Methods are also accessed through a dot, but, unlike a property, parentheses should be written after the method name. Let's command our object:

myCar.go(); myCar.turn(); myCar.stop();
