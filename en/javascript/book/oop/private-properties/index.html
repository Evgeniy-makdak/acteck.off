Private properties in OOP in JavaScript

Properties of an object that can be read and written externally are calledpublic. There are alsoprivateproperties that will be available only within the class.

Private property names must begin with the symbol#. In addition, such properties must be declared at the beginning of the class code. Let's do this:

class User { #name; }

Now let's write the data to our property. This can be done, for example, in the class constructor:

class User { #name; constructor(name) { this.#name = name; } }

Let's now create a method that will return the value of our property:

class User { #name; constructor(name) { this.#name = name; } show() { return this.#name; } }

Let's create a class object by passing the user name as a parameter:

let user = new User('john');

Trying to access our property directly outside of the class will result in an error:

console.log(user.#name); // ошибка

And calling our method will allow us to read this property:

console.log(user.show()); // выведет 'john'
