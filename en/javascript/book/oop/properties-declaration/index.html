Declaring properties inside classes in OOP in JavaScript

At the beginning of a class, you can declare what properties this class will have. Let's, for example, declare that the user will have the propertyname:

class User { name; show() { return this.name; } }

You can immediately write some value into it:

class User { name = 'john'; show() { return this.name; } }

Let's now create an object of our class:

let user = new User;

Now let’s display the initially specified value:

console.log(user.name); // 'john'
