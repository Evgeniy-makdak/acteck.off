Properties of objects in OOP in JavaScript

Let's now learn how to work with object properties. Properties allow you to write to an object and then read some data from it. Let's look at an example. Let us have a class objectUserfrom the previous lesson:

let user = new User;

Let's write some data to the properties of our object:

user.name = 'john'; user.surn = 'smit';

Now let's read these properties:

console.log(user.name); console.log(user.surn);
