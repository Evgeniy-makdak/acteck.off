Prediction site in JavaScript

Now we will make a website that will give predictions. Let there be a button on this site, when clicked, a timer will be started, which will be every0.1seconds, display in some div a random number from1up to some maximum.

Under the div, let there be another button, by clicking on which the user of our site can stop the timer and record a certain number in the div. This number will be the prediction number. After this, show the user the prediction with this number, and remove all unnecessary buttons from the screen so that the user cannot receive another prediction. That is, for one visit to the site - one prediction.
