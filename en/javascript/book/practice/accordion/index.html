Discussion

Technically, the accordion HTML code is more convenient than the tab code: in it, the links are located directly above the spoiler, which they open and close. In addition, classactivein our layout a common parent is given links and spoilers.

This means that when you click on a link, you just need to deactivate the old tab and activate the new one. To deactivate the old tab you need to find the tab with the classactiveand remove this class from it.

To activate a new tab, you need to click on the link to find the parent tab of this link and give it a classactive. Technically, such a search can be done like this: simply find the closest parent of the link that has the classtab.
