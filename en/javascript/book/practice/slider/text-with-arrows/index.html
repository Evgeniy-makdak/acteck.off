Text slider with arrows in JavaScript

Let's now make a text slider with arrows. This means that the text will change not by a timer, but by clicking on the arrow. Let's add arrows to our HTML code:

<a href="" id="left">←</a> <a href="" id="right">→</a> <div id="slider"></div>

The main subtlety of this slider solution is that the text counter variable should be common to the click handlers of our arrows:

let i = 0; // внешняя переменная left.addEventListener('click', function() { // уменьшаем i на 1 // и выводим текст с номером i }); right.addEventListener('click', function() { // увеличиваем i на 1 // и выводим текст с номером i });

And the second subtlety is that, as when reducingi, and when increasing, it is impossible to reach numbers less than zero and greater than the last element of the array.
