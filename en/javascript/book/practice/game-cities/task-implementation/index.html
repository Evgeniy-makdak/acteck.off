Implementation of a city game in JavaScript

Let's discuss how we will implement our project.

I would start by making an array that will record the cities that our players enter:

let cities = [];

Let's say it's the beginning of the game and the first player goes. He hits the city and pressesEnter. Obviously, we can immediately add this city to our array of cities.

Now the second player will move. Since our array is no longer empty, this second move will be described by the same algorithm as all subsequent moves.

What kind of algorithm is this? First, you need to check that the first letter of the entered city matches the last letter of the previous city. Secondly, you need to check that such a city has not yet appeared in this game.

If both of these conditions are met, then the entered city is written to our array and the turn passes to the next player. If some condition is not met, then you need to display a message about this.

Implement the game in accordance with the described algorithm.
