Game of cities in JavaScript

There is a game of cities. Its rules: two or more people play cities, in which each participant, in turn, names a real-life city of any country, the name of which begins with the letter with which the name of the previous participant ends.

Let's implement this game. Let two people play on one computer. They take turns. Cities are entered in one input and sent by pressing a keyEnter. The browser should remember cities that have already existed and not accept such cities.

Let's immediately make a simple layout of our project. We will have an input for entering cities and divs for outputting messages from the browser side:

<input id="field"> <div id="message"></div>

We can immediately get these elements into variables:

let field = document.querySelector('#field'); let message = document.querySelector('#message');
