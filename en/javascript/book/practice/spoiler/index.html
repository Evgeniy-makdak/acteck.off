JavaScript spoiler

Now we will learn how to make spoilers. By spoilers I mean blocks that can be expanded or collapsed using a special link.

The following code example has a paragraph with the classspoileris a spoiler, and the link is with the classtoggleshould show or hide this spoiler on click:

<p> абзац с текстом <a href="" class="toggle">развернуть</a> </p> <p class="spoiler"> скрытый спойлер </p> <p> абзац с текстом </p>

The text, of course, may contain many spoilers and each should have its own link:

<p> абзац с текстом </p> <p> абзац с текстом <a href="" class="toggle">развернуть спойлер 1</a> </p> <p class="spoiler"> скрытый спойлер 1 </p> <p> абзац с текстом </p> <p> абзац с текстом <a href="" class="toggle">развернуть спойлер 2</a> </p> <p class="spoiler"> скрытый спойлер 2 </p> <p> абзац с текстом </p>

Obviously, the link for expanding and the spoiler itself must be somehow connected. You can come up with various options, but in this case I suggest doing it by location: let's say that a link with a classtoggleopens or closes an element with a classspoiler, located immediately below the parent of this link.

Let the spoiler be hidden by default, and if we want to show it, we will give it a classactive. Let's write the corresponding CSS code:

.spoiler:not(.active) { display: none; }

Copy the given HTML and CSS codes for yourself. Implement the described operation of spoilers.
