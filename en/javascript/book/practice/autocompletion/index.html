Autocompletion in JavaScript

Now we are implementing autocompletion. This term refers to the drop-down hint when entering text into the input. Let's look at an example. Below I have made an input in which you can enter the name of the country. At the same time, if you enter some letters, a list of countries that begin with the entered line will appear under the input.

If you wish, you can, in order not to enter the entire country name, click on any country and it will appear in the input. For this, in general, autocompletion is needed. In the example, for simplicity, I made only three countries: Belarus, Belgium and Bulgaria. First enter the symbol in the input below'В'English and then'e'and see what happens:

Discussion

Let's discuss how to solve the problem. We need to make an array with the names of countries. I have it like this:

let arr = ['Belarus', 'Belgium', 'Bulgaria'];

When entering text into the input, you need to iterate through the array with countries after entering each character and get countries that begin with the entered line. This can be conveniently done using the methodsfilterAndstartsWith.

By usingfilteryou can get an array of countries starting with the entered string. Then you need to loop through this array and fill the listulsurplus with countries. And this should be done for each character input, first removing previously created excesses from the trick.
