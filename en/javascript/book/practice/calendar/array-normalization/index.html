Normalizing an array in a JavaScript calendar

In previous lessons you had to make an array of numbers from1until the last day of the month:

let arr = range(getLastDay(year, month));

Also you have the day of the week number for the first day of the month and for the last:

let firstWeekDay = getFirstWeekDay(year, month); let lastWeekDay = getLastWeekDay(year, month);

Let's now pad our array with empty strings on the right and left. Let us have a function for thisnormalize, the first parameter is an array, the second is how many empty lines to add on the left, and the third is how many empty lines on the right:

function normalize(arr, left, right) { }

I remind you that on the left we must addfirstWeekDayempty elements, and on the right -6minuslastWeekDayelements. That is, we will use our functionnormalizelike this:

let res = normalize(arr, firstWeekDay, 6 - lastWeekDay); console.log(res);
