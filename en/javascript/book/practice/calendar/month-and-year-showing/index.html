Displaying the month and year in a calendar in JavaScript

In previous lessons, we made the calendar that was the sample in the first lesson. Let's improve it now.

First, let's make sure the current month and year are shown above the calendar. Let's make the appropriate changes to the HTML code:

<div id="parent"> <div id="calendar"> <div class="info">Янв 2020</div> <table> <thead> <tr> <th>пн</th> <th>вт</th> <th>ср</th> <th>чт</th> <th>пт</th> <th>сб</th> <th>вс</th> </tr> </thead> <tbody class="body"></tbody> </table> </div> </div>

Let's also make changes to the CSS code:

#parent { text-align: center; } #calendar { display: inline-block; } #calendar td, #calendar th { padding: 10px; border: 1px solid black; text-align: center; } #calendar .info { text-align: center; }
