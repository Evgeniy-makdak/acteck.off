Arrows to change the month in a JavaScript calendar

Let's now make arrows under the calendar to change the month. Let's make changes to the HTML code:

<div id="parent"> <div id="calendar"> <div class="info">Янв 2020</div> <table> <thead> <tr> <th>пн</th> <th>вт</th> <th>ср</th> <th>чт</th> <th>пт</th> <th>сб</th> <th>вс</th> </tr> </thead> <tbody class="body"></tbody> </table> <div class="nav"> <a href="#" class="prev">←</a> <a href="#" class="next">→</a> </div> </div> </div>

Let's make changes to the CSS code:

#parent { text-align: center; } #calendar { display: inline-block; } #calendar td, #calendar th { padding: 10px; border: 1px solid black; text-align: center; } #calendar .nav, #calendar .info { text-align: center; } #calendar a { color: blue; font-size: 25px; text-decoration: none; } #calendar a:hover { color: red; } #calendar .active { color: red; }

Let's get references to arrows in variables:

let prev = calendar.querySelector('.prev'); let next = calendar.querySelector('.next');

Discussion

Let's now discuss how to solve the problem. In fact, our calendar code is written to be highly extensible: we have a functiondraw, which draws a calendar for a given month. Thus, when we click on the arrow, we can redraw the calendar with a different month (and possibly year) number.

Here's an example of what we can do by clicking on the next month arrow:

next.addEventListener('click', function() { draw(body, getNextYear(year, month), getNextMonth(month)); });

As you can see, the functions are used heregetNextYearAndgetNextMonth. Let's discuss what they should do.

As the name suggests, the functiongetNextYeargets next year. What this means is that if it is now December, then when switching to the next month the year should increase by1. If it is not December, then the function should simply return the current year.

FunctiongetNextMonthshould receive next month. For all months except December, it will add one to the month number. Well, for December the next month will be January with the number0.

Similar operations should be performed when clicking on the button for the previous month:

prev.addEventListener('click', function() { draw(body, getPrevYear(year, month), getPrevMonth(month)); });
