Discussion of implementing a calendar in JavaScript

Our calendar is a table with numbers from1until the last day of the month. The problem is that the numbers in the table in the first row should not come from the first cell.

This is generally obvious. It is not obvious which cell to start with. Let's think about it.

Let the week start on Monday and it is day zero. In this case, the number of the day of the week on the first day of the month corresponds to the number of empty cells before it. For example, Thursday would have the number3and in front of him will be3empty cells.

A similar problem will await us at the end of the table. There you also need to leave empty cells after the last day of the month. In this case, the number of empty cells will be equal to six, from which you need to subtract the number of the day of the week of the last day of the month.

Let's now think about how we will form our table. Obviously, it is most convenient to have a two-dimensional array, each subarray of which will contain7elements (= number of days in a week). In this case, some of the elements in the first and last subarrays must be empty.

Algorithm of actions

Let's make a function that will make an array of numbers from1until the last day of the month. Then we will make a function that will add the required number of empty elements to the beginning of the array. A similar function should add empty elements to the end of the array.

After this, we will have a one-dimensional array with the number of elements a multiple of7. Let's transform this array into a two-dimensional one7elements in the subarray.

Then we take this two-dimensional array, loop through it and form our table.
