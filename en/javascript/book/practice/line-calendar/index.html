Linear calendar in JavaScript

Output as a listulall dates of the current month. Use CSS to arrange list items in a row.

Make sure that the current day on the calendar is highlighted in some color.

Make sure that the name of the current month in Russian and the year number are written above the list.

Make sure there are forward and backward links above the calendar so you can change the month. The month and year displayed above the calendar must match the month displayed.
