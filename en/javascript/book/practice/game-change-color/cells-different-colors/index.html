Coloring cells in different colors

The next logical step would be to paint our cells in random colors. According to the conditions of the problem, we have a certain limited set of colors, let’s say three: red, green, blue.

If you think about it, it’s convenient to make these colors CSS classes. We will give these classes to the cells of our table. Let's make them:

.red { background: red; } .green { background: green; } .blue { background: blue; }

In JavaScript code, it is convenient to collect our classes into an array:

let colors = ['red', 'green', 'blue'];

Having such an array, we can solve our problem: at the moment of creating a cell, we will assign CSS to it, randomly selecting it from our array.

To do this, it is better to make some kind of auxiliary function that will take an array as a parameter and return its random element.

Implement the described function. Test her work.

Using the function you created, make the cells appear in random colors when creating a table.
