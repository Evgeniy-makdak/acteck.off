Changing cell color

The next logical step would be to make sure that when you click on a cell, this cell changes its color.

According to the game description, the colors should change in order. That is, technically in our case, in the order in which they are written in our array of colors:

let colors = ['red', 'green', 'blue'];

This means that when we click on a cell, we must read its color, find this color in the array of colors, then get the next color from the array and set it to the color of our cell.

In this case, it is convenient to have a function that will take an array and an element of this array as a parameter and return the next element. It is clear that everything should go in a circle: if we pass the last element of the array to a function, then it should return the first as its result.

Implement the described function. Test her work.

Using the function you created, make each cell change color to the next one when you click on it.
