Winning the game of changing colors using JavaScript

In the previous lessons, we basically implemented the entire gameplay. All that remains is to make sure that the game determines the moment of victory.

Having thought about the question, we can come to the conclusion that the moment of victory is when all the cells have the same color. That is, technically - the same class.

This means that at the moment of clicking on any cell, we need to take the class of any cell (the one we clicked on will do) and see if all other cells have this class.

If yes, then that's it, victory. If not, we continue to play.
