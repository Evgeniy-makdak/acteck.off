Тест с вариантами ответов в массиве на JavaScript

Пусть теперь у нас есть массив, в котором написаны номера правильных ответов для каждого вопроса:

let answers = [0, 1, 2];

To есть в первом элементе этого массива лежит номер правильного ответа первого вопроса, во втором элементе - для второго вопроса и так далее.
