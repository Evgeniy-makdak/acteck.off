Test with questions and answers in HTML code using JavaScript

In this and the next few lessons we will create tests with questions and answers. Let's start with the simplest options and gradually increase complexity.

Let us first want to make a set of questions, each of which has only one possible answer. Let the user enter the answers into the inputs under the questions themselves.

I will illustrate what was written with HTML code:

<div id="test"> <div> <p>вопрос 1?</p> <input> </div> <div> <p>вопрос 2?</p> <input> </div> <div> <p>вопрос 3?</p> <input> </div> </div>

As you can see, we have questions and inputs to answer. All we have to do is figure out where we will store the correct answers.

The simplest thing you can think of is to store the responses directly in the HTML input code, in somedata-attribute:

<div id="test"> <div> <p>вопрос 1?</p> <input data-right="ответ 1"> </div> <div> <p>вопрос 2?</p> <input data-right="ответ 2"> </div> <div> <p>вопрос 3?</p> <input data-right="ответ 3"> </div> </div>

The next thing to consider is: at what point will we check the answers for correctness? Here we can offer two options: either let each input check whether the answer is correct immediately upon entering an answer, or let there be a button that, when clicked, will check the answers to all questions at once.

Whatever option we choose, if the answer is correct, then the input border will be painted green, and if the answer is incorrect, red. Let's create the corresponding CSS classes for this:

.right { border: 1px solid green; } .wrong { border: 1px solid red; }

Make sure that entering a response in the input ends with a key pressEnter. In this case, let the input immediately check the answer for correctness.

Modify your code so that responses are checked when a button is clicked.
