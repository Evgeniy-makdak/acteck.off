Тест с ответами в массиве на JavaScript

Пусть теперь у нас дан массив правильных ответов:

let answers = [ 'ответ 1', 'ответ 2', 'ответ 3', ];

Это значит, что мы теперь не будем хранить ответы в data- атрибутах инпутов:

<div id="test"> <div> <p>вопрос 1?</p> <input> </div> <div> <p>вопрос 2?</p> <input> </div> <div> <p>вопрос 3?</p> <input> </div> <button id="button">проверить ответы</button> </div>

Сделайте так, чтобы по нажатию на кнопку выполнилась проверка правильности ответов.
