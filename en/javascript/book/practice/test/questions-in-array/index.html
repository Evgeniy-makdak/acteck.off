Test with answers in an array in JavaScript

Let us now have an array of correct answers:

let answers = [ 'ответ 1', 'ответ 2', 'ответ 3', ];

This means that we will no longer store responses in the data attributes of inputs:

<div id="test"> <div> <p>вопрос 1?</p> <input> </div> <div> <p>вопрос 2?</p> <input> </div> <div> <p>вопрос 3?</p> <input> </div> <button id="button">проверить ответы</button> </div>

Make sure that when you click a button, the answers are checked for correctness.
