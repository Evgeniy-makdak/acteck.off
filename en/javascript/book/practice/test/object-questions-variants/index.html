Test with questions and answer options in a JavaScript object

Let now questions, answer options and numbers of correct answers be stored in one object:

let questions = [ { text: 'вопрос 1?', right: 0, variants: [ 'вариант 1', 'вариант 2', 'вариант 3' ] }, { text: 'вопрос 2?', right: 1, variants: [ 'вариант 1', 'вариант 2', 'вариант 3' ] }, { text: 'вопрос 3?', right: 2, variants: [ 'вариант 1', 'вариант 2', 'вариант 3' ] }, ];

Our HTML code will then be reduced to the following:

<div id="test"></div> <button id="button">проверить ответы</button>

Now, upon entering the page, you will have to loop through our object and generate the HTML code for the questions.

Complete the task.
