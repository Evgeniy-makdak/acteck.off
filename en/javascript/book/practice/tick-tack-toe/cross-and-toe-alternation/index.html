Alternating cross and zero

After solving the problem in the previous lesson, you should have the following code:

let cells = document.querySelectorAll('#field td'); start(cells); function start(cells) { for (let cell of cells) { cell.addEventListener('click', function() { this.textContent = 'X'; }); } }

Let's now make alternating crosses and toes. To do this, I suggest introducing a move counter:

function start(cells) { let i = 0; // начальное значение счетчика for (let cell of cells) { cell.addEventListener('click', function() { this.textContent = 'X'; i++; // увеличиваем счетчик }); } }

Having such a counter, we can easily implement the alternation of a cross and a zero: obviously, a cross will appear on even counter values, and a zero on odd ones.
