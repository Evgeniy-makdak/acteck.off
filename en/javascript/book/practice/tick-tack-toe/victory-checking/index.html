Checking victory in the game tic-tac-toe in JavaScript

Now it's time to write the code that will determine the victory and display the name of the winner.

Having thought about the problem, you can realize that the check for victory should be performed every turn:

function init(selector) { let cells = document.querySelectorAll('#field td'); let i = 0; for (let cell of cells) { cell.addEventListener('click', function step() { this.textContent = ['X', 'O'][i % 2]; this.removeEventListener('click', step); // здесь мы должны проверять победу или ничью i++; }); } }

Let's make a functionisVictory, which will take a cell array as a parameter and returntrue, if there is a victory on the field, andfalseif not. We will use this function as follows:

function start(cells) { let i = 0; for (let cell of cells) { cell.addEventListener('click', function step() { this.textContent = ['X', 'O'][i % 2]; this.removeEventListener('click', step); if (isVictory(cells)) { alert('имя победителя'); } i++; }); } }

How can we find out the name of the winner? Obviously, his name is stored in each of the winning cells. Let me explain: if we have a victory, it means that some3cells horizontally, vertically or diagonally contain the same value: either a cross or a zero.

Obviously, the value of any of these cells is the name of the winner. But you can also go another, less obvious way: since we determine the presence of victory every move, it is logical that the winner will be the player who made the last move. The name of this player is contained in the text of the clicked cell:

function start(cells) { let i = 0; for (let cell of cells) { cell.addEventListener('click', function step() { this.textContent = ['X', 'O'][i % 2]; this.removeEventListener('click', step); if (isVictory(cells)) { alert(this.textContent); // выводим имя победителя } i++; }); } }

Implementation of isVictory

How can we technically determine the presence of a victory on the field? If you think about it carefully, you can come to the conclusion that there is a limited number of triples of cells on the field.

That is, to check for victory, you can simply check all these threes. It is necessary to check that the triples have the same non-empty values ​​(that is, there is either a cross or a zero).

Let's make a two-dimensional array, each subarray of which will contain the cell numbers of one of the triples:

let combs = [ [0, 1, 2], [3, 4, 5], [6, 7, 8], [0, 3, 6], [1, 4, 7], [2, 5, 8], [0, 4, 8], [2, 4, 6], ];

Let's write the definition of victory using this array:

function isVictory(cells) { let combs = [ [0, 1, 2], [3, 4, 5], [6, 7, 8], [0, 3, 6], [1, 4, 7], [2, 5, 8], [0, 4, 8], [2, 4, 6], ]; for (let comb of combs) { if ( cells[comb[0]].textContent == cells[comb[1]].textContent && cells[comb[1]].textContent == cells[comb[2]].textContent && cells[comb[0]].textContent != '' ) { return true; } } return false; }
