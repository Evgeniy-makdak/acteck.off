Removing purchases in the grocery calculator in JavaScript

Let's now implement the deletion of the purchase. We already have a link to delete, we create it with this line:

createCell(tr, 'удалить', 'remove');

I remind you that the functioncreateCellreturns the created cell. This means that various operations can be performed with this cell. For example, let's bind a click handler to it:

let remove = createCell(tr, 'удалить', 'remove'); remove.addEventListener('click', function() { // удалим ряд таблицы });

You don’t have to enter an intermediate variable:

createCell(tr, 'удалить', 'remove').addEventListener('click', function() { // удалим ряд таблицы });
