Product calculator in JavaScript

In this section we implement a product calculator. It will be a table into which the user of our site will make his purchases.

Let purchases be made using the form. In addition, we will provide a deletion link for each product. We will also make it possible to edit the name, price and quantity of the product. Let such editing occur by double clicking on a table cell.

Let the total cost of products be displayed below the table. Let's make sure that this amount is recalculated when deleting and editing products.
