Total amount in JavaScript product calculator

Let's now make sure that the total amount of purchases is displayed below the table. There are nuances here.

If you think about the situation, it becomes clear that the amount should be adjusted not only when adding a new purchase, but also when deleting and editing.

In this case, I would suggest making some function, let's call itrecountTotal, which will perform a complete recalculation of the amount. That is, it will loop through all purchases and summarize the contents of their columncost.

Having such a function, we can call it in any place where some changes in the total amount are expected. Of course, it is a little less optimal to calculate the total amount every time. But we will get a significant simplification of the code.

We use the intended function when creating a new purchase:

add.addEventListener('click', function() { let tr = document.createElement('tr'); createCell(tr, name.value, 'name'); createCell(tr, price.value, 'price'); createCell(tr, amount.value, 'amount'); createCell(tr, price.value * amount.value, 'cost'); createCell(tr, 'удалить', 'remove'); table.appendChild(tr); recountTotal(); // пересчитаем общую сумму });
