Drawing circles using canvas in JavaScript

Next methodarcdraws an arc with its center at a certain point. It accepts the following parameters:x, y, radiusr, starting anglestartAngle, final angleendAngle, draw clockwise or counterclockwisedirection.

Parameterdirectiontakes the following values:truemakes you draw clockwise,falsecounterclockwise (default).

In this case, the angles in the methodarcmeasured inradians, not in degrees. To convert degrees to radians you can use the following function:

function getRadians(degrees) { return (Math.PI / 180) * degrees; }

Let's draw a circle

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.arc(100, 100, 75, 0, getRadians(360)); ctx.stroke();
