Drawing rectangles using canvas in JavaScript

Squares and rectangles do not have to be drawn using combinationsmoveTo, lineTo- There are simpler methods for this. Let's look at them.

strokeRect method

MethodstrokeRect(x, y, width, height) draws the outline of a rectangle at a given point. The first two parameters specify the coordinates of the point where the upper left corner of the drawn rectangle will be.

Let's draw a rectangle usingstrokeRect(we believe thatctxalready exist):

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.strokeRect(50, 50, 100, 75);

:

fillRect method

MethodfillRect(x, y, width, height) works the same way asstrokeRect, only draws a filled rectangle. Let's look at an example:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.fillRect(50, 50, 100, 75);

:

rect method

Next methodrect(x, y, width, height) also draws a rectangle. But this rectangle will become visible only if you apply the methodstrokeorfill. In the first case there will be an outline, and in the second there will be a figure.

Let's draw the outline usingrect:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.rect(50, 50, 100, 75); ctx.stroke();

:

Now let's draw a figure:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.rect(50, 50, 100, 75); ctx.fill();

:

clearRect method

Next methodclearRect(x, y, width, height) works like an eraser, clearing a rectangular area and making the contents completely transparent.

Let's draw a square usingfillRectand erase part of it withclearRect:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.fillRect(50, 50, 100, 100); ctx.clearRect(75, 75, 50, 50);
