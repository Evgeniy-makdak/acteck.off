Getting started with canvas in JavaScript

Canvas is a special field for drawing in the browser using JavaScript. This field is made using the tag<canvas>, which should be given width and height:

<canvas width="200" height="200"></canvas>

To work with this field in JavaScript, we first get a link to this tag in some variable:

let canvas = document.querySelector('canvas');

Then you need to apply the method to this variablegetContext:

let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d');

You can simplify:

let ctx = document.querySelector('canvas').getContext('2d');

As a result, into a variablectxan object containing the so-calledexecution context. All drawing will happen using the methods of this object. We will study them in the following lessons.
