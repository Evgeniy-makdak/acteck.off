Drawing lines via canvas in JavaScript

Let's start with something simple - by drawing lines. To understand how this is done, imagine that you have a piece of paper and a pencil. You can draw on this sheet, or you can simply move the pencil to the desired point without drawing.

JavaScript works the same way: it has a pencil (or pen for drawing), you can move it to the desired location using the methodmoveTo, and draw using the methodlineTo.

Both methods take as parameters the point to which to move the tip of the pen - the first parameter is the horizontal coordinate, and the second is the vertical coordinate. Origin - point0, 0is the top left corner of the element.

When the pen moves to the desired point - methodmoveTojust moves itlineTowhile moving, draws a line from the current position of the pen to the point to which it will move.

However, if you run the above code, nothing will happen: you need to write two more commands: the first commandbeginPathmust be specified before starting to draw the line, and the second commandstrokemust be called after all combinationsmoveToAndlineTofor the line to be drawn.

So let's write some working code to draw a line:

Let's draw a line

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.stroke();

Closing shapes

To draw a closed figure, it is not necessary to draw all the lines - the last line can be drawn automatically and close the figure. To do this, before the methodstrokemethod should be calledclosePath.

This method attempts to close the shape by drawing a straight line from the end point to the start point. If the figure has already been closed or is just a point, then the method does nothing.

Let's draw a triangle usingclosePath:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.moveTo(50, 50); ctx.lineTo(100, 100); ctx.lineTo(150, 50); //ctx.lineTo(50, 50); - этот шаг опускаем ctx.closePath(); ctx.stroke();
