Separately catching exceptions in promises in JavaScript

INthenYou can specify only the exception handler function by passing instead of the first parameternull:

promise.then( null, function(error) { console.log(error); } );

In this case, it is convenient to use the shortened syntax through the methodcatch:

promise.catch( function(error) { console.log(error); } );

Rewrite the following code through the methodcatch:

let promise = new Promise(function(resolve, reject) { setTimeout(function() { let isError = false; if (!isError) { resolve('success'); } else { reject(new Error('error')); } }, 3000); }); promise.then( res => console.log(res), err => console.log(err.message), );
