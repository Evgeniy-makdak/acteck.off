Object with Promise Error in JavaScript

More accepted into functionrejectpass not a string with an error, but an object with an error:

let promise = new Promise(function(resolve, reject) { setTimeout(function() { reject(new Error('error in promise')); // объект с ошибкой }, 3000); });

You can also throw away objects with errors usingthrow- this will be equivalent to transferring them toreject:

let promise = new Promise(function(resolve, reject) { setTimeout(function() { throw new Error('error in promise'); // эквивалентно reject }, 3000); });

Modify the following code according to what you learned:

let promise = new Promise(function(resolve, reject) { setTimeout(function() { let isError = false; if (!isError) { resolve('success'); } else { reject('error'); } }, 3000); });
