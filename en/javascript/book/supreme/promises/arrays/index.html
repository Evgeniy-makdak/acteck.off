Working with Arrays of Promises in JavaScript

Now we will look at two useful methods that allow you to work with arrays of promises. MethodPromise.allallows you to execute the code upon completion of all promises passed to it in the form of an array, and the methodPromise.racewaits for the first promise to be loaded from the array, discarding the rest.

Both methods return a new promise as their result. For methodPromise.allthe result of this promise will be an array of results of all passed promises (the order of the results corresponds to the order of the promises in the array), and forPromise.race- the result of the first triggered promise.

Let's see in practice. Let's say we have an array of promises:

let promises = [ new Promise(resolve => setTimeout(() => resolve(1), 1000)), new Promise(resolve => setTimeout(() => resolve(2), 2000)), new Promise(resolve => setTimeout(() => resolve(3), 3000)), ];

Let's usePromise.allLet's wait until all promises from our array are loaded:

Promise.all(promises).then(function(res) { console.log(res); // выведет [1, 2, 3] - результаты всех промисов });

And now with the helpPromise.raceLet's wait for the first of the promises to finish loading:

Promise.race(promises).then(function(res) { console.log(res); // выведет 1 - результат первого выполнившегося промиса });

If at least one of the promises in the array is rejected, then the promise with the result will immediately go into the staterejected. Therefore, the exception that has arisen can be caught in the usual way that you have already learned, for example, throughcatch:

Promise.all(promises).then(function(res) { console.log(res); }).catch(function(err) { console.log(err); });
