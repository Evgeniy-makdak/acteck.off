Creating Triggered Promises in JavaScript

Sometimes you may need to create a promise that has already been fulfilled. There are two methods for this: the methodPromise.resolvecreates a successful promise,Promise.rejectcreates a rejected promise. These methods receive as a parameter what will be the result or error of the promise, respectively.

When might we need a promise that has already been fulfilled? Let's look at an example. Let us have some function that takes a number as a parameter, does something with it asynchronously and returns a promise with the result:

function func(num) { return new Promise(function(resolve) { setTimeout(function() { resolve(num * num); }, 3000); }); }

We will use our function as follows:

func(5).then(function(res) { console.log(res); // выведет 25 });

Let us now decide that we will perform our asynchronous operation only if a number greater than zero is passed. Otherwise, the result of the function must be zero:

function func(num) { if (num > 0) { return new Promise(function(resolve) { setTimeout(function() { resolve(num * num); }, 3000); }); } else { return 0; // нулевой результат } }

However, now it turns out that the function returns either a promise or a number. Because of this we can no longer apply the methodthento the function result, since if the function returns a number, we will receive an error:

func(0).then(function(res) { // ошибка, применяем метод then к нулю });

Help us fix the problemPromise.resolve:

function func(num) { if (num > 0) { return new Promise(function(resolve) { setTimeout(function() { resolve(num * num); }, 3000); }); } else { return Promise.resolve(0); // возвращаем промис, а не число } }

Let us now decide that for the passed zero we must return zero, and for numbers less than zero we must return an exception. The method will help us with thisPromise.reject:

function func(num) { if (num > 0) { return new Promise(function(resolve) { setTimeout(function() { resolve(num * num); }, 3000); }); } else if (num === 0) { return Promise.resolve(0); } else { return Promise.reject('incorrect number'); // вернем отклоненный промис } }
