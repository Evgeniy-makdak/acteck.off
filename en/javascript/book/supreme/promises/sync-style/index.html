Synchronous Style Promises in JavaScript

To solve the promise hell problem, a special promise syntax was invented, which allows you to write asynchronous code much easier - in a synchronous style.

Let's start studying it. First, let's take the code from the previous lesson:

function func() { getSmth(2).then(res => { console.log(res); // выведет 4 }); } func();

If our functiongetSmthwas synchronous, then the function codefuncwe could rewrite it like this:

function func() { let res = getSmth(2); console.log(res); // выведет 4 }

FunctiongetSmthhowever, is asynchronous, so the above code will not work. But by using the synchronous promise style, we can get something similar. Let's do this.

First we need to declare our functionfuncasynchronous using a special commandasync:

async function func() { }

After this we can inside the functionfuncuse special commandawait. This command, written before a promise, will cause JavaScript to wait until the promise is fulfilled. After which the command will return the result of the promise, and code execution will continue.

In our case we have to writeawaitbefore the callgetSmth. Since the result of calling this function will be a promise, further code execution will continue only after this promise has been fulfilled. Well, the result of the promise can be written into a variable. Let's do the following:

async function func() { let res = await getSmth(2); console.log(res); // выведет 4 } func();

Let's callgetSmthseveral times:

async function func() { let res1 = await getSmth(2); let res2 = await getSmth(3); console.log(res1 + res2); // выведет 13 } func();

Now let's callgetSmthin a loop:

async function func() { let arr = [1, 2, 3, 4, 5]; let sum = 0; for (let elem of arr) { sum += await getSmth(elem); } console.log(sum); } func();

Rewrite the following code using synchronous syntax:

function func() { getSmth(2).then(res1 => { getSmth(3).then(res2 => { getSmth(4).then(res3 => { console.log(res1 + res2 + res3); }); }); }); } func();
