'or' command in JavaScript regular expressions

In this lesson we will look at the command'|', which is a more powerful option'или'Compared to the [ ] command. This command allows you to divide the regular sequence into several parts. In this case, what you are looking for can fall either under one part of the regular sequence or under another. Let's look at examples.

Example

In this example, the search template is:three letters'a'or three letters'b':

let str = 'aaa bbb abb'; let res = str.replace(/a{3}|b{3}/g, '!');

As a result, the following will be written to the variable:

'! ! abb'

Example

In this example, the search template is:three letters'a'or from1and more letters'b':

let str = 'aaa bbb bbbb bbbbb axx'; let res = str.replace(/a{3}|b+/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! axx'

Example

In this example, the search template is:one or more letters or three numbers:

let str = 'a ab abc 1 12 123'; let res = str.replace(/[a-z]+|\d{3}/g, '!');

As a result, the following will be written to the variable:

'! ! ! 1 12 !'

Example

A vertical bar can divide a regular expression not into two parts, but into any number of parts:

let str = 'aaa bbb ccc ddd'; let res = str.replace(/a+|b+|c+/g, '!');

As a result, the following will be written to the variable:

'! ! ! ddd'

Example

If the vertical bar is inside parentheses, then'или'only works inside these brackets.

For example, let's find strings using the following pattern:stands at the beginning or'a', or'b'one or more times, and then two letters'x':

let str = 'axx bxx bbxx exx'; let res = str.replace(/(a|b+)xx/g, '!');

As a result, the following will be written to the variable:

'! ! ! exx'

Practical problems

let str = 'aeeea aeea aea axa axxa axxxa';

let str = 'aeeea aeea aea axa axxa axxxa';

Write a regular expression that finds strings using the pattern:there are letters on the edges'a', and between them - or a letter'e'twice or a letter'x'any number of times.
