Ignoring case in JavaScript regular expressions

Using a modifieriYou can ignore case. Let's see how it's done.

Example

In the following example, the regular engine will only find small letters:

let str = 'aaa bbb CCC DDD'; let res = str.replace(/[a-z]+/g, '!');

As a result, the following will be written to the variable:

'! ! CCC DDD'

Example

Now let's add a modifieriand the regular engine will start searching for characters in all registers:

let str = 'aaa AAA bbb BBB'; let res = str.replace(/[a-z]+/ig, '!');

As a result, the following will be written to the variable:

'! ! ! !'

Practical problems

Simplify the code using the modifier we learned:

let res = str.replace(/[a-zA-Z]/g, '!');
