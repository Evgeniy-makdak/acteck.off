search method in JavaScript regular expressions

In this lesson we will look at the methodsearch, which searches a string using a regular expression. It accepts the following parameters:

строка.search(регулярка)

The method returns the position of the first found substring, and if it is not found, then-1.

In the following example we find the position of a substring'aaa'in our line:

let str = 'a aa aaa aaaa aaaa'; let res = str.search(/aaa/);

As a result, the following will be written to the variable:

5
