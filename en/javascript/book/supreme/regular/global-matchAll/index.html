Global matchAll in JavaScript regex

MethodmatchAllreturns all matches asiterable object, each element of which contains an array of the found one and its pockets. The method can only be called with a modifierg.

Let's look at an example. Let's say we have the following line:

let str = '12 34 56';

Let's find all pairs of numbers and divide their digits into pockets:

let matches = str.matchAll(/(\d)(\d)/g);

Let's loop through the resulting iterable object and print the matches found:

for (let match of matches) { console.log(match); // [12, 1, 2], [34, 3, 4], [56, 5, 6] }

Given the following line:

let str = '12:37 15:48 17:59';

Find all the substrings with time in it and for each one you find, put the hours and minutes in your pockets.

Given the line:

let str = 'site.ru sss site.com zzz site.net';

Get an array of domain names from this string by putting the domain name and its zone in different pockets in this array.
