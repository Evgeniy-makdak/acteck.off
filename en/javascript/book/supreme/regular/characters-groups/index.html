Character groups in JavaScript regular expressions

There are special commands that allow you to select entire groups of characters at once. Team\dmeans the number from0to9. Team\wdenotes a number, a Latin letter, or an underscore. Team\sdenotes a space or whitespace character: space, newline, tab. You can invert the meaning of a command by capitalizing it: for example, if\d- number, then\D- not a number.

Example

Let's find all the numbers:

let str = '1 12 123'; let res = str.replace(/\d/g, '!');

As a result, the following will be written to the variable:

'! !! !!!'

Example

Repeat operators treat group commands as one unit, meaning grouping parentheses are not needed. In the following example, the search pattern looks like this:figure from0to9one or more times:

let str = '1 12 123 abc @@@'; let res = str.replace(/\d+/g, '!');

As a result, the following will be written to the variable:

'! ! ! abc @@@'

Example

In the following example, the search pattern looks like this:anything one or more times, but not the number from0until 9:

let str = '123abc3@@'; let res = str.replace(/\D+/g, '!');

As a result, the following will be written to the variable:

'123!3!'

Example

In this example, the search template looks like this:space character once:

let str = '1 12 123 abc @@@'; let res = str.replace(/\s/g, '!');

As a result, the following will be written to the variable:

'1!12!123!abc!@@@'

Example

In this example, the search template looks like this:NOT whitespace one or more times. All substrings separated by spaces will be replaced with'!':

let str = '1 12 123 abc @@@'; let res = str.replace(/\S+/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! !'

Example

In this example, the search template looks like this:number or letter one or more times. All substrings consisting of numbers and letters will be replaced with'!':

let str = '1 12 123a Abc @@@'; let res = str.replace(/\w+/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! @@@'

Example

In this example, the search template looks like this:NOT a number and NOT a letter one or more times. This definition in our case includes'@@@'and all the spaces (they are not numbers or letters either). Please note that at the end one'!'- the string was converted into it' @@@'- with a space in front:

let str = '1 12 123 Abc @@@'; let res = str.replace(/\W+/g, '!');

As a result, the following will be written to the variable:

'1!12!123!Abc!'

Practical problems

Given the line:

let str = 'a1a a2a a3a a4a a5a aba aca';

Write a regular expression that will find lines in whichthere are letters on the edges'a', and there is one digit between them.

Given the line:

let str = 'a1a a22a a333a a4444a a55555a aba aca';

Write a regular expression that will find lines in whichthere are letters on the edges'a', and between them any number of digits.

Given the line:

let str = 'aa a1a a22a a333a a4444a a55555a aba aca';

Write a regular expression that will find lines in whichthere are letters on the edges'a', and between them any number of digits (including zero digits, that is, a string'aa').

Given the line:

let str = 'avb a1b a2b a3b a4b a5b abb acb';

Write a regular expression that finds strings like this:there are letters on the edges'a'And'b', and between them is not a number or a space.

Given the line:

let str = 'ave a#b a2b a$b a4b a5b a-b acb';

Write a regular expression that finds strings like this:there are letters on the edges'a'And'b', and between them - not a letter, not a number or a space.

Given the line:

let str = 'ave a#a a2a a$a a4a a5a a-a aca';

Write a regular expression that will replace all spaces with'!'.
