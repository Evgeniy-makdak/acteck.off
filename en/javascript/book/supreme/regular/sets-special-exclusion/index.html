Exceptions within sets in JavaScript regex

You already know that special characters are inside[]become ordinary symbols. There are exceptions, however: if you need square brackets as characters inside[ ]- then they need to be escaped with a backslash. For example, in the following code, the search template looks like this:there is a square bracket between the X's:

let str = 'x]x xax x[x x1x'; let res = str.replace(/x[\[\]]x/g, '!');

As a result, the following will be written to the variable:

'! xax ! x1x'

Given the line:

let str = 'x[]z x[[]]z x()z';

Write a regular search engine that will find all words according to the pattern:letter'x', then square brackets any number of times, then a letter'z'.

Given the line:

let str = 'x[]z x{}z x.z x()z x([])z';

Write a regular search engine that will find all words according to the pattern:letter'x', then any number of any parentheses, then a letter'z'.
