Hyphen inside sets in JavaScript regular expressions

The hyphen is also a special character inside[ ](but outside - no). If you need the hyphen itself as a symbol, then put it where it iswill not be interpreted as a group separator.

Why this is important: You can make a group of symbols without noticing. For example, like this -'[:-@]'- you think you are choosing a colon, a hyphen and a dog, but in fact you get a group of characters between:And@. This group includes the following symbols:? < = > : ;

Where did they come from? From the tableASCII- the colon has a number less than the dog - and the group is obtained. That is, all groups are obtained from the ASCII table (you can use this if you wish).

How to deal with this: place the hyphen character where it will definitely not be perceived as a group character, for example, at the beginning or end (that is, after[or before]).

You can also escape the hyphen - then it will represent itself regardless of position. For example, instead of[:-@]write[:\-@]- and the group will no longer exist, but there will be three characters - a colon, a hyphen and a dog@.

Example

In the following example, the search pattern is:number1, then the letter from'a'to'z', then a number2:

let str = '1a2 1-2 1c2 1z2'; let res = str.replace(/1[a-z]2/g, '!');

As a result, the following will be written to the variable:

'! 1-2 ! !'

Example

Let's now escape the hyphen. The resulting search pattern is:number1, then the letter'a', or hyphen, or letter'z', then a number2:

let str = '1a2 1-2 1c2 1z2'; let res = str.replace(/1[a\-z]2/g, '!');

As a result, the following will be written to the variable:

'! ! 1c2 !'

Example

You can simply rearrange the hyphen without escaping it:

let str = '1a2 1-2 1c2 1z2'; let res = str.replace(/1[az-]2/g, '!');

As a result, the following will be written to the variable:

'! ! 1c2 !'

Example

In the following example, the search pattern is:the first character is small letters or a hyphen'-', then two letters'x':

let str = 'axx Axx -xx @xx'; let res = str.replace(/[a-z-]xx/g, '!');

As a result, the following will be written to the variable:

'! Axx ! @xx'

Example

In the following example, the search pattern is:the first character is small, capital letters or a hyphen'-', then two letters'x':

let str = 'axx Axx -xx @xx'; let res = str.replace(/[a-zA-Z-]xx/g, '!');

As a result, the following will be written to the variable:

'! ! ! @xx'

Example

You can place a hyphen between two groups - there it definitely won’t create another group:

let str = 'axx 9xx -xx @xx'; let res = str.replace(/[a-z-0-9]xx/g, '!');

As a result, the following will be written to the variable:

'! ! ! @xx'

Practical problems

Given the line:

let str = 'xaz xBz xcz x-z x@z';

Find all lines using the following pattern:letter'x', capital or small letter or hyphen, letter'z'.

Given the line:

let str = 'xaz x$z x-z xcz x+z x%z x*z';

Find all lines using the following pattern:letter'x', then either a dollar, or a hyphen or a plus, then a letter'z'.
