The split method in JavaScript regular expressions

In this lesson we will look at the methodsplit, which is already familiar to you - it splits a string into an array based on the delimiter. You should already know that the delimiter is passed as a parameter and it is a string.

However, this is not always the case - a regular expression can also be passed as a parameter. In this case, the separator will be all substrings that fall under the regular expression.

In the following example we will split a string into an array by delimiter'-'or by separator'+':

let str = 'a-b+c-d+e'; let res = str.split(/[-+]/);

As a result, the following will be written to the variable:

['a', 'b', 'c', 'd', 'e']

Given a string with date and time:

let str = '2025-12-31 12:59:59';

Break up this string so that all the date and time parts are in one array. That is, you should get the following array:

['2025', '12', '31', '12', '59', '59'];
