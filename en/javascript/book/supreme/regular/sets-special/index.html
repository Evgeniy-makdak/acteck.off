Special characters inside square brackets in JavaScript

Special characters inside[ ]become ordinary symbols. This means that they do not need to be escaped with a backslash.

Example

In this example, the search template looks like this:any letter between the X's'a', 'b', 'c', or point:

let str = 'xax xbx xcx xdx x.x x@x'; let res = str.replace(/x[abc.]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! xdx ! x@x'

Example

In this example, the search template looks like this:between the X's any small Latin letter or dot:

let str = 'xax xbx xcx x@x'; let res = str.replace(/x[a-z.]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! x@x'

Practical problems

Given the line:

let str = 'aba aea aca aza axa a.a a+a a*a';

Write a regular expression that finds strings'a.a', 'a+a', 'a*a'without affecting the others.

Given the line:

let str = 'xaz x.z x3z x@z x$z xrz';

Write a regular expression that finds strings using the pattern:letter'x', then NOT a dot, NOT a dog, and NOT a dollar, and then a letter'z'.
