Curly braces in JavaScript regular expressions

Operators'+', '*', '?'are good, however, they cannot be used to indicate a specific number of repetitions. In this case, the operator will come to your aid{}.

It works like this:{5}- five repetitions,{2,5}– repeats from two to five (both inclusive),{2,}- repeated two or more times. Please note that this option -{,2}- No. Look at the examples:

Example

In this example, the search template looks like this:letter'x', letter'a'once or twice, letter'x':

let str = 'xx xax xaax xaaax'; let res = str.replace(/xa{1,2}x/g, '!');

As a result, the following will be written to the variable:

'xx ! ! xaaax'

Example

In this example, the search template looks like this:letter'x', letter'a'two or more times, letter'x':

let str = 'xx xax xaax xaaax'; let res = str.replace(/xa{2,}x/g, '!');

As a result, the following will be written to the variable:

'xx xax ! !'

Example

In this example, the search template looks like this:letter'x', letter'a'three times, letter'x':

let str = 'xx xax xaax xaaax'; let res = str.replace(/xa{3}x/g, '!');

As a result, the following will be written to the variable:

'xx xax xaax !'

Example

In this example, the search template looks like this:letter'a'ten times:

let str = 'aaa aaaaaaaaaa aaa'; let res = str.replace(/a{10}/g, '!');

As a result, the following will be written to the variable:

'aaa ! aaa'

Example

In this example, the code author wanted a template like this:letter'x', letter'a'three times times or less, letter'x', but, unfortunately, this - {,3} - does not work. You need to specify explicitly:

let str = 'xx xax xaax xaaax'; let res = str.replace(/xa{1,3}x/g, '!');

As a result, the following will be written to the variable:

'xx ! ! !'

Example

Zero is also acceptable:

let str = 'xx xax xaax xaaax'; let res = str.replace(/xa{0,3}x/g, '!');

As a result, the following will be written to the variable:

'! ! ! !'

Practical problems

Given the line:

let str = 'aa aba abba abbba abbbba abbbbba';

Write a regular expression that finds strings'abba', 'abbba', 'abbbba'and only them.

Given the line:

let str = 'aa aba abba abbba abbbba abbbbba';

Write a regular expression that will find strings like'aba', in which'b'less common3-x times (inclusive).

Given the line:

let str = 'aa aba abba abbba abbbba abbbbba';

Write a regular expression that will find strings like'aba', in which'b'occurs more4-x times (inclusive).
