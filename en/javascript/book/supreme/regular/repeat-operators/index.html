Character repetition operators in regular expressions

There are situations when we want to indicate that a character is repeated a specified number of times. If we know the exact number of repetitions, then we can simply write it several times -/aaaa/. But what if we want to say this:repeat one or more times?

There are repetition operators (quantifiers) for this: plus+(one or more times), asterisk*(zero or more times) and a question?(zero or one time). These operators act on the symbol that comes before them.

Let's look at how these operators work with examples.

Example

Find all substrings according to the patternletter'x', letter'a'one or more times, letter'x':

let str = 'xx xax xaax xaaax xbx'; let res = str.replace(/xa+x/g, '!');

As a result, the following will be written to the variable:

'xx ! ! ! xbx'

Example

Find all substrings according to the patternletter'x', letter'a'zero or more times, letter'x':

let str = 'xx xax xaax xaaax xbx' let res = str.replace(/xa*x/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! xbx'

Example

Find all substrings according to the patternletter'x', letter'a'zero or one times, letter'x':

let str = 'xx xax xaax xbx'; let res = str.replace(/xa?x/g, '!');

As a result, the following will be written to the variable:

'! ! xaax xbx'

Practical problems

Given the line:

let str = 'aa aba abba abbba abca abea';

Write a regular expression that finds strings'aba', 'abba', 'abbba'according to the template:letter'a', letter'b'any number of times, letter'a'.

Given the line:

let str = 'aa aba abba abbba abca abea';

Write a regular expression that finds strings'aa', 'aba', 'abba', 'abbba'according to the template:letter'a', letter'b'any number of times (including none), letter'a'.

Given the line:

let str = 'aa aba abba abbba abca abea';

Write a regular expression that finds strings'aa', 'aba'according to the template:letter'a', letter'b'once or not at all, letter'a'.

Given the line:

let str = 'aa aba abba abbba abca abea';

Write a regular expression that finds strings'aa', 'aba', 'abba', 'abbba'without capturing'abca'And'abea'.
