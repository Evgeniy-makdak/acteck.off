Backslash in regex strings in JavaScript

In lines with regular expressions, you need to escape (actually double) backslashes. Let's look at an example. Let's say we have the following line:

let str = 'xyz';

Let's say we have the following code with a regular expression:

let reg = /\w+/; let res = str.match(reg);

Let's convert the regular expression into a string. In this case we have a problem with the backslash:

let reg = new RegExp('\w+'); // не работает let res = str.match(reg);

To solve the problem, double the backslash:

let reg = new RegExp('\\w+'); // работает let res = str.match(reg);

Practical problems

Convert the regular expression to a string:

let str = 'x1y x12y x123y'; let reg = /x\d+y/; let res = str.replace(reg, '!');

Convert the regular expression to a string:

let str = 'x.y xay xby'; let reg = /x\.y/; let res = str.replace(reg, '!');

Convert the regular expression to a string:

let str = 'x\\y'; let reg = /x\\y/; let res = str.replace(reg, '!');
