exec method in JavaScript regex

Methodexecsearches by string. The result returns the found substring and its pockets. In this case, each subsequent call of this method will begin the search from the place where the previous found substring ended.

Let's look at an example. Let's say we have the following line:

let str = '12 34 56';

Let's say we have the following regular expression:

let reg = /\d+/g;

Let's call our method sequentially on our string:

let res1 = reg.exec(str); console.log(res1[0]); // 12 let res2 = reg.exec(str); console.log(res2[0]); // 34 let res3 = reg.exec(str); console.log(res3[0]); // 56

After three calls, since our string no longer matches the regular expression, the next method call will returnnull:

let res4 = reg.exec(str); console.log(res4); // null

This feature of the method is convenient to use in a loop:

let str = '12 34 56'; let reg = /\d+/g; let res; while (res = reg.exec(str)) { console.log(res); // [12], [34], [56] }

You can not only find a match, but also put it in your pockets:

let str = '12 34 56'; let reg = /(\d)(\d)/g; let res; while (res = reg.exec(str)) { console.log(res); // [12, 1, 2], [34, 3, 4], [56, 5, 6] }

let str = '12:37:57 15:48:58 17:59:59';
