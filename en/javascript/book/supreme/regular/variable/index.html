Regular variable in JavaScript

Regular expressions can be put into variables. Let's look at an example. Let's say we have the following line:

let str = 'aaa bbb ccc';

Let's say we have the following code with a regular expression:

let res = str.replace(/\w+/, '!');

Let's put the regular expression into a variable:

let reg = /\w+/; let res = str.replace(reg, '!');

Place the regular expression in a variable in the following code:

let str = 'a aa aaa aaaa aaaa'; let res = str.replace(/a+/, '!');
