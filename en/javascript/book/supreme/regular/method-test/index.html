Test Method in JavaScript Regular Expressions

In this tutorial we will look at the methodtest, which checks if the string has at least one regular match. If there is - it is returnedtrue, and if not -false. The method takes a string as a parameter and is applied to a regular expression, like this:

регулярка.test(где искать)

Let's check some string for compliance with the regular expression as an example:

/a+/.test('eee aaa bbb'); // вернет true

This method is often used to check whether an entire string matches a regular expression. In this case, a cap is placed at the beginning of the regular season, and a dollar at the end:

/^a+$/.test('aaaaaaaaa'); // вернет true
