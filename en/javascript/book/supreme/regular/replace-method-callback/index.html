Callback in the replace method in JavaScript regex

The replace method can take not only a string as its second parameter, but alsocallback function, which will be applied to each match found. Each substring that the regular engine finds will be replaced with what this function returns for exactly this substring.

You can pass parameters to this function: the first parameter will contain the found string, the second parameter - the first pocket, the third parameter - the second pocket, and so on - you can make as many parameters as there are pockets in the regular expression.

The penultimate parameter will contain the position of the match found, and the last parameter will contain the entire string used to search.

Let's look at how it all works using practical examples.

Example

Let's be given a string with numbers:

let str = '2 3 4 5';

Let's replace these numbers with their squares. First, let's just print our numbers one by one to the console in the callback function:

str.replace(/\d+/g, function(match) { console.log(match); });

Our code will first output'2', Then'3', '4'And'5'. That is, into a variablematchthe lines that the regular engine found are sequentially included.

Let's solve the problem to the end - we will buildmatchinto a square and return it usingreturn. It turns out that for the found two it will return4and the two will be replaced by this four, for the found three it will return9and the three will be replaced by this nine - and so on:

let result = str.replace(/\d+/g, function (match) { return match**2; }); console.log(result); // выведет '4 9 16 25'

Example

Let now the line contain constructions of the form'2+3=':

let str = '2+3= 4+5= 6+7=';

Let's make sure that the sums of the corresponding numbers are inserted after the equal numbers. That is, our line should turn into the following:

'2+3=5 4+5=9 6+7=13'

To solve the problem, let's experiment again - put the first and second terms into separate pockets:

str.replace(/(\d+)\+(\d+)=/g, function (match0, match1, match2) { console.log(match0, match1, match2); });

Now let’s finally solve the problem: for each found substring, we sum the first and second pockets, take the zero pocket (the found string, for example'2+3='), add the result to it and return it all throughreturn:

let result = str.replace(/(\d+)\+(\d+)=/g, function(match0, match1, match2) { let sum = Number(match1) + Number(match2); return match0 + sum; }); console.log(result);

Practical problems

Given the line:

let str = 'aaa [2] bbb [3] ccc [12] ddd';

Find the numbers in brackets and double them. That is, our line should look like this:

'aaa [4] bbb [6] ccc [24] ddd'

Given the line:

let str = '123 456 789';

Find all the numbers and reverse their digits. That is, our line should look like this:

'321 654 987'

Given a string with dates:

let str = '31.12.2025 30.11.2024 29.10.2023';

Find all the dates and convert them to another format so that you get the following string:

'2025-12-31 2024-11-30 2023-10-29'
