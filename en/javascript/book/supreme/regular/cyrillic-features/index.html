Features of the Cyrillic alphabet in regular JavaScript

Cyrillic characters are not included in the group\w. To get them you need to use a group in square brackets, like this:[а-я]. But even with this group there is a problem - the letter will not be included here'ё'. To enable it you need to do this:[а-яё].

Given the line:

let str = 'wйw wяw wёw wqw';

Write a regular expression that finds strings using the pattern:there are letters on the edges'w', and between them is a Cyrillic letter.

Given the line:

let str = 'ааа ббб ёёё ззз ййй ААА БББ ЁЁЁ ЗЗЗ ЙЙЙ';

Write a regular search engine that will find all words according to the pattern:any Cyrillic letter any number of times.
