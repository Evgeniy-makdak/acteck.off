Global match in JavaScript regex

Methodmatch, called with modifierg, returns all found matches as an array. Let's look at examples.

Example

Let's get an array of substrings consisting of letters'a':

let str = 'a aa aaa aaaa'; let res = str.match(/a+/g);

As a result, the following will be written to the variable:

['a', 'aa', 'aaa', 'aaaa']

Example

Let's get an array of numbers:

let str = '1 23 456 789'; let res = str.match(/\d+/g);

As a result, the following will be written to the variable:

['1', '23', '456', '789']

Example

Let's get an array of all numbers:

let str = '1 23 456 789'; let res = str.match(/\d/g);

As a result, the following will be written to the variable:

['1', '2', '3', '4', '5', '6', '7', '8', '9']

Practical problems

Given the line:

let str = 'site.ru sss site.com zzz site.net';

Get an array of domain names from this string.

Given the line:

let str = 'a1b c34d x567z';

Find the sum of all numbers in this line.
