Multi-line in JavaScript regular expressions

Regular codes have their own characteristics when working with multi-line strings. Let's understand them using the following line as an example, containing regular characters, newlines and tabs:

let str = `111 222 333 444`;

Line feed

Line feed characters can be caught using the command'\n':

let res = str.replace(/\n/g, '!');

As a result, the following will be written to the variable (the spaces between the lines are tabs):

`111! 222! 333! 444`;

Tabulation

Tab characters can be caught using the command'\t':

let res = str.replace(/\t/g, '!');

As a result, the following will be written to the variable:

` 111 !222 !333 !444 `;

Work point

Team'.'for a multiline string does not catch newline characters:

let res = str.replace(/./g, '!');

As a result, the following will be written to the variable:

` !!! !!!! !!!! !!!! `;

Any character

To catch all the characters in a multi-line string, a clever trick is used in the form of a combination[\s\S]. This construction will find all regular characters and all newlines:

let res = str.replace(/[\s\S]/g, '!');

As a result, the following will be written to the variable:

`!!!!!!!!!!!!!!!!!!`;

hat

Using a modifier'm'You can enable multi-line mode. In this case, the cap will catch the beginning of each line:

let res = str.replace(/^/gm, '!');

As a result, the following will be written to the variable:

` !111 !222 !333 !444 `;

Dollar

Team'$'in multiline mode will catch the end of each line:

let res = str.replace(/$/gm, '!');

As a result, the following will be written to the variable:

` 111! 222! 333! 444! `;

Practical problems

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

` abc! def! ghi! jkl! `;

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

` ! abc ! def ! ghi ! jkl `;

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

`! abc def ghi jkl !`;

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

`! ! abc ! def ! ghi ! jkl !`;

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

`! abc! def! ghi! jkl! !`;

Given the line:

` abc def ghi jkl `;

Write a regular expression that turns this line into the following:

` !abc !def !ghi !jkl `;
