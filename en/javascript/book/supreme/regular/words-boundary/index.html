Word Boundaries in JavaScript Regulars

Using the command'\b'you can refer to a word boundary, and using the command'\B'- not to the border. Let's see how these commands work using examples.

Example

Let's wrap each word in a symbol'!':

let str = 'aaa aaa aaa'; let res = str.replace(/\b/g, '!');

As a result, the following will be written to the variable:

'!aaa! !aaa! !aaa!'

Example

Let's add a symbol'!'in the spaces between letters:

let str = 'aaa aaa aaa'; let res = str.replace(/\B/g, '!');

As a result, the following will be written to the variable:

'a!a!a a!a!a a!a!a'
