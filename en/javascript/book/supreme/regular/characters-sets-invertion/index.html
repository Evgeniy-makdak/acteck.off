Inverting character sets in regular expressions

With a hat'^'at the beginning of the square brackets you can invert what you want. That is, if, for example, the command[ab]looking for a letter'a'or'b', then the command[^ab]will search for all characters except'a'And'b'.

Example

In this example, the search template looks like this:letter'x', then NOT letter'a', Not'b'and not'c', then the letter'z':

let str = 'xaz xbz xcz xez'; let res = str.replace(/x[^abc]z/g, '!');

As a result, the following will be written to the variable:

'xax xbx xcx !'

Example

In this example, the search template looks like this:letter'x', then NOT a small Latin letter, then the letter'z':

let str = 'xaz xbz x1z xCz'; let res = str.replace(/x[^a-z]z/g, '!');

As a result, the following will be written to the variable:

'xaz xbz ! !'

Practical problems

Write a regular expression that finds strings using the pattern:number'1', then the symbol is not'e'and not'x', number'2'.

Write a regular expression that finds strings using the pattern:letter'x', then NOT the number from2to7, letter'z'.

Write a regular expression that finds strings using the pattern:letter'x', then NOT a capital letter from1and more times, letter'z'.

Write a regular expression that finds strings using the pattern:letter'x', then NOT a capital or small Latin letter or a number from1to5from1and more times, letter'z'.
