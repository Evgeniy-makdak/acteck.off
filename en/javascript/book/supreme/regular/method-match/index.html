Match Method in JavaScript Regulars

Methodmatchfinds the first match of the regular expression. It returns an array, the zeroth element of which will contain the found substring.

Let's look at an example. Let's say we have the following line:

let str = 'xax xaax xaaax';

Let's apply our regular expression method to this string:

let res = str.match(/xa+x/);

The zero element of the array containing the result will contain the first match with the regular expression:

console.log(res[0]); // выведет 'xax'

In this case, the result will have additional properties. In propertyindexwill be the position where the match is found:

console.log(res.index); // выведет 0

And in the propertyinput- the line used to search (information of dubious value):

console.log(res.input); // выведет 'xax xaax xaaax'
