Positive and Negative Lookup in JavaScript Regulars

Sometimes you need to solve a problem of this type: find a string'aaa'and replace it with'!', but only if after'aaa'costs'x', and himself'x'do not replace it. If we try to solve the problem'в лоб', then nothing will work for us:

'aaax baaa'.replace(/aaax/g, '!'); // вернет '! baaa', а хотели '!x baaa'

Look ahead

To solve the problem we need a way to say that'x'should not be replaced. This is done using special brackets(?= )who just look at it but don’t take it with them.

These brackets are calledpositive looking forward. Positive - because'x'(in our case) there must be - only then will the replacement occur.

Let's use these brackets to solve our problem:

'aaax aaab'.replace(/aaa(?=x)/g, '!'); // вернет '!x aaab'

There are alsonegative look-ahead - (?! )- he, on the contrary, says that something should not exist. In the following example, the replacement will only occur if after'aaa'worth NOT'x':

'aaax aaab'.replace(/aaa(?!x)/g, '!'); // вернет 'aaax !b'

View back

Likewise there ispositive look back - (?<= ). In the following example, the replacement will only occur if before'aaa'costs'x':

'xaaa'.replace(/(?<=x)aaa/g, '!'); // вернет 'x!'

And there is alsonegative look back - (?<! ). In the following example, the replacement will only occur if before'aaa'not worth it'x':

'baaa'.replace(/(?<!x)aaa/g, '!'); // вернет 'b!'

Practical problems

Given a string containing the names of the functions:

let str = 'func1() func2() func3()';

Get an array of function names from a string.

Given a line with a tag:

let str = '<a href="" class="eee" id="zzz">';

Get an array of attribute names for this tag.

Given a line with variables:

let str = '$aaa $bbb $ccc xxxx';

Get substrings preceded by a dollar symbol.
