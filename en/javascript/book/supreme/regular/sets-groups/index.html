Groups of characters within sets in JavaScript regular expressions

Character groups\d, \D, \w, \W, \s, \Sinside[]will denote groups, that is, they will still be teams.

Example

In this example, the search template looks like this:between the X's is any number or letter from'a'to'f':

let str = 'xax xbx x1x x2x xhx x@x'; let res = str.replace(/x[\da-f]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! xhx x@x'

Example

In this example, the search template looks like this:letter'x', then not a number, not a dot, and not a small Latin letter, then a letter'z':

let str = 'xaz x1z xAz x.z x@z'; let res = str.replace(/x[^\d.a-z]z/g, '!');

As a result, the following will be written to the variable:

'xaz x1z ! x.z !'
