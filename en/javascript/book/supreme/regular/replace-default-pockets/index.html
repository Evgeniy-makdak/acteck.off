Default pockets in the replace method in JavaScript regex

In the replace method, in addition to pockets with your numbers, standard pockets are always available:$&- all matches found,$`And$'- part of the string before and after the match. Let's see how to work with them using examples.

Example

Let's find all the numbers and wrap them in parentheses:

let str = '1 23 456'; let res = str.replace(/\d+/g, '($&)');

As a result, the following will be written to the variable:

'(1) (23) (456)'

Example

Let's find the @ symbol and replace it with what comes before it, the dog, and what comes after it. Let's write all this in parentheses:

let str = '123@456'; let res = str.replace(/@/g, "($`@$')");

As a result, the following will be written to the variable:

'123(123@456)456'

Example

Let's say we want to find a dollar and wrap it in quotes``. In this case, to$`was not perceived as a command, the dollar needs to be doubled:

let str = 'aaa $ bbb'; let res = str.replace(/\$/g, '`$$`');

As a result, the following will be written to the variable:

'aaa `$` bbb'

Practical problems

Given the line:

let str = 'a1b2c3';

Write a regular expression that will write the same next to each number.

Given the line:

let str = 'sss site.ru zzz site.com kkk';

Replace domains with links like:

<a href="http://site.ru">site.ru</a>
