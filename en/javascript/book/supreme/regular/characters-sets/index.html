Character sets in JavaScript regular expressions

Character groups\dAnd\wnot very flexible. Even something as simple asfind all letters but not numbers- cannot be solved by them. For such problems, you should use square brackets, which represent the operation'или'.

Square brackets replace a single character, any of those listed inside. For example, like this:/x[abc]x/- we say that there should be the letters X on the edges, and one symbol inside: or'a', or'b', or'c'.

You can write repetition operators after square brackets. For example, like this:/x[abc]+x/- we say that there can be any number of characters inside the X's'a', 'b'And'c'- in any combinations.

You can not only list symbols, but also create groups of symbols by writing a hyphen between two symbols. For example, like this:[a-d]- we get all the characters from'a'to'd'.

Let's look at examples.

Example

In this example, the search template looks like this:between the X's is any letter from'a'to'z':

let str = 'xax xbx xcx x@x'; let res = str.replace(/x[a-z]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! x@x'

Example

In this example, the search template looks like this:between the X's is any letter from'a'to'k':

let str = 'xax xbx xmx x@x'; let res = str.replace(/x[a-k]x/g, '!');

As a result, the following will be written to the variable:

'! ! xmx x@x'

Example

In this example, the search template looks like this:between the X's is any letter from'A'to'Z':

let str = 'xax xBx xcx x@x'; let res = str.replace(/x[A-Z]x/g, '!');

As a result, the following will be written to the variable:

'xax ! xcx x@x'

Example

In this example, the search template looks like this:between the X's any number from0until 9:

let str = 'xax x1x x3x x5x x@x'; let res = str.replace(/x[0-9]x/g, '!');

As a result, the following will be written to the variable:

'xax ! ! ! x@x'

Example

In this example, the search template looks like this:between the X's any number from3up to 7:

let str = 'xax x1x x3x x5x x@x'; let res = str.replace(/x[3-7]x/g, '!');

As a result, the following will be written to the variable:

'xax x1x ! ! x@x'

Example

In this example, the search template looks like this:between the X's is any letter from'a'to'z'or number from1until 9:

let str = 'xax x1x x3x x5x x@x'; let res = str.replace(/x[a-z1-9]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! x@x'

Example

In this example, the search template looks like this:between the X's is any letter from'a'to'z'or letter from'A'to'Z':

let str = 'xax xBx xcx x5x x@x'; let res = str.replace(/x[a-zA-Z]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! x5x x@x'

Example

In this example, the search template looks like this:between the X's is any letter from'a'to'z'or numbers1, 2:

let str = 'xax xbx x1x x2x x3x'; let res = str.replace(/x[a-z12]x/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! x3x'

Example

In this example, the search template looks like this:Between the X's are the letters from'a'to'z'in quantities from1and more:

let str = 'xx xabesx xaadx x123x xa3x'; let res = str.replace(/x[a-z]+x/g, '!');

As a result, the following will be written to the variable:

'xx ! ! ! x123x xa3x'

Example

Let's make it so that the number of letters can be zero:

let str = 'xx xabesx xaadx x123x xa3x'; let res = str.replace(/x[a-z]*x/g, '!');

As a result, the following will be written to the variable:

'! ! ! ! x123x xa3x'

Practical problems

let str = 'a1a a3a a7a a9a aba';

Write a regular search engine that matches the following pattern:there are letters on the edges'a', and between them is the number from3-x to6-ty.

let str = 'aba aea afa aha aga';

Write a regular search engine that matches the following pattern:there are letters on the edges'a', and between them - the letter from a to g.

let str = 'aba aea afa aha aga';
