Introduction to Regular Expressions in JavaScript

Regular expressions are commands for complex search and replace (or just search). They allow you to do very interesting things, but, unfortunately, are quite difficult to master.

There are several JavaScript methods for working with regular expressions. We will begin to get acquainted with them using an examplereplace- you are already familiar with this method: it takes as the first parameter what to change, and the second - what to change to. And the method itself is applied to the line in which the replacement is made:

'bab'.replace('a', '!'); // вернет 'b!b'

The first parameter of this method can be passed not just a string, butregular expression. A regular expression is a set of commands inside slashes/. These slashes are calledlimitersregular expressions.

Regular expressions themselves consist of two types of characters: those that represent themselves and command characters calledspecial characters.

Letters and numbers represent themselves. In the following example, we will use a regular expression to replace the letter'a'on!:

'bab'.replace(/a/, '!'); // вернет 'b!b'

But the dot is a special symbol and meansany character. In the following example, we will find a string using this pattern:letter'x', then any symbol, then a letter again'x':

'xax eee'.replace(/x.x/, '!'); // вернет '! eee'

After the delimiters you can writemodifiers- commands that change the general properties of a regular expression. For example, the modifiergturns on the global search and replace mode - without it, the regular engine searches only for the first match, and with it - all matches.

The following example does not specify a modifiergand the regular engine will find only the first match:

'aab'.replace(/a/, '!'); // вернет '!ab'

And now the regular engine will find all the matches:

'aab'.replace(/a/g, '!'); // вернет '!!b'

let str = 'ahb acb aeb aeeb adcb axeb';

Given the line:

let str = 'aba aca aea abba adca abea';

Write a regular expression that finds strings'abba', 'adca', 'abea'according to the template:letter'a', 2any symbol, letter'a'.

Given the line:

let str = 'aba aca aea abba adca abea';

Write a regular expression that finds strings'abba'And'abea'without capturing'adca'.
