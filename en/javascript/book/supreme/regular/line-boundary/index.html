Start and end of line in JavaScript regular expressions

There are special characters that indicate the beginning'^'or end of line'$'. Let's see their work with examples.

Example

In this example, the search template is:replace'aaa'on'!'only if it is at the beginning of the line:

let str = 'aaa aaa aaa'; let res = str.replace(/^aaa/g, '!');

As a result, the following will be written to the variable:

'! aaa aaa'

Example

In this example, the search template is:replace'aaa'on'!'only if it is at the end of the line:

let str = 'aaa aaa aaa'; let res = str.replace(/aaa$/g, '!');

As a result, the following will be written to the variable:

'aaa aaa !'

Example

When at the beginning of the regular season it is'^', and at the end -'$', then in this way we check the entire string for compliance with the regular sequence.

In the following example, the search pattern is:letter'a'repeated one or more times, replace the entire line with'!'only it consists of only letters'a'.

let str = 'aaa'; let res = str.replace(/^a+$/g, '!');

As a result, the following will be written to the variable:

'!'

Practical problems

Given the line:

let str = 'abc def xyz';

Write a regular expression that finds the first substring of letters.

Given the line:

let str = 'abc def xyz';

Write a regular expression that finds the last substring of letters.
