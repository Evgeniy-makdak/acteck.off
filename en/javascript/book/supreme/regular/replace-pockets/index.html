Pockets in the replace method in JavaScript regex

When working with the methodreplace, if we put something in a pocket in a regular sequence, then in the replacement line we can insert the contents of this pocket by writing a dollar sign$and pocket number. For example,- first pocket,- second pocket and so on.

Why is this needed and how to use it, let's look at examples.

Example

Let's find all the numbers and insert the same numbers instead, but in parentheses. To do this, we will replace all the numbers found with them themselves, but in brackets:

let str = '1 23 456 xax'; let res = str.replace(/(\d+)/g, '($1)');

As a result, the following will be written to the variable:

'(1) (23) (456) xax'

Example

Let's find all the lines that are numbers with X's around them and replace these numbers with them, but with'!'signs around:

let str = 'x1x x23x x456x xax'; let res = str.replace(/x(\d+)x/g, '!$1!');

As a result, the following will be written to the variable:

'!1! !23! !456! xax'

Example

Let's solve the following problem: given lines of the form'aaa@bbb'- letters, then a dog, then letters. You need to swap the letters before and after @.

let str = 'aaa@bbb ccc@ddd'; let res = str.replace(/([a-z]+)@([a-z]+)/g, '$2@$1');

As a result, the following will be written to the variable:

'bbb@aaa ddd@ccc'

Practical problems

Given the line:

let str = '12 34 56 78';

Swap the digits in all two-digit numbers.

Given a string with a date:

let str = '31.12.2025';

Convert this date to'2025.12.31'.
