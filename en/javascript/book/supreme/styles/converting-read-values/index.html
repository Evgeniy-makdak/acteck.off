Converting read values ​​in JavaScript

FunctiongetComputedStylesdoes not receive exactly the values ​​that were set in the CSS. It converts them to absolute values, usually pixels.

Let's look at an example. Let us set the width as a percentage:

#elem { width: 100%; }

However, when reading the width, we get the value not in percentage, but in pixels:

console.log(computedStyle.width); // значение в px

The element has the following styles:

#elem { font-size: 2em; }

By clicking on the button, display the font size of the element.
