Styling with Classes in JavaScript

Style elements via attributestyleSometimes it’s convenient, but more often than not it’s not a good idea. The problem is that the styles will be scattered throughout the JavaScript file and will be difficult to change. It is much more convenient to set styles in CSS files so that they can be easily changed without delving into JavaScript code.

For example, let's say we have an element that displays a message. The message can be "good" and displayed in green, or "bad" and displayed in red. The best solution in this case would be to make the appropriate CSS classes for this:

.success { color: green; } .error { color: red; }

Now, when outputting a “good” message, we will give the element a “good” class:

elem.textContent = 'good'; elem.classList.add('success');

And when outputting a “bad” message, we will give the element a “bad” class:

elem.textContent = 'bad'; elem.classList.add('error');

Paragraphs with numbers are given. Go through these paragraphs in a loop and color the paragraphs containing even numbers red, and those containing odd numbers green.
