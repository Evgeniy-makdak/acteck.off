Alternating styles using the data attribute in JavaScript

To solve the problem described in the previous lesson, we will not assign classes to the element, but will change the valuedata-attribute. This will be convenient because such an attribute can have only one value, and when written, the new value will itself overwrite the old one.

For example, this is how we will set a successful style:

<div id="elem" data-type="success"> text </div>

And this is the style for the error:

<div id="elem" data-type="error"> text </div>

We will set the styles of our states usingattribute selectors:

[data-type="success"] { color: green; } [data-type="warning"] { color: orange; } [data-type="error"] { color: red; }

Now we can easily color our element a successful color:

elem.dataset.type = 'success';

And so we will color it in the color of the error:

elem.dataset.type = 'error';

Input given. When you lose focus, check the number entered into it. If this number is up to ten, then color the input green, if from ten to twenty - yellow, and if more than twenty - red.
