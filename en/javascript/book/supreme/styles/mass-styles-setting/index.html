Bulk styling in JavaScript

You can set several styles at once with one command. This is done using the propertycssText:

elem.style.cssText = ` width: 100px; height: 100px; display: block; `;

This property is rarely used because the assignment removes all existing styles: it replaces them rather than adding them. You might accidentally delete something you need. But it can be used, for example, for new elements when we know for sure that we will not delete the existing style.

Rewrite the code using the learned property:

elem.style.width = '100px'; elem.style.height = '100px'; elem.style.margin = '10px auto'; elem.style.color = 'red';
