Unpacking CSS Property Values ​​in JavaScript

When reading an attributestyleshortened CSS properties are unpacked into their full properties. Let's see what this means. Let us set the indentation for all sides:

elem.style.margin = '20px';

Let's output it:

console.log(elem.style.margin); // выведет '20px';

In this case, reading properties for individual parties will also be available:

console.log(elem.style.marginLeft); // выведет '20px';

Dan div:

<div id="elem" style="width: 300px; height: 200px; border: 1px solid red;"> text </div>

A button is also given. By clicking on the button, display the border thickness, its type and color.
