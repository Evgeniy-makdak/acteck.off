Units of measurement in the style attribute in JavaScript

When specifying property values ​​through an attributestylewe must indicate the units of measurement:

elem.style.width = '100px';

When reading values ​​they will also be returned with units:

let width = elem.style.width; console.log(width); // выведет '100px';

If necessary, we can get rid of units of measurement usingparseInt:

let width = elem.style.width; console.log(parseInt(width)); // выведет 100

If fractional values ​​are expected, you can use the functionparseFloat:

let width = elem.style.width; console.log(parseFloat(width));

Given a div and a button. By clicking the button, set the width to400px, and the height is300px.

Dan div:

<div id="elem" style="width: 300px; height: 200px;"> text </div>

A button is also given. By clicking on the button, display the width and height of the div without units of measurement.

Dan div:

<div id="elem" style="font-size: 1.5em;"> text </div>

A button is also given. By clicking on the button, display the div font size without units of measurement.
