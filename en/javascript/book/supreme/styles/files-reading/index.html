Reading properties from CSS files in JavaScript

Properties set in a CSS file can be read using a special functiongetComputedStyles.

Let's see how it's done. Let us have the following element:

<div id="elem"> text </div>

Let the following styles be set for this element:

#elem { width: 100px; color: red; font-size: 20px; }

Let's get a reference to the element in a variable:

let elem = document.querySelector('#elem');

Now let's pass a link to the received element as a function parametergetComputedStyles:

let computedStyle = getComputedStyle(elem);

As a result, we will receive an object containing the CSS property values ​​for our element. Let's use it to read, for example, color:

console.log(computedStyle.color); // выведет 'red'

And now the width:

console.log(computedStyle.width); // выведет '100px'

And now the font size:

console.log(computedStyle.fontSize); // выведет '20px'

The element has the following styles:

#elem { width: 200px; height: 200px; }

By clicking on the button, display the width and height of the element.
