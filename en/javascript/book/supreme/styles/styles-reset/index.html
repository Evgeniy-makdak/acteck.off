Resetting styles via style in JavaScript

Let us decide to hide an element due to some event:

elem.style.display = 'none';

Let us now decide to show it back. This means that the propertydisplayyou need to return it to its original value. For example, like this:

elem.style.display = 'block';

The problem is that it's not convenient. After all, the original meaning was not necessarily exactlyblock. It could have beenflex, Andinline-block, and something else. It could be set by us in the CSS file, or it could be taken by the browser by default. Keeping track of the correct value is difficult.

Luckily, there is a way to easily change the property back to its original value. You just need to assign it an empty string as the value of the property:

elem.style.display = '';
