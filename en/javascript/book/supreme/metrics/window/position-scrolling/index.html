Method to scroll window to position in JavaScript

MethodscrollToScrolls the window to the specified position. It takes the horizontal position as the first parameter, and the vertical position as the second. Let's scroll the window to a given position:

window.scrollTo(100, 200);

The method has an alternative syntax in which an object is passed as a parameter. In the keytopthis object's vertical position is indicated in the keyleft- horizontal position, and in the keybehavior- scroll type. Type can be'auto'or'smooth'. In the second case, scrolling will be smooth:

element.scrollTo({ top: 100, left: 100, behavior: 'smooth' });
