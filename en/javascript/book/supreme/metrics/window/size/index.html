Window sizes in JavaScript

The following properties contain browser window sizes that do not include the scrollbar:

let w = document.documentElement.clientWidth; // ширина let h = document.documentElement.clientHeight; // высота

The following properties include the scrollbar:

let w = window.innerWidth; // ширина let h = window.innerHeight; // высота

The difference between the two property types gives the scrollbar size:

let w1 = document.documentElement.clientWidth; let w2 = window.innerWidth; console.log(w2 - w1);
