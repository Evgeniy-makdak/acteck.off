Method to scroll a window by an amount in JavaScript

MethodscrollByScrolls the window by a specified amount from the current position. It takes the horizontal shift as the first parameter, and the vertical shift as the second. Let's scroll the window by the specified amounts:

window.scrollBy(100, 200);

Let's scroll in the opposite direction:

window.scrollBy(-100, -200);

Let's perform smooth scrolling:

window.scrollBy({ top: 100, left: 100, behavior: 'smooth' });

By clicking the button, scroll the window to100pxdown.

By clicking the button, scroll the window to100pxup.

By pressing the button, smoothly scroll the window to300pxdown.

By pressing the button, smoothly scroll the window to300pxup.
