Getting and changing window scrolling in JavaScript

The following properties contain how much the window is scrolled:

let t = document.documentElement.scrollTop; // сверху let l = document.documentElement.scrollLeft; // слева

These properties can be changed by scrolling the window to any position:

document.documentElement.scrollTop = 200;

By clicking the button, scroll the window to a position300pxabove.

By clicking the button, scroll the window to a position100pxbelow.

By clicking the button, scroll the window to the very bottom.
