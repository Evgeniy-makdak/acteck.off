Method to scroll window to element in JavaScript

MethodscrollIntoViewScrolls the window to the specified element. The method is applied to the element to which the window should be scrolled. The method parameter controls where the element should appear: at the top of the window or at the bottom.

Let's look at examples. Scroll the window so that the element is at the top of the window:

elem.scrollIntoView(true);

Now let’s scroll the window so that the element is at the bottom of the window:

elem.scrollIntoView(false);

You can also pass an object with settings as a parameter. Let's smoothly scroll the window to the element:

elem.scrollIntoView({ behavior: 'smooth', });

The settings object also has settings for page alignment relative to the element. Settingsblockspecifies the vertical alignment, and the settinginline- horizontal. They can take the following values:'start', 'center', 'end', 'nearest'.

Let's try:

elem.scrollIntoView({ behavior: 'smooth', block: 'start', inline: 'end', });
