Scroll-aware window sizing in JavaScript

Let's get the window dimensions taking into account the scrolled part. Unfortunately, there is no convenient method that works in all browsers. Below are working solutions. There is no need to delve into them, just use them.

Height including scrolled part:

let scrollHeight = Math.max( document.body.scrollHeight, document.documentElement.scrollHeight, document.body.offsetHeight, document.documentElement.offsetHeight, document.body.clientHeight, document.documentElement.clientHeight ); console.log(scrollHeight);

Width including scrolled part:

let scrollWidth = Math.max( document.body.scrollWidth, document.documentElement.scrollWidth, document.body.offsetWidth, document.documentElement.offsetWidth, document.body.clientWidth, document.documentElement.clientWidth ); console.log(scrollWidth);

By pressing the button, display the height taking into account the scrolled part.

When you click the button, display the width taking into account the scrolled part.

By clicking the button, display the height of the part hidden under the scroll.
