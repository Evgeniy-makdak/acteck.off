Expanding an element in JavaScript

Let's open the element to its full height:

elem.style.height = elem.scrollHeight + 'px';

Let's expand the element to its full width:

elem.style.width = elem.scrollWidth + 'px';

Given an element and a button. By pressing the button, open the element to its full height.
