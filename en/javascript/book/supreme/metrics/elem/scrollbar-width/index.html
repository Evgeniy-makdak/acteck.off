Scrollbar width in JavaScript

Scrollbar width varies by browser. We can get the value of this width. To do this, create an element with scrolling, but without borders and padding:

let div = document.createElement('div');

Let's add some styles to it:

div.style.overflowY = 'scroll'; div.style.width = '50px'; div.style.height = '50px';

Let's insert the element into the document, otherwise the dimensions will be zero:

document.body.append(div);

Let's get the scrollbar size:

let scrollWidth = div.offsetWidth - div.clientWidth;

Let's remove the element:

div.remove();

A button is given. By clicking the button, you will get the width of the scroll bar in your browser.
