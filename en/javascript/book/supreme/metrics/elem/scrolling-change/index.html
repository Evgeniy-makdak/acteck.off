Changing element scrolling in JavaScript

Properties containing scrollable elements can be modified. In this case, the element will scroll to the specified value. Let's for example scroll the element to position200pxabove:

elem.scrollTop = 200;

Now let's scroll the element to100pxdown from the current value:

elem.scrollTop = elem.scrollTop + 100;

Now let's scroll the element to the end:

elem.scrollTop = elem.scrollHeight - elem.clientHeight;

Given an element and a button. On click of a button, scroll the element to the scroll position100pxvertically and50pxhorizontally.

Given an element and a button. When you click the button, scroll the element to50pxdown from the current scroll position.

Given an element and a button. By clicking the button, scroll the element to the very top.

Given an element and a button. By clicking the button, scroll the element to the very bottom.
