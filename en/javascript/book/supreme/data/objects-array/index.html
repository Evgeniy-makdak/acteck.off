Array of objects with id in JavaScript

Quite often you will have to work with arrays of objects in which each element will have some randomid. For example, the array could be like this:

let arr = [ { id: 'GYi9GauC4gBF1e2SixDvu', prop1: 'value11', prop2: 'value12', prop3: 'value13', }, { id: 'IWSpfBPSV3SXgRF87uO74', prop1: 'value21', prop2: 'value22', prop3: 'value23', }, { id: 'JAmjRlfQT8rLTm5tG2m1L', prop1: 'value31', prop2: 'value32', prop3: 'value33', }, ];

In such arrays of objects, elements are accessed by theirid. The problem here is that theseidare inside the elements, which means that for any work with the element, you will first need to find it. We will learn how to work with such arrays in the following lessons.
