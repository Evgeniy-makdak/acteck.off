Changing element by id in JavaScript

Let in the variablearrlies an array of objects:

let arr = [ { id: 'GYi9GauC4gBF1e2SixDvu', prop1: 'value11', prop2: 'value12', prop3: 'value13', }, { id: 'IWSpfBPSV3SXgRF87uO74', prop1: 'value21', prop2: 'value22', prop3: 'value23', }, { id: 'JAmjRlfQT8rLTm5tG2m1L', prop1: 'value31', prop2: 'value32', prop3: 'value33', }, ];

Let's say we want to change some element of the array. Let the new data be stored in a variable, for example, like this:

let data = { id: 'IWSpfBPSV3SXgRF87uO74', prop1: 'value21 !', prop2: 'value22 !', prop3: 'value23 !', };

In the given objectidcoincides withidthe second element of the array, and the property values ​​are different. In other words, indatain propertyidwe storeidthe array element that we want to change.

Let's make this change. To do this, we will iterate through the elements of the array in a loop and, ifidmatches the one we are looking for, we will replace the element, and if it does not match, we will leave the element unchanged:

let res = arr.map(elem => { if (elem.id === data.id) { return data; } else { return elem; } });

You can shorten the code by using the ternary operator:

let res = arr.map(elem => elem.id === data.id ? data : elem);
