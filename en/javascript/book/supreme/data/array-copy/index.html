Копии массивов в JavaScript

Давайте научимся копировать массивы. Пусть для примера у нас есть некоторый массив:

let arr = [1, 2, 3, 4, 5];

Давайте сделаем его копию. Это делается с помощью специального метода, вот так:

let copy = Object.assign([], arr);

Есть и второй подход - через деструктуризацию:

let copy = [...arr];
