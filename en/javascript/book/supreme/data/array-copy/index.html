Copies of arrays in JavaScript

Let's learn how to copy arrays. For example, let's say we have some array:

let arr = [1, 2, 3, 4, 5];

Let's make a copy of it. This is done using a special method, like this:

let copy = Object.assign([], arr);

There is a second approach - through destructuring:

let copy = [...arr];
