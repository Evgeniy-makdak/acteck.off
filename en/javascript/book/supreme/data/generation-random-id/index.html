Generating a random id in JavaScript

Let's make a function that will generate a string with a randomid:

function getId(length = 16) { let chars = '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'; let res = ''; for (let i = 0; i < length; i++) { res += chars[Math.floor(Math.random() * chars.length)]; } return res; }

Let's use our function:

let id = getId(); console.log(id);
