Getting an element by id in JavaScript

You may need to get an array element by itsid. Let's see how it's done.

Let in the variablearrlies an array of objects:

let arr = [ { id: 'GYi9GauC4gBF1e2SixDvu', prop1: 'value11', prop2: 'value12', prop3: 'value13', }, { id: 'IWSpfBPSV3SXgRF87uO74', prop1: 'value21', prop2: 'value22', prop3: 'value23', }, { id: 'JAmjRlfQT8rLTm5tG2m1L', prop1: 'value31', prop2: 'value32', prop3: 'value33', }, ];

Letidelement is stored in a variable:

let id = 'IWSpfBPSV3SXgRF87uO74';

Let's get an element like thisid. For this we use the methodfind:

let res = arr.find(elem => { if (elem.id === id) { return true; } else { return false; } });

The code can be shortened:

let res = arr.find(elem => elem.id === id);
