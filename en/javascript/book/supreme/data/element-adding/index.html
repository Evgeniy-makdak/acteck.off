Immutable adding elements to an array in JavaScript

Let's now learn how to do immutable addition of elements to an array. Let, for example, we have some array:

let arr = [1, 2, 3, 4, 5];

Let's add a new element to it. According to our approach, we must make a copy of the array and add a new element to this copy. Let's do this:

let copy = Object.assign([], arr); copy.push(6); let res = copy;

Now let’s use the second approach with destructuring:

let res = [...arr, 6];

Make a button that, when clicked, will immutably add a new element to the array.
