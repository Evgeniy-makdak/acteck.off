Immutable modification of array elements in JavaScript

Let's now learn how to make an immutable change to the elements in an array. For example, let's say we have some array:

let arr = [1, 2, 3, 4, 5];

Let the change number be stored in a variableind:

let ind = 3;

Let's change the element with the given number. According to our approach, we have to make a copy of the array and change the element in the copy. Let's do this:

let copy = Object.assign([], arr); copy[ind] = '!'; let res = copy;

Let's use the second approach:

let res = [ ...arr.slice(0, ind), '!', ...arr.slice(ind + 1) ];
