Throwing custom exception types in JavaScript

The number of exceptions built into JavaScript is not that large and most often they cannot satisfy all our needs for different types of exceptions. Therefore, JavaScript has a built-in ability to create exceptions with its own type.

There are different ways to do this. The simplest one is inthrowpass an object with keysnameAndmessage:

try { throw {name: 'MyError', message: 'текст исключения'}; } catch (error) { console.log(error.name); // 'MyError' console.log(error.message); // 'текст исключения' }

Previously, we made a function that throws an exception when dividing by zero:

function div(a, b) { if (b !== 0) { return a / b; } else { throw new Error('ошибка деления на ноль'); } }

Rework this function so that it throws an exception with some type we invented, for example,DivisionByZeroError.

Previously, you made a function that threw an exception when you tried to take the root of a negative number. Rework your function so that it throws an exception with the type you came up with. Think carefully about the name of the exception so that the name is successful.
