Catching Exceptions in JavaScript

To catch exceptions, use the constructtry-catch. It has the following syntax:

try { // код } catch (error) { // обработка ошибки }

In the blocktryyou should post code that might contain an exception. If suddenly an exception occurs while executing this code, then our script will not crash with an error in the console, but will start executing the block codecatch.

In this block we must somehow adequately respond to the error. For example, if we wanted to send some data over the Internet and the Internet does not work, in the blockcatchwe can somehow cope with the situation: we can, for example, issue a message to the user, or after a while we can try to send the data again - suddenly the Internet is already working.

If, when executing a blocktryno exceptions occurred, then the useful code will simply be executed, and the code from the blockcatch- No.

For example, let's try to parse JSON and if it is incorrect, we will display a message about this on the screen:

try { let data = JSON.parse('{1,2,3,4,5}'); } catch (error) { alert('невозможно выполнить операцию разбора JSON'); }
