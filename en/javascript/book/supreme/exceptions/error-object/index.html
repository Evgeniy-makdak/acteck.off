JavaScript Error Object

If an exception occurs, then the first parameter of the blockcatchan object with an error will be received. Propertynameof this object contains the name of the error (in fact, its type), and the propertymessage- text of this error:

try { } catch (error) { console.log(error.name); // имя ошибки console.log(error.message); // текст ошибки }

Specifically throw an exception when attempting to parse invalid JSON. Print the name and text of this error to the console.

Purposely throw a local storage overflow exception. Print the name and text of this error to the console.
