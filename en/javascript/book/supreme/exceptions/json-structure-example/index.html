Example of exception with JSON in JavaScript

Let JSON with a product come to us from somewhere in the outside world:

let json = '{"product": "apple", "price": 1000, "amount": 5}'; let product = JSON.parse(json); alert(product.price * product.amount);

You already know that the methodJSON.parsewill throw an exception if the JSON is invalid. Let's catch this exception:

try { let json = '{"product": "apple", "price": 1000, "amount": 5}'; let product = JSON.parse(json); alert(product.price * product.amount); } catch (error) { // как-то реагируем на исключение }

However, it may be that the JSON itself is correct, but does not contain the fields we need, for example, there is no field with a price:

let json = '{"product": "apple", "amount": 5}'; // нет цены

Let's say that this is also an exceptional situation and in this case we will throw our own custom exception:

try { let json = '{"product": "apple", "amount": 5}'; let product = JSON.parse(json); if (product.price !== undefined && product.amount !== undefined) { alert(product.price * product.amount); } else { throw { name: 'ProductCostError', message: 'отсутствует цена или количество у продукта' }; } } catch (error) { // как-то реагируем на исключение }

Now the blockcatchwill receive two types of exceptions: either the JSON is generally incorrect, in which case there will be an exception likeSyntaxError, or the JSON is correct, but does not contain the fields we need, and then there will be an exception likeProductCostError.

Let's blockcatchWe will catch these types of exceptions:

try { let json = '{"product": "apple", "amount": 5}'; let product = JSON.parse(json); if (product.price !== undefined && product.amount !== undefined) { alert(product.price * product.amount); } else { throw { name: 'ProductCostError', message: 'отсутствует цена или количество у продукта' }; } } catch (error) { if (error.name == 'SyntaxError') { alert('Некорректный JSON продукта'); } else if (error.name == 'ProductCostError') { alert('У продукта отсутствует цена или количество'); } }

Let JSON come to you like this:

let json = `[ { "name": "user1", "age": 25, "salary": 1000 }, { "name": "user2", "age": 26, "salary": 2000 }, { "name": "user3", "age": 27, "salary": 3000 } ]`;

Check this JSON for general correctness when parsed, and after parsing, check that the result is an array and not something else. If the result is not an array, throw an exception.
