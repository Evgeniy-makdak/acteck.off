Throwing Different Types of Exceptions in JavaScript

Let's throw our exception and see how the error object behaves in this case:

try { throw new Error('текст исключения'); } catch (error) { console.log(error.name); // 'Error' console.log(error.message); // 'текст исключения' }

As you can see, our exception type is'Error'. Any exceptions thrown in this manner will have the same type. This, however, will not always be convenient, since if we may have several exceptions, we will not be able to distinguish them from each other.

JavaScript provides a solution to this problem: you can throw exceptions not only of typeError, but also any type of error built into JavaScript, for example,TypeError, SyntaxError, RangeError.

Let's throw an exception likeSyntaxError:

try { throw new SyntaxError('текст исключения'); } catch (error) { console.log(error.name); // 'SyntaxError' console.log(error.message); // 'текст исключения' }
