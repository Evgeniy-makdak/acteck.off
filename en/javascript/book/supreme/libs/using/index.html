Using Libraries in JavaScript

Let's look at the general principle of using libraries. Suppose we needed a library with convenient functions for working with arrays. Let us google and google the librarylodash.

We go to the library website and look for links to download the library. Typically, libraries are provided in two versions: regular and minified. The normal view is useful if you want to examine the source code of a library. The minimized view is convenient because it takes up less space.

So, we are looking for a download link. Download the library. Sometimes it happens that the documentation provides a link to a JavaScript file. In this case, instead of being downloaded, this file will simply open in the browser. In this case, you need to select save in the browser menu or clickCtrl + Sand save the library as a file.

You need to connect the downloaded library to your HTML file:

<!DOCTYPE html> <html> <head> <title></title> <script src="lodash.js"></script> </head> <body> </body> </html>

The linked library creates one or more variables in a shared scope. In the case of lodash, this variable is called_. Let's now connect our file below the library connection, in which we will write the code for our site:

<!DOCTYPE html> <html> <head> <title></title> <script src="lodash.js"></script> <script src="test.js"></script> </head> <body> </body> </html>

In this file, a global variable will be available, which is created by the previously included lodash library:

console.log(_); // в этой переменной библиотека lodash

Let's use one of the methods of the connected library:

let res = _.chunk(['a', 'b', 'c', 'd'], 2); console.log([['a', 'b'], ['c', 'd']]);

Download your libraryunderscorejs. Connect it to your HTML file. Try out several functions from this library.
