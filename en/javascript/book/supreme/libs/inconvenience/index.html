Disadvantages of JavaScript Libraries

Often some libraries depend on others. Therefore, before you include the library you need, you need to include itdependencies.

For example, consider the libraryslickto create sliders. This library depends on the jQuery library. This means that to connect the slider you also need to connect jQuery:

<!DOCTYPE html> <html> <head> <title></title> <script src="jquery-2.2.0.min.js"></script> <script src="./slick/slick.js"></script> </head> <body> </body> </html>

The disadvantage of this approach is that you need to manually download the library and its dependencies and at the same time ensure that the versions of the libraries match each other. If you use several libraries on your site, this can become a problem. For example, each of your libraries may require jQuery in its dependencies and at the same time different versions. You will have to manually resolve this conflict.

Download your libraryslick. Connect it to yourself and create a slider. Play around with the settings described in the documentation.
