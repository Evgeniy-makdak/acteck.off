Getting the name of a symbol in JavaScript

Having a variable with a symbol, you can get its name in the global registry. This is done using the methodSymbol.keyFor. Let's try it. Let's create a symbol with the name:

let sym = Symbol.for('test');

Let's get his name from his variable:

let key = Symbol.keyFor(sym); console.log(key); // выведет 'test'

Two characters are given:

let sym1 = Symbol.for('test1'); let sym2 = Symbol.for('test2');
