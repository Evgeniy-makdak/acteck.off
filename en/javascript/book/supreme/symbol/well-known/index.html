Well Known Symbols in JavaScript

There are special built-in characters that are used by JavaScript for utility purposes. Such symbols are calledwell-known symbols (well known symbols).

An example of such a symbol isSymbol.iterator, containing a function that iterates through arrays. Let's check its presence in some array:

let arr = [1, 2, 3]; let func = arr[Symbol.iterator]; console.log(func);
