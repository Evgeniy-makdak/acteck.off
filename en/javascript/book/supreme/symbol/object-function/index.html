Function in an object in JavaScript

Using symbols, you can add functions to objects and these functions will not participate in the search. Let's try it. Let us have an object:

let obj = {a: 1, b: 2, c: 3};

Let's create a symbol:

let sym = Symbol();

Let's write a function with a key in the form of our symbol into the object:

obj[sym] = function() { console.log('!!!'); };

When iterating over an object, our function will not iterate:

for (let key in obj) { console.log(obj[key]); // 1, 2, 3 }

Let's get our function by passing our symbol (the same variable) as the key:

let func = obj[sym]; func();

You can shorten the code:

obj[sym]();
