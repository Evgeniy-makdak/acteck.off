Operations on objects in functions in JavaScript

Let's add a function to the object that will perform some useful operation. For example, it will find the sum of the elements of an object.

Let us have an object:

let obj = {a: 1, b: 2, c: 3};

Let's create a symbol:

let sym = Symbol();

Let's write the function:

obj[sym] = function() { };

In a function bound to an objectthiswill point to the object itself:

obj[sym] = function() { console.log(this); // {a: 1, b: 2, c: 3} };

Let's find the sum of the elements of an object using our function:

obj[sym] = function() { let sum = 0; for (let key in this) { sum += this[key]; } return sum; };

Let's call our function, receiving the sum of the object's elements:

let sum = obj[sym](); console.log(sum); // выведет 6

Given an array:

let arr = [1, 2, 3];

Using the method described in the lesson, add a function to the array that will return the sum of the array elements.

Call the created function and make sure that it finds the sum correctly.

Add multiple elements to an array using the methodpush. Make sure that the function will find the sum taking into account the new elements.
