Iterating over an object with characters in JavaScript

Type advantageSymbolas an object key is that such keys will not participate in the loop.

Let's look at an example. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let's add a new element to this object with a key from the symbol:

let sym = Symbol(); obj[sym] = 'text';

Let's loop through this object. As a result, we will see all the elements except ours with the key from the symbol:

for (let key in obj) { console.log(obj[key]); // 1, 2, 3 }

Make an object with a key from a symbol. Loop through it. Make sure that the symbol is not being searched.
