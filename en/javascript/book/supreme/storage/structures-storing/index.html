Storing structures in localStorage in JavaScript

Local storage can only store strings. However, it is possible to store arrays and objects - for this you can simply useJSON format.

Let's save the array:

let arr = [1, 2, 3, 4, 5]; localStorage.setItem('arr', JSON.stringify(arr));

Now we get it back:

let str = localStorage.getItem('arr'); let res = JSON.parse(str); console.log(res);

Inputs and a button are given. By clicking the button, you will receive the texts of all inputs in the form of an array and save this array to local storage.

Modify the previous problem. Make sure that the next time you visit the page, the inputs contain the values ​​saved in the local storage.
