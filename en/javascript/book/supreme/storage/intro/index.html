Browser local storage in JavaScript

In the browser there is something calledlocal storage, which allows you to store data between user visits to the site (5-10megabytes of information).

Data is accessed by key: you save data with some key, and then you can retrieve it using this key or delete it. In this case, only rows are allowed to be saved. There are special methods for working with data. Let's look at them.

Two types of storages are used to store data:sessionStorageAndlocalStorage. The first one stores data only until the browser is closed, and the second one always.
