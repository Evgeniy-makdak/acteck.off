Deleting data in localStorage in JavaScript

Using the methodremoveItemyou can delete data and the associated key. Let's delete the contents of a key as an example:

localStorage.removeItem('key');

Let's now check the availability of data:

let value = localStorage.getItem('key'); console.log(value); // выведет null
