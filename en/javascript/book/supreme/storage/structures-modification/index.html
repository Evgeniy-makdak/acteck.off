Modifying stored structures in localStorage in JavaScript

Let's say some array is stored in local storage:

let arr = [1, 2, 3, 4, 5]; localStorage.setItem('data', JSON.stringify(arr));

Suppose now we need to somehow modify this array, for example, add another element to the end or change an existing one.

To solve the problem, we will obtain a string with an array stored in the storage, convert this string into an array, perform the necessary manipulations with this array, convert this array back to a string and write it back to the storage:

let json = localStorage.getItem('data'); let data = JSON.parse(json); data.push(6); data[0] = '!'; localStorage.setItem('data', JSON.stringify(data));
