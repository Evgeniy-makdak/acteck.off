Overwriting data in localStorage in JavaScript

You can overwrite data stored in storage under a specific key. Let's try it. Let's write down some data:

localStorage.setItem('key', '1'); let value1 = localStorage.getItem('key'); console.log(value1); // выведет '1'

Now let's rewrite this data:

localStorage.setItem('key', '2'); let value2 = localStorage.getItem('key'); console.log(value2); // выведет '2'

The user visits the site, then refreshes the page, then refreshes it again, and so on. Make a page refresh counter. Each time you update, display the counter value on the screen.
