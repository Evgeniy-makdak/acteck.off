Numbering DOM elements in JavaScript

Understanding iterators makes it easy to add numbers to DOM elements. Let, for example, we have paragraphs:

<p>text</p> <p>text</p> <p>text</p>

Let's get a collection of these paragraphs into a variable:

let elems = document.querySelectorAll('p');

Let's use the built-in iterator to iterateentries:

for (let entry of elems.entries()) { console.log(entry); }

Let's use destructuring to separate numbers from elements:

for (let [num, elem] of elems.entries()) { console.log(num, elem); }

Add a serial number to each paragraph at the end:

for (let [num, elem] of elems.entries()) { elem.textContent += num; }
