Generator Loops in JavaScript

You can use loops inside generators. At the same time, we can pause the cycles usingyield. Let's look at an example. We will pause the loop every iteration:

function *func() { for (let i = 1; i <= 3; i++) { yield i; } }

We get an iterator:

let iter = func();

Let's check the operation of our iterator:

console.log(iter.next()); // {value: 1, done: false} console.log(iter.next()); // {value: 2, done: false} console.log(iter.next()); // {value: 3, done: false} console.log(iter.next()); // {value: undefined, done: true}
