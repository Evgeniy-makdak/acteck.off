Generators in JavaScript

There are special functions with which you can create your own iterators. Such functions are calledgenerators. The names of such functions must begin with an asterisk:

function *func() { }

The keyword is used inside generatorsyield, indicating what the iterator should return on the next call. For example, let's make the first call return1, second -2, and the third -3:

function *func() { yield 1; yield 2; yield 3; }

The generator returns an iterator as its result:

let iter = func();

Let's check the operation of our iterator:

console.log(iter.next()); // {value: 1, done: false} console.log(iter.next()); // {value: 2, done: false} console.log(iter.next()); // {value: 3, done: false} console.log(iter.next()); // {value: undefined, done: true}
