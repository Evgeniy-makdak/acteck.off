Object Iterator in JavaScript

Let's create an iterator that can be used to iterate over an object. First, let's create a generator that takes an object as a parameter and iterates through it:

function *func(obj) { for (let key in obj) { yield obj[key]; } }

Let's now create an iterator:

let iter = func({a: 1, b: 2, c: 3});

Let's loop through the iterator:

for (let elem of iter) { console.log(elem); // 1, 2, 3 }

Make an iterator that will iterate over an object and return an array with each call, the zeroth element of which will contain the key, and the first element will contain the value of the object element. Example:

let iter = func({a: 1, b: 2, c: 3}); for (let elem of iter) { console.log(elem); // ['a', 1], ['b', 2], ['c', 3] }
