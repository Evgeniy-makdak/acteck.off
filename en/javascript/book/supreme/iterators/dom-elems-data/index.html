Retrieving DOM element data in JavaScript

Destructuring allows us to retrieve text and DOM attribute values ​​directly in a loop. Let's figure out how it works. Let's say we have the following paragraphs:

<p id="id1">text1</p> <p id="id2">text2</p> <p id="id3">text3</p>

Let's get a collection of these paragraphs into a variable:

let elems = document.querySelectorAll('p');

Let's go through the elements in a loop, separating the numbers and the elements themselves:

for (let [key, elem] of elems.entries()) { console.log(key, elem); }

Now let's destructure the elements, getting them from themidand texts:

for (let [key, {id, textContent}] of elems.entries()){ console.log(key, id, textContent); }

The following code is given:

<input id="id1" value="111"> <input id="id2" value="222"> <input id="id3" value="333">
