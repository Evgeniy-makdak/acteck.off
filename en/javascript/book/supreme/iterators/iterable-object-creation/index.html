Creating an Iterable Object in JavaScript

Let's create a real iterable object that can be iterated throughfor-of. To do this, the object needs to add the corresponding function toSymbol.iterator.

Let's implement it. Let us have the following object:

let obj = { a: 1, b: 2, c: 3, };

Let's add a function toSymbol.iterator:

obj[Symbol.iterator] = function() { }

Let's turn this function into a generator:

obj[Symbol.iterator] = function *() { }

Inside our functionthiswill refer to the object itself:

obj[Symbol.iterator] = function *() { console.log(this); // сам объект }

Let's start iterating over the object:

obj[Symbol.iterator] = function *() { for (let key in this) { yield obj[key]; } }

That's it, the object can be iterated through a loopfor-of:

for (let elem of obj) { console.log(elem); // 1, 2, 3 }

Make an object that can be looped overfor-of. Let the element of the loop contain an object in the keykeywhich will be the key of the object being iterated, and in the keyval- meaning. Example:

let obj = {a: 1, b: 2, c: 3}; for (let elem of obj) { console.log(elem); }

Will output:

{key: 'a', val: 1} {key: 'b', val: 2} {key: 'c', val: 3}
