Iterator function in JavaScript

All iterable objects have a special structure callediterator. An iterator is a function that allows you to iterate over an object.

Let's look at an example array:

let arr = [1, 2, 3];

The iterator function for any iterable object is stored with a key in the form of a well-known symbolSymbol.iterator:

let func = arr[Symbol.iterator];

We can verify that a function is stored under this key (but its code will be hidden from us by the browser):

console.log(func); // функция

Check the various types of iterable objects you know to see if they contain the described function.
