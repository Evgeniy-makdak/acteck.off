Three built-in iterators in JavaScript

In fact, each iterable object contains not one iterator, but three:values, keysAndentries. Each object type has its own default iterator. For arrays this isvalues, and, for example, for a collectionMap, Thisentries.

With this knowledge, we can now apply this principle to any collection. For example, let's say we have a collectionMap:

let map = new Map(); map.set('a', 1); map.set('b', 2); map.set('c', 3);

The default iterator for this collection isentries:

for (let elem of map) { console.log(elem); // ['a', 1], ['b', 2], ['c', 3] }

But we can easily get the keys of our collection:

for (let elem of map.keys()) { console.log(elem); // 'a', 'b', 'c' }

We can also get the values:

for (let elem of map.values()) { console.log(elem); // 1, 2, 3 }
