The spread operator and iterators in JavaScript

Operatorspreadexpands any object that has an iterator. This will take the default iterator. Let's look at an example. Let us have the following collectionMap:

let map = new Map(); map.set('a', 1); map.set('b', 2); map.set('c', 3);

By default, decomposition will result in a two-dimensional array:

let arr = [...map]; console.log(arr); // [['a', 1], ['b', 2], ['c', 3]]

To obtain an array of keys, we expand the corresponding iterator:

let arr = [...map.keys()]; console.log(arr); // ['a', 'b', 'c']

Let's do the same to get an array of values:

let arr = [...map.values()]; console.log(arr); // [1, 2, 3]
