Generator via Computed Property in JavaScript

In the previous lesson, we assigned a generator for an object with a separate command. This was our object:

let obj = { a: 1, b: 2, c: 3, };

This is how we set the generator:

obj[Symbol.iterator] = function *() { for (let key in this) { yield obj[key]; } }

In fact, we can rewrite our code differently viaвычисляемое свойство:

let obj = { a: 1, b: 2, c: 3, [Symbol.iterator]: function *(){ for (let key in this){ yield this[key]; } } };

Let's check the search:

for (let elem of obj) { console.log(elem); // 1, 2, 3 }
