Built-in String Iterators in JavaScript

Strings also have a built-in iterator. For example, let's loop through the characters of a string:

for (let elem of 'abc') { console.log(elem); // 'a', 'b', 'c' }

Now let's expand the string through the operatorspread:

console.log([...'abc']); // ['a', 'b', 'c']
