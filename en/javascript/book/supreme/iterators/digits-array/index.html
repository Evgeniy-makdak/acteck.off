Array of number digits in JavaScript

Let's get an array of digits of a number. Let the following number be given:

let num = 12345;

An attempt to expand a number throughspreadwill result in an error since the numbers are not iterable:

let num = 12345; let arr = [...num]; // ошибка console.log(arr);

To solve the problem, convert the number to a string:

let num = 12345; let arr = [...String(num)]; console.log(arr); // ['1', '2', '3', '4', '5']

However, we ended up with an array of strings, not numbers. Let's fix the problem with a tricky trick using the methodmap:

let num = 12345; let arr = [...String(num)].map(Number); console.log(arr); // [1, 2, 3, 4, 5]
