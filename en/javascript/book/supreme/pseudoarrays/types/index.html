Types of pseudo-arrays in JavaScript

Pseudo-arrays come in various types. The type depends on the method of receipt.

In the following example we will get a collection of one type:

let elems = document.querySelectorAll('p'); console.log(elems); // NodeList

And the following example shows a collection of a different type:

let elems = document.getElementsByTagName('p'); console.log(elems); // HTMLCollection

Check what type of collection will be in the propertychildNodes.

Check what type of collection will be in the propertychildren.
