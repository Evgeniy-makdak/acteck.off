Similarities to Arrays in JavaScript

Let's see how pseudo-arrays are similar to regular arrays.

Indexes

The elements of a pseudo-array, like the elements of a regular array, can be referred to by their number:

console.log(elems[0]); console.log(elems[1]); console.log(elems[2]);

Length

You can also find out the number of elements using the propertylength:

console.log(elems.length);

Cycle

You can loop through the elements of a pseudo-array:

for (let elem of elems) { console.log(elem); }

Practical problems

Paragraphs given:

let elems = document.querySelectorAll('p');
