Asynchrony of the event model in JavaScript

You know that using the methodaddEventListenerYou can subscribe to various events that occur on page elements. For example, let’s subscribe to a click on some element, the link to which is stored in a variableelem:

elem.addEventListener('click', function() { console.log('1'); }); console.log('2');

The above code will print the second output to the console first. When will the first one be executed? Once: execution of this code waits for an event to occur - a click on an element. As soon as this happens, our code will be executed.

Tell us in what order the numbers will be output to the console:

elem1.addEventListener('click', function() { console.log('1'); }); elem2.addEventListener('click', function() { console.log('2'); });
