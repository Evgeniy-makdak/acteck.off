Setting up a Webpack build to run in JavaScript

Now on filepackage.jsonyou must have the following modules enabled:

{ "type" : "module" }

Let's also place the setting in this file"scripts"for convenient launch of Webpack:

{ "type" : "module", "scripts" : { "build" : "webpack" } }

After this, our collector can be called with the following command:

npm run build

Make the appropriate changes to the filepackage.json.
