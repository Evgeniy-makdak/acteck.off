Installing the Webpack bundler in JavaScript

To build modules, we will use the popular Webpack builder.

Let's install the collector itself:

npm install webpack --save-dev

Now let's also install the Webpack CLI, which allows you to access the bundler via the command line:

npm install webpack-cli --save-dev

Install Webpack and Webpack CLI.

Make sure Webpack is installed by running the check version command.
