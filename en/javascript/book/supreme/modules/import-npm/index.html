Importing npm modules into JavaScript

You can connect not only your own modules, but also modules installed via npm. In this case, for your modules you need to indicate in front./, but for npm modules it is not necessary.

Let's look at an example. Let's install the libraryunderscore:

npm install underscore

Let's connect it:

import _ from 'underscore';

Let's use some functions of the connected library:

import _ from 'underscore'; let arr = [1, 2, 3, 4, 5] let res = _.first(arr) + _.last(arr); console.log(res);

You can import not all functions, but only the necessary ones:

import {first, last} from 'underscore'; let arr = [1, 2, 3, 4, 5] let res = first(arr) + last(arr); console.log(res);
