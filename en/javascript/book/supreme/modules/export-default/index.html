Default export in JavaScript

Only one value can be exported from an ES module. This is done using the commandexport default. When exporting using this command, the imported value will be included in the variable when importing the module.

Let's look at examples. Let's export one function:

export default function() { return 'text'; };

Let's import:

import test from './test.js';

Let's check the function:

let res = test(); console.log(res);
