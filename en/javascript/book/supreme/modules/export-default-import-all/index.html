Import all content when combining exports in JavaScript

With a combination of exports, you can import the default function and all other functions as an object.

Let's see how it's done. Let's say we have the following exports:

export function func1() { return '1' } export function func2() { return '2' } export default function() { return 'text'; };

Import the default function and all other functions with the namemod:

import test, * as mod from './test.js';

Let's check how the default function works:

let res = test(); console.log(res);

Let's check the operation of other functions:

let res1 = mod.func1(); let res2 = mod.func2(); console.log(res1, res2);
