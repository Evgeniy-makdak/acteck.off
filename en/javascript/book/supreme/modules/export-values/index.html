Exporting values ​​from ES modules to JavaScript

You can export not only functions, but also other values. Let's see how it's done.

Export strings

Export the string:

export default 'test';

Let's import:

import str from './test.js'; console.log(str);

Exporting arrays

Exporting the array:

export default [1, 2, 3, 4];

Let's import:

import arr from './test.js'; console.log(arr);

Exporting objects

Exporting the object:

export default { a: 1, b: 2, c: 3 };

Let's import:

import obj from './test.js'; console.log(obj);

Practical problems

Make a module that exports an array of numbers. Connect this module to another file and find the sum of the elements of the connected array.

Make a module that exports three numbers. Connect this module to another file and find the sum of these numbers.
