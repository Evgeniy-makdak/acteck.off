Renaming when importing into JavaScript

You can rename functions when importing. This is done using the commandas. For example, let's rename the functionroot2into a functionsqrt:

import {root2 as sqrt, root3} from './math.js';

Let's check the code:

let res = sqrt(2) + root3(3); console.log(res);

When importing your module, rename the functionpow2Vsquare, and the functionpow3Vcube.
