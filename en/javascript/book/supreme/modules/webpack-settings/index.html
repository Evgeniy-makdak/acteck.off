Setting up the Webpack bundler in JavaScript

The installed collector must be configured. This is done in a special configuration filewebpack.config.js. In this file the settingmodeadjusts the build mode, settingentryspecifies the entry point, and the settingoutput- name of the bundle.

Let's create a settings file with the following content:

export default { mode: 'development', entry: './src/index.js', output: { filename: './dist/bundle.js', } };

Create the described configuration file in your working folder.
