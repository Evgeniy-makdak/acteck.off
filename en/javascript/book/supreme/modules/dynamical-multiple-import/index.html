Dynamic Multiple Imports in JavaScript

You can dynamically import several modules at once:

btn.addEventListener('click', function() { Promise.all([ import('./module1.js'), import('./module2.js'), import('./module3.js'), ]) .then(([module1, module2, module3]) => { }); });

Make a module with an array of names of days of the week and a module with the names of months.

By clicking the button, import these modules. Display the names of days and months as listsul.
