Connecting ES module in JavaScript

Let's now connect the module to another file. This is done using the commandsimportAndfrom.

After the commandimportThe names of the imported functions should be written in curly braces. And after the commandfrom- path to the imported file.

In our case we will import functionsroot2Androot3:

import {root2, root3} from './math.js';

After such a connection, we will have access to the functionsroot2Androot3. Let's use them:

let res = root2(2) + root3(3); console.log(res);
