Перенос npm проекта в JavaScript

Папка node_modules занимает много места и содержит много мелких файлов. Поэтому, когда вы заливаете свой проект на хостинг или даете проект коллеге, вам не стоит отдавать папку с библиотеками.

Дело в том, что ваш коллега легко сможет установить библиотеки, прописанные в файле package.json. Давайте посмотрим, как это делается. Разместите в пустой папке следующий файл:

{ "dependencies": { "jquery": "^3.6.1", "lodash": "^4.17.21" } }

Теперь выполните команду терминала, которая установит все проекты, описанные в package.json:

npm install

Пусть вам прислали следующий package.json:

{ "dependencies": { "jquery": "^3.6.1", "underscore": "^1.13.6" }, "devDependencies": { "webpack": "^5.74.0" } }

Положите его в пустую папку и запустите установку библиотек из этого файла.
