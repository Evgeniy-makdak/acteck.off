Running scripts via npm in JavaScript

In filepackage.jsonthere is a special settingscripts, which allows you to run any terminal commands via npm. This is used to make long commands more enjoyable.

Let's see how to work with this. For example, let us have the utility installedsloc:

npm install -D sloc

Suppose we need to use it to read the contents of a foldersrc:

npx sloc src

Let's make the above command nicer. First, let's see what we have in the file nowpackage.json:

{ "devDependencies": { "sloc": "^0.2.1" } }

Let's now write the setting into itscripts. This setting will be an object with command names and their values. Let's come up with some name for the counting command, for examplecount. As the value, we specify the terminal command that we want to run:

{ "devDependencies": { "sloc": "^0.2.1" }, "scripts": { "count": "npx sloc src" } }

Now to call this command we must write in the terminalnpm, then the commandrunand then the team name we came up with:

npm run count
