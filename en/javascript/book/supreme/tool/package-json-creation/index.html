Создание файла package.json в JavaScript

Можно создать файл package.json со всеми нужными настройками через терминал. Для этого нужно запустить следующую команду:

npm init

После запуска команды терминал будет спрашивать у вас значения параметров, а после этого создаст файл package.json.

Если вы не хотите отвечать на вопросы, то с помощью флага -y можно заставить терминал ответить "да" на все вопросы системы:

npm init -y

Откройте в терминале папку и создайте в ней через терминал файл package.json.
