Creating a package.json file in JavaScript

You can create a filepackage.jsonwith all the necessary settings via the terminal. To do this you need to run the following command:

npm init

After running the command, the terminal will ask you for parameter values, and then create a filepackage.json.

If you do not want to answer questions, then use the flag-yYou can force the terminal to answer “yes” to all system questions:

npm init -y

Open a folder in the terminal and create a file in it through the terminalpackage.json.
