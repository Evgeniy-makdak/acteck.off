Updating packages via npm in JavaScript

To update packages use the following command:

npm update

Upon receiving this command, npm will check all packages for new versions, and if it finds new versions that meet the package version restrictions specified inpackage.json, then it will install them.

You can also update a separate package:

npm update lodash
