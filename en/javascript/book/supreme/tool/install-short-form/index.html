Short installation form via npm in JavaScript

All npm commands have two forms: long and short. The long form is more understandable, and the short form is faster to write, which is often used by professionals.

For example, the commandinstallwill have a short formi. Let's try the short form:

npm i lodash

Rewrite the command in a short form:

npm install underscore
