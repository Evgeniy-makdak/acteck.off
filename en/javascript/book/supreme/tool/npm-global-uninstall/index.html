Global package removal via npm in JavaScript

To remove libraries installed globally, use the commanduninstallwith a flag-g:

npm uninstall -g somelib

Depending on how you installed NodeJS, you may need to writesudobefore the command (usually for Linux):

sudo npm uninstall -g somelib

Let's remove the previously installed utilitysloc:

npm uninstall -g sloc
