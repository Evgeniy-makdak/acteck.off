Dev dependencies in npm in JavaScript

There are also libraries that are needed only during the development process as a tool for the programmer's work. An example of such libraries is the Webpack or Gulp bundlers (we will study them later).

Such libraries are also installed via npm. However, they should not be installed in regular dependencies, but in special dependencies for the developer.

To do this, you need to call the installation with the flag--save-dev:

npm install --save-dev webpack

Or in shorter form:

npm install -D webpack

Or even shorter:

npm i -D webpack

As a result, packages installed by such a command will be included in the settingsdevDependencies:

{ "devDependencies": { "webpack": "^5.74.0" } }

Install the Gulp library:

npm install --save-dev gulp
