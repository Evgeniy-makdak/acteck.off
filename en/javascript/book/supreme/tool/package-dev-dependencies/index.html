Dev зависимости в npm в JavaScript

Существуют также библиотеки, которые нужны только в процессе разработки как инструмент для работы программиста. Примером таких библиотек служат сборщики Webpack или Gulp (мы их будем изучать в дальнейшем).

Такие библиотеки также устанавливаются через npm. Однако, их следует устанавливать не в обычные зависимости, а в специальные зависимости для разработчика.

Для этого нужно вызывать установку с флагом --save-dev:

npm install --save-dev webpack

Или в более короткой форме:

npm install -D webpack

Или еще короче:

npm i -D webpack

В результате пакеты, установленные такой командой попадут в настройку devDependencies:

{ "devDependencies": { "webpack": "^5.74.0" } }

Установите библиотеку Gulp:

npm install --save-dev gulp

Изучите файл package.json.
