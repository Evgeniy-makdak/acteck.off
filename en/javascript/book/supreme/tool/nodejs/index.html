Introduction to NodeJS Tool

NodeJS is JavaScript that runs on the server. A server means a special computer on the Internet on which your website will be hosted. The computer on which the site is being developed can also be made a server.

NodeJS allows the server to generate HTML code, which will then be sent to the browser (toclient). In addition, NodeJS allows you to work with the database and the server file system.

Additionally, NodeJS is a framework that will run handy JavaScript tools on your computer. It is in this capacity that we still need him. And dedicated to using NodeJS as a serverseparate textbook.
