Dependencies in the package.json file in JavaScript

When installing packages, npm creates a special filepackage.json, in which it lists all installed dependencies and their versions.

For example, if we have the jquery library and the lodash library installed, then this file could look like this:

{ "dependencies": { "jquery": "^3.6.1", "lodash": "^4.17.21" } }
