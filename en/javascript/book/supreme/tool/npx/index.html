npx tool in javascript

Currently, installing packages globally is considered bad form. The fact is that such an installation clutters the operating system and also does not allow you to simultaneously keep several versions of the library for different projects.

It is better to install software packages locally in the project, and use the special npx tool to launch them.

Let's see how it's done. Let's install the utility locally in the projectsloc(in dev notation):

npm install -D sloc

Now, being in our project folder, we can call the installed utility:

npx sloc .
