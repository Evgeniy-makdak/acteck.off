Global installation via npm in JavaScript

Some npm packages are not just libraries, but programs that, once installed, add a new command to the terminal.

Such programs should be installedglobally. This means that they will not be installed in the foldernode_modulesproject, but simply on your computer as a program.

To start the global installation mode, you need to pass the flag-g:

npm install -g somelib

Depending on how you installed NodeJS, you may need to writesudobefore the command (usually for Linux):

sudo npm install -g somelib

Let's look at an example. Let's take the utilitysloc, which counts the number of lines of code in the project. Let's install it globally:

npm install -g sloc

While in the folder with any project, run the command in the terminal to count lines, passing the path to the folder (dot - current folder) as a command parameter:

sloc .

Install the library globallycowsay. Once installed in the terminal, refer to this library by making the cow say something to the terminal.
