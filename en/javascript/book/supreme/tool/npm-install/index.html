Installing libraries via npm in JavaScript

Using npm, installing the required library is very easy. You need to open a terminal in your project folder and run the following command:

npm install имя_библиотеки

For example, let's install the librarylodash:

npm install lodash

The installed library will go into the foldernode_modules. At the same time, if the library has dependencies, they will also be installed in this folder as separate libraries.

Libraries installed via npm can be included in the HTML file. But this should not be done, since they representmodules. Such libraries should be connected in a special way. We will go through this in the following sections of the tutorial. For now, let's just learn how to install libraries via npm.
