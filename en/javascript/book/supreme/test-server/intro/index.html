Тестовый сервер на JavaScript

Для дальнейшей работы вам понадобится развернуть тестовый сервер на своем компьютере. Этот сервер написан и подготовлен мною под наши задачи.

Для начала скачайте сервер по следующей ссылке: test-http-server.

Откройте папку с сервером в терминале и выполните следующую команду:

npm install

После завершения установки выполните следующую команду:

npm run start

После этого сервер запустится и вы сможете зайти на него через браузер открыв в нем адрес http://localhost:3001/.

Установите тестовый сервер. Проверьте его работу, открыв в браузере указанную страницу.
