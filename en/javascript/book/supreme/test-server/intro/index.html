JavaScript test server

For further work, you will need to deploy a test server on your computer. This server was written and prepared by me for our tasks.

To get started, download the server from the following link:test-http-server.

Open the server folder in a terminal and run the following command:

npm install

Once the installation is complete, run the following command:

npm run start

After this, the server will start and you can access it through a browser by opening the address in ithttp://localhost:3001/.

Install a test server. Check its operation by opening the specified page in your browser.
