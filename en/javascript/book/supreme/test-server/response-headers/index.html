Response headers in a JavaScript test server

Our server can send various HTTP headers. To do this, use the second callback parameter:

export default { '/handler/': function(data, resp) { // параметр resp для заголовков } }

Using the methodsetHeaderwe can give any title. For example, let's give the MIME type for JSON:

export default { '/handler/': function(data, resp) { resp.setHeader('Content-Type', 'application/json'); return '[1,2,3]'; } }
