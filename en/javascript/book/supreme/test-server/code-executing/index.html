Executing code through a test server in JavaScript

In the server folder, find the fileserver.js. In this file you can write JavaScript that will be executed on the server.

This file contains an object withroutes, the keys of which will be addresses, and the values ​​will be functions that will be executed upon entering the corresponding pages through the browser.

Let's look at an example. Place the following code in this file:

export default { '/test/': function() { return 'hello world'; } }

After that, stop the server by pressing the keyboard shortcut in the consoleCtrl + C. Then restart it with the command:

node index.js

Now you can access through your browser athttp://localhost:3001/test/. As a result, our function will be executed and the text that it will return viareturn.

We can place any JavaScript code in this function. For example, let's write the following:

export default { '/test/': function() { let num1 = 1; let num2 = 2; return num1 + num2; } }

Make address handlers/page1/, /page2/, /page3/. Access them through your browser.
