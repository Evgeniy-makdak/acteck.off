Using a test server in JavaScript

Open the folder where the server is located. There you will see a folderpublic. This folder will be the root folder of your site. This means that any file placed in this folder will be accessible from the browser via a direct link.

Let's look at an example. Let's file/public/page.htmlLet's create the HTML page layout:

<!DOCTYPE html> <html lang=""> <head> <meta charset="utf-8"> <title>page</title> </head> <body> page.html works! </body> </html>

To access this page in a browser, you will need to go tohttp://localhost:3001/page.html- that is, without specifying a folderpublic.

Make a page/public/dir/test.html. Access it through your browser.

Make a page/public/style.css. Access it through your browser.

Make a page/public/script.js. Access it through your browser.
