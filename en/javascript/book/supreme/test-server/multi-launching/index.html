Multiple launches of a test server in JavaScript

You can run multiple copies of the test server. To do this, each copy needs to specify its own port.

For example, let's start the first server on3001port:

export default { port: '3001', }

And the second server is on3002port:

export default { port: '3002', }

Make several copies of the server. Run each copy on a different port.
