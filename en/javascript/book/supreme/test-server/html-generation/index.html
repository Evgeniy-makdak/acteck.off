HTML generation via JavaScript test server

Test server handlers can also send HTML code:

export default { '/test/': function() { return '<b>text</b>'; } }

You can also generate HTML code in a loop:

export default { '/test/': function() { let str = '<ul>'; for (let i = 1; i < 9; i ++) { str += '<li>' + i + '</li>'; } str += '</ul>'; return str; } }

Let the server be given an array:

let arr = [1, 2, 3, 4, 5];

Make a route that will write each element of the array into its own paragraph and send the result to the server.
