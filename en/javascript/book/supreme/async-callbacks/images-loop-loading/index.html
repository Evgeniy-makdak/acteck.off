Asynchronous loading of images in a loop in JavaScript

Let the paths to the pictures be stored in an array:

let arr = ['img1.png', 'img2.png', 'img3.png'];

We can load these images in a loop:

for (let path of arr) { loadImage(path, function(image, err) { document.body.append(image); }); }

The code turned out beautiful without callback hell, however, we returned to our two problems: the order of the pictures is not guaranteed and it is impossible to catch the moment when all the pictures are loaded.

And there is no solution in this situation: it is impossible to start a loop, use an asynchronous function inside it, and then catch the moment of completion of all functions of the loop. Either you don’t need to catch this moment and the above code will suit you, or welcome to callback hell.

But don’t be upset - the solution to the problem is possible through promises, which we will study in the following lessons.
