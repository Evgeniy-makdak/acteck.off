Asynchronous code with callbacks in JavaScript

Let's say we have some asynchronous function:

function make() { setTimeout(function() { console.log('1'); }, 3000); }

Let's use this function like this:

make(); console.log('2'); // выполнится первым

Let's say we want to make the second output to the console happen after the asynchronous operation inside the function is completed. One approach used for this is to usecallback: wrap the waiting code in an anonymous function and pass it as a parameter to the functionmake:

make(function() { console.log('2'); });

Of course, this alone will not solve our problem. For now we just made the following agreement:optionally execute the code after triggeringmakepass this code as a callback to the callmake.

Let's fix the function codemakeso that it starts working according to our agreement:

function make(callback) { setTimeout(function() { console.log('1'); // некая асинхронная операция, может не одна callback(); // затем наш коллбэк }, 3000); }

Tell us in what order the numbers will be output to the console:

function make(callback) { setTimeout(function() { console.log('1'); callback(); }, 3000); } make(function() { console.log('2'); console.log('3'); });
