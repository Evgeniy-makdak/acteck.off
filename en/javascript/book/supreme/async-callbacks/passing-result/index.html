Asynchronously passing the result to a callback in JavaScript

Now suppose that the asynchronous operation, after its completion, does not output anything to the console, but receives a certain result. Let this be an array of data, which, for example, could be obtained via AJAX. But since we don’t know how to work with AJAX yet, we’ll simply simulate this receipt:

function make() { setTimeout(function() { let res = [1, 2, 3, 4, 5]; // массив с результатом }, 3000); }

Let's make sure that the array with the result is passed to the callback parameter:

function make(callback) { setTimeout(function() { let res = [1, 2, 3, 4, 5]; callback(res); // передаем результат параметром }, 3000); }

Now, when passing the callback to the function callmakewe can write a parameter in it - and the result of the asynchronous operation will go into this parameter:

make(function(res) { console.log(res); // наш массив });

Add the callback code so that it finds the sum of the array elements with the result.
