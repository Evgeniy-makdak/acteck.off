Loading images via asynchronous callbacks in JavaScript

Let's implement the functionloadImage, which will load pictures. Let this function take the path to the image as the first parameter, and the second parameter be the callback that will be executed when the image is loaded:

loadImage('img.png', function() { // выполнится по загрузке картинки });

Let the first parameter of our callback contain a reference to the DOM element of the image, and the second parameter contain an error if an exception occurs:

loadImage('img.png', function(image, err) { console.log(image, err); });

We can use our function like this:

loadImage('image.png', function(image, err) { document.body.append(image); // разместим картинку по загрузке });

Or with exception handling:

loadImage('image.png', function(image, err) { if (!err) { document.body.append(image); } else { console.log('произошла ошибка: ' + err); } });
