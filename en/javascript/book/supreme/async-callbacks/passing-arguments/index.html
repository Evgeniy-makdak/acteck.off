Passing parameters to an asynchronous callback in JavaScript

Let’s now make it possible to pass input parameters to an asynchronous function. For example, let as the first parameter of the functionmakewe will pass the number of the array element that we want to receive as a result. For example, let's get the third element of the array:

make(3, function(res) { console.log(res); // третий элемент массива });

Let's rewrite our function codemakeas described:

function make(num, callback) { setTimeout(function() { let arr = [1, 2, 3, 4, 5]; callback(arr[num]); // результатом передаем элемент массива }, 3000); }

Make the functionmaketook two parameters: the number of one and the other element of the array. Let the result of an asynchronous operation be that this function returns the sum of the specified elements.
