Changing JSON Data in JavaScript

Let's say we have a string containing an array in JSON format:

let json = '[1, 2, 3, 4, 5]';

Let's add another element to the end of this array. To do this we need to go through a number of steps.

First, let's unpack the JSON into a JavaScript array:

let arr = JSON.parse(json);

Let's add a new element to the array:

arr.push(6);

Let's convert the modified array back to JSON:

let res = JSON.stringify(arr);

The final code will look like this:

let json = '[1, 2, 3, 4, 5]'; let arr = JSON.parse(json); arr.push(6); let res = JSON.stringify(arr); console.log(res);
