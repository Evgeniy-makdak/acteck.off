Convert JavaScript data structures to JSON format

Using the methodJSON.stringifyYou can convert JavaScript arrays and objects to JSON format. Let's look at an example.

Let us have an array with data:

let arr = [1, 2, 3, 4, 5, 'a', 'b'];

Let's convert our array to a string:

let json = JSON.stringify(arr);
