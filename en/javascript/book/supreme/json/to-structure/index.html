Convert JSON to JavaScript data structure

Using the methodJSON.parseit is possible to convert JSON to a JavaScript data structure. Let's look at an example.

Let's say we have a string containing an array in JSON format:

let json = '[1, 2, 3, 4, 5, "a", "b"]';

Let's convert our string to an array:

let arr = JSON.parse(json);

If the line contains incorrect JSON, our method will throw an error:

let json = '[1, 2, 3, 4, 5,]'; // некорректный JSON let arr = JSON.parse(json); // выдаст ошибку
