Introduction to JSON Format in JavaScript

Sometimes you need to convert a data structure, such as an array or an object, into a string. This may be needed, for example, to send this structure over the Internet or save it to some kind of storage.

For this purpose, a special JSON format was invented in JavaScript.

The JSON format can contain one of two structures: either an array or an object with pairskey-value. Arrays and objects are constructed in the same way as in JavaScript, but have the limitation that all strings and object string keys must be enclosed in double quotes.

Let's for example make a string containing an array in JSON format:

let json = '[1, 2, 3, 4, 5, "a", "b"]';

Now let's make a string containing an object in JSON format:

let json = `{ "a": "aaa", "b": "bbb", "c": "ccc", "111": "ddd" }`;

You can combine these structures in any order:

let json = `[ { "a": "aaa", "b": "bbb" }, { "c": "ccc", "d": "ddd" } ]`;

Unlike JavaScript structures, JSON does not allow commas after the last elements of arrays and objects:

let json = '[1, 2, 3, 4, 5,]'; // неверно let json = '[1, 2, 3, 4, 5]'; // верно

In addition to numbers and strings, values ​​can also serve as elementstrue, falseAndnull:

let json = '[null, true, false]';

Given an array:

let arr = [1, 2, 3, 'a', 'b', 'c'];

Manually convert this array to a string in JSON format.

Given object:

let obj = { a: 1, b: 2, c: 'eee', d: true, };

Manually convert this object to a JSON string.

Given object:

let obj = { a: ['a', 'b', 'c',], b: '111', c: 'eee', };

Manually convert this object to a JSON string.
