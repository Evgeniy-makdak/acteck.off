Working with URLs in JavaScript

In JavaScript, sometimes you have to work with link addresses. This is what the class is forURL, which allows you to receive and change parts of addresses.

Let's see how it works using the following address as an example:

let path = 'http://site.ru:3000/dir/eee/page.html#show?a=1&b=2&c=3';

Let's create a class objectURL, passing it our address as a parameter:

let url = new URL(path);

Let's see what properties the created object has.

Protocol

Using propertyprotocolyou can get the protocol:

let res = url.protocol; console.log(res); // выведет 'http://'

Host

Using propertyhostyou can get the domain name with the port:

let res = url.host; console.log(res); // выведет 'site.ru:3000'

Hostname

Using propertyhostnameyou can get the domain name:

let res = url.hostname; console.log(res); // выведет 'site.ru'

Port

Using propertyportyou can get the port:

let res = url.port; console.log(res); // выведет '3000'

Path

Using propertypathnameyou can drop the domain name, getting the path:

let res = url.pathname; console.log(res); // выведет '/dir/eee/page.html#show?a=1&b=2&c=3'

Hash

Using propertyhashyou can get the hash value:

let res = url.hash; console.log(res); // выведет 'show'

GET parameters

Using propertysearchyou can get a string of GET parameters:

let res = url.search; console.log(res); // выведет 'a=1&b=2&c=3'

Using propertysearchParamsyou can get GET parameters as a class objectURLSearchParams:

let res = url.searchParams; console.log(res); // выведет объект класса URLSearchParams
