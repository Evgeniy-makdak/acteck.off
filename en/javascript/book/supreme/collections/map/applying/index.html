Using Map Collections

Let us have paragraphs:

<p>aaa</p> <p>bbb</p> <p>ccc</p> <p>ddd</p> <p>eee</p>

Let's make sure that when you click on each paragraph, its serial number in the list of paragraphs is written at the end. Let's solve the problem throughMap.

Let's get started. First, let's put our paragraphs into a variable:

let elems = document.querySelectorAll('p');

Let's create a new collectionMap:

let map = new Map;

Let's fill our collection so that the keys are our paragraphs, and their values ​​are the ordinal numbers:

let i = 1; for (let elem of elems) { map.set(elem, i++); }

Let's loop through the paragraphs and attach a click handler to them:

for (let elem of elems) { elem.addEventListener('click', function() { }); }

Let's now click to add a serial number to the end of the paragraph text. In this case, we will receive this number from our collection:

for (let elem of elems) { elem.addEventListener('click', function() { this.textContent += map.get(this); }); }

Inputs are given. Loop through these inputs and create a collectionMap, the keys of which will be inputs, and the value will be their serial number on the page. Make it so that when you click on any input, it willvalueits serial number was recorded.
