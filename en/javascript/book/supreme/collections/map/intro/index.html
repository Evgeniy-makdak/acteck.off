Map Collections in JavaScript

CollectionsMaprepresent objects in which both primitives and objects can act as keys.

Let's try it with an example. First, let's create a collection using the following command:

let map = new Map;

After this, you can add elements to the collection using the methodsetand get them using the methodget.

Let's try it. Suppose we want to make two arrays as keys:

let arr1 = [1, 2]; let arr2 = [3, 4];

Let's assign some values ​​to our keys:

map.set(arr1, 'data1'); map.set(arr2, 'data2');

Now let’s read our values ​​by key:

console.log(map.get(arr1)); console.log(map.get(arr2));
