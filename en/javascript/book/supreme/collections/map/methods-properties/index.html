Useful features of Map collections

CollectionsMaphave a number of useful properties and methods. Let's look at them.

Collection size

In propertysizecontains collection size:

console.log(map.size);

Checking for a key

You can check the presence of a key in the collection using the methodhas:

console.log(map.has(arr1));

Delete by key

You can remove an element from a collection using the methoddelete:

map.delete(arr1);

Clearing a collection

You can clear the entire collection using the methodclear:

map.clear();
