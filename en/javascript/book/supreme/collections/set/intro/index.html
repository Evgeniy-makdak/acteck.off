Set Collections in JavaScript

CollectionsSetallow you to create arrays without duplicates. Let's see how to work with such collections. First you need to create a collection using the following command:

let set = new Set;

After creating a collection, you can add new elements to it using the methodadd:

let set = new Set; set.add(1); set.add(2); set.add(3);

However, when you try to add an element that already exists in the collection, a new addition will not occur:

let set = new Set; set.add(1); set.add(2); set.add(3); set.add(3); // не добавится, т.к. уже есть

Create a collectionSet. Using the methodaddadd elements to it. Make sure that duplicate elements are not added.
