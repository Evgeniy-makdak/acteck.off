Removing duplicates from an array using Set

Using CollectionsSetYou can easily remove duplicates from arrays. To do this, the array needs to be converted into a collectionSet. Since this collection cannot contain duplicate elements, they will disappear during conversion. If we then convert the collection back to an array, we will get an array without duplicates.

Let's look at an example. Let's say we have the following array with duplicates:

let arr = [1, 2, 3, 1, 3, 4];

Let's create a collection based on itSet:

let set = new Set(arr);

Now let's convert our collection back to an array:

let arr = [1, 2, 3, 1, 3, 4]; let res = [...new Set(arr)]; console.log(res); // выведет [1, 2, 3, 4]

Write a function that will take an array as a parameter and return this array without duplicates.
