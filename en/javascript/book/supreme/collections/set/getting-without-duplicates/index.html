Getting DOM elements without duplicates

Let's say we have paragraphs and a button:

<p>aaa</p> <p>bbb</p> <p>ccc</p> <p>ddd</p> <p>eee</p> <button>click me</button>

Let the user click on these paragraphs in any order. Let's make sure that when a button is clicked, an exclamation mark is added to the end of each clicked paragraph. Let's solve the problem throughSet.

Let's get started. First, let's put our elements into variables:

let button = document.querySelector('button'); let elems = document.querySelectorAll('p');

Let's create a new collectionSet:

let set = new Set;

Let's loop through the paragraphs and attach a click handler to them:

for (let elem of elems) { elem.addEventListener('click', function() { }); }

Let's now click on a paragraph and add this paragraph to the collection:

for (let elem of elems) { elem.addEventListener('click', function() { set.add(this); }); }

Due to the fact that we are using a collectionSetClicking on a paragraph again will not result in adding a duplicate paragraph.

Let's now, by clicking on the button, iterate through the contents of our collection and add an exclamation point to the end of each paragraph:

button.addEventListener('click', function() { for (let elem of set) { elem.textContent = elem.textContent + '!'; } });

Paragraphs and a button are given. The user clicks on these paragraphs in random order. Make sure that when the button is clicked, an exclamation mark is added to the end of each clicked paragraph.
