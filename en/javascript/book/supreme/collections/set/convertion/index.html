Converting Set Collections

CollectionsSetcan be converted to arrays. You can also perform the reverse operation - convert arrays toSet. Let's see how it's done.

Convert to array

Let us have some collectionSet:

let set = new Set([1, 2, 3]);

You can convert it to an array using the destructuring technique:

let arr = [...set];

Or you can use the methodArray.from:

let arr = Array.from(set);

Collection givenSet. Convert it to an array using the two methods described.

Converting an Array to a Set

Let's say we have an array:

let arr = [1, 2, 3];

Let's convert it to a collectionSet:

let set = new Set(arr);
