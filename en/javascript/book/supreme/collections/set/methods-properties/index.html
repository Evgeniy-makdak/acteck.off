Useful features of Set collections

CollectionsSethave a number of useful properties and methods. Let's look at them.

Collection size

In propertysizecontains collection size:

let set = new Set; set.add(1); set.add(2); set.add(3); console.log(set.size);

Checking for the presence of a value

You can check the presence of a value in the collection using the methodhas:

let set = new Set; set.add(1); set.add(2); set.add(3); console.log(set.has(1));

Removing items

You can remove an element from a collection using the methoddelete:

let set = new Set; set.add(1); set.add(2); set.add(3); set.delete(1);

Clearing a collection

You can clear the entire collection using the methodclear:

let set = new Set; set.add(1); set.add(2); set.add(3); set.clear();
