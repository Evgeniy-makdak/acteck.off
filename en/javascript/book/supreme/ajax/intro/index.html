Introduction to AJAX

AJAX technology is a way to update part of a page without completely reloading it. The technology is based on JavaScript, the HTTP protocol, and some server part, for example, PHP, NodeJS, or some other.

Currently, working with AJAX is implemented throughfetch. This is modern technology, but does not work in older browsers. An older technology is to use an objectXMLHttpRequest. You may still come across this technology, but you shouldn’t delve into it too much.

To learn how AJAX works, you must have a server. We will use ourtraining server. Let's create an HTML file in the folder with the server, which we will open in the browser:

<!DOCTYPE html> <html> <head> <title></title> <script src="client.js" defer></script> </head> <body> </body> </html>

Let's connect the JavaScript file to it:

alert('test');

Create the described files and check their operation in the browser.
