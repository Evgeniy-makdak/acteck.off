Sending GET requests via AJAX in JavaScript

Let's now learn how to transfer data to the server in an AJAX request. The server will do something with this data and send us the result back.

Our test server will process data sent to/handler/:

button.addEventListener('click', function() { let promise = fetch('/handler/'); });

First, let's send a GET request:

button.addEventListener('click', function() { let promise = fetch('/handler/?num=3'); });

Let's get the sent data on the server side:

export default { '/handler/': function({get}) { console.log(get.num); // выведет 3 } }

Let's do something with this data and send it back:

export default { '/handler/': function({get}) { return get.num ** 2; } }

On the client side, we will get the result and output it somewhere:

button.addEventListener('click', function() { fetch('/handler/?num=3').then( response => { return response.text(); } ).then( text => { console.log(text); } ); });

The client is given a div and a button. By clicking the button, send two numbers to the server. Let the server find the sum of the transmitted numbers. Write the result in div.

Let there be an array on the server. Let the server expect a number to be passed as a parameter and return the array element corresponding to that number. When you click the button, send a number to the server, and print the server’s response in a paragraph.
