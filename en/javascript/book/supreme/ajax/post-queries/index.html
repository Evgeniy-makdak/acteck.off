Sending POST requests via AJAX in JavaScript

Let's now send a POST request to the server. To do this, the functionfetchThere is a second parameter with settings. In settingmethodindicates the HTTP request method. Let's specify the POST method:

button.addEventListener('click', function() { let promise = fetch('/handler/', { method: 'post', }); });

Let's also set the data we want to send to the server. In POST requests, data is transmitted in the body of the HTTP request. To do this, you need to specify the data in the settingsbody. Let's specify them as a Query String, setting their type to the appropriate MIME:

button.addEventListener('click', function() { let promise = fetch('/handler/', { method: 'post', body: 'num1=1&num2=2', headers: { 'Content-Type': 'application/x-www-form-urlencoded', }, }); });

Let's get this data on the server, do something with it and send it back:

export default { '/handler/': function({ post }) { return Number(post.num1) + Number(post.num2); } }

The client is given a div and a button. When you click the button, send three numbers to the server using the POST method. Let the server find the sum of the transmitted numbers. Write the result in div.
