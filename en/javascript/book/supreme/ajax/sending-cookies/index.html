Passing cookies in AJAX requests in JavaScript

By default, cookies are not sent in AJAX requests. This means that the server session will not work. Usually we still need to transfer cookies. It can be enabled by settingcredentials.

Meaning'include'will force cookies to be sent in an AJAX request (even if the request is cross-origin):

fetch('https://example.com', { credentials: 'include' });

Meaning'same-origin'will also force cookies to be sent, but only to the site on which the script is run:

fetch('https://example.com', { credentials: 'same-origin' });

Meaning'omit'prohibits the transmission of cookies:

fetch('https://example.com', { credentials: 'omit' });
