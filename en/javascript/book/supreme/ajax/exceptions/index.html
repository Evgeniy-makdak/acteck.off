Exceptional situations with AJAX requests in JavaScript

If an exception occurs when executing an AJAX request (for example, the Internet goes down), the promise will fail with an error. We can catch this error in a way convenient for us, for example, throughcatch:

button.addEventListener('click', function() { let promise = fetch('/ajax.html') .then( response => { return response.text(); }, ).then( text => { console.log(text); } ).catch( error => { alert(error); } ); });

Print the page text if the request was successful and an error if a network error occurred.
