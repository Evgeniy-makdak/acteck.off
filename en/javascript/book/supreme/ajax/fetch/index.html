Basics of working with fetch in AJAX

Let's now see how to load part of a page using AJAX. Let on our pageindex.htmla div and a button will be located (the content is shown herebody):

<div></div> <button>click me</button>

Let us also have a pageajax.html, which contains three paragraphs (no page tags, just paragraphs):

<p>1</p> <p>2</p> <p>3</p>

Let's make it so that when you click on a button on the pageindex.htmlthe contents of the page were loaded into the divajax.html.

To do this, first we’ll get variable references to our elements:

let div = document.querySelector('div'); let button = document.querySelector('button');

Let's attach a click handler to the button:

button.addEventListener('click', function() { // тут будем выполнять AJAX запрос });

Let's now execute the AJAX request. To do this, use the functionfetch, which takes as a parameter the address of the page whose contents we want to receive. We will indicate the page addressajax.htmlon our server:

button.addEventListener('click', function() { let result = fetch('/ajax.html'); });

By its resultfetchwill returnpromise. The fact is that requesting a page via AJAX is an asynchronous operation, because some time will pass before the page responds to us.

Let's get the result of the promise:

button.addEventListener('click', function() { let promise = fetch('/ajax.html'); promise.then(function(response) { // ответ сервера лежит в переменной response }); });

Let's simplify by getting rid of the extra variable:

button.addEventListener('click', function() { fetch('/ajax.html').then(function(response) { }); });

Let's simplify by using the arrow function:

button.addEventListener('click', function() { fetch('/ajax.html').then(response => { }); });

Let's present it in a more readable form:

button.addEventListener('click', function() { fetch('/ajax.html').then( response => { } ); });

In variableresponsecontains a rather complex class objectResponse:

button.addEventListener('click', function() { fetch('/ajax.html').then(response => { console.log(response); // объект класса Response }); });

We will study this object in depth in the following lessons, but for now we need the simplest thing - to get the text of the requested page. For this purpose, there is a method in this objecttext. This method, however, does not return the page text, but a promise with its text:

button.addEventListener('click', function() { fetch('/ajax.html').then( response => { console.log(response.text()); // выведет Promise } ); });

To get to the page text, you need a promise derived fromresponse.text, process again:

button.addEventListener('click', function() { fetch('/ajax.html').then( response => { return response.text(); } ).then( text => { console.log(text); // текст страницы } ); });

So, we finally have the text of the requested page and we can, for example, write it into our div:

button.addEventListener('click', function() { fetch('/ajax.html').then( response => { return response.text(); } ).then( text => { div.innerHTML = text; } ); });

Three buttons and a div are given. There are three pages on the server. Make sure that each button loads the corresponding page into the div.

Let there be five pages on the server. Let the first click on the button load the first page, the second click load the second, and so on until there are no more pages.
