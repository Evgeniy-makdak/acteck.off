Receiving JSON via AJAX in JavaScript

A common practice is for the server to return data in the formatJSON. Let's see how to work with this. Let the server have the following file:

{ "num1": "1", "num2": "2" }

Let's get the data from this file:

button.addEventListener('click', function() { fetch('/data.json').then(response => { return response.text(); }).then(data => { console.log(JSON.parse(data)); }); });

You can use the methodjsonso that the JSON is automatically converted to a JavaScript structure:

button.addEventListener('click', function() { fetch('/data.json').then(response => { return response.json(); }).then(data => { console.log(data); }); });
