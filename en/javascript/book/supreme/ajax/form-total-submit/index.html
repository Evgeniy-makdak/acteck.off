Submitting a form via AJAX using FormData JavaScript

By usingFormDatayou can automatically send all form fields to the server. Let's see how it's done.

Let us have the following form, in which the attributes are given to the inputsname:

<form action="" method="GET"> <input name="num1"> <input name="num2"> <input type="submit"> </form>

Let's submit this form via AJAX. First, let's get a link to the form into a variable:

let form = document.querySelector('form');

Now in the body of the request we will indicate the object as dataFormData. And we’ll pass a link to the form to the object itself as a parameter:

form.addEventListener('submit', function(event) { let promise = fetch('/handler/', { method: 'POST', body: new FormData(this) // передаем ссылку на форму }); event.preventDefault(); });

As a result, on the server we will be able to get the input values ​​by their names from the attributesname:

export default { '/handler/': function({post}) { console.log(post.num1); console.log(post.num2); return 'form data received'; } }

A form is given with five inputs into which numbers are entered. Submit this form to the server using the POST method. Let the server find the arithmetic mean of the entered numbers and send the result back to the browser.
