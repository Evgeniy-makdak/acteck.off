Submitting Forms in JavaScript

You already know how to work with form elements in JavaScript, processing them in the browser. Forms, however, can also be submitted to the server to be processed by the site's backend, written in PHP or NodeJS.

This is necessary in order to add and change site data stored on the server.

Let's see how it's done. Let's say we have several inputs and a button:

<input> <input> <input type="submit">

Let's wrap our elements in a tagform. In this case, we will get a form that, when the button is clicked, will be sent to the server:

<form> <input> <input> <input type="submit"> </form>

In order to receive form data on the server, you need to give each input its own name:

<form> <input name="test1"> <input name="test2"> <input type="submit"> </form>

In this case, the form data will arrive to the server in the form of pairskey-value, where the keys are the names of the form elements, and the values ​​are the entered data.

There is no need for a server to submit the form. You can copy the form code provided to any file, run it in your browser and click on the submit button - the form will be sent and the browser page will reload.

If there is no server, however, the form data will not go anywhere. Therefore, let's check the operation of the form together with the server. You don’t know any server-side languages ​​yet, so let’s practice sending the form using ourtraining HTTP server.

Let's place our form in a file so that it can be accessed athttp://localhost:3001/form.html:

<form> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Let's add an attribute to our formaction, indicating the sending address of our form:

<form action="/handler/"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Now in fileserver.jsLet's write a handler for the form's sending address:

export default { '/handler/': function() { return 'form data received'; } }

The parameter of our handler will contain the submitted form data:

export default { '/handler/': function(data) { console.log(data); // выведется в консоль сервера return 'form data received'; } }

Create a form and a file to process it. Submit the form and check that the data actually arrived on the server.
