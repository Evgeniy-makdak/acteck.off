Submitting forms using the GET method in JavaScript

Let's implement form submission using the GET method using an example. Let us have a form in which numbers are entered into the inputs:

<form action="/handler/" method="GET"> <input name="num1"> <input name="num2"> <input type="submit"> </form>

Let the server find the sum of the received numbers and send the result back to the browser. First, let's get the sent numbers:

export default { '/handler/': function({get}) { console.log(get.num1); console.log(get.num2); return 'form data received'; } }

Let’s now find their sum and send it back to the browser:

export default { '/handler/': function({get}) { return Number(get.num1) + Number(get.num2); } }

Make a form with five inputs into which numbers are entered. Let the server calculate the arithmetic average of the numbers passed in and send the result back to the browser.

Ask the user for his first, last and patronymic names. Send the data to the server. Let the server return a successful message as a response.

Ask the user for a date in the formatyear-month-day. Send the date to the server. Let the server check that the date format is correct. If the date is correct, let it return a success message, and if it is incorrect, let it return a failure message.
