Form Buttons in JavaScript

You already know that the form is submitted when you click a button. However, there are nuances here. The button must have an attributetypein meaningsubmit:

<form> <input name="test1"> <input name="test2"> <input type="submit"> </form>

The form will also be sent by the tagbuttonwith attributetypein meaningsubmit:

<form> <input name="test1"> <input name="test2"> <button type="submit">btn</button> </form>

The form will also be sent if you put focus in any of the inputs and clickEnter.

Button with attributetypein meaningbuttonis used as a page element and will not submit the form:

<form> <input name="test1"> <input name="test2"> <input type="button"> </form>

The tag will work in a similar waybutton:

<form> <input name="test1"> <input name="test2"> <button>btn</button> </form>

Button with attributetypein meaningresetwill clear the completed form:

<form> <input name="test1"> <input name="test2"> <input type="reset"> <input type="submit"> </form>

Correct the following code so that the button submits the form to the server:

<form> <input name="name"> <input name="surn"> <button>send</button> </form>
