Submitting a form via JavaScript

You can force a form to submit via JavaScript. Let's see how it's done. Suppose we have a form and some button, upon clicking which we want to submit the form:

<form action="/handler/" method="POST"> <input name="test1"> <input name="test2"> </form> <button>отправить</button>

Let's get references to our elements in variables:

let form = document.querySelector('form'); let button = document.querySelector('button');

Let's assign a click handler to the button:

button.addEventListener('click', function(event) { });

By clicking on the button, we will submit the form using the methodsubmit:

button.addEventListener('click', function(event) { form.submit(); });
