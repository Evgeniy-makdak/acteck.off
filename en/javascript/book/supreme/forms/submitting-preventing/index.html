Preventing form submission in JavaScript

If necessary, you can prevent the form from being submitted. This is done through what is already known to youpreventDefault. Let's look at an example.

Let us have the following form:

<form action="/handler/" method="POST"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Let's get a link to it in a variable:

let form = document.querySelector('form');

Submitting a form can be caught via an eventsubmit:

form.addEventListener('submit', function() { });

Let's now prevent the form from being submitted:

form.addEventListener('submit', function(event) { event.preventDefault(); });

A form with an input is given. When you try to send, make sure that the correct email is entered in the input. If this is not the case, disable submitting the form and display a message about this.
