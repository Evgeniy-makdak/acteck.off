Working with FormData in JavaScript

Let us have some form:

<form action="/target/" method="POST"> <input name="test1" value="123"> <input name="test2" value="456"> <input type="submit"> </form>

Suppose we want to receive data of this form in the form of pairskey-value. It is clear that for this we will have to start a cycle and form what we want in it. In JavaScript, however, there is a simpler way - you can use a special objectFormData, which allows you to receive form data in an ordered form.

Let's look at working with this object. First, let's get a link to our form:

let form = document.querySelector('form');

Now let's create an object with our form:

let formData = new FormData(form);

You can output our object to the console, but with this output we will not see the form data:

console.log(formData);

Let's convert our object to an array to see the form data:

console.log(Array.from(formData));

Given a form with three inputs. A button is also given. By clicking the button, you will receive the form data as an objectFormData.
