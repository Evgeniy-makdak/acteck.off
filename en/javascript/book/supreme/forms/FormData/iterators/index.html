Iterators in FormData in JavaScript

FormDataisiterable object. Accordingly, iterators are built into itvalues, keys, entries.

Let's loop through the values ​​of the form elements:

for (let value of formData.values()) { console.log(value); }

Now let’s loop through the names of the form elements:

for (let key of formData.keys()) { console.log(key); }

Now let's loop through the pairskey-valueform elements:

for (let entry of formData.entries()) { console.log(entry); }
