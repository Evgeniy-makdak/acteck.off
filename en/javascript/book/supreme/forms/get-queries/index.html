GET запросы в JavaScript

На самом деле для отправки формы методом GET не обязательно наличие формы. Мы можем просто взять и вручную вбить в адресную строку GET Parameters и нажать на Enter. Тем самым мы имитируем отправку формы.

Этим приемом часто пользуются при разработке сайтов. Конечно, не вбивают вручную в адресную строку (хотя это технически возможно для продвинутых пользователей), а создают ссылки, при переходе по которым будет имитироваться отправка формы.

Давайте сделаем подобную ссылку:

<a href="/handler/?test1=1&test2=2">link</a>

Поймаем данные из ссылки на сервере:

export default { '/handler/': function({get}) { console.log(get); return 'text'; } }

С помощью GET параметра отправьте на сервер число. Возведите полученное число в квадрат и выведите результат в консоль сервера.

С помощью GET параметров отправьте на сервер два числа. Найдите сумму переданных чисел и выведите результат в консоль сервера.
