GET requests in JavaScript

In fact, you don't need to have a form to submit a form using the GET method. We can simply manually enter the GET parameters into the address bar and click onEnter. Thus, we simulate submitting the form.

This technique is often used when developing websites. Of course, they don’t manually type them into the address bar (although this is technically possible for advanced users), but create links that, when clicked, will simulate submitting the form.

Let's make a link like this:

<a href="/handler/?test1=1&test2=2">link</a>

Let's catch the data from the link on the server:

export default { '/handler/': function({get}) { console.log(get); return 'text'; } }

Use the GET parameter to send a number to the server. Square the resulting number and display the result in the server console.

Using GET parameters, send two numbers to the server. Find the sum of the transmitted numbers and display the result in the server console.
