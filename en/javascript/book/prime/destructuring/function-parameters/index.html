Destructuring Function Parameters in JavaScript

Destructuring has another very important application - passing function parameters. The point is this: if a function takes an array as a parameter, we can specify directly in the function declaration how to destructure this array.

Let's look at an example. Let us have a function that takes an array with year, month and day as a parameter:

func([2025, 12, 31]);

Let's indicate directly in the function parameter which variables this array should be divided into:

function func([year, month, day]) { console.log(year); // выведет 2025 console.log(month); // выведет 12 console.log(day); // выведет 31 }

The above construction should be considered as one function parameter. You can add more parameters if you wish:

func('str1', [2025, 12, 31], 'str2'); function func(param1, [year, month, day], param2) { console.log(param1); // выведет 'str1' console.log(year); // выведет 2025 console.log(month); // выведет 12 console.log(day); // выведет 31 console.log(param2); // выведет 'str2' }

And in the following example, arrays are passed as the first and second parameters to the function and we destructure both of them:

func([2025, 12, 31], [2026, 11, 30]); function func([year1, month1, day1], [year2, month2, day2]) { console.log(year1); // выведет 2025 console.log(month1); // выведет 12 console.log(day1); // выведет 31 console.log(year2); // выведет 2026 console.log(month2); // выведет 11 console.log(day2); // выведет 30 }
