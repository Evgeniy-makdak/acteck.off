Destructuring Arrays in JavaScript

In this lesson we will look at the operationdestructuringarrays. This operation is intended for mass recording of array elements into variables in one line of code.

The syntax for this operation is as follows:

let [переменная1, переменная2, переменная3] = массив;

As you can see, to the left of the sign=The variable names are listed in square brackets. These variables are declared in bulk vialet, which is placed before the opening parenthesis.

As a result of the operation, the first element of the array (that is, with the key zero) will be written to the first variable, the second to the second variable, and the third to the third variable.

Let's look at a practical example. Let us have an array storing the year, month and day:

let arr = [2025, 12, 31];

Let's write the year, month and day into the corresponding variables using destructuring:

let arr = [2025, 12, 31]; let [year, month, day] = arr;

Let's look at the contents of our variables:

console.log(year); // выведет 2025 console.log(month); // выведет 12 console.log(day); // выведет 31

For comparison, look at how inconvenient and long the code would be without destructuring:

let arr = [2025, 12, 31]; let year = arr[0]; let month = arr[1]; let day = arr[2]; console.log(year); // выведет 2025 console.log(month); // выведет 12 console.log(day); // выведет 31

The following code writes parts of the array into corresponding variables:

let arr = ['John', 'Smit', 'development', 'programmer', 2000]; let name = arr[0]; let surname = arr[1]; let department = arr[2]; let position = arr[3]; let salary = arr[4];
