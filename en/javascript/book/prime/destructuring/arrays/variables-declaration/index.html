Declaring variables when destructuring an array in JavaScript

It is not necessary to declare variables when destructuring. They can be declared in advance and then when assigning there will be no need to write a commandlet:

let arr = [2025, 12, 31]; let year; let month; let day; [year, month, day] = arr;

There are, however, nuances. Look at the following code:

let arr = [2025, 12, 31]; let year; let [year, month, day] = arr;

As you can see, the variableyearwas declared in advance, but the variablesmonthAndday- No. Therefore, the author of the code decided to writeletbefore destructuring assignment.

This, however, will lead to an error, since you cannot declare the same variable twicelet(it turns out thatyearannounced twice).
