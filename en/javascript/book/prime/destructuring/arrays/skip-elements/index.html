Skipping array elements when destructuring in JavaScript

You can start writing into variables not from the array first, but skip some values. Let's, for example, skip the year, and write the month and day into variables. To do this, when specifying variables, we put a comma before the first variable, like this:[, month, day]:

let arr = [2025, 12, 31]; let [, month, day] = arr; console.log(month); // выведет 12 console.log(day); // выведет 31

You can skip more than one value, but several:

let arr = [2025, 12, 31]; let [,, day] = arr; console.log(day); // выведет 31

The following code writes parts of the array into corresponding variables:

let arr = ['John', 'Smit', 'development', 'programmer', 2000]; let department = arr[2]; let position = arr[3];
