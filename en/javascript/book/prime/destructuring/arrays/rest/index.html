The remainder of an array when destructuring in JavaScript

If the array has more elements than variables, you can optionally write the extra elements into the array using the operatorrest:

let arr = [2025, 12, 31, 23, 59, 59]; let [year, month, day, ...time] = arr; console.log(year); // выведет 2025 console.log(month); // выведет 12 console.log(day); // выведет 31 console.log(time); // выведет [23, 59, 59]

The following code writes parts of the array into corresponding variables:

let arr = ['John', 'Smit', 'development', 'programmer', 2000]; let name = arr[0]; let surname = arr[1]; let info = arr.slice(2); // все элементы со второго до конца массива
