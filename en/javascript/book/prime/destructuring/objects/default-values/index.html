Default values ​​when destructuring objects in JavaScript

When destructuring objects, you can also specify default values. In this case, unlike destructuring arrays, any variable can be optional - not necessarily from the end of the array. Let's, for example, specify a default value for a variableyear:

let obj = { month: 12, day: 31, }; let {year = 2025, month, day} = obj; console.log(year); // выведет 2025 console.log(month); // выведет 1 console.log(day); // выведет 31

The following code writes parts of the object to the corresponding variables:

let options = { width: 400, height: 500, }; let color; if (options.color !== undefined) { color = options.color; } else { color = 'black'; } let width = options.width; let height = options.height;
