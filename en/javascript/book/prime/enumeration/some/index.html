some method to check an array in JavaScript

Next methodsomechecks array elements and returnstrue, if the callback returned for at least one elementtrue, otherwise the method returnsfalse.

Let's check, for example, that the array contains at least one even number:

let arr = [2, 4, 6, 8]; let result = arr.some(function(elem) { return elem % 2 == 0; }); console.log(result);

Let's simplify:

let arr = [2, 4, 6, 8]; let result = arr.some(elem => elem % 2 == 0); console.log(result);
