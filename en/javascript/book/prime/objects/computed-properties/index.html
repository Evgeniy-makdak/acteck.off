Computed Properties in JavaScript

When declaring an object, the names of its properties can be taken from variables. Such properties are calledcomputable. Let's see how it's done. Let us have the following object:

let obj = { a: 1, b: 2, c: 3 };

Let the name of some property be stored in a variable:

let key = 'a';

Let's make sure that instead of the property name, the value from our variable is taken. To do this, the variable should be placed in square brackets:

let obj = { [key]: 1, b: 2, c: 3 };

When you declare computed properties, you can execute some code. Example:

let obj = { [key + '1']: 1, [key + '2']: 2, [key + '3']: 3 };

In the following code, the key should have been taken from a variable. Correct the mistake:

let key = 'x'; let obj = { key: 1, y: 2, z: 3 };

Given object:

let obj = { x: 1, y: 2, z: 3 };

Given variables:

let key1 = 'x'; let key2 = 'y'; let key3 = 'z';

Make the object's keys come from these variables.
