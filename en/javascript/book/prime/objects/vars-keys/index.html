Object Keys from Variables in JavaScript

Object keys can be stored in variables. Let's look at an example. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let a variable store a key:

let key = 'a';

Let's display an object element by key:

console.log(obj[key]); // выведет 1

let obj = {x: 1, y: 2, z: 3};
