Finding errors in code with JavaScript objects

In the following problems, some programmer wrote the code and may have made mistakes in it. You should check if the code does what it says. If the code does not work correctly, you must fix the errors.

The code should output the object element:

let obj = {x: 1, y: 2, z: 3}; console.log(obj[x]);

The code should derive the object element by key from the variable:

let obj = {x: 1, y: 2, z: 3}; let key = 'x'; console.log(obj.key);

The code should print the sum of the elements of the object:

let obj = {x: 1, y: 2, z: 3}; let sum = obj[x] + obj[y] + obj[x]; console.log(obj);

The code should display the number of elements of the object:

let obj = {x: 1, y: 2, z: 3}; console.log(obj.length);
