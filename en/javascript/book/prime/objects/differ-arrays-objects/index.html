Distinguishing arrays from objects in JavaScript

As stated above, the operatortypeofdoes not allow you to distinguish between arrays and objects. Sometimes, however, this must be done. The function will come to the rescueArray.isArray():

console.log( Array.isArray([]) ); // выведет true console.log( Array.isArray({}) ); // выведет false

Without running the code, determine what will be printed to the console:

console.log( Array.isArray([1, 2, 3]) );

Without running the code, determine what will be printed to the console:

console.log( Array.isArray({x: 1, y: 2, z: 3}) );
