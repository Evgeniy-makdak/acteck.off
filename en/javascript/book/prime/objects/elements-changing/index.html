Changing Object Elements in JavaScript

Object elements can be changed by accessing them using keys. Let's see how it's done. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let's change the value of its element:

obj['a'] = '!';

Let's use property access:

obj.b = '!';

Let's check if the object has changed:

console.log(obj);

Given object:

let obj = {x: 1, y: 2, z: 3};

Square each element of this object.
