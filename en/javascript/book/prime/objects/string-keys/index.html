String Object Keys in JavaScript

Object keys can be strings:

let obj = {'a': 1, 'b': 2, 'c': 3};

In this case, it is customary not to put string keys in quotes:

let obj = {a: 1, b: 2, c: 3};

Let's infer the element of our object by key:

console.log(obj['a']); // выведет 1
