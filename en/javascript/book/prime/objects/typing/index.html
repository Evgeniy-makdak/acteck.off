Object Typing in JavaScript

Objects have their own data type called'object'. Let's check using the operatortypeof:

console.log(typeof {}); // выведет 'object'

Without running the code, determine what will be printed to the console:

console.log( typeof {x: 1, y: 2, z: 3} );

Without running the code, determine what will be printed to the console:

console.log( typeof {} );

Without running the code, determine what will be printed to the console:

let obj = {x: 1, y: 2, z: 3}; console.log( typeof obj );

Without running the code, determine what will be printed to the console:

let obj = {x: 1, y: 2, z: 3}; console.log( typeof obj['x'] );
