Properties of objects in JavaScript

There is an alternative way to access the elements of an object. In this method, we put a dot after the object name and write the name of the key. In this case the key is calledpropertyobject. Let's see how it's done. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let's display the value of its property:

console.log(obj.a); // выведет 1
