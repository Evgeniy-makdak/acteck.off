Constants with arrays or objects in JavaScript

Constants containing arrays and objects work in interesting ways. JavaScript does not allow you to change the values ​​of these constants, but you can change the properties of objects and the values ​​of array elements.

Let's look at examples. Let us be given the following object:

const obj = {a: 1, b: 2, c: 3};

Let's try to write something else into it:

obj = 123; // ошибка

Let's try to write another object into a constant:

obj = {x: 1, y: 2, z: 3}; // ошибка

However, if we try to change a property of the object, it works:

obj.a = '+'; // работает!

What will be output as a result of executing the following code:

const arr = ['a', 'b', 'c']; arr[1] = '!'; console.log(arr);

What will be output as a result of executing the following code:

const arr = ['a', 'b', 'c']; arr = [1, 2, 3]; console.log(arr);

What will be output as a result of executing the following code:

const arr = ['a', 'b', 'c']; arr = ['a', 'b', 'c']; console.log(arr);
