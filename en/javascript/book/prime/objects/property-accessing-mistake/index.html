Error accessing object property in JavaScript

When accessing a property of an object, the name of that property cannot be stored in a variable. Let's see why. Let us be given the following object:

let obj = {a: 1, b: 2, c: 3};

Let a variable store the name of the property whose value we want to get:

let key = 'a';

Let's try to access a property of an object using this variable:

console.log(obj.key); // undefined

The above code won't work because we're actually looking for a property called'key', not a property whose name is stored in a variablekey. That is, in fact, our code is equivalent to the following:

console.log(obj['key']);

The only way to access an element by key from a variable is to write that variable in square brackets:

console.log(obj[key]);

Correct the error in the following code:

let obj = {x: 1, y: 2, z: 3}; let prop = 'x'; console.log(obj.prop);

Correct the error in the following code:

let obj = {x: 1, y: 2, z: 3}; let prop = 'x'; console.log(obj['prop']);
