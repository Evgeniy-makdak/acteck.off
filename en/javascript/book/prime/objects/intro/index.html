Objects in JavaScript

JavaScript has special data structures calledobjects(in other programming languages ​​they are calledassociativearrays orhashes).

Objects are created using curly braces{ }, inside which the elements of this object are written in the formatkey: value.

Let's create an empty object:

let obj = {};

Now let's create an object with data:

let obj = {1: 'a', 2: 'b', 3: 'c'};

Let's display an object element by its key:

console.log(obj[1]); // выведет 'a'
