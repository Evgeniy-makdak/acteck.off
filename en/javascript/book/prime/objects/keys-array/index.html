Array of object keys in JavaScript

You can get an array of object keys. Let's see how it's done. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let's get an array of its keys:

let keys = Object.keys(obj);

Let's output this array to the console:

console.log(keys);

Get the array of keys of the following object:

let obj = {x: 1, y: 2, z: 3};
