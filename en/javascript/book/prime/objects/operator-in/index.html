The in operator in JavaScript

Using an operatorinyou can check for the presence of a property in an object. Let's look at an example. Let us be given the following object:

let obj = {a: 1, b: 2, c: 3};

Let's check if there is any property in our object:

console.log('b' in obj); // выведет true

Now let's check for a non-existent property:

console.log('x' in obj); // выведет false

Tell me what will be printed to the console as a result of executing the following code:

let obj = {x: 1, y: 2, z: 3}; console.log('x' in obj); console.log('w' in obj);
