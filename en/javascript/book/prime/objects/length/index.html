Object Length in JavaScript

Objects have no propertylength, allowing you to find their length. Let's make sure of this. Let us have the following object:

let obj = {a: 1, b: 2, c: 3};

Let's try to find out the number of its elements:

console.log(obj.length); // выведет undefined

Let's solve the problem in a roundabout way - get an array of object keys and find its length:

console.log(Object.keys(obj).length); // выведет 3
