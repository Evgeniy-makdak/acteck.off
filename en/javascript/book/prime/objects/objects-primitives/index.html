Objects and Primitives in JavaScript

Objects (including arrays, as you already know) are considered complex data types. Strings, numbers, booleans are considered prime, orprimitivedata types. They are often called that -primitives, meaning everything that is not an object.

JavaScript has seven primitive data types:string, number, boolean, null, undefined, symbol, bigint. Remember their number and names - this is often asked during interviews.

The following code is given:

let test = {x: 1, y: 2, z: 3}; console.log(test);

What type of data will be output to the console? Is it a primitive or an object?

The following code is given:

let test = {x: 1, y: 2, z: 3}; console.log(test.x);

What type of data will be output to the console? Is it a primitive or an object?

The following code is given:

let test = [1, 2, 3]; console.log(test);

What type of data will be output to the console? Is it a primitive or an object?

The following code is given:

let test = [1, 2, 3]; console.log(test[1]);

What type of data will be output to the console? Is it a primitive or an object?

The following code is given:

let test1 = [1, 2, 3]; let test2 = 1; console.log(test1);

What type of data will be output to the console? Is it a primitive or an object?

The following code is given:

let test1 = [1, 2, 3]; let test2 = 1; console.log(test1[test2]);

What type of data will be output to the console? Is it a primitive or an object?

Without looking at the textbook, list all the primitive data types in JavaScript.
