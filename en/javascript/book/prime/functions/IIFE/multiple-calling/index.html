Multiple IIFE calls in JavaScript

Let them offer you a tricky task: make several calling parentheses, like this:

(function() { // какой-то код })()(); // несколько вызывающих скобок

If you think about it, it becomes obvious that in this case, calling the function in place should return an anonymous function, like this:

(function() { return function() { console.log('!'); }; })()(); // выведет '!'

Add the following code so that running it displays'!':

(function() { // какой-то код })()()();

Add the following code so that its launch displays the sum of the numbers passed as parameters:

(function() { // какой-то код })(1)(2);

Add the following code so that its launch displays the sum of the numbers passed as parameters:

(function() { // какой-то код })(1)(2)(3);
