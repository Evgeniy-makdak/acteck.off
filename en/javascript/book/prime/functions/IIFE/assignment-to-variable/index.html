Assigning a function to a variable in JavaScript

Let now our locally called function not output something to the console, but return it viareturn:

+function() { return '!'; }();

Let's assign the result of our function to a variable and output it to the console. Since in this case there is an assignment to a variable, the plus is no longer needed:

let result = function() { return '!'; }(); console.log(result); // выведет '!'

Determine, without running the code, what will be output to the console:

let result = function() { return '!'; }(); console.log(result);

Determine, without running the code, what will be output to the console:

let result = function() { return '!'; }; console.log(result);

Determine, without running the code, what will be output to the console:

let result = function() { return '!'; }; console.log(result());
