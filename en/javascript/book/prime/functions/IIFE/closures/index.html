Closures and in-place function calls in JavaScript

As you already know, you can call an anonymous function in place and assign the result to a variable:

let result = function() { return '!'; }(); console.log(result); // выведет '!'

Often a function that is called in place is then enclosed in parentheses, although this is not necessary. This is done so that it is immediately clear that the function is called in place:

let result = (function() { return '!'; })(); console.log(result); // выведет '!'

Now let the result of the function called in place be an anonymous function:

let func = (function() { return function() { console.log('!'); } })(); func(); // выведет '!'

Let's make the return function store a variable in the closurenum:

let func = (function() { let num = 1; return function() { console.log(num); } })();

If you add morenum++, then the functionfuncwill become a ready counter:

let func = (function() { let num = 1; return function() { console.log(num); num++; } })(); func(); //выведет 1 func(); //выведет 2 func(); //выведет 3 func(); //выведет 4 func(); //выведет 5

Modify the counter so that it counts up to5, and then the countdown started all over again.
