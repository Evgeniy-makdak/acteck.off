Применение вызова функции на месте в JavaScript

Давайте применим вызов функции на месте в качестве одного из слагаемых:

let sum = 1 + function() { return 2; }(); console.log(sum); // выведет 3

Определите, не запуская код, что выведется в консоль:

let result = function() {return 1;}() + function() {return 2;}(); console.log(result);
