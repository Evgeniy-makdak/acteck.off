Calling a function in place in JavaScript

Now we will look at a technique that allows you to call a function right where it is declared. This design is calledImmediately Invoked Function Expression (IIFE).

Let's look at an example. Let us have this functional expression:

let func = function() { console.log('!'); }; func(); // выведет '!'

Let's now not assign our function to a variable, but call it immediately, “on the spot”. To do this, we put parentheses after the function:

+function() { console.log('!'); // выведет '!' }();

The presence of a plus in this case is a prerequisite, since without it the function will become a Function Declaration, and they cannot be called on the spot (and even without a name). Of course, instead of a plus, anything can be used - the main thing is that our function is a functional expression.

Determine, without running the code, what will be output to the console:

!function() { console.log('!'); }();

Determine, without running the code, what will be output to the console:

function() { console.log('!'); }();
