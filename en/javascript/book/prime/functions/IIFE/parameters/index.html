Function parameters when called in place in JavaScript

Let our function, which we are going to call in place, take as a parameter a string for output:

function(str) { console.log(str); }

Let's call our function in place, passing it a string to output to the console:

(function(str) { console.log(str); // выведет '!!!' })('!!!');

Determine, without running the code, what will be output to the console:

(function(num1, num2) { console.log(num1 + num2); })(1, 2);
