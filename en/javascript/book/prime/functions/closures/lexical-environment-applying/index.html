Using the lexical environment of functions in JavaScript

Let us have a function that returns another function as its result:

function test() { return function() { } }

If the parent function has any variables, then these variables will be contained in the lexical environment of the returned function:

function test() { let num = 1; // переменная родительской функции return function() { // лексическое окружение = {num: 1} } }

Let's write in the code of our return functionconsole.log, outputting the value of a variablenum:

function test() { let num = 1; return function() { console.log(num); } }

Let's now call the parent functiontestand write the result of its work into a variablefunc:

function test() { let num = 1; return function() { console.log(num); } } let func = test();

To a variablefuncthe return function will be written. Let's call our function - its result will display the contents of the variablenum:

function test() { let num = 1; return function() { console.log(num); } } let func = test(); func(); // выведет 1

If you just try to output the variablenumoutside the function - it will not be available:

function test() { let num = 1; return function() { console.log(num); } } console.log(num); // переменная num тут недоступна

As you can see, the local variablenumtied to the lexical environment of our function and now, by calling this function anywhere in the code, we can get the value of the variablenum, even if this variable itself is not available at the call site.

In fact, a similar result can be achieved by making a variablenumglobal:

function test() { return function() { console.log(num); } } let num = 1; // глобальная переменная let func = test(); func(); // выведет 1

Here, however, there will be a significant difference: in the new version, the variablenumcan be changed outside of functions (since it is global), but not in the old one.

Determine, without running the code, what will be output to the console:

function test() { let num1 = 1; let num2 = 2; return function() { return num1 + num2; } } let func = test(); console.log(func());

Determine, without running the code, what will be output to the console:

function test() { let num1 = 1; let num2 = 2; return function() { return num1 + num2; } } console.log(test()());

Determine, without running the code, what will be output to the console:

function test() { let num1 = 1; return function() { return num1 + num2; } } let num2 = 2; let func = test(); console.log(func());

Determine, without running the code, what will be output to the console:

function test() { let num = 1; return function() { return num; } } let num = 2; let func = test(); console.log(func());
