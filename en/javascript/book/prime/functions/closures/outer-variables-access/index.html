Accessing External Function Variables in JavaScript

Let's look at the following code:

let num = 1; // задаем значение переменной function func() { console.log(num); // выводим его в консоль } func(); // вызываем функцию

As I mentioned earlier, the value of a variable does not have to come before the function definition, as long as it comes before the function call:

function func() { console.log(num); } let num = 1; func();

In fact, this is not entirely true. Our function knows the value of the variable even before it is callednum:

let num = 1; function func() { console.log(num); // функция уже знает, что num = 1 }

Here's a more complex example:

let num = 1; // функция в этот момент узнает, что num = 1 function func() { console.log(num); } num = 2; // функция в этот момент узнает, что num = 2

Let's add function calls:

let num = 1; // функция в этот момент узнает, что num = 1 func(); // выведет 1 function func() { console.log(num); } func(); // выведет 1 num = 2; // функция в этот момент узнает, что num = 2 func(); // выведет 2

Once again: in fact, the function knows the values ​​of external variables without even being called.
