Example with a parameter via recursion in JavaScript

Let's use recursion to sequentially display the elements of the array. Let the array be initially passed to the function parameters:

func([1, 2, 3]);

Let's use the method without recursion for nowshiftLet's display all the elements of the array one by one:

function func(arr) { console.log(arr.shift()); // выведет 1 console.log(arr); // выведет [2, 3] - массив уменьшился console.log(arr.shift()); // выведет 2 console.log(arr); // выведет [3] - массив уменьшился console.log(arr.shift()); // выведет 3 console.log(arr); // выведет [] - массив пуст } func([1, 2, 3]);

As you can see, the methodshiftcuts and returns the first element of the array, while the array itself is reduced by this element.

Let's now use recursion:

function func(arr) { console.log(arr.shift(), arr); if (arr.length != 0) { func(arr); } } func([1, 2, 3]);

In fact, of course, the easiest way is to iterate through the elements of an array in a loop. The examples given so far simply demonstrate how recursion works using simple examples (not real-life ones). More useful examples of using recursion are simply more complex, we will look at them below.
