Element Manipulation in JavaScript

Let's do something with the array elements being iterated, for example, write a sign at the end of them'!'. In this case we will have to use a regular loopfor, notfor-of, like this:

function func(arr) { for (let i = 0; i < arr.length; i++) { if (typeof arr[i] == 'object') { arr[i] = func(arr[i]); } else { arr[i] = arr[i] + '!'; } } return arr; } console.log(func([1, [2, 7, 8], [3, 4, [5, 6]]]));
