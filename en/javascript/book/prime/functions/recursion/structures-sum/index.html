Sum of array elements via recursion in JavaScript

Let's find the sum of the primitive elements of our array:

function func(arr) { let sum = 0; for (let elem of arr) { if (typeof elem == 'object') { sum += func(elem); } else { sum += elem; } } return sum; } console.log(func([1, [2, 7, 8], [3, 4, [5, [6, 7]]]]));

Given a multidimensional object of an arbitrary nesting level, for example, this:

{a: 1, b: {c: 2, d: 3, e: 4}, f: {g: 5, j: 6, k: {l: 7, m: {n: 8, o: 9}}}}

Use recursion to find the sum of the elements of this object.

Given a multidimensional array of an arbitrary nesting level, containing strings inside it, for example, like this:

['a', ['b', 'c', 'd'], ['e', 'f', ['g', ['j', 'k']]]]

Using recursion, merge the elements of this array into one line:

'abcdefgjk'
