Sum of array elements during recursion in JavaScript

Let's now not output the elements of the array to the console, but find the sum of the elements of this array:

function getSum(arr) { let sum = arr.shift(); if (arr.length !== 0) { sum += getSum(arr); } return sum; } console.log(getSum([1, 2, 3]));
