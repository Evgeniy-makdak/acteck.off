Working with Recursion in JavaScript

In programming there is such a thing asrecursion- this is when a function calls itself. Let's look at an example. Using recursion, we will derive the numbers from1to10:

let i = 1; function func(){ console.log(i); i++; if (i <= 10){ func(); // здесь функция вызывает сама себя } } func();

Let's discuss how this code works.

We have a global variableiand functionfunc, inside which the contents of the variable are output to the consoleiand then done++.

If our variableiless than or equal to10, then the function is called again. Since the variablei- global, then with each new function call it will contain the value of the variable specified in the previous calli.

It turns out that the function will call itself untiliit won't get any bigger10.

Please note that in our case we cannot run the function withoutif- if you do this, you will end up with an endless function call.
