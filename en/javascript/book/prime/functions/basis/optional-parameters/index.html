Optional Parameters in JavaScript

Function parameters can be made optional. To do this, the parameters need to be set to default values. For example, let's say we have the following function:

function func(num) { console.log(num ** 2); }

Let's make this default value0:

function func(num = 0) { console.log(num ** 2); }

Let's check the operation of our function with the parameter:

func(2); // выведет 4

Let's check how our function works without a parameter:

func(); // выведет 0

Given function:

function func(num = 5) { console.log(num * num); }

This function is called as follows:

func(2); func(3); func();

Tell us what the result of each function call will be.

Given function:

function func(num1 = 0, num2 = 0) { console.log(num1 + num2); }

This function is called as follows:

func(2, 3); func(3); func();

Tell us what the result of each function call will be.
