Calling functions sequentially in JavaScript

You can pass the result of one function as a parameter to another. In the following example we will first find the square of a number2and then square the result:

function func(num) { return num ** 2; } let res = func(func(2)); console.log(res); // выведет 16

The functions, of course, do not have to be the same. Let, for example, we have a function that returns the square of a number and a function that returns the cube of a number:

function square(num) { return num ** 2; } function cube(num) { return num ** 3; }

Let's use these functions to build a number2squared, and then the result of this operation is cubed:

let res = cube(square(2)); console.log(res);

Let us now have a function that returns the square of a number, and a function that finds the sum of two numbers:

function square(num) { return num ** 2; } function sum(num1, num2) { return num1 + num2; }

Using these functions, we find the sum of the square of a number2and the square of the number3:

let res = sum(square(2), square(3)); console.log(res);

Let's say you have a function that returns the square root of a number, and a function that rounds a fraction to three decimal places:

function sqrt(num) { return Math.sqrt(num); } function round(num) { return num.toFixed(3); }

Use these functions to find the square root of a number2and round it to three decimal places.

Let's say you have a function that returns the square root of a number, and a function that returns the sum of three numbers:

function sqrt(num) { return Math.sqrt(num); } function sum(num1, num2, num3) { return num1 + num2 + num3; }

Use these functions to find the sum of the roots of numbers2, 3And4and write it to a variableres.
