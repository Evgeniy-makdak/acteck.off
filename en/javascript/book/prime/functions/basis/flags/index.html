Flags in JavaScript Functions

You can use flags implicitly in functions using the statementreturn. Let's see how it's done. Let's say we have the following function that checks that all elements of an array are positive numbers:

function isPositive(arr) { let flag = true; for (let elem of arr) { if (elem < 0) { flag = false; } } return flag; }

Let's rewrite the function code using implicit flags:

function isPositive(arr) { for (let elem of arr) { if (elem < 0) { return false; } } return true; }

How it works: if the desired element is found in the array, we exit the function (and the loop too) usingreturn. But if the required element is not found in the array, the function will not exit and execution will proceed to the commandreturn true. And it turns out that the function will returntrueas a sign that all elements in the array are positive.
