Multiple Function Parameters in JavaScript

A function can take more than one parameter, but several. In this case, they must be listed separated by commas. Let's make an example of a function that takes two numbers as a parameter and displays their sum to the console:

function func(num1, num2) { console.log(num1 + num2); }

Let's check the operation of our function:

func(1, 2); // выведет 3
