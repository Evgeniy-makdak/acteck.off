Параметры функций в JavaScript

Функции могут принимать параметрами данные. Для примера давайте сделаем функцию, которая параметром будет принимать число:

function func(num) { }

Давайте сделаем так, чтобы функция возводила переданное число в квадрат:

function func(num) { console.log(num ** 2); }

Проверим работу функции, вызвав ее с различными числами:

func(2); // выведет 4 func(3); // выведет 9

Сделайте функцию, которая параметром принимает число и выводит в консоль куб этого числа.

Сделайте функцию, которая параметром принимает число и проверяет, положительное это число или отрицательное. В первом случае пусть функция выводит в консоль текст '+++', а во втором '---'.
