Logical operators without if in JavaScript functions

Let us have a function with if. Here it is:

function func(a, b) { if (a > b) { return true; } else { return false; } }

As you already know from previous lessons, constructionsif, which return boolean values, can be rewritten in a shorthand form. Let's do this:

function func(a, b) { return a > b; }
