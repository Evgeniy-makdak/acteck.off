Expression on the right side of a function in JavaScript

Note that the expression that involves the function must be to the left of it. Trying to do something to the right of a function doesn't make it a function expression. Example:

function func() { // это Function Declaration console.log('!'); } + 1;

Why so: because in this case this +1 is simply a new command written after the function. If you write this command on a new line, everything becomes clearer:

function func() { console.log('!'); } +1; // просто команда

Let's make our function a Function Expression. For example, let's put before the function+:

+function func() { // это Function Expression console.log('!'); } + 1;

Or let's do the assignment:

let test = function func() { // это Function Expression console.log('!'); } + 1;

Or we’ll pass it as a parameter toconsole.log:

console.log(function func() { // это Function Expression console.log('!'); } + 1);

Determine whether the represented function is a Function Declaration or a Function Expression:

function func() { console.log('!'); } +1;

function func() { console.log('!'); } + 1;

+ 1 function func() { console.log('!'); } + 1;
