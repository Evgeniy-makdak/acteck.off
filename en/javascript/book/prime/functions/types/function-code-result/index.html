Source code of the function and its result in JavaScript

In JavaScript, you can write the function name without parentheses. In this case, we will see not the result of the function, but its source code.

Let's try it. Let us have some function, for example, this:

function func() { return '!'; }

Let's call this function by writing its name and parentheses. In this case, we will see the result of the function:

function func() { return '!'; } console.log(func()); // выведет '!'

Now let's try omitting the parentheses after the function name. In this case we will see the source code of our function:

function func() { return '!'; } console.log(func); // увидим код функции

Run the above code on your own and make sure that the source code of the function is output to the console.
