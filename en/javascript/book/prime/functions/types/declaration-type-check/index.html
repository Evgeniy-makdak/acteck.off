How to check the type of a function in JavaScript

In the problems below, you will need to determine whether a function is defined as a Function Declaration or a Function Expression.

In simple cases this is not difficult to do visually. But how can you check that you did it correctly? Use the difference between Function Declaration or Function Expression: the former can be called above its definition, while the latter cannot.

Let's say we have this function:

let test = function() { console.log('!'); }

Let's look at this function before it's defined:

test(); // выведет ошибку в консоль, значит Function Expression let test = function() { console.log('!'); }

Here's another example:

func(); // выведет '!', значит Function Declaration function func() { console.log('!'); }

Let's put a plus sign in front of our function:

func(); // выведет ошибку в консоль, значит Function Expression +function func() { console.log('!'); }

Since the function above is a Function Expression and it is not assigned to any variable, there is no way to call it, because it will not be accessible by the name func.

Determine whether the represented function is a Function Declaration or a Function Expression:

let test = function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

console.log( function func() { console.log('!'); } );

Determine whether the represented function is a Function Declaration or a Function Expression:

+function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

function func() { console.log('!'); }
