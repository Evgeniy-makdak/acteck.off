Function as a variable in JavaScript

So, in the previous lesson we found out that by calling a function without parentheses, we get its source code.

In fact, everything is much more interesting. In JavaScript, unlike other languages, functions are variable values ​​just like numbers, strings, and arrays.

Let's look at an example. Let us have, for example, a functionfunc:

function func() { console.log('!'); }

We don't actually have a functionfunc. We have a variablefunc, which stores the source code of the function.

We can, for example, overwrite the variablefuncsomething else, such as a string. In this case the functionfuncwill no longer be a function, but a string. Look at the example:

function func() { console.log('!'); } func(); // выведет '!' func = 'string'; // затрем переменную func console.log(func); // выведет 'string'
