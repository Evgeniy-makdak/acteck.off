Semicolon when declaring functions in JavaScript

When declaring a function as a Function Declaration after a curly brace}There is no semicolon:

function func() { console.log('!'); }

If the function is declared Function Expression, then after}a semicolon is added:

let func = function() { console.log('!'); };

Why: because in JavaScript any expression must end with a semicolon, and in this case we have just an expression. This semicolon is not required, since JavaScript generally allows them not to be used, but it is desirable. Always bet :)

Place semicolons in all necessary places:

let func1 = function() {console.log('!')} let func2 = function() { console.log('!') } function func3() { console.log('!') }
