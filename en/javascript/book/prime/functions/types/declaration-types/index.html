Function expression and Function declaration in JavaScript

In JavaScript, a function can be declared in two ways.

The first way is to simply declare the function viafunction, immediately asking her name:

function func() { console.log('!'); } func(); // выведет '!'

The second way is to make an unnamed function and write it to some variable:

let func = function() { console.log('!'); }; func(); // выведет '!'

Scientifically, the first method is called Function Declaration (function declaration), and the second one is Function Expression (functional expression).

Determine whether the represented function is a Function Declaration or a Function Expression:

function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

let func = function() { console.log('!'); }
