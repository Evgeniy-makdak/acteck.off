Expression on the left of a function in JavaScript

Let's look at the following code:

+function func() { console.log('!'); }

As you already know, this function is a functional expression, despite the fact that it has a name (we have already found out that having a name is not a criterion at all). Let's remove this plus and get a Function Declaration:

function func() { console.log('!'); }

Let's put+on the line before the function - it will again become a Function Expression:

+ function func() { console.log('!'); }

Now after the plus we put a number1and a semicolon - our function will become a Function Declaration:

+1; function func() { console.log('!'); }

Why is this so: because the first line contains one complete command, closed with a semicolon. Therefore, this command does not affect our function in any way.

In fact, the semicolon can be removed, because in JavaScript it is not required - the function will still remain a Function Declaration:

+1 function func() { console.log('!'); }

But if after1add one more plus, then the function will become a Function Expression:

+1+ function func() { console.log('!'); }

Why is this so: because on the first line there is an incomplete expression - there is a plus and nothing after it. Therefore, the JavaScript interpreter thinks that this plus refers to the line below, that is, our function.

If there is a completed expression on the first line, then JavaScript automatically puts a semicolon in it and this expression does not affect our function in any way.

Determine whether the represented function is a Function Declaration or a Function Expression:

- function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

-1; function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

-1 function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

1 function func() { console.log('!'); }

Determine whether the represented function is a Function Declaration or a Function Expression:

-1- function func() { console.log('!'); }
