Using an Object with Functions in JavaScript

Let's make an object with two functions, each of which will take a number as a parameter. Let the first function square the given number, and the second function cube it. We implement:

let math = { square: function(num) {return num * num}, cube: function(num) {return num * num * num}, };

Let's use our functions:

let math = { square: function(num) {return num * num}, cube: function(num) {return num * num * num}, }; console.log( math.square(2) ); // выведет 4 console.log( math.cube(2) ); // выведет 8
