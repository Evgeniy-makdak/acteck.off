Matching function name and variable name in JavaScript

From the above it follows that the existence of a variable and a function with the same name can easily lead to a problem.

In the following example, the functionfuncwill be overwritten and replaced by the line'string':

function func() { return '!'; } func = 'string'; // затрем переменную с функцией строкой func(); // получим ошибку, т.к. func - не функция

To prevent this from happening, you should adhere to a rule common to all programming languages: functions should be verbs, and variables should be nouns.
