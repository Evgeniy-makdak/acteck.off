Local Variables in Functions in JavaScript

Variables defined within a function are calledlocal. Unlike global variables, local variables are visible only inside the function, and not visible from the outside:

function func() { let num = 5; // локальная переменная console.log(num); } console.log(num); // ничего не выведет, а выдаст ошибку в консоль

Determine, without running the code, what will be output to the console:

function func() { let num = 5; return num; } console.log(num);

Determine, without running the code, what will be output to the console:

function func() { let num = 5; return num; } console.log(func());
