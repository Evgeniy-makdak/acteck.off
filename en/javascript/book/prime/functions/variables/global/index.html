Global Variables in Functions in JavaScript

In JavaScript (not often the case in other languages), variables defined outside a function will be visible inside that function. Such variables are calledglobal. Let's look at an example:

let num = 1; // переменная снаружи функции function func() { console.log(num); // переменная num видна внутри функции } func(); // выведет 1

In fact, the variable should not be defined before the function definition, but before it is called:

function func() { console.log(num); } let num = 1; // переменная снаружи функции func(); // выведет 1

If you change the value of a variable and then call the function each time -console.logwill produce different results each time:

function func() { console.log(num); } let num; // объявим переменную num = 1; // задаем значение 1 func(); // выведет 1 num = 2; // задаем значение 2 func(); // выведет 2

If we have several functions, then the global variable will be available in each of these functions:

function func1() { console.log(num); } function func2() { console.log(num); } let num = 1; func1(); // выведет 1 func2(); // выведет 1

If changes occur to a global variable in one of the functions, then this variable will change in all functions that use this variable:

function func1() { console.log(num); num++; // меняем глобальную переменную } function func2() { console.log(num); } let num = 1; func1(); // выведет 1 func2(); // выведет 2

Since any of the functions can easily change a global variable, their use is a breeding ground for subtle errors. For this reason, the use of global variables in a script should be kept to a minimum. It is advisable that there should be none at all or a very minimal amount.

Determine, without running the code, what will be output to the console:

let num = 1; function func() { console.log(num); } func();

Determine, without running the code, what will be output to the console:

let num = 1; function func() { console.log(num); } num = 2; func();

Determine, without running the code, what will be output to the console:

function func() { console.log(num); } let num; num = 1; func(); num = 2; func();
