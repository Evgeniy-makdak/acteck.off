Matching names with parameters in JavaScript

The parameters of a function are local variables within it. Let's call the function parameter the same name as the external global variable:

function func(num) { console.log(num); } let num = 1; func(num);

In this case, we will have a variable outside the functionnumand inside the function there will be a variablenum. But these will be different variables: changing a variable inside a function will lead to changing a local variable of the function. And the external variable inside the function will be inaccessible and cannot be changed in any way. Let's check:

function func(num) { num = 2; // меняем локальную переменную } let num = 1; func(num); console.log(num); // выведет 1 - ничего не поменялось

Determine, without running the code, what will be output to the console:

function func(num) { num = 2; } let num = 1; func(num); console.log(num);

Determine, without running the code, what will be output to the console:

function func() { num = 2; } let num = 1; func(); console.log(num);

Determine, without running the code, what will be output to the console:

function func() { let num = 2; } let num = 1; func(); console.log(num);
