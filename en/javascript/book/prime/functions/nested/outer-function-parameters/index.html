External Function Parameters in JavaScript

The parameters of the external function will also be available in the internal one:

function test(num) { function func() { console.log(num); // выведет 1 } func(); // вызываем внутреннюю функцию }; test(1); // передаем параметром число

Determine, without running the code, what will be output to the console:

function test(num1, num2) { function func() { console.log(num1 + num2); } func(); } test(1, 2);

Determine, without running the code, what will be output to the console:

function test(num1, num2) { function func() { console.log(num1 + num2); } num1 = 2; func(); } test(1, 2);
