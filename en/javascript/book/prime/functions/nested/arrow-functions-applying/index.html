Using Arrow Functions in JavaScript

Arrow functions have a particular advantage as callbacks. Let's look at an example to see how much the code is simplified in this case. Let us have the following function as an examplefilter:

function filter(arr, callback) { let res = []; for (let elem of arr) { if ( callback(elem) === true ) { res.push(elem); } } return res; }

Let's call this function, passing it a callback:

let result = filter([1, 2, 3, 4, 5], function(elem) { if (elem % 2 === 0) { return true; } else { return false; } });

Let's now simplify our function. First, let's get rid of the designifand write the condition simply through the operator===:

let result = filter([1, 2, 3, 4, 5], function(elem) { return elem % 2 == 0; });

Let us now replace the usual function with an arrow function:

let result = filter([1, 2, 3, 4, 5], elem => elem % 2 == 0);

Given the following function with a callback:

let result = filter([1, 2, 3, 4, 5], function(elem) { if (elem > 0) { return true; } else { return false; } });

Simplify the callback using an arrow function.
