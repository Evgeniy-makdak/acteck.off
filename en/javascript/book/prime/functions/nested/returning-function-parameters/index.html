Return parameters of a function in JavaScript

You can pass parameters to the function calls we studied. In the following example, the internal function expects a string as a parameter and prints it to the console:

function func() { return function(str) { return str; }; }

The internal function corresponds to the second bracket when called, so we pass the desired string into this second bracket:

function func() { return function(str) { return str; }; } console.log( func()('!') ); // выведет '!'

Let's make sure that both the first function takes a parameter and the second one. And the result of the call will be the sum of these parameters:

function func(num1) { return function(num2) { return num1 + num2; }; } console.log( func(1)(2) ); // выведет 3
