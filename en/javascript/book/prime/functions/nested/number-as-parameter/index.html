Passing a number as a parameter in JavaScript

Now let's not store the number that the passed function does something with insidetest, and pass as the first parameter:

function test(num, func) { // первым параметром приходит число console.log(func(num)); }

Let's use our function:

function test(num, func) { console.log(func(num)); } // Выведет 4: test(2, function(num) { return num * num; });

In the convenience of our design: we have one functiontest, which takes a number as a parameter. But what will happen to the number is not hardcoded into the functiontest.

We can, for example, use the second parameter of the functiontestwe can pass a function that squares, or we can, for example, cube it:

function test(num, func) { console.log(func(num)); } // Найдем квадрат числа: test(2, function(num) { return num * num; // возвращает квадрат }); // Найдем куб числа: test(2, function(num) { return num * num * num; // возвращает куб });

Let the functiontestthe first parameter takes a number, and the second and third parameters take functions, also taking numbers as a parameter. Let the functiontestreturns the sum of the results of the passed functions:

function test(num, func1, func2) { return func1(num) + func2(num); }

Call the functiontest, passing the number as the first parameter3, the second parameter is a function that squares a number, and the third is a function that cubes a number. Output the result of your work to the console.
