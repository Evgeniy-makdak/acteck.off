Callback functions in JavaScript

Functions can be passed as parameters to other functions. Such parameter functions are calledcallbacks (callback). Let's see how to work with them using an example.

Suppose we want to make a function whose first parameter will take an array, and the second will be a callback that will be applied to each element of the array:

function each(arr, callback) { // тут какой-то код }

Let's write the implementation of our function:

function each(arr, callback) { let result = []; for (let elem of arr) { result.push( callback(elem) ); // вызываем функцию-коллбэк } return result; }

Our functioneach- universal. This means that we can pass various callbacks to it, performing different operations on arrays. At the same timeour function code will remain unchanged- only the transmitted callbacks will change.

For example, let's use our function to square each element of an array. To do this, we pass the corresponding callback as a parameter:

let result = each([1, 2, 3, 4, 5], function(num) { return num ** 2; }); console.log(result);

Now let’s build the elements of the array into a cube. To do this, we will pass as a parameter another callback that performs this operation:

let result = each([1, 2, 3, 4, 5], function(num) { return num ** 3; }); console.log(result);

Given an array of numbers. Using the function we createdeachdouble each element.

Given an array with strings. Using the function we createdeachreverse the characters of each line.

Given an array with strings. Using the function we createdeachcapitalize the first character of each line.
