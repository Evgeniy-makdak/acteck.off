Function returning a function in JavaScript

Let us have a function like this that returns a string:

function func() { return '!'; } let result = func(); console.log(result); // выведет '!'

Let's now return an anonymous function instead of a string:

function func() { return function() { return '!'; }; } let result = func(); // в переменной result будет функция console.log(result); // выведет 'function() {return '!';}'

As you can see, the variableresultnow represents a function. Let's output the result of its work to the console. To do this, we will write it in parentheses:

function func() { return function() { return '!'; }; } let result = func(); console.log( result() ); // выведет '!'

Since the challengefunc()returns a function, then we can immediately call this returned function, like this:func()()- the first parentheses get the result of the functionfunc(which is itself a function) and the second parentheses are applied to the resultfunc.

Let's try:

function func() { return function() { return '!'; }; } console.log( func()() ); // выведет '!'
