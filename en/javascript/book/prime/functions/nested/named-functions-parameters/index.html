Named parameter functions in JavaScript

Functions that are passed as parameters do not have to be anonymous.

Let's make them as Function Declaration. Let's call the first functionget1, and the second -get2:

function get1() { return 1; } function get2() { return 2; }

Let's pass it into the function parameterstestfunction namesget1Andget2(that is, their source code, not the result):

function get1() { return 1; } function get2() { return 2; } test(get1, get2); // выведет 3 function test(func1, func2) { console.log( func1() + func2() ); }

Let's change it to Function Expression:

let get1 = function() { return 1; } let get2 = function() { return 2; } test(get1, get2); // выведет 3 function test(func1, func2) { console.log( func1() + func2() ); }
