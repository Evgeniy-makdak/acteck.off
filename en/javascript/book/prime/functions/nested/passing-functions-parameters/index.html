Parameters of passed functions in JavaScript

Let us have a functiontest, which takes another function as a parameter and displays the result of this passed function in the console:

function test(func) { console.log( func() ); }

Let the passed functionfunctakes a number as a parameter and does something with it. Let's give her, for example, the number3:

function test(func) { console.log( func(3) ); }

Let's now call the functiontest, passing an anonymous function as a parameter. This anonymous function will take a number as a parameter and return the square of that number.

As a result of all this, our construction will output the square of the number3, that is9:

// Выведет 9: test( function(num) { return num * num; } ); function test(func) { console.log(func(3)); }

Let's format the code more elegantly:

// Выведет 9: test(function(num) { return num * num; }); function test(func) { console.log(func(3)); }
