Arrow functions in JavaScript

Let's now look atarrow functions, simplifying the syntax of functions.

In the following code example, the first is a regular function, and the second is its corresponding arrow function (both functions do the same thing):

let func1 = function(num1, num2) { let result = num1 * num2; return result; } let func2 = (num1, num2) => { let result = num1 * num2; return result; }

If the function contains one line of code, then in arrow functions you don’t have to writereturnand curly braces:

let func1 = function(num1, num2) { return num1 * num2 } let func2 = (num1, num2) => num1 * num2;

If there is only one parameter of the arrow function, you don’t have to write parentheses:

let func1 = function(num) { return num * num; } let func2 = num => num * num

If the function has no parameters at all, you need to write empty parentheses:

let func1 = function() { console.log('!!!'); } let func2 = () => console.log('!!!')
