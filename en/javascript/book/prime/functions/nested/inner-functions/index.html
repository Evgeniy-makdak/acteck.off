Nested Functions in JavaScript

Let us have a function that takes two numbers as a parameter and returns the sum of the squares of these numbers:

function func(num1, num2) { return num1 * num1 + num2 * num2; } console.log(func(2, 3)); // выведет 13 (результат 2 * 2 + 3 * 3)

Let's move the squaring operation into an auxiliary functionsquare:

function square(num) { return num * num; } function func(num1, num2) { return square(num1) + square(num2); } console.log(func(2, 3)); // выведет 13

Let us be sure that the functionsquarewill only be used in the functionfuncand no other.

In this case, it would be convenient to make sure that no one accidentally uses it (you yourself or another programmer working with you).

JavaScript has an elegant solution: you can use our helper functionsquareput insidefunc. In this case, our helper function will only be available internallyfunc, but outside it won’t.

Let's do this:

function func(num1, num2) { function square(num) { return num * num; } return square(num1) + square(num2); } console.log(func(2, 3)); // выведет 13

Trying to callsquareoutsidefuncwill result in an error:

function func(num1, num2) { function square(num) { return num * num; } return square(num1) + square(num2); } console.log(square(2)); //!! выдаст ошибку

Make a functionfunc, which will take two numbers as parameters and return the sum of the square of the first number with the cube of the second number. Make a helper function for thissquare, which squares a number, and an auxiliary functioncube, raising a number to cube.
