Any level of nesting in JavaScript

There may be such function calls:func()()()Andfunc()()()()- and so on ad infinitum.

To do this, you need the inner function to also return a function, that one to return another one, and so on. Here's an example:

function func() { return function() { return function() { return '!'; }; }; } console.log( func()()() ); // выведет '!'

Make a functionfunc, which when called like this:func()()()()(), will return'!'.
