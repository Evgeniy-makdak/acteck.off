Logical OR in JavaScript

Operator||is a logical OR and requires at least one condition to be met.

In the following example, if the variablenum1more0or variablenum2more0, then the condition will be true (it is enough to fulfill at least one of the conditions):

let num1 = 10; let num2 = -5; if (num1 > 0 || num2 > 0) { console.log('+++'); // сработает это } else { console.log('---'); }
