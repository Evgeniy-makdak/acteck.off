Ternary operator in JavaScript

In this lesson we will look at a specialternary operator, which is a shortened version of the designif-else. Its syntax is as follows:

let переменная = условие ? значение1 : значение2;

The operator works like this: if the condition is true, then it returnsзначение1, otherwise -значение2. Let's make an example code using this operator:

let age = 17; let adult = age >= 18 ? true: false; console.log(adult);

This code in its entirety can be rewritten as follows:

let age = 17; let adult; if (age >= 18) { adult = true; } else { adult = false; } console.log(adult);

The ternary operator should only be used in the simplest cases, as its use makes the code difficult to understand.

Rewrite the following code using a ternary operator:

let num = 1; let res; if (num >= 0) { res = '1'; } else { res = '2'; } console.log(res);
