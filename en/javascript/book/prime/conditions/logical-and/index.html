Logical AND in JavaScript

You can group comparison operations using the operator&&, which is a logical AND. In the following example, if the variablenumgreater than zero andsimultaneouslyless10, only then will it be displayed'+++':

let num = 3; if (num > 0 && num < 10) { console.log('+++'); } else { console.log('---'); }

Conditions can be imposed not on one variable, but on different ones. In the following example, if the variablenum1equal to2and at the same time a variablenum2equal to3, only then the condition will be true:

let num1 = 2; let num2 = 3; if (num1 == 2 && num2 == 3) { console.log('+++'); } else { console.log('---'); }
