Common shorthand form of validation in JavaScript

The shortened form of validation works for other data types as well. In this case, this data is first converted to a Boolean type and then compared withtrue. See example:

let test = 3; if (test) { console.log('+++'); // сработает это } else { console.log('---'); }

In fact, the given code is equivalent to the following:

let test = 3; if (Boolean(test) === true) { console.log('+++'); } else { console.log('---'); }
