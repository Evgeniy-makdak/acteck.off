Checking the remainder of a division in JavaScript

Let us have two variables with numbers:

let a = 10; let b = 3;

Let's find the remainder when dividing one variable by another:

let a = 10; let b = 3; console.log(a % b); // выведет 1

Let now the variables store such values ​​that one variable is divisible by the second:

let a = 10; let b = 5; console.log(a % b); // выведет 0

Let's write a script that will check whether one number is divisible by a second number:

let a = 10; let b = 3; if (a % b === 0) { console.log('делится нацело'); } else { console.log('делится с остатком'); }

Let us now assume that if a number is divisible by a remainder, output this remainder to the console:

let a = 10; let b = 3; if (a % b === 0) { console.log('делится нацело'); } else { console.log('делится с остатком ' + a % b); }

In the above code, it turns out that the remainder is calculated in two places, which is not optimal. Let's fix the problem:

let a = 10; let b = 3; let rest = a % b; if (rest === 0) { console.log('делится нацело'); } else { console.log('делится с остатком ' + rest); }
