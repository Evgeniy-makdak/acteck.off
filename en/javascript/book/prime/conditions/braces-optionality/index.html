Curly braces are optional in JavaScript

If in curly bracesiforelseThere will be only one expression, you don’t have to write these curly braces. Let, for example, be given the following code with all the brackets:

if (test === 0) { console.log('+++'); } else { console.log('---'); }

You can shorten it like this:

if (test === 0) console.log('+++'); else console.log('---');

Or like this:

if (test === 0) { console.log('+++'); } else console.log('---');

You can also remove all the brackets, but arrange everything not in a line, but like this:

if (test === 0) console.log('+++'); else console.log('--');

Rewrite the following code in shortened form:

if (test > 0) { console.log('+++'); } else { console.log('---'); }

Rewrite the following code in shortened form:

if (test > 0) { console.log('+++'); }
