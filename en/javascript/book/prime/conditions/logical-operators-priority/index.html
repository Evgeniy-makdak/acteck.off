Приоритет логических операторов в JavaScript

Операция && имеет приоритет над ||. В следующем примере условие будет истинно, если переменная num от 0 до 5 ИЛИ от 10 до 20:

let num = 3; if (num > 0 && num < 5 || num > 10 && num < 20) { console.log('+++'); } else { console.log('---'); }

Расскажите порядок выполнения сравнения в следующем условии:

if (num == 0 || num > 1 && num < 5 ) { console.log('+++'); } else { console.log('---'); }
