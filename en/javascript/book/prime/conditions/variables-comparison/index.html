Сравнение переменных в JavaScript

В примерах выше мы сравнивали переменную с каким-то числом. Но никто не запрещает нам сравнивать две переменные. Смотрите пример:

let test1 = 1; let test2 = 2; if (test2 > test1) { console.log('+++'); // сработает это } else { console.log('---'); }

Даны переменные test1 и test2. Проверьте, значение какой из этих переменных больше и выведите соответствующее сообщение.
