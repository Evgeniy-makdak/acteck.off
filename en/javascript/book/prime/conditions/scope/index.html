Variable scope in if-else in JavaScript

For simplicity, let us have a condition that is always met:

if (true) { }

Let's look at the following code:

if (true) { let res = '!'; } console.log(res);

As you can see, into the variableresthe value should be written'!'. However, if you run this code, an error will be displayed in the console!

The point is that the variablesdeclared inside curly braces are visible only inside those braces, and not visible outside. Scientifically speaking, we are talking aboutVariable scopes.

We can say that the scope of variables declared inside curly braces is only those curly braces. However, if a variable is declared outside the curly braces, it is visible both inside and outside them:

let res; // переменная объявлена снаружи if (true) { res = '!'; } console.log(res); // выведет '!'

In the following example, into a variableresDepending on the condition, either the value is written1, or value2:

let num = 5; if (num >= 0) { let res = 1; } else { let res = 2; } console.log(res);

However, if you run this code, an error will be displayed in the console. As you now understand, it's all about the visibility of variables. To solve the problem, let's declare a variable outside the condition:

let test = true; let res; // объявим переменную снаружи условия if (test) { res = 1; } else { res = 2; } console.log(res); // выведет 1
