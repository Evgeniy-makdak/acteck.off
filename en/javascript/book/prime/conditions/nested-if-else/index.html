Nested if-else constructs in JavaScript

Constructionsif-elsecan be nested inside each other in any way. See example:

let num = 3; if (num >= 0) { if (num <= 5) { console.log('меньше или равно 5'); } else { console.log('больше 5'); } } else { console.log('меньше нуля'); }

Let in the variablenumnumber is stored. If this number falls within the range of10to99, then find the sum of the digits of this number. If the amount received is less than or equal to9, then print a message that the sum of the digits is single-digit, otherwise print a message that the sum of the digits is two-digit.
