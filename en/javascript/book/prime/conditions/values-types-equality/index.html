Equality by value and type in JavaScript

Suppose you want to compare so that the number in quotes is not equal to the same number without quotes. In other words, you want to compare so that the comparison is not only by value, but also by data type.

To do this, instead of the operator==operator should be used===. In the following example, when comparing a string'3'and numbers3the condition will be false, since the variables, although equal in value, are not equal in type:

if ('3' === 3) { console.log('+++'); } else { console.log('---'); // сработает это }

But when comparing two strings'3'the condition will be true:

if ('3' === '3') { console.log('+++'); // сработает это } else { console.log('---'); }

The same as when comparing numbers:

if (3 === 3) { console.log('+++'); // сработает это } else { console.log('---'); }

Difference between operator==and operator===It appears precisely when the values ​​are the same, but the data type is different. In other cases, these operators work the same. For example, when comparing different numbers, of course, it will display'---':

if (2 === 3) { console.log('+++'); } else { console.log('---'); // сработает это }
