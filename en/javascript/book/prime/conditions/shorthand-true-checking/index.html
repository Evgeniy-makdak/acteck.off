Shorthand form of testing for truth in JavaScript

Suppose we want to find out whether a variable is equal totestmeaningtrue. In this case, the designifcan be written like this:

let test = true; if (test == true) { console.log('+++'); } else { console.log('---'); }

In programming, such checks are required very often, so there is a more elegant shorthand for them: instead ofif (test == true)you can simply writeif (test).

Let's rewrite our code in a shortened form:

let test = true; if (test) { console.log('+++'); } else { console.log('---'); }

Rewrite the following code using the shortened form:

let test = true; if (test === true) { console.log('+++'); } else { console.log('---'); }
