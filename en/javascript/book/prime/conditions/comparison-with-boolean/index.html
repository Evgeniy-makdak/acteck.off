Comparing values ​​with boolean type in JavaScript

In the previous example, we used the operator for comparison===. In this case, our variable was compared for equalitytrueboth by value and type.

For such a comparison, you can also use the operator==. If in a variabletestthere will always be one of the valuestrueorfalse, then nothing will change:

let test = true; // тут пишем или true, или false if (test == true) { console.log('+++'); } else { console.log('---'); }

But if in a variabletestAny values ​​can fall, then everything becomes much more complicated.

In this case, if in the variabletestIf there is a non-Boolean value, then this value will first be converted to a Boolean and only then will be compared.

Let, for example, in the variabletestthere will be a number1. In this case, it is first converted to a logical type, that is, totrue. And then the comparison will be done:

let test = 1; if (test == true) { console.log('+++'); // сработает это } else { console.log('---'); }

But, for example, the number0converted tofalse. And our condition will result in'неверно':

let test = 0; if (test == true) { console.log('+++'); } else { console.log('---'); // сработает это }

In fact, such a comparison can be explicitly rewritten as follows:

let test = 1; if (Boolean(test) == true) { console.log('+++'); } else { console.log('---'); }
