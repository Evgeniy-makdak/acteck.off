Inverting Boolean Expressions in JavaScript

Consider the following code:

if (num > 0 && num < 5) { console.log('+++'); } else { console.log('---'); }

Now our condition is:nummust be from0to5. Let's invert this condition, that is, turn it into its opposite. The opposite condition would look like this:nummust be less than or equal0OR greater than or equal to5:

if (num <= 0 || num >= 5) { console.log('+++'); } else { console.log('---'); }

As you can see, in order to invert the condition, you have to think a little. It will be much easier to use the operator!, which is a logicalНЕ. Using this operator, we just need to put the sign!before the original condition - and it inverts itself:

if ( !(num > 0 && num < 5) ) { console.log('+++'); } else { console.log('---'); }

The following code is given:

if (num1 >= 0 || num2 <= 10) { console.log('+++'); } else { console.log('---'); }

Using the operator!invert the given condition.
