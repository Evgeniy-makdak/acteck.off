Checking for inequality in JavaScript

Operator!=checks for inequality. Let's check for example that the variabletestnot equal0:

let test = 1; if (test != 0) { console.log('+++'); // сработает это } else { console.log('---'); }

Let's change the value of the variable:

let test = 0; if (test != 0) { console.log('+++'); } else { console.log('---'); // сработает это }
