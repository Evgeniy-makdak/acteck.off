else if construct in JavaScript

Designelse ifallows you to set conditions in a blockelse. Let's look at an example:

let num = 1; if (num == 1) { console.log('value1'); } else if (num == 2) { console.log('value2'); } else if (num == 3) { console.log('value3'); }

Advantage of usingelse ifinstead of several ifs, it is possible to catch a situation when the value of a variablenumdoes not meet any of the following conditions:

let num = 1; if (num == 1) { console.log('value1'); } else if (num == 2) { console.log('value2'); } else if (num == 3) { console.log('value3'); } else { console.log('неверное значение переменной num'); }
