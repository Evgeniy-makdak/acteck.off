Switch-case construct in JavaScript

By designswitch-caseyou can execute different code depending on the value of the variable. Here is the syntax for this construct:

switch (переменная) { case 'значение1': /* здесь код, который выполнится в случае, если переменная имеет значение1 */ break; case 'значение2': /* здесь код, который выполнится в случае, если переменная имеет значение2 */ break; case 'значение3': /* здесь код, который выполнится в случае, если переменная имеет значение3 */ break; default: /* здесь код, который выполнится в случае, если не совпала ни с одним значением */ break; }

Let's write an example code with the constructionswitch-case:

let num = 1; switch (num) { case 1: console.log('value1'); break; case 2: console.log('value2'); break; case 3: console.log('value3'); break; default: console.log('incorrect value'); break; }

Rewrite the following code viaswitch-case:

let lang = 'ru'; if (lang == 'ru') { console.log('rus'); } else if (lang == 'en') { console.log('eng'); } else if (lang == 'de') { console.log('deu'); } else { console.log('language not supported'); }
