if-else construct in JavaScript

Now we will study the special designif-else, which allows you to execute some code depending on the fulfillment of some condition.

By designif-elsewe can ask the program, for example, the question “is the variable greaterathan variableb", and depending on the answer, execute either one code or another.

The syntax of the construct looks like this:

if (операция сравнения) { /* расположенный здесь код выполниться, если результат операция сравнения верный */ } else { /* расположенный здесь код выполниться, если результат операция сравнения неверный */ }
