Button to start a timer in JavaScript

In previous lessons, our timer began its work immediately upon loading the page. Let's now make a button that, when clicked, will start our timer:

<button id="start">start</button>

Let's write the corresponding JavaScript:

let start = document.querySelector('#start'); start.addEventListener('click', function() { let i = 0; setInterval(function() { console.log(++i); }, 1000); });
