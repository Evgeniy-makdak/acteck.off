Basics of working with the setInterval function in JavaScript

In this tutorial we will learn how to work with timers in JavaScript. Once you understand timers, you can automatically perform any operations on the page after a specified period of time. For example, it will be possible to make a picture slider in which the pictures will change every second.

To work with timers in JavaScript, use the functionsetInterval, which runs specified code at certain intervals.

This function works as follows: as the first parameter it takesfunction source code, and the second parameter is the interval after which this function will be automatically called. The second parameter is specified in milliseconds (1000 milliseconds = 1 second).

Let's study the function using an example. For example, let's write a code that will output something to the console every second.

First, let's make a function that prints something to the console:

function timer() { console.log('!'); }

And now with the helpsetIntervalLet's force the function we created to be executed every second:

setInterval(timer, 1000); function timer() { console.log('!'); }

It is not necessary to create a separate function - you can just add it to the first parametersetIntervalpass an anonymous function like this:

setInterval(function() { console.log('!'); }, 1000);

Start a timer that every3seconds it will print something to the console.
