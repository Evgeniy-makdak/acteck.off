A button is given. Given a paragraph whose text is a number, for example,10. By clicking the button, start a timer that will reduce the text of the paragraph by every second.1. As soon as the paragraph value becomes zero, stop the timer.

An input with a number is given. Make sure that every second the square of the number written in it appears in it.

Input given. A number is entered into it. When you lose focus, make the paragraph below start counting down, starting from the entered number. When the countdown reaches zero, it should end.
