Buttons to start and stop a timer in JavaScript

Let's now make two buttons: by clicking on the first, let the timer start, and by clicking on the second, it will stop. Here everything will not be so simple and a catch awaits us. In order to understand the essence of this trick, I will describe the creation of the code step by step.

So, we make two buttons:

<button id="start">start</button> <button id="stop">stop</button>

We get links to these buttons in variables:

let start = document.querySelector('#start'); let stop = document.querySelector('#stop');

By clicking on the first button, we start the timer by writing its number into a variable:

start.addEventListener('click', function() { let timerId = setInterval(function() { console.log('!') }, 1000); });

Now let’s stop the timer by pressing the second button:

start.addEventListener('click', function() { let timerId = setInterval(function() { console.log('!') }, 1000); }); // Останавливаем таймер: stop.addEventListener('click', function() { clearInterval(timerId); });

However, if we try to run the above code, we are in for a surprise: when we try to stop the timer, it turns out that the variabletimerIdequal toundefined! Why did this happen? Therefore, when starting the timer, we made our variabletimerIdlocal inside a function bound to a buttonstart.

To solve the problem, let's create a variabletimerIdglobal - in this case it will be available both in the timer start function and in the stop function:

let timerId; // сделаем переменную глобальной start.addEventListener('click', function() { timerId = setInterval(function() { console.log('!') }, 1000); }); stop.addEventListener('click', function() { clearInterval(timerId); });

Let a variable be given that initially stores a number100. Two buttons are also given. By clicking on the first button, start a timer that will decrease the value of the variable by every second.1and output the new value to the console. As soon as the variable value reaches zero, stop the timer.

By pressing the second button, stop the timer. Also stop the timer if the second button was not pressed but the variable value reached zero.

Some programmer wrote a code that, when a button is pressed, starts a timer that displays the current moment in time to the console:

<input type="submit" id="start" value="start"> <input type="submit" id="stop" value="stop"> let start = document.querySelector('#start'); let stop = document.querySelector('#stop'); start.addEventListener('click', function() { let timerId = setInterval(function() { let date = new Date; console.log(date.getMinutes() + ' ' + date.getSeconds()); }, 1000); }); stop.addEventListener('click', function() { clearInterval(timerId); });

After running the code, however, it turned out that the stop button did not work. Fix the code author's mistake.
