Stopping a JavaScript Timer

You already know how to start a timer, now let's learn how to stop it. To do this, you need to know that every timer started using the functionsetInterval, has a unique number. This number is returned by the functionsetIntervalwhen the timer starts:

let timerId = setInterval(function() { console.log('!') }, 1000); alert(timerId); // выведет номер таймера

To stop the timer, use the functionclearInterval, which takes the unique number of the timer that needs to be stopped.

For example, let's start a timer that displays numbers in ascending order to the console, starting with1. Stop the timer as soon as the number is displayed on the screen10:

let i = 0; let timerId = setInterval(function() { console.log(++i); if (i >= 10) { clearInterval(timerId); } }, 1000);

Let a variable be given that initially stores a number10. Start a timer that will decrease the value of this variable by every second.1and output this value to the console. As soon as the variable value reaches zero, stop the timer.
