Timers and loss of context in JavaScript

When using timers in event handlers, we encounter problems with loss of context. Let's look at an example.

Let us have an input:

<input id="elem" value="text">

Let an anonymous function be triggered by clicking on this input, and a timer will start inside this function, outputting something to the console every second:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { setInterval(function() { console.log('!!!'); // что-нибудь выводим в консоль }, 1000); });

So far everything is working correctly. But now we want to output to the consolevalueour input - a surprise awaits us: it will be output to the consoleundefined:

elem.addEventListener('click', function() { setInterval(function() { console.log(this.value); // будет выводится undefined }, 1000); });

The thing is that we have a function within a function: there is an external anonymous function that is called when clicked and an internal anonymous function that is started by a timer. In an external functionthisindicates an input, but not internally. There is a loss of context.

Why is it displayed?undefined, and doesn’t an error appear in the console, as was the case in previous lessons? Becausethisinside a function called viasetInterval, indicateswindow.

This means that we are trying to read the value property of the objectwindow, like this:window.value, but there is no such property in it, and we getundefined(not a mistake).

Let's fix the problem by introducingself:

elem.addEventListener('click', function() { let self = this; setInterval(function() { console.log(self.value); }, 1000); });

Let the following code be given:

<input type="button" id="elem" value="1"> let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { setInterval(function() { this.value = Number(elem.value) + 1; }, 1000); });

The author of the code wanted that when you press a button, the value of this button increases by every second1. However, when you press the button, nothing happens at all.
