Starting a Timer in JavaScript

Although the functionsetTimeoutand is not intended for creating timers, but they can still be made using it if you userecursion:

let i = 0; function timer() { setTimeout(function() { console.log(++i); timer(); // вызовем сами себя }, 1000); } timer();

You can stop such a timer simply by preventing recursion from happening:

let i = 0; function timer() { setTimeout(function() { console.log(++i); if (i < 10) { // запускаем, только если счетчик меньше 10 timer(); } }, 1000); } timer();

Print the number to the console0. After a second, print the number1, after two seconds print the number2, through3seconds print the number3. And so on ad infinitum.
