Adding Elements to Multidimensional JavaScript Objects

Let us be given the following object with students:

let students = { 'group1': ['student11', 'student12', 'student13'], 'group2': ['student21', 'student22', 'student23'], 'group3': ['student31', 'student32'], };

Let's add another student to the first group:

students.group1.push('student14');

Let's now make another, fourth group and add a student to it. To do this, you must first declare the new group as an array:

students.group4 = [];

And only then you can add students to it:

students.group4.push('student41');

let affairs = { '2019-12-28': ['data11', 'data12', 'data13'], '2019-12-29': ['data21', 'data22', 'data23'], '2019-12-30': ['data31', 'data32', 'data33'], }
