Adding Elements to Multidimensional JavaScript Arrays

Let's say we have this array of users:

let users = [ { name: 'name1', surn: 'surn1', }, { name: 'name2', surn: 'surn2', }, { name: 'name3', surn: 'surn3', }, ];

Let's add one more user to this array. To do this, the method parameterpushLet's pass a new object with the user:

users.push({ name: 'name4', surn: 'surn4', });
