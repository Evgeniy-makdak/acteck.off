Multidimensional Structures in JavaScript

You can combine multidimensional arrays and objects to create different structures. Let's, for example, make an object containing arrays inside it:

let users = { 'boys': ['boy1', 'boy2', 'boy3'], 'girls': ['girl1', 'girl2', 'girl3'], };

Let's derive some element from this object:

console.log(users['boys'][1]); // выведет 'boy2'

Given the following object with students:

let students = { 'group1': ['name11', 'name12', 'name13'], 'group2': ['name21', 'name22', 'name23'], 'group3': ['name31', 'name32', 'name33'], };

Display the first student from the third group.
