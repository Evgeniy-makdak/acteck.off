Keys from Variables in Multidimensional JavaScript Structures

Let the following object be given:

let obj = { 'sub1': ['11', '12', '13'], 'sub2': ['21', '22', '23'], };

Let's derive some element from our object, for example, element'22':

console.log(obj['sub2'][1]);

Now let the keys be stored in variables:

let key1 = 'sub2'; let key2 = 1;

Using our variables, we display an element of the multidimensional structure:

console.log(obj[lang][day]);

Here is the structure for storing a to-do list for years, months and days:

let affairs = { '2018': { 11: { 29: ['name111', 'name112', 'name113'], 30: ['name121', 'name122', 'name123'], }, 12: { 30: ['name211', 'name212', 'name213'], 31: ['name221', 'name222', 'name223'], }, }, '2019': { 12: { 29: ['name311', 'name312', 'name313'], 30: ['name321', 'name322', 'name323'], 31: ['name331', 'name332', 'name333'], } }, }

Let there also be three variables containing the year, month and day. Print the case corresponding to the values ​​of the variables.
