Multidimensional Arrays in JavaScript

Array elements can be not only strings and numbers, but also arrays. In this case we will get an array of arrays ormultidimensional array.

In the following example, the arrayarrconsists of three elements, which in turn are arrays:

let arr = [['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i']];

Let's rewrite it in a more understandable form:

let arr = [ ['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i'], ];

Depending on the level of nesting, arrays can be two-dimensional - an array of arrays, three-dimensional - an array of arrays of arrays (and so on - four-dimensional, five-dimensional, etc.).

The above array is two-dimensional, since inside one array there are other subarrays and there are no other arrays in these subarrays.

To display any element from a two-dimensional array, you should write not one pair of square brackets, but two:

let arr = [ ['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i'], ]; console.log(arr[0][1]); // выведет 'b' console.log(arr[1][2]); // выведет 'f'

Given the following array:

let arr = [ ['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i'], ['j', 'k', 'l'], ];

Use it to display elements with text'l', 'e', 'g'And'a'.

let arr = [[1, 2], [3, 4], [5, 6]];

Referring to each element of the array, find the sum of all its elements.
