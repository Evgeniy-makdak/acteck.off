Adding elements via push in JavaScript

Using a special methodpushYou can add elements to the end of the array. Let's see how it's done. Let's say we have the following array:

let arr = [];

Let's add three elements to the end:

arr.push('a'); arr.push('b'); arr.push('c');

Let's see what happened:

console.log(arr); // выведет ['a', 'b', 'c']
