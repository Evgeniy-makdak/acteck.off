Increment and Decrement of Array Elements in JavaScript

You can also use increment and decrement operations:

let arr = [1, 2, 3, 4]; arr[0]++; ++arr[1]; arr[2]--; --arr[3]; console.log(arr); // выведет [2, 3, 2, 3]

Create an array with elements1, 2And3. Using an operator++increment each array element by one.
