Array length in JavaScript

The length of the array is found using the propertylength:

let arr = ['a', 'b', 'c']; console.log(arr.length) // выведет 3

Knowing the length of the array, you can get its last element:

let arr = ['a', 'b', 'c']; console.log(arr[arr.length - 1]) // выведет 'c'
