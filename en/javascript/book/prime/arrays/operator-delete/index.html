The delete operator in arrays in JavaScript

You can remove array elements using the operatordelete. Let's look at examples. Let us be given the following array:

let arr = ['a', 'b', 'c'];

Let's remove one of the elements from our array:

delete arr[1];

As a result, the element will be removed, but the array will become sparse:

console.log(arr); // выведет ['a',, 'c']

Given an array:

let arr = ['a', 'b', 'c', 'd', 'e'];

Remove two elements from it. Check what the property value becomeslengthafter that.
