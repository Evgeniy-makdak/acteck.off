Finding errors in code with JavaScript arrays

In the following problems, some programmer wrote the code and may have made mistakes in it. You should check if the code does what it says. If the code does not work correctly, you must fix the errors.

The code should print the last element of the array:

let arr = [1, 2, 3, 4, 5]; console.log(arr[arr.length]);

The code should find the sum of the array elements:

let arr = [1, 2, 3, 4, 5]; console.log(arr[1] + arr[2] + arr[3] + arr[4] + arr[5]);

The code should print the length of the array:

let arr = {1, 2, 3, 4, 5}; console.log(arr.length]);

The code should print the length of the array:

let arr = [1, 2, 3, 4, 5]; console.log(arr.lenght);
