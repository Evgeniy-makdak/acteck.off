Sparse Arrays in JavaScript

If the array contains keys with gaps, then you can getsparsearray. Let's see what it looks like. Let us have some array:

let arr = ['a', 'b', 'c'];

Let's add one more element to this array so that the keys have spaces:

arr[4] = '!';

As a result, a hole is formed in the array with the valueundefined:

console.log(arr); // выведет ['a', 'b', 'c', undefined, '!']

The length of the array will include all the holes. That is, in our case it will be5, not4:

console.log(arr.length); // выведет 5
