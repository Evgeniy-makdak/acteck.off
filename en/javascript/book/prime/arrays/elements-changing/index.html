Changing array elements in JavaScript

Array elements can be changed:

let arr = ['a', 'b', 'c']; arr[0] = '!'; console.log(arr); // выведет ['!', 'b', 'c']

Create an array with elements'a', 'b', 'c'. Write a number instead of the first element1, instead of the second -2, instead of the third -3.
