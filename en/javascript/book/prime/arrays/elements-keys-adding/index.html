Adding elements by JavaScript keys

Elements in an array do not have to be added immediately at the time the array is declared. You can first declare this array, and then add the necessary elements to it, like this:

let arr = []; arr[0] = 'a'; arr[1] = 'b'; arr[2] = 'c'; console.log(arr); // выведет ['a', 'b', 'c']

You can also add elements to an array that is already filled with data:

let arr = ['a', 'b', 'c']; arr[3] = 'd'; console.log(arr); // выведет ['a', 'b', 'c', 'd']
