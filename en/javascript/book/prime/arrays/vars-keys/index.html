Array keys from variables in JavaScript

Let us be given the following array:

let arr = ['a', 'b', 'c'];

Let's display the element with the key0:

let arr = ['a', 'b', 'c']; console.log(arr[0]); // выведет 'a'

Let's now not write the key of the output element directly in square brackets, but write it into a variable:

let arr = ['a', 'b', 'c']; let key = 0; // запишем ключ в переменную

We now use our variable to display the corresponding element:

let arr = ['a', 'b', 'c']; let key = 0; // запишем ключ в переменную console.log(arr[key]); // выведет 'a'

Given the following array:

let arr = ['a', 'b', 'c'];

A variable is also given:

let key = 2;

Display an element whose key is stored in a variablekey.

Given the following array:

let arr = [1, 2, 3, 4, 5];

Variables are also given:

let key1 = 1; let key2 = 2;

Find the sum of the elements whose keys are stored in our variables.
