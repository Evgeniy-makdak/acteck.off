Arrays in JavaScript

Now we'll start exploring a special data type calledarray. An array is a variable in which a whole set of values ​​can be stored in an ordered form.

The values ​​that an array stores are calledelements. Elements are separated by a comma. It is customary to put spaces after this comma.

Square brackets are used to create an array:

let arr = [];

Let's fill our array with strings:

let arr = ['a', 'b', 'c'];

Now let's fill the array with numbers:

let arr = [1, 2, 3];

In addition to strings and numbers, you can store all valid data types in an array, and also mix them together in one array, example:

let arr = [1, 2, 'a', 'b', null, true, false];
