Getting Array Elements in JavaScript

Array elements are accessed similarly to string characters. Each element of the array has its own number: the first element is the number0, second - number1and so on. These numbers are calledkeysarray elements. Let's look at an example. Let us be given the following array:

let arr = ['a', 'b', 'c'];

Let's display its elements:

console.log(arr[0]); // выведет 'a' console.log(arr[1]); // выведет 'b' console.log(arr[2]); // выведет 'c'
