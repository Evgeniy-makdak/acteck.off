Overwriting Array Elements in JavaScript

You can also read the current value of an element, perform some operations on it, and write the changed value back to that element:

let arr = ['a', 'b', 'c']; arr[0] = arr[0] + '!'; arr[1] = arr[1] + '!'; arr[2] = arr[2] + '!'; console.log(arr); // выведет ['a!', 'b!', 'c!']

The previous code can be rewritten using the operator+=:

let arr = ['a', 'b', 'c']; arr[0] += '!'; arr[1] += '!'; arr[2] += '!'; console.log(arr); // выведет ['a!', 'b!', 'c!']
