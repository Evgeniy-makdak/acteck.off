Function that merges arrays into one in JavaScript

Let's now implement the functionmerge, which takes an arbitrary number of arrays as parameters and merges their elements into one array.

Here is an example of our function working:

let result = merge([1, 2, 3], [4, 5, 6], [7, 8, 9]); console.log(result); // выведет [1, 2, 3, 4, 5, 6, 7, 8, 9]

Let's start implementation. Let's first get the passed arrays in the form of one two-dimensional one:

merge([1, 2, 3], [4, 5, 6], [7, 8, 9]); function merge(...arrs){ console.log(arrs); // выведет [ [1, 2, 3,] [4, 5, 6], [7, 8, 9] ] }

Let's now merge this two-dimensional array into a one-dimensional one. For this we use the methodconcatand the spread operator:

let arrs = [ [1, 2, 3,], [4, 5, 6], [7, 8, 9] ]; let result = [].concat(...arrs); console.log(result); // выведет [1, 2, 3, 4, 5, 6, 7, 8, 9]

Let's add this code to our functionmerge:

function merge(...arrs) { return [].concat(...arrs); } let result = merge([1, 2, 3], [4, 5, 6], [7, 8, 9]); console.log(result); // выведет [1, 2, 3, 4, 5, 6, 7, 8, 9]

As you can see, this function also turned out to be very concise.
