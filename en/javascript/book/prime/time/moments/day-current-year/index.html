Getting the day of the current year in JavaScript

Let's get a date object containing8March 1st of the current year. Under the wordscurrent yearWhat this means is that the script should work in any year, always getting the year that is currently on the planet.

To do this, we need to make another auxiliary date object containing the current point in time. With this object we can get the current year and then use that year when creating an object with the date we want, like this:

let now = new Date(); // получаем текущий момент let date = new Date(now.getFullYear(), 2, 8); // получаем нашу дату

We can use the resulting point in time, for example, to determine the day of the week corresponding to this date:

console.log(date.getDay());

Determine what day of the week it will be31December of this year.
