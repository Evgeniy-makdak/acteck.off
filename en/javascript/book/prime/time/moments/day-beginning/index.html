Start of the day in JavaScript

Let's get a date object containing the start of the current day:

let now = new Date(); let date = new Date(now.getFullYear(), now.getMonth(), now.getDate(), 0, 0, 0);

As you already know, object parametersDatecan be lowered from the end. In this case hours, minutes and seconds will matter0. Let's omit them:

let now = new Date(); let date = new Date(now.getFullYear(), now.getMonth(), now.getDate());

But the day cannot be omitted, because if it is omitted, it will take on the value1, and we need the current day. It is also impossible to omit the month without omitting the year according to the rules of working with Date.

It is also impossible to omit the year, month and day at the same time - in this case the current moment in time will be taken. Why is this bad, since we need the current year, the current month and the current day? The point is that we need midnight, that is, hours, minutes and seconds should matter0, and at the current moment in time they will have current values, and not midnight.
