Date falling within range

Let there be three intervals: from1January to8March, from9March to17June, from18June to31December. Let us also have some date containing a month and a day. Let's determine what period this date falls into:

let date = '08-20'; if (date >= '01-01' && date <= '03-08') { console.log('1 промежуток'); } if (date >= '03-09' && date <= '06-17') { console.log('2 промежуток'); } if (date >= '06-18' && date <= '12-31') { console.log('3 промежуток'); }
