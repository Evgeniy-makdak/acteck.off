String comparison of dates in JavaScript

Let us have two dates in the following text format:

let date1 = '2020-12-01'; let date2 = '2019-12-01';

In this case, you can compare these dates and find out which of these dates is greater:

console.log(date1 > date2); // выведет true

How are these dates compared? The thing is that our dates are strings and JavaScript compares them as strings. That is, it first compares the first characters of two dates: if they are the same, then JavaScript compares the second characters, and so on until it finds differences. Due to the fact that in our date format the year is located first, then the month, and then the day, such a comparison is possible.

The fact is that if the number of the first year turns out to be greater than the number of the second year, then it no longer matters what happened to the months and days - the first year is definitely greater. If the years coincide, then the date with more than a month will be greater. And if the months coincide, then the date with the longer day will be greater. Well, if the days are the same, then the dates are equal.

It is also important that the dates are in the same format. In our case, the date separators are hyphens. This is, of course, not necessary. For example, you can put points:

let date1 = '2020.12.01'; let date2 = '2019.12.01';

Or remove separators altogether:

let date1 = '20201201'; let date2 = '20191201';

The main thing is that the comparison is correct, the placement should be as follows: first the year, then the month, then the day.
