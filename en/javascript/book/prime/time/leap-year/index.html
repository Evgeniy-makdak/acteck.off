Defining Leap Year in JavaScript

Using the techniques we learned earlier, we can easily determine for any year whether it is a leap year or not: we just need to find out how many days there are in February. To do this, we need to take the zero day of March:

let date = new Date(2020, 2, 0); console.log(date.getDate()); // выведет 29, тк 2020 год - високосный

Let's improve our code so that the year type is displayed in text:

let date = new Date(2020, 2, 0); if (date.getDate() == 29) { console.log('високосный'); } else { console.log('обычный'); }

Make a functionisLeap, which will take the year as a parameter and returntrue, if this year is a leap year, andfalse- if not.
