Getting time as timestamp in JavaScript

When working with dates, there is a special timestamp format, which in JavaScript shows the number of milliseconds that have passed since1January th1970year to the current (or specified) point in time.

There is a special methodgetTime, with which you can get the time in timestamp format. Let's, for example, get the current moment in time in this format:

let date = new Date(); console.log(date.getTime());

Let's get the given point in time in timestamp format:

let date = new Date(2015, 11, 4, 23, 59, 59); console.log(date.getTime());
