Setting the time in a Date object in JavaScript

When we created a date object like this:new Date(), - we left the parentheses empty. In this case, we received the current moment in time.

However, you can do it differently - pass it parameters in the formatnew Date(year, month, day, hours, minutes, seconds, milliseconds)and in this case into a variabledateIt will not be the current moment in time that is recorded, but the one that we specified in the parameters. In this case, when passed as a parameter, the month count starts from zero.

Parameters can be omitted from the end. In this case, missing parameters for milliseconds, seconds and hours are considered equal to zero, and for days - one. The year and month cannot be omitted.

Let's create an object with a date for a specific point in time:

let date = new Date(2025, 10, 5, 12, 59, 59);

The ability to specify a point in time can be used, for example, to find out the day of the week for a specific date:

let date = new Date(2025, 10, 5); // установим 5 ноября 2025 года let day = date.getDay(); let days = ['вс', 'пн', 'вт', 'ср', 'чт', 'пт', 'сб']; console.log(days[day]);
