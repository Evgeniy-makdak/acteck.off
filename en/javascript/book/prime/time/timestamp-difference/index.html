Difference between dates in timestamp format in JavaScript

The timestamp format is designed to find the difference between dates. For example, let's get the difference in milliseconds between the current and a given point in time:

let now = new Date(); let date = new Date(2015, 11, 4, 23, 59, 59); let diff = now.getTime() - date.getTime(); console.log(diff);

Obviously, most often we need a difference not in milliseconds, but in days or years. To do this, we just need to convert milliseconds to the value we need.

For example, to convert milliseconds to seconds, you need to divide milliseconds by1000To convert seconds to minutes, you need to divide the seconds by60and so on.

Let's, for example, convert the difference between dates into minutes:

console.log(diff / (1000 * 60));

And now - on the clock:

console.log(diff / (1000 * 60 * 60));
