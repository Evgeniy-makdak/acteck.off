Printing date parts as words in JavaScript

You already know how to display the day of the week as a number using the methodgetDay. Often, however, we do not need the number of the day of the week, but its text name. Let's write code that will convert the numbers returned by the methodgetDay, in the corresponding names of the days of the week.

First, let's write the code that will display the number of the current day of the week:

let date = new Date(); let day = date.getDay(); console.log(day);

Now let’s modify this code so that the screen displays not the number of the day of the week, but its name (for brevity, Sunday will be'вс', Monday'пн'and so on).

To solve the problem, let's create an array of days of the weekdaysand start it from Sunday (since it is day zero):

let days = ['вс', 'пн', 'вт', 'ср', 'чт', 'пт', 'сб'];

How to use this array to display, for example,'вт'? You need to pass the number of this day in square brackets (Tuesday has the number2):

let days = ['вс', 'пн', 'вт', 'ср', 'чт', 'пт', 'сб']; console.log(days[2]); // выведет 'вт'

However, we don't want to pass the day of the week manually, but rather let JavaScript determine the current day and display its name. To do this, you need to combine what the method returnsgetDay, and our arraydays:

let date = new Date(); let day = date.getDay(); let days = ['вс', 'пн', 'вт', 'ср', 'чт', 'пт', 'сб']; console.log(days[day]);
