Getting the day of the week in JavaScript

Using an objectDateyou can also get the number of the current day of the week. This is done using the methodgetDay. This method returns numbers from0to6-ty, andweek starts on sundayand this day has a number0. Monday is day number1, Tuesday - number2and so on.

Let's see how the method worksgetDayby example. Let, for example, today be Tuesday. Then the following code will print the number2:

let date = new Date(); console.log(date.getDay());
