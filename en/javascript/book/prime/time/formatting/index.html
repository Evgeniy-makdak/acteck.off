Formatting Dates in JavaScript

Let's now learn how to display a date in a specific format. Let, for example, we want to display the current day, month and year in the formatyear-month-day.

Let's do this:

let date = new Date(); console.log(date.getFullYear() + '-' + date.getMonth() + '-' + date.getDate());

Our code, however, has a problem: the month numbers will start with a zero, but we would most likely want them to start with a one.

Let's fix the problem:

let date = new Date(); console.log(date.getFullYear() + '-' + (date.getMonth() + 1) + '-' + date.getDate());

Our code, however, is still not perfect. But you can notice this only in certain months and on certain days. Let, for example, now5March th2020year.

In this case, our date will be displayed in the format2020-3-5. And we would like the date to be displayed in the format2020-03-05- with zeros before single-digit day and month numbers.

To solve the problem, let's write a functionaddZero, which will add zeros before the numbers from0to9:

function addZero(num) { if (num >= 0 && num <= 9) { return '0' + num; } else { return num; } }

Let's apply the created function and now we will actually get the date in the format we need:

let date = new Date(); console.log( addZero(date.getFullYear()) + '-' + addZero(date.getMonth() + 1) + '-' + addZero(date.getDate()) );
