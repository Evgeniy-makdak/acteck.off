Working with the Date Object in JavaScript

Now we'll start learning how to work with dates in JavaScript. For this we need an objectDate, with which you can perform various manipulations with dates in JavaScript.

A date object is created as follows:

new Date();

Let's write the created object to some variable, for example, to the variabledate:

let date = new Date();

After the operation has been performed, the variabledatewill be an object that stores the current moment in time (second, minute, hour, and so on).

Using this object with the help of special methods, we can obtain the time characteristics we need, for example, the current hour, the current day or the current month.

For example, the current hour can be obtained like this:date.getHours(), and the current month is like thisdate.getMonth(). See all options:

let date = new Date(); console.log(date.getFullYear()); // год console.log(date.getMonth()); // месяц console.log(date.getDate()); // день console.log(date.getHours()); // часы console.log(date.getMinutes()); // минуты console.log(date.getSeconds()); // секунды

Please note that the month that the method returnsgetMonth, starts from scratch - January zero, February first and so on.
