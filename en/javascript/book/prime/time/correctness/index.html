Validating Dates in JavaScript

Let's now learn how to check the date for correctness. For example,31January is the correct date, and32January - incorrect. As you know, JavaScript automatically adjusts dates. In our case this means that32January will automatically become1February.

This JavaScript property can be used to check if a date exists. How we will check: create an object with a date and see whether parts of the date have changed or not. In other words, whether JavaScript has adjusted our date or not. If you have completed it, then the date we transmitted is incorrect, and if you have not completed it, it is correct.

Let's do the following:

let year = 2025; let month = 0; let day = 32; let date = new Date(year, month, day); if (date.getFullYear() == year && date.getMonth() == month && date.getDate() == day) { console.log('корректна'); } else { console.log('некорректна'); }

Make a functioncheckDate, which will perform the described check. Let the function returntrue, if the date is correct andfalseif not. An example of how this function works for31January and32January:

console.log(checkDate(2025, 0, 31)); // выведет true console.log(checkDate(2025, 0, 32)); // выведет false
