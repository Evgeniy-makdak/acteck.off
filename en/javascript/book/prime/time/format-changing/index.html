Changing date format in JavaScript

Let's now learn how to change the date format. Let, for example, we have a string with a date in the formatyear-month-day. Let's change the format of this date to another one, for example, to this:day/month/year.

Let's solve the problem using a specific example. Let in the variabledatelies the date'2025-12-31'. Let's convert this date to'31/12/2025'

To solve the problem, let's split our string'2025-12-31'to an array using the methodsplitwith a hyphen separator. As a result, the zero element of the array will contain the year, the first - the month, and the second - the day:

let str = '2025-12-31'; let arr = str.split('-'); console.log(arr); // получим массив ['2025', '12', '31']

Now, referring to different elements of the array by their keys, we will form the string we need:

let str = '2025-12-31'; let arr = str.split('-'); let res = arr[2] + '/' + arr[1] + '/' + arr[0]; console.log(res); // получим строку '31/12/2025'

You can also use a combination of methodssplit, reverseAndjoin:

let str = '2025-12-31'; let res = str.split('-').reverse().join('/'); console.log(res); // получим строку '31/12/2025'

The date is given in the formatyear-month-day. Convert this date to formatday.month.year.
