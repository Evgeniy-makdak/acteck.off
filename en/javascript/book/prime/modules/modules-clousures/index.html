Modules via Closures in JavaScript

The problem described above is typical for any programming language. As a solution, the so-calledmodules.

A module is a kind of structure made so that the variables and functions of this structure are visible only inside it and do not interfere with anyone outside.

There are several types of modules in JavaScript. The simplestmodules via closuresare created by calling a function in place, like this:

;(function() { // тут код модуля })();

Variables and functions created in such a module will not be visible outside of that module:

;(function() { let str = 'переменная модуля'; function func() { alert('функция модуля'); } })(); // Тут переменные и функции модуля недоступны: alert(str); alert(func);
