JavaScript DOM Basics

In previous lessons, we studied the basic capabilities of the JavaScript language. Starting with this lesson, we will do what JavaScript is actually designed for - we will modify HTML page elements and respond to user actions. Our scripts will become more spectacular and useful.

In all of the following tutorials, your JavaScript code should be placed below the HTML code it is accessing. That is, the structure of your code should look like this:

<!DOCTYPE html> <html> <head> </head> <body> ваши html теги <script> ваш JavaScript код </script> </body> </html>

Or as follows, if you write JavaScript code in a separate file:

<!DOCTYPE html> <html> <head> </head> <body> ваши html теги <script src="index.js"></script> </body> </html>

In the future, for brevity, I will not give the full code, but will simply indicate what needs to be written in the HTML part and what in the JavaScript part.

Make a file with the specified code structure.
