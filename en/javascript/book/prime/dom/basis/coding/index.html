Tips for writing code using the DOM in JavaScript as an example

Suppose you are faced with a task of sufficient complexity, for the implementation of which you need to write a number of lines of code.

The wrong approach would be to try to write the entire solution code and then start testing it. In this case, there is a high probability that nothing will work for you, and you will have to look for the error in a large amount of code.

The correct approach is to break down the task into small, elementary steps that you will implement and immediately check their correctness. In this case, even if you make a mistake somewhere, you will immediately notice the problem and correct it.

Let's try it in practice. Let's say you have paragraphs:

<p>1</p> <p>2</p> <p>3</p> <p>4</p> <p>5</p> <p>6</p>

Let your task be to find paragraphs with numbers that are multiples of3, and find the sum of their numbers.

The first small step I would suggest is to get our paragraphs as an array and print this array to the console to see that we got everything right. Let's do this:

let elems = document.querySelectorAll('p'); console.log(elems);

The next small step is to loop through our paragraphs and output each of them to the console individually:

let elems = document.querySelectorAll('p'); for (let elem of elems) { console.log(elem); }

Now let’s display the texts of our paragraphs in a loop:

let elems = document.querySelectorAll('p'); for (let elem of elems) { console.log(elem.textContent); }

Now let's print the texts of those paragraphs whose number is divisible by3:

let elems = document.querySelectorAll('p'); for (let elem of elems) { let text = +elem.textContent; if (text % 3 === 0) { console.log(text); } }

After making sure that we get the correct paragraphs, we can start summing their numbers:

let elems = document.querySelectorAll('p'); let sum = 0; for (let elem of elems) { let text = +elem.textContent; if (text % 3 === 0) { sum += text; } } console.log(sum);
