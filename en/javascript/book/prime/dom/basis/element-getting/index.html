Getting a DOM element in JavaScript

Now we will learn how to obtain DOM elements of the page in order to perform some manipulations with them in the future.

Let's say we have a button:

<input id="button" type="submit">

Let's get a link to this button into a variable. To do this you should use the methodquerySelectorspecial objectdocument. This method takes a CSS selector as a parameter and returns a link to the element found using this selector.

Our button has an attributeidwith meaningbutton. So we can find this button by the selector#button. So, let's find our button and write the link to it into a variable:

let button = document.querySelector('#button'); console.log(button);
