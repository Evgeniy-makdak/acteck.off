DOM elements in JavaScript

Each page tag in JavaScript corresponds to an object with its own properties. This object stores the text of the tag, the values ​​of its attributes, as well as other useful things. In JavaScript, such objects are calledDOM elements.

Let's say our HTML code contains the following tag:

<div id="elem">text</div>

Let in the variableelemthere is a link to this tag. In this case the variableelemwill be an object with propertiesidAndtextContent. Propertyidwill contain the attribute valueidof our tag, and the propertytextContent- his text.
