This object in JavaScript

Now we will work with a special objectthis, available in the event handler function. This object points to the element where the event occurred.

Objectthisconvenient when the element in which the event occurred and the element with which actions are performed as a result of the event are the same element.

For example, if we have an input, we can attach a focus loss handler to it and, when this event occurs, do something with the text of the input. Let's do what is described. Let us be given an input:

<input id="elem" value="text">

Let's get a reference to it in a variableelem:

let elem = document.querySelector('#elem');

Let's bind an event handler function to itblur:

elem.addEventListener('blur', func);

Inside this functionfuncobject will be availablethis, indicating our input:

function func() { console.log(this); // содержит ссылку на наш элемент }

Display the contents of the attributevalueour input:

function func() { console.log(this.value); // выведем содержимое атрибута }

Well, now let’s write some text into the input:

function func() { this.value = '!!!'; }

You can also use an anonymous function:

elem.addEventListener('blur', function() { this.value = '!!!'; });

Input given. Upon receiving focus, use this input to write a number into it1, and by loss of focus - the number2. To access an input inside a handler function, use the objectthis.
