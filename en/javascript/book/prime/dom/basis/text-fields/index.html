Working with Text Fields in JavaScript

Now we will learn how to receive text from users of our site. There is a special tag in HTML for this.input, which is a text field for entering data.

<input>

This tag has a special attributevalue, which specifies the initial text that will be written in the input upon entering the page:

<input value="text">

A user of our site can change the text of the input after visiting the page. Moreover, if we have a variable containing a reference to this input, then the propertyvaluethis variable will always contain the current value of the input text.

Let's try it in practice. Let us have an input with an attributevalue:

<input id="elem" value="text">

Let's get a link to this input into the variable:

let elem = document.querySelector('#elem');

Now let’s display the current input text:

console.log(elem.value);

Now let’s change the input text to something else:

elem.value = 'new text';
