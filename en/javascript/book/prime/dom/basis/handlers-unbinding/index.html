Unbinding Events in JavaScript

In this lesson we will learn how to unbind event handlers that we previously bound to elements. Let the following button be given as an example:

<input id="button" type="submit">

Let's bind a function to this buttonfunc:

let button = document.querySelector('#button'); button.addEventListener('click', func); function func() { console.log('!!!'); }

Let's now make the event handler fire on the first click and then unbind from the button. There is a special method for thisremoveEventListener. This method takes the event type as its first parameter, and the second parameter is a reference to the function that needs to be unbinded.

Typically this means that the event handler is unbound in the same way it was bound. That is, if we bind it like this:addEventListener('click', func), then we’ll untie it with the same parameters, like this:removeEventListener('click', func).

So, let's solve the problem we set:

let button = document.querySelector('#button'); button.addEventListener('click', func); function func() { console.log('!!!'); this.removeEventListener('click', func); }
