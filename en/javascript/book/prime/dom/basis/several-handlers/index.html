Multiple handlers for the same event in JavaScript

Several functions can be attached to one element at once. Let's look at an example. Let's say we have a button:

<input id="button" type="submit">

Let us have two functions:

function func1() { console.log('1'); } function func2() { console.log('2'); }

Let's get a link to our button in a variable:

let button = document.querySelector('#button');

Now let’s bind both the first and second functions to our button as click handlers:

button.addEventListener('click', func1); button.addEventListener('click', func2);

Given paragraph:

<p id="elem">text</p>

The following functions are given:

function func1() { console.log('1'); } function func2() { console.log('2'); } function func3() { console.log('3'); }

Link all these functions to our paragraph.
