Tag attributes as properties in JavaScript

Let's now learn how to get tag attributes. The following rule applies here:each tag attribute corresponds to the DOM element property of the same name.

Let's look at an example. Let's say we have this tag:

<input id="elem" type="text">

Let's get a link to our element in a variable:

let elem = document.querySelector('#elem');

Let's display the values ​​of the attributes we need:

console.log(elem.id); // выведет 'elem' console.log(elem.type); // выведет 'text'

Now, for example, let’s change the attribute value:

elem.type = 'submit';

<input id="elem" type="email">
