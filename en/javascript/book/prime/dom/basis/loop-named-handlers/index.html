Adding loop handlers in JavaScript

Let's now learn how to bulk add event handlers to elements. Let, for example, we have paragraphs:

<p>text1</p> <p>text2</p> <p>text3</p>

Let us also have a function:

function func() { console.log('!'); }

Let's loop through our paragraphs and add a function to each paragraph as a click handlerfunc:

let elems = document.querySelectorAll('p'); for (let elem of elems) { elem.addEventListener('click', func); }

Let's go further and make sure that when you click on any paragraph, the text of that paragraph is displayed. There is, however, a problem: there are many paragraphs, but only one handler function. How can we distinguish our paragraphs inside the handler function?

The object will help us with thisthis- when calling a function at the time of an event, this object will point to the element where this event occurred. Let's rework the code of our functionfuncin accordance with what has been said:

function func() { console.log(this.textContent); // выводим текст абзаца }

Given the following function:

function func() { this.value = Number(this.value) + 1; }

Inputs are also given. Make sure that when we lose focus in any of our inputs, the above function will be executed.

Paragraphs with numbers are given. Make it so that when you click on any paragraph, its number in it is squared.
