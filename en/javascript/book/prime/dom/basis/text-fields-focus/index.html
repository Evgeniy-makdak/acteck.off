Focus text fields in JavaScript

Let us have an input. You can click on this input and after that the wand cursor will blink in it and you can enter text into it.

About this state they say that the input now hasinput focus. In practice, this means that if you start entering text from the keyboard, this text will fall into the input that currently has the input focus. If you then click somewhere outside the input, then this inputwill loseinput focus and you will not be able to enter text into it.

In order to catch the moment an input gains or loses focus, JavaScript provides special events: eventfocusallows you to catch receiving focus by input, and the eventblur- loss. Let's try it in practice. Let us have an input:

<input id="elem" value="text">

Let's get a link to it in a variable:

let elem = document.querySelector('#elem');

Now let’s make sure that upon receiving focus, the current input text is displayed in the console:

elem.addEventListener('focus', function() { console.log(elem.value); });

Input given. When you receive focus, write a number in it1, and by loss of focus - the number2.

Input given. Let a number be entered into it. When you lose focus, display the square of this number.

An input is given, which initially contains some text. When an input receives focus, clear the contents of that input.
