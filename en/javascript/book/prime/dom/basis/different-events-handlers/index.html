Handlers for various events in JavaScript

In addition to clicking on an element, there are other events. For example, using the eventdblclickyou can catch a double click on an element using an eventmouseover- hovering the cursor over an element, and using an eventmouseout- the cursor leaves the element.

In this case, handlers of various types of events can be bound to one element. Let's, for example, bind the hover response and the leave response to one element:

button.addEventListener('mouseover', function() { console.log('1'); }); button.addEventListener('mouseout', function() { console.log('2'); });

A button is given. By double clicking on it, display a message.

A button is given. When you hover over it, display a message.

A button is given. When the cursor leaves it, print some message.

A button is given. When you hover over it, display one message, and when you leave it, display another.
