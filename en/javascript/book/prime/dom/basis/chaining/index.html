Method and Property Chains in JavaScript

Let us have an input:

<input id="elem" value="text">

Let's display the input text on the screen:

let elem = document.querySelector('#elem'); console.log(elem.value); // выведет 'text'

As you can see, we first get the element by itsid, write this element to a variableelem, and then display the propertyvaluefrom this variable.

In fact, you don't have to enter a variableelem, and build a chain of points in this way:

console.log( document.querySelector('#elem').value ); // выведет 'text'

In the same way - in a chain - you can rewrite attributes:

document.querySelector('#elem').value = 'www';

The following code is given:

<img id="image" src="avatar.png"> let image = document.querySelector('#image'); console.log(image.src);
