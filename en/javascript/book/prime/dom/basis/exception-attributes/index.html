Exceptions when working with attributes in JavaScript

There is an exception when working with attributes - this attributeclass. This word is special in JavaScript and therefore we cannot simply writeelem.classto read the attribute valueclass. Instead you should writeelem.className.

Let's look at an example. Let us be given the following input:

<input id="elem" class="aaa bbb">

Let's print the attribute valueclassfor our input:

let elem = document.querySelector('#elem'); console.log(elem.className); // выведет 'aaa bbb'
