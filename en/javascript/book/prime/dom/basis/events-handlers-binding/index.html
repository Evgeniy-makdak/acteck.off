Bind handlers to elements in JavaScript

Let's now teach our DOM elements to respond to site user actions. For example, the user will click somewhere with the mouse, and our code in response will have to process this click and display some information on the screen.

User actions that we can track through JavaScript are calledevents. Events can be the following: a mouse click on a page element, a mouse hover over a page element, or vice versa - the mouse cursor leaves the element, and so on. In addition, there are events that do not depend on user actions, for example, the event when an HTML page is loaded into the browser.

As an example, let’s make a button that, when clicked, will display some text on the screen. First, let's create the HTML code for the button:

<input id="button" type="submit">

Now we get a link to the button in a variable:

let button = document.querySelector('#button');

Now we need to set the reaction of our button when it is clicked. There is a special method for this in JavaScript.addEventListener, the first parameter taking the name of the event (a click on the button has a name'click'), and the second parameter iscallback function, executed when this event occurs.

Let's, for example, by clicking on a button, display some text:

button.addEventListener('click', function() { console.log('!!!'); });
