Unbinding event handlers in a loop in JavaScript

Let us now have not one element, but several. For example, a few paragraphs:

<p>text1</p> <p>text2</p> <p>text3</p>

Let's bind a function to each of these paragraphs as a click handlerfunc:

let elems = document.querySelectorAll('p'); for (let elem of elems) { elem.addEventListener('click', func); } function func() { console.log(this.textContent); }

Let's now rework the code so that each paragraph only responds to the first click on it. To do this, when you click on a paragraph, we will untie the attached handler from it. In this case, the unbinding will be specifically from this paragraph, without affecting the rest in any way.

As you already know, the element in which the event occurred can be obtained in the handler function viathis. This means that you need to unbind the handler fromthis, like this:

let elems = document.querySelectorAll('p'); for (let elem of elems) { elem.addEventListener('click', func); } function func() { console.log(this.textContent); this.removeEventListener('click', func); // отвязываем обработчик }

Paragraphs are given. When you click on any paragraph, add an exclamation point to the end. Make sure that this addition occurs only on the first click.
