Working with HTML code of elements in JavaScript

DOM elements have a propertyinnerHTML, allowing you to read the HTML code of these elements. Let's look at an example. Let's say we have the following tag:

<p id="elem"><b>text</b></p>

Let's get a link to this tag into a variable:

let elem = document.querySelector('#elem');

Let's read the HTML code of the tag:

console.log(elem.innerHTML); // выведет <b>text</b>

Let's change the tag text:

elem.innerHTML = '<i>!!!</i>';
