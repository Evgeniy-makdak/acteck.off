Getting a group of elements in JavaScript

In previous lessons we used the methodquerySelectorreceived one page element. Now it's time to learn how to receive a group of elements and perform some operations with many elements at once.

There is a method for thisquerySelectorAll, which gets all tags that match the CSS selector as an array of elements. To do something with the found elements, you need to work with the resulting array, for example, iterate through it in a loop and perform some operation with each element separately in the loop.

Let, for example, we have paragraphs with the classwww:

<p class="www">text1</p> <p class="www">text2</p> <p class="www">text3</p>

Let's get an array of these paragraphs, go through them in a loop and in a loop we will output the texts of the found paragraphs to the console:

let elems = document.querySelectorAll('.www'); for (let elem of elems) { console.log(elem.textContent); }

Now let's add an exclamation point to the end of the text of each paragraph:

let elems = document.querySelectorAll('.www'); for (let elem of elems) { elem.textContent = elem.textContent + '!'; }

Paragraphs and a button are given. By clicking the button, find all the paragraphs, loop through them and set the text of each paragraph to the value'text'.

Paragraphs with text and a button are given. By clicking the button, write down its serial number at the end of the text of each paragraph.

Inputs with numbers, a paragraph and a button are given. By clicking on the button, find the sum of the numbers from the inputs and write this sum in the text of the paragraph.
