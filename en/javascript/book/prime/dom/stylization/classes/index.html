Styling with CSS classes in JavaScript

In the previous lesson, we learned how to change the CSS styles of elements by changing the style attribute. Most often this is not a good idea. The fact is that if you scatter CSS styles throughout the JavaScript code, it will be problematic to change the site design in the future, since you will have to tinker with the JavaScript code in search of the styles embedded there.

It would be more convenient for further support to add or remove CSS classes to the element, thereby styling them as needed.

Let's look at an example. Let's say we have several paragraphs:

<p>text1</p> <p>text2</p> <p>text3</p>

Let's make sure that when you click on any paragraph, this paragraph will be painted in some color, for example, green. To do this, in the CSS file we will create a special class for coloring paragraphs:

.colored { color: green; }

The advantage of using the class is that now you can easily change the desired color - to do this you just need to make a change to the CSS file without messing with the JavaScript code. This will be especially convenient if you write the JavaScript code, and in the future it will be styled by someone else (perhaps a less qualified person who only knows how to work with CSS).

So, now, after introducing the class, by clicking on any paragraph you can change its color by simply adding our class to it:

let elems = document.querySelectorAll('p'); for (let elem of elems) { elem.addEventListener('click', function() { this.classList.add('colored'); // добавляем абзацу класс }); }

Explain why I chose the wordcolored, not a wordgreen, clearly indicating the color we want.
