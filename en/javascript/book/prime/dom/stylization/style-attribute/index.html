Styling elements using the style attribute in JavaScript

Let's now learn how to add CSS styles to elements. This is done by changing the attributestyle. For example, to change the color, you need to build the following chain:elem.style.color, and assign it the desired color value.

Let's look at an example. Let us have this element:

<p id="elem">text</p>

Let's make this element red:

let elem = document.querySelector('#elem'); elem.style.color = 'red';
