Finding elements inside another element in JavaScript

Let us have some parent element, and inside it there are other elements:

<div id="parent"> <div class="child">text</div> <div class="child">text</div> <div class="child">text</div> </div>

Let the reference to the parent have already been received in a variable:

let parent = document.querySelector('#parent');

In this case, if necessary, we can search for the selector within this parent, rather than throughout the entire document. To do this, the search method must be applied not todocument, but to the variable containing our parent.

For example, let's find elements with the class inside our parentchild:

let elems = parent.querySelectorAll('.child');

Given element#parent:

<div id="parent"> <p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p class="ggg">text</p> <p class="ggg">text</p> <p class="ggg">text</p> </div> let parent = document.querySelector('#parent');

Find elements with class inside the parentwwwand write them to a variableelems1. Then look inside the parent for elements with the classgggand write them to a variableelems2.
