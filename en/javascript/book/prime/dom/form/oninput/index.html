JavaScript input event

The next event we will study is calledinput. It occurs every time a new character is entered into the input ortextarea. Let's look at an example. Let us have an input:

<input id="elem">

Let's output the current contents of the input to the console every time we press a button on the keyboard:

let elem = document.querySelector('#elem'); elem.addEventListener('input', function() { console.log(this.value); });
