Working with Radio Buttons in JavaScript

Let's now work withradio buttons. They are a group of radio buttons from which only one can be selected. For multiple radio buttons to be a group, they must have the same attribute valuename:

<input type="radio" name="elem"> <input type="radio" name="elem"> <input type="radio" name="elem">

And to distinguish in JavaScript which button was selected, value attributes with a different meaning are added to each radio button in the group:

<input type="radio" name="elem" value="1"> <input type="radio" name="elem" value="2"> <input type="radio" name="elem" value="3">

To make a radio button marked by default, you need to set the attributechecked:

<input type="radio" name="elem" value="1" checked> <input type="radio" name="elem" value="2"> <input type="radio" name="elem" value="3">

Let's see in practice how to work with such buttons in JavaScript. Let's say we have the group of radio buttons presented above, as well as a regular button:

<input type="submit" id="button">

Let's make it so that when you press a button, the screen displaysvaluethe radio button that is currently marked.

To solve the problem, you need to cycle through all our radio buttons and determine which one is selected. To do this you need to read the property valuecheckedeach of the radio buttons being selected. Which has this property equaltrue- that radio button is selected.

Let's implement the described:

let radios = document.querySelectorAll('input[type="radio"]'); let button = document.querySelector('#button'); button.addEventListener('click', function() { for (let radio of radios) { if (radio.checked) { console.log(radio.value); } } });
