Working with textarea in JavaScript

In this section of the tutorial, we'll explore working with more complex shape capabilities. Let's start studying with the tagtextarea, which is a multi-line input field.

There is some nuance when working with this tag. The bottom line is this: the text that will appear in the tag by default is located between the opening and closing tagstextarea:

<textarea id="elem">text</textarea>

:

However, despite this, the tag text should not be read and written through the propertyinnerHTML, and throughvalue. For example, let's display the text abovetextarea:

let elem = document.querySelector('#elem'); console.log(elem.value); // выведет 'text'
