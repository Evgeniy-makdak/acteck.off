Getting dropdown list items in JavaScript

Let us be given some drop-down list:

<select id="select"> <option value="one">один</option> <option value="two">два</option> <option value="three">три</option> </select>

Question: how do we get all the items on this list? The easiest way is to use the appropriate selector:

let options = document.querySelectorAll('#select option'); console.log(options);

You can not search throughout the entire document, but specifically in our select:

let select = document.querySelector('#select'); let options = select.querySelectorAll('option'); console.log(options);
