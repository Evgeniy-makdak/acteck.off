Working with Dropdown Lists in JavaScript

In this lesson we will learn how to work with the tagselect, which is a drop-down list. If you do not know how to work with this tag in HTML, then first study its structure in the reference book at this link:select.

Let's see how to work with this tag through JavaScript. Let us have a drop-down list like this:

<select id="select"> <option>one</option> <option selected>two</option> <option>three</option> </select>

Let's get a link to the select into the variable:

let select = document.querySelector('#select');

Let's now display the text of the selected list item when changing the select. To do this, you need to read the value property of our select:

select.addEventListener('change', function(){ console.log(this.value); });

Given a select, a paragraph and a button. By clicking on the button, display the text of the selected list item in a paragraph.

Make a dropdown list of years from2020to2030. When you select an item in the list, display a message indicating whether this year is a leap year or not.
