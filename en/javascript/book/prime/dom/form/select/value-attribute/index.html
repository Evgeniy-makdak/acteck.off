The value attribute in dropdown lists in JavaScript

As a rule, the attribute is added to list itemsvalue. In this case, the value property of the selector will not contain textoption, and the value of its attributevalue.

Let's redo our select:

<select id="select"> <option value="1">one</option> <option value="2" selected>two</option> <option value="3">three</option> </select>

Let's check what the selected value is now:

select.addEventListener('change', function(){ console.log(this.value); });
