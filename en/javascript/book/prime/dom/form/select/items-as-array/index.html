Array of dropdown list items in JavaScript

When working with selects, there is the following feature: the variable in which the link to the select is stored is an array, the elements of which are tagsoption. This means that this variable can be looped through:

let select = document.querySelector('#select'); for (let option of select) { console.log(option); }

Well, or just go to the item with the desired number:

let select = document.querySelector('#select'); console.log(select[0]);

Loop through all the list items and add the value of its attribute to the end of the text of each itemvalue.
