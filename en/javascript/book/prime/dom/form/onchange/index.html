Change event in JavaScript

In this lesson you will analyze the eventchange, which appears in input fields when they are changed. What does it mean? Let, for example, you have an input and there is some text in it. If you change this text, then this event will occur.

Let's look at an example. Let us have an input:

<input id="elem" value="text">

Let's change it and display its new value in the console:

let elem = document.querySelector('#elem'); elem.addEventListener('change', function() { console.log(this.value); });
