Working with checkboxes in JavaScript

Now we will learn how to work withcheckboxes. A checkbox is a special checkmark that can be in two states: checked and unchecked.

The checkbox is created as follows:

<input type="checkbox" id="elem">

To make a checkbox checked, you need to write an attribute to itchecked:

<input type="checkbox" checked id="elem">

Let's find out the state of our checkbox as an example:

let elem = document.querySelector('#elem'); console.log(elem.checked);

There is a checkbox and two buttons. By clicking on the first button, set the checkbox to the checked state, and by clicking on the second - to the unchecked state.

Given a checkbox, a button and a paragraph. By clicking the button, if the checkbox is checked, display the word in a paragraph'привет', and if the checkbox is not checked, then the word'пока'.
