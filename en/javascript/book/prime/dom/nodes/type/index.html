Type of nodes in JavaScript

All nodes have the propertynodeType. Its value is numeric:1for elements,3for text nodes,8for comments. The remaining values ​​are currently either little used or outdated.

Dan div:

<div id="elem">txt<b>tag</b><!--com-->txt<b>tag</b><!--com--></div>

Loop through all the nodes of this div and output to the console only element nodes and text nodes.
