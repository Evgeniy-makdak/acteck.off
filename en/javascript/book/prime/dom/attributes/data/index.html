Custom Attributes in JavaScript

HTML allows you to add your own custom attributes to tags. Such attributes must begin withdata-, and then should come any attribute name that is convenient for you.

Custom attributes can be used in a huge number of different ways. We will study many of these methods further in the textbook, and you can invent even more on your own in the future.

Access to such attributes is not arranged in a standard way. You can’t just access the element’s property of the same name, as we did before, but you should use a special propertydataset, after which the attribute name is written through a dot withoutdata-. For example, if our attribute is calleddata-test, then to refer to it we will writeelem.dataset.test, Whereelem- a variable with our element.

Let's look at an example. Let us be given the following element:

<div id="elem" data-num="1000"></div>

Let's display the value of its attributedata-num:

let elem = document.querySelector('#elem'); console.log(elem.dataset.num); // выведет 1000

Now let's assign another value to this attribute:

let elem = document.querySelector('#elem'); elem.dataset.num = 123;

The following code is given:

<div id="elem" data-text="str">text</div>

Make sure that when you click on a div, the contents of its attribute are added to the end of its textdata-text.

Given divs containing in the attributedata-numyour serial number:

<div data-num="1">text</div> <div data-num="2">text</div> <div data-num="3">text</div> <div data-num="4">text</div> <div data-num="5">text</div>

Make sure that when you click on any of the divs, its serial number is written at the end.

A button is given. Make this button count the number of clicks on it, recording them in some custom attribute. Suppose that when you click on another button, the screen displays how many clicks were made on the first button.

Given input:

<input id="elem" data-length="5">

In this input in the attributedata-lengthcontains the number of characters that need to be entered in the input. Make sure that when focus is lost, if the number of characters entered does not match the specified number, a message about this is displayed.

Given input:

<input id="elem" data-min="5" data-max="10">

In this input the attributesdata-minAnddata-maxcontain a range. Make sure that when focus is lost, if the number of characters entered does not fall within this range, a message about this is displayed.
