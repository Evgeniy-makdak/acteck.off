Activating elements in JavaScript

Let's say we have some HTML table#tablewith empty cells.

Let's make sure that when you click on any cell, it somehowactivated, for example, became a red background. To do this, we will give the activated cells some CSS class:

.active { background: red; }

Let's implement the activation:

let tds = document.querySelectorAll('#table td'); for (let td of tds) { td.addEventListener('click', function() { this.classList.add('active'); }); }
