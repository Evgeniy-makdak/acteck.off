Changing HTML Table Cells in JavaScript

Let some HTML table be given#table. Let us be faced with the task of doing something with each cell of the table, for example, writing text into each of them'!'.

Let's discuss the nuances of solving such a problem.

As you know, an HTML table has a two-dimensional structure: there are rows and cells in them. We can solve our problem as follows: loop through the rows and in each row loop through the cells and do the action we need with them. That is, the given solution scheme is similar to how we would create such a table, filling it with rows and cells.

However, in this case, two nested loops will be redundant: you can simply get everythingtdand go through them in a loop, performing the desired operation. Let's do this:

let tds = document.querySelectorAll('#table td'); for (let td of tds) { td.textContent = '!'; }
