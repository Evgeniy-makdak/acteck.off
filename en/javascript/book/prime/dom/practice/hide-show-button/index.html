Buttons to hide and show an element in JavaScript

In this section, we will learn how to hide and show page elements. As usual, let's start with something simple and gradually increase complexity.

Let us have a paragraph and two buttons for warm-up:

<p id="elem">text</p> <input type="submit" id="show" value="show"> <input type="submit" id="hide" value="hide">

Let's get references to our elements in variables:

let elem = document.querySelector('#elem'); let show = document.querySelector('#show'); let hide = document.querySelector('#hide');

Let's now make sure that by clicking on one button our paragraph is hidden, and on the other it is shown. To do this, we will give or remove the corresponding CSS class to the element:

.hidden { display: none; }

Let's solve our problem:

hide.addEventListener('click', function() { elem.classList.add('hidden'); }); show.addEventListener('click', function() { elem.classList.remove('hidden'); });
