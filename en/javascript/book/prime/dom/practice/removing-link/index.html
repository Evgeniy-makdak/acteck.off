Link to remove an element in JavaScript

Let us have a paragraph. Let's make a link next to it so that this paragraph can be deleted.

We implement:

<div id="parent"> <p id="elem">text</p> <a href="#" id="remove">remove</a> </div> let elem = document.querySelector('#elem'); let remove = document.querySelector('#remove'); remove.addEventListener('click', function() { elem.remove(); });

Please note that the href attribute of the link contains #. If we remove this grid, we will get a link click and, as a result, a page update.

In fact, the paragraph will also be deleted, but we won’t notice it, since the page will be refreshed and everything will return to its original position.

To solve the problem, you need to prevent clicking on the link usingpreventDefault:

<div id="parent"> <p id="elem">text</p> <a href="" id="remove">remove</a> </div> let elem = document.querySelector('#elem'); let remove = document.querySelector('#remove'); remove.addEventListener('click', function(event) { elem.remove(); event.preventDefault(); // отменяем переход по ссылке });
