Styling Elements in JavaScript

Let's say we have several paragraphs:

<p>text1</p> <p>text2</p> <p>text3</p>

Let's go through these paragraphs in a loop and add a link to the end of each paragraph; when clicked, some stylization will be added to the text of the paragraph. For example, the text of a paragraph will become strikethrough (this is done by the CSS propertytext-decoration).

Let's discuss two nuances of such a task.

Firstly, as already discussed in previous lessons, you should not change paragraph styles directly through JavaScript - it will be much more convenient to add some CSS classes.

Secondly, when implementing such a task, some surprise awaits you. To understand its essence, let's look at the HTML code of the paragraphs after adding links:

<p>text1<a href="">link</a></p> <p>text2<a href="">link</a></p> <p>text3<a href="">link</a></p>

Now imagine that when you click on the link, the text of the paragraph will be crossed out. However, the link in this case is also part of the paragraph and will also be crossed out! Most likely we do not need such an effect. We want the text to strike out, but the link doesn't.

To solve the problem, you just need to wrap the paragraph text in a tagspan, like this:

<p><span>text1</span><a href="">link</a></p> <p><span>text2</span><a href="">link</a></p> <p><span>text3</span><a href="">link</a></p>

Given the following HTML code:

<p>text1</p> <p>text2</p> <p>text3</p>

Add a link to the end of each paragraph; when clicked, the text of the paragraph will be crossed out (but the link will not).

Modify the previous problem so that when you click on a link, the link is removed from the paragraph (and the paragraph text becomes strikethrough).

Given some HTML table. Add another column with a link to this table. When you click on this link, the row with this link should have a green background.

Modify the previous task so that the first click on the link paints the row with a green background, and the second click cancels this action.
