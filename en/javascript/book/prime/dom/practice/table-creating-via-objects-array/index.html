Creating an HTML table from an array of objects

Let us be given a certain array of objects, for example, this array with users:

let users = [ { name: 'name1', surname: 'surname1', patronymic: 'patronymic1' }, { name: 'name2', surname: 'surname2', patronymic: 'patronymic2' }, { name: 'name3', surname: 'surname3', patronymic: 'patronymic3' }, ];

Let's make an HTML table from this array, in each row of which we will record the data of an individual user.

With this data storage structure, they usually run one loop through the array and create each table cell manually, like this:

<table id="table"></table> let table = document.getElementById('table'); for (let user of users) { let tr = document.createElement('tr'); let td1 = document.createElement('td'); td1.textContent = user.name; tr.appendChild(td1); let td2 = document.createElement('td'); td2.textContent = user.surname; tr.appendChild(td2); let td3 = document.createElement('td'); td3.textContent = user.patronymic; tr.appendChild(td3); table.appendChild(tr); }

This approach gives greater flexibility - we can ourselves regulate the order of data arrangement in the table cells (we can, for example, swap the first and last names).

In addition, if desired, we can assign events to certain cells. For example, you can attach some kind of click action to a cell with a surname, and so on.
