Simultaneous editing and deleting of elements

Let us have a certain set of paragraphs:

<div id="parent"> <p>text1</p> <p>text2</p> <p>text3</p> </div>

For these paragraphs, let's make it so that you can edit the text of each paragraph and at the end of each paragraph there is a link to delete.

When implementing the described task, we are faced with some problem.

In order to understand the essence of the problem, let's look at the HTML code that will be produced when delete links are added to the end of each paragraph:

<div id="parent"> <p>text1<a href="">remove</a></p> <p>text2<a href="">remove</a></p> <p>text3<a href="">remove</a></p> </div>

Now imagine that by clicking on any paragraph, an input for editing text will appear in it. In this case, the entire text of the paragraph will be included in the input - along with a link to delete!

This is, of course, not correct.

A better solution would be to wrap paragraph texts in tagsspan, like this:

<div id="parent"> <p><span>text1</span><a href="">remove</a></p> <p><span>text2</span><a href="">remove</a></p> <p><span>text3</span><a href="">remove</a></p> </div>

For such code, you can simply attach an editing event not to the paragraph itself, but tospanwith text. In this case, the input for editing will appear in the tagspan, and our delete link will remain intact.

Given the following HTML code:

<div id="parent"> <p><span>text1</span></p> <p><span>text2</span></p> <p><span>text3</span></p> </div>

Add a delete link to the end of each paragraph.

Make it so that by clicking onspanan input for editing appeared in it.

Let now initially tagsspanNo:

<div id="parent"> <p>text1</p> <p>text2</p> <p>text3</p> </div>

Wrap paragraph text in tags firstspan, add editability to these tags, and then add a delete link at the end of each paragraph.
