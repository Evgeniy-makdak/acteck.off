Editing a Single Element in JavaScript

Now we will learn how to edit element texts using input fields. Let's start with simple things and gradually complicate them.

So, let's say we have a paragraph and an input located in the same parent:

<div id="parent"> <p id="elem">text</p> <input id="input"> </div>

Let's make sure that when the input loses focus, its text appears in the paragraph:

let elem = document.querySelector('#elem'); let input = document.querySelector('#input'); input.addEventListener('blur', function() { elem.textContent = this.value; });

Let's now make sure that when you enter the page, the input will already contain the text of the paragraph. Thus, using the input, we can edit the text located in the paragraph.

We implement:

let elem = document.querySelector('#elem'); let input = document.querySelector('#input'); input.value = elem.textContent; // записываем в инпут текст абзаца input.addEventListener('blur', function() { elem.textContent = this.value; });

Modify the above code so that the paragraph text changes not when focus is lost, but as text is entered into the input.

Appearance of input

Let's now make sure that the input is not initially on the page, but appears when you click on the paragraph. That is, our initial HTML will look like this:

<div id="parent"> <p id="elem">text</p> </div>

To begin with, we simply implement the appearance of the input, without editing:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { let input = document.createElement('input'); input.value = elem.textContent; elem.parentElement.appendChild(input); });

Now let’s make sure that when the input loses focus, the text of the paragraph changes:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { let input = document.createElement('input'); input.value = elem.textContent; input.addEventListener('blur', function() { elem.textContent = this.value; }); elem.parentElement.appendChild(input); });

Our code, however, is imperfect, since each click on a paragraph will result in a new input appearing.

To solve the problem, we will simply delete the current input when focus is lost:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { let input = document.createElement('input'); input.value = elem.textContent; input.addEventListener('blur', function() { elem.textContent = this.value; this.remove(); // удалим инпут }); elem.parentElement.appendChild(input); });

Solve the described problem yourself, without looking at my code.
