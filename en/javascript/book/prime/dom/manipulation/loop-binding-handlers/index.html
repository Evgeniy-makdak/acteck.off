Hang handlers in a loop in JavaScript

In the previous lesson we learned how to create new elements in a loop. Let's now attach event handlers when creating new elements.

Let us again have this div parent:

<div id="parent"></div>

Let's run a loop that will add to the end of our div9new paragraphs by attaching a click handler to them:

let parent = document.querySelector('#parent'); for (let i = 1; i <= 9; i++) { let p = document.createElement('p'); p.textContent = '!'; // Навешиваем обработчик клика: p.addEventListener('click', function() { console.log(this.textContent); }); parent.appendChild(p); }
