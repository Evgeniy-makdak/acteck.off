Bind events when inserting elements

Let's now bind event handlers to them when inserting elements.

Let, for example, we have a parent element like this:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> </div>

Let's add another paragraph to the end of this parent, giving it a click handler:

let parent = document.querySelector('#parent'); let p = document.createElement('p'); p.textContent = '!'; p.addEventListener('click', function() { console.log(this.textContent); // по клику выведем текст абзаца }); parent.appendChild(p);
