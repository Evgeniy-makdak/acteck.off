Looping elements in JavaScript

In previous lessons we created one new element at a time. Let's now make sure that new elements are created in a loop.

For example, let's say we have a parent div like this:

<div id="parent"></div>

Let's run a loop that will append to the end of our div9new paragraphs:

let parent = document.querySelector('#parent'); for (let i = 1; i <= 9; i++) { let p = document.createElement('p'); p.textContent = '!'; parent.appendChild(p); }
