Creating and inserting elements in JavaScript

Now we will learn how to create new DOM elements using JavaScript and then add them to the page. The method is designed for thiscreateElement. The parameter of this method should be the name of the tag that is to be created.

If you write down the result of the workcreateElementinto a variable, then this variable will contain an element as if we had received it throughquerySelector.

The only difference is that our element will not be placed on the page. And this way we can change its text, attributes, attach event handlers, and ultimately place it on the page.

To place a new element on the page, use the methodappendChild. This method should be applied to the elementin whichwe want to place our element. And the method parameter should be passed to our new element, created earlier viacreateElement.

Let's look at a practical example. Let us have a div and several paragraphs in it:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> </div>

Let's create a paragraph, set its text and place it on the page at the end of the block#parent:

let parent = document.querySelector('#parent'); let p = document.createElement('p'); p.textContent = '!'; parent.appendChild(p);

Result of executing the code:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> <p>!</p> </div>
