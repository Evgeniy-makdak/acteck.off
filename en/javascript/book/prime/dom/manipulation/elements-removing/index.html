Removing elements in JavaScript

Let's now learn how to remove elements. To do this you need to use the methodremove. Let's look at an example of how this is done. Let us have paragraphs:

<p>elem 1</p> <p>elem 2</p> <p>elem 3</p> <p>elem 4</p> <p>elem 5</p>

Let's make sure that any paragraph can be deleted by clicking on it:

let elems = document.querySelectorAll('p'); for (let elem of elems) { elem.addEventListener('click', function() { elem.remove(); }); }

The following code is given:

<ul> <li>1</li> <li>2</li> <li>3</li> </ul>

Make sure that anyliwas deleted by clicking on it.

The following code is given:

<ul id="parent"> <li>1</li> <li>2</li> <li>3</li> </ul> <input type="submit" id="button">

Make sure that every click on the button removes the last element from#parent.
