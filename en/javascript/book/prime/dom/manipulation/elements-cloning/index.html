Cloning elements in JavaScript

In this lesson we will learn how to get copies of elements. You can work with these copies as with ordinary elements - change them and insert them into the desired place on the page. The process of obtaining copies of elements is calledcloning.

You can clone an element using the methodcloneNode. This method must be passed true or false as a parameter. If true is passed, then the element is cloned completely, along with all attributes and child elements, and if false, only the element itself is cloned.

Let's look at an example. Let's say we have this code:

<div id="parent"> <div class="elem"> <p>первый абзац</p> <p>второй абзац</p> </div> </div>

Let's make a copy of the block with the classelemand insert it at the end of the block#parent:

let parent = document.querySelector('#parent'); let elem = parent.querySelector('.elem'); let clone = elem.cloneNode(true); parent.appendChild(clone);

The result will be the following:

<div id="parent"> <div class="elem"> <p>первый абзац</p> <p>второй абзац</p> </div> <div class="elem"> <p>первый абзац</p> <p>второй абзац</p> </div> </div>

Input given. A button is given. By clicking the button, clone this input.
