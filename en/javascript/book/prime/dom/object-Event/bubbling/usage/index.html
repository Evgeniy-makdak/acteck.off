Using event bubbling termination in JavaScript

Let us have a button and some block inside one parent:

<div id="parent"> <button>click me</button> <div id="block"> text </div> </div>

Let's get references to our elements in variables:

let parent = document.querySelector('#parent'); let button = document.querySelector('button'); let block = document.querySelector('#block');

Let our block be initially hidden:

#block:not(.active) { display: none; }

Let's make our block appear when you click on the button:

button.addEventListener('click', function() { block.classList.add('active'); });

Now let’s make sure that when you click on any place in the parent, our block will disappear:

parent.addEventListener('click', function() { block.classList.remove('active'); });

However, we are in for an unexpected surprise: since the button is located inside the parent, thenClicking on the button also means clicking on the parent. This means that first our block will appear, and then, due to the event bubbling, the handler in the parent will fire and our block will disappear.

This is where the ability to cancel bubbling comes in handy: we can make it so that when the cancel bubbling button is clicked, the parent does not react to this click.
