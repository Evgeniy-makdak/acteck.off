Stopping event bubbling in JavaScript

Event bubbling can be stopped on any element through which the event is bubbling. To do this, you should call the method in the element codestopPropagationobjectEvent.

In the following example, clicking on the red block will work on itself, then on the blue block and that’s it - the blue block stops further popping up and the green block will not react in any way:

elem1.addEventListener('click', function() { console.log('зеленый'); }); elem2.addEventListener('click', function(event) { console.log('голубой'); event.stopPropagation(); // остановим всплытие }); elem3.addEventListener('click', function() { console.log('красный'); });
