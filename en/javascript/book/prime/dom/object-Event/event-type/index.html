Event type in an Event object in JavaScript

ObjectEventalso contains the type of event that occurred. Type refers to the nameclick, mouseoverand so on. The event type contains a propertytype:

elem.addEventListener('click', function(event) { console.log(event.type); // выведет 'click' });

The following code has the same handler attached to two events:

<button id="elem">text</button> let elem = document.querySelector('#elem'); elem.addEventListener('click', func); elem.addEventListener('dblclick', func); function func() { }

Add the function codefuncso that when you click on an element, this element turns green, and when you double-click, it turns red.
