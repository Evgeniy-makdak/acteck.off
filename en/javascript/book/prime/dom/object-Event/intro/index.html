Basics of Working with the Event Object in JavaScript

In this lesson we will learn the basics of working with an objectEvent. This object contains information about the event that occurred. For example, if an element was clicked, we can find out the coordinates of this click, whether a key was pressedCtrl, AltorShiftat the moment of clicking and so on.

Let's see how to get an objectEvent. Let's say we have a button:

<button id="elem">text</button>

Let a certain function be performed by clicking on this button:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { });

There is already an object inside the bound functionEvent- we just don’t know how to get it yet. It turns out as follows: when declaring our function, we need to pass any variable into it as a parameter (as a rule,event- but the name can be anything) and into this variablethe browser will automatically place the objectEvent:

elem.addEventListener('click', function(event) { console.log(event); // увидим объект с событием });

Perform the described manipulations yourself and output the object with the event to the console. Study the structure of this object.
