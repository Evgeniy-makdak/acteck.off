Overriding a default action in JavaScript

Sometimes using JavaScript you want to override the default action of a tag. For example, by clicking on a link, you can cancel following that link. As a rule, this is required if we simply use this link to run some code. In this case, we don’t really need the fact that when you click on a link it will be followed.

Overriding the default action is done using the objectEvent. He has a special method for thispreventDefault(), which should be called anywhere in the event handler.

Let's try it in practice. Let us have this link:

<a href="/" id="elem">ссылка</a>

Let's make sure that clicking on a link does not lead to another page:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function(event) { event.preventDefault(); console.log('Вы не можете перейти по этой ссылке!'); });

Links are provided. Make sure that when you click on a link, its href is written at the end, and the link is not followed.

Two inputs are given, a paragraph and a link. Let numbers be entered into the inputs. Make sure that when you click on the link, the sum of the entered numbers is written into the paragraph.
