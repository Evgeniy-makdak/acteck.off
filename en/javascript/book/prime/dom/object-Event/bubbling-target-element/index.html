Getting the target element when bubbling events

Let's say we have two elements:divand paragraphp, lying inside this div:

<div> <p></p> </div>

Let's get a link to our div into a variable:

let div = document.querySelector('div');

Let's add some styles to our tags:

div { padding: 20px; border: 1px solid red; } p { width: 200px; height: 200px; border: 1px solid green; }

Let's now have a click handler attached to the div:

div.addEventListener('click', function() { console.log('click'); });

Due to the fact that our div haspadding, then when we click on a div, we can go to a paragraph, or we can go to a place where this paragraph does not exist, that is, directly to the div.

At the same time, in the click handlerthiswill always point to the element on which the handler is attached. In our case, this is our div:

div.addEventListener('click', function() { console.log(this); // див });

However, we can get exactly the tag in which the event occurred. To do this, we can see what lies inevent.target:

div.addEventListener('click', function(event) { console.log(event.target); // или див, или абзац });

You can distinguish between these two options using conditions:

let div = document.querySelector('div'); div.addEventListener('click', function(event) { if (event.target.tagName === 'DIV') { console.log('клик именно по диву'); } if (event.target.tagName === 'P') { console.log('клик именно по абзацу'); } });

Maybe insteadtagNameusematches:

let div = document.querySelector('div'); div.addEventListener('click', function(event) { if (event.target.matches('div')) { console.log('клик именно по диву'); } if (event.target.matches('p')) { console.log('клик именно по абзацу'); } });

The following elements are given:

<div> <ul> <li>1</li> <li>2</li> <li>3</li> </ul> </div> div, ul, li { padding: 20px; } div { border: 1px solid red; } ul { border: 1px solid orange; } li { border: 1px solid green; }

Attach a click handler to the div. In this handler, determine which tag triggered the event.
