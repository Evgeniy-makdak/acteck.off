Universal Event Delegation in JavaScript

The code given in the previous lesson is working, however, not without its shortcomings. Let's look at these shortcomings and write a more universal solution.

The flaw in our code will become apparent when insideliThere will be some nested tags. In our case, let these be tagsi:

<ul> <li>item <i>italic</i> item</li> <li>item <i>italic</i> item</li> <li>item <i>italic</i> item</li> <li>item <i>italic</i> item</li> <li>item <i>italic</i> item</li> </ul>

In this case, clicking on the tagiwill add an exclamation point toend of tag i, not a taglias we would like - after all, inevent.targetthe exact tag in which the event occurred is hit.

You can solve the problem using the methodclosest:

list.addEventListener('click', function(event) { let li = event.target.closest('li'); if (li) { li.innerHTML = li.innerHTML + '!'; } });
