Bubbling Events in JavaScript

Imagine that you have several blocks nested inside each other:

<div id="elem1"> <div id="elem2"> <div id="elem3"></div> </div> </div>

Let's get links to our divs into variables:

let elem1 = document.querySelector('#elem1'); let elem2 = document.querySelector('#elem2'); let elem3 = document.querySelector('#elem3');

Now let’s attach click handlers to them:

elem1.addEventListener('click', function() { alert('зеленый'); }); elem2.addEventListener('click', function() { alert('голубой'); }); elem3.addEventListener('click', function() { alert('красный'); });

Let's display our blocks on the screen by adding some CSS to them:

Now click on the innermost red block - and you will see how the event will first fire in the red block, then in the blue, then in the green. And this is logical, because when you click on the internal block, you simultaneously click on all external ones.

That is, it turns out that when you click on the innermost block, the click event occurs first in it, then fires in its parent, in the parent of its parent, and so on until it reaches the very top.

This behavior is calledby surfacing (bubbling) events - by analogy with the ascent of an air bubble from the bottom. Just like the bubble, our event seems to float to the top, each time triggering on higher blocks.

Write your own code that will demonstrate the bubbling of events. Check for different types of events to pop up on it.

Not all events may pop up. Experimentally find at least one event that will not pop up.
