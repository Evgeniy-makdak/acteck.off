Attaching handlers to new elements in JavaScript

Let us have a listuland button:

<ul> <li>item</li> <li>item</li> <li>item</li> <li>item</li> <li>item</li> </ul> <button>add</button>

Let's get our elements into the corresponding variables:

let button = document.querySelector('button'); let list = document.querySelector('ul'); let items = list.querySelectorAll('li');

Let's make it so that by clicking on anylian exclamation point was added to the end:

for (let item of items) { item.addEventListener('click', function() { this.textContent = this.textContent + '!'; }); }

Let's now make sure that when you click a button, a new one is added to the end of the listli:

button.addEventListener('click', function() { let item = document.createElement('li'); item.textContent = 'item'; list.appendChild(item); });

We will, however, get a problem: clicking on the new one addedliwill not cause an exclamation point to be added to the end. The point is that we add a click handler only to thoseli, which existed initially, butnot for new ones.

Let's fix the problem by assigning a new oneliclick handler:

button.addEventListener('click', function() { let item = document.createElement('li'); item.textContent = 'item'; item.addEventListener('click', function() { // обработчик клика this.textContent = this.textContent + '!'; }); list.appendChild(item); });

However, now the code of the handler function is duplicated in two places - for the initially existingliand for new ones. Let's fix this by moving it into a separate function:

function handler() { this.textContent = this.textContent + '!'; }

Let's use our function to avoid code duplication:

for (let item of items) { item.addEventListener('click', handler); } button.addEventListener('click', function() { let item = document.createElement('li'); item.textContent = 'item'; item.addEventListener('click', handler); list.appendChild(item); });

The problem is generally solved and we have avoided duplicating the code of the handler function. However, we still have to attach event handlers in two places: and in the loop for existingli, and in when you click on the button. In the next lesson we will look at a way to get rid of this inconvenience.
