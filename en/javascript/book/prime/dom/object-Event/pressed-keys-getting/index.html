Getting pressed keys in JavaScript

Using an objectEventyou can receive the keys pressed. Let's see how it's done. Let us have an input:

<input>

Let's get a link to it in a variable:

let elem = document.querySelector('input');

Let's attach an event to our input that fires on each key press:

elem.addEventListener('keypress', function(event) { });

In propertykeyThe object with the event will contain the pressed symbol:

elem.addEventListener('keypress', function(event) { console.log(event.key); });

And in the propertycodeThe code of the key pressed will be:

elem.addEventListener('keypress', function(event) { console.log(event.code); });
