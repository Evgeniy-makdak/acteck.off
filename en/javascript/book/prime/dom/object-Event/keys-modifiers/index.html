Tracking modifier keys in JavaScript

Using an objectEventyou can find out whether the keys were pressed at the time of the eventCtrl, AltAndShift. This is done using propertiesctrlKey, altKeyAndshiftKey- they mattertrueorfalsedepending on whether this key was pressed at the time of the event or not.

Let's look at an example. Let's say we have the following button:

<button id="elem">text</button>

When you click on the button, we will display a message indicating whether one of the keys was pressedCtrl, AltAndShift:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function(event) { if (event.ctrlKey) { console.log('нажат Ctrl'); } if (event.altKey) { console.log('нажат Alt'); } if (event.shiftKey) { console.log('нажат Shift'); } });

Given element. Make it so that when you click on it, it turns red, but only if a key is pressed at the time of clickingAlt.

Make it so that by clicking on anyli, a number was added to the end of its text1, if the key is pressedCtrl, and number2, if the key is pressedShift.
