Passing a sequence number to a callback in JavaScript

Let's make sure that if the second parameter is written in our callback function, let it contain the serial number of the element in the found set of elements:

forEach('.elem', function(elem, index) { console.log(elem); // выведет элемент console.log(index); // выведет порядковый номер элемента });

Using this parameter, we can, for example, add its serial number to the end of each element:

forEach('.elem', function(elem, index) { elem.textContent = elem.textContent + index; });

Let's rewrite our forEach function code to do this:

function forEach(selector, func) { let elems = document.querySelectorAll(selector); for (let i = 0; i < elems.length; i++) { func(elems[i], i); } }

Paragraphs are given. Using the function we createdforEachFor each paragraph, add its serial number to the beginning.
