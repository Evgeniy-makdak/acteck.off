Functions for working with a group of DOM elements

Let's now write a function that will take a selector of a group of elements and their new text as a parameter. Let this function set all elements matching the selector to a new text.

Let's implement the described function:

function setText(selector, text) { let elems = document.querySelectorAll(selector); for (let elem of elems) { elem.textContent = text; } }

Let's test it on a group of the following elements:

<p class="elem"></p> <p class="elem"></p> <p class="elem"></p>

Let's set all elements with the classelemnew text:

setText('.elem', 'text');

Make a functionappendText, which will take a selector as its first parameter and text as its second. Make this function append text to the end of the passed elements.
