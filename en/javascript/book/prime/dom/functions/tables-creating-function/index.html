Function for creating HTML tables in JavaScript

Let's now make a functioncreateTable, which will create a table of the given size and append it to the end of the given element.

Let our function take the number of rows as the first parameter, the number of columns as the second parameter, and the third parameter as a link to the DOM element within which our table will be created.

Let's see how we will use the described function once it is created. Let, for example, we have the following div:

<div id="elem"></div>

Let's make a table inside this div3on4:

let div = document.querySelector('#elem'); createTable(3, 4, div);

Let us now have two divas:

<div id="elem1"></div> <div id="elem2"></div>

Let's make our own table in each of these divs:

let div1 = document.querySelector('#elem1'); createTable(3, 4, div1); let div2 = document.querySelector('#elem2'); createTable(5, 6, div2);

To make the created tables immediately visible, you can add some CSS, for example, this:

td { width: 50px; height: 50px; border: 1px solid black; }
