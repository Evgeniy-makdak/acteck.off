Functions for working with DOM elements in JavaScript

Now we will learn how to make auxiliary functions that perform any operations with the DOM. For example, let's make a function whose first parameter will beidelement, and the second is the text of the element, and will set the new text to this element.

Here is the function described:

function setText(id, text) { let elem = document.getElementById(id); elem.textContent = text; }

Let's try it out. Let's say we have two paragraphs:

<p id="elem1"></p> <p id="elem2"></p>

Let's change the text of these paragraphs using the function we created:

setText('elem1', 'text1'); setText('elem2', 'text2');

Modify the function I created so that it takes notidelement, but a custom CSS selector.

Make a functionsetAttr, which will change the DOM attribute of the element. Let the function take the element selector as the first parameter, the attribute name as the second, and its new value as the third.
