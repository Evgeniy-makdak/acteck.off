Passing a callback to work with the DOM in JavaScript

Let us have some group of DOM elements:

<p class="elem">1</p> <p class="elem">2</p> <p class="elem">3</p> <p class="elem">4</p> <p class="elem">5</p>

Let's make a functionforEach, which will take the element group selector as the first parameter, and the second parameter -callback function, which will be applied in turn to each of the found elements:

forEach('.elem', function() { // функция применится к каждому элементу });

Let the elements that fall under the selector, in turn, fall into the first callback parameter:

forEach('.elem', function(elem) { console.log(elem); // выведет по очереди найденные элементы });

Let's use our function to find all elements with the classelemand for each element found, apply a callback that will square the text of each element:

forEach('.elem', function(elem) { elem.textContent = elem.textContent ** 2; });

Let's now write the implementation of the function we have in mindforEach:

function forEach(selector, func) { let elems = document.querySelectorAll(selector); for (let elem of elems) { func(elem); } }

Paragraphs are given. Using the forEach function we created, for each paragraph, add an exclamation point to the end of its text.
