Passing a DOM element as a function parameter

In previous lessons, we passed element selectors to our functions, and our functions themselves received references to these elements within their code. There is another approach: you can pass previously received references to elements to functions.

Let's look at an example. Let's say we have the following code:

<p id="elem1"></p> <p id="elem2"></p>

Let's make a functionsetText, which will take a link to a DOM element as its first parameter, and the text of the element as its second parameter:

function setText(elem, text){ elem.textContent = text; }

We use the function we created to set the text of our paragraphs:

let elem1 = document.getElementById('elem1'); setText(elem1, 'text1'); let elem2 = document.getElementById('elem2'); setText(elem2, 'text2');
