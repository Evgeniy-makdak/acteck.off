Changing an Array in a JavaScript Loop

Array elements can be changed in loops. To do this, you need to iterate through the arrays in a regular loop.for. For example, let's multiply the elements of the array by2:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { arr[i] = arr[i] * 2; } console.log(arr); // выведет [2, 4, 6, 8, 10]

Now let’s increase each element of the array by one:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { arr[i]++; } console.log(arr); // выведет [2, 3, 4, 5, 6]

Now let's increase each element of the array by5:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { arr[i] += 5; } console.log(arr); // выведет [6, 7, 8, 9, 10]
