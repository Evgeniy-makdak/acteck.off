Looping through an array with a for-of loop in JavaScript

Cyclefor-ofallows you to sequentially iterate through array elements. It has the following syntax:

for (let переменнаяДляЭлемента of массив) { /* В переменнуюДляЭлемента по очереди будут попадать элементы перебираемого массива. */ }

Let's look at an example. Let the following array be given:

let arr = [1, 2, 3, 4, 5];

Let's loop through this array and output its elements to the console:

for (let elem of arr) { console.log(elem); }

let arr = ['a', 'b', 'c', 'd', 'e'];
