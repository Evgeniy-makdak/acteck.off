Loop for JavaScript

Cycleforallows you to repeat some code a specified number of times. Here is its syntax:

for ( начальные команды; условие окончания; команды после прохода ) { тело цикла }

The initial commands are what will be executed before the loop starts. They will only be executed once. Usually the initial values ​​of the counters are placed there, for example:i = 0.

The loop ending condition is the condition under which the loop will spin,as long as it's true, For example:i <= 10.

Команды после прохода цикла - это команды, которые будут выполнятся каждый раз при окончании прохода цикла. Обычно там увеличивают счетчики, например: i++.

Давайте с помощью цикла for выведем последовательно числа от 1 до 9:

for (let i = 1; i <= 9; i++) { console.log(i); // выведет 1, 2... 9 }

А теперь будем увеличивать счетчик не на 1, а на 2:

for (let i = 1; i <= 9; i += 2) { console.log(i); // выведет 1, 3, 5... }

Можно выполнить обратный отсчет:

for (let i = 10; i > 0; i--) { console.log(i); // выведет 10, 9, 8... }

С помощью цикла for выведите в консоль нечетные числа в промежутке от 1 до 99.
