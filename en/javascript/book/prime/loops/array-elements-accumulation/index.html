Накопление элементов массивов в циклах JavaScript

Давайте с помощью цикла найдем сумму элементов массива:

let arr = [1, 2, 3, 4, 5]; let res = 0; for (let elem of arr) { res += elem; } console.log(res); // искомая сумма
