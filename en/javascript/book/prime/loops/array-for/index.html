For loop for arrays in JavaScript

Arrays can also be iterated through in a loopfor. Let's see how it's done. Let us be given the following array:

let arr = [1, 2, 3, 4, 5];

Let's display the elements of this array in a loop:

for (let i = 0; i <= arr.length - 1; i++) { console.log(arr[i]); }

You can not subtract one from the length of the array, but use a strict comparison:

for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Looping through an arrayforgives more control over what is happening. For example, you can display elements not from zero, but from first:

for (let i = 1; i < arr.length; i++) { console.log(arr[i]); }

You can display the elements in reverse order:

for (let i = arr.length - 1; i >= 0; i--) { console.log(arr[i]); }

Print all elements of the following array to the console:

let arr = ['a', 'b', 'c', 'd', 'e'];

Print to the console all elements of the following array, except for the zero and last ones:

let arr = ['a', 'b', 'c', 'd', 'e'];

Print the elements of the following array to the console in reverse order:

let arr = ['a', 'b', 'c', 'd', 'e'];
