Changing objects through a loop in JavaScript

Objects can also be changed in a loop. Let, for example, we have this object:

let obj = {a: 1, b: 2, c: 3};

Let's loop through itfor-inand increase each of its elements by2times:

for (let key in obj) { obj[key] = obj[key] * 2; }

Let's check the result:

console.log(obj);

Given object:

let obj = {x: 1, y: 2, z: 3};

Loop through this object and square each element of this object.

Given object:

let obj = {x: 1, y: 2, z: 3};

Loop through this object and increment each element of this object by one.
