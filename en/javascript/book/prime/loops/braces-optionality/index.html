Optionality of parentheses in loops in JavaScript

Curly braces are not required in loops. If they are omitted, the loop will only execute the one line below it.

Let's look at an example. Let's say we have a loop with curly braces:

for (let i = 0; i <= 9; i++) { console.log(i); // выведет числа от 0 до 9 }

Let's omit the curly braces and the result will not change:

for (let i = 0; i <= 9; i++) console.log(i); // выведет числа от 0 до 9

Rewrite the following code without the curly braces:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); }
