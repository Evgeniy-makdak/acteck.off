Forming strings through loops in JavaScript

Using loops you can form strings. Let's make an example of a string filled with ten letters'x':

let str = ''; for (let i = 0; i < 10; i++) { str += 'x'; } console.log(str); // выведет 'xxxxxxxxxx'

Now let's make the line'12345'. To do this, we will add a loop counter to our variable:

let str = ''; for (let i = 1; i <= 5; i++) { str += i; } console.log(str); // выведет '12345'
