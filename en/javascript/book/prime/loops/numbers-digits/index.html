Digits of a number when iterating through a JavaScript loop

Suppose that when searching through numbers, we want to get some digit from these numbers. For example, we want to make sure that a number is printed to the console if its first digit is equal to1or2.

Let me remind you that you can’t just refer to number symbols, that is, it won’t work like this:

for (let i = 1; i <= 100; i++) { if (i[0] == 1 || i[0] == 2) { // так не будет работать console.log(i); } }

First you need to convert the number into a string and then get a specific character of the resulting string, like this:

for (let i = 1; i <= 100; i++) { let str = String(i); // преобразуем число в строку if (str[0] === '1' || str[0] === '2') { console.log(i); } }
