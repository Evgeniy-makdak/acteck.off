Working with Flags in JavaScript

Now we will learn how to work withflags. A flag is a special variable that can only take two values:trueAndfalse. Using flags, you can solve problems that check the absence of something: for example, you can check that there is no element with a certain value in an array. Let's get started.

Let's solve the following problem: given an array with numbers, we need to check if it contains an element with a value3or not. If there is, we will bring it out'+++', if not, we will withdraw it'---'.

First, let's try to derive'+++'. To do this, let's go through all the elements of our array and ask if the current element is equal to the value3. If equal, we will output'+++':

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem == 3) { console.log('+++'); } }

But our solution is not very good: after all, if there is more than one value in the array3, but several, then'+++'will appear several times. Let's fix our array (make two elements with value 3) and make sure of this:

let arr = [1, 2, 3, 4, 3, 5]; for (let elem of arr) { if (elem == 3) { console.log('+++'); // выведет несколько раз } }

Let's fix the problem: let's end the loop withbreakif the element is already found:

let arr = [1, 2, 3, 4, 3, 5]; for (let elem of arr) { if (elem == 3) { console.log('+++'); break; // завершим цикл } }

Let's now try to make sure that if there are no elements in the array with the value3, was output'---'. A common misconception is to addelseto our Ifu - in this case'---'will be output to all elements that are not3:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem == 3) { console.log('+++'); // выведет на элементе 3 } else { console.log('---'); // выведет на элементах 1, 2, 4, 5 } }

So the idea is to addelse- bad idea, not working. To solve problems of this type (such problems occur quite often) they use so-called flags.

As mentioned above, a flag is a variable that can take two values:trueorfalse.

So let's make a variableflagwith this value: if it is equaltrue, then the array has an element3, and iffalse, then there is no such element.

Initially, let's set the variableflagin meaningfalse- that is, we will assume that the element3not in the array:

let arr = [1, 2, 3, 4, 5]; let flag = false; // считаем, что элемента 3 нету в массиве

Then let's run the if loop as we did before. If the loop detects that there is element 3 in the array, then we set the variableflagin meaningtrueand exit the loop withbreak:

let arr = [1, 2, 3, 4, 5]; let flag = false; // считаем, что элемента 3 нет в массиве for (let elem of arr) { if (elem == 3) { flag = true; // элемент есть - переопределим переменную flag break; // выйдем из цикла } }

Answer to the question if in an array3or not, we can only give after the cycle. And we already have this answer: after the loop the variableflagcould have stayedfalseor could change its meaning totrueif the loop found in the array3:

let arr = [1, 2, 3, 4, 5]; let flag = false; for (let elem of arr) { if (elem == 3) { flag = true; break; } } // тут переменная flag равна или true, или false

Now after the loop we can make an if that will look at the variableflagand will display'+++'or'---':

let arr = [1, 2, 3, 4, 5]; let flag = false; for (let elem of arr) { if (elem == 3) { flag = true; break; } } if (flag === true) { console.log('+++'); } else { console.log('---'); }
