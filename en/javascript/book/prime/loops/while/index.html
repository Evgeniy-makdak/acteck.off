While loop in JavaScript

Cyclewhilewill be executed untiltruethe expression passed to it as a parameter. It allows you to perform an arbitrary number of iterations. Here is its syntax:

while ( пока выражение истинно ) { выполняем этот код циклически; в начале каждого цикла проверяем выражение в круглых скобках }

The loop will end when the expression is no longer true. If it was false initially, then it will never be executed.

For example, let's output sequentially using a loopwhilenumbers from one to five:

let i = 1; while (i <= 5) { console.log(i); i++; }

Now let's count down:

let i = 10; while (i > 0) { console.log(i); i--; }

Now we will divide the given number by2so many times until the result becomes smaller10:

let num = 500; while (num > 10) { num = num / 2; } console.log(num); // результат
