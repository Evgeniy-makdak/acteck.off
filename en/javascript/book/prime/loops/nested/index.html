Nested Loops in JavaScript

Cycles that you already know how to work with can be nested within each other. For example, let's solve the following problem: display the line on the screen:

111222333444555666777888999

Here one loop is not enough - you need to run two nested loops: the first loop will iterate over the numbers, and the second loop will repeat these numbers three times. Let's implement:

for (let i = 1; i <= 9; i++) { for (let j = 1; j <= 3; j++) { document.write(i); } }

Please note: the first loop has a counteri, secondj, and if there is also a third cycle, then its counter will be a variablek. These are standard common names and should be used.

111222333444555666777888999
