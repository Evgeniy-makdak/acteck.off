Accumulating numbers in JavaScript loops

Let's use a loop to find the sum of integers from1to100. To solve such a problem, numbers are sorted through in a loop and their sum is sequentially written into some variable:

let res = 0; for (let i = 1; i <= 100; i++) { res = res + i; } console.log(res); // искомая сумма

You can simplify the solution using the operator+=:

let res = 0; for (let i = 1; i <= 100; i++) { res += i; } console.log(res);
