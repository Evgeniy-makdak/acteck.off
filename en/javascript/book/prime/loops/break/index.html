Break Statement in JavaScript

Instructionsbreakallows you to terminate the cycle early. Let's see when this might come in handy. Let's say we have a loop that prints array elements to the console:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); }

Let us be faced with the task of determining whether there is a number in the array3. If there is, we will output the word to the console'есть'(and if not, we won’t do anything).

Let's solve our problem:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem === 3) { console.log('есть'); } }

The problem is solved, however, there is a problem: after the number3has already been found, the array still continues to pointlessly move on, wasting valuable processor resources and slowing down the work of our script.

It would be more optimal to terminate our loop immediately after finding the number. This can be done using special instructionsbreak, which allows the cycle to be completed ahead of schedule.

So let's end the loop as soon as we encounter a number3:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem == 3) { console.log('есть'); break; // выйдем из цикла } }

Instructionsbreakcan complete any cycles: normalfor, whileand so on.
