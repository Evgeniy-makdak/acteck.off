Tips for debugging code using loops in JavaScript as an example

Now I will teach you a technique for finding errors in code. Let's imagine that, due to some misunderstanding, you did not follow the advice from the previous lesson and ended up with a code that doesn't work. Let's see how to find errors and make it work.

Many beginners use the wrong search method. They take a look at the code in the hope that they will notice an error. And so on for a couple of hours. It doesn't work that way.

The correct technique is to output variables to the console and determine whether what you need is there or not. In this case, you need to start from the expected location of the problem and move up through the code. Let's see in practice.

For example, let’s say you were faced with the task of iterating through an array of two-digit numbers and finding numbers whose first digit is one more than the second. For simplicity, let the array store numbers as strings.

Let's say you solved the problem and got the following code:

let arr = ['21', '32', '34', '43', '45', '54', '55']; let sum = 0; for (let elem of arr) { if (elem[0] === elem[1] + 1) { sum += elem; } } console.log(sum); // выдает 0

The code, however, produces0. In this case, the first thing you need to check is to see if the code execution variable is included in the if. Let's display something in it to the console:

let arr = ['21', '32', '34', '43', '45', '54', '55']; let sum = 0; for (let elem of arr) { if (elem[0] === elem[1] + 1) { console.log('!'); // ничего не выводит sum += elem; } } console.log(sum);

Since nothing appeared in the console, this means that the code execution simply does not enter the condition. Let's see with our own eyes what we are comparing:

let arr = ['21', '32', '34', '43', '45', '54', '55']; let sum = 0; for (let elem of arr) { console.log(elem[0], elem[1] + 1); // '2' и '11', '3' и '21' ... if (elem[0] === elem[1] + 1) { sum += elem; } } console.log(sum);

As a result, it immediately becomes clear that the second term is two-valued. It's easy to see that this is because one is added as a string. Let's fix the problem:

let arr = ['21', '32', '34', '43', '45', '54', '55']; let sum = 0; for (let elem of arr) { if (+elem[0] === +elem[1] + 1) { // исправляем sum += elem; } } console.log(sum); // '021324354'

After the correction, we already see that in the variablesumsomething appeared, albeit incorrect. It is clear, however, that there is what we need there, but it is in the form of strings, not numbers. Let's fix the problem:

let arr = ['21', '32', '34', '43', '45', '54', '55']; let sum = 0; for (let elem of arr) { if (+elem[0] === +elem[1] + 1) { sum += +elem; // исправляем } } console.log(sum); // все работает

Correct the errors in the following code:

let obj = {a: 10, b: 20, c: 30, d: 40, e: 50}; let sum = 0; for (let elem in obj) { if (elem[0] === '1' && elem[0] === '2') { sum += +elem; } } console.log(sum);
