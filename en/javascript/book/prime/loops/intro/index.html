Introduction to Loops in JavaScript

In this lesson we will start learningcycles. Loops are used to execute a section of code several times in a row.

Why is this necessary - imagine that you need to square100array elements. If you access each element separately by its key, it will take100lines of code, and in order to write this code, you will need to spend quite a lot of time.

But this is not necessary - we have the opportunity to have JavaScript perform some operation for us the required number of times. For example, I squared all the elements of an array. This is done using loops.

Loops can repeat some code a specified number of times. Each such pass of the loop is callediterationcycle.

Loops often use special variables that increase their value by one each iteration. Such variables are calledcycle counters. Counters are used to count how many times a loop has been executed. It is customary to use letters for countersi, jAndk.
