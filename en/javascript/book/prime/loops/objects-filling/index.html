Filling Objects Through a Loop in JavaScript

Objects can also be populated with data in loops. Let's look at an example. Let's say we have two arrays:

let keys = ['a', 'b', 'c', 'd', 'e']; let values = [1, 2, 3, 4, 5];

Let's use them to make an object, taking the keys for this object from the first array, and the values ​​from the second. To do this, let’s run a loop and form our object in the loop:

let obj = {}; for (let i = 0; i <= 4; i++) { let key = keys[i]; let value = values[i]; obj[key] = value; } console.log(obj);

You don't have to enter variables for the key and value, you can just do it like this:

let obj = {}; for (let i = 0; i <= 4; i++) { obj[keys[i]] = values[i]; } console.log(obj);
