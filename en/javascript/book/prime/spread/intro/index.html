spread operator in JavaScript

ES6 introduced a special operator...calledspread.

When the spread operator precedes an array, it splits that array into individual values, turning the array into a set of parameters needed to call a function.

This probably sounds confusing, so let's look at a simple example. Let us have a function that accepts3parameters and returning their sum:

function func(num1, num2, num3) { return num1 + num2 + num3; }

Let us also have an array of three elements:

let arr = [1, 2, 3];

Obviously, we can’t just pass this array as a function parameter, like this:

let arr = [1, 2, 3]; func(arr);

Indeed, in this case, the entire array will end up in the first parameter of the function, and a number must be passed to this parameter (and to the other parameters too).

Of course, you can do this:

let arr = [1, 2, 3]; func(arr[0], arr[1], arr[2]);

But there is an easier way - use the spread operator:

let arr = [1, 2, 3]; func(...arr);

The presence of a variable is not required - you can apply spread directly to the array:

func(...[1, 2, 3]);

Given an array:

let arr = [1, 2, 3, 4, 5];

The function is also given:

function func(num1, num2, num3, num4, num5) { return num1 + num2 + num3 + num4 + num5; }

Use the given function to find the sum of the array elements.
