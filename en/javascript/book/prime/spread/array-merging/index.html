The spread operator and array merging

Let's say we have two arrays:

let arr1 = ['a', 'b', 'c']; let arr2 = [1, 2, 3];

Let's make it so that between the first and second elements of the arrayarr2array elements insertedarr1.

In other words, we want to write code that from the current arrayarr2will make the following array:

[1, 'a', 'b', 'c', 2, 3]

The problem, in general, can be solved using the methodsplice. However, this problem can be solved much easier using spread:

let arr1 = ['a', 'b', 'c']; let arr2 = [1, ...arr1, 2, 3]; console.log(arr2); // выведет [1, 'a', 'b', 'c', 2, 3]

Without running the code, determine what will be printed to the console:

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; let arr = ['a', ...arr1, 'b', 'c', ...arr2]; console.log(arr);

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; let arr = ['a', ...arr1, ...arr1, 'b', 'c']; console.log(arr);

let arr1 = [1, 2, 3]; let arr2 = [...arr1, 4, 5, 6]; let arr = ['a', 'b', 'c', ...arr2]; console.log(arr);

let arr1 = [1, 2, 3]; let arr2 = [...arr1, 4, 5, 6]; let arr3 = [...arr2, 7, 8, 9]; let arr = [0, ...arr3]; console.log(arr);
