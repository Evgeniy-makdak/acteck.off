Context via parameter in JavaScript

There is another solution to the problem. Let's make the functionchildaccepted the parameter:

function child(param) { // тут будет код }

And when calling this function we will pass into itthis:

function parent() { child(this); // передаем параметром this function child(param) { // в переменной param лежит переданное содержимое this } }

Since the challengechildis carried out in the parent function, then the passed onethisindicates what is needed. Then this onethisgets into the parameterparamand in this form it will be available inside the function.

Here is the final code:

let elem = document.querySelector('#elem'); elem.addEventListener('blur', parent); function parent() { child(this); // передаем параметром this function child(param) { console.log(param.value); // выводим value инпута } }
