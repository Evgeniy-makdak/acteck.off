Bind a context using the bind method in JavaScript

Next methodbindallows you to permanently bind a context to a function. As a result, this method returns a new function, inside whichthiswill have a hard-coded value.

Let's look at an example.

Let us have an input:

<input id="elem" value="text">

Let the link to this input be written into a variableelem:

let elem = document.querySelector('#elem');

Let us also have the following functionfunc:

function func(param1, param2) { console.log(this.value + param1 + param2); }

Let's usebindlet's make a new function that will be a copy of the functionfunc, Butthisit will always be equalelem:

let newFunc = func.bind(elem);

Now in the variablenewFunclies the function. Let's call it by passing it as the first parameter'1', and in the second'2'(I remind you that elem contains an input with value equal to'text'):

newFunc('1', '2');

Let's put it all together:

let elem = document.getElementById('elem'); function func(param1, param2) { console.log(this.value + param1 + param2); } let newFunc = func.bind(elem); newFunc('1', '2'); // выведет 'text12'

It is not necessary to write down the result of the workbindto a new functionnewFunc, you can just rewritefunc. After thatfuncwill be the same function as it was, but with a tightly boundthis:

func = func.bind(elem);

Let the following code be given:

<input id="elem" value="hello"> let elem = document.getElementById('elem'); function func(name, surname) { console.log(this.value + ', ' + name + ' ' + surname); } // тут напишите конструкцию с bind() func('John', 'Smit'); // тут должно вывести 'hello, John Smit' func('Eric', 'Luis'); // тут должно вывести 'hello, Eric Luis'
