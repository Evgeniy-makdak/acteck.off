Infinity and -Infinity in JavaScript

ValuesInfinityAnd-Infinitydenote infinity and minus infinity, respectively. Such values ​​are obtained if a number is divided by zero - in this case, JavaScript does not throw an error, as in other programming languages, but returns these values:

alert( 1 / 0); // выведет Infinity alert(-1 / 0); // выведет -Infinity
