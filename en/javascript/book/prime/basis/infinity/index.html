Infinity и -Infinity в JavaScript

Значения Infinity и -Infinity обозначают соответственно бесконечность и минус бесконечность. Такие значения получаются, если какое-то число поделить на ноль - в этом случае JavaScript не выдает ошибку, как в других языках программирования, а возвращает эти значения:

alert( 1 / 0); // выведет Infinity alert(-1 / 0); // выведет -Infinity
