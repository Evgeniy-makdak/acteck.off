Exponentiation in JavaScript

There is also a special operator for raising a number to a power**. Let's use it to build a number10to the third power:

alert(10 ** 3); // выведет 1000

Let's raise the value of the variable to a power:

let a = 10; alert(a ** 3); // выведет 1000

It may be that both the number and the degree will be contained in variables:

let a = 10; let b = 3; alert(a ** b); // выведет 1000
