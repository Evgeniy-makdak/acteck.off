Outputting text to a document in JavaScript

JavaScript can be used to manipulate HTML page tags, for example, to display text in them. We'll learn this later when we go through the section on working with the DOM.

Sometimes, however, for debugging purposes you need to display some text in the browser window. This is done using the commanddocument.write. Let's see how it works using examples.

Let's print the numerical value:

document.write(123);

Let's print some text:

document.write('text');

Let's display the text enclosed in tags:

document.write('<b>text</b>');

Let's output the text from the variable:

let str = 'text'; document.write(str);

Let's enclose the text from the variable in tags:

let str = 'text'; document.write('<b>' + str + '</b>');

Let's display the text in two rows:

document.write('text <br> text');

Let's display a lot of text in a column:

document.write('text1<br>'); document.write('text2<br>'); document.write('text3<br>');

You can do it like this:

document.write('text'); document.write('<br>'); document.write('text');
