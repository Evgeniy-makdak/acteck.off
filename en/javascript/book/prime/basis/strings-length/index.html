JavaScript String Length

The number of characters per line is contained in the propertylength:

let str = 'abcde'; alert(str.length); // выведет 5

Propertylengthcan be applied directly to a string:

alert('abcde'.length); // выведет 5

Space is also a symbol:

alert('ab de'.length); // выведет 5

Write a string to a variable. Display the length of your string.
