Shorthand conversion to number in JavaScript

Often to shorten code insteadNumberput a sign in front of the line+. Thus, an operation is performed on the string that is only valid for numbers and the string is converted to a number.

See example:

let a = +'2'; // в переменную запишется число 2 let b = +'3'; // в переменную запишется число 3 alert(a + b); // выведет 5

Here's another example:

let a = '2'; let b = +a; // в b запишется число 2

But the next example, although working, does not look very nice. It would be more appropriate to use the functionNumber:

let a = '2'; let b = '3'; alert(+a + +b); // выведет 5

Code given:

let a = '2'; let b = '3'; alert(a + b); // выведет '23'
