Declaring Multiple Variables in JavaScript

Let's declare some variables:

let a = 1; let b = 2; let c = 3;

The above code can be simplified by writingletonce and after it listing the necessary variables with their values, like this:

let a = 1, b = 2, c = 3;

You can first declare all the variables and then assign values ​​to them:

let a, b, c; // объявляем все 3 переменные // Присваиваем переменным значения: a = 1; b = 2; c = 3;

Try all the described methods of declaring variables yourself.
