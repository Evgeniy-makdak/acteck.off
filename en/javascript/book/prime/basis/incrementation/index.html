Increment and Decrement Operations in JavaScript

Consider the following code:

let num = 0; num = num + 1; // прибавляем к переменной num число 1 alert(num); // выведет 1

As you already know, this code can be rewritten in a shortened form using the operator+=:

let num = 0; num += 1; // прибавляем к переменной num число 1 alert(num); // выведет 1

In fact, adding one occurs so often in programming that an even more abbreviated syntax was invented for this operation - a special operatorincrement ++, increasing the value of the variable by1.

Let's rewrite our code using it:

let num = 0; num++; // прибавляем к переменной num число 1 alert(num); // выведет 1

There is also an operationdecrement --, decreasing the value of the variable by1. See example:

let num = 0; num--; // отнимаем от переменной num число 1 alert(num); // выведет -1

Rework this code to include increment and decrement operations:

let num = 10; num = num + 1; num = num + 1; num = num - 1; alert(num);
