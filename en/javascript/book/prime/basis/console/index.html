Working with the Console in JavaScript

For the convenience of programmers, each browser has a specialdeveloper panel. To open this panel, you need to right-click anywhere on the site page. In the menu that appears, you need to select the lowest item (it will be called Inspect or something similar).

In the panel that opens, you will mainly use two tabs: Elements and Console. In the first tab you can get information on page tags, and in the second - JavaScript debugging information.

Let's look at the capabilities of the console.

With the help of a special teamconsole.logYou can output data to the console in your scripts. This is used to practicedebuggingprograms.

Let's output something to the console as an example:

console.log(123);

Now let's display the value of the variable:

let num = 123; console.log(num);

You can display several variables in turn:

let num1 = 123; let num2 = 456; console.log(num1); console.log(num2);

You can display multiple variables with a single command by separating the variables with commas:

let num1 = 123; let num2 = 456; console.log(num1, num2);
