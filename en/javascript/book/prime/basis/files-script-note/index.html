Features of the script tag in JavaScript

Please note that in the tagscriptyou can either write code or include a file. Trying to do both at the same time will not work. Therefore the following code does not work:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title></title> <script src="script.js"> alert('text'); </script> </head> <body> </body> </html>
