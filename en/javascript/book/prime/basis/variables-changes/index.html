Variable Value Changes

Let us have a variable to which we have assigned some value. We can then perform the assignment operation again and assign a different value to the variable:

let a; // объявляем переменную a = 1; // записываем в нее значение 1 alert(a); // выведет 1 a = 2; // записываем теперь значение 2, затирая значение 1 alert(a); // выведет 2

That is, the value of a variable is not something rigidly tied to it. We can freely write some data to a variable, read it, then write something else - and so on.
