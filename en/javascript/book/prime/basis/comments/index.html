Comments in JavaScript

Like HTML and CSS, JavaScript allows you to leave comments. They are single-line and multi-line.

Here's an example of a one-line comment:

alert('text'); // комментарий

Here is an example of a multiline comment:

/* комментарий */ alert('text');

Try both types of comments.
