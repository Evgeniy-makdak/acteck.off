Highlighting Numbers in JavaScript

FunctionNumberonly works with strings consisting of numbers. However, when programming in JavaScript, there are often situations where you need to deal with strings that begin with a number and then letters.

An example of such a string would be a value in pixels:'12px'. Suppose we need to get the number at the beginning, discarding the string part. For such an operation there is a functionparseInt. Let's see how it works with an example:

let num = parseInt('12px'); alert(num); // выведет 12

Pixels, however, are sometimes fractional:'12.5px'. In this case the functionparseIntpasses and prints only the whole part:

let num = parseInt('12.5px'); alert(num); // все равно выведет 12

In general, perhaps this behavior is exactly what you need. But if not, use the functionparseFloatwhich extracts a number along with its fractional part:

let num = parseFloat('12.5px'); alert(num); // выведет 12.5

Of course, the absence of a fractional part does not interfere with the functionparseFloatwork correctly:

let num = parseFloat('12px'); alert(num); // выведет 12
