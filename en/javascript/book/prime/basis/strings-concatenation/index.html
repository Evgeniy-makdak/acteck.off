Adding Strings in JavaScript

To add strings, just like to add numbers, you use the operator+:

let str = 'abc' + 'def'; // складываем две строки alert(str); // выведет 'abcdef'

Strings can also be stored in variables:

let str1 = 'abc'; let str2 = 'def'; alert(str1 + str2); // выведет 'abcdef'

You can also add variables and strings:

let str1 = 'abc'; let str2 = 'def'; alert(str1 + '!!!' + str2); // выведет 'abc!!!def'

Let's say two strings are stored in variables, and when we add them we want to insert a space between them. This is done like this:

let str1 = 'abc'; let str2 = 'def'; alert(str1 + ' ' + str2); // выведет 'abc def'

Let there be only one variable:

let str = 'abc'; alert(str + ' ' + 'def'); // выведет 'abc def'

In this case, there is no point in highlighting the space as a separate line - insert it as part of the second term:

let str = 'abc'; alert(str + ' def'); // выведет 'abc def'
