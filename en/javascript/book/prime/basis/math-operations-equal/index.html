Equal precedence of mathematical operations in JavaScript

Multiplication and division have equal priority and are performed in turn from left to right. Let's look at an example of what this means. The following code will do the division first and then the multiplication:

let a = 8 / 2 * 4; alert(a); // выведет 16 (результат 4 * 4)

If you rearrange the signs, then multiplication will be performed first, and then division:

let a = 8 * 2 / 4; alert(a); // выведет 4 (результат 16 / 4)

In the following example, each new division operation will apply to the previous one:

let a = 16 / 2 / 2 / 2; alert(a); // выведет 2

Without running the code, determine what will be displayed on the screen:

let a = 8 / 2 * 2; alert(a);

Without running the code, determine what will be displayed on the screen:

let a = 8 * 4 / 2 / 2; alert(a);
