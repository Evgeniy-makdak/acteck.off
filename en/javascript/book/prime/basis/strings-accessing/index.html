Referring to the digits of a number in JavaScript

Let us now store in a variable not a number in quotes, but just a number. In this case, an attempt to access its individual symbol will produceundefined, since this treatment only works for strings:

let test = 12345; alert(test[0]); // выдаст undefined

To solve the problem, let's convert our number to a string:

let test = String(12345); // строка alert(test[0]); // выведет '1' - все работает

Let us now want to find the sum of the first two digits:

let test = String(12345); // строка alert(test[0] + test[1]); // выведет '12' - суммирует как строки

Let's add a functionNumberto make the characters sum as numbers:

let test = String(12345); // строка alert(Number(test[0]) + Number(test[1])); // выведет 3

Let me remind you that this kind of problem will only arise during summation. When multiplying, for example, the conversion to numbers need not be performed:

let test = String(12345); // строка alert(test[1] * test[2]); // выведет 6
