Remainder in JavaScript

There is a special operator%, with which you can find the remainder when dividing one number by another:

alert(10 % 3); // выведет 1

If one number is evenly divided by the second, the remainder will be zero:

alert(10 % 2); // выведет 0

Operator%, of course, can be applied not only to numbers, but also to variables:

let a = 10; let b = 3; alert(a % b); // выведет 1
