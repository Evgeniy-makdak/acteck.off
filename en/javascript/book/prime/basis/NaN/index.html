NaN value

MeaningNaN (Not-A-Number) denotesnot a number. This value is obtained when you try to do mathematical operations not with numbers, but with something else. For example, an attempt to multiply a string by a number will give this value:

alert('abc' * 3); // выведет NaN

Any operations withNaNlead to the fact that the result is stillNaN:

alert(NaN + 1); // выведет NaN
