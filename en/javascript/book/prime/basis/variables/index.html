Variables in JavaScript

The basic concept of any programming language isvariable. A variable is a container in which we can store some data, such as strings or numbers.

Each variable must have a name, which can consist of Latin letters, numbers, symbols$and underscores. In this case, the first character of the variable name must not be a number. Examples of variable names:str, my_str, myStr, a1, $, $a, $$a.
