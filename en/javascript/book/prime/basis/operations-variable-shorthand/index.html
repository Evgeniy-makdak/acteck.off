JavaScript Shortcuts

For such operations, when its current value plus something else is written to a variable, there is a special shorthand syntax. This syntax uses a special operator instead of a regular assignment+=. Let's look at an example:

let num = 1; num += 3; // эквивалентно num = num + 3;

There are similar operators for other mathematical operations:

let num = 2; num -= 3; // эквивалентно num = num - 3;
let num = 2; num *= 3; // эквивалентно num = num * 3;
let num = 2; num /= 3; // эквивалентно num = num / 3;

Rework this code to use the following shortcuts:

let num = 47; num = num + 7; num = num - 18; num = num * 10; num = num / 15; alert(num);
