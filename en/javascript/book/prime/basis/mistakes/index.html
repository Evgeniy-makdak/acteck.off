Finding errors in the code

In the following problems, some programmer wrote the code and may have made mistakes in it. You should check if the code does what it says. If the code does not work correctly, you must fix the errors.

The code should find the sum of numbers:

let num1 = 1; let num2 = 2; console.log('сумма: ' + num1 + num2);

The code should find the sum of numbers:

let a = 1; let b = 2; console.log(a + b + c);

The code should find the sum of the digits of a number:

let num = '123'; let sum = num[0] + num[1] + num[2]; console.log(sum);

The code should print the first digit of the number:

let num = 123; console.log(num[0]);

The code should have output a number to the console1, but outputs0:

let a = 0; console.log(a++);

The code should print the number of digits in a number:

let num = 123; console.log(num.length);

The code should find the number of seconds in a day:

let a = 24 / 60 / 60; console.log(a);

The code should print the number of digits in a number:

let num = 123; let str = String(num); console.log(num.length);

The code should output the last digit of the number:

let num = 123; let str = String(num); console.log(num[num.length]);
