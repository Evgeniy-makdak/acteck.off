The value is undefined

Meaningundefineddenotes uncertainty. A variable will have this value if we declared it, but did not write a value to it:

let a; alert(a); // выведет undefined

This value can be assigned to a variable explicitly:

let a = undefined; alert(a); // выведет undefined

Declare a variable without giving it a value. Display the value of the created variable on the screen.
