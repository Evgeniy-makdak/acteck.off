Fractions in JavaScript

You can work with decimals in JavaScript. In this case, the integer and fractional parts are separated from each other by a dot. See example:

let a = 0.5; alert(a); // выведет 0.5

See another example:

let a = 0.5 + 0.5; alert(a); // выведет 1
