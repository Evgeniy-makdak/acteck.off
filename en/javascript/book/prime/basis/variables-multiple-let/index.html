Multiple lets for one variable

The same variable cannot be declared several times throughlet. For example, the following code will result in an error:

let a = 1; alert(a); let a = 2; alert(a);

There are two options to solve the problem. You can simply enter two different variables:

let a = 1; alert(a); let b = 2; alert(b);

Or you can first declare a variablea, and then do operations with it:

let a; a = 1; alert(a); a = 2; alert(a);
