Convert to a number in JavaScript

Let's look at the following code:

let a = '2'; let b = '3'; alert(a + b); // выведет '23'

As you can see, in our case, both variables contain strings and are added, respectively, as strings.

Suppose we would like the values ​​of our variables in this case to be added not as strings, but as numbers. In this case, we can force the type of our variable to be converted to a number using a special functionNumber:

let a = '2'; let b = '3'; alert(Number(a) + Number(b)); // выведет 5

You can convert strings to numbers not in addition operations, but immediately when writing to a variable - this will work:

let a = Number('2'); // в переменную запишется число 2 let b = Number('3'); // в переменную запишется число 3 alert(a + b); // выведет 5
