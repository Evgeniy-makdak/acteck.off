Using Variables

In order to use a variable, it must first beannounce: write a keyword before her namelet. Let's declare, for example, a variable calleda:

let a;

After declaring a variable, you can write to it (they sayassignher) any value, for example, a number or a string.

Writing data to a variable is carried out using the assignment operation=. Let's, for example, write into a variableanumber3:

let a = 3;

Now let's display the contents of this variable on the screen using the functionalert:

let a = 3; // объявляем переменную и задаем ей значение alert(a); // выведет 3

It is not necessary to write the value to the variable immediately after the declaration. You can first declare a variable and then assign a value to it:

let a; // объявим переменную a = 3; // присвоим ей значение alert(a); // выведем значение переменной на экран

As you can see,letIt is written before the variable name only once - when declaring this variable. Then, to use a variable, you simply write the name of that variable.
