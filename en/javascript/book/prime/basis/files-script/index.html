Including a file with JavaScript scripts

JavaScript code can be written in a separate file, which will then be connected to the HTML file. Let's see how it's done.

First, let's create a file with our script. This file must have the extension.js. For example, let's call itscript.js. Let's put some code in it:

alert('text');

Let's now connect our script file to the HTML file. For this purpose in the tagscriptin attributesrcyou need to specify the path to the file with the script:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title></title> <script src="script.js"></script> </head> <body> </body> </html>

Make a script file. Connect it to your HTML file. Check its operation.
