Multi-line

In JavaScript strings created using single or double quotes, line breaks are not allowed. That is, it won’t work like this:

let str = 'abc def'; // так не будет работать

And it won't work like this:

let str = "abc def"; // так не будет работать

But slash quotes are specifically designed to create multiline strings:

let str = `abc def`; // так будет работать

Write to variablestra string consisting of characters'a', 'b', 'c', and after each character, except the last one, there must be a line break.
