Converting invalid numbers in JavaScript

When trying to convert a string containing not only numbers, but also other characters, the functionNumberwill return the value as its resultNaN:

alert(Number('2s')); // выведет NaN

Using a plus will have the same effect:

alert(+'2s'); // выведет NaN
