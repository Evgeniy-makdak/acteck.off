Convert to string in JavaScript

Sometimes you need to perform the reverse conversion - from a number to a string. This is the purpose of the functionString:

let str = String(123); // теперь в переменной строка '123'

Another example:

let num1 = 1; let num2 = 2; alert(String(num1) + String(num2)); // выведет '12'
