Automatic data type conversion

As you already know, strings in JavaScript must be quoted. It may happen that all the characters in the string will be numbers:

let a = '123';

In this case the variableais also a string, but with some nuance.

The essence of the nuance is this: if you perform any mathematical operation on a string with numbers, this operation will be performed as if we really had numbers, and not strings:

alert('2' * '3'); // выведет 6

In this case, JavaScript sees that we are trying to perform an operation that is not valid for strings, but valid for numbers.

He also sees that our strings are actually numbers in quotes. Therefore, JavaScript automatically converts these strings to numbers and performs the corresponding mathematical operation on them.

This works if our strings contain only numbers. If there is at least one letter there, JavaScript will consider the operation incorrect and make its result a valueNaN:

alert('2' * '3s'); // выведет NaN

Additionally, automatic conversion occurs if the operation is mathematically correct but not valid for strings.

For example, strings cannot (but numbers can) be multiplied, divided, subtracted, so in all such operations there will be automatic conversion of strings with digits into numbers.

However, strings, like numbers, can be added using the operator+. This means that in case of addition there will be no automatic conversion:

let a = '1' + '2'; alert(a); // выведет '12' - произошло сложение строк

If one of the terms is a number and the other is a string, then the string always wins:

let a = 1 + '2'; alert(a); // выведет '12'

In such cases, JavaScript, on the contrary, automatically converts numbers into strings and performs string addition.

There are, however, nuances when there are more than two terms: in this case, the conversion to a specific type depends on the order of operations.

In the following example, first2And3will add up like numbers, and then the result will be added to the string'1'already like a line'5', which will ultimately give the line'15':

let a = '1' + (2 + 3); alert(a); // выведет '15'

If you remove the brackets, then everything will add up from left to right. That is, first to the line'1'the number will be added2and the result will be the line'12'. Then the number will be added to this line3and the result will be the line'123':

let a = '1' + 2 + 3; alert(a); // выведет '123'

But in the next example the numbers2And3are written at the beginning, so they will first add up like numbers, resulting in the number5, and then the line will be added to this number'1', resulting in the line'51':

let a = 2 + 3 + '1'; alert(a); // выведет '51'
