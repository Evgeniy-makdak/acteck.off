Template strings in JavaScript

There is a special type of quotes - slash quotes:

let str = `abc`; alert(str); // выведет 'abc'

You can insert variables within slash quotes. To do this, the variable name must be written in the construction${}.

Let's look at an example. Let's say we want to add strings and a variable:

let str = 'xxx'; let txt = 'aaa ' + str + ' bbb';

This code can be rewritten as follows:

let str = 'xxx'; let txt = `aaa ${str} bbb`;

Rewrite the following code using variable insertion:

let str1 = 'xxx'; let str2 = 'yyy'; let txt = 'aaa ' + str1 + ' bbb ' + str2 + ' ccc';
