Boolean Conversion in JavaScript

Now we will look at how special values ​​behavetrueAndfalsewhen converting them to strings or numbers. You will need this knowledge in the following lessons.

So, let's try to convert to a string using the functionString:

alert(String(true)); // выведет 'true' alert(String(false)); // выведет 'false'

As you can see, the valuetrueconverted to string'true', and the valuefalse- per line'false'. That is, when you try to add a string and a logical value, this logical value is converted to a string and string addition occurs:

alert('a' + true); // выведет 'atrue'

When casting to a number, the valuetrueconverted to number1, and the valuefalse- in number0:

alert(Number(true)); // выведет 1 alert(Number(false)); // выведет 0

In all mathematical operations, Boolean values ​​are first converted to a number. Example:

alert(true + 1); // выведет 2 alert(true + true); // выведет 2
