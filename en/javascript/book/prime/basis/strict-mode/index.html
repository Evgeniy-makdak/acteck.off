Strict mode in JavaScript

In modern JavaScript, when writing code, the first line needs to include the so-calledstrict regime. This mode forces the browser to use all modern language features.

To enable strict mode, the first line of the script must include the command"use strict". Let's rewrite our code in strict mode:

"use strict"; alert('text!');

In the following, for simplicity, I will not indicate that the code works in strict mode (but it is assumed that it does, and you always enable it).
