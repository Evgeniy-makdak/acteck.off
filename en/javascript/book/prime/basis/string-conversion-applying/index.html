Applying conversion to strings in JavaScript

Let us have a variablenumwith some number:

let num = 12345;

Let's find the number of digits in this number. As you already know, to find the length of a string, you can use the propertylength.

It, however, only works with strings, and if applied to a number, it will not work:

let num = 12345; alert(num.length); // выведет undefined

To solve the problem, we convert our number to a string and find the length of this string:

let num = 12345; let str = String(num); // преобразуем наше число к строке alert(str.length); // найдем длину строки

You don't have to enter an intermediate variablestr, and apply the propertylengthstraight to the function resultString:

let num = 12345; alert(String(num).length); // найдем длину строки
