Constants in JavaScript

In addition to variables in JavaScript, you can also createconstants. They can only be written to once, and then their values ​​cannot be changed. Constants are declared using the operatorconst.

Typically constants are used for values ​​that should not be accidentally changed in code. For example, let's create a constant that will store the value of pi:

const pi = 3.14;

Constants are usually written in capital letters to distinguish them from regular variables:

const PI = 3.14;

Let's try changing the value of our constant:

const PI = 3.14; PI = 3; // выдаст ошибку

Make it a constantPIand use it to calculate the length of a circle based on its radius.
