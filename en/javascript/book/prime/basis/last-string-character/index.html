Last character of a line in JavaScript

Let's print the last character of the line. At the same time, we will make sure that our script itself determines the number of the last character, regardless of the length of the line.

Let's say we have this line:

let str = 'abcde';

As you can see, the number of characters in this line is5. If you think a little, it becomes obvious that the number of the last character of this line will be on1less than its length, since the numbering of characters starts from zero.

It turns out that knowing the length of the string, we can subtract from it1and get the number of the last character, and then using this number you can get the last character itself.

As you already know, the length of a string can be found using the propertylength. Based on this, we find the number of the last character:

let str = 'abcde'; let last = str.length - 1; // номер последнего символа

We use the found number to display the symbol on the screen:

let str = 'abcde'; let last = str.length - 1; // номер последнего символа alert(str[last]); // выведет 'e'

Intermediate variablelastYou don't have to enter:

let str = 'abcde'; alert(str[str.length - 1]); // выведет 'e'
