Getting characters from a string in JavaScript

Let's say we have some string. Each character in this line has its own serial number: the first character is number0, second character is number1, third character number2and so on.

As you have already noticed, the numbering of characters starts from zero (zero as the beginning of numbering is often found in programming).

If necessary, you can access a specific character in a string by its number. To do this, write the name of the variable, after this name square brackets are placed and the symbol number is indicated in these brackets.

Let's look at an example. Let us be given this line:

let str = 'abcde';

Let's look at some characters in this line:

let str = 'abcde'; alert(str[0]); // выведет 'a' alert(str[1]); // выведет 'b' alert(str[2]); // выведет 'c'

The symbol number can also be stored in a variable:

let str = 'abcde'; let num = 3; // номер символа в переменной alert(str[num]); // выведет 'd'
