JavaScript prompt function

You already know about the existence of the functionalert, displaying the specified message on the screen. There is also a similar functionprompt, which allows you to receive some text from the user.

Let's ask for the username as an example:

prompt('Ваше имя?');

Run the above code to see the window output by the functionprompt. Enter your name and click on the button. If you have completed all the manipulations described above, the name you entered will be included in our script.

In order to access the entered name, the result of the functionpromptyou need to assign it to some variable, for example, like this:

let name = prompt('Ваше имя?');

Let's display the previously entered name on the next line of code using the functionalert:

let name = prompt('Ваше имя?'); alert('Ваше имя: ' + name);

You must understand that when calling a functionpromptfurther execution of the script is blocked until the appropriate data is entered. In general, the function also worksalert, only it waits for the corresponding button to be pressed.

By the way, it is not necessary to use a variable:

alert('Ваше имя: ' + prompt('Ваше имя?'));

Ask the user's age using the functionprompt. Withdraw usingalertuser entered age.
