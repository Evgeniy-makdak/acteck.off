Grouping brackets in JavaScript

If desired, you can indicate the precedence of operations using parentheses. Let's, for example, rewrite our code so that the addition is performed first, and only then the multiplication:

let a = 2 * (2 + 3); alert(a); // выведет 10 (результат 2 * 5)

Note: there can be any number of brackets, including nested ones:

let a = 2 * (2 + 4 * (3 + 1)); alert(a);

You can enclose operations that have priority in brackets - this will not be an error. For example, let's put the product of numbers in brackets:

let a = (2 * 2) + 3; alert(a); // выведет 7 (результат 4 + 3)

In this case, the parentheses are superfluous (multiplication already has priority), but the code is valid.

Sometimes such a grouping is used in places where the priority of operations is not obvious. For example, consider the following code:

let a = 8 / 2 * 4; alert(a);

As you already know, it will first perform division and then multiplication. But this may not be obvious at first glance.

You can use grouping brackets to explicitly show precedence:

let a = (8 / 2) * 4; alert(a);
