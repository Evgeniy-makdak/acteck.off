Type problem with the prompt function in JavaScript

Please note that the functionpromptalways returns a string, even if the user entered a number. For example, consider the following script:

let num1 = prompt('Введите первое число'); let num2 = prompt('Введите второе число'); alert(num1 + num2); // сложит числа как строки

Rework the above code to sum numbers rather than strings.

Using the functionpromptask the user for the side of the square. Display the area of ​​the entered square.

With two functionspromptask the user for the sides of the rectangle. Display the perimeter of the entered rectangle.
