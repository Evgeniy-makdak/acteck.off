escape method

Methodescapeencodes any string into a URL query string using the character'%'. Encoding is necessary to correctly transfer the characters in a string into a format that can be transmitted over the Internet. Also some unsafe ASCII characters are replaced with a safe sequence with'%'and two hexadecimal digits.

Syntax

querystring.escape(строка);

Example

Let's convert the string to a URL:

import querystring from 'querystring'; console.log(querystring.escape('abc def'));

Result of the executed code:

'abc%20def'
