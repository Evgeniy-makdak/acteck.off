throws method

Methodthrowsreturns an error object throws an exception when testing the function. In the first parameter we indicate the function that we will test, in the second optional parameter we indicate an error, which can be a regular expression or a function. The third optional parameter can be used to specify an error message.

Syntax

assert.throws(функция, [ошибка], [сообщение об ошибке]);

Example

Let's check if there is an error when executing the function:

import assert from 'assert'; let testNum = function (num) { if (num > 1) { console.log('number is correct') } else { console.log('number is incorrect'); } }; assert.throws(function () { testNum(2); });

Result of the executed code:

'number is correct'

Example

Now let’s test a number that is obviously inappropriate for the conditions of our function:

import assert from 'assert'; let testNum = function (num) { if (num > 1) { console.log('number is correct') } else { console.log('number is incorrect'); } }; assert.throws(function () { testNum(1); });

Result of the executed code:

'number is incorrect'
