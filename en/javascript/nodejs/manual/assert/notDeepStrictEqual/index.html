notDeepStrictEqual method

MethodnotDeepStrictEqualchecks the given parameters for deep strict inequality, i.e. also checks child parameter objects. In the first parameter of the method we indicate the first value for comparison, in the second parameter - the second value, in the third optional parameter - the error message. If two parameters are not equal to each other, then returnsundefined, if they are equal, an error is thrown.

Syntax

assert.notDeepStrictEqual(первое значение, второе значение, [сообщение об ошибке]);

Example

Let's compare two numbers1And2:

import assert from 'assert'; console.log(assert.notDeepStrictEqual(1, 2));

Result of the executed code:

undefined

Example

Now let's compare the numbers1With1:

import assert from 'assert'; console.log(assert.notDeepStrictEqual(1, 1));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: Expected "actual" not to be strictly deep-equal to: 1'
