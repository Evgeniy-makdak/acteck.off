deepStrictEqual method

MethoddeepStrictEqualchecks the given parameters for deep strict equality, i.e. also checks child parameter objects. In the first parameter of the method we indicate the first value for comparison, in the second parameter - the second value, in the third optional parameter - the error message.

Syntax

assert.deepStrictEqual(первое значение, второе значение, [сообщение об ошибке]);

Example

Let's compare the numbers1And2:

import assert from 'assert'; console.log(assert.deepStrictEqual(1, 2));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: Expected values to be strictly deep-equal: 1 !== 2'
