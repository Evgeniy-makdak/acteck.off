doesNotMatch method

MethoddoesNotMatchchecks whether there is a mismatch between the string specified in the first parameter and the regular expression specified in the second parameter. The third optional parameter can be used to specify an error message. If the string and the regular expression do not match each other, then the method returnsundefined, if they match, it throws an error.

Syntax

assert.doesNotMatch(строка, регулярное выражение, [сообщение об ошибке]);

Example

Let's compare the string'abcde'and regular expression/abc/:

import assert from 'assert'; console.log(assert.doesNotMatch('abcde', /abc/));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: The input was expected to not match the regular expression /abc/. Input: 'abcde' '

Example

Now let's check the string mismatch'abcde'and regular expression/fg/:

import assert from 'assert'; console.log(assert.doesNotMatch('abcde', /abc/));

Result of the executed code:

'undefined'
