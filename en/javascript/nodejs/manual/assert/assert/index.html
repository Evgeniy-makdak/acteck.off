assert method

Methodassertchecks the truth of the statement specified in the first parameter. If we give an expression that is equal to0orfalse, then the method returns an error. The second optional parameter can be used to specify an error message. Synonymous with methodassertis the methodok.

Syntax

assert(утверждение для проверки, [сообщение об ошибке]);

Example

Let's check if the statement is true1more2:

import assert from 'assert'; console.log(assert(1 > 2));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: false == true'

Example

Now let's check whether the statement is true2more1:

import assert from 'assert'; console.log(assert(2 > 1));

After executing the code we will see a messageundefined, which indicates that the statement is true:

'undefined'
