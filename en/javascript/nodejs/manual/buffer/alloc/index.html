alloc method

Methodalloccreates a new buffer object of the specified size. In the first parameter of the method we indicate the buffer size we need in bytes, in the second optional parameter - the value for filling the buffer (by default it is0). In the third optional parameter you can set the encoding type (default is'utf8').

Syntax

Buffer.alloc(размер буфера, [значение для заполнения], [кодировка]);

Example

Let's create a new buffer object with size12byte:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(12); console.log(buffer);

Result of the executed code:

<Buffer 00 00 00 00 00 00 00 00 00 00 00 00>

Example

Now let's pass the value to fill the buffer:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(12, 1); console.log(buffer);

Result of the executed code:

<Buffer 01 01 01 01 01 01 01 01 01 01 01 01>

See also

  • methodallocUnsafe,
    which creates a new unsafe unfilled buffer object
  • methodallocUnsafeSlow,
    which creates an empty buffer object without initialization
  • methodfrom,
    which creates a new buffer object from the specified value
  • propertypoolSize,
    which determines the size of the allocated buffer instances
