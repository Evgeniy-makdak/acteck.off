concat method

Methodconcatcombines all buffer objects from a given array into one object. The method returns a buffer. In the first parameter of the method we specify an array of buffer objects, in the second optional parameter we specify the size of the combined buffer.

Syntax

Buffer.concat(массив объектов буфера, [размер объединенного буфера]);

Example

Let's concatenate the buffer objects from the array:

import { Buffer } from 'buffer'; let buffer1 = Buffer.alloc(4); let buffer2 = Buffer.alloc(6); let arr = [buffer1, buffer2]; let newBuffer = Buffer.concat(arr) console.log(newBuffer);

Result of the executed code:

<Buffer 00 00 00 00 00 00 00 00 00 00>
