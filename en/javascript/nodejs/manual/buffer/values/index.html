values ​​method

Methodvaluesreturns an iterator of the values ​​or bytes in the buffer object. We do not specify anything in the method parameter.

Syntax

буфер.values();

Example

Let's find the indexes on our buffer object:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); for (let value of buffer.values()) { console.log(value); }

Result of the executed code:

97 98 99 100 101 102

See also

  • methodkeys,
    which returns an iterator of indices in the buffer object
  • methodentries,
    which returns the index-byte pair from the buffer object
