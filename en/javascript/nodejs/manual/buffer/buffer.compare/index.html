Buffer.compare property

PropertyBuffer.comparecompares a buffer object with another object. The property returns a number indicating their differences. If the method returns0- the buffer and the object are the same size,1- the object must be in front of the buffer when sorting,-1- when sorting, the object must be placed after the buffer. In the first parameter of the method we indicate the object that we want to compare with the buffer. In the second optional parameter we set the byte index of the object from which the comparison begins (by default it is equal to0). In the third optional parameter we indicate the index of the bytes of the object at which the comparison should end. The fourth optional parameter can specify the index of the buffer bytes from which the comparison should begin. In the fifth optional parameter, you can specify the index of the buffer bytes at which the comparison should end.

Syntax

буфер.compare(объект для сравнения, [начало сравнения объекта], [конец сравнения объекта], [начало сравнения буфера], [конец сравнения буфера]);

Example

Let's compare the buffer and our object:

import { Buffer } from 'buffer'; let buffer1 = Buffer.from('123'); let buffer2 = Buffer.from('456'); console.log(buffer1.compare(buffer2));

Result of the executed code:

-1
