toString method

MethodtoStringconverts a buffer object to a string value. In the first optional parameter of the method, you can specify the encoding (by default it is equal to'utf8'). In the second optional parameter, you can specify the index from which the transformation will begin (by default it is equal to0). In the third optional parameter, you can specify the index at which the conversion will end (by default it is equal to the length of the buffer object).

Syntax

буфер.toString();

Example

Let's convert our buffer object to a string:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer.toString());

Result of the executed code:

'abcde'
