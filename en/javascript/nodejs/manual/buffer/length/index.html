length property

Propertylengthreturns the number of bytes in the buffer.

Syntax

буфер.length;

Example

Let's find out how many bytes are in our buffer object:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer.length);

Result of the executed code:

5
