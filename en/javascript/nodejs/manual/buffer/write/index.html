write method

Methodwritewrites a string to a buffer object. In the first parameter of the method we indicate the string that we want to write to the buffer. In the second optional parameter, you can specify the byte index from which to start recording. The third optional parameter allows you to specify the maximum number of bytes to write. In the fourth optional parameter of the method, you can set the encoding (by default it is equal to'utf8'). The method returns the number of bytes written.

Syntax

буфер.write(строка для записи, [индекс байтов перед записью], [максимальное количество байтов для записи], [кодировка]);

Example

Let's write the string to our buffer object'abc':

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(4); console.log(buffer.write('abc'));

Result of the executed code:

3
