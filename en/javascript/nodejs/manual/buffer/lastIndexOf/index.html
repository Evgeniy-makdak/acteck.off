lastIndexOf method

MethodlastIndexOflooks for the last index of a given value in a buffer object. In the first parameter of the method we indicate the value that we want to find in the buffer object. The value can be a string, a buffer, or a number. In the second optional parameter, you can specify the byte index from which the search for the value will begin, in the third optional parameter - the encoding (by default it is equal to'utf8'). The method returns the index of the value we need; if the value is not found, it will return-1.

Syntax

буфер.lastIndexOf(значение, [начало поиска], [кодировка]);

Example

Let's find the index'a'in our buffer object:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcabc'); console.log(buffer.lastIndexOf('a'));
