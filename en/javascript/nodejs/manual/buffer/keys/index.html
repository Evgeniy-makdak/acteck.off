Keys method

Methodkeysreturns an iterator of the keys or indexes in the buffer object. We do not specify anything in the method parameter.

Syntax

буфер.keys();

Example

Let's find the indexes on our buffer object:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); for (let key of buffer.keys()) { console.log(key); }

Result of the executed code:

0 1 2 3 4 5

See also

  • methodentries,
    which returns the index-byte pair from the buffer object
  • methodvalues,
    which returns an iterator of bytes in the buffer object
