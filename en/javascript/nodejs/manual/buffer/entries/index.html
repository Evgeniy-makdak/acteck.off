entries method

Methodentriesreturns an index-byte pair from a buffer object. We do not specify anything in the method parameter.

Syntax

буфер.entries();

Example

Let's get the index-byte pairs from our buffer object:

import { Buffer } from 'buffer'; let buffer = Buffer.from('123'); for (const elem of buffer.entries()) { console.log(elem); }

Result of the executed code:

[ 0, 49 ] [ 1, 50 ] [ 2, 51 ]

See also

  • methodkeys,
    which returns an iterator of indices in the buffer object
  • methodvalues,
    which returns an iterator of bytes in the buffer object
