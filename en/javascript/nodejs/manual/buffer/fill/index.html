fill method

Methodfillfills a buffer object with the data specified in the first parameter of the method. In the second optional parameter, you can specify the byte index from which filling of the buffer object will begin, in the third optional parameter - the index at which filling will end. In the fourth optional parameter you can set the encoding (by default it is equal to'utf8').

Syntax

буфер.fill(данные, [начало заполнения], [конец заполнения], [кодировка]);

Example

Let's fill the buffer with data:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(4); buffer.fill('a'); console.log(buffer.toString());

Result of the executed code:

'aaaa'
