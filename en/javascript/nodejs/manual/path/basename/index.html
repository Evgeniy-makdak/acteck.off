basename method

Methodbasenamereturns the file name from the file path specified in the first parameter. In the second optional parameter we indicate the type of file extension -.txt, .htmletc.

Syntax

path.basename(путь к файлу, [расширение файла]);

Example

Let's find out the file name from its path:

import path from 'path'; console.log(path.basename('./dir/file'));

Result of the executed code:

'file'

Example

And now in the second parameter we indicate the name of the file with what extension you need to find out:

import path from 'path'; console.log(path.basename('./dir/file', 'html'));

Result of the executed code:

'file'

    which returns the folder name from the path
