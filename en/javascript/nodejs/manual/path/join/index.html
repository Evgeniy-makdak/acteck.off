join method

Methodjoinconcatenates all path segments using a platform-specific delimiter. After merging, normalizes the resulting path. All segments specified in the method parameter must have string values.

Syntax

path.join(сегменты пути);

Example

Let's combine the segments into one common path:

import path from 'path'; console.log(path.join('/dir', 'file.txt'));

Result of the executed code:

'\dir\file.txt'

See also

  • methodnormalize,
    which normalizes the path
  • methodresolve,
    which returns the absolute path from the segments
