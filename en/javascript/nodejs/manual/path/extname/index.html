extname method

Methodextnamereturns the file extension from the path specified in the parameter. When defining an extension, the method focuses on the values ​​after'.'. If there's no way'.', then an empty string will be returned.

Syntax

path.extname(путь к файлу);

Example

Let's find out the extension from the file path:

import path from 'path'; console.log(path.extname('./dir/file.txt'));

Result of the executed code:

'.txt'

Example

Let there be no extension now:

import path from 'path'; console.log(path.extname('./dir/file'));

After executing the code we will see:

''
