posix property

Propertyposixreturns POSIX implementations of path methods as an object.

Syntax

path.posix;

Example

Let's find out the implementations of path methods for POSIX:

import path from 'path'; console.log(path.posix);

See also

  • propertywin32,
    which returns the Windows implementations of the path methods as an object
