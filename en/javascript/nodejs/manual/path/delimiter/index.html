delimiter property

Propertydelimiterreturns the path separator for the current platform as a string value. For for Windows the property will return';', for POSIX -':'.

Syntax

path.delimiter;

Example

Let's define a path separator for our platform:

import path from 'path'; console.log(path.delimiter);
