parse method

Methodparseconverts the path from a string value to an object with the following keys:dir- folder name,root- root path,base- file with extension,name- name of the halyard,ext- extension. We pass the path into the method parameter, the syntax of which differs depending on the platform used. For example, for POSIX the path is written as'/dir/file.txt', and for Windows -'\\dir\\file.txt'.

Syntax

path.parse(путь к файлу);

Example

Let's convert the path from a string value to an object on the Windows platform:

import path from 'path'; console.log(path.parse('./dir/file.txt'));

Result of the executed code:

{ root: '\\', dir: '\\dir', base: 'file.txt', ext: '.txt', name: 'file' }
