Метод group

Метод group в модуле console группирует сообщения в консоли. В необязательном параметре метода можно указывать метку, с которой мы хотим сгруппировать сообщения.

Синтаксис

console.group([метка]);

Пример

Давайте сгруппируем сообщения в консоли:

console.log('ab'); console.log('cd'); console.group(); console.log('cd');

Результат выполнения кода:

'ab' 'cd' 'cd'

Смотрите также

  • метод groupCollapsed,
    который сворачивает группы сообщений в консоли
  • метод groupEnd,
    который завершает группировку сообщений в консоли
