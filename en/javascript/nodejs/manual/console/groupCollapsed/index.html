groupCollapsed method

MethodgroupCollapsedin the moduleconsolecollapses groups of messages in the console, and also launches them. In an optional parameter of the method, you can specify a label for grouping messages. The method works similar to the methodgroup.

Syntax

console.groupCollapsed([метка]);

Example

Let's collapse message groups in the console:

console.log('ab'); console.log('cd'); console.groupCollapsed(); console.log('cd');

Result of executing the code:

'ab' 'cd' 'cd'

    which groups messages on the console
  • methodgroupEnd,
    which completes the grouping of messages on the console
