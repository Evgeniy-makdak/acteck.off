clear method

Methodclearin the moduleconsoleclears the console output. The operation of the method varies depending on the operating system used and the types of terminals. For Linux operating systems, the method works the same as the shell clear command in the terminal. On Windows, the method clears only the current terminal browsing window for the NodeJs binary. We do not specify anything in the method parameter.

Syntax

console.clear();

Example

Let's clear our console:

console.clear();
