trace method

Methodtracein the moduleconsolereturns a stack trace of the executed code to the console. In the first optional parameter we indicate the message, in the second optional parameter - values ​​that can be substituted into the message from the first parameter.

Syntax

console.trace([сообщение], [значение для сообщения]);

Example

Let's find out the stack trace when the code is executed:

function func1(){ function func2() { console.trace(); }; func2(); }; func1();

Result of the executed code:

'Trace at func2 (file://путь к функции, включая номер строки редактора кода) at func1 (file://путь к функции, включая номер строки редактора кода) at file://путь к файлу, в котором выполняется код at ModuleJob.run (node:internal/modules/esm/module_job:192:25)'
