count method

Methodcountin the moduleconsoledisplays to the console the number of calls with the label specified in the method parameter. A string value is used as a label; by default it is equal to'default'.

Syntax

console.count([метка]);

Example

Let's find out how many calls we have with a tag'default'And'abc':

console.count(); console.count('abc'); console.count('abc');

Result of executing the code:

'default: 1' 'abc: 1' 'abc: 2'

    which restarts the console call counter
