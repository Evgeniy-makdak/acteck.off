countReset method

MethodcountResetin the moduleconsolerestarts the console call counter with the label specified in the method parameter. A string value is used as a label; by default it is equal to'default'.

Syntax

console.countReset([метка]);

Example

Let's find out how many calls we have with a tag'abc'after restarting the call counter:

console.countReset(); console.countReset('abc'); console.countReset('abc');

Result of executing the code:

'abc: 1' 'default: 1' 'abc: 1'
