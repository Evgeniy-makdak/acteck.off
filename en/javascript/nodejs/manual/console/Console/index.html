Console class

ClassConsolecreates a new console with one or more writable threads. The class parameter specifies an object with options.

Syntax

let newConsole = new Console({ stdout: writableStream, stderr: writableStream, ignoreErrors: true, colorMode:true });

Object with options

Option Description
stdout Specifies the stream to be written for outputting logs or information.
stderr Stream for warning or error output.
ignoreErrors Ignores errors when writing to underlying streams (default istrue).
colorMode The mode for setting the color in the console may make a differencetrue, falseorauto.
inspectOptions An object that specifies the parameters for a methodutil.inspect().
groupIndentation Sets group access (by default its value is2)

For the class to work, only the first two keys are required -stdoutAndstderr. If the keystderris not specified, it is used by defaultstdout.

Example

Let's create a new console:

import { Console } from 'console'; import fs from 'fs'; let out = fs.createWriteStream('./stdout.log'); let err = fs.createWriteStream('./stderr.log'); let newConsole = new Console({ stdout: out, stderr: err }); let count = 2; newConsole.log('count: %d', count); console.log('console created');
