assert method

Methodassertin the moduleconsoletests the truth of a value or statement. In the first parameter of the method we indicate the value that we want to check, in the second optional parameter - the message. If the value is true the method does not return anything, otherwise we will get an error.

Syntax

console.assert(значение или утверждение, [сообщение]);

Example

Let's check the truth of two statements:

console.assert( 2 > 1 , 'error 2 > 1'); console.assert(1 > 2, 'error 1 > 2');

Result of executing the code:

'Assertion failed: error 1 > 2'
