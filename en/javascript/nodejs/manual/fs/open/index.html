open method

Methodopenopens a file and returns a class objectFileHandle. The first parameter of the method specifies the path to the file that we want to open. In the second optional parameter we set a flag - this is a string or number that indicates the operation in which the file should be opened. By default the flag is'r'. In the third optional parameter you can set the mode of interaction with the file; by default it is equal to'чтение-запись'. In the fourth optional parameter we specify a callback, which in turn accepts parameterserr(error) andfd(file descriptor).

Syntax

fs.open(путь к файлу, [флаги], [режим], [коллбэк]);

Example

Let's open the file asynchronously viaawaitand then using the methodFileHandle.readFileLet's read its contents:

import fs from 'fs/promises'; try { let filehandle = await fs.open('file.txt'); let data = await filehandle.readFile('utf-8') console.log(data); } catch (err) { console.log(err); }

Example

Now let's asynchronously open and read the file viathen:

import fs from 'fs/promises'; fs.open('file.txt').then((filehandle) => { filehandle.readFile('utf-8').then(data => { console.log(data); }) }).catch((err) => { console.log(err); });

Example

Let's asynchronously open and read the file from the previous examples through a callback. First we create a buffer and then through the callback parameterfdLet's read the contents of the file:

import fs from 'fs'; const buffer = new Buffer.alloc(1024); fs.open('file.txt', function(err, fd) { if (!err) { fs.readSync(fd, buffer); let data = buffer.toString('utf8'); console.log(data); } else { console.log(err); } });

Example

Now let's synchronously open and read the file viaopenSync:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); const buffer = new Buffer.alloc(1024); try { fs.readSync(fileDescriptor, buffer); let data = buffer.toString('utf8'); console.log(data); } catch (err) { console.log(err); }
