watchFile method

MethodwatchFileused to continuously monitor changes only in a file, as opposed to the methodwatch, which applies to both files and folders. In the first parameter of the method we indicate the name of the file we need or set the path to it. In the second optional parameter we specify an object with options. In the third parameter we set a listener in which we indicatecurrentAndprevious- these are objects containing information about the current and previous changes in the file, respectively.

Syntax

fs.watchFile(имя файла или путь к нему, опции, слушатель);

Object with options

Option Description
bigint Iftrue, specifies the display of numeric values ​​in the formatbigint. Default valuefalse.
persistent Specifies that the process continues to run while viewing files. Default valuetrue.
interval Sets the frequency of accessing the target file. Default value5007.

Example

Let's start observing changes in the file asynchronously:

import fs from 'fs'; fs.watchFile('file.txt', (curr, prev) => { console.log('file watched'); console.log('previous modified:', prev.mtime); console.log('current modified:', curr.mtime); console.log('file content is ', fs.readFileSync('file.txt', 'utf8') ); }); fs.writeFileSync('file.txt', 'text' );
