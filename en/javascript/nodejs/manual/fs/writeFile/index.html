writeFile method

MethodwriteFilewrites data to a file. The first parameter of the method specifies the file name, the second parameter specifies the data that we want to write, and the third and fourth optional parameters contain options and a callback.

Syntax

fs.writeFile(имя файла, данные, [опции], [коллбэк]);

Example

Let's write data to a file asynchronously viaawait:

import fs from 'fs/promises'; try { await fs.writeFile('file.txt', 'text'); console.log('file written'); } catch (err) { console.log(err); }

Example

Let's write data to a file asynchronously viathen:

import fs from 'fs/promises'; fs.writeFile('file.txt', 'text').then(() => { console.log('file written'); }).catch(err => { console.log(err); });

Example

Let's asynchronously write data to a file via a callback:

import fs from 'fs'; fs.writeFile('file.txt', 'text', err => { if (!err) { console.log('file written'); } else { console.log(err); } });

Example

Let's synchronize the data to the file viawriteFileSync:

import fs from 'fs'; try { let text = fs.writeFileSync('file.txt', 'text'); console.log('file written'); } catch (err) { console.log(err); }
