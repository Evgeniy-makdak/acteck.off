link method

Methodlinkchanges the path to the file and can accordingly set a new file name. The method returns a new file, and the original file remains in the same path. In the first parameter we indicate the original path to the file, in the second parameter - the new path to the file, in the third optional parameter we indicate the callback.

Syntax

fs.link(исходный путь, новый путь, [коллбэк]);

Example

Let's say we have the following file structure:

  • /dir1/
    • /dir1.2/
      • file1.txt
      • file2.txt
  • /dir2/
      • file3.txt

Let's set a new file path asynchronouslyfile1throughawait:

import fs from 'fs/promises'; try { await fs.link('./dir1/dir1.2/file1.txt', './dir2/file1.txt'); console.log('path changed'); } catch (err) { console.log(err); }

Example

Now let's asynchronously set the new file path viathen:

import fs from 'fs/promises'; fs.link('./dir1/dir1.2/file1.txt', './dir2/file1.txt').then(() => { console.log('path changed'); }).catch((err) => { console.log(err); });

Example

Let's set a new file path asynchronously via a callback:

import fs from 'fs'; fs.link('./dir1/dir1.2/file1.txt', './dir2/file1.txt', err => { if (!err) { console.log('path changed'); } else { console.log(err); } });

Example

Now let's synchronously set the new file path vialinkSync:

import fs from 'fs'; try { fs.linkSync('./dir1/dir1.2/file1.txt', './dir2/file1.txt'); console.log('path changed'); } catch (err) { console.log(err); }
