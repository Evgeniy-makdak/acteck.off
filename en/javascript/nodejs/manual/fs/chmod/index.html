chmod method

Methodchmodchanges access rights to a file or folder. In the first parameter we indicate the file we need, in the second parameter - the access mode, in the third optional parameter - the callback.

Types of access modes

Constant Code Description
fs.constants.S_IRUSR 0o400 Readable by owner only.
fs.constants.S_IWUSR 0o200 Entry by owner only.
fs.constants.S_IXUSR 0o100 Extract/search by file owner.
fs.constants.S_IRGRP 0o40 Group reading.
fs.constants.S_IWGRP 0o20 Group recording.
fs.constants.S_IXGRP 0o10 Extract/search by group.
fs.constants.S_IROTH 0o4 Reading by other users.
fs.constants.S_IWOTH 0o2 Recording by other users.
fs.constants.S_IXOTH 0o1 Retrieve/search by other users.

Syntax

fs.chmod(файл, режим доступа, [коллбэк]);

Example

Let's get throughawaitAsynchronously set permissions for the file to be read only by the owner:

import fs from 'fs/promises'; try { await fs.chmod('file.txt', 0o400); console.log('read file by owner'); } catch (err) { console.log(err); }

Example

Now let's get throughthenAsynchronously set the permissions for the file to be written only by the owner:

import fs from 'fs/promises'; fs.chmod('file.txt', 0o200).then(() => { console.log('write by owner'); }).catch((err) => { console.log(err); });

Example

Let's asynchronously set up read access for a file for a group of users via a callback:

import fs from 'fs'; fs.chmod('file.txt', 0o40, err => { if (!err) { console.log('read by group'); } else { console.log(err); } });

Example

Now let's get throughchmodSyncsynchronously establish access to file recording by a group of users:

import fs from 'fs'; try { fs.chmodSync('file.txt', 0o20); console.log('write by group'); } catch (err) { console.log(err); }
