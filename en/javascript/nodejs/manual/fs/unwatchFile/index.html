unwatchFile method

MethodunwatchFilestops monitoring changes in the file. In the first parameter we indicate the name of the file we need or set the path to it. In the second and third optional parameters we specify the listener and callback, respectively.

Syntax

fs.unwatchFile(имя файла или путь к нему, [слушатель], [коллбэк]);

Example

Let's asynchronously stop observing changes to a file through a callback. First we need to use the methodwatchFilestart monitoring the file, then make changes to the file and only then stop monitoring:

import fs from 'fs'; fs.watchFile('file.txt', () => { console.log('file watched'); }); setTimeout(() => fs.writeFileSync('file.txt', 'text'), 1000); setTimeout(() => { fs.unwatchFile('file.txt'); console.log('file has been stopped watching'); }, 7000);
