symlink method

Methodsymlinkcreates a symbolic link. The first parameter of the method specifies the path from which we want to create a symbolic link. In the second parameter we set a new path for our symbolic link. In the third optional parameter we indicate the type, which can have the following values:'dir', 'file'or'junction'. In the fourth optional parameter we set the callback. In order for the methodsymlinkworked correctly, you need to run the terminal as an administrator.

Syntax

fs.symlink(исходный путь, конечный путь, [тип], [коллбэк]);

Example

Let's say we have the following file structure:

  • /dir/
    • file.txt

Let's create a symbolic link asynchronously viaawait:

import fs from 'fs/promises'; try { await fs.symlink('./dir/file.txt', './dir/symlink', 'file'); console.log('symlink created'); } catch (err) { console.log(err); }

Example

Now let's create a symbolic link asynchronously viathen:

import fs from 'fs/promises'; fs.symlink('./dir/file.txt', './dir/symlink', 'file').then(() => { console.log('symlink created'); }).catch((err) => { console.log(err); });

Example

Let's create a symbolic link asynchronously via a callback:

import fs from 'fs'; fs.symlink('./dir/file.txt', './dir/symlink', 'file', err => { if (!err) { console.log('symlink created'); } else { console.log(err); } });

Example

Now let's synchronously create a symbolic link viasymlinkSync:

import fs from 'fs'; try { fs.symlinkSync('./dir/file.txt', './dir/symlink', 'file'); console.log('symlink created'); } catch (err) { console.log(err); }
