realpath method

Methodrealpathdetermines the actual location of the path. In the first parameter we indicate the source path to the file or folder, in the second optional parameter - the encoding (by default'utf-8'). In the third optional parameter we specify the callback.

Syntax

fs.realpath(путь к файлу, [кодировка], [коллбэк]);

Example

Let's say we have the following file structure:

  • /dir/
    • file.txt

Let's asynchronously determine the actual location of the file pathfile.txtthroughawait:

import fs from 'fs/promises'; try { let data = await fs.realpath('./dir/file.txt'); console.log(data); } catch (err) { console.log(err); }

Example

Now let's asynchronously find out the location of the file pathfile.txtthroughthen:

import fs from 'fs/promises'; fs.realpath('./dir/file.txt').then(data => { console.log(data); }).catch((err) => { console.log(err); });

Example

Let's asynchronously find out the location of the file path from the previous example via a callback:

import fs from 'fs'; fs.realpath('./dir/file.txt', (err, resolvedPath) => { if (!err) { console.log(resolvedPath); } else { console.log(err); } });

Example

Now let's synchronously find out the location of the file path viarealpathSync:

import fs from 'fs'; try { let data = fs.realpathSync('./dir/file.txt'); console.log(data); } catch (err) { console.log(err); }
