Метод lstat

Метод lstat возвращает статистическую информацию о файлах, папки и символической ссылке в виде объекта. В первом параметре метода указывается путь к нужному нам файлу, папки или символической ссылке. Во втором и третьем необязательных параметрах - объект с опциями и коллбэк.

Синтаксис

fs.lstat(путь к файлу, [опции], [коллбэк]);

Объект с опциями

Опция Описание
bigint Если true, задает отображение числовых значений в формате bigint. Значение по умолчанию false.

Пример

Давайте асинхронно узнаем статистическую информацию о файле через коллбэк:

import fs from 'fs'; fs.lstat('./file.txt', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Пример

Давайте синхронно прочитаем статистическую информацию о файловом дескрипторе через lstatSync:

import fs from 'fs'; try { let data = fs.lstatSync('./file.txt'); console.log(data); } catch (err) { console.log(err); }

  • метод fstat,
