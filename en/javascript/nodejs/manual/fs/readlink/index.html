readlink method

Methodreadlinkreads the contents of a symbolic link. The first parameter of the method specifies the path of the symbolic link. In the second optional parameter we set the encoding (by default it is equal to'utf8'), in the third optional parameter we specify the callback.

Syntax

fs.readlink(путь символической ссылки, [кодировка], [коллбэк]);

Example

Let's say we have the following file structure:

  • /dir/
    • file.txt

Let's first create a symbolic link asynchronously and then read it throughawait:

import fs from 'fs/promises'; try { await fs.symlink('./dir/file.txt', './dir/symlink', 'file'); console.log('symlink created'); } catch (err) { console.log(err); } try { let data = await fs.readlink('./dir/symlink'); console.log(data); } catch (err) { console.log(err); }

Result of executing the code:

'.\dir\file.txt'

Example

Now let's asynchronously viathenLet's read the symbolic link created in the previous example:

import fs from 'fs/promises'; fs.readlink('./dir/symlink').then((data) => { console.log(data); }).catch((err) => { console.log(err); });

Example

Now let's read the symbolic link asynchronously via a callback:

import fs from 'fs'; fs.readlink('./dir/symlink', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Now let's synchronously read the symbolic link viareadlinkSync:

import fs from 'fs'; try { let data = fs.readlinkSync('./dir/symlink'); console.log(data); } catch (err) { console.log(err); }
