cp method

Methodcpcopies the folder structure, including subfolders and files. In the first parameter of the method we indicate the path to the source folder, in the second parameter - the path to the folder to which we are copying, in the third parameter - an object with options. In the fourth optional parameter you can specify a callback.

Syntax

fs.cp(исходная папка, куда копируем, опции, коллбэк);

Object with options

Option Description
recursive Iftrue, copies the folder recursively, that is, with all its contents.

Example

Let's say we have the following file structure:

  • /dir1/
    • /dir1.2/
      • file1.txt
      • file2.txt
  • /dir2/
      • file3.txt

Example

Let's copy the folder asynchronouslydir1throughawait:

import fs from 'fs/promises'; try { await fs.cp('./dir1', './dir2', { recursive: true }); console.log('directory copied'); } catch (err) { console.log(err); }

Example

Now let's asynchronously copy the folder from the previous example viathen:

import fs from 'fs/promises'; fs.cp('./dir1', './dir2', { recursive: true }).then(data => { console.log('directory copied'); }).catch(err => { console.log(err); });

Example

Let's copy the folder asynchronously via a callback:

import fs from 'fs'; fs.cp('./dir1', './dir2', { recursive: true }, err => { if (!err) { console.log('directory copied'); } else { console.log(err); } });

Example

Now let's synchronously copy the folder viacpSync:

import fs from 'fs'; try { fs.cpSync('./dir1', './dir2', { recursive: true }); console.log('directory copied'); } catch (err) { console.log(err); }
