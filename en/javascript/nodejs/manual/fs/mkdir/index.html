mkdir method

Methodmkdircreates a new folder. In the first parameter we indicate the path to the file, in the second optional parameter - an object with options. In the third optional parameter we specify the callback.

Syntax

fs.mkdir(путь к папке, [опции], [коллбэк]);

Object with options

Option Description
recursive Iftrue, creates a folder recursively, that is, with all the contents.
mode Access mode. Default value0o777. Not supported on Windows OS.

Example

Let's make a new folder asynchronously viaawait:

import fs from 'fs/promises'; try { await fs.mkdir('./dir1/'); console.log('directory created'); } catch (err) { console.log(err); }

Example

Now let's make a new folder asynchronously viathen:

import fs from 'fs/promises'; fs.mkdir('./dir1/').then(() => { console.log('directory created'); }).catch((err) => { console.log(err); });

Example

Let's create a new folder asynchronously via a callback:

import fs from 'fs'; fs.mkdir('./dir1/', err => { if (!err) { console.log('directory created'); } else { console.log(err); } });

Example

Now let's create a new folder asynchronously viamkdirSync:

import fs from 'fs'; try { fs.mkdirSync('./dir1/'); console.log('directory created'); } catch (err) { console.log(err); }
