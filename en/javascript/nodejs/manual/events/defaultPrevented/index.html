defaultPrevented property

PropertydefaultPreventedchecks whether the given event is canceled. If the event can be canceled, the property returns a boolean valuetrue, otherwise -false.

Syntax

event.defaultPrevented;

Example

Let's find out if the click on the button is cancelled:

<button>Click</button> let button = document.querySelector('button'); button.addEventListener('click', function(event) { console.log(event.defaultPrevented); });

Result of the executed code:

false
