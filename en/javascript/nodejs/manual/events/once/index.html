Once method

Methodoncecreates a promise that is executed when the class objectEventEmittergenerates this event. A promise can be rejected ifEventEmittergenerates an error while waiting. The first parameter of the method specifies the class objectEventEmitter, in the second parameter - the name of the event, in the third optional parameter - you can set options.

Syntax

once(объект класса EventEmitter, название события, [опции]);

Example

Let's create a promise using the methodonce:

import { EventEmitter, once } from 'events'; const eventEmitter = new EventEmitter(); once(eventEmitter, 'event') .then(() => console.log('text')) .catch((err) => console.error(err.message)); eventEmitter.emit('event');

Result of the executed code:

'text'
