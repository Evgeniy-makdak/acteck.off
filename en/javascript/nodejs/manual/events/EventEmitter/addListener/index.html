addListener method

MethodaddListenermoduleEventEmitteradds listeners for the event specified in the first parameter of the method. In the second parameter of the method we set the listener, which is the callback. The new listener is added to the end of the listener array. The method works similar to the methodaddListener.

Syntax

eventEmitter.addListener(название события, слушатель);

Example

Let's create a listener:

import { EventEmitter } from ':events'; const eventEmitter = new EventEmitter(); eventEmitter.addListener('event', function listener() { console.log('text'); }); console.log(eventEmitter.listenerCount('event'));
