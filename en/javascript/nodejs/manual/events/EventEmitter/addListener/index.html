Метод addListener

Метод addListener модуля EventEmitter добавляет слушателей для события, указанного в первом параметре метода. Во втором параметре метода задаем слушателя, который является коллбэком. Новый слушатель добавляется в конец массива слушателей. Метод работает аналогично методу addListener.

Синтаксис

eventEmitter.addListener(название события, слушатель);

Пример

Давайте создадим слушателя:

import { EventEmitter } from ':events'; const eventEmitter = new EventEmitter(); eventEmitter.addListener('event', function listener() { console.log('text'); }); console.log(eventEmitter.listenerCount('event'));

Смотрите также

  • метод on,
    который добавляет слушателя для события в конец массива слушателей
  • метод prependListener,
    который добавляет слушателя для события в начало массива слушателей
  • метод once,
    который однократно добавляет слушателя события в конец массива
  • метод prependOnceListener,
    который однократно добавляет слушателя события в начало массива
  • метод removeAllListeners,
    который удаляет всех слушателей для события
  • метод removeListener,
    который удаляет одного слушателя для события
