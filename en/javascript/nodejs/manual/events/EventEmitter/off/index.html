off method

MethodoffmoduleEventEmitterRemoves one listener for the event specified in the first parameter of the method. In the second parameter of the method, we set the listener as a callback. The method works similar to the methodremoveListener.

Syntax

eventEmitter.off(название события, слушатель);

Example

Let's remove one event listener'event1':

import { EventEmitter } from 'node:events'; const eventEmitter = new EventEmitter(); let listener1 = () => { console.log('text1'); }; let listener2 = () => { console.log('text2'); }; eventEmitter.on('event1', listener1); eventEmitter.on('event2', listener2); eventEmitter.off('event1', listener1); console.log(eventEmitter.listenerCount('event1'));

Result of the executed code:

0
