EventEmitter constructor

ConstructorEventEmittercreates an event for listeners. Nothing is specified in the method parameter.

Syntax

let eventEmitter = new EventEmitter();

Example

Let's create a new event for the listener:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener() { console.log('text'); });
