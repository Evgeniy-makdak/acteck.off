listenerCount method

MethodlistenerCountmoduleEventEmitterreturns the number of listeners listening to the specified event. In the first parameter of the method we indicate the name of the event, in the second optional parameter - the listener. If a listener is specified, its occurrence count will also be returned.

Syntax

eventEmitter.listenerCount(название события, [слушатель]);

Example

Let's find out the number of listeners:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event1', function listener1() { console.log('text1'); }); eventEmitter.on('event1', function listener2() { console.log('text2'); }); eventEmitter.on('event2', function listener2() { console.log('text3'); }); console.log(eventEmitter.listenerCount('event1'));

Result of the executed code:

2
