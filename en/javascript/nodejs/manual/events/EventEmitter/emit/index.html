emit method

MethodemitmoduleEventEmittersynchronously calls listeners for a specific event in the order in which they were registered. In the first parameter of the method we set the name of the event, in the second optional parameter we pass values ​​for the listeners. If the event we need has listeners, the method returnstrue, otherwise -falseor an empty string.

Syntax

eventEmitter.emit(имя события, [значения для слушателей]);

Example

Let's check, call the event listeners and pass values ​​to one of them in the parameters:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener1() { console.log('text1'); }); eventEmitter.on('event', function listener2(num1, num2) { console.log('text2', num1 + num2); }); eventEmitter.emit('event', 1, 2);

Result of the executed code:

'text1' 'text2' 3

Example

Now let's call a listener for a non-existent event:

import { EventEmitter } from 'node:events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event1', function listener1() { console.log('text1'); }); eventEmitter.emit('event');

After executing the code, we will see an empty line:

''
