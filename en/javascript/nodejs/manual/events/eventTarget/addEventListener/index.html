addEventListener method

ConstructoraddEventListenerobjectEventTargetadds a listener for the event. The first parameter of the method specifies the event type, and the second parameter specifies the listener. In the third optional parameter we specify an object with options.

Syntax

eventTarget.addEventListener(тип события, слушатель, [опции]);

Object with options

Option Description
once If specifiedtrue, then immediately after adding the listener is removed (default valuefalse).
passive If specifiedtrue, then the listener will not be able to call the methodpreventDefaultfor the event, (default valuefalse).
capture Not directly used by NodeJS, but included in the options stack (default valuefalse).
signal Indicates that the listener will be removed when the method is calledabort.

Example

Let's add a listener to our event:

let listener = () => { console.log('text'); }; const target = new EventTarget(); target.addEventListener('event', listener);
