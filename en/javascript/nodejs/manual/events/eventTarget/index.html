Конструктор EventTarget

Конструктор EventTarget создает события и слушателей для них. В параметре метода ничего не указывается.

Синтаксис

let eventTarget = new EventTarget();

Пример

Давайте создадим новое событие для слушателя:

let eventTarget = new EventTarget(); console.log(eventTarget);

В полученном коде мы увидим пустой объект EventTarget:

EventTarget {}
