devNull property

PropertydevNullreturns the path to the null device file depending on the platform used.Null or empty deviceis a special file in UNIX-class systems, writing to which always occurs successfully, regardless of the amount of information written. For Windows meaningdevNullequals'\.\nul', for POSIX -'/dev/null'.

Syntax

os.devNull;

Example

Let's find out the path to the zero device file of the platform we are using:

import os from 'os'; console.log(os.devNull);
