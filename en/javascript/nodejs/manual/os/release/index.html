Метод release

Метод release возвращает информацию о версии операционной системы в виде строчного значения.

Синтаксис

os.release();

Пример

Давайте узнаем версию операционной системы:

import os from 'os'; console.log(os.release());

Смотрите также

  • метод platform,
    который возвращает информацию о платформе операционной системы
  • метод type,
    который возвращает имя операционной системы
  • метод arch,
    который возвращает архитектуру процессора операционной системы
