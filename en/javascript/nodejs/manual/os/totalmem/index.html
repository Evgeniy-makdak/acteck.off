Метод totalmem

Метод totalmem возвращает общий объем системной памяти в байтах.

Синтаксис

os.totalmem();

Пример

Давайте узнаем общий объем системной памяти нашей машины (ПК):

import os from 'os'; console.log(os.totalmem());

Смотрите также

  • метод freemem,
    который возвращает свободную память
  • метод endianness,
    который возвращает эндианальность процессора
  • метод loadavg,
    который возвращает средние значения нагрузки системы
