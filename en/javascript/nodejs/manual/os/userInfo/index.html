Метод userInfo

Метод userInfo возвращает информацию о текущем действующем пользователе в виде объекта. Для Windows возвращаемый объект содержит следующие ключи uid, gid, username, homedir, shell. В необязательном параметре метода можно задать объект с опциями.

Синтаксис

os.userInfo([опции]);

Объект с опциями

Опция Описание
encoding Режим кодировки данных. Значение по умолчанию 'utf8'.

Пример

Давайте узнаем информацию о текущем действующем пользователе:

import os from 'os'; console.log(os.userInfo());

Смотрите также

  • свойство machine,
    которое возвращает тип машины (ПК)
  • метод platform,
    который возвращает информацию о платформе операционной системы
  • метод release,
    который возвращает информацию о версии операционной системы
  • метод type,
    который возвращает имя операционной системы
