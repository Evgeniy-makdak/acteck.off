Machine property

Propertymachinereturns the user's machine or PC type as a string value. Possible values ​​include'arm', 'arm64', 'aarch64', 'mips', 'ppc64', 'ppc64le', 's390', 's390x', 'i386', 'i686', 'x86_64'.

Syntax

os.machine();

Example

Let's find out the type of machine we use:

import os from 'os'; console.log(os.machine());
