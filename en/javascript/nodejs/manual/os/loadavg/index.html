loadavg method

Methodloadavgreturns average system load values ​​for1, 5And15minutes. The load average is calculated by the operating system and expressed as a fraction. Load average is a concept unique to the Unix operating system. On Windows the return value is always[0, 0, 0].

Syntax

os.loadavg();

Example

Let's find out the average load values ​​of our operating system:

import os from 'os'; console.log(os.loadavg());
