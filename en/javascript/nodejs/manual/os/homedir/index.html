Метод homedir

Метод homedir возвращает путь к домашнему каталогу текущего пользователя в виде строки.

Синтаксис

os.homedir();

Пример

Давайте узнаем путь к нашему домашнему каталогу:

import os from 'os'; console.log(os.homedir());

Смотрите также

  • свойство devNull,
    которое возвращает путь к файлу нулевого устройства
  • метод userInfo,
    который возвращает информацию о текущем пользователе
