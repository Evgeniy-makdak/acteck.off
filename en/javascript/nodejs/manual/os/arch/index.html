arch method

Methodarchreturns the architecture of the operating system processor for which the current NodeJS file is compiled. Return values ​​can be as follows:'x32', 'x64', 'arm', 'arm64', 's390', 's390x', 'mipsel', 'ia32', 'mips', 'ppc', 'ppc64'.

Syntax

os.arch();

Example

Let's find out the processor architecture of our operating system:

import os from 'os'; console.log(os.arch());
