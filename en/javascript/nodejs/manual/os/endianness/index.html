Метод endianness

Метод endianness возвращает эндианальность процессора, для которого скомпилирован текущий Node.js файл. Возможные значения данного метода включают 'BE для big endian и 'LE для little endian.

Синтаксис

os.endianness();

Пример

Давайте узнаем эндианальность нашего процессора:

import os from 'os'; console.log(os.endianness());
