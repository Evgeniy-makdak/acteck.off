Type method

Methodtypereturns the operating system name as a string. For Linux the method will return'Linux', for macOS -'Darwin', for Windows -'Windows_NT'.

Syntax

os.type();

Example

Let's find out the name of our operating system:

import os from 'os'; console.log(os.type());

    which returns the machine type (PC)
  • methodplatform,
    which returns information about the operating system platform

    which returns information about the current user
