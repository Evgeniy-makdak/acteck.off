getPriority method

MethodgetPriorityreturns the scheduling priority for the process specifiedpidis an optional parameter specifying the ID of the process whose scheduling priority should be returned. Its default value is0. If the method parameter has the valuepidnot specified or equal0, then the methodgetPriorityreturns the priority of the current process.

Syntax

os.getPriority([pid]);

Example

Let's find out the default scheduling priority for our process:

import os from 'os'; console.log(os.getPriority());
