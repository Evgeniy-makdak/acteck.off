EOL property

PropertyEOLshows the end-of-line character or marker specific to the operating system. For Windows propertyos.EOLreturns'\r\n', for Linux -'\n', for Mac -'\n\r'.

Syntax

os.EOL;

Example

Let's find out the end of line character of the operating system we are using:

import os from 'os'; console.log(os.EOL);

See also

  • methodplatform,
    which returns information about the operating system platform
  • methodtype,
    which returns the operating system name
