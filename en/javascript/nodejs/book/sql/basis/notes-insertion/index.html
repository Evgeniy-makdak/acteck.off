Inserting records via SQL query in NodeJS

Let's now learn how to add new records to the table. This is done using the commandINSERT INTO. It has the following syntax:

let query = "INSERT INTO таблица (поле1, поле2...) VALUES (значение1, значение2...)";

Let's go to our tableusersLet's add a new user. Also note that no column is specified when insertingidand its meaning. And this is correct, since the value of this column will be entered automatically by the base:

let query = "INSERT INTO users (name, age, salary) VALUES ('user', 30, 1000)";

After successful execution of the code, the following information will be displayed:

ResultSetHeader { fieldCount: 0, affectedRows: 1, insertId: 7, info: '', serverStatus: 2, warningStatus: 0, changedRows: 0 }
