Grouping result in NodeJS

To group the result obtained from the database, use the commandGROUP BY.

Let's group users by age so that the minimum salary is displayed for each group:

let query = "SELECT age, MIN(salary) as min FROM users GROUP BY age";

Since two users have the same age23after executing the code it will display5groups. At the same time, opposite the group23Only the minimum salary will be noted400:

[ { age: 23, min: 400}, { age: 25, min: 500}, { age: 30, min: 900}, { age: 27, min: 500}, { age: 28, min: 900} ]
