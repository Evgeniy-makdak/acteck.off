Отсутствие столбцов при вставке через SQL запрос в NodeJS

Что будет, если не указать значение какого-либо столбца? Давайте, например, укажем только имя и возраст:

let query = "INSERT INTO users (name, age) VALUES ('user', 30)";

В таком случае неуказанные столбцы возьмут значение по умолчанию. Если такое значение не задано, то это приведет к ошибке и такой запрос не выполнится:

Error: Field 'salary' doesn't have a default value

Добавьте нового юзера 'xxxx', не указав ему возраст и зарплату.
