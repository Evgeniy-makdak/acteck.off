Missing columns when inserting via SQL query in NodeJS

What happens if you don't specify the value of a column? Let's, for example, specify only the name and age:

let query = "INSERT INTO users (name, age) VALUES ('user', 30)";

In this case, unspecified columns will take the default value. If such a value is not set, this will result in an error and such a request will not be executed:

Error: Field 'salary' doesn't have a default value
