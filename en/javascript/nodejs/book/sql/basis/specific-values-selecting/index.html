Fetching records by specific values ​​in NodeJS

If you need to select a record from any group of field values, you can use the commandIN. К примеру, можно выбрать записи, у которых полеnamematters'user1', 'user3'or'user4'. It will look like this:WHERE id IN ('user1', 'user3', 'user4').

In principle, for such things you can use the commandOR, then the first example will look like this:WHERE id=1 OR id=3 OR id=7 OR id=14 OR id=28. Option withINIt still looks simpler and more elegant.

Example

Let's select users whose fieldidmatters1, 3, 5:

let query = "SELECT * FROM users WHERE id IN (1, 3, 5)";

Result of the executed code:

[ { id: 1, name: 'user1', age: 23, salary: 400 }, { id: 3, name: 'user3', age: 23, salary: 500 }, { id: 5, name: 'user5', age: 27, salary: 500 } ]

Example

TeamINcan be combined with other commands in the request. Let's rewrite the previous example, indicating that the age of the users should be more than23years:

let query = "SELECT * FROM users WHERE id IN (1, 3, 5) AND age>23";

Result of the executed code:

[ { id: 5, name: 'user5', age: 27, salary: 500 } ]
