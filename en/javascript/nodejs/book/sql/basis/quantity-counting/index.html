Counting quantities via SQL query in NodeJS

Using the commandCOUNTyou can count the number of rows in the sample. The asterisk character specified in the command parameter means that all records from the table are selected.

Example

Let's, for example, count all the users in the table:

let query = "SELECT COUNT(*) FROM users";

Result of the executed code:

[ { 'COUNT(*)': 6 } ]

Example

Now let's count everyone whose salary is equal900:

let query = "SELECT COUNT(*) FROM users WHERE salary=900";

Result of the executed code:

[ { 'COUNT(*)': 2 } ]

Example

You can also specify in the command parameterCOUNTfield name. And the command will count all records for which the given field is not equalNULL:

let query = "SELECT COUNT(salary) FROM users";

Result of the executed code:

[ { 'COUNT(salary)': 6 } ]
