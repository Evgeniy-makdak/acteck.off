Удаление записей через SQL запрос в NodeJS

С помощью команды DELETE можно удалять записи из таблицы. Ее синтаксис похож на изученное вами ранее:

let query = "DELETE FROM таблица WHERE условие";

Будьте внимательны, если не указать в запросе блок с WHERE, то удалятся все записи!
