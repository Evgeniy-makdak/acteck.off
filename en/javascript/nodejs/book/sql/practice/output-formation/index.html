Design of outputting records from a database in NodeJs

To work with data, it is important not only to be able to obtain it from the database, but also to output it into HTML markup. As a rule, this is done using a template engine. Let's take Handlebars as an example.

So let's print out the records of our test tableusersin the following form:

<p> <b>user1</b> <b>23</b> <b>400</b> </p> <p> <b>user2</b> <b>24</b> <b>500</b> </p> <p> <b>user3</b> <b>25</b> <b>600</b> </p>

To begin with, in the fileapp.jsLet's write a route that will render the Handlebars template engine file. It will contain information about all users, so let's call itusers.hbs. Next in the route we will declare a variablequery, which will store an SQL query to display all users from the table. Then we unpack the query result into two variables:resultsAndfields. And to the presentation fileusers.hbspass the variableresults:

app.get('/', async function(req, res) { let query = "SELECT * FROM users"; let [results, fields] = await connection.query(query); res.render('users', {results: results}); });

In variableresultsan array of objects is stored:

[ {id: 1, name: 'user1', age: 23, salary: 400}, {id: 2, name: 'user2', age: 24, salary: 500}, {id: 3, name: 'user3', age: 25, salary: 600} ]

Sinceresultsis an array, all its objects can be displayed in a view using a loop. To do this, a special command is written in double curly braces#each. To get the values ​​of each object, you need to specify the corresponding keys in the body of the loop:

<p> {{#each results}} <br>name: <b>{{ name }}</b> <br>age: <b>{{ age }}</b> <br>salary: <b>{{ salary }}</b> {{/each}} </p>

Output the records of our table in the following form:

<div> <h2>user1</h2> <p> 23 years, <b>400$</b> </p> </div> <div> <h2>user2</h2> <p> 24 years, <b>500$</b> </p> </div> <div> <h2>user3</h2> <p> 25 years, <b>600$</b> </p> </div>

Output the records of our table in the following form:

<table> <tr> <th>id</th> <th>name</th> <th>age</th> <th>salary</th> </tr> <tr> <td>1</td> <td>user1</td> <td>23</td> <td>400</td> </tr> <tr> <td>2</td> <td>user2</td> <td>25</td> <td>500</td> </tr> <tr> <td>3</td> <td>user3</td> <td>23</td> <td>500</td> </tr> </table>

Output the records of our table in the following form:

<ul> <li>user1</li> <li>user2</li> <li>user3</li> </ul>
