Connecting to a MySQL database using Open Server Panel

To develop applications on Windows OS, you can (not necessarily) use special software -Open Server Panel(OSP). OSP includes a set of necessary server software, including a MySQL database.

It is recommended to install Open Server Panel (OSP) according todocumentation. After installing OSP, you can select the desired version of MySQL in the plug-in settings.
