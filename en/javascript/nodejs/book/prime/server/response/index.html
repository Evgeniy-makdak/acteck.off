Setting up a server response in NodeJS

Using propertystatusCodewe specify the HTTP response code. Typically this is the code200, telling the browser that everything is ok and the page has been found:

http.createServer((request, response) => { response.statusCode = 200; response.write('hello world'); response.end(); });

Change the response code to404. As the page text, output the text that the page was not found.

HTTP headers

Using the methodsetHeaderHTTP headers can be sent. For example, let's send the headerContent-Language:

http.createServer((request, response) => { response.setHeader('Content-Language', 'ru'); response.statusCode = 200; response.write('hello world'); response.end(); });

Submit titleCache-Controlwith meaning'no-cache'.

Send HTML

For now, however, the browser treats our response as normal (plain) text. Let's make our response interpreted as HTML. To do this, use the methodsetHeadersend the corresponding HTTP header:

http.createServer((request, response) => { response.setHeader('Content-Type', 'text/html'); response.statusCode = 200; response.write('<b>hello world</b>'); response.end(); }).listen(3000);

Make sure that when requested, your server gives the current time, formatted in some tag.

Response code

The response code and headers can be combined in one methodwriteHead. This method takes the response code as the first parameter, and an object with headers as the second:

http.createServer((request, response) => { response.writeHead(200, {'Content-Type': 'text/html'}); response.end(); });

The following code is given:

http.createServer((request, response) => { response.setHeader('Content-Type', 'text/plain'); response.statusCode = 404; response.write('page not found'); response.end(); }).listen(3000);

Simplify it through a methodwriteHead.

The following code is given:

http.createServer((request, response) => { response.setHeader('Content-Type', 'text/html'); response.setHeader('Content-Language', 'ru'); response.statusCode = 200; response.write('{}'); response.end(); }).listen(3000);

Simplify it through a methodwriteHead.
