Checking file existence in NodeJS

Let's learn how to check the existence of a file or folder. In modern NodeJS this is done in a special way through the methodfs.access, which checks whether the file can be accessed.

First, let's import the asynchronous part of the modulefs:

import fs from 'fs/promises';

Now from the general part of the modulefsimport special constants:

import { constants } from 'fs';

After this in the variableconstantsthere will be a special object with constants containing special flags, which we will pass as a parameter to the methodaccess, changing his behavior.

The following flag is used to check if a file exists:

constants.F_OK

Let's use this flag to check the existence of the filetest.txt:

fs.access('test.txt', constants.F_OK);

Our methodaccessreturns a promise as its result. If the file exists, but the promise will be resolved, and if not, it will be rejected:

fs.access('test.txt', constants.F_OK).then(() => { console.log('file exists'); }).catch(() => { console.error('file does not exists'); });

Let's rewrite our code viaawait. To do this, let's turn the callaccessinto the designtry-catch:

try { await fs.access('test.txt', constants.F_OK); console.log('file exists'); } catch { console.error('file does not exists'); }

Check if the file existstest.txt. If it exists, read its text.

Other flags

In our methodaccessOther flags can also be passed. For example, the following flag is used to check the read permission of a file:

constants.R_OK

And the following flag is used to check the file's write permission:

constants.W_OK

Let's check the file for readability:

try { await fs.access('test.txt', constants.R_OK); console.log('can read'); } catch { console.error('cannot read'); }

Now let’s check the file for writability:

try { await fs.access('test.txt', constants.W_OK); console.log('can write'); } catch { console.error('cannot write'); }

Now let’s check the file for both:

try { await fs.access('test.txt', constants.R_OK | constants.W_OK); console.log('can access'); } catch { console.error('cannot access'); }
