Asynchronous work with fs via then in NodeJS

With module methodsfsYou can work asynchronously not only through callbacks, but also through promises. There is a special property for thispromises, containing promise analogues of methods for working with the file system. For example, for the methodfs.readFileits promise analogue will befs.promises.readFile.

Reading files

Let's output the contents of a file to the console:

fs.promises.readFile('readme.txt', 'utf8').then(data => { console.log(data); });

Let the file contain a number. Read this file and print the sum of the digits of this number to the console.

Exception Handling

Let's now add exception handling:

fs.promises.readFile('readme.txt', 'utf8').then(data => { console.log(data); }).catch(err => { console.log('ошибка'); });

Try reading a non-existent file. Make sure that this throws an exception. Modify your code to handle this situation.

Read and write

You can read a file, do something with its text, and then write it back:

fs.promises.readFile('readme.txt', 'utf8').then(data => { return fs.promises.writeFile('readme.txt', data + '!'); }).catch(err => { console.log('ошибка'); });

Let the numbers be written in the file separated by commas. Make a script that will write each of these numbers to a separate file.

Mass work

Let's say we have several files. Let's read these files, merge their text into one line and write it to a new file.

Unlike callbacks, in this case we do not need to read files one by one. When working with promises, we can write all promises for reading files into an array, and then usePromise.allto write to a file only when all files have been read.

Let's do this. Let us have the file names in the form of an array:

let names = ['1.txt', '2.txt', '3.txt'];

Let's start a loop in which we will read files, writing promises with the results to an array:

let names = ['1.txt', '2.txt', '3.txt']; let files = []; for (let name of names) { files.push(fs.promises.readFile(name, 'utf8')); } console.log(files); // массив промисов

Given such an array, we can callthenonly once, when all promises are fulfilled:

Promise.all(files).then(data => { fs.promises.writeFile('res.txt', data.join('')); });

Let's add exception handling:

Promise.all(files).then(data => { fs.promises.writeFile('res.txt', data.join('')); }).catch(err => { console.log('что-то пошло не так'); });

Let's put all our code together:

let names = ['1.txt', '2.txt', '3.txt']; let files = []; for (let name of names) { files.push(fs.promises.readFile(name, 'utf8')); } Promise.all(files).then(data => { fs.promises.writeFile('res.txt', data.join('')); }).catch(err => { console.log('что-то пошло не так'); });

Let you have5files with numbers. Find the sum of these numbers and write it to a new file.
