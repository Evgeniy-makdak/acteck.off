Sending a file as a response in Express

In previous lessons, we provided some text upon request, and we wrote this text directly in the request handler function:

app.get('/page/', function(req, res) { res.send('page'); });

In real life, this is, of course, inconvenient, and files will be returned as a response. Let, for example, we have a filepage.html, which we want to give to some address.

This can be done using the methodsendFileresponse object. However, there are nuances here. Relative file path won't work (for some reason):

app.get('/page/', function(req, res) { res.sendFile('page.html'); // не будет работать });

Therefore we will have to use an absolute path. Let's do it using__dirname:

import __dirname from './dirname';

Now let's send the file to the browser:

app.get('/page/', function(req, res) { res.sendFile(__dirname + '/page.html'); });

Make sure that along the way/page/1/the file was givenpages/1.html, on the way/page/2/- filepages/2.html, and along the way/page/3/- filepages/3.html.
