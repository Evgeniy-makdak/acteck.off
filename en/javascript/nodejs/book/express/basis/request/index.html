Request object in Express

Now we will understand the capabilities of the request object. To do this, we will assume that the URL entered into the address bar looks like this:

http://localhost:3000/dir/page.html?get1=1&get2=2

Let's write a handler for this URL:

app.get('/dir/page.html', function(req, res) { });

As you can see, in the handler we specify the address without protocol, domain and port, as well as without a query string. Let's see what kind of request information we can get using the request object.

Path

Using propertypathwe can get the requested address without protocol, host, port and query string:

app.get('/dir/page.html', function(req, res) { console.log(req.path); // выведет '/dir/page.html' });

Url

Using propertyurlwe can get the requested address with path and query string:

app.get('/dir/page.html', function(req, res) { console.log(req.url); // выведет '/dir/page.html?get1=1&get2=2' });

Moreover, the propertyurlmay be changed by Express when routing is running. Therefore there is a special propertyoriginalUrl, which is guaranteed to contain the original URL:

app.get('/dir/page.html', function(req, res) { console.log(req.originalUrl); // выведет '/dir/page.html?get1=1&get2=2' });

Query string

Using propertyquerywe can get the query string as an object with keys and values:

app.get('/dir/page.html', function(req, res) { console.log(req.query); // содержит объект {get1: '1', get2: '2'} });

For example, let's get the value of a GET parameter with the nameget1:

app.get('/dir/page.html', function(req, res) { console.log(req.query.get1); // выведет '1' });

Protocol

Using propertyprotocolwe can get the protocol:

app.get('/dir/page.html', function(req, res) { console.log(req.protocol); // выведет 'http' });

There is also a propertysecure, which containstruefor HTTPS protocol, andfalse- for regular HTTP:

app.get('/dir/page.html', function(req, res) { console.log(req.secure); // выведет false });

Headings

Using propertyheaderswe can get the request headers:

app.get('/dir/page.html', function(req, res) { console.log(req.headers); });

Languages

Using the methodacceptsLanguageswe can get the user's preferred languages:

app.get('/dir/page.html', function(req, res) { console.log(req.acceptsLanguages()); });

IP address

Using propertyipwe can get the user's IP address:

app.get('/dir/page.html', function(req, res) { console.log(req.ip); });
