Submitting a form using the GET method to the current page in Express

Let's now make the form submit to the current page. To do this, let's make it emptyactionor remove it completely:

<form action="" method="GET"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Let our form be shown and sent to the following handler:

app.get('/', function(req, res) { res.render('form'); });

Let's output the submitted form data to the console:

app.get('/', function(req, res) { console.log(req.query); res.render('form'); });

Our handler will be executed twice. The first time we go to the page and see the form. In this case, an empty object with form data will be displayed in the console. The second time we will get to the handler after submitting the form and see the entered data in the console.

We display the result near the form

For example, let's show the form on the first entry, and on the second entry the form, and under it - the sum of the entered numbers.

Let's create a handler:

app.get('/', function(req, res) { res.render('form', { sum: +req.query.test1 + +req.query.test2, }); });

Let's modify the code of the file with the form:

<form action="" method="GET"> <input name="test1"> <input name="test2"> <input type="submit"> </form> {{#if sum}} {{ sum }} {{/if}}
