Submitting a form via POST to the current page in Express

Let's now submit our form using the POST method to the current page:

<form method="POST"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

In this case, unlike the GET method, we will have to write two handlers. The first handler will be a methodgetand it will be triggered when a page with a form is entered:

app.get('/', function(req, res) { res.render('form'); });

And the second handler will be the POST method and it will fire after the form is submitted:

app.post('/', function(req, res) { res.send('result'); });

This is a form sent using the POST method. Make sure that the form and its result are on the same URL.
