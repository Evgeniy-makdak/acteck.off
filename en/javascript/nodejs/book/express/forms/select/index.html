Select element in Express

Let's now learn how to work with the tagselect. Let's create a drop-down list like this in our form:

<form action="" method="POST"> <select name="test"> <option>item1</option> <option>item2</option> <option>item3</option> </select> <input type="submit"> </form>

After submitting the form toreq.dataThe select will contain the value of the selected tagoption:

app.post('/', function(req, res) { console.log(req.body.test); // 'item1', 'item2' или 'item3' });

Using a drop-down list, prompt the user to select the country in which he lives.

value attribute

Let's specify an attribute for list itemsvalue:

<form action="" method="POST"> <select name="test"> <option value="1">item1</option> <option value="2">item2</option> <option value="3">item3</option> </select> <input type="submit"> </form>

In this case, exactly the value of this attribute will be sent to the server:

app.post('/', function(req, res) { console.log(req.body.test); // '1', '2' или '3' });
