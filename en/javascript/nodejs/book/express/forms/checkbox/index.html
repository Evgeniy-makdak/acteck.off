Checkbox in Express

Let's now learn how to work with checkboxescheckbox. Let's create a checkbox like this in our form:

<form action="" method="POST"> <input type="checkbox" name="flag"> <input name="text"> <input type="submit" name="submit"> </form>

After submitting the form, the checkbox data will contain the string'on', if the checkbox was checked andundefinedif not:

app.post('/', function(req, res) { console.log(req.body.flag); // 'on' или undefined });

Let's display something on the screen depending on whether the checkbox was checked or not. To do this, let's send our form data to the view:

app.post('/', function(req, res) { res.render('form', { body: req.body, }); });

In the view, we will check whether the checkbox has been checked or not and display the corresponding message about this:

{{#if body.submit}} {{#if body.flag}} отмечен {{else}} не отмечен {{/if}} {{/if}}

Make a form with an input and a checkbox. Using input, ask the user for a name. After submitting the form, if the box was checked, welcome the user, and if it was not checked, say goodbye.
