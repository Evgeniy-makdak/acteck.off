Submitting a form using the GET method in Express

Now we will learn how to receive submitted form data on the server. Let us have the following form:

<form action="/target/" method="GET"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Let's submit this form when accessing the main page of the site:

app.get('/', function(req, res) { res.render('form'); });

Our form will be sent to the address written in the attributeaction. Let's write a handler for this address:

app.get('/target/', function(req, res) { res.send('result'); });

The submitted form will be sent to this handler, and the form data will be contained inreq.queryin the form of an object. In this object, the keys will be the attribute valuesnameinputs from the form, and values ​​- the entered data.

Let, for example, let us enter a number into the first input1, and in the second - the number2. Then the data object will look like this:

app.get('/target/', function(req, res) { console.log(req.query); // выведет {test1: '1', test2: '2'} res.send('result'); });

You can display each value separately:

app.get('/target/', function(req, res) { console.log(req.query.test1); console.log(req.query.test2); res.send('result'); });

You can do something with the entered data and send the result back. Let's find, for example, the sum of the entered numbers:

app.get('/target/', function(req, res) { let num1 = +req.query.test1; let num2 = +req.query.test2; let sum = num1 + num2; res.send('result: ' + sum); });

Make a form with three inputs. After submitting the form, send the sum of the numbers entered in the inputs to the browser.
