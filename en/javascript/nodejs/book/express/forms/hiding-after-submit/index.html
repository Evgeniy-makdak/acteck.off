Hiding the form after sending it to Express

Let's now make sure that the form is hidden after submission, and the result of the submission is displayed instead.

To do this, we will send the form data to the view:

app.get('/', function(req, res) { res.render('form', { query: req.query, }); });

Now in the view, depending on whether the form was submitted, we will either show the form itself or its data:

{{#if query.submit}} {{query.test1}} {{query.test2}} {{else}} <form action="" method="GET"> <input name="test1"> <input name="test2"> <input type="submit" name="submit"> </form> {{/if}}

Use the form to ask for the user's name. After submitting the form, greet the user by name and remove the form.
