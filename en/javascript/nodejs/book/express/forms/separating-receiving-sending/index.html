Separating form receipt and submission in Express

Let's say we have a form that goes to the current page:

<form method="GET"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

Let's write a handler for this form:

app.get('/', function(req, res) { res.render('result'); });

As you already know, in our case the handler will be executed twice: when you first visit the page and when you submit the form. Let's write a condition that separates these two situations inside our handler.

Obviously, to check we need to usereq.query:

app.get('/', function(req, res) { console.log(req.query); // выведет {} или объект с данными res.render('result'); });

There is, however, a problem. INreq.queryThere is always an object - either empty or filled. And objects are always equaltrue. Therefore the following check will not work:

app.get('/', function(req, res) { if (req.query) { res.render('form'); } else { res.render('result'); } });

Let's make a working test.

First option

You can check the presence of each field in an object with the result:

app.get('/', function(req, res) { if (req.query.test1 && req.query.test2) { res.render('form'); } else { res.render('result'); } });

This option, however, has disadvantages. The check we wrote will not miss an empty input value, since it is cast tofalse. Let's fix the problem:

app.get('/', function(req, res) { if (req.query.test1 !== undefined && req.query.test2 !== undefined) { res.render('form'); } else { res.render('result'); } });

However, the method still remains inconvenient, since the check needs to list all the form inputs. It turns out that if we have five inputs, then the condition will need to check all these five inputs. Inconvenient.

Second option

The described inconvenience has a classic solution - you need to give the submit button a name:

<form action="" method="GET"> <input name="test1"> <input name="test2"> <input type="submit" name="submit"> </form>

In this case, we don't need to check each field to see if the form has been submitted. Let's just check the presence of our button:

app.get('/', function(req, res) { if (req.query.submit) { res.render('result'); } else { res.render('form'); } });

Practical problems

Use the form to ask the user for five numbers. After submitting the form, display the sum of these numbers.
