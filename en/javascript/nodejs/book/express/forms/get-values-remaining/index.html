Saving form values ​​after submission

Let's now make sure that after submitting the form, the entered data remains in it. To do this, let's pass the data to the view:

app.get('/', function(req, res) { res.render('form', { query: req.query }); });

And we will display this data invalueinputs:

<form action="" method="GET"> <input name="test1" value="{{query.test1}}"> <input name="test2" value="{{query.test2}}"> <input type="submit"> </form>

Let's figure out how this will work. On first entryquery.test1Andquery.test2will lieundefined. In this case, Handlebars will output empty lines as inputs and we will simply see an empty form. But on the second visit after submitting the form, we will see the previously entered data.

A form is given that asks the user for his name, age and salary. Make sure that after submitting the form, the entered data remains in it.
