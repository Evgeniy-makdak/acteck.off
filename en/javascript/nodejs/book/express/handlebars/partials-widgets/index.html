Handlebars Express Partial Templates

It is often convenient to split the page into parts, which will be placed in separate files. Such blocks are calledwidgetsor in terms of Handlebarspartial templates (partials).

For example, you can include the text of the header, footer, and sidebar. This is convenient, firstly, because the text of these blocks will not clutter up the site layout, making its structure more transparent and understandable. Secondly, there may be several templates on the site. In this case, it will be possible to use the block in several templates at once.

Let's try it in practice. Let's say we have the following site layout:

<!DOCTYPE html> <html> <head> <title>{{titt}}</title> </head> <body> <header> <nav> <a href="/">главная</a> <a href="/about">о нас</a> <a href="/contacts">контакты</a> </nav> </header> <main> {{{body}}} </main> <footer> footer </footer> </body> </html>

Let's put the contents of the header into a separate file in the folderpartials:

<nav> <a href="/">главная</a> <a href="/about">о нас</a> <a href="/contacts">контакты</a> </nav>

In the file with the layout we will connect our header using a special command{{>:

<!DOCTYPE html> <html> <head> <title>{{titt}}</title> </head> <body> <header> {{> header}} </header> <main> {{{body}}} </main> <footer> footer </footer> </body> </html>

Take the website layout and place the header, footer, and sidebar into separate widgets.

Subfolders

In a folderpartialsyou can place subfolders by laying outhbsfiles in them. In this case, in the widget connection command you will need to specify a subfolder in addition to the file name.

Let's look at an example. Let's say you have this structure:

  • views/
    • partials/
      • header/
        • menu.hbs
        • logo.hbs
      • footer
        • contacts.hbs
        • form.hbs

Let's connect our widgets in the layout file:

<!DOCTYPE html> <html> <head> <title>{{titt}}</title> </head> <body> <header> {{> header/logo}} {{> header/menu}} </header> <main> {{{body}}} </main> <footer> footer {{> header/contacts}} {{> header/form}} </footer> </body> </html>
