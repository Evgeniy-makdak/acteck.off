Transferring a title to Handlebars in Express

When page content is connected to a site layout, the title must match that connected page. This can be easily implemented by passing the title as the second parameter of the methodrender, because this data will be available both in the content file and in the general layout.

Let's assign different titles to the three pages:

app.get('/page/1/', function(req, res) { res.render('1', {title: 'title 1'}); }); app.get('/page/2/', function(req, res) { res.render('2', {title: 'title 2'}); }); app.get('/page/2/', function(req, res) { res.render('2', {title: 'title 2'}); });

You can simplify:

app.get('/page/:page/', function(req, res) { res.render(req.params.page, {title: 'title ' + req.params.page}); });

Now let's display our title in the page layout:

<!DOCTYPE html> <html> <head> <title>{{ title }}</title> </head> <body> <header> header </header> <main> {{{body}}} </main> <footer> footer </footer> </body> </html>

Let your website have the following pages:

  • views/
    • index.hbs
    • about.hbs
    • conctacs.hbs
    • price.hbs

Let the titles of your pages be stored in an object whose keys are page addresses, and whose values ​​are their titles:

let titles = { index: 'главная страница', about: 'о нас', conctacs: 'контакты', price: 'наш прайс' }
