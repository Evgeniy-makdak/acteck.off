The with command in Handlebars in Express

Let an object be passed to the view:

app.get('/page/', function(req, res) { res.render('page', { user: {name: 'john', age: 30} }); });

Let's display the parts of this object in a view:

{{user.name}} {{user.age}}

Sometimes it may not be very convenient that for each property of an object you have to write the name of this object. Our team will help make your application easierwith. This command is applied to the object we want to work with:

{{#with user}} {{/with}}

Within the teamwithwe can access a property of an object without specifying the name of that object:

{{#with user}} {{name}} {{age}} {{/with}}
