Context in Handlebars Express

Let an object with a user and some variable be passed to the viewname:

app.get('/page/', function(req, res) { res.render('page', { name: 'test', user: {name: 'john', age: 30} }); });

Let's display the passed data in a view:

имя из переменной: {{name}} {{#with user}} имя из юзера: {{name}} {{/with}}

As you can see the variable name isnamematches the key. Suppose we want to output in a designwithboth the name from the variable and the name from the object with the user. It won’t be possible to do this so easily, because the output commands will be the same:

{{#with user}} имя из юзера: {{name}} имя из переменной: {{name}} - выведет имя юзера {{/with}}

In order to derive exactly the name from a variable, we must tell the template engine that we want to take the valuenamefrom outsidecontext. To do this, before the variable name you should write../:

{{#with user}} имя из юзера: {{name}} имя из переменной: {{../name}} {{/with}}

res.render('page', { amount: 10, product: { name: 'prod', cost: 1000, amount: 5 } });
