Displaying objects in Handlebars in Express

Let us render some pagepage.hbs:

app.get('/page/', function(req, res) { res.render('page'); });

Let an object with some data be passed to this page:

app.get('/page/', function(req, res) { res.render('page', { user: {name: 'sam', age: 30} }); });

Let's display the user's name and age in the view file:

<div> {{user.name}} {{user.age}} </div>
