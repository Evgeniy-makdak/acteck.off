Conditions in Handlebars Express

Using the command{{#if}}it is possible to make certain text to be shown if some variable has a valuetrue, and hidden iffalse. Let's look at an example.

Let some logical variable be passed to the view:

app.get('/page/', function(req, res) { res.render('page', { show: true }); });

Let's show some text if the value of our variable istrue:

<div> {{#if show}} <p>hello</p> {{/if}} </div>

else command

You can also use the command{{else}}to show either one or the second text:

<div> {{#if show}} <p>text1</p> {{else}} <p>text2</p> {{/if}} </div>

unless command

Team{{#unless}}reverse command{{#if}}, that is, shows text if the value of the variable is false:

<div> {{#unless show}} <p>text1</p> {{/unless}} </div>

Practical problems

Let the following variables be passed to the view:

res.render('page', { show1: true, show2: false, show3: true, });

Let us have three blocks:

<div> block 1 </div> <div> block 2 </div> <div> block 3 </div>

Make each block visible if its corresponding variable is true.

Comment

The commands described can only test variables for true or false. More complex conditions are not available.
