Introduction to the Handlebars template engine in Express

Let's now start exploringtemplate developers. They allow us to conveniently separate the site layout file, its content and its blocks into separate files. They also provide a number of commands that can be written in HTML files to generate HTML code.

One of the most popular template engines in Express is Handlebars. To install it, run the command:

npm install express-handlebars

Now we need to connect and configure the installed template engine. Let's do this:

import expressHandlebars from 'express-handlebars'; const handlebars = expressHandlebars.create({ defaultLayout: 'main', extname: 'hbs' }); import express from 'express'; let app = express(); app.engine('hbs', handlebars.engine); app.set('view engine', 'hbs');

After that in the folderviewsyou will store your content files, and in the folderviews/layouts- your layouts to which content will be connected. These files will need to have the extension.hbs, so that the template engine understands that these are its files.

Handlebars will take a template file, add content, title and widgets in the right place and send the finished code to the browser. We will look at how this is done in the following lessons.
