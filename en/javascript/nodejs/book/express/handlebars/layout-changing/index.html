Changing the layout in Handlebars Express

Sometimes you may need to change the site layout for some pages. Let, for example, in addition to the default site layout, we also have this one:

<!DOCTYPE html> <html> <head> <title>{{ title }}</title> </head> <body> <header> admin header </header> <main> {{{ body }}} </main> <footer> admin footer </footer> </body> </html>

Let us also have some page:

app.get('/page/', function(req, res) { res.render('page', { test: 'data' }); });

Let's make our page appear in our site layout. To do this, we will pass the name of our layout to the view in the propertylayout:

app.get('/page/', function(req, res) { res.render('page', { test: 'data', layout: 'admin' }); });
