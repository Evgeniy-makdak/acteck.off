Helper combinations in Handlebars

Helpers can be combined within one team. Let's look at an example. Let us have the following helpers:

helpers: { square: function(num) { return num * num; }, cube: function(num) { return num * num * num; }, sum: function(num1, num2) { return num1 + num2; } }

Let's use these helpers to first square the number, and then cube the result:

<p> {{cube (square 3)}} </p>

Let's find the sum of the square and cube of some numbers:

<p> {{sum (square 3) (cube 4)}} </p>

Let's find the sum of the square and just some number:

<p> {{sum (square 3) 4}} </p>
