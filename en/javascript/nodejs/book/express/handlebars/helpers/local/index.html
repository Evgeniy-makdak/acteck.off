Локальные хелперы в Handlebars

Создаваемы нами хелперы получаются глобальными - они будут доступны в любом представлении. Можно, однако, сделать локальный хелпер, доступный только при обработке определенного урла:

app.get('/', function(req, res) { res.render('test', { data1: 'str1', data2: 'str2', helpers: { // объект с локальными хелперами } }); });

При этом можно как создавать новые хелперы, так и переопределять глобальные хелперы, меняя их поведение.
