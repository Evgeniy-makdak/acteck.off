Passing parameters to helpers in Handlebars

Helpers can receive parameters and return a result based on those parameters. For example, let's make a helper that will receive a number as a parameter and return the square of this number:

helpers: { square: function(num) { return num * num; } }

Let's use the ones created by the helper in the view, passing it a number as a parameter. The parameter is passed separated by a space after the helper name:

<p> {{square 3}} </p>

Multiple options

You can pass several parameters to the helper. For example, let's make a helper that will receive two numbers as parameters and return their sum:

helpers: { sum: function(num1, num2) { return num1 + num2; } }

Let's use the ones created by the helper in the view, passing two numbers as parameters:

<p> {{square 3 4}} </p>

Make a helper that will take three numbers as parameters and return the sum of these numbers.

Parameter string

Let's make a helper that will receive a string as a parameter and return this string in uppercase:

helpers: { upper: function(str) { return str.toUpperCase(); } }

Let's use the ones created by the helper in the view, passing it the string as a parameter:

<p> {{upper 'abcde'}} </p>
