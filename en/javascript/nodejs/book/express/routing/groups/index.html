Route groups using Router in Express

It is convenient to combine routes with partially matching addresses into groups. Let's see how it's done. Let's say we have the following set of routes:

app.get('/user/show/:id', function(req, res) { }); app.get('/user/edit/:id', function(req, res) { });

As you can see, these routes have a common part/user. Let's take it out separately. To do this, for our group of routes we must create a separate router using the methodRouter:

let userRouter = express.Router();

Let’s now attach our routes to the created router, removing the matching part of the address:

userRouter.get('/show/:id', function(req, res) { }); userRouter.get('/edit/:id', function(req, res) { });

Now let's connect our router toappas a route, indicating the matching part of the address:

app.use('/user/', userRouter);

Divide the following routes into groups:

app.get('/city/show/:id', function(req, res) { }); app.get('/city/edit/:id', function(req, res) { }); app.get('/country/list', function(req, res) { }); app.get('/country/show/:id', function(req, res) { }); app.get('/country/edit/:id', function(req, res) { });
