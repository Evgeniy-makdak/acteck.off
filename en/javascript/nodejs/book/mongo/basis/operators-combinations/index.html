Operator Combinations in MongoDB

The operators we studied in the previous lesson can be combined in any way. Let's look at examples.

Example

let cond = {age: {$gt: 26, $lt: 29}}; let res = await coll.find(cond).toArray();

Tell us what the given code does.

Example

let cond = {age: {$gte: 26, $lte: 29}}; let res = await coll.find(cond).toArray();

Tell us what the given code does.

Example

let cond = { age: {$gte: 26, $lte: 29}, salary: {$in: [400, 500]}, }; let res = await coll.find(cond).toArray();

Tell us what the given code does.
