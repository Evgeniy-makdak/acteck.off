Deleting data in MongoDB

In this lesson we will look at methods that allow you to delete documents and collections from the database.

Removal

The method removes the first document from the collection that matches the filter. For example, let's delete the first user with age25:

await coll.deleteOne({age: 25});

Remove the first product with a price400.

Remove the first product in the collection.

Mass deletion

MethoddeleteManydeletes all documents in the collection that match the filter. For example, let's delete all users with age26:

await coll.deleteMany({age: 26});

Remove all products with a price500.

findOneAndDelete method

The method not only removes a document from the collection, but also returns it.

let res = await coll.findOneAndDelete({age: 25}); console.log(res);

Print the first product to the console. At the same time, remove it from the database.

drop method

The method removes the collection from the database. Let's delete our collectionusers:

await coll.drop();
