Adding Documents to MongoDB

To add new documents to the collection, use the methodinsertOne. Let's see how to work with it in practice.

Let us have an object with a new user:

let user = {name: 'john', age: 25, salary: 900};

Let's add this user to the collectionusers:

await coll.insertOne(user);

Given an object with a product:

let prod = {name: 'test', cost: 300, rest: 30};

Add this product to your collectionprods.
