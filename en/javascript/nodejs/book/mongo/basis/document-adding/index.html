Добавление документов в MongoDB

Для добавления новых документов в коллекцию используется метод insertOne. Давайте посмотрим, как работать с ним на практике.

Пусть у нас есть объект с новым юзером:

let user = {name: 'john', age: 25, salary: 900};

Давайте добавим этого юзера в коллекцию users:

await coll.insertOne(user);

Дан объект с продуктом:

let prod = {name: 'test', cost: 300, rest: 30};
