Sorting data in MongoDB

MongoDB allows you to sort documents when fetching. There is a special method for thissort. This method takes as a parameter an object with fields by which sorting is performed.

In this case, the keys of the object will be the names of the fields, and the values ​​will be numbers1or-1. The first value forces the data to be sorted in ascending order, and the second - in descending order. Let's look at examples.

Example

Let's sort our users in ascending age order:

let res = await coll.find().sort({age: 1}).toArray(); console.log(res);

Example

Let's sort our users in descending order of age:

let res = await coll.find().sort({age: -1}).toArray(); console.log(res);

Example

Let's sort our users first by age, and within groups with the same age, sort by salary:

let res = await coll.find().sort({age: 1, salary: -1}).toArray(); console.log(res);

Natural sorting

Parameter$naturalallows you to sort documents in the order in which they were added to the collection, or in reverse order.

Let's sort our users in reverse order:

let res = await coll.find().sort({$natural: -1}).toArray(); console.log(res);
