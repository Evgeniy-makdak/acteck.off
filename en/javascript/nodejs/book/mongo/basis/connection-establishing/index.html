Establishing a connection to MongoDB

Let's now work with MongoDB from a NodeJS file. Для начала нужно импортировать установленный драйвер:

import mongodb from 'mongodb';

Next, you need to create a new object containing the MongoDB connection settings. The first parameter specifies localhost and the standard MongoDB port 27017, and the second parameter specifies the connection settings:

let mongoClient = new mongodb.MongoClient('mongodb://localhost:27017/', { useUnifiedTopology: true });

Then you need to establish a connection to MongoDB:

mongoClient.connect(async function(error, mongo) { });

Let's now check if the connection to MongoDB has been successfully established:

mongoClient.connect(async function(error, mongo) { if (!error) { console.log('connection is successful'); } else { console.error(err); } });

Let's now, if the connection is successfully established, connect to the database namedtest:

mongoClient.connect(async function(error, mongo) { if (!error) { let db = mongo.db('test'); } else { console.error(err); } });

Now we get a collection with the nameusers:

mongoClient.connect(async function(error, mongo) { if (!error) { let db = mongo.db('test'); let coll = db.collection('users'); } else { console.error(err); } });

Let's output data from this collection to the console:

mongoClient.connect(async function(error, mongo) { if (!error) { let db = mongo.db('test'); let coll = db.collection('users'); let res = await coll.find().toArray(); console.log(res); } else { console.error(err); } });

In the following lessons, for the sake of brevity, I will omit the extra code and write only the essential part, like this:

let res = await coll.find().toArray(); console.log(res);

Copy the code you received in the lesson. Run it for yourself and make sure everything works.
