MongoDB syntax options

Working with MongoDB, like everything else in NodeJS, has several syntax options: viaasync-await, throughthenand through callbacks.

You may encounter any of these options, so in this tutorial I will show all the syntaxes. But in the following lessons I will use the first option, as it is more modern.

First syntax

You already know the first syntax for getting data from the previous lesson:

let res = await coll.find().toArray(); console.log(res);

It would be a good idea to add exception handling. Let's do this:

try { let res = await coll.find().toArray(); console.log(res); } catch (error) { console.log(error); }

Second syntax

Here is the second version of the promise syntax:

coll.find().toArray().then(res => { console.log(res); });

Let's add exception handling:

coll.find().toArray().then(res => { console.log(res); }).catch(error => { console.log(error); });

Third syntax

Here is the third option via callbacks:

coll.find().toArray((error, res) => { console.log(res); });

Let's add exception handling:

coll.find().toArray((error, res) => { if (!error) { console.log(res); } else { console.log(error); } });
