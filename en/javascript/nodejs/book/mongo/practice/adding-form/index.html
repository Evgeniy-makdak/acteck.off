Form for adding a document

Let's now create a form to add a new user to our database:

<form action="" method="POST"> <input name="name"> <input name="age"> <input name="salary"> <input type="submit"> </form>

Let's make a route to display this form:

app.get('/user/add', function(req, res) { res.render('add'); });

Our form is submitted via POST to the same page. This means that the result of form submission should be caught at the same URL, but using the methodpost:

app.post('/user/add', function(req, res) { });

Let's display the submitted form data:

app.post('/user/add', function(req, res) { let user = req.body; console.log(user); });

Saving a new user to the database:

app.post('/user/add', function(req, res) { let user = req.body; await coll.insertOne(user); res.send('added successfully'); });
