Types of Hooks in React

In this tutorial we will look at different types of hooks. In total, React has12built-in hooks. Widely used are considereduseState, useEffectAnduseContext. All hooks can be divided into several groups, according to their application.

First group

The first group is hooks for working with component states or state hooks. This group includesuseStateAnduseReducer.

Second group

The second group is context hooks (we will look at the concept of context in React in one of the lessons). There is only one hook in this group -useContext.

Third group

The third group is ref hooks (we will get acquainted with the concept of refs later). In this group we have hooksuseRefAnduseImperativeHandle.

Fourth group

The fourth group includes effect hooks (we'll talk about effects later), such asuseEffect, useLayoutEffect, useInsertionEffect.

Fifth group

And finally, the last group contains hooks to improve application performance. These are hooks likeuseMemo, useCallback, useTransitionAnduseDeferredValue.

Uncategorized

There are more hooksuseDebugValue, useId, useSyncExternalStore, which do not belong to any category.

Your own hooks

Also, in addition to using the built-in hooks, you can create your own hook if necessary.

What's next

In future lessons we'll take a closer look at most of the hooks covered in this tutorial.
