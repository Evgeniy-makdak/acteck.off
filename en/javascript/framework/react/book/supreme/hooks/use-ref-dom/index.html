Using the useRef hook on the DOM in React

In one of the previous lessons, we mentioned that refs are most often used to access DOM elements.

Refs will come to the rescue when, for example, you need to focus on an element, scroll or calculate its size and position, or perhaps use some of its methods or properties, because there are no built-in methods for such things in React.

Let's look at an example with an input to see how this is done. Let's create a component with an input and a button:

return ( <div> <input /> <button>focus</button> </div> );

First, let's import it into the componentuseRef:

import { useRef } from 'react';

And let's start a refrefin the component. We initialize it with the valuenull:

const ref = useRef(null);

Let's attach a click handler to the button:

<button onClick={handleClick}>focus</button>

To have access to input, let’s write down our refrefto attributerefinput. This way, when React creates a DOM node for this input, it will put a reference to it inref.currentand we can use input methods:

<input ref={ref} />

And finally, we’ll add a click processing function so that when you click on the button, the input gets focus. Now we can do this through the ref, which contains a link to our input, by accessing its methodfocus:

function handleClick() { ref.current.focus(); }

Press the button and make sure that the input has focus.

Take the component codeApp, which we wrote in this lesson and make sure that when you click on a button in the input, in addition to setting focus, the input field is also cleared. Use the property for thisvalueinput.
