Global handlers via useEffect in React

Suppose that now we want the color to change not only when the button is clicked, but also when pressedEnteron the keyboard. In normal React life, listen to the global objectwindowusingaddEventListenerit is forbidden. For this we have effects. Let's add this action withuseEffect.

Let's write a click handler functionEnter. Let the color in this case change to red:

function handleEnter(event) { if (event.keyCode === 13) { setColor('red'); } }

Let's now bind event listening towindowin the hookuseEffect:

useEffect(() => { document.body.style.color = color; window.addEventListener('keydown', handleEnter); }, [color]);

If we have some kind of installation function, then we always need touseEffectreturn a cleanup or unsubscribe code to avoid problems in the future. In this case, we need after attachingaddEventListenerreturn its removal:

useEffect(() => { document.body.style.color = color; window.addEventListener('keydown', handleEnter); return () => { window.removeEventListener('keydown', handleEnter); }; }, [color]);

Component is given. Make sure that when you click anywhere on the page, the background of this component changes color.
