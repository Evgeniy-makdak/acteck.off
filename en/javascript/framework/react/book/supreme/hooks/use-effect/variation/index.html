Variations of the useEffect hook in React

At the hookuseEffectThere are also variations that are specific and very rarely used, so we will dwell on them briefly.

The first variation is a hookuseLayoutEffect, which fires before the browser redraws the screen. You can use this feature to calculate the size and position of a component depending on other elements, so that when rendered, the browser will display it immediately as and where it is needed. Thanks to this, for example, the user will not see intermediate jumps of the component when changing its position or size, but only their final versions.

The second variation is a hookuseInsertionEffect, which fires before any changes to the DOM tree, that is, when there are still references are not attached and the DOM is not updated. Mainly used where you need to insert dynamic styles from CSS-in-JS libraries before DOM changes.
