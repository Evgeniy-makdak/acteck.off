Hooks in React

In this tutorial we will begin to get acquainted with hooks. We have already encountered them when we studiedstates- there we learned how to work with a state hookuseState.

In general, hooks are regular JavaScript functions whose names begin with the word'use'.

Hooks allow us to use states and other features of React without using classes. They make it easier to reuse code for different tasks, as well as test it, and help break complex components into simpler functions based on their intended purpose. Some hooks can even replace functionalityRedux. In addition, hooks help improve application performance.

If you want to use hooks, then follow two basic rules. First, use them only at the top level - don't call them inside loops, conditional statements, or nested functions. Secondly, hooks should only be called from React functions (from React components or custom hooks), and not from regular JavaScript functions.

Another mistake that programmers make is the uncontrolled use of hooks in their code. Before using a particular hook, make sure that it is really necessary.
