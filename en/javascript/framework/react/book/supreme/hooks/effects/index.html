Effects in React

Before you consider using a hookuseEffect, we need to understand what effects are in React.

Effectsor side effects (side effects) are additional actions that allow the component to connect and synchronize withexternal systems.

External systems refer to those pieces of code or application that are not controlled by React. This means working with the network, browser DOM, animation, widgets written using third-party UI libraries, and other applications not created in React.

Effects allow you to run code after rendering, thus giving you the ability to sync your component with systems outside of React.

Effects can be executed when the component is first rendered, when an observed value changes, or when the component's lifecycle has ended.

You need to remember that effects are a kind of "escape" hatch from React, you shouldn't abuse them.
