Routing in React using the Router library

In this section we will get acquainted with a very interesting and important section when creating web applications -routing. It is thanks to routing that the user can navigate among many web pages depending on the actions or requests sent.

In this tutorial we will learn how to use an external libraryReact Routerto create React applications with routing.

React Route allows us to work with routing both on the client side (i.e. in the browser) and on the server side. It allows you to create complex single-page web applications that use navigation without refreshing the entire page (that is, without additional calls to the server), and also work with browser history (back arrows).

React Router uses a component-based architecture. Thus, this library allows you to navigate among multiple components of a React application, change URLs in the browser, and keep your application state in sync with the URL.
