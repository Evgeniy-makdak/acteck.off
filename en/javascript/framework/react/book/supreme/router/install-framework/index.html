Установка фреймворка React Router

Для данного раздела учебника, кроме React Router, мы будем использовать Vite в качестве сборщика и тестового сервера.

Для начала давайте создадим React-приложение my-app с использованием Vite. Для этого откроем терминал и введем следующие команды:

npm create vite@latest my-app -- --template react cd my-app npm install

Для того, чтобы работать с маршрутизацией нам необходим модуль react-router-dom, давайте установим и его:

npm install react-router-dom

Теперь все готово для начала работы. Запустим наше приложение:

npm run dev

Скопируем линк http://localhost:5173/, указанный в терминале в адресную строку браузера и увидим запущенное приложение (можете понажимать кнопочку со счетчиком).

Создайте приложение в React с Vite, как описано в уроке, установите модуль react-router-dom. Убедитесь, что приложение запускается.
