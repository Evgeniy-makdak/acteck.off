Getting loader data by URL parameters in React Router

In the last lesson we wrote a function to get product page data, let's importgetProductin the fileproduct.jsx:

import { getProduct } from '../forStorage';

Let's write a loader functionloaderimmediately after import and before the functionProduct. Let's pass URL parameters to it:

export async function loader({ params }) { const product = await getProduct(params.productId); return { product }; }

We also import the hookuseLoaderDatato use the data received by the bootloader:

import { useLoaderData } from 'react-router-dom';

ApplicableuseLoaderData, respectively replacing the line with the creation of the objectproductat the beginning of the functionProductto the next one:

const { product } = useLoaderData();

And, of course, we’ll change our layout a little, since we’ve removed the old objectproduct. Let's replace the contents of the paragraphs:

<p>Name: {product.name}</p> <p>Cost: {product.cost}</p> <p>Amount: {product.amount}</p>

To the following:

<p>Name: {product.name ? product.name : <i>unnamed</i>}</p> <p>Cost: {product.cost ? product.cost : <i>unknown</i>}</p> <p>Amount: {product.amount ? product.amount : <i>unknown</i>}</p>

We just have to openmain.jsxand add import thereloader, let's call itproductLoader:

import Product, { loader as productLoader, } from './routes/product';

And register it as a loader for the route object of the product page, adding it tochildrenafter propertyelement:

children: [ { path: 'products/:productId', element: <Product />, loader: productLoader, }, ],

That's it, we've sorted out loading data for the product page! True, we have nothing to upload yet. Launch the application, add products and click on them in the list to make sure everything works.
