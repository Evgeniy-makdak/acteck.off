Adding a router in React Router

Having become a little familiar with routers, we can add one of them to the application.

But first, we need to clean up our application template that we created in the last lessons a little. More specifically, let's take care of daddysrc. Throw out the folderassets, filesApp.css, App.jsx. In general, you will only havemain.jsxAndindex.css.

Openindex.cssand clean it completely. Let it be empty.

Now let's move on to our main filemain.jsx. In it we will have the following code:

import './index.css' import React from 'react'; import ReactDOM from 'react-dom/client'; ReactDOM.createRoot(document.getElementById('root')).render( <React.StrictMode> </React.StrictMode> );

Now we can add the router. We useBrowserRouter, since it is most often used in web applications. Let's not forget about modern syntax. Let's add an import line:

import { createBrowserRouter, RouterProvider, } from 'react-router-dom';

Then, let's create a constantrouterand create ourBrowserRouterafter the import lines. Let's do this so that on the main page we display a div with the inscription'Hello Router!'. To do this, we will write down the path to the main page and the element that we will display:

const router = createBrowserRouter([ { path: '/', element: <div>Hello Router!</div>, }, ]);

We still need to add a componentRouterProviderand enter the router we created above. It accepts all router objects, renders our application and connects other APIs:

ReactDOM.createRoot(document.getElementById('root')).render( <React.StrictMode> <RouterProvider router={router} /> </React.StrictMode> );

The complete code will look like this:

import './index.css' import React from 'react'; import ReactDOM from 'react-dom/client'; import { createBrowserRouter, RouterProvider, } from 'react-router-dom'; const router = createBrowserRouter([ { path: '/', element: <div>Hello world!</div>, }, ]); ReactDOM.createRoot(document.getElementById('root')).render( <React.StrictMode> <RouterProvider router={router} /> </React.StrictMode> );

Let's save the changes. Congratulations! Our application no longer argues with you. Now, if we run it, then on the main page in the browser we will see'Hello Router!'.

Using the material provided, make sure that on the main page of your application, which you created in homework for previous lessons, you display a paragraph with the text'I'm number one in React!'.
