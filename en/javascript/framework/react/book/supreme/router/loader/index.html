Route data loader in React Router

In the last lesson, we retrieved product data from the warehouse. Now this data needs to be loaded into the route element. To do this, use the bootloader function (loader). Now we will create it.

Let's openroot.jsx, add an import line heregetProductsfromforStorage.js:

import { getProducts } from '../forStorage'

Immediately after the import lines and before the functionRootlet's write the function codeloader, which is throughgetProductswill return us data with products from the storage. In general, such functions should be written in a separate file, but we’ll be a little lazy, so don’t pay attention to the warning:

export async function loader() { const products = await getProducts(); return { products }; }

Let's now add our bootloader import tomain.jsx. We will have this loader for the root, so let's call itrootLoader:

import Root, { loader as rootLoader } from './routes/root';

And add it to our route object as a propertyloader:

const router = createBrowserRouter([ { path: '/', element: <Root />, errorElement: <ErrorPage404 />, loader: rootLoader, children: [ { path: 'products/:productId', element: <Product />, }, ], }, ]);

Take the application you created in the assignments for previous lessons. Using the lesson materials, write in the fileroot.jsxfunctionloaderto load student data and add it to the route object in main.jsx.
