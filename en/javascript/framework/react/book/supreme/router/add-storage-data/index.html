In this tutorial we will be adding a product to the repository. First we need a functioncreateProductto create a product, add it toforStorage.jsafter functiongetProducts:

First we add againsomeNetwork:

export async function createProduct() { await someNetwork(); }

But next we needunique idfor each product, you've already seen this in the React tutorial. Let's use the library for thisnanoid. Enter the following command in the terminal:

npm install --save nanoid

We import toforStorage.jslibrary:

import { nanoid } from 'nanoid'

Let us haveidBy6characters:

Initially, when creating a product, we will only have in the objectid. Let's callgetProducts, add the generated product and update the list of our products in the repository. Ready:

export async function createProduct() { await someNetwork(); let id = nanoid(6); let product = { id }; let products = await getProducts(); products.unshift(product); await setProducts(products); return product; }

The function for updating the list in the repository will be as follows (we will add products to it under the keyproducts), place it after the functioncreateProduct:
