Creating a root route in React Router

Often the main route leading to the main page is called the root (root route), since other routes will be rendered inside it. It is very convenient to imagine it as a separate component. So let's take our application again and in the foldersrclet's create another one and call itroutes. Now let's create a file in this folderroot.jsxfor a separate React componentRoot. The code in the file will look like this:

function Root() { return <div>Hello Router!</div>; } export default Root;

Now let's make changes to a piece of code in the filemain.jsx, because as a valueelementwe will now pass the componentRoot, and its contents are now in a separate fileroot.jsx:

const router = createBrowserRouter([ { path: '/', element: <Root />, }, ]);

Let's not forget the import line with our component in main.jsx:

import Root from './routes/root';
