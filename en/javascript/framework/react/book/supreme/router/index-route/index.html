Index route in React Router

In this lesson we will learn what index routes are.

First, let's pay attention to one thing. When we are on the main page, on the right we see just a blank screen. All because none of the child routes match the address/, but we didn’t say what to output in this case. In this case, in order not to display a blank page, you can useindex route- a child route that does not have a path, and its element is displayed in the parentOutletwhen none of the set of children is suitable. Using such routes you can post any information, or statistics, or something else like that. As with regular ones, they can be used to download data.

Let's create a fileindex.jsxin a folderroutes, and in it the componentIndex:

export default function Index() { return ( <div> <p>Hi, React Router!</p> <p>This is an application for my products :)</p> </div> ); }

Now let's go tomain.jsxand import our component thereIndex:

import Index from './routes/index';

And the first element in the arraychildrenlet's add an object where the propertyindexinstalled intrue, and our component will be the element to be displayedIndex:

{ index: true, element: <Index /> },
