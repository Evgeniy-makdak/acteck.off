Adding another route to React Router

We have a root route that goes to the main page where we have links for products. Let's now add another route so that when one of the links is clicked, we will see a page for the product, and not an error screen.

First, let's open our application, which we made in previous lessons and create it in the folderroutesfileproduct.jsx. Let's make a component in itProduct:

function Product() {} export default Product;

Now let's create insideProductobjectproduct, with propertiesname, costAndamountfor now these will be some fixed values:

function Product() { const product = { name: 'Product', cost: 400, amount: 5, }; } export default Product;

And on the product page we will display, respectively, its name, cost and quantity:

return ( <div> <h2>Product page</h2> <p>Name: {product.name}</p> <p>Cost: {product.cost}</p> <p>Amount: {product.amount}</p> </div> );

Be sure to add component importProductto filemain.jsx:

import Product from './routes/product';

And finally, let's add another route to our application, do it right after the root one. As the path we indicate'products/:productId', and the element to be displayed is the componentProduct:

{ path: '/', element: <Root />, errorElement: <ErrorPage404 />, }, { path: 'products/:productId', element: <Product />, },

Now we launch the application, click on the links and find ourselves on the page with the product (for now it is the same for all links), and not with an error.
