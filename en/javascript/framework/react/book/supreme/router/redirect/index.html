Redirect to another route in React Router

Now the data entered into the form when editing a product is saved, but there is one BUT - after saving the data, we remain on the page with the form, and we need to get back to the product page. A redirect will help us with this, which we will get acquainted with in this lesson.

Let's open the fileedit.jsxand importredirectfrom the library:

import { redirect } from 'react-router-dom';

And then let's make the functionactionwill now not return1, but redirect us to the desired route. In this case, we need to go back to the product page:

return redirect(`/products/${params.productId}`);

Now, by clicking the save button, we are taken back to the product page with updated data.

We can do the same thing to add a new product, because it would probably be more convenient to immediately go to the form and fill it out. Let's do just that.

First, we now need to open the fileroot.jsxand importredirect, since adding products occurs on the root page:

import { redirect } from 'react-router-dom';

Let's take the function againactionand now we will not returnproduct, and redirect to the editing page:

return redirect(`/products/${product.id}/edit`);
