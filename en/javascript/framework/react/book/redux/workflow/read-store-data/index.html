Getting data from store in a component in Redux

In this lesson we will display data from store in a React component.

Let's open our application with products, go to the folderparts/productsand create a file in itProductsList.jsx. First, let's import the hook into ituseSelector, with which we will receive data from the store:

import { useSelector } from 'react-redux'

Now let's get the products from the store. We will not create the selector function separately; we will write its code directly in the parameter foruseSelector:

export const ProductsList = () => { const products = useSelector((state) => state.products) }

We will then output the products in the standard way in the body of the functionProductsListafter the line with receiving products usingmap:

const dispProducts = products.map((product) => ( <div key={product.id}> <h3>{product.name}</h3> <p>{product.desc.substring(0, 100)}</p> <p>Price: {product.price}</p> <p>Amount: {product.amount}</p> </div> ))

And below we will displaydispProductsin the following layout:

return ( <div> <h2>Products</h2> {dispProducts} </div> )

Now all that remains is to display our list of products on the main page. Let's open the fileroot.jsxand fix the title:

<h2>This is my first Redux app!</h2>

On to the next one:

<h2>My Products App</h2>

Then we import the component into the fileProductsList:

import { ProductsList } from '../parts/products/ProductsList'

And insert it into the div with IDmain-pageimmediately after the closing taghr:

<ProductsList />

Let's launch our application. Our list of two products was successfully displayed on the main page.
