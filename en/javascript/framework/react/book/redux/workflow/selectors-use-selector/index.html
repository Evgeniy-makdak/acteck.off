Selectors and the useSelector hook in Redux

In previous lessons, we recorded product data in the store. How can we get them from the store?

In order to do this, we need to writeselector function. This function knows how to extract a specific piece of information from a state value stored in the store and helps avoid writing repetitive code as the application grows. This function takes state as input, and outputs the required state slice. For example like this:

const selectValue = state => state.some.value

At the very beginning of the tutorial it was mentioned that store has a methodgetState. And we could easily use it to get the state value:

const value = selectValue(store.getState())

The problem is that our React components cannot directly access the store because it is not allowed to be imported into the component file. But we have the opportunity to get the data in the component using the React-Redux hookuseSelector. In addition, with this hook our components will always receive only current data:

const count = useSelector(selectValue)
