Reducer and action in Redux

In the last lesson, we completed the form for adding a product and displayed it on the main page. But while the added data is not saved, in other words, the new product is not added to the existing ones in the store. Let's fix this.

To begin, in our products application we will open the fileproductsSlice.jsxand write in the propertyreducerForcreateSlicefunctionproductAdded, which will be responsible for adding a product to the store based on the current state and action passed to it. It should be noted that here we are not given the entire state, but only that part of it that is responsible for the products (sliceproductsknows only about her). The object with the new product will be located in the propertypayloadobjectaction, which will be generated by the handler when the save button is clicked on the form. As a result, the reducerproductAddedwill look like in codecreateSliceSo:

const productsSlice = createSlice({ name: 'products', initialState, reducers: { productAdded(state, action) { state.push(action.payload) }, }, })

What about action? At the very beginning of the tutorial, we mentioned that an action is an event, represented as an object, that describes what happened in the application. We also talked about the fact that we can use the action creator function, which will create such an object for us, for example like this:

const addProduct = newProduct => { return { type: 'products/productAdded', payload: newProduct } }

The good news is that we don't need to do anything, and the function will do it for uscreateSlice, which we use. Once we have written the reducer, it will automatically create an action creator with the same name for us. All we have to do is export the resulting action creator for further use in components. Let's do this at the end of the file before exporting the reducer, like this:

export const { productAdded } = productsSlice.actions export default productsSlice.reducer

Open your application with students and then the filestudentsSlice.jsx, add the field valuereducerForcreateSlice, as shown in the lesson.

Add to the end of the filestudentsSlice.jsxexport the resulting action creator function.
