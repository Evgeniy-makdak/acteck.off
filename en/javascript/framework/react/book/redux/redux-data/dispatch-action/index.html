Sending an action in a Redux application

Now we have a reducer to update the slice with reactions. Now we need to add an action sending that will be triggered when the user clicks on a button with a reaction. But first, in order for everything to fully work, we need to make a couple of changes.

Let's open our application with products, and in it there is a fileproductsSlice.js. Look at the definitioninitialState. Do you see? Objects do not have properties with reactions. Let's fix this by setting the counters for each of the reactions in0:

{ id: '1', title: 'Product1', desc: 'This is an amazing product', price: '300', amount: '30', reactions: { good: 0, soso: 0, bad: 0, }, },

Add a property to the second object to be initializedreactionsyourself. We need to add the same property below to the code for the reducerproductAddedso that new products also have this functionality. Let's do it for his methodprepare:

prepare(title, desc, price, amount, sellerId) { return { payload: { id: nanoid(), title, desc, price, amount, seller: sellerId, reactions: { good: 0, soso: 0, bad: 0, }, }, } },

Now let's start sending action. Let's open the fileUserReactions.jsx, then import the hook into ituseDispatchAndreactionClicked:

import { useDispatch } from 'react-redux' import { reactionClicked } from './productsSlice'

At the beginning of the code in the function bodyUserReactionsfirst line (beforeconst userReactions ... ) let's write:

const dispatch = useDispatch()

And add sending to the reaction buttonreactionClickedwhen clicked, we will receive the product id and the name of the clicked reaction:

<button key={name} type="button" className="reaction-button" onClick={() => dispatch(reactionClicked({ productId: product.id, reaction: name })) } > {image} {product.reactions[name]} </button>

All we have to do now is to display the component with reactions in the layout for our pages. Let's do this in a fileProductsList.jsxbefore the elementLink:

<UserReactions product={product} />

And exactly the same before the elementLinkon the product page in the fileProductPage.jsx:

<UserReactions product={product} />

Now that everything is in place, we can launch our application. Let's try clicking on the buttons with reactions. We can add a new product and also click on them. Now, wherever we click on the reaction buttons, the changed values ​​will be displayed for this product on all pages. Look at the page for an individual product (on which you clicked on reactions) and the page with their list and vice versa, for all pages the number of reactions for a particular product will be the same.

This lesson was the final lesson in the chapter on working with data in Redux. We worked more in-depth with the data in the Redux store and now we know how to add the functionality we need to the application and use the data in React components.
