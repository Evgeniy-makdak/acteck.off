List of sellers in Redux

In previous lessons we finished with the basic material. Now using thunk we get the list of products from the server and can save the new product on the server. Let's deal with the sellers. After all, we receive data for them too. Let's make a separate page with a list of sellers.

Let's open our application with products, and in it the foldersellers. Let's create a file in this folderSellersList.jsx. Here we will useuseSelector, LinkAndselectAllSellers, the code of which we will write insellersSlice.jsa little later:

import { useSelector } from 'react-redux' import { Link } from 'react-router-dom' import { selectAllSellers } from './sellersSlice'

Next, let's create it ourselvesSellersList. In it we will get all the sellers using a hookuseSelector, then we get the layout for the list of sellers in a loopmap, and each name of the seller will lead to his page. And at the end we will return the layout with the title and the resulting list:

export const SellersList = () => { const sellers = useSelector(selectAllSellers) const sellersToRender = sellers.map((seller) => ( <li key={seller.id}> <Link to={`/sellers/${seller.id}`}>{seller.name}</Link> </li> )) return ( <div> <h2>Sellers:</h2> <ul>{sellersToRender}</ul> </div> ) }

Then let's go to the filesellersSlice.jsand by analogy with selectors inproductsSlice.jsLet's create two selectors at the very end of the file after exporting the reducer:

export const selectAllSellers = (state) => state.sellers export const selectSellerById = (state, sellerId) => state.sellers.find((seller) => seller.id === sellerId)

Now we need to register a route for the page with the list; to do this, open the fileApp.jsxand into an array of child routeschildrenlet's add another object (don't forget to import the componentSellersList):

{ path: '/sellers', element: <SellersList />, },

To our link'Sellers'in the menu it's working, let's go to the fileroot.jsx(how long ago it was...) and replace the tagaper elementNavLink:

<NavLink to="/sellers" end> Sellers </NavLink>

And one last thing. To make the active link stand out in our menu and make it clear what page we are on, we will add a style toindex.css:

nav a.active { color: purple; }

Let's launch our application. Now we can go to the sellers page by clicking on'Sellers'on the menu. First, of course, you need to click on'Products'in the menu, otherwise our products will not be loaded.
