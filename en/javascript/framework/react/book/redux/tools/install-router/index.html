Installing React Router in a Redux application

In previous lessons, we installed the basic tools for working with Redux. It often happens that an application needs a router to navigate its web pages, and our application is no exception. To do this we will install the libraryReact Router.

Let's open our project and type the following command in the terminal:

npm install react-router-dom

Looking ahead a little, I will say that our Redux application is traditionally dedicated to products, information about which is posted by certain sellers. Therefore, now we will slightly rework the main componentApp, and also add routing functionality to the application.

First, let's go to the foldersrc, and then inappand create a file hereroot.jsx. This will be the root of our site: on the left we will have a menu, and on the right - content (for now we only have the title here):

function Root() { return ( <div id="main"> <div id="menu"> <nav> <a>Products</a> <a>Sellers</a> </nav> </div> <div id="main_page"> <h2>This is my first Redux app!</h2> <hr></hr> </div> </div> ) } export default Root

A componentAppwe'll redo it. First, let's completely delete its contents. Then we import two functions for the router and our root componentRoot:

import { createBrowserRouter, RouterProvider } from 'react-router-dom' import Root from './app/root'

Below, after import, we will create a router and register the first route, passing it ourRootas a display element and, accordingly, setting the path to'/'. In the future inAppwe will add other necessary routes:

const router = createBrowserRouter([{ path: '/', element: <Root /> }])

Below we write the function codeApp:

function App() { return <RouterProvider router={router} /> } export default App

All. Let's launch. In the next section, we will implement the components for running the Redux application.

Finally, let's add a little beauty by writing styles for this inindex.css:

body { font-size: 24px; line-height: 1.5; } nav { display: flex; flex-direction: column; } ul { list-style: none; padding: 0; } h1, h2, h3 { margin: 0; } a { text-decoration: none; color: blue; cursor: pointer; } #main { display: flex; margin: 20px; } #menu { margin-right: 50px; padding-top: 50px; padding-right: 50px; border-right: 2px solid lightgray; font-weight: bold; }
