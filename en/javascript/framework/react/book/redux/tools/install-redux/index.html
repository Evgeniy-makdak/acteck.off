Installing Redux and RTK in a React application

In this lesson, we will install the React-Redux package in our application, which allows React components to work in conjunction with Redux and the Redux Toolkit (RTK for short).

Let's open our project and type the following command in the terminal:

npm install @reduxjs/toolkit react-redux

Now we need to create a store store that will store the global state of our application. To begin with, in the foldersrcwe will create a folderapp, and in it the filestore.js. Then in this file we need to importconfigureStorefrom RTK:

import { configureStore } from '@reduxjs/toolkit'

Next, let's create an empty store and pass at least one reducer:

export default configureStore({ reducer: {} });

In order for our React components to work with store, we will do the following. Let's go to our filemain.jsx, which already contains the code:

import React from 'react' import ReactDOM from 'react-dom/client' import App from './App.jsx' import './index.css' ReactDOM.createRoot(document.getElementById('root')).render( <React.StrictMode> <App /> </React.StrictMode>, )

Let’s add to the import the lines with the one created in the previous stepstoreand componentProvider:

import store from './app/store' import { Provider } from 'react-redux'

Then we wrap our main componentAppVProvider, and as a prop we’ll pass it to himstore:

<React.StrictMode> <Provider store={store}> <App /> </Provider> </React.StrictMode>

And finally, in the main React componentAppLet's place the following code:

function App() { return <h2>This is my first Redux app!</h2> } export default App

We run our project again and see in the browser window:'This is my first Redux app!'. For now, we do not pay attention to the warning about the absence of a valid reducer. You did it all, congratulations :).

Install Redux and RTK in your application as described in the tutorial. Make sure your application works.
