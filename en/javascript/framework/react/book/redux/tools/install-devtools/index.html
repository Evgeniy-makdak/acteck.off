Установка Redux DevTools Extension

На этом занятии мы добавим в брузер инструмент Redux DevTools Extension для отладки приложения на Redux.

Для Chrome мы скачаем и установим его по предложенной ссылке.

RTK-функция configureStore, которой мы воспользовались на прошлом уроке, не только создает store, но и автоматически конфигурирует React DevTools Extension, так что больше нам ничего не придется делать.

Конечно, если мы сейчас запустим наш проект и включим Redux DevTools Extension, нажав на иконку в расширениях, мы ничего практически пока и не увидим, поскольку мы еще не создавали стейты и логику в приложении.

Установите для вашего браузера расширение Redux DevTools Extension. Убедитесь, что все работает.
