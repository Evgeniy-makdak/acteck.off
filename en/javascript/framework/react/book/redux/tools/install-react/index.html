Installing and preparing a React application

In this tutorial, we will start by installing and preparing a React application, since in the future we will use it in conjunction with Redux. There are faster ways to create a React app template with Redux (which by the way is not always easy, especially for beginners) that are listed in the official documentation, but we will go from scratch.

First, let's create a React applicationmy-appusingVite(this is a fashionable replacementWebpack). For this we will openterminaland enter the following commands (in the proposed options, if any, select React and then JavaScript):

npm create vite@latest my-app -- --template react cd my-app npm install

Let's launch the created application:

npm run dev

Open the suggested link in your browserhttp://localhost:5173/and see a familiar screensaver. Everything works.

Now let's remove the excess. Let's clear the folderpublic. In a foldersrclet's just leavemain.jsx, index.cssAndApp.jsx(Vindex.cssAndApp.jsxcompletely delete the contents).

In fileindex.htmlreplace the title text with:

<title>React Redux App</title>

Build a React app and make sure it works.

Remove unnecessary files and folders.
