Passing id to a component in React

You already know that component tag attributes go into props. This applies to all attributes except the attributekey, which is needed for use in loops, like this:

const items = prods.map(prod => { return <Product key ={prod.id} name={prod.name} cost={prod.cost} />; });

In this case, the attributes will be included in the propsnameAndcost, Akey- won't hit. We may, however, need to conveyidin the component props. In this case, we will need to introduce one more attribute:

const items = prods.map(prod => { return <Product key ={prod.id} id ={prod.id} name={prod.name} cost={prod.cost} />; });
