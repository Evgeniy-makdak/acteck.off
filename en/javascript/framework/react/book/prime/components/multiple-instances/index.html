Multiple instances of a component in React

You can insert multiple products. To do this, you just need to write a few component tags:

import React from 'react'; import Product from './Product'; function App() { return <div> <Product /> <Product /> <Product /> </div>; } export default App;

The result after rendering will be the following:

<div> <p> product </p> <p> product </p> <p> product </p> </div>
