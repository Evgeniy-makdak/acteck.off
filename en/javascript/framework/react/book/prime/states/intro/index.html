Introduction to states in React

The next concept we'll cover is calledstates. States representreactivecomponent variables.

Reactivity means that when a state changes, changes will occur in all places where this state is used. Technically, this is achieved by rerendering the entire component when any state changes.

To use states, you first need to import the functionuseState:

import React, { useState } from 'react';

FunctionuseStatetakes the initial value of the state as a parameter, and returns a special array of two elements as its result. The first element of the array will store the current value of the state, and the second will store the function for changing the state.

For states to work correctly, they cannot be changed directly, but you should use a function to change them - only then will reactivity work.
