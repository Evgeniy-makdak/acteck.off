Using states in React

Let's use the functionuseStateto create a state containing the product name:

const state = useState('prod');

As a result, the constantstatewill be an array, the first element of which will store the name of the product, and the second will contain a function for changing the name:

const state = useState('prod'); const name = state[0]; const setName = state[1];

For brevity, one usually does not use an intermediate constant for an array, but rather usesdestructuring:

const [name, setName] = useState('prod');

Let's now display our state with the product name in some layout:

return <p> <span>{name}</span> </p>;

Let's put everything together and get the following code:

import React, { useState } from 'react'; function App() { const [name, setName] = useState('prod'); return <div> <span>{name}</span> </div>; } export default App;

If you run this code, the initial state value will be displayed in the div, in our case'prod'. When changing the state through the functionsetNameand a new state value will automatically appear in the layout.

Let's say you want to display user data on the screen: his first name, last name, age. Make appropriate states for this with initial values.
