Checkboxes and conditional rendering in React

Let's make sure that, depending on the checkbox, either one piece of code or another is displayed on the screen. We use conditional rendering for this:

function App() { const [checked, setChecked] = useState(true); let message; if (checked) { message = <p>сообщение 1</p>; } else { message = <p>сообщение 2</p>; } return <div> <input type="checkbox" checked={checked} onChange={() => setChecked(!checked)} /> <div>{message}</div> </div>; }

A checkbox is given. Using the checkbox, ask the user if he already18years. If the checkbox is checked, show the user the following block of code:

<div> <h2>Ура, вам уже есть 18</h2> <p> здесь расположен контент только для взрослых </p> </div>

And if the checkbox is not checked, then the following:

<div> <p> увы, вам еще нет 18 лет:( </p> </div>

Given a checkbox and a paragraph. If the checkbox is checked, let the paragraph be visible on the screen, and if it is not checked, hide it.
