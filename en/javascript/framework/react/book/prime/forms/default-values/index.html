Defaults in React

Sometimes you may need to make sure that the input already has some value by default. In this case, it is necessary that only the initial value of the input be taken from the state, and the input itself should not be tied to this state.

To solve this problem you need to use the attributedefaultValue:

function App() { const [value, setValue] = useState('text'); return <div> <input defaultValue={value} /> </div>; }

There is a similar attribute for checkboxesdefaultChecked, with which you can set the initial:

function App() { const [checked, setChecked] = useState(true); return <div> <input type="checkbox" defaultChecked={checked} /> </div>; }

Try these attributes out for yourself.
