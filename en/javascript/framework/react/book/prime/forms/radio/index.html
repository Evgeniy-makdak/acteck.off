Working with radio in React

Working withradio radiosomewhat different, for example, from the same checkboxes. The problem is that several radios will have the same state, but differentvalue.

Therefore, the work occurs in the following way: each radio button has an attributevaluewrite their value, and into the attributechecked- a special condition that checks whether the state is equal to a certain value. If equal, the radio button will become checked, and if not equal, the radio button will be unchecked.

Here is the implementation of what is described:

function App() { const [value, setValue] = useState(1); function changeHandler(event) { setValue(event.target.value); } return <div> <input type="radio" name="radio" value="1" checked={value === '1' ? true : false} onChange={changeHandler} /> <input type="radio" name="radio" value="2" checked={value === '2' ? true : false} onChange={changeHandler} /> <input type="radio" name="radio" value="3" checked={value === '3' ? true : false} onChange={changeHandler} /> </div> }
