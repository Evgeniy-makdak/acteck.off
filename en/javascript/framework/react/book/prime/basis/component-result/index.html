Result of the component

The div with the result will display what is returned throughreturncomponent function. In the following example it will be a div with text:

function App() { return <div> text </div>; }

Please note that we write divs without quotes - we just write the tag in JavaScript code! This is the core feature of React. In fact, in React we do not write in JavaScript, but in JSX, which we will study in the following lessons.
