Using props in conditional rendering with Styled Components in React

Props can also be used for conditional rendering. Let's take a React componentBlock, which we worked with in the last lesson.

Let's make sure that the background for the first input is yellow, and for the other inputs it is green. To do this, in the styles for the componentInputLet's add another line and it will be:

const Input = styled.input` background: ${(props) => (props.first ? 'yellow' : 'green')}; margin: 5px; font-size: 18px; `;

Let's add propsfirstto the first input:
