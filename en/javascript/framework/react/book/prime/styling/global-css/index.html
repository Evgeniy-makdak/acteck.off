Using global CSS when styling in React

Let's say we have a React componentApp, in which there is a div, and inside the div there are three paragraphs:

function App() { return ( <div> <p>text</p> <p>text</p> <p>text</p> </div> ); }

Let's style this component. For this, in the same foldersrcwith our component we will create a regular CSS file with stylesstyles.cssfor our tags.

In this file we will create a class for the divclass1with styles:

.class1 { width: 200px; border: 2px solid brown; padding: 10px; text-align: center; }

Now let's add the classclass2with styles for the first paragraph:

.class2 { color: orangered; font-weight: bold; }

Classclass3with styles for the second paragraph:

.class3 { font-style: italic; color: brown; }

And finally, let's create a classclass4for the last paragraph:

.class4 { background-color: orange; font-weight: bold; color: white; }

We're done with the CSS file. All that remains is to apply our CSS styles that we wrote for the tags. Let's go back to our fileApp.jswith the component.

The first thing we must do is add the import line of our style file to the beginning of the filestyles.css:

import './styles.css';

Now, to apply the CSS classes from the file to the component, we will use the attributeclass. For the first paragraph we had a classclass2, apply it:

<p class="class2">text</p>

Let's add classes for the remaining tags in a similar way. As a result, we get the following code:

<div class="class1"> <p class="class2">text</p> <p class="class3">text</p> <p class="class4">text</p> </div>

Create a React component that will contain a div, and the div will have two buttons. Create a filestyles.csswith CSS styles for your tags. Apply the written styles to the component tags.
