Inserting data into the style attribute from variables in React

In the last lesson we wrote an object directly in the attributestyle. We can also use data insertion into this object from variables.

Let's take our React component without CSS styles:

function App() { return ( <div> <p>text</p> <p>text</p> <p>text</p> </div> ); }

Now we will create variables and assign CSS property values ​​to them. Let's start in order with the styles for the diva.

In the component file before the commandreturnlet's create a variablewd1and assign it a value'200px':

const wd1 = '200px';

In a similar way, we will create variables for all property values ​​of our div:

const wd1 = '200px'; const br1 = '2px solid brown'; const pd1 = '10px'; const ta1 = 'center';

Now let's add variables for the first paragraph:

const co1 = 'orangered'; const fw1 = 'bold';

For the second paragraph:

const fs1 = 'italic'; const co2 = 'brown';

And finally for the last one. Here you only need two, not three, variables. It makes no sense to repeat myself, because the meaningboldwe already have for the first paragraph:

const bco1 = 'orange'; const co3 = 'white';

Now we will substitute variables with style values. Let's do this for the first paragraph. Let's insert variablesco1Andfw1instead of values ​​in the object:

<p style = {{ color: co1, fontWeight: fw1 }}> text </p>

We will do the same with the rest of the tags.

As a result, our component code will look like this:

function App() { const wd1 = '200px'; const br1 = '2px solid brown'; const pd1 = '10px'; const ta1 = 'center'; const co1 = 'orangered'; const fw1 = 'bold'; const fs1 = 'italic'; const co2 = 'brown'; const bco1 = 'orange'; const co3 = 'white'; return ( <div style = {{ width: wd1, border: br1, padding: pd1, textAlign: ta1 }}> <p style = {{ color: co1, fontWeight: fw1 }}> text </p> <p style = {{ fontStyle: fs1, color: co2 }}> text </p> <p style = {{ backgroundColor: bco1, fontWeight: fw1, color: co3 }}> text </p> </div> ); }

Modify the code for solving the problem from the previous lesson so that the values ​​of CSS properties are stored in variables.
