Advanced Component Styling in React

Let's say we have a React componentBlockand we created components in it styled using Styled ComponentsButtonAndContainer:

const Container = styled.div` display: flex; flex-direction: column; width: 150px; `; const Button = styled.input` background-color: orange; font-size: 18px; margin: 5px; `;

Now imagine that we need one more componentButton, only with white button text and a green background.

To do this, we just need to use our componentButtoncreate a new componentMdButtonand write in it only those properties that we want to change:

const MdButton = styled(Button)` color: white; background-color: green; `;

Let's place it inBlockall our styled components:

function Block() { return ( <Container> <Button>btn0</Button> <MdButton>btn1</MdButton> </Container> ); }
