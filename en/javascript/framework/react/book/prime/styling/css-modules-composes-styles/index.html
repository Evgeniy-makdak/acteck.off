Composes command for classes in CSS modules in React

In this tutorial we will look at a very useful technique for reusing CSS styles from one class in another in CSS modules.

Back to the applicationbuttonswhich we did in previous lessons. Let's look at the fileButtons.module.css:

.btn1 { background-color: orange; border: 2px solid brown; color: white; } .btn2 { background-color: red; border: 2px solid green; color: yellow; } .btn3 { background-color: brown; border: 2px solid yellowgreen; color: orange; }

Let's say we want all buttons to have the same rounding and font size. It would, of course, be possible to create identical properties in each class. But thanks to the teamcomposesfrom CSS modules, we will do things differently.

First, let's create another class, for examplecommon-btn, into which we place these identical properties. And to apply properties from this class to others, you need to use the syntax in them:

селектор { composes: название класса; }

Now our codeButtons.module.csswill look like this:

.common-btn { font-size: 16px; border-radius: 3px; } .btn1 { composes: common-btn; background-color: orange; border: 2px solid brown; color: white; } .btn2 { composes: common-btn; background-color: red; border: 2px solid green; color: yellow; } .btn3 { composes: common-btn; background-color: brown; border: 2px solid yellowgreen; color: orange; }

Take your React applicationinputswhich you did in the problems for previous lessons. Add a couple of identical CSS styles with the same values ​​for all inputs. Create an additional class in which these styles will be written. By usingcomposesreplicate them across all other classes for inputs.
