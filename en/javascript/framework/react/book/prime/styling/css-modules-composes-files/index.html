Composes command for files in CSS modules in React

Teamcomposescan also be used to reuse styles from one file in another.

Back to the applicationbuttonswhich we did in previous lessons. Let's say we want to add the same shadows, cursor style and bold font to all buttons. Let these styles be located in the fileApp.module.css. Let's open this file and add a new class at the beginningbeautywith these styles:

.beauty { box-shadow: rgba(50, 50, 50, 0.2) 0 5px 5px 0; font-weight: bold; cursor: pointer; }

Let's use it to style the buttons. For this we will openButtons.module.cssand make changes to the classcommon-btn. We will add a line with the commandcomposes, where we indicate the name of the classbeauty, which we want to apply and the fileApp.module.csswhich contains this class:

.common-btn { composes: beauty from "../App.module.css"; font-size: 16px; border-radius: 3px; }

Take your application code from the problem in the last lesson and make some kind of shadow to your inputs using the method given in this lesson.
