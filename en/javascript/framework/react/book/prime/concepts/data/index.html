Component data in React

Component data can be stored in three places: in props, in component variables and constants, and in states. Props store data that is sent to a component from outside by its parent component. At the same time, props inside the component must remain unchanged.

Variables, constants, and states should store local data that is personally important to the component and that the parent should not know about. At the same time, states should store data that can change in various events and at the same time you want the changes to be reactive.
