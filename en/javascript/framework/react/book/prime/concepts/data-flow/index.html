Unidirectional data flow in React

A component can pass its state down the child tree in the form of props to child components. This process is calleddownward (top-down) orunidirectional (unidirectional) data stream. A state always belongs to a specific component, and that state can only affect components that are descendants of that component.

In addition to downstream data flow, there is alsoascendingorback. It contains data that appears in descendant components, for example, entered into inputs by the user. This data stream is passed to the parent by calling a function passed in the props of the child component.
