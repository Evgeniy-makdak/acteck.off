JSX tag closure

All tags in JSX must be closed, including tags that do not require a pair, such asinputorbr.

The following code example will throw an error because the input is not closed:

function App() { return <div> <input> </div>; }

Let's close it with a backslash:

function App() { return <div> <input /> </div>; }

The function in your main component should now look like this:

function App() { return <div> text </div>; }

Add several inputs to the div, separated by tagsbr.
