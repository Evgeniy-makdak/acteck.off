Features of storing multiple tags in JSX constants

It is necessary to take into account that several tags stored in a constant must be wrapped in some kind of common tag. That is, the code will not work like this:

function App() { const str = <p>text1</p><p>text2</p>; return <main> {str} </main>; }

But this is how it will be, since the paragraphs are wrapped in one tagdiv:

function App() { const str = <div><p>text1</p><p>text2</p></div>; return <main> {str} </main>; }

You can also use empty tags:

function App() { const str = <><p>text1</p><p>text2</p></>; return <main> {str} </main>; }

The following code is given:

function App() { const li1 = <li>text1</li>; const li2 = <li>text2</li>; const li3 = <li>text3</li>; }

Make the rendering result look like this:

<ul> <li>text1</li> <li>text2</li> <li>text3</li> </ul>
