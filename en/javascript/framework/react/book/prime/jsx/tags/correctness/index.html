Correct layout in React

Layout in JSX must be correct. In particular, not all tags can be nested within each other. For example, if in the tagulplace a paragraph will result in an error.

In addition to the more or less obvious invalid nestings, there are also unexpected ones: tables.

In tables we are used to going straight into the tagtableinsert tagstr, like this:

function App() { return <table> <tr> <td>1</td> <td>2</td> </tr> <tr> <td>3</td> <td>4</td> </tr> </table>; }

This code in React will result in a warning in the console becausetrmust be nested either in a tagtbody, or inthead, or intfoot.

Let's fix the problem by making our table correct:

function App() { return <table> <tbody> <tr> <td>1</td> <td>2</td> </tr> <tr> <td>3</td> <td>4</td> </tr> </tbody> </table>; }

Make a table with three rows and three columns.
