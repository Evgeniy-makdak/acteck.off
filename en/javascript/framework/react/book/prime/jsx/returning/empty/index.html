Function returning an empty tag in JSX

Let's say we want to return an empty tag:

function App() { return <div></div>; }

In this case, the code can be shortened like this:

function App() { return <div />; }

At the same time, in the resulting HTML code, React automatically converts the abbreviated form into the normal one.

The author of the following code wants to return three inputs:

function App() { return <input><input><input>; }

The code, however, doesn't work. Fix the code author's mistake.
