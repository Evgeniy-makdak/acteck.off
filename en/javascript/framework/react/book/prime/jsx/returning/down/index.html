Rules for a function to return when moving a tag down in JSX

The opening tag must be written on the same line as the commandreturn. For example, the following code will not work:

function App() { return <div> <p>text1</p> <p>text2</p> </div>; }

In order for this tag to move downward to work, our tag must be placed in parentheses:

function App() { return ( <div> <p>text1</p> <p>text2</p> </div> ); }

In the following code, the coder wants to return the ul tag:

function App() { return <ul> <li>text1</li> <li>text2</li> <li>text3</li> </ul>; }

The code, however, doesn't work. Fix the code author's mistake.
