Returning an unclosed JSX tag

As a result, you can return a tag that does not need to be closed, for example,input. As you already know, according to React rules, in this case in such a tag the closing slash is written in the opening tag:

function App() { return <input />; }

The author of the following code wants to return an input:

function App() { return <input>; }

The code, however, doesn't work. Fix the code author's mistake.
