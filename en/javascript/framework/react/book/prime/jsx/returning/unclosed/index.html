Возврат незакрытого тега JSX

В качестве результата можно возвращать тег, который не нужно закрывать, например, input. Как вы уже знаете, по правилам React в этом случае в таком теге слеш для закрытия пишется в открывающем теге:

function App() { return <input />; }

Автор следующего кода хочет вернуть инпут:

function App() { return <input>; }

Код, однако, не работает. Исправьте ошибку автора кода.
