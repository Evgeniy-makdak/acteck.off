Introduction to JSX in React

The JSX language is regular JavaScript, but with some additions that allow you to write tags directly in the code, without quotes.

Tags can be returned viareturn:

function App() { return <div> text </div>; }

Tags can have attributes:

function App() { return <div class="eee"> text </div>; }

Tags can be written to variables or constants:

function App() { const elem = <div>text</div>; return elem; }

Your main component has the following function:

function App() { return <div> text </div>; }
