Parameters in a JSX function

Let our functionshowMesstakes the name of the person we want to say hello to as a parameter:

function showMess(name) { alert('hello, ' + name); }

You can pass this parameter when binding a function to an event. To do this, our function call should be wrapped inarrow function:

function App() { function showMess(name) { alert('hello, ' + name); } return <div> <button onClick={() => showMess('user')}>show</button> </div>; }

This way we can bind the same function to several buttons with different parameters:

function App() { function showMess(text) { alert(text); } return <div> <button onClick={() => showMess('user1')}>show1</button> <button onClick={() => showMess('user2')}>show2</button> </div>; }

Three buttons are given:

function App() { return <div> <button>act1</button> <button>act2</button> <button>act3</button> </div>; }

Make it so that when you click on the first button throughalertnumber was displayed1, by clicking on the second button - number2, and by clicking on the third - the number3.
