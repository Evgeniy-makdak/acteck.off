Inverting for shorthand conditions in JSX

The opposite situation may also occur: you need to insert text if the condition is false, and do nothing if it is true. To do this, you need to invert the constant using the operator!.

In the following example, if inhidewillfalse, then a paragraph with the text will be inserted:

function App() { const hide = false; return <div> {!hide && <p>text</p>} </div>; }
