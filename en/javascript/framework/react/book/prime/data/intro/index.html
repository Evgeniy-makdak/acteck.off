Data reactivity in React

In React, we will store our data in various structures: in arrays, in objects, in arrays of objects. These structures in React are reactive. React reacts reactively to all data changes: adding, deleting, changing elements, as well as changing their order.

At the same time, the React rulesit is prohibited to change the data of arrays and objects from the state. You need to make a copy of the data, change it and write it back to the state. This means that our data must be immutable. Read aboutimmutability of arrays, objects and arrays of objectsin the corresponding section of the JavaScript tutorial.
