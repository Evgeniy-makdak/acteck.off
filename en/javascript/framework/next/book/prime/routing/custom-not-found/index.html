Кастомизация 404 ошибки в NextJS

В предыдущем уроке мы видели, что в NextJS уже встроена красиво оформленная страница 404 ошибки.

Но при желании мы можем создать свою страницу для ошибки, оформив ее в нужном нам стиле (говорят, кастомизировать).

Для этого нужно создать файл со специальным именем not-found.jsx и разместить его в папке app:

export default function NotFound() { return <> <h1>Error 404</h1> <p> this page is not found </p> </> }

Сделайте кастомный вывод 404 ошибки. Убедитесь, что в случае ошибки отдается ваш файл, а не встроенный в NextJS.
