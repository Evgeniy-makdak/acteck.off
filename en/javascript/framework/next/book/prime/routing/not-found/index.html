Built-in 404 error in NextJS

Let the user enter a URL into the address bar of the browser that does not exist on our site. In this case we have to show him the error page.

In addition, we must give the HTTP status with the code404- this is important for SEO, so that search engines that follow a broken link understand that such a page does not exist.

NextJS does this automatically. Try it! Enter a non-existent address and you will see an error page. If you go to the browser debugger and open the Network tab, you will see404response status.
