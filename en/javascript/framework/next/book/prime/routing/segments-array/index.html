An array of segments in a dynamic route in NextJS

If necessary, you can get an array of all segments into one variable. To do this, the parameter is placed in square brackets and an ellipsis is placed before the parameter name. Let's look at an example.

Let us have a route like this:/prod/:category/:name/:id/, where parameters marked with a colon are dynamic. Let's get an array of values ​​of these parameters into some variable. The name of this variable can be anything. For example, let's call itslugs.

Let's make the following file structure:

  • /app/
    • /prod/
      • /[...slugs]/

Let's make the corresponding component:

export default function Prod({ params }) { console.log(params); // массив значений }

Make a route that processes addresses like/prod/:category/:name.

Let the following array be given:

let prods = [ { category: 'catg1', name: 'prod1', cost: 100, }, { category: 'catg1', name: 'prod2', cost: 200, }, { category: 'catg2', name: 'prod1', cost: 300, }, { category: 'catg2', name: 'prod3', cost: 400, }, ];

Make sure that depending on the value of the parameters in the component layout, the corresponding product is displayed.
