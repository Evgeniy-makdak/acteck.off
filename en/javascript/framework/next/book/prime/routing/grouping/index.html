Grouping routes in NextJS

Sometimes we may need to conveniently group several routes into one folder so that the grouping folder does not create a separate segment in the URL.

To do this, the name of the grouping folder should be placed in parentheses. Let's look at an example.

Let by URL/regwe want to register, and by URL/log- authorization. Let's combine the folders corresponding to these URLs into one common folder called(auth).

This gives us the following file structure:

  • /app/
    • /(auth)/
      • /reg/
        • page.jsx
      • /log/
        • page.jsx

Let's create a file for the registration page:

export default function Reg() { return <h1>register</h1>; }

Let's create a file for the login page:

export default function Log() { return <h1>login</h1>; }

Perform the manipulations described in the lessons. Make sure everything works as expected.
