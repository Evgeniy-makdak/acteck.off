Default layout in NextJS

In the previous lesson, we found out that we must have a general site layout into which the content, title and meta description of the page are inserted. NextJS by default already has a file with the general layout of the site. It's in the fileapp/layout.jsx.

After installation, there is already some code in the layout file. This code, however, contains a lot of unnecessary stuff. Let's make a minimal website layout that will be easy to understand. To do this, open the layout file, delete everything from it and paste the following:

export const metadata = { title: 'Test site', description: 'This is my page description.', }; export default function RootLayout({children}) { return ( <html lang="en"> <body> {children} </body> </html> ); }

Correct the default content in the main site layout.
