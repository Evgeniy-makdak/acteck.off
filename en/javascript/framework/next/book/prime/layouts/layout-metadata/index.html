Metadata in the main site layout in NextJS

So, in the previous lesson we found out that there are different ways to insert metadata. The simplest one is to define them in the layout file using the following construct:

export const metadata = { title: 'Test site', description: 'This is my page description.', };

In this case, the title and meta description will change to those specified by us. However, this approach has a significant problem. It lies in the fact that metadata is hard-coded into the main layout of the site. That is, it turns out that for each URL they will be the same. This is wrong both from an SEO point of view and from a site usability point of view.

Change the title in your main site layout. Check that it has changed.

Change the meta description in your main site layout. Then open the source code of the site and check that it has changed.
