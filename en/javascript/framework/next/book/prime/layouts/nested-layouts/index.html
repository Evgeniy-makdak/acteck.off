Nested Layouts in NextJS

As you know, in the folder/appin the file/layout.jsxThe main layout of the site is located.

If you wish, however, you can create your own layouts in subfolders. In this case, the page content will be inserted into the sublayout first, and then the resulting result will be inserted into the main layout.

Let's try it in practice. Let our main layout look like this:

export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> header </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }

Let's make another, nested layout for all content from the folder/users:

export default function UsersLayout({children}) { return ( <div className="info"> message for all users </div> <div className="content"> {children} </div> ); }

As you can see, in this layout we do not specifymain page tags, since they are already in our main site layout.

Let's also make a file with content to display to the user:

export default function User() { return <> <h1>User</h1> <p> user description </p> </>; }

After these manipulations, our file structure will look like this:

  • /app/
    • layout.jsx
    • /users/
      • layout.jsx
      • /show/
        • page.jsx

Now, if we access the appropriate URL, NextJS will put everything together and in the browser we will get the following final HTML code:

<html lang="en"> <body> <header> header </header> <main> <div class="info"> message for all users </div> <h1>User</h1> <p> user description </p> </main> <footer> footer </footer> </body> </html>

Make a section with users and a section with posts on the site. Create your own layout for each section, which will inherit from the main site layout.
