Content in the site layout in NextJS

Let's look at the basic layout of the site, which we started looking at in the last lesson:

export default function RootLayout({children}) { return ( <html lang="en"> <body> {children} </body> </html> ); }

As you can see, the site layout is a component with a function. In this case, the object from which we extract the key is passed to the function parameterchildrento the corresponding variable. This variable stores the contents of the filepage.jsx, which matches the requested URL.

That is in the filelayout.jsxWe store the site layout, and using a variablechildrenwe indicate the location in the site layout where the page content should be inserted.

For example, let's format the page content in some tag:

export default function RootLayout({children}) { return ( <html lang="en"> <body> <main> {children} </main> </body> </html> ); }
