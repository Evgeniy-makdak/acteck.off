Importing components into NextJS

You can import not only data files, but also components, just as we are used to when working in regular React. Let's see how it works.

First, let's introduce some conventions. As you already know, in the foldersrc/appsite pages are posted. Let's place the child components in a foldersrc/comp.

Let's now connect the child component to some page. Let's say we have a page like this:

export default function Test() { return <> <h1>Test</h1> </>; }

Let's say we have the following child component:

export default function Child() { return <p> child </p>; }

Let's import our child component to our page:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> </>; }

Let's insert the imported component. To do this, we use a tag with the name of the component variable:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> <Child /> </>; }

Create several child components. Import them to your pages.
