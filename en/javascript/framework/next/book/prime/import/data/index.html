Importing data into NextJS

In NextJS we can import and export arbitrary files. You should already be able to do this if you have worked withES6 modules. It is logical that this will work in NextJS, but this is not always obvious to beginners. So let's talk about imports.

For example, let's say we have a page. On this page we want to display some data. Let, for example, we have some array with data. Let this data be so large that it seems convenient to us to put it in a separate file.

The name of this file can be anything. Let it bedata.js. In this file we will place an array with data. And in the filepage.jsxwe will have a page that will be processed by NextJS itself and sent to the browser.

That is, we will have the following file structure:

  • /src/
    • /app/
      • /test/
        • page.jsx
        • data.js

We will place it in our filedata.jssome array with data:

export default [ 1, 2, 3, 4, 5, ];

Let's make a pagepage.jsx:

export default function Test() { return <h1>Test</h1>; }

Let's connect an array with data to our page:

import data from './data.js'; export default function Test() { return <h1>Test</h1>; }

Let's output this data in a loop:

import data from './data.js'; export default function Test() { let list = data.map(function(item) { return <li>{item}</li> }); return <> <h1>Test</h1> <ul> {list} </ul> </>; }
