Проект Витрина в NextJS

Давайте реализуем проект с витриной интернет-магазина.

Витрина должна представлять собой плиточку из карточек товаров. В каждой карточке должна быть картинка товара, название товара и его цена.

Карточка должна быть ссылкой, ведущей на страницу товара. На странице товара должно быть полное описание товара.

Сделайте файл с многомерной структурой данных, в которой будут хранится товары.

На главной странице сайта выведите список всех товаров нашей витрины.

Реализуйте страницу товара.
