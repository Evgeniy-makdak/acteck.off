Country-City Project in NextJS

Let's implement a project with countries and cities, as well as their descriptions.

The description of the country must include the language of the country, area, and population.

For each country, you need to store an array of cities in that country.

The description of the city must include area, population, and year of foundation.

Make a file with a multidimensional data structure that will store countries and their cities.

On the main page of the site, display a list of all countries in the form of links to country pages.

On the country page, display a description of the country, as well as a list of its cities in the form of links to city pages.

On the city page, display a description of the city, as well as a link to the country page for this city.
