Build Modes in NextJS

In the previous tutorial we installed the framework. Now we need to run our project.

Before that, however, let's discuss the build modes available to us. NextJS has two modes: modedevelopmentand modeproduction.

When you develop your project on your local computer, you need to run the project in development mode. In this mode, the framework will work in a manner convenient for the programmer.

When the project is ready, it will need to be uploaded to hosting so that it can be accessed via the Internet. In this case, we will need to build it in production mode.

In the following lessons we will discuss the features of these two assembly options.
