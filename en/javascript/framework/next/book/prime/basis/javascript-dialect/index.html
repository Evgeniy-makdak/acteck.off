JavaScript dialect in the NextJS framework

Development in NextJS can be carried out either onpure JavaScript, or onTypeScript. The language dialect can be selected when installing NextJS.

Currently, TypeScript development is more popular. To keep things simple, however, we'll start with pure JavaScript, and then in a separate section of the tutorial we'll talk about what we'll need to change in our code to move to TypeScript.
