Introduction в фреймворк NextJS

NextJS - фреймворк для веб-разработки с открытым исходным кодом, написанном на основе React. К основным преимуществам NextJS относятся встроенная маршрутизация, оптимизированный рендеринг как на стороне клиента, так и сервера, удобная сборка приложения для работы в продакшене. Благодаря современной архитектуре веб-разработки, основанной на JavaScript, API и предварительно отрендеренной разметке, NextJS работает быстрее и эффективнее React.

Фреймворк используется для создания разнообразных полнофункциональных веб-приложений - от одностраничного сайта до соцсетей и интернет-магазинов.

В данном учебнике будут рассмотрены основные аспекты работы с фреймворком NextJS. Однако для его изучения вам необходимо знать чистый JavaScript, работу с React, основы командной строки, работу с npm и npx, работу с модулями ES6, основы работы с протоколом HTTP, работу со сборщиком Vite.
