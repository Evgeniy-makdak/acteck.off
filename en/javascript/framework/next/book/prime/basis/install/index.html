Installing the NextJS framework

In this tutorial we will install NextJS. To install this framework you will need NodeJS version22or higher.

To start the installation interminalyou need to run the following command:

npx create-next-app@latest

After this, the installation process will start and you will be asked to answer a series of questions that will configure your framework.

Question Answer
What is your project named? Here we need to give our project a name. You can set anything. Writemy-app.
Would you like to use TypeScript? Here we are encouraged to use TypeScript. We won't do this since we'll be working in JavaScript for now for simplicity.
Choose No.
Would you like to use ESLint? ESLint is a special library that checks that your code complies with design standards.
Select Yes.
Would you like to use Tailwind CSS? Tailwind is a custom CSS framework for React.
Select Yes.
Would you like to use `src/` directory? This asks if we want our project to be in the foldersrc.
Select Yes.
Would you like to use App Router? Includes a modern routing option.
Select Yes.
Would you like to customize the default import alias (@/*)? This setting allows you to simplify file paths when using file imports.
Select Yes.

After answering all the suggested questions, the installation of the framework will begin. Once the project is created, go to its folder:

cd my-app

Create a NextJS project.
