Working with Tailwind in NextJS

In NextJS you can use the CSS framework for stylingTailwind, which is often used with React. Let's see how to use it.

To work with Tailwind styles, you need two files:tailwind.config.jsAndpostcss.config.js. Since we selected the “use Tailwind styles” option when installing the project, these files are already generated in our project and are located in its root. If there are no such files, then you need to install Tailwind according toofficial documentation.

Let's add bold and yellow to the paragraph text using Tailwind styles. In Tailwind CSS, the numbers used in color classes indicate the intensity or hue of the color. These numbers range from50to900. For the paragraph text, choose a medium shade:

export default function Test() { return <p className="font-bold text-yellow-500"> text text text </p>; }

Try using Tailwind styles in your project.
