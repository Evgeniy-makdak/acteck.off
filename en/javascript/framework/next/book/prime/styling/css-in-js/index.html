CSS-in-JS in NextJS

NextJS also allows you to use the CSS-in-JS approach. In this approach, the CSS code is written directly in the component file. You have already encountered this approach in the React tutorial in the lesson aboutStyled Components.

There are various libraries available for the CSS-in-JS approach in React. You can get acquainted with their connection on the pageNextJS documentation on CSS-in-JS.
