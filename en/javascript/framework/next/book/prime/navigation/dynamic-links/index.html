Dynamic link generation in NextJS

Let's say we have some file with user data:

export default users = [ { id: 1, name: 'name1', surn: 'surn1', }, { id: 2, name: 'name2', surn: 'surn2', }, { id: 3, name: 'name3', surn: 'surn3', }, ];

Let's make two components. The first component will show a list of users with links to their detailed description. The second component will output this detailed description.

Let by URL/users/lista list of all users will be shown, and by URL/users/show/:id- a specific user by his ID.

To do this, we will create the following file structure:

  • /app/
    • /users/
      • users.js
      • /list/
        • page.jsx
      • /show/[id]/
        • page.jsx

Let's create the first component in which we will dynamically generate links in a loop, substituting in themideach user:

import users from '../users.js'; import Link from 'next/link'; export default function List() { let list = users.map(user => { return <li> <Link href={`/users/show/${user.id}`}> {user.name} </Link> </li>; }); return <ul> {list} </ul>; }

Let's make a component to display a specific user by hisid:

import users from '../../users.js'; export default function User({params}) { let user = users[params.id]; return <div> <span>{user.id}</span> <span>{user.name}</span> <span>{user.surn}</span> </div>; }
