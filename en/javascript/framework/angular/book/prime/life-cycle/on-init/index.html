ngOnInit hook in Angular

HookngOnInitFires when the component is initialized. It is similar in action to a class constructor, but you can perform more complex tasks in it, for example, loading data from the server.

Let's see how to use this hook. First you need to import its interface:

import { OnInit } from '@angular/core';

Next you need to include the interface in the component class:

export class UserComponent implements OnInit { }

After this, we can write a method in the component classngOnInitand this method will be executed automatically when the class is initialized. Let's check its operation:

export class UserComponent implements OnInit { constructor() { console.log('constructor is started'); } ngOnInit() { console.log('onInit is applied'); } }

Make the method privateshowand call it upon component initialization.
