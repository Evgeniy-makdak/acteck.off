Binding event handlers in the Angular template

In Angular, event handlers are bound to tags in a special way - using attributes. The name of the event is indicated in parentheses, and the attribute value calls the component class method that should be executed on this event.

For example, let's make sure that when a button is clicked, the method will be executedshowcomponent class:

<button (click)="show()"> button </button>

Let's now write the implementation of the methodshowin the component class:

export class AppComponent { public show(): void { alert('test'); } }

Make two buttons. By clicking on the first one, greet the user, and by clicking on the second one, say goodbye to him.
