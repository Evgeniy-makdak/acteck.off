Event object in an event handler in an Angular template

In event handlers you can receive an objectEvent. To do this, you need to pass a special variable to the parameter of the bound method$event(with dollar at the beginning):

<button (click)="show($event)"> button </button>

Let's get the passed object in a class method:

export class AppComponent { public show(event: MouseEvent): void { console.log(event); } }

By clicking, get the object with the event and display it in the console.
