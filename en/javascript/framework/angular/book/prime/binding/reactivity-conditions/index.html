Reactivity of conditions in Angular

Reactivity will also work with conditionsngIf. This means that we can show or hide the block depending on what is contained in the class property.

Let's try it in practice. Let us have the following block

<div> text </div>

Let's show or hide it according to the condition:

<div *ngIf="isShow"> text </div>

Let's now make two buttons. By clicking on the first one we will show our block, and by clicking on the second we will hide it:

<button (click)="show()"> show </button> <button (click)="hide()"> hide </button>

Let's add a property to the component class that will regulate whether the block is shown or not:

export class AppComponent { public isShow: boolean = true; }

Now let’s write the implementation of our methods for showing and hiding the block:

export class AppComponent { public isShow: boolean = true; public show(): void { this.isShow = true; } public hide(): void { this.isShow = false; } }

Make a button that, when clicked, will toggle the block: show if it is hidden, and hide if it is shown.
