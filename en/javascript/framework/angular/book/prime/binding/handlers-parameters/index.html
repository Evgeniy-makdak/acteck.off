Параметры обработчиков событий в темплейте Angular

В методы-обработчики событий можно передавать параметры. Давайте, например, передадим какую-нибудь строку:

<button (click)="show('test')"> button </button>

В методе класса компонента зададим параметр, указав ему строковый тип:

export class AppComponent { public show(arg: string): void { alert(arg); } }

По клику на кнопку передайте параметром метода два числа. В методе выведите сумму этих чисел в консоль.
