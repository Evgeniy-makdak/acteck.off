Event handler parameters in the Angular template

You can pass parameters to event handler methods. Let's, for example, pass some string:

<button (click)="show('test')"> button </button>

In the component class method, we will set a parameter, indicating its string type:

export class AppComponent { public show(arg: string): void { alert(arg); } }

By clicking on the button, pass two numbers as a parameter to the method. In the method, print the sum of these numbers to the console.
