Two-way data binding in Angular

In Angular, you can make it so that as you enter text into an input, this text automatically ends up in a class property. It's calledtwo-way data binding.

For this binding to work, you must first enable it. To do this, import in the component fileFormsModule:

import { FormsModule } from '@angular/forms';

And then add it to the propertyimportsdecorator@Component:

@Component({ selector: 'app-root', standalone: true, imports: [FormsModule, UserComponent], templateUrl: './app.component.html', styleUrl: './app.component.css' })

After this, two-way data binding will be enabled and we can use it. Let's do this.

First, let's declare a class property, which we will then subject to two-way binding:

export class AppComponent { public text: string = ''; }

Let us have a div and an input. Let a div display some class property:

<div> {{ text }} </div> <input>

Let's bind our propertytextto change the input. To do this, you need to write a special directive in the input[(ngModel)], the value of which should indicate the property of our class:

<div> {{ text }} </div> <input [(ngModel)]="text">

Now, if you run the code and start entering text into the input, this text will immediately begin to display in the div.
