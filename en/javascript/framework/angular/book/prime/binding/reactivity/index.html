Reactivity in Angular

For any changes to public properties, the template will reactreactively, that is, to change instantly. This is how it worksreactivityin Angular.

Let's look at an example. Let's say we have a property containing text:

export class AppComponent { public text: string = ''; }

Let's display our property in some tag:

<div> {{ text }} </div>

Now let's create a button that, when clicked, will call the class method:

<button (click)="show()"> show </button>

In this method we change the text:

export class AppComponent { public text: string = ''; public show(): void { this.text = 'hello'; } }

Now, if you run the code and click on the button, then immediately after clicking the text in the div will change.

Make a div and two buttons. Make sure that when you click the first button, one text appears in the div, and when you click on the second, another text appears.
