Операторы сравнения в директиве if в Angular

В директиве *ngIf можно использовать привычные нам операторы сравнения.

Давайте посмотрим на примере. Пусть у нас есть следующая числовая переменная:

export class AppComponent { public num: number = 1; }

Покажем блок текста в зависимости от условия:

<div *ngIf="num === 1"> text </div>

Пусть в переменной age хранится возраст пользователя. Покажите текст для взрослых, если его возраст больше 18 лет.
