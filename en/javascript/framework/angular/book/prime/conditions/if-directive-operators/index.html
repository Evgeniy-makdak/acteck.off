Comparison Operators in If Directive in Angular

In the directive*ngIfYou can use the familiar comparison operators.

Let's look at an example. Let us have the following numeric variable:

export class AppComponent { public num: number = 1; }

Let's show a block of text depending on the condition:

<div *ngIf="num === 1"> text </div>

Let in the variableageThe user's age is stored. Show adult text if age is older18years.
