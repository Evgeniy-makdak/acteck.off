Конструкция else-if в Angular

В Angular также доступна конструкция @else-if:

@if (isAdmin) { <div> admin </div> } @else if (isUser) { <div> user </div> } @else { <div> guest </div> }

Пусть в свойстве day лежит какое-то число из интервала от 1 до 31. Определите в какую декаду месяца попадает это число (в первую, вторую или третью).
