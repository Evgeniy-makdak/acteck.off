If Directive in Angular

In Angular, you can make a tag show or hide depending on the value of a variable. A special directive is used for this*ngIf. Let's see how to work with it.

First, you need to import this directive:

import {NgIf} from "@angular/common";

And add it to the imports section in the decorator:

@Component({ ..... imports: [....., NgIf], .... })

Let us now have the following variable:

export class AppComponent { public isAdmin: boolean = true; }

Let's make a div that will be shown if this variable has a valuetrue:

<div *ngIf="isAdmin"> text </div>

Now let's make the div show if our variable has a valuefalse:

<div *ngIf="!isAdmin"> text </div>

Make a propertyisAdult. Show adult text if our variable contains a valuetrue.
