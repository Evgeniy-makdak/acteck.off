if construct in Angular

The condition in Angular can also be done using a special construct@if. This design allows you to show or hide blocks of code based on condition. Let's see how to work with it.

Let us have the following variable:

export class AppComponent { public isAdmin: boolean = true; }

Let's make a div that will be shown if this variable has a valuetrue:

@if (isAdmin) { <div> text </div> }

Let's make the div show if our variable has a valuefalse:

@if (!isAdmin) { <div> text </div> }

Make a propertyisAdult. Show adult text if our variable contains a valuetrue.
