Конструкции switch и case в Angular

В Angular также доступны конструкции @switch, @case и @default:

@switch (str) { @case ('a') { <div> aaa </div> } @case ('b') { <div> aaa </div> } @case ('c') { <div> ccc </div> } @default { <div> error </div> } }

Пусть в свойстве lang лежит название языка: 'ru', 'en' или 'de'. В зависимости от содержимого покажите фразу на заданном языке или ошибку, если такого языка нет.
