Передача текста в дочерний компонент в Angular

Можно передавать текст из родительского компонента в дочерний. Для этого нужно передаваемый текст написать между открывающим и закрывающим тегом дочернего компонента. Давайте сделаем это:

<user-data> text from parent </user-data>

Теперь в темплейте дочернего компонента мы должны указать место, в которое будет осуществлена вставка переданного родительского текста. Это делается с помощью специального тега ng-content. Давайте поставим этот тег в темплейте дочернего компонента:

child text <br> <ng-content></ng-content>

Пусть в родительском компоненте есть некоторое свойство. Вставьте значение этого свойства в текст. Передайте этот текст описанным в уроке способом в дочерний компонент.
