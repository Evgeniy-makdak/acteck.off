Changing child component tag in Angular

If desired, you can change the tag name to call the child component inside the parent template. This name is set in the propertyselectordecorator@Componentdescendant In our case this is the tagapp-user:

@Component({ selector: 'app-user', // вот оно templateUrl: './user.component.html', styleUrls: ['./user.component.css'] })

Let's change it to something else:

@Component({ selector: 'user-data', // меняем templateUrl: './user.component.html', styleUrls: ['./user.component.css'] })
