Смена тега дочернего компонента в Angular

При желании можно сменить имя тега для вызова дочернего компонента внутри темплейта родителя. Это имя задано в свойстве selector декоратора @Component потомка. В нашем случае это тег app-user:

@Component({ selector: 'app-user', // вот оно templateUrl: './user.component.html', styleUrls: ['./user.component.css'] })

Давайте поменяем его на другое:

@Component({ selector: 'user-data', // меняем templateUrl: './user.component.html', styleUrls: ['./user.component.css'] })

Смените тег для вызова вашего дочернего компонента.
