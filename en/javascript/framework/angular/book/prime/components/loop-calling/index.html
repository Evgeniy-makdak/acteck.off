Calling a component in a loop in Angular

You can also call a component in a loop, passing various parameters to it as attributes. Let's try it in practice. Let us have the following property with users:

interface User { name: string, age: number } class UserComponent { public users: User[] = [ { name: 'user1', age: 25, }, { name: 'user2', age: 26, }, { name: 'user3', age: 27, }, ]; }

Let's go through the users in a loop, in each iteration calling the child component with the user and passing it the name and age as parameters:

<div *ngFor="let user in users"> <user-data [name]="user.name" [age]="user.age"></user-data> </div>

Given the following array of products:

interface Product { name: string, cost: number } class ProductComponent { public products: Product[] = [ { name: 'prod1', cost: 100, }, { name: 'prod2', cost: 200, }, { name: 'prod3', cost: 300, }, ]; }

Loop through this array, each time creating a new component with a product.
