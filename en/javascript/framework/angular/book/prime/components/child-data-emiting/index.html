Passing data to parent component in Angular

You can pass data from a child component to a parent component. This is done by emitting events using a special objectEventEmitterand methodemit. The child component will emit events by passing some data along with the event, and the parent component will catch events by receiving data from the parent component.

Let's try it in practice. To do this we need to go through a number of steps. First, we import the decorator into our child component@Outputand classEventEmitter:

import { Component, Output, EventEmitter } from '@angular/core';

Now in the child class we will create a propertyonData, whose value will be a new objectEventEmittercontaining the line:

export class UserComponent { @Output() public onData = new EventEmitter<string>(); }

Now let's make a methodsend, whose call will emit an eventonDatato the parent component with the given text:

export class UserComponent { @Output() public onData = new EventEmitter<string>(); send() { this.onData.emit('xxx'); } }

In the template of the child component we will create a button, upon clicking which the method will be calledsend:

<button (click)="send()"> send data </button>

Let's create a method in the parent componentonData, which will be automatically called when the corresponding event is emitted from the child component. The method parameter will contain the passed data:

export class AppComponent { public onData(data: string) { console.log(data); } }

In the template of the parent component, we bind the event to the tag of the child component:

<app-user (onData)="onData($event)"></app-user>
