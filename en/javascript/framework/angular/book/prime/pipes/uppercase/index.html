Converting to uppercase via pipe in Angular

Using the channelUpperCasePipeYou can convert the string to lower case. Let's try it in practice. Let's say in the component class we have the following line:

export class AppComponent { name: string = 'john'; }

Let's display our line in the template file, using our channel in it:

<div> {{ name | uppercase }} </div>

Result of executing the code:

<div> JOHN </div>

Let's say you have an array:

export class AppComponent { users: string[] = ['john', 'alex', 'kate', 'mary']; }

Print each element of the array in uppercase.
