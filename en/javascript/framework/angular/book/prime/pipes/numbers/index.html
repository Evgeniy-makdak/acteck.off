Formatting Numbers Using Channels in Angular

Using the channelDecimalPipenumbers can be formatted. In the first channel parameter we pass the minimum number of digits in the integer part. In the second parameter we indicate the minimum number of digits in the fractional part. In the third optional parameter we set the maximum number of digits in the fractional part.

Despite the fact that our channel is calledDecimalPipe, to call it we write like this:number.

Let's try it in practice. In the component class, set the following number:

export class AppComponent { num: number = 18; }

Example

Let's output our number so that its integer part contains two digits. In this case, the minimum number of digits in the fractional part will be1:

<div>{{ num | number:'2.1' }}</div>

Result of executing the code:

<div>18.0</div>

Example

Now let's set the fractional part of our number to display two digits:

<div>{{ num | number:'2.2' }}</div>

Result of executing the code:

<div>18.00</div>

Example

Let's set a new number:

export class AppComponent { num: number = 18.67899; }

And display it in the HTML template so that the fractional part displays a maximum of two digits:

<div>{{ num | number:'2.1-2' }}</div>

Result of executing the code:

<div>18.68</div>

Example

Now let's format our number so that its integer part contains three digits. And five digits were displayed in the fractional part:

<div>{{ num | number:'3.5-6' }}</div>

Result of executing the code:

<div>018.67899</div>

Example

Let's try to output the minimum number of digits in the fractional part that exceeds the specified number in the number. In our case, we will output six digits:

<div>{{ num | number:'3.6-8' }}</div>

As a result of executing the code, we will see that the channelDecimalPipeadded a zero to the fractional part:

<div>018.678990</div>
