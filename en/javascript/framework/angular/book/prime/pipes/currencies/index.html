Formatting Currency Using a Channel in Angular

Using the channelCurrencyPipeYou can format the currency output. The first parameter of the channel contains the currency code according to the ISO specification4217(default USD). In the second parameter we set the currency display. It can take the following values:'code'(currency code),'symbol'(her symbol),'symbol-narrow'(for countries that have multiple currency symbols),'string'(to output any string). In the third optional parameter we can set a number format similar to the channelDecimalPipe. The fourth optional parameter can specify the locale code to be used:

<div>{{ value | number : код валюты : отображение валюты : формат числа : код локали }}</div>

Examples

Let's try the channel using examples. To do this, set the following property in the component classsalary, containing a salary in some currency:

export class AppComponent { salary: number = 867.564; }

Example

Let's convert our number to currency:

<div>{{ salary | currency }}</div>

Result of executing the code:

<div>$867.56</div>

Example

Now let's specify the display of currency in Russian rubles:

<div>{{ salary | currency:'RUB':'code' }}</div>

Result of executing the code:

<div>RUB867.56</div>

Example

Let's display the Russian ruble symbol next to the number:

<div>{{ salary | currency:'RUB':'symbol-narrow' }}</div>

Result of executing the code:

<div>₽867.56</div>

Example

Now let's use a number with four fractional digits:

<div>{{ salary | currency:'RUB':'symbol':'2.4-5' }}</div>

Result of executing the code:

<div>RUB867.5640</div>

Example

Let's add the following line to the currency display:

<div>{{ salary | currency:'RUB':'this is a new currency - ' }}</div>

Result of executing the code:

<div>this is a new currency - 867.56</div>

Practical problems

Given number:

salary: number = 134.89;

Format it as follows:

'$134.89'

Given number:

salary: number = 134.89;

Format it as follows:

'€134.89'

Given number:

salary: number = 134.89;

Format it as follows:

'$134.890'

Given number:

salary: number = 134.89;

Format it as follows:

'$0,134.8900'

Given number:

salary: number = 134.89;

Format it as follows:

'this currency is changed 134.89'
