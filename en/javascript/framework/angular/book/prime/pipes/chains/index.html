Channel chains in Angular

To apply multiple views to a single value, you can use channel chains. To do this, we register all the channels we need through the symbol|.

Let's set the property in the main component classname:

export class AppComponent { name: string = 'john d.'; }

In the HTML template, we will write the username output in upper case, excluding part of the line' d.':

<div>{{ name | uppercase | slice:0:4 }}</div>

Set your main component class to a string consisting of10characters. Display its last five characters in uppercase.
