Trimming a string through a channel in Angular

Let's learn how to use the channelSlicePipe. This channel allows you to trim a line. At the same time, it accepts parameters that specify the initial and final cutting indices. To pass a parameter, place a colon after the channel name and list the parameters.

Let's look at an example. In the component class we will set the propertyname:

export class AppComponent { name: string = 'john'; }

And now using the channelSlicePipeand truncate the string by specifying the starting and ending indexes for it:

<div> {{ name | slice:0:2 }} </div>

The result will be the following:

<div> jo </div>
