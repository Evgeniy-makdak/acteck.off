Converting to lowercase via channel in Angular

Using the channelLowerCasePipe, you can convert the string to lower case. Let's try it in practice. Let's say in the component class we have the following line:

export class AppComponent { name: string = 'JOHN'; }

Let's display our line in the template file, using our channel in it:

<div> {{ name | lowercase }} </div>

Result of executing the code:

<div> john </div>

Let's say you have an array:

export class AppComponent { products: string[] = ['Milk', 'Bananas', 'Fish', 'Bread']; }

Print each element of the array in lowercase.
