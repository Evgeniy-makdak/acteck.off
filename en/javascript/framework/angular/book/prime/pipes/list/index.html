List of channels in Angular

Let's collect all the channels in one table so that you can peep in the future:

Channel Description
UpperCasePipe Converts all characters in a string to uppercase.
LowerCasePipe Converts all characters in a string to lowercase.
SlicePipe Trims a string at the start and end indexes specified in the method parameters.
CurrencyPipe Formats the specified number into a currency string, according to the ISO specification4217. Using this method, you can display the currency code, its symbol, and also set your own string with additional information.
PercentPipe Converts a number to a percent string formatted according to the local standard specifying the group size and separator, decimal point character, and other configurations.
DatePipe Converts date output according to the format specified in the method parameter.
DecimalPipe Formats a number according to the specified number of decimal and integer digits.
