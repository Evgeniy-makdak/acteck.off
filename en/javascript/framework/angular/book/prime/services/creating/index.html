Creating a service in Angular

Let's start exploring services by creating your own. To do this in the foldertest/src/applet's make a filedata.service.ts. Let our service be responsible for displaying some data on the screen.

Let's create a class for our service:

export class DataService { }

Now we need a decoratorInjectable. Its use will make it possible for the service class to bedeployto the component class. We'll talk about implementation in the next lesson. Now let’s import the desired decorator:

import { Injectable } from '@angular/core'; export class DataService { }

Let's apply it to our class:

import { Injectable } from '@angular/core'; @Injectable() export class DataService { }

Now let's make sure that our class has some data. Typically, Angular applications load data from the server, but for educational purposes we will use an array. Let's make it in a private property:

export class DataService { private data: string[] = ['a', 'b', 'c']; }

Let's now write a method to receive data:

export class DataService { private data: string[] = ['a', 'b', 'c']; getData(): string[] { return this.data; } }

Our service is ready. In the next tutorial we will connect it to the component class.
