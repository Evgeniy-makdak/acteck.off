Using a service in Angular

In the previous lesson we created our own serviceDataService. Let's now import it into the component:

import { Component } from '@angular/core'; import { FormsModule } from '@angular/forms'; import { DataService } from './data.service'; // тут

In the decorator@Componentin propertyprovidersyou need to specify our service:

@Component({ ..... providers: [ DataService ] // тут })

Now we'll use a special approach calleddependency injection (dependency injection). Its essence is that in the component constructor we can pass our service as a parameter and it will automatically go into the private property of our component. Look in the code:

export class AppComponent { constructor(private dataService: DataService) { // внедряем зависимость console.log(this.dataService); // сервис в приватном свойстве } }

The injection for our service works because we previously applied a decorator to itInjectable.

Let's now get data from our service in the component class constructor by calling the appropriate method:

export class AppComponent { public data: string[] = []; constructor(private dataService: DataService) { this.data = this.dataService.getData(); } }

Let's display the received data in the component template:

<ol> <li *ngFor="elem of data"> {{ elem }} </li> </ol>
