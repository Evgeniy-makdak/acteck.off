Introduction to Angular Services

Services- classes that perform certain tasks, for example, logging, data output. Services do not work with HTML template markup, but perform a rather narrow task, for example, outputting messages to the browser console.

Standard service tasks include:

  • Working with data - the service can store data in its own file or take it from some external source, for example, a server.
  • Organization of interaction between project components.
  • Unloading the component code, from which calculation, logging, etc. tasks can be transferred to the service. Based on this, the component itself will be focused strictly on working with its templates.
