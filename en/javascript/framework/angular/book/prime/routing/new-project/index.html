Создание нового проекта для роутинга в Angular

Для изучения роутинга нам необходимо создать новый проект. При создании вам будут предлагать различные опции и одной из опций будет включение роутинга. Включите его.

Напомню команду на установку нового проекта:

ng new my-app

Установите новый проект с роуитнгом.
