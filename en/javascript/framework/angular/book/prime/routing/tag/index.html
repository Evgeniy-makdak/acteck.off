Angular routing tag

In the previous lesson we configured routes. Now we need to specify in the parent component the location where our child components will be displayed.

This location is indicated using a special tagrouter-outlet, which we write in the parent component.

First you need to connect this tag:

import { RouterOutlet } from '@angular/router';

Register in the keyimportsdecorator:

@Component({ ...... imports: [RouterOutlet], ...... })

Now in the parent component we can write a tagrouter-outlet:

parent <router-outlet></router-outlet>

One of the child components will be displayed in this tag, depending on which URL is entered in the address bar.

Please note that you do not need to import child components into the parent component. We import them into a file with routes.

Enter the URLs one by one into the address bar/aaaa/And/bbbb/. Make sure the correct components are connected.
