Capture all URLs in Angular routing

You can capture all possible addresses using the command**. This is used to show an error if none of the addresses matched. Let's do this:

const appRoutes: Routes =[ { path: 'aaaa', component: AaaaComponent}, { path: 'bbbb', component: BbbbComponent}, { path: '**', component: NotFoundComponent }, ];

Make a component that will be shown if none of the routes match the URL.
