Problem with switching links in Angular

Let's look at the solution to the problem with switching links in routing. This problem can be solved using the libraryRxJS, which is already included in Angular.

First, let's import the classSubscription:

import { Subscription } from 'rxjs';

Let's make a private property:

export class AaaaComponent { private subscription: Subscription; }

Let's create a property to store a parameter with the nameid:

export class AaaaComponent { id: number | undefined; private subscription: Subscription; }

Now let’s subscribe to change the parameteridfrom route:

export class AaaaComponent { id: number | undefined; private subscription: Subscription; constructor(private activateRoute: ActivatedRoute) { this.subscription = activateRoute.params.subscribe(params => this.id = params['id']); } }
