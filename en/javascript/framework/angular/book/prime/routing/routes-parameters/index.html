Route options in Angular

In Angular, routes can not only be hard-coded, but also support parameters. This way we can define a whole group of URLs in one route.

Let, for example, we want the same component to be available at the following addresses:aaaa/1, aaaa/2, aaaa/3and so on.

To do this, you need to specify the changing part of the address in the route as a parameter. To do this, you need to write a colon and the parameter name. In our case, we'll give our parameter a nameid(we can give anything):

const routes: Routes = [ { path: 'aaaa/:id', component: AaaaComponent }, { path: 'bbbb', component: BbbbComponent }, ];

Make a route with parameters. Verify that the specified component opens for different parameter values.
