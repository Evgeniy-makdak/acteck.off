Angular Component Styles

Let's now style our layout using CSS. For this we need the fileapp.component.css, in which CSS styles are written for the layout of the component. At the same time, Angular itself takes care of ensuring that the styles work as a CSS module. This means that all selectors will only work for the layout of the component and will not capture the layout from the outside.

Let's see in practice. Let our component have the following layout:

<div> abcde </div>

Let's set the div's color:

div { color: red; }

At the same time, we can be sure that the style will be applied only to the divs from our component, and not to any others.
