Binding CSS Styles in Angular

In Angular you can also directly add styles to a block. This is done using a tag attribute written in the following format:

[style.styleProperty]="property value"

In this case, the names of CSS properties that have a hyphen inside them, in our case, will be written incamelCase. Let's look at examples of how this works.

Example

Let's set the background to the element:

<div [style.backgroundColor]="'blue'"> text </div>

Example

Let's say we have some property containing the background color:

export class AppComponent { public value: string = 'red'; }

Let's use this property as a value for the style:

<div [style.backgroundColor]="value"> text </div>

Example

Let us have some Boolean property:

export class AppComponent { public isActive: boolean = true; }

We will bind different CSS property values ​​depending on the contents of the boolean property. We use for thisternary operator:

<div [style.backgroundColor]="isActive ? 'blue' : 'red'"> text </div>

Practical problems

Given a block. Make the first click on a block color it red, and the second click turn it green.

Given a block. A button is given. Make sure that pressing a button doubles the width of the block.
