Dynamically changing styles in Angular

Using directivesngClassAndngStyleWe can bind expressions to elements, allowing our styles to change dynamically.

Let's make the text break or show using a propertyactivecomponent class:

export class AppComponent { active: boolean = true; toggle() { this.active = !this.active; } }

To do this, let's write a functiontoggle, which will alternately enable or disable this property:

export class AppComponent { active: boolean = true; toggle() { this.active = !this.active; } }

In the CSS file of the component, set the following style to the class:

.hidden { display: none; }

Let's make a div in the component template and add a CSS class to ithidden, which will be turned on or off depending on the propertyactivefrom the component class:

<div [ngClass]="{hidden: active}"> text </div>

Let's also make a button, when clicked the method will be calledtoggleshowing or hiding our component:

<button (click)="toggle()"> toggle </button>
