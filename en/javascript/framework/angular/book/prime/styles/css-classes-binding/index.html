Binding CSS classes in Angular

In Angular, you can turn on/off the application of a CSS class for a block. This is done using a tag attribute written in the following format:

[class.имя_класса]="true или false"

Let's see in practice. Let us have the following classes:

.blue { color: blue; } .bold { font-weight: bold; }

Let's look at examples of how to enable and disable these classes for a block.

Example

Let's turn on one class and turn off the other:

<div [class.blue]="true" [class.bold]="false"> text </div>

Example

Let class states be stored in class properties:

export class AppComponent { public isBlue: boolean = true; public isBold: boolean = false; }

Let's bind the visibility of the class depending on the values ​​of our properties:

<div [class.blue]="isBlue" [class.bold]="isBold"> text </div>

Practical problems

Given the following CSS class:

.hidden { visibility: hidden; }

Bind this class to a block. Make a button that, when clicked, will hide or show the block.
