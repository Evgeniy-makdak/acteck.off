Interpolation in Angular

In the template file, you can not only display text, but also write JavaScript code, performing various operations. In Angular terms this is calledinterpolation.

In order to perform an operation, it must be written in double curly braces. For example, let's find the sum of two numbers:

<div> {{ 1 + 2 }} </div>

Now let's find the square root of the number:

<div> {{ Math.sqrt(2) }} </div>

Using interpolation, print the product of two numbers.

Use interpolation to round a fraction to the specified decimal place.
