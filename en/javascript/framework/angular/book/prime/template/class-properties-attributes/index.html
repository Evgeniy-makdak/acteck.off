Component class properties in Angular attributes

You can display public properties of a class not only in the text of tags, but also as attribute values. Let's see in practice. Let our class have the following property:

export class AppComponent { public test: string = 'abcde'; }

Let's display the contents of our property in the tag attribute:

<div id="{{ test }}"> text </div>

The following layout is given:

<img src="test.png" alt="picture">

Make the attribute contents pass from the component class.
