Template Variables in Angular

In Angular you can access page tags usingtemplate variables. They allow you to bind a tag to a variable and reference it while the component is running. In order to declare such a variable, you need to put a sign in front of its name#.

Let's see in practice. Let, for example, we have a div. Let's write it into a template variable:

<div #userName> text </div>

Now in the variableuserNamethere will be a link to the DOM element of our div. And we can treat it the way it is done inpure JavaScript. For example, let's display the text of this div in another tag:

<div> {{ userName.textContent }} </div>
