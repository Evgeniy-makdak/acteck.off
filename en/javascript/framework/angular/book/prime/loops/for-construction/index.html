for construct in Angular

A loop through array elements can also be started using the construct@for.

Let's try it in practice. Let again our component have an array:

export class AppComponent { public arr: number[] = [1, 2, 3, 4, 5]; }

Let's look at the syntax of the construction:

@for () { теги }

Let's indicate the array to be iterated and the variable into which the elements will go:

@for (elem of arr) { теги }

This, however, is not all. For Angular to work correctly, we must specify a special commandtrack. After it, we must specify some unique value for each element of the array. This is necessary so that Angular can work more optimally with the DOM. Let's take the array element number as such a unique value. To do this we must specify a variable$index. So here is our final code:

@for (elem of arr; track $index) { <div> {{ elem }} </div> }
