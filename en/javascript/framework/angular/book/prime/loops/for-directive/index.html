for directive in Angular

Let's say we have an array:

export class AppComponent { public arr: string[] = ['a', 'b', 'c', 'd']; }

Let's loop through this array in the template file. This is what the directive is for.*ngFor. It can iterate through an array and display each element in a separate tag.

First, you need to import this directive:

import {NgFor} from "@angular/common";

And add it to the imports section in the decorator:

@Component({ ..... imports: [....., NgFor], .... })

Now you can use it. Let's see how it's done. Let us have a listul:

<ul> </ul>

Let's make sure that our array is looped through and each element is displayed in tagsli. This is done as follows:

<ul> <li *ngFor="let el of arr"> {{ el }} </li> </ul>

export class AppComponent { public arr: number[] = [1, 2, 3, 4, 5]; }
