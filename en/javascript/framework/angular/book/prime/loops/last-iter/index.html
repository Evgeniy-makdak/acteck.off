Last iteration of a loop in Angular

When searching through*ngForAnd@foryou can define the last iteration of a loop using a variablelast.

Example for a directive*ngFor:

<ul> <li *ngFor="let el of arr; let last = last"> {{ last ? 'первый' : '' }} {{ el }} </li> </ul>

Example for design@for:

<ul> @for (el of arr; track $index) { <li> {{ $last ? 'последний' : '' }} {{ el }} </li> } </ul>

Given an array:

export class AppComponent { public arr: string[] = ['a', 'b', 'c', 'd']; }

Output array elements as a listul, having painted the last oneliin red.

export class AppComponent { public arr: string[] = ['a', 'b', 'c', 'd']; }
