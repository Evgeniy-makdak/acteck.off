Odd loop iteration in Angular

When searching through*ngForAnd@foryou can define an odd loop iteration using a variableodd.

Example for a directive*ngFor:

<ul> <li *ngFor="let el of arr; let odd = odd"> {{ odd ? 'нечетный' : 'нет' }} {{ el }} </li> </ul>

Example for design@for:

<ul> @for (el of arr; track $index) { <li> {{ $odd ? 'нечетный' : 'нет' }} {{ el }} </li> } </ul>

Given an array:

export class AppComponent { public arr: string[] = ['a', 'b', 'c', 'd']; }

Output array elements as a listul, coloring the even onesliin red, and odd ones in green.
