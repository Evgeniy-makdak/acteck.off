Angular Component Structure

In a foldersrc/appwill host the main component created automatically when the project is created. It will have the following file structure:

  • /src/app/
    • app.component.html
    • app.component.css
    • app.component.ts

In fileapp.component.htmlthe layout of our component will be placed.

In fileapp.component.cssour component's styles will be placed. The file extension depends on the settings made when creating the project.

In fileapp.component.tsTypeScript code that you write for the functionality of our component will be placed.

We will look at working with these files in more detail in the following lessons.
