Структура компонента Angular

В папке src/app будет размещаться основной компонент, созданный автоматически при создании проекта. Он будет иметь следующую файловую структуру:

  • /src/app/
    • app.component.html
    • app.component.css
    • app.component.ts

В файле app.component.html будет размещаться верстка нашего компонента.

В файле app.component.css будут размещаться стили нашего компонента. При этом расширение файла зависит от настроек, сделанных при создании проекта.

В файле app.component.ts будет размещаться TypeScript код, который вы напишете для функционала нашего компонента.
