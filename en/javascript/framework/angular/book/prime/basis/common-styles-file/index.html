Angular Project Common Style File

After creating the project in the foldersrcThere will be a file with general site styles. This file will be namedstyles, but its extension may vary depending on the settings selected when creating the project:.css, .lessor.sass.

This file should contain styles that relate to the structure of the site, but not to individual components. The fact is that for each Angular component, its styles will be stored in the folder with this component.

Write a code that will arrange the blocks of your site on the page: header, footer, sidebars and content.
