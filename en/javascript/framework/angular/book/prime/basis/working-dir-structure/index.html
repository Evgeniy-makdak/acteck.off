Структура рабочий папки проекта Angular

Давайте теперь разберем структуру папки src, в которой будет вестись основная работа. Вот нужные нам папки и файлы (остальное служебное и пока не нужно):

  • /src/
    • /app/
    • /assets/
    • index.html
    • styles.css
    • favicon.ico

В папке app будут размещаться ваши компоненты. Изначально по умолчанию там будет основной компонент с именем App. В дальнейшем мы будем добавлять туда и другие компоненты.

В папке assets можно размещать различные ресурсы нашего приложения. Под ресурсами понимаются картинки и тому подобные вещи.

Файл index.html представляет собой основной макет сайта с версткой. К этой верстке особым образом подключаются компоненты в указанные места.

В файл styles.css можно размещать общие CSS стили сайта.

В файле favicon.ico хранится фавиконка вашего сайта. Эта та картинка, которая отображается на вкладке с сайтом перед его именем.

В следующих уроках мы поговорим подробнее об этих файлах и папках.
