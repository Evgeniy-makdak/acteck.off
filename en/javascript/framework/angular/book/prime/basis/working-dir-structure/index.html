Angular project working folder structure

Let's now look at the folder structuresrc, where the main work will be carried out. Here are the folders and files we need (the rest is service and not needed yet):

  • /src/
    • /app/
    • /assets/
    • index.html
    • styles.css
    • favicon.ico

In a folderappyour components will be placed. Initially, by default, there will be a main component namedApp. In the future we will add other components there.

In a folderassetsyou can place differentresourcesour application. Resources mean pictures and similar things.

Fileindex.htmlrepresents the basic layout of the site with layout. Components are connected to this layout in a special way in the specified places.

To filestyles.cssYou can place general CSS site styles.

In filefavicon.icoThe favicon of your site is stored. This is the picture that is displayed on the website tab before its name.

In the following lessons we will talk more about these files and folders.
