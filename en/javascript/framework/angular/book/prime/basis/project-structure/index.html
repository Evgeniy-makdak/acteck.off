Angular Project Structure

After executing the command to create a project, a certain file structure will appear in its folder. Basically there are service files that support the operation of the framework. For now we are only interested in the following folder and file:

  • /src/
  • README.md

In a foldersrcyou will do all the work. We'll look at it in more detail in the next lesson.

FileREADME.mdcontains instructions for starting the application. It is usually used when they receive an archive with a project from a colleague and do not understand how to start the project. In this case, the necessary terminal commands can be found in this file.
