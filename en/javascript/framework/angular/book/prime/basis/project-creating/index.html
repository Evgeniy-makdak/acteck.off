Creating an Angular Project

After installing the Angular framework, we can make our first project. To do this, create a folder in which we will work. Open it through the terminal and write the following command in itng new, and after it the name of the folder with our project.

For example, let's choose as the project nametest:

ng new test

Our project will be installed in the foldertest. For further actions we need to go to this folder in the terminal:

cd test

Once the installation is complete, we can run our project. To do this, while in the folder with the site, run the following command:

ng serve --open

Install your project.
