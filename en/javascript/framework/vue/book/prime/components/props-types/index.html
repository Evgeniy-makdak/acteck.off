Specifying the type of props in Vue

It is better when each input parameter is of a certain type. In these cases, you can list the input parameters by an object, where the property and value will contain information about the input parameter name and its type, respectively:

props: { title: String, likes: Number, isPublished: Boolean, commentIds: Array, author: Object, callback: Function, contactsPromise: Promise // или любой другой конструктор }

This not only documents the component, but also alerts programmers in the browser console if they are passing in the wrong type.

Let's specify the type of props for our user component:

export default { props: { name: String, surn: String }, data() { return { } } }
