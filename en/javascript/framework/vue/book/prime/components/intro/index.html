Working with Components in Vue

If we look at a website, we can identify some blocks on it: header, content, sidebar, footer.

When creating a website in Vue, we can also use such independent blocks. They will allow us to better organize our code and simplify its further maintenance.

In Vue these blocks are calledcomponents. In large applications, component separation becomes a must to keep the development process manageable

It is recommended to place components in a foldercomponents. Each component in a separate file in the formES6 module.
