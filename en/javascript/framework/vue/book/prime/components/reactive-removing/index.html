Reactive Component Removal in Vue

Let's learn how to reactively remove child components from a list. Let's create a special button for this in each child component. When this button is clicked, an event will be emitted and the specified child component will be deleted in the parent component by itsid.

Let's start implementation. Let the parent component be given the following array of objects:

data() { return { users: [ { id: 1, name: 'name1', surn: 'surn1' }, { id: 2, name: 'name2', surn: 'surn2' }, { id: 3, name: 'name3', surn: 'surn3' }, ], } }

Let's write in the parent component a method for deleting a user by hisid:

methods: { remove(id) { this.users = this.users.filter((user) => { return user.id !== id; }) } }

Let's create components in a loop, passing them first name, last name,idand the method to remove:

<template> <User v-for ="user in users" :id ="user.id" :name ="user.name" :surn ="user.surn" @remove ="remove" :key ="user.id" /> </template>

Let's enter the incoming data in the settingspropsand emitted event in setupemits:

props: { id: Number, name: String, surn: String, }, emits: ['remove'],

Let's display the user's first and last name in the component view:

<template> {{ name }} {{ surn }} </template>

Now let’s make a button, when clicked, an event for deletion will be emitted. The parameter of this event will be transmittedidcomponent. When the parent component receives an event, it will remove this user from the array of objects and it will reactively disappear from the list of users.

So, let's implement this button:

<template> {{ name }} {{ surn }} <button @click="$emit('remove', id)"> remove </button> </template>
