Creating Components in a Loop in Vue

Components can be created in a loop by passing them the appropriate data. Let's see how it's done. Let the parent component be given the following array of objects:

data() { return { users: [ { id: 1, name: 'name1', surn: 'surn1' }, { id: 2, name: 'name2', surn: 'surn2' }, { id: 3, name: 'name3', surn: 'surn3' }, ], } }

Let's loop through this array of objects and create components with users in the loop:

<template> <User v-for="user in users" /> </template>

Specify the attribute valueskeyfor the loop to work correctly:

<template> <User v-for="user in users" :key="user.id" /> </template>

We will also pass their data, first and last name of each user to the components:

<template> <User v-for="user in users" :name="user.name" :surn="user.surn" :key="user.id" /> </template>

The following array with employee data is given:

data() { return { users: [ { id: 1, name: 'name1', salary: 100, age: 30, }, { id: 2, name: 'name2', salary: 200, age: 40, }, { id: 3, name: 'name3', salary: 300, age: 50, }, ], } }
