Creating Child Components in Vue

The components inside have the same structure as the main component we worked with earlier. That is, in the file of each component there will be tagsscriptAndtemplate.

Let's make an example component with the nameUser. Let's place its code in the appropriate file:

<script> export default { data() { return { } } } </script>

In the objectdatacomponent you can place some data:

<script> export default { data() { return { name: 'john' } } } </script>

This data can be displayed in the component view:

<template> {{ name }} </template>

Let's now connect the component we created to the main component. To do this, first import it:

<script> import User from './components/User.vue' export default { } </script>

Let's write his name in the settingscomponents:

<script> import User from './components/User.vue' export default { components: { User } } </script>

In a parent component's view, you can display a child component's view. To do this, you need to write a tag whose name matches the name of the component. Let's do this:

<template> <User /> </template>
