Component for adding data to Vue

Let's now implement a form for adding new users. We will make this form as a separate component. Let's start implementation. First, in the parent component we will create a method for adding a user:

methods: { add(name, surn) { let id = this.users.length + 1; this.users.push({ id, name, surn }); } }

In the view of the parent component, connect the child one:

<template> <UserForm @add="add" /> </template>

In the child component we will create properties for controlling inputs:

data() { return { newName: '', newSurn: '', } }

In the child component we will create a method for saving data:

methods: { save() { this.$emit('add', this.newName, this.newSurn); } }

Let's make views of the child component:

<template> <input v-model="newName"> <input v-model="newSurn"> <button @click="save"> save </button> </template>
