Emitting an event on an attribute in Vue

You can emit the event directly in the handler attribute:

<template> <button @click="$emit('show')"> btn </button> </template>

You can pass the following parameters:

<template> <button @click="$emit('show', 'xxx', 'yyy')"> btn </button> </template>

Make a button that, when clicked, will emit an event.
