Getting keys when iterating over arrays in Vue

Sometimes, in addition to array elements, we would also like to access keys. The following extended syntax is used for this:

<template> <p v-for="(elem, key) in arr"> {{ key }} {{ elem }} </p> </template>

You can perform various operations with keys during withdrawal. For example, let's make the numbers start with one:

<template> <p v-for="(elem, key) in arr"> {{ key + 1 }} {{ elem }} </p> </template>

Given the following array:

data() { return { arr: ['x', 'y', 'z'], } }
