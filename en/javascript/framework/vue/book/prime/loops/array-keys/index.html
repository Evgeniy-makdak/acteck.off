Получение ключей при переборе массивов в Vue

Иногда кроме элементов массива, мы хотели бы получить доступ еще и к ключам. Для этого используется следующий расширенный синтаксис:

<template> <p v-for="(elem, key) in arr"> {{ key }} {{ elem }} </p> </template>

С ключами при выводе можно делать различные операции. К примеру, сделаем так, чтобы номера начинались с единицы:

<template> <p v-for="(elem, key) in arr"> {{ key + 1 }} {{ elem }} </p> </template>

Дан следующий массив:

data() { return { arr: ['x', 'y', 'z'], } }

Выведите ключи этого массива в виде списка ul.
