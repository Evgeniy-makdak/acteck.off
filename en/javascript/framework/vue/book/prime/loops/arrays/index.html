Iterating over arrays in Vue

Vue allows you to form tags in a loop. This is done using a special directivev-for. Let's see how it works for arrays. To do this, we will create the following array:

data() { return { arr: ['a', 'b', 'c'], } }

Let's print each element of this array in a separate paragraph. To do this, first make a paragraph in the view:

<template> <p></p> </template>

Now let's write a directive for our paragraphv-for. The value of this directive should indicate the name of the array being iterated and the variable into which the elements of this array will sequentially be placed. In our case, the array name will bearr, and for the variable we will come up with a nameelem:

<template> <p v-for="elem in arr"></p> </template>

As a result, our paragraph will be repeated as many times as there are elements in our array. Let's display the elements being iterated in the text of our paragraphs:

<template> <p v-for="elem in arr">{{ elem }}</p> </template>
