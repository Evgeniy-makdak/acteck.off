Conditions in loops in Vue

When iterating through elements in loops, you can impose conditions. But the directivesv-forAndv-ifyou need to write on different tags, otherwise there will be a conflict. To avoid conflict directivev-forapply to tagtemplate, and the directivev-if- directly to the inserted tag.

Let's look at an example. Let's say we have the following array:

data() { return { arr: [1, 2, 3, 4, 5], } }

Let's loop through this array:

<template> <ul> <li v-for="elem in arr"> {{ elem }} </li> </ul> </template>

Now let’s impose a condition on the displayed elements:

<template> <ul> <template v-for="elem in arr"> <li v-if="elem % 2 === 0"> {{ elem }} </li> </template> </ul> </template>

Given the following array:

data() { return { items: [1, -2, 3, -4, 5], } }

Loop through this array and print only the positive elements of the array.
