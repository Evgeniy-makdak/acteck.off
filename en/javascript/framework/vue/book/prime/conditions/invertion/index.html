Inverting a condition in Vue

You can invert the condition using an exclamation point:

<template> <p v-if="!visible">text</p> </template>

In this case, the paragraph will be shown ifvisiblemattersfalseand hidden - iftrue.

Invert the condition:

<template> <p v-if="hidden">text</p> </template>
