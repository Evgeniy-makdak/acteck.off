Complex conditions in Vue

In the directivev-ifYou can make more complex conditions. Let, for example, in the propertynumcould be one of the numbers1, 2or3:

data() { return { num: 3, // пусть сейчас там 3 } }

Let's now make three paragraphs, of which only one will be shown depending on the value of the propertynum:

<template> <p v-if="num === 1">one</p> <p v-if="num === 2">two</p> <p v-if="num === 3">three</p> </template>

You can make more complex conditions:

<template> <p v-if="num === 1 || num === 3"> one or two </p> </template>
