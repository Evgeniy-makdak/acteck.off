Changing text when toggling in Vue

Let's make sure that when toggling, the text of the operation that the button will perform at the moment is written on the button. For this we need to useternary operatoras follows:

<template> <button @click="toggle"> {{ visible ? 'hide' : 'show' }} </button> </template>

Three paragraphs and three buttons are given. Make each button toggle its own paragraph and change its text accordingly.
