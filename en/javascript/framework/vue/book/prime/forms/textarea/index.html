Работа с textarea в Vue

Работа с textarea происходит абсолютно также, как и с инпутами - через директиву v-model. Давайте посмотрим на практике. Сделаем свойство, которое будем привязывать к нашему тегу:

data() { return { text: '', } }

Привяжем свойство к тегу:

<template> <textarea v-model="text"></textarea> </template>

Сделаем еще и абзац, в который будет выводиться вводимый текст:

<template> <textarea v-model="text"></textarea> <p>{{ text }}</p> </template>

Дан textarea. Сделайте так, чтобы при наборе текста в этом textarea, этот текст одновременно выводился в абзаце под ним.

Дана кнопка textarea. В него вводится текст. По нажатию на кнопку получите массив введенных слов и выведите элементы этого массива в виде списка ul.
