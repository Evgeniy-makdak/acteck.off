Button to remove array element in Vue

Let's make a reactive button to remove elements from the list. Let's start implementation. Let us be given an array:

data() { return { items: ['a', 'b', 'c', 'd', 'e'], } }

Let's display the contents of the array as a list:

<template> <ul> <li v-for="(item, index) in items" :key="index"> {{ item }} </li> </ul> </template>

Let's create a delete button for each list item. In this button we will bind a method whose parameter will pass the number of the array element that we are going to delete:

<template> <ul> <li v-for="(item, index) in items" :key="index"> {{ item }} <button @click="removeItem(index)">remove</button> </li> </ul> </template>

Implement deletion in the methodremoveItem:

methods: { removeItem: function(index) { this.items.splice(index, 1); } }
