Button to remove from an array of objects in Vue

Let's now implement a button to remove data from an array of objects. Let's start implementation. Let there be the following array:

data() { return { users: [ { id: 1, name: 'name1', surn: 'surn1', }, { id: 2, name: 'name2', surn: 'surn2', }, { id: 3, name: 'name3', surn: 'surn3', }, ] } }

Let's display the contents of the array as a list:

<template> <ul> <li v-for="user in users" :key="user.id"> {{ user.name }} {{ user.surn }} </li> </ul> </template>

Let's create a delete button for each list item. In this button we will bind a method whose parameter we will passidthe array element we are going to remove:

<template> <ul> <li v-for="user in users" :key="user.id"> {{ user.name }} {{ user.surn }} <button @click="removeItem(user.id)">remove</button> </li> </ul> </template>

Implement deletion in the methodremoveItem:

methods: { removeItem: function(id) { this.users = this.users.filter((user) => { return user.id !== id; }) } }

The following array with employee data is given:

data() { return { users: [ { id: 1, name: 'name1', salary: 100, age: 30, }, { id: 2, name: 'name2', salary: 200, age: 40, }, { id: 3, name: 'name3', salary: 300, age: 50, }, ], } }

Output employee data as an HTML table. Make a column with links to remove workers.
