Two-way data binding to inputs in Vue

Now we will learn how to work with inputs in the Vue framework. With their help, we will reactively add data to the page. Let's get started. Let us have an input:

<template> <input> </template>

Let us also have the propertymessage:

data() { return { message: 'hello', } }

We can bind this property and input in such a way thatchanging any of them led to a change in the other. This is done using the directivev-model, which specifies the property bound to the input.

Let's bind a property to our inputmessage:

<template> <input v-model="message"> </template>

After running the code, the input will contain the property textmessage. And if you edit the data in the input, the property will also change accordingly. To see this, let's output the input content somewhere in a paragraph:

<template> <input v-model="message"> <p>{{ message }}</p> </template>

Input given. Make sure that the text entered into the input is immediately displayed in the paragraph below it.

Modify the previous problem so that the text is displayed in uppercase.

Input given. Let a number be entered into it. Make sure that the square of the entered number is displayed in the paragraph as you type.
