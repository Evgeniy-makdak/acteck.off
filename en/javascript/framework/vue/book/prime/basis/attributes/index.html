Working with attributes in the Vue framework

Insert property values ​​fromdataYou can not only add text, but also tag attributes. This is done using the directivev-bind.

Let's use this directive to set the attribute valuesrcfor tagimg. Let the desired value be stored in a property:

data() { return { attr: 'img.png', } }

Let's write the value from the property to the attributesrc:

<template> <img v-bind:src="attr"> </template>

Usually everyone uses the shortened versionv-bind. It is simply a colon before the attribute name:

<template> <img :src="attr"> </template>

Let indatathe text and link address are stored:

data() { return { text: 'page', href: 'page.html', } }

Generate the following code using this data:

<a href="page.html">page</a>
