Objects in Vue

Let now indatawe have an object written:

data() { return { obj: {a: 1, b: 2, c: 3}, } }

Let's display the contents of this object on the screen:

<template> {{ obj.a }} {{ obj.b }} {{ obj.c }} </template>

Or you can use an alternative method:

<template> {{ obj['a'] }} {{ obj['b'] }} {{ obj['c'] }} </template>
