Reactivity in the Vue framework

Any change to data fromdataresults in an instant change in the display of this data on the page. This Vue behavior is calledreactivity.

Let's check it in practice. Let us have the following property:

data() { return { text: 'xxx', } }

Let's display the value of this property:

<template> {{ text }} </template>

Let's make a button that, when clicked, will change our property:

<template> {{ text }} <button @click="change">text</button> </template>

Let’s now write a method that will be called when the button is clicked and change the property:

methods: { change: function() { this.text = 'yyy'; } }

If you put all the code together and click on the button - text on the screenreactivelywill change to another.
