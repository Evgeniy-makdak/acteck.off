Styling via the style attribute in Vue

CSS styles can be directly added via attributestyle. To do this, an object is passed to the attribute, in which the desired CSS properties must be the keys of this object, and the corresponding CSS property values ​​must be the values ​​of this object.

For example, let's color a paragraph red and add a green border:

<template> <p :style="{color: 'red', border: '1px solid green'}"> text </p> </template>

Let's now set a border for himfont-sizeV30px. Since an object cannot have a key with a hyphen, we must quote it:

<template> <p :style="{color: 'red', 'font-size': '30px'}"> text </p> </template>

You can also insteadfont-sizewritefontSize- this will also work:

<template> <p :style="{color: 'red', fontSize: '30px'}"> text </p> </template>
