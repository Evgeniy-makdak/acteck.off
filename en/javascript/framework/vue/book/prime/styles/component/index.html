Styling a component in the Vue framework

To style components in the component file you need to write a tagstyle:

<script> </script> <template> </template> <style> </style>

It will be possible to write CSS code in this tag, and it will only be applied to the HTML code of the component, and will not be applied anywhere else. This is done specifically for convenience.

Let's look at an example. Let's say we have some paragraphs:

<template> <p> 1 </p> <p> 2 </p> </template>

Let's color them red:

<style> p { color: red; } </style>
