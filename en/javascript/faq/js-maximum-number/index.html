JS maximum number

Finding the maximum number

To find the maximum number in JavaScript useMath.max. See example:

console.log(Math.max(10, 45, 50, 344, 30));

Result of executing the code:

344

Maximum element in array via apply

A tricky way to find the maximum value of an array is to useMath.maxin conjunction with the methodapply:

let arr = [10, 45, 50, 344, 30]; console.log(Math.max.apply(null, arr));

Result of executing the code:

344

Maximum element in array via spread

Using the methodapplyto find the minimum element in an array is deprecated with the advent of the operator spread. Let's see an example:

let arr = [10, 45, 50, 344, 30]; console.log(Math.max(...arr));

Result of executing the code:

344
