JS get input value

In this tutorial we will look at ways in which you can get the input value through JavaScript. Let us have the following input:

<input value="text">

Let's get a link to this input using the methodquerySelector:

let input = document.querySelector('input');

Let's consider various options for obtaining the input value.

Option 1

Let's get and display the input value:

let value = input.value; console.log(value);

Option 2

Let's make sure that the input value is displayed when focus is lost.

To do this, attach a focus loss handler to the input:

input.addEventListener('blur', function() { });

And now, when focus is lost, we get and display the value of the input:

input.addEventListener('blur', function() { let value = this.value; console.log(value); });
