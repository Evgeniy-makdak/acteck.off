Add a new JS array element

In this tutorial, we'll look at the different ways to add new elements to an array in JavaScript and look at code examples for each.

To the beginning of the array

Using the methodunshiftyou can add a new element to the beginning of the array. See example:

let arr = ['a', 'b', 'c']; arr.unshift('x'); console.log(fruits); // выведет: ['x', 'a', 'b', 'c']

To the end of the array

Inserting a new element at the end of the array is done using the methodpush. See example:

let arr = [1, 2, 3]; arr.push(4); console.log(arr); // выведет: [1, 2, 3, 4]

By specified index

Sometimes we may need to insert a new element at a specific position. This can be done using the methodsplice.

For example, let's insert a new element at position2:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.splice(2, 0, '1'); console.log(arr); // выведет ['a', 'b', '1', 'c', 'd', 'e']

Through unification

You can put elements of one array into another using the methodconcat. See example:

let arr = [1, 2, 3]; arr = arr.concat([4, 5]); console.log(arr); // выведет: [1, 2, 3, 4, 5]

Via spread

You can put elements of one array into another using the operatorspread. See example:

let arr = ['a', 'b']; let res = [1, 2, ...arr, 3]; console.log(arr); // выведет: [1, 2, 'a', 'b', 3]
