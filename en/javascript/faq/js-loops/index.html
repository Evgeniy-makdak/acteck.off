A quick reference to loops in JavaScript

while loop

Cyclewhilewill be executed as long as the expression passed to it as a parameter is true. See example:

let i = 0; while (i < 5) { console.log(i); i++; }

Result of executing the code:

01234

for loop

Cycleforis a more compact version of while. See example:

for (let i = 1; i <= 5; i++) { console.log(i); }

Result of executing the code:

12345

Cycleforcan also be used to iterate over an array:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Result of executing the code:

12345

for-of loop

Cyclefor-ofused to iterate over arrays:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); }

Result of executing the code:

12345

forEach method

You can also use the methodforEach. See example:

let arr = [1, 2, 3, 4, 5]; arr.forEach(elem => { console.log(elem); });

Result of executing the code:

12345

for-in loop

A loop is used to iterate over an objectfor-in. See example:

let obj = { a: 1, b: 2, c: 3 }; for (let key in obj) { console.log(key); }

Result of executing the code:

abc

Let's display not the keys, but the elements of the object:

let obj = { a: 1, b: 2, c: 3 }; for (let key in obj) { console.log(obj[key]); }

Result of executing the code:

123
