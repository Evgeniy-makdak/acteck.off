JS convert string to number

There are several ways to convert a string to a number in JavaScript. Let's look at them.

Method 1

Using the functionNumberto convert any value to a number:

let num = Number('10'); console.log(num); // 10

Method 2

Using the operator+to convert any value to a number:

let num = +'10'; console.log(num); // 10

Method 3

Using the functionparseIntto convert a string to an integer;

let num = parseInt('10'); console.log(num); // 10

Method 4

Using the functionparseFloatto convert a string to a float:

let num = parseFloat('10.5'); console.log(num); // 10.5
