Space js string

In this tutorial we will look at various manipulations with spaces in JavaScript.

Removing a space

You can remove the first space from a string using the methodreplace:

let str = 'aaa bbb ccc'; let res = str.replace(' ', ''); console.log(res); // выведет 'aaabbb ccc'

Removing spaces

You can remove all spaces from a string using the methodreplaceAndregular expressions:

let str = 'aaa bbb ccc'; let res = str.replace(/\s/g, ''); console.log(res); // выведет 'aaabbbccc'

Removing trailing spaces

You can remove trailing spaces using the methodtrim:

let str = ' aaa bbb ccc '; let res = str.trim(); console.log(res); // выведет 'aaa bbb ccc'

Split by space

You can split a string by space and get an array of strings using the methodsplit:

let str = 'aaa bbb ccc'; let res = str.split(' '); console.log(res); // выведет ['aaa', 'bbb', 'ccc']
