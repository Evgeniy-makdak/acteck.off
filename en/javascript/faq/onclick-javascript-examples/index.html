onclick javascript examples

Eventonclickis a JavaScript event that occurs when the user clicks on an element. This event can be attached to any HTML element and is used to execute specific code when clicked. In this article we will look at various examples of using this event.

Let's say we have the following button:

<button id="btn">btn</button>

Let's get a link to our button using the methodquerySelector:

let btn = document.querySelector('#btn');

Let's look at three different ways to label an event.onclickto our button.

Example 1

Attach an event handler to the buttononclickusing the methodaddEventListener:

btn.addEventListener('click', function() { console.log('!!!'); });

Example 2

We attach an event handler to the button using the propertyonclick:

btn.onclick = function() { console.log('!!!'); };

Example 3

Given the following function:

function func() { console.log('!!!'); };

Let's attach this function to the button as a response to a click using the HTML attributeonclick:

<button id="btn" onclick="func()">btn</button>
