Array element to string JS

In this tutorial we will look at various options for converting array elements into a string in JavaScript. See the options that may exist.

Option 1

You can merge array elements into a string using the methodjoin:

let arr = ['a', 'b', 'c', 'd', 'e']; let res = arr.join(''); console.log(res); // 'abcde'

Option 2

There are situations when we have a numeric array, and we would like to convert it to a string. Let's look at an example. Let's say we have the following array of numbers:

let arr = [1, 2, 3];

Let's convert its elements to strings using a special trick with the methodmap:

let res = arr.map(String);

As a result, we get an array of strings:

console.log(res); // ['1', '2', '3']
