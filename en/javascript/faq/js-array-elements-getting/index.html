How to get array elements in JS

Using a for-of loop

To get and display all elements from an array, you can use a loopfor-of. Let's see an example:

let arr = ['a', 'b', 'c', 'd']; for (let elem of arr) { console.log(elem); }

Result of the executed code:

'a' 'b' 'c' 'd'

Using a for loop

You can also use a loop to get all elements from an arrayfor, in which we will set the condition to iterate and display all elements as many times as the length of the array. See example:

let arr = ['a', 'b', 'c', 'd']; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Result of the executed code:

'a' 'b' 'c' 'd'
