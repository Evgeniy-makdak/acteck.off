JS reset form

The form can be reset using the methodreset. This method resets all form values ​​using JavaScript.

Let's look at an example. Let us have the following form:

<form id="form"> <input type="text" name="user" value="john"> <button type="button" id="btn">сброс</button> </form>

Let's get links to the form and the button using the methodquerySelector:

let form = document.querySelector('#form'); let btn = document.querySelector('#btn');

Let's bind a click handler to the button:

btn.addEventListener('click', function() { });

Inside the handler, let's reset the form:

btn.addEventListener('click', function() { form.reset(); });
