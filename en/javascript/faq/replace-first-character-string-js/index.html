Replace first character of string JS

There are several ways to replace the first character in a string in JavaScript. The most obvious way is to stupidly write a new value to the zero position, however, it does not work:

let str = 'abcde'; str[0] = '!'; // не работает

The fact is that strings in JavaScript are immutable. Therefore, you have to use special cunning techniques. Let's look at them.

Method 1

We use the methodreplace: and regular expressions:

let str = 'abcde'; let res = str.replace(/^./, '!'); console.log(res); // выведет '!bcde'

Method 2

We use the methodsplitto split the string into an array, then change the first element of the array and merge it back into a string using the methodjoin:

let str = 'abcde'; let arr = str.split(''); arr[0] = '!'; let res = arr.join(''); console.log(res); // выведет '!bcde'
