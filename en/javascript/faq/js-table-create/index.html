How to make a table in JS

In order to make a table using a user-defined function, you should write two nested loops in it, of which the first will create table rows, and the second will create cells in each row. You also need to create a parent document in the HTML document, to which the created table will subsequently be attached. See example:

let elem = document.querySelector('#elem'); createTable(elem, 3, 3); function createTable(parent, cols, rows) { let table = document.createElement('table'); for (let i = 0; i < rows; i++) { let tr = document.createElement('tr'); for (let j = 0; j < cols; j++) { let td = document.createElement('td'); tr.appendChild(td); } table.appendChild(tr); } parent.appendChild(table); }
