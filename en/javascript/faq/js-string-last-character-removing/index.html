Remove last character of string in JS

Using the slice method

To remove the last character in a string, you can use the methodslice, specifying the number in the first parameter0, and the second parameter --1. See example:

let str = 'abcde'; let res = str.slice(0, -1); console.log(res);

Result of executing the code:

'abcd'

Using the substring method

You can also remove all characters in a string using the methodsubstring, in the first parameter of which we pass the number0, and the second parameter is the difference obtained by subtracting one from the length of the string. See example:

let str = 'abcde'; let res = str.substring(0, str.length-1); console.log(res);

Result of executing the code:

'abcd'
