JS array sorting

Methodsortsorts the array in lexicographic order. See example:

let arr = ['c', 'b', 'a', 'e', 'd']; arr.sort(); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd', 'e']

Creating a sort function

Methodsortalso accepts a comparison function as a second parameter. The comparison function takes as arguments2meanings. If the first argument is less than the second, the function should return-1, if the first argument is greater than the second, the function should return1, and if the arguments are equal, then0. Let's see an example:

function func(num1, num2) { if (num1 < num2) { return -1; } if (num1 > num2) { return 1; } return 0; } let arr = ['c', 'b', 'a', 'e', 'd']; console.log(arr.sort(func));

Result of executing the code:

['a', 'b', 'c', 'd', 'e']
