How to make an array in JS

JavaScript provides several ways to make an array. The simplest one is to create an empty array using square brackets:

let arr = [];

You can also create an array with specific values, for example:

let arr = [1, 2, 3];

Or with the lines:

let arr = ['a', 'b', 'c'];
