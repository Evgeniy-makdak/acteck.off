Array index by JS value

In this article, we will look at various ways to get the index of an array by value in JavaScript.

Through a for loop

The easiest way to get the index of an array element is to use a loopforand compare each element of the array with the given value:

let arr = [1, 2, 3, 4, 5]; let val = 3; for (let i = 0; i < arr.length; i++) { if (arr[i] === val) { console.log(i); // индекс элемента break; } }

indexOf method

MethodindexOfreturns the index of the first occurrence of an element in an array:

let arr = [1, 2, 3, 4, 5]; let val = 3; let pos = arr.indexOf(val); console.log(pos);

findIndex method

MethodfindIndexexecutes the function on each element of the array and returns the index of the first element for which the function returns true:

let arr = [1, 2, 3, 4, 5]; let val = 3; let pos = arr.findIndex(elem => elem === val); console.log(pos);
