Add JS array keys

When working with arrays in JavaScript, there is often a need to add new keys or change existing elements. Let's look at ways how this can be done.

Method 1: Direct Addition

You can directly write the value to the new key:

let arr = ['a', 'b', 'c']; arr[3] = 'x'; console.log(arr); // ['a', 'b', 'c', 'x']

This method, however, has problems: if the new key is out of order, we will getsparse array:

let arr = ['a', 'b', 'c']; arr[5] = 'x'; console.log(arr); // ['a', 'b', 'c', , , 'x']

Method 2: Using the push method

Using the methodpushyou can add a new element to the end of the array. Example:

let arr = ['a', 'b', 'c']; arr.push('x'); console.log(arr); // ['a', 'b', 'c', 'x']
