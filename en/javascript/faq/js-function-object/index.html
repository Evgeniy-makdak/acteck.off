JS function object

In JavaScript, a function is an object. This object has certain properties and methods that allow you to perform various operations on functions.

Let's look at examples. Let's say we have the following function:

function func(num1, num2) { return num1 + num2; }

Example of function object properties:

console.log(func.name); // имя функции console.log(func.length); // количество параметров

Example of function object methods:

func().call(2, 3); // вызывает функцию с заданными параметрами

Function objects are created automatically when a function is defined and can be obtained using the keywordthisinside a function. For example, let's get the name of the function inside this function:

function func(num1, num2) { console.log(this.name); return num1 + num2; }

All properties and methods of a function can be viewed in the console if you display the function in it as follows:

console.dir(func);
