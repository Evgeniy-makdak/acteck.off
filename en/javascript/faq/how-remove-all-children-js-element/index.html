How to remove all children of a JS element

Let's look at several ways to remove all children of an element using JavaScript. Let us have the following parent:

<ul id="list"> <li>1</li> <li>2</li> <li>3</li> </ul>

Method 1: removing children via textContent

The easiest way to remove all children is for the parent totextContent. write empty value. Let's do this.

Let's get a link to the parent using the methodquerySelector:

let parent = document.querySelector('#list');

Let's remove all its children:

parent.textContent = '';

Method 2: Using the remove method

The second method is based on using the methodremove. This method removes the specified child element from the parent element. We can get an array of children, loop through them and remove each one. Let's do this. Let's get an array of child elements:

let elems = document.querySelectorAll('#list li');

Let's go through them in a loop and for each we execute a method for deleting:

for (let elem of elems) { elem.remove(); }
