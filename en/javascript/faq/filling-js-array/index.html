Filling a JS Array

When announcedarrayyou can immediately fill it with data:

let arr = ['a', 'b', 'c', 'd', 'e'];

You can fill it out not at the stage of its creation, but later. In the following example, the array is first declared and then filled:

let arr = []; arr[0] = 1; arr[1] = 2; arr[2] = 3; console.log(arr);

Result of executing the code:

[1, 2, 3]

Filling via push

Methodpushadds an unlimited number of elements to the end of the array. The elements are passed as a method parameter. The method modifies the original array and returns the new length of the array. See example:

let arr = ['a', 'b', 'c']; arr.push('d', 'e'); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd', 'e']

Filling via unshift

Methodunshiftadds any number of new elements to the beginning of the array. The method modifies the original array and returns the new length of the array. See example:

let arr = ['c', 'd', 'e']; arr.unshift('a', 'b'); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd', 'e']
