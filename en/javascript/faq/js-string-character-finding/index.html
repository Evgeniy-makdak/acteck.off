Find characters in a string in JS

Using the includes method

To find characters in a string you can use the methodincludes. The method takes as the first parameter the character that needs to be found, and the second optional parameter is the position from which the search should begin. After execution the method returnstrueorfalse. See example:

let res = 'abcde'.includes('ab'); console.log(res);

Result of executing the code:

true

Using the indexOf method

Searching for characters in a string can be done using the methodindexOf. In the first parameter of the method, we indicate the character we are looking for in the case we need (capital letters or small letters). The method will return the position of the first match, and if it is not found, it will return-1. The second optional parameter can be the symbol number where the search should start. See example:

let str = 'ab cd cd cd ef'; let res = str.indexOf('cd'); console.log(res);

Result of executing the code:

3

Using the startsWith method

Tocheck if the string starts with the desired charactershould use the methodstartsWith. If it starts, the method returnstrue, and if it doesn’t start, thenfalse. As the second optional parameter, the method takes the position from which to start checking (by default, from the beginning of the line). See example:

let str = 'abcde'; let res = str.startsWith('abc'); console.log(res);

Result of executing the code:

true

Using the endsWith method

In order tofind the character from the end of the stringshould use the methodendsWith. If it ends, the method returnstrue, and if it doesn’t end, thenfalse. The second optional parameter can force you to specify the length of the string. In this case, the check will not be the actual end of the line, but the specified one. See example:

let str = 'abcde'; let res = str.endsWith('cde'); console.log(res);

Result of executing the code:

true

Using the at method

You can search for a character in a string by its index using the methodat. In the method parameter we specify an integer, which can be positive or negative (in this case, the search is carried out from the end of the string). See example:

let res = 'abcde'.at(0); console.log(res);

Result of executing the code:

'a'
