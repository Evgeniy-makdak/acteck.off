Change text using JS

JavaScript provides several ways to change text in HTML pages. Let's look at different examples.

Option 1

Tag with text:

<p id="elem">aaa</p>

You can change the text using the propertytextContent:

let elem = document.querySelector('#elem'); elem.textContent = 'xxx';

Option 2

Tag with text:

<p id="elem">aaa</p>

To apply tags, you need to use the propertyinnerHTML:

let elem = document.querySelector('#elem'); elem.innerHTML = '<b>xxx</b>';

Option 3

Tag with text from attribute:

<input id="elem" value="aaa">

We change the input text by changing the attribute value:

let elem = document.querySelector('#elem'); elem.value = 'xxx';

Option 4

Tagtextarea:

<textarea>aaa</textarea>

This tag is special - its text is written inside the tags, but to change its text, you need to access the propertyvalue. Let's do this:

let elem = document.querySelector('#elem'); elem.value = 'xxx';
