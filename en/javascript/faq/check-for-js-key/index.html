Check for JS key

When working with objects in Javascript, sometimes you may need to check for the presence of a certain key. Let's see how it's done. For example, let's say we have the following object:

let obj = { name: 'john', surn: 'smit', };

Let's look at ways to check for the presence of a key in an object.

Direct check

You can check the presence of a key using a direct check:

if (obj['key'] !== undefined) { console.log('ключ существует') } else { console.log('ключа нет') }

in operator

You can check the presence of a key using the operatorin. It works like this: if the key exists in the object, it will returntrue, otherwise -false. Usage example:

if ('key' in obj) { console.log('ключ существует') } else { console.log('ключа нет') }

hasOwnProperty method

MethodhasOwnPropertychecks whether the key belongs to the object itself and is not inherited through a prototype. He returnstrueif the key belongs to the object andfalseif not. Usage example:

if (obj.hasOwnProperty('key')) { console.log('ключ существует в объекте') } else { console.log('ключа нет') }
