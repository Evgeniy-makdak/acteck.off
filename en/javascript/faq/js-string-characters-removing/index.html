Remove characters in a string in JS

Using the replace method

To remove all characters in a string, you can use the methodreplace, in the first parameter of which we pass the character that needs to be deleted, and in the second - an empty string. See example:

let str = 'abcde'; let res = str.replace('abcde', ''); console.log(res); console.log(typeof(res));

Result of executing the code:

'' string

Using the substring method

You can also remove all characters in a string using the methodsubstring, in the first parameter of which we pass the number0, and the second parameter-1. See example:

let str = 'abcde'; let res = str.substring(0, -1); console.log(res); console.log(typeof(res));

Result of executing the code:

'' string
