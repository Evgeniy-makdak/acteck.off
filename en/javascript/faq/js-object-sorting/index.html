How to sort an object in JS

Objects in JavaScript are unordered, i.e. their elements do not have clearly defined positions or indexes, so they cannot be sorted directly. For this purpose, you first need to create an empty array, then add all the elements from the object to it, sort them using the methodsortand a custom function. Only after this the resulting array can be converted into an object. Let's sort the object in ascending order of values:

let obj = {'d': 2, 'a': 4, 'c': 1, 'b': 3}; let arr = []; for (let elem in obj) { arr.push([elem, obj[elem]]); } arr.sort(function(a, b) { return a[1] - b[1]; }); let res = Object.fromEntries(arr); console.log(res);

Result of the executed code:

{'c': 1, 'd': 2, 'b': 3, 'a': 4}
