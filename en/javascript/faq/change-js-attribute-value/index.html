Change JS attribute value

In this tutorial we will look at ways to change the value of a tag attribute using JavaScript. For example, let us have the following input:

<input value="text">

Let's change the attribute valuevalueinput. First, we get a link to this input using the methodquerySelector:

let input = document.querySelector('input');

Now let's look at ways to change our attribute.

Via property

The easiest way to change the value of an attribute is to change the property:

input.value = 'xxx';

Via method

You can also change the attribute value using the methodsetAttribute:

input.setAttribute('value', 'xxx');

class attribute

Attributeclassis an exception and must be addressed viaclassName:

input.className = 'xxx';

Changing CSS classes

Using an objectclassListyou can add and remove individual CSS classes to the attributeclasswithout affecting other classes:

input.classList.add('xxx');

Attributes without values

There are attributes in HTML that have no values. These attributes are either present or not. An example would be the attributedisabled:

<input disabled>

To include such an attribute in the corresponding property, you need to write a Boolean value. For example, let's set our attribute:

input.disabled = true;

Now let's remove our attribute:

input.disabled = false;
