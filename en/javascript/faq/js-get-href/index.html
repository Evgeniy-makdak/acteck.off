JS get href

Let's say we have the following link:

<a href="index.html" id="link">text</a>

Let's get a link to this link using the methodquerySelector:

let link = document.querySelector('#link');

Let's now read the attribute valuehref.

Method 1: getAttribute method

One way to get attribute values ​​is to use the methodgetAttribute. This method returns the value of the specified element attribute ornull, if the attribute is not found.

Usage example:

let href = link.getAttribute('href'); console.log(href);

It is important to note that the methodgetAttributedoes not work with elements created using inline JavaScript (for example,document.createElement). For such elements you must use the methodsetAttributeAndgetAttribute.

Method 2. Through a property

You can also get the attribute valuehrefvia the element's DOM property. Usage example:

console.log(link.href);

Difference

The difference between the two methods is as follows: methodgetAttributegets exactly the value that is written in the attributehref. But when we gethrefthrough the property, we already seenormalizedthe value in which JavaScript has automatically corrected the URL to absolute form.
