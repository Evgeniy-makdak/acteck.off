How to reverse a string in JS

In order to reverse a string, you first need to split it using the methodsplit, then turn it over using the methodreverseand finally combine using the methodjoin. See example:

let str = 'abcde'; let res = str.split('').reverse().join(''); console.log(res);

Result of executing the code:

'edcba'

See also

  • methodsplit,
    which splits a string into an array at a specified delimiter
  • methodreverse,
    which reverses the order of the elements in the array
  • methodjoin,
    which merges array elements into a string
