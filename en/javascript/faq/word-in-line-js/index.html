Is there the word JS in the line

Often during development there is a need to check whether a given line contains a certain word. In this article, we'll look at different ways to solve this problem using JavaScript.

For example, let's say we have the following line:

let str = 'hello world';

Let's check that it contains the word'hello'.

Method 1

Let's split the string into an array of words using the methodsplit, using spaces as a separator:

let words = str.split(' ');

Let's check if there is a word in the string using the methodincludes:

let isWord = words.includes('hello'); console.log(isWord);

Method 2

We useregular expressionsand team\b, indicating a word boundary:

let isWord = /\bhello\b/.test(str); console.log(isWord);

If the word is stored in a variable, then we will do the following:

let word = 'hello'; let reg = new RegExp('\\b' + word + '\\b') let isWord = reg.test(str); console.log(isWord);
