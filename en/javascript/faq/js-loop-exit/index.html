JS выход из цикла

Чтобы выйти из цикла в JavaScript используйте инструкцию break Как она работает: как только выполнение цикла дойдет до нее, цикл закончит свою работу.

В следующем примере мы перебираем массив с помощью цикла и, как только в массиве встретится число 3 - выйдем из цикла с помощью break:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); if (arr[i] == 3) { break; } }

Результат выполнения кода:

123
