JS loop exit

To break out of a loop in JavaScript use the statementbreakHow it works: Once the execution of the loop reaches it, the loop will finish its work.

In the following example, we iterate through an array using a loop and as soon as a number is encountered in the array3- let's exit the loop usingbreak:

let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); if (arr[i] == 3) { break; } }

Result of executing the code:

123
