Unix time JS

Unix time in JavaScript is the number of seconds that have passed since1January1970year. To get the current time in Unix format, you can use the following code:

let now = new Date().getTime();

This will return the time in milliseconds. To get it in seconds, you need to divide the result by1000:

let now = new Date().getTime() / 1000;

Now the variablenowcontains the current time in the formatUnix.
