Natural numbers JS

When working with numbers in JavaScript, we often come across natural numbers. Natural numbers are numbers used to count objects. They represent an endless series starting with the number1. In JavaScript, natural numbers are stored as numbers likeNumber.

You can work with natural numbers using various functions and methods. For example, you can use the functionMath.floorto round a number down, use the functionMath.ceil- for rounding up methodtoFixed- for rounding with a given accuracy, functionsparseIntAndparseFloat- to convert a string to a number, etc.

In addition, JavaScript has a special data type for working with natural numbers -BigInt. This type allows you to work with very large numbers without loss of precision. However, it should be remembered thatBigIntis not supported in some browsers, so you should check its support before using this data type.

For example, let's print the first hundred natural numbers using a loop:

for (let i = 1; i <= 100; i++) { console.log(i); }
