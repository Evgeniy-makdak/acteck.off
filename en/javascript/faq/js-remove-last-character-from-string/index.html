JS remove last character from string

There are several ways to remove the last character from a string in JavaScript. However, due to the fact that strings in JS are not mutable, all these methods are based on obtaining a new string without the last character.

Method 1

Using the methodsubstringyou can get a new line without the last character:

let str = 'abcde'; let res = str.substring(0, str.length - 1); console.log(res); // 'abcd'

Method 2

Using the methodsliceyou can remove the last character of the line in a more convenient way:

let str = 'abcde'; let res = str.slice(0, -1); console.log(res); // 'abcd'

Method 3

Using the methodreplaceAndregular expressions:

let str = 'abcde'; let res = str.replace(/.$/, ''); console.log(res); // 'abcd'
