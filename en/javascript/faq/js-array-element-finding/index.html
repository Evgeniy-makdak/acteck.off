How to find an element in an array in JS

Using the indexOf method

To find an element in an array you can use the methodindexOf. In the first parameter of the method we indicate the position number of the searched symbol. The method returns the number of the first element found, or-1, if there is no such element. The second optional parameter of the method specifies the position from which the search should begin. Let's see an example:

let arr = [1, 2, 3, 3, 3, 4, 5]; let res = arr.indexOf(3); console.log(res);

Result of the executed code:

2

Using the includes method

You can also use the method to find an element in an arrayincludes. As a parameter, the method takes the value to be searched. If such an element is in the array, then the method returnstrue, and if not, thenfalse. See example:

let arr = [1, 2, 3, 4, 5]; let res = arr.includes(3); console.log(res);

Result of the executed code:

true

Using the find method

To find the first element in an array, you can use the methodfind, which will search for it according to the parameter passedcallback. If there is no element, then it is returnedundefined. See example:

let arr = [1, 2, 3, 4]; let res = arr.find(function(currentValue) { return currentValue > 0; }); console.log(res);

Result of the executed code:

1
