JS dictionary length

In JavaScript terms, a dictionary is calledobjectis an unordered collection of key-value pairs. Sometimes you may need to know the length of a dictionary to determine the number of key-value pairs it contains.

Unfortunately, directly, through the propertylengthIt is impossible to find the length of a dictionary in JS. Therefore, we use a cunning technique. To begin with, let us be given a dictionary:

let obj = { key1: 'value1', key2: 'value2', key3: 'value3' };

Now we will get the array of keys of our object using the methodObject.keys:

let keys = Object.keys(obj);

And the length of the array can be found through the propertylength:

console.log(keys.length);

This will be the number of elements in our dictionary.
