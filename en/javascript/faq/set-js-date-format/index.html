Set JS date format

Method 1

You can set the date format in JavaScript manually by concatenating parts of the date through the desired separator. For example, let's display the date in the formatday.month.year:

let d = new Date(); let res = d.getDate() + '.' + (d.getMonth() + 1) + '.' + d.getFullYear(); console.log(formattedDate);

You can improve this code by adding a zero to the unique day or month number. To do this, we will create a special function:

funciton addZero(num) { if (num <= 9) { return '0' + num; } else { return num; } }

Let's use it:

let d = new Date(); let res = addZero(d.getDate()) + '.' + addZero((d.getMonth() + 1) + '.' + d.getFullYear(); console.log(formattedDate);

Method 2

You can set the date format in JavaScript using the methodtoLocaleDateStringobjectDate. This method takes a date format string as an argument. For example, the following code will output the current date in the format 'DD.MM.YYYY':

let d = new Date(); let formattedDate = d.toLocaleDateString('ru-RU', { day: '2-digit', month: '2-digit', year: '4-digit' }); console.log(formattedDate);

This example uses the localized date format for Russian ('ru-RU'), but you can specify any other language or format you need.
