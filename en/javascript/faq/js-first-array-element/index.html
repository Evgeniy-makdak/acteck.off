JS first array element

To get the first element of an array, refer to it by its number. The first element is numbered zero. See example:

let arr = ['a', 'b', 'c']; console.log(arr[0]);

Result of executing the code:

'a'

You can also use the methodshift, which retrieves and returns the first element of the array. After extraction, the element disappears from the array. See example:

let arr = ['a', 'b', 'c']; console.log(arr.shift());

Result of executing the code:

'a'
