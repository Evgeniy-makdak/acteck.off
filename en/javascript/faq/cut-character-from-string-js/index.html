Cut character from JS string

Cutting a character from a string in JavaScript can be done in several ways. Let's look at them:

substring method

Using the methodsubstring- This method returns characters starting at the specified index and length of the specified length. The string itself is not changed, but the removed substring is returned. For example, let's cut the first character from the string:

let str = 'abcde'; let res = str.substring(1); console.log(res); // 'b'

slice method

Similarly, you can cut characters using the methodslice:

let str = 'abcde'; let res = str.slice(1); console.log(res); // 'b'

replace method

Using the methodreplaceto search and replace in a string. To cut a character using it, you need to replace this character with an empty string. For example, let's cut the letter from the string'a':

let str = 'abcde'; let res = str.replace('a', ''); console.log(res); // 'bcde'

Regular season

Regular Expressionsallow you to cut characters or substrings based on certain rules. This is done using the methodreplace.

For example, let's remove all letters from a string:

let str = 'a-b-c-d-e'; let res = str.replace(/[a-z]/g, ''); console.log(res); // '----'

For example, let's remove all numbers from a string:

let str = '1-2-3-4-5'; let res = str.replace(/[0-9]/g, ''); console.log(res); // '----'
