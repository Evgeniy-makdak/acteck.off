JS date difference in days

In this tutorial we will find the difference between two dates in days. To begin, we need to first create two dates. We can do this using a classDate.

Let's make two dates:

let date1 = new Date(2021, 11, 15); let date2 = new Date(2021, 10, 10);

Now, to find the difference in days between these two dates, we can use the following code:

let diffInSecs = date2.getTime() - date1.getTime(); let diffInDays = Math.round(diffInSecs / (1000 * 3600 * 24)); console.log(`разница в днях: ${differenceInDays}`);

Here we first get the time in milliseconds for each date using the methodgetTime. We then subtract the time of the second date from the time of the first date to get the time difference in milliseconds. We then divide this difference by the number of milliseconds in one day (1000 * 60 * 60 * 24) to get the number of days between the two dates. Finally, we round the result using the functionMath.round.
