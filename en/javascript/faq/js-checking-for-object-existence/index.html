JS object existence check

You can check the existence of an object using the operatortypeof. If the variable is an object, thentypeofwill return'object'. Usage example:

let obj; if (typeof obj === 'object') { console.log('+++'); } else { console.log('---'); }

Checking the existence of an object property

Operatorinreturnstrueif the specified property exists on the object, andfalse- if not. For example, let's check whether the property exists in the object'a':

let obj = {a: 1, b: 2, c: 3}; if ('a' in obj) { console.log('свойство существует'); } else { console.log('свойство не существует'); }

Using the hasOwnProperty method

MethodhasOwnPropertyallows you to check whether the specified property is directly owned by a given object, and not inherited from its prototype. Let's see in practice. Let us have the following object:

let obj = { name: 'john', sayName: function() { console.log(this.name); } };

Let's check its propertynameown or not:

if (obj.hasOwnProperty('name')) { console.log('+++'); } else { console.log('---'); }
