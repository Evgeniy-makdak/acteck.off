JS array of records

To organize a collection of records in JavaScript, you can use arrays that will store objects. Such arrays can be iterated using standard JavaScript methods and searched for the required entries. Let's see an example:

let obj = [ { header: 'запись1', text: 'Lorem ipsum dolor sit amet.' }, { header: 'запись2', text: 'Lorem ipsum dolor sit amet, consectetur adipisicing.' } ]; for (let elem of obj) { console.log(elem.header, elem. text); }

Let's try to iterate through records using a loopfor-ofand output them:

for (let elem of obj) { console.log(elem.header, elem.text); }
