Changing JS Strings

In JavaScript, strings are immutable, meaning that you cannot change an existing string. Instead, you always create a new row with the changed content.

There are several ways to change strings in JavaScript. Let's look at them.

Method 1

Slicesallow you to extract part of a string and use it separately. You can also use them to insert a new substring into an existing string. Sections are made using methodssliceAndsubstring.

Example:

let arr = ['a', 'b', 'c', 'd', 'e']; let sub = arr.slice(0, 2); console.log(sub); // ['a', 'b']

Method 2

Methodreplaceallows you to replace occurrences of one substring with another substring. See example:

let str = 'abcde'; let res = str.replace('a', '!'); console.log(res); // '!bcde'

This method, however, only changes the first occurrence of the string. To replace all occurrences, you need to use special techniques.

Method 3

Even though the methodreplacechanges only the first match, you can replace all matches using a loop:

let elem = 'a'; let str = 'aaa bbb'; while (str.includes(elem)) { str = str.replace(elem, '!'); } console.log(str); // '!!! bbb'

Method 4

You can also make all replacements usingregular expressions:

let str = 'aaa bbb'; let res = str.replace(/a/g, '!'); console.log(str); // '!!! bbb'

Method 5

You cannot change a string character directly, since strings in JS are immutable:

let str = 'aaa bbb'; str[0] = '!'; console.log(str); // не работает

The desired effect, however, can be achieved using combinations of methodssplitAndjoin:

let str = 'aaa bbb'; let arr = str.split(''); arr[0] = '!'; let res = arr.join(''); console.log(res); // '!aa bbb'
