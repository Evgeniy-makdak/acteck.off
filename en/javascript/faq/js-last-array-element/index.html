JS last element of array

To get the last element of a JavaScript array, you can use the following method:

let arr = ['a', 'b', 'c']; let last = arr[arr.length - 1]; console.log(last);

Result of executing the code:

'c'

Via the pop method

You can also use the methodpop, which retrieves and returns the last element of the array. After extraction, the element disappears from the array. See example:

let arr = ['a', 'b', 'c']; console.log(arr.pop());

Result of executing the code:

'c'
