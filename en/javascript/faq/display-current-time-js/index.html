Вывести текущее время JS

Вывести текущее время можно с помощью метода getTime объекта Date. Он возвращает количество миллисекунд, прошедших с 1 января 1970 года до текущего момента. Пример использования:

let currentTime = new Date().getTime(); console.log(currentTime);

Также можно вывести текущее время в формате, удобном для пользователя. Для этого можно использовать методы getHours, getMinutes и getSeconds объекта Date. Пример:

let d = new Date(); console.log(d.getHours() + ':' + d.getMinutes() + ':' + d.getSeconds());
