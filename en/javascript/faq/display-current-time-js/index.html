Print current time JS

You can display the current time using the methodgetTimeobjectDate. It returns the number of milliseconds that have passed since1January1970years to date. Usage example:

let currentTime = new Date().getTime(); console.log(currentTime);

You can also display the current time in a user-friendly format. For this you can use methodsgetHours, getMinutesAndgetSecondsobjectDate. Example:

let d = new Date(); console.log(d.getHours() + ':' + d.getMinutes() + ':' + d.getSeconds());
