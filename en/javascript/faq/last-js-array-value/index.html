Last JS array value

The last value in a JavaScript array can be obtained in various ways, depending on what exactly you need to do. Let's look at these methods.

Here are some ways to get the last element of an array:

Method 1

You can get the last element of an array by index:

let arr = [1, 2, 3, 4, 5]; let last = arr[arr.length - 1]; console.log(last);

Method 2

You can get the last element of an array using the methodpop:

let arr = [1, 2, 3, 4, 5]; let last = arr.pop(); console.log(last);

Method 3

You can reverse an array using the methodreverseand take the zero element:

let arr = [1, 2, 3, 4, 5]; arr.reverse(); let last = arr[0]; console.log(last);
