Последнее значение массива JS

Последнее значение в массиве JavaScript можно получить различными способами, в зависимости от того, что именно нужно сделать. Давайте разберем эти способы.

Вот несколько способов получения последнего элемента массива:

Способ 1

Можно получить последний элемент массива по индексу:

let arr = [1, 2, 3, 4, 5]; let last = arr[arr.length - 1]; console.log(last);

Способ 2

Можно получить последний элемент массива с помощтю метода pop:

let arr = [1, 2, 3, 4, 5]; let last = arr.pop(); console.log(last);

Способ 3

Можно перевернуть массив с помощтю метода reverse и взять нулевой элемент:

let arr = [1, 2, 3, 4, 5]; arr.reverse(); let last = arr[0]; console.log(last);
