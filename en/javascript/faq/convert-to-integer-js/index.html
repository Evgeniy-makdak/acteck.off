Cast to JS integer

Integer casting is the process of converting a floating point number to an integer. JavaScript has several methods to perform this operation.

Math.round method

MethodMath.roundRounds a number to the nearest whole number. If the number is equal to or greater0.5, then it is rounded up, otherwise - down.

Example 1:

let num = 5.3; let res = Math.round(num); console.log(res); // выведет 5

Example 2:

let num = 5.7; let res = Math.round(num); console.log(res); // выведет 6

Math.floor method

MethodMath.floorrounds a fraction to a whole number, always rounding down. See example:

let num = 5.8; let res = Math.floor(num); console.log(res); // выведет 5

Math.ceil method

MethodMath.ceilrounds a fraction to a whole number, always rounding up. See example:

let num = 5.8; let res = Math.ceil(num); console.log(res); // выведет 6

parseInt function

FunctionparseIntconverts a string to an integer. This is needed for values ​​like'12px'- when the number comes first, and then the units of measurement. If you apply the functionparseIntTo'12px', then the result will be the number12(and it will actually be a number, not a string).

Example 1:

let num = '6.3px'; let res = parseInt(num); console.log(res); // выведет 6

Example 2:

let num = '6.8px'; let res = parseInt(num); console.log(res); // выведет 6
