How to round a number in JS

Upwards

Round number uppossible using the methodMath.ceil. See example:

let num = 4.32277; let res = Math.ceil(num); console.log(res);

Result of the executed code:

5

Downward

Forrounding numbers downyou can use the methodMath.floor. See example:

let num = 4.62277; let res = Math.floor(num); console.log(res);

Result of the executed code:

4

Until the whole

Toround a number to the nearest integer valueshould use the methodMath.round. See example:

let num = 4.32277; let res = Math.round(num); console.log(res);

Result of the executed code:

4

Using the Math.toFixed method

Round the number to the specified decimal placepossible through the methodMath.toFixed, whose parameter specifies the number of characters. If the number of characters is not specified, then the default is taken0, i.e., rounding to a whole number is performed. See example:

let num = 4.62577; let res = num.toFixed(2); console.log(res);

Result of the executed code:

4.63

Using the toPrecision method

In case you needround a number to a given lengthyou can use the methodtoPrecision. UnlikeMath.roundRounding can be done not only in the fractional part. The method parameter specifies how many digits should remain in the number. The remaining digits will be discarded. The last remaining number will be rounded using mathematical rounding rules. If the parameter is empty, the original number will be returned. If the specified number of digits is not achievable by cutting off the decimal part, converts the number to exponential form. See example:

let num = 4.6257714; let res = num.toPrecision(4); console.log(res);

Result of the executed code:

4.626
