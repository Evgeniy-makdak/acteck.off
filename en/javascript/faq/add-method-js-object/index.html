Add a method to a JS object

In this lesson we will study4way to add a new method to an object. For example, let us have the following object with a user:

let obj = { name: 'john', surn: 'smit', };

Let's add a new method to this object using various methods.

Method 1. Directly

You can add a method immediately when declaring an object:

let obj = { name: 'john', surn: 'smit', full: function() { return this.name + ' ' + this.surn; } };

Let's call our method:

console.log(obj.full()); // выведет 'john smit'

Method 2. After the announcement

You can add a method after an object is declared. Let's first declare the object:

let obj = { name: 'john', surn: 'smit', full: function() { return this.name + ' ' + this.surn; } };

Now let's add a new method:

let obj.full = function() { return this.name + ' ' + this.surn; };

Let's call our method:

console.log(obj.full()); // выведет 'john smit'

Method 3. Through Object.defineProperty

MethodObject.definePropertyalso allows you to add a new method to an object. Let's do this:

Object.defineProperty(obj, 'full', { get: function() { return this.name + ' ' + this.surn; } });

Let's call our method:

console.log(obj.full()); // выведет 'john smit'

Method 4. Through Object.assign

MethodObject.assignallows us to combine several objects into one. We can use it to add methods to an existing object. Let's see in practice. Let us have the following object:

let obj = { name: 'john', surn: 'smit', };

Let us also have a function:

function full() { return this.name + ' ' + this.surn; }

Let's combine our object and the new object with our function together:

obj = Object.assign({}, obj, {full: full});

Or you can use an anonymous function:

obj = Object.assign({}, obj, {full: function() { return this.name + ' ' + this.surn; }});

Let's call our method:

console.log(obj.full()); // выведет 'john smit'
