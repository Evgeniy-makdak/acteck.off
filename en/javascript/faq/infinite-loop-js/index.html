JS infinite loop

If the condition for exiting the loop is not met in the body of the loop, the loop can run indefinitely. Look at the example:

let bool = true; while (bool) { console.log('!'); }

And in the following example, the counter increment is forgotten in the loop. This loop will also run forever:

let i = 1; while (i < 100) { console.log('!'); }
