JS loop from the end

Example 1

Let's run a regular loopforfrom the end:

for (let i = 100; i >= 0; i--) { console.log(i); }

Example 2

Let's start the cycleforthrough the array from the end:

let arr = [1, 2, 3, 4, 5]; for (let i = arr.length - 1; i >= 0; i--) { console.log(arr[i]); } console.log(arr);

Example 3

We use the methodreverseto flip the array and then loop through itfor-of:

let arr = [1, 2, 3, 4, 5]; arr.reverse(); for (let elem of arr) { console.log(elem); }
