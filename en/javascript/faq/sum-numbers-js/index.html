Сумма чисел JS

Сумма чисел в JavaScript может быть найдена с помощью встроенного метода, цикла или свертки. Давайте разберем все эти способы.

Встроенный метод

В JavaScript есть встроенный метод Math.sum, который может быть использован для нахождения суммы чисел из массива. Пример:

let arr = [1, 2, 3, 4, 5]; console.log(Math.sum(arr));

Цикл

Можно использовать цикл для перебора всех чисел и добавления их к сумме. Например:

let arr = [1, 2, 3, 4, 5]; let sum = 0; for (let num of arr) { total += num; } console.log(sum);

Свертка

Можно также использовать метод reduce:

let arr = [1, 2, 3, 4, 5]; let res = arr.reduce(function(sum, elem) { return sum + elem; }, 0); console.log(res);
