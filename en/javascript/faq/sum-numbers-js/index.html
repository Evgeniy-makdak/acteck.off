Sum of numbers JS

The sum of numbers in JavaScript can be found using a built-in method, loop or convolution. Let's look at all these methods.

Built-in method

JavaScript has a built-in methodMath.sum, which can be used to find the sum of numbers from an array. Example:

let arr = [1, 2, 3, 4, 5]; console.log(Math.sum(arr));

Cycle

You can use a loop to iterate through all the numbers and add them to the sum. For example:

let arr = [1, 2, 3, 4, 5]; let sum = 0; for (let num of arr) { total += num; } console.log(sum);

Convolution

You can also use the methodreduce:

let arr = [1, 2, 3, 4, 5]; let res = arr.reduce(function(sum, elem) { return sum + elem; }, 0); console.log(res);
