Add key-value to JS object

Let's look at how to add a new key-value pair to an object. For example, let's say we have the following object:

let obj = { name: 'john', surn: 'smit', };

Adding via key

The easiest way to add a new key-value pair is to write directly to the new key:

obj['age'] = 30;

Or via an alternative method:

obj.age = 30;

Let's check the result:

console.log(person);

Object.assign method

You can also add new key-values ​​using the methodObject.assign:

let res = Object.assign(obj, { age: 25 });

Let's check the result:

console.log(res);

Via spread

Operatorspreadallows you to create a new object from an existing one by adding new properties. Here is an example of use:

let res = { ...obj, age: 25 };

Let's check the result:

console.log(res);
