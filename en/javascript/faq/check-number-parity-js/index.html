Check number parity JS

There are several ways to check the parity of a number in JavaScript. Let's look at them.

Checking the remainder of a division

The easiest way to determine the parity of a number is to check the remainder when the number is divided by2. For an even number the remainder will be0, and for an odd number -1. Let's try it in practice. Let us have the following number:

let num = 10;

Let's check it for parity:

if (num % 2 == 0) { console.log('четное'); } else { console.log('нечетное'); }

Using bit operations

You can also check parity using bit operations. Let's try it in practice. Let us have the following number:

let num = 10;

Let's check it for parity:

let num = 10; if (num & 1 === 0) { console.log('четное'); } else { console.log('нечетное'); }

Bit operations are faster, especially for large numbers, but are not very intuitive for novice developers.
