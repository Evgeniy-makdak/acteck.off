Проверить четность числа JS

В JavaScript существует несколько способов проверки четности числа. Давайте их разберем.

Проверка остатка от деления

Самый простой способ определить четность числа - проверить остаток от деления этого числа на 2. Для четного числа остаток будет 0, а для нечетного числа - 1. Давайте попробуем на практике. Пусть у нас есть следующее число:

let num = 10;

Проверим его на четность:

if (num % 2 == 0) { console.log('четное'); } else { console.log('нечетное'); }

Использование битовых операций

Можно также проверить четность с помощью битовых операций. Попробуем на практике. Пусть у нас есть следующее число:

let num = 10;

Проверим его на четность:

let num = 10; if (num & 1 === 0) { console.log('четное'); } else { console.log('нечетное'); }

Битовые операции работают быстрее, особенно для больших чисел, но не очень понятные для начинающих разработчиков.
