Swap the value of JS variables

In this article we will look at 3 ways to swap the value of two variables. Here they are:

Method 1. Not working

Directly assigning the value of variables to each other is not a working way:

let a = 1; let b = 2; a = b; b = a; console.log(a); // выведет 2, верно console.log(b); // выведет 2, неверно - нужно 1

Method 2. Temporary variable

In order to correctly swap the values ​​of variables, you need to add a temporary variable:

let a = 1; let b = 2; let tmp = a; a = b; b = tmp; console.log(a); // выведет 2, верно console.log(b); // выведет 1, верно

Method 3. Destructuring

In an elegant way, you can change the values ​​of variables usingdestructuring

let a = 5; let b = 10; [a, b] = [b, a]; console.log(a); // выведет 2, верно console.log(b); // выведет 1, верно
