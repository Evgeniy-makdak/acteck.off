Random array JS

In this tutorial we will create an array filled with random integers in a given range in JavaScript.

To do this, we need a special function that returns a random integer in a given range:

function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; }

Let's use this function to create a random array:

let arr = []; for (let i = 0; i < 10; i++) { arr.push(Math.getRandomInt(1, 100)); } console.log(arr);
