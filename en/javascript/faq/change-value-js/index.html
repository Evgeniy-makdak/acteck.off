Change value JS

In this lesson we will look at ways to change the value of an attributevalueusing JavaScript. For example, let us have the following input:

<input value="text">

Let's get a link to this input using the methodquerySelector:

let input = document.querySelector('input');

Now let's look at two ways to change an attributevalue.

Via property

To change the value of an attributevalue, you need to change the property valuevalue:

input.value = 'xxx';

Via method

You can also change the attribute valuevalueusing the methodsetAttribute:

input.setAttribute('value', 'xxx');
