Checking an Array Element

You can check if an element with a given value exists in an array. This is done using the methodincludes. Let's try:

let arr = [1, 2, 3, 4, 5]; let isElem = arr.includes(3); console.log(isElem); // выведет true

Checking the DOM element

You can check for the presence of a DOM element using the methodquerySelector. If the element is found, the method will return it. If the element does not exist, the method will return the valuenull. Here's how you can use it:

let elem = document.querySelector('селектор элемента'); if (elem) { // используем элемент } else { console.error('элемент не найден'); }
