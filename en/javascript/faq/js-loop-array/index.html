JS loop iterating through an array

for loop

You can iterate through an array using a loopfor. See example:

let arr = ['a', 'b', 'c', 'd', 'e']; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Result of executing the code:

12345

for-of loop

Cyclefor-oftraverses iterable objects (arrays, set and map collections, list of arguments, etc.):

let arr = ['a', 'b', 'c', 'd', 'e']; for (let elem of arr) { console.log(elem); }

Result of executing the code:

'abcde'

forEach method

You can also use it to iterate through array elementsforEach method. See example:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.forEach(function(elem) { console.log(elem); });

Result of executing the code:

'abcde'
