JS how to make a link

In this lesson we will look at how to dynamically create an HTML link using JavaScript and add it to an element.

Let us have the following block to which we will add a link:

<div id="elem"></div>

Let's get this block into a variable using the methodquerySelector:

let elem = document.querySelector('#elem');

Let's create a new HTML link using the methodcreateElement:

let link = document.createElement('a');

Add text to the link using the propertytextContent:

link.textContent = 'text'

Let's add our created link to the end of the block using the methodappend:

elem.append(link);
