JS Button Click Function

In this tutorial we'll look at how to make a function that runs when a button is clicked in JavaScript. Let's say we have the following button:

<button id="btn">btn</button>

Let's get a link to our button using the methodquerySelector:

let btn = document.querySelector('#btn');

Using the methodaddEventListenerLet's attach a function to the button that will be executed when the button is pressed:

btn.addEventListener('click', function() { });

Let's do something in this function:

btn.addEventListener('click', function() { alert('!!!'); });
