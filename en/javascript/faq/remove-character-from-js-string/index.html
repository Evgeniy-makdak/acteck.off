Remove character from JS string

Removing a character from a string in JavaScript is done using the methodreplace, which searches and replaces across a string. In the first parameter of the method we indicate the character to be removed, and in the second - empty quotes. Example:

let str = 'a-b-c'; let res = str.replace('-', ''); console.log(res); // выведет 'ab-c'

As you can see, only the first character found is removed. To remove all characters you need to useregular expressionsand modifierg:

let str = 'a-b-c'; let res = str.replace(/-/g, ''); console.log(res); // выведет 'abc'
