JS get checkbox value

In this tutorial we will look at ways in which you can get the value of a checkbox through JavaScript. Let us have the following checkbox:

<input type="checkbox" value="1">

Let's get a link to this checkbox using the methodquerySelector:

let checkbox = document.querySelector('input[type="checkbox"]');

Let's consider various options for obtaining the value of a checkbox.

Option 1

Let's get and display the value of the checkbox (the contents of the attributevalue):

let value = checkbox.value; console.log(value); // выведет 1

Option 2

Let's get and display the state of the checkbox (checked or not):

let checked = checkbox.checked; console.log(value);
