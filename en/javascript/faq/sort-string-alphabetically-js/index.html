Sort a string alphabetically in JS

In this lesson we will look at how to sort words in a string in alphabetical order. For this we use the methodsplitto split the string into an array of words, then sort the array using the methodsortand merge it back into a string using the methodjoin:

let fruits = 'apple cherry banana'; fruits.split(' ').sort().join(' '); console.log(fruits); // 'apple banana cherry'
