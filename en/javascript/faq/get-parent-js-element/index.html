Get the parent of a JS element

In this tutorial we will look at ways to get the parents of an element in JavaScript.

First, let's make the following HTML code with a parent and child element:

<div id="grand"> <div id="parent"> <div id="child">text</div> </div> </div>

Let's get a reference to our child element using the methodquerySelector:

let elem = document.querySelector('#child');

Let's now, having a reference to a child element, get its parent element. Let's look at various methods.

parentElement property

The immediate parent of an element can be obtained using the propertyparentElement:

let parent = elem.parentElement; console.log(parent);

closest property

Using the methodclosestYou can get the closest parent that matches a given CSS selector:

let parent = elem.closest('div'); console.log(parent);

parents property

Using propertyparentsYou can get all the parent elements of the current element, starting from the top level:

let parents = elem.closest('div'); console.log(parents);
