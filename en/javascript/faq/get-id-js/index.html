Получить id JS

В этом уроке мы рассмотрим способ получения id элемента. Для примера пусть у нас есть следующий элемент:

<div id="elem">text</div>

Получим ссылку на наш элемент с помощью метода querySelector:

let elem = document.querySelector('div');

Прочитаем id элемента:

console.log( elem.id );
