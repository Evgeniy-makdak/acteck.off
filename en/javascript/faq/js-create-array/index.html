JS create array

An array is a variable in which a whole set of values ​​can be stored in an ordered form. Square brackets are used to create an array:

let arr = []; // создаем массив arr

Currently the created array does not contain any data. Let's fill it with data:

let arr = [1, 2, 3, 4, 5];

Each list value that we wrote into the array is calledelementarray.

Elements are separated by a comma. You can put spaces after this comma, or you don’t have to (for code readability, it’s customary to put spaces).

In addition to numbers, an array can storelines. Each line element must be placed in quotes:

let arr = ['a', 'b', 'c', 'd', 'e'];
