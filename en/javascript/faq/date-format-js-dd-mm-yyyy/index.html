js date format dd-mm-yyyy

You can set the date format in JavaScript manually by concatenating parts of the date through the desired separator. For example, let's display the date in the formatdd.mm.yyyy, that isday.month.year:

let d = new Date(); let res = d.getDate() + '.' + (d.getMonth() + 1) + '.' + d.getFullYear(); console.log(formattedDate);

Let's make sure that the day or month number is appended with a zero if necessary. To do this, we will create a special function:

funciton addZero(num) { if (num <= 9) { return '0' + num; } else { return num; } }

Let's use our function:

let d = new Date(); let res = addZero(d.getDate()) + '.' + addZero((d.getMonth() + 1) + '.' + d.getFullYear(); console.log(formattedDate);
