Cut JS string

Let's give the string str and see how we can truncate the string to a specific character:

let str = 'abcde'; // будем работать с этой строкой

substr method

To get part of a substring up to a specific character, you can use the methodsubstr. See example:

console.log(str.substr(0, 2));

Result of executing the code:

'ab'

substring method

To get part of a substring up to a specific character, you can use the methodsubstring. See example:

console.log(str.substring(0, 2));

Result of executing the code:

'ab'

slice method

To get part of a substring up to a specific character, you can use the methodslice. See example:

console.log(str.slice(0, 2));

Result of executing the code:

'ab'
