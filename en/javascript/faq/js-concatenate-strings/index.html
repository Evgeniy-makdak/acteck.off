JS concatenate strings

Concatenating strings in JavaScript can be done in several ways. The choice of method depends on the specific situation and preferences of the developer.

Method 1

We use the '+' operator:

let str1 = 'aaa'; let str2 = 'bbb'; let res = str1 + str2; console.log(res); // 'aaabbb'

Method 2

Let's combine the lines, separating them with a space:

let str1 = 'aaa'; let str2 = 'bbb'; let res = str1 + ' ' + str2; console.log(res); // 'aaa bbb'

Method 3

Can be usedtemplate strings:

let str1 = 'aaa'; let str2 = 'bbb'; let res = `${str1} ${str1}`; console.log(res); // 'aaa bbb'

Method 4

Using the methodconcat:

let str1 = 'aaa'; let str2 = 'bbb'; let res = ''.concat(str1, str2); console.log(res); // 'aaabbb'

Method 5

Using the methodjoinYou can concatenate strings from an array with a given delimiter:

let arr = ['aaa', 'bbb']; let res = arr.join(' '); console.log(res); // 'aaa bbb'
