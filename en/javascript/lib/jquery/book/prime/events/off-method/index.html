jQuery off method

Unbinding events bound viaon, is produced using the methodoff. It takes the event type as its first parameter (for example,'click'), and the second is the name of the function to unbind.

Consider the following example. Let's say we have HTML code:

<ul> <li>text</li> <li>text</li> <li>text</li> </ul>

The CSS specifies the style for the tagli:

li { width: 100px; cursor: pointer; }

Let's make sure that after the first click onli, the event was untied from it:

$('li').on('click', function() { $(this).append('!'); $(this).off('click', func); });

Click on the list items. You will see that when you click on the same list item again, nothing happens because our functionfuncafter executing the methodappenduntied by methodoff.

Bind an event to all links - when you hover over a link, it is appended to the end of its texthrefin parentheses. After the first hover over the link, you should unbind an event from it that addshrefto the end of the text.

For all inputs, make them output their ownvaluewhen you click on any of them, but only on the first click. Pressing the input again should not cause a reaction.

Paragraphs with numbers are given. By clicking on a paragraph, the square of the number it contains should appear, but only on the first click. By double-clicking on a paragraph, the number in the paragraph should double, but also only the first time.
