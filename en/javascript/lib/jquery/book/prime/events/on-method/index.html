Generic on method in jQuery

You can also use the generic method to bind eventson. It takes the name of the event as its first parameter (for example,'click'), and the second is the function to bind.

Let's look at an example using the following HTML code:

<ul> <li>text</li> <li>text</li> <li>text</li> </ul>

The CSS looks like this:

li { width: 100px; cursor: pointer; }

Now let's rewrite the code forli, which we wrote earlier through the methodon. To check the operation, click on the list items:

$('li').on('click', function() { $(this).append('!'); });

You can simultaneously bind one function to several types of events - to do this, you need to list them separated by a space:'click mousemove и т.д.'. For example, like this:

$('li').on('click mousemove', function func() { $(this).append('!'); });

Bind an event to all links - when you hover over a link, it is appended to the end of its texthrefin parentheses.

Bind an event to all inputs - when focus is lost, each input outputs its ownvaluein paragraph withid=#test.
