Event binding in jQuery

Almost every JavaScript event has its own jQuery method. For example, a click on elements can be caught like this:$(селектор).click(функция).

Consider the following HTML code:

<ul> <li>text</li> <li>text</li> <li>text</li> </ul>

The CSS for it is written:

li { width: 100px; cursor: pointer; }

Let's make it so that when you click on anylian exclamation mark was displayed. Click on the list items:

$('li').click(function() { alert('!'); });
