jQuery submit method

Previously we used this construction:

$('form').on('submit', function(event) { event.preventDefault(); });

Instead, we can also directly use the jQuery methodsubmit, which binds a form handler or executes an event:

$('form').submit(function(event) { event.preventDefault(); });

In the following example, we have a simple form to which we will attach an event handler using the methodsubmit, with a text field and a button - input with typesubmit:

<p>jQuery</p> <form action="/"> <div> <input type="text"> <input type="submit"> </div> </form> <span></span>

Let's enter text into a text field, and when the button is clicked, check this text. If we enter'jQuery', then we will show the text in the span'Good!', using methodstextAndshow, and if something else, then -'Bad...':

$('form').submit(function(event) { event.preventDefault(); if ($('input').first().val() === 'jQuery') { $('span').text('Good!').show(); return; } $('span').text('Bad...').show(); });
