Overriding default action in jQuery

As you should already know, a user action in JavaScript is canceled usingevent.preventDefault(). Let's prevent the form data from being submitted, as well as further event bubbling:

$('form').on('submit', function(event) { event.preventDefault(); });

You can also prevent just the form submission (but not the event from bubbling) in jQuery style - to do this, the event handler function must returnfalse:

$('form').on('submit', function() { return false; });

Of course, you can cancel any event, such as clicking on a link. For example, we have the following link in the HTML code:

<a href="/">link</a>

Now let's use undo:

$('a').on('click', function () { alert('Вы не можете перейти по этой ссылке!'); return false; });
