One-time event in jQuery

When we were learning about unbinding events using the methodoff, we used the following construction:

$('li').on('click', function() { $(this).append('!'); $(this).off('click', func); });

First we attached the handler function usingon, then untied usingoff.

jQuery has a convenient methodone, which allows you to bind a one-time event - it will be executed only once, and then automatically unbound. This method takes the event type as the first parameter and the bound function as the second.

We will look at the next example based on the HTML code below:

<ul> <li>text</li> <li>text</li> <li>text</li> </ul>

The CSS code looks like this:

li { width: 100px; cursor: pointer; }

Now to eachlibind a one-time event:

$('li').one('click', function() { $(this).append('!'); });

Click on the list items. As you can see, we got the same effect thanks to the methodone.
