Getting started with jQuery

Working with the jQuery library is done using the universal function$, which for brevity consists of just one character.

After the dollar, you should write parentheses (this is a function), which indicate the parameters (usually one parameter), something like this:$(параметр).

Most often using the dollar$get a group of HTML elements using a CSS selector.

Let's get all elements with classwww:

let elems = $('.www');

And now we get only paragraphs with the classp.www:

let elems = $('p.www');

Now we get the element withid, equalwww:

let elems = $('#www');

As you can see, regular CSS selectors are used, similar to the methodquerySelectorAll. OnlyquerySelectorAllappeared much later than jQuery, and has fewer capabilities (but works faster).
