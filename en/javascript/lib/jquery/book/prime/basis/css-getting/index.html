Working with CSS via jQuery

To work with CSS, jQuery provides a methodcss. It takes the name of the CSS property as its first parameter, and its new value as its second.

Let's say we have the following code:

<p class="www">text</p>

Let's get all elements with classwwwand change their color to red:

$('.www').css('color', 'red');
