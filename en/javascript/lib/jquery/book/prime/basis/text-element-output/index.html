Displaying element text using jQuery

MethodshtmlAndtextcan be used not only to change the text of elements, but also to display it on the screen. Let's get the element and print its text:

<p id="test">text</p> alert($('#test').html());

If you try to get the text of more than one element, you will only see the contents of the first element. Use methodeachto get the texts of all elements, it will be discussed in the following lessons.

In the example below we get all paragraphs with the classwww:

<p class="www">text1</p> <p class="www">text2</p> <p class="www">text3</p>

Then we output the contents usinghtml, in this case only the text of the first paragraph from the received ones will be displayed:

alert($('.www').html()); // выведет 'text1'

With methodtextYou also need to be careful, in our case it will display the contents of all found elements:

alert($('.www').text()); // выведет 'text1text2text3'
