Working with the each method in jQuery

When we have a group of jQuery-derived elements, we can only make changes to them for all the elements at once.

For example, if we want to change the text of all found paragraphs, we will not be able to make it different.

In this case, a special method will help useach, which allows you to apply a function to all elements of a jQuery set. Moreover, within this function we will be able to separate the elements and act differently with each of them.

Essentiallyeachis a loop that can be used to iterate through all found elements. The link to the element through which the cycle is now passing will be inthis.

We can just take advantage of thisthisin pure JavaScript, as we did before, for example, like this -this.innerHTML- and display the internal contents of our elements. But it's better to wrap it upthisto dollar jQuery this way -$(this)- in this case we will be able to apply all jQuery methods and chains to it.

Let's look at the following HTML code:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

In this example, we need to get all elements with the classwwwand display their contents. How this is done: using$('.www')we get the elements we need, then usingeach(test)we apply the function to each received elementtest. It will first be applied to the first paragraph, then to the second, to the third, and so on.

Inside a functionthiswill point to the element to which the function is appliedtest- first it will be the first element, then the second, and so on. With this design$(this)we instead of usualthisWe get a jQuery element from JavaScript and apply the method to ithtmlwhich will receive the text of our element. Well, then it will simply be displayed on the screen:

/* Имя функции test пишем без кавычек и (), так как нам нужен ее код, а не результат: */ $('.www').each(test); function test() { alert($(this).html()); }

To solve this problem, you can also use anonymous functions - this is what is done most often:

$('.www').each(function() { alert($(this).html()); });

We can also pass to the methodeachcallback function with parameters.

Let's add to the end of all in the next examplelitheir serial number is on the page. Now we will pass the element number and the element itself to our anonymous function:

$('li').each(function (index, elem) { $(elem).append(index); });

And now insteadelemwe usethis:
