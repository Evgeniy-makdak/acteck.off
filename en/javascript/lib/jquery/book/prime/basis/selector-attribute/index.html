Attribute selector in jQuery

Will select elements that contain an attribute or an attribute with a specific value, for example:

$("input[type='text']");

The attribute selector also looks for elements starting with a specific value, for example:

$("a[href^='http://']");

And elements ending with a specific value:

$("a[href$='.ru']");

In addition, using this selector you can select elements containing a specific value anywhere, for example:

$("a[target*='blank']");
