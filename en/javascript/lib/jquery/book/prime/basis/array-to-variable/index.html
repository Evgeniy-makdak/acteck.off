Writing a set of elements to a variable using jQuery

You don't have to build a chain$('.www').html('!!!'), and first write down the set of elements$('.www')to a variable (I called itelems), and then apply the method to this variable.html('!!!'). For example, our HTML code looks like this:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

And this is what the code written in JavaScript will look like:

let elems = $('.www'); elems.html('!!!');

There is a convention that it is advisable to use for convenience: the names of variables that contain a jQuery wrapped set usually begin with a dollar. That is, in our case it is not necessaryelems, A$elems.

If you are confused by this variable name -$elems- note that the dollar in JavaScript is a regular symbol and can be used anywhere. You can even make the function name consist of one character$, as is done in jQuery and some other libraries.

So, once again - if you want to write a jQuery array into a variable, then that variable usually starts with a dollar. This is not necessary, but it allows you to understand at first glance that this variable contains a group of elements and all jQuery methods are applicable to this variable.

Let's use this in our example with the below HTML code:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

The JavaScript looks like this:

let $elems = $('.www'); $elems.html('!!!');

Similarly, to insert text, you can use the methodtext:

$('.www').text('!!!');
