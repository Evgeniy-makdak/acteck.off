What is jQuery?

jQuery is a JavaScript library - a set of ready-made functions that help you do some things easier and more convenient than is done in pure JavaScript.

Compare two codes that do the same thing - the first is in pure JavaScript:

document.getElementById('elem').innerHTML = '!';

And the second one is in jQuery:

$('#elem').html('!');

jQuery code looks much more compact and easier to write.

Compare two more codes: in the first we get all elements with the classwwwand set their text color to red using pure JavaScript:

let elems = document.getElementsByClassName('www'); for (let i = 0; i < elems.length; i++) { elems[i].style.color = 'red'; }

And in the second - in jQuery:

$('.www').css('color', 'red');

There is already a huge difference here - in pure JavaScript we had to make a loop, but the jQuery code hasn’t changed much compared to the first example.
