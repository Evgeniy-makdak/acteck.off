Селектор соседей jQuery

С помощью jQuery можно выбрать элементы p, расположенные после h3 и являющимися его соседями:

$('h3 ~ p');

Также с помощью данного селектора можно выбирать элементы p, идущие непосредственно за h3 и являющимися для него соседями:

$('h3 + p');
