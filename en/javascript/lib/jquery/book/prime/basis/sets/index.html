jQuery Sets

Notice the variableelems:

let elems = $('.www');

A group of selected elements, the so-calledjQuery elements set.

You can change all elements of a jQuery set at the same time. UnlikequerySelectorAllno cycles required.

Basically, the variableelemsmost often it is not needed, since jQuery allows you to apply useful functions directly to the set, in the form of methods.

For example, this is how we get all elements with the classwwwand give them red color:

$('.www').css('color', 'red');

Almost all jQuery methods return the set to which the method was applied, allowing you to chain methods of any length.

Let's get all elements with classwww, give them red color, and change their text to'новый текст':

$('.www').css('color', 'red').html('new text');
