Inserting tagged text using html method in jQuery

Methodtextallows you to work only with text, in contrast to it usinghtmlYou can insert not only text content, but also text with tags, in other words, an element with text. Source HTML code:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Then we use the html method:

$('.www').html('<span>!!!</span>');

And the HTML code will look like this:

<p class="www"><span>!!!</span></p> <p class="www"><span>!!!</span></p> <p class="www"><span>!!!</span></p> <p>text</p>

In everythingh2insertspanwith text'!'.
