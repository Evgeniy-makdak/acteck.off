Positioning in jQuery

jQuery has methods for working with element coordinatespositionAndoffset.

These methods are used to get or change the current coordinates of an element and return an object containing properties as a resulttopAndleft.

Calculation errors may occur if the user resizes the page. Also, the methods do not receive the coordinates of hidden elements.

Methodoffsetgets the element's position relative to the document, andpositionthe element's position relative to the parent's padding.

Let's say we have the following HTML code:

<div id="parent"> <p id="test">text</p> </div> <p id="text1"></p> <p id="text2"></p>

CSS styles are attached to paragraphs:

#parent { width: 150px; border: 1px solid blueviolet; } #test { margin: 5px; text-align: center; outline: 1px solid green; }

Let's get the diva positiontestusing these methods, and referring to the keys of the resulting object, we will display this information below in paragraphs:

let offset = $('#test').offset(); $('#text1').text( '(OFFSET()) left: ' + offset.left + ', top: ' + offset.top ); let position = $('#test').position(); $('#text2').text( '(POSITION()) left: ' + position.left + ', top: ' + position.top );
