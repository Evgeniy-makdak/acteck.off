Changing a single property attribute in jQuery

It is important not to confuse element properties and attributes. When creating a DOM tree, attributes are translated into their corresponding DOM element properties. Their names do not always coincide, nor do their meanings.

Some properties may also not have corresponding attributes; for these you must use the methodprop, which takes the property name as the first parameter, andtrueorfalse.

If you puttrue- the property will be installed, and iffalse- will be deleted.

Consider the following example. Let us have an input:

<input type="text" id="test">

Let's set the input attributedisabled:

$('#test').prop('disabled', true);

The HTML code will look like this:

<input type="text" id="test" disabled>

And now, on the contrary, let’s removedisabled:

$('#test').prop('disabled', false);

The HTML code will look like this:

<input type="text" id="test">
