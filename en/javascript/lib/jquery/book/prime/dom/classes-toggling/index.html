Working with the toggleClass method in jQuery

There is also a useful methodtoggleClass, which will add the specified class if it does not exist, and remove it if it exists. It may be needed so that when you click on the same button with a page element, periodic changes occur.

Let's look at this method using an example. Let's say we have the following CSS styles:

.red { color: red; } .zzz { font-style: italic; }

There is also a paragraph with a classzzz:

<p class="zzz" id="test">text</p> <button>click</button>

Let's, for example, make sure that when the button is clicked for the first time, the class will be added to the elementred, which adds red color, and when repeated, this class will be removed and the red color will disappear. If you press the button again, everything will happen again.

Try running the following code and see for yourself (classzzzhighlightsitalicsto show thattoggleClasswill not interfere with other classes' work):

$('button').click(function() { $('#test').toggleClass('red'); });

After the first click, the HTML code will look like this (the class will be addedred):

<p class="zzz red" id="test">text</p> <button>click</button>

And after the second click - like this (the class will disappearred):

<p class="zzz" id="test">text</p> <button>click</button>
