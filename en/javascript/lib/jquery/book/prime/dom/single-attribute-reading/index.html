Считывание одного атрибута-свойства в jQuery

Можно не только изменять значения атрибутов-свойств, но и считывать их. К примеру, в инпуте указано свойство disabled:

<input type="text" id="test" disabled>

Указываем для атрибута только свойство disabled, без булевых значений:

alert($('#test').prop('disabled')); // выведет 'true' - атрибут есть
