Reading a single property attribute in jQuery

You can not only change the values ​​of property attributes, but also read them. For example, the input specifies the propertydisabled:

<input type="text" id="test" disabled>

Specify only the property for the attributedisabled, without boolean values:

alert($('#test').prop('disabled')); // выведет 'true' - атрибут есть
