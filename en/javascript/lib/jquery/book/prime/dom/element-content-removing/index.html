Removing element content in jQuery

Using the methodemptyyou can delete the contents of the elements (the elements themselves will not go anywhere). For example, there are the following paragraphs:

<p>text</p> <p>text</p> <p>text</p>

Let's delete their contents:

$('p').empty();

The HTML code will look like this:

<p></p> <p></p> <p></p>
