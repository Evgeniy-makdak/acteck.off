Adding a class to an element in jQuery

I remind you that in the attributeclassIn HTML code you can write several classes separated by spaces.

Let's imagine this situation - you want to add a new class to the existing ones without overwriting them. In this case, just go ahead and do it.attr('class', 'new-class')will not work (using the methodattr) - you will overwrite those classes that are already in the attribute.

You can, of course, first count the classes that are there, then add our class to them separated by a space and write everything back to the attribute, but there is an easier way - jQuery has a method for such thingsaddClass.

MethodaddClassallows you to add a class to an element (or elements) without losing other classes. The class you want to add to the element is taken as a parameter to this method.

In the following example, we have an element with two classes -class="www zzz":

<p class="www zzz" id="test">text</p>

We can add a third class to it -kkk. As a result, the contents of the attribute will become like this -class="www zzz kkk":

$('#test').addClass('kkk');

The HTML code will look like this:

<p class="www zzz kkk" id="test">text</p>
