Removing an element's parent using the unwrap method in jQuery

On the contrary, you can not wrap the element, but expand it (that is, remove the parent) using the methodunwrap.

In the following example we have paragraphs that are inside tagsdiv:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p>text</p></div>

Let's for paragraphs with classwwwperform the unfolding operationunwrap:

$('.www').unwrap();

The HTML code will look like this:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <div><p>text</p></div>

You can also pass a selector to the method as a string, then it will fire if the parent of the element matches the specified selector:

$('i').unwrap('p');
