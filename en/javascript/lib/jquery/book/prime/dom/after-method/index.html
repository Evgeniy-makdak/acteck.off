jQuery after method

Methodafterallows you to add text after elements.

Consider the following example, let's say we have a paragraph:

<p>text</p>

Now we write the following Javascript code:

$('p').after('<b>!!!</b>');

After executing the code, the paragraph will look like this:

<p>text</p><b>!!!</b>

It should be noted that all the above insertion methods areprepend, append, beforeAndafteralso provide the ability to apply a function to each element in the set.

Let's find all the paragraphs and at the end of each of them put its serial number in the set:

<p>text</p> <p>text</p> <p>text</p> $('p').append(function(index){return index;});

After eachh2insert'<p>?</p>'.

One chain in front of eachh2insert'<p>!</p>', and afterh2 - '<p>?</p>'.
