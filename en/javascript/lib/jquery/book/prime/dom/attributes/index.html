Working with attributes in jQuery

Now we will learn how to read, add and change HTML tag attributes using jQuery. All this is done using the methodattr. It works similar to the methodhtml- depending on the number of parameters, it will read or change the attribute value.

The first parameterattrtakes the name of the attribute, and the second is its new value. For example, like this -.attr('value', 'www')- we will write the line'www'to attributevalue.

If you callattrwith one parameter, then it will return the values ​​of the attribute whose name we specify in the first parameter. For example, like this -.attr('value')- we will get the attribute valuevalue.

In the following example we have an input:

<input type="text" value="!!!" id="test">

Next we receive input from itidand display the value of its attributevalue:

alert($('#test').attr('value')); // выведет '!!!'
