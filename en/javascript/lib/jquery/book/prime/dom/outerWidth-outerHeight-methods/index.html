jQuery outerWidth and outerHeight methods

The last couple of methods areouterWidthAndouterHeight, with the help of which it is possible to obtain the width and height of an element, taking into account all the padding and borders, or you can ignore the external paddingmargin.

Let's look at an example. Let's say we have a paragraph:

<p id="test">text</p> <p id="out"></p>

The CSS looks like this:

p { margin: 10px; padding: 5px; border: 2px solid blue; }

Let's get the paragraph width and height values#test, taking into account padding and borders, to do this, you need to pass these methodsfalseor leave the parentheses empty (since it is already in the method by default):

let w = $('#test').outerWidth(false); let h = $('#test').outerHeight(); $('#out').text('Width: ' + w + ' Height: ' + h);

Now our values ​​will differ by another4px, since the thickness of the boundaries is2pxon each side.

Now let's get the paragraph width and height values#test, taking into account all the indents and borders, for this purpose in the methodsouterWidthAndouterHeightmust be transferredtrue:

let w = $('#test').outerWidth(true); let h = $('#test').outerHeight(true); $('#out').text('Width: ' + w + ' Height: ' + h);

Now our values ​​will differ from the previous ones by another20px, since we have external margins according to10pxon each side.

Complete the solution to the previous problems - display in the third paragraph under the div#wrapperdiv width and height values#wrapper- taking into account internal padding and borders, but without external padding. In the fourth paragraph, print the width and height of the div.#wrappertaking into account all indents and borders.

Also using methodsouterWidthAndouterHeightwe can change the width and height values ​​of the elements. For example, let's set the width of the div#test1equal to250px, and the height#test2 - 200px:

$('#test1').outerWidth(250, true); $('#test2').outerHeight(200);

All of the above methods for working with element sizes also provide the ability to apply a function to each element in the set.
