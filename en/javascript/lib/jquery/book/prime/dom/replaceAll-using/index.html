Using the replaceAll method in jQuery

You can replace some elements with others using the equivalent methodreplaceAll.

Let's take the following paragraphs as an example:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Let's replace all the paragraphs with the classwwwon div with text'!!!':

$('<div>!!!</div>').replaceAll('.www');

The HTML code will look like this:

<div>!!!</div> <div>!!!</div> <div>!!!</div> <p>text</p>
