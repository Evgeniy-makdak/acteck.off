Wrapping elements in jQuery

Tags can be wrapped in other tags, for example, if we have paragraphs, then we can additionally wrap them in tagsdiv. For such things the method is usedwrap, which allows you to wrap each element found in a specified tag.

For example, let's say we have paragraphs with the classwww:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Let's wrap them with a tagdiv:

$('.www').wrap('<div>');

The HTML code will look like this:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <p>text</p>
