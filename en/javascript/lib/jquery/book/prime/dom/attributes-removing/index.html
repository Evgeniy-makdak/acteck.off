Удаление атрибутов в jQuery

Если вы хотите удалить какой-нибудь атрибут совсем, для этого следует использовать метод removeAttr. Давайте посмотрим, как он работает, на следующем примере. Пусть у нас есть инпут:

<input type="text" value="!!!" id="test">

Для удаления атрибута value мы прописываем его в переменной метода removeAttr.

$('#test').removeAttr('value');

Теперь в HTML коде атрибута value больше нет:

<input type="text" id="test">

Добавьте всем li класс www, если его нет, и удалите, если есть.

Удалите у li класс www.
