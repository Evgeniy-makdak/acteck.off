Element sizes in jQuery

There are several methods for working with element sizes in jQuery:width, innerWidth, outerWidth, height, innerHeight, outerHeight. It must be remembered that these methods receive a 'calculated value' (computed value).

They are convenient for mathematical calculations because they return a dimensionless quantity (for example400). The default units are pixels, but you can use others when setting values. It must be remembered that the resulting values ​​may be fractional.

The resulting values ​​may change if the element or its parent is hidden or the user resizes the page, which can lead to errors.
