jQuery innerWidth and innerHeight methods

Using methodsinnerWidthAndinnerHeightwe get the width and height of the elements, but taking into account not only the content, but also the internal padding -padding.

Let's get the paragraph width and height values#test, but with the helpinnerWidthAndinnerHeight, and display them in another paragraph. Let's say we have two paragraphs:

<p id="test">text</p> <p id="out"></p>

The CSS looks like this:

p { margin: 10px; padding: 5px; border: 2px solid blue; }

In Javascript we write the following code:

let w = $('#test').innerWidth(); let h = $('#test').innerHeight(); $('#out').text('Width: ' + w + ' Height: ' + h);

As you can see, the values ​​​​already differ by10px, since internal indents are taken into account, which in our case amount to5pxfrom all sides.

Also just using methodsinnerWidthAndinnerHeightwe can change the width and height values ​​of the elements. For example, let's set the width of the div#test1equal to250px, and the height#test2 - 200px:

$('#test1').innerWidth(250); $('#test2').innerHeight(200);
