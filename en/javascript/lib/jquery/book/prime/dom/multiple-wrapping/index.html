Wrapping multiple elements in jQuery

You can wrap not each element individually, but all found elements together using the methodwrapAll.

The HTML code shows a group of paragraphs:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Let's find all paragraphs with classwwwand wrap them in one tagdiv:

$('.www').wrapAll('<div></div>');

The HTML code will look like this:

<div> <p class="www">text</p> <p class="www">text</p> <p class="www">text</p> </div> <p>text</p>

Please note that if the wrapped elements are not next to each other, then the methodwrapAllwill first move the elements into one place and then wrap them. See the following example:

<p class="www">text1</p> <p class="www">text2</p> <p>text</p> <p class="www">text3</p>

Applying the methodwrapAll:

$('.www').wrapAll('<div></div>');

As a result, we get the following HTML code:

<div> <p class="www">text1</p> <p class="www">text2</p> <p class="www">text3</p> </div> <p>text</p>

Element wrapping methods also provide the ability to apply a function to each element in a set.
