Применение метода before в jQuery

Метод before позволяет добавлять текст в теги перед элементами.

Например, есть следующий абзац:

<p>text</p>

Теперь, давайте добавим теги с текстом:

$('p').before('<b>!!!</b>');

HTML код станет выглядеть так:

<b>!!!</b><p>text</p>

Перед каждым h2 вставьте '<p>!</p>'.
