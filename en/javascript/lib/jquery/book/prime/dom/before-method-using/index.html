Using the before method in jQuery

Methodbeforeallows you to add text to tags before elements.

For example, there is the following paragraph:

<p>text</p>

Now, let's add tags with text:

$('p').before('<b>!!!</b>');

The HTML code will look like this:

<b>!!!</b><p>text</p>
