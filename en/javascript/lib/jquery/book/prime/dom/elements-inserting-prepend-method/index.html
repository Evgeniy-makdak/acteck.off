prepend method for inserting elements in jQuery

For inserting elements in jQuery, it is provided4method:append, prepend, after, before.

Methodprependallows you to add text to the beginning of elements. For example, let's say we have a paragraph:

<p>text</p>

Applying the methodprepend, inside the tags indicating the text we need:

$('p').prepend('<b>!!!</b>');

The HTML code will look like this:

<p><b>!!!</b>text</p>

Inside everyoneh2insert at the beginning'!'.
