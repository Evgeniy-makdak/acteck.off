Removing only the value of an attribute in jQuery

It happens that we need to leave an attribute, but at the same time delete its current value. Let's look at the following example. Let us have an input:

<input type="text" value="!!!" id="test">

In this case, we register the attribute, but add an empty value in quotes (without a space) to it as the second parameter.

$('#test').attr('value', '');

The HTML code will look like this - attributevaluewill not disappear from the HTML code:

<input type="text" value id="test">

Change the solution to the previous problem so that only the value is removedwww, but not the attribute itself.
