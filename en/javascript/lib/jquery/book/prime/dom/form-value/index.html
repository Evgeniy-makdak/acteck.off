value of forms in jQuery

To work with an attributevalueinputs you can also use the methodval, which allows you to read and write new values.

Let's look at an example. Let's say we have the following HTML code:

<input type="text" id="test" value="!!!">

Let's write down the new value'www'to attributevalue:

$('#test').val('www');

The HTML code will look like this:

<input type="text" id="test" value="www">

Now let's display the current value on the screen:

alert($('#test').val());

Also for the methodvalthe function can be applied to each element in the set. The function takes the current number in the set as its first parameter, and its current value as its second parameter.

There is one more nuance - for the tagtextareayou can't get the internal contents through a methodhtml. To do this you need to use the methodval.

Consider the following HTML code:

<textarea id="test">!!!</textarea>

in Javascript code we write a methodval:

alert($('#test').val()); // выведет '!!!'

Most likely, this was done to ensure uniformity in working with forms.
