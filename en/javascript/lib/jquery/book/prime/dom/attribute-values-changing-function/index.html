Changing Attribute Values ​​Using a Function in jQuery

You can also apply a function to each attribute. In this example there are three inputs:

<input type="text" value="aaa"> <input type="text" value="bbb"> <input type="text" value="ccc">

Let's change the attribute of each inputvalue- add to the current onevalueserial number of the element in the set:

$('input').attr('value', function(index, value) { return value + ' ' + index; });

Thus, all attribute valuesvaluewill be replaced by those that the function returns to us. The HTML code will look like this:

<input type="text" value="aaa 0"> <input type="text" value="bbb 1"> <input type="text" value="ccc 2">

Change the valuevalueevery second input, adding to the currentvaluethe serial number of the element in the set.
