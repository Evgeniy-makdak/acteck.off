Strictly removing and adding a class using toggleClass method in jQuery

Also intoggleClassyou can pass the state parameter liketrueorfalse, then the class will either only be added or only removed.

Let's say we have the following CSS classes:

.red { color: red; } .zzz { font-style: italic; }

The class is applied to the paragraphzzz:

<p class="zzz" id="test">text</p> <button>click</button>

Let's just add the classredwhen pressed:

$('button').click(function() { $('#test').toggleClass('red', true); });

As you can see, when you press it again, no changes occur.

All three listed methods for working with classes -addClass, removeClass, toggleClasscan also take multiple classes as a parameter, and also allow a function to be applied to each element in the set.
