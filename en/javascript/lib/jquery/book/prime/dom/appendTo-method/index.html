appendTo method in jQuery

There are also alternative forms of methods for inserting elements. For example, instead ofappendcan be usedappendTo.

The difference is in the method of application: ifappendaccepts the text to be inserted as a parameter, thenappendToon the contrary, it takes as a parameter a selector of elements for which append should be applied.

The following entries are equivalent:

$('.www').append('<b>!!!</b>') $('<b>!!!</b>').appendTo('.www')
