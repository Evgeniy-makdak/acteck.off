Width and height methods in jQuery

MethodswidthAndheightget, respectively, the width and height of the element’s content without taking into accountpadding, borderAndmargin.

Let's get the paragraph width and height values#testand put them in another paragraph. Let's take the following HTML code as an example:

<p id="test">text</p> <p id="out"></p>

The CSS paragraph style looks like this:

p { margin: 10px; padding: 5px; }

Now we write the Javascript code:

let w = $('#test').width(); let h = $('#test').height(); $('#out').text('Width: ' + w + ' Height: ' + h);

We can also simply change the width and height values ​​of elements using these methods. For example, let's set the width of the div#test1equal to250px, and the height#test2 - 200px:

$('#test1').width(250); $('#test2').height(200);

We have this layout:

<div id="wrapper"> <div id="test">text</div> </div> <p id="one"></p> <p id="two"></p> <p id="three"></p> <p id="four"></p>

The following CSS styles:

div { margin: 10px; padding: 5px; border: 2px solid green; } #wrapper { border-color: blueviolet; }

Get and display div width and height values#wrapperwithout taking into account indents and borders. Place the values ​​in the first paragraph under this div.
