remove method in jQuery

There are two methods for removing elements in jQuery:removeAnddetach.

Methodremovedeletes items permanently.

Let's say we have the following HTML code:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Let's deletepwith classwww:

$('p').remove('.www');

The HTML code will look like this:

<p>text</p>

And you can do it like this:

$('p.www').remove();
