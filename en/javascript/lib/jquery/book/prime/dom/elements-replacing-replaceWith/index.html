Replacing elements with the replaceWith method in jQuery

You can replace some elements with others using the methodreplaceWith.

Let's look at an example, let's say we have the following paragraphs:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Let's replace all the paragraphs with the classwwwon'<div>!!!</div>':

$('.www').replaceWith('<div>!!!</div>');

The HTML code will look like this:

<div>!!!</div> <div>!!!</div> <div>!!!</div> <p>text</p>
