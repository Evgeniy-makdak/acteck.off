Creating and Wrapping Elements in jQuery

Before wrapping a tag, you can first create the element in the document. Let's wrap the paragraphs below in tags'<div></div>':

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

To do this, you can first create an element in the document usingdocument.createElement('div'):

$('.www').wrap(document.createElement('div'));

Or:

let div = document.createElement("div"); $('.www').wrap(div);

As a parameter, you can pass not only the name of the tag, but also the following construction:'<div></div>'- in this case the effect will be exactly the same:

$('.www').wrap('<div></div>');

The HTML code will look like this:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <p>text</p>

When using the second method, in the opening tag (in our case'<div>') you can write any attributes, and the wrapping will be along with these attributes.

Let's wrap our paragraphs in a div with a classzzz:

$('.www').wrap('<div class="zzz"></div>');

The HTML code will look like this:

<div class="zzz"><p class="www">text</p></div> <div class="zzz"><p class="www">text</p></div> <div class="zzz"><p class="www">text</p></div> <p>text</p>
