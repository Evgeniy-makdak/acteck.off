Изменение значений нескольких атрибутов в jQuery

Можно поменять значения сразу нескольких атрибутов, тогда их нужно передать в виде объекта. Например, пусть у нас есть следующий HTML код:

<input type="text" value="!!!" id="test">

В Javascript коде записываем атрибуты value, id и их значения в виде пары ключ-значение:

$('#test').attr({value: 'text', type: 'password'});

Теперь HTML код выглядит так:

<input type="password" value="text" id="test">
