slideToggle method in jQuery

Next method -slideToggleIt works like this: if an element is hidden, it shows it, and if it is shown, it hides it. The convenience of this method is that all this can be hung on one button, and not on two, as was the case in previous lessons.

Let's look at the following example. Let our HTML code look like this:

<button id="toggle">click me</button> <div id="elem">text...</div>

There is also CSS:

#elem { padding: 10px; width: 150px; height: 150px; border: 1px solid green; margin-top: 10px; }

Now let’s write the code so that when you click one button, the element is either hidden or shown:

$('#toggle').click(function() { $('#elem').slideToggle(1000); });
