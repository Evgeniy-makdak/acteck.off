Метод fadeTo в jQuery

Метод fadeTo позволяет уменьшить прозрачность opacity элемента до определенного значения (0 - полностью прозрачный, 1 - полностью непрозрачный). Первым параметром метод принимает до какого значения изменить прозрачность, а вторым - время выполнения.

Давайте сделаем так, чтобы по клику на кнопку наш элемент стал полупрозрачным за 1 секунду. Для начала возьмем следующий HTML код:

<button id="fade">click me</button> <div id="elem">text...</div>

Пусть также у нас есть CSS:

#elem { padding: 10px; width: 150px; height: 150px; border: 1px solid green; margin-top: 10px; }

Теперь прописываем следующий Javascript код:

$('#fade').click(function() { $('#elem').fadeTo(1000, 0.5); });
