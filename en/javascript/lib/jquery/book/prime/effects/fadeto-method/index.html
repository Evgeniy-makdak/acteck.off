fadeTo method in jQuery

MethodfadeToallows you to reduce transparencyopacityelement up to a certain value (0- completely transparent,1- completely opaque). The first parameter the method takes is the value to change the transparency to, and the second parameter is the execution time.

Let's make sure that when the button is clicked, our element becomes translucent for1second. Let's start with the following HTML code:

<button id="fade">click me</button> <div id="elem">text...</div>

Let's also say we have CSS:

#elem { padding: 10px; width: 150px; height: 150px; border: 1px solid green; margin-top: 10px; }

Now we write the following Javascript code:

$('#fade').click(function() { $('#elem').fadeTo(1000, 0.5); });
