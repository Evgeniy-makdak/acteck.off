slideUp and slideDown methods in jQuery

MethodsslideUpAndslideDownallow you to “roll up” and “roll out” the element back (they also take the effect execution time as a parameter; if you do not specify it, the default time will be taken,400мс). See the following example for how this looks.

Let's say we have two buttons anddiv:

<button id="hide">hide</button> <button id="show">show</button> <div id="elem">text...</div>

The CSS looks like this:

#elem { padding: 10px; width: 150px; height: 150px; border: 1px solid green; margin-top: 10px; }

Now let's write the methodsslideUpAndslideDown:

$('#hide').click(function() { $('#elem').slideUp(1000); }); $('#show').click(function() { $('#elem').slideDown(1000); });
