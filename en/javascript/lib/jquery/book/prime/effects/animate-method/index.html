jQuery animate method

All the methods that we discussed in previous lessons are somewhat limited - they do exactly what they are designed to do. If you need more control over the animation - use the universal methodanimate.

As the first parameter, this method takes an object in which to set the new values ​​of the CSS properties that you want to animate. And the second parameter is the animation execution time.

Let's implement this method using the following HTML code:

<button id="button">click me</button> <div id="elem">text...</div>

Let the CSS look like this:

#elem { width: 150px; height: 150px; padding: 10px; border: 1px solid green; }

Let's make sure that when the button is pressed, the element shrinks to the following dimensions: height -50px, width -100px:

$('#button').click(function() { $('#elem').animate({height: 50, width: 100}, 1000); });

If you specify a value in this formatheight: '+=50', then the animation will work like this: the current height value will be added50px(in our case) and the element will smoothly animate to the new value. Let's look at an example:

$('#button').click(function() { $('#elem').animate({height: '+=50', width: '+=50'}, 1000); });

For more information about the animate method, see the jQuery reference book.
