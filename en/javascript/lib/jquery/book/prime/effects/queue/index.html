Working with a queue in jQuery

To work with animation in jQuery, we may need such a useful thing as working with a queue. Queues allow you to carry out a sequence of actions with an element asynchronously, without interrupting program execution. Can be usedfx- a standard effects queue, which is done quite often, or you can use your own - custom queues.
