Метод has в jQuery

Если нам нужно отфильтровать набор по элементам, которые имеют определенных потомков, то мы можем использовать метод has:

Например, пусть у нас есть следующий HTML код:

<p>text</p> <p>text</p> <p><b>bold</b> text</p> <p><b>bold</b> text</p>

Давайте выберем только те абзацы, у которых внутри есть тег b и поставим им в конец текст '???':

$('p').has('b').append('???');

Найдите те li в наборе, у которых внутри есть тег i и покрасьте их в красный цвет.
