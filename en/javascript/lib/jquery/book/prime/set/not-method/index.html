jQuery not method

The method we will cover in this tutorial will benot. This is a method that does negation.

Let's get all paragraphs not with a classwww:

$('p').not('.www');

When working with this method, we first get all the paragraphs, and then among them we select paragraphs not with the classwww.

In the following example we have paragraphs:

<p class="www">text</p> <p class="www">text</p> <p>text</p> <p>text</p>

Let's find all the paragraphs and put the text at the beginning of them'!', then find paragraphs not with the classwwwand put the text at the end of them'?':

$('p').prepend('!').not('.www').append('?');

The HTML code will look like this:

<p class="www">!text</p> <p class="www">!text</p> <p>!text?</p> <p>!text?</p>
