eq method in jQuery

In order to select a specific number in the set, there is a methodeq. The number of an element in a set can be either positive or negative. In the second case, the countdown is from the end,-1will be the last element,-2- penultimate and so on:

Let's look at an example where the following paragraphs are written in the HTML code:

<p>text</p> <p>text</p> <p>text</p> <p>text</p>

Let's find all the paragraphs, and then the third one (numbered2, since the numbering starts from scratch), let’s color it red:

$('p').eq(2).css('color', 'red');
