Finding all parents in jQuery

Methodparentgets the element's immediate parent. In order to get all the parents of an element - not only the immediate one, but also the grandparent, great-grandparent, and so on up to the root element, use the methodparents.

In the next example, we will look at the following HTML code:

<div> <div class="www"> <div class="www"> <p id="test">text</p> </div> </div> </div>

Then we will find the element#testand all his parents with the helpparentsand if these are parentsdivwith classwww, then we will put the text at the beginning'!':

$('#test').parents('div.www').prepend('!');
