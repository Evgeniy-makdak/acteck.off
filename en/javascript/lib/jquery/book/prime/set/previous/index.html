Previous set in jQuery

Sometimes when building jQuery chains there is a need to return to a previous set. This is done using the methodend.

The following example shows the HTML code:

<p class="www">text</p> <p class="www">text</p> <p>text</p> <p>text</p>

Our task is to first find all paragraphs, then select among those found paragraphs with the classwwwusing the methodfilterand, using the methodappend, add text to them at the end'!'. Then we will return to the previous set (this is the set$('p')) by usingendand add text to the end'?'.

It turns out that paragraphs with a classwwwthere will be text'!?', and for regular paragraphs - text'?':

$('p').filter('.www').append('!').end().append('?');
