Using the contents method in jQuery

To return all children, including text, use the methodcontents.

Below is the HTML code:

<p id="test">text1 <b>bold</b> text2</p>

Now let's use the methodcontents:

$('#test').contents().each( function() { console.log($(this).text()); // выведет 'text1', 'bold', 'text2' } );
