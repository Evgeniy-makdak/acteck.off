Adding elements using addBack in jQuery

There is also a similar methodaddBack, which adds the previous one to the current set.

Consider the following example. Let the HTML code look like this:

<div> <p>text</p> <p id="test">text</p> <p>text</p> </div>

Let's find a paragraph#test, then find the parent of this paragraph usingparent, add the found paragraph to its parent usingaddBackand put the text at the beginning'!'using the methodprepend. It turns out that both the paragraph and its parent will have text at the beginning'!':

$('p').parent().addBack().prepend('!');
