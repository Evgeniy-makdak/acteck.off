Добавление элементов с помощью addBack в jQuery

Существует также похожий метод addBack, который добавляет в текущий набор предыдущий.

Рассмотрим следующий пример. Пусть HTML код выглядит следующим образом:

<div> <p>text</p> <p id="test">text</p> <p>text</p> </div>

Давайте найдем абзац #test, затем найдем родителя этого абзаца с помощью parent, добавим найденный абзац к его родителю с помощью addBack и поставим им в начало текст '!' с помощью метода prepend. Получится, что и абзац, и его родитель в начале будут иметь текст '!':

$('p').parent().addBack().prepend('!');

Найдите родителя li с классом www (класс у элемента li), добавьте к родителю этот li с помощью addBack и покрасьте их в красный цвет.
