jQuery parent method

We have already discussed the method a littleparentwhen discussing pseudo-classparent. I remind you that the methodparentand selectorparentdo different things, the first one finds the parent of the element, and the second one finds all non-empty elements (which are the parents).

In the following example we will find the element#test, then find its parent usingparentand give him the text'!'. Let's say we have the following HTML code:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p>

And now we write the methodparentin Javascript code:

$('#test').parent().html('!');

Don't forget that you can pass a selector to this method for additional filtering of parents.
