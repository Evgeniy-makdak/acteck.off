jQuery prev and next methods

Using the methodprevyou can contact the neighbor above for a certain element, and usingnext- to the neighbor below (note that elements are considered neighbors only within one parent).

For example, take the following paragraphs:

<p>text</p> <p>text</p> <p id="test">text</p> <p>text</p> <p>text</p>

Now let's find the element#testand we’ll put the text on his upstairs neighbor'!':

$('#test').prev().html('!');

Now let’s perform a similar operation with the neighbor below:

$('#test').next().html('!');
