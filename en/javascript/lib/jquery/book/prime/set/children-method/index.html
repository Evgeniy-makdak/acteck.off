The children method for working with children in jQuery

To work with descendants of an element, we first look at the methodchildren, which can be used to get only the immediate descendants of an element.

Consider the following HTML code:

<p id="test">text1 <b>bold</b> text2</p>

To obtain descendants, we will write the following function in the method:

$('#test').children().each( function() { console.log($(this).text()); // выведет 'bold' } );

We have divs with tags:

<div id="text"><i>text1</i> <b>text2</b> text3 text4</div>

Print all immediate children of this div to the console using the methodchildren.
