Метод children для работы с потомками в jQuery

Для работы с потомками элемента, мы вначале рассмотрим метод children, с помощью которого можно получить только непосредственных потомков элемента.

Рассмотрим следующий HTML код:

<p id="test">text1 <b>bold</b> text2</p>

Для получения потомков пропишем в методе следующую функцию:

$('#test').children().each( function() { console.log($(this).text()); // выведет 'bold' } );

У нас есть див с тегами:

<div id="text"><i>text1</i> <b>text2</b> text3 text4</div>

Выведите в консоль всех непосредственных потомков этого дива с помощью метода children.
