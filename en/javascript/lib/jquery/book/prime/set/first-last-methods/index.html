jQuery first and last methods

There are also separate methodsfirstAndlastto select the first and last element of the set.

Let's look at how the above methods work using the following paragraphs as an example:

<p>text</p> <p>text</p> <p>text</p>

Let's find all the paragraphs, and then select the first one among the found ones and put the text to it'!':

$('p').first().text('!');

Now, let's select the last one among those found and give it the text'!':

$('p').last().text('!');
