Методы first и last в jQuery

Есть также отдельные методы first и last, чтобы выбирать первый и последний элемент набора.

Рассмотрим работу вышеперечисленных методов на примере следующих абзацев:

<p>text</p> <p>text</p> <p>text</p>

Давайте найдем все абзацы, а затем выберем среди найденных первый и поставим ему текст '!':

$('p').first().text('!');

А теперь, давайте выберем среди найденных последний и поставим ему текст '!':

$('p').last().text('!');
