Работа с данными в элементах в jQuery

В DOM элементы можно записывать и считывать из них данные при помощи специальных методов. Речь пойдёт не о data-аттрибутах, хотя эти методы позволяют работать и с ними. В основном это нужно для работы с внутренним хранилищем jQuery, чтобы размещать и получать из него некоторую информацию, связанную с определённым DOM-элементом текущей страницы.

data removeData hasData
