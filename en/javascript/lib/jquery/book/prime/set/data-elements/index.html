Working with Element Data in jQuery

In DOM elements, data can be written to and read from them using special methods. We will not talk about data attributes, although these methods allow you to work with them. This is mainly needed to work with jQuery's internal storage to place and retrieve from it some information associated with a specific DOM element of the current page.
