Cloning elements in jQuery

Methodclonecreates deep copies of selected page elements and returns them as a jQuery object.

Let's look at how this method works using a paragraph as an example:

<p class="www">text</p>

After the paragraph, insert another one of the same using the methodinsertAfter:

$('.www').clone().insertAfter('.www');
