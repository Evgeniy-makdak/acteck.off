jQuery siblings method

Using methodsiblingsyou can find all the neighbors of an element. Let's look at the HTML code:

<p>text</p> <p>text</p> <p id="test">text</p> <p>text</p> <p>text</p>

Then we apply the methodsiblings:

$('#test').siblings().html('!');

By passing a selector to the method, you can perform additional filtering of neighbors, for example, like this:

$(#test).siblings('.zzz').html('!');
