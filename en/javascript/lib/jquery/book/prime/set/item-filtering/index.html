Filtering elements in jQuery

Using the methodfilterYou can keep in the set only those elements that satisfy a certain selector.

Let's look at the HTML code:

<p>text</p> <p>text</p> <p class="www">text</p> <p class="www">text</p>

Let's find all the paragraphs and put the text at the beginning of them'!'by usingprepend, then from the found ones we get only paragraphs with the classwwwusing the methodfilterand put the text at the end of them'?'by usingappend:

$('p').prepend('!').filter('.www').append('?');

Find everythingliin the set, then usingfilterselect allliwith classwwwand put them at the end'!'.
