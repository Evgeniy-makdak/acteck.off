Converting to an Array in jQuery

MethodtoArrayconverts a set of jQuery elements to a regular JavaScript array. This is necessary so that JavaScript methods and functions can be applied to this array, for example,sortthis array orturn over.

For example, consider the following HTML code:

<p>text1</p> <p>text2</p> <p>text3</p> <div>text4</div>

Let's get all elements with tagpas an array using the methodtoArray, let's flip it usingreverse. Then we will output the text contents of the elements as a string using the JavaScript methodjoin.

To extract the text of the elements and output the resulting array as a string, we will write a functionprint, whose parameter will be an inverted array:

function print(elems) { let arr = []; for (let i = 0; i < elems.length; i++) { arr.push(elems[i].innerHTML); } alert(arr.join(' ')); } print($('p').toArray().reverse());
