jQuery length property

To find out the number of elements in a set, use the propertylength.

For example, we have the following HTML code:

<p>text</p> <p>text</p> <div>text</div> <p>text</p> <p>text</p> <div>text</div>

Let's find out the number of paragraphs and display it in the console:

let num = $('p').length; console.log(num);
