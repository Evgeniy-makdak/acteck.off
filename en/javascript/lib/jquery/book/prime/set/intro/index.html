jQuery Typing Basics

Now we'll start learning jQuery methods for working with a set of elements. They are very convenient to use when building chains of methods.

First, let's get a set of elements corresponding to a jQuery object. This can be done using the methodget, this way we can get all the paragraphs:

console.log($('p').get());

You can also get a specific paragraph, then you need to specify a specific number of the element in the set. The number can also have a negative value (then the countdown is carried out in the opposite direction from the last element). Let's get the second paragraph using the methodget, counting starts from0and print it to the console:

<p>text</p> <p>test</p> <p>text</p> <p>text</p> console.log($('p').get(1));
