Основы работы с набором в jQuery

Сейчас мы начнём изучать методы jQuery для работы с набором элементов. Их очень удобно использовать при выстраивании цепочек методов.

Для начала давайте получим набор элементов, соответствующих объекту jQuery. Это можно сделать при помощи метода get, таким образом мы можем получить все абзацы:

console.log($('p').get());

Можно получить и какой-то определённый абзац, тогда нужно указать определенный номер элемента в наборе. Номер может иметь и отрицательное значение (тогда отсчет ведется в обратную сторону с последнего элемента). Давайте получим второй абзац, используя метод get, отсчет ведется с 0 и выведем его в консоль:

<p>text</p> <p>test</p> <p>text</p> <p>text</p> console.log($('p').get(1));

Получите 5-й li и выведите его в консоль.
