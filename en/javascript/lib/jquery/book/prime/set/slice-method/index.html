Метод slice в jQuery

С помощью метода slice можно можно обратиться к части элементов набора, например, со второго по пятый.

Давайте разберем работу метода на следующем примере - пусть у нас есть следующий HTML код:

<p>text0</p> <p>text1</p> <p>text2</p> <p>text3</p> <p>text4</p> <p>text5</p>

Теперь абзацам с 2-го по 4-тый (включительно) поставим в конец текст '!' (нумерация с нуля, второй абзац будет иметь номер один):

$('p').slice(1, 5).append('!');

HTML код станет выглядеть так:

<p>text0</p> <p>text1!</p> <p>text2!</p> <p>text3!</p> <p>text4!</p> <p>text5</p>

Обратите внимание на то, что slice(1, 5) включает первый аргумент, а второй нет, то есть (1, 5) находит элементы с номером 1, 2, 3, 4, а элементы номер 0 и 5 не берет.
