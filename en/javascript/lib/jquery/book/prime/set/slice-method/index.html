slice method in jQuery

Using the methodsliceyou can refer to part of the elements of the set, for example, from the second to the fifth.

Let's look at how the method works using the following example - let's say we have the following HTML code:

<p>text0</p> <p>text1</p> <p>text2</p> <p>text3</p> <p>text4</p> <p>text5</p>

Now paragraphs with2th4-th (inclusive) put at the end of the text'!'(numbering from zero, the second paragraph will be numbered one):

$('p').slice(1, 5).append('!');

The HTML code will look like this:

<p>text0</p> <p>text1!</p> <p>text2!</p> <p>text3!</p> <p>text4!</p> <p>text5</p>

Note that slice(1, 5) includes the first argument but not the second, i.e.(1, 5)finds elements with number1, 2, 3, 4, and the elements number0And5doesn't take it.
