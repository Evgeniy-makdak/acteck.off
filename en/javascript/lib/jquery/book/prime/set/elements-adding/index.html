Adding elements to the current set in jQuery

Sometimes there are situations when we already have a set of methods, we have performed some operations on it, and now we would like to add more elements to this set and perform some more operations on this common set. This is done using the methodadd.

Take for example the following HTML code:

<div>ddd</div> <h1>hhh</h1> <p>ppp</p> <div id="test"><h2>hhh</h2></div> <p>ppp</p> <h2>hhh</h2> <p>ppp</p>

Let's find all the paragraphs and add text to the end of them'!', then add headings to the found paragraphsh2, and only those that lie inside the element#test, and at the same time set the selected headings and paragraphs to red:

$('p').append('!').add('#test h2').css('color', 'red');
