slideDown method

MethodslideDownsmoothly reveals hidden elements, rolling them out from top to bottom. You can hide elements using the methodslideUp.

Syntax

Display for a specified time,400мсdefault:

.slideDown(длительность);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.slideDown('slow' или 'fast');

If you do not specify parameters, there will be no animation, the elements will be shown instantly:

.slideDown();

You can also pass the smoothness function as the second parameter, as well as a callback function as the third - it will work after the animation is completed. Both parameters are optional:

.slideDown(длительность, [функция плавности], [callback-функция]);

You can pass various options to the method, in the form of a JavaScript object containing pairskey: value:

.slideDown(options);

Such an object can pass the following parameters and functions -duration, easing, queue, specialEasing, step, progress, complete, start, done, fail, always. You can see the description of these parameters for the methodanimate. For example, let's set the duration and smoothness function:

.slideDown( {duration: 600, easing: easeInSine} );

Example

Let's smoothly show the hidden paragraph after pressing the button using the methodslideDown. By passing the keywordslow, we will set the speed to600мс. After the animation is finished, we will output it to the console'Animation complete':

<button>show text</button> <p style="display: none">text</p> $('button').click(function() { $('p').slideDown('slow', function() { console.log('Animation complete') }); });
