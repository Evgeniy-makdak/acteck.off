innerHeight method

MethodinnerHeightallows you to get and change the height of an element taking into account its internal padding. It's important to remember that we will be getting the 'calculated value' of the height (computed height). Calculation errors can also occur if the user resizes the page, or if an element or its parent is hidden. The height value does not take into account the thickness of the element's border.

Syntax

Get the height of an element. In some cases, the resulting values ​​may be fractional:

$(селектор).innerHeight();

To change the height of an element, you can simply pass a number (for example400), then the units of measurement will be pixels, or a string indicating the units of measurement (for example'10em'):

$(селектор).innerHeight(новое значение);

We can also apply a given function to each element in the set. In this case, the first parameter the function will receive is the number of the element in the set, and the second parameter will be the current value of the specified height for a specific element.thisinside the function will point to the current element. The element's height value will change to the one returned by the function:

$(селектор).innerHeight(function(номер в наборе, текущее значение высоты));

Example

Let's display paragraph height information for comparison.#test, obtained by methodsheightAndinnerHeight:

<p id="test">text</p> <p id="out1"></p> <p id="out2"></p> p { margin: 10px; padding: 5px; border: 2px solid blue; } let h1 = $('#test').height(); let h2 = $('#test').innerHeight(); $('#out1').text(h1); $('#out2').text(h2);

We will see the difference in10px, which is the sum of our top and bottom padding.
