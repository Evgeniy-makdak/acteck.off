offset method

Methodoffsetallows you to get the position of an element relative to the document, unlike the methodposition, which receives coordinates relative to the parent's padding. Methodoffsetmore useful when positioning a new element on top of an existing one for global manipulations, such as implementing drag and drop (drag-and-drop). As a result, the method returns an object containing propertiestopAndleft. Calculation errors may occur if the user resizes the page. Also, the method does not receive the coordinates of hidden elements.

Syntax

Get current coordinates. In some cases, the resulting values ​​may be fractional:

$(селектор).offset();

To change the coordinates of an element, you need to pass an object containing propertiestopAndleft:

$(селектор).offset({top: 40, left: 40});

We can also apply a given function to each element in the set. In this case, the function will receive the number of the element in the set as the first parameter, and the object with coordinates as the second parametertopAndleft. thisinside the function will point to the current element. The element's coordinate values ​​will change to those returned by the function:

$(селектор).width(function(номер в наборе, {текущие координаты}));

Example

Let's get the position of the div (green square) using the methodoffset, and then, referring to the keys of the resulting object, we display this information:

<div id="result">click a square ...</div> <div id="test"></div> #test { position: absolute; top: 40px; left: 40px; width: 100px; height: 100px; background: green; color: white; margin-top: 10px; cursor: pointer; } $('*', document.body).click(function(event) { let offset = $(this).offset(); event.stopPropagation(); $('#result').text('Left: ' + offset.left + ', Top: ' + offset.top); });

Example

Let's change the coordinates of the second paragraph:

<p style="margin-left: 10px;">text1</p> <p style="margin-left: 10px;">text2</p> $('p').last().offset({top: 40, left: 60});
