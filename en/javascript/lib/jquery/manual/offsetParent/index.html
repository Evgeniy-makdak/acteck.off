offsetParent method

MethodoffsetParentallows you to get the closest positioned ancestor of an element. Under the concept'positioned'implies an element with a CSS propertyposition, set to valuerelative, absolute, orfixed. The method is useful in animation and when placing elements on the page.

Syntax

Get the closest positioned ancestor:

$(селектор).offsetParent();

Example

Let's find the closest positioned (relatively in our case) ancestor of the paragraph with#testand color its background green:

<div> <p>aaa</p> <div style="position: relative;"> <p>bbb</p> <p>bbb</p> <p>bbb</p> <div> <p id="test">ccc</p> <div> <p>ddd</p> <p>ddd</p> <p>ddd</p> </div> <p>ccc</p> <p>ccc</p> </div> </div> <p>aaa</p> <p>aaa</p> </div> $('#test').offsetParent().css('background-color', 'green');
