Methodsubmitallows us to work with JavaScript eventsubmit. It allows us to bind a handler to a JavaScript eventsubmitor fire this event on the element.

Syntax

We can pass a handler function to the method, which will be triggered every time an event occurs, to which an event object can in turn be passed:

.submit(функция-обработчик);

In this case, the method will work similarly to the design.on('submit', handler)methodon.

We can also use an optional parameter to pass the method an object with data that will be passed to the handler function:

.submit([данные события], функция-обработчик);

Or you can not pass any parameters, then the method will work similarly.trigger('submit')methodtrigger:

.submit();
