addClass method

MethodaddClassadds the given CSS class to an element.

Syntax

One CSS class:

$(селектор).addClass(имя CSS класса);

Several CSS classes (by line):

$(селектор).addClass('класс1 класс2 класс3 ...');

Several CSS classes (array):

$(селектор).addClass(['класс1', 'класс2', 'класс3', '...']);

Applying a function to each element in a set:

$(селектор).addClass(function(номер в наборе, текущие CSS классы элемента));

Example

Our paragraph has no classes at all. Let's add a class to itwww:

<p id="test">text</p> $('#test').addClass('www');

The HTML code will look like this:

<p id="test" class="www">text</p>

Example

Our paragraph already has a class. Let's add some class to it toowww:

<p id="test" class="eee">text</p> $('#test').addClass('www');

The HTML code will look like this:

<p id="test" class="eee www">text</p>

Example

Our paragraph already has two classes. Let's add some class to it toowww:

<p id="test" class="eee ggg">text</p> $('#test').addClass('www');

The HTML code will look like this:

<p id="test" class="eee ggg www">text</p>

Example

Our paragraph already has a class. Let's add two more classes to it -wwwAndggg:

<p id="test" class="eee">text</p> $('#test').addClass('www ggg');

The HTML code will look like this:

<p id="test" class="eee www ggg">text</p>
