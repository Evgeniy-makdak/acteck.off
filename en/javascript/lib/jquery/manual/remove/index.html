Метод remove

Метод remove полностью удаляет выбранные элементы из DOM. Чтобы удалить элементы без удаления jQuery данных и событий, связанных с элементами, используйте метод detach.

Синтаксис

Так мы удаляем элементы. Additional мы можем отфильтровать выбранные элементы, передав методу необязательный параметр selector:

.remove([селектор]);

Пример

Давайте удалим все абзацы с классом www:

<p>text</p> <p>text</p> <p class="www">text</p> <p class="www">text</p> $('p.www').remove();

HTML код станет выглядеть так:

<p>text</p> <p>text</p>

Смотрите также

  • метод detach,
    который удаляет элементы с возможностью их восстановления
  • метод empty,
    который удаляет содержимое из выбранных элементов
  • JavaScript метод remove,
    который позволяет удалить элемент
