remove method

Methodremovecompletely removes the selected elements from the DOM. To remove elements without removing jQuery data and events associated with the elements, use the methoddetach.

Syntax

This is how we remove elements. Additionally, we can filter the selected elements by passing an optional parameter to the methodselector:

.remove([селектор]);

Example

Let's delete all paragraphs with classwww:

<p>text</p> <p>text</p> <p class="www">text</p> <p class="www">text</p> $('p.www').remove();

The HTML code will look like this:

<p>text</p> <p>text</p>

See also

    which deletes items with the ability to restore them
  • methodempty,

  • JavaScript methodremove,
