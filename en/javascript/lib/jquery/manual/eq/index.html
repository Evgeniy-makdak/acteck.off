eq method

Methodeqselects the nth element among those found, that is, an element with a certain number, for example, the third or fifth. The first element found will have the number0. The number of an element in a set can be either positive or negative. In the second case, the countdown is from the end,-1will be the last element,-2- penultimate and so on.

Syntax

$(селектор).eq(номер);

Example

Let's find all the paragraphs, and then the third one (numbered2, since the numbering starts from scratch), let’s color it red:

<p>text</p> <p>text</p> <p>text</p> <p>text</p> $('p').eq(2).css('color', 'red');

The HTML code will look like this:

<div class="result"> <p>text</p> <p>text</p> <p style="color: red;">text</p> <p>text</p> </div>
