prependTo method

MethodprependToadds text to the beginning of the element. There is also a methodprepend, which works in a similar way.

Syntax

Inserting text at the beginning of an element:

$(текст).prependTo(селектор);

The content can be not only plain text, but also a DOM element, selector, array of elements, string or jQuery object.

Example

Let's insert tagged text at the beginning of a given paragraph:

<p id="test">абзац</p> $('<b>!!!</b>').prependTo('#test');

Result of executing the code:

<p id="test"><b>!!!</b>абзац</p>

See also

  • methodprepend,
    which adds text to the beginning of the element
  • methodsappendTo, insertBefore, insertAfter,
    allowing you to add content to a specific location
  • pseudo elementbefore,
    which adds text to the end of an element in CSS
