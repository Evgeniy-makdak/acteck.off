html method

Methodhtmlallows you to change the text of an element and receive it along with tags.

Syntax

Receiving text:

$(селектор).html();

Changing the text:

$(селектор).html(новый текст);

Additionally

Methodhtmlcan apply a given function to each element in a set. In this case, the function will receive the number of the element in the set as the first parameter, and the current text of the element as the second parameter:

$(селектор).html(function(номер в наборе, текущий текст элемента));

Variable names in a function can be anything. For example, if for the first parameter we give the nameindex- then the variable will be available inside our functionindex, which will contain the number in the set for the element that the function is processing at a given time. Similarly, if for the second parameter you give, for example, a namevalue- then the variable will be available inside our functionvalue, which will contain the text of the element that the function is processing at a given time:

$(селектор).html(function(index, value) { // тут доступны переменные index и value });

The text of each element will change to the one that the function will return specifically for this element.

Example

Let's display the contents of our paragraph:

<p id="test">text</p> let text = $('#test').html(); alert(text);

Example

Let's change the content of our paragraph:

<p id="test">text1</p> $('#test').html('text2');

The HTML code will look like this:

<p id="test">text2</p>

Example

Let's change the content of our paragraph to text with tags:

<p id="test">text1</p> $('#test').html('<span>text2</span>');

The HTML code will look like this:

<p id="test"><span>text2</span></p>

Example

Let's add to each paragraph at the end its serial number in the set:

<p>text</p> <p>text</p> <p>text</p> $('p').html(function(index, value){ return value + ' ' + index; });

The HTML code will look like this:

<p>text 1</p> <p>text 2</p> <p>text 3</p>
