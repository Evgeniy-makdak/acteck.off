prev method

Methodprevreturns the element that comes before the given one within one parent.

Syntax

Getting immediate neighbor:

.prev();

You can also select an immediate neighbor only if it satisfies the given selector:

.prev(селектор);

Example

Let's find the element#testand we’ll put the text on his upstairs neighbor'!'using the methodhtml:

<p>text</p> <p>text</p> <p id="test">text</p> <p>text</p> <p>text</p> $('#test').prev().html('!');

The HTML code will look like this:

<p>text</p> <p>!</p> <p id="test">text</p> <p>text</p> <p>text</p>
