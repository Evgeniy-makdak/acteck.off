event object

Objecteventgenerated when an event is triggered and contains information about it. When an event occurs, the objecteventpassed to the handler function.

Properties and methods of the event object

The table below shows the properties and methods of the objecteventand their description:

Name Description
event.currentTarget The current DOM element at the event bubbling stage. Typically this property is equivalentthisfunctions.
event.data Optional parameter. The data object passed to the event method when attaching an executable handler.
event.delegateTarget The element to which the jQuery event handler that was just called was attached. This property is most useful in delegated events, where the handler is bound to an ancestor of the element being processed. For undelegated handlers bound directly to an element, the valueevent.delegateTargetcorresponds to the valueevent.currentTarget.
event.isDefaultPrevented The method checks whether the method has been calledevent.preventDefaultfor this event object.
event.isImmediatePropagationStopped The method checks whether the method has been calledevent.stopImmediatePropagationfor this event object.
event.isPropagationStopped The method checks whether the method has been calledevent.stopPropagationfor this event object.
event.isPropagationStopped The method checks whether the method has been calledevent.stopPropagationfor this event object.
event.metaKey The method checks whether a key has been pressedMETAat the time the event is triggered. The key may differ depending on the platform. Returnstrueorfalse.
event.namespace The namespace specified at the time the event was raised. This property will be useful for plugin authors whose tasks depend on the namespace used.
event.pageX Shows the mouse position relative to the left edge of the document.
event.pageY Shows the mouse position relative to the top edge of the document.
event.preventDefault If this method is called, the default action for this event will not be performed. For example, clicking on a link will not take you to a new URL. To check whether this method was called, you can use the methodevent.isDefaultPrevented.
event.relatedTarget Returns the other DOM element involved in the event, if there is one. Formouseoutshows which element the mouse is hovering over, formouseoverfrom which element the mouse cursor was moved away.
event.result The last value returned by the triggered event handler is not equal toundefined. The property can be useful for getting the values ​​of your own events.
event.stopImmediatePropagation Cancels all remaining event handlers associated with the element and prevents the event from bubbling up the DOM tree. To simply prevent an event from bubbling to ancestor elements, but allow other event handlers to work, you can use the methodevent.stopPropagation. Use methodevent.isImmediatePropagationStoppedto check whether it was calledevent.stopImmediatePropagationfor a given event object.
event.stopPropagation Prevents an event from bubbling up the DOM tree. Remember that other handlers will continue to work for this element. This method works for native events raised using the methodtrigger. To check if this method has been called, useevent.isPropagationStopped.
event.target The DOM element that triggered the event. This can be an element registered for the event or its descendant. Very useful to compareevent.targetAndthis, to determine the event bubbling. The property is useful when delegating events when events bubble up.
event.timeStamp The time difference in milliseconds between the time the browser generated the event and January 1, 1970. The property can be useful in determining the performance of an event by obtaining the difference in valuesevent.timeStampfor two points in time in the code. If you just want to get the current time inside an event handler, use the methodgetTime.
event.type This property specifies the event type.
event.which This property indicates which keyboard or mouse key was pressed. For mouse:1- left button,2- wheel,3- right button. Useevent.whichinstead ofevent.button.

Other properties of the event object

There are other properties that are copied to the objectevent:

altKey, button, buttons, cancelable, char, charCode, clientX, clientY, ctrlKey, detail, eventPhase, key, keyCode, offsetX, offsetY, originalTarget, screenX, screenY, shiftKey, toElement, view.

To access properties not listed above, you can use the objectevent.originalEvent.

Example

Let's display in a div which keys were pressed:

<input id="test" value="type something"> <div id="text"></div> $('#test').on('keydown', function(event) { $('#text').html(event.type + ": " + event.which); });
