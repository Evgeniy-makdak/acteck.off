toggle method

Methodtogglealternates smooth showing/hiding of elements. If an element is shown, it will be hidden and vice versa.

Syntax

Display for a specified time in milliseconds,400мсdefault:

.toggle(длительность);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.toggle('slow' или 'fast');

You can use the parameterdisplaypass valuetrueorfalse. Iftrue, then the element will only be shown,false- hide:

.toggle(display);

If you do not specify parameters, there will be no animation, elements will be shown/hidden instantly:

.toggle();

You can also pass the smoothness function as the second parameter, as well as a callback function as the third - it will work after the animation is completed. Both parameters are optional:

.toggle(длительность, [функция плавности], [callback-функция]);

You can simply pass a callback function as the second optional parameter - it will work after the animation is completed:

.toggle(длительность, [callback-функция]);

You can pass various options to the method, in the form of a JavaScript object containing pairskey: value:

.toggle(options);

Such an object can pass the following parameters and functions -duration, easing, queue, specialEasing, step, progress, complete, start, done, fail, always. You can see the description of these parameters for the methodanimate. For example, let's set the duration and smoothness function:

.toggle( {duration: 800, easing: easeInSine} );

Example

Let's smoothly show the hidden paragraph after pressing the button using the methodtoggle. By passing the keywordslow, we will set the speed to600мс. Click again to hide:

<button>show text</button> <p style="display: none">text</p> $('button').click(function() { $('p').toggle('slow'); });
