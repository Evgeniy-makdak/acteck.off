toggleClass method

MethodtoggleClassalternates adding/removing a CSS class: if it is not there, it will be added, if it is there, it will be removed.

Syntax

One CSS class:

$(селектор).toggleClass(имя класса);

Just add or remove a CSS class depending on the second parameter, which takes valuestrueorfalse:

$(селектор).toggleClass(имя CSS класса, состояние);

Several CSS classes:

$(селектор).toggleClass(['класс1', 'класс2', 'класс3', '...']);

Only add or remove CSS classes depending on the second parameter:

$(селектор).toggleClass(['класс1', 'класс2', 'класс3', '...'], состояние');

Applying a function to each element in a set, the state can betrueorfalse, and tells the method whether to only add or only remove the CSS class:

$(селектор).toggleClass(function(номер в наборе, текущие классы элемента, состояние), [состояние]);

Example

Our paragraph already has two classes. Let's add some class to it toowww:

<p id="test" class="eee ggg">text</p> $('#test').toggleClass('www');

The HTML code will look like this:

<p id="test" class="eee ggg www">text</p>

Example

And now classwwwis already in the element - let's remove it:

<p id="test" class="eee ggg www">text</p> $('#test').toggleClass('www');

The HTML code will look like this:

<p id="test" class="eee ggg">text</p>
