off method

Methodoffallows you to unbind an event handler from an element. To bind a handler, you can use the methodoff.

Syntax

So we remove the event handler from the element, one or more events separated by spaces are passed as the first parameter as a string, and as the second we pass an additional filtering selector of descendants inside the element (it must match what we passed to the methodon, when the handler was attached). To remove all delegated events, pass the value'**'. The second parameter is optional. To the third we pass the handler function (which we attached), to which the event object is passed, or we passfalse:

$(селектор).off(события, [селектор], функция-обработчик(объект события));

You can use the methodoffdifferently, then the first parameter is passed a JavaScript object, where the keys are the event type, and the values ​​are the handler functions that we added to the element:

$(селектор).off({'тип события': handler}, [селектор]);

We can simply pass one parameter as an objectjQuery.Event:

$(селектор).off(событие);

If we do not pass parameters to the method, we will unbind all attached handlers from the element:

$(селектор).off();

Example

Let's add a handler function to the paragraphstestFunc. But let’s immediately remove the handler from the second paragraph, i.e. clicking on the second paragraph will not lead to anything (if we comment out the last line of code, we will see how the event is triggered when clicking on the second paragraph):

<p id="test1">click1</p> <p id="test2">click2</p> function testFunc(event) { alert(event.data.text); } $('#test1').on('click', {text: 'aaa'}, testFunc); // добавляем обработчик $('#test2').on('click', {text: 'bbb'}, testFunc); // добавляем обработчик $('#test2').off('click', testFunc); // удаляем обработчик
