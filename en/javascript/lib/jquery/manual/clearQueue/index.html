clearQueue method

MethodclearQueueallows us to remove from the queue all functions that have not been run.

Syntax

This way we can clear the function queue. You can pass the queue name as a string as an optional parameter (by defaultfx- standard effects queue). If you do not pass any parameters, then from the queuefxall remaining functions will be removed:

.clearQueue([имя очереди]);

This method is similar to the methodstop. But, if the latter can only work with animation, thenclearQueueallows you to work with any jQuery queue that was added using thequeue.

Example

Let's press the button#start, we will run the animation. When you press the button#stopwe will stop the animation and clear the queue using the methodclearQueue. When you press again#startthe animation will start again:

<button id="start">start</button> <button id="stop">stop</button> <div></div> div { position: absolute; margin: 3px; width: 50px; height: 50px; left: 0px; top: 30px; background: red; display: none; } div.newcolor { background: green; } $('#start').click(function() { let myDiv = $('div'); myDiv.show('slow'); myDiv.animate({ left: '+=200' }, 5000); myDiv.queue(function() { let that = $(this); that.addClass('newcolor'); that.dequeue(); }); myDiv.animate({ left: '-=200' }, 1500); myDiv.queue(function() { let that = $(this); that.removeClass('newcolor'); that.dequeue(); }); myDiv.slideUp(); }); $('#stop').click(function() { let myDiv = $('div'); myDiv.clearQueue(); myDiv.stop(); });
