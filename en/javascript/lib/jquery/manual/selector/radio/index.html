radio selector

Selector:radioselects radio switches. See tagradio. Equivalent$(':radio')is$('[type=radio]'). Because:radiodoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='radio'].

Syntax

This is how we select radio buttons:

$(':radio');

As with other pseudo-class selectors (starting with':'), before':'it is better to put the name of the tag or other selector, otherwise the selector will be applied'*', that is$(':radio')will be perceived as$('*:radio'), so it's better to use instead$('input:radio'). To select associated radio switches you can use$('input[name=gender]:radio').

Example

Let's select the radio switches and wrap them in span. Then we will create a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="text"> </form> <+javascript+> $('form input:radio') .wrap('') .parent() .css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // prevents form submission });<-javascript->
