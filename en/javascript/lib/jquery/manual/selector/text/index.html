Text selector

Selector:textselects inputs with typetext. See taginput. Because:textdoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='text'].

Syntax

This is how we select inputs with typetext:

$(':text');

As with other pseudo-class selectors (starting with':'), before':'it is better to put the name of the tag or other selector, otherwise the selector will be applied'*', that is$(':text')will be perceived as$('*:text'), so it's better to use instead$('input:text'). It must be remembered that:textwill also select all inputs that do not have an attribute specifiedtype, you can see the differences in behavior in the following example:

$('<input>').is('[type=text]'); // вернет false $('<input>').is(':text'); // вернет true

Example

Let's select all inputs with typetextand give it a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <input type="checkbox"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> $('form input:text').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // предотвращает отправку формы });
