file selector

Selector:fileselects elements with typefile, which are file upload fields. Equivalent$(':file')is$('[type=file]'). Because:filedoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='file'].

Syntax

This is how we select elements with typefile:

$(':file');

As with other pseudo-class selectors (starting with':'), before':'it is better to put the name of the tag or other selector, otherwise the selector will be applied'*', that is$(':file')will be perceived as$('*:file'), so it's better to use instead$('input:file').

Example

Let's select all inputs with typefileand give it a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <input type="password"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> <+javascript+>$('form input:file').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // prevents form submission });<-javascript->
