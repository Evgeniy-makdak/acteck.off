Селектор contains

Селектор :contains выбирает элементы, которые содержат заданный текст.

Синтаксис

Так мы выбираем элементы по заданному тексту. Текст может быть в кавычках или без. Также этот параметр чувствителен к регистру:

$(':contains(текст)');

Пример

Давайте выберем абзац с текстом bbb, и поставим ему в конец текст '!!!':

<p>aaa</p> <p>bbb</p> <p>ccc</p> $('p:contains("bbb")').append('!!!');

HTML код станет выглядеть так:

<p>aaa</p> <p>bbb!!!</p> <p>ccc</p>

Смотрите также

  • метод find,
    который ищет элементы внутри уже найденных
  • метод text,
    который получает и изменяет текст элемента
  • метод contents,
    который позволяет получить потомков элемента и текст внутри этого элемента
