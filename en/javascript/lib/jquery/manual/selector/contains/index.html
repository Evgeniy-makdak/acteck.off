The contains selector

Selector:containsselects elements that contain the specified text.

Syntax

This is how we select elements based on the given text. The text may be in quotes or without. This parameter is also case sensitive:

$(':contains(текст)');

Example

Let's select a paragraph with textbbb, and put the text at the end of it'!!!':

<p>aaa</p> <p>bbb</p> <p>ccc</p> $('p:contains("bbb")').append('!!!');

The HTML code will look like this:

<p>aaa</p> <p>bbb!!!</p> <p>ccc</p>

See also

    which searches for elements within those already found
  • methodtext,

  • methodcontents,
