input selector

Selector:inputselects form controls - inputs, textarea, drop-down lists and buttons. See tags:input, textarea, button, select. Because:inputdoes not apply to the CSS specification, then to improve performance in modern browsers it is better to first filter elements using a pure CSS selector, and then apply.filter(':input').

Syntax

This is how we select the form controls:

$(':input');

Example

Let's, according to the above theory, select all the form controls and display their number in the console using the propertylength. We will see that everyone got there13tags:
