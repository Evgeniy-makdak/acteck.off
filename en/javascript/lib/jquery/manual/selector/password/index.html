password selector

Selector:passwordselects elements with typepassword. Lookpassword fields. Equivalent$(':password')is$('[type=password]'). Because:passworddoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='password'].

Syntax

So we select fields with typepassword:

$(':password');

As with other pseudo-class selectors (starting with':'), before':'it is better to put the name of the tag or other selector, otherwise the selector will be applied'*', that is$(':password')will be perceived as$('*:password'), so it's better to use instead$('input:password').

Example

Let's select all inputs with typepasswordand give it a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <input type="password"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> <+javascript+>$('form input:password').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // prevents form submission });<-javascript->
