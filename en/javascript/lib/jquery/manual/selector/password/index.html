Селектор password

Селектор :password выбирает элементы с типом password. Смотрите поля для ввода пароля. Эквивалентом $(':password') является $('[type=password]'). Так как :password не относится к CSS спецификации, то для улучшения производительности в современных браузерах вместо него лучше применить [type='password'].

Синтаксис

Так мы выбираем поля с типом password:

$(':password');

Как и с другими селекторами псевдо-классами (начинающимися с ':'), перед ':' лучше поставить название тега или другого селектора, в противном случае будет применен селектор '*', то есть $(':password') будет восприниматься как $('*:password'), поэтому вместо этого лучше использовать $('input:password').

Пример

Давайте, выберем все инпуты с типом password и сделаем им зеленый фон и красную рамку с помощью метода css:

<form> <input type="button" value="button"> <input type="file"> <input type="password"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> <+javascript+> $('form input:password').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // предотвращает отправку формы }); <-javascript->
