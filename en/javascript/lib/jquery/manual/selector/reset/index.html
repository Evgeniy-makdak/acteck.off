Selector reset

Selector:resetselects elements with typereset, which are buttons for clearing the form. Equivalent$(':reset')is$('[type=reset]'). Because:resetdoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='reset'].

Syntax

This is how we select elements with typereset:

$(':reset');

Example

Let's select all inputs with typeresetand give it a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <input type="password"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> <+javascript+>$('input:reset').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // prevents form submission });<-javascript->
