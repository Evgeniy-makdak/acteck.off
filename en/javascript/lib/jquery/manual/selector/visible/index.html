visible selector

Selector:visibleselects all visible elements on the page. Elements withvisibility: hiddenor withopacity: 0are considered visible because they take up space in the layout. Elements are considered visible if they take up space in the document. Visible elements have a height and width different from0. Because:visibledoes not apply to the CSS specification, then to improve performance in modern browsers it is better to first filter elements using a pure CSS selector, and then apply.filter(':visible'). Tracking element visibility using other methods, such as a class, may provide better performance.

Syntax

This is how we select visible elements:

$(':visible');

Example

Let's click on the button to show invisible squares. When you click on the “visible” squares, their background will change to yellow:

<button id="show">show</button> <div></div> <div style="display:none;"></div> <div></div> <div></div> <div style="display:none;"></div> div { width: 40px; height: 40px; margin: 5px; border: 2px outset green; float: left; } $('div:visible').click(function() { $(this).css('background', 'yellow'); }); $('#show').click(function() { $('div:hidden').show('slow'); });
