Selector submit

Selector:submitselects elements with typesubmit, which are form submit buttons. Typically the selector:submitapplies to buttons or input elements. Please be aware that some browsers accept the elementbuttonas having[type='submit']implicitly, while others do the opposite. To ensure proper operation, always specify the propertytype. Because:submitdoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='submit'].

Syntax

This is how we select elements with typesubmit:

$(':submit');
