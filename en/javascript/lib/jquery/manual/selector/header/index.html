header selector

Selector:headerselects headingsh1, h2, h3, h4, h5, h6. Because:headerdoes not apply to the CSS specification, then to improve performance in modern browsers it is better to first filter elements using a pure CSS selector, and then apply.filter(':header').

Syntax

This is how we select the elements:

$(':header');

Example

Let's change the background of all headers to gray:

<h1>Header h1</h1> <p>text</p> <h2>Header h2</h2> <p>text</p> $(':header').css({background: '#ccc'});

  • methodcss,
