button selector

Selector:buttonselects button elements and all elements with typebutton. See tagsbutton, inputwith typebutton. Equivalent$(':button')is$('button'), $("input[type='button']"). Because:buttondoes not apply to the CSS specification, then to improve performance in modern browsers it is better to first filter elements using a pure CSS selector, and then apply.filter(':button').

Syntax

This is how we select the buttons:

$(':button');

Example

Let's, according to the above theory, select all the buttons and give them a green background and a red border, adding a class to themmarkedusing the methodaddClass:

<form> <input type="button" value="button"> <input type="file"> <button>button</button> <input type="reset"> <input type="radio"> <input type="text"> </form> .marked { background-color: green; border: 2px red solid; } $(':button').addClass('marked'); $('form').submit(function(event) { event.preventDefault(); // предотвращает отправку формы });
