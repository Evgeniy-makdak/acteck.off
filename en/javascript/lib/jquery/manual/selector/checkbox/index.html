checkbox selector

Selector:checkboxselects checkboxes. See tagcheckbox. Equivalent$(':checkbox')is$('[type=checkbox]'). Because:checkboxdoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead[type='checkbox'].

Syntax

This is how we select checkboxes:

$(':checkbox');

As with other pseudo-class selectors (starting with':'), before':'it is better to put the name of the tag or other selector, otherwise the selector will be applied'*', that is$(':checkbox')will be perceived as$('*:checkbox'), so it's better to use instead$('input:checkbox').

Example

Let's select all the checkboxes and wrap them in span. Then we will create a green background and a red frame using the methodcss:

<form> <input type="button" value="button"> <input type="file"> <input type="checkbox"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> $('form input:checkbox') .wrap('<span></span>') .parent() .css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // предотвращает отправку формы });
