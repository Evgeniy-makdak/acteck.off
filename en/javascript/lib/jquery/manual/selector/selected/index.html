Selector selected

Selector:selectedselects the elements that are selected, that is, those that have the attributeselected. Selector:selectedworks for elements with tagoption. It doesn't work with checkboxes (checkbox) or radio switches (radio), for them use a selectorchecked. Because:selecteddoes not apply to the CSS specification, then to improve performance in modern browsers it is better to first filter elements using a pure CSS selector, and then apply.filter(':selected').

Syntax

This is how we select elements with the attributeselected:

$(':selected');

Example

Let's display it in green below the list each time we select an item in the drop-down list. As you can see, before your first click, the default selected items will be displayed, these are'bbb'And'ddd':
