Selector empty

Selector:emptyselects elements that have no children (that is, empty elements that have neither element nor text).

Syntax

Getting empty elements:

$(':empty');

Example

We have a signtablewith green background, let's find the cellstd, which have neither text nor element inside them, and color them gray:

<table> <tr><td>aaa</td><td></td></tr> <tr><td>bbb</td><td></td></tr> <tr><td>ccc</td><td></td></tr> </table> table { border: 1px; } td { width: 40px; background: green; } $('td:empty').css({background: '#ccc'});
