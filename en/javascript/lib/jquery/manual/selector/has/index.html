Selector has

Selector:hasselects elements that contain at least one element by the given selector, not necessarily an immediate descendant. Because:hasdoes not belong to the CSS specification, then to improve performance in modern browsers it is better to use it instead$("ваш css-селектор").has(селектор/DOM элемент).

Syntax

This is how we select the elements:

$(':has(селектор)');

This is how we select elements if we need direct nesting of the selector:

$(':has(>селектор)');

Example

Let's select only those paragraphs that have a tag insideband put the text at the end of them'!!!':

<p>text</p> <p>text</p> <p><b>bold</b> text</p> <p><b>bold</b> text</p> $('p:has(b)').append('!!!');

The HTML code will look like this:

<p>text</p> <p>text</p> <p><b>bold</b> text!!!</p> <p><b>bold</b> text!!!</p>
