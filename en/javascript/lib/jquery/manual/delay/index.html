delay method

Methoddelayallows us to delay the execution of queued functions. It can be applied to both the standard effects queue and custom queues. It can only be applied to queued events. Duration can be set not only in milliseconds, but also using keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation. Because at the methoddelaythere are certain limitations, it is not a replacement for native JavaScript functionssetTimeout, which may be more convenient in certain cases.

Syntax

This is how we set the delay. The first parameter sets the duration of the delay for the execution of the next element in the queue in milliseconds. The second optional parameter can be the queue name - by defaultfx- name of the standard effects queue:

.delay(длительность, [имя очереди]);

Example

Let's first hide the element withhide, we will make a delay in3seconds and then show it back withshow:

<p id="test">text text text</p> $('#test').hide(1000).delay(3000).show(1000);
