Метод removeClass

Метод removeClass удаляет заданный CSS класс из атрибута class. Можно удалить один или несколько классов одновременно.

Синтаксис

Один CSS класс:

$(селектор).removeClass(имя CSS класса)

Несколько CSS классов:

$(селектор).removeClass('класс1 класс2 класс3 ...')

Применение функции к каждому элементу в наборе:

$(селектор).removeClass(function(номер в наборе, текущие CSS классы элемента))

Пример

У нашего абзаца уже есть два класса. Давайте удалим класс www:

<p id="test" class="eee www">text</p> $('#test').removeClass('www');

HTML код станет выглядеть так:

<p id="test" class="eee">text</p>

Пример

У нашего абзаца уже есть класс. Давайте удалим этот класс:

<p id="test" class="www">text</p> $('#test').removeClass('www');

HTML код станет выглядеть так:

<p id="test" class="">text</p>

Пример

У нашего абзаца уже есть 3 класса. Давайте удалим два класса - www и ggg:

<p id="test" class="eee www ggg">text</p> $('#test').removeClass('www ggg');

HTML код станет выглядеть так:

<p id="test" class="eee">text</p>

  • JavaScript свойство classlist,
    с помощью которого можно добавлять и удалять CSS классы элемента на чистом JavaScript
