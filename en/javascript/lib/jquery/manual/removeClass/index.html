removeClass method

MethodremoveClassremoves the specified CSS class from an attributeclass. You can delete one or more classes at the same time.

Syntax

One CSS class:

$(селектор).removeClass(имя CSS класса)

Several CSS classes:

$(селектор).removeClass('класс1 класс2 класс3 ...')

Applying a function to each element in a set:

$(селектор).removeClass(function(номер в наборе, текущие CSS классы элемента))

Example

Our paragraph already has two classes. Let's delete the classwww:

<p id="test" class="eee www">text</p> $('#test').removeClass('www');

The HTML code will look like this:

<p id="test" class="eee">text</p>

Example

Our paragraph already has a class. Let's remove this class:

<p id="test" class="www">text</p> $('#test').removeClass('www');

The HTML code will look like this:

<p id="test" class="">text</p>

Example

Our paragraph already has3class. Let's remove two classes -wwwAndggg:

<p id="test" class="eee www ggg">text</p> $('#test').removeClass('www ggg');

The HTML code will look like this:

<p id="test" class="eee">text</p>
