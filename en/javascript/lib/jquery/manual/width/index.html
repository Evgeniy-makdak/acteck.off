width method

Methodwidthallows you to get and change the width of an element. It's important to remember that we will be getting the 'calculated value' of the width (computed width). Unlike the methodcss('width'), it returns a dimensionless quantity (for example400) and is convenient for mathematical calculations. The method gets the width of the element's content, regardless of the CSS property specifiedbox-sizing. To avoid the inconvenience associated with this, it is recommended to usecss('width'). Calculation errors can also occur if the user resizes the page, or if an element or its parent is hidden. The width value does not take into account padding or border values.

Syntax

Get the width of the element. In some cases, the resulting values ​​may be fractional:

$(селектор).width();

To change the width of an element, you can simply pass a number (for example400), then the units of measurement will be pixels, or a string indicating the units of measurement (for example'10em'):

$(селектор).width(новое значение);

We can also apply a given function to each element in the set. In this case, the first parameter the function will receive is the number of the element in the set, and the second parameter will be the current value of the specified width for a specific element. Meaningthisinside the function will point to the current element. The element width value will change to the one returned by the function:

$(селектор).width(function(номер в наборе, текущее значение ширины))ж

Example

Let's click ondivchange its width to value40pxusing the methodwidth, and also with the helpcssLet's change its background to green:
