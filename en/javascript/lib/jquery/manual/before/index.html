before method

Methodbeforeadds text before the given element. There is also a methodinsertBefore, which works in a similar way.

Syntax

Inserting text before an element:

$(селектор).before(текст);

This is how text will be added before the selected elements, which will be returned by the user function:

$(селектор).before(function(номер в наборе));

Another option for using the function, the second parameter is the old html value of the element:

$(селектор).before(function(номер в наборе, html строка));

The text can also be not only plain text, but also a DOM element or a jQuery object. In this case, these elements will move from their position in the HTML code.

Example

Let's insert text before the given paragraph:

<p id="test">text</p> $('#test').before('!!!');

The HTML code will look like this:

!!!<p id="test">text</p>

Example

Let's insert tagged text before the given paragraph:

<p id="test">text</p> $('#test').before('<p>!!!</p>');

The HTML code will look like this:

<p>!!!</p><p id="test">text</p>

Example

Let's put one paragraph above the other (that is, cut the paragraph from the old place and put it in the new one):

<p id="p1">text1</p> <p id="p2">text2</p> $('#p2').before('#p1');

The HTML code will look like this:

<p id="p2">text2</p> <p id="p1">text1</p>

See also

    which adds text before a given element
  • methodsafter, append, prepend,
    allowing you to add content to a specific location on the page
  • methodclone,
    which creates copies of the selected elements
