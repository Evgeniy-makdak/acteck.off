after method

Methodafteradds text after the given element. There is also a methodinsertAfter, which works in a similar way.

Syntax

Inserting text after an element:

$(селектор).after(текст);

This is how text will be added after the selected elements and returned by the custom function. The function is called separately for each of the selected elements. The first parameter of this function contains the number in the set of each of the selected elements (in turn), the second - the current contents of the element:

$(селектор).after(function(номер в наборе, текущее содержимое элемента));

The text can also be not only plain text, but also a DOM element or a jQuery object. In this case, these elements will move from their position in the HTML code.

Example

Let's insert text after the given paragraph:

<p id="test">text</p> $('#test').after('!!!');

The HTML code will look like this:

<p id="test">text</p>!!!

Example

Let's insert tagged text after the given paragraph:

<p id="test">text</p> $('#test').after('<p>!!!</p>');

The HTML code will look like this:

<p id="test">text</p><p>!!!</p>

Example

Let's move one paragraph under another (that is, cut the paragraph from the old place and put it in the new one):

<p id="p1">text1</p> <p id="p2">text2</p> $('#p1').after('#p2');

The HTML code will look like this:

<p id="p2">text2</p> <p id="p1">text1</p>

See also

    which adds text after a given element
  • methodsbefore, append, prepend,
    allowing you to add content to a specific location on the page
  • methodclone,
    which creates copies of the selected elements
