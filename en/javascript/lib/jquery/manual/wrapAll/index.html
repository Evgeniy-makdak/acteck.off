wrapAll method

MethodwrapAllwraps all elements of a set in one given tag. If these elements are in different places, then jQuery first places them next to each other, and then wraps them in the desired tag.

Syntax

The method parameter can be text (possibly with tags), a DOM element, or a jQuery object:

.wrapAll(text);

We can also use a function for each element:

.wrapAll(function(index))

Example

Let's wrap paragraphs with classwwwto tagdiv:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrapAll('<div></div>');

The HTML code will look like this:

<div> <p class="www">text</p> <p class="www">text</p> <p class="www">text</p> </div> <p>text</p>
