Метод empty

Метод empty удаляет содержимое из выбранных элементов (сами элементы не удаляются, а остаются пустыми).

Синтаксис

.empty();

Пример

Давайте удалим содержимое элемента #test:

<p id="test">Hello, <b>world</b>!</p> $('#test').empty();

HTML код станет выглядеть так:

<p id="test"></p>

Смотрите также

  • метод remove,
    который удаляет элементы без возможности их восстановления
  • метод detach,
    который удаляет элементы с возможностью их восстановления
  • JavaScript метод remove,
    который позволяет удалить элемент
