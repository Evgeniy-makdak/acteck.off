jQuery Events

The tables below show the methods for individual jQuery events and their descriptions:

Mouse events

Name Description
click The mouse button is clicked on the control element. The event occurs after the mouse button has been released.
mousedown Mouse button pressed.
mouseup The mouse button is released.
dblclick Double click.
mousemove The mouse pointer moves within the element's display area.
mouseover The mouse pointer is inside the element's display area.
mouseout The mouse pointer has moved out of the element's display area.
mouseenter The mouse pointer is inside the element's display area.
mouseleave The mouse pointer has moved out of the element's display area.
hover Mouse pointing and moving.

Keyboard events

Name Description
keydown A button on the keyboard is pressed.
keypress A key on the keyboard has been pressed and not released for longer than the repeat interval. The duration of the repetition interval is a system parameter and depends on the user's operating system settings.
keyup A previously pressed button has been released.

Events of forms and page elements

Name Description
change An element loses input focus, and the content of the element has changed while the element had focus.
select Some part of the text inside the element becomes highlighted.
submit The "Submit" button has been clicked on the form, but the form has not yet been submitted to the server.
blur The control loses input focus, i.e. the cursor moves to another element.
focus The displayed element has received input focus. For text fields, this event indicates that the cursor is already in the element.
focusin The displayed element has received input focus. For text fields, this event indicates that the cursor is already in the element.
focusout The control loses input focus, i.e. the cursor moves to another element.

Document Load Events

Name Description
ready The DOM of the page is ready.
load Completed loading items.
unload The document is exited (closing or redirecting the page to another address).

Browser Events

Name Description
scroll Fires when an element is scrolled.
resize Fires when the browser page size changes.

See also

  • methodon,
    which allows you to bind an event handler to an element
  • methodtrigger,
    which allows you to run all event handlers bound to an element for events of a given type
  • objectevent,
    which contains properties and methods containing information about the event that was fired
