index method

Methodindexreturns the number of an element in a jQuery set (numbered from zero).

Syntax

Returns the position number of the first selected element, relative to adjacent elements on the page (neighboring refers to elements that have a common parent):

.index();

Returns the position number of the first selected element among elements that match the selector. If such an element is not found there, the method will return-1:

.index(селектор);

The parameter can also be a DOM element or a jQuery object (if it contains multiple elements, the method will be executed on the first one):

.index(элемент);

Example

Let's display the numberliWith#test:

<ol> <li>text</li> <li id="test">text</li> <li>text</li> </ol> let num = $('li').index('#test'); alert(num);
