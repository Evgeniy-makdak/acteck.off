toArray method

MethodtoArrayConverts a set of jQuery elements to a JavaScript array. This is necessary so that JavaScript methods and functions can be applied to this array, for example,sortthis array orturn over.

Syntax

This is how we get the entire set of elements:

.toArray();

Example

Let's get all elements with tagpas an array using the methodtoArray, let's flip it usingreverseand output the text content of the elements as a string using the JavaScript methodjoin:

<p>text1</p> <p>text2</p> <p>text3</p> <div>text4</div> function print(elems) { let arr = []; for (let i = 0; i < elems.length; i++) { arr.push(elems[i].innerHTML); } console.log(arr.join(' ')); } alert($('p').toArray().reverse());
