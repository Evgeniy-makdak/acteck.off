nextAll method

MethodnextAllreturns all elements that are after the given one within one parent.

Syntax

Getting all neighbors below:

$(селектор).nextAll();

You can also filter neighbors by a given selector:

$(элемент).nextAll(селектор);

Example

Let's find the element#testand to all its neighbors below we will put the text'!'using the methodhtml:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').nextAll().html('!');

The HTML code will look like this:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>!</p> <p>!</p> </div> <p>outside</p>

See also

  • methodsiblings,
