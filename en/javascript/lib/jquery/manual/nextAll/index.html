Метод nextAll

Метод nextAll возвращает все элементы, которые находятся после заданного внутри одного родителя.

Синтаксис

Получение всех соседей снизу:

$(селектор).nextAll();

Можно также пофильтровать соседей по заданному селектору:

$(элемент).nextAll(селектор);

Пример

Давайте найдем элемент #test и всем его соседям снизу поставим текст '!' с помощью метода html:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').nextAll().html('!');

HTML код станет выглядеть так:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>!</p> <p>!</p> </div> <p>outside</p>

Смотрите также

  • метод prevAll,
    который возвращает элементы, находящиеся перед заданным внутри одного родителя
  • метод next,
    который позволяет соседа элемента снизу
  • метод nextUntil,
    который возвращает элементы, находящиеся после заданного
  • метод siblings,
    который позволяет получить соседей элемента внутри одного родителя
