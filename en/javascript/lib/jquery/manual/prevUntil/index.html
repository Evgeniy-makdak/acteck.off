prevUntil method

MethodprevUntilreturns elements that are before the given one within one parent. The method returns elements from the given one to the first element that matches the selector specified by the method parameter. If you do not pass parameters to a method, it will work as a methodprevAll.

Syntax

The first parameter can be passed an expression in the form of a string by which the selection is made, a DOM node or a jQuery object. The second parameter can be used to pass an additional filtering condition in the form of a string. Both parameters are optional:

.prevUntil([селектор], [фильтр]);

Example

Let's find the element#testand all its neighbors from above to the paragraph with the classwwwput the text'!'using the methodhtml:

<p>outside</p> <div> <p>inside</p> <p class="www">inside</p> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> <p class="www">inside</p> <p>inside</p> </div> <p>outside</p> $('#test').prevUntil('p.www').html('!');

The HTML code will look like this:

<p>outside</p> <div> <p>inside</p> <p class="www">inside</p> <p>!</p> <p>!</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> <p class="www">inside</p> <p>inside</p> </div> <p>outside</p>
