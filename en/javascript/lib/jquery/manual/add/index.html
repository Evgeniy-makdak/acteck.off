add method

Methodaddadds the specified elements to an already existing set of elements.

Syntax

Adding by selector:

.add(селектор);

The elements to be added are specified by one or more (array) DOM elements:

.add(DOM элемент);

The elements to be added are specified by the jQuery object:

.add(объект jQuery);

The elements to be added are specified in html text:

.add(html текст);

The elements to be added are searched on the page using the specified selector, within the area specified by the second parameter. The search scope can be specified by a DOM element, a jQuery object, or a document object:

.add(селектор, контекст);

Example

Let's find all the paragraphs and put the text at the end of them'!', then add headings to the found paragraphsh2and at the same time set the color red for headings and paragraphs:

<div>ddd</div> <h1>hhh</h1> <p>ppp</p> <div id="test"><h2>hhh</h2></div> <p>ppp</p> <h2>hhh</h2> <p>ppp</p> $('p').append('!').add('h2').css('color', 'red');

Example

Let's introduce search context - add only thoseh2, which lie inside the element#test:

let $context = $('#test'); $('p').append('!').add('h2', $context).css('color', 'red');

Example

Let's create a search context as a DOM element using the JavaScript methodquerySelector:

let context = document.querySelector('#test'); $('p').append('!').add('h2', context).css('color', 'red');

Example

Most often, you can do without context by simply making a more complex selector:

$('p').append('!').add('#test h2').css('color', 'red');
