animate method

Methodanimateallows you to create animation effects for various CSS properties. The only required parameter is an object with CSS properties, similar to what we pass to the methodcss. Many properties that have non-numeric values ​​or multiple values ​​cannot be animated by core jQuery functionality (for examplebackground-color). Also, abbreviated names for CSS properties are not always supported, for examplefont- should be used insteadfontSizeorfont-size. Before the values ​​you can put'+='or'-=', in this case the value specified below will be added or subtracted from the current value of the property. Also, instead of numerical values, you can putshow, hideortoggle. All jQuery effects, includinganimate, can be disabled globally using the settingjQuery.fx.off = true, which sets the duration value to 0.

Syntax

The first parameter of the animation is an object with CSS properties and their values ​​in the formatkey: value, which we are going to animate. The remaining parameters are considered optional. The second parameter determines the duration of the animation in milliseconds,400мсdefault. The third parameter is the smoothness function for the transition (by default this isswing), as well as the fourth - a callback function that will run after the animation is completed:

.animate(свойства, [длительность ], [функция плавности ], [callback-функция ]);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation. You can pass various options to the method as the second parameter, in the form of a JavaScript object containing pairskey: value:

.animate(свойства, options);

For example, let's set the CSS properties towidthAndheightmeaningtoggle. Duration for5000мс, we will transfer the smoothness functions in the form of an object, for width it will be a linear function, and for height -easeOutBounce, also upon completion of the animation, we will execute the function once (see the keycomplete), which is after the element with#testwill place a div with the inscription'Animation complete.':

$('#test').animate({ width: 'toggle', height: 'toggle' }, { duration: 5000, specialEasing: { width: 'linear', height: 'easeOutBounce' }, complete: function() { $(this).after('<div>Animation complete.</div>'); } });

The table below shows the properties and methods of the objectoptionsand their description:

Example

Let's press the button#left, we will move the green square to the left, and to the button#rightright by50px, also set the duration to600мс- teamslow:

<button id="left">left</button> <button id="right">right</button> <div class="block"></div> div { position: absolute; background-color: green; left: 50px; width: 100px; height: 100px; margin: 5px; } $('#right').click(function() { $('.block').animate({'left': '+=50px'}, 'slow'); }); $('#left').click(function() { $('.block').animate({'left': '-=50px'}, 'slow'); });
