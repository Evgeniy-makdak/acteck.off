attr method

Methodattrallows you to get and change any attribute of an element.

It is important not to confuse element properties and attributes. When creating a DOM tree, attributes are translated into their corresponding DOM element properties. Their names do not always coincide, nor do their meanings. Some properties may also not have attributes corresponding to them, these can only be obtained and set by using the methodprop, for example propertytagName.

Syntax

Getting the attribute value:

$(селектор).attr(имя атрибута);

Changing the attribute value:

$(селектор).attr(имя атрибута, новое значение);

Changing the values ​​of several attributes:

$(селектор).attr({первый_атрибут: новое_значение, второй_атрибут: новое_значение, ...});

Applying a function to each element in a set:

$(селектор).attr(имя атрибута, function(номер в наборе, текущее значение атрибута));

Example

In the following example we will receive input by itsidand display the value of its attributevalue:

<input type="text" value="!!!" id="test"> let value = $('#test').attr('value'); console.log(value);

Example

Now let's write down the new value'www'to the same attribute:

<input type="text" value="!!!" id="test"> $('#test').attr('value', 'www');

The HTML code will look like this:

<input type="text" value="www" id="test">

Example

Let's change the attributes at the same timetypeAndvalue:

<input type="text" value="!!!" id="test"> $('#test').attr({value: 'www', type: 'password'});

The HTML code will look like this:

<input type="password" value="www" id="test">

Example

Let's change the attribute of each inputvalue- add to the current onevalueserial number of the element in the set:

<input type="text" value="aaa"> <input type="text" value="bbb"> <input type="text" value="ccc"> $('input').attr('value', function(index, value) { return value + ' ' + index; });

The HTML code will look like this:

<input type="text" value="aaa 1"> <input type="text" value="bbb 2"> <input type="text" value="ccc 3">
