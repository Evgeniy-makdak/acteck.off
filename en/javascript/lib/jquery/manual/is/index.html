The is method

Methodischecks whether at least one of the selected elements matches the given selector. Instead of a selector, a DOM element or a jQuery object can be specified. If you specify a function as a parameter, the condition check will be assigned to it. Returns a value of typeboolean (trueorfalse).

Syntax

The method checks whether at least one of the selected elements satisfies the given selector:

.is(селектор);

The method checks whether among the selected elements there is at least one of the elements of the specified jQuery object or DOM element:

.is(jQuery объект или DOM элемент);

This is how the specified function will be called for each of the selected elements. As a parameter, it will receive the position number of the element in the set, the element itself will be available in a variablethis. The function must returntrueorfalse. If at least one of the selected elements returns a valuetrue, Thatiswill also returntrue. Otherwise, the method will returnfalse.

.is(function(index));
