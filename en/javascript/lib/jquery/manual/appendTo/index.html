appendTo method

MethodappendToadds text to the end of the element. There is also a methodappend, which works in a similar way.

Syntax

Inserting text at the end of an element:

$(текст).appendTo(селектор);

The content can be not only plain text, but also a DOM element, selector, array of elements, string or jQuery object.

Example

Let's insert tagged text at the end of a given paragraph:

<p id="test">абзац</p> $('<b>!!!</b>').appendTo('#test');

Result of executing the code:

<p id="test">абзац<b>!!!</b></p>

    which adds text to the end of the element
  • pseudo elementafter,
    which adds text to the end of an element in CSS
  • methodsprependTo, insertBefore, insertAfter,
    allowing you to add content to a specific location
