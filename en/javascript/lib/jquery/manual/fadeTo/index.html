fadeTo method

MethodfadeTochanges smoothlytransparencyelement to a given value, in the range from0to1.

Syntax

Duration and transparency are set for any use of the method. Changing transparency over a given time,400мсdefault:

.fadeTo(длительность, значение прозрачности);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.fadeTo('slow' или 'fast', значение прозрачности);

You can also pass a callback function as a third (optional) parameter - it will work after the animation is completed:

.fadeTo(длительность, значение прозрачности, [callback-функция]);

You can also pass a smoothness function and a callback function (optional parameters) - it will work after the animation is completed:

.fadeTo(длительность, значение прозрачности, [функция плавности], [callback-функция]);

Example

In the following example, using the methodfadeTowhen you press a button#hideelement with#testwill be hidden by reducing the transparency to0.5, and on#show- show up:

<button id="hide">hide</button> <button id="show">show</button> <div id="test"></div> #test { width: 200px; height: 200px; background: green; color: white; margin-top: 10px; } $('#show').on('click', function() { $('#test').fadeTo(1000, 1); }); $('#hide').on('click', function() { $('#test').fadeTo(1000, 0.5); });
