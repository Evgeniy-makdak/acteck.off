parentsUntil method

MethodparentsUntilgets all the parents of the element - not only the immediate one, but also the grandparent, great-grandparent, and so on until the root element (that is, until the taghtml). In this case, the search for parents ends upon reaching a parent that falls under the selector specified by the parameter of this method.

Syntax

Search for parent to parent that matches the selector, the selector can be a DOM node, expression or jQuery object, you can also use the second parameter for additional filtering. Both parameters are optional:

.parentsUntil([селектор], [фильтр]);

The selector may not be specified, in which case the method will behave similarly to the methodparents:

.parentsUntil();

Example

Let's find the element#testthen we find all its parents up to the element with the classzzzby usingparentsUntiland put the text at the beginning'!'by usingprepend:

<div> <div class="zzz"> <div class="www"> <div class="www"> <p id="test">text</p> </div> </div> </div> </div> $('#test').parentsUntil('.zzz').prepend('!');

The HTML code will look like this:

<div> <div class="zzz"> <div class="www"> ! <div class="www"> !<p id="test">text</p> </div> </div> </div> </div>
