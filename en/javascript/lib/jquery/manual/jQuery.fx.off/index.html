jQuery.fx.off property

PropertyjQuery.fx.offallows you to globally disable animation. When you set this property totrueall animation methods will set the elements to their final state, without showing any effects. To undo this setting, set this property back tofalse.

Example

Let's assume we have a green rectangle, we will start the animation when the button is pressed#startusing the methodtoggle, and using the button#togglewe will turn the animation on and off using the settingjQuery.fx.off. With animation turned off, our rectangle will simply appear and disappear without any effects:

<input id="start" type="button" value="start"> <button id="toggle">toggle fx</button> <div></div> div { width: 50px; height: 30px; margin: 5px; float: left; background: green; } let toggleFx = function() { $.fx.off = !$.fx.off; }; $('#toggle').click(toggleFx); $('#start').click(function() { $('div').toggle('slow'); });
