addBack method

MethodaddBackadds elements from the previous set to the current one (we are talking about one chain of methods), you can also selectively pass a selector to the method. The previous set refers to the set of elements that can be obtained using the methodend.

Syntax

.addBack([селектор]);

Example

Let's find the paragraph with#test, then find the parent of this paragraph usingparent, add the found paragraph to its parent usingaddBackand put the text at the beginning'!'by usingprepend. It turns out that both the paragraph and its parent will have text at the beginning'!':

<div> <p>text</p> <p id="test">text</p> <p>text</p> </div> $('p').parent().addBack().prepend('!');

The HTML code will look like this:

<div> ! <p>!text</p> <p id="test">!text</p> <p>!text</p> </div>
