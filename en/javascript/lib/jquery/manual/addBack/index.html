Метод addBack

Метод addBack добавляет элементы из предыдущего набора к текущему (речь идет об одной цепочке методов), в метод можно также выборочно передать селектор. Под предыдущим набором подразумевается набор элементов, который можно получить с помощью метода end.

Синтаксис

.addBack([селектор]);

Пример

Давайте найдем абзац с #test, затем найдем родителя этого абзаца с помощью parent, добавим найденный абзац к его родителю с помощью addBack и поставим им в начало текст '!' с помощью prepend. Получится, что и абзац, и его родитель в начале будут иметь текст '!':

<div> <p>text</p> <p id="test">text</p> <p>text</p> </div> $('p').parent().addBack().prepend('!');

HTML код станет выглядеть так:

<div> ! <p>!text</p> <p id="test">!text</p> <p>!text</p> </div>

Смотрите также

  • метод add,
    который добавляет заданные элементы в набор
  • метод end,
    который возвращает предыдущий набор элементов в текущей цепочке методов
