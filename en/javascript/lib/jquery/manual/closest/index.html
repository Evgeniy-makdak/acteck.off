closest method

Methodclosestfinds the closest parent element that satisfies the selector specified by the method parameter. This method is similar to the methodparent. The search is carried out like this: first, the immediate parent of the element is checked, then the grandparent, great-grandparent, and so on until the root element (that is, to the taghtml).

Syntax

The selector can be a DOM element, an expression, or a jQuery object. The search begins with the DOM element passed as a parameter:

.closest(селектор);

You can also pass a DOM element as the second parameter, then the search will be carried out in this element:

.closest(селектор, контекст);

Example

Let's for element with#testfind the closest parent with the classwwwand at the beginning of this element we insert a paragraph with text'!', and at the end - a paragraph with text'?':

<div> <div class="www"> <div> <p id="test">text</p> </div> </div> </div> $('#test').closest('.www').append('<p>?</p>').prepend('<p>!</p>');

The HTML code will look like this:

<div> <div class="www"> <p>!</p> <div> <p id="test">text</p> </div> <p>?</p> </div> </div>
