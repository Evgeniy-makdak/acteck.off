detach method

Methoddetachdeletes selected items with the option to restore them. The method is convenient when we want to insert deleted elements later.

Syntax

This is how we remove elements. Additionally, we can filter the selected elements by passing an optional parameter to the methodselector:

.detach([селектор]);

Example

Let's delete the paragraph from#testusing the methoddetachand restore it by using it to insert an element intodivmethodappendTo:

<p id="test">text</p> <div></div> let $elem = $('#test'); $elem.detach(); // удаляем элемент $elem.appendTo('div'); // вставляем элемент обратно на страницу

The HTML code will look like this:

<div><p id="test">text</p></div>
