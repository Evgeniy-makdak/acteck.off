hide method

Methodhidesmoothly hides elements. You can show elements using the methodshow.

Syntax

Disappearance within a given time,400мсdefault:

.hide(длительность);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.hide('slow' или 'fast');

If you do not specify the parameters, there will be no animation, the elements will be hidden instantly:

.hide();

You can also pass the smoothness function as the second parameter, as well as a callback function as the third - it will work after the animation is completed. Both parameters are optional:

.hide(длительность, [функция плавности], [callback-функция]);

You can pass various options to the method, in the form of a JavaScript object containing pairskey: value:

.hide(options);

Such an object can pass the following parameters and functions -duration, easing, queue, specialEasing, step, progress, complete, start, done, fail, always. You can see the description of these parameters for the methodanimate. For example, let's set the duration and smoothness function:

.hide( {duration: 800, easing: easeInSine} );

Example

Let's after pressing the button#test, smoothly hide the paragraph using the methodhide. By passing the keywordslow, we will set the speed to600мс. After the animation is finished, we will output it to the console'Animation complete':

<button id="test">hide text</button> <p>text</p> $('#test').click(function() { $('p').hide('slow', function() { console.log('Animation complete') }); });
