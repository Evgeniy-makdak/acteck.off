position method

Methodpositionallows you to get the position of an element relative to the parent's padding, unlike the methodoffset, which receives coordinates relative to the document. Methodpositionmore useful when positioning a new element next to another in the same parent. As a result, the method returns an object containing propertiestopAndleft. Calculation errors may occur if the user resizes the page. Also, the method does not receive the coordinates of hidden elements.

Syntax

Get current coordinates. In some cases, the resulting values ​​may be fractional:

$(селектор).position();

Example

Let's get the position of the paragraph which is located in the div using the methodposition, and then, referring to the keys of the resulting object, we display this information in another paragraph:

<div style="padding: 15px;"> <p style="margin-left: 10px;">text</p> </div> <p></p> let position = $('p').first().position(); $('p').last().text('left: ' + position.left + ', top: ' + position.top);
