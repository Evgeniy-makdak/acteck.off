Метод parent

Метод parent получает непосредственного родителя элемента.

Синтаксис

Получение непосредственного родителя для элементов:

.parent();

Можно также выбрать непосредственного родителя, только если он удовлетворяет заданному селектору:

.parent(селектор);

Пример

Давайте найдем элемент #test затем найдем его родителя с помощью parent и поставим ему текст '!' с помощью html:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').parent().html('!');

HTML код станет выглядеть так:

<p>outside</p> <div> ! </div> <p>outside</p>

Пример

Давайте найдем элемент #test затем найдем его родителя с помощью parent и, если этот родитель div с классом www, то поставим ему текст '!' с помощью html:

<p>outside</p> <div class="www"> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').parent('div.www').html('!');

HTML код станет выглядеть так:

<p>outside</p> <div class="www"> ! </div> <p>outside</p>

Смотрите также

  • метод parents,
    который позволяет получить всех родителей элемента
  • метод closest,
    который позволяет получить ближайший родительский элемент, удовлетворяющий селектору
  • метод parentsUntil,
    который позволяет получить всех родителей элемента до указанного родителя
  • метод children,
    который позволяет получить потомков элемента
  • метод contents,
    который позволяет получить потомков элемента и текст внутри этого элемента
