parent method

Methodparentgets the element's immediate parent.

Syntax

Getting the immediate parent of elements:

.parent();

You can also select the immediate parent only if it satisfies the given selector:

.parent(селектор);

Example

Let's find the element#testthen find its parent usingparentand give him the text'!'by usinghtml:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').parent().html('!');

The HTML code will look like this:

<p>outside</p> <div> ! </div> <p>outside</p>

Example

Let's find the element#testthen find its parent usingparentand if this parentdivwith classwww, then we’ll give it the text'!'by usinghtml:

<p>outside</p> <div class="www"> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').parent('div.www').html('!');

The HTML code will look like this:

<p>outside</p> <div class="www"> ! </div> <p>outside</p>
