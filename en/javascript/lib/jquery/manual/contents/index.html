contents method

Methodcontentsgets the element's descendants, including text and comments.

Syntax

The method does not accept parameters:

.contents();

Methodcontentsworks like a methodchildren, the difference appears when trying to access the text of descendants. Compare the two examples below:

Example

We use the methodchildren:

<p id="test">text1 <b>bold</b> text2</p> $('#test').children().each( function() { console.log($(this).text()); // выведет 'bold' } );

Example

We use the methodcontents:

<p id="test">text1 <b>bold</b> text2</p> $('#test').contents().each( function() { console.log($(this).text()); // выведет 'text1', 'bold', 'text2' } );

See also
