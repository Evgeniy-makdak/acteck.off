filter method

Methodfilterleaves in the set only those elements that satisfy a certain selector.

Syntax

Filtering by selector:

.filter(селектор);

The parameter can also be DOM elements or a jQuery object:

.filter(DOM элемент или объект jQuery);

You can also filter items using a function that is called individually for each of the selected items. If she returnstrue, then the current element will be included in the final result iffalse- not included. The elements themselves are available in the function, in the variablethis, and their serial numbers in the set are in the variableindex:

.filter(function(index, elem))

Example

Let's find all the paragraphs and put the text at the beginning of them'!'by usingprepend, then from the found ones we get only paragraphs with the classwwwby usingfilterand put the text at the end of them'?'by usingappend:

<p>text</p> <p>text</p> <p class="www">text</p> <p class="www">text</p> $('p').prepend('!').filter('.www').append('?');

The HTML code will look like this:

<p>!text</p> <p>!text</p> <p class="www">!text?</p> <p class="www">!text?</p>

Example

Let's use the function to find all paragraphs that have the tagstrongand give them the text'!!!':

<p><strong>text</strong></p> <p><strong>text</strong></p> <p>text</p> <p>text</p> $('p').filter(function() { return $('strong', this).length === 1; }).html('!!!');

The HTML code will look like this:

<p>!!!</p> <p>!!!</p> <p>text</p> <p>text</p>
