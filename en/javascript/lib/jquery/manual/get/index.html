get method

Methodgetgets a set of elements corresponding to a jQuery object.

Syntax

This way we get the entire set of elements in the form of an array:

.get();

You can specify a specific element number in a set and get it. The number can also have a negative value (then the countdown is carried out backwards from the last element):

.get(номер);

Example

Let's get the second paragraph using the methodget, counting starts from0:

<p>text</p> <p class="aaa">text</p> <p>text</p> <p>text</p> alert( $('p').get(1) );

See also

  • methodfind,
    which searches for elements within those already found
  • methodfilter,
    which filters the elements in a set
