fadeToggle method

MethodfadeTogglesmoothly shows or hides elements in turn, animating their transparency.

Syntax

Show/hide for a specified time,400мсdefault:

.fadeToggle(длительность);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.fadeToggle('slow' или 'fast');

If you do not specify parameters, there will be no animation, elements will be shown/hidden instantly:

.fadeToggle();

You can also pass the smoothness function as the second parameter, as well as a callback function as the third - it will work after the animation is completed. Both parameters are optional:

.fadeToggle(длительность, [функция плавности], [callback-функция]);

You can pass various options to the method, in the form of a JavaScript object containing pairskey: value:

.fadeToggle(options);

Such an object can pass the following parameters and functions -duration, easing, queue, specialEasing, step, progress, complete, start, done, fail, always. You can see the description of these parameters for the methodanimate. For example, let's set the duration and smoothness function:

.fadeToggle( {duration: 800, easing: easeInSine} );

Example

In the following example, the first time you click the button#toggleelement with#testwill be hidden, and when repeated, will be shown. For this we use the methodfadeToggle. We will also set the speed to1000мсand the value of the smoothness function inlinear:

<button id="toggle">toggle</button> <div id="test"></div> #test { width: 200px; height: 200px; background: green; color: white; margin-top: 10px; } $('#toggle').on('click', function() { $('#test').fadeToggle(1000, 'linear'); });
