removeData method

MethodremoveDataallows us to remove values ​​set using the methoddata.

Syntax

By calling a method with the name of a key, we will delete the value stored in the object with that key:

.removeData(имя ключа);

We can also pass a list to the method as an array or string, with the key names separated by spaces, to remove multiple values ​​at once:

.removeData(список);

If we do not pass any parameters to the method, we will delete all data:

.removeData();

Example

Let's usedataset the data for the tagpby passing the key name'test'and meaning21and then usingremoveDataLet's remove them again:

<p>text</p> $('p').data('test', 21); // данные записываются console.log($('p').data('test')); // выведет 21 $('p').removeData('test'); // данные для 'test' удалены console.log($('p').data('test')); // выведет undefined
