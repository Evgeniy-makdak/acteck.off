siblings method

Methodsiblingsfinds all neighbors of an element within the same parent.

Syntax

Getting neighbors:

.siblings();

You can also filter neighbors by a given selector (only those neighbors that satisfy this selector will remain in the set):

.siblings(селектор);

Example

Let's find the element#testand we will put a text to all his neighbors'!'using the methodhtml:

<p>outside</p> <div> <p>inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p>inside</p> </div> <p>outside</p> $('#test').siblings().html('!');

The HTML code will look like this:

<p>outside</p> <div> <p>!</p> <p>!</p> <p id="test">inside</p> <p>!</p> <p>!</p> </div> <p>outside</p>

Example

Let's find the element#testand all his neighbors with classwwwput the text'!'using the methodhtml:

<p>outside</p> <div> <p>inside</p> <p class="www">inside</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p class="www">inside</p> <p>inside</p> </div> <p>outside</p> $('#test').siblings('.www').html('!');

The HTML code will look like this:

<p>outside</p> <div> <p>inside</p> <p class="www">!</p> <p>inside</p> <p id="test">inside</p> <p>inside</p> <p class="www">!</p> <p>inside</p> </div> <p>outside</p>
