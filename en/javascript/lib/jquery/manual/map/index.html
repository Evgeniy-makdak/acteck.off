map method

MethodmapCalls the given function for each element of the set. It is especially useful for getting or setting the values ​​of a collection of elements.Methodmapreturns a newly formed set of various elements and objects, unlike a methodeach, which returns the original set of elements.

Syntax

The elements themselves are available in the function, in the variablethis, and their serial numbers in the set are in the variableindex:

$(селектор).map(function(index, element));

Example

Let's display a list of namesidelements separated by commas. We also use methodsgetAndjoin:

<p id="aaa">text1</p> <p id="bbb">text2</p> <p id="ccc">text3</p> console.log( $('p').map( function() { return this.id; } ).get().join() );
