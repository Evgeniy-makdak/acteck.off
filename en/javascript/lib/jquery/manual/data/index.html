data method

Methoddataallows you to bind data of any type to DOM elements in a safe way, which avoids memory leaks. We can set several different values ​​for an element and retrieve them later. Using the methoddatato update data does not affect attributes in the DOM.

Syntax

This is how we set the value for some element and the key by which it can be found later:

.data(ключ, значение);

We can also pass an object that contains key-value pairs to update the data:

.data(объект);

You can get the data associated with the first element in a jQuery collection by passing only the key:

.data(ключ);

Without passing any parameters, we can read data previously associated with DOM elements. The method will return a JavaScript object containing each value as a property:

.data();

Example

Let's usedataset the data for the tagp, passing as the first parameter'test'and meaning21for the second one. Then we read the data stored with the key'test':

<p>text</p> $('p').data('test', 21); // здесь данные записываются console.log($('p').data('test')); // выведет число 21 console.log($('p').data()); // выведет объект {test: 21}
