outerHeight method

MethodouterHeightallows you to get and change the height of an element, taking into account the internal padding, border and optionally the external padding of the element. It's important to remember that we will be getting the 'calculated value' of the height (computed height). Calculation errors can also occur if the user resizes the page, or if an element or its parent is hidden.

Syntax

This way we can get the height of the element. This method can take an optional parameter of the formtrueorfalse (falseis the default), depending on whether margins will be taken into account or not. In some cases, the resulting values ​​may be fractional:

$(селектор).outerWidth([включая_margin]);

To change the width of an element, you can simply pass a number (for example400), then the units of measurement will be pixels, or a string indicating the units of measurement (for example'10em'):

$(селектор).outerHeight(новое значение, [включая_margin]);

We can also apply a given function to each element in the set. In this case, the first parameter the function will receive is the number of the element in the set, and the second parameter will be the current value of the specified height for a specific element.thisinside the function will point to the current element. The element width value will change to the one returned by the function:

$(селектор).outerHeight(function(номер в наборе, текущее значение высоты));

Example

Let's display paragraph height information for comparison.#test, obtained by methodsheight, innerHeightAndouterHeight:

<p id="test">text</p> <p id="out1"></p> <p id="out2"></p> <p id="out3"></p> <p id="out4"></p> p { margin: 10px; padding: 5px; border: 2px solid blue; } let h1 = $('#test').height(); let h2 = $('#test').innerHeight(); let h3 = $('#test').outerHeight(); let h4 = $('#test').outerHeight(true); $('#out1').text(h1); $('#out2').text(h2); $('#out3').text(h3); $('#out4').text(h4);
