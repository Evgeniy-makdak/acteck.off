unwrap method

Methodunwrapperforms the inverse operation of a methodwrap.

Syntax

So we will use the method:

.unwrap();

You can also pass an optional parameter to the method in the form of a string, as a selector; if the parent of the element does not match the specified selector, then the method will not be applied to it:

.unwrap(селектор);

Example

Let's do itunwrapfor all paragraphs with classwww:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p>text</p></div> $('.www').unwrap();

The HTML code will look like this:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <div><p>text</p></div>
