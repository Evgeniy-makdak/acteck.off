Метод unwrap

Метод unwrap осуществляет операцию, обратную методу wrap.

Синтаксис

Так мы воспользуемся методом:

.unwrap();

Также в метод можно передать необязательный параметр в виде строки, в качестве селектора, если родитель элемента не подходит заданному селектору, то метод к нему применяться не будет:

.unwrap(селектор);

Пример

Давайте выполним unwrap для всех абзацев с классом www:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p>text</p></div> $('.www').unwrap();

HTML код станет выглядеть так:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <div><p>text</p></div>
