Val method

Methodvalallows you to get and change the current value of an element.

Syntax

This way we can get the current value of the first element in the set:

$(селектор).val();

MethodvalMainly used to get the values ​​of form elements such as:input, selectAndtextarea. If called on an empty collection, it will returnundefined. When working withselectwith attribute settingmultiple, the method will return an array of values ​​for each selectedoption. If none of them are selected, then an empty array will be returned. This way we can set the value of each element in the set. You can pass a string, array or number as a parameter:

$(селектор).val(значение);

Apply a function to each element in a set. The function takes the current number in the set as its first parameter, and its current value as its second parameter:

$(селектор).val(имя атрибута, function(номер в наборе, текущее значение));

Example

In the following example, let's get the values ​​entered into the input and output them below in a paragraph:

<input type="text" value="text"> <p></p> p { color: green; margin: 8px; } $('input').keyup(function() { let value = $(this).val(); $('p').text(value); }).keyup();

Example

And now using the methodvalwe will record the values ​​of the buttons that we will press in the input below:

<div> <button>one</button> <button>two</button> <button>three</button> <button>four</button> </div> <input type="text" value="click buttons"> button { margin: 4px; cursor: pointer; } input { margin: 4px; color: green; } $('button').click(function() { let text = $(this).text(); $('input').val(text); });
