prop method

Methodpropallows you to get and change a property of an element. It is important not to confuse element properties and attributes. When creating a DOM tree, attributes are translated into their corresponding DOM element properties. Their names do not always coincide, nor do their meanings. Some properties may also not have attributes corresponding to them, these can only be obtained and set by using the methodprop, for example propertytagName. To get the attributes of an element, use the methodattr.

Syntax

Getting the property value:

$(селектор).prop(имя свойства);

Changing the property value:

$(селектор).prop(имя свойства, новое значение);

Changing the values ​​of several properties:

$(селектор).prop({первое_свойство: новое_значение, второе_свойство: новое_значение, ...});

Applying a function to each element in a set:

$(селектор).prop(имя свойства, function(номер в наборе, текущее значение свойства));

Example

In the following example we will get the paragraph by itsidand display the value of its propertyclassName:

<p id="test" class="aaa">text</p> let value = $('#test').prop('className'); console.log(value);

Example

Let's change the properties forclassNameAnddisabled:

<input type="checkbox" class="aaa" id="test" /> $("#test").prop({ className: 'bbb', disabled: 'true' });
