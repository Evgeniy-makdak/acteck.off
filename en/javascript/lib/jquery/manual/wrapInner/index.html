wrapInner method

MethodwrapInnerwraps the content inside an element in the given tag.

Syntax

The method parameter can be text (possibly with tags), a DOM element, or a jQuery object:

.wrapInner(text);

This is how the selected elements are wrapped with content that will be returned as html text by the custom function. The function is called separately for each of the selected elements:

.wrapInner(function(index));

Example

Let's wrap paragraphs with classwwwto tagb:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrapInner('<b></b>');

The HTML code will look like this:

<p class="www"><b>text</b></p> <p class="www"><b>text</b></p> <p class="www"><b>text</b></p> <p>text</p>
