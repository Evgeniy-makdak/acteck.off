jQuery.hasData Method

MethodjQuery.hasDatachecks whether an element has data set using the methoddata. If there is no data object associated with this element, then the method will returnfalse, otherwisetrue.

Syntax

This is how we check if the DOM element has data:

jQuery.hasData(DOM элемент);

Example

Let's usedataset the data for the tagp, and check whether the data is set for it:

<p>text</p> let $p = jQuery('p'), p = $p[0]; console.log(jQuery.hasData(p)); // выведет false $('p').data('test', 21); console.log(jQuery.hasData(p)); // выведет true
