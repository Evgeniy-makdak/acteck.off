clone method

Methodclonecreates deep copies of selected page elements and returns them as a jQuery object. The elements are copied along with all the elements they contain. Optional parameters control whether event handlers bound to these elements should be copied (defaultfalse- do not copy).

Syntax

You can pass two optional parameters to the method, which can take valuestrueorfalse (falsedefault). For the first parameter in the valuetrue- event handlers and data for the element will also be copied, whenfalse- No. For the second parameter in the valuetrue- event handlers and data for all descendants of the cloned element will also be copied, whenfalse- No.

.clone([флаг1], [флаг2]);

Example

Let's insert another one after the paragraph using the methodcloneand methodinsertAfter:

<p class="www">text</p> $('.www').clone().insertAfter('.www');
