each method

MethodeachCalls the given function for each element of the set. Unlike a similar methodmap, which returns a newly generated set of various elements and objects, methodeachreturns the original set of elements.

Syntax

The elements themselves are available in the function, in the variablethis, and their serial numbers in the set are in the variableindex:

$(селектор).each(function(index, elem));

Example

Let's display the contents of each paragraph sequentially using the methodseachAndtext:

<p>text1</p> <p>text2</p> <p>text3</p> $('.www').each( function() { console.log($(this).text()); // выведет 'text1', 'text2' и 'text3' } );
