insertAfter method

MethodinsertAfteradds text after the given element. There is also a methodafter, which works in a similar way.

Syntax

Inserting text after an element:

$(текст).insertAfter(селектор);

The content can be not only plain text, but also a DOM element, selector, array of elements, string or jQuery object.

Example

Let's insert tagged text after the given paragraph:

<p id="test">абзац</p> $('<p>!!!</p>').insertAfter('#test');

The HTML code will look like this:

<p id="test">абзац</p><p>!!!</p>

See also

  • methodafter,
    which adds text after a given element
  • methodsinsertBefore, appendTo, prependTo,
    allowing you to add content to a specific location on the page
  • methodclone,
    which creates copies of the selected elements
