wrap method

Methodwrapwraps elements in the given tag.

Syntax

The method parameter can be text (possibly with tags), a DOM element, or a jQuery object:

.wrap(text or tag);

This is how the selected elements are wrapped with the content that will be returned by the user function. The function is called separately, for each of the selected elements, taking the position of the element in the set as a parameter:

.wrap(function(номер в наборе));

Example

Wrap all paragraphs with classwwwtagdiv:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrap(document.createElement('div'));

The HTML code will look like this:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <p>text</p>

Example

As a parameter, you can pass not only the tag name, but also the following construction:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrap('<div></div>');

The HTML code will look like this:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <p>text</p>

Example

You can write any attributes into the opening tag, and the wrapping will be along with these attributes. Let's wrap paragraphs with classwwwdiva with classzzz:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrap('<div class="zzz"></div>');

The HTML code will look like this:

<div class="zzz"><p class="www">text</p></div> <div class="zzz"><p class="www">text</p></div> <div class="zzz"><p class="www">text</p></div> <p>text</p>
