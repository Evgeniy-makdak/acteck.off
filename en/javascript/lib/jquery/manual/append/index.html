append method

Methodappendadds text to the end of the element. There is also a methodappendTo, which works in a similar way.

Syntax

Inserting text at the end of an element:

$(селектор).append(текст);

You can add text to the end of the selected elements, which will be returned by the custom function. The function is called separately for each of the selected elements. When called, the following parameters are passed to it: the position of the element in the set, the current contents of the element:

$(селектор).append(function(номер в наборе, текущее содержимое элемента));

The content can also be not only plain text, but also a DOM element or a jQuery object. In this case, these elements will move from their position in the HTML code.

Example

Let's insert text at the end of the given paragraph:

<p id="test">text</p> $('#test').append('!!!');

The HTML code will look like this:

<p id="test">text!!!</p>

Example

Let's insert tagged text at the end of a given paragraph:

<p id="test">text</p> $('#test').append('<b>!!!</b>');

The HTML code will look like this:

<p id="test">text<b>!!!</b></p>

Example

Let's find all the paragraphs and at the end of each of them put its serial number in the set:

<p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> $('p').append(function(index, text){return index;});

The HTML code will look like this:

<p>text0</p> <p>text1</p> <p>text2</p> <p>text3</p> <p>text4</p>
