end method

Methodendreturns the previous set of elements in the current method chain.

Syntax

.end();

Example

In the following example, we will find all paragraphs, then select among those found paragraphs with the classwwwby usingfilterand add text to the end'!'using the methodappend, then go back to the previous set (this is the set$('p')), usingend, and add text to the end'?'. It turns out that paragraphs with a classwwwthere will be text'!?', and for regular paragraphs - text'?':

<p class="www">text</p> <p class="www">text</p> <p>text</p> <p>text</p> $('p').filter('.www').append('!').end().append('?');

The HTML code will look like this:

<p class="www">text!?</p> <p class="www">text!?</p> <p>text?</p> <p>text?</p>
