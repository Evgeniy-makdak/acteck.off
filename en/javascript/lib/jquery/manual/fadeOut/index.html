fadeOut method

MethodfadeOutsmoothly hides elements. You can show elements using the methodfadeIn, making them opaque.

Syntax

Display for a specified time,400мсdefault:

.fadeOut(длительность);

Time can be set not only in milliseconds, but also with keywordsslow (600мс) Andfast (200мс), the higher the value, the slower the animation:

.fadeOut('slow' или 'fast');

If you do not specify the parameters, there will be no animation, the elements will be hidden instantly:

.fadeOut();

You can also pass the smoothness function as the second parameter, as well as a callback function as the third - it will work after the animation is completed. Both parameters are optional:

.fadeOut(длительность, [функция плавности], [callback-функция]);

You can pass various options to the method, in the form of a JavaScript object containing pairskey: value:

.fadeOut(options);

Such an object can pass the following parameters and functions -duration, easing, queue, specialEasing, step, progress, complete, start, done, fail, always. You can see the description of these parameters for the methodanimate. For example, let's set the duration and smoothness function:

.fadeOut( {duration: 800, easing: easeInSine} );

Example

In the following example, when you click on a button#hideelement with#testwill be hidden using the methodfadeOut, and to the button#show- show up, with the helpfadeIn. We will also set the speed to1000мс:

<button id="hide">hide</button> <button id="show">show</button> <div id="test"></div> #test { width: 200px; height: 200px; background: green; color: white; margin-top: 10px; } $('#show').on('click', function() { $('#test').fadeIn(1000); }); $('#hide').on('click', function() { $('#test').fadeOut(1000); });

Example

And in the following example, after the end of the animation when showing the element#testwe will display a message with the text'!', and when hidden with text'?':

<button id="hide">hide</button> <button id="show">show</button> <div id="test"></div> #test { width: 200px; height: 200px; background: green; color: white; margin-top: 10px; } $('#show').on('click', function() { $('#test').fadeIn(1000, function() { alert('!'); }); }); $('#hide').on('click', function() { $('#test').fadeOut(1000, function() { alert('?'); }); });
