removeProp method

MethodremovePropremoves element properties. Note: use this method to remove artificially introduced properties, you should not remove native properties of elements (i.e. the original, native properties of elements), this may cause browser errors. In particular, do not useremovePropto remove attributes such ascheckedAnddisabled. When these attributes are missing ininput-elements, their properties of the same name cease to exist, they simply take on a valuefalse. So to disable these properties you need to set them tofalseusing the methodprop.

Syntax

$(селектор).removeProp(имя свойства);

Example

Let's set the propertytextwith meaningaaaat the paragraph and display it in the text of the paragraph:

<p></p> elem = $('p'); elem.prop('text', 'aaa').html(elem.prop('text'));

The HTML code will look like this:

<p>aaa</p>

Let's now remove this property:

<p></p> elem = $('p'); elem .prop('text', 'aaa') .html(elem.prop('text')) .removeProp('text') .html(elem.prop('text'));
