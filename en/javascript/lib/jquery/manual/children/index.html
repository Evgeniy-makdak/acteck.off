children method

Methodchildrenreturns only the element's immediate descendants; does not return text nodes. To return all children, including text and comments, use the methodcontents.

Syntax

Getting immediate descendants:

.children();

You can also filter descendants by a given selector:

.children(селектор);

Example

Let's find all immediate children of a tagdivand at the end we will add them'!'using the methodappend:

<div> <p>text</p> <p>text</p> <p>text</p> </div> $('div').children().append('!');

The HTML code will look like this:

<div> <p>text!</p> <p>text!</p> <p>text!</p> </div>

Example

Let's now find all immediate children with classwww:

<div> <p class="www">text</p> <p class="www">text</p> <p>text</p> </div> $('div').children('.www').append('!');

The HTML code will look like this:

<div> <p class="www">text!</p> <p class="www">text!</p> <p>text</p> </div>

See also
