parents method

Methodparentsgets all the parents of the element - not only the immediate one, but also the grandparent, great-grandparent, and so on until the root element (that is, until the taghtml).

Syntax

Getting all parents for elements:

.parents();

You can also select parents only if they satisfy the given selector:

.parents(селектор);

Example

Let's find the element#test, then find all his parents usingparentsand if these are parentsdivwith classwww, then we will put the text at the beginning'!'by usingprepend:

<div> <div class="www"> <div class="www"> <p id="test">text</p> </div> </div> </div> $('#test').parents('div.www').prepend('!');

The HTML code will look like this:

<div> <div class="www"> ! <div class="www"> !<p id="test">text</p> </div> </div> </div>
