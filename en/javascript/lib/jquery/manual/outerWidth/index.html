outerWidth method

MethodouterWidthallows you to get and change the width of an element taking into account the internal padding, border and optionally the external padding of the element. It's important to remember that we will be getting the 'calculated value' of the width (computed width). Calculation errors can also occur if the user resizes the page, or if an element or its parent is hidden.

Syntax

This way we can get the width of the element. This method can take an optional parameter of the formtrueorfalse (falseis the default), depending on whether margins will be taken into account or not. In some cases, the resulting values ​​may be fractional:

$(селектор).outerWidth([включая_margin]);

To change the width of an element, you can simply pass a number (for example400), then the units of measurement will be pixels, or a string indicating the units of measurement (for example'10em'):

$(селектор).outerWidth(новое значение, [включая_margin]);

We can also apply a given function to each element in the set. In this case, the first parameter the function will receive is the number of the element in the set, and the second parameter will be the current value of the specified width for a specific element.thisinside the function will point to the current element. The element width value will change to the one returned by the function:

$(селектор).outerWidth(function(номер в наборе, текущее значение ширины));

Example

Let's display information about paragraph width for comparison#test, obtained by methodswidth, innerWidthAndouterWidth:

<p id="test">text</p> <p id="out1"></p> <p id="out2"></p> <p id="out3"></p> <p id="out4"></p> p { margin: 10px; padding: 5px; border: 2px solid blue; } let w1 = $('#test').width(); let w2 = $('#test').innerWidth(); let w3 = $('#test').outerWidth(false); let w4 = $('#test').outerWidth(true); $('#out1').text(w1); $('#out2').text(w2); $('#out3').text(w3); $('#out4').text(w4);
