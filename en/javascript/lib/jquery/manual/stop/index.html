stop method

Methodstopallows us to immediately stop a running animation. The animation will continue from the next function without finishing the previous one.

Syntax

This is how we stop the running animation. You can pass two optional parameters that accept Boolean values. If the first parameter is set astrue, then the remaining functions in the animation queue will be deleted and will never run. If the second parameter is set totrue, then when the animation stops, the element’s CSS properties will immediately take on their final values ​​(i.e., if the goal was to smoothly hide the element, then it will be hidden instantly):

.stop([clearQueue], [jumpToEnd]);

You can also pass an optional parameter specifying the name of the queue:

.stop([имя очереди], [clearQueue],[jumpToEnd]);

Animation can be disabled globally using the settingjQuery.fx.off = true, which sets the duration value to 0.

Example

Let's make sure that every time the button is clicked#toggleour rectangle was either expanded or collapsed using the methodslideToggle. Press the button without waiting for the animation to complete - the next stage will start without waiting for the previous one to complete, thanks to thestop:

<button id="toggle">toggle</button> <div id="test"></div> #test { background-color: green; border: 1px solid black; width: 200px; height: 100px; margin: 10px; } $('#toggle').on('click', function() { $('#test').stop().slideToggle(1500); });
