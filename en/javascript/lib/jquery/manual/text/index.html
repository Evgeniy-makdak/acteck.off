text method

Methodtextallows you to get the text of elements, including its descendants, and change it.

Syntax

Receiving text:

$(селектор).text();

Changing the text:

$(селектор).text(новый текст);

Replaces the contents of the selected elements with the value returned by the user-defined function. The function is called separately for each of the selected elements. When called, the following parameters are passed to it:index— the position of the element in the set,value— current text content:

$(селектор).text(function(номер в наборе, текущее содержимое элемента));

Example

Let's display the contents of our paragraph:

<p id="test">text</p> let text = $('#test').text(); alert(text);
