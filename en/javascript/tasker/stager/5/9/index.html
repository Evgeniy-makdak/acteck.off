Level 5.9 of the JavaScript problem book

Given an array with numbers, a button and an input. By clicking the button, write all the numbers from the array into the input, separating them with commas.

Paragraphs with text are given. Make sure that when you click on any paragraph, its text turns red.

Input given. A number is entered into it. If an odd number is entered, then color the input border red, and if an even number, then color it green.

The date is entered into the input in the format2025-12-31. When you lose focus, make sure that the input contains a date in the format35.12.2025.

Given an empty listuland textarea. It contains text with punctuation marks. When you lose focus, write each sentence from the text into your own tagli.
