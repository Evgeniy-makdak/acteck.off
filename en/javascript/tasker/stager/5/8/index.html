Level 5.8 of the JavaScript problem book

Inputs are given. Numbers are entered into them. Make it so that when you lose focus in any of the inputs, the square of the number entered into it appears in it.

Given an array, a button and an input. Make sure that when you click on a button, a random array element is displayed in the input.

Given an input and a paragraph. Numbers are entered into it separated by commas. When you lose focus, find the sum of the entered numbers and print it in a paragraph.

Two inputs and a button are given. Numbers are entered into the inputs. By clicking the button, you will receive an array of common divisors of these numbers and output it to the console.

An input and a button are given. Numbers are entered into the input separated by commas. By clicking the button, check if the number entered is13.

An input and a button are given. A six-digit number is entered into it. By clicking on the button, check that the number represents a lucky ticket, that is, the sum of the first three digits is equal to the sum of the second three digits.
