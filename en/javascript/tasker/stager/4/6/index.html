Level 4.6 of the JavaScript problem book

Make a function that will take a date in the format as a parameteryear-month-day, and determine whether such a date exists or not.

Make a function that will generate a string of a given length, filled with random Latin letters.

Make a function that takes a string of words as a parameter and returns a string in upper case, consisting of the first letters of the words.

Make a function that takes an array of numbers as a parameter and replaces each number with an array of its divisors.

Make a function that takes seconds as a parameter and returns the number of days, hours, minutes and seconds corresponding to these seconds, as the following object:

{ d: 12, h: 10, m: 59, s: 59, }
