Level 4.7 of the JavaScript problem book

Make a function that will take text with words as a parameter and return text in which these words will be sorted in alphabetical order.

Make a function that takes two arrays as a parameter and returns an array of their common elements.

Make a function that will return a random number. A function should not return the same number twice in a row.

Make a function that takes an array and an element as a parameter and returns the next element. See example:

let arr = [1, 2, 3, 4, 5]; func(arr, 1); // 2 func(arr, 4); // 5 func(arr, 5); // 1
