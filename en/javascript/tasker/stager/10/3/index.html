Level 10.3 of the JavaScript problem book

Make an input in which you can enter only numbers. The remaining characters should not be entered into the input.

Given a block. By clicking on the block, show a tooltip with some fairly long text. Show the tooltip either above or below, taking into account that it fits entirely on the screen and does not disappear under scrolling.

Given an array of six digits. See if you can collect a lucky ticket from these numbers.

Given a rectangular table. By pressing the button, color the cells of one diagonal red and the cells of the second diagonal green.

Write a code that will generate a password of a given size. The password must contain small and capital letters, a number and some special character.

A browser window is given. Make sure that the elements entirely located in the left half of the window are painted green, those entirely located in the right half of the window are painted blue, and those located in both of them are painted red.

Two numbers are given. Display the process of dividing these numbers into a column, just like in school.

Given a string containing two numbers and a mathematical operation between them:

'10 + 20'

Write code that calculates the result of a written mathematical operation.
