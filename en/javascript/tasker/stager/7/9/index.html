Level 7.9 of the JavaScript problem book

A paragraph and an input are given. In a paragraph, words are written separated by spaces. A word is entered into the input. If you lose focus, check if there is such a word in the paragraph, and if not, add it.

Write a program that will generate the following string:

'-1-12-23-34-45-'

Write a program that will generate the following array:

[ 12, 34, 56, 78, 90, ]

The following layout is given:

<a href="1.html">111</a> <a href="2.html">222</a> <a href="3.html">222</a>

Get the contents of addresses and link texts in the form of the following structure:

[ { text: '111', href: '1.html', }, { text: '222', href: '2.html', }, { text: '333', href: '3.html', }, ]
