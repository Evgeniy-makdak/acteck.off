Level 2.6 of the JavaScript problem book

Given a string with letters and numbers. Get an array of positions of all digits from this string.

Given an array with some numbers, for example, like this:

[123, 456, 789]

Write code that will reverse the numbers in this array using the following principle:

[321, 654, 987]

Given a string with a number:

'1234567'

Separate triplets of digits with spaces, starting at the end of the number. In our case it should look like this:

'1 234 567'

Given some string:

'AbCdE'

Change the case of the letters in this string to the opposite case. In our case it should look like this:

'aBcDe'

Given some array with numbers, for example, like this:

[1, 2, 3, 4, 5, 6]

Merge pairs of elements together:

[12, 34, 56]

Given a line with the words:

'aaa bbb ccc eee fff'

Capitalize the first character of every second word on this line. In our case it should look like this:

'aaa Bbb ccc Eee fff'
