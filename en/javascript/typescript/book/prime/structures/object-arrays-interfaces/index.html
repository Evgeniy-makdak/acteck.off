Interface as an object type with a TypeScript array

You can also set an interface as the type for the entire object.

Let's look at an example. First, let's create an interface for the product:

interface Product { name: string, colors: string[] };

Then we create an object, specifying its type as the corresponding interface:

let prod: Product = { name: 'notebook', colors: ['red', 'black', 'gray'] }
