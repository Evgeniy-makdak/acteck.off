Interfaces in TypeScript

A more advanced option for typing objects in TypeScript isinterfaces. They allow you to create new data types that describe the structure of objects.

Interfaces are created using a keywordinterface, followed by the name of the interface (written with a capital letter), and then the structure of the object is described in curly braces.

As an example, let's make an interface that describes the structure of an object with a user:

interface User { name: string, age: number }

Let's now create an object that implements this interface. To do this, we specify the name of our interface as the object type:

let user: User = {name: 'john', age: 30};

Create three objects that implement the following interface:

interface Employee { name: string, age: number, salaryday: number }

Create three objects that implement the following interface:

interface Time { hour: number, minute: number, second: number }
