Complex objects in TypeScript

Objects can have any nesting structure within them, and this structure can be described when declaring an object or in an interface.

For example, let's create the following interface that describes the user:

interface User { name: string, age: number, parents: { mother: string, father: string } }

Let's create an object that implements this interface:

let user: User = { name: 'john', age: 30, parents: { mother: 'jane', father: 'eric' } }

Given object:

let event = { name: 'my new event', time: { start: '2025-11-01', finish: '2025-12-31' } };

Make an interface that describes the structure of this object.

Given object:

let employee { name: 'andrew', potision: { name: 'programmer' salary: 1000, } addr: { country: 'belarus', city: 'minsk' } };

Make an interface that describes the structure of this object.
