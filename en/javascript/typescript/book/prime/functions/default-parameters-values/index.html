Default Parameter Values ​​in TypeScript

Optional parameters can also be assigned a default value. In this case, a question mark is not written to indicate that the parameter is optional. For example, let's assign the value to those users who do not have a last name'snow':

function func(first: string, last: string = 'snow') { return first + ' ' + last; }

Let's now use our function. Let's call it with two parameters:

func('john', 'smit'); // вернет 'john smit'

Let's call it with one parameter:

func('john'); // вернет 'john snow'

Make a function that will raise a number to a given power. Let the function take a number as its first parameter, and a degree as its second. Let the second parameter be optional, and by default the function raises the number to the second power.
