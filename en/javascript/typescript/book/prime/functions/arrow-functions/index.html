Стрелочные функции в TypeScript

В TypeScript также можно делать стрелочные функции. Давайте посмотрим на примере. Пусть у нас есть следующая функция:

let func = function(num: number): number { return num ** 2; }

Перепишем эту функцию на стрелочный вариант:

let func = (num: number): number => num ** 2;

Переделайте следующую функцию на стрелочную:

let func = function(num1: number, num2: number): number { return num1 + num2; }

Переделайте следующую функцию на стрелочную:

let func = function(str: string): string[] { return str.split(''); }

Перепишите следующий JavaScript код на TypeScript вариант:

let arr = [1, 2, 3]; let res = arr.map(num => num ** 2); console.log(res);
