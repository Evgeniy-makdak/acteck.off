Tuples in TypeScript

Sometimes we may need to store an array of values ​​of different types. TypeScript provides us with a data type calledmotorcade (tuple). A tuple is an array, each element of which has its own hard-coded type. And the array itself has a strictly defined length.

For example, let's create a tuple in which we will store the user's name and age. In the first element of the tuple we will have a name and it will be a string, and in the second element we will have an age and it will be a number. Let's declare the described tuple:

let user: [string, number];

Let's fill our tuple with data when declaring:

let user: [string, number] = ['john', 31];

Let's print out the elements of our tuple:

console.log(user[0]); // 'john' console.log(user[1]); // 31
