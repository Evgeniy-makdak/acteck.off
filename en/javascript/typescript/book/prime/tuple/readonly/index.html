Tuple to read in TypeScript

You can create read-only tuples whose elements cannot be modified. To do this, the keyword is specified before the tuple typereadonly:

let user: readonly [string, number] = ['john', 31];

An attempt to modify such a tuple will result in an error:

user[0] = 'eric'; // ошибка

Tell us what the result of running the following code will be:

let time: readonly [number, number, number] = [12, 59, 59]; time[0] = 13; console.log(time);
