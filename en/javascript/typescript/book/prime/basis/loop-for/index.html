for loop in TypeScript

In a loopforyou need to specify the counter type:

for (let i: number = 0; i <= 10; i++) { console.log(i); }

Rewrite the following code via TypeScript:

let res = 0; for (let i = 0; i < 100; i++) { res += i; } console.log(res);
