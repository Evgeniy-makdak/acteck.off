Type aliases in TypeScript

In TypeScript you can createpseudonymstypes. This is done using the operatortype.

For example, let's give another name for the string data type:

type str = string;

Let's declare a variable using our new type:

let test: str = 'abc';

In general, renaming standard types does not provide any practical benefit. Let's look at a more useful example of using aliases.

Application

Let's create a new data type using type union:

type stumber = string | number;

Let's declare a variable with our new type:

let test: stumber;

Let's write a number in it:

test = 123;

Let's write the line in it:

test = 'abc';
