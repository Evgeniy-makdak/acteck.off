Operations on data in TypeScript

TypeScript, unlike other strongly typed languages, allows you to perform operations on different data types without converting them to one common type.

Let's look at an example to see what this means. Let us have a string and a numeric variable:

let test1: number = 123; let test2: string = 'abc';

Let's add these variables:

console.log(test1 + test2); // выведет '123abc'

That is, you can add, for example, strings and numbers, and this will not lead to an error. That is, TypeScript only makes sure that the programmer does not change the data type of the variable.

For example, in the following code we will try to write a string to a numeric variable and this will already result in an error:

let test1: number = 123; let test2: string = 'abc'; let test3: number; test3 = test1 + test2; // пытаемся записать строку

let test1: number = 123; let test2: number = 456; console.log(test1 + test2);
