String literal type in TypeScript

Using the type union operator, you can combine not only built-in types, but also strings. For example, let's make sure that the variable can only take one of two string values:

let str: 'success' | 'error';

Let's check it out. Let's write a variable in one of the lines:

str = 'success';

Let's write the second one:

str = 'error';

But trying to write a different string will result in an error:

str = 'eee'; // ошибка

Make the variable take one of three values:'error', 'warning'or'success'.
