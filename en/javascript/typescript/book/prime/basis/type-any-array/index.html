Type any in arrays in TypeScript

Using typeanyYou can declare an array containing values ​​of an arbitrary type:

let arr: any[] = ['abs', true, 3];

Rewrite the following code via TypeScript:

let arr = [1, '2', '3', 4, 5]; console.log(arr);
