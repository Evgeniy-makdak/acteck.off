Data Types in TypeScript

In TypeScript, when declaring a variable, we are required to indicate its type. To do this, after the variable name we must write a colon and then indicate the desired type.

For example, let's make a string variable:

let test: string = 'abc';

And now the numbers:

let test: number = 123;

And now the logical one:

let test: boolean = true;

It is not necessary to set the value of a variable immediately when declaring it. You can first declare a variable and then assign a value to it later in the code. Example:

let test: string; test = 'abc';

Make three variables: one string, one numeric and one boolean.
