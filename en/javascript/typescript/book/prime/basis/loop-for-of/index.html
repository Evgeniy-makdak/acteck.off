for-of loop in TypeScript

If we need to iterate through the elements of an array, we use a loopfor-of. However, there is one caveat in TypeScript - the type of the variable for the element in this loop is not specified, since TypeScript is able to calculate it automatically:

let arr: number[] = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); }

Rewrite the following code via TypeScript:

let arr = [1, 2, 3, 4, 5]; let res = 0; for (let elem of arr) { res += elem; } console.log(res);
