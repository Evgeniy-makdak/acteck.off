for-in loop in TypeScript

We can iterate through the elements of an object using a loopfor-in. In this case, the type of the variable for the key is not specified:

let obj = {a: 1, b: 2, c: 3}; for (let key in obj) { console.log(key); }

Rewrite the following code via TypeScript:

let obj = {a: 1, b: 2, c: 3}; let res = 0; for (let key in obj) { let elem = obj[key]; res += elem; } console.log(res);
