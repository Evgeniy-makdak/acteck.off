Indexable type for objects in TypeScript

Let's now make an indexable type for the object. To do this, let's create an interfaceIObject. Let's specify that the object key will be a string and the value will be a number:

interface IObject { [index: string]: number; }

Let's set our variable's value as an object with string keys and numeric values:

let obj: IObject = {'a': 1, 'b': 2, 'c': 3};
