Indexable type for arrays in TypeScript

In TypeScript, you can make interfaces that will specify the type of key-value pair in arrays and objects. This type is calledindexable.

Let's see in practice. Let's create an interfaceIArray, which will specify the key-element pair for the array. Let's indicate that the key of the array will be a number (here we have no other options), and the value will be a string:

interface IArray { [index: number]: string; }

Now let's declare the variablearrwith our indexed type:

let arr: IArray;

Let's set our variable's value as an array of strings:

let arr: IArray = ['a', 'b', 'c'];

Now, if you make numbers as array elements, TypeScript will throw an error:

let arr: IArray = [1, 2, 3]; // ошибка
