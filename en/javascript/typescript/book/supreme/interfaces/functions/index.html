Interface for a function in TypeScript

An interface can be made not only for an object, but also for a function. To do this, indicate in the body of the interfacecall signaturefunctions: parameters for the function and their types, as well as the type of the function's result.

Consider the following example. Let's make a type using an interfaceIMathFunc. We indicate two numeric parameters in parentheses. Let's set the return value to a Boolean type:

interface IMathFunc { (num1: number, num2: number): boolean; }

Now let's make a functionmyFuncbased on our interface:

let myFunc: IMathFunc = function(num1: number, num2: number): boolean { if(num1 == num2) { return true; } else { return false; } } console.log(myFunc(2, 2));
