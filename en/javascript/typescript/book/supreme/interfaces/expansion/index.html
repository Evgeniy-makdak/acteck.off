Extending Interfaces into OOP in TypeScript

Interfaces in TypeScript can inherit from each other. This interaction is calledexpansioninterfaces. Let's look at an example. Let us have an interfaceISize:

interface ISize { height: number; width: number; }

Let's make an interfaceIStyle, which will expandISize:

interface IStyle extends ISize { color: string; }

Now let's create a classFigure, implementing the interfaceIStyle. Our class will need to implement both the property of the interface itself and its parent. Let's do this:

class Figure implements IStyle { height: number; width: number; color: string; constructor(height: number, width: number, color: string) { this.height = height; this.width = width; this.color = color; } }

Let's check the work:

let fig = new Figure(130, 200, 'green'); console.log(fig);
