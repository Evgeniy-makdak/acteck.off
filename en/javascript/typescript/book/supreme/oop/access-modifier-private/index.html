The private modifier in TypeScript

Modifierprivateused in cases where you need to block access to properties and methods outside the class. In this case, even in descendant classes it will not be possible to access these properties.

Let's look at an example. Let's do it in classUserprivate propertyname:

class User { private name: string; constructor(name: string) { this.name = name; } }

Let's create a class object by passing the value of our property to the constructor:

let user: User = new User('john');

Now trying to read this private property outside the class will result in an error:

console.log(user.name); // ошибка

An attempt to write something to this property outside the class will also result in an error:

user.name = 'eric'; // ошибка

But inside the class methods it will be possible to both read and change the value of our private property:

class User { private name: string; constructor(name: string) { this.name = name; } public getName() { return this.name; // прочитаем свойство } public setName(name: string) { this.name = name; // запишем новое значение свойства } }
