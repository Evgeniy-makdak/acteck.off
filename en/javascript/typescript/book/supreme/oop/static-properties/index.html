Static Properties in TypeScript

In TypeScript, you can make special properties that will belong to the class, not the object. Such properties are calledstatic.

Static properties can be called without creating an object, simply by accessing the class in which they are declared. In order to make a property static, you need to write a keyword after the access modifierstatic.

Let's go to classUserset a static propertysalary:

class User { public name: string; public static salary: number = 1000; constructor(name: string) { this.name = name; } }

Now let's access this property without creating an object:

console.log(User.salary); // 1000
