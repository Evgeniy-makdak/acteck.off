Properties of an OOP class in TypeScript

Let's now learn how to declare class properties. In TypeScript, all properties must be declared with their type. Let's see how this is done in practice.

Let's create a classUser. Let's give it the propertyname, indicating that it will be a string:

class User { name: string; }

This code, however, will generate an error because the initial value of the property is not set.

Let's set it:

class User { name: string = 'john'; }

Now let's create an object of our class:

let user: User = new User;

Now let's display the property value:

console.log(user.name); // 'john'

Now let’s change the value of the property after creating the object:

user.name = 'eric';
