Модификатор public в TypeScript

Свойства и методы, объявленные через модификатор public, доступны для чтения и записи снаружи класса. Давайте посмотрим на практике. Сделаем у нашего класса публичное свойство name:

class User { public name: string = ''; }

Создадим объект класса:

let user = new User();

Поменяем значение свойства:

user.name = 'john';

Прочитаем свойство:

console.log(user.name);

Сделайте класс User, который будет содержать публичные свойства с именем и годом рождения пользователя. Сделайте публичный метод getAge, который будет получать возраст пользователя по году его рождения.
