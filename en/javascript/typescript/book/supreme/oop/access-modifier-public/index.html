Public modifier in TypeScript

Properties and methods declared through a modifierpublic, are readable and writable from outside the classroom. Let's see in practice. Let's make our class a public propertyname:

class User { public name: string = ''; }

Let's create a class object:

let user = new User();

Let's change the property value:

user.name = 'john';

Let's read the property:

console.log(user.name);

Make a classUser, which will contain public properties with the user's name and year of birth. Make the method publicgetAge, which will get the user's age based on his year of birth.
