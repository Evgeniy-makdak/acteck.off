Introduction to Generics in TypeScript

The main feature of TypeScript is that all its structural components must be given a type. However, when developing complex programs, combining types is often not enough. And for this purpose they are usedgenerics- special variables that help declare a complex type. OOP interfaces and classes can be used as a generic type.
