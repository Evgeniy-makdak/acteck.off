Using class types in generics in TypeScript

Classes can also be used as a generic type. You just need to set the type of the class using its constructor. Therefore, instead of the parametertype:Twe need to indicatetype: {new(): T;}. Let's make a function that in turn will create a new user from the classUser:

function getUser <T> (type: { new (): T; }): T { return new type(); } class User { constructor() { console.log('You create a new user!'); } } let user: User = getUser(User);

Result of the executed code:

'You create a new user!'
