event.clientY property

Propertyevent.clientYcontains the coordinates of the mouse cursor along the Y axis. There are also properties for determining the coordinatesevent.clientX, event.pageX, event.pageY. Let's see the difference betweenclientX/clientYAndpageX/pageY.

How they workclientXAndclientY: if you have a window1000on1000pixels and the mouse is in the center, thenclientXAndclientYboth will be equal500. If you then scroll the page horizontally or vertically without moving the cursor, the valuesclientXAndclientYwill not change, since they are measured relative to the window, not the document.

How they workpageXAndpageY: if you have a window1000on1000pixels and the cursor is in the center, thenpageXAndpageYwill be equal500. If you then scroll to250pixels down, thenpageYwill become equal750. ThuspageXAndpageYcontain event coordinates taking into account scrolling.

Syntax

event.clientY;

Example

When the mouse moves across the page, we will display its coordinates relative to the browser window (clientXAndclientY):

<div id="elem">0 : 0</div> let elem = document.getElementById('elem'); document.addEventListener('mousemove', function(event) { elem.innerHTML = event.clientX + ' : ' + event.clientY; });
