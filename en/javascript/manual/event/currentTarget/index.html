event.currentTarget property

Propertyevent.currentTargetcontains the element for which the event was assigned. Unlikeevent.target, this is not the deepest tag in which the event occurred, but rather the element to which it was attached. More oftenevent.currentTargetcoincides withthis, but sometimes for some reason this may not be the case. In this case, you needcurrentTarget.

Syntax

event.currentTarget;
