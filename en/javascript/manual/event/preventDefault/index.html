event.preventDefault method

Methodevent.preventDefaultAllows you to override your browser's default actions. For example, make sure that when you click on a link, there is no transition to another page, or when you click on a button, the form is not sent to the server. How to use: just inside a function that is tied to an event, you should call it anywhereevent.preventDefault(), where event is an objectEvent.

Syntax

event.preventDefault();

Example

Let's make sure that clicking on a link does not lead to another page:

<a href="/" id="elem">ссылка</a> let elem = document.querySelector('#elem'); elem.addEventListener('click', function(event) { event.preventDefault(); alert('вы не можете перейти по этой ссылке!'); });
