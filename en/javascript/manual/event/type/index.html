event.type property

Propertyevent.typecontains the type of event that occurred.

Syntax

event.type;

Example

In the following example, a click handler is attached to the element and therefore the event type will be'click':

<button id="elem">button</button> let elem = document.querySelector('#elem'); elem.addEventListener('click', function(event) { console.log(event.type); });

Result of executing the code:

'click'

Example

In the following example, a focus loss handler is attached to the input and therefore the event type will be'blur':

<input id="elem"> let elem = document.querySelector('#elem'); elem.addEventListener('blur', function(event) { console.log(event.type); });

Result of executing the code:

'blur'
