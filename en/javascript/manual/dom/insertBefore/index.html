insertBefore method

MethodinsertBeforeallows you to insert an element before another element. Most often used after creating an element usingcreateElement. The method is applied to the parent of the element before which the insertion will occur.

Syntax

родитель.insertBefore(элемент, перед кем вставить);

Example

Let's create a paragraph and place it before the second paragraph:

<div id="parent"> <p>elem 1</p> <p id="before">elem 2</p> <p>elem 3</p> </div> let parent = document.querySelector('#parent'); let before = document.querySelector('#before'); let p = document.createElement('p'); p.textContent = '!'; parent.insertBefore(p, before);

Result of executing the code:

<div id="parent"> <p>elem 1</p> <p>!</p> <p>elem 2</p> <p>elem 3</p> </div>

Example

Add a paragraph to the beginning of the element#parent. To do this, insert our paragraph before the first child#parent. This child can be found usingfirstElementChild:

<div id="parent"> <p>elem 1</p> <p>elem 2</p> </div> let parent = document.querySelector('#parent'); let p = document.createElement('p'); p.textContent = '!'; parent.insertBefore(p, parent.firstElementChild);

Result of executing the code:

<div id="parent"> <p>!</p> <p>elem 1</p> <p>elem 2</p> </div>

Example

When passing as the second parameternullmethodinsertBeforeworks likeappendChild. At the same time, if the element has no child elements,firstElementChildreturnsnull. Therefore, you can add to the beginning of an element even when it has no child elements:

<div id="parent"></div> let parent = document.querySelector('#parent'); let p = document.createElement('p'); p.textContent = '!'; parent.insertBefore(p, parent.firstChild);

Result of executing the code:

<div id="parent"> <p>!</p> </div>
