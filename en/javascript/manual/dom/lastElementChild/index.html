lastElementChild property

PropertylastElementChildstores the last child element. Child elements are all tags that are directly located inside the block. If the element has no child elements, it is returnednull.

Syntax

элемент.lastElementChild;

Example

Let's get the contents of the last descendant of the element:

<div id="parent"> <p>1</p> <p>2</p> </div> let parent = document.querySelector('#parent'); let text = parent.lastElementChild.textContent; console.log(text);

Result of executing the code:

'2'

Example

And now the element has no child elements and therefore will be displayednull:

<div id="parent"></div> let parent = document.querySelector('#parent'); console.log(parent.lastElementChild);

Result of executing the code:

null
