contains method

Methodcontainsallows you to check whether one element contains another within it. The method parameter passes an element that will be checked to ensure that it is inside the element to which the method was applied.

Syntax

родитель.contains(элемент);

Example

Let's check if the paragraph is#childin the block#parent:

<div id="parent"> <p id="child"></p> </div> let parent = document.querySelector('#parent'); let child = document.querySelector('#child'); let contains = parent.contains(child); console.log(contains);

Result of executing the code:

true

Example

And now there is no passed element in the parent and therefore the method returnsfalse:

<div id="parent"></div> <p id="child"></p> let parent = document.querySelector('#parent'); let child = document.querySelector('#child'); let contains = parent.contains(child); console.log(contains);

Result of executing the code:

false

Example

You can pass the same element on which the method was called, in which case the method will also returntrue:

<div id="parent"></div> let parent = document.querySelector('#parent'); let contains = parent.contains(parent); console.log(contains);

Result of executing the code:

true
