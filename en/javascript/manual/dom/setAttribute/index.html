setAttribute method

MethodsetAttributeallows you to change the value of a given attribute of a tag.

Syntax

элемент.setAttribute(имя атрибута, новое значение);

Example

Let's change the attribute value of the elementvalue:

<input id="elem" value="abcde"> let elem = document.querySelector('#elem'); elem.setAttribute('value', '!!!');

The HTML code will look like this:

<input id="elem" value="!!!">
