children property

Propertychildrenkeeps in itselfpseudo-arraychild elements. Child elements are all tags that are directly located inside the block.

Syntax

элемент.children;

Example

Let's loop through all the element's descendants and print their contents:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> <p>4</p> <p>5</p> </div> let parent = document.querySelector('#parent'); let elems = parent.children; for (let elem of elems) { console.log(elem.textContent); }
