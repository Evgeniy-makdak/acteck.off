addEventListener method

MethodaddEventListenerallows you to assign event handlers to an element. With its help, you can specify, for example, what to do when you click on a button or what to do when typing in a text field. In the first parameter we indicate the type of event to be transmitted, in the second - the function that will be triggered after the event specified in the first parameter. In the third optional parameter we pass the characteristics of the object (once, capture, passive) or parameteruseCapture.

Syntax

элемент.addEventListener('тип события', функция, [характеристики объекта]); или элемент.addEventListener('тип события', функция, [useCapture]);

Example

Let's make sure that when the button is clicked, a message is displayed:

<input type="button" id="button" value="click me"> let button = document.querySelector('#button'); button.addEventListener('click', function() { alert('message'); });

Example

Let's write the code so that when we lose focus in an input, a message is displayed with the text of this input:

<input id="input" value="text"> let input = document.querySelector('#input'); input.addEventListener('blur', function() { alert(this.value); });

:

Example

Let's make sure that when you click on the button, the message is displayed in the console only once. To do this, we will use the second parameter:

<input type="button" id="button" value="click me"> let button = document.querySelector('#button'); button.addEventListener('click', function() { console.log('message'); }, { once: true } );

Example

Parameterpassiveprohibits calling the methodevent.preventDefaultwhen processing an event. Let's apply the method to the previous exampleevent.preventDefault, and also indicate in the parameterpassivemeaningtrue:

<input type="button" id="button" value="click me"> let button = document.querySelector('#button'); button.addEventListener('click', function(event) { event.preventDefault(); console.log('No message'); }, { passive: true } );

After executing the code, we will see the following messages:

'No message'; 'Unable to preventDefault inside passive event listener invocation.'
