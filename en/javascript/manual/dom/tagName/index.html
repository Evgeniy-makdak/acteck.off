tagName property

PropertytagNamecontains the tag name in uppercase (capital letters).

Syntax

элемент.tagName;

Example

Let's get the element#elemand display the name of its tag:

<div id="elem"></div> let elem = document.getElementById('elem'); console.log(elem.tagName);

Result of executing the code:

'DIV'

Example

Let's print the tag name in lowercase. To do this, we will use the methodtoLowerCase:

<div id="elem"></div> let elem = document.getElementById('elem'); console.log(elem.tagName.toLowerCase());

Result of executing the code:

'div'
