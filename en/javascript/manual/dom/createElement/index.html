createElement method

MethodcreateElementallows you to create a new element by passing the tag name as a parameter. After creation, the element can be worked with as with a regular element, and it can also be added to the page using the methodsprepend, append, appendChild, insertBeforeorinsertAdjacentElement.

If you write down the result of the workcreateElementinto a variable, then this variable will contain an element as if we had received it throughquerySelectororgetElementById. The only difference is that our element will not be placed on the page. And so we can change its innerHTML, attributes, attach event handlers and ultimately place it on the page.

Syntax

document.createElement('имя тега');

Example

Let's create a paragraph, set its text and place it on the page at the end of the block#parent:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> </div> let parent = document.querySelector('#parent'); let p = document.createElement('p'); p.textContent = '!'; parent.appendChild(p);

Result of executing the code:

<div id="parent"> <p>1</p> <p>2</p> <p>3</p> <p>!</p> </div>

<ul id="parent"></ul> let parent = document.querySelector('#parent'); for (let i = 1; i <= 9; i++) { let li = document.createElement('li'); li.textContent = i; parent.appendChild(li); }
