querySelectorAll method

MethodquerySelectorAllallows you to get page elements using an arbitrary CSS selector. You can perform various manipulations with the resulting elements: change their text, attributes, CSS styles, and so on.

Syntax

document.querySelectorAll(селектор);

Example

Let's get all elements with classwwwand change their text using the propertytextContent:

<p class="www">elem 1</p> <p class="www">elem 2</p> <p class="www">elem 3</p> let elems = document.querySelectorAll('.www'); for (let elem of elems) { elem.textContent = '!!!'; }

Result of executing the code:

<p class="www">!!!</p> <p class="www">!!!</p> <p class="www">!!!</p>
