createTextNode method

MethodcreateTextNodecreates a new text node. In the parameter, the method receives the data that will be inserted into the new text node.

Syntax

let text = document.createTextNode(данные);

Example

Let's make a new text node in the document:

<div id="parent"> <p>text1</p> <p>text2</p> </div> let parent = document.getElementById('parent'); let text = document.createTextNode('text3'); parent.appendChild(text);

Result of executing the code:

<div id="parent"> <p>text1</p> <p>text2</p> text3 </div>
