nextElementSibling property

PropertynextElementSiblingcontains the next element located in the same parent. If there is no such element, it is returnednull.

Syntax

элемент.nextElementSibling;

Example

Given element#elem. Let's print the text of its neighbor below:

<p id="elem">elem</p> <p>sibling</p> let elem = document.querySelector('#elem'); let text = elem.nextElementSibling.textContent; console.log(text);

Result of executing the code:

'sibling'

Example

If there is no neighbor below or it is not located in the parent of our element, it is also returnednull:

<div> <p id="elem">elem</p> </div> <p>sibling</p> let elem = document.querySelector('#elem'); console.log(elem.nextElementSibling);

Result of executing the code:

null
