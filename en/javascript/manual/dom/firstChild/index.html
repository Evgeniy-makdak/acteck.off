firstChild property

PropertyfirstChildstores the first child node of an element. If the element has no child elements, it is returnednull.

Syntax

элемент.firstChild;

Example

Let's get the text of the first element node:

<div id="parent">text<p>parag</p><!--com--></div> let parent = document.querySelector('#parent'); let text = parent.firstChild.textContent; console.log(text);

Result of executing the code:

'text'

  • propertylastChild,
