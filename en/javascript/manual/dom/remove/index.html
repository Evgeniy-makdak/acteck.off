remove method

Methodremoveallows you to delete an element. Applies to the element that needs to be removed.

Syntax

элемент.remove();

Example

Let's delete the paragraph#elem:

<div id="parent"> <p id="elem">elem 1</p> <p>elem 2</p> </div> let elem = document.querySelector('#elem'); elem.remove();

Result of executing the code:

<div id="parent"> <p>elem 2</p> </div>
