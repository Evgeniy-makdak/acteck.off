removeEventListener method

MethodremoveEventListenerallows you to delete a previously assigned one viaaddEventListenerevent handler. To do this, you need to pass in the parameters the event type and the same function that were passed when assigning the event.

Syntax

элемент.removeEventListener('тип события', функция);

Example

Let's make sure that clicking on an element only works once:

<input type="button" id="button" value="click me"> let button = document.querySelector('#button'); button.addEventListener('click', func); // назначаем событие function func() { alert('!!!'); button.removeEventListener('click', func); // после клика удаляем его };
