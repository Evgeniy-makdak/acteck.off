hasAttribute method

MethodhasAttributechecks the presence of a given attribute on an element. If the attribute exists, it will displaytrue, if not, it will printfalse.

Syntax

элемент.hasAttribute(имя атрибута);

Example

Let's check for the presence of the attributevalueat the element:

<input id="elem" value="abcde"> let elem = document.querySelector('#elem'); console.log(elem.hasAttribute('value'));

Result of executing the code:

true

Example

And now the attributevalueThere is not:

<input id="elem"> let elem = document.querySelector('#elem'); console.log(elem.hasAttribute('value'));

Result of executing the code:

false
