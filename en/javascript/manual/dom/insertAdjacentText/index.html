insertAdjacentText method

MethodinsertAdjacentTextallows you to insert a line anywhere on the page. The row is inserted relative tosupport element. You can make an insert in front of the support element (insertion method'beforeBegin'), after it (insertion method'afterEnd'), as well as to the beginning (insertion method'afterBegin') or at the end (insertion method'beforeEnd') supporting element.

Syntax

опорный элемент.insertAdjacentText(способ вставки, текст для вставки);

Example. WaybeforeBegin

Let the support element be the element#target. Let's insert some text in front of it:

<div id="target"> <p>elem</p> </div> let target = document.querySelector('#target'); target.insertAdjacentText('beforeBegin', 'text');

Result of executing the code:

text <div id="target"> <p>elem</p> </div>

Example. WayafterEnd

Now let’s insert a new paragraph after the anchor element:

<div id="target"> <p>elem</p> </div> let target = document.querySelector('#target'); target.insertAdjacentText('afterEnd', 'text');

Result of executing the code:

<div id="target"> <p>elem</p> </div> text

Example. afterBegin

Insert a new paragraph at the beginning of the anchor element:

<div id="target"> <p>elem</p> </div> let target = document.querySelector('#target'); target.insertAdjacentText('afterBegin', 'text');

Result of executing the code:

<div id="target"> text <p>elem</p> </div>

Example. WaybeforeEnd

Insert a new paragraph at the end of the anchor element:

<div id="target"> <p>elem</p> </div> let target = document.querySelector('#target'); target.insertAdjacentText('beforeEnd', 'text');

Result of executing the code:

<div id="target"> <p>elem</p> text </div>
