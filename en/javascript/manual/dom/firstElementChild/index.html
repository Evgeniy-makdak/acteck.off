firstElementChild property

PropertyfirstElementChildcontains the first child element. Child elements are all tags that are directly located inside the block. If the element has no child elements, it is returnednull.

Syntax

элемент.firstElementChild;

Example

Let's get the contents of the first child of the element:

<div id="parent"> <p>1</p> <p>2</p> </div> let parent = document.querySelector('#parent'); let text = parent.firstElementChild.textContent; console.log(text);

Result of executing the code:

'1'

Example

And now the element has no child elements and therefore will be displayednull:

<div id="parent"></div> let parent = document.querySelector('#parent'); console.log(parent.firstElementChild);

Result of executing the code:

null
