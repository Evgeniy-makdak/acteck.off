classList property

PropertyclassListcontainspseudo-arrayCSS element classes, and also allows you to add and remove element classes, check the presence of a certain class among the element classes.

It's about the attributeclass, inside which you can write several classes separated by a space, for examplewww ggg zzz. By usingclassListyou can delete, for example, the classgggwithout affecting other classes.

Syntax

элемент.classList;

Example. Number of classes

Let's find out the number of element classes:

<p id="elem" class="www ggg zzz"></p> let elem = document.querySelector('#elem'); let length = elem.classList.length; console.log(length);

Result of executing the code:

3

Example. Let's go through the classes

Let's display the element classes one by one:

<p id="elem" class="www ggg zzz"></p> let elem = document.querySelector('#elem'); let classNames = elem.classList; for (let className of classNames) { console.log(className); }

Result of executing the code:

'www' 'ggg' 'zzz'
