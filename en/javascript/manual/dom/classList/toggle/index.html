The toggle method of the classList object

MethodtoggleobjectclassListalternates the given CSS class of an element: adds the class if it does not exist and removes it if it does.

Syntax

element.classList.contains(class);

Example

In this example, when using the methodtoggleClasszzzwill be removed since it is already in the element:

<p id="elem" class="www ggg zzz"></p> let elem = document.querySelector('#elem'); elem.classList.toggle('zzz');

Result of executing the code:

<p id="elem" class="www ggg"></p>

Example

In this example, when using the toggle method, the classzzzwill be added since it is not in the element:

<p id="elem" class="www ggg"></p> let elem = document.querySelector('#elem'); elem.classList.toggle('zzz');

Result of executing the code:

<p id="elem" class="www ggg zzz"></p>
