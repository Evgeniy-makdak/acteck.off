bind method

Methodbindbind context to function. The context should be passed as the first parameter, and the function parameters as subsequent parameters. The method returns a new function, inside whichthiswill be equal to the passed context.

Syntax

функция.bind(контекст, параметр1, параметр2...);

Example

Let us have an input:

<input id="elem" value="text">

Let the link to this input be written into a variableelem:

let elem = document.querySelector('#elem');

Let us also have the following functionfunc:

function func(param1, param2) { console.log(this.value + param1 + param2); }

Let's usebindlet's make a new function that will be a copy of the functionfunc, Butthisit will always be equalelem:

let newFunc = func.bind(elem);

Now in the variablenewFunclies the function. Let's call it by passing it as the first parameter'1', and in the second'2':

newFunc('1', '2');

Let's put it all together:

let elem = document.getElementById('elem'); function func(param1, param2) { console.log(this.value + param1 + param2); } let newFunc = func.bind(elem); newFunc('1', '2'); // выведет 'text12'

Example

It is not necessary to write down the result of the workbindto a new functionnewFunc, you can just rewritefunc:

let func = func.bind(elem);
