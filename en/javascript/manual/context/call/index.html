call method

Methodcallallows you to call a function with a given context. The first parameter of the method should indicate the function context, and the remaining parameters should indicate the function parameters.

Syntax

функция.call(контекст, параметр1, параметр2...);

Example

Let us have an input:

<input id="elem" value="text">

Let also be given a function that takes three parameters:

function func(param1, param2, param3) { console.log(this.value + param1 + param2 + param3); }

Let's call our function like this:thisinside the function was equal to our input, and at the same time passing numbers to it as parameters1, 2And3:

let elem = document.querySelector('#elem'); func.call(elem, 1, 2, 3);

Example

Let the function take no parameters:

function func() { console.log(this.value); }

In this case, when calling this function viacallit is enough to pass only the first parameter with the context:

let elem = document.querySelector('#elem'); func.call(elem);
