Math.floor method

MethodMath.flooralways rounds numbers down to integers.

Syntax

Math.floor(число);

Example

Let's round the number2.9999to a lesser extent:

console.log(Math.floor(2.9999));

Result of executing the code:

2

    which rounds a number up
  • methodMath.round,

    which round a number to a given sign
