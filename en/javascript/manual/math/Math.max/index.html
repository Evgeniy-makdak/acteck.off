Math.max method

MethodMath.maxreturns the maximum number from a group of numbers passed to the function. If nothing is passed to the function, it will be returned-Infinity.

Syntax

Math.max(число, число, число...);

Example

Let's print the maximum number from a group of numbers:

console.log(Math.max(1, 5, 10, 34, 100));

Result of executing the code:

100

Example

Let's print the maximum number from a group of numbers:

console.log(Math.max(-1, 0, -20, -56, -100));

Result of executing the code:

0

Example

The following example will output-Infinity, since no parameters were passed to the method:

console.log(Math.max());

Result of executing the code:

-Infinity

Example

By default, the function does not work with arrays. However, she can be forced to do this withspread operator. Let's use it to display the maximum value of the array:

let arr = [1, 5, 10, 34, 100]; let max = Math.max(...arr); console.log(max);

Result of executing the code:

100

Example

You can also force the function to work with arrays using the methodapply:

let arr = [1, 5, 10, 34, 100]; let max = Math.max.apply(null, arr); console.log(max);

Result of executing the code:

100
