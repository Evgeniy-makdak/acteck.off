Метод Math.abs

Метод Math.abs возвращает модуль числа, то есть из отрицательного числа делает положительное.

Синтаксис

Math.abs(число);

Пример

Давайте выведем модуль числа -3:

console.log(Math.abs(-3));

Результат выполнения кода:

3

Пример

Давайте выведем модуль числа 3:

console.log(Math.abs(3));

Результат выполнения кода:

3
