Math.log method

MethodMath.logreturns the natural logarithm of the passed number.

Syntax

Math.log(число);

Example

In this example, the value of the natural logarithm of35:

console.log(Math.log(35));

Result of executing the code:

3.5553480614894135

Example

In this example, the value of the natural logarithm of-10:

console.log(Math.log(-10));

Result of executing the code:

NaN
