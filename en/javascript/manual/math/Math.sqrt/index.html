Math.sqrt method

MethodMath.sqrtreturns the square root of a number.

Syntax

Math.sqrt(положительное число);

Example

Let's take the square root of4:

console.log(Math.sqrt(4));

Result of executing the code:

2

Example

Let's get the square root of15:

console.log(Math.sqrt(15));

Result of executing the code:

3.872983346207417

Example

Let's get the square root of-100. Since a negative number was passed, it will be outputNaN:

console.log(Math.sqrt(-100));

Result of executing the code:

NaN
