toFixed method

MethodtoFixedrounds a number to the specified decimal place. The number of characters is specified by the parameter. If the number of characters is not specified, then the default is taken0signs, that is, rounding to a whole number.

Syntax

число.toFixed([количество знаков в дробной части]);

Example

Let it be a fraction. Let's round it up to3signs in the fractional part:

let num = 1.1111; console.log(num.toFixed(3));

Result of executing the code:

1.111

Example

The following example also leaves only3sign in the fractional part, but the last digit has increased by1, since rounding is done according to the rules of mathematics:

let num = 1.1119; console.log(num.toFixed(3));

Result of executing the code:

1.112

Example

The following example will round the fraction to a whole number because the method parameter is empty:

let num = 1.111; console.log(num.toFixed());

Result of executing the code:

1

Example

In the following example, the method parameter exceeds the number of decimal places in the original fraction, so the method will add two digits0to the end of our fraction:

let num = 1.1111; console.log(num.toFixed(6));

Result of executing the code:

1.111100
