Math.pow method

MethodMath.powRaises a number to a given power. The first parameter is the number, the second - to what power to raise it.

Syntax

Math.pow(число, степень);

Example

Let's raise the number3to the degree4:

console.log(Math.pow(3, 4));

Result of executing the code:

81

Example

Let's raise the number2to the degree-5:

console.log(Math.pow(2, -5));

Result of executing the code:

0.03125

Example

There is also an operator**, raising numbers to powers:

console.log(3 ** 4);

Result of executing the code:

81
