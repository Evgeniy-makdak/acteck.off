Math.acos method

MethodMath.acosreturns the arc cosine of a number. The method parameter must be between-1to1, otherwise it will be returnedNaN.

Syntax

Math.acos(число);

Example

Let's print the arc cosine of the number0.34:

console.log(Math.acos(0.37));

Result of executing the code:

1.191787306098946

Example

Let's print the arc cosine of the number1:

console.log(Math.acos(1));

Result of executing the code:

0

Example

The following example will outputNaN, since the parameter is greater1:

console.log(Math.acos(3));

Result of executing the code:

NaN
