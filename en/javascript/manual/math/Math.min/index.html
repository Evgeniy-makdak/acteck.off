Math.min method

MethodMath.minreturns the minimum number from a group of numbers passed as parameters. If nothing is passed as parameters, it will be returnedInfinity.

Syntax

Math.min(число, число, число...);

Example

Let's derive the minimum number from a group of numbers:

console.log(Math.min(40, 20, 42, 100, 67));

Result of executing the code:

20

Example

Let's derive the minimum number from a group of numbers:

console.log(Math.min(-1, -100, -30, -25, 40));

Result of executing the code:

-100

Example

The following example will outputInfinity, since no parameters were passed to the method:

console.log(Math.min());

Result of executing the code:

Infinity

Example

By default, the function does not work with arrays. However, she can be forced to do this withspread operator. Let's use it to display the minimum value of the array:

let arr = [1, 5, 10, 34, 100]; let min = Math.min(...arr); console.log(min);

Result of executing the code:

1

Example

You can also force the function to work with arrays using the methodapply:

let arr = [1, 5, 10, 34, 100]; let min = Math.min.apply(null, arr); console.log(min);

Result of executing the code:

1
