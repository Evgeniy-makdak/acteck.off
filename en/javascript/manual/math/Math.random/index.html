Math.random method

MethodMath.randomreturns a random fractional number from0to1.

Syntax

Math.random();

Application

To get a random number in a certain range (fractional or integer), you should use special techniques. Getting a random fractional number betweenminAndmaxit happens like this:

function getRandomArbitary(min, max) { return Math.random() * (max - min) + min; }

Now let's get a random integer betweenminAndmax:

function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; }

Example

Let's print a random number from0to1:

console.log(Math.random());

Result of executing the code:

0.5416665468657356

Example

Let's print a random integer from10to100:

function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; } console.log(getRandomInt(10, 100));

Result of executing the code:

12
