isNaN function

FunctionisNaNchecks whether the passed parameter is a number or not. Will returntrue, if the parameter is not a number andfalse, if it is. How it works: the passed parameter is converted to a number. If it is not a number (string, array, etc.), then it is converted toNaN. Well, wellisNaNchecks -NaNwhether we succeeded after the transformation or not. Buttruenot converted toNaN, and to the number1. There are other similar pitfalls, see them in the descriptionNaN.

Syntax

isNaN(значение);

Example

NowisNaNwill returnfalse, since the parameter is a number:

console.log(isNaN(3));

Result of executing the code:

false

Example

NowisNaNwill returntrue, since the parameter is not a number:

console.log(isNaN('abcde'));

Result of executing the code:

true
