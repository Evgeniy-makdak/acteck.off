Конструкция function

Конструкция function создает пользовательскую функцию. Эту функцию затем можно будет вызвать в другом месте кода, передав ей какие-нибудь параметры.

Синтаксис

function имяФункции(параметры) { команды, которые выполняет функция. };

Пример

Давайте создадим и вызовем функцию:

// Создаем функцию: function func() { alert('!'); } // Вызываем функцию: func(); // выведет '!'

Пример

Сделаем функцию, параметром принимающую два числа и выводящую на экран их сумму:

function func(num1, num2) { alert(num1 + num2); } func(1, 2); // выведет 3
