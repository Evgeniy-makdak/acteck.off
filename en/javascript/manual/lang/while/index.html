while construct

Designwhiledefines a cycle. The code placed inside it will be repeated a certain number of times.

Syntax

while ( пока выражение истинно ) { выполнять этот код };

Example

Let's print the numbers from1to10:

let i = 1; while (i <= 10) { console.log(i); i++; }

Example

Let's print even numbers from2to10:

let i = 2; while (i <= 10) { console.log(i); i += 2; }
