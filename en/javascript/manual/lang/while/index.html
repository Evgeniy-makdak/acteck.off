Конструкция while

Конструкция while задает цикл. Код, размещенный внутри него, будет повторен определенное количество раз.

Синтаксис

while ( пока выражение истинно ) { выполнять этот код };

Пример

Выведем числа от 1 до 10:

let i = 1; while (i <= 10) { console.log(i); i++; }

Пример

Выведем четные числа от 2 до 10:

let i = 2; while (i <= 10) { console.log(i); i += 2; }
