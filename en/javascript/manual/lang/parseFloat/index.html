parseFloat function

FunctionparseFloatConverts a string to a floating point number. This is needed for values ​​like'12.5px'- when the number comes first, and then the units of measurement. If you apply the functionparseFloatTo'12.5px', then the result will be the number12.5(and it will actually be a number, not a string). The conversion will occur if only the integer is at the beginning of the line, otherwise it will be printedNaN. The second optional parameter can specify the number system of the number, and the function will return the number converted from the specified number system to decimal.

Syntax

parseFloat(строка, [система счисления]);

Example

Let's extract the fractional number from the beginning of the string:

console.log(parseFloat('10.3px'));

Result of executing the code:

10.3

Example

In this example, the function will not be able to read the number (since it is not at the beginning of the line) and will returnNaN:

let str = 'width: 100px'; console.log(parseFloat(str));

Result of executing the code:

NaN
