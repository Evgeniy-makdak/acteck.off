typeof operator

Operatortypeofallows you to determine the type of parameter (number, string, object). The operator returns a string containing the type ('number', 'string', 'object').

Fornulloperator returns'object'(this is a recognized language error). For functions, the operator returns'function'. This is done for convenience, as'function'does not exist.

Syntax

At the operatortypeofThere is2syntax (both syntaxes work the same):

typeof параметр; typeof(параметр);

Example

Let's see howtypeofworks with numbers:

typeof 1;

As a result of the executed code we will get the valuenumber:

'number'

Example

Now let's set the parameter to the string:

typeof 'str';

Result of executing the code:

'string'

Example

Now let's specify a boolean value in the parametertrue:

typeof true;

Result of executing the code:

'boolean'

Example

Let's see what type the value isundefined:

typeof undefined;

After executing the code we will also getundefined:

'undefined'

Example

Now let's find out the type of an empty object:

typeof {};

Result of executing the code:

'object'

Example

Now let’s find out the type of an empty array:

typeof [];

As a result we also get'object':

'object'

Example

Let's find out the value typenull:

typeof null;

As a result we also get'object', which is a recognized language error:

'object'

Example

Now let's define the type of the empty function:

typeof function() {};

After executing the code we will get'function', despite the fact that such a type does not exist. This line is needed for user convenience when defining a function:

'function'

Example

Let's write a function that will output only numbers:

function printNumber(number) { if (typeof number === 'number') { console.log(number); } } printNumber(2); printNumber('str'); printNumber(3);
