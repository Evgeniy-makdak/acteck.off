Boolean function

FunctionBooleanConverts the passed value to a boolean type. The result of executing the function will be eithertrue, orfalse.

At the same time, tofalseare converted only:0, empty string,null, undefined, NaN. All other values ​​(numbers, strings, objects, arrays, functions) are converted totrue.

Syntax

Boolean(что нужно преобразовать);

Example

Let's convert the number0to boolean type:

Boolean(0);

As a result of executing the code we will getfalse:

false

Example

Let's convert the valueNaN:

Boolean(NaN);

Also after executing the code we will getfalse:

false

Example

Now let's convert the number1:

Boolean(1);

After executing the code we will gettrue:

true

Example

Now let's convert the negative number:

Boolean(-1);

As a result of executing the code we will gettrue:

true
