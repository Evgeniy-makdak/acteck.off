if construct

Designifspecifies a condition. Using this construction, we can ask the program, for example, the question “is the variable largerathan variableb", and depending on the answer, execute either one code or another.

As a rule, this design is used in combination with the designelse.

Syntax

if (логическое выражение) { /* расположенный здесь код выполниться, если логическое выражение верное */ };

If there is only one expression in curly braces, you can omit these curly braces.

Example

If the value of the variable is equal to one, we will display some message on the screen:

let test = 1; if (test == 1) { alert('+++'); }

Example

Let's check whether the value of the variable is greater than zero or not:

let test = 1; if (test > 0) { alert('+++'); } else { alert('---'); }

Example

Constructionsif-elseYou can nest each other in any way:

let num = 3; if (num >= 0) { if (num <= 5) { alert('меньше или равно 5'); } else { alert('больше 5'); } } else { alert('меньше нуля'); }
