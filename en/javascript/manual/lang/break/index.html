break instruction

Instructionsbreakperforms a forced exit from the loop. Can complete any loops:for, for-of, for-in, while.

Syntax

break;

Example

Let's determine if there is a number in the array3. If there is, we will output it to the console'+++'and exit the loop:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem == 3) { console.log('+++'); break; // выйдем из цикла } }
