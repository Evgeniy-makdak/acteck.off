Object construct

DesignObjectcreates an object containing keys and their corresponding values.

Syntax

let object = new Object(элементы объекта через запятую);
let object = {элементы объект через запятую}

Example

Let's create a new object:

let object = new Object({'a': 1, 'b': 2, 'c': 3}); console.log(object);

Result of executing the code:

{'a': 1, 'b': 2, 'c': 3}
