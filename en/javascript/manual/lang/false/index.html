Значение false

Значение false обозначает ложь. Его можно использовать в условных выражениях с if и elseif для проверки их на истинность, а также с помощью false можно определить наличие элемента или свойства.

Синтаксис

false;

Пример

Давайте проверим, есть ли в массиве буква 'e':

let arr = ['a', 'b', 'c', 'd']; console.log(arr.includes('e'));

Результат выполнения кода:

false

Пример

Давайте проверим равно ли значение переменной 2:

let num = 1; console.log(num === 2);

Результат выполнения кода:

false

Пример

Давайте проверим результат выполнения условия:

let test = false; if(test === false) { console.log('+++'); } else { console.log('---'); }

Результат выполнения кода:

'+++'

Смотрите также

  • значение true,
    которое обозначает, что значение элемента истинно
  • значение null,
    которое обозначает отсутствие объекта или элемента
  • значение undefined,
    которое обозначает, что значение элемента "не определено"
