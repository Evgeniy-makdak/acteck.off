The value is false

Meaningfalsemeans false. It can be used in conditional expressions withifAndelseifto check them for truth, as well as usingfalseyou can determine the presence of an element or property.

Syntax

false;

Example

Let's check if there is a letter in the array'e':

let arr = ['a', 'b', 'c', 'd']; console.log(arr.includes('e'));

Result of executing the code:

false

Example

Let's check if the value of the variable is equal to2:

let num = 1; console.log(num === 2);

Result of executing the code:

false

Example

Let's check the result of the condition:

let test = false; if(test === false) { console.log('+++'); } else { console.log('---'); }

Result of executing the code:

'+++'
