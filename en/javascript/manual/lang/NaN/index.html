NaN value

MeaningNaN(short forNot-A-Number) means "not a number". This value occurs when attempts are made to perform an invalid mathematical operation.

Example

Let's multiply a string of letters by a number:

console.log('aaa' * 3);

Result of executing the code:

NaN

Example

Let's try to split one line into another:

console.log('aaa' / 'bbb');

After executing the code we will also getNaN:

NaN

Example

Now let's add up the valueNaNwith number:

console.log(NaN + 3);

Result of executing the code:

NaN
