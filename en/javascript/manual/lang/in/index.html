Example

Let's see how to use the operatorinto check a property in an array:

let arr = ['green', 'red', 'blue', 'pink'] console.log(0 in arr); // true, есть элемент с таким индексом console.log(4 in arr); // false, нет элемента с таким индексом console.log('four' in arr); // false, нужен индекс, а не значение элемента console.log('length' in arr); // true, у массива есть такое свойство
