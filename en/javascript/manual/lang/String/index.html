String function

FunctionStringconverts the passed value to a string. Conversions for primitives (numbers, strings and other simple types) occur in the most obvious way. For example:true - 'true', 5 - '5'. And objects are most often transformed to'[object Object]', but there may be exceptions.

Syntax

String(что нужно преобразовать);

Example

Let's convert the number0to the line:

String(0);

Result of executing the code:

'0'

Example

Now let's set the function parameter to a non-numeric valueNaN:

String(NaN);

As a result, we will get the line:

'NaN'

Example

Let's convert the boolean valuefalse:

String(false);

Result of executing the code:

'false'

Example

Converting a string using the functionString:

String('str');

After executing the code, we will get the same line as output:

'str'

Example

Now let's set the function parameter to the valuenull:

String(null);

As a result of executing the code, we will get the value converted to a string:

'null'

Example

Let's set the function parameter to the valueundefined:

String(undefined);

Result of executing the code:

'undefined'

Example

Now let's set curly braces in the parameter to denote an empty object:

String({});

After executing the code, we will receive a line indicating the type of the given object:

'[object Object]'

Example

Now let's convert square brackets or an empty array:

String([]);

As a result, instead of describing the object type, we get an empty string:

''

Example

Let's add an array filled with numbers to the function parameter:

String([1, 2, 3]);

As a result of executing the code, all array values ​​are converted into one string:

'1,2,3'
