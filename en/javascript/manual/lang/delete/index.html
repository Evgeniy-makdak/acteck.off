delete operator

Operatordeleteremoves a property of an object. After removal this property will have the valueundefined.

Syntax

delete объект.свойство;

Example

Let's create an object and remove one of its properties:

let obj = { name: 'john', age: 23, }; delete user.name; console.log(user.name);

Result of executing the code:

undefined

Example

Let's create an array and remove an element from it:

let arr = ['a', 'b', 'c', 'd']; delete arr[2]; console.log(arr);

Result of executing the code:

['a', 'b', empty, 'd']
