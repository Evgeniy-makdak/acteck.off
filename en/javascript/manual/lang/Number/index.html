Number function

FunctionNumberconverts the passed value to a number. In the case of a string, whitespace characters at the edges are trimmed, then, if the string can be converted to a number, then this number is returned, and if not, it is returnedNaN. Meaningtrueconverted to number1, meaningfalse- to the number0. Objects during numerical transformations turn intoNaN.

Syntax

Number(что нужно преобразовать);

Example

Converting a Boolean Valuetrueto number:

Number(true);

Result of executing the code:

1

Example

Now let's convert the boolean valuefalse:

Number(false);

Result of executing the code:

0

Example

Let's enter a string containing a number into the function parameter:

Number('53');

As a result of the transformation we get the number:

53

Example

Now let’s add spaces to the number at the beginning and end of the line:

Number(' 5 ');

After executing the code, the extra spaces are removed and we end up with a number:

5

Example

Let's convert the empty string:

Number('');

Result of executing the code:

0

Example

Now let's convert a string containing a letter and a number:

Number('a5');

After executing the code we get the valueNaN, indicating an invalid mathematical operation:

NaN

Example

Let's swap letters and numbers in the string:

Number('5a');

The result of executing the code will remain the same:

NaN

Example

Now let's transform the string in which the numbers are separated by a space:

Number('5 5');

As a result of executing the code, we will again see that this mathematical operation is invalid:

NaN

Example

Let's convert the boolean valuetrue, enclosed in the line:

Number('true');

After executing the code we get againNaN:

NaN

Example

Now let's convert the empty object:

Number({});

Result of executing the code:

NaN

Example

Let's set the function parameter to an empty array:

Number([]);

As a result we get the number0:

0

Example

Now let's add a number to the array:

Number([1]);

The array is converted to a number:

1

Example

Let's convert an array containing two digits:

Number([1, 2]);

As a result, we will again be told that this mathematical operation is invalid:

NaN
