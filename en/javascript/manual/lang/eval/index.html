Функция eval

Функция eval выполняет строку как JavaScript код.

Синтаксис

eval(строка с кодом);

Пример

В данном примере будет вызван alert с сообщением:

eval("alert('Hello word')");

Пример

В данном примере будет выведен результат от суммы двух переменных:

let code = "let a=10; let b=20; document.write(a+b)"; eval(code);
