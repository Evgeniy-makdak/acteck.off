The for construct

Designfordefines a loop: the code placed inside it will be repeated a specified number of times.

Syntax

for (начальные команды; условие окончания цикла; команды после прохода цикла) { тело цикла };

Example

Let's print a sequence of numbers from0to10:

for (let i = 0; i <= 10; i++) { console.log(i); }

Example

Let's print a sequence of numbers from10to0:

for (let i = 10; i >= 0; i--) { console.log(i); }

Example

Let's print even numbers from0to10:

for (let i = 2; i <= 10; i += 2) { console.log(i); }

Example

Let's display the elements of the array:

let arr = ['a', 'b', 'c', 'd', 'e']; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Example

Initial commandsAndcommands after loopmay consist of not one, but several commands, separated by commas. For example, let's make two counters: let the first one increase by one each iteration of the loop, and let the second one increase by two:

for (let i = 0, j = 0; i <= 9; i++, j += 2) { console.log(i, j); }
