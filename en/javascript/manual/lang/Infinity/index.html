Infinity meaning

MeaningInfinitymeans infinity, andInfinity- positive infinity (equal to1/0), A-Infinity- negative infinity (equal to-1/0).

Syntax

Infinity;

Example

Let's determine the result of the following mathematical operation:

let res = 1 / 0; console.log(res);

Result of executing the code:

Infinity

Example

Now let's find out the result of division with negative numbers:

let res = -1 / 0; console.log(res);

Result of executing the code:
