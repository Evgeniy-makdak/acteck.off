The value is undefined

Meaningundefinedmeans "undefined". JavaScript has a similar meaningnull, specifying the absence of a value. The difference is thatnulldenotes intentional absence (explicitly written in code), andundefined- simply the absence of any information about the element type and its meaning.

Syntax

undefined;

Example

Let's find out the value of a variable that was declared without a value:

let test; console.log(test);

Result of executing the code:

undefined

Example

Let's find out the value of a non-existent array element:

let arr = ['a', 'b', 'c']; console.log(arr[5]);

Result of executing the code:

undefined

Example

Let's find out the value of a non-existent object element:

let obj = {'a': 1, 'b': 2, 'c': 3}; console.log(obj['e']);

Result of executing the code:

undefined
