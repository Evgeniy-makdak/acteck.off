offsetTop property

PropertyoffsetTopcontains the offset of the element from above relative tooffsetParent. Contains distance fromoffsetParentto the border of the element.

Syntax

элемент.offsetTop;

Example

Let's find the offset of the element relative to itsoffsetParent:

<div id="parent" style="position: relative;"> <div id="elem" style="position: absolute; top: 100px;"></div> </div> let elem = document.querySelector('#elem'); console.log(elem.offsetTop);

Result of executing the code:

100

Example

Indentationsmarginalso summed up into offset:

<div id="parent" style="position: relative;"> <div id="elem" style="position: absolute; top: 100px; margin: 50px;"></div> </div> let elem = document.querySelector('#elem'); console.log(elem.offsetTop);
