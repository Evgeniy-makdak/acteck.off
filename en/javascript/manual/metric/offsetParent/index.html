offsetParent property

PropertyoffsetParentcontains the closest parent relative to which the element is positioned. This will either be the closest parent that has the CSS propertypositionnot equalstatic, or tagbody, if there is no parent with such positioning.

Syntax

элемент.offsetParent;

Example

Let's find the parent element with the propertyposition, different fromstatic:

<div id="parent3"> <div id="parent2" style="position: relative"> <div id="parent1"> <p id="elem">text</p> </div> </div> </div> let elem = document.querySelector('#elem'); console.log(elem.offsetParent.id);

Result of executing the code:

'parent2'

Example

If parent elements have no positioning, thenoffsetParentcontainsbody:

<div> <p id="elem">text</p> </div> let elem = document.querySelector('#elem'); console.log(elem.offsetParent.tagName);

Result of executing the code:

'BODY'
