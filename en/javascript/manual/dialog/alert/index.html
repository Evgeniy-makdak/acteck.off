Функция alert

Функция alert показывает модальное окно с заданным сообщением. Пользователь сайта не сможет продолжить работу, пока не нажмет на кнопку ОК. Отображаться будет только текст, теги в модальном окне будут игнорироваться.

Синтаксис

alert(сообщение);

Пример

Давайте сделаем так, чтобы по нажатию на кнопку появилось модальное окно с сообщением:

<button id="button">click me</button> let button = document.querySelector('#button'); button.addEventListener('click', function(event) { alert('abcde'); });
