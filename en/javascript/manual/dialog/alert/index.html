alert function

Functionalertshows a modal window with the specified message. The site user will not be able to continue working until he clicks on the buttonОК. Only text will be displayed, tags in the modal window will be ignored.

Syntax

alert(сообщение);

Example

Let's make sure that when the button is clicked, a modal window with a message appears:

<button id="button">click me</button> let button = document.querySelector('#button'); button.addEventListener('click', function(event) { alert('abcde'); });
