confirm function

Functionconfirmcalls a confirmation window with buttonsOKAndCancel. Returnstrue, if the button was pressedOK, Andfalse, if the button was pressedCancel. The parameter accepts the text that the user will see.

Syntax

confirm(сообщение);

Example

Let's use the functionconfirmdisplay a dialog box. If in the window that appears, clickOK, then into the variablereswill sign uptrue, and if you pressCancel- Thatfalse:

<button id="button">click me</button> let button = document.querySelector('#button'); button.addEventListener('click', function(event) { let res = confirm('Are you 18 years old?'); console.log(res); });
