Метод setMonth

Метод setMonth позволяет изменить значение месяца объекту с датой. Принимает первым параметром месяц, а вторым необязательным параметром день месяца.

Синтаксис

дата.setMonth(месяц, [день]);

Пример

Давайте установим месяц для текущей даты:

let date = new Date(); date.setMonth(1); console.log(date);

Пример

Давайте установим месяц и день для текущей даты:

let date = new Date(); date.setMonth(1, 25); console.log(date);
