setMonth method

MethodsetMonthallows you to change the month valueobject with date. Takes the month as the first parameter, and the day of the month as the second optional parameter.

Syntax

дата.setMonth(месяц, [день]);

Example

Let's set the month for the current date:

let date = new Date(); date.setMonth(1); console.log(date);

Example

Let's set the month and day for the current date:

let date = new Date(); date.setMonth(1, 25); console.log(date);
