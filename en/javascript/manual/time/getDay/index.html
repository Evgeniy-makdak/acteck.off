getDay method

MethodgetDayapplies toobject with dateand returns the day of the week number as a number from0to6. Moreover, Sunday is day number0, Monday -1etc.

Syntax

дата.getDay();

Example

Let's print the number of the current day of the week:

let date = new Date(); let res = date.getDay(); console.log(res);

Example

Let's print the day of the week of a given date:

let date = new Date(2025, 11, 31); let res = date.getDay(); console.log(res);

Result of executing the code:

3
