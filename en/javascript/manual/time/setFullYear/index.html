Метод setFullYear

Метод setFullYear позволяет изменить значение года объекту даты. Принимает первым параметром год, также может принимать месяц и день.

Синтаксис

дата.setFullYear(год, [месяц], [день]);

Пример

Давайте поменяем год в объекте с датой:

let date = new Date(); date.setFullYear(2026); console.log(date.getFullYear());

Результат выполнения кода:

2026

Пример

Поменяем сразу год, месяц, и день:

let date = new Date(); date.setFullYear(2026, 5, 6);

Смотрите также

  • метод setMonth,
    который устанавливает месяц
  • метод setDate,
    который устанавливает день месяца
  • метод setHours,
    который устанавливает часы
  • метод setMinutes,
    который устанавливает минуты
  • метод setSeconds,
    который устанавливает секунды
