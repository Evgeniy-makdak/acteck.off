Date.now method

MethodDate.nowreturns the number of milliseconds that have passed since midnight1January1970years to date. This date format is calledtimestamp.

Syntax

Date.now();

Example

Let's print the number of milliseconds that have passed since1January1970year:

let res = Date.now(); console.log(res);
