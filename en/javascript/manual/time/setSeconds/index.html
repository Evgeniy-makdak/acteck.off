setSeconds method

MethodsetSecondsallows you to change the number of secondsobject with date. Takes the number of seconds as the first parameter; it can also take milliseconds.

Syntax

дата.setSeconds(секунды, [миллисекунды]);

Example

Let's set the seconds for the current time:

let date = new Date(); date.setSeconds(59); console.log(date);

Example

Let's set the seconds and milliseconds for the current time:

let date = new Date(); date.setSeconds(59, 999); console.log(date);
