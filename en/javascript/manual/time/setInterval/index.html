setInterval function

FunctionsetIntervalexecutes code at a specified time interval. The first parameter should be passedcallback, and the second is the time in milliseconds, indicating after what interval the code specified by the first parameter will be repeated. The function returns a unique identifier that can be used to stop the timer. To do this, this identifier should be passed to the functionclearInterval.

Syntax

setInterval(функция, время);

Example

Let's start a timer that outputs some text to the console every second:

setInterval(function() { console.log('text'); }, 1000);

Example

Let's start a timer that outputs ascending integers to the console every second:

let i = 0; setInterval(function() { console.log(i++); }, 1000);

Example

Stop the timer when the counter reaches the value10:

let i = 0; let id = setInterval(function() { i++; if (i == 10) { clearInterval(id); } else { console.log(i); } }, 1000);
