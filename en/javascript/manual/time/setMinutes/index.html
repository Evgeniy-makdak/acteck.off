setMinutes method

MethodsetMinutesallows you to change the number of minutesobject with date. Takes the number of minutes as the first parameter, and can also take seconds and milliseconds as the second and third optional parameters.

Syntax

дата.setMinutes(минуты, [секунды], [миллисекунды]);

Example

Let's set the minutes for the current time:

let date = new Date(); date.setDate(59); console.log(date);

Example

Let's set the minutes and seconds for the current time:

let date = new Date(); date.setDate(59, 59); console.log(date);
