Date object

ObjectDate- the main object for working with dates.

Syntax

let date = new Date();

Now the variabledateis an object with a date that storescurrent moment in time(second, minute, hour, etc.). Using special functions we can get the time characteristics we need, for example, the current hour, the current day or the current month.

For example, the current hour can be obtained like this:date.getHours(), and the current month is like thisdate.getMonth(). See all options:

let date = new Date(); console.log(date.getSeconds()); // секунды console.log(date.getMinutes()); // минуты console.log(date.getHours()); // часы console.log(date.getDate()); // дни console.log(date.getMonth()); // месяцы с нуля console.log(date.getFullYear()); // год console.log(date.getDay()); // текущий день недели

We set a specific point in time

You can specify not the current moment of time, but a given one. To do this, pass the parameters in the formatnew Date(год, месяц, день, часы, минуты, секунды, миллисекунды)and in this case into a variabledateIt will not be the current moment in time that will be recorded, but the one that we specified in the parameters. Features of this format: the counting of months starts from zero, missing parameters, starting from the hour, are considered equal to zero, and for the year, months and days - to one.

Example

Let's display the current onesday, month and yearin format'год-месяц-день'(the month will be on1less than the present, since the numbering of months is from zero):

let date = new Date(); let str = date.getFullYear() + '-' + date.getMonth() + '-' + date.getDate(); console.log(str);
