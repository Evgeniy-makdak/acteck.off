Метод getSeconds

Метод getSeconds применяется к объекту с датой и возвращает текущее значение секунд (число от 0 до 59).

Синтаксис

дата.getSeconds();

Пример

Давайте выведем текущее количество секунд:

let date = new Date(); let res = date.getSeconds(); console.log(res);
