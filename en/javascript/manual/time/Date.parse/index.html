Date.parse method

MethodDate.parsereturns the number of milliseconds before the date passed in the parameter as an ISO format string. ISO format looks like this:'YYYY-MM-DDTHH:mm:ss.sssZ', Where:'YYYY-MM-DD'- date;'T'- separator;'HH:mm:ss.sss'- time;'Z'- Time zone. It is not necessary to transmit the full format, it is enough to transmit'YYYY-MM-DD'or justYYYY.

Syntax

Date.parse('YYYY-MM-DDTHH:mm:ss.sssZ');

Example

Let's use the methodDate.parsefind out how many milliseconds have passed since1January1970year to16.11.2016, 12:59:59:

let res = Date.parse('2016-11-16T12:59:59'); console.log(res);

Result of executing the code:

1479290399000

Example

Now let's print the number of milliseconds, specifying only the year:

let res = Date.parse('2016'); console.log(res);

Result of executing the code:

1451606400000

Example

Let's set in the method parameterDate.parsewrong date:

let res = Date.parse('20162016'); console.log(res);

Result of executing the code:

NaN
