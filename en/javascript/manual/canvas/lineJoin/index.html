lineJoin property

PropertylineJoinspecifies how two lines are combined. Can take the following values:miter- acute angle (default),round- rounded corner,bevel- flat angle.

Syntax

контекст.lineJoin = 'miter' или 'round' или 'bevel';

Example

Let's set the propertylineJoinin meaningmeter:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.lineWidth = 10; ctx.lineJoin = 'meter'; ctx.moveTo(50, 140); ctx.lineTo(100, 30); ctx.lineTo(150, 140); ctx.stroke();
