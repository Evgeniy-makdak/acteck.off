arc method

Methodarcdraws an arc with a center at a given point and a given radius. This arc will become visible only if you apply the methodsstrokeorfill. In the first case there will be an outline, and in the second there will be a figure.

The last optional parameter controls the drawing direction. It takes meaningtrueorfalse. Meaningtruedraws an arc clockwise, and the valuefalse- counterclockwise (default).

You can set the start and end angles when drawing. These angles are measured inradians, not in degrees. To convert degrees to radians you can use the following function:

function getRadians(degrees) { return (Math.PI / 180) * degrees; }

Syntax

контекст.arc(x, y, радиус, начальный угол, конечный угол, [направление = false]);

Example

Let's draw a circle:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.arc(100, 100, 75, 0, getRadians(360)); ctx.stroke(); function getRadians(degrees) { return (Math.PI / 180) * degrees; }
