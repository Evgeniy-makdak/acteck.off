Свойство font

Свойство font задает настройки шрифта при его рисовании на canvas с помощью метода fillText или метода strokeText. Метод принимает такие же параметры, как и CSS свойство font.

Синтаксис

контекст.font = настройки шрифта;

Пример

Давайте нарисуем на canvas текст с помощью fillText и зададим ему размер в 30px и тип Arial с помощью font:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.font = '30px Arial'; ctx.fillText('Hello world', 30, 50);

:
