direction property

Propertydirectionspecifies the direction of text drawn using the methodfillTextor methodstrokeText. Takes one of the possible values:ltr(from left to right),rtl(from right to left),inherit(inherited). The default value isinherit(for Arabic and Hebrew it will bertl, since they write there from right to left, and for everyone else -ltr).

Syntax

контекст.direction = ltr или rtl или inherit;

Example

Let's write some text and give it different directions using the propertydirection:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> const canvas = document.getElementById("canvas"); const ctx = canvas.getContext("2d"); ctx.font = "16px arial"; ctx.fillText("text1", 150, 50); ctx.direction = "rtl"; ctx.fillText("text1", 50, 120);
