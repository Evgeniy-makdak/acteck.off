Метод lineTo

Метод lineTo проводит линию от точки к точке. Линия проводится от текущего положения пера для рисования и до указанных координат. Координаты указываются параметрами метода.

Синтаксис

контекст.lineTo(x, y);

Пример

Давайте нарисуем линию:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.stroke();

Пример

Давайте нарисуем квадратик:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.lineTo(150, 150); ctx.lineTo(50, 150); ctx.lineTo(50, 50); ctx.stroke();

:

Смотрите также

  • метод moveTo,
    который перемещает перо для рисования
  • метод closePath,
    который замыкает линии в фигуру
  • свойство lineWidth,
    которое задает толщину линии
  • свойство lineJoin,
    которое задает стиль излома линии
