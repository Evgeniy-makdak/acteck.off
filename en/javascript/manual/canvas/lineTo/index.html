lineTo method

MethodlineTodraws a line from point to point. A line is drawn from the current position of the drawing pen to the specified coordinates. The coordinates are specified by the method parameters.

Syntax

контекст.lineTo(x, y);

Example

Let's draw a line:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.stroke();
