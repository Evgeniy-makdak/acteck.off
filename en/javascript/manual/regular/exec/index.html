exec method

Methodexecsearches a string for a givenregular expression. The result returns the found substring and its pockets. In this case, each subsequent call of this method will begin the search from the place where the previous found substring ended. If a match is not found, it is returnednull.

Syntax

регулярное_выражение.test(строка);

Example

Let's check the operation of the method:

let str = '12 34 56'; let reg = /\d+/g; let res1 = reg.exec(str); console.log(res1); let res2 = reg.exec(str); console.log(res2); let res3 = reg.exec(str); console.log(res3); let res4 = reg.exec(str); console.log(res4);

Result of executing the code:

[12] [34] [56] null

Example

Let's use the method in a loop:

let str = '12 34 56'; let reg = /\d+/g; let res; while (res = reg.exec(str)) { console.log(res); }

Result of executing the code:

[12] [34] [56]

Example

The matches found can be sorted into pockets:

let str = '12 34 56'; let reg = /(\d)(\d)/g; let res; while (res = reg.exec(str)) { console.log(res); }

Result of executing the code:

[12, 1, 2] [34, 3, 4] [56, 5, 6]

Example

Using propertylastIndexyou can set the position from which to start the search:

let str = '12 34 56'; let reg = /\d+/g; reg.lastIndex = 2; let res1 = reg.exec(str) console.log(res1); let res2 = reg.exec(str) console.log(res2);

Result of executing the code:

[34] [56]

Example

Using a modifieryyou can fix the search start position:

let str = '12 34 56'; let reg = /\d+/y; reg.lastIndex = 2; let res1 = reg.exec(str) console.log(res1); let res2 = reg.exec(str) console.log(res2);

Result of executing the code:

null [12]
