match method

Methodmatchreturns an array of matches withregular expression. If there are no matches, it will returnnull. Being called without a modifiergThe method returns an array, the zero element of which will contain the found substring, and the remaining elements will contain pockets. If the method is called with a modifiergit returns all matches found as an array.

Syntax

строка.match(регулярное выражение);

Example

Let's find a regular expression match and put it into pockets:

let str = '12:34'; let res = str.match(/(\d+):(\d+)/); console.log(res[0]); // найденное console.log(res[1]); // 1 карман console.log(res[2]); // 2 карман

Result of executing the code:

'12:34' '12' '34'

Example

Let's get an array of substrings consisting of letters'a':

let str = 'a aa aaa aaaa'; let res = str.match(/a+/g); console.log(res);

Result of executing the code:

['a', 'aa', 'aaa', 'aaaa']
