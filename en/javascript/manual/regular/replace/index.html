replace method

Methodreplacesearches and replaces parts of a string. The first parameter is takenregular expression, and the second is the substring with which we are replacing.

Syntax

строка.replace(регулярка, замена);

Example

Let's find and replace the symbol'a':

let str = 'bab'; let res = str.replace(/a/, '!'); console.log(res);

Result of executing the code:

'b!b'

Example

By default, only the first match is replaced. Let's replace the symbol again'a'in the line:

let str = 'baaab'; let res = str.replace(/a/, '!'); console.log(res);

Result of executing the code:

'b!aab'

Example

Let's replace all matches using a global search:

let str = 'baaab'; let res = str.replace(/a/g, '!'); console.log(res);

Result of executing the code:

'b!!!b'

Example

Let's find and replace the string using this pattern:letter'x', then any symbol, then a letter again'x':

let str = 'xax eee'; let res = str.replace(/x.x/, '!'); console.log(res);

Result of executing the code:

'! eee'
