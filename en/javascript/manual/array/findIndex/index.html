findIndex method

MethodfindIndexallows you to find the index of the first element according to the parameter passedcallback. If the element is not found, then it is returned-1.

Syntax

массив.findIndex(функция);

Example

Let's find the index of the first even element of the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.findIndex(function (elem){ return elem % 2 == 0; }); console.log(res);

Result of executing the code:

1

Example

Now let's set the following conditions in the function that will not match any element in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.findIndex(function (elem){ return elem < 0; }); console.log(res);

Result of executing the code:

-1
