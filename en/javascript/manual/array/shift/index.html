shift method

Methodshiftremoves the first element from the array. In this case, the original array is modified, and the removed element is returned as the result of the method.

Syntax

массив.shift();

Example

Let's remove the first element from the array:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.shift(); console.log(arr);

Result of executing the code:

['b', 'c', 'd', 'e']

Example

Let's remove the first element from the array and print it to the screen:

let arr = ['a', 'b', 'c', 'd', 'e']; let elem = arr.shift(); console.log(elem);

Result of executing the code:

'a'

Example. Application

Let's make a string from the array'16-25-34'. To solve the problem we use a combination of methodsshift, pop, pushAndjoin:

let arr = ['1', '2', '3', '4', '5', '6']; let res = []; while (arr.length > 0) { // массив уменьшается в цикле пока не достигнет нуля let first = arr.shift(); let last = arr.pop(); let str = first + last; // тут будет строка '16', потом '25', потом '34' res.push(str); } // После цикла в res лежит массив ['16', '25', '34']. Сольем его в строку: res = res.join('-'); console.log(res);

Result of executing the code:

'16-25-34'
