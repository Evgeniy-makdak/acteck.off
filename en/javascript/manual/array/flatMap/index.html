flatMap method

MethodflatMapreturns a new array created after each element of the original array has been appliedcallback, specified in the method parameter. At first glance the methodflatMapsimilar to methodmap, but its main difference is that it always returns a one-dimensional array.

Syntax

let newArray = массив.flatMap(функция);

Example

Let's get a new array using the methodflatMap:

let arr = ['a', 'b', 'c']; let res = arr.flatMap(function(elem) { return elem + '!'; }); console.log(res);

Result of executing the code:

['a!', 'b!', 'c!']

Example

Let's get a new array from the original one consisting of numbers:

let arr = [1, 2, 3, 4, 5]; let res = arr.flatMap(function(elem) { return elem + 2; }); console.log(res);

Result of executing the code:

[3, 4, 5, 6, 7]

Example

Let's try the methodflatMapto an array with the first nesting level:

let arr = [1, 2, 3, [4, 5]]; let res = arr.flatMap(function getElem(elem){ return elem + 2; }); console.log(res);

After executing the code, we will see that the function conditions have been applied to the subarray, counting it as one element. Therefore, calling this method in nested arrays does not make sense:

[3, 4, 5, '4,52']

Example

If we don’t want to bother with the nesting level, but want to combine the entire array, then we use the parameterInfinity:

let arr = ['a', 'b', ['c', 'd', ['e', 'f', ['g', 'h', ['i']]]]]; let res = arr.flatMap(Infinity); console.log(res);

Result of executing the code:

[ 'a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i' ]

Example

Let's apply two methods to the array:flatMapAndmap, and the parameter of both methods will contain a function that makes the array two-dimensional:

let arr = [1, 2, 3, 4, 5, 6]; let res1 = arr.map(elem => [elem * 3]); let res2 = arr.flatMap(elem => [elem * 3]); console.log('метод map', res1); console.log('метод flatMap', res2);

Result of executing the code:

'метод map' [[3], [6], [9], [12], [15], [18]] 'метод flatMap' [3, 6, 9, 12, 15, 18]
