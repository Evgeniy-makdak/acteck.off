indexOf method

MethodindexOfsearches for an element in an array. In the first parameter we specify the element to search. The method returns the number of the first element found, or-1, if there is no such element. The second optional parameter of the method specifies the position from which the search should begin.

Syntax

массив.indexOf(элемент, [откуда начинать]);

Example

Let's find the position of the first three in the array:

let arr = [1, 2, 3, 3, 3, 4, 5]; let res = arr.indexOf(3); console.log(res);

Result of executing the code:

2

Example

Let now the element being checked is not in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.indexOf(6); console.log(res);

Result of executing the code:

-1

Example

Let's start the search from a given position:

let arr = [1, 2, 3, 4, 5, 3]; let res = arr.indexOf(3, 4); console.log(res);

Result of executing the code:

5

Common mistake

Sometimes the method is used to check the presence of an element in an array. In this case, it is easy to make a mistake if you check as follows:

if (arr.indexOf(3)) { console.log(true); } else { console.log(false); }

This check will work for all positions except zero - after all, it is interpreted asfalse.

The correct way to check for the presence of an element is as follows:

if (arr.indexOf(3) !== -1) { console.log(true); } else { console.log(false); }
