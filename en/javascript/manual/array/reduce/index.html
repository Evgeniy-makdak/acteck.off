reduce method

Methodreducecollapses the array to a single value (reduces). For example, using this method you can easily find the sum of the elements of an array (that is, the array will be reduced to one value - the sum of the elements).

The first parameter of the methodreducegets a function that will be executed sequentially for each element of the array, starting from the first. You can pass this function4parameter. If these parameters exist (they are not required), then the first one will automatically includeintermediate result, the second will contain the array element, the third will contain its number in the array (index), and the fourth will contain the array itself.

The intermediate result is a variable in which the value returned by the method will be accumulatedreduce, when it will iterate through all the elements of the array. For example, you can sequentially accumulate the sum of array elements there: first put the first element, on the next pass of the loop the sum of the first element and the second, on the next pass - the sum of the first, second and third. And so on until the array ends. The function it takesreduce, must return the new intermediate result value.

The second parameter of the methodreducethe initial value of the intermediate result is indicated. If you do not specify it, then it will be equal to the first element of the array, and processing of elements will begin from the second element.

Syntax

массив.reduce(function(промежуточный результат, элемент, индекс, массив) { return новый промежуточный результат; }, начальное значение);

Example

Let's find the sum of the array elements:

let arr = [1, 2, 3, 4, 5, 6]; let res = arr.reduce(function(sum, elem) { return sum + elem; }, 0); console.log(res);

Result of executing the code:

21

Example

Let's find the sum of all positive numbers in the array:

let arr = [1, -2, -3, 4, 5, -6]; let res = arr.reduce(function(sum, elem) { if (elem >= 0) { return sum + elem; } else { return sum; } }, 0); console.log(res);

Result of executing the code:

10
