join method

Methodjoinconcatenates array elements into a string with the specified delimiter (it will be inserted between array elements). The separator is specified as a method parameter and is optional. If it is not specified, the default separator will be a comma. If you want to merge array elements without a delimiter, specify it as an empty string''.

Syntax

массив.join([разделитель]);

Example

Let some array be given. Let's concatenate the elements of this array into a delimited string'-':

let arr = [1, 2, 3]; let str = arr.join('-'); console.log(str);

Result of executing the code:

'1-2-3'

Example

Now let's not specify a separator and the default separator will be a comma:

let arr = [1, 2, 3]; let str = arr.join(); console.log(str);

Result of executing the code:

'1,2,3'

Example

Let's merge the array elements without any separator:

let arr = [1, 2, 3]; let str = arr.join(''); console.log(str);

Result of executing the code:

'123'

Example. Application

Let's reverse the characters in the string. To do this, we split the string into an array usingsplitby separator''(this delimiter will put each character of the string into a separate element of the array), let's reverse this array withreverseand then merge the inverted array back usingjoin:

let str = '123456789'; let arr1 = str.split(''); let arr2 = arr1.reverse(); let res = arr2.join(''); console.log(res);

Result of executing the code:

'987654321'

Example. Application

Let's simplify the solution to the previous problem - merge all the commands into a chain:

let str = '123456789'; let res = str.split('').reverse().join(''); console.log(res);

Result of executing the code:

'987654321'

Example. Application

The date is given in the format'2025-12-31'. Let's make it into a format'31.12.2025'. To do this, we split the string into an array usingsplit, let's flip this array usingreverseand then merge the inverted array back usingjoin:

let date = '2025-12-31'; let res = date.split('-').reverse().join('.'); console.log(res);

Result of executing the code:

'31.12.2025'
