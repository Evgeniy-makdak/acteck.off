at method

Methodatreturns the array element at the given index. The method parameter specifies a positive or negative integer. In the first case, the element is searched from the beginning of the array, in the second, from the end.

Syntax

массив.at(число);

Example

Let's find the first element of the array:

let res = ['a', 'b', 'c'].at(0); console.log(res);

Result of executing the code:

'a'

Example

Now let's set the method parameter to a negative number:

let res = ['a', 'b', 'c'].at(-1); console.log(res);

Result of executing the code:

'c'

Example

Now let's find the element by specifying a fractional number in the parameter:

let res = ['a', 'b', 'c'].at(1.8); console.log(res);

As can be seen from the resulting result, the method returns an element that corresponds to the integer part of the fractional number:

'b'
