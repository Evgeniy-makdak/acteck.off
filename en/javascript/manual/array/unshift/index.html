unshift method

Methodunshiftadds an unlimited number of new elements to the beginning of the array. In this case, the original array is changed, and the result is the new length of the array.

Syntax

массив.unshift(элемент, элемент, элемент...);

Example

Let's add two more new elements to the beginning of the array and output the changed array:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.unshift('1', '2'); console.log(arr);

Result of executing the code:

['1', '2', 'a', 'b', 'c', 'd', 'e']

Example

Let's add two new elements and output the new length of the array:

let arr = ['a', 'b', 'c', 'd', 'e']; let length = arr.unshift('1', '2'); console.log(length);

Result of executing the code:

7
