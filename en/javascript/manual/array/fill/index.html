fill method

Methodfillfills the array with the given values. The first parameter of the method specifies the desired value. In the second and third optional parameters, we specify the starting and ending position for filling, respectively.

Syntax

массив.fill(элемент, [начальная позиция, [конечная позиция]]);

Example

Let's fill the array using this method:

let res = [1, 2, 3, 4].fill('!'); console.log(res);

Result of executing the code:

['!', '!', '!', '!']

Example

Now let’s indicate from what position the array needs to be filled:

let res = ['a', 'b', 'c'].fill('!', 1); console.log(res);

Result of executing the code:

['a', '!', '!']

Example

Let's fill the array by specifying, in addition to the starting position, also the ending position:

let res = ['a', 'b', 'c', 'e'].fill('!', 2, 3); console.log(res);

Result of executing the code:

['a', 'b', '!', 'e']

Example

Now let’s start filling the array from the very end, specifying the index as the second parameter-1:

let res = ['a', 'b', 'c', 'e'].fill('!', -1); console.log(res);

Result of executing the code:

['a', 'b', 'c', '!']
