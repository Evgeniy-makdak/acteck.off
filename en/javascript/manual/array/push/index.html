push method

Methodpushadds an unlimited number of elements to the end of the array. In this case, the original array is changed, and the result is the new length of the array.

Syntax

массив.push(элемент, элемент, элемент...);

Example

Let's add two more new elements to the end of the array:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.push('1', '2'); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd', 'e', '1', '2']

Example

Let's add two new elements to the array and print the new length of the array:

let arr = ['a', 'b', 'c', 'd', 'e']; let length = arr.push('1', '2'); console.log(length);

Result of executing the code:

7

Example. Application

Let's fill the array with numbers from1to9:

let arr = []; for (let i = 1; i <= 9; i++) { arr.push(i) } console.log(arr);

Result of executing the code:

[1, 2, 3, 4, 5, 6, 7, 8, 9]
