slice method

Methodslicecuts and returns the specified portion of the array. The array itself does not change.

The first parameter indicates the number of the array element from which cutting begins, and the second parameter indicates the number of the element at which cutting will end (in this case, the element with this number will not be included in the cut part). The second parameter is optional. If you do not specify it, the subarray will be taken from the element specified in the first parameter to the end of the array.

It can also take negative values. In this case, the counting of the element at which the trimming ends begins from the end of the array. The last element will have the number-1.

Syntax

массив.slice(откуда отрезать, [докуда отрезать]);

Example

Let's cut out elements from the array from zero to the second, not inclusive (the second one will not be cut):

let arr = ['a', 'b', 'c', 'd', 'e']; let sub = arr.slice(0, 2); console.log(sub);

Result of executing the code:

['a', 'b']

Example

Let's cut from the first element to the end of the array. To do this, we do not set the second parameter:

let arr = ['a', 'b', 'c', 'd', 'e']; let sub = arr.slice(1); console.log(sub);

Result of executing the code:

['b', 'c', 'd', 'e']

Example

Let's cut out the elements from the second to the penultimate (-1points to the last element and it will not be included in the extracted part):

let arr = ['a', 'b', 'c', 'd', 'e']; let sub = arr.slice(1, -1); console.log(sub);

Result of executing the code:

['b', 'c', 'd']

The advantage of this approach is that part of the array will always be cut, not including the last element, regardless of the size of the array.
