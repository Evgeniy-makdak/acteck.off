Свойство length

Свойство length позволяет узнать и установить длину массива.

Синтаксис

массив.length;

Пример

Давайте выведем длину массива arr:

let arr = ['a', 'b', 'c', 'd', 'e']; console.log(arr.length);

Результат выполнения кода:

5

Пример

Давайте сократим массив до 2-х элементов:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.length = 2; console.log(arr);

Результат выполнения кода:

['a', 'b']
