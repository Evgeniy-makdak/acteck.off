sort method

Methodsortsorts the array in lexicographical order and returns the already modified array. An optional parameter can be used to specify your own sorting function.

Syntax

массив.sort([функция]);

Example

Let's sort the array:

let arr = ['d', 'b', 'a', 'c']; console.log(arr.sort());

Result of executing the code:

['a', 'b', 'c', 'd']

Example

Let's sort an array of numbers:

let arr = [4, 1, 7, 2]; console.log(arr.sort());

Result of executing the code:

[1, 2, 4, 7]

Example

Let's add numbers to the array and sort it:

let arr = [1, 123, 2, 4, 111, 7]; console.log(arr.sort());

As a result of executing the code, we will see that the numbers are not arranged in ascending order, but in lexicographic order, i.e. numbers are compared to each other as strings. In this case, the comparison is made for each character, whether the code of the first character is greater than the code of the neighboring one, etc. Result:

[1, 111, 123, 2, 4, 7]

Example

Now let's sort the array using a function in which we will write a condition - if the first parameter is greater than the second, then we rearrange it (for this inreturnany positive number). Otherwise, no permutation is required and we indicate0or any negative number:

let arr = [1, 123, 2, 4, 111, 7]; arr.sort(function(a, b) { if (a > b) { return 1; } else { return -1; } }); console.log(arr);

The result of executing the code:

[1, 2, 4, 7, 111, 123]

Example

For convenience, let's shorten the code in our function from the previous example:

let arr = [1, 123, 2, 4, 111, 7]; arr.sort(function(a, b) { return a - b; }); console.log(arr);

The result of executing the code:

[1, 2, 4, 7, 111, 123]

Example

Now, using the function, we will sort the array in descending order:

let arr = [1, 123, 2, 4, 111, 7]; arr.sort(function(a, b) { return b - a; }); console.log(arr);

The result of executing the code:

[123, 111, 7, 4, 2, 1]

Example

Let's sort an array of objects by key'one'in ascending order:

let arr = [ {one: 1, two: 2}, {one: 7, two: 1}, {one: 3, two: 3} ]; arr.sort(function(a, b) { return a.one - b.one; }); console.log(arr);

The result of executing the code:

[ {one: 1, two: 2}, {one: 3, two: 3}, {one: 7, two: 1} ]

Example

Now let's sort the array of objects by key'two':

let arr = [ {one: 1, two: 2}, {one: 7, two: 1}, {one: 3, two: 3} ]; arr.sort(function(a, b) { return a.two - b.two; }); console.log(arr);

The result of executing the code:

[ {one: 7, two: 1}, {one: 1, two: 2}, {one: 3, two: 3} ]

Example

Let's check if the array has changed after applying the methodsort:

let arr = ['b', 'a', 'd', 'c']; let res = arr.sort(); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd']
