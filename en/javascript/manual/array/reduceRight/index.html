reduceRight method

MethodreduceRightworks exactly the same asreduce- watch it for full understanding. The only difference:reduceiterates elements from left to right, andreduceRight- from right to left.

Syntax

массив.reduceRight(function(промежуточный результат, элемент, индекс, массив) { return новый промежуточный результат; }, начальное значение);

Example

Let's find the sum of the array elements:

let arr = [1, 2, 3, 4, 5, 6]; let res = arr.reduceRight(function(sum, elem) { return sum + elem; }, 0); console.log(res);

Result of executing the code:

21

Example

Let's merge a two-dimensional array into a one-dimensional one:

let arr = [['a', 'b'], ['c'], ['d', 'e']]; let res = arr.reduceRight(function(elem1, elem2) { return elem1.concat(elem2); }, []); console.log(res);

Result of executing the code:

['d', 'e', 'c', 'a', 'b']
