map method

Methodmapallows you to apply a given function to each element of the array. In this case, the method does not change the original array, but returns the modified one.

The method in the parameter receives a function that will be executed for each element of the array. What this function will return viareturnfor an array element, will become the new value of this element (see examples).

You can pass it to a function3parameter. If these parameters are present (they are not required), then the first will automatically include the array element, the second will include its number in the array (index), and the third will contain the array itself.

Syntax

let новый массив = массив.map(function(элемент, индекс, массив) { код return измененный элемент; });

Example

Let's create an array, each element of which is twice the size of the corresponding element of the initial array:

let arr = [1, 2, 3, 4, 5]; let res = arr.map(function(elem) { return elem * 2; }); console.log(res);

Result of executing the code:

[2, 4, 6, 8, 10]

Example

Let's create an array, each element of which is obtained like this - the value of the element is multiplied by its serial number in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.map(function(elem, index) { return elem * index; }); console.log(res);

Result of executing the code:

[0, 2, 6, 12, 20]

Example

If necessary, you can pass the array itself into the third parameter:

let arr = [1, 2, 3, 4, 5]; let res = arr.map(function(elem, index, arr) { тут будет доступен массив arr });

Example

Methodmapcan also be used to iterate through multidimensional arrays. Let, for example, be given the following array:

let arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

Let's iterate through this arraymapand display its elements to the console:

let arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]; let result = arr.map(function(elem) { console.log(elem); });

As a resultconsole.logwill output[1, 2, 3], Then[4, 5, 6], Then[7, 8, 9].

As you can see, into the variableelemsubarrays fall into. Let's now apply the methodmapto each subarray and square each element:

let arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]; let result = arr.map(function(elem) { return elem.map(function(num) { return num * num; }); }); console.log(result);
