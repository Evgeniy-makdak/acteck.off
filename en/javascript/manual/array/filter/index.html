filter method

Methodfilterallows you to filter array elements, leaving only elements that meet a certain condition. The method in the parameter receives a function that will be executed for each element of the array. As a result, the method returns a new array, which will include only those elements for which the passed function will returntrue.

You can pass three parameters to the function. If these parameters are present (they are not required), then the first will automatically include the array element, the second will include its number in the array (index), and the third will contain the array itself.

Syntax

let новый массив = массив.filter(function(элемент, индекс, массив) { код return true или false });

Example

Let's filter the array, leaving only positive numbers in it:

let arr = [-2, 5, 1, -5, -1, 1, 3, 4, -1]; let res = arr.filter(function(elem) { if (elem >= 0) { return true; } else { return false; } }); console.log(res);

Result of executing the code:

[5, 1, 1, 3, 4]

Example

Let's shorten the code usingarrow function:

let arr = [-2, 5, 1, -5, -1, 1, 3, 4, -1]; let res = arr.filter(elem => { if (elem >= 0) { return true; } else { return false; } }); console.log(res);

Result of executing the code:

[5, 1, 1, 3, 4]

Example

Let's shorten the code usingreduction of logical operations:

let arr = [-2, 5, 1, -5, -1, 1, 3, 4, -1]; let res = arr.filter(elem => { return elem >= 0; }); console.log(res);

Result of executing the code:

[5, 1, 1, 3, 4]

Example

Let's shorten the code using the features of arrow functions:

let arr = [-2, 5, 1, -5, -1, 1, 3, 4, -1]; let res = arr.filter(elem => elem > 0); console.log(res);

Result of executing the code:

[5, 1, 1, 3, 4]

Example

If necessary, you can pass the array itself into the third parameter:

let res = arr.filter(function(elem, index, arr) { тут будет доступен массив arr });
