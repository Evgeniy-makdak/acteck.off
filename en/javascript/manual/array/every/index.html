Every method

Methodeverychecks the array elements according to the passed function. Method returnstrue, if for all elements of the array the passed function returnstrue, otherwise the method returnsfalse.

You can pass three parameters to the function. If these parameters are present (they are not required), then the first will automatically include the array element, the second will include its number in the array (index), and the third will contain the array itself.

Syntax

массив.every(function(элемент, индекс, массив) { return true или false; });

Example

Let's check that all elements in the array of numbers are positive:

let arr = [1, 2, 3, 4, 5]; let check = arr.every(function(elem) { if (elem >= 0) { return true; } else { return false; } }); console.log(check);

Result of executing the code:

true

Example

Let's check that the product of an element and its ordinal number is always less30:

let arr = [1, 2, 3, 4, 5]; let check = arr.every(function(elem, index) { if (elem * index < 30) { return true; } else { return false; } }); console.log(check);

Result of executing the code:

true

Example

If necessary, you can pass the array itself into the third parameter:

let check = arr.every(function(elem, index, arr) { тут будет доступен массив arr });
