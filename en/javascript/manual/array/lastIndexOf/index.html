lastIndexOf method

MethodlastIndexOfsearches for an element in an array. Returns the number of the last element found, or-1, if there is no such element. The first parameter we specify is the element to search for, the second (optional) is the position from which to start the search. The search is carried out from the end of the array to the beginning.

Syntax

массив.lastIndexOf(элемент, [откуда начинать]);

Example

Let's find the position of the last three in the array:

let arr = [1, 2, 3, 3, 3, 4, 5]; let res = arr.lastIndexOf(3); console.log(res);

Result of executing the code:

4

Example

Now let's try to find an element that is not in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.lastIndexOf(6); console.log(res);

Result of executing the code:

-1

Example

Let's start the search from a given position. As a result, the last three will be found, excluding the missing ones:

let arr = [1, 2, 3, 3, 4, 5, 3]; let res = arr.lastIndexOf(3, 4); console.log(res);

Result of executing the code:

3
