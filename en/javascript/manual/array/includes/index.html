includes method

Methodincludeschecks the presence of an element in the array. The parameter takes the value to search for. If such an element is in the array, then the method returnstrue, and if not, thenfalse.

Syntax

массив.includes(элемент);

Example

Let's check for the presence of an element in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.includes(3); console.log(res);

Result of executing the code:

true

Example

Let now the element being checked is not in the array:

let arr = [1, 2, 3, 4, 5]; let res = arr.includes(6); console.log(res);

Result of executing the code:

false
