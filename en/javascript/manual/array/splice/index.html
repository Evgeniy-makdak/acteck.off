splice method

Methodspliceremoves or adds elements to an array. You can only remove elements, only add, or do both at the same time. The method is very universal and difficult to understand. The method modifies the array itself and returns an array of deleted elements.

The first parameter the method takes is the number of the array element that needs to be removed. The second parameter is how many array elements should be removed. If you put it in0- then the elements will not be deleted (only new ones will be added). Next, separated by commas, are the elements that need to be added to the array (they are optional parameters). These elements will be added instead of the removed array elements.

If there was no deletion (when the second parameter0) - elements will be inserted into the array starting from the position specified by the first parameter of the method. The first parameter can have a negative value. In this case, position counting will begin not from the beginning of the array, but from the end. The last element will have the number-1.

Syntax

массив.splice(откуда удаляем, сколько удаляем, [вставить], [вставить]...);

Example

Let's remove three elements, starting with the first one:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.splice(1, 3); console.log(arr);

Result of executing the code:

['a', 'e']

Example

Let's print an array of deleted elements:

let arr = ['a', 'b', 'c', 'd', 'e']; let del = arr.splice(1, 3); console.log(del);

Result of executing the code:

['b', 'c', 'd']

Example

Let's first remove element number2, and then instead of it we will insert three more new elements:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.splice(2, 1, '1', '2', '3'); console.log(arr);

Result of executing the code:

['a', 'b', '1', '2', '3', 'd', 'e']

Example

Let's now not delete anything, but to the position2, insert three more new elements:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.splice(2, 0, '1', '2', '3'); console.log(arr);

Result of executing the code:

['a', 'b', '1', '2', '3', 'c', 'd', 'e']

Example. Negative value

Let's remove the penultimate element:

let arr = ['a', 'b', 'c', 'd', 'e']; arr.splice(-2, 1); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'e']
