some method

Methodsomechecks the array elements according to the passed function. This function is passed as a method parameter and is executed for each element of the array. Method returnstrue, if for at least one array element the passed function returnstrue, otherwise the method returnsfalse.

You can pass three parameters to the function. If these parameters are present (they are not required), then the first will automatically include the array element, the second will include its number in the array (index), and the third will contain the array itself.

Syntax

массив.some(function(элемент, индекс, массив) { });

Example

Let's check if the array of numbers contains at least one positive number:

let arr = [-1, -2, -3, 4]; let check = arr.some(function(elem) { if (elem >= 0) { return true; } else { return false; } }); console.log(check);

Result of executing the code:

true

Example

Let's check that at least one product of an element and its ordinal number is greater than or equal to20:

let arr = [1, 2, 3, 4, 5]; let check = arr.some(function(elem, index) { if (elem * index >= 20) { return true; } else { return false; } }); console.log(check);

Result of executing the code:

true

Example

If necessary, you can pass the array itself into the third parameter:

let check = arr.some(function(elem, index, arr) { тут будет доступен массив arr });
