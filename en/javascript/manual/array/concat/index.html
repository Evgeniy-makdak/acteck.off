concat method

Methodconcatmerges the specified arrays into one common array. The method is applied to one of the arrays, and the remaining arrays for merging are passed in the method parameters. In this case, the method does not change the original array, but returns a new one.

Syntax

массив.concat(еще массив, и еще массив, и еще...);

Example

Let's mix it up3array into one using the methodconcat:

let arr1 = [1, 2]; let arr2 = [3, 4]; let arr3 = [5, 6]; let res = arr1.concat(arr2, arr3); console.log(res);

Result of executing the code:

[1, 2, 3, 4, 5, 6]

Example

Now let's merge the two arrays together:

let arr1 = [1, 2]; let arr2 = [3, 4]; let res = arr1.concat(arr2); console.log(res);

Result of executing the code:

[1, 2, 3, 4]

Example

You can pass not only arrays, but also specific values:

let arr1 = [1, 2]; let arr2 = [3, 4]; let res = arr1.concat(arr2, 5, 6); console.log(res);

Result of executing the code:

[1, 2, 3, 4, 5, 6]

Example

You can apply the method to an empty array. In this case, all merged arrays will be passed as parameters of the method:

let arr1 = [1, 2]; let arr2 = [3, 4]; let arr3 = [5, 6]; let res = [].concat(arr1, arr2, arr3); console.log(res);

Result of executing the code:

[1, 2, 3, 4, 5, 6]
