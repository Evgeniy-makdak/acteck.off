flat method

Methodflatreduces the nesting level of a multidimensional array. Can either make the array one-dimensional or reduce the dimension by a given value.

Syntax

let newArray = массив.flat(уровень подъема);

Example

Let's get a new array without specifying any value in the parameter:

let arr = ['a', 'b', ['c', 'd']]; let res = arr.flat(); console.log(res);

As a result of executing the code, we will see that the subarrays have moved up one level:

['a', 'b', 'c', 'd']

Example

Now let's apply the methodflatto an array with two levels of nesting:

let arr = ['a', 'b', ['c', 'd', ['e', 'f']]]; let res = arr.flat(); console.log(res);

Result of executing the code:

['a', 'b', 'c', 'd', ['e', 'f']]

Example

Let's indicate the nesting level:

let arr = ['a', 'b', ['c', 'd', ['e', 'f']]]; let res = arr.flat(2); console.log(res);

Result of executing the code:

['a', 'b', 'c', 'd', 'e', 'f']

Example

If we don’t want to bother with the nesting level, but want to combine the entire array, then we use the parameterInfinity:

let arr = ['a', 'b', ['c', 'd', ['e', 'f', ['g', 'h', ['i']]]]]; let res = arr.flat(Infinity); console.log(res);

Result of executing the code:

[ 'a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i' ]
