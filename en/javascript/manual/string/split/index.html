split method

Methodsplitsplits strings into an array using the separator specified in the first parameter. If it is not specified, the entire string will be returned. If it is specified as empty quotes, then each character in the string will end up in a separate array element. The second optional parameter can specify the maximum number of elements in the resulting array.

Syntax

строка.split([разделитель], [максимальное количество элементов]);

Example

Let's be given some string with hyphens. Let's split this string into an array by delimiter'-':

let str = 'ab-cd-ef'; let arr = str.split('-'); console.log(arr);

Result of executing the code:

['ab', 'cd', 'ef']

Example

Let's split the string by delimiter again'-', however, as the second parameter we will indicate the maximum number of elements in the resulting array, for example,2. In this case, only2element:

let str = 'ab-cd-ef'; let arr = str.split('-', 2); console.log(arr);

Result of executing the code:

['ab', 'cd']

Example

Let's write each character of the string into a separate array element:

let str = 'abcde'; let arr = str.split(''); console.log(arr);

Result of executing the code:

['a', 'b', 'c', 'd', 'e']

Example

Let's be the first3We will write the string character into separate elements of the array:

let str = 'abcde'; let arr = str.split('', 3); console.log(arr);

Result of executing the code:

['a', 'b', 'c']

Example

Let's usesplitLet's split the string with numbers. Note that the result will be an array of strings, not an array of numbers:

let str = '12345'; let arr = str.split(''); console.log(arr);

Result of executing the code:

['1', '2', '3', '4', '5']

Example. Application

Let's reverse the characters in the string. To do this, we split the string into an array usingsplitby separator''(this will place each character of the string into a separate element of the array), let's reverse this array withreverseand then merge the inverted array back usingjoin:

let str = '123456789'; let arr1 = str.split(''); let arr2 = arr1.reverse(); let res = arr2.join(''); console.log(res);

Result of executing the code:

'987654321'

Example. Application

Let's simplify the solution to the previous problem - merge all the commands into a chain:

let str = '123456789'; let res = str.split('').reverse().join(''); console.log(res);

Result of executing the code:

'987654321'

Example. Application

Given a string with numbers. Let's find the sum of the digits from this line. To do this, we will split the string into an array, and then iterate over this array and find its sum. There's a catch waiting for us:splitreturns strings, so when summing, we convert these string digits into real numbers usingNumber:

let str = '12345'; let arr = str.split(''); let sum = 0; for (let i = 0; i < arr.length; i++) { sum += Number(arr[i]); } console.log(sum);

Result of executing the code:

15

Example. Application

A number is given. Let's write each digit of this number into a separate array element. There is a catch here -splitapplies only to strings, and we have a number. First we convert the number to a string usingString, and then applysplit:

let num = 12345; let str = String(num); let arr = str.split(''); console.log(arr);

Result of executing the code:

['1', '2', '3', '4', '5']
