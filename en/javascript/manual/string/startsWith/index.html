startsWith method

MethodstartsWithchecks whether the string begins with the substring specified in the first parameter. If it starts, it returnstrue, and if it doesn’t start, thenfalse. As the second optional parameter, the method takes the position from which to start checking (by default, from the beginning of the line).

Syntax

строка.startsWith(что ищем, [начало проверки]);

Example

Let's check if the string starts with a given substring:

let str = 'abcde'; let res = str.startsWith('abc'); console.log(res);

Result of executing the code:

true

Example

Let's check if the string starts with a given substring:

let str = 'abcde'; let res = str.startsWith('xxx'); console.log(res);

Result of executing the code:

false

Example

Let's start checking from a given position:

let str = 'abcde'; let res = str.startsWith('bc', 1); console.log(res);

Result of executing the code:

true
