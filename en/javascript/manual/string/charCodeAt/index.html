charCodeAt method

MethodcharCodeAtreturns the character code (numeric value) at a specific position in the string. Character numbering starts with0. If the specified number is greater than the last character of the string, then the method returnsNaN.

Syntax

строка.charCodeAt(номер символа в строке);

Example

Let's print the letter code'A'(character with position0):

let str = 'ABC'; let code = str.charCodeAt(0); console.log(code);

Result of executing the code:

65

Example

Let's look for a character whose number is not in the line:

let str = 'ABC'; let code = str.charCodeAt(4); console.log(code);

Result of executing the code:

NaN
