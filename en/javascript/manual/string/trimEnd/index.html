trimEnd method

MethodtrimEndremoves spaces from the end of a string and returns a new string without changing the original string.

Syntax

строка.trimEnd();

Example

Let's remove extra spaces in the string' abcde ':

let res = ' abcde '.trimEnd(); console.log(res);

As a result of executing the code, we will see that there are no longer spaces at the end of the line:

' abcde'

See also

  • methodtrimStart,
    which removes spaces at the beginning of a line
  • methodtrim,
    which removes whitespace from the edges of a string
