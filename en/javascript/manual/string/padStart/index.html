padStart method

MethodpadStartpads the beginning of the current line with another line (several times if necessary) to a specified length. The first parameter in the method is the desired length of the string, and the second optional parameter is the string with which it will be filled.

Syntax

строка.padStart(длина строки, [строка для заполнения]);

Example

Let's complete the line'ab'so that its length is4symbol:

let res = 'ab'.padStart(4); console.log(res);

Result of executing the code:

' ab'

Example

Let's increase the string length to8characters by filling it'0':

let res = 'abcde'.padStart(8, '0'); console.log(res);

Result of executing the code:

'000abcde'

Example

Let's set the first parameter of the method to a number that is less than the length of the string:

let res = 'ab'.padStart('1'); console.log(res);

As a result of executing the code, the entire string will be returned:

'ab'

  • propertylength,
