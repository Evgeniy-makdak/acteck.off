substr method

Methodsubstrreturns a substring from a string (the original string is not changed). The first parameter specifies the position number from which the method will start cutting the character (numbering starts from zero), and the second parameter specifies how many characters to cut.

The first parameter can take negative values. In this case, the character from which the trimming begins will be counted from the end of the line. The last character has a number-1. The second parameter is optional; if it is not specified, then all characters up to the end of the line will be cut.

Syntax

строка.substr(откуда отрезать, [сколько символов отрезать]);

Example

Let a string be given. Let's cut the first ones out of it3symbol:

let str = 'abcde'; let sub = str.substr(0, 3); console.log(sub);

Result of executing the code:

'abc'

Example

Let's now cut out the characters with2-th position and until the end of the line (for this we will not set the second parameter of the method):

let str = 'abcde'; let sub = str.substr(2); console.log(sub);

Result of executing the code:

'cde'

Example

Let's cut out the substring with3th character from the end and take2symbol:

let str = 'abcde'; let sub = str.substr(-3, 2); console.log(sub);

Result of executing the code:

'cd'

Example

Now let's cut out the substring with3th character from the end and take the entire remaining line to the end (for this we will not set the second parameter of the method):

let str = 'abcde'; let sub = str.substr(-3); console.log(sub);

Result of executing the code:

'cde'

Example

Let's cut out the last character of the line:

let str = 'abcde'; let sub = str.substr(-1); console.log(sub);

Result of executing the code:

'e'
