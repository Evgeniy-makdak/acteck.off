substring method

Methodsubstringreturns a substring from a string (the original string is not changed). The first parameter specifies the number of the character from which the method starts cutting (numbering starts from zero), and the second parameter specifies the number of the character at which cutting should end (the character with this number is not included in the cut part). The second parameter is optional; if it is not specified, then all characters up to the end of the line will be cut.

Syntax

строка.substring(откуда начать отрезать, [докуда отрезать]);

Example

Let a string be given. Let's cut out characters from the first to the third:

let str = 'abcde'; let sub = str.substring(1, 3); console.log(sub);

The result of executing the code (symbol with number3will not fit into the cut out part):

'bc'

Example

Let's now cut out the characters with1-th and until the end of the line (for this we will not set the second parameter of the method):

let str = 'abcde'; let sub = str.substring(1); console.log(sub);

Result of executing the code:

'bcde'
