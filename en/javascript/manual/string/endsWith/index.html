endsWith method

MethodendsWithchecks whether the string ends with the substring specified in the first parameter. If it ends, it returnstrue, and if it doesn’t end, thenfalse. The second optional parameter can force you to specify the length of the string. In this case, the check will not be the actual end of the line, but the specified one.

Syntax

строка.endsWith(что ищем, [длина строки]);

Example

Let's check if the string ends with a given substring:

let str = 'abcde'; let res = str.endsWith('cde'); console.log(res);

Result of executing the code:

true

Example

Let's check if the string ends with a given substring:

let str = 'abcde'; let res = str.endsWith('xxx'); console.log(res);

Result of executing the code:

false

Example

Let's force the length of the string to be checked:

let str = 'abcde'; let res = str.endsWith('bc', 3); console.log(res);

Result of executing the code:

true
