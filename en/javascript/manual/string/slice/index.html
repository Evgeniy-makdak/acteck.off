slice method

Methodslicereturns a substring from a string (the original string is not changed). The first parameter specifies the number of the line character from which cutting begins, and the second parameter specifies the number of the character at which cutting ends (in this case, the character with this number will not be included in the cut part). The second parameter is optional. If you do not specify it, the substring will be taken from the character specified in the first parameter to the end of the line. It can also take negative values. In this case, the counting of the character at which the trimming ends begins from the end of the line. The last character has a number-1.

Syntax

строка.slice(откуда отрезать, [докуда отрезать]);

Example

Let a string be given. Let's cut out characters from the first to the third:

let str = 'abcde'; let sub = str.slice(1, 3); console.log(sub);

The result of executing the code (symbol with number3will not fit into the cut out part):

'bc'

Example

Let's now cut out the characters with1-th and until the end of the line (for this we will not set the second parameter of the method):

let str = 'abcde'; let sub = str.slice(1); console.log(sub);

Result of executing the code:

'bcde'

Example

Let's now cut out the characters from the position1by position-2:

let str = 'abcde'; let sub = str.slice(1, -2); console.log(sub);

The result of executing the code (symbol with number-2will not fit into the cut out part):

'bc'
