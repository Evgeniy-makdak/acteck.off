repeat method

Methodrepeatcreates a new string containing the specified number of copies of the original string concatenated together.

Syntax

строка.repeat(число);

Example

Let's specify a number in the method parameter1:

let res = 'abcde'.repeat(1); console.log(res);

As a result of executing the code, our line will remain the same:

'abcde'

Example

Now we will copy the line2times:

let res = 'abcde'.repeat(2); console.log(res);

Result of executing the code:

'abcdeabcde'

Example

Let's try to copy the line-1once:

let res = 'abcde'.repeat(-1); console.log(res);

After execution we receive an error:

'RangeError: Invalid count value: -1'

Example

If we specify a number in the parameter0:

let res = 'abcde'.repeat(0); console.log(res);

As a result of executing the code, we will get an empty string:

''
