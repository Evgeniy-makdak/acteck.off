toUpperCase method

MethodtoUpperCaseconverts the string to upper case (makes capital letters from small ones). This returns a new string, but does not change the original string.

Syntax

строка.toUpperCase();

Example

Let's convert all letters to capital letters:

let str = 'abcde'; console.log(str.toUpperCase());

Result of executing the code:

'ABCDE'

Example

Also using the methodsliceYou can convert individual letters to uppercase:

let str = 'abcde'; let res = str.slice(0, 1).toUpperCase() + str.slice(1); console.log(res);

Result of executing the code:

'Abcde'
