length property

Propertylengthallows you to find out the length of the string. Length refers to the number of characters in it.

Syntax

строка.length;

Example

Let in the variablestra string is stored. Let's find its length:

let str = 'abcde'; console.log(str.length);

Result of executing the code:

5

Example

The length of the string also includes spaces. Let's find the length of a string that has a space in it:

let str = 'word1 word2'; console.log(str.length);

Result of executing the code:

11
