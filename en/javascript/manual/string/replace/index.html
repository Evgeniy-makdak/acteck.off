replace method

Methodreplacesearches and replaces parts of a string. The first parameter is the substring that we are replacing, and the second is the substring that we are replacing with.

Syntax

строка.replace(что заменяем, на что заменяем);

Example

Let's replace in the line'ab'on'!':

let str = 'abcde'; let res = str.replace('ab', '!'); console.log(res);

Result of executing the code:

'!cde'

Example

Let's try to replace all matches in the string:

let str = 'ab cde ab'; let res = str.replace('ab', '!'); console.log(res);

We will see that the replacement occurred only in the first match:

'! cde ab'

Example

Even though the methodreplacechanges only the first match, you can replace all matches using a loop:

let elem = 'ab'; let str = 'ab cde ab'; while (str.includes(elem)) { str = str.replace(elem, '!'); } console.log(str);

Result of executing the code:

'! cde !'
