includes method

Methodincludesperforms a case-sensitive search for the specified string in the current string. The first parameter the method takes is the string to be found, and the second optional parameter is the position from which to start the search. After execution the method returnstrueorfalse.

Syntax

строка.includes(что ищем, [откуда начинать поиск]);

Example

Let's check if there is a string'ab'in line'abcde':

let res = 'abcde'.includes('ab'); console.log(res);

Result of executing the code:

true

Example

Now let's search from the fifth character of the current line:

let res = 'ab cd ef'.includes('cd', 5); console.log(res);

Result of executing the code:

false

Example

Let's do a case-sensitive search for a string:

let res = 'abcde'.includes('AB'); console.log(res);

Result of executing the code:

false
