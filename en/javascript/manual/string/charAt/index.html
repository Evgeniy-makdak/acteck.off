charAt method

MethodcharAtreturns the character at the specified position in the string. The position is specified by the method parameter (numbering starts from zero). If the specified position is greater than the position of the last character, an empty string will be returned.

Syntax

строка.charAt(позиция символа);

Example

Let's print the character at position zero:

let str = 'abcde'; let char = str.charAt(0); console.log(char);

Result of executing the code:

'a'

Example

Now an empty string will be output, since the position entered is greater than the position of the last character:

let str = 'abcde'; let char = str.charAt(30); console.log(char);

Result of executing the code:

''
