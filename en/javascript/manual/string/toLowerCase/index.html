toLowerCase method

MethodtoLowerCaseconverts string characters to lower case (makes small ones from capital letters). In this case, we get a new string, but the original string remains unchanged.

Syntax

строка.toLowerCase();

Example

Let's convert all capital letters of the string to small letters:

let str = 'ABCDE'; console.log(str.toLowerCase());

Result of executing the code:

'abcde'

Example

Combining methodtoLowerCaseand methodsliceYou can set lowercase for individual parts of a string:

let str = 'ABCDE'; let res = str.slice(0, 1) + str.slice(1).toLowerCase(); console.log(res);

Result of executing the code:

'Abcde'
