Gracefully exchange variable values ​​using destructuring in ES6

Let us have two variables a and b. Let's make sure that variable a is filled with the value of variable b, and variable b is filled with the value of variable a. In ES5, you can't solve this problem without introducing a temporary variable, but ES6 allows you to solve the problem much more gracefully!

Code sources

let a = 1; let b = 2; [a, b] = [b, a]; console.log(a, b);
let a = 1; let b = 2; let c = 3; [a, b, c] = [b, c, a]; console.log(a, b, c);
