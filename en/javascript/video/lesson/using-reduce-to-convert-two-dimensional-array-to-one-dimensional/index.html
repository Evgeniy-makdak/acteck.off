Using the reduce method to convert a two-dimensional array to a one-dimensional one

In this video, I'll show you how to use the reduce method and arrow functions to convert a two-dimensional array into a one-dimensional array while preserving the order of the elements.

Code sources

let array = [[1, 2, 3], [4, 5], [6]]; // [1, 2, 3, 4, 5, 6] console.log(array.reduce((flat, current) => flat.concat(current), []));
