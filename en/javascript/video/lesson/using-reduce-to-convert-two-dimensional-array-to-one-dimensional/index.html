Использование метода reduce для преобразования двухмерного массива в одномерный

В данном видео я покажу вам применение метода reduce и стрелочных функций для преобразования двухмерного массива в одномерный с сохранением порядка элементов.

Исходники кода

let array = [[1, 2, 3], [4, 5], [6]]; // [1, 2, 3, 4, 5, 6] console.log(array.reduce((flat, current) => flat.concat(current), []));
