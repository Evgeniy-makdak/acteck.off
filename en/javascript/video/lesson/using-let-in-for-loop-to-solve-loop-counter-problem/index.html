Using let in a for loop to solve the loop counter problem

Let's say you have a set of DOM elements that you loop through. And let’s say you want to click on any element to display its serial number. In this case, a catch may await you - all elements will display the number of the last element + 1, and not their own! In this tutorial I will show you how to useletinstead ofvarwill easily solve this problem.

Code sources

p { height: 20px; border: 1px solid red; } <p></p> <p></p> <p></p> <p></p> <p></p> var elems = document.querySelectorAll('p'); for (let i = 0; i < elems.length; i++) { elems[i].addEventListener('click', function() { this.innerHTML = i; }); } //console.log(i);
