Getting a copy of an array using ES6 and using it in practice

Let us have some kind of array and at the same time we want to get a copy of it. Simply assigning an array to another variable won't copy it - you'll just end up with two variables referencing the same array. Let's see how ES6 handles this problem with a practical example.

Code sources

let arr = [1, 2, 3, 4, 5, 6, 10]; // [6, 5, 4, 3, 2, 1] let [...clone] = arr; clone.reverse(); let result = []; for (let i = 0; i < arr.length; i++) { result.push(arr[i] + clone[i]) } console.log(result);
