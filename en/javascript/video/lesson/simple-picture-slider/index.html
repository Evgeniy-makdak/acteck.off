Простой слайдер картинок на JavaScript

В уроке показана реализация простого слайдера картинок средствами JavaScript. В окне браузера всегда видна одна картинка, которая сменяется другой каждую секунду.

Исходники кода

<div id="slider"><img src=""></div> var images = ['1.png', '2.png', '3.png', '4.png', '5.png']; var slider = document.querySelector('#slider'); var img = slider.querySelector('img'); var i = 1; img.src = 'images/' + images[0]; window.setInterval(function() { img.src = 'images/' + images[i]; i++; if (i == images.length) { i = 0; } }, 1000);
