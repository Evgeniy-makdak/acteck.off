We pass an array to Math.max using the ES6 spread operator

As you know, the Math.max and Math.min functions cannot work with arrays. There are some clever tricks to get them to do this, but a more elegant way to do this is with the ES6 spread operator.

Code sources

let arr = [1, 20, 7, 6, 5]; console.log(Math.max.apply(null, arr));
let arr = [1, 20, 7, 6, 5]; console.log(Math.max(...arr));
