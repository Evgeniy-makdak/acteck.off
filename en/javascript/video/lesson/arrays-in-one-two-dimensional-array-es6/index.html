Passed arrays into one two-dimensional one on ES6

Let's make a function that will take an arbitrary number of arrays as a parameter and turn them into a two-dimensional array. On ES6 the solution will be elementary!

Code sources

function func(...arrs) { return arrs; } console.log(func([1, 2, 3], [4, 5, 6], [7, 8, 9]));
