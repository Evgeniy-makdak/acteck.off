The sequence number of a click on the screen using JavaScript

The lesson shows how to display the serial number of this click at the location of the click on the screen. The task itself is not very practical, but it demonstrates some useful features.

Code sources

var i = 0; window.addEventListener('click', function(event) { i++; var elem = document.createElement('div'); elem.innerHTML = i; document.body.appendChild(elem); elem.style.position = 'absolute'; elem.style.left = event.clientX + 'px'; elem.style.top = event.clientY + 'px'; });
