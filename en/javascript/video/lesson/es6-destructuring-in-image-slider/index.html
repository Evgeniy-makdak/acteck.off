Using ES6 destructuring in an image slider

Let's say we want to make a slider from3-x pictures successively replacing each other. Using ES6 destructuring will allow you to gracefully solve this problem.

Code sources

<img src="images/1.png"> <img src="images/2.png"> <img src="images/3.png"> let img = document.querySelectorAll('img'); setInterval(function() { [img[0].src, img[1].src, img[2].src] = [img[1].src, img[2].src, img[0].src]; }, 1000);
