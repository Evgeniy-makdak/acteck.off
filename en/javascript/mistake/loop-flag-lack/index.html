Error missing flag in JavaScript loop

Let us have some number:

let num = '12345';

Let's check if this number contains a given digit. A certain programmer has already written code that implements this:

for (let digit of num) { if (digit === '3') { console.log('есть'); } else { console.log('нет'); } }

This code, however, does not work correctly, outputting'есть'or'нет'for each array element. We need the result to be displayed only once. In this case, in combination with a loop, the problem is solved only through a flag:

let hasDigit = false; // флаг for (let digit of num) { if (digit === '3') { hasDigit = true; break; } } if (hasDigit) { console.log('есть'); } else { console.log('нет'); }
