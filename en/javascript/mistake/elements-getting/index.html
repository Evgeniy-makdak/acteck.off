Error getting elements in JavaScript

Sometimes novice programmers make the mistake of using the methodquerySelectorinstead of a methodquerySelectorAll.

Let's look at the characteristic features of this error. Let the paragraphs be given:

<p>1</p> <p>2</p> <p>3</p>

A certain programmer decided to obtain and display the texts of these paragraphs in the console. To do this, he obtained references to these elements into a variable by mistakenly using the wrong method:

let elems = document.querySelector('p');

Then he decided to loop through the paragraphs and print the text of each paragraph to the console:

for (let elem of elems) { console.log(elem.textContent); }

As a result, a characteristic error will appear in the consoleelems is not iterable. She says the variableelemsNotiterable, that is, it is not an array or something that can be iterated throughfor-of.
