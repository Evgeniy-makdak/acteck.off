Error accessing array of elements in JavaScript

Sometimes novice programmers try to work with an array of elements as if they had one element. Let's see what this error is all about.

Let the paragraphs be given:

<p>1</p> <p>2</p> <p>3</p>

Let a certain programmer decide to write the same meaning in the text of each paragraph. To do this, he received references to these elements in a variable:

let elems = document.querySelectorAll('p');

Then our programmer mistakenly took and treated our variable as if it contained one element, and not an array. As a result, the text of the paragraphs will not change, but, characteristically, there will be no error in the console:

elems.textContent = '!';

The correct solution here would be to loop through the array of elements and perform the required operation for each element separately:

for (let elem of elems) { elem.textContent = '!'; }
