Error changing array element in JavaScript

Let's look at the error that occurs when an array element is changed incorrectly. Let us be given the following array:

let arr = [1, 2, 3, 4, 5];

Let us also have a function that takes a number as a parameter and returns the square of that number:

function func(num) { return num ** 2; }

Let's use a loopfor-ofLet's iterate through our array and apply our function to each of its elements:

for (let elem of arr) { elem = func(elem); }

This is where beginners often make a mistake. The point is that changing the variableelemdoes not change the element in the array itself. Let's make sure of this:

console.log(arr); // массив не изменился

The array does not change due to the fact that in the variableelema copy of the element is stored, not a reference to it. Changeelemaffects the copy, but not the array itself.

To solve the problem, you need to change exactly the elements of the array:

for (let i = 0; i < arr.length; i++) { arr[i] = func(arr[i]); } console.log(arr); // теперь массив изменился
