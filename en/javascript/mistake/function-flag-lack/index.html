Error missing flag in function in JavaScript

Suppose we want to make a function that will take a digit and a number as a parameter, and check whether the given digit is in the number:

function func(needle, num) { // тут будет код }

This is how we will use our function:

console.log(func('3', '12345')); // выведет true

A certain programmer wrote a solution to this problem:

function func(needle, num) { for (let digit of num) { if (digit === needle) { return true; } else { return false; } } }

This solution, however, does not work correctly. The fact is that the cycle contains a condition in which in the first iteration either one will be fulfilledreturn, or the second, thereby exiting both the loop and the function.

Let's rewrite our code correctly usingimplicit flags in functions:

function func(needle, num) { for (let digit of num) { if (digit === needle) { return true; } } return false; }
