Key and element confusion in JavaScript for loop

Let us be given an array with the names of the months:

let arr = [ 'янв', 'фев', 'март', 'апр', 'май', 'июнь', 'июль', 'авг', 'сен', 'окт', 'ноя', 'дек' ];

Let in the variablemonththe number of the current month is stored:

let month = 10;

Let's use a loop to display all the months, with the current month displayed in italics.

A certain programmer has already solved this problem, however, he made a mistake and did not pay attention to the fact that in the variablemonththe month number is stored, not its name:

for (let i = 0; i < arr.length; i++) { if (month === arr[i]) { // ошибка тут document.write('<i>' + arr[i] + '</i>'); } else { document.write(arr[i]); } document.write('<br>'); }

Our programmer compares an array element with a variablemonth. But this variable stores the month number, not its name!

For the code to work correctly, you need to comparemonthwith the month number, that is, with the counter valuei:

for (let i = 0; i < arr.length; i++){ if (month === i) { // исправили ошибку document.write('<i>' + arr[i] + '</i>'); } else { document.write(arr[i]); } document.write('<br>'); }
