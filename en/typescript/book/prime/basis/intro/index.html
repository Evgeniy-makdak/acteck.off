Introduction to TypeScript

TypeScript is a dialect of the JavaScript language. This dialect extends JavaScript by adding some additional features to it.

The main feature of TypeScript is that it is a language withstatic typing. This means that the type of the variable cannot be changed during program execution. This makes it faster and easier to write large, complex programs that are easier to maintain, develop, scale, and test.

TypeScript does not run directly in browsers. Code in this language must be compiled into JavaScript using special programs. We will look at compilation in more detail in the next lesson.
