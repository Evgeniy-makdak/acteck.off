Controlling the type of a variable in an object in TypeScript

TypeScript controls the type of a variable with an object, prohibiting data of a different type from being written to it. Let's look at an example. Let us have an object with a user:

let user = {name: 'john', age: 30};

Let's try to write data of a different type into this variable, for example, a string. We will receive an error:

user = 'eric'; // ошибка

Without running the code, determine what the result of executing the code will be:

let date = {year: 2025, month: 12, day: 31}; date = '2025-12-31'; console.log(date);
