Arrays in TypeScript

Arrays in TypeScript are strongly typed. This means that they can only contain data of one type.

The data type of an array is determined in two ways. Let's study them.

First way

Let's make an array with strings. To do this, after the variable name we indicate the data type, and after it we write square brackets to indicate that we have an array:

let arr: string[] = ['a', 'b', 'c', 'd', 'e'];

Let's print some element of the array:

console.log(arr[0]); // выведет 'a'

Second way

There is an alternative way to declare an array. In it we indicate the keywordArrayfollowed by the data type in angle brackets. See example:

let arr: Array<string> = ['a', 'b', 'c', 'd', 'e'];

Practical problems

Using the first method, specify the data type in the following array:

let arr = [1, 2, 3, 4, 5];

Using the second method, specify the data type in the following array:

let arr = [1, 2, 3, 4, 5];
