Compiling TypeScript

The TypeScript compiler can be installed vianpm:

npm install -g typescript

You can then compile files written in TypeScript. These files must have the extension.ts. Let's make a file like this:

let test: string = 'it works'; alert(test);

Let's compile it into a file with the same name, but with the extension.js. To do this, in the terminal, being in the folder with our file, run the commandtsc:

tsc app.ts

To quickly get started with TypeScript, there isonline compiler. Try working with him too.
