HTMLElement type in TypeScript

DOM elements also have their own data types. Let's look at working with DOM elements using an example. Let us have the next onediv:

<div></div>

Let's get a link to this onedivto a variable. All DOM elements are of typeHTMLElement. Let's indicate this type of our variable:

let elem: HTMLElement;

Let us now write into this variable a link to ourdiv:

let elem: HTMLElement = document.querySelector('div'); console.log(elem);

All divs, in addition to being DOM elements with typeHTMLElement, also belong to the typeHTMLDivElement(other tags have similar types). Let's specify a more precise type for our element:

let elem: HTMLDivElement = document.querySelector('div'); console.log(elem);
