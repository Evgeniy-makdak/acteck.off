Date in TypeScript

Let the variable store the date, which we will create using JavaScript using the commandnew Date.

It turns out that our variable will have the typeDate:

let date: Date;

Let's write an object with a date containing the current point in time into this variable:

let date: Date = new Date;

Now let’s write an object with a date containing a given point in time:

let date: Date = new Date(2030, 11, 31);
