Object Structure in TypeScript

Instead of relying on TypeScript to define the object's structure, you can specify it as the type when declaring the variable.

Let's, for example, when declaring a variableuserlet's say that she is an object, in a field'name'which is stored - a string, and in the field'age'- number:

let user: {name: string, age: number};

Let us now write the corresponding object into our variable:

user = {name: 'john', age: 30};

You can merge both operations into one line: declare an object and immediately write a value to it:

let user: {name: string, age: number} = {name: 'john', age: 30};

TypeScript will then take care of monitoring the structure and data types of the object and will throw an error if you try to change something in an incorrect way. Example:

user.name = 123; // ошибка

Without running the code, determine what the result of executing the code will be:

let date: {year: number, month: number, day: number}; date = {year: 2025, month: 12, day: '01'};
