Read-only properties in an interface in TypeScript

Read-only properties are set only once when the object is created. They cannot be changed in the future. The keyword is used to declare these propertiesreadonly.

Let's take the interfaceIFigure, created in the previous lesson. But now let's set the propertywidthread-only access:

interface IFigure { height: number; readonly width: number; }

Let's create an objectrectangle:

let rectangle: IFigure = { height: 200, width: 300 }

Let's change the height:

rectangle.height = 150;

Now let’s change the width and see an error, since the width is declared immutable:

rectangle.width = 400; // ошибка
