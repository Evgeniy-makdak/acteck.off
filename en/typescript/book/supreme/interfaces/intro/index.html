Interfaces in OOP in TypeScript

In this section we will master advanced work with interfaces, which you have already studied earlier in the main textbook,here.

As you already know, interfaces are declared using the keywordinterface. To avoid conflicts between interface and class names, it is customary to put the first letter of the interface name'I'.

Let's, for example, make an interfaceIUser, describing the user:

interface IUser { }
