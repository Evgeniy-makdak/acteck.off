Interface Methods in TypeScript

You can also add object methods to object interfaces. Let's add a method to our interface that should greet the user. To do this, we must specify the method name, parameter types, and return type:

interface IUser { name: string; age: number; greet(text: string): string; }

Let's implement our method in an object:

let user: IUser = { name: 'john', age: 30, greet(text: string): string { return text + ', ' + this.name } }

Let's call our method:

console.log(user.greet('hello')); // 'hello, john'
