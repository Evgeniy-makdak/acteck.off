Interface Properties in TypeScript

Let us have an interface that describes the user:

interface IUser { }

Let's add properties and their type to this interface:

interface IUser { name: string; age: number; }

Now let's create an interface object. To do this, we declare a variable in which we indicate the selected interface as the type:

let user: IUser;

Now we can set values ​​for the properties specified in the interface:

let user: IUser = { name: 'john', age: 30 }

If we now try to create a user without giving him the necessary properties, we will see an error:

let user: IUser = { name: 'john', // ошибка }

There will also be an error if you try to make an extra property:

let user: IUser = { name: 'john', surn: 'smit', // ошибка age: 30 }
