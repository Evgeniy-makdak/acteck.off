Static Methods in TypeScript

Not only properties, but also methods can be static. Such methods can be called without creating a class object.

Let's look at an example. Let's create a class that performs the summation of array elements. Let's make it a static method that takes an array as a parameter and returns the sum of the array elements:

class Calc { public static getSum(arr: number[]): number { let sum: number = 0; for (let elem of arr) { sum += elem; } return sum; } }

Let's use this method to find the sum of the elements of an array:

let num: number = Calc.getSum([1, 2, 3, 4, 5]); console.log(num);

To classCalcadd methods that will find the sum of squares and the sum of cubes of array elements.
