useContext context hook in React

In this lesson we will continue working with context. Now let's build a component tree. First, let's create a React component in a separate file.BigBox:

function BigBox() { return ( <p>bigbox</p> ); }

Let's import it and place it in our main componentApp:

import BigBox from './BigBox'; function App() { return ( <BigBox /> ); }

For greater clarity, a littlestylizedivas. To do this, let's create a filestyles.css:

div { border: 1px solid blue; margin: 10px; text-align: center; max-width: 300px; }

Let's not forget to import it intoApp.js:

import './styles.css';

Now let’s create a component in a separate fileMiddleBox:

function MiddleBox() { return ( <p>middlebox</p> ); }

And put it in a big boxBigBox:

import MiddleBox from './MiddleBox'; function BigBox() { return ( <div> <MiddleBox /> </div> ); }

Let's do the same with a small box.SmallBox:

function SmallBox() { return ( <p>I am ... </p> ); }

Let's place two such boxes inMiddleBox:

import SmallBox from './SmallBox'; function MiddleBox() { return ( <div> <SmallBox /> <SmallBox /> </div> ); }

We built the tree. And now we want from ourApptransfer, suppose, toSmallBoxstring value'small box :)', not using props, but using the context (suddenly our boxed application grows to enormous sizes).

Context fileMyContext.jswe have already created and connected in the last lesson.

The next step we will take is to wrapBigBoxVcontext provider, which is a property of our context objectMyContext. Now all the components enclosed in this design (and these are all the boxes enclosed inBigBox) will be able to access and subscribe to context changes. We pass what we want as the context value'small box :)':

function App() { return ( <MyContext.Provider value="small box :)"> <BigBox /> </MyContext.Provider> ); }

Now all that remains is to read the meaning of the context. We wanted to use it inSmallBox, which means we import the context file thereMyContext.jsand hookuseContext:

import { useContext } from 'react'; import { MyContext } from './MyContext.js';

Let's read withuseContextcontext value into variablename:

function SmallBox() { const name = useContext(MyContext); return ( <div> <p>I am ... </p> </div> ); }

We signedSmallBoxto the given context and if it changes, this component will also be updated.

And finally, we substitute the value of the variablenameinstead of an ellipsis:

function SmallBox() { const name = useContext(MyContext); return ( <div> <p>I am {name} </p> </div> ); }
