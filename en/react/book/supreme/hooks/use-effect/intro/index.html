useEffect effect hook in React

A hook is used to work with effectsuseEffect. Let's take a look at his work.

First, let's import our hook:

import { useEffect } from 'react';

Let's create a componentAppcontaining the header:

function App() { return ( <div> <h1>React App</h1> </div> ); } export default App;

Let's say that we need to do some action after rendering, for example, set the background color of the entire page. In this case, the external system will be the browser DOM.

Let's apply our hook:

function App() { useEffect(() => { document.body.style.backgroundColor = 'green'; }, []); ... }

The second parameter passes an array of dependencies. These include values ​​used by the component's functions. For now we left them empty. In this case, the color will only be set to green once after rendering. You cannot completely remove the second parameter, since your component may enter an infinite loop.
