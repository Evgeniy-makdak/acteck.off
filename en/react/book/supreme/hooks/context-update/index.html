Updating Context in React

In this tutorial we will look at updating components when the context value changes. Let's say you want the context to change.

In this case, it is useful to use context in conjunction with states. Let's slightly change our application with boxes, which we did in the last lesson and make sure that when the button is pressed, the value transferred becomes'metal box :)'.

So let's take our fileAppand afterBigBoxLet's draw a button:

function App() { return ( <MyContext.Provider value="small box :)"> <BigBox /> <button>click</button> </MyContext.Provider> ); }

Next we import the hookuseStateand let's start a statename, the value of which will change when the button is pressed. Let’s make it the initial value that we immediately passed as the context value, that is'small box :)':

function App() { const [name, setName] = useState('small box :)'); }

This time we will be passing a state rather than a string as contextname:

<MyContext.Provider value={name}> <BigBox /> <button>click</button> </MyContext.Provider>

As a final step, we will call the button click handler using the functionsetNameset new state value in'metal box :)'. After this, you can click on the button and check the result:

<button onClick={() => { setName('metal box :)'); }}>click</button>
