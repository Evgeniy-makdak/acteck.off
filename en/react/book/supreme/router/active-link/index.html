Highlighting the active link in React Router

Let's say we have added several products to our application and are browsing through each of the products. How can we understand which product page we are currently on? Let's highlight this product in the list, for example, in bold and brown, and if the link is still being loaded, then in gray.

To get started, go to the main page, add to the list4product and enter data for each of them. Give them any names.

The component will help us highlight the active linkNavLink, this is a subtype of componentLink, which knows about the state of the data. It is usually used when creating a navigation menu.

Let's openroot.jsxand importNavLink(don't forget to remove importLink, otherwise React might get angry, because we don’t use it anymore):

import { NavLink } from 'react-router-dom';

Then in the layout we replace the componentLinkonNavLink:

<NavLink key={product.id} to={`products/${product.id}`}> {product.name ? product.name : <i>unnamed</i>} </NavLink>

And add props to himclassName, and also use conditional logic. If the page has loaded, then the link will go into the active state, if it is loading, then into the loading state, but if nothing happens, it will remain the same. Depending on this, the link class will change:

<NavLink key={product.id} to={`products/${product.id}`} className={({ isActive, isPending }) => isActive ? 'active' : isPending ? 'loading' : '' } > {product.name ? product.name : <i>Unnamed</i>} </NavLink>

All we have to do is add styles foractiveAndloadingclasses to our css file:

a.active { font-weight: bold; color: brown; } a.loading { color: gray; }

Ready! Now click on the products in the list.
