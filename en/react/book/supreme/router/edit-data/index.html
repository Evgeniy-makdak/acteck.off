Writing and editing data using URL parameters in React Router

Our application can now load data for each individual product from the store. In this lesson we will begin to implement adding information about a product and editing it using URL parameters.

In order to enter or change data we will again use the componentForm. First, let's openproduct.jsxand add a button at the end of the layout to edit product data - after the last paragraph (don’t forget to importFormto file):

<Form action="edit"> <button type="submit">edit</button> </Form>

Now we have an edit button on the product page. Next, we will make sure that when we click on this button, we will be redirected to a page with a form where we can enter data. To do this, we will create another route withproducts/:productId/edit. Let's make a layout for this.

So, in the routes folder we will create a new fileedit.jsx. Here we will have a form with fieldsname, costAndamountto fill out, as well as a save button. We will have all this in the componentEditProduct. Be sure to write down the attributesname, we will need them in the following lessons:

import { Form } from 'react-router-dom'; function EditProduct() { return ( <Form method="post" id="product-form"> <div> <span>Name:</span> <input placeholder="name" type="text" name="name" /> </div> <div> <span>Cost:</span> <input placeholder="cost" type="text" name="cost" /> </div> <div> <span>Amount:</span> <input placeholder="amount" type="text" name="amount" /> </div> <p> <button type="submit">save</button> </p> </Form> ); } export default EditProduct;

Take the application you created in the assignments for previous lessons. Using the lesson materials, add a button to edit student data. Create a fileedit.jsxfor editing with functionEditStudent, which will contain a form for filling out student data.
