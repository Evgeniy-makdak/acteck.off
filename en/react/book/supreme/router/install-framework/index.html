Installing the React Router framework

For this section of the tutorial, in addition to React Router, we will useViteas a collector and test server.

First, let's create a React applicationmy-appusing Vite. To do this, open a terminal and enter the following commands:

npm create vite@latest my-app -- --template react cd my-app npm install

In order to work with routing we need a modulereact-router-dom, let's install it too:

npm install react-router-dom

Now everything is ready to start working. Let's launch our application:

npm run dev

Let's copy the linkhttp://localhost:5173/, specified in the terminal in the address bar of the browser and we will see the running application (you can press the button with the counter).
