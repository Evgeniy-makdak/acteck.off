Adding Data to React Router

In previous lessons, we prepared a loader and unloading data along a specific route from the storage. Let's now use our application as an example to learn how to add a new product and write its data to the storage.

Let's open the fileroot.jsxand add a button to the layout to add a new product beforenav, wrapping it in a form tag. Let's also now wrap the button and our list in another div with#menu. The layout will now look like this:

return ( <div id="main"> <div id="menu"> <form method="post"> <button type="submit">add product</button> </form> {products.length ? ( <nav> {products.map((product) => ( <Link key={product.id} to={`products/${product.id}`}> {product.name ? product.name : <i>Unnamed</i>} </Link> ))} </nav> ) : ( <p> <i>no products here ...</i> </p> )} </div> <div id="product"> <Outlet /> </div> </div> );

If we go to the tab now'Сеть'developer panel, and then click on our button, we will see an erroneous document request to the server. With React Router, we will again use client-side routing, eliminating the request to the server.

Let's change the tag for thisformper componentFormReact Router. Let's add import firstForm:

import { Form } from 'react-router-dom';

Now let's replace the tagsformin a piece of code:

<Form method="post"> <button type="submit">add product</button> </Form>

Reload our application and look again at the developer panel. Click the add product button - now there is no longer a request to the server (ignore the error for now).
