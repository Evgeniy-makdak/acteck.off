Using data received by the loader in React Router

In the last lesson, we retrieved data from the storage usingloader. In order to use this data, we use a hookuseLoaderData. Let's open the fileroot.jsxand add a hook to the import:

import { Outlet, Link, useLoaderData } from 'react-router-dom';

In the Root function we will now receive data into a constantproductsand work with her. If we have some products, the list will be displayed, if not, then we will display it in a paragraph'no products here ...'. In the tagnavwe now use a standard loop to output the resulting products. Each element will be wrapped in a componentLinkand everyone will have their own address (we'll talk about this later). Accordingly, if the product does not have a name, it will display'Unnamed'default. So with that said, now our code for the function isRootwill be like this:

function Root() { const { products } = useLoaderData(); return ( <div id="main"> {products.length ? ( <nav> {products.map((product) => ( <Link key={product.id} to={`products/${product.id}`}> {product.name ? product.name : <i>Unnamed</i>} </Link> ))} </nav> ) : ( <p><i>no products here ...</i></p> )} <div id="product"> <Outlet /> </div> </div> ); }

If we now restart the application and go to the main page, we will only see'no products here ...', since we have not yet put any products in storage.
