Removing a route in React Router

We wrote a function to delete data from storage. Let's now add a piece of code with a button to delete a product to the layout of the product page. Let's add it right after the edit button. Let's wrap both forms with buttons in divs#control. Then inactionwrite down the value'delete'. We will also add a dialog box so that the user can once again confirm whether he wants to remove the product:

<div id="control"> <Form action="edit"> <button type="submit">edit</button> </Form> <Form method="post" action="delete" onSubmit={(event) => { if (!confirm('Do you want to delete this product?')) { event.preventDefault(); } }} > <button type="submit">delete</button> </Form> </div>

Let's also add styles to our CSS file:

div#control { display: flex; } button { margin-right: 5px; }

Now we will create a new route to remove the product. To do this, open the folderroutesand create a file in itdelete.jsx. Let's immediately add import to itredirectand delete functionsdeleteProduct:

import { redirect } from 'react-router-dom'; import { deleteProduct } from '../forStorage';

And then, of course, we write our functionaction, which will causedeleteProductByid, and after deletion it will redirect us to the main page:

export async function action({ params }) { await deleteProduct(params.productId); return redirect('/'); }

All we have to do is openmain.jsx. Import ouraction:

import { action as deleteAction } from './routes/delete';

And set it as the value for the methodactionremoval route. We will add the route object itself to the end of the arraychildren:

{ path: 'products/:productId/delete', action: deleteAction, },

Now we can click on any product and remove it using the delete button. You can look at the localforage developer panel and see for yourself.
