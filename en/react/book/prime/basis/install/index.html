Installing the React framework

To install the framework you will need NodeJS22+on your computer. To create a project on the command line, run the following command:

npm create vite@latest my-app -- --template react

And then run the following commands:

cd my-app npm install npm run dev

After this you will have a foldermy-app, containing the framework. Find the folder in this foldersrc- this will be your working folder in which you will develop your project.

In a foldersrcfind the fileApp.js. For the upcoming lessons, this file will be your main working file. In it, you will write code and watch its results in the browser window.

Open this file in an editor and remove everything unnecessary from it, bringing it to this form:

import React from 'react'; function App() { return <div> text </div>; } export default App;

After your project is installed, for the next launch (for example, after restarting the computer) you will only need to go through the terminal to the foldermy-appand run the following command:

npm run dev
