Event Hinging in JSX

Let's now learn how to work with events in React. For example, let’s make it so that when you click on a block, it will displayalertwith some text.

Let us have a functionshowMess, which outputsalertwith a message, and in the HTML code there is a button, upon clicking on which we would like to see thisalert:

function App() { function showMess() { alert('hello'); } return <div> <button>show</button> </div>; }

Let's bind an event to our buttononclickso that when you click on this div the function is triggeredshowMess. To do this you need to add the attributeonClick(exactly in camelCase, that isonClick, notonclick), and in it indicate the function that will be executed on this event:

function App() { function showMess() { alert('hello'); } return <div> <button onClick={showMess}>show</button> </div>; }

This is how events work: adding an attribute (for example,onClickoronFocus), the attribute value specifies the method that will be called on this event.

The following code is given:

function App() { function show1() { alert(1); } function show2() { alert(2); } return <div> <button>act1</button> <button>act2</button> </div>; }

Make it so that clicking the first button triggers the first function, and clicking the second button triggers the second function.
