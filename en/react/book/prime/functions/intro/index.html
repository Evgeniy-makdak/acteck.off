Using Functions in React

You can create auxiliary functions inside the main function of a component. For example, let's use auxiliary functions to find the sum of powers of two numbers and display it in the tag text:

function App() { function square(num) { return num ** 2; } function cube(num) { return num ** 3; } const sum = square(3) + cube(4); return <div> {sum} </div> }
