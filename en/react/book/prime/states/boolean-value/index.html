Boolean value in state in React

Let's make a stateinCart, which will show whether the product is in the cart or not:

function App() { const [inCart, setInCart] = useState(false); return <div> </div>; }

Let the valuefalsemeans that the product is not in the cart, but the valuetrue- what's in the basket. Let's display information about this using the ternary operator:

function App() { const [inCart, setInCart] = useState(false); return <div> <span>{inCart ? 'в корзине' : 'не в корзине'}</span> </div>; }

Now let’s create a button that, when clicked, will add the product to the cart:

function App() { const [inCart, setInCart] = useState(false); return <div> <span>{inCart ? 'в корзине' : 'не в корзине'}</span> <button onClick={() => setInCart(true)}>btn</button> </div>; }

Let's modify our code so that the first click on the button adds to the cart, and the second click removes from it:

function App() { const [inCart, setInCart] = useState(false); return <div> <span>{inCart ? 'в корзине' : 'не в корзине'}</span> <button onClick={() => setInCart(!inCart)}>btn</button> </div>; }

Make a button that, when clicked, will ban the user and a button that, when clicked, will unban the user.

Modify the previous task so that of the two buttons, only the corresponding one is always visible. That is, if the user is banned, then a button for unbanning is visible, and if not banned, a button for banning is visible.
