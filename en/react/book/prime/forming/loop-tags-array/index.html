Forming an array of tags in a React loop

An array with tags can be created in a loop:

function App() { const arr = []; for (let i = 0; i <= 9; i++) { arr.push(<p>{i}</p>); } return <div> {arr} </div>; }

Using a loopforgenerate the following code:

<ul> <li>1</li> <li>2</li> <li>3</li> <li>4</li> <li>5</li> </ul>
