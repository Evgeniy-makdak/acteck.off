Tags on different lines in JSX

Tags written to constants do not have to be written on the same line. You can also do this:

function App() { const str = <p> text </p>; return <div> {str} </div>; }

Or you can do it like this:

function App() { const str = ( <p> text </p> ); return <div> {str} </div>; }

The author of the following code wanted to write several tags into a constantliand paste them intoul:

function App() { const items = <li>text1</li><li>text2</li><li>text3</li>; return <ul> {items} </ul> }

The code, however, doesn't work. Fix the code author's mistake.
