Returning nested JSX tags with functions

In the previous lesson you got the following code:

function App() { return <div> text </div>; }

Let's look at the features of returning nested tags from a function when working with JSX. Inside a tag that is returned viareturn, there can be any number of nested tags:

function App() { return <div> <p>text1</p> <p>text2</p> </div>; }

You have the following function:

function App() { return <ul> </ul>; }
