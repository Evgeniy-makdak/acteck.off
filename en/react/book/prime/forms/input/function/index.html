Changing data using a function in React

It is not necessary to perform certain operations on the state directly on the output. You can use the function:

function square(num) { return num ** 2; } function App() { const [value, setValue] = useState(0); function handleChange(event) { setValue(event.target.value); } return <div> <input value={value} onChange={handleChange} /> <p>{square(value)}</p> </div>; }

Given an input and a paragraph. The user's age is entered into the input. Make sure that when typing text, the user's year of birth automatically appears in the paragraph.
