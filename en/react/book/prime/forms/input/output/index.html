Changing input data when outputting it in React

Let us enter a number into the input. Let's make sure that as you enter a number into the input, the square of the entered number is displayed in the paragraph:

function App() { const [value, setValue] = useState(0); function handleChange(event) { setValue(event.target.value); } return <div> <input value={value} onChange={handleChange} /> <p>{value ** 2}</p> </div>; }

Input given. A paragraph is given. Make sure that when you enter text into the input, the number of characters entered into the input is displayed in the paragraph.
