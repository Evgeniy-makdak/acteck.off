Inserting data into the style attribute from an object in React

You can also add CSS styles to elements using the attributestyle. In this and several subsequent lessons we will look at waysinline styling.

Now we will not include the filestyles.css, and pass the corresponding values ​​to the attributestylein the form of an object with styles for each tag, which we will write directly in the component file.

So, let's take our component without CSS styles, which we made in the last lesson:

function App() { return ( <div> <p>text</p> <p>text</p> <p>text</p> </div> ); }

Let's fileApp.jsin front of the teamreturnlet's create an object with CSS styles for the div in a variableclass1. Remember that property names here are written incamelCasenotation, and property values ​​must be enclosed in quotes:

const class1 = { width: '200px', border: '2px solid brown', padding: '10px', textAlign: 'center' };

Now let's add an objectclass2with styles for the first paragraph:

const class2 = { color: 'orangered', fontWeight: 'bold' }

Objectclass3with styles for the second paragraph:

const class3 = { fontStyle: 'italic', color: 'brown', }

And finallyclass4for the last one:

const class4 = { backgroundColor: 'orange', fontWeight: 'bold', color: 'white', }

Now, to apply CSS classes in the component, we will use the attributestyle. For the first paragraph we had a variableclass2, pass it as a value:

<p style={class2}>text</p>

Similarly, we will add styles from objects for the remaining tags.

As a result, the component code will look like this:

function App() { const class1 = { width: '200px', border: '2px solid brown', padding: '10px', textAlign: 'center', }; const class2 = { color: 'orangered', fontWeight: 'bold', }; const class3 = { fontStyle: 'italic', color: 'brown', }; const class4 = { backgroundColor: 'orange', fontWeight: 'bold', color: 'white', }; return ( <div style={class1}> <p style={class2}>text</p> <p style={class3}>text</p> <p style={class4}>text</p> </div> ); }
