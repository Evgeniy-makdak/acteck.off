Using Props with Styled Components in React

From the last lesson, we learned that components styled using the Styled Components library can be used like regular React components. In this tutorial we will see that props will work in a similar way here.

Let's say we have a React componentBlockand we created components in it styled using Styled ComponentsInputAndContainer:

const Container = styled.div` display: flex; flex-direction: column; width: 150px; `; const Input = styled.input` margin: 5px; font-size: 18px; `;

Let's arrange three componentsInputVContainer:

function Block() { return ( <Container> <Input /> <Input /> <Input /> </Container> ); }

Using props, we can set various attributes in components. Let's set the attributes for the second inputplaceholderAndtypewith meaningsnameAndtext, respectively, and the third input of the attributetypeset the valuepassword:

function Block() { return ( <Container> <Input /> <Input placeholder="name" type="text" /> <Input type="password" /> </Container> ); }
