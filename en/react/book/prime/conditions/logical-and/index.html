Using the && Operator in JSX

We may need to insert text if the condition is true and do nothing if it is false. This is done using the operator&&.

Let's see how it works with an example. In the following code, if inshowwilltrue, then insert a paragraph with text, and if there isfalse, then nothing will be inserted:

function App() { const show = true; return <div> {show && <p>text</p>} </div>; }

The following code is given:

function App() { const isAuth = true; return <div> <p>вы авторизованы</p> </div>; }

Make the following paragraph of text appear only ifisAuthwritten true.
