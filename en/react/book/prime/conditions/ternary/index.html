Ternary operator in JSX

As you already know, you can execute JavaScript code inside curly braces. In fact, this code may not be anything, but only the simplest.

In particular, the conditionsifcannot be used there. Instead, shortened versions of the terms and conditions should be used.

Let's depending on the value of the constantshowLet's display one or the other text. We use for thisternary operator:

function App() { const show = true; return <div> {show ? 'text1' : 'text2'} </div>; }

You can work not only with texts, but also with tags:

function App() { const show = true; return <div> {show ? <p>text1</p> : <p>text2</p>} </div>; }

The following code is given:

function App() { const age = 19; return <div> </div>; }

If age contains more18years, then in the tag textdivshow the user a paragraph with one text, and if less, then with another.
