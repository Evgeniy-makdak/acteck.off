Installing Redux DevTools Extension

In this lesson, we will add the Redux DevTools Extension tool to the browser to debug a Redux application.

For Chrome we will download and install it as suggestedlink.

RTK functionconfigureStore, which we used in the last lesson, not only creates a store, but also automatically configures the React DevTools Extension, so we don’t have to do anything else.

Of course, if we now launch our project and enable the Redux DevTools Extension by clicking on the icon in the extensions, we will not see practically anything yet, since we have not yet created states and logic in the application.

Install the Redux DevTools Extension for your browser. Make sure everything works.
