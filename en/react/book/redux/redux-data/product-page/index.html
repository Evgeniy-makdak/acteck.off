Product page in Redux

In this tutorial we will make a separate page to display information about a specific product.

Let's open our application with products and in the folderproductslet's create a fileProductPage.jsx. As you remember, each of our products has a unique id. Thanks to this id, we will be able to generate a unique page URL for each product. Traditionally it will look like:/products/id123. And the id will be contained in its dynamic part (we will deal with the route itself a little later).

So let's inProductPage.jsxcreate an empty componentProductPage:

export const ProductPage = () => {}

The first thing we'll do here is use a hookuseParamsto pull out the dynamic part of the URL for the product page we're on:

export const ProductPage = () => { let params = useParams() const { productId } = params }

Next, using the received id, we will look for the desired product in the sliceproductsin store, using a hook already familiar to ususeSelector:

const product = useSelector((state) => state.products.find((product) => product.id === productId) )

Don't forget to import both hooks into the file:

import { useSelector } from 'react-redux' import { useParams } from 'react-router-dom'

If the product we need is not available (for example, the user entered an incorrect address), then we will display information about this on the screen. In this step the code for the componentProductPageshould look like this:

export const ProductPage = () => { let params = useParams() const { productId } = params const product = useSelector((state) => state.products.find((product) => product.id === productId) ) if (!product) { return <p>No such product</p> } }

And all we have to do is display the received product data in the layout:

return ( <div> <h2>{product.name}</h2> <p>Description: {product.desc}</p> <p>Price: {product.price}</p> <p>Amount:{product.amount}</p> </div> )
