Product page in browser in Redux

In the last lesson we made a separate page for the product. Now we need to make sure that when you click on a button, our page appears in the browser.

First, let's attach an address to it where it will be displayed. Let's open the fileApp.jsx, in which we register the routes, and add tochildrenanother child route (don't forget to importProductPage.jsx). Let's indicate the path and component that we will output:

{ path: '/products/:productId', element: <ProductPage />, },

Now let's openProductsList.jsxin a folderproductsand change the code a little fordispProducts. Now we have a separate page with complete information about each product. This means that in the list of products we will display only the product name and abbreviated description text. We will also add a link as a navigation elementLinkfrom the router library, when clicked you will be taken to the product page. Let's also add a diva classproduct-excerptto unstick the products. Now our code fordispProductswill be like this:

const dispProducts = products.map((product) => ( <div key={product.id} className="product-excerpt"> <h3>{product.name}</h3> <p>{product.desc.substring(0, 100)}</p> <Link to={`/products/${product.id}`} className="link-btn"> view </Link> </div> ))

We importLink:

import { Link } from 'react-router-dom'

And add styles for classeslink-btnAndproduct-excerptVindex.css:

.product-excerpt { margin-top: 30px; border: 1px solid gray; border-radius: 5px; padding: 10px; } .link-btn { border: 1.5px solid gray; border-radius: 10px; background-color: coral; color: black; padding: 1px 12px 1px 12px; }

Finally, let's also make a working link in the menu on the left, leading to the page with the product list, so that we can get to it from any other place. To do this, let's open ourroot.jsxand replace this line of code:

<a>Products</a>

To the next one:

<NavLink to="/products" end> Products </NavLink>

Also, don't forget to importNavLinkfrom the library for React router:

import { Outlet, NavLink } from 'react-router-dom'

Let's launch our application. Now we can go to the information page about any product by clicking on the view button. Now try adding a new product and look at information about it on a separate page by clicking the view button. Also now, to return to the list of products, just click on'Products'in the menu on the left. While on different pages, look at how the URL changes in the address bar of your browser.
