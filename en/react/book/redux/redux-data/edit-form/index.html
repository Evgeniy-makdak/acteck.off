Form for editing data in Redux

Now we have a reducer for changing data in the store. In this tutorial we will create a form with which we can edit product data.

Let's open our products app and create in the folderproductsfileEditProductForm.jsx. Creating a ComponentEditProductFormit will be similarNewProductForm, with the exception of some differences, which we will dwell on. So copy the entire codeNewProductForm.jsxand paste it into the created fileEditProductForm.jsx. Now let's start in order.

Remove nanoid from import, we don’t need to generate id here. Replace importproductAddedonproductUpdated, because here we will use action to update, and not to add a product.

Next, let's change the name of our component's function fromNewProductFormonEditProductForm.

Before we add local states to functionsEditProductFormForname, desc, priceAndamount, let's insert a few more lines of code. As we mentioned earlier, we don't need to generate id here. Now our task is to get it from the data of the product being changed. We already did this when creating a separate page for the product. So, let's get the id using a hookuseParams, and then find the product we need using the hookuseSelector(don't forget to import both hooks at the beginning of the file):

let params = useParams() const { productId } = params const product = useSelector((state) => state.products.find((product) => product.id === productId) )

After that, we will change the block with the declaration of local states. Now we need to set the product data received from the store as the initial value. This is what the first state will look like, redo the remaining three yourself:

const [name, setName] = useState(product.name)

After assigning a variable touseDispatchlet's add another line of code this time for the hookuseNavigate. We will use it to return to the product page when the user saves the changes made to the form:

const navigate = useNavigate()

We also import this hook at the beginning of the file:

import { useNavigate, useParams } from 'react-router-dom'

Next we have handlers for input fields. And after them we need to correct the functiononSaveProductClick. Now, when clicked, we will send an actionproductUpdatedwith the received id and modified data in the form of an object. Then we will add ournavigate, to go to the page of the changed product:

const onSaveProductClick = () => { if (name && desc && price && amount) { dispatch( productUpdated({ id: productId, name, desc, price, amount, }) ); navigate(`/products/${productId}`); } }

All that remains is to find the line in our returned layout:

<h2>Add a New Product</h2>

And replace it with:

<h2>Edit Product</h2>
