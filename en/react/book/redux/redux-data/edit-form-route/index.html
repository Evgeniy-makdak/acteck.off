Product edit page in browser in Redux

In the last lesson we made a form for editing a product. Let's attach a route for her.

Let's open our application with products, and in it there is a fileApp.jsxand add another child route object to the array for the propertychildren(don't forget to importEditProductForm):

{ path: '/editProduct/:productId', element: <EditProductForm />, },

Also, to go to the editing form, we need to make a link. Let's open for thisProductPage.jsxand place this link in the returned layout after the last paragraph with the quantity of the product and before the closing div:

<Link to={`/editProduct/${product.id}`} className="link-btn"> edit </Link>

We also importLinkfrom the router library:

import { Link } from 'react-router-dom'

Let's launch our application and click on the button to view a product. Let's try to edit it and go back to the list of products by clicking on the link'Products'in the menu on the left. Also in Redux DevTools you can see a new actionproductUpdatedand look at the changes in the product object in the store.

Open your application with students. In fileApp.jsxadd another child route to edit student data.

Add an elementLinkin the layout on the student’s page to go to the page for editing his data.

Launch your application, try to edit the data of a student. Pay attention to your browser's address bar when you are on the edit page. Check out the changes in the browser's Redux DevTools.
