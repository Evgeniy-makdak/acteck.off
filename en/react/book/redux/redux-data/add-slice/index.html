Adding another slice to Redux

In previous lessons we worked closely on the sliceproductsfor products. In a real Redux application, a state can have many slices for different purposes. Therefore, in this lesson we will add another slice to state for sellers, who will actually add products in our application.

Let's open our products application and the first thing we will do is in the folderpartslet's create another foldersellers, this is where we will place all the code and components related to sellers.

Now, just like with products, let's create a slice for sellers. To begin with, in the foldersellerslet's create a filesellersSlice.jsand import it into itcreateSlice:

import { createSlice } from '@reduxjs/toolkit'

We continue to take the standard steps. Let's create four sellers as the initial value for the slice. For simplicity they will have two fields -idAndname:

const initialState = [ { id: '0', name: 'Super Power' }, { id: '1', name: 'Miracle Life' }, { id: '2', name: 'Dolls&Toys' }, { id: '3', name: 'Granny' }, ]

Below usingcreateSlicelet's create a slicesellers, give it the initial value. Since we will neither add sellers nor update their data in the future, let the field with reducers remain empty for now:

const sellersSlice = createSlice({ name: 'sellers', initialState, reducers: {}, })

At the end of the file, don’t forget about exporting the resulting reducer:

export default sellersSlice.reducer

Now we need to open the filestore.jsin a folderappand import the resulting reducer:

import sellersReducer from '../parts/sellers/sellersSlice'

And then assign it toconfigureStoreas a reducer forsellers:

export default configureStore({ reducer: { products: productsReducer, sellers: sellersReducer, }, })

Open your application with students. Create another folderteachers, here you will have code files related to teachers. Create a file in itteachersSlice.js. Using the lesson materials, create a slice in itteacherswith the helpcreateSlice.

Let each object with teacher data have three properties:id, name(which will contain the last name and initials, for example'Петров А.В.') and a subject (any one you can think of - physics, mathematics, biology, etc.). Make 3 of these objects with teachers as the initial value for the slice.

Import the resulting reducerteachersReducerVstore.jsand assign it as a reducer forteachers, as shown in the lesson.
