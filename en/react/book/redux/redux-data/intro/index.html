Working with data in Redux

In the previous section, we implemented the basic scheme of how Redux works and now you know the basic principles of how a Redux application works. In the following lessons we will begin to add functionality to our application with products and work with data.

As the application grows with new components and routes, let's do one thing about routing. Open the products app, then the fileroot.jsx. Import the component into itOutlet, at the same time delete the lines with importProductsListAndNewProductForm:
