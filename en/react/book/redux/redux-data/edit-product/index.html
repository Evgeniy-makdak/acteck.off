Changing product data in Redux store

We've learned how to add products, but what if we want to change the product information? Let's look at how to do this in this and subsequent lessons.

Let's open our application with products and go to the fileproductsSlice.js. Let's start with what we need for the sliceproductswrite another reducer that will be launched when the product is updated. Let's call himproductUpdatedand add it to the fieldreducersafter the reducerproductAdded. We will also immediately pass it the parametersstateAndaction, on the basis of which, as we remember, the reducer works:

reducers: { productAdded(state, action) { state.push(action.payload) }, productUpdated(state, action) {}, },

Now let's think about what data we need to update the product. First of all - thisid, using it we can find the product we are interested in in the store. The rest of the data - name, description, price and quantity will be with us for change. We will get all these values ​​from the propertypayloadaction object that will come to us, and if we wrote it by hand, it would look like the one shown below. Note that the expected value of the property ispayload- one argument, so we will pass an object here (we will talk about this later):

{ type: 'products/productUpdated', payload: {id, name, desc, price, amount} }

With the basics out of the way, we can now write the code in curly braces toproductUpdated. First we get from the objectactionchanged data:

productUpdated(state, action) { const { id, name, desc, price, amount } = action.payload },

Now we will find from the receivedidproduct to change:

productUpdated(state, action) { const { id, name, desc, price, amount } = action.payload const desiredProduct = state.find(product => product.id === id) },

If such a product is found in the store, then we will replace its data with new values:

productUpdated(state, action) { const { id, name, desc, price, amount } = action.payload const desiredProduct = state.find(product => product.id === id) if (desiredProduct) { desiredProduct.name = name desiredProduct.desc = desc desiredProduct.price = price desiredProduct.amount = amount } },

All we have to do is export the action creator for use in the component, which will kindly create for uscreateSlice. Let's add it to the export at the end of the file, along with the existing oneproductAdded:

export const { productAdded, productUpdated } = productsSlice.actions
