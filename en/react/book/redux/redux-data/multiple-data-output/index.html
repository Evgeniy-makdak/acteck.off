Reusing data from a slice in Redux

In the last lesson, we added another field to the form for adding a product with a drop-down list for selecting the seller who posts it. Let's say we now also need to display this information in several other places in the application. To do this, we will not duplicate the code every time, but will simply create a separate component in which we will extract data from the slice and display it in those parts of the application where we need it.

Let's open our application with products. Let's go to the folder nowproductslet's create another fileSellerOfProd.jsx. First, let's import the hookuseSelector, with the help of which we will receive a slice with the desired product:

import { useSelector } from 'react-redux'

Then we will extract the desired product from the store by seller id, which we will pass to our new component in props:

export const SellerOfProd = ({ sellerId }) => { const seller = useSelector((state) => state.sellers.find((seller) => seller.id === sellerId) ) }

And if there is such a seller, we will return his name, and if not, we will return it'unknown':

return <span>by {seller ? seller.name : 'unknown'}</span>

Let's now import our new component intoProductsList.jsxand insert it into the functionsProductsListimmediately after the heading with the name of the product. As you can see, we passed it id in props:

<SellerOfProd sellerId={product.seller} />

Now, if we launch our application, add a new product and find it in the list, we will see that it now has a seller, and the products automatically added when starting the application will have'unknown'.

A small digression: if your eslint in the VS Code editor swears atsellerIdand the application is running, you can ignore it for now or open the file.eslintrc.cjsyour application andrulesadd"react/prop-types": "off".

It goes without saying that we would like to add seller information on the product page as well. To do this, insert in the layoutProductPageafter the header with the product name, add this line again and check that everything works:

<SellerOfProd sellerId={product.seller} />
