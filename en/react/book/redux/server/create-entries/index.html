Creating Database Entries in Redux

In previous lessons we prepared everything to populate our database with data. Let's get started.

Let's open our application with products, and in it there is a fileserver.js. First, we will decide how many and what kind of data objects we need. Sellers with us4, let each of them have3product. To do this, we introduce two constants immediately after the import lines:

const NUM_SELLERS = 4 const PRODS_PER_SELLER = 3

Now at the end of the file, before exporting the worker, we will make a for loop, in which we usecreateLet's create database entries for our4sellers:

for (let i = 0; i < NUM_SELLERS; i++) { const newSeller = db.seller.create(createSellerData(i)) }

For every seller we have3product, so in the loop for sellers we will make another nested for, in which we will create records for products:

for (let i = 0; i < NUM_SELLERS; i++) { const newSeller = db.seller.create(createSellerData(i)) for (let j = 0; j < PRODS_PER_SELLER; j++) { const newProduct = createProductData(newSeller) db.product.create(newProduct) } }

Great! At this point we have finished working with the database.
