Sending data from the server using a GET request in Redux

So we have a database with data. The first thing we will do when running our Redux application is to receive product data from the server. To do this, we need to send a GET request from our application. Let's process it on the server side.

Let's open our application with products, and in it there is a fileserver.js. In order to work with HTTP requests, we need to import several tools from msw into the file, let's do this:

import { http, HttpResponse, delay } from 'msw'

We will also immediately make adjustments to reality, that is, possible network delays. Let the value of our artificial delay be2seconds so that we can see how the data is loaded. We'll write it laterconst PRODS_PER_SELLER = 2:

const ARTIFICIAL_DELAY_MS = 2000

And now at the end of the file, before exporting the worker, let’s create an array for the request handler API:

export const handlers = []

And then in square brackets we will write the first handler to intercept the GET request for products:

export const handlers = [http.get()]

The first parameter forhttp.getwe will transmit some kind of fake address, for example'/fakeServer/products', and the second is an asynchronous callback:

export const handlers = [http.get('/fakeServer/products', async () => {})]

Now let's write the code for this callback. Using it we will extract all products from the database. Here we also use the functionserializeProduct, the code for which we wrote in the last lesson. Let's pass it on tomap. Then we’ll make a delay (as if the network is slowing down) and return toresponseobjects with product data in the form of JSON:

export const handlers = [ http.get('/fakeServer/products', async () => { const products = db.product.getAll().map(serializeProduct) await delay(ARTIFICIAL_DELAY_MS) return HttpResponse.json(products) }), ]

And one last thing. We have written the first handler for our worker, let's pass an array to this workerhandlers. To do this, let's correct the last line of code in the file to:

export const worker = setupWorker(...handlers)

After studying the material in this lesson, create an empty array of handlers to intercept requests. Write the code in it to handle the students' GET request. As a way let you have'/fakeServer/students'.

At the end of the file, don't forget to pass an array of handlershandlersto the worker.
