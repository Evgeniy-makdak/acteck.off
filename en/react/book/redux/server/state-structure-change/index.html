Changing the state structure in Redux

The first stage that we will implement for the interaction of our Redux application with some external server is to receive the data that is stored on it when the application is launched. But first, let's make some changes to our application.

Let's open our application with products, and in it there is a fileproductsSlice.js. Pay attention to the definitioninitialState. Initially our slice with productsproductsSlice- this is an array in which there are objects (in our case there are two of them) with product data. Firstly, since we will now download them from the server, we need to remove them frominitialState. Secondly, we will send API requests to the server and it will be important for us to know about their status. Given these two things, let's redoinitialState. Now let it be just an object with fieldsproducts(initially there will be no data here, just an empty array),statusAnderror. Now our code forinitialStatewill be a little shorter:

const initialState = { products: [], status: 'idle', error: null, }

While we have set the status'idle'(inaction), because initially we do not send any requests, but in general it will change and can take values'загрузка', 'выполнен', 'ошибка загрузки'. Remember that to indicate your status, you can choose names that are convenient for you.

Let's go down the code. Following the changeinitialState, we should also change the code in which we work with this state in reducer functions. Let's look at the code forproductAdded:

state.push(action.payload)

Now we will add new products not just to the state for products, but to its propertystate.products. Therefore, let's replace the above line with:

state.products.push(action.payload)

Accordingly, let's make the same changes for the reducerreactionClicked. Instead of:

const currentProduct = state.find((product) => product.id === productId)

Now it will be:

const currentProduct = state.products.find((product) => product.id === productId)

Replace it yourselfstateonstate.productsand in the reducer codeproductUpdated.
