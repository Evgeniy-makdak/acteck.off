Getting selectors in Redux

Sometimes it is very useful to get rid of unnecessary code in an application, as well as to encapsulate data.

Let's open our products application. Please note that in several places in the application we use selectors to pull out a slice with products in order to perform some actions with it.

And now we will open ourproductsSlice.js, let's go down to the very end of the file and add a couple of lines of code in which we will write two selectors and export them. We will have one function to get all products:

export const selectAllProducts = (state) => state.products.products

And after it the second one is for getting one product by id:

export const selectProductById = (state, productId) => state.products.products.find((product) => product.id === productId)

Of course, you may have a fair question - where do we get such bloat:state.products.products? The answer is simple, in the last lesson we changed the structure of the slice and now our products will be located in the form of an array in a separate propertyproductsslice of productsstate.products(remember that herestateis the root Redux state object that contains all the slices).

So we've simply moved the code for the selector functions to one place in the application and now we need to make the appropriate changes where we used them. Let's start with the fileProductsList.jsx. We import into itselectAllProducts, since here we will need to get all the products:

import { selectAllProducts } from './productsSlice'

And at the beginning of the function codeProductsList, replace:

const products = useSelector((state) => state.products)

On the line:

const products = useSelector(selectAllProducts)

Now let's go to the fileProductPage.jsx. Here we needed a product by id. Let's replace the string (don't forget about import):

const product = useSelector((state) => state.products.find((product) => product.id === productId) )

On:

const product = useSelector((state) => selectProductById(state, productId))

Make a similar replacement yourself in the fileEditProductForm.jsx. Here we also need product by id.

Remember that such optimization is only for your convenience and you can do it when you consider it necessary and convenient. And using our example, imagine that we would write the selector function code in different places each time, and then change the state structure, and then we would have to correct it in all components.

Open your application with students. Open the file in itstudentsSlice.js. At the very bottom of the file, write and export, as shown in the lesson, two selectors to get all studentsselectAllStudentsand to get one student by idselectStudentById.

Now importselectAllStudentsAndselectStudentByIdto filesStudentsList.jsx, StudentPage.jsxAndEditStudentForm.jsx, make the appropriate changes in these files as shown in the lesson.
