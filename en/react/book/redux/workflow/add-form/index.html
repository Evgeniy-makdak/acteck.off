Adding a form to Redux

In the last lesson, we learned that for form input fields you can apply states that will only work inside one component, and we created an empty form.

Let's inside the tagsformWe will place a layout for each field in which information will be entered. Our first input will be responsible for the name of the product and its layout will be like this:

<form> <p> <label htmlFor="productName">Name:</label> <input id="productName" name="productName" value={name} onChange={onNameChanged} /> </p> </form>

The second we will havetextareafor product description:

<p> <label htmlFor="productDesc">Description:</label> <textarea id="productDesc" name="productDesc" value={desc} onChange={onDescChanged} /> </p>

Write the code yourself for the two remaining inputs for price and quantity with IDproductPriceAndproductAmountrespectively.

After all four input fields before the closing tagformadd a save button:

<button type="button">save</button>

Our form component is ready. Let's display it on the main page. To do this, open the fileroot.jsx, import the component into it:

import { NewProductForm } from '../parts/products/NewProductForm'

And insert it between the taghrand componentProductsListlike this:

<hr></hr> <NewProductForm /> <ProductsList />

Let's launch our application and admire the form and list of products. Let's add more styles toindex.css:

#main-page { max-width: 600px; } select, textarea, input, button { font-size: 18px; } form { margin-bottom: 20px; border-bottom: 2px solid lightgray; padding-bottom: 10px; } form p { margin: 5px; }
