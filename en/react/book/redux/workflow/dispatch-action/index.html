Dispatch action and useDispatch hook in Redux

We have implemented almost all the components for a Redux application to work. All that remains is to figure out sending an action from a React component. Let's get started.

Open our products app. In component formNewProductFormwe have a button to save new product data. Let's attach a click handler to it. Let the handler be triggered when it is clickedonSaveProductClick:

<button type="button" onClick={onSaveProductClick}> save </button>

When creating a new product, we will need a new id for it. We will generate it using the well-known librarynanoid. By the way, there is no need to install it, since it already comes with the RTK package. Let's add nanoid to the import:

import { nanoid } from '@reduxjs/toolkit'

We also remember that the only way to change state is to call the methoddispatch, which store has and pass the action object. To get to this method, we'll use the React-Redux hookuseDispatch. Let's import it too:

import { useDispatch } from 'react-redux'

In the function codeNewProductFormlet's create a constant for itdispatchright after we created constants for our local states:

const dispatch = useDispatch()

We also need to import action creatorproductAdded, which we received in the last lesson:

import { productAdded } from './productsSlice'

Now we can write the handler function itselfonSaveProductClick. Let's do it in front of the teamreturn:

const onSaveProductClick = () => {}

Inside the function we calldispatch, we form the action object into the propertypayloadwhich we will have the generated id, name, description, price and quantity of the product. We take all this data from local states. At the same time, at the beginning we check in the condition whether all fields are filled in - only in this case we calldispatch. At the end of the function code, we return the local states to their original state:

const onSaveProductClick = () => { if (name && desc && price && amount) { dispatch( productAdded({ id: nanoid(), name, desc, price, amount }) ) setName('') setDesc('') setPrice(0) setAmount(0) } }

Now let's launch our application, enter data into the form fields and press the save button. Hurray, a new product has appeared at the bottom of the page. Having opened Redux DevTools, and in it the Inspector tab, we see that on the left, in addition to @@INIT, our action has also appearedproducts/productAdded. And on the right side we can click on the Action and State tabs, while switching @@INIT andproducts/productAdded. Now we see our new product being added. Also on the Action tab we see the action generated automatically usingcreateSlice(click on Raw for example). Hurray: we didn’t have to write it manually.
