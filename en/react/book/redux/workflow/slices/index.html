Adding a slice in Redux

We have prepared the basic tools for creating our Redux application, now we can move directly to implementing its components.

In this lesson we will start with the conceptslices (slices). A Redux application has one global state, which is stored in the store. This state is split into slices. In fact, a slice is a piece of storage containing the source state, reducer logic and actions for a specific part of the application.

Our application will feature products, so the first slice we make will contain information about the products. Also, when creating a slice, you need to add a reducer that knows how to process this information.

Let's go to the foldersrcand create a folderparts, and then a folder in itproducts- here we will put all files with code related to the products. In a folderproductswe will create a fileproductsSlice.js, and then import the RTK function into itcreateSlice, with which we will create slices:

import { createSlice } from '@reduxjs/toolkit'

Now let's create a slice for the products and call it'products'. For propertyreducersWe'll leave the empty curly braces for now:

const productsSlice = createSlice({ name: 'products', initialState, reducers: {} })

We also need to determineinitialState- the initial state of the state slice, which will be loaded when the application is first launched. Let's write it immediately after import and before creating the slice. Let this be an array of a pair of objects containing product data. We will indicate for each productid, name, description, price and quantity:

const initialState = [ { id: '1', title: 'Product1', desc: 'This is an amazing product', price: '300', amount: '30', }, { id: '2', title: 'Product2', desc: 'This is a nice product', price: '700', amount: '12', }, ]

At the end of the file code we have the result of the functioncreateSliceLet's take the reducer generated by it and export it. It will be useful to us in other application files:

export default productsSlice.reducer

Open your student application. Create a file in itstudentsSlice.js, as described in the lesson. Create in it usingcreateSliceslicestudents. As the initial value of the slice, let's say you have an array of a pair of objects containing student data.

For each student, make the following fields:id, first name, last name, age and specialty.

At the end of the filestudentsSlice.jsexport the resulting reducer as described in the tutorial.
