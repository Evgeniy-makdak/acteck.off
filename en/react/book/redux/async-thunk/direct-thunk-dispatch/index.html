Directly sending thunk to Redux

When we sent actions from components, we used a hookuseDispatch, which is returned by the store methoddispatch, since the components do not have direct access to store.

This time we need to get objects with sellers and do it once when we start our application. We can do this directly from the main filemain.jsxapplication after the worker starts working.

Let's open our application with products, and in it there is a filemain.jsxand import thunk into itfetchSellers:

import { fetchSellers } from './parts/sellers/sellersSlice'

We have store available in this file, so no hooks are needed here and we can directly use its methoddispatch. Let's after the line:

await worker.start({ onUnhandledRequest: 'bypass' })

Let's add the following:

store.dispatch(fetchSellers())
