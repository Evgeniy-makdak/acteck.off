Vendor page in Redux

The matter remains small. Let's do the last thing in this lesson - add a separate page for each seller to our application. There will only be actions that are already familiar to you. Let's get started.

Let's open our application with products, and in it the foldersellers. Let's create a file in this folderSellerPage.jsx. Now let's start writing codeSellerPagefor our component:

export const SellerPage = () => {}

First, we get the seller’s id and use it to find the object of the desired seller in the slice:

export const SellerPage = () => { let params = useParams() const { sellerId } = params const seller = useSelector((state) => selectSellerById(state, sellerId)) }

Further below in the code forSellerPageAfter receiving the seller, we will receive all the products, and then select from them only those that are posted by this seller:

const productsOfSeller = useSelector((state) => { const allProducts = selectAllProducts(state) return allProducts.filter((product) => product.seller === sellerId) })

And then from the names of the selected products we will form a list usingmap. Moreover, each product name in this list will be a link to the page of this product:

const productNames = productsOfSeller.map((product) => ( <li key={product.id}> <Link to={`/products/${product.id}`}>{product.name}</Link> </li> ))

And at the end of the code forSellerPageLet's return the layout: a header with the name of the seller and a list of products that this seller posted:

return ( <div> <h2>Seller: {seller.name}</h2> <ul>{productNames}</ul> </div> )

Of course, we also need a route for the seller's page. Let's open the fileApp.jsxand add it last to the child routes:

{ path: '/sellers/:sellerId', element: <SellerPage />, },

Don't forget to import the necessary hooks and components intoSellersPage.jsxAndApp.jsx.

Let's launch our application, load the products, and then click on the link in the menu on the left'Sellers'. Now we can go to the page of any seller by clicking on his name, and then see all his products on his page. And by clicking on any of his products, we will go to the page with this product.

That's all for now. We've covered the basics of Redux for building an app and even went a little deeper. We got acquainted with various useful tools. I wish you good luck in creating your Redux applications!
