Проверка билда React через локальный static сервер

Давайте проверим наш React билд с помощью локального static сервера, работающего на NodeJS. Для этого воспользуемся специальной библиотекой serve.

Установим ее глобально через npm:

npm install -g serve

Находясь в папке с проектом вызовем команду на запуск, где build - это имя папки с билдом (может быть и другим, если вам нужно):

serve -s build

После запуска наш билд автоматически откроется во вкладке браузера.
