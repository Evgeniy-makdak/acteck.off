Checking a React build via a local static server

Let's test our React build using a local static server running NodeJS. To do this, we will use a special libraryserve.

Let's install it globally vianpm:

npm install -g serve

While in the folder with the project, call the command to run, wherebuild- this is the name of the folder with the build (can be different if you need):

serve -s build

After launch, our build will automatically open in a browser tab.

Test your build via a local static server.
