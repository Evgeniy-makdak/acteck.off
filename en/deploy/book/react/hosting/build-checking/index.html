Checking the React build

So, we received the build of our React project. Before uploading to hosting, it is advisable to test the build on your local computer.

However, just open the fileindex.htmlYou can't do it in a browser - it won't work. The problem here is in the paths to the static files, which begin with/and thus are absolute.

If you wish, you can manually change absolute paths to relative ones, but this is an inconvenient method. It is more convenient to run the project build on a local server.

There are different types of local servers. In the following lessons we will explore several options.
