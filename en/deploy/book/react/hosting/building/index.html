Building React for hosting

Let your React project reach the point where you are ready to host it. To do this, you need to prepare a project build.

To create a build, while in the project folder, run the following command:

npm run build

After this, a folder will appear in the root of your projectbuild, containing the site assembly. We will upload the contents of this folder to the hosting in the following lessons.
