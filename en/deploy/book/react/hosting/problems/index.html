Проблемы выкладки React на хостинг

Код проекта на React представляет собой обычные JavaScript файлы. Мы, однако, не можем просто залить проект на React на хостинг, как обычную статику.

Дело в том, что в процессе разработки сайта на React нам требуется NodeJS. Но NodeJS требуется лишь для разработки сайта, а когда сайт создан и работает, NodeJS уже не требуется.

Поэтому мы не можем просто залить файлы нашего сайта на хостинг - они просто не заработают без NodeJS.

Правильный подход здесь следующий: когда проект будет готов к деплою, нам нужно будет выполнить его сборку для продакшена.

В результате сборки у нас получится билд, представляющий собой голую статику. Этот билд и нужно загружать на хостинг. В следующих уроках мы обсудим все нюансы этого процесса.

Объясните, почему мы не можем просто залить файлы React проекта на хостинг?
