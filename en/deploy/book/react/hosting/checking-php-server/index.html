Проверка билда React через локальный PHP сервер

Для проверки React билда можно также воспользоваться локальным PHP сервером, если он у вас уже есть.

В этом случае нужно просто скопировать файлы билда в виртуальный сайт на вашем локальном PHP сервере, а затем открыть этот сайт через браузер.

Способ удобен тем, кто кроме React, также работает и с PHP. Если вы не работаете, то просто знайте, что есть такой способ, и переходите дальше к следующим урокам.
