Checking a React build via a local PHP server

To test your React build, you can also use a local PHP server if you already have one.

In this case, you just need to copy the build files to a virtual site on your local PHP server, and then open this site through a browser.

The method is convenient for those who, in addition to React, also work with PHP. If you don't work, then just know that there is such a way, and move on to the next lessons.
