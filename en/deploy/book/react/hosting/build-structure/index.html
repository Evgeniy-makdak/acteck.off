React build structure

Let's look at the basic structure of the project build. If you look in the folderbuild, then you will see the file thereindex.htmland folderstatic. Let's study their purpose.

Fileindex.htmlrepresents the main site file that will be launched in the browser. It contains the initial layout of the project and the inclusion of script and style files.

Folderstaticcontains scripts (folderjs), styles (foldercss) and media files (pictures, etc., foldermedia)
