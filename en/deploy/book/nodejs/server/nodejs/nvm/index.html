Installing NVM on a VPS server

Let's now install NVM. First, let's find out the latest version of NVM by looking at it as follows:link. I see that this version is nowv0.39.3(you may have more).

Now let’s download and install NVM, indicating the version we need in the URL:

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

After NVM is installed, restart the terminal. Then check that NVM is installed:

nvm -v

Install NVM and check that it is installed.
