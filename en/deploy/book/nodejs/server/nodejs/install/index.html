Installing NodeJS via NVM on a VPS server

Let's look at the list of NodeJS versions available for installation:

nvm ls-remote

Let's install the latest LTS version (your version number may be higher):

nvm install 18.13.0

Let's look at the list of installed versions:

nvm list

Let's tell the manager which version to use:

nvm use 18.13.0

Let's check which version of NodeJS is used:

node -v

Install and activate the desired version of NodeJS.
