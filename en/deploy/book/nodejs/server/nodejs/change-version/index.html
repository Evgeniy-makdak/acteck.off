Смена версии NodeJS через NVM на VPS сервер

Как только появятся более новые версии NodeJS, их можно установить и активировать в системе. Для этого сначала нужно деактивировать старую версию NodeJS:

nvm deactivate 18.13.0

После этого при желании ее можно удалить:

nvm uninstall 18.13.0
