Changing the NodeJS version via NVM to a VPS server

As soon as newer versions of NodeJS are available, they can be installed and activated on the system. To do this, you first need to deactivate the old version of NodeJS:

nvm deactivate 18.13.0

After that, if desired, you can delete it:

nvm uninstall 18.13.0
