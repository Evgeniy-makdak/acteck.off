Частота процессора VPS для NodeJS хостинга

Одним из важных показателей является частота процессора или CPU. Частота указывается в виде записи 1x2.2. Первое число указывает на количество ядер, а второе - на их частоту в ГГц.

Увеличение количества ядер слабо увеличивает стоимость хостинга. Поэтому можно взять 1x2.2 или 2x2.2. Этого хватит как для тестовых целей, так и для боевых проектов, если они не сильно нагруженные.

В настройках конфигурации мы можем менять количество ядер, а также выбрать одну из частот: 2.2 или 3.4.

Повыбирайте различные частоты и разное количество ядер и посмотрите как меняется стоимость сервера.
