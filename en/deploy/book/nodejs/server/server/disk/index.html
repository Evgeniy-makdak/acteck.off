VPS disk size and type for NodeJS hosting

Disk size doesn't really matter for regular projects. The fact is that by default 20 GB is allocated, which is more than enough for any project, unless you have files, music or movies stored there.

Disk type - HDD or SSD affects the speed of data reading. For educational purposes, HDD is suitable for us. In combat conditions, it is permissible to take both options.

Play around with the size and type of disk and see how the cost of the server changes.
