OS for VPS for NodeJS hosting

Servers can run either Windows or Linux. You shouldn't buy servers for Windows - they are not reliable, they work slower, and they require more resources.

Therefore, our choice is servers running Linux. And here there are already options, since Linux has many distributions, that is, OS options. Ubuntu, Debian and CentOS are available on our hosting.

There is no particular difference between these distributions. You should choose the distribution that you know how to work with. I use Ubuntu in my daily life, so I recommend choosing this distribution. Moreover, later in the textbook the story will be about this distribution :)

So, select Ubuntu, and then in the list of versions of this OS, select the latest LTS version.
