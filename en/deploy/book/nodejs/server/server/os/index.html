ОС для VPS для NodeJS хостинга

Сервера могут работать либо под Windows, либо под Linux. Сервера под Windows брать не стоит - они не надежные, медленнее работают, требуют больше ресурсов.

Поэтому наш выбор - это сервера под управлением Linux. И тут уже есть варианты, так как в Linux есть много дистрибутивов, то есть вариантов ОС. На нашем хостинге доступны Ubuntu, Debian и CentOS.

Особой разницы между этими дистрибутивами нет. Следует выбирать тот дистрибутив, с которым вы умеете работать. Я в повседневной жизни пользуюсь Ubuntu, поэтому рекомендую выбрать именно этот дистрибутив. Тем более, что далее в учебнике рассказ будет именно об этом дистрибутиве:)

Итак, выбираем Ubuntu, а затем в списке версий этой ОС выбираем самую последнюю LTS версию.

Выберите нужную операционную систему.
