VPS RAM for NodeJS hosting

Random access memory, or RAM, is the most important and expensive indicator.

For educational purposes, 1 GB is enough for us. You should not take less, since in this case the server will periodically crash due to lack of memory.

For combat unloaded projects, 1-2 GB will be enough. I recommend initially buying 1 GB, then if there is not enough memory, you can simply buy more. To do this, you will need to write to hosting support and say that you need more RAM for the specified server.
