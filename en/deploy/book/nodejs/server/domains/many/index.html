Multiple domains on a VPS server for NodeJS

In order for us to have several websites with their own domains on one VPS server, we needreverse proxy server. This is the kind of program that will listen80port and resolve requests. We will use the Nginx server as such a program.

How it will work. Let's say we have two domains. Let's say that on our VPS the script for the first domain is running on3001port, and the script for the second domain is on3002port Nginx reverse proxy will listen80port, see what domain was requested and will forward the request to the appropriate port.

In the following lessons we will figure out how to install and configure Nginx in reverse proxy mode.
