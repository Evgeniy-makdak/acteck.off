Установка Nginx на VPS сервер

Давайте установим Nginx на наш VPS сервер. Для начала выполним следующую команду:

apt install software-properties-common

Теперь получим последнюю версюю Nginx:

apt-add-repository ppa:nginx/stable

Дальше обновим списки пакетов из репозиториев:

apt update

Установим Nginx:

apt install nginx

Добавим Nginx в автозагрузку, чтобы он запускался автоматически, после перезагрузки сервера:

systemctl enable nginx

Проверим, что Nginx установился:

nginx -v

Установите Nginx.
