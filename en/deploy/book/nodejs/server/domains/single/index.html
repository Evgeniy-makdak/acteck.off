Linking one domain to a VPS server on NodeJS

Let's try the easiest way to link a domain to a server. This method will work if there is only one domain on your server.

To use this method, you need to run the test site script on80port:

app.listen(80, function() { console.log('running'); });

A site launched on such a port will be accessible only by IP address, without specifying the port:

http://12.345.678.90

Next, you need to have a domain or subdomain purchased from a registrar. In the domain registrar panel, go to the sectionДомены, we are looking thereУправление зонойand there we look for our domain (or subdomain) and change the A-record for it to the IP of our server.

We wait a few minutes (no more10) and after that we try to access the domain name. If everything is done correctly, we should see the result of the test script.
