Launching a test site via NodeJS on a VPS server

Let's now launch our site so that it is accessible online through a browser. To do this, go to the folder with the site:

cd /var/www/test.com

And run the launch command:

node app.js

After this, you should go to the address in your browser, which should contain the IP of your server and the port of the test script:

http://12.345.678.90:3000/

Launch the site and open it through your browser.
