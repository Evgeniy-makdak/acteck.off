Uploading a test NodeJS site to a VPS server

After installing NodeJS, we are ready to deploy and launch our first test site. This is what we will do now.

First, let's go to the folder with sites:

cd /var/www

Now let's create a folder for our test site. I called himtest.com. But it’s better to name this folder the same way your domain will be called. This is not necessary, but will help avoid confusion.

So, let's create a folder for the site:

mkdir test.com

Create a folder for the test site.

Upload the test site files there via SFTP (without the foldernode_modules).
