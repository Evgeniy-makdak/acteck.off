Problem with a test site using NodeJS on a VPS server

So, we launched our test site online. There is, however, a problem - if you close the terminal, the site will not work.

We, of course, need the site to work constantly, without our connection to the server. This can be done, but to do this you need to install a special program that will keep the script running even after the terminal is disconnected from the server.

NodeJS has several libraries for this. ThisforeverAndpm2. We will use PM2 since it keeps the script running even after a server reboot. In the next section of the tutorial we will study this library.

Make sure that disconnecting the terminal from the server stops the site.
