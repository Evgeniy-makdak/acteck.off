Connecting to the server via SSH

So, in the previous section we bought a server and got access to it. Now we can connect to this server through a terminal and control it as if we were our own computer.

Through the terminal we can install any programs, work with the file system and run NodeJS scripts.

This is done through a special SSH protocol. It's already built into your terminal. You just need to know the server's IP address and SSH password. The hosting sent this information to you by email.

Let's take the IP of your server as an example:12.345.678.90(you substitute yours). Then we can connect to our server by typing the following command in the terminal:

ssh root@12.345.678.90

If the IP address is correct, you will be asked to enter your SSH password. Copy the password that the hosting sent you in the letter and clickEnter.
