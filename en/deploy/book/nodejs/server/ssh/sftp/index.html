Доступ к VPS серверу через SFTP

Закачивать и изменять файлы через терминал можно, но не очень удобно. Гораздо удобнее будет подключиться к файловой системе нашего сервера через FTP.

Проблема в том, что у нас есть доступы к SSH и нет доступов к FTP. Поэтому мы воспользуемся специальным протоколом SFTP, который позволяет закачивать файлы через SSH.

Давайте подключимся через SFTP с помощью Filezilla. Для этого в меню программы зайдем в File, там выберем Site Manager и создадим новый сайт.

В настройках сайта выберем протокол SFTP. В поле Host введем IP сервера, в поле логин - root, в поле пароль - ваш пароль от SSH.

Подключитесь к вашему серверу через SFTP.
