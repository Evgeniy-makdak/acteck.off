Access to VPS server via SFTP

It is possible to upload and change files through the terminal, but it is not very convenient. It will be much more convenient to connect to the file system of our server via FTP.

The problem is that we have SSH access and no FTP access. Therefore, we will use the special SFTP protocol, which allows you to upload files via SSH.

Let's connect via SFTP using Filezilla. To do this, go to File in the program menu, select Site Manager there and create a new site.

In the site settings, select the SFTP protocol. In the Host field enter the server IP, in the login field -root, in the password field - your SSH password.

Connect to your server via SFTP.
