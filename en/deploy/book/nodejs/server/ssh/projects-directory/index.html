Folder for projects on VPS via SSH

In the Ubuntu operating system, projects with websites are usually placed in a foldervar/www. Let's create it.

First, let's go to the foldervar:

cd /var/

Let's create a folder in itwww:

mkdir www

Let's check that the folder has been created:

ls
