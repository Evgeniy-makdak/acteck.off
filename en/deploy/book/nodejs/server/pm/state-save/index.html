Saving state via PM2 on a VPS server

Using PM2, you can save the current list of scripts so that they are automatically restarted after the server is rebooted.

To do this, you first need to initialize the save:

pm2 startup

After that, every time you change the list of scripts, you need to save the current state:

pm2 save

Run your test scripts as usual. Reboot the server. Make sure the sites are down.

Save running scripts. Reboot the server. Make sure the sites are now working.
