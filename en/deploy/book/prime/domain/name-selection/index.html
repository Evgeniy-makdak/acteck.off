Choosing a domain name

In this tutorial we will learn how to choose domain names. Below are a number of generally accepted recommendations. It is better to follow them, but not necessary. These are the recommendations:

  • The domain name should be short, memorable and consistent with the theme of the site.
  • Use keywords in your domain name to improve your site's promotion.
  • Avoid hyphens and numbers, but don't have to.
  • Avoid other people's trademarks - it's illegal.

  • Select the right zone for your domain.
  • Check the domain to see if it is busy. If it is busy, check the name in other zones.
