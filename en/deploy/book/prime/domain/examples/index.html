Example domains

Some domain names are reserved for use in examples (in documentation or in a tutorial) in order to avoid conflicts with real-life sites. These names are not available for registration. Here they are:example.com, example.net, example.org

Unfortunately, these names are somewhat long, and therefore textbooks often use names likesite.comorsite.ru, corresponding to real existing sites.
