Buying shared hosting

Let's now buy our first hosting. For simplicity, we will buy it on the same site where we registered our first domain (we will study a more complex option later).

To do this, on the hoster’s website, among the variety of hosting options, you should find virtual hosting. For example, for hoster.by ithere.

Buy the simplest plan, but one that allows you to work with PHP, create databases and subdomains. On some hosting sites, the most basic tariff may be for bare static data, so you will have to take a slightly higher one.

For Belarus I have a promotional code for a small discount fromhoster.by. A promotional code was issued specifically for readers of this textbook. Here it is: U23YQ5S8. To use it, enter it into the appropriate field when purchasing virtual hosting.
