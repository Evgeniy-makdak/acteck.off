What is hosting technically?

Technically, hosting is a computer connected to a good Internet connection. This computer is located somewhere on the hoster’s territory, in some room (it’s calledserver room). As a rule, there are many computers in this room at once.

Websites run on all these computers. As a rule, many sites run on one computer at once, and these sites have different owners.

The server room is provided with good Internet, with high bandwidth.
