Тестовый сайт со статикой

Давайте теперь сделаем тестовый сайт со статикой. У него будет HTML файл, CSS файл и файлы с картинками.

Сделайте тестовый сайт. Проверьте его работу на вашем компьютере.

Подключитесь с помощью FileZilla к вашему сайту.

Залейте на ваш сайт тестовый код. Зайдите на сайт через браузер и убедитесь, что все работает.

Внесите изменения в код сайта на вашем компьютере и залейте изменения на хостинг.
