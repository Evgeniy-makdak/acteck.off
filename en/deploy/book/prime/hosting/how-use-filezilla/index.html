How to use FileZilla

Let's now learn how to use FileZilla. At the top there are input fields in which you need to enter your FTP accesses:

After you enter your FTP accesses and click on the button, FileZilla will begin connecting to your hosting.

When the connection is established, the program will be divided into two parts. On the left will be your computer's file system:

And on the right there will be the file system of the site on the hosting:

In order not to re-enter accesses each time, it is better to make FileZilla remember them. To do this, open the File tab, and in it the Site Manager tab. Click on the button to create a new site:

And then we enter FTP accesses:

And the next time you need to connect to a site, you will no longer need to enter access, but you can go to the Site Manager tab and select the site you need there.
