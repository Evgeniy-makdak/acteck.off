Parts of FTP Access

As you already know, to connect to the site you needFTP accesses.

They consist of three parts:hostname, usernameAndpassword. Sometimes, extremely rarely, a fourth part is also needed -port number.

What a username and password are, I think it’s obvious to you.

As the host, you must specify your domain name or its IP address. As a rule, the hoster gives you the host name in a letter with access. But not always. If it's not there, just write the domain name - it should work.

The port number is rarely needed. Usually we do not specify it and the default value is taken. However, this setting often causes problems because people try to write something there, for example, the port number for a NodeJS script or something else. Here, follow the rule: unless you are explicitly told that this is the port number for FTP, then do not touch this setting.
