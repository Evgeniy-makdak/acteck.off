Backups on VH

Backup- This is the saved state of the site. On shared hosting, this is usually done automatically once every few days.

Site files, databases, various accounts (FTP, database, etc.) are saved.

Backups are needed so that in case of any problems, you can quickly roll back to the previous version of the site.

Let's learn how to use backups in our hosting control panel. To do this, find a section with a name similar to Backup. Go into it and study what is in it.

Find the backups section in your hosting control panel.

Roll back the site to the previous version. To do this, upload the files with the site to the hosting. Wait a few days for these files to be included in the next backup. Then change the files. After that, in the backup management section, roll back to the previous version of the site. Check that the site files are back to the previous version.
