Выкладка сайта PHP на хостинге

В данном учебнике мы будем учиться выкладывать PHP сайты на виртуальный хостинг (VH).

Эта операция немного сложнее, чем выкладка обычного сайта со статикой, так как тут есть некоторые важные нюансы, которые нужно предварительно изучить.

Для изучения этого учебника вам нужно пройти основной учебник по выкладке сайтов на хостинг, а также учебник по работе с SQL на виртуальном хостинге. У вас уже должен быть домен и виртуальный хостинг с PHP и SQL.

Учебник состоит из двух частей: базовая выкладка и продвинутые нюансы. Вторую часть изучать не обязательно, но очень желательно иметь хотя бы общее представление. Итак, давайте приступим!
