Layout of PHP website on hosting

In this tutorial we will learn how to host PHP sites on virtual hosting (VH).

This operation is a little more complicated than laying out a regular static website, since there are some important nuances that need to be studied first.

To study this tutorial you need to go throughbasic tutorial on hosting websites, and alsotutorial on working with SQL on shared hosting. You should already have a domain and shared hosting with PHP and SQL.

The textbook consists of two parts: basic layout and advanced nuances. It is not necessary to study the second part, but it is very advisable to have at least a general understanding. So let's get started!
