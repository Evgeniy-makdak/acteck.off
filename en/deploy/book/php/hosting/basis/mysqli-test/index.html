Testing mysqli connection to a database on hosting

Let's now test connecting to the database through the mysqli extension using the following test code:

<?php $host = ''; // имя хоста $name = ''; // имя базы данных $user = ''; // имя пользователя $pass = ''; // пароль $link = mysqli_connect($host, $user, $pass, $name); $query = 'SELECT * FROM users'; $res = mysqli_query($link, $query) or die(mysqli_error($link)); var_dump($res); ?>

Substitute your accesses into the test code, then upload this code to your website and test the database connection.
