Checking the PHP version on the hosting

The versions of PHP installed on your computer and on the hosting may differ. This may lead to the fact that a script running locally will not work on the hosting due to differences in versions.

In order to find out what version of PHP is installed, you need to make a PHP file with the following content:

<?php phpinfo(); ?>

Then you need to access this file through the address bar of your browser. The current PHP version will be written at the very top of the page.
