Disabling PHP errors on VS

On hosting, you need to disable the output of PHP errors to the browser. Firstly, it scares users and indicates poor quality of the site, and secondly, errors can give hints to hackers in the process of hacking the site.

Let's turn off error output. First, let's create an error, for example, by outputting a non-existent variable:

<?php echo $test; ?>

Now throughini_setLet's turn on error output and make sure that it is output to the browser:

<?php ini_set('display_errors', 'on'); echo $test; ?>

Now let’s turn off error output and make sure that it is no longer displayed:

<?php ini_set('display_errors', 'off'); echo $test; ?>

Disable error output to the browser.
