Changing file name with PHP errors on VS

You can change the file name for storing errors. This is what the setting is forerror_log:

<?php ini_set('display_errors', 'off'); ini_set('log_errors', 'on'); ini_set('error_log', 'errors.log'); echo $test; ?>

Change the name of the error file.

Intentionally create a mistake. Make sure it appears in the correct file.
