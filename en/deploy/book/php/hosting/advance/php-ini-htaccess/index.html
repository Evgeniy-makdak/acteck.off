Changing php.ini settings via htaccess

You can also change the settingsphp.iniusing the htaccess file. To do this, in this file we first write the command to change, then, separated by a space, the name of the setting, and then, separated by a space, its value.

There are two commands to change. Teamphp_flagis intended for boolean values ​​that can beonoroff, and the teamphp_value- for everyone else.

For example, let's enable error output:

php_flag display_errors on

Now let's set the maximum upload file size:

php_value upload_max_filesize 10M
