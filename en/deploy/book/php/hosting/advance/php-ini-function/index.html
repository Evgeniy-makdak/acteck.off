Изменение настроек php.ini через ini_set на хостинге

На виртуальном хостинге можно также менять настройки php.ini с помощью функции ini_set. Давайте сделаем это:

<?php ini_set('memory_limit', '64M'); ?>

Проверим Master Value и Local value:

<?php phpinfo(); ?>

Описанным способом поменяйте настройку max_execution_time.
