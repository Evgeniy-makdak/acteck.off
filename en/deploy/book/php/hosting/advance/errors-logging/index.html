Логгирование ошибок PHP на VS

Давайте теперь сделаем так, чтобы PHP ошибки выводились в специальный файл на хостинге. В случае проблем можно будет открыть этот файл через FTP и посмотреть, нету ли там ошибок.

Воспользуемся для этого настройкой log_errors. Она по умолчанию делает так, что ошибки будут выводиться в файл с именем error_log (без расширения). Включим эту настройку:

<?php ini_set('display_errors', 'off'); ini_set('log_errors', 'on'); echo $test; ?>

Включите вывод ошибок в файл.

Намеренно создайте ошибку. Убедитесь, что она появится в файле.
