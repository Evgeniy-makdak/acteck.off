Logging PHP errors on VS

Let's now make sure that PHP errors are output to a special file on the hosting. In case of problems, you can open this file via FTP and see if there are any errors there.

Let's use the setting for thislog_errors. By default, it makes it so that errors will be output to a file callederror_log(no extension). Let's enable this setting:

<?php ini_set('display_errors', 'off'); ini_set('log_errors', 'on'); echo $test; ?>
