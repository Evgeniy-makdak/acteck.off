PHP configuration via php.ini on hosting

The PHP engine can be configured using a special filephp.ini. This file contains various settings and their meanings.

There are different ways to change these values:

  • You can edit the settings manually by opening the file in an editor. In this case, the settings are applied to all sites.
  • You can change settings through the htaccess file. In this case, the settings are applied to a specific site.
  • You can change settings through a special PHP functionini_set. In this case, the settings are applied to a specific script.

On hosting we do not have access to direct file editingphp.ini. However, the hosting control panel may have an interface for changing a limited number of parameters.

The hosting option will also work withini_setand htaccess file, but also in a limited way. We will go into more detail in the following lessons.
