Useful values ​​from the php.ini file on the hosting

Let's look at the useful settings from the filephp.ini. These settings are usually available for editing on shared hosting:

Settings Description
display_errors Displaying errors in the browser. Meaningonincludes the output and the valueoffdisables.
memory_limit Limit on the amount of RAM allocated to one script. Measured in megabytes.
max_execution_time Script execution time limit Measured in seconds.
post_max_size Limit on the size of data sent using the POST method. Measured in megabytes.
upload_max_filesize Limit on the size of files sent via input with typefile. Measured in megabytes.
