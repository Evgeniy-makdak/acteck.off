Writing to a file in PHP on hosting

As a rule, shared hostings operate via Linux. In thisOSthere is a feature associated withaccess rightsto folders. On shared hosting, it may be that the folder in which you want to write your file is closed for writing.

For example, let's try to write a file to a foldertest(this folder should already be created by you):

<?php file_put_contents('test/file.txt', '123'); ?>

If the recording is successful, then everything is in order. If not, then you need to give write access to our folder. One option to do this is through FileZilla.

Let's do this. Open your site through FileZilla and right-click on the folder you want to change permissions on. In the menu that appears, select File permissions.
