Creating a user for a database on hosting

Let's say you have created a database. This, however, is not enough for your program to connect to this database. The fact is that onlyusersassociated with this database.

Each database must have one or more users associated with it. In this case, our program will be able to connect to the specified database under the specified user.

Different users are needed to differentiate access rights to the database. Technically, this means that users will not be able to execute all SQL queries, but only those that are allowed for a given user.

Let's create a new user and link it to our database. To do this, in your hosting control panel, find where new users for databases are created. As a rule, this is done on the same page where databases are created.

You need to do two steps.

Step 1. Create a new user and set his login and password.

Step 2. Link your user to your database.

At the time of binding you will be asked to chooseprivilegeuser for this database. For security, we should give a minimum of privileges. Typically these are SELECT, INSERT, UPDATE and DELETE.

Create a new database user.

Link the new user to your database. Give it the correct privileges.
