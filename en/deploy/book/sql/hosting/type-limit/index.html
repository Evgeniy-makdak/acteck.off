Ограничения на тип SQL СУБД на хостинге

Существуют различные системы управления баз данными (СУБД), работающие на SQL: MySQL, PostgreSQL, Oracle, MS SQL и другие. Все они имеют свои плюсы и минусы и используются для разработки сайтов.

На виртуальном хостинге, однако, обычно доступны не все популярные движки баз данных, а только MySQL.

Если вам нужны другие СУБД, то вам придется покупать хостинг на сервере.
