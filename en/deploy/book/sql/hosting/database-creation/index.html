Creating a database on hosting

Our shared hosting plan says that we have this many databases. In fact, initially they do not exist, but if necessary we must create them.

But we need to create a database. This is not done in PMA, as we are used to locally. The fact is that on hosting PMA is usually limited and does not allow the creation of databases.

This is done to ensure that the number of databases you can create is limited. There is no physical need for this. This is just an artificial restriction to create a line of tariffs with different prices.

Therefore, the database must be created in the hosting control panel.

There is a small nuance with the name of the database. When creating a database, you will have to come up with a name for it. However, a small random string will still be appended to this name.

Find database management in your hosting control panel.
