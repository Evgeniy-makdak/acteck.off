Client components in the NextJS framework

Let's now convert our server component to a client component. To do this, write a directive at the beginning of the component file'use client':

'use client'; export default function Test() { return <h1>hello, user!</h1>; }

Now we will have access to states created throughuseState. Let's create a state as an example and display its contents in the layout:

'use client'; import { useState } from 'react'; export default function Test() { let [name, setName] = useState('user'); return <h1>hello, {name}!</h1>; }

Make a client component with two inputs and a button. Let numbers be entered into the inputs. By clicking the button, display the sum of the entered numbers in a paragraph.

Remove the directive'use client'. Make sure that in this case NextJS will throw an error about the reason that states cannot be used inside server components.
