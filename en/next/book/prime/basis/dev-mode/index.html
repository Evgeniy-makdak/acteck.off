Development mode in NextJS

Development mode is designed to make the process of writing code and testing it convenient and flexible. This mode includes tools that allow you to find errors, test and quickly make the necessary changes to the code.

Main features of development mode:

  • Hot reload - when you make any change to the code, the server automatically restarts and the page with the code is updated.
  • Display errors on the page in the browser to make them easier to see and fix. The browser also shows the location of files with errors, making them easier to find.
  • The code is not optimized for maximum performance, so there may be additional checks, warnings, and even pages loading a little slower.
  • Detailed log output in the terminal to make it easier to track what is happening on the server.

Development mode is started with the following command:

npm run dev

After that, an address will appear in the terminal, by clicking on which you can see your project.

To exit developer mode, you must press the key combinationCTRL+C.
