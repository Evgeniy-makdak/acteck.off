Project file structure in the NextJS framework

After creating a project, a certain file structure will appear in its folder. Let's look at its most important components:

  • /node_modules/
  • /src/
    • /app/
      • globals.css
      • layout.jsx
      • page.jsx
  • next.config.mjs
  • package.json
  • tailwind.config.js

The files that we will make while working with the project will be stored in folderssrcAndsrc/app. Currently in the foldersrcthere is only one working folderapp. And it stores files for working with CSS settings (global.css), a file with the main page layout (layout.jsx) and a file with the main project route (page.jsx).

In filenext.config.mjscontains basic project settings.

In filetailwind.config.jscontains basic settings for the Tailwind styles used in this project.
