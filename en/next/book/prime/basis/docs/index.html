NextJS framework documentation

This textbook is designed to make it easy and simple to learn. Therefore, I am omitting some of the information. After working through the textbook, be sure to read the entireofficial NextJS documentation.

Please note that the documentation's code examples will default to TypeScript, but you can switch to JavaScript.

It also makes sense to refer to the documentation while reading this textbook. Especially if something described in the textbook does not work or does not work as it should. In this case, the documentation is the primary source in which you should check what you read.
