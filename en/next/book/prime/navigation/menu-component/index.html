NextJS Menu Component

In the following lessons we will explore the special features of NextJS for creating a website menu. To do this, we need a component with a site menu. In this lesson we will make a blank for this component, and we will insert links in the following lessons.

Our preparation will look like this:

export default function Menu() { return <> ... </>; }

Let's insert this template into the main site layout:

import Menu from '@/comps/menu/menu.jsx'; export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> <nav> <Menu /> </nav> </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }
