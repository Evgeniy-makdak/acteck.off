Problem with regular menu in NextJS

In the previous lesson we made a menu from regular links:

export default function Menu() { return <> <a href="/">home</a> <a href="/about">about</a> <a href="/price">price</a> <a href="/contacts">contacts</a> </>; }

Let us now discuss some of the problems associated with this. The fact is that following the links in this menu leads to a reload of the entire page. This is correct from an SEO point of view, but not very optimal from a performance point of view. The fact is that when the page is reloaded, we again download the entire site layout, although only the content changes. It would be optimal to download only the page content.

NextJS provides a solution to this problem. We will look at it in the next lesson.

Make sure that clicking regular links causes the page to reload.
