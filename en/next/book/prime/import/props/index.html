Пропсы компонентов в NextJS

В NextJS так же, как и в React, данные из родительских компонентов в дочерние можно передавать через пропсы.

Давайте посмотрим, как это делается на практике. Пусть наш дочерний компонент может принимать пропсы:

export default function Child({text1, text2}) { return <p> child </p>; }

Давайте выведем их в верстке компонента:

export default function Child({text1, text2}) { return <p> child {text1} {text2} </p>; }

Пусть в родительском компоненте подключается наш дочерний компонент:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> <Child /> </>; }

Давайте передадим в дочерний компонент данные из родительского:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> <Child text1="aaa" text2="bbb" /> </>; }

Сделайте дочерний компонент Product для вывода продукта. Пусть пропсами передаются название и цена продукта.

Пусть дан следующий массив с продуктами:

let prods = [ { id: 1, name: 'prod1', cost: 100, }, { id: 2, name: 'prod2', cost: 200, }, { id: 3, name: 'prod3', cost: 300, }, ];

Выведите в цикле данные продуктов, используя дочерний компонент Product.
