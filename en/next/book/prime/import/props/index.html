Component props in NextJS

In NextJS, just like in React, data from parent components to child components can be passed throughprops.

Let's see how this is done in practice. Let our child component be able to accept props:

export default function Child({text1, text2}) { return <p> child </p>; }

Let's display them in the component's layout:

export default function Child({text1, text2}) { return <p> child {text1} {text2} </p>; }

Let our child component be connected in the parent component:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> <Child /> </>; }

Let's pass data from the parent component to the child component:

import Child from '@/comp/child/child.jsx'; export default function Test() { return <> <h1>Test</h1> <Child text1="aaa" text2="bbb" /> </>; }
