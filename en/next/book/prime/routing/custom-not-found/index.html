Customizing 404 errors in NextJS

In the previous tutorial, we saw that NextJS already has a beautifully designed page built into it404errors.

But if we wish, we can create our own error page, designing it in the style we need (they saycustomize).

To do this you need to create a file with a special namenot-found.jsxand place it in a folderapp:

export default function NotFound() { return <> <h1>Error 404</h1> <p> this page is not found </p> </> }

Make a custom output404errors. Make sure that in case of an error, your file is returned, and not the one built into NextJS.
