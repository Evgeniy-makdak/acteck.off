Nested routes in NextJS

Suppose we have some URL consisting of several segments, for example, like this:/user/show/name.

To make a file that will be shown at a given URL, we will need to make several subfolders. In our case like this:

  • /app/
    • /user/
      • /show/
        • /name/

Let's make the corresponding file:

export default function Name() { return <p> user </p>; }
