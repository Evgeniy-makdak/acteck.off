Metadata in NextJS

Let's now see how to add a title and meta description to the page. In NextJS, work with them is carried out jointly and therefore they are called one common wordmetadata.

Metadata is specified using a special object containing a title and meta description:

{ title: '...', description: '...', };

In this case, NextJS itself will insert metadata into the appropriate tags. We don't need to care about this. Therefore, there is no tag in the main site layouthead, so it will be generated automatically:

export default function RootLayout({children}) { return ( <html lang="en"> <body> {children} </body> </html> ); }

There are different ways to set metadata. We will study them in the following lessons.
