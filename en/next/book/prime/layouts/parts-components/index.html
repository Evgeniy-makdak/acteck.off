Site parts in components in NextJS

In the previous lesson, we wrote the text of parts of the site directly in the layout file:

export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> header </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }

This, however, can be inconvenient when parts of the site contain a lot of code and text. Therefore, usually parts of the site are placed in separate component files.

Let's take the site header as an example into a separate component. Let's create the corresponding file:

export default function Header() { return <> header </>; }

We import our component in a file with a layout:

import Header from '@/comp/header/header.jsx'; export default function RootLayout({children}) { }

Let's display the text of our component using a tag with its name:

import Header from '@/comp/header/header.jsx'; export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> <Header /> </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }
