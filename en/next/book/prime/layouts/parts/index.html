Site parts in the main layout in NextJS

Let's take a look at our site layout. Now it contains a command for inserting changing page content:

export default function RootLayout({children}) { return ( <html lang="en"> <body> <main> {children} </main> </body> </html> ); }

However, there are usually other parts of a site that remain the same across different pages. For example, this could be a header and footer. Let's add them to our layout:

export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> header </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }
