Global CSS in NextJS

Inside a folderappby default there is a fileglobals.csswith the global CSS styles of our project.

Global styles can be imported into any layout, page, or component. Without import they will not apply.

Let's import the global styles file into the main site layout:

import '@/app/globals.css';

Change the background of your page.

In fileglobals.cssset some CSS class. Apply it in some component.
