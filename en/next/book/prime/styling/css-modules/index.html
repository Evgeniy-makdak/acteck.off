CSS modules in NextJS

NextJS already has built-in support for CSS modules. They are separate CSS files in which class names are in local scope, which avoids conflicts between class names from different components.

Let's see how to use them. Let us have some component:

export default function Test() { return <p> text text text </p>; }

Let's create a CSS module for it. To do this, in the same folder we will create a file with the extension.module.cssand an arbitrary name. We will write the following CSS class in it:

.text { color: blue; }

Let's import our CSS module into the component file:

import styles from './test.module.css'; export default function Test() { return <p> text text text </p>; }

Now let's apply the CSS class to the paragraph inside our component:

import styles from './test.module.css'; export default function Test() { return <p className={styles.text}> text text text </p>; }
