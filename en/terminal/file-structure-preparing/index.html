Preparing the file structure

Let's look at the basic terminal commands. Let us create the following file structure for this:

  • /xxx/
    • /aaa/
      • dddd.txt
      • eeee.txt
      • /zzz/
    • /bbb/
      • cccc.txt
      • ffff.txt
      • /kkk/
      • /yyy/

Let the terminal open on the folderxxx. In the code examples, I will indicate the working directory above the code along with the terminal commands:

your command

On your command line, the current working directory will likely appear before your commands and end with a dollar sign:

xxx$ your command
