Delete a file using the command line

Teamrmdeletes the file. It takes the path to the file as a parameter:

rm dir/file.txt

You can delete multiple files by listing them separated by a space:

rm dir/file1.txt dir/file2.txt

Use the appropriate command to delete one of your files.
