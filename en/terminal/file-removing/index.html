Удаление файла с помощью командной строки

Команда rm выполняет удаление файла. Параметром она принимает путь к файлу:

rm dir/file.txt

Можно удалить несколько файлов, перечисляя их через пробел:

rm dir/file1.txt dir/file2.txt

С помощью соответствующей команды удалите один из ваших файлов.
