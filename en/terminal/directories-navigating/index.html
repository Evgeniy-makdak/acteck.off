Navigating folders using the command line

Let's first learn how to navigate through folders. This is what the team doescd.

Let the working folder of our terminal be the folderxxx. Let's go to the folderaaa, located inside it:

cd aaa

Now let's go to the folderzzz, located insideaaa:

cd zzz

If you specify two dots as the folder name, you will move up one folder:

cd ..

Let's take it one more level:

cd ..

A dot as a file name indicates the current directory. In the following example we will stay put:

cd .

The following command will change to the previous directory:

cd-

You can specify not only the directory name as a path, but also the path to it:

cd aaa/zzz

You can start searching for the catalog by going up a level. For example, being in the folderaaawe want to go to the folderbbb/kkk. Obviously, to do this we need to first go up a level - to the folderxxx. This can be done as follows:

cd ../bbb/kkk

You can go up as many levels as you like:

cd ../../bbb/kkk
