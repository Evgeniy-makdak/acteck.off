Terminal Basics

In this lesson we will look at the basic command line commands for working with JavaScript tools. Command line (terminal, console) is a program that allows you to enter text commands. These commands can be used to navigate the file system and run programs.

If you are a Linux or Mac user, then you probably know that the command line is organically built into your operating system, and you know how to at least launch it.

If you work on Windows, then you should be aware that not all of the commands you learn below will work on this OS if you use the built-in command line. In your case, you need to take a proper command line program, such as Power Shell.
