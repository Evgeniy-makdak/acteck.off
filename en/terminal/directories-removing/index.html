Removing a directory using the command line

Teamrmdirperforms deletion of an empty directory:

rmdir test

Teamrmwith a flag-Rused to remove directories and contents inside them:

rm aaa -R

Be very careful with this command and always check what directory you are in. It deletes everything and cannot be undone.
