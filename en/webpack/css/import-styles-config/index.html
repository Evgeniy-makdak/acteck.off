Setting up import of CSS styles into Webpack

After installing the loaders, we can use them in the configuration file. Let's do this:

export default { entry: './src/index.js', module: { rules: [ { test: /\.css$/i, use: ['style-loader', 'css-loader'], }, ], }, };
