Loaders for CSS styles in Webpack

First, we need to install loaders that will load CSS from a file and add it to the JavaScript bundle.

Loadercss-loaderneeded so that Webpack can process imports of CSS files. Let's install it:

npm install css-loader --save-dev

Loaderstyle-loaderneeded so that Webpack can embed CSS code into an HTML page. Let's install it:

npm install style-loader --save-dev
