Importing CSS styles into Webpack

In Webpack you can import CSS styles directly into a JavaScript file. Like this:

import './styles.css';

The imported styles will be inserted directly into the JavaScript assembly file. Then, when connecting the bundle to the HTML file, the script itself will insert these styles into the tagstyle.

However, Webpack cannot import styles that easily. To do this you need to make some settings. We will deal with them in the following lessons.
