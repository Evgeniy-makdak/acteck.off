Connecting CSS libraries in Webpack

You can install CSS libraries via npm and connect them to CSS files. Let's look at an example librarynormalize.css. Let's install it:

npm install normalize.css

Import the library directly into the CSS file. In this case, you should put a tilde at the beginning of the path~. This will tell Webpack to look for files innode_modules. Let's import:

@import '~normalize.css'

Install and import the librarymodern-css-reset.

Assemble the project. Look where the installed library ends up during the build.
