Testing CSS bundle assembly in Webpack

Let's test assembling the CSS into a separate bundle. Let's say we have two CSS files.

First:

body { background: red; }

Second:

p { color: green; }

We import them to the entry point:

import './styles1.css'; import './styles2.css';

The command to build the project will collect everything into a separate file calledmain.css:

body { background: red; } p { color: green; }

Import three CSS files to your entry point.

Build in development mode. Examine the bundle's CSS file.

Build in production mode. Examine the bundle's CSS file.
