Начальная настройка сборщика Webpack

Установленый сборщик должен быть настроен. Это делается в специальном конфигурционном файле webpack.config.js, размещаемым в корне сайта.

Для начала нам нужна одна настройка entry, которая задает точку входа. Также нам нужна настройка entry, задающая режим сборки (с ним мы будем разбираться далее в учебнике).

Итак, создадим файл настроек со следующим содержимым:

export default { mode: 'development', entry: './src/index.js', };

Создайте описанный конфигурционный файл в вашей рабочей папке.
