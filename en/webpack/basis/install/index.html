Установка Webpack

Давайте теперь установим Webpack через npm. Для начала установим сам сборщик:

npm install webpack --save-dev

Теперь установим также Webpack CLI, позволяющий обращаться к сборщику через командную строку:

npm install webpack-cli --save-dev

Установите Webpack и Webpack CLI.

Убедитесь, что Webpack установлен, выполнив команду на проверку версии.
