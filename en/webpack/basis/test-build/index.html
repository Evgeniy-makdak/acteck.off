Test build via Webpack

Let's make a test build. To do this, let's create a test file with an entry point:

alert('it works');

In the terminal, while in the root folder of your site, run the build:

npm run build

As a result, in the folderdistthe file should appearmain.jscontaining the assembly. Webpack gives this name to the bundle by default, since we did not specify our name.

Perform a test build. Check that the collected file appears.
