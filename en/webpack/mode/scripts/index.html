Build modes via scripts in Webpack

You can set the build mode not in the configuration file, but through the settingsscriptsfilepackage.json. Now in this file we have the commandbuild, which builds the project in the mode specified in the configuration file:

"scripts": { "build": "webpack" }

Let's add a command that builds the project in development mode:

"dev": "webpack --mode development"

Let's also add a command that builds the project in production mode:

"prod": "webpack --mode production"

As a result, our filepackage.jsonwill look like this:

"scripts": { "build": "webpack", "dev": "webpack --mode development", "prod": "webpack --mode production", }

Now in the terminal we can use various commands to call the assembly in different modes. To build in development mode we will write the following:

npm run dev

And for building in production mode - the following:

npm run prod

Add to yourpackage.jsonappropriate settings.

Build your project in development mode.
