Webpack build file names and hashes

You can simultaneously specify names and hashes for assembly files:

export default { entry: { test1: './src/test1.js', test2: './src/test2.js' }, output: { filename: '[name].[contenthash].js', } };

Make three entry points. Make sure that they are collected into three files with names and a hash.
