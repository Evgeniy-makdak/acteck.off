Webpack entry and exit points

In this section we will study the settings of entry and exit points. For now we only have one settingentry, which specifies the entry point:

export default { entry: './src/index.js', };

Since the exit point is not specified, by default Webpack will collect everything into one filemain.js, located in the folderdist. In the following lessons we will learn how to set various entry and exit point settings.
