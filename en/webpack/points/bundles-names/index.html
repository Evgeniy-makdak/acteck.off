Webpack Build File Names

Using the command[name]you can specify the location where the entry point name will be inserted. This feature allows you to configure the names of the assembly files. For example, let's make the build files namedf1.build.jsAndf2.build.js:

export default { entry: { f1: './src/test1.js', f2: './src/test2.js' }, output: { filename: '[name].build.js', } };

The following settings are given:

export default { entry: { test1: './src/test1.js', test2: './src/test2.js', test3: './src/test3.js' }, };

Modify the settings so that the resulting files have namestest1.bund.js, test2.bund.jsAndtest3.bund.js.
