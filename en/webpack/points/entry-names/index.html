Webpack Entry Point Names

Entry points can be given names. For this settingentrymake it into an object. The object keys will be the names of the entry points. Named entry points will be compiled into separate build files.

Let's try it. In the following example, the result after building in the folderdistwe will have two files:file1.jsAndfile2.js.

export default { entry: { file1: './src/test1.js', file2: './src/test2.js' }, };

Make three entry points. Collect them into separate files.
