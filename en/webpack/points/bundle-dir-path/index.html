Path to build folder in Webpack

You can specify an absolute path to the build folder. Typically this is done to ensure that Webpack's advanced features work properly. Let's set this path.

First, let's import the modulepath:

import path from 'path';

Now let's specify the path to the assembly folder using the settingspath:

export default { entry: [ './src/test1.js', './src/test2.js' ], output: { filename: 'build.js', path: path.resolve('dist'), } };
