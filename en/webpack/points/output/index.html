Webpack exit point

The output point is adjustable by settingoutput. This setting represents an object. In this setting, you can specify the name of the assembly file using the keyfilename:

export default { entry: './src/index.js', output: { filename: 'build.js', } };

As a result, the assembly will still be located in the folderdist, but already in the file we specified.

Have your assembly compiled into a filebundle.js.
