Multiple entry points into Webpack

You can make several entry points. For this settingentryshould be made into an array:

export default { entry: [ './src/test1.js', './src/test2.js' ], };

All entry points will be collected into one common build file. If necessary, you can name this file:

export default { entry: [ './src/test1.js', './src/test2.js' ], output: { filename: 'build.js', } };

Make three entry points in your project. Let several modules from other files be connected to each point.

Build the project into a filebundle.js.
