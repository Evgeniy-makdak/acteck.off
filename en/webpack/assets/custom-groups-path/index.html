Customizing the path to asset groups in Webpack

You can set the saving path for each group of assets. This is done using the settinggeneratorspecified for each group:

export default { entry: './src/index.js', output: { filename: 'bundle.js', path: path.resolve('dist'), }, module: { rules: [ { test: /\.png$/, type: 'asset/resource', generator: { filename: 'images/[name].[hash][ext]' } } ] }, plugins: [ new HtmlWebpackPlugin(), ], };

Add all the pictures with the extension.pngto a folderimages/png.

Add all the pictures with the extension.jpgto a folderimages/jpg.

Add all files with the extension.zipto a folderdownload.

Add all files with the extension.zipor.rarto a folderdownload.
