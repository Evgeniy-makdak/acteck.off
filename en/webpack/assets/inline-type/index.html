Inline assets in Webpack

Let's now try inline assets. With this setting, the images will be converted to base64. Let's try:

export default { entry: './src/index.js', output: { filename: 'bundle.js', path: path.resolve('dist'), }, module: { rules: [ { test: /\.png$/, type: 'asset/inline' } ] }, plugins: [ new HtmlWebpackPlugin(), ], };

Make the pictures.svgbecame inline assets. Check your settings using an example image.
