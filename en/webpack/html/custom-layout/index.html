Custom layout file in Webpack

In practice, we usually already have our own HTML website layout. In this case, we need Webpack to take our layout rather than generate its own. Let's see how it's done.

Let's say we have the following layout:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>My layout</title> </head> <body> </body> </html>

Let's force Webpack to take our layout. To do this, we will register it in the following setting:

export default { entry: './src/index.js', plugins: [ new HtmlWebpackPlugin({ template: './src/index.html', }), ], };

As a result, after assembly in the folderdistthere will be our layout and all bundles will be automatically connected in it:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>My layout</title> <script defer src="main.js"></script> </head> <body> </body> </html>

Make your own layout. Configure Webpack to take your layout.
