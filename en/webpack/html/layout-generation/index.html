Generating HTML Layout in Webpack

In the previous lesson we installed and connected HtmlWebpackPlugin. Now, during assembly, an HTML layout file will be automatically generated and scripts and styles will be automatically included in this layout.

Let's try the simplest case. Let us have one entry point:

export default { entry: './src/index.js', plugins: [ new HtmlWebpackPlugin(), ], };

As a result, after assembly, our JavaScript will be assembled into a bundle with the namemain.jsand this bundle will be connected to the automatically created site layout:

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Webpack App</title> <script defer src="main.js"></script> </head> <body> </body> </html>

Build with HtmlWebpackPlugin enabled. Check that you have a layout file. Study its source code. Open the file in your browser.
