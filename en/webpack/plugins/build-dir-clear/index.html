Cleaning up the build folder in Webpack

By default, Webpack places new files in the build folder without deleting previous ones. It's a little inconvenient. The following plugin will help you clean up the folder:clean-webpack-plugin.

Let's install it:

npm install clean-webpack-plugin --save-dev

We import:

import {CleanWebpackPlugin} from 'clean-webpack-plugin';

Let's include:

export default { entry: { test1: './src/test1.js', test2: './src/test2.js', test3: './src/test3.js', }, output: { filename: '[name].[contenthash].js', path: path.resolve('dist'), // обязательно для работы плагина }, plugins: [ new CleanWebpackPlugin(), ], };

Install this plugin and check its operation.
