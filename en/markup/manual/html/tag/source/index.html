Source tag

Tagsourceinserts an audio or video file for tagsaudioAndvideo. This tag is needed in order to set paths to audio or video recording files in various formats (since browsers support different audio formats, for examplemp3orwav). For this purpose in the tagaudioorvideothere should be several tagssourcewith files in different formats. The browser will iterate through these tags in order until it finds a file in a format that it can play.

The closing tag is optional.

Attributes

Attribute Description
src Specifies the path to the audio or video file.
Required attribute.
media Determines the device for which the audio or video file will be played.
type File type and codec names for opening it. Example:video/ogg; codecs="theora, vorbis.

Example.

Let's go for the tagaudioadd a tagsource:

<audio controls> <source src="track.mp3"> Ваш браузер не поддерживает audio на HTML5. </audio>

Example.

Let's go for the tagvideoadd a tagsource:

<video> <source src="moovie.mp4" controls> Ваш браузер не поддерживает video на HTML5. </video>
