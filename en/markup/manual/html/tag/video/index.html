Video tag

Tagvideoallows you to add a video to an HTML page. It can also be used to control this video. The path to the video file is specified via the attributesrcor nested tagsource.

If you want to see a player to control video playback, you should add the attributecontrols. Without it, you won't see anything - there will be no sound or image either.

Sometimes posting a video on the site itself is not very convenient (for example, your hosting has a limit on the size of the site). In this case, the video can be uploaded to YouTube and connected using the tagiframe.

Attributes

Attribute Description
src Specifies the path to the video file. But it is better to use a tag for thissource(in this case it will be possible to set the video in different formats).
preload Used to download a video file simultaneously with loading a website page.
Accepted values:none(do not download file, default),metadata(upload only service information: duration of sound and so on),auto(download the entire video when loading the HTML page).
autoplay The sound starts playing immediately after the page loads, and the presence of the attributepreloadwill be ignored.
Attribute without value.
controls Adds a control panel to the video.
Attribute without value. By default (if there is no attribute), the panel is not added.
loop Repeats the audio recording in a “circle”: after completion it will start playing again.
Attribute without value. By default (if there is no attribute), the recording will only be played1once.
poster Path to the image that will be displayed while the video is not available or playing. If attributeposteris not specified, the browser will try to display the first frame of the video.
height Attributeheightsets the height of the video playback area (in pixels or percentage). The video itself changes its size up or down, adjusting to the given height, but its proportions do not change.
Default value: height is taken from video parameters, if this value is not available then the attribute valueheightis considered equal to the height of the image from the attributeposter. If andposternot specified - height is set to150pixels.
width Attributewidthsets the width of the video playback area (in pixels or percentage). The video itself changes its size up or down, adjusting to the given width, but its proportions do not change.
Default value: width is taken from video parameters, if this value is not available then the attribute valuewidthis considered equal to the width of the image from the attributeposter. If andposternot specified - width is set to150pixels.

Example. Player on the website page

Let's add a video player to the page. For those browsers that do not support the tagvideoThere was a special note about this:

<video src="moovie.mp4" controls> Ваш браузер не поддерживает video на HTML5. </video>

Example.
