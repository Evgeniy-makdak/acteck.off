script tag

TagscriptconnectsJavaScriptto the HTML page (and also, much less frequently, other scripts).

You can connect a separate JavaScript file (in this case the attribute must be specifiedsrc) or write JavaScript directly on the HTML page. Essentiallyscriptbehaves like a taglinkand tagstyleat the same time, but not for CSS, but for JavaScript.

This tag must be placed inside a taghead, however this is not necessary - tagscriptcan also be placed inside a tagbodyand it will work.

Attributes and Properties

Attribute/property Description
src Path to the included file.
type The data type of the included file. For JavaScript you should usetext/javascript. This is not necessary in HTML5.
language Sets the script language. This attribute is deprecated and the attribute should be used insteadtype.
defer Attributedeferdelays script execution until the entire page has loaded completely.

Example

Let's connect a separate JavaScript file:

<script src="test.js"></script>

Example

And so we will write JavaScript code directly on the HTML page:

<script> let test = '123'; </script>
