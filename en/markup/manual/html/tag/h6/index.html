h6 tag

Tagh6creates a header6-th level. A heading is the name of a block, usually in large and bold font. Headingh1- this is the most important heading on the page; it should reflect its main idea.

Headings in HTML have different levels: from the first (h1) to sixth (h6). If you compare an HTML page with a book, then the titleh1can be considered the title of the entire book,h2- chapters,h3- paragraph in the chapter and so on. Accordingly, regular text lies inparagraphs.

This tag is bold and large by default. This behavior can be changed using CSS propertiesfont-weightAndfont-size.

Example

Let's see what headings at different levels and paragraphs look like in the browser:

<h1>Heading h1</h1> <h2>Heading h2</h2> <h3>Heading h3</h3> <h4>Heading h4</h4> <h5>Heading h5</h5> <h6>Heading h6</h6> <p>This is paragraph.</p> <p>This is paragraph.</p> <p>This is paragraph.</p>
