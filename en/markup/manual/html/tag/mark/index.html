Tag mark

TagmarkMarks text as selected. This text is no different from regular text, but its appearance can be changed using CSS.

Unlike tagsstrongAndem, which are intended to indicate the importance of the text, tagmarkis necessary in order to identify specific parts of the text that the user is currently working with and whichare important to him at the moment.

In Chrome and Firefox browsers, the background color of the text insidemarkhighlighted with a yellow background (this can be reversed using the CSS propertybackground-color).
