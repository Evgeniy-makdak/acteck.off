Тег footer

Тег footer задает верхнюю часть страницы в HTML. Используется в основной структуре страницы:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title></title> </head> <body> <div id="wrapper"> <header> ... </header> <main> ... </main> <aside> ... </aside> <footer> ... </footer> </div> </body> </html>

Смотрите также

  • тег header,
    который задает верхнюю часть сайта
  • тег main,
    который задает основную часть сайта
  • тег nav,
    который задает главное меню сайта сайта
  • тег aside,
    который задает побочную часть сайта
