code tag

To tagcodeIt is customary to include the program code of any languages ​​that you want to place in the text of your HTML page.

Please note that the tags insidecodewill work like tags. To prevent the browser from treating them as a command, you should replace all angle brackets with special HTML characters&lt;And&gt;or use a custom tagxmp.

Text placed inside a tagcode, the default will bemonospace(this means that the length of all letters will be the same). This behavior can be changed using CSS propertiesfont-family.
