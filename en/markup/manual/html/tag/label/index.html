Label tag

Taglabelspecifies a label for an input field of typeinput, textarea, selectand so on.

The label is bound to a specific input field using an attributefor. It should indicate the attribute valueidthe input field to which the label is attached. Also, the label is attached to an element if this element is placed inside the taglabel. In this case the attributeforno need to indicate. See examples for better understanding.

By clicking on a label, linked to itcheckboxesorradio buttonswill change their state from marked to unmarked and vice versa. In this case, the label is needed for convenience: small form elements are difficult to hit with the cursor, but a long label with text is much easier.

Type input text field bound to a labelinputAndtextareawill receive input focus. Focus is when the cursor blinks in the input field, in this case, if you write something on the keyboard, the text will fall into this field (see pseudo-classfocusfor advanced understanding of focus).

Labellabelcan be used to simulatecheckboxorradio buttons. This is necessary in order to make a checkbox or radio with your own design (which is prohibited in CSS, but with the help of clever tricks it is possible).

Example. Checkbox

Let's go to the checkboxcheckboxbind the label using the attributefor. Click on the label and you will see the checkbox state change from checked to unchecked and vice versa:

<input type="checkbox" id="checkbox"> <label for="checkbox">Я привязан к чекбоксу с id checkbox.</label>

Example. Text input field

Let's taginputbind the label using the attributefor. Clicking on the label will cause the input field to gain focus. In this case, the text contained in the field (add using the attributevalue) will become highlighted. If you start writing something on the keyboard, this text will be erased. You can get rid of the selection without deleting the text by clicking on the input field: the focus will not disappear, but the selection will disappear:

<input type="text" id="input1" value="какой-то текст"> <label for="input1">Я привязан к инпуту с id input1.</label>
