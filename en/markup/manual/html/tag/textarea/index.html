Textarea tag

Tagtextareacreates a multi-line input field for use in HTML forms.

Unlike the taginput, Vtextareayou can enter many lines of text, separating them with a keyEnter.

Tagtextareamust be inside a tagform. This is not necessary, it is necessary to send data to the server and then process it through PHP.

By default, the user can resize the tagtextareaby pulling the bottom right corner. This behavior can be changed using CSS propertiesresize.

Example. Just textarea

Let's see how the tag workstextarea:

<textarea></textarea>

:

Example. Default text

Let's try tegutextareaadd default text. Please note that, unlike the taginput, tagtextareaattribute not setvalue, and the default text is written between the opening and closing tags:

<textarea>какой-то текст</textarea>

:

Example. Parasitic spaces

Please note that if you space the opening and closing tags on different lines, then as a result, extra spaces will appear in this input field (how to deal with this: do not space them on different lines):

<textarea> какой-то текст </textarea>

:

Example. Tooltip in textarea

Now let's tagtextareaadd a hintplaceholder:

<textarea placeholder="i am a hint"></textarea>
