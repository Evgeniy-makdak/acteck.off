Тег figure

Тег figure создан для группировки HTML изображений и подписей под ними. Подписи для изображений делаются с помощью тега figcaption.

Пример

Давайте попробуем img вложим в тег figure, а подпись к ней сделаем с помощью тега figcaption:

<figure> <img src="monkey.jpg" alt=""> <figcaption>Рис 1. Картинка обезьянки.</figcaption> </figure>

Смотрите также

  • тег img,
    который задает изображение
  • свойство background-image,
    которое задает фоновую картинку
