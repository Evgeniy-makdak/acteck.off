figure tag

Tagfigurecreated for groupingHTML imagesand signatures under them. Captions for images are made using a tagfigcaption.

Example

Let's try imgput it in a tagfigure, and we’ll create a signature for it using the tagfigcaption:

<figure> <img src="monkey.jpg" alt=""> <figcaption>Рис 1. Картинка обезьянки.</figcaption> </figure>
