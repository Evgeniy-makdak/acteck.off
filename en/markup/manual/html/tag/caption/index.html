Caption tag

Tagcaptionis designed to create the header of the entire HTML table. This title is the text that appears before the table by default and describes its contents.

This tag can only be placed inside a tagtable, and immediately after the opening tag.

Example

Let's add a table titlecaption:

<table> <caption>this is the title of the entire table</caption> <tr> <th>Name</th> <th>Surn</th> <th>Salary</th> </tr> <tr> <td>name1</td> <td>surn1</td> <td>200$</td> </tr> <tr> <td>name2</td> <td>surn2</td> <td>1000$</td> </tr> </table> table, td, th { border: 1px solid black; }
