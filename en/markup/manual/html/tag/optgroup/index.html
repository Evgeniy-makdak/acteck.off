Тег optgroup

Тег optgroup группирует пункты option выпадающего списка select.

Группировка нужна для того, чтобы дать общий заголовок определенной группе пунктов списка. Этот заголовок задается в атрибуте label тега optgroup.

Атрибуты

Атрибут Описание
label Задает текст, который будет заголовком для группы пунктов списка.

Пример

Давайте с помощью тега optgroup сгруппируем пункты списка в две группы:

<select> <optgroup label="Country 1"> <option>City 1</option> <option>City 2</option> </optgroup> <optgroup label="Country 2"> <option>City 3</option> <option>City 4</option> </optgroup> </select>
