head tag

Tagheadcontains the service content of the page, which specifiesdocument encoding, includes CSS files on the page, connects JavaScript files to the pageand so on.

Most often insideheadthere are tagstitle, meta, script, link, style.

Example

Let's see what the structure of a simple HTML page looks like:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Это заголовок тайтл</title> </head> <body> Это основное содержимое страницы. </body> </html>

See also

  • tagdoctype,
    which contains the doctype of the page
  • taghtml,
    which is the main tag on the page
  • tagbody,
    which contains the main content
  • tagtitle,
    which contains the page title
  • tagmeta,
    which contains meta data
