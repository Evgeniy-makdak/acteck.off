title tag

Tagtitlesets the title of the entire page, which will be visible in the browser tab. This tag should be placed inside a taghead, which contains the service content of the page.

Example

Let's look at the structure of a simple HTML page:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Это заголовок тайтл</title> </head> <body> Это основное содержимое страницы. </body> </html>
