Tag rt

Tagrtadds an annotation above the text. The annotation in this case is a transcript of the text below it. The tag should be used inside a tagruby(see it for a more complete understanding). By default, the annotation appears smaller than the text itself. This can be fixed using CSS propertiesfont-size.

Example

Let's make an annotation above the text:
