sup tag

Tagsupdisplays text in superscript, that is, just above the line of normal text and in a smaller font size. This may be necessary to write down mathematical or chemical formulas, and so on.

Example

Let's use a superscript to write the formula for a quadratic equation:

Формула квадратного уравнения: ax<sup>2</sup> + bx + c = 0.
