Tag th

Tagthspecifies the header cell of the HTML table.

By default, the text in the header cell is centered and bold. This behavior can be changed using CSS propertiestext-alignAndfont-weight.

Example

Let's create a table with cellsthAndtd:

<table> <tr> <th>Name</th> <th>Surn</th> <th>Salary</th> </tr> <tr> <td>name1</td> <td>surn1</td> <td>200$</td> </tr> <tr> <td>name2</td> <td>surn2</td> <td>1000$</td> </tr> </table> table, td, th { border: 1px solid black; }
