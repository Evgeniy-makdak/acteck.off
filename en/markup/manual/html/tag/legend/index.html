Legend tag

Taglegendsets a label over a grouping of HTML fieldsformsmade using the tagfieldset.

If you assign a taglegendany CSS styles for positioning (float, positionetc.), then its contents will leave the frame and go to the right place (there will no longer be a gap in the frame).

Example

Let's break the form form into two logical blocks using two tagsfieldset, and we’ll make captions for the blocks using the taglegend:

<form> <fieldset> <legend>Данные покупателя</legend> <input type="text"> </fieldset> <fieldset> <legend>Данные продавца</legend> <input type="text"> </fieldset> </form>
