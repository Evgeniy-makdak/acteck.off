Тег a

Тег a создает ссылку на другую страницу своего или чужого сайта. Адрес ссылки следует писать в атрибуте href. Links по умолчанию подчеркнуты. Отменить подчеркивание можно с помощью CSS свойства text-decoration.

Атрибуты

Атрибут Описание
href Адрес страницы, на которую ссылается ссылка.
Обязательный атрибут.
target Указывает ссылке в какой вкладке ей открываться: в новой, в текущей или во фрейме.
Необязательный атрибут.

Значения атрибута target

Значение Описание
_blank Page откроется в новой вкладке браузера.
_self Page откроется в текущей вкладке браузера.
_parent Page откроется во фрейме-родителе.
_top Отменяет все фреймы и загружает страницу в полном окне браузера.

Значение по умолчанию: _self.

Пример

Давайте сделаем ссылку, которая будет вести home страницу сайта google.com. При этом она откроется в новой вкладке браузера, так как мы зададим ей атрибут target в значении _blank:

<a href="http://google.com" target="_blank">google.com</a>
