Radio switch

Radio switch is an HTML elementformsin the form of a round button, which may or may not be checked.

A radio button is created using a taginputwith attributetypein meaningradio. Appearance:

One radio switch is almost never used, they should be used in a group. In this case, only one of the radio buttons in the group can be checked. Radio buttons will only form a group if they have the same attribute valuename(see examples below).

Example

Let's make two radio switches: the first one will be marked (set the attributechecked), but the second one does not. In this case, the radio buttons will be a group in which only one of them can be checked (since they have the same attribute valuename).

Please note that these radio buttons are given different attribute valuesvalue. This is necessary so that after sending the data to the server, PHP can determine which of the radio buttons was selected:

<input type="radio" name="test" checked value="1"> <input type="radio" name="test" value="2">

:

Example

Now let's make two groups of radio switches: the first group will have one attribute valuename, and the second is different. Let's try to click on them, and we will see that the groups switch independently of each other:

<input type="radio" name="radio1" checked> <input type="radio" name="radio1"> <input type="radio" name="radio1"> <br> <input type="radio" name="radio2" checked> <input type="radio" name="radio2"> <input type="radio" name="radio2">
