checkbox

Checkboxcheckboxrepresents an element (check or bird) of HTMLforms, which can have two states: checked and unchecked.

A checkbox is created using a taginputwith attributetypein meaningcheckbox. Checkbox appearance:

The checked checkbox will send the contents of the attribute to the servervalue. If attributevaluethere is no checkbox - it will send a string"on". If the checkbox is not checked, nothing will be sent to the server, regardless of the presence of the attributevalue.

Example

Let's make two checkboxes: let the first one be checked (since it is given the attributechecked), but the second one does not:

<input type="checkbox" checked> <input type="checkbox">
