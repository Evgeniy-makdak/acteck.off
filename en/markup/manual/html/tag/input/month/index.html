Поле для ввода месяца

С помощью тега input с атрибутом type в значении month можно создать поле для ввода месяца.

Пример

<input type="month">

:

Смотрите также

  • поле date,
    с помощью которого можно ввести дату
  • поле datetime,
    с помощью которого можно ввести дату-время
  • поле week,
    с помощью которого можно ввести неделю
