progress tag

Tagprogressused to display the progress of a task's completion. Changing the value occurs through JavaScript. Inside the tag, specify the text that will be displayed if the browser does not understand this tag.

Attributes

Attribute Description
value Current progress value.
max Maximum progress value.

Attribute values ​​are integer or fractional numbers.

Example

<progress max="100" value="25"> Загружено на 25% </progress>

:

<progress max="100" value="75"> Загружено на 75% </progress>
