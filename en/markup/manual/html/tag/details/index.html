Details tag

Tagdetailscreates a spoiler in HTML, that is, it is used for information that can be hidden or shown at the request of the user.

By default, the content of the tag is hidden; to display it at the user's request, you must add the tag summary, which specifies the title for the tagdetails. This header is always visible on the page (even if the contentdetailsis in a collapsed state) and can be clicked to expand or collapse hidden text.

You can also use the attribute to show hidden contentopentagdetails(for example, to display contentdetailsusing JavaScript).

Example

Let's see how the tag worksdetails:

<details> <summary>click me</summary> hidden text </details>
