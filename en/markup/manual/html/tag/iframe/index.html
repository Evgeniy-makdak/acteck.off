iframe tag

Tagiframeallows you to embed content from one site into another. It is used to display content such as videos, images or articles without having to open a new window or tab.

The content of the inserted site will have the design of that site. In this case, the CSS styles of our site, as well as our JavaScript code, will not affect the content of the tagiframe.

Go to contentiframeYou can use our JavaScript, but for this you need to use special methods.

Tagiframewidely used on websites to embed videos from YouTube and other platforms, as well as display articles and images from other sources.

Attributes

Attribute Description
src Specifies the URL of the page that should be embedded.
srcdoc Allows you to set the contents of the iframe directly in the attribute.
frameborder Specifies the frame of the iframe.
Meaningyesspecifies the frame (default) and the valuenoremoves the frame.
scrolling Specifies scrollbars within an iframe.
Meaningauto- scroll as needed (default), valueyes- scrolling always, valueno- no scrolling.

Example

Let's insert an iframe with another site into our website (let's take code.mu as an example):

<iframe src="https://code.mu/" width="600" height="400"></iframe>

:

Example

Let's insert a video from YouTube onto our website:

<iframe src="https://www.youtube.com/embed/KzQrHpB1H8U" allowfullscreen width="400" height="400"> </iframe>

:

Example

Let's write the content directly in the iframe:

<iframe srcdoc="<h1>head</h1><p>text</p>" width="300" height="200"> </iframe>
