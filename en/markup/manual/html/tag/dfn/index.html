Тег dfn

Тег dfn задает встретившийся первый раз в документе термин (определение чего-либо нового).

По умолчанию браузеры показывают содержимое тега dfn курсивом, однако это можно поменять с помощью свойства font-style.

Пример

Давайте в текст введем новый термин и включим его в тег dfn, чтобы показать это:

<dfn>Shovel</dfn> - a tool for digging up soil.
