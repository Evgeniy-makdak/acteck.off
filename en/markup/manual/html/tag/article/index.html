Article tag

Tagarticleis intended for independent self-contained website blocks that can be used as a separate page.

For example, if you have a site with articles, then the article can be placed in this tag. If you have user comments, then each of these comments can be placed in this tag.

That being said, if you have a tagmainfor site content, thenarticlecan be placed insidemain, but not vice versa.

Example

Let's format the text of the article into a tagarticle:

<div id="wrapper"> <header> ... </header> <main> <article> <h1>Header</h1> <p> text </p> <p> text </p> </article> </main> <footer> ... </footer> </div>

Example

Let's format posts in tagsarticle:

<div id="wrapper"> <header> ... </header> <main> <h1>New posts</h1> <article> <h2>Post 1</h2> <p> text </p> <p> text </p> </article> <article> <h2>Post 2</h2> <p> text </p> <p> text </p> </article> </main> <footer> ... </footer> </div>
