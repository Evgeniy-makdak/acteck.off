Style tag

Tagstyleallows you to write CSS directly inside an HTML page. Please note that inside the tagstylewe write following the syntaxCSS language(selectors, curly braces, etc.).

This tag must be placed inside a taghead, however this is not necessary - tagstylecan also be placed inside a tagbodyand it will work.

The tag should not be abusedstyle- should most often be used insteadseparate CSS file.

Example

Let's write CSS inside HTML code using a tagstyle:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Это заголовок тайтл</title> <style> p { color: red; } </style> </head> <body> <h1>Это заголовок стандартного (черного) цвета.</h1> <p>Это абзац красного цвета.</p> </body> </html>
