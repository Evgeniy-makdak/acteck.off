Тег xmp

Внутри тега xmp все остальные теги будут отображаться как обычный текст.

Текст, размещенный внутри тега xmp, по умолчанию станет моноширинным (это значит, что длина всех букв будет одинакова). Это поведение можно изменить с помощью CSS свойства font-family.

К сожалению, данный тег считается не стандартным и валидатор будет на него ругаться.

Пример

Давайте тег b возьмем в тег xmp и отобразим в браузере (обратите внимание на то, что текст начнется с новой строки, это потому, что тег xmp блочный):

Next comes <xmp><b>bold</b></xmp> text.
