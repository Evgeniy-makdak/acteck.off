button tag

Tagbuttoncreates a button for use in HTMLforms, and also simply as a design element (in this case it is often included inlink).

Attributes

Attribute Description
type Sets the button type.
Optional attribute. The default value isbutton.

Type attribute values

Meaning Description
button Normal button (default value).
submit Creates a button that submits form data to the server.
reset Creates a button that clears a completed form.

Example

Let's create a button using a tagbutton:

<button>click me</button>
