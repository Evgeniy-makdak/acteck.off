Тег button

Тег button создает кнопку для использования в HTML формах, а также просто как элемент дизайна (в этом случае ее часто вкладывают в ссылку).

Атрибуты

Атрибут Описание
type Задает тип кнопки.
Необязательный атрибут. По умолчанию имеет значение button.

Значения атрибута type

Значение Описание
button Обычная кнопка (значение по умолчанию).
submit Создает кнопку, которая отправляет данные формы на сервер.
reset Создает кнопку, которая очищает заполненную форму.

Пример

Давайте создадим кнопку с помощью тега button:

<button>click me</button>
