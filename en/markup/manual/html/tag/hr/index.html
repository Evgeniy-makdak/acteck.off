Tag hr

Taghrspecifies a horizontal dividing line. The advantage of the tag is that in order to draw a single line, you do not need to create extra blocks. Does not require a closing tag.

Example

Let's see how the tag workshr:

<hr>

:

Example

Let's try changing the line color using a CSS propertyborder-color:

<hr style="border-color: red;">

:

Example

The line color can also be changed using the propertycolor(at the same time, if the propertiescolorAndborder-color- the second has priority):

<hr style="color: red">

:

Example

Let's now change the line color using a CSS propertyborder. Please note that in this case the line will become double thick (since we set the border both above and below):

<hr style="border: 1px solid red;">

:

Example

Let's try to change the line color using CSS propertiesborder-top, and set only the upper bound. Now there will be no double line:

<hr style="border-top: 1px solid red;">

:

Example

Let's make a dotted line using CSS propertiesborder-top, set the valuedottedinstead ofsolid:

<hr style="border-top: 1px dotted red;">

:

Example

Let's increase the line thickness using a CSS propertyborder-width:

<hr style="border-width: 10px; border-color: red;">

:

Example

Now let's add height to the linesheightand the border acrossborder. The boundary will then split into upper and lower:

<hr style="height: 10px; border: 1px solid red;">

:
