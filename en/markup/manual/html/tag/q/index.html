Tag q

Tagqintended for highlighting short quotes in the text. In attributecitea link to the source of the quotation should be provided.

Default tag contentqappears in quotation marks. The type of quotation marks depends on the text language specified by the attributelang(if there is no attribute, the browser will detect the language automatically). The type of quotes can also be changed using CSS propertiesquotes.

Tagqcan be used in conjunction with a tagcite, which is needed to indicatelinks to source quotes.

Attributes

Attribute Description
cite Link to the source of the quote (if necessary, if such a source exists).

Example

Let's tag the quoteqand display it in quotes:

<q>Все хорошо, что хорошо кончается</q>, сказал он.
