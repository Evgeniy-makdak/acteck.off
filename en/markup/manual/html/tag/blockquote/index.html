blockquote tag

Tagblockquoteintended for highlighting long quotations within a document. The tag can be used in conjunction with the tagcite, which is needed to indicatelinks to source quotes.

By default, the text taken into the tagblockquotehas indentation on the left. It can be removed using CSS propertiesmargin.

Attributes

Attribute Description
cite Link to the source of the quote (if necessary, if such a source exists).

Example

Example of using a tag:

<p> Обычный текст. </p> <blockquote> Длинная длинная цитата. </blockquote> <p> Обычный текст. </p>
