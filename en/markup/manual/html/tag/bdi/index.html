bdi tag

Tagbdispecifies a piece of text that should be isolated from changes in text direction. This behavior is important for texts that simultaneously contain different languages, read from left to right and from right to left.

The direction of the text can be changed using a tagbdo, attributediror CSS propertiesunicode-bidiAnddirection.

Example

Let's change the direction of the text:

<bdo dir="rtl">This text <bdi dir="ltl">except for this piece</bdi> has a direction from right to left.</bdo>
