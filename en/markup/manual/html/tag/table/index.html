Table tag

Tagtablecreates a table. Used in conjunction with tagstr, td, th.

Tagtrcreates a table row and tagstdAndthcreate the cells of this table. Difference betweentdAndth: The first one is a normal cell and the second one is a header cell.

Attributes

Attribute Description
cellspacing Sets the distance between cells.
Deprecated, CSS property should be used insteadborder-spacing.
cellpadding Specifies the distance between text and the cell border.
Deprecated, CSS property should be used insteadpadding.
border Sets the border of cells and tables.
Deprecated, CSS property should be used insteadborder.

Example

Let's create a table with cellsthAndtd:

<table> <tr> <th>Name</th> <th>Surn</th> <th>Salary</th> </tr> <tr> <td>name1</td> <td>surn1</td> <td>200$</td> </tr> <tr> <td>name2</td> <td>surn2</td> <td>1000$</td> </tr> </table> table, td, th { border: 1px solid black; }

:
