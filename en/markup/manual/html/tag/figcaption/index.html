figcaption tag

Tagfigcaptionspecifies a description of the picture or pictures for the tagfigure. This tag must be the first or last element insidefigure.

Example

Give me a picture imgput it in a tagfigure, and we’ll create a signature for it using the tagfigcaption:

<figure> <img src="monkey.jpg" alt=""> <figcaption>Картинка обезьянки.</figcaption> </figure>

    which specifies the image

    which sets the background image
