img tag

Tagimgcreates a picture. The path to the image is specified in the attributesrc. Does not require a closing tag.

Attributes

Attribute Description
src Sets the path to the image.
Required attribute.
alt Alternative text that will be shown instead of the image if it is not found (for example, the path to it is incorrectly specified).
Required attribute. If it is missing, the validator (a program that checks the correctness of HTML or CSS) will complain.
width The width of the image, in pixels (in this case the units of measurement are not specified) or as a percentage of the image’s parent.
height The height of the image, in pixels (in this case the units of measurement are not specified) or as a percentage of the image’s parent.

Nuances

If neither width nor height is specified for the image, the image will have its actual size. If the height is specified, the picture will become the specified height, and the width will be adjusted so that its proportions are not distorted.

If only the width is specified, similarly, the image will be adjusted in height so as to maintain the proportions.

If both width and height are specified, the proportions of the picture may be distorted (or maybe not, depending on your guess). If the width or height (or both together) is larger than the real one, the picture will increase, but will lose quality.

It is recommended to set the width and height of images in attributes (rather than via CSS) - in this case, the browser will load images faster - it does not need to calculate the size of each image after receiving it.

It is not recommended to reduce the actual size of the image unless necessary. For example, the actual size of the picture1000on1000pixels, and you give it a width of100px. In this case, the picture on the screen will look like100pixels, but have a size of a whole thousand and, accordingly, take much longer to load.

Example

Let's add a picture to the site and not set attributesheightAndwidth. The picture will have its actual size:

<img src="monkey.png" alt="обезьянка">

:

Example

Let's try to add width to the image using the attributewidth, the height should be adjusted so as to maintain the proportions of the picture:

<img src="monkey.png" width="200" alt="обезьянка">

:

Example

Now let's add height to the image using the attributeheight, the width will be adjusted so as to maintain the proportions of the image:

<img src="monkey.png" height="100" alt="обезьянка">

:

Example

Let's add both height and width to the picture at the same time. The proportions of the picture should become distorted (not necessarily, but in this case the height and width are chosen so that the proportions are distorted):

<img src="monkey.png" height="100" width="300" alt="обезьянка">

:

Example

Let's put the wrong path to the image (for simplicity, we'll leave it empty). Instead of a picture, we will see the contents of the attributealt(it seems that this is ordinary text - but try to copy it - you won’t succeed, it will stretch like a picture):

<img src="" alt="обезьянка">
