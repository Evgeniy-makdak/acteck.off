doctype tag

Tagdoctypetells the browser the HTML version. To indicate the current, fifth version of HTML, write this:

<!DOCTYPE html>

Before the release of HTML5, you had to write terrible things that were completely impossible to remember by heart. For example, these:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

Fortunately, the doctype has now been shortened and such long structures are no longer needed. There is no need not only to remember them, but even to understand what they did.

Tagdoctypewritten at the very beginning of the document, before the taghtml. The tag is required, but the page will work without it, but in this case some things will work differently than if it was present.doctype. Therefore, always write it - you will avoid some problems.

Example

Let's look at the structure of a simple html page:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Это заголовок тайтл</title> </head> <body> Это основное содержимое страницы. </body> </html>
