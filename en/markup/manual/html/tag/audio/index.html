Audio tag

Tagaudiocontrols the playback of audio recordings on an HTML page. When you add it, a player appears on the page, which will have buttons “Back”, “Forward”, “Stop” and so on (their type and number depends on the browser). The path to the file is specified via the attributesrcor nested tagsource.

Tagsourceneeded in order to set paths to audio files in different formats (since browsers support different audio formats, for examplemp3orwav). If you have a recording in one of the formats, then you need to convert it to others.

Inside a tagaudioYou can write text that will be displayed in browsers that do not support HTML audio (this is to inform users about this).

If you want to see a player to control audio playback, you should add the attributecontrols. Without it, you won't see anything - and there will be no sound either. To have sound when there is no player, add an attributeautoplay.

Attributes

Attribute Description
src Specifies the path to the audio file. But it is better to use a tag for thissource(in this case it will be possible to set audio in different formats).
preload Used to download an audio file simultaneously with loading a website page.
Accepted values:none(do not download file, default),metadata(upload only service information: duration of sound and so on),auto(download the entire audio file when the HTML page loads).
autoplay The sound starts playing immediately after the page loads, and the presence of the attributepreloadwill be ignored.
Attribute without value.
controls Adds a control panel to an audio file. while the presence of the attributepreloadwill be ignored.
Attribute without value.
loop Repeats the audio recording in a “circle”: after completion it will start playing again. while the presence of the attributepreloadwill be ignored.
Attribute without value. By default (if there is no attribute), the recording will only be played1once.
muted Mute audio recordings.
Attribute without value. By default (if there is no attribute), the recording has sound turned on.

Example. Player on the website page

Let's add a player with audio recording to the page. For those browsers that do not support the tagaudioThere was a special note about this:

<audio src="track.mp3" controls> Ваш браузер не поддерживает audio на HTML5. </audio>

Example. Player on a website page with the source tag

Now let's replace the attributesrcfor tagaudioadd a tagsource:

<audio controls> <source src="track.mp3"> Ваш браузер не поддерживает audio на HTML5. </audio>
