Tag wbr

Tagwbrspecifies a place where the browser can make a line break if necessary (if the text does not fit into the width of the element). Such transfers are called soft.

Does not require a closing tag.

When wrapping a word through a tagwbrThe hyphen "-" is not added. If you need it, use the soft hyphen symbol&shy;(the semicolon at the end is required, this is not a typo).

Soft transfer&shy;tells the browser where it can hyphenate the word (if necessary) by adding a hyphen "-".

Soft transfers&shy;will not work if the propertyhyphensgiven in valuenone(and hyphenswbrwill be).

Example. Text without hyphens

Let's look at a sample text that has long words that extend beyond the block boundary. We will set the block to a small width so that a large word will not fit in it:

<div id="elem"> this is supersupersuperlong text </div> #elem { width: 200px; border: 1px solid black; }

:

Example. Tag wbr

And here, let's add a tagwbrin places where we recommend that the browser hyphenate words if necessary (note that the browser will not hyphenate everywhere we specify):

<div id="elem"> this is super<wbr>super<wbr>super<wbr>long text </div> #elem { width: 200px; border: 1px solid black; }

:

Example. The symbol ­

Let's try instead of tagwbrput a symbol&shy;. Hyphens will appear at hyphens:

<div id="elem"> this is super­super­super­long text </div> #elem { width: 200px; border: 1px solid black; }

:
