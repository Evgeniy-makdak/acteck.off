Time tag

TagtimeMarks text as a date, time, or both date and time. The date can be specified inside the tag itself or through an attributedatetime.

The meaning of this tag is semantic markup. The search engine sees not neutralspan, but data that it can identify as date or time.

Also, the meaning of this tag is that inside it you can insert a date or time (or both) in any format convenient for you, and in the attributedatetimeindicate the same date in a form convenient for search engines. For example:

<time datetime="2015-03-12">Прошлый вторник</time>

:

Attributes

Attribute Description
datetime Specifies the date, time, or both for text. Date and time are specified in international ISO format8601(year-month-day hour:minute:second, see format for details).
