Datalist tag

Tagdatalistcreates autocomplete for input fieldsinputin HTMLform.

The essence of autocomplete: when the user tries to type something in the input field, a tooltip will appear below, which will be a list of available selection options. The user will be able to select one of the proposed options without entering it completely.

To bind to a taginputautocomplete, it needs to be given an attributelist, in which the attribute should be specifiedidanchor tagdatalist.

Choices are stored in tagsoption, which in turn are stored in the tagdatalist.

Example

Let's enter the country in the input field. Let's enter the letter first'b'- and we will see a tooltip from three countries. Then we also enter the letter'e'- the number of proposed countries will be reduced to two:

<input type="text" list="country"> <datalist id="country"> <option>Belarus</option> <option>Belgium</option> <option>Bulgaria</option> </datalist>

:

Example

Let's go to the tag nowoptionadd an attributevalue. When selecting a specific country, the linked input will contain not the name of the country, but the contents of the attributevalue:

<input type="text" list="country-value"> <datalist id="country-value"> <option value="Belarus">Belarus - a country of lakes and potatoes</option> <option value="Belgium">Belgium - a country where Belgians live</option> <option value="Bulgaria">Bulgaria - a country for vacation</option> </datalist>

:

Example. Impact of the autocomplete attribute

The tooltip dropdown list is affected by the attributeautocomplete. If it is enabled (and it will be by default), then to the values ​​that you specified in the tagdatalist, the values ​​that the user previously entered into this field will be mixed in.

Drive into the field'Brazil'and click the send button (without sending, the browser will not remember this country). Then go back to this example and enter the letter'b'- in the list that appears you will see not only3countries fromdatalist, but also the previously entered country (in addition, if you do the same with the word'Belarus'- in the drop-down list this value will appear twice).

If this bothers you, disable the attributeautocomplete, adding a value to itoff.

Perform the manipulations described above with the example:

<form action=""> <input type="text" list="country" autocomplete="on"> <datalist id="country"> <option>Belarus</option> <option>Belgium</option> <option>Bulgaria</option> <input type="submit"> </datalist> </form>
