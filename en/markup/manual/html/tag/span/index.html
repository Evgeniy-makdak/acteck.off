span tag

TagspanAllows you to select a piece of text. The tag itself doesn't do anything, but when combined with attributesclass, idorstyleallows you to access the text that is inside it via CSS (for example, in order to changetext color, its sizeand so on).

Let's highlight a piece of text using a tagspan, but let's leave it the same as regular text (since this span has no CSS styles assigned):
