Tag ol

Tagolcreates the so-calledordered listwith markers in the form of numbers.

Tagolshould be used in conjunction withtag li, which creates list items (aolcreates the list itself, see example below). Without taglitagolnot used.

Tag advantageolis that it automatically arranges the numbering. If you want to insert a couple of items in the middle of the list, the numbering will adjust to suit you.

You can change or completely remove list markers using CSS propertieslist-style-type.

Example. List ol

Let's see what the list looks likeolin default browser:

<ol> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ol>
