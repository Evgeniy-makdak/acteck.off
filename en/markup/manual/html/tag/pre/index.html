Tag pre

Tagprespecifies the output of text as it is typed in the document: with all indents (tabs), spaces and line breaks.

Text placed inside a tagpre, will becomemonospace(this means that the length of all letters will be the same). For more information about monospaced text, read the propertyfont-familyin meaningmonospace.
