Тег bdo

Тег bdo устанавливает направление текста: слева направо (как русский, английский и многие другие языки) или справа налево (как арабский язык или иврит).

По умолчанию текст, взятый в данный тег имеет "нормальное" направление слева направо, однако это поведение можно поменять с помощью атрибута dir.

Направление текста также можно сменить с помощью CSS свойств unicode-bidi и direction или атрибута dir для любого тега.

Атрибуты

Атрибут Описание
dir Задает направление текста. Значение ltr задает направление слева направо, а значение rtl - справа налево.

Пример

Давайте изменим направление русского текста:

<bdo dir="rtl"> Этот текст имеет направление справа налево. </bdo>
