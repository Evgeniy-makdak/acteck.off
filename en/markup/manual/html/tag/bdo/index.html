bdo tag

Tagbdosets the direction of the text: left to right (like Russian, English and many other languages) or right to left (like Arabic or Hebrew).

By default, the text taken in this tag has a "normal" direction from left to right, but this behavior can be changed using the attributedir.

The text direction can also be changed using CSS propertiesunicode-bidiAnddirectionor attributedirfor any tag.

Attributes

Attribute Description
dir Sets the direction of the text. Meaningltrsets the directionfrom left to right, and the valuertl - from right to left.

Example

Let's change the direction of the Russian text:

<bdo dir="rtl"> Этот текст имеет направление справа налево. </bdo>
