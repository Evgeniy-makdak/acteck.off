Link tag

Taglinkconnects CSS files to the HTML page. Besides,linkincludes some other files, for example, favicon. (A favicon is a website shortcut that is visible in a browser tab. It can also be seen in some search engines when searching opposite sites, for example in Yandex).

This tag has a lot of attributes and they can take on a variety of values, but in real life they are practically not used. The most popular is connecting CSS and adding a favicon.

How to include CSS:

<link rel="stylesheet" href="style.css">

How to connect a favicon:

<link href="favicon.ico" rel="shortcut icon" type="image/x-icon">

Taglinkdoes not require a closing tag.

Attributes

Attribute Description
href Path to the included file.
media The type of device for which the file should be attached.
What this means is that you can include a CSS file only for large screens (meaningscreen) or only for small screens: for mobile phones or tablets (meaninghandheld).
Possible values:all, braille, handheld, print, screen, speech, projection, tty, tv. See below for more details.
rel The type of the included file.
Possible values:stylesheet | alternate. Meaningstylesheetindicates that a CSS file is being included, the valuealternateused, for example, to indicate a link to a file in XML format to describe a news feed or article announcements.
charset Encoding of the included file. Currently the standard isutf-8.
type The data type of the included file.

Media attribute values

In HTML5, media queries can be specified as values.

Meaning Description
all All devices.
screen Monitor screen.
handheld Phones, smartphones, devices with small screens.
braille Devices based on the Braille system are intended for blind people.
speech Speech synthesizers, as well as programs for playing text out loud. This also includes speech browsers.
print Printers.
projection Projectors.
tty Teletypewriters, terminals, portable devices with limited screen capabilities. They should not use pixels as units of measurement.
tty TVs that can open web pages (this also happens).

Default value:all.

Example

Let's see what the structure of a simple HTML page looks like, to which we will add the connection of CSS files and favicons:

<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="style1.css"> <link rel="stylesheet" href="style2.css" media="screen"> <link href="favicon.ico" rel="shortcut icon" type="image/x-icon"> <meta charset="utf-8"> <title></title> </head> <body> </body> </html>
