select tag

Tagselectcreates a dropdown list for use in HTMLforms.

A single list item should be stored in a tagoption.

Attributes

Attribute Description
multiple The presence of this attribute createsmultiselect- a drop-down list in which you can select several items by holding down the keyCtrlor by selecting them with the mouse.
Optional attribute.
name The name of the dropdown list. Needed to get input field data in PHP. For the form to work correctly, the names of the input fields should not coincide with each other (in the same form). If they match, PHP will receive the data from the input field that is below in the HTML code.

Example

Let's see how the dropdown list works:

<select> <option>City1</option> <option>City2</option> <option>City3</option> <option>City4</option> </select>

:

Example

Let's make the width of the dropdown list equal to the width of the largest element (unless it is explicitly specified using the CSS propertywidth):

<select> <option>Big City1</option> <option>City2</option> <option>City3</option> <option>City4</option> </select>

:

Example. Default selection

Now let's try to select the default city. Let's do this using the attributeselected:

<select> <option>Big City1</option> <option selected>City2</option> <option>City3</option> <option>City4</option> </select>

:

Example. Multiselect

Let's now turn a regular select into a multi-select using the attributemultiple:

<select multiple name="city[]"> <option>City1</option> <option>City2</option> <option>City3</option> <option>City4</option> </select>

:

Please note that the select name specified in the attributename, must have square brackets at the end. This is necessary so that PHP receives all selected values ​​(otherwise only one will arrive - the last one).

Example. Multiple default values ​​in multiselect

Now, in the default multiselect, let's try to select more than one value:

<select multiple name="city[]"> <option>City1</option> <option selected>City2</option> <option>City3</option> <option selected>City4</option> </select>

:
