div tag

Tagdivdefines a block that combines other tags. The tag itself doesn't do anything, but when combined with attributesclassAndidallows you to access a group of combined tags through CSS.

Example

Let's use the tagdivlet's group several paragraphs and apply some styles to the group:

<div id="elem"> <p>text</p> <p>text</p> <p>text</p> </div> #elem { color: red; padding: 10px; border: 1px solid black; }

:

See also

  • tagspan,
    which allows you to select a piece of text
  • tagsection,
    which also groups tags
