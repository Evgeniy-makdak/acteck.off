ul tag

Tagulcreates the so-calleddisorderedlist with markers in the form of dots.

Tagulshould be used in conjunction with the tagli, which creates list items (aulcreates the list itself, see example below). Without taglitagulnot used.

You can change or completely remove list markers using CSS propertieslist-style-type.

Example

Let's look at an example of what a list looks likeul:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul>
