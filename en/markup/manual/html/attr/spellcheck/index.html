spellcheck attribute

Attributespellchecktells the browser whether it should check the spelling of entered text. Applies to tagsinputortextarea, as well as to elements that have the attribute specifiedcontenteditable.

Meaningtrueincludes checking andfalse- turns off. Instead oftrueYou can specify an empty value or not write it at all.

Example

Let's look at the text that contains errors and turn on the spell check. If your browser supports the attributespellcheckand spell checking is enabled, thenupon receiving focus(!) element, the browser should highlight incorrectly spelled words:

<textarea lang="en" spellcheck>This text has eror</textarea> textarea { width: 200px; height: 100px; }
