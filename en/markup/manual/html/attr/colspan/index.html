colspan attribute

Attributecolspancontrols the concatenation of HTML columnstables. Applies to tagsthAndtd.

Accepted value: integer starting from1.

How to use: add this attribute to any celltdorth. If, for example, you setcolspanin meaning2- then the cell to which this is specified will occupy two columns of the table. In this case, the neighboring cells on the right will not go anywhere, our expanded cell will displace them and the table will fall apart. To prevent this from happening, you need to delete one of the cells on the right. See examples for better understanding.

Example. Table without joins

Let's look at the table without joins that we will work with next:

<table> <tr> <td>cell1</td> <td>cell2</td> <td>cell3</td> </tr> <tr> <td>cell4</td> <td>cell5</td> <td>cell6</td> </tr> <tr> <td>cell7</td> <td>cell8</td> <td>cell9</td> </tr> </table> table, td, th { border: 1px solid black; }

:

Example. Expand the cell by two columns

Let's try to expand the cell1into two columns by giving it the attributecolspanin meaning2. At the same time, it will displace the cells on the right and the table will fall apart:

<table> <tr> <td colspan="2">cell1</td> <td>cell2</td> <td>cell3</td> </tr> <tr> <td>cell4</td> <td>cell5</td> <td>cell6</td> </tr> <tr> <td>cell7</td> <td>cell8</td> <td>cell9</td> </tr> </table> table, td, th { border: 1px solid black; }

:

Example. Let's expand the cell without collapsing the table

Now let’s try to fix the problem with the table falling apart from the previous example, let’s delete one of the cells to the right of ours (this is the cell2or3- doesn't matter):

<table> <tr> <td colspan="2">cell1</td> <td>cell2</td> </tr> <tr> <td>cell4</td> <td>cell5</td> <td>cell6</td> </tr> <tr> <td>cell7</td> <td>cell8</td> <td>cell9</td> </tr> </table> table, td, th { border: 1px solid black; }

:

Example. Expand the cell by three columns

Let's expand the cell1on3column and give it an attributecolspanin meaning3. At the same time, we will delete another cell on the right so that the table does not fall apart:

<table> <tr> <td colspan="3">cell1</td> </tr> <tr> <td>cell4</td> <td>cell5</td> <td>cell6</td> </tr> <tr> <td>cell7</td> <td>cell8</td> <td>cell9</td> </tr> </table> table, td, th { border: 1px solid black; }

Example. Simultaneously colspan and rowspan

Let's try to merge cells with number4, 5, 7And8, using the attributescolspanAndrowspan(we will remove the previous joins from the table):

<table> <tr> <td colspan="3">cell1</td> </tr> <tr> <td colspan="2" rowspan="2">cell4</td> <td>cell6</td> </tr> <tr> <td>cell9</td> </tr> </table> table, td, th { border: 1px solid black; }
