form attribute

Attributeformassociates a form element with a tagform.

This is necessary when the form element lies outside the tagform, but must be sent to the server along with the contents of this tagform.

For example, you have a buttonsubmit, but by design it is located in a completely different place on the page. In this case the attributeformwill solve the problem: set this attribute to the button, indicating the attribute value in itidfrom the desired tagform. After these manipulations, the button will submit the linked form.

You can bind not only buttons, but also any other form elements.

Example

In the following example, the button is outside the field, but is associated with it through an attributeform:

<input type="submit" form="test"> <form id="test"> <input type="text" name="age"> </form>
