width attribute

Attributewidthsets the width of the image that is specifiedimg tag.

Currently, this attribute can only be used forpictures, iframe, videoand some other tags.

You can specify values ​​in pixels or percentages. If you specify the size as a percentage, they will be taken from the width of the parent. If you specify a size in pixels, the units of measurement are not specified.

If neither width nor height is specified for the image, the image will have its actual size. If the height is specified, the picture will become the specified height, and the width will be adjusted so that its proportions are not distorted. If only the width is specified, similarly, the image will be adjusted in height so as to maintain the proportions. If both width and height are specified, the proportions of the picture may be distorted (or maybe not, depending on your guess). If the width or height (or both together) is larger than the real one, the picture will increase, but will lose quality.

It is recommended to set the width and height of images in attributes - in this case, the browser will load images faster - it does not need to calculate the size of each image after receiving it.

It is not recommended to reduce the actual size of the image unless necessary. For example, the actual size of the picture1000on1000pixels, and you give it a width of100px. In this case, the picture on the screen will look like100pixels, but have a size of a whole thousand and, accordingly, take much longer to load.
