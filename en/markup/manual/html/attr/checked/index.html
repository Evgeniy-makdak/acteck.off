The checked attribute

Attributecheckedmakes the checkbox checked by defaultcheckboxor switchradio.

Is an attribute without a value.

Example. Checkboxes

In the following example, two checkboxes are made: the first one will be checked (since it is given the attributechecked), but the second one does not:

<input type="checkbox" checked> <input type="checkbox">

:

Example. Radio switches

Let's make two radio switches - the first one will be marked (since we will give it the attributechecked), but the second one does not:

<input type="radio" name="radio" checked> <input type="radio" name="radio">

:
