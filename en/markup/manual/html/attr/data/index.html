data attribute

Attributedata-used to create attributes with their own names. Such attributes are used to store various data in HTML layout.

How to create your own attributes: their names must start withdata-, and then the name you came up with should come. Attributes created in this way will be validated.

Example

Let's use the attributedata-Let's save the day, month and year:

<span data-day="12" data-month="03" data-year="2015">text</span>
