value attribute

Attributevaluesets the default value for an HTML input fieldforms. It will be automatically inserted when you visit the page.

Applies to tagsinput, optionand others.

Text input

In case of text input fieldinputattribute contentvaluewill set the default text visible in the field (you can delete it and write your own).

Button

In case the attribute is applied to a button created through the taginputwith attributetypein meaningsbutton, submitorreset, attributevaluewill set the text for the button.

Checkbox

In case the attribute is applied tocheckboxno external changes will be observed, but the checked checkbox will send the contents of this attribute to the server. If attributevaluethere is no checkbox - it will send a string'on'. If the checkbox is not checked, nothing will be sent to the server, regardless of the presence of the attributevalue.

Radio switch

For a groupradio switchesit is necessary to set different attribute valuesvalueso that after sending to the server, the server language, for example,PHPorNodeJScould determine which radio button was selected.

option tag

In case the tagoptionattribute not setvalue, then the contents of the tag will be sent to the serveroption. If the attribute is specified, its contents will be sent.

Let's set the input field to default text:

<input type="text" value="default text">
