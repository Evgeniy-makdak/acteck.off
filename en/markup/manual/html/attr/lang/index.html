lang attribute

Attributelangsets the text language. This is necessary so that the browser correctly displays some language characters. An example is quotes (see propertyquotes) and text hyphens (see propertyhyphens).

The meaning corresponds to the Russian languageru, Englishenand so on (see the table below the example for other languages).

Example

Let's set the text to Russian:

<p lang="en">русский язык</p>
