Атрибут autofocus

Атрибут autofocus задает автоматический фокус ввода по умолчанию для полей HTML формы при загрузке страницы.

Toт элемент, которому задан атрибут autofocus, будет иметь фокус при загрузке страницы (в нем будет моргать курсор, если начать вводить текст с клавиатуры - он будет попадать в поле с фокусом ввода).

Атрибут является атрибутом без значения. Его следует применять к тегам input или textarea и только одному элементу на странице, в противном случае будет конфликт.

Пример

Давайте посмотрим при заходе на страницу первый тег input который имеет фокус ввода, так как ему мы задали атрибут autofocus (только не нажимайте никуда мышкой, так как фокус из инпута пропадет):

<input type="text" autofocus> <input type="text"> <input type="text">
