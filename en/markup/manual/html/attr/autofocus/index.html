autofocus attribute

Attributeautofocussets automatic default input focus for HTML form fields when the page loads.

The element to which the attribute is givenautofocus, will have focus when the page is loaded (the cursor will blink in it, if you start entering text from the keyboard, it will fall into the field with the input focus).

An attribute is an attribute without a value. It should be applied to tagsinputortextareaand only one element on the page, otherwise there will be a conflict.

Example

Let's look at the first tag when entering the pageinputwhich has input focus, since we set the attribute to itautofocus (just don't click anywhere with your mouse, since the focus from the input will be lost):

<input type="text" autofocus> <input type="text"> <input type="text">
