required attribute

Attributerequiredindicates that this HTML elementformsmust be filled out.

The attribute should be applied to tagsinputortextarea.

When you try to submit a form if the field with this attribute is not filled in, the browser will not allow you to submit the form and will display an error in the form of a tooltip. Unfortunately, the error text and appearance cannot be changed using HTML or CSS.

Please note that the presence of an attributerequireddoes not free you from checking that the form is filled out correctly on the PHP server side (since attribute protection is easy to bypass).

Example

Let's taginputadd an attributerequired. Without entering anything in the field, try clicking the button to submit the form. The browser will not allow you to submit the form, will add a shadow to the input and display an error message:

<form action=""> <input type="text" placeholder="введите что-нибудь" required> <input type="submit"> </form>
