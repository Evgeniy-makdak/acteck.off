Атрибут selected

Атрибут selected задает выбранный по умолчанию пункт option выпадающего списка select.

Является атрибутом без значения.

Пример

Давайте выберем город Минск по умолчанию. Сделаем это с помощью атрибута selected:

<select> <option>city1</option> <option selected>city2</option> <option>city3</option> <option>city4</option> </select>

:

Пример . Несколько значений по умолчанию в мультиселекте

В мультиселекте по умолчанию может быть выбрано более, чем одно значение:

<select multiple name="city[]"> <option>city1</option> <option selected>city2</option> <option>city3</option> <option selected>city4</option> </select>

:
