placeholder attribute

Attributeplaceholdersets a tooltip in an HTML input fieldforms.

This hint is needed to indicate to the user what exactly should be entered in this field. The tooltip itself will be located in the input field, but, unlike the attributevalue, will disappear automatically when you try to enter text.

Applies to tagsinput, textarea.

Apply CSS styles to tooltipplaceholderyou can using pseudo elementplaceholder.

Example

Let's set the input field to a hintplaceholder:

<input type="text" placeholder="введите имя">

:

Example. The value attribute simultaneously with placeholder

And here let's set the input field to a hint at the same timeplaceholderand attributevalue. If you remove the attribute contentvalue, you will see a hint hidden under it:

<input type="text" placeholder="введите имя" value="Иван">
