autocomplete attribute

Attributeautocompletecancels automatic form completion.

Automatic autofill is when the browser offers to substitute in a field the data that has already been entered into this field previously. This may not always be convenient, since, for example, the browser can tell your username or password to an attacker, etc. For fields where sensitive information is entered, autofill should be disabled.

Attributeautocompletecan be used not only to hide secret information. For example, you can disable auto-completion in test answers. This is necessary so that when you take the test again, the browser does not suggest answers (all the interest of the tests is lost).

It can be used as an attribute without a value, or you can specify one of the values.

Values

Meaning Description
on Autocomplete is enabled (this is the default).
off Autofill is disabled.

Example

Let's haveinputLet's turn on autofill. Enter, for example, the word “autocomplete” into it and click on the send button (this is required, sincethe browser remembers only the words sent). After that, go back to the example and enter the letter "a" - the browser will prompt you for the word "autocomplete" and you can select it without entering the whole thing:

<form action=""> <input type="text" autocomplete="on" name="test"> <input type="submit"> </form>

Example

Now let's turn off autofill. Do the same manipulations as in the previous example, but no prompts will appear:

<form action=""> <input type="text" autocomplete="off" name="test"> <input type="submit"> </form>
