disabled attribute

Attributedisabledblocks HTML elementforms, that is, makes it inactive. Is an attribute without a value.

Locking elements is usually needed to prevent the user from changing certain values ​​(which, nevertheless, must be shown to the user as form elements). Sometimes elements are blocked (and unlocked) using JavaSctipt.

Behavior

In the case of the button (buttonorinputwith attributetypein meaningsbutton, resetorsubmit) blocking means that the button cannot be pressed. In the case of a text input field (inputortextarea) it will not be possible to change or copy the text in it. In the case ofcheckboxesAndradiotheir state (checked or not) cannot be changed. In the case of drop-down listsselectIt will not be possible to change the selected list item.

A blocked element has a gray background by default. He will also not participate inTab key transitions.
