alt attribute

Attributealtalternative text that will be shown if the browser cannot find the image specified in the attributesrctagimg. Is a required attribute for the tagimg, if it is absent, the validator (a program that checks the correctness of HTML and CSS) will complain.

Example

Let's try to put the wrong path to the picture (leave it empty). Instead of a picture, we will see the contents of the attributealt(it seems that this is ordinary text - but try to copy it - you won’t succeed, it will stretch like a picture):

<img src="" alt="Картинка веселой обезьянки">
