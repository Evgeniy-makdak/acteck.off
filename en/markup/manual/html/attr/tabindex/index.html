tabindex attribute

Attributetabindexsets the order in which focus is received when moving between elements using a keyTab.

By pressing this key you can activate sequentially some elements of the page (linksand elementsforms). Active elements will receive input focus.

For input fields likeinputAndtextareathe trick will be that the cursor will blink in the field and text can be entered into it,for linksand other elements, this will be highlighted in some way (underlining, dotted border, etc., depending on the browser).

If an element was not visible on the page when it received focus (due to scrolling), then the page will scroll to that element.

If elements are not given an attributetabindexor does it matter0, then move through them with the keyTabwill go in the order of the elements in the HTML code.

If there are elements on the page that are giventabindex, then first the transition will follow them, starting from the lower attribute valuetabindex(the smallest can be one) and so on in increasing attribute values, and when such elements run out, the transition will go through those elements that are not giventabindexor does it matter0.

The attribute values ​​are integers from1ad infinitum. If some numbers are missing, nothing bad will happen (for example, if there is no number2, then focus will first fall on the element withtabindex, equal1, and then to the element withtabindex, equal3).

If on the page at the moment the key is pressedTabthere is an element in focus (focus could be obtained not only by pressingTab, but also by clicking the mouse on an element or attributeautofocus), then the next key pressTabwill cause the next element after the one with focus to receive focus (for example, if the element with focus is currently in focustabindex 3, then the next element to receive focus istabindex 4).

If the input field is given the attributedisabled, then it will be ignored by key transitionsTab, even if this field is given the attributetabindex.

Attributetabindexapplies to tagsa, input, textarea, button, select, area.

Example

Let'sinputsset the attributetabindex. Press the key successivelyTaband you will see how the input focus will move from the first input to the fourth, and then follow the links from the beginning of the page (since there are more elements with the given attributetabindexNo):

<input type="text" tabindex="3" placeholder="number: 3"> <input type="text" tabindex="1" placeholder="number: 1"> <input type="text" tabindex="2" placeholder="number: 2"> <input type="text" tabindex="4" placeholder="number: 4">
