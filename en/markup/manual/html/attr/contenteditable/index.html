contenteditable attribute

Attributecontenteditableallows you to make any page element editable through the browser.

Meaningtrueincludes editing, andfalse- turns off. Instead oftrueYou can specify an empty value or not write it at all.

You should not think that after editing, changes on the page will be saved somewhere. You should implement this yourself via JavaScript.

Example

Let's edit the paragraph text through the browser (just click on it and start writing):

<p contenteditable> Текст данного абзаца можно редактировать через браузер. </p>

:

Example

Now let’s disable text editing (for this you can use the attributecontenteditabledo not write at all):

<p contenteditable="false"> Текст данного абзаца нельзя редактировать через браузер. </p>
