class attribute

Attributeclassspecifies one or more classes for an element (by element we mean a tag).

This is done in order to then access a group of elements that have the same class via CSS and apply certain properties to it (for example, changetext color, font sizeand so on).

There is alsoid attribute, which is similar to the attributeclassallows you to select elements on an HTML page.

Difference between attributeclassAndid attributeis that class selects a group of elements (even if it is given to one element, it can later be given to another), andidselects a unique element (there should not be more elements with the same id on the site page, otherwise there will be a conflict).

How do you know what to give an element - a class or an id? The class is given to those elements that are repeated on the pages of the site (so as not to write the same CSS code several times). Even if you currently have only one element, but you feel that similar elements may appear in the future, give this element a class. If you are sure that such an element is unique, then give it an id. Although the current trend is to give all elements a class and leave the id to JavaScript, it is not universally accepted.

An element can be given several classes; in this case, they should be listed separated by a space.

Class names must be typed in English letters, numbers, without spaces (a space separates classes from each other; an underscore or hyphen can be used instead). Classes should not start with a number (this is already possible in HTML5, but will not work in older browsers).

Class names should be given in English (and not in Russian, just in English letters!). Names should be meaningful and reflect the essence of the class.

Example

Let's all paragraphs with classtestset the text color to red:

<p class="test">Абзац с классом test.</p> <p>Контрольный абзац без класса.</p> .test { color: red; }

Example. Multiple classes for an element

And here let's give the first paragraph several classes -test1Andtest2(let's write them separated by a space). Classtest1sets redtext color, and classtest2setsfont sizeV20px. The second paragraph is given only the classtest1(this paragraph will turn red), and the third paragraph will be given a classtest2(this paragraph will have a font size of 20px). The first paragraph, which has two classes, will have both a red color and a font size of20px:

<p class="test1 test2">Абзац с двумя классами test1 и test2.</p> <p class="test1">Абзац с классом test1.</p> <p class="test2">Абзац с классом test2.</p> <p>Контрольный абзац без классов.</p> .test1 { color: red; } .test2 { font-size: 20px; }
