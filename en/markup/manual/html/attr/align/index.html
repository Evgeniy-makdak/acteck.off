Атрибут align

Атрибут align задает выравнивание текста или элемента. Является устаревшим, хотя по-прежнему работает. Вместо него следует использовать CSS свойство text-align для выравнивания текста и CSS свойство margin в значении auto для выравнивания блочных элементов по центру.
