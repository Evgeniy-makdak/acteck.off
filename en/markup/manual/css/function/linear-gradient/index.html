linear-gradient function

Functionlinear-gradientspecifies a linear gradient. Applies to properties that specify the background image:background, background-imageor a picture of the border:border-image, background-image-source.

Syntax

селектор { background: linear-gradient([угол или направление], цвет1 размер1, цвет2 размер2...); }

Values

Meaning Description
direction Sets a specific gradient direction in anyunits for anglesor keywordstop, left, right, bottomor a combination of them:top left, top rightand so on. The order of words is not important: you can writetop leftAndleft top, there is no difference. The parameter is optional: if you do not write it, the gradient will go from top to bottom (as withto top). The direction is preceded by the wordto.
corner Any angleunits for angles. Can be positive or negative. The parameter is optional. Either the angle or the direction (or nothing at all) can be specified at the same time.
color1 The initial color of the gradient in anyunits for color.
color2 The final color of the gradient in anyunits for color.
size Sets the extent of a specific gradient color in anyunits for size.

Example. The simplest option

Syntax:

селектор { background: linear-gradient(начальный цвет, конечный цвет); }

Let's look at an example:

<div id="elem"></div> #elem { background: linear-gradient(black, red); width: 200px; height: 200px; }

:

Example. Adding an angle

Syntax:

селектор { background: linear-gradient(угол, начальный цвет, конечный цвет); }

Let's look at an example:

<div id="elem"></div> #elem { background: linear-gradient(45deg, black, red); width: 200px; height: 200px; }

:

Example. Adding a direction

Instead of an angle, you can add a directiontop, left, right, bottomor a combination of them:top left, top rightThe direction is preceded by the wordto.

Syntax:

селектор { background: linear-gradient(направление, начальный цвет, конечный цвет); }

Let's look at an example:

<div id="elem"></div> #elem { background: linear-gradient(to left, black, red); width: 200px; height: 200px; }

:

Example. Adding a direction

Let's point in another direction:

<div id="elem"></div> #elem { background: linear-gradient(to top left, black, red); width: 200px; height: 200px; }

:

Example. Adding size

Syntax:

селектор { background: linear-gradient(цвет1 размер1, цвет2 размер2); }

In the following example the behavior would be: pure red would be from0pxto30px, from30pxto50pxthere will be a gradient from red to blue, and from50pxand until the end - pure blue:

<div id="elem"></div> #elem { background: linear-gradient(red 30px, blue 50px); width: 200px; height: 200px; }

:

Example. We add more than2colors

Syntax:

селектор { background: linear-gradient(цвет1 размер1, цвет2 размер2, цвет3 размер3...); }

In the following example the behavior would be: pure red would be from0pxto30px, from30pxto50pxthere will be a gradient from red to blue, and from50pxand up to70px- gradient from blue to green, and then70px- pure green:

<div id="elem"></div> #elem { background: linear-gradient(red 30px, blue 50px, green 70px); width: 200px; height: 200px; }

:

Example. Abrupt transitions

In the following example the behavior would be: pure red would be from0pxto30px, pure blue - from30pxto60px, pure green - after60px (red 0pxyou don't have to write it):

<div id="elem"></div> #elem { background: linear-gradient(red 0px, red 30px, blue 30px, blue 60px, green 60px); width: 200px; height: 200px; }

:

Example. Dimensions can also be specified as percentages

In the following example the behavior would be: pure red would be from0%to30%, pure blue - from30%to60%, pure green - after60% (red 0%you don't have to write it):

<div id="elem"></div> #elem { background: linear-gradient(red 0%, red 30%, blue 30%, blue 60%, green 60%); width: 200px; height: 200px; }

:

Example. Combination with background-size

In the following example the behavior would be: pure red would be from0pxto30px, pure blue - from30pxto60px, and all this will be repeated in pieces60px(because ofbackground-size: 60px;):

<div id="elem"></div> #elem { background: linear-gradient(to right, red 30px, blue 30px, blue 60px); background-size: 60px 60px; width: 200px; height: 200px; }
