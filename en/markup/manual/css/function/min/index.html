min function

Functionminallows you to select the minimum of the transmitted values. The function takes two or more values ​​separated by commas as parameters and returns the minimum of them.

Syntax

селектор { свойство: min(значение1, значение2); }

Example

Now the width of the block will be or300pxor50%from the parent, whichever is smaller:

<div id="parent"> <div id="child"></div> </div> #child { width: min(300px, 50%); height: 100px; margin: 0 auto; border: 1px solid red; } #parent { border: 1px solid green; }
