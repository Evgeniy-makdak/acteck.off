translateY function

FunctiontranslateYspecifies the shift of the element along the Y axis. Used in conjunction with the propertytransform. The property value is anyunits for size. A positive value moves down, a negative value moves up.

Syntax

селектор { transform: translateY(сдвиг); }

Example

If you hover your mouse over a block, it will move to30pxdown:

<div id="elem"></div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: translateY(30px); }

:

Example

If you hover your mouse over a block, it will move to30pxup:

<div id="elem"></div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: translateY(-30px); }

:
