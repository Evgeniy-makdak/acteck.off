rotate function

Functionrotate()sets the rotation of an element at a given angle. Used in conjunction with the propertytransform. The value of the property is the angle in anyunits for angles. A positive value rotates clockwise, a negative value rotates counterclockwise. Rotation is carried out around the point specified by the propertytransform-origin.

Syntax

селектор { transform: rotate(угол); }

Example

In this example the block is rotated to30degrees clockwise:

<div id="elem">lorem ipsum</div> #elem { transform: rotate(30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example the block is rotated to30degrees counterclockwise:

<div id="elem">lorem ipsum</div> #elem { transform: rotate(-30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, the hover block will rotate to1revolution, since the rotation value is set in1turn(the same effect can be achieved if you set the rotation angle to360deg). For smooth rotation, a property has been addedtransition:

<div id="elem">lorem ipsum</div> #elem { border: 3px solid black; width: 100px; height: 50px; transition: transform 3s linear; } #elem:hover { transform: rotate(1turn); }

:
