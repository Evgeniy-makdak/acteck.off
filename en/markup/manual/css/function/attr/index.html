attr function

Functionattrallows you to get the value of an HTML tag attribute in CSS.

Syntax

селектор { свойство: attr(имя, тип, значение); }

Options

Parameter Description
Name The element attribute name passed to the CSS.
type CSS property value type. The following keywords are allowed:string(strings),color(color),url(web address),integer(integer),number(real number),angle(corner),time(time),deg(degree), as well as CSS units likeem, pxetc.
meaning The value to be used if the attribute does not exist or the browser cannot obtain it.

Example

Let's display the value of its attribute after the link texthrefin parentheses:

<a href="http://google.com">ссылка</a> a::after { content: ' (' attr(href) ')'; }
