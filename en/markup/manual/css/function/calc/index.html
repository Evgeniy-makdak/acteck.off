Функция calc

Функция calc позволяет производить вычисления значений CSS свойств. Функция в качестве параметра принимает математическое выражение, результат вычисления которого можно использовать как значение CSS свойства. Выражение может включать операторы +, -, *, /.

Синтаксис

селектор { свойство: calc(выражение); }

Пример

Давайте сделаем так, чтобы ширина блока была равна 100% от ширины родителя, минус 50px:

<div id="parent"> <div id="child"></div> </div> #child { width: calc(100% - 50px); height: 100px; margin: 0 auto; border: 1px solid red; } #parent { border: 1px solid green; }
