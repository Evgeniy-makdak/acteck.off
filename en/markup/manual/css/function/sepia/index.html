sepia function

Functionsepiaallows you to turn a background image into sepia - a black and white image with a brown tint. Meaning0indicates that the image will remain original. Meaning100%or1turns the background completely to sepia. Any number located in this range sets the application of sepia linearly. A negative value cannot be specified. The default value is0.

Syntax

селектор { filter: sepia(число); }

Example

Let's make our image full sepia:

<div id="elem"></div> #elem { filter: sepia(100%); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }

:

Example

Now let's tone down the sepia by50%:

<div id="elem"></div> #elem { filter: sepia(50%); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }

:

Example

Now let's set in the functionsepiavalue equal0:

<div id="elem"></div> #elem { filter: sepia(0); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }
