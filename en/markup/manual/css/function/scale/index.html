scale function

Functionscalescales an element: increases or decreases it several times. Scaling is carried out around the point specified by the propertytransform-origin.

Can take one or two parameters, separated by commas. If there are two parameters, the first parameter specifies horizontal scaling, and the second parameter specifies vertical scaling. If there is only one parameter, then it sets horizontal and vertical scaling at the same time.

The parameter value is an integer or fractional number. If it is more1- the element is increased if less (for example,0.5) - the element decreases. If you set1- nothing will change (this is the default value).

Syntax

селектор { transform: scale(одно или два числа); }

Example

By pointing at the block, we will zoom in1.5times on both axes:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scale(1.5); }

:

Example

Let's zoom out to2times on both axes:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scale(0.5); }

:

Example

Let's zoom in2times along the X axis:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scale(2, 1); }

:

Example

Let's zoom in2times along the Y axis:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scale(1, 2); }

:
