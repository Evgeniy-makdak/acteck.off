skewX function

FunctionskewXsets the bevel of the element along the X axis, thereby turning it into a parallelogram. Used in conjunction with the propertytransform. The value of the property is the angle in anyunits for angles. The angle can be either positive or negative. A positive value skews to the left, a negative value skews to the right. The bevel is carried out around the point specified by the propertytransform-origin.

Syntax

селектор { transform: skewX(угол); }

Example

In this example, the block is beveled at30degrees left:

<div id="elem">lorem ipsum</div> #elem { transform: skewX(30deg); border: 1px solid black; width: 100px; height: 50px; }

Example

In this example, the block is beveled at30degrees to the right:

<div id="elem">lorem ipsum</div> #elem { transform: skewX(-30deg); border: 1px solid black; width: 100px; height: 50px; }
