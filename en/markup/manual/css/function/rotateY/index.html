rotateY function

FunctionsrotateYspecifies rotation about the Y axis in three-dimensional space. Used in conjunction with the propertytransform. The value of the property is the angle in anyunits for angles. A positive value turns towards us, a negative value turns away from us. Rotation is carried out around the point specified by the propertytransform-origin.

Syntax

селектор { transform: rotateY(угол); }

Example

In this example, the hover block will rotate to180degrees along the Y axis. For smooth rotation, a property has been addedtransition:

<div id="elem">lorem ipsum</div> #elem { border: 3px solid black; width: 100px; height: 50px; transition: transform 3s linear; } #elem:hover { transform: rotateY(180deg); }

:

Example

Let's change the axis of rotation using the propertytransform-origin:

<div id="elem">lorem ipsum</div> #elem { border: 3px solid black; width: 100px; height: 50px; transition: transform 3s linear; transform-origin: right center; } #elem:hover { transform: rotateY(180deg); }
