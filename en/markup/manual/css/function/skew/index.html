skew function

Functionskewsets the bevel of the element, thereby turning it from a rectangle into a parallelogram. Used in conjunction with the propertytransform. The value of the property is the angle in anyunits for angles. The bevel is carried out around the point specified by the propertytransform-origin.

Can take one or two parameters, separated by commas. If there are two parameters, the first parameter sets the bevel horizontally, and the second - vertically. If there is only one parameter, then it specifies the bevel horizontally (and not on both sides). The parameters can be either positive or negative.

Syntax

селектор { transform: skew(один или два угла); }

Example

In this example, the block is beveled at30degrees left:

<div id="elem">lorem ipsum</div> #elem { transform: skew(30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, the block is beveled at30degrees to the right:

<div id="elem">lorem ipsum</div> #elem { transform: skew(-30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, the block is beveled at30vertical degrees (positive value):

<div id="elem">lorem ipsum</div> #elem { transform: skew(0deg, 30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, the block is beveled at30degrees vertical (negative value):

<div id="elem">lorem ipsum</div> #elem { transform: skew(0deg, -30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

We set the bevel along the X and Y axis simultaneously:

<div id="elem">lorem ipsum</div> #elem { transform: skew(30deg, 30deg); border: 1px solid black; width: 100px; height: 50px; }

:
