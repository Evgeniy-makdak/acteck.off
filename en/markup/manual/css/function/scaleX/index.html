scaleX function

FunctionscaleXscales the element along the X axis. Scaling occurs around the point specified by the propertytransform-origin. The function value is an integer or fraction. If it is more1- the element is increased if less (for example,0.5) - the element decreases. If you set1- nothing will change (this is the default value).

Syntax

селектор { transform: scaleX(число); }

Example

By pointing at the block, we will zoom in1.5times along the X axis:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scaleX(1.5); }

:

Example

By pointing at the block, reduce the scale by2times along the X axis:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform: scaleX(0.5); }

:
