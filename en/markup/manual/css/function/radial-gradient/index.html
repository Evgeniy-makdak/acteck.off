radial-gradient function

Functionradial-gradientspecifies the radial gradient. This function applies to properties that specify the background image:background, background-imageor a picture of the border:border-image, background-image-source.

Syntax

селектор { background: radial-gradient([форма тип позиция], цвет1 размер1, цвет2 размер2...); }

Values

Meaning Description
form Accepted values:ellipseelliptical gradient (default),circlecircular gradient.
type Sets the gradient stretch. Accepted values:closest-side, closest-corner, farthest-side, farthest-corner.
position After the keywordatposition is indicated in anyunits for sizeor using keywordstop, bottom, left, right, centerin various combinations.
color1 The initial color of the gradient in anyunits for color.
color2 The final color of the gradient in anyunits for color.
size Sets the extent of a specific gradient color in anyunits for size.

Values ​​for type

Meaning Description
closest-side The shape of the gradient matches the side of the block closest to it.
closest-corner The shape of the gradient is calculated based on information about the distance to the nearest corner of the block.
farthest-side The gradient extends to the far side of the block.
farthest-corner The shape of the gradient is calculated based on information about the distance to the far corner of the block.

Comment

The type and shape can be swapped, but the position should go at the end of the first parameter. The gradient type and its size do not work with each other (logically, they conflict). Size wins.

Example. The simplest option

Let's just set the start and end colors:

<div id="elem"></div> #elem { background: radial-gradient(red, blue); width: 200px; height: 200px; }

:

Example. Let's add a position

Let's set the center position:30px- indentation on the left,100px- top indent:

<div id="elem"></div> #elem { background: radial-gradient(at 30px 100px, red, blue); width: 200px; height: 200px; }

:

Example. Position in percentage

Let's set the center position as a percentage:30%- indentation on the left,50%- top indent:

<div id="elem"></div> #elem { background: radial-gradient(at 30% 50%, red, blue); width: 200px; height: 200px; }

:

Example. Position by keyword

Keywords can be usedtop, bottom, left, right, centerin various combinations. Let's put, for example, a gradient right in the center:

<div id="elem"></div> #elem { background: radial-gradient(at right center, red, blue); width: 200px; height: 200px; }

:

Example. Gradient size

In this case, the gradient will work like this: from0pxto20pxthere will be a pure red color, from20pxto60px- gradient from red to blue, after60px- pure blue:

<div id="elem"></div> #elem { background: radial-gradient(red 20px, blue 60px); width: 200px; height: 200px; }

:

Example. Several flowers in a row

In this case, the gradient will work like this: from0pxto20pxthere will be a pure red color, from20pxto40pxthere will be a pure blue color, from40pxto60px- gradient from blue to green, after60px- pure green:

<div id="elem"></div> #elem { background: radial-gradient(red 20px, blue 20px, blue 40px, green 60px); width: 200px; height: 200px; }

:

Example. Size + position

Let's simultaneously set the sizes for different colors and the position of the gradient center:

<div id="elem"></div> #elem { background: radial-gradient(at 30px 60px, red 20px, blue 60px); width: 200px; height: 200px; }

:

Example. Gradient shape

The shape of the gradient is specified by the first parameter and can take2values:circle(circular gradient) orellipse(elliptical gradient, default). Why haven’t we seen elliptical gradients before, even though it is set by default? Because before that the shape of the blocks was square. If we change the shape to rectangular, we will see an elliptical gradient:

<div id="elem"></div> #elem { background: radial-gradient(red 20px, blue 60px); width: 300px; height: 200px; }

:

Example. Round gradient

Let's make a circular gradient in a rectangular box. To do this, set the first parameter to the shape of the gradient using the keywordcircle:

<div id="elem"></div> #elem { background: radial-gradient(circle, red 20px, blue 60px); width: 300px; height: 200px; }

:

Example. Let's add a position

Let's add the position of the gradient to the previous code:

<div id="elem"></div> #elem { background: radial-gradient(circle at 30px 60px, red 20px, blue 60px); width: 300px; height: 200px; }

:

Example. Type farthest-corner + circle

The shape of the gradient is calculated based on information about the distance to the far corner of the block:

<div id="elem"></div> #elem { background: radial-gradient(circle farthest-corner at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type farthest-corner + ellipse

The shape of the gradient is calculated based on information about the distance to the far corner of the block:

<div id="elem"></div> #elem { background: radial-gradient(ellipse farthest-corner at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type farthest-side + circle

The gradient extends to the far side of the block:

<div id="elem"></div> #elem { background: radial-gradient(circle farthest-side at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type farthest-side + ellipse

The gradient extends to the far side of the block:

<div id="elem"></div> #elem { background: radial-gradient(ellipse farthest-side at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type closest-corner + circle

The shape of the gradient is calculated based on information about the distance to the nearest corner of the block:

<div id="elem"></div> #elem { background: radial-gradient(circle closest-corner at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type closest-corner + ellipse

The shape of the gradient is calculated based on information about the distance to the nearest corner of the block:

<div id="elem"></div> #elem { background: radial-gradient(ellipse closest-corner at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. closest-side + circle type

The shape of the gradient matches the side of the block closest to it:

<div id="elem"></div> #elem { background: radial-gradient(circle closest-side at 60px 80px, red, blue); width: 300px; height: 200px; }

:

Example. Type closest-side + ellipse

The shape of the gradient matches the side of the block closest to it:

<div id="elem"></div> #elem { background: radial-gradient(ellipse closest-side at 60px 80px, red, blue); width: 300px; height: 200px; }
