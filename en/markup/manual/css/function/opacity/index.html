opacity function

Functionopacityallows you to change the transparency of the background image. Meaning100%or1Leaves the image at its original transparency. Meaning0indicates that the image will become completely transparent. Any number located in this range specifies partial transparency of the background. A negative value cannot be specified. The default value is1.

Syntax

селектор { filter: opacity(число); }

Example

Let's set transparency to our image50%:

<div id="elem"></div> #elem { filter: opacity(50%); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }

:

Example

Now let's set the transparency100%:

<div id="elem"></div> #elem { filter: opacity(100%); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }

Example

Now let's set in the functionopacityvalue equal0:

<div id="elem"></div> #elem { filter: opacity(0); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; }
