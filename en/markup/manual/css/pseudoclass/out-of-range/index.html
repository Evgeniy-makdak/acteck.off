out-of-range pseudo-class

Pseudo-classout-of-rangesets the style for elements whose values ​​fall outside the specified range. The range of values ​​must be specified by the attributesminAndmax.

Syntax

селектор:out-of-range { }

Example

Let's set a blue color for the background of the input into which numbers falling within the range from1to5. We will also set a pink background when the entered value does not fall within the selected limit:

<form> <p>Enter number from 1 to 5</p> <p> <input type="number" min="1" max="5" value="1"> </p> </form> input:in-range { background: #C2DDFD; } input:out-of-range { background: #E37D76; }

:

See also

  • pseudo-class:invalid,

  • pseudo-class:valid,
    which sets the style of the input that entered the correct value
