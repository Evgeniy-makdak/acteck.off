Last-of-type pseudo-class

Pseudo-classlast-of-typeselects the element that is the last child of the parent of the given type. That is, if I writeh2:last-of-type- the last one will be foundh2in the parent (as opposed tolast-child, which only one will findh2, which is the last one in the parent).

Syntax

селектор:last-of-type { }

Example

Let's find the titleh2, which is the last oneh2in parent:

<div> <h2>заголовок</h2> <p>абзац</p> <h2>заголовок</h2> <p>абзац</p> <h2>заголовок</h2> <p>абзац</p> </div> h2:last-of-type { color: red; }
