Pseudo-class link

Pseudo-classlinksets styles for unvisited links. Typically this pseudo-class is used in conjunction with pseudo-classeshover, visitedAndactive.

Syntax

a:link{ }

Example

An unvisited link will be red, a visited link will be blue. If you hover over it with the mouse, it will turn green, and if you press the left mouse button and hold it, it will turn black (stateactive):

<a href="#">ссылка</a> a:link { color: red; } a:visited { color: blue; } a:hover { color: green; } a:active { color: black; }
