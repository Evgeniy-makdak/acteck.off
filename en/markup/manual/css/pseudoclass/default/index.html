Pseudo-class default

Pseudo-classdefaultsets the style to one element from a group that has a default type. These elements must be inside the parent tag<form>.

Syntax

селектор:default { }

Example

An example of a default type is attributetypewith meaningsubmit. Let's set one of these buttons to have a blue background:

<form> <p> <label>Enter number</label> <input type="number"> </p> <p> <input type="submit"> <input type="submit"> <input type="reset"> </p> </form> :default { background-color: #C2DDFD; }

:
