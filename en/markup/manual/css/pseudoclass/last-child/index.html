Pseudo-class last-child

Pseudo-classlast-childselects the element that is the last child of its parent.

Syntax

селектор:last-child { }

Example

In this example we will make it redli, which is the last child of its parent (the last one insideol):

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li:last-child { color: red; }
