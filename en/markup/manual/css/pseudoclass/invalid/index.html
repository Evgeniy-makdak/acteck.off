Pseudo-class invalid

Pseudo-classinvalidsets the style for an input that has an incorrect value or a value that does not match the specified type.

Syntax

селектор:invalid { }

Example

Let's recolor the input background to pink when an incorrect value is entered and back to green when the value passes the check for the selected type:

<p>Enter Your email</p> <p><input type="email"></p> input:invalid { background: #E37D76; } input:valid { background: #9DDFA4; }

:

See also

  • pseudo-class:valid,
    which sets the style of the input that entered the correct value
  • pseudo-class:default,
    which sets the default style for elements
  • pseudo-class:in-range,
    which sets the style to elements with a given range of values
  • pseudo-class:out-of-range,
    which styles elements whose values ​​are out of range
