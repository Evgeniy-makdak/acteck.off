Pseudo-class lang

Pseudo-classlangselects text based on the language specified in the attributelang. The following values ​​can be used as a language:ru- Russian,en- English,de- German,fr- French,it- Italian and so on.

Syntax

селектор:lang(язык) { }

Example

Let's put different quotes for different languages ​​(we usequotes):

<p> Цитата на английском: <q lang="en">To be or not to be</q>. </p> <p> Цитата на немецком: <q lang="de">Versuche die Gotter nicht</q>. </p> q:lang(en) { quotes: "\201C" "\201D"; } q:lang(de) { quotes: "\201E" "\201C"; }
