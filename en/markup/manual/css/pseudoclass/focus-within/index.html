Focus-within pseudo-class

Pseudo-classfocus-withinsets the style of the elements that are in focus. Unlike pseudo classfocus, which sets the style directly to the element itself,focus-withinsets the style for the parent element.

Syntax

селектор:focus-within { }

Example

Let's set the parent element's background color to blue when its child input element has focus:

<div> Введите ваше имя: <input name="user"> </div> div { width: 300px; height: 50px; padding: 10px; } div:focus-within { background-color: #C2DDFD; }
