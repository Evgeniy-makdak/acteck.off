Pseudo-class enabled

Pseudo-classenabledsets styles for the activeinput. By active we mean an input that does not have an attribute specifieddisabled.

Syntax

input:enabled { }

Example

Apply styles for the active input:

<input> <input disabled> input:enabled { border: 1px solid green; box-shadow: 0 0 3px green; }

:

See also

  • pseudo-classdisabled,
    which sets styles for the inactive input
