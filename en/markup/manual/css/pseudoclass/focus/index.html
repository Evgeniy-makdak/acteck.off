Pseudo-class focus

Pseudo-classfocussets styles forinput, textareaorlinkswith input focus.Input focus- this is when we clicked on the input and the wand cursor blinks (you can enter text).

Syntax

input:focus { }

Example

After receiving focus, the input will receive a red border and shadow:

<input> input:focus { border: 1px solid red; box-shadow: 0 0 3px red; }
