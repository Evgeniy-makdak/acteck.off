Pseudo-class optional

Pseudo-classoptionalsets the style of an input that does not have an attribute specifiedrequired, i.e. this pseudo-class applies to optional inputs.

Syntax

селектор:optional { }

Example

Let's give the optional input a gray background:

<form> <p> <label>Enter number</label> <input type="number" required> </p> <p> <label>Enter text</label> <input type="text"> </p> </form> input:optional { background-color: #D8DFE8; }

:
