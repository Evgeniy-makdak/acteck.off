Pseudo-class indeterminate

Pseudo-classindeterminatesets the style for switches that are in an undefined state, i.e. are not checked.

Syntax

селектор:indeterminate { }

Example

Let's recolor the text font green when the radio button is in an undefined state:

<form> <p> <input type="checkbox"> <span>Text</span> </p> </form> input:indeterminate, input:indeterminate + span { color: green; } input, span { color: black; } document.getElementsByTagName('input')[0].indeterminate = true;

:
