Pseudo-class only-of-type

Pseudo-classonly-of-typeselects the element that is the only child of the given type.

Syntax

селектор:only-of-type { }

Example

Let's color the titleh2in red if it is the only oneh2in its parent:

<div> <h2>заголовок</h2> <p>абзац</p> <p>абзац</p> <p>абзац</p> </div> <div> <h2>заголовок</h2> <p>абзац</p> <h2>заголовок</h2> <p>абзац</p> <p>абзац</p> </div> div { border: 1px dashed black; padding: 15px; margin-bottom: 10px; } h2:only-of-type { color: red; }
