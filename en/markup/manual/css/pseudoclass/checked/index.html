Pseudo-class checked

Pseudo-classcheckedallows you to catch the markedcheckboxorradio.

Unfortunately, styles for decorating them cannot be specified. For example, you cannot setbackgroundorborder color(you can set width and height, for example, but they won't have as much impact as you might expect).

Despite this, you can set styles for adjacent elements, for example, like this:input:checked + p- set styles for the paragraph that is located directly below the marked input.

Syntax

элемент:checked { }

Example

Check the checkboxes and you will see the marks light up in redlabel, which are located immediately after the checked checkboxes:

<input type="checkbox" id="elem1"> <label for="elem1">метка 1</label> <br> <input type="checkbox" id="elem2"> <label for="elem2">метка 2</label> input:checked + label { color: red; }
