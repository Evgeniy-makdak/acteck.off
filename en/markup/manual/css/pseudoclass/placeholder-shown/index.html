Pseudo-class placeholder-shown

Pseudo-classplaceholder-shownsets the style of the input or textarea that displays the so-called. hint text that disappears after you start typing.

Syntax

селектор:placeholder-shown {}

Example

Let's set a red frame for the input, which displays the hint text, and after entering it, let the frame turn green:

<form> <input placeholder="enter the text"> </form> input { border: 1px solid green; } input:placeholder-shown { border-color: red; }

:

    which sets the tooltip for the input field
