read-write pseudo-class

Pseudo-classread-writesets the style of inputs whose values ​​are available for change.

Syntax

селектор:read-write {}

Example

Let's set blue borders to the input where data can be entered:

<p> <label>Enter text</label> <input type="text"> </p> <p> <label>Read text</label> <input type="text" value="text" readonly> </p> input:read-write { border: 1px solid #467CBC; } input:read-only { border: 1px solid #D8DFE8; }

:
