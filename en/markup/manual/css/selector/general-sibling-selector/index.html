Related selector in CSS

Related selector~allows you to select all elements after a given element within one parent.

Syntax

селектор1 ~ селектор2 { }

Example

Let's go through all the tagsplocated after the tagsh2, and color them red:

<div> <h2>text</h2> <p> + </p> <p> + </p> <p> + </p> </div> <p> - </p> h2 ~ p { color: red; }

:

Example

Let's go through all the tagsp, located after elements with the class.test, and color them red:

<div> <p class="test"> text </p> <p> + </p> <p> + </p> </div> <p> - </p> .test ~ p { color: red; }

:

Example

Let's access all elements with a class.elem, located after elements with the class.test, and color them red:

<div> <p class="test"> text </p> <p class="elem"> + </p> <p> - </p> <p class="elem"> + </p> </div> <p> - </p> .test ~ .elem { color: red; }
