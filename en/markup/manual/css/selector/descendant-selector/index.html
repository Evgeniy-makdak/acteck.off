Context selector in CSS

The context selector, which is the space command(' '), allows you to select tags nested within a given parent

Syntax

селектор1 селектор2 { }

Example

Let's go through all the tagspinside tagsdivand color them red:

<div> <p> text </p> <p> text </p> </div> div p { color: red; }

:

Example

Let's go through all the tagspinside element with#elemand color them red:

<div id="elem"> <p> text </p> <p> text </p> </div> #elem p { color: red; }

:

Example

Let's go through all the tagspinside elements with class.blockand color them red:

<div class="block"> <p> text </p> <p> text </p> </div> .block p { color: red; }

:

Example

Let's go through all the tagsbinside tagsplocated inside tagsdivand color them red:

<div> <p> text <b>+</b> </p> </div> div p b { color: red; }
