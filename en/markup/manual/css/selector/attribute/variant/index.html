Selector by one of the attribute values

A selector based on one of the attribute values ​​sets a style for an element that has at least one of the specified values ​​in the attribute. In this case, all attribute values ​​must be separated by a space.

Example

Let's find everything<p>containing in one of the valuestest, and recolor them blue:

<p class="test main">text1</p> <p class="text">text2</p> <p class="tag test">text3</p> p[class~="test"] { color: blue; }
