Селектор по концу значения атрибута

Селектор по концу значения атрибута применяет стиль к элементу, у которого значение атрибута тега заканчивается на определенный текст.

Пример

Давайте обратимся к тегу a, у которого значение атрибута href заканчивается на .html и перекрасим его в красный цвет:

<a href="file.html">link1</a> <a href="https://code.mu">link2</a> a[href$=".html"] { color: red; }
