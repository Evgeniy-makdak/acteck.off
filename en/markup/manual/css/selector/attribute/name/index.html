Selector by attribute name

The attribute name selector styles an element that has only the specified tag attribute.

Example

Let's go to the tag<input>, with attributetype, and give it a blue background color:

<p> <label>Enter text</label> <input> </p> <p> <label>Enter number</label> <input type="number"> </p> input[type] { background-color: #C2DDFD; }
