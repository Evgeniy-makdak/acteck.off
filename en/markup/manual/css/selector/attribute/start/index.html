Selector by beginning of attribute value

The start attribute value selector applies a style to an element whose tag attribute value begins with specific text.

Example

Let's go to the tag<a>, whose attribute valuehrefstarts withhttpsand recolor it red:

<a href="file.html">link1</a> <a href="https://code.mu">link2</a> a[href^="https://"] { color: red; }
