Attribute selector by text

A text attribute selector applies a style to an element whose tag attribute value contains specific text.

Example

Let's find links containing in the attributestest, and recolor them green:

<a href="https://test.com">link1</a> <a href="https://code.mu">link2</a> <a href="http://my-test.com">link3</a> a[href*="test"] { color: green; }
