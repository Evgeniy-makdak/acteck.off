Universal selector in CSS

Universal selector*allows you to select all elements.

Syntax

* { }

Example

Let's select all the elements and color them red:

* { color: red; }

Example

Let's select all the elements inside the tagdiv, and color them red:

div * { color: red; }

Example

Let's select all elements directly inside the tagdiv, and color them red:

div > * { color: red; }

Example

Let's select all the elements inside the tagspan, which in turn is inside any tag, which in turn is inside a tagdiv:

div * span { color: red; }
