text-overflow property

Propertytext-overflowadds an ellipsis to the end of trimmed text to indicate that the text did not fit in the block and was trimmed.

For the property to work, the text must be trimmed through the propertyoverflowor propertyoverflow-xin meaninghidden, autoorscroll. If setvisible(and it is by default) -text-overflowwon't work.

Syntax

селектор { text-overflow: ellipsis | clip; }

Values

Meaning Description
ellipsis Adds an ellipsis to the end of the trimmed text.
clip Does not add an ellipsis at the end (this is the default, needed to cancel the property if necessary).

Default value:clip.

Example. Pop-out text

In this example, a very long word will not fit into the container and will fall outside its boundaries. No cutting occurs:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur loooooooooooooooooooooooooooooooongword adipiscing elit. </div> #elem { width: 200px; overflow: visible; border: 1px solid red; }

:

Example. Let's add the overflow property

Now everything that has gone beyond the boundaries of the container will simply be cut off:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur loooooooooooooooooooooooooooooooongword adipiscing elit. </div> #elem { width: 200px; overflow: hidden; border: 1px solid red; }

:

Example. Meaning of ellipsis

Now in addition to the propertyoverflowlet's also addtext-overflowin meaningellipsis. An ellipsis will be added to the end of the trimmed text:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur loooooooooooooooooooooooooooooooongword adipiscing elit. </div> #elem { width: 200px; overflow: hidden; text-overflow: ellipsis; border: 1px solid red; }

:

Example. With scroll bars

If you setoverflow: autoAndtext-overflow: ellipsis, the scroll bar will appear, but the ellipsis will still be added. Try scrolling the scrollbar in the example:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur loooooooooooooooooooooooooooooooongword adipiscing elit. </div> #elem { width: 200px; overflow: auto; text-overflow: ellipsis; border: 1px solid red; }

:

Example. If there are no very long words

If there are no words so long that they extend beyond the border of the container, then trimming will not occur (nothing will extend). Look at the following example:

<div id="elem"> Lorem ipsum dolor sit amet adipiscing elit. </div> #elem { width: 200px; overflow: hidden; text-overflow: ellipsis; border: 1px solid red; }

:

Example. One line text

However, there are situations when we want text to be cut off if it is too long (text in general, not individual words) and not wrap to the next line. This is done by adding a propertywhite-spacein meaningnowrap, which will prevent text from wrapping to another line. Look at the example, now the text is cut off:

<div id="elem"> Lorem ipsum dolor sit amet adipiscing elit. </div> #elem { width: 200px; overflow: hidden; white-space: nowrap; text-overflow: ellipsis; border: 1px solid red; }

:

Example. Width in percentage

If you set the block width in %, then cropping will also work correctly:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur loooooooooooooooooooooooooooooooongword adipiscing elit. </div> #elem { width: 80%; overflow: hidden; white-space: nowrap; text-overflow: ellipsis; border: 1px solid red; }

:
