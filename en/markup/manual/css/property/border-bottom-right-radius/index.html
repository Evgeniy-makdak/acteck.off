Свойство border-bottom-right-radius

Свойство border-bottom-right-radius создает скругление для нижнего правого угла границы и фона. Значением свойства служат любые единицы для размеров. Значение по умолчанию: 0.

Синтаксис

селектор { border-bottom-right-radius: значение; }

Пример

Установим скругление 10px для нижнего правого угла:

<div id="elem"></div> #elem { border-bottom-right-radius: 10px; border: 1px solid black; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-top-left-radius,
    которое задает скругление для верхнего левого угла
  • свойство border-top-right-radius,
    которое задает скругление для верхнего правого угла
  • свойство border-bottom-right-radius,
    которое задает скругление для нижнего левого угла
  • свойство border-radius,
    которое задает скругление одновременно для всех углов
  • свойство border,
    которое является свойством-сокращением для границы
