justify-items property

Propertyjustify-itemsspecifies the alignment of elements inside grid cells along the horizontal axis. The alignment can be seen most clearly when viewing the grid in the browser debugger. Applies to the parent element.

Syntax

селектор { justify-items: flex-start | flex-end | center ; }

Values

Meaning Description
flex-start The elements are pressed to the beginning of the horizontal axis.
flex-end The blocks are pressed against the end of the horizontal axis.
center The blocks are centered on the horizontal axis.

Example. Alignment to the origin of the horizontal axis

Let's align our elements inside the cells to the beginning of the horizontal axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: grid; justify-items: start; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); gap: 10px; padding: 10px; height: 200px; width: 400px; border: 2px solid #696989; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }
