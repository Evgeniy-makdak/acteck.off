background-repeat property

Propertybackground-repeatspecifies how to repeat the element's background image. By default, the image is repeated along both the X and Y axis, thus covering the entire available area.

Syntax

селектор { background-repeat: no-repeat | repeat | repeat-x | repeat-y | space | round; }

Values

Meaning Description
no-repeat The picture will not be repeated at all.
repeat-x The picture will be repeated along the X axis.
repeat-y The picture will be repeated along the Y axis.
repeat The picture will be repeated along the X-axis and Y-axis.
space The picture will be repeated so many times to completely fill the area; if this fails, empty space is added between the pictures.
round The picture will be repeated so that the area fits an integer number of pictures; if this cannot be done, then the background pictures will be scaled.

Default value:repeat- covers the entire screen with a pattern.

Example

By default, the background image will cover the entire element:

<div id="elem"></div> #elem { background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

Let's make sure the picture doesn't repeat itself:

<div id="elem"></div> #elem { background-repeat: no-repeat; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

Now let the picture repeat along the X axis:

<div id="elem"></div> #elem { background-repeat: repeat-x; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

And now along the Y axis:

<div id="elem"></div> #elem { background-repeat: repeat-y; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

Images that repeat along the axes can be positioned using the propertybackground-position:

<div id="elem"></div> #elem { background-repeat: repeat-y; background-position: top center; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

Let's see how the value worksspace:

<div id="elem"></div> #elem { background-repeat: space; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }

:

Example

Let's see how the value worksround:

<div id="elem"></div> #elem { background-repeat: round; background-image: url("bg.png"); width: 400px; height: 300px; border: 1px solid black; }
