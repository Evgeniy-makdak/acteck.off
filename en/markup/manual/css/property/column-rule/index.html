column-rule property

Propertycolumn-rulesets the border between columns in multi-column text. The accepted values ​​coincide with the propertyborder.

Example. Type solid

In this example, the text will be split into3columns with borders between columns in the form of a solid line with a thickness of1px(the distance between the columns will be generated automatically, as specifiedcolumn-gapin meaningnormal):

<div id="elem"> some long text </div> #elem { column-rule: 1px solid black; column-count: 3; column-gap: normal; text-align: justify; }

:

Example. Impact of column-gap

In this example, the text will be split into3columns with a gap between them of50px. The border will pass through the middle of the gap (to the right and left of the border there will be25px):

<div id="elem"> some long text </div> #elem { column-rule: 1px solid black; column-count: 3; column-gap: 50px; text-align: justify; }
