text-align-last property

Propertytext-align-lastsets the alignment of the last line of text: left, right, center, left and right simultaneously.

Note that if in an element (for example, a paragraph),only one line- then it is considered both the first and the last and its alignment will be as specified intext-align-last, not intext-align.

Syntax

селектор { text-align-last: center | left | right | justify | auto | start | end; }

Values

Meaning Description
center In the center.
left On the left edge.
right On the right edge.
justify Simultaneously both on the right edge and on the left.
auto Does not change the element's alignment.
start Equalsleft, if the text goes from left to right andright, when the text goes from right to left (Arabic, etc.).
end Equalsright, if the text goes from left to right andleft, when the text goes from right to left.
inherit Inherits the value of the parent.

Default value:auto.
