transition-duration property

Propertytransition-durationsets the duration of the smooth transitiontransition.

Syntax

селектор { transition-duration: время в s или ms; }

Values

Meaning Description
s Specifies the time in seconds (for example3s). You can specify fractional values, for example,0.5s- half a second.
ms Specifies the time in milliseconds (for example3ms). One second is1000milliseconds

Default value:0s.

Example

Hover your mouse over the block - it will smoothly change its width within2seconds.

<div id="elem"></div> #elem { transition-duration: 2s; transition-property: width; border: 1px solid black; height: 50px; width: 100px; } #elem:hover { width: 400px; }
