text-decoration-line property

Propertytext-decoration-linespecifies the position of the line of the text. The property should be used in conjunction with propertiestext-decoration-styleAndtext-decoration-color. These properties extend the effect of the propertytext-decoration, being more advanced analogues.

Syntax

селектор { text-decoration-line: underline | line-through | overline | none; }

Values

Meaning Description
underline Doesemphasizedtext.
line-through Doescrossed outtext.
overline Makes a lineabove the text.
none Undoes underlining of the underlined element. Most often used for links, as they are underlined by default.

Default value:none. For links the default value is:underline.

Example

Let's cross out the text with a red line:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-decoration-style: solid; text-decoration-color: red; text-decoration-line: line-through; }
