position property

Propertypositionspecifies the way elements are positioned. This property is most often used in conjunction with the propertiestop, right, bottom, left, which set the padding at the top, right, bottom, left, respectively.

Syntax

селектор { position: absolute | relative | fixed | static | sticky; }

Meanings for position

Meaning Description
absolute Absolute positioning. The element is removed from the normal flow of the document and positioned relative to its nearest positioned ancestor (if any) or relative to the browser window.
relative Relative positioning. The element is displaced from its normal position in the document flow, but the space it occupied remains reserved.
fixed Fixed positioning. The element is removed from the normal flow of the document and positioned relative to the browser window. Stays in place as you scroll the page.
static Static positioning. The value means there is no positioning and the element behaves as usual.
sticky Sticky positioning. The element behaves likerelativeuntil it reaches the specified scroll position, after which it sticks to the specified location (asfixed).

Default value:static.

Example. Absolute positioning

Let's place the element in the upper left corner of the screen with small indents:

<div class="elem"></div> <div class="main"> some long text </div> .elem { position: absolute; top: 40px; left: 50px; width: 100px; height: 100px; background-color: #98fb98; } .main { width: 400px; text-align: justify; margin: 20px auto; }

Example. Absolute positioning

Now let’s place the element in the upper right corner of the screen with small indents:

<div class="elem"></div> <div class="main"> some long text </div> .elem { position: absolute; top: 40px; right: 50px; width: 100px; height: 100px; background-color: #98fb98; } .main { width: 400px; text-align: justify; margin: 20px auto; }

:

Example. Fixed positioning

With fixed positioning, the element will remain in place when scrolled:

<div class="elem"></div> <div class="main"> some long text </div> .elem { position: fixed; top: 40px; left: 50px; width: 100px; height: 100px; background-color: #98fb98; } .main { width: 400px; text-align: justify; margin: 20px auto; }

:

Example. Relative positioning

With relative positioning, the element is offset from its normal position, but other elements behave as if the element had not been offset:

<div class="container"> <div class="elem elem1"></div> <div class="elem elem2"></div> <div class="elem elem3"></div> </div> .container { display: flex; justify-content: flex-row; } .elem { margin: 0 3px; width: 100px; height: 100px; background-color: #add8e6; } .elem2 { position: relative; top: 20px; left: 30px; background-color: #e6addf; }

:

Example. Nesting

If the parent has the propertypositionmattersrelative, then absolutely positioned elements will be positioned relative to the parent:

<div class="container"> <div class="elem"></div> </div> .container { position: relative; border: 1px solid red; width: 500px; height: 300px; } .elem { position: absolute; top: 0px; right: 0px; width: 100px; height: 100px; background-color: #add8e6; }

:

Example. Nesting

If the parent has the propertypositionmattersabsolute, then absolutely positioned elements will be positioned relative to the parent:

<div class="container"> <div class="elem"></div> </div> .container { position: absolute; top: 30px; left: 30px; border: 1px solid red; width: 500px; height: 300px; } .elem { position: absolute; top: 0px; right: 0px; width: 100px; height: 100px; background-color: #add8e6; }

:

Example. Sticky positioning

If the element has a propertypositionmatterssticky, then the element behaves likerelative, until it reaches the specified position when scrolling, after which it sticks in place. Let's make a sticky header:

<h1>Main Site Header</h1> <div class="header">header header header</div> <div class="main"> some long text </div> h1 { text-align: center; } .header { position: sticky; top: 0; padding: 20px; background: #f0f0f0; text-align: center; font-weight: bold; } .main { width: 400px; margin: 20px auto; border: 1px solid #ccc; padding: 20px; text-align: justify; }

:

Example. No position

If you give an element absolute positioning without specifying a position, then the element will remain in the place where it stood without positioning, but other elements will no longer notice it:

<div class="container"> <div class="elem elem1"></div> <div class="elem elem2"></div> <div class="elem elem3"></div> </div> .container { position: relative; border: 1px solid red; width: 500px; height: 300px; } .elem { display: inline-block; } .elem2 { position: absolute; width: 50px; height: 50px; background-color: #e6addf; } .elem1, .elem3 { width: 100px; height: 100px; background-color: #add8e6; }

:

Example. One axis

With absolute positioning, you can specify a position along only one axis. For example, if we set the propertytop, then the vertical element will be in the desired position, and the horizontal element will remain standing where it stood:

<div class="container"> <div class="elem elem1"></div> <div class="elem elem2"></div> <div class="elem elem3"></div> </div> .container { position: relative; border: 1px solid red; width: 500px; height: 300px; } .elem { display: inline-block; } .elem2 { position: absolute; top: 20px; width: 50px; height: 50px; background-color: #e6addf; } .elem1, .elem3 { width: 100px; height: 100px; background-color: #add8e6; }

:

Example. All positions

With absolute positioning, you can specify positions on all sides without specifying width and height. In this case, the element will be centered in the parent block:

<div class="container"> <div class="elem"></div> </div> .container { position: relative; border: 1px solid red; width: 500px; height: 300px; } .elem { position: absolute; top: 20px; bottom: 20px; left: 20px; right: 20px; background-color: #add8e6; }
