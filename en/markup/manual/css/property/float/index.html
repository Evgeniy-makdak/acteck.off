float property

Propertyfloat- sets floating blocks that will be wrapped with text. Propertyfloatsometimes everything is used in conjunction with the propertyclear, which cancels the element's wrapping.

Although initiallyfloatwas invented for inserting wraparound blocks into text, it is now used very widely and in creative ways to build layouts.

Syntax

селектор { float: left | right | none; }
селектор { clear: both | left | right | none; }

Meanings for float

Meaning Description
left The block will floatleft, and the text will wrap around itright.
right The block will floatright, and the text will wrap around itleft.
none There is no flow around.

Default value:none.

Meanings for clear

Meaning Description
left Cancels left wrapping.
right Cancels wrapping to the right.
both Cancels wrapping on both the left and right.
none There is no wrap cancellation.

Default value:none.

Example

We usefloatin a non-standard way, putting the list items in a line:

<ul> <li>1</li> <li>2</li> <li>3</li> <li>4</li> <li>5</li> </ul> li { float: left; border: 1px solid red; }
