caret-color property

Propertycaret-colorsets the color of the text cursor in the input, which appears when entering text. The property can take two values:auto(default) and user-selected color.

Syntax

селектор { caret-color: auto или цвет; }

Example

Let's set the default text cursor color:

<input> input { padding: 5px; caret-color: auto; }

:

Example

Now let's recolor the text cursor blue:

<input> input { padding: 5px; caret-color: blue; }
