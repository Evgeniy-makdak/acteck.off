Свойство caret-color

Свойство caret-color задает цвет текстового курсора в инпуте, который появляется при вводе текста. Свойство может принимать два значения: auto (по умолчанию) и выбранный пользователь цвет.

Синтаксис

селектор { caret-color: auto или цвет; }

Пример

Давайте зададим цвет текстового курсора по умолчанию:

<input> input { padding: 5px; caret-color: auto; }

:

Пример

А теперь давайте перекрасим текстовой курсор в синий цвет:

<input> input { padding: 5px; caret-color: blue; }

:

Смотрите также

  • псевдокласс :fullscreen,
    который задает стиль элементам в полноэкранном режиме
  • псевдокласс :indeterminate,
    который задает стиль переключателям, которые находятся в неопределенном состоянии
  • псевдокласс :focus-within,
    который задает стиль элементам в фокусе
