border-image-width property

Propertyborder-image-widthcontrols the width of the visible part of the frame and scales it. If this value is greater than widthborder-width, the frame image will crawl under the content.

For more details see propertyborder-image.

Syntax

селектор { border-image-width: CSS единицы | проценты | число | auto; }

Values

Meaning Description
CSS units Value in givenCSS units.
Interest Percentage values ​​relative to sizeblock, which is given a boundary.
Number Numeric value to be multiplied byborder-width.
auto Keyword. If given, the value is equal to the correspondingborder-image-slice. If there is no suitable size, the value is usedborder-width, while the picture fills the entire corner of the frame, crawling under the content. See examples for better understanding.

Default value:1.

Example. Number

Set the valueborder-image-widthV2by hovering the mouse over an element. In this case, the border will become the size26px*2- V2times larger than its width specified inborder-width. The border will go under the text (like a background):

<div id="elem"></div> #elem:hover { border-image-width: 2; } #elem { border-image-repeat: repeat; border-image-source: url("border-image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }

:

Example. Change border-image-repeat to round

In the previous hover example, the border will not contain an integer number of diamonds. Let's fix this by settingborder-image-repeatin meaninground:

<div id="elem"></div> #elem:hover { border-image-width: 2; } #elem { border-image-repeat: round; border-image-source: url("border-image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }
