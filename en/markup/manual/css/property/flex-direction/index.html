Flex-direction property

Propertyflex-directionsets the direction of the main and transverse axis or, in other words, arranges elements in a row or column.

Applies to the parent element for flex blocks. Included in the abbreviation propertyflex-flow.

Syntax

селектор { flex-direction: row | row-reverse | column | column-reverse; }

Values

Meaning Description
row The main axis is directed from left to right. The elements are arranged in a row, by default pressed to the left edge, their numbering is in the usual order -from left to right.
row-reverse The main axis is directed from right to left. The elements are arranged in a row, by default pressed to the right edge, their numbering is in reverse order -from right to left.
column The main axis is directed from top to bottom. The elements are arranged in a column, pressed to the top by default, their numbering is in the usual order -top down.
column-reverse The main axis is directed from bottom to top. The elements are arranged in a column, pressed to the bottom by default, their numbering is in reverse order -from bottom to top.

Default value:row.

Example

The elements are arranged in a row, by default pressed to the left edge, their numbering is in the usual order -from left to right:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { flex-direction: row; display: flex; } #parent > div { border: 1px solid #696989; width: 100px; height: 50px; }

:
