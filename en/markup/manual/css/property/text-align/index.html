text-align property

Propertytext-alignsets text alignment: left, right, center, left and right simultaneously.

Syntax

селектор { text-align: center | left | right | justify | auto | start | end; }

Values

Meaning Description
center The text will be centered.
left The text will be left aligned.
right The text will be aligned to the right.
justify The text will be aligned both right and left.
auto Does not change element alignment (default).
start Equalsleft, if the text goes from left to right, andright, when the text goes from right to left (Arabic, etc.).
end Equalsright, if the text goes from left to right andleft, when the text goes from right to left.

Default value:auto. In most cases, text is aligned by defaulton the left edge(in Arabic - to the right). However, there are exceptions, for example, text inthwill be centered.

Example. left value

<p> some long text </p> p { text-align: left; width: 400px; }
