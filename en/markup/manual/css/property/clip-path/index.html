clip-path property

Propertyclip-pathdefines the display area of ​​an element, cropping it to a specified shape. Takes a trim function or SVG path as a parameter.

Syntax

селектор { clip-path: функция обрезки; }

Values

Meaning Description
circle Trims an element in a circle. Parameters: radius and center position.
ellipse Trims an element to an ellipse. Parameters: X/Y radii and center position.
polygon Clips an element to a polygon. Parameters: coordinates of points separated by commas.
url Uses the SVG path for trimming. Parameter: SVG path ID.
path Trims an element along an SVG path using the path data string. Parameter: SVG path string.
inset Trims the element into a rectangle with the option of rounding the corners. Parameters: top, right, bottom, left margins and rounding radius.
margin-box Uses the outer margin of the element as the cropping area.
border-box Uses the element's border as the cropping area.
padding-box Uses the element's padding as the clipping area.
content-box Uses the element's content as the clipping area.
none Disables cropping (default).

Example

Let's cut the element to a hexagon:

<div id="hexagon"></div> #hexagon { width: 200px; height: 200px; background: #3498db; clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0% 75%, 0% 25%); }
