background-image property

Propertybackground-imagesets the background image for the element. By default, the image will cover the entire block with its copies, however, this behavior can be canceled using the propertybackground-repeat.

Syntax

селектор { background-image: url(путь к картинке); }
селектор { background-image: none; }

Values

Meaning Description
url Path to the file with the image. The name of the picture can be in double quotes, single quotes or without quotes at all.
none Overrides the background image for an element.

Default value:none.

Example

Let's set the background image usingbackground-image, prohibiting its repetition using the propertybackground-repeat:

<div id="elem"></div> #elem { background-image: url("bg.png"); background-repeat: no-repeat; width: 400px; height: 300px; border: 3px solid black; }

:

Example

Withoutbackground-repeatthe background image will cover the entire block:

<div id="elem"></div> #elem { background-image: url("bg.png"); width: 400px; height: 300px; border: 3px solid black; }

:

Example

You can set the background image and set the background color at the same time usingbackground-color. In this case, where there is no background image, there will be a background color:

<div id="elem"></div> #elem { background-color: orange; background-image: url("bg.png"); background-repeat: no-repeat; width: 400px; height: 300px; border: 3px solid black; }

:
