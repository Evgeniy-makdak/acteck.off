columns property

Propertycolumnssets both the width and number of columns in multi-column text. Is a shorthand property forcolumn-countAndcolumn-width. Default value:auto auto (?).

Syntax

селектор { columns: ширина количество; }

The order of the properties does not matter. One of them may be omitted.

See also

  • propertycolumn-gap,
    which specifies the space between columns
  • propertycolumn-rule,
    which sets the boundary between columns
  • propertycolumn-span,
    which specifies the number of columns the element should stretch across
