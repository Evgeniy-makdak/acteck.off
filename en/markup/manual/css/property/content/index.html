Content property

Propertycontentallows you to insert arbitrary generated text into tags. Used in conjunction with pseudo-elementsafterAndbefore.

Syntax

селектор::after { content: 'текст'; }
селектор::before { content: 'текст'; }

Example

Let's go to the beginning and end of the tag textliinsert additional text:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li::before { content: '!!!'; } li::after { content: '???'; }

:

Example

You can apply different styles to pasted text. Let's paint, for example, this text some color:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li::before { color: red; content: '!!!'; } li::after { color: green; content: '???'; }
