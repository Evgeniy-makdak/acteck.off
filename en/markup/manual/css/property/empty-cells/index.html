empty-cells property

Propertyempty-cellstells the browser how to display a background and border that are emptycells tdorcells th HTML tables: show or not.

A cell is considered empty in the following cases: there are no characters at all, the cell contains only a space (one or more), a line feed or tab character, or a propertyvisibilityset ashidden.

To make a cell not empty, but without visible text, use the following technique: write a non-breaking space into the cell&nbsp;.

Property does not work if setborder-collapsein meaningcollapse.

Syntax

селектор { empty-cells: show | hide; }

Values

Meaning Description
show The background and border are shown on an empty cell.
hide The background and border are not shown for an empty cell.

Default value:show.

Example. show meaning

Now in the table some cells are empty, but they still have a border and background:

<table> <tr> <td>cell</td> <td></td> <td>cell</td> </tr> <tr> <td></td> <td>cell</td> <td></td> </tr> <tr> <td>cell</td> <td>cell</td> <td>cell</td> </tr> </table> table { width: 400px; empty-cells: show; } td { border: 1px solid black; background-color: #f3f3f3; text-align: center; }

Example. Hide value

But now empty cells will not have a background or border:
