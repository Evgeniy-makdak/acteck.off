grid-column-end property

Propertygrid-column-endspecifies the final position of an element in a grid or mesh up to a specific column. The property value can be a positive or negative number. If we specify a positive number, then the final position of the element is counted from left to right. If you specify a negative number, the element will be placed in the reverse order, i.e. from right to left.

An important nuance of the propertygrid-column-endis that the number of the specified column is not included in the final position of the element - if we specify a number3, then the element will only occupy the first and second columns.

Syntax

селектор { grid-column-end: положительное или отрицательное число; }

Example

Let's give the elements in the grid end positions:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #parent { display: grid; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-column-start: 1; grid-column-end: 2; } #elem2 { grid-column-start: 1; grid-column-end: 3; } #elem3 { grid-column-start: 1; grid-column-end: 4; }

Example

When columns occupied by adjacent elements overlap, each subsequent element is moved one row lower. Let's use this feature to make sure that the first element is located in the first row, the second in the second, and the third and fourth in the third row:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> <div id="elem4">4</div> </div> #parent { display: grid; padding: 10px; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-column-start: 1; grid-column-end: 4; } #elem2 { grid-column-start: 2; grid-column-end: 3; } #elem3 { grid-column-start: 1; grid-column-end: 2; } #elem4 { grid-column-start: 3; grid-column-end: 4; }
