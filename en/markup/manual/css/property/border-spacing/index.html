border-spacing property

Propertyborder-spacingsets the space between cellstdor cellsth HTML tables(and between the border of the cell and the table itself). The property value is anyunits for sizesexcept interest.

Syntax

селектор { border-spacing: одно или два значения; }

Values

Meaning Description
one value One value specifies equal vertical and horizontal space between cells.
two meanings The first value specifies the horizontal padding, and the second value specifies the vertical padding.

Default value:0. However, each browser has its own non-zero attribute valuecellspacing, so by default you will see padding between cells.

Notes

Apply the property to the table, not to its cells (it will not work for cells).

A similar effect cannot be achieved usingmargin, becausemargindoes not work for table cells.

If the property is setborder-collapsein meaningcollapse - border-spacingwon't work.

Example. One value

Let's add some padding between cells (and between the cell and the table border) in10px:

<table> <tr> <td>1</td> <td>2</td> <td>3</td> </tr> <tr> <td>4</td> <td>5</td> <td>6</td> </tr> <tr> <td>7</td> <td>8</td> <td>9</td> </tr> </table> table { border-spacing: 10px; border-collapse: separate; width: 400px; border: 1px solid red; } td { border: 1px solid red; text-align: center; }

:

Example. Two meanings

Now let's indent10pxhorizontally and30pxvertical:

<table> <tr> <td>1</td> <td>2</td> <td>3</td> </tr> <tr> <td>4</td> <td>5</td> <td>6</td> </tr> <tr> <td>7</td> <td>8</td> <td>9</td> </tr> </table> table { border-spacing: 10px 30px; border-collapse: separate; width: 400px; border: 1px solid red; } td { border: 1px solid red; text-align: center; }

:

Example. Set border-collapse: collapse

And nowborder-spacingwill not work due to the propertyborder-collapsein meaningcollapse:

<table> <tr> <td>1</td> <td>2</td> <td>3</td> </tr> <tr> <td>4</td> <td>5</td> <td>6</td> </tr> <tr> <td>7</td> <td>8</td> <td>9</td> </tr> </table> table { border-collapse: collapse; border-spacing: 10px 30px; width: 400px; border: 1px solid red; } td { border: 1px solid red; text-align: center; }
