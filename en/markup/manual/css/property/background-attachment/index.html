background-attachment property

Propertybackground-attachmentspecifies how to scroll the background image of the element: together with the text, or the text will slide over the image.

Syntax

селектор { background-attachment: fixed | scroll | local; }

Values

Meaning Description
fixed The background image will be stationary, and the text will slide across it.
scroll The background image will scroll along with the text.
local The background is fixed based on the behavior of the element. If the element has a scroll, the background will scroll along with the content, but the background outside the element will remain in place.

Default value:scroll.

Example. Scroll value

Now propertybackground-attachmentset toscroll. Scroll the element vertically - you'll see the text scroll along with the background:

<body> <div id="elem"> some long text </div> </body> body { background-attachment: scroll; background-image: url("bg.png"); } #elem { width: 400px; margin: 0 auto; text-align: justify; font-weight: bold; font-size: 20px; }

:

Example. fixed value

And now the propertybackground-attachmentset tofixed. Scroll the element vertically and you'll see the text sliding across the background:

<body> <div id="elem"> some long text </div> </body> body { background-attachment: fixed; background-image: url("bg.png"); } #elem { width: 400px; margin: 0 auto; text-align: justify; font-weight: bold; font-size: 20px; }
