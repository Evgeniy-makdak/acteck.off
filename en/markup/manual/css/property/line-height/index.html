Line-height property

Propertyline-heightsets the spacing between lines of text (line spacing).

The property does not specify the space between lines of text, as it might seem, it specifiestext line height. This means that the actual space between lines will be calculated like this:line-height - font-size= distance between lines of text. Or vice versaline-height = font-size+ distance between lines of text.

Syntax

селектор { line-height: значение; }

Values

The property value is anyunits for sizes. When specifying a value in %, the line spacing will be a percentage of the font size.

In addition, you can set the value to any number greater than zero. In this case, it is perceived as a multiplier of the font size. For example, iffont-sizematters20px, Aline-height - 1.5, then it’s the same as writingline-height: 30px (20px * 1.5 = 30px).

By default the property has the valuenormal, in this case the browser selects the line spacing automatically.

Example

In this example, the distance between lines of text will beline-height - font-size = 38px - 18px = 20px:

<p> some long text </p> p { font-size: 18px; line-height: 35px; text-align: justify; }
