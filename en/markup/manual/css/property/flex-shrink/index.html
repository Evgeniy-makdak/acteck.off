flex-shrink property

Propertyflex-shrinkdetermines how much the flex block will shrink relative to adjacent elements inside the flex container if there is not enough free space.

For example, if all flex boxes inside a flex container haveflex-shrink:1, then they will be the same size. If one of them hasflex-shrink:2, then he will be in2times less than everyone else.

Applies to:specific flex block.

This property is included as an integral part of the abbreviation propertyflex.

Syntax

селектор { flex-shrink: положительное число; }

Default value:1.

Example

Let us have3element. The width of each of them is200pxand the total is equal600px, which is greater than the width of the parent container, which is equal to350px:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> <div class="child elem2">3</div> </div> .parent { display: flex; width: 350px; height: 200px; border: 1px solid red; } .child { width: 200px; height: 50px; border: 1px solid green; } .elem1 { flex-shrink: 1; } .elem2 { flex-shrink: 2; } .elem3 { flex-shrink: 3; }

:

Let's calculate the negative free space using the formula:

600px - 350px = 250px

Total weighted width of elements based on property valuesflex-shrinkfor each element will be equal to:

200px * 1 + 200px * 2 + 200px * 3 = 1200px

Now let's determine how much the first element will be reduced:

250px * 1 (flex-shrink) * 200 (width) / 1200 = 41,6px

Second element:

250px * 2 (flex-shrink) * 200 (width) / 1200 = 83,3px

Third element:

250px * 3 (flex-shrink) * 200 (width) / 1200 = 125px

It turns out that the third element, whose valueflex-shrinkequals3will shrink more than other elements.
