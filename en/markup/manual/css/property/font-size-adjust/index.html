font-size-adjust property

Propertyfont-size-adjustallows you to control the font size, while keeping it constantfont-size(increase or decrease).

Application: let infont-familytwo fonts are specified, separated by commas, for example,Georgia, "Times New Roman". Let the user's computer not have the Georgia font - in this case, "Times New Roman" will be used. However, a problem awaits us - with the same valuefont-sizefonts will look different. In the example below, both paragraphs are givenfont-sizeV16px, but differentfont-family. See how the sizes differ:

<p style="font-size: 16px; font-family: Georgia;"> Lorem ipsum dolor sit amet. </p> <p style="font-size: 16px; font-family: 'Times New Roman';"> Lorem ipsum dolor sit amet. </p>

:

This occurs because the displayed font size does not depend largely on the propertyfont-size, which determines only the overall font size, and from the ratio of property valuesfont-sizeand magnitudex-height (x-heightis the difference between the size of a lowercase letter "x" and an uppercase letter "X" for a particular font).

This ratio is calledaspectfont and is calculated by the formula:aspect=font-size/x-height.

The size problem is solved by the propertyfont-size-adjust(it allows you to change the aspect of the font) which will make the second font appear to be the same size as the first.

Let's make Times New Roman the same size as Georgia. We know the aspect of Georgia - this0.5(see table below).

Let's installfont-size-adjustfor text with Times New Roman in value0.5and we will see that the second text has become the same size as the first:

<p style="font-size: 16px; font-family: Georgia;"> Lorem ipsum dolor sit amet. </p> <p style="font-size: 16px; font-family: 'Times New Roman'; font-size-adjust: 0.5;"> Lorem ipsum dolor sit amet. </p>

:

Syntax

селектор { font-size-adjust: число | none; }

Values

Meaning Description
number The number specifies the aspect of the font.
none No font size adjustment.

Default value:none.

How to determine the aspect value of a font?

Check out this excerpt from the W3C specification:The proportionality factor (aspect) for selected fonts can be calculated by comparing the same text but with different valuesfont-size-adjust. If the property value is selected correctly, then with the same font size, the text will remain unchanged for all fonts used on the page.

Here are the aspect values ​​known to me for some fonts: Georgia -0.5, Times New Roman - 0.46, Verdana - 0.58.

Example

Compare what text looks like with different meaningsfont-size-adjustand the samefont-sizeAndfont-family:

<p style="font-size: 16px; font-family: 'Times New Roman'; font-size-adjust: none;"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p> <p style="font-size: 16px; font-family: 'Times New Roman'; font-size-adjust: 0.5;"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p> <p style="font-size: 16px; font-family: 'Times New Roman'; font-size-adjust: 0.6;"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
