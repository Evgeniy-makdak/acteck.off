border-left-color property

Propertyborder-left-colorsets the color of the left border. The property value is anyunits for color. Default value: same color as the block text.

Syntax

селектор { border-left-color: цвет; }

Example

Let's set the left border to red:

<div id="elem"></div> #elem { border-left-color: red; border-left-width: 1px; border-left-style: solid; background: #f9f9f9; width: 300px; height: 100px; }
