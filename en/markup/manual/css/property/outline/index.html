outline property

Propertyoutlinesets the color, type, and thickness for the outer border (a border that doesn't take up space) for all sides at the same time, but unlikebordersuch a border does not extend the element. Is a shorthand property foroutline-width, outline-styleAndoutline-color(but not foroutline-offset).

Syntax

селектор { outline: толщина тип цвет; }

The order of the properties does not matter.

Example. Outline only

<div id="elem"></div> #elem { outline: 1px solid black; width: 300px; height: 100px; }

:

Example. Outline and border properties together

Border fromoutlinewill be outside, and the border will be fromborder- inside:

<div id="elem"></div> #elem { outline: 4px solid green; border: 4px solid red; width: 300px; height: 100px; }

:
