grid-template property

Propertygrid-templatespecifies the number and width of rows and columns that an element will occupy in a grid or mesh and is a short form of propertiesgrid-template-rowsAndgrid-template-columns. The rows and columns on which the element will be placed are indicated by a slash. Propertygrid-templateis specified in the parent element and determines the location of child elements. In the property value we indicate the width of rows or columns in anyunits for sizes.

When specifying values ​​in pixels in properties, the sizes of elements will correspond exactly to them. If we give the wordauto, then the columns and rows will fill all available space. Unit usagefr(fraction) means that the entire space will be divided into equal parts. Advantagefris its adaptability to different containers or screen resolutions, sincefrsimply divides them into the specified number of fractions without reference to the exact size in pixels.

Syntax

селектор { grid-template: ширина и количество рядов / ширина и количество столбцов; }

Example

Let's use the propertygrid-templateLet's make a table:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> <div>6</div> <div>7</div> <div>8</div> <div>9</div> </div> #parent { display: grid; grid-template: 1fr 1fr 1fr / 1fr 1fr 1fr; border: 2px solid #696989; padding: 10px; width: 400px; height: 400px; } #parent > div { padding: 10px; border: 1px solid #696989; }

Example

Now let's give the second and third rows the same width, and each column a different width:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> <div>6</div> <div>7</div> <div>8</div> <div>9</div> </div> #parent { display: grid; grid-template: 60px 1fr 60px / 20% 1fr 15%; border: 2px solid #696989; padding: 10px; width: 400px; height: 300px; } #parent > div { padding: 10px; border: 1px solid #696989; }

Example

Now in the table from the previous example, let’s make the top row two fractions wide, and the first column half a fraction wide:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> <div>6</div> <div>7</div> <div>8</div> <div>9</div> </div> #parent { display: grid; grid-template: 2fr 1fr 1fr / 0.5fr 1fr 1fr; border: 2px solid #696989; padding: 10px; width: 400px; height: 400px; } #parent > div { padding: 10px; border: 1px solid #696989; }
