background-clip property

Propertybackground-clipdefines how the background fill or background image will be placed relative to the element: below the border, only insidepaddingor just above the content.

Peculiarities

  • When usingborder-boxwith transparent borders the background will be visible underneath them
  • Meaningtextrequires vendor prefixes for full support
  • The property works with gradients and multiple backgrounds
  • Atborder-radiusbackground is cropped at rounded corners

Browser support

All modern browsers supportborder-box, padding-boxAndcontent-box. Meaningtextrequires a prefix-webkit-and is supported in Chrome, Safari, Edge.

Syntax

селектор { background-clip: border-box | padding-box | content-box | text; }

Values

Meaning Description
border-box The background extends to the outer edge of the border (under the border).
padding-box The background is trimmed along the inner edge of the border (does not go underborder).
content-box The background is only shown above the content (cut offpadding).
text The background is cropped to fit the text.

Default value:border-box.

Example. border-box meaning

The background goes under the semi-transparent border:

<div id="elem"></div> #elem { background-clip: border-box; background-color: #ffd6d6; background-repeat: no-repeat; border: 10px dashed rgba(0,0,0,0.3); padding: 30px; width: 250px; height: 150px; }

:

Example.Padding-box value

The background does not go under the border:

<div id="elem"></div> #elem { background-clip: padding-box; background-color: #ffd6d6; background-repeat: no-repeat; border: 10px dashed rgba(0,0,0,0.5); padding: 30px; width: 250px; height: 150px; }

:

Example. Content-box meaning

The background doesn't fitpadding:

<div id="elem"> text text text text text text text text text text text text text text text </div> #elem { background-clip: content-box; background-color: #ffd6d6; background-repeat: no-repeat; border: 10px dashed rgba(0,0,0,0.3); padding: 30px; width: 250px; height: 150px; }

:

Example. Text value

The background is cropped to the text (required-webkit-text-fill-colorin meaningtransparent):

<div id="elem">text text text</div> #elem { display: inline-block; background: linear-gradient(to right, red, blue); -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text; font-size: 40px; font-weight: bold; }
