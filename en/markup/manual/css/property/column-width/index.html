column-width property

Propertycolumn-widthsets the width of columns in multi-column text. The property value is anyunits for sizes, except percent, or keywordauto(default). When valueautoThe browser itself selects the optimal column width based oncolumn-countAndcolumn-gap.

Syntax

селектор { column-width: значение; }

Example. Specified width

In this example, the column width is specifiedcolumn-widthV150px, and their numbercolumn-countstands in meaningauto. As a result, the width of the columns will be150px, their number and the interval between them will be selected by the browser itself:

<div id="elem"> some long text </div> #elem { column-count: auto; column-width: 150px; text-align: justify; }

:

Example. Auto value

In this example, the text will be split into3columns (sincecolumn-widthstands in meaning3), but their width will be calculated automatically:

<div id="elem"> some long text </div> #elem { column-width: auto; column-count: 3; text-align: justify; }

:
