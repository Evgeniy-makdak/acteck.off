font-style property

Propertyfont-stylesets italics or slanted text.

Syntax

селектор { font-style: italic | oblique | normal; }

Values

Meaning Description
italic Doesitalictext.
oblique Italic text. Unlike italic, which mimics handwriting, italic is normal text with the letters slanted to the side.
normal Cancels italics or italics. Usually necessary if you need to cancel italics for a certain section of text (for example, all paragraphs are made in italics, but this one is not).

Default value:normal.

Example. Italic meaning

The text will now be italicized:

<p> Lorem ipsum dolor sit amet. </p> p { font-style: italic; }

:

Example. Oblique meaning

The text will now become italic (depending on the browser, it may be indistinguishable from italic):

<p> Lorem ipsum dolor sit amet. </p> p { font-style: oblique; }

:
