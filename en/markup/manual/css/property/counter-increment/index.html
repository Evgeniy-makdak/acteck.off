counter-increment property

Propertycounter-incrementsets automatic numbering of elements, such as paragraphs or headings. Used in conjunction with the propertycounter-reset, pseudo-elementsafterAndbefore, propertycontent, inside which the function is usedcounter. For a better understanding, I recommend looking at examples.

Syntax

селектор { counter-increment: название [шаг] | none; }

Values

Meaning Description
Name Counter name. A regular word (like a class name or id). You can specify multiple names, separating them with a space. In this case, several counters will change simultaneously.
step A positive or negative integer. Optional parameter.
none Disables incrementing of the counter for the current selector.

Default value:none.

Example

Let's make paragraphs numbered automatically:

<div id="parent"> <p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> </div> #parent { counter-reset: test; } #parent p::before { counter-increment: test; content: counter(test); }

:

Example

Let some text be added in addition to numbering. In our case№before the number and a dot after:

<div id="parent"> <p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> </div> #parent { counter-reset: test; } #parent p::before { counter-increment: test; content: "№" counter(test) "."; }

:

Example

Let's start numbering with10-ti. To do this, set the initial value of the counter to nine, that is, to1less than we need:

<div id="parent"> <p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> </div> #parent { counter-reset: test 9; } #parent p::before { counter-increment: test; content: "№" counter(test) "."; }

:

Example

Now let the numbering be in increments "2". To do this, put a two as a step:

<div id="parent"> <p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> </div> #parent { counter-reset: test; } #parent p::before { counter-increment: test 2; /* ставим двойку */ content: "№" counter(test) "."; }

:

Example

In the previous example, the numbering started from two, but we would like to start from one. Why did this happen? Becausecounter-resetresets the counter to zero and thencounter-incrementadds its own step: the default is one, so previously our numbering began with1. And now a two is added - and the numbering begins with a two.

To fix the problem, set the initial value of the counter to-1and now the numbering will start from1and will increase by2:

<div id="parent"> <p>text</p> <p>text</p> <p>text</p> <p>text</p> <p>text</p> </div> #parent { counter-reset: test -1; } p::before { counter-increment: test 2; content: "№" counter(test) "."; }
