text-transform property

Propertytext-transformsets the case of text characters: uppercase (capital letters), lowercase (lowercase small letters), can capitalize the first letter of each word.

Syntax

селектор { text-transform: capitalize | lowercase | uppercase | none; }

Values

Meaning Description
capitalize The first character of each word in a sentence will be capitalized.
lowercase All text characters are converted tolower case.
uppercase All text characters are converted toUPPER CASE.
none Does not change character case.

Default value:none.

Example. Capitalize value

Now the first letter of each word will be capitalized:

<p> Lorem ipsum dolor sit amet. </p> p { text-transform: capitalize; }

:

Example. Meaning of uppercase

Now all characters will be capitalized:

<p> Lorem ipsum dolor sit amet. </p> p { text-transform: uppercase; }

:

Example. Meaning of lowercase

Now all characters will become lowercase (that is, small, not uppercase):

<p> LOREM IPSUM DOLOR SIT AMET. </p> p { text-transform: lowercase; }
