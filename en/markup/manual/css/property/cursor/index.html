cursor property

Propertycursorsets the appearance of the cursor when hovering over an element.

Syntax

селектор { cursor: значение; }

Note

The property has many meanings. I have grouped these values ​​below to make it easier to navigate. The appearance of the cursor differs in different browsers, so I did not take photos of the cursors, but simply made a special column in the tables with values, hovering over the cell of which you can see the appearance of the described cursor.

Values

Meaning Description Appearance
default Standard appearance in the form of an arrow pointer.
pointer Used to indicate that the element is active and clickable (this is the default for links).
help Used to show that there is some kind of help or question associated with an item.
not-allowed Used to indicate that an operation is unacceptable.

Text

Meaning Description Appearance
text Standard text cursor in the form of a vertical stick.
vertical-text Text cursor for vertical text.

Expectation

Meaning Description Appearance
progress Used to indicate that some operation is in progress and the user needs to wait.
wait Used to indicate that the user needs to wait.

Cursor when resizing

When changing the size of an element, the property value is formed as follows: wordresize, and in front of it, separated by a hyphen, is some letter or several letters, for example:n-resizeorne-resize.

These letters are the first letters of the cardinal directions: north - north, south - south, east - east, west - west. In this case, the conventions adopted for terrain maps are used: north at the top, south at the bottom, west on the left, east on the right. Of course, this is confusing, but you have to use it as intended :)

So here are the meanings:

w-resize left west

Resizing two elements

Meaning Description Appearance
col-resize Change horizontally.
row-resize Vertical change.

Changing the scale

Meaning Description Appearance
zoom-in Zooming in.
zoom-out Zoom out.

Movement

Meaning Description Appearance
move Used to indicate that an element can be moved, or when it is being moved.
all-scroll Used to indicate that the element is scrollable by the mouse in all directions.
