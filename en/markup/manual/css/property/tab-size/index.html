tab-size property

Propertytab-sizesets the amount of indentation created by a keyTab.

Works only if property is setwhite-spacein meaningpreorpre-wrap, and also for the tagpre.

The value of the property takes a positive integer that specifies the number of spaces that the key will correspond toTab. Default value:8spaces.

Syntax

селектор { tab-size: целое положительное число; }

Example

Let's set the indentationTab, equal0spaces:

<div id="elem"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </div> #elem { tab-size: 0; white-space: pre; border: 1px solid black; }

:

Example

Set the Tab indent equal to1space:

<div id="elem"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </div> #elem { tab-size: 1; white-space: pre; border: 1px solid black; }

Let's set the indentationTab, equal2-th spaces:

<div id="elem"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. </div> #elem { tab-size: 2; white-space: pre; border: 1px solid black; }
