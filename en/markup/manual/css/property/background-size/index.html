background-size property

Propertybackground-sizesets the size of the background image. The property value is anyunits for sizes, or keywordsauto, coverorcontain.

Syntax

селектор { background-size: значение; }

Keywords

Meaning Description
auto The background will have a natural size, the same as the real size of the background image. Ifautois specified only for one side, then on this side the background will be scaled so as to have undistorted proportions.
cover Scales the image so that it covers the entire block while maintaining its proportions. The picture will try to fit entirely, but this will not always work, so some part of it will be cut off. The block will always be covered by the entire image.
contain Scales the image so that it fits entirely into the block while maintaining its proportions. In this case, it can occupy either the entire width or the entire height (depending on the proportions of the image and the size of the element). In the general case, the block will not be completely covered by the image (but the entire image will always be visible, albeit in a reduced version).

Default value:auto.

Usage

Units for sizes andautocan be used in various combinations, for example like this:auto 20px, or30% 20px, orauto 30%and so on. In this case, the first parameter sets the background width size, and the second parameter sets the background height size. If one parameter is specified, it will set the background size in both width and height at the same time.

Example

Now the background image will have its natural size:

<div id="elem"></div> #elem { background-repeat: no-repeat; background-image: url("bg.png"); border: 1px solid black; width: 550px; height: 400px; }

:

Example

Now the background image will be the size300pxon400px(in our case, the proportions of the picture will be distorted):

<div id="elem"></div> #elem { background-size: 300px 400px; background-repeat: no-repeat; background-image: url("bg.png"); border: 1px solid black; width: 400px; height: 400px; }

:

Example

Now the background image will be the size400pxon400px(in our case, the proportions of the picture will be distorted):

<div id="elem"></div> #elem { background-size: 400px; background-repeat: no-repeat; background-image: url("bg.png"); border: 1px solid black; width: 400px; height: 400px; }

:

Example

Now the background image will be the size400pxhorizontally, and vertically its size will be adjusted so that the proportions are not distorted:

<div id="elem"></div> #elem { background-size: 400px auto; background-repeat: no-repeat; background-image: url("bg.png"); border: 1px solid black; width: 400px; height: 400px; }

:

Example

Now the background image will be the size400pxvertically, and horizontally its size will be adjusted so that the proportions are not distorted:

<div id="elem"></div> #elem { background-size: auto 400px; background-repeat: no-repeat; background-image: url("bg.png"); border: 1px solid black; width: 400px; height: 400px; }

:
