table-layout property

In some cases, the browser may shrink cells even if they are given a width. Propertytable-layoutallows you to instruct the browser that table cells should always be of a given width.

Syntax

селектор { table-layout: fixed | auto | inherit; }

Values

Meaning Description
auto The browser can resize cells.
fixed The browser will not resize the cells.
inherit Inherits the value of the parent.

Default value:auto.
