font-weight property

Propertyfont-weightsets or deselects bold text.

Syntax

селектор { font-weight: bold | normal; }

Values

Meaning Description
bold Doesfattytext.
normal Cancels bold. Usually necessary if you need to make a tag that is bold by default (for example, headings) non-boldh1-h6orth).

Default value:for each element its own- Paragraphs are not bold by default, but headings are bold.

Example

Let's set the bold font for the paragraph:

<p> Lorem ipsum dolor sit amet. </p> p { font-weight: bold; }

:

Example

Let's cancel the bold font for the title (it's bold by default):

<h2>Lorem ipsum</h2> h2 { font-weight: normal; }
