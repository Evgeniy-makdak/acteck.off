border-color property

Propertyborder-colorsets the border color for all sides at the same time or separately for each side. The property value is anyunits for color. Default value: same color as the block text.

The property is a shorthand property for the following properties:border-left-color, border-right-color, border-top-color, border-bottom-color.

Syntax

селектор { border-color: цвет; }

Number of values

The property can take1, 2, 3or4values ​​specified separated by spaces:

Quantity Description
1 Type for all sides at the same time.
2 The first value is for the top and bottom, the second is for the left and right borders.
3 The first value is for the top, the second is for the left and right borders, the third is for the bottom.
4 The first value is for the top border, the second - for the right, the third - for the bottom, the fourth - for the left border.

Example

Let's color the block border red:

<div id="elem"></div> #elem { border-width: 1px; border-style: solid; border-color: red; width: 300px; height: 100px; }

:

Example

Let's set a red border for the top and bottom sides, and a blue border for the right and left sides:

<div id="elem"></div> #elem { border-width: 1px; border-style: solid; border-color: red blue; width: 300px; height: 100px; }

:

Example

Let's set a red border for the top side, blue for the right side, green for the bottom, black for the left side:

<div id="elem"></div> #elem { border-width: 1px; border-style: solid; border-color: red blue green black; width: 300px; height: 100px; }

:

Example

Let's simultaneously set different thickness, type and color of the border:

<div id="elem"></div> #elem { border-width: 1px 2px 3px 4px; border-style: dashed dotted solid double; border-color: red blue green black; width: 300px; height: 100px; }

:

Example

If the border color is not specified inborder-color- it will be taken from the propertycolor:

<div id="elem"></div> #elem { color: red; border-style: solid; border-width: 2px; width: 300px; height: 100px; }
