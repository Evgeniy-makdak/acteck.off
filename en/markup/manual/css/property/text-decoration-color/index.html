Свойство text-decoration-color

Свойство text-decoration-color задает цвет линии у текста. Свойство следует использовать совместно со свойствами text-decoration-line и text-decoration-style. Эти свойства расширяют действие свойства text-decoration, являясь более продвинутыми аналогами. Значением свойства служат любые единицы для цвета.

Синтаксис

селектор { text-decoration-color: цвет в любом CSS формате; }

Значение по умолчанию: такой же цвет, как и у текста.

Пример

Сейчас у текста будет подчеркивание снизу красного цвета, так как задано свойство text-decoration-color в значении red:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-decoration-style: solid; text-decoration-color: red; text-decoration-line: underline; }
