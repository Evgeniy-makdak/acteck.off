text-decoration-color property

Propertytext-decoration-colorsets the line color of the text. The property should be used in conjunction with propertiestext-decoration-lineAndtext-decoration-style. These properties extend the effect of the propertytext-decoration, being more advanced analogues. The property value is anyunits for color.

Syntax

селектор { text-decoration-color: цвет в любом CSS формате; }

Default value: same color as the text.

Example

Now the text will have a red underline at the bottom, since the property is settext-decoration-colorin meaningred:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-decoration-style: solid; text-decoration-color: red; text-decoration-line: underline; }
