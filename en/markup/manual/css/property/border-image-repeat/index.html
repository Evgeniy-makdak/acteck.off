border-image-repeat property

Propertyborder-image-repeatspecifies how to repeat the inside of the border as a picture. For more details see propertyborder-image.

Syntax

селектор { border-image-repeat: stretch | repeat | round; }

Values

Number of parameters Description
stretch Stretches the border drawing to the size of the element. This is the default value.
repeat Repeats the border pattern.
round Repeats the image and scales it so that there is an integer number of images on the element side.

Default value:stretch.

Number of parameters

Can take1or2parameter. If two parameters are given, then the first one will be for the top and bottom border, and the second parameter will be for the left and right.

Example. Meaning of stretch

<div id="elem"></div> #elem { border-image-source: url("image.png"); border-image-slice: 26; border-image-repeat: stretch; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }

:

Example. repeat meaning

<div id="elem"></div> #elem { border-image-source: url("image.png"); border-image-slice: 26; border-image-repeat: repeat; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }

:

Example. Meaning of round

Now in normal state the value is set torepeat, and on guidance -round. Please note that before hovering, not a whole number of diamonds are placed in the border, but after hovering - a whole number. That's how it worksround:

<div id="elem"></div> #elem:hover { border-image-repeat: round; } #elem { border-style: solid; border-width: 42px; border-image-source: url("image.png"); border-image-slice: 26; border-image-repeat: repeat; width: 200px; height: 200px; background: green; margin: 0 auto; }

:

Example. Two words

Meaningrepeatfor width andstretchfor height:

<div id="elem"></div> #elem { border-image-repeat: repeat stretch; border-image-source: url("image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }
