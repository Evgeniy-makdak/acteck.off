visibility property

Propertyvisibilitysets the element's invisibility. In this case, all other elements will behave as if the element had never disappeared.

Syntax

селектор { visibility: visible или hidden или collapse; }

Values

Meaning Description
visible The element is visible.
hidden The element becomes invisible, as if transparent, and at the same time continues to participate in the formatting of the page.
collapse If you usecollapsefor the contents of table cells, they seem to disappear, and the table is rebuilt with a new one.
If this value is not applied to table rows or columns, then the result of its use will be the same ashidden.

Default value:visible.

Example

Let's make the first paragraph text initially invisible, but when hovering over the button'Show text'we could read it:

<button>Show text</button> <div class="text">text1</div> <div>text2</div> div{ border: 1px solid black; width: 100px; } .text { visibility: hidden; } button:hover + .text { visibility: visible; }
