padding-inline-end property

Propertypadding-inline-endspecifies the final internal indent of an element along the line (horizontal) axis.

Syntax

селектор { padding-inline-end: размер; }

Example

Let's set the ending padding for the child element's inline axis to20px:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-end: 20px; width: 100px; height: 50px; border: 1px solid red; }

:

Example

Now let's set the ending padding for the inline child element to20%:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-end: 20%; width: 100px; height: 50px; border: 1px solid red; }

:
