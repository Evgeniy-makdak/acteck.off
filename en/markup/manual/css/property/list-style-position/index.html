list-style-position property

Propertylist-style-positionsets whether the list marker will be displayed: inside or outside the listulorol.

Syntax

селектор { list-style-position: inside | outside; }

Values

Meaning Description
inside The marker is inside the list.
outside The marker is located outside the list.

Default value:outside.

Example. Meaning inside

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> ul { list-style-position: inside; border: 1px solid red; width: 200px; padding-left: 0; margin-left: 10px; }
