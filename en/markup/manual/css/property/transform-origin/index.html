transform-origin property

Propertytransform-originspecifies the point relative to which element transformations specified by the property will occurtransform. By default this point iselement center, and, for example, rotation will occur relative to its center. However, this behavior can be changed and make the element rotate relative to its side, corner, or even relative to a point that lies outside the element.

Syntax

селектор { transform-origin: X-ось Y-ось Z-ось; }

The values ​​for the Y and Z axes are optional and can be omitted (they will take the default values). Shifting a point along the Z axis is needed for 3D transformations.

X-axis values

Meaning Description
CSS units The meaning is anyunits for sizes, which specify the indentation of the transformation center from the left border of the element. A positive value moves the transformation center to the right (inside the element), and a negative value moves the transformation center to the left (outside the element) relative to the left border of the element.
left The horizontal rotation point is on the left edge of the element.
right The horizontal rotation point is on the right edge of the element.
center The horizontal rotation point is at the center of the element.

Default value:center.

Y axis values

Meaning Description
CSS units The meaning is anyunits for sizes, which specify the indentation of the transformation center from the top border of the element. A positive value moves the center of the transformation down (inside the element), and a negative value moves the center of the transformation up (outside the element) relative to the top border of the element.
top The vertical rotation point is at the top border of the element.
bottom The vertical rotation point is at the bottom border of the element.
center The vertical rotation point is at the center of the element.

Default value:center.

Z-axis values

Meaning Description
CSS units The meaning is anyunits for sizes, which specify the offset of the transformation center from the plane of the element. A positive value shifts it towards us (away from the screen plane), and a negative value away from us.

Default value:0px.

Example

Now the value of the transform-origin property is not set and the block will rotate relative to its center. Hover your mouse over the block to see the effect:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: center center; transform: rotate(30deg); }

:

Example

And now, when hovered, the block will rotate relative to the upper left corner:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: 0px 0px; transform: rotate(30deg); }

:

Example

And now, when hovered, the block will rotate relative to the lower right corner. To do this, you should assign the transformation point to100%to the right and to100%down (you could also set px, but when changing the size of the element, the transformation point would remain in place, so it’s better to do it in %):

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: 100% 100%; transform: rotate(30deg); }

:

Example

Let's rotate the block relative to the upper right corner:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: 100% 0%; transform: rotate(30deg); }

:

Example

Let's rotate the block relative to the center of the left side. To do this, for the X axis we setleft(the rotation point will be on the left), and for the Y axis -center(the rotation point will be vertically centered):

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: left center; transform: rotate(30deg); }

:

Example

The property can be set not only for rotation, but also for other transformations. Let's zoom in withscale, specifying the transformation point as the lower left corner:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: left bottom; transform: scale(1.5); }

:

Example

Now let’s indicate the transformation point as the upper right corner:

<div id="elem">lorem ipsum</div> #elem { border: 1px solid black; width: 100px; height: 50px; } #elem:hover { transform-origin: right top; transform: scale(1.5); }

:

Example

The transformation point can also be set outside the element. In the following example, when you point the mouse at a red block, the black block will rotate relative to a point that is located outside:

<div id="hover"></div> <div id="elem"></div> #hover { border: 1px solid red; width: 50px; height: 50px; } #elem { border: 1px solid black; width: 100px; height: 50px; transition: transform 1s; transform-origin: -100px -100px; } #hover:hover + #elem { transform: rotate(30deg); }
