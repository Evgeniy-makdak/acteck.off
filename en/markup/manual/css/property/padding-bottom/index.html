padding-bottom property

Propertypadding-bottomSets the inner bottom padding of an element. The property value is anyunits for sizes.

Syntax

селектор { padding-bottom: значение; }

Example

Let's make the inner bottom padding in30px:

<div id="elem"> some long text </div> #elem { padding-bottom: 30px; width: 300px; border: 1px solid black; text-align: justify; }
