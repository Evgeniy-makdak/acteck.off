border-image property

Propertyborder-imagespecifies the image for the border, it is a property shorthand for propertiesborder-image-source, border-image-slice, border-image-repeat, border-image-widthAndborder-image-outset. However, the shorthand property was introduced in CSS before shorthand properties and is therefore supported in more older browsers.

Syntax

селектор { border-image: url(путь к картинке) смещение/толщина/сдвиг повторение; }

Description

The image we want to use for the border must be designed in a special way:4mini pictures for corners and4pictures for the parties. An example of such a picture:

In this case, the central part is left white (since we don't want it to fall into the background of the element). An example of a picture with a filled central part:

Meaningslicetells the browser which parts of the image will go to the corners and which to the sides (and which part will be the center). They go to the corners4parts go to the sides4parts and one part (central) goes to the background of the element (optional, keywordfill).

The picture is “cut” into pieces as follows: for the meaningslicefrom one to four values ​​are indicated. Let's look at it with an example. Let the following values ​​be specified:10 20 30 40(px pixels are not indicated, this is due to the fact that the image can be both raster and vector). The values ​​indicate the following:10cut off from the top20cut off on the right30cut off from the bottom40cut off on the left. What part of the picture will fall into the upper left corner? This will be a piece:10above,40left. In the upper right corner will be10above,20right. And so on.

Most often the picture is symmetrical (like the diamonds above) and we need to cut equal pieces into corners. In this case, one value is specified, which will set the same offsets on all sides. To cut off the orange diamonds, we indicatesliceV26(since the orange diamond is the size26pxon26px). Yellow diamonds will fall on the border lines and the following will happen to them: they will either stretch across the entire block or will be repeated along the border (depending on the valuerepeat):

<div id="elem"></div> #elem { border-image-source: url("image.png"); border-image-slice: 26; border-image-repeat: stretch; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }

<div id="elem"></div> #elem { border-image-source: url("image.png"); border-image-slice: 26; border-image-repeat: repeat; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }

:

Propertyborder-imagedoes not allow you to specify the thickness of the element's border. The browser simply stretches the image along the border with the existing width. Therefore, it must be set through the propertyborder.

Users of older browsers (or with images disabled) will see the default border set inborder, so it makes sense to give it a suitable style and color.

Example. border-image-repeat: round value

<div id="elem"></div> #elem:hover { border-image: url("image.png") 26 round; } #elem { border-style: solid; border-width: 42px; border-image: url("image.png") 26 repeat; width: 200px; height: 200px; background: green; margin: 0 auto; }

Example. border-image-slice

Let's take another picture in which different pieces are not equal:

We will indicate which parts of the picture need to be cut off -25% 30% 10% 20%. It works like this:25%- indentation at the top,30%- right indentation,10%- bottom indentation,20%- left indentation. Essentially, we cut off the corners with these pieces. The upper left corner will be25%above the picture, and20%left. The upper right corner will be25%above the picture, and30%right and so on.

Also, if you hover your mouse over the block, the keyword will workfill, and the central part of the image will become the background of our block:

<div id="elem"></div> #elem:hover { border-image: url("image.png") fill 25% 30% 10% 20% stretch; } #elem { border-style: solid; border-width: 52px; border-image: url("image.png") 25% 30% 10% 20% stretch; width: 200px; height: 200px; margin: 0 auto; }
