border-right-style property

Propertyborder-right-stylespecifies the style of the right border. The property value is the same as in the propertyborder-style.

Syntax

селектор { border-right-style: значение; }

Example

Let's set the right border of the typedashed:

<div id="elem"></div> #elem { border-right-style: dashed; border-right-width: 1px; border-right-color: black; background: #f9f9f9; width: 300px; height: 100px; }

:
