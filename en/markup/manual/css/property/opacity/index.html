Opacity property

Propertyopacitysets the element's translucency. If an element has translucency set, this means that the elements that lie underneath it will be visible through it.

The property takes as its value a fractional number from0to1. In this case, the value0is complete transparency, and the value1- complete opacity. An intermediate value, for example0.3, sets translucency. Default value:1.

Syntax

селектор { opacity: число от 0 до 1; }

Example

In this example, translucency is added to the block and through the border and the text of this block the background of the parent shines through:

<div id="parent"> <div id="elem"> Lorem ipsum dolor sit amet. </div> </div> #parent { background-image: url("bg.png"); background-repeat: no-repeat; background-size: cover; display: inline-block; padding: 30px; } #elem { width: 300px; height: 200px; border: 20px solid red; padding: 10px; font-size: 50px; font-weight: bold; color: red; background: black; opacity: 0.7; }
