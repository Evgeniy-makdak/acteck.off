box-sizing property

Propertybox-sizingallows you to change the way element sizes are calculated.

Default addpaddingexpands the element: if we set the widthwidth 100pxAndpadding-leftV20px, then the actual width of the element will be120px.

And if we also askborder-leftV10px, then the actual width of the element will become130px. That ispaddingAndborderexpand the block (both in width and height). This behavior can be changed usingbox-sizing.

Syntax

селектор { box-sizing: content-box | border-box; }

Values

Meaning Description
content-box ANDpadding, Andborderexpand the block.
border-box Neitherpadding, norborderdo not expand the block.

Default value:content-box.

Example. Standard Behavior

Now both blocks have the same height and width, but the second blockpaddingis given, but the first one is not. Look how their sizes differ:

<div id="elem1">elem1</div> <div id="elem2">elem2</div> #elem1 { width: 300px; height: 100px; margin-bottom: 10px; background: #f1f1f1; } #elem2 { padding: 50px; width: 300px; height: 100px; background: #f1f1f1; }

Example. Standard Behavior

Currently both blocks have the same height and width, but the second block has a border set to10px, but the first one does not. Look how their sizes differ:

<div id="elem1">elem1</div> <div id="elem2">elem2</div> #elem1 { width: 300px; height: 100px; background: #f1f1f1; } #elem2 { border: 10px solid black; width: 300px; height: 100px; background: #f1f1f1; }

Example. Standard Behavior

Now the second block haspadding, and the border in10px:

<div id="elem1">elem1</div> <div id="elem2">elem2</div> #elem1 { width: 300px; height: 100px; background: #f1f1f1; } #elem2 { padding: 50px; border: 20px solid black; width: 300px; height: 100px; background: #f1f1f1; }

Example. border-box meaning

Let's add a value to the second elementborder-boxfor propertybox-sizing. Now the sizes of both the first and second elements will be the same:

<div id="elem1">elem1</div> <div id="elem2">elem2</div> #elem1 { width: 300px; height: 100px; background: #f1f1f1; } #elem2 { box-sizing: border-box; padding: 50px; border: 10px solid black; width: 300px; height: 100px; background: #f1f1f1; }
