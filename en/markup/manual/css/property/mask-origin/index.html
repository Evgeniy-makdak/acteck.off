mask-origin property

Propertymask-originsets the reference point for positioning the mask. It determines which part of the element (borders, padding, or content) the mask will be placed relative to.

Syntax

селектор { mask-origin: content-box | padding-box | border-box; }

Values

Meaning Description
border-box The mask is positioned relative to the element's border (including border).
padding-box The mask is positioned relative to the padding area (default).
content-box The mask is positioned relative to the element's content.

Example. border-box

The mask is positioned relative to the element's border:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: border-box; }

:

Example. padding-box

The mask is positioned relative to the padding area:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: padding-box; }

:

Example. content-box

The mask is positioned relative to the content:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: content-box; }
