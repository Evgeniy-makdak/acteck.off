border-style property

Propertyborder-stylesets the border style for all sides at the same time or separately for each side. Is a shorthand property for the following properties:border-left-style, border-right-style, border-top-style, border-bottom-style.

Syntax

селектор { border-style: значение; }

Values

Meaning Description
solid Solid line.
dotted Border in the form of dots.
dashed Border in the form of a dash.
ridge The border is in the form of a convex line.
double Double line border. To see the effect, the border thickness should be at least3px.
groove Concave border.
inset Depressed border.
outset Convex border.
none No border.

Default value:none.

Number of values

The property can take1, 2, 3or4values ​​specified separated by spaces:

Quantity Description
1 Type for all sides at the same time.
2 The first value is for the top and bottom, the second is for the left and right borders.
3 The first value is for the top, the second is for the left and right borders, the third is for the bottom.
4 The first value is for the top border, the second - for the right, the third - for the bottom, the fourth - for the left border.

Example

This example specifies different border types for different sides of the elements:

<div id="elem"></div> #elem { border-width: 1px; border-style: solid dotted dashed dotted; border-color: black; width: 300px; height: 100px; }

:

Example

And now different sides of the element are also given different border thickness and color:

<div id="elem"></div> #elem { border-width: 1px 2px 3px 4px; border-style: dashed dotted solid double; border-color: red blue green black; width: 300px; height: 100px; }

:

Example

The top and bottom borders are now set to typesolid, and for right and left -dotted:
