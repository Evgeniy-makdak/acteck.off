margin property

Propertymarginsets the outer padding of an element. The property value is anyunits for sizesor key propertyauto, which lets the browser calculate the element's indentation on its own. By default, each browser can add different padding to elements.

Property is short for propertiesmargin-top, margin-right, margin-bottom, margin-left.

Syntax

селектор { margin: значение; }

Number of values

Propertymarginaccepts1, 2, 3or4values ​​listed separated by spaces:

Quantity Description
1 One value specifies the padding on all sides of the element.
2 The first value specifies the top and bottom padding, and the second specifies the right and left padding.
3 The first value sets the top margin, the second - right and left, and the third - bottom.
4 The first value specifies the top indent, the second the right, the third the bottom, and the fourth the left.

Example

Now we will have a block without indents:

<div id="parent"> <div id="elem"></div> </div> #parent { border: 1px solid black; display: inline-block; } #elem { margin: 0; border: 1px solid red; width: 100px; height: 100px; }

:

Example

Now let’s indent the block by10px:

<div id="parent"> <div id="elem"></div> </div> #parent { border: 1px solid black; display: inline-block; } #elem { margin: 10px; border: 1px solid red; width: 100px; height: 100px; }

:

Example

Let's set the block indent to10pxtop and bottom and 20px - left and right:

<div id="parent"> <div id="elem"></div> </div> #parent { border: 1px solid black; display: inline-block; } #elem { margin: 10px 20px; border: 1px solid red; width: 100px; height: 100px; }

<div id="parent"> <div id="elem"></div> </div> #parent { border: 1px solid black; display: inline-block; } #elem { margin: 5px 15px 25px 35px; border: 1px solid red; width: 100px; height: 100px; }
