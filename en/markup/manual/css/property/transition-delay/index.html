transition-delay property

Propertytransition-delaysets the delay before the fade startstransition.

Syntax

селектор { transition-delay: время в s или ms; }

Values

Meaning Description
s Specifies the time in seconds (for example3s). You can specify fractional values, for example,0.5s- half a second.
ms Specifies the time in milliseconds (for example3ms). One second is1000milliseconds

Default value:0s.

Example

Hover your mouse over the block -3nothing will happen for seconds (there is a delay3s), and then it will smoothly change its width in2seconds. If you then remove the mouse, then again3nothing will happen for a second, and then the width will smoothly decrease to its original value:

<div id="elem"></div> #elem { border: 1px solid black; transition-duration: 2s; transition-property: width; transition-delay: 3s; height: 50px; width: 100px; } #elem:hover { width: 400px; }

:

Example

Hover your mouse over the block - it will first change its width for2seconds, and then changes its height in4seconds. The properties will change sequentially since the height is set to3seconds (time of width change). If you remove the mouse, the changes will occur in the reverse order: first the width will decrease, and then the height will decrease:

<div id="elem"></div> #elem { border: 1px solid black; transition-duration: 2s, 4s; transition-property: width, height; transition-delay: 0s, 3s; width: 100px; height: 25px; } #elem:hover { width: 400px; height: 50px; }
