column-gap property

Propertycolumn-gapspecifies the distance between columns in multi-column text, as well as the distance between columns in a grid. The property value is anyunits for sizesor keywordnormal(default), in which the browser itself selects the optimal distance between columns.

Syntax

селектор { column-gap: значение; }

Example

This example shows the number of columnscolumn-count (3pieces) and the space between themcolumn-gapV50px. And the width of the columns will be automatically selected to satisfy these conditions (the number of columns and the distance between them):

<div id="elem"> some long text </div> #elem { column-count: 3; column-gap: 50px; text-align: justify; }

:

Example. Normal value

In this example, the column width is specifiedcolumn-widthV150px, their numbercolumn-countstands in meaningauto, and the intervalcolumn-gapbetween them - in meaningnormal. Thus, the browser itself will select the required number of columns and the space between them (but their width will be150px):

<div id="elem"> some long text </div> #elem { column-gap: normal; column-count: auto; column-width: 150px; text-align: justify; }

:

Example

Let's set the distance between columns in the grid:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #parent { display: grid; column-gap: 10px; padding: 10px; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-column: 1 / 2; } #elem2 { grid-column: 2 / 3; } #elem3 { grid-column: 3 / 4; }

:

Example

Now let's set the distance between the columns in%:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #parent { display: grid; column-gap: 5%; padding: 10px; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-column: 1 / 2; } #elem2 { grid-column: 2 / 3; } #elem3 { grid-column: 3 / 4; }
