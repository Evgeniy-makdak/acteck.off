border-top-style property

Propertyborder-top-stylespecifies the style of the top border. The property value is the same as in the propertyborder-style.

Syntax

селектор { border-top-style: значение; }

Example

Set the upper bound of the typedashed:

<div id="elem"></div> #elem { border-top-style: dashed; border-top-width: 1px; border-top-color: black; background: #f9f9f9; width: 300px; height: 100px; }

:

See also

  • propertyborder-top-color,
    which sets the color of the top border
  • propertyborder-top-width,
    which sets the thickness of the upper border
  • propertyborder-top,
    which sets the border for the top side
  • propertyborder-width,
    which sets the thickness of all sides of the border
