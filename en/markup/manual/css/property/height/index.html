height property

Propertyheightsets the height of the element. The property value is anyunits for sizesor key propertyauto, which allows the browser to calculate the element's size on its own (the default behavior).

Syntax

селектор { height: значение; }

Example

Set the height of the block to100px:

<div id="elem"></div> #elem { border: 1px solid black; width: 300px; height: 100px; }
