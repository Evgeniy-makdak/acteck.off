backdrop-filter property

Propertybackdrop-filterallows you to apply filters (blur, contrast changes, color effects) to the area behind the element. This creates a "glass" or "frosted glass" effect.

Syntax

селектор { backdrop-filter: none | <фильтр> [<фильтр>]*; }

Values

Meaning Description
none No filters are applied (default).
blur() Applies a blur effect (for example:blur(5px)).
brightness() Changes the background brightness (for example:brightness(0.5)).
contrast() Changes the background contrast (for example:contrast(200%)).
drop-shadow() Adds shadow to the background.
grayscale() Converts the background to grayscale (for example:grayscale(100%)).
hue-rotate() Changes the background color tone (for example:hue-rotate(90deg)).
invert() Inverts the background colors (for example:invert(100%)).
opacity() Changes the transparency of the background (for example:opacity(50%)).
sepia() Applies sepia to the background (for example:sepia(100%)).
saturate() Changes the saturation of background colors (for example:saturate(200%)).

You can combine several filters by listing them separated by a space.

Example. Simple blur

Basic background blur effect:

<div class="blur-example"> <div class="blur-box"> Backdrop blur effect </div> </div> .blur-example { background: linear-gradient(135deg, #667eea, #764ba2); padding: 40px; height: 200px; } .blur-box { backdrop-filter: blur(6px); background-color: rgba(255, 255, 255, 0.2); border-radius: 8px; padding: 20px; width: 80%; margin: 0 auto; text-align: center; color: white; font-size: 22px; }

Browser support

The property is supported in most modern browsers, but may require a prefix-webkit-for full cross-browser compatibility.
