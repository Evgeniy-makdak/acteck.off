Свойство border

Свойство border задает цвет, тип и толщину границы для всех сторон одновременно. Является свойством-сокращением для border-width, border-style и border-color. В отличие от сокращаемых свойств не может задавать разную границу для 4-х сторон сразу, а только одинаковую для всех.

Синтаксис

селектор { border: толщина тип цвет; }

Порядок свойств значения не имеет.

Пример

Давайте сделаем элементу черную границу толщиной в 1px:

<div id="elem"></div> #elem { border: 1px solid black; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-left,
    которое задает левую границу
  • свойство border-right,
    которое задает правую границу
  • свойство border-top,
    которое задает верхнюю границу
  • свойство border-bottom,
    которое задает нижнюю границу
  • свойство outline,
    которое задает рамку для элемента
  • свойство border-radius,
    которое скругляет уголки границы
