border property

Propertybordersets the color, type and thickness of the border for all sides at the same time. Is a shorthand property forborder-width, border-styleAndborder-color. Unlike shorthand properties, it cannot specify a different boundary for4- x sides at once, but only the same for everyone.

Syntax

селектор { border: толщина тип цвет; }

The order of the properties does not matter.

Example

Let's give the element a black border of thickness1px:

<div id="elem"></div> #elem { border: 1px solid black; width: 300px; height: 100px; }
