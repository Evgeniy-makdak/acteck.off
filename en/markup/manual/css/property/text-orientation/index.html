text-orientation property

Propertytext-orientationspecifies the orientation of characters in vertical text, i.e. to which the property is setwriting-modewith meaningsvertical-lrorvertical-rl. The property takes two values:mixed(symbols rotate to90°clockwise) orupright(symbols are placed one below the other).

Syntax

селектор { text-orientation: mixed или upright; }

Example

Let's set all vertical text characters to rotate to90°:

<p> Text </p> body { width: 200px; height: 100px; } p { text-orientation: mixed; background: #467CBC; color: white; padding: 10px; font-size: 2rem; writing-mode: vertical-lr; }

:

Example

Let's place all the vertical text characters below each other:

<p> Text </p> body { width: 200px; height: 100px; } p { text-orientation: upright; background: #467CBC; color: white; padding: 10px; font-size: 2rem; writing-mode: vertical-lr; }

:
