font-family property

Propertyfont-familysets the font type.

Syntax

селектор { font-family: имя шрифта; }

Comment

If the font name contains spaces, e.g.Times New Roman, it must be enclosed in single or double quotes.

Comment

You can list several similar fonts separated by commas. When the browser encounters the first font in the list, it checks for its presence on the user's computer. If there is no such font, the next font from the list is taken and also analyzed for presence. Therefore, multiple fonts increases the likelihood that at least one of them will be detected on the client computer.

They usually end the list with a keyword that describes the font type (all fonts belong to some type) -serif, sans-serif, cursive, fantasyormonospace. If the browser does not find any of the specified fonts on the user's computer, it will select one of the fonts of the specified type.

Font types

Type Description
serif Serif font.
sans-serif Sans serif font.
cursive Italic font.
fantasy A decorative font with beautiful curls and unusual effects.
monospace A monospaced font in which the width of each character is the same.

Default: The browser's default font. This is usually Times New Roman.

Example

Assign a family to the fontArial:

<p> Lorem ipsum dolor sit amet. </p> p { font-family: Arial; }

:

Example

Assign a family to the fontTimes New Roman. Since the font name consists of several words, we put it in quotes:

<p> Lorem ipsum dolor sit amet. </p> p { font-family: "Times New Roman"; }

:

Example

Assign a font from the group to the fontserif(most likely Times New Roman will be used):

<p> Lorem ipsum dolor sit amet. </p> p { font-family: serif; }

:

Example

Assign a font from the group to the fontsans-serif(most likely Arial will be used):

<p> Lorem ipsum dolor sit amet. </p> p { font-family: sans-serif; }

:

Example

Assign a font from the group to the fontfantasy(it may not work for you due to the lack of such a font):

<p> Lorem ipsum dolor sit amet. </p> p { font-family: fantasy; }

:

Example

Assign a font from the group to the fontmonospace(the letters will become the same size):

<p> Lorem ipsum dolor sit amet. </p> p { font-family: monospace; }
