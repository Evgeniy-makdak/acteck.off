User-select property - disable text selection

Propertyuser-selectcontrols the highlighting behavior of text and other elements on the page.

Syntax

селектор { user-select: none | contain | all | auto | text; }

Values

Meaning Description
none The text is not highlighted.
contain A selection started inside an element will not extend beyond that element.
all If a double click occurs on a nested element, all parent content with that property value will be selected.
auto Calculated value, automatically determined as follows: for pseudo-elementsafterAndbeforethe value isnone, for the edited element the value iscontainif the element's parent has a valueallornone- the element itself will have the same, otherwise the value will betext.
text The user can select text in an element.

Default value:text.

Note

Some values ​​may not work in all browsers.

Example. Value none

The following text cannot be selected:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { user-select: none; }
