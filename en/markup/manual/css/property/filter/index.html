filter property

PropertyfilterGives the background image a specific art style or filter. The property can take as values ​​the names of artistic styles generated by the functionfilterornone(cancel the selected filter).

Syntax

селектор { filter: художественный стиль или none; }

Example

Let's set a black and white filter for our image, and cancel it on hover:

<div id="elem"></div> #elem { filter: grayscale(100%); background: url("bg.png") center / cover no-repeat; width: 400px; height: 300px; border: 1px solid black; } #elem:hover { filter: none; }
