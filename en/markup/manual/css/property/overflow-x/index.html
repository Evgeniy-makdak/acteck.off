Свойство overflow-x

Свойство overflow-x указывает браузеру, как поступать с содержимым, которое вылазит за границы блока по горизонтали. Работа свойства аналогична свойству overflow.

Синтаксис

селектор { overflow-x: hidden | scroll | auto | visible; }
