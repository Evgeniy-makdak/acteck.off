background-position property

Propertybackground-positionspecifies the location of the element's background image. The location can be set using anyunits for sizes. The first value denotes the left indent, the second - the top indent.

You can also specify the position using keywords. In this case, the order of the values ​​is not important. Keywords for vertical:top, center, bottom. Horizontal Keywords:left, center, right.

How to use keywords

To place an image with a keyword, you need to specify one value for the vertical and one for the horizontal. For example, if you specify the valuetop right, then the picture will appear on the top right.

The order of the words does not matter: you can writetop right, or maybe -right top. If there is a keywordcenter- it can be lowered. For example,top center, it's just liketop. Acenter centerit's just like thatcenter.

Syntax

селектор { background-position: два значения через пробел; }

Example

By default, the background image will be in the top left corner:

<div id="elem"></div> #elem { background-image: url("bg.png"); background-repeat: no-repeat; width: 400px; height: 300px; border: 1px solid black; }
