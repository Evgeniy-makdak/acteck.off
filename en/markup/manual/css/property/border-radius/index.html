border-radius property

Propertyborder-radiuscreates rounded corners for border and background. The property value is anyunits for sizes. Default value:0. Is short for propertiesborder-top-left-radius, border-top-right-radius, border-bottom-left-radius, border-bottom-right-radius.

Syntax

селектор { border-radius: значение; }

Number of values

The property can take1, 2, 3or4values ​​specified separated by spaces:

Quantity Description
1 For all angles at the same time.
2 The first value sets the rounding for the top right and bottom left corners, the second - for the top left and bottom right.
3 The first value sets the rounding for the upper left corner, the second - for both the upper right and lower left, and the third - for the lower right corner.
4 The first value sets the rounding for the top left corner, the second for the top right, the third for the bottom right, and the fourth for the bottom left.

Elliptical fillet

Two values ​​separated by a slash set an elliptical fillet. The value before the slash indicates a horizontal fillet, and the values ​​after the slash indicate a vertical fillet:

селектор { border-style: горизонтальное / вертикальное; }

If fillets are specified for several corners, then all horizontal fillets are listed before the slash, and all vertical ones after it.

Example

Set the rounding10pxfor all angles:

<div id="elem"></div> #elem { border-radius: 10px; border: 1px solid black; width: 300px; height: 100px; }

Example

If you put a fillet for a square block equal to half the side of the square, you get a circle:

<div id="elem"></div> #elem { border-radius: 100px; border: 1px solid black; width: 200px; height: 200px; }

Example

If you put a fillet larger than the side of the square, you will still get a circle:

<div id="elem"></div> #elem { border-radius: 200px; border: 1px solid black; width: 200px; height: 200px; }

Example

The circle can also be obtained by installingborder-radiusV50%(the advantage is that when changing the size of the square you will not have to change the fillet):
