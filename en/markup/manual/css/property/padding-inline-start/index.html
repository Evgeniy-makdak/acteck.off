Свойство padding-inline-start

Свойство padding-inline-start задает начальный внутренний отступ элемента по строчной (горизонтальной) оси.

Синтаксис

селектор { padding-inline-start: размер; }

Пример

Давайте установим начальный внутренний отступ для дочернего элемента по строчной оси на 20px:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-start: 20px; width: 100px; height: 50px; border: 1px solid red; }

:

Пример

А теперь давайте установим начальный внутренний отступ для дочернего элемента по строчной оси на 20%:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-start: 20%; width: 100px; height: 50px; border: 1px solid red; }

:

Смотрите также

  • свойство padding-inline-end,
    которое задает конечный внутренний отступ элемента по строчной оси
  • свойство padding-block-end,
    которое задает конечный внутренний отступ элемента по блочной оси
  • свойство padding-block-start,
    которое задает начальный внутренний отступ элемента по блочной оси
  • свойство padding,
    которое задает внутренний отступ
  • свойство margin-bottom,
    которое задает внешний отступ снизу
  • свойство margin-left,
    которое задает внешний отступ слева
  • свойство margin-right,
    которое задает внешний отступ справа
  • свойство margin,
    которое является сокращением для внешнего отступа
