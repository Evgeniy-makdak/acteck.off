font-size property

Propertyfont-sizesets the text font size. The property value is anyunits for sizes(usually this ispx, emorrem) or special keywords (used extremely rarely).

Syntax

селектор { font-size: значение; }

Values ​​as keywords

Meaning Description
xx-small The smallest one. Example:Lorem ipsum dolor sit amet.
x-small The smallest one. Example:Lorem ipsum dolor sit amet.
small Small. Example:Lorem ipsum dolor sit amet.
medium Average. Example:Lorem ipsum dolor sit amet.
large Big. Example:Lorem ipsum dolor sit amet.
x-large Very big. Example:Lorem ipsum dolor sit amet.
xx-large The biggest one. Example:Lorem ipsum dolor sit amet.
larger Larger than the parent font by some value.
smaller Smaller than the parent font by some value.

Default value:medium.

Example

Let's set the paragraph font size to20px:

<p> Lorem ipsum dolor sit amet. </p> p { font-size: 20px; }

:

Example

In this example, the paragraph size is set to16px, and for the span inside it -150%. In this case, the font size for span will be150%from its parent, that is, a paragraph, and its actual size will be16px * 150% = 24px:

<p> Lorem <span>ipsum dolor</span> sit amet. </p> p { font-size: 16px; } span { font-size: 150%; }

:

Example

In this example, the paragraph size is set to16px, and forspaninside it -larger. In this case, the font size forspanwill be greater than its parent (paragraph):

<p> Lorem <span>ipsum dolor</span> sit amet. </p> p { font-size: 16px; } span { font-size: larger; }

:

Example

Please note that fonts with differentfont-familyand visually the same font-size may not be the same size (to solve this problem, see the propertyfont-size-adjust).

In the example below, both paragraphs are givenfont-sizeV16px, but differentfont-family:

<p id="elem1"> Lorem ipsum dolor sit amet. </p> <p id="elem2"> Lorem ipsum dolor sit amet. </p> #elem1 { font-size: 16px; font-family: Arial; } #elem2 { font-size: 16px; font-family: "Times New Roman"; }
