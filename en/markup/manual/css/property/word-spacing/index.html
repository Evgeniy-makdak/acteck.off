word-spacing property

Propertyword-spacingsets the spacing between words. The property value is anyunits for sizesexcept interest. A negative value is acceptable. MeaningnormalAllows the browser to calculate the spacing between words itself (default).

If text alignment is set totext-alignin meaningjustify, then the interval between words will be set automatically, but not less than the value specified throughword-spacing.

Syntax

селектор { word-spacing: значение; }

Example

Let's put the distance between words at30px:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: 30px; text-align: left; }

:

Example

Let's installword-spacingto a negative value - the distance between words will practically disappear:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: -7px; }

:

Example

Setting this property to zero does not merge words:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: 0; }

:

Example

Let's installword-spacingin meaningnormal(this is the default):

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: normal; }
