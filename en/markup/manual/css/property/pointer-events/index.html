pointer-events property

Propertypointer-eventsallows you to disable the element's response to mouse events. The property takes two values. Meaningnoneremoves the reaction and the meaningautoCauses the element to behave as usual (default).

Syntax

селектор { pointer-events: none | auto; }

Example

In the following example, the link will not respond to hover or click:

<a href="http://google.com">ссылка</a> a { pointer-events: none; }
