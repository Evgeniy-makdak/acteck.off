align-self property

Propertyalign-selfsets the alignment along the transverse axis for an individual flex block and along the vertical axis for an individual element in the grid. In essence, this property is a propertyalign-items, but for a specific block.

Syntax

селектор { align-self: auto | flex-start | flex-end | center | baseline | stretch; }

Values

Meaning Description
flex-start The block is pressed to the beginning of the transverse axis.
flex-end The block is pressed against the end of the transverse axis.
center The block stands in the center of the transverse axis.
baseline The block is aligned to its baseline. The baseline is an imaginary line that runs along the bottom edge of characters without taking into account overhangs, such as those of letters'p', 'q', 'y', 'g'.
stretch The block is stretched, occupying all the available space along the transverse axis, while still taking into accountmin-widthAndmax-width, if they are specified. If the width and height for the element are specified -stretchwill be ignored.
auto The block will be aligned as specified in the propertyalign-items.

Default value:auto.

Example. Meaning of stretch

In this example, all blocks are set to flex-start (propertyalign-items), and the third block -align-selfin meaningstretch:

<div id="parent"> <div>1</div> <div>2</div> <div id="elem">3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: flex-start; flex-direction: row; border: 1px solid #696989; height: 100px; } #parent > div { min-width: 100px; border: 1px solid #696989; } #elem { align-self: stretch; }

:

Example. Flex-end value

In this example, all blocks for the propertyalign-itemsgiven valueflex-start, and the third block -align-selfin meaningflex-end:

<div id="parent"> <div>1</div> <div>2</div> <div id="elem">3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: flex-start; flex-direction: row; border: 1px solid #696989; height: 100px; } #parent > div { min-width: 100px; border: 1px solid #696989; } #elem { align-self: flex-end; }

:

Example. Alignment to the origin of the vertical axis in the grid

Let's set the alignment of the first element to the origin of the vertical axis:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> <div id="elem4">4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { align-self: start; }

:

Example. Aligning to the center of the vertical axis in the grid

Let's set the alignment of the first element to the center of the vertical axis:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { align-self: center; }

:

Example. Aligning to the end of the vertical axis in the grid

Let's set the alignment of our first element in the grid to the end of the vertical axis:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { align-self: end; }
