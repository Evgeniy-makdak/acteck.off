display property

Propertydisplay- sets how the browser displays the element.

Syntax

селектор { display: значение; }

Values

Meaning Description
block Block element.
inline Inline element.
inline-block Inline block element.
list-item The item becomes a list item and a list marker appears in front of it.
flex Sets the element to be a block element and its children will become flex elements.
inline-flex Sets the element to be an inline-block element, and its descendants will become flex elements.
grid Creates a mesh element.
none The element will disappear altogether and all other elements will behave as if this element does not exist.
table The element will behave like a table.
table-cell The element will behave like a table cell.
inline-table The element will behave like a table, but the table is an inline element and wraps around other elements, such as text.
table-caption The element will behave like a tagcaption.
table-column The element will behave like a table column.
table-row The element will behave like a table row.
table-column-group The element will behave like a tagcolgroup.
table-footer-group The element will behave like a tagtfoot.
table-header-group The element will behave like a tagthead.
table-row-group The element will behave like a tagtbody.
run-in Sets the element to be block-level or inline depending on the context.

By default, elements can have different valuesdisplay.
