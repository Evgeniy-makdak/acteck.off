border-image-slice property

Propertyborder-image-slicetells the browser which parts of the image will go to the corners and which to the sides (and which part will be the center). They go to the corners4parts go to the sides4parts and one part (central) goes to the background of the element (optional, keywordfill).

For more details see propertyborder-image.

Syntax

<+css+>selector { border-image-slice: from1th before4-x numbers | from1th before4-x percent; }<-css->

With numbers and percentages, a keyword can be separated by a spacefill.

Values

Meaning Description
interest Percentages are calculated relative to the image size. Horizontal relative to width, vertical relative to height.
numbers The numbers are pixels (for a raster image) or coordinates (for a vector image). Units of measurement are not indicated.
fill The default behavior is that the center portion of the image will be discarded. In order to use it, you must use the keywordfillin addition to numbers or percentages.

Number of parameters

The offset can take1, 2, 3or4parameter. Please note that offset units are not written (for example, just20, not20px). You can also set the thickness in%.

Number of parameters Description
1 Thickness for all sides at the same time.
2 1 2; - 1pxfor top and bottom,2pxfor left and right.
3 1 2 3; - 1pxfor the top,2pxfor left and right,3pxfor the bottom.
4 1 2 3 4; - 1pxfor the top,2pxfor the right one,3pxfor the bottom,4pxfor the left.

Default value:100%(?).

Example

<div id="elem"></div> #elem { border-image-source: url("image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 0 auto; }
