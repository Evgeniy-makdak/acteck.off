letter-spacing property

Propertyletter-spacingsets the spacing between letters.

The property value is anyunits for sizesexcept interest. A negative value is acceptable. MeaningnormalAllows the browser to calculate letter spacing itself (default).

Syntax

селектор { letter-spacing: значение; }

Example

<p> Lorem ipsum dolor sit amet. </p> p { letter-spacing: 4px; }
