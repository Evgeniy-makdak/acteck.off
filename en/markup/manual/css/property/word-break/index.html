word-break property

Propertyword-breakallows you to move the letters of a long word to a new line if this word does not fit into the width of the container.

Syntax

селектор { word-break: break-all | keep-all | normal; }

Values

Meaning Description
break-all Causes long words to wrap to a new line if the word does not fit in the container.
keep-all For transferring hieroglyphs.
normal Standard behavior: if a long word does not fit the width of the container, it will simply crawl outside its border (and start on a new line).

Default value:normal.

Example. Normal value

In this example, a very long word will not fit into the container and will fall outside its boundaries (and will start on a new line):

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur adipiscing elit. </div> #elem { width: 200px; word-break: normal; border: 1px solid red; }

:

Example. Break-all meaning

And now those letters that did not fit will simply be moved to the next line:

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur adipiscing elit. </div> #elem { width: 200px; word-break: break-all; border: 1px solid red; }

:
