outline-style property

Propertyoutline-stylesets the style of the outer frame - a border that does not take up space.

Syntax

селектор { outline-style: значение; }

Values

Meaning Description
solid Solid line.
dotted Frame in the form of dots.
dashed Frame in the form of a dash.
ridge Frame in the form of a convex line.
double Double line frame. To see the effect, the border thickness should be at least3px.
groove Concave frame.
inset Recessed frame.
outset Convex frame.
none No border.

Default value:none.

Example. Solid value

<div id="elem"></div> #elem { outline-width: 1px; outline-style: solid; outline-color: black; width: 300px; height: 100px; }
