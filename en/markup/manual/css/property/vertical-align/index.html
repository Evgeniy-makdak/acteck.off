vertical-align property

Propertyvertical-alignsets the vertical alignment of the text.

The property only works for inline elementsinline, inline-block (inline-block) and for tables. Will not work for block elements.

Syntax

селектор { vertical-align: bottom | top | middle | sub | super | baseline | text-bottom | text-top; }

Values

Meaning Description
bottom Along the bottom edge.
top Along the top edge.
middle In the center.
sub Subscript.
super Superscript.
baseline Along the baseline.
text-bottom At the very bottom edge of the line.
text-top At the very top edge of the line.

Default value:middle.
