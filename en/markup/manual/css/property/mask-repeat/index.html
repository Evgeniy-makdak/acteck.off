mask-repeat property

Propertymask-repeatspecifies how the mask will be repeated if its dimensions are smaller than the dimensions of the element. Works similarbackground-repeat, but applies to masks.

Syntax

селектор { mask-repeat: repeat-x | repeat-y | repeat | space | round | no-repeat; }

Values

Meaning Description
repeat The mask is repeated on both axes (default).
no-repeat The mask is not repeated.
repeat-x The mask is repeated only horizontally.
repeat-y The mask is repeated only vertically.
space The mask is repeated at regular intervals.
round The mask is scaled to fill it evenly.

Example. No-repeat

One heart in the center without repetition:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: 100px; mask-repeat: no-repeat; }

:

Example. Repeat horizontally (repeat-x)

Hearts are repeated only horizontally:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: left center; mask-size: 50px; mask-repeat: repeat-x; }

:

Example. Vertical repetition (repeat-y)

Hearts are repeated only vertically:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center top; mask-size: 50px; mask-repeat: repeat-y; }

:

Example. Repeat with spaces (space)

The hearts are evenly distributed at intervals:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-size: 80px; mask-repeat: space; }

:

Example. Scalable repetition (round)

Hearts are scaled to fill evenly:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-size: 80px; mask-repeat: round; }

:
