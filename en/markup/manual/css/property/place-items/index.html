place-items property

Propertyplace-itemssets the simultaneous alignment of columns and rows, which significantly reduces the code we use. With the first value we specify the location of the element along the vertical axes, and the second value along the horizontal axes. Propertyplace-itemsis specified in the parent element. Doing work withplace-itemsconvenient to view through the browser debugger.

Syntax

селектор { place-items: значение выравнивания по горизонтали значение выравнивания по вертикали; }

Example. Alignment to the center of the vertical and the beginning of the horizontal axis

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: grid; place-items: center start; grid-template-columns: 100px 100px; gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }
