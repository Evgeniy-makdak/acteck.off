animation-direction property

Propertyanimation-directionsets the direction of the animation. By default, the animation will only repeat1times and then returns to its original state, then if a delay is specifiedanimation-delay, will wait the specified time and then the cycle will start over.

This property allows you to change this behavior, for example, so that the animation, after finishing, remains in the place where it ended, rather than jumping to the starting position.

You can also set the following behavior: the animation will reach the extreme point and return back (as intransition). See description below.

Syntax

селектор { animation-direction: normal | reverse | alternate | alternate-reverse; }

Values

Meaning Description
reverse The animation will play in reverse.
alternate The animation will play first in the forward direction, and then in the opposite direction (like a transition).
alternate-reverse The animation will play from the end position to the start position and back. Essentially these are the meaningsalternateAndreversein one bottle.
normal The animation will play from beginning to end, and after finishing it will jump to the starting position.

Default value:normal.

Example

Now the element will move in one direction and then return back, since the value is setalternate. At the same timeanimation-durationmatters3seconds and this means that movements “there” and “back” will last for3seconds:

<div id="elem"></div> @keyframes move { from { margin-left: 0; } to { margin-left: 400px; } } #elem { animation-direction: alternate; animation-duration: 3s; animation-name: move; animation-iteration-count: infinite; animation-timing-function: linear; border: 1px solid black; width: 50px; height: 50px; }

:

Example

Now the element will move in the opposite direction (should be from left to right, but will be from right to left):

<div id="elem"></div> @keyframes move { from { margin-left: 0; } to { margin-left: 400px; } } #elem { animation-direction: reverse; animation-duration: 3s; animation-name: move; animation-iteration-count: infinite; animation-timing-function: linear; border: 1px solid black; width: 50px; height: 50px; }

:

Example

Now the element will move back and forth, but in the opposite direction (should start on the left, but will start on the right):

<div id="elem"></div> @keyframes move { from { margin-left: 0; } to { margin-left: 400px; } } #elem { animation-direction: alternate-reverse; animation-duration: 3s; animation-name: move; animation-iteration-count: infinite; animation-timing-function: linear; border: 1px solid black; width: 50px; height: 50px; }
