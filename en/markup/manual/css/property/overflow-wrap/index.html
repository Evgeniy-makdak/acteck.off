overflow-wrap property

Propertyoverflow-wrapallows you to move the letters of a long word to a new line if this word does not fit into the width of the container.

Syntax

селектор { overflow-wrap: break-word | normal; }

Values

Meaning Description
break-word Causes long words to wrap to a new line if the word does not fit in the container. The word will start on a new line.
normal Standard behavior: if a long word does not fit the width of the container, it will simply crawl out beyond its border (and start on a new line).

Default value:normal.

Example. Normal value

In this example, a very long word will not fit into the container and will fall outside its boundaries (and will start on a new line):

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur adipiscing elit. </div> #elem { word-wrap: normal; overflow-wrap: normal; border: 1px solid red; width: 200px; }

:

Example. Break-word meaning

And now those letters that did not fit will simply be moved to the next line (in this case, the long word will still start on a new line):

<div id="elem"> Lorem ipsum dolor sit amet loooooooooooooooooooooooooooooooongword, consectetur adipiscing elit. </div> #elem { overflow-wrap: break-word; border: 1px solid red; width: 200px; }

:
