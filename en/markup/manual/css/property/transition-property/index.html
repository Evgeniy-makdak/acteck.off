transition-property

Propertytransition-propertysets the property that will be animated by the fade transitiontransition. The value should be the name of the CSS property that will be animated. If you set the valueall(it is set by default) - all properties will be animated at once.

Syntax

селектор { transition-property: значение; }

Example

Hover your mouse over the block - it will smoothly change its width, sincetransition-propertymatterswidth:

<div id="elem"></div> #elem { border: 1px solid black; transition-duration: 2s; transition-property: width; height: 50px; width: 100px; } #elem:hover { width: 400px; }

:

Example. All meaning

In this exampletransition-propertymattersall- therefore, all properties written for the state will change smoothlyhover(in our case it iswidthAndheight). All changes will take place in one time, which is specified intransition-duration:

<div id="elem"></div> #elem { border: 1px solid black; transition-duration: 2s; transition-property: all; height: 25px; width: 100px; } #elem:hover { height: 50px; width: 400px; }

:
