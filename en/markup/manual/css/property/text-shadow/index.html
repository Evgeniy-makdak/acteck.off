text-shadow property

Propertytext-shadowsets a shadow for the text. The property takes as its value4parameters, separated by spaces, or a keywordnone, which cancels the shadow completely (this is the default).

Syntax

селектор { text-shadow: сдвиг_по_x сдвиг_по_y размытие цвет; }
селектор { text-shadow: none; }

Options

Parameter Description
x-shift Sets the shadow offset along the X axis.
A positive value shifts to the right, a negative value shifts to the left.
y shift Sets the shadow offset along the Y axis.
A positive value moves down, a negative value moves up.
blur Sets the shadow blur.
The higher the value, the more blurry the shadow will be.
Optional parameter. If not specified, the shadow will be clear.
color Sets the shadow color in anyunits for color.
Optional parameter. If not specified, the shadow color matches the text color.

Axis displacements and blur are specified in anyunits for sizesexcept interest.

Example

Let's move the shadow down and to the left and add a little blur:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-shadow: 5px 5px 3px black; }

:

Example. Clear shadow

Now let’s make a red shadow, move it down and to the left, but remove the blur - the shadow will be clear:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-shadow: 10px 10px red; }

:

Example. Uniform shadow

Now the shadow is not moved, but a blur has been added to it:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { text-shadow: 0px 0px 3px black; }

:
