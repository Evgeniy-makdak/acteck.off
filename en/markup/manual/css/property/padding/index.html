padding property

PropertypaddingSets the padding of an element. The property value is anyunits for sizes. By default, each browser can add different padding to elements. Property is short for propertiespadding-top, padding-right, padding-bottom, padding-left.

Syntax

селектор { padding: значение; }

Number of parameters

Propertypaddingaccepts1, 2, 3or4values ​​listed separated by spaces:

Quantity Description
1 One value specifies the padding on all sides of the element.
2 The first value specifies the top and bottom padding, and the second specifies the right and left padding.
3 The first value sets the top margin, the second - right and left, and the third - bottom.
4 The first value specifies the top indent, the second the right, the third the bottom, and the fourth the left.

Example

In this case, by default the padding will be zero and the text will be pressed against the block border:

<div id="elem"> some long text </div> #elem { padding: 0; width: 300px; border: 1px solid black; text-align: justify; }

:

Example

Now let's make an inner indent in30px:

<div id="elem"> some long text </div> #elem { padding: 30px; width: 300px; border: 1px solid black; text-align: justify; }

:

Example

Now the indentation above and below will be20px, and on the right and left -5px:

<div id="elem"> some long text </div> #elem { padding: 20px 5px; width: 300px; border: 1px solid black; text-align: justify; }

:

Example

Now indented from above5px, right -15px, below -25px, left -35px:

<div id="elem"> some long text </div> #elem { padding: 5px 15px 25px 35px; width: 300px; border: 1px solid black; text-align: justify; }
