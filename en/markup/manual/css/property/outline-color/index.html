outline-color property

Propertyoutline-colorsets the color of the outer frame - the border that does not take up space. The property value is anyunits for color. Default value: same color as the block text.

Syntax

селектор { outline-color: цвет; }

Example

Let's give the element a border:

<div id="elem"></div> #elem { outline-width: 1px; outline-style: solid; outline-color: red; width: 300px; height: 100px; }
