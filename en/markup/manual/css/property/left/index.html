left property

Propertyleftspecifies the left edge of the element. Values ​​for a property can be written in common units of size, such as pixels, inches, points, percentages. The specified values ​​can also be negative numbers, in which case the elements will overlap one another. The coordinate count is determined by the propertyposition, which usually takes the valuerelative(relative position) orabsolute(absolute position).

Syntax

селектор { left: размер или auto; }

Example

Let's hover over a child element and set its left border position to25px:

<div id="elem"> <p></p> </div> #elem { width: 400px; height: 300px; border: 1px solid black; } p { position: relative; width: 100px; height: 100px; border: 1px solid red; margin-left: 50px; } p:hover { left: 25px; width: 100px; height: 100px; border: 1px solid red; }

:

Example

Now let the left border of the child element be located on hover-50px:

<div id="elem"> <p></p> </div> #elem { width: 400px; height: 300px; border: 1px solid black; } p { position: relative; width: 100px; height: 100px; border: 1px solid red; margin-left: 50px; } p:hover { left: -50px; width: 100px; height: 100px; border: 1px solid red; }
