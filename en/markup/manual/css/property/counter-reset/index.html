The counter-reset property

Propertycounter-resetresets the counter to zero. This property must be used when working withcounter-increment, and it is installedfor parent.

Syntax

селектор { counter-reset: название [начальное значение] | none; }

Values

Meaning Description
Name An ordinary word (like a class name orid). You can specify multiple names, separating them with a space. In this case, several counters will change simultaneously.
initial value A positive or negative integer. Optional parameter.
none Disables incrementing of the counter for the current selector. Is the default value.
