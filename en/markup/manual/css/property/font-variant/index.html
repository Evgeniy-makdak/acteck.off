font-variant property

Propertyfont-variantallows you to make small (lowercase) letters CAPITAL, butsmall size. This entry is calledcapital.

Syntax

селектор { font-variant: small-caps | normal; }

Values

Meaning Description
small-caps Makes small letters in small capitals.
normal Cancels actionsmall-caps. This is usually necessary if you need to cancel capitals for a certain section of text (for example, all paragraphs are made with small caps, but this one is not).

Default value:normal.

Example. Meaning of small-caps

In this example, the letters will become capitalized, but at the same time small in size:

<p> Lorem ipsum dolor sit amet. </p> p { font-variant: small-caps; }

:

Compare the previous example (capitals), with the text written in capital letters:

<p> LOREM IPSUM DOLOR SIT AMET. </p>

:

Example. Normal value

And now there will be regular text:

<p> Lorem ipsum dolor sit amet. </p> p { font-variant: normal; }

:
