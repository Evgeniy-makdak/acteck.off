resize property

Propertyresizeprohibits or allows resizingtextareaor other elements (does not work with other elements in all browsers).

Syntax

селектор { resize: both | horizontal | vertical | none; }

Values

Meaning Description
both You can stretch the element in width and height.
horizontal You can only stretch an element along its width.
vertical You can only stretch an element in height.
none You cannot resize an element.

Default value:none. Fortextareain some browsers:both. In browsers where there is no supportresize, textareawill behave as if the value is givennone.

Example. Meaning both

The element stretches in both directions:

<textarea></textarea> textarea { resize: both; width: 200px; height: 200px; }

:

Example. The meaning of horizontal

The element only stretches horizontally:

<textarea></textarea> textarea { resize: horizontal; width: 200px; height: 200px; }

:

Example. The meaning of vertical

The element only stretches vertically:

<textarea></textarea> textarea { resize: vertical; width: 200px; height: 200px; }

:

Example. Value none

The element doesn't stretch at all:

<textarea></textarea> textarea { resize: none; width: 200px; height: 200px; }

:

Example. Width and Height Limit

You can limit the minimum and maximum width and height using propertiesmax-width, min-width, max-height, min-height:

<textarea></textarea> textarea { resize: both; min-height: 100px; max-height: 300px; min-width: 100px; max-width: 300px; }
