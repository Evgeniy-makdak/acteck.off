Flex property

Propertyflexabbreviation forflex-basis, flex-shrinkAndflex-grow.

Applies to:specific flex block.

The order doesn't matter. Second and third parameters (flex-shrink, flex-basis) are not required.

Values

See related properties.

Default value:0 1 auto.

Example

Let us have3element. The width of each of them is200pxand the total is equal600px, which is greater than the width of the parent container, which is equal to300px. Let's assign a width to the first element200px, for the second element -300px, for the third element -100px. Let all elements have a valueflex-basiswill be equal1, Aflex-shrink - 1, 2, 3in accordance with the serial number of the element:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> <div class="child elem2">3</div> </div> .parent { display: flex; width: 350px; height: 200px; border: 1px solid red; } .child { width: 200px; height: 50px; border: 1px solid green; } .elem1 { flex: 200px 1 1; } .elem2 { flex: 300px 1 2; } .elem3 { flex: 100px 1 3; }
