transition property

Propertytransition- shorthand for smooth transitions, allows you to simultaneously settransition-duration, transition-property, transition-timing-functionAndtransition-delay.

Syntax

селектор { transition: значение; }

The order of the properties does not matter, but the execution time (transition-duration) must come before the delay (transition-delay).

Values

See related properties. Default value:all 0s ease 0s.

Meaningnonecancels all transitions (they will happen instantly).

Example

Hover your mouse over the block -3nothing will happen for a second (there is a delay of 3s), and then it will smoothly change its width in2seconds. If you then remove the mouse, then again3nothing will happen for a second, and then the width will smoothly decrease to its original value:

<div id="elem"></div> #elem { border: 1px solid black; transition: width 2s ease 3s; width: 100px; height: 50px; } #elem:hover { width: 400px; }
