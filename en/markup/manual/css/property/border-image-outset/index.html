border-image-outset property

Propertyborder-image-outsetallows you to move the frame outside the element. The value is a number without specifying units of measurement or anyunits for sizesexcept interest. If a number without units is specified, then it is a multiplier by which property values ​​are multipliedborder-width. Negative values ​​are not supported.

For more information on pictures for the border, see the propertyborder-image.

Syntax

селектор { border-image-outset: положительное число; }

Example. Number

Set the valueborder-image-outsetV3by hovering the mouse over an element. In this case, the border will shift by26px*3- V3times further than its width specified inborder-width:

<div id="elem"></div> #elem:hover { border-image-outset: 3; } #elem { border-image-repeat: repeat; border-image-source: url("image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 100px auto; }

:

Example. Pixels

Set the valueborder-image-outsetV30pxby hovering the mouse over an element. In this case, the border will shift by30px:

<div id="elem"></div> #elem:hover { border-image-outset: 30px; } #elem { border-image-repeat: repeat; border-image-source: url("image.png"); border-image-slice: 26; border-style: solid; border-width: 26px; width: 200px; height: 200px; background: green; margin: 50px auto; }
