list-style property

Propertylist-styleallows you to simultaneously set the marker typelist-style-type, picturelist-style-imageand his positionlist-style-positionfor listsulorol.

The order of the properties does not matter.

Example

Let's set our list to square markers:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> ul { list-style: square outside; }

:

Example

Now let's set in the propertylist-styleimage for the marker and its location inside the text field:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> ul { list-style: url("li.gif") inside; }
