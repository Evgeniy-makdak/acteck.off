z-index property

Propertyz-indexsets who will be on top in case several elements overlap each other.

Syntax

селектор { z-index: число | auto; }

The number must be an integer, positive or negative. May be zero.

Values

Meaning Description
Number An integer specifies the order in which elements are stacked: when elements are stacked on top of each other, the one withz-indexmore.
auto The stacking order is built automatically: the element that is located below in the HTML code will be higher.

Default value:auto.

Example

In this example, the blocks will be superimposed on each other in the order they appear in the HTML code (z-indexis not specified and will have the default value -auto). The first block will be at the very bottom (red), and the last one will be at the top (green):

<div id="div1"></div> <div id="div2"></div> <div id="div3"></div> #div1 { position: absolute; top: 30px; left: 30px; width: 50px; height: 50px; background: red; } #div2 { position: absolute; top: 60px; left: 60px; width: 50px; height: 50px; background: blue; } #div3 { position: absolute; top: 90px; left: 90px; width: 50px; height: 50px; background: green; }

:

Example

Let's change the overlay order by settingz-index. To the red block -3, blue -2, green -1. The stacking order will be reversed (the block withz-index 3):

<div id="div1"></div> <div id="div2"></div> <div id="div3"></div> #div1 { position: absolute; top: 30px; left: 30px; width: 50px; height: 50px; background: red; z-index: 3; } #div2 { position: absolute; top: 60px; left: 60px; width: 50px; height: 50px; background: blue; z-index: 2; } #div3 { position: absolute; top: 90px; left: 90px; width: 50px; height: 50px; background: green; z-index: 1; }
