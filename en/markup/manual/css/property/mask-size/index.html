mask-size property

Propertymask-sizesets the size of the mask image. You can set specific sizes in pixels/percentages or use the cover and contain keywords.

Syntax

селектор { mask-size: auto | <length> | <percentage> | cover | contain; }

Values

Meaning Description
auto Original image size (default).
cover Scales the mask while maintaining its proportions so that it completely covers the element.
contain Scales the mask while maintaining its proportions so that it fits completely within the element.
100px 50px Specific dimensions in pixels.
50% 100% Dimensions as a percentage of the element dimensions.

Example. cover size

The mask scales to completely cover the element:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: cover; mask-repeat: no-repeat; }

:

Example. Contain size

The mask is scaled to fit entirely within the element:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: contain; mask-repeat: no-repeat; }

:

Example. Fixed size in pixels

Mask size 150x150 pixels:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: 150px; mask-repeat: no-repeat; }

:

Example. Size in percentage

The mask takes up 80% of the element's width and 60% of the element's height:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: 80% 60%; mask-repeat: no-repeat; }

:

Example. Different sizes along the axes

Mask 200px wide and 100px high:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask-image: url("heart.svg"); mask-position: center; mask-size: 200px 100px; mask-repeat: no-repeat; }

:
