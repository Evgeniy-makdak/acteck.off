outline-offset property

Propertyoutline-offsetspecifies the indentation of the outer frame - a border that does not take up space. The property value is anyunits for sizesexcept interest. Default value:0. A positive value moves the frame outward from the element, a negative value moves the border inward.

Syntax

селектор { outline-offset: значение; }

Example. Positive value

The frame moves away from the element (it is highlighted in the background):

<div id="elem"></div> #elem { outline-offset: 5px; outline-width: 1px; outline-style: solid; outline-color: black; background-color: #e4f1ed; width: 300px; height: 100px; }

:

Example. Negative value

The frame is inside the element:

<div id="elem"></div> #elem { outline-offset: -10px; outline-width: 1px; outline-style: solid; outline-color: black; background-color: #e4f1ed; width: 300px; height: 100px; }

:

Example. Negative outline + border

You can create such an interesting effect (pay attention to the thickness of the white space between the borders, it10px, not14because I ate some of itoutlinewith its thickness of4px):

<div id="elem"></div> #elem { outline-offset: -14px; outline: 4px solid green; border: 4px solid red; width: 300px; height: 100px; }

:
