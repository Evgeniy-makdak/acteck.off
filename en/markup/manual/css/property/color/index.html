color property

Propertycolorsets the color of the text. The property value is anyunits for color. The default value is usually black.

Syntax

селектор { color: цвет; }

Example

Let's give the element a red color:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { color: red; }

:

Example

Propertycolorsets the color not only for the text, but also for the list markers:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li { color: red; }

:

Example

Propertycoloralso sets the border color (if it is not specified inborder-color):

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { color: red; border-style: solid; border-width: 2px; }
