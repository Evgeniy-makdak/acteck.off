clear property

Propertyclearcancels the wrapping of one element by another, specified by the propertyfloat. Can take the following values:none(cancels the action of itself),both(cancelsfloatsimultaneously right and left)left(cancelsfloatleft),right(cancelsfloatright).

Syntax

селектор { clear: none или left или right или both }

Example

Let's stop the text from wrapping around the left side of the image:

<img class="image" src="bg.png" alt=""> <div class="txt"> <p> text text text text text text text text text text text text text text text text text text text text text text text text </p> </div> .image { float: left; padding-right: 10px; } .txt { clear: left; }

:

Example

Now let's stop the text from wrapping on the right side of the image:

<img class="image" src="bg.png" alt=""> <div class="txt"> <p> text text text text text text text text text text text text text text text text text text text text text text text text </p> </div> .image { float: right; padding-right: 10px; } .txt { clear: right; }
