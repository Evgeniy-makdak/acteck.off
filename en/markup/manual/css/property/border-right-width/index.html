border-right-width property

Propertyborder-right-widthsets the thickness of the left border. The property value is anyunits for sizes, except percentages, or keywordsthin(border in2pixel),medium(border in4pixel) orthick(border in6pixels). Default value:medium.

Syntax

селектор { border-right-width: значение; }

Example

Let's set the right border to one pixel thick:

<div id="elem"></div> #elem { border-right-width: 1px; border-right-style: solid; border-right-color: black; background: #f9f9f9; width: 300px; height: 100px; }
