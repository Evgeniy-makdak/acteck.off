transition-timing-function property

Propertytransition-timing-functionsets the rate of change (acceleration) of the smooth transitiontransition. For example, first slowly, then quickly, then slowly, etc.

Syntax

селектор { transition-timing-function: значение; }

Values

Meaning Description
ease Slowly at first, then quickly, and finally slowly again.
ease-in Starts off slowly and gradually speeds up.
ease-out Starts quickly and gradually stops.
ease-in-out Slowly at first, then quickly, and finally slowly again. It differs from ease in speed.
linear Always the same speed.
step-start There is no animation; the property immediately takes on its final value.
step-end There is no animation, the property waits for the time specified intransition-duraton, and then instantly takes on its final value.
steps The property value changes intermittently.
cubic-bezier A Bezier curve that can define arbitrary animation. Cm.Bezier curve generator.

Default value:ease.
