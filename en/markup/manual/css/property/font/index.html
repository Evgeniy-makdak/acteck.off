The font property

Propertyfontis a shorthand property in which you can use the following CSS properties:font-size, font-style, font-weight, line-height, font-family, font-variant.

Syntax

селектор { font: [font-style|font-variant|font-weight] font-size [/line-height] font-family; }

The required properties arefont-sizeAndfont-family, the rest can be omitted while maintaining order.

Propertiesfont-style, font-variant, font-weightcan be rearranged in any order relative to each other.

Please note that the propertyline-heightwritten with a slash after the propertyfont-size.

Example

Let's make italic text size15px, with type Arial, and line spacing in30px:

<p> some long text </p> p { font: italic 15px/30px Arial; text-align: justify; }

:

Example

Another example:

<p> some long text </p> p { font: bold 16px/20px "Times New Roman", Georgia, serif; text-align: justify; }

:

Example

Let's assign only the font size and family, omitting other properties:

<p> some long text </p> p { font: 18px Arial; text-align: justify; }

:

Example

If you set only the font size, it will not be applied, since the font family must also be specified:

<p> some long text </p> p { font: 30px; text-align: justify; }
