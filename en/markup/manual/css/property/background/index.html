background property

Propertybackgroundsets all background properties at once:background-attachment, background-color, background-image, background-position, background-repeat, background-size, background-origin, background-clip.

The order of the properties does not matter. Unneeded properties can be omitted - in this case it will take the default value. Propertiesbackground-positionAndbackground-sizeare specified through a slash and in this case the order matters (first the position, then the size).

Also propertybackgroundtakes on the valuenone, corresponding to the absence of background.

Syntax

селектор { background: сокращаемые свойства; }
