transform property

Propertytransformspecifies element transformations, such asrotation, scaling, bevel, shiftand others. The corresponding transformations are written separated by a space:

селектор { transform: rotate(30deg) scale(3); }

Example

In this example the block is rotated to30degrees clockwise:

<div id="elem">lorem ipsum</div> #elem { transform: rotate(30deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, in addition to rotation, a bevel is also added:

<div id="elem">lorem ipsum</div> #elem { transform: rotate(30deg) skew(20deg); border: 1px solid black; width: 100px; height: 50px; }

:

Example

In this example, in addition to rotation and bevel, a scale change is added:

<div id="elem">lorem ipsum</div> #elem { transform: rotate(30deg) skew(20deg) scale(2, 3); border: 1px solid black; width: 100px; height: 50px; }
