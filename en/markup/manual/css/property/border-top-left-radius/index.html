border-top-left-radius property

Propertyborder-top-left-radiusCreates a fillet for the top left corner of the border and background. The property value is anyunits for sizes. Default value:0.

Syntax

селектор { border-top-left-radius: значение; }

Example

Set the rounding10pxfor top left corner:

<div id="elem"></div> #elem { border-top-left-radius: 10px; border: 1px solid black; width: 300px; height: 100px; }
