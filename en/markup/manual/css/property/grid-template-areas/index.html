grid-template-areas property

Propertygrid-template-areasspecifies the placement of elements in the grid. First, we give each descendant element a name using the propertygrid-area. Then on the parent element we specify the propertygrid-template-areas, in which we list the names of descendant elements in the order in which we want to place them in the grid.

Syntax

селектор { grid-template-areas: имена элементов-потомков; }

Example

Let's set the location of our elements in the grid:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #elem1 { grid-area: first; } #elem2 { grid-area: second; } #elem3 { grid-area: third; } #parent { display: grid; grid-template-areas: 'second first third'; border: 2px solid #696989; padding: 10px; } #parent > div { padding: 10px; border: 1px solid #696989; }

:

Example

Now let's replace the names of some elements with a dot'.'. As can be seen from the result of the executed code, in this case an empty block will be placed in the diagram:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #elem1 { grid-area: first; } #elem2 { grid-area: second; } #elem3 { grid-area: third; } #parent { display: grid; grid-template-areas: 'first . second third'; border: 2px solid #696989; padding: 10px; } #parent > div { padding: 10px; border: 1px solid #696989; }
