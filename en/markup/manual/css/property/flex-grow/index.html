flex-grow property

Propertyflex-growdetermines how much larger an individual flex block can be than neighboring elements, if necessary.

For example, if all flex boxes inside a flex container haveflex-grow:1, then they will be the same size. If one of them hasflex-grow:2, then he will be in2times more than everyone else.

If the total width of the elements is less than the width of the parent, therefore there is empty space on the right. If desired, this empty space can be proportionally divided between our elements. This is done using the propertyflex-grow, given to flex elements. The value of this property is a dimensionless number.

Applies to:specific flex block.

This property is included as an integral part of the abbreviation propertyflex.

Syntax

селектор { flex-grow: положительное число; }

Default value:0.

Example

Now we have two flex blocks with a width of100px. Their total width of elements200px, and the width of the parent is300px. It turns out that there is free space left in100px.

If the elements are not givenflex-grow, then we'll just see this free space. If it is given to them, then the actual width of the elements will be greater than the specified one - they will proportionally divide the free space among themselves and add it to their width.

Let, for example, the first elementflex-growequals1, and the second -3. Let's calculate how much free space each element will receive.

First you need to get the total number of unitsflex-growall our elements. For the first element it is equal1, and in the second -3. This means that in total it is equal to4.

Let's divide now100pxfree space on4and we get that25pxaccounts for one unitflex-grow. It turns out that one unit will be added to the first elementflex-grow, that is25px, and to the second - three units, that is75px.

The width of the first element will be125px, and the second -175px:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> </div> .parent { display: flex; width: 300px; height: 200px; border: 1px solid red; } .child { height: 50px; border: 1px solid green; } .elem1 { width: 100px; flex-grow: 1; } .elem2 { width: 100px; flex-grow: 3; }
