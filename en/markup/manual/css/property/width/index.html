width property

Propertywidthsets the width of the element. The property value is anyunits for sizesor key propertyauto, which allows the browser to calculate the element's size on its own (the default behavior).

Syntax

селектор { width: значение; }

Example

Set the width of the element to300px:

<div id="elem"></div> #elem { width: 300px; height: 100px; border: 1px solid black; }

:

Example

Let's make the element take70%from the width of its parent (in our case this will be the browser window):

<div id="elem"></div> #elem { width: 70%; height: 100px; border: 1px solid black; }

:
