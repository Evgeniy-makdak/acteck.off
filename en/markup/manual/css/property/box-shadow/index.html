box-shadow property

Propertybox-shadowsets the shadow of the block. The property takes as its value6parameters listed separated by a space or keywordnone, canceling the shadow completely.

Syntax

селектор { box-shadow: inset сдвиг_по_x сдвиг_по_y размытие размер_тени цвет; }
селектор { box-shadow: none; }

Options

Parameter Description
inset Optional parameter.
If it is specified, then the shadow will be inside the container; if not specified, then it will be outside.
x-shift Sets the shadow offset along the X axis.
A positive value shifts to the right, a negative value shifts to the left.
y shift Sets the shadow offset along the Y axis.
A positive value moves down, a negative value moves up.
blur Sets the shadow blur.
The higher the value, the more blurry the shadow will be.
Optional parameter. If not specified, the shadow will be clear.
shadow size Sets the size of the shadow.
A positive value stretches the shadow, a negative value, on the contrary, compresses it.
Optional parameter. If not specified, the shadow will be the same size as the element.
color Sets the shadow color in anyunits for color.
Optional parameter. If not specified, the shadow color matches the text color.

Axis offsets, blur and shadow size can be set in anyunits for sizesexcept interest.

Example

In this example, the shadow is moved down and to the right and a slight blur is added:

<div id="elem"></div> #elem { box-shadow: 5px 5px 3px black; border: 1px solid black; width: 300px; height: 50px; }

:

Example. Clear shadow

In this example, the shadow is moved down and to the right, but there is no blur (the shadow will be clear):

<div id="elem"></div> #elem { box-shadow: 2px 2px black; border: 1px solid black; width: 300px; height: 50px; }

:

Example. Uniform shadow

In this example, the shadow is not moved, but a blur has been added to it:

<div id="elem"></div> #elem { box-shadow: 0px 0px 3px black; border: 1px solid black; width: 300px; height: 50px; }

:

Example. Shadow size

In this example, the shadow is not moved, the blur is zero, but dimension is added to it (black is the border, red is the shadow):

<div id="elem"></div> #elem { box-shadow: 0 0 0 3px red; border: 3px solid black; width: 300px; height: 50px; }

:

Example. Blur + shadow size

In this example, the shadow is not moved, but blur and size have been added to it (black is the border, red is the shadow):

<div id="elem"></div> #elem { box-shadow: 0 0 3px 3px black; border: 1px solid black; width: 300px; height: 50px; }

:

Example. Inner shadow

In this example, the shadow is inside the container:

<div id="elem"></div> #elem { box-shadow: inset 0 0 6px black; border: 1px solid black; width: 300px; height: 50px; }

:
