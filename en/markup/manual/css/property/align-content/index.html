align-content property

Propertyalign-contentsets the alignment of elements along the transverse axis for flex blocks and along the vertical axis for grids. Applies to the parent element.

Syntax

селектор { align-content: flex-start | flex-end | center | space-between | space-around; }

Values

Meaning Description
flex-start The blocks are pressed to the beginning of the transverse (vertical) axis.
flex-end The blocks are pressed against the end of the transverse (vertical) axis.
center The blocks stand in the center of the transverse (vertical) axis.
space-between The blocks are distributed along the transverse (vertical) axis, with the first element pressed to the beginning of the axis, and the last to the end.
space-around The blocks are distributed along the transverse (vertical) axis, with the same gap between the first block and the beginning of the axis, the last block and the end of the axis as between the other blocks.
However, they are not equal as it might seem: the gaps are summed up and the distance between two blocks is twice as large as between the block and the start/end of the axis.

Example. Flex-start meaning

In this example, the axis along which the alignment occurs is directed from top to bottom, i.e. it is transverse. As you can see from the result, all our elements are pressed to its top part:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: flex; align-content: flex-start; flex-wrap: wrap; height: 200px; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Flex-end value

In this example, the blocks are pressed against the underside of the transverse axis, since the propertyalign-contentgiven in valueflex-end:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: flex; align-content: flex-end; flex-wrap: wrap; height: 200px; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Meaning of center

Now the blocks are aligned to the center of the transverse axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: flex; align-content: center; flex-wrap: wrap; height: 200px; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Alignment to the origin of the vertical axis in the grid

Let's set the alignment of our elements in the grid to the beginning of the vertical axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: grid; align-content: start; grid-template-columns: 100px 100px; gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

:

Aligning to the center of the vertical axis in the grid

Now let's set the alignment of the elements to the center of the vertical axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: grid; align-content: center; grid-template-columns: 100px 100px; gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

:

Aligning to the end of the vertical axis in the grid

Let's set the alignment of the elements to the end of the vertical axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: grid; align-content: end; grid-template-columns: 100px 100px; gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }
