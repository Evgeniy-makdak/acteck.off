Hyphens property

PropertyhyphensEnables word hyphenation in the browser using the hyphen "-". To support the Russian language, you need to add the attributelang="en"to the tag to which hyphens are applied (or to the parent of that tag). You can simply add this attribute to the taghtml. Without this, transfers will not work.

Syntax

селектор { hyphens: auto | manual | none; }

Values

Meaning Description
auto Places transfer signs where necessary.
manual Adds hyphens only in those places where it is addedwbrtag or special character&shy;and only if necessary. Such transfers are calledsoft.
none Cancels hyphenation (words will not be hyphenated, even if there are soft hyphens).

Default value:manual.
