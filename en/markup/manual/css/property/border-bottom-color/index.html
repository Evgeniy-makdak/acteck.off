Свойство border-bottom-color

Свойство border-bottom-color задает цвет нижней границы. Значением свойства служат любые единицы для цвета. Значение по умолчанию: такой же цвет, как и у текста блока.

Синтаксис

селектор { border-bottom-color: цвет; }

Пример

Зададим нижнюю границу красного цвета:

<div id="elem"></div> #elem { border-bottom-color: red; border-bottom-width: 1px; border-bottom-style: solid; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-bottom-style,
    которое задает тип нижней границы
  • свойство border-bottom-width,
    которое задает толщину нижней границы
  • свойство border-bottom,
    которое задает границу для нижней стороны
  • свойство border-width,
    которое задает толщину всех сторон границы
