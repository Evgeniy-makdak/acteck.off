align-items property

Propertyalign-itemssets the alignment of elements along the transverse axis for flex blocks and along the vertical axis for grids. Applies to the parent element.

Syntax

селектор { align-items: flex-start | flex-end | center | baseline | stretch; }

Values

Meaning Description
flex-start The blocks are pressed to the beginning of the transverse (vertical) axis.
flex-end The blocks are pressed against the end of the transverse (vertical) axis.
center The blocks stand in the center of the transverse (vertical) axis.
baseline Elements are aligned to their baseline. The baseline is an imaginary line that runs along the bottom edge of characters without taking into account overhangs, such as those of letters'p'And'y'.
stretch The blocks are stretched, taking up all the available space along the transverse axis, while still being taken into accountmin-widthAndmax-width, if they are specified. If the width and height for the elements are specified -stretchwill be ignored.

Default value:stretch.

Example. Meaning of stretch

Now the main axis is directed from left to right, and along the transverse axis the elements are stretched to their full height:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: stretch; flex-direction: row; border: 1px solid #696989; } #parent > div { border: 1px solid #696989; }

:

Example. stretch value + element dimensions

Now the elements are given width and height, so the valuestretchfor propertyalign-itemswill be ignored:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: stretch; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Flex-start value without element dimensions

Now the elements will be pressed to the top:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: flex-start; flex-direction: row; border: 1px solid #696989; } #parent > div { border: 1px solid #696989; }

:

Example. flex-start value + element dimensions

Now the elements will still be pressed to the top, but they will have the specified width and height:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: flex-start; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Flex-end value + element dimensions

Now the elements will be pressed to the bottom:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: flex-end; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Center value + element dimensions

Now the elements will be centered along the transverse axis (in this case, vertical):

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: center; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 100px; height: 50px; border: 1px solid #696989; }

:

Example. Center value, elements of different sizes

Now the elements have different sizes in height (currently they are moved apart by text, but you can also setheight), all have the same width, since the property is setwidth:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: center; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 100px; border: 1px solid #696989; }

:

Example. Baseline value, elements of different sizes

And this is what baseline alignment looks like:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: flex; align-items: baseline; flex-direction: row; border: 1px solid #696989; } #parent > div { width: 130px; line-height: 1; border: 1px solid #696989; }

:

Example. Alignment to the origin of the vertical axis in the grid

Let's align our elements inside the cells to the beginning of the vertical axis:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: grid; align-items: flex-start; grid-template-columns: 100px 100px; gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }
