Unit px

Absolute unit of measurementpxrepresents a cell on the matrix of an electronic device. For example, if the width of the smartphone screen is480px, this means that the screen is located480cells, each of which is equal1px.

Example

Let's give the rectangle a width of200pxand height100px:

<div> </div> div { border: 1px solid black; width: 200px; height: 100px; }

:

Example

Let's set the paragraph text to font30pixels:

<p> text </p> p { font-size: 30px; }

:
