Unit %

Relative unit of measurement%sets the size of the child element relative to the size of its parent. For example, the width of a rectangle is equal to50%means that it is half the size of its parent.

Example

Let's give the child element's rectangle a width of20%and height40%:

<div> <p></p> </div> div { border: 1px solid black; width: 200px; height: 100px; } p { border: 1px solid red; width: 20%; height: 40%; }
