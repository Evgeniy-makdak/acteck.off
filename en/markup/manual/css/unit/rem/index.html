Unit rem

Relative unit of measurementremsets the font size of an element depending on the font of the root element -<html>. For example, a font equal to2remmeans it is twice the size of the root element.

Example

Let's give the title text a font one and a half times larger than the font of the root element, and the paragraph -1rem:

<h1>Text1</h1> <p>text2</p> body{ font-size: 14px; } h1 { font-size: 1.5rem; } p { font-size: 1em; }
