Unit em

Relative unit of measurementemsets the font size of a child element based on the size of its parent. For example, a font equal to2emmeans that it is twice the size of its parent.

Example

Let's set the element's text<p>font is one and a half times larger than its parent's font<div>:

<div> text1 <p> text2 </p> </div> div { font-size: 14px; } p { font-size: 1.5em; }
