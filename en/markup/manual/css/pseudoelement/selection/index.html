The selection pseudo-element

Pseudo-elementselectionallows you to set the color and background of the selected text.

Syntax

селектор::selection { }

Example

Select the text and you will see that it will be red on a black background:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem::selection { color: red; background-color: black; }

:

See also

  • propertyuser-select,
    which disables text selection
  • propertypointer-events,
    which removes the reaction to the mouse
