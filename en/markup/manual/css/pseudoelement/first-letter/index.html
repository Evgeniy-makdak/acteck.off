first-letter pseudo-element

Pseudo-elementfirst-letterallows you to style the first letter of text in an element.

Syntax

селектор::first-letter { }

Example

Let's apply styles to the first letters of each paragraph:

<p> Lorem ipsum dolor sit amet. </p> <p> Lorem ipsum dolor sit amet. </p> <p> Lorem ipsum dolor sit amet. </p> p::first-letter { color: red; font-size: 27px; }
