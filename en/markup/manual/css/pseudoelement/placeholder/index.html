Placeholder pseudo-element

Pseudo-elementplaceholdersets tooltip stylesplaceholderin tagsinputortextarea.

Syntax

селектор::placeholder { color: red; }

Example

Let's color the tooltip red:

<input type="text" placeholder="подсказка"> input::placeholder { color: red; }

:

Example. Tooltip in textarea

Let's color the tooltip red in the tagtextarea:

<textarea placeholder="подсказка"></textarea> textarea::placeholder { color: red; }

:

See also

  • attributeplaceholder,
    which sets the tooltip for the input field
  • pseudo-classplaceholder-shown,
    which sets styles for the input field with a tooltip
