The before pseudo-element

Pseudo-elementbeforeinserts text before the element. Can only be used in conjunction with the propertycontent, which specifies the text to be inserted.

Syntax

селектор::before { content: 'текст'; }

Example

Let's make sure that by pointing atlithe text was inserted at the beginning:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li:hover::before { content: '!!!'; }

:

Example

You can apply different styles to pasted text. Let's paint, for example, this text some color:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li:hover::before { color: red; content: '!!!'; }

:
