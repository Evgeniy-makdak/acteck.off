import command

Team@importis intended for importing a CSS file into the current style sheet. In this command you can specify what types of media the import is intended for:all(all types, default),print(printing devices),screen(monitor screen).

Syntax

import url('имя файла') [типы носителей]; @import 'имя файла' [типы носителей];

See also

  • team@media,
    which specifies the style for the media type
  • team@supports,
    which sets the style for elements whose capabilities are supported by the browser
  • team!important,
    which sets style priority
