Media Team

Team@mediais intended for importing a CSS file into the current style sheet. The following media types can be specified in this command:all(all types, default),print(printing devices),screen(monitor screen),speech(speech synthesizers).

Syntax

media тип носителя { }

See also

  • team@import,
    which imports the CSS file
  • team@supports,
    which sets the style for elements whose capabilities are supported by the browser
  • team!important,
    which sets style priority
