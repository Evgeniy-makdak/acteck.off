!important command

Team!importantgives priority to the style or any CSS property for which it is specified.

Syntax

селектор { свойство: значение !important; }

Example

Let's make the blue paragraph font color take precedence over the green color:

<div>text</div> div{ color: blue !important; width: 600px; height: 100px; font-size: 20px; } div { color: green; }

:

  • team@import,
    which imports the CSS file
  • team@media,
    which specifies the style for the media type
  • team@supports,
    which sets the style for elements whose capabilities are supported by the browser
