Testing the compiler in LESS

Let's now test the compiler. To do this you need to make a file with LESS test code. This file must have the extension.less. Let's add the following content to it:

div { p { font-size: 20px; } }

Following the instructions from the documentation of the compiler you installed, compile your.lessfile in.css. As a result, you should see the following code:

div p { font-size: 20px; }

Perform the described manipulations. Make sure your compiler is working.
