CSS Keywords and LESS Variables

There are keywords and commands in CSS that start with dog, for example,@media, @keyframesand others. LESS can process them correctly and does not perceive them as commands. See example:

@media (max-width: 600px) { div { width: 300px; } }

Another example:

@keyframes anim { from { width: 100px; } to { width: 200px; } }

Write some media queries. Make sure they work correctly.
