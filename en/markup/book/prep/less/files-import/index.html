Импортирование файлов LESS

Можно разделять код LESS по нескольким файлам, а затем собирать их вместе в один CSS файл. Это делается с помощью команды @import.

Давайте посмотрим на примере. Пусть у нас есть файл lib.less, в котором мы будем размещать вспомогательные функции:

.mix() { color: red; }

Давайте импортируем этот файл к нашему основному файлу styles.less:

@import url('lib.less');

После импорта в нашем файле становятся доступны все данные из импортированного файла. Поэтому мы можем воспользоваться примешниванием .mix:

div { .mix; }

Сделайте файл с библиотекой полезных функций. Импортируйте его к вашему основному файлу.
