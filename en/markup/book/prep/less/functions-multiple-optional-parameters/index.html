Several optional parameters in functions in LESS

If a function has several parameters, then the optional parameters should be placed at the end. Let's look at an example.

Let us have a function that sets the text color and at the same time the background color:

.color(@c1, @c2) { color: @c1; background-color: @c2; }

Let's make the second parameter optional:

.color(@c1, @c2: white) { color: @c1; background-color: @c2; }

Now let's make the first parameter optional:

.color(@c1: black, @c2: white) { color: @c1; background-color: @c2; }
