Media queries in LESS

Let's make a function that will add to an elementmedia query. Let's take this example as a query that regulates styles for mobile devices. Let's name our function accordingly:

.mobile(@code) { }

Our function will take a block of code as a parameter. Let's output this block inside a media query:

.mobile(@code) { @media (max-width: 600px) { @code(); } }

Now let's use our function:

div { .mobile({ width: 300px; }); }

As a result, after compilation, our div will be inside the media query, receiving the appropriate styles:

@media (max-width: 600px) { div { width: 300px; } }
