Variables inside selectors in LESS

Variables can also be used inside selectors. To do this, however, you need to use a slightly different syntax for inserting a variable: after@The variable name must be enclosed in curly braces. Let's look at examples. Let us have the following variable:

@var: div;

Let's insert this variable into the selector. Let's consider different situations.

Example

Let's insert our variable as a selector:

@{var} { font-size: 20px; }

Compilation result:

div { font-size: 20px; }

Example

Let our variable now contain part of the selector:

main @{var} { font-size: 20px; }

Compilation result:

main div { font-size: 20px; }

Example

Let us now say that our variable contains not the name of the tag, but itsid. Add a hash in front of the variable name:

#@{var} { font-size: 20px; }

Compilation result:

#div { font-size: 20px; }

Example

Let us now say that our variable contains not the name of the tag, but its class. Add a dot before the variable name:

.@{var} { font-size: 20px; }

Compilation result:

.div { font-size: 20px; }

Practical problems

Tell us what the result of compiling the following code will be:

@var: ul; @{var} { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: ul; div @{var} { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: ul; div @{var} { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: ul; @{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: list; .@{var} { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: list; #@{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: list; .@{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: list; ul.@{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: list; ul.@{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: .block; ul@{var} { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: .block; ul@{var} li { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@var: a; @{var}:hover { text-decoration: none; }

Tell us what the result of compiling the following code will be:

@var1: div; @var2: a; @{var1} @{var2}:hover { text-decoration: none; }

Tell us what the result of compiling the following code will be:

@var1: block1; @var2: block2; .@{var1}.@{var2} { font-size: 20px; }
