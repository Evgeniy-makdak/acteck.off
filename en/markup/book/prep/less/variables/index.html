Variables in LESS

The LESS language allows you to usevariables. A variable is a container in which you can store some data and use it in the right places in the code.

To create a variable, you need to come up with its name and put a sign in front of it@. Then after the variable name we put a colon and indicate the value of this variable. Let's make an example variable with the namevar:

@var: 20px;

Let's now use this variable as a property value:

p { font-size: @var; }

As a result, after compilation in CSS, our variable will be replaced with its value:

p { font-size: 20px; }

Tell us what the result of compiling the following code will be:

@color: #000; p { color: @color; }
