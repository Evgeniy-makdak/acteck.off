Using a Code Block for Pseudo-Classes in LESS

Let's make a function that will set link styles in the statehover.

Let this function take a block of code as a parameter:

.hover(@code) { }

Let's output the passed block of code inside the statehover:

.hover(@code) { &:hover { @code(); } }

Let's use our function inside the link:

a { color: red; .hover({ color: blue; text-decoration: none; }); }

Compilation result:

a { color: red; } a:hover { color: blue; text-decoration: none; }
