Multiple parameters in LESS functions

Functions in LESS can pass multiple parameters. To do this, they need to be listed separated by commas. For example, let's make a functionsize, specifying the dimensions of the element:

.size(@w, @h) { width: @w; height: @h; }

Let's use our function:

div { .size(200px, 100px); }

The result after compilation is the following code:

div { width: 200px; height: 100px; }

Make a function.fsh, which will take the property value as its first parameterfont-size, and the second is the property valueline-height.

Make a function.absTL, which will specify absolute positioning. In this case, its first parameter will take the value of the propertytop, and the second is the property valueleft.

Make similar functions.absTR, .absBLAnd.absBR, specifying absolute positioning from different sides.
