Несколько параметров в функциях LESS

В функциях в LESS можно передавать несколько параметров. Для этого их нужно перечислить через запятую. Для примера давайте сделаем функцию size, задающую размеры элемента:

.size(@w, @h) { width: @w; height: @h; }

Воспользуемся нашей функцией:

div { .size(200px, 100px); }

В результате после компиляции получится следующий код:

div { width: 200px; height: 100px; }

Сделайте функцию .fsh, которая первым параметром будет принимать значение свойства font-size, а вторым - значение свойства line-height.

Сделайте функцию .absTL, которая будет задавать абсолютное позиционирование. Пи этом она первым параметром будет принимать значение свойства top, а вторым - значение свойства left.

Сделайте аналогичные функции .absTR, .absBL и .absBR, задающие абсолютное позиционирование с различных сторон.
