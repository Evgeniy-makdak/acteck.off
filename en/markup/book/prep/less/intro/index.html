Introduction to LESS

The LESS language is an add-on to the CSS language that adds additional features to it that simplify your work.

Technically LESS ispreprocessor. This means that you will write code in LESS, and a special program will convert this code into CSS, and you will connect the result of the conversion to an HTML file.

Special programs are used to convert the codecompilers. There are quite a lot of such programs and you can choose one of them to suit your taste. To get started, I suggest you use one of the following programs:KoalaorPrepros.
