Grouping selectors in LESS

If nesting involves grouping of selectors, then LESS will correctly expand this construct during compilation. Let's see in practice. Here is an example of a grouping:

#header { .menu, .logo { color: red; } }

As a result, when compiled, you will get the following code:

#header .menu, #header .logo { color: red; }

Tell us what the result of compiling the following code will be:

#block { ul, ol { color: blue; li { font-size: 20px; } } }
