Optional parameter in functions in LESS

Parameters in functions may be optional. To do this, they need to set a default value. In this case, when calling the function, we can not specify the value of the parameter - and the specified default value will be taken. Let's look at an example.

Let's say we have a function that sets the color of an element:

.color(@c) { color: @c; }

Let's make the color parameter optional and default tored:

.color(@c: red) { color: @c; }

Let's now call our function with a parameter:

div { .color(green); }

Now let's call our function without a parameter:

div { .color(); }
