Comments in LESS

There are two types of comments in LESS: multi-line and single-line. Let's study their features.

Multiline comments will be the same as in CSS:

/* комментарий */ div { font-size: 20px; }

Or like this:

div { font-size: 20px; /* комментарий */ }

LESS also adds one-line comments. These comments are more convenient because they are easier to write, and they do not end up in the compiled code. See example:

div { font-size: 20px; // комментарий }

Make two types of comments in your code. Compile the code. Look at the compilation result.
