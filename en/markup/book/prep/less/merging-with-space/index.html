Joining via space in LESS

There are properties whose values ​​need to be combined not through a comma, but through a space. There is a special operator for such unions+_.

Let's look at how it works using the property exampletransform. Let's make functions.rotateAnd.skew, defining the corresponding transformations:

.rotate(@r) { transform+_: rotate(@r); } .skew(@s) { transform+_: skew(@s); }

Let's use our functions:

div { .rotate(10deg); .skew(10deg); }

As a result of compilation, our transformations will be combined into one propertytransform:

p { transform: rotate(10deg) skew(10deg); }
