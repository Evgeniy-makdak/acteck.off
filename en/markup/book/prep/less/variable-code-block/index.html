Code block in LESS variable

You can write not only a value, but an entire block of code to a variable, like this:

@test: { width: 100px; height: 100px; }

In order to display a block of code written in a variable, you need to write the name of this variable and parentheses, like this:

div { @test(); }

As a result, after compilation, we will get the following code:

div { width: 100px; height: 100px; }
