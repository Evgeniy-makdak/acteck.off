Global variables inside selectors in SASS

If we set variables located inside selectors, then we can use them only within the specified nesting level. However, if we need a variable to be available globally, then we add a label to its value!global. Let's look at an example:

div { $height: 20px !global; height: $height; } #navbar { height: $height; }

The result of compiling the above code will be as follows:

div { height: 20px; } #navbar { height: 20px; }

Tell us what the result of compiling the following code will be:

#main { $font-family: Arial !global; font-family: $font-family; } #navbar { font-family: $font-family; }
