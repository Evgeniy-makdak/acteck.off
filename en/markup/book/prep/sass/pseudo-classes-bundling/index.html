Connecting Pseudo-Classes in SASS

The ampersand is convenient to use when working with pseudo-classes. See example:

a { color: blue; &:hover { color: red; } }

Compilation result:

a { color: blue; } a:hover { color: red; }

Tell us what the result of compiling the following code will be:

a.class { &:link { color: red; } &:visited { color: blue; } &:hover { color: green; } }

Simplify the given code:

#block { width: 300px; } #block a:link, #block a:visited { color: red; } #block a:hover { color: blue; }
