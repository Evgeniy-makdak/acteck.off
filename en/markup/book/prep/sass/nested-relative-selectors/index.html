Related selectors when nesting SASS

Let's look at how to add various selectors in SASS when nesting, selecting elements by relatedness. See examples.

Example

Child selector:

div { > p { color: red; } }

Compilation result:

div > p { color: red; }

Example

Adjacent selector

div { + p { color: red; } }

Compilation result:

div + p { color: red; }

Example

Related selector:

div { ~ p { color: red; } }

Compilation result:

div ~ p { color: red; }

Practical problems

Tell us what the result of compiling the following code will be:

ul { color: red; > li { padding: 20px; } }

Tell us what the result of compiling the following code will be:

div { ul { > li { padding: 20px; } } }
