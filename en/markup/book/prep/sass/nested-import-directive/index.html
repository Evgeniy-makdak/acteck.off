Nested import in SASS

Usually@importare used at the beginning of the document, but the SASS preprocessor allows you to import files into any parts of the file we need, including inside selectors.

Consider the following example:

.active { color: blue; }

Next, import it into the filestyle.scss:

div { @import "main"; }

Now let's look at the compilation result:

div .active { color: blue; }

Create a filenavbar.scsswith the following content:

.link { color: red; font-size: 12px; }

Then import this fragment into a block#mainfilestyles.scssand compile tostyles.css.
