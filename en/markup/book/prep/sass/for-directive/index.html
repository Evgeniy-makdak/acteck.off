@for directive in SASS

Using a directive@forYou can output certain styles in the amount specified by the counter variable. First recording option@forlooks like this:

@for $var from (begin) through (end)

In this case, the range strictly includes the values ​​(begin) and (end). It is worth paying attention to the fact that (begin) and (end) must return integers, since they specify the direction of counting, i.e. if (begin) is less than (end), the counter will increase and, accordingly, if on the contrary, it will decrease.

Let's look at how the directive works@forin the first version of the syntax using the following example:

@for $count from 1 through 5 { .li-#{$count} { width: $count + px; } }

Compilation result:

.li-1 { width: 1px; } .li-2 { width: 2px; } .li-3 { width: 3px; } .li-4 { width: 4px; } .li-5 { width: 5px; }

In the second version, the entry looks like this:

@for $var from (begin) to (end)

And here the value (end) is discarded.

Let's take the previous example and write it in the second version:

@for $count from 1 to 5 { .li-#{$count} { width: $count + px; } }

Now pay attention to the resulting code:

.li-1 { width: 1px; } .li-2 { width: 2px; } .li-3 { width: 3px; } .li-4 { width: 4px; }

Tell us what the result of compiling the following code will be:

@for $var from 5 through 1 { li-#{$var} { padding: 10 * $var + px; } }

Tell us what the result of compiling the following code will be:

@for $var from 1 to 3 { .navbar-link-#{$var} { color: blue; width: 10px * $var; } }
