Директива @while в SASS

При создании циклов в языке Sass очень удобно пользоваться директивой @while, в которую обязательно нужно прописать переменную-счетчик. Давайте более подробно рассмотрим ее работу на следующем примере:

$i: 4; @while $i > 0 { .li-#{$i} { font-size: 4px * $i; } $i: $i - 1; }

Результат компиляции:

.li-4 { font-size: 16px; } .li-3 { font-size: 12px; } .li-2 { font-size: 8px; } .li-1 { font-size: 4px; }

Расскажите, каким будет результат компиляции следующего кода:

$count: 10; @while $count > 0 { .link-#{$count} { padding: 4px + $count; } $count: $count - 2; }
