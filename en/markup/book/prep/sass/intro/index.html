Introduction to SASS Language

In this tutorial we will study the SASS preprocessor. SASS is an alternative to the LESS preprocessor that also allows programming in CSS.

Actually SASS has two syntax variants, the first variant is also called SASS and the second one is SCSS. They both support the same set of features and differ only slightly in syntax. However, since SCSS is newer and more widely used, it is what we will be studying in this tutorial.

So, the syntax of SCSS is similar to the syntax of CSS, that is, we will see the same curly braces and the same semicolons as in CSS. Let's look at an example of SCSS syntax, where we immediately assign font and color to variables, and then assign these variables to properties in the selectorbody:

$font: Helvetica, sans-serif; $color: #333; body { font: $font; color: $color; }
