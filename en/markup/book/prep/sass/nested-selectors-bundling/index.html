Connecting selectors in SASS

Sometimes we need that when compiling nested structures, they are connected not through a space, but together. To do this, you need to put an ampersand before the name of the nested selector:

div { &.block { width: 300px; } }

As a result, after compilation, we will get the following code:

div.block { width: 300px; }

Tell us what the result of compiling the following code will be:

#block { &.xxx { width: 300px; } }

Tell us what the result of compiling the following code will be:

#block { .xxx { width: 300px; } }

Tell us what the result of compiling the following code will be:

#block { &.xxx { &.zzz { width: 300px; } } }

Tell us what the result of compiling the following code will be:

#block { &.xxx { .zzz { width: 300px; } } }

Tell us what the result of compiling the following code will be:

#block { .xxx { &.zzz { width: 300px; } } }
