Extending with a compound selector in SASS

Application of the directive@extendis not limited to just class selectors. In principle, any selector can be expanded, even single elements. Let's look at this with an example:

Let's look at an example:

.active-link { @extend a:active; } a:active { color: blue; }

As a result of compilation we will see that.active-linkinherits the propertycolorata:active:

a:active, .active-link { color: blue; }

Tell us what the result of compiling the following code will be:

button { @extend .box:hover; } .box:hover { color: #14147e; }

Tell us what the result of compiling the following code will be:

button:active { @extend input:focus; } button:blur { @extend input:blur; } input:focus { color: #14147e; border-color: #b5b5d1; } input:blur { color: #040433; background-color: #87878f; }
