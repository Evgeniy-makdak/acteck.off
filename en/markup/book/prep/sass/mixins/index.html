Mixins in SASS

In this lesson we will get acquainted withhagfishes(mixing), allowing one class to be included within another. Let's figure out what it is and how to work with it.

Essentially, mixing is the use of some classes inside others, when one class is a template - a specific piece of code that we can use in many places.

How is this convenient? With pure CSS, we would have to type a piece of code wherever we need it, but in SASS we can write it in one place and then connect it to the necessary places in the code.

The directive is used to declare mixins@mixin, after which we indicate its name, parameters and a block with the body:

@mixin mix { width: 100px; height: 100px; }
