Тестируем компилятор в SASS

Давайте теперь потестируем работу компилятора. Для этого вам нужно сделать файл с тестовым кодом SCSS. Этот файл должен иметь расширение .scss. Давайте добавим в него следующее содержимое:

div { p { font-size: 20px; } }

Следуя инструкциям из документации установленного вами компилятора скомпилируйте ваш .scss файл в .css. В результате вы должны увидеть следующий код:

div p { font-size: 20px; }

Проделайте описанные манипуляции. Убедитесь в том, что ваш компилятор работает.
