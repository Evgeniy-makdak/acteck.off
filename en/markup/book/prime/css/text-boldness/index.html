Bold text in CSS

In this lesson we will study the propertyfont-weight, which allows you to make the text bold or vice versa - cancel the boldness (for example, for headings, which are bold by default). To make text bold, specify a valuebold, and to cancel the boldness - the valuenormal.

This property works similar to the tagb, which we have already discussed. The difference is that using CSS to control boldness is much more convenient - you can, for example, make all paragraphs become bold and all headings become non-bold by making just a couple of entries in the CSS file.

If I did bold through the tagb- I would have to wrap the contents of each paragraph in this tag, and this would be very time-consuming (and if I then want to remove the boldness of the paragraphs, I will have to remove all these tags, imagine how much unnecessary, useless work this is).

In the following example we will make all paragraphs bold and all headingsh2- low-fat:

<h2>Title</h2> <p> This is a paragraph with text. </p> h2 { font-weight: normal; } p { font-weight: bold; }
