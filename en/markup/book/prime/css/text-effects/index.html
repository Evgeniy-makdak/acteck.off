Text Effects in CSS

Propertytext-decorationallows you to set some effects for text:underlining, crossing out, top line, and also disable such effects if any tag has them by default. The most common use of this property is to de-underline links (they are underlined by default).

In the following lessons we will take a closer look at property valuestext-decoration.
