Red line of text in CSS

Let's now figure out how to make a red line, that is, indent the first line of text. This is what the property is fortext-indent. Let's use it to make a red line for paragraphs:

<p> some long text </p> <p> some long text </p> <p> some long text </p> p { text-indent: 50px; text-align: justify; width: 400px; }
