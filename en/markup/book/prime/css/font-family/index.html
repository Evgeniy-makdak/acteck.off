Font type in CSS

Propertyfont-familyallows you to setfamilyfont. Most common fonts:Times New Roman(set by default),Arial, Georgia, Verdana, Trebuchet MS, Impact, Arial Black, Comic Sans MS, Courier New.

Set the font for all paragraphsArial:

<p> This is a paragraph with text. </p> p { font-family: Arial; }

:

If a font consists of several words, then the name of such a font should be placed in quotes:

<p> This is a paragraph with text. </p> p { font-family: "Comic Sans MS"; }

:

Check the functionality of all fonts from the list above.
