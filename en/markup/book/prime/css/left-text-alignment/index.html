Align text left in CSS

And here is the meaningleftfor paragraphs you don’t have to set it - they are already aligned to the left by default. However, there are elements that are centered by default (for example, tagsth, which make the table header cell). And it is for them that alignment to the left side may be required. Let's do this:

<table border="1"> <tr> <th>Name</th> <th>Surname</th> <th>Salary</th> </tr> <tr> <td>John</td> <td>Smith</td> <td>200$</td> </tr> <tr> <td>Nick</td> <td>Mayers</td> <td>300$</td> </tr> <tr> <td>Alex</td> <td>Jonson</td> <td>400$</td> </tr> </table> th { text-align: left; } table { width: 400px; }
