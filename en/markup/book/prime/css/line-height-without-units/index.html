Line-height value without units in CSS

Property valueline-heightdoes not necessarily have to serve as a number in some units. You can also simply write a number or fraction. In this case the real meaningline-heightcan be found by multiplying it by the valuefont-size.

For example,font-sizeequals10px, Aline-height - 1.5. In this case the real valueline-heightwill10px * 1.5 = 15px. Well, the visible white space between the lines of text will be5px: 15px - 10px = 5px.

The advantage of this method of assignmentline-heightis that when you change the font size, the line spacing will automatically change.

Let's see the application described in an example:

<p> some long text </p> p { font-size: 20px; line-height: 1.5; text-align: justify; width: 400px; }

:

Let the font size for paragraphs be30px. Set the propertyline-heightso that the visible white space between paragraphs is equal to15px. Use the value for thisline-heightno units.
