Width and height of elements in CSS

Properties are used to set the width and height of elements in CSS.widthAndheight. The values ​​of these properties are typically specified in pixels. Pixels are designated aspxand added to the number. For example, to set the size to100pixels, the property value should be specified as follows:100px.

Let's look at how these properties work using an example. Let's give the cells of some table a width of200pxand height100px:

<table border="1"> <tr> <td>1</td> <td>2</td> <td>3</td> </tr> <tr> <td>4</td> <td>5</td> <td>6</td> </tr> <tr> <td>7</td> <td>8</td> <td>9</td> </tr> </table> td { width: 200px; height: 100px; }
