How to work with CSS

Each tag in HTML corresponds to a so-calledselectorCSS. For example, tagpmatches CSS selectorp, with which we can access all paragraphs of an HTML page and, for example, color them all red at the same time.

After the selector you should put curly braces, inside of which you should write CSSproperties. Properties also set color, font size and other interesting things. They should be written in this format: the name of the property, then a colon, then the value of that property (for example, the property is a color, and “red” is a value). Then you need to put a semicolon and you can write the next property.

Let's, for example, color all paragraphs red:

p { color: red; }

In your filestyles.csspost code that colors paragraphs red. Open the pages of your site in a browser and make sure that all paragraphs turn red.
