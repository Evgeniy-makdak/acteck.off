Basics of working with CSS language

We have already discussed the main tags of the HTML language and now it’s time to make it look beautiful. I would like to be able to change the text color, size, background and much more. This is done using the CSS language.

Typically, CSS commands (styles) are stored in a separate file, which is connected with a special tag to all HTML pages of our site.

The advantage of this approach is that there is only one CSS file, but there can be any number of HTML files, even a thousand. If we make a change in one place in the CSS file, for example, coloring all the paragraphs red, then these changes will be applied to the entire thousand HTML pages to which our CSS file is connected. Very convenient and fast.

The style file must have the extension.css. To connect such a file to an HTML page, in the tagheadyou should write the following construction:

<link rel="stylesheet" href="fileName.css">

In the following example, a CSS file is connected to our HTML filestyles.css:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>This is the title</title> <link rel="stylesheet" href="styles.css"> </head> <body> <p> This is a paragraph with text. </p> </body> </html>

Create and connect a file to all your pagesstyles.css.
