Font size in CSS

Let's learn how to change the font size. This is what the property is forfont-size, which takes a value in pixels.

Let's set the paragraph text to font30pixels:

<p> This is a paragraph with text. </p> p { font-size: 30px; }

:

Copy the following HTML code onto your page:

<h1>Main page title</h1> <p> This is a paragraph with text. </p> <h2>Secondary heading 1</h2> <p> This is a paragraph with text. </p> <p> This is a paragraph with text. </p> <h2>Secondary heading 2</h2> <p> This is a paragraph with text. </p> <p> This is a paragraph with text. </p>

Set titleh1font size in50pxand center alignment, headingsh2font size in40pxand cancel the boldness, and set the font for paragraphs to30px.
