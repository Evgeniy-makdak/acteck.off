The font shorthand property in CSS

In the previous lesson we looked at many properties for text. It is often quite expensive to write each of these properties separately, so CSS has a special shortcut property for greater convenience.font. This property allows you to simultaneously set the font size, family, boldness, italics and line spacing.

The described property has the following syntax:

font-style font-weight font-size / line-height font-family

In this case, the order of properties matters, and mandatory ones arefont-sizeAndfont-family. Note that properties other than those described in the syntax above are not included in this abbreviation property.

Let's look at examples.

Example

Let's say we have the following styles:

p { font-size: 16px; font-family: Arial; }

Let's rewrite them using a shorthand property:

p { font: 16px Arial; }

Example

Let's say we have the following styles:

p { font-size: 16px; font-family: Arial; line-height: 50px; }

Let's rewrite them using a shorthand property:

p { font: 16px/50px Arial; }

Example

Let's say we have the following styles:

p { font-size: 16px; font-family: Arial; font-weight: bold; }

Let's rewrite them using a shorthand property:

p { font: bold 16px Arial; }

Example

Let's say we have the following styles:

p { font-size: 16px; font-family: Arial; font-weight: bold; line-height: 50px; font-style: italic; }

Let's rewrite them using a shorthand property:

p { font: bold italic 16px/50px Arial; }

Practical problems

Shorten your code using the shorthand propertyfont:

p { font-family: "Times New Roman"; font-size: 13px; line-height: 20px; }

Shorten your code using the shorthand propertyfont:

p { width: 300px; font-family: Arial; color: red; font-size: 40px; font-weight: bold; }

Shorten your code using the shorthand propertyfont:

p { font-family: Arial; font-size: 40px; font-weight: bold; text-indent: 50px; font-style: italic; line-height: 60px; }
