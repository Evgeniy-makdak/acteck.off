Line spacing in CSS

Let's now learn how to set line spacing for text. Line spacing refers to the distance between lines of text, that is, the white space between them. This is what the property is forline-height, which specifies the height of the text line.

When using the propertyline-heightThere's a catch here: this property doesn't set the space between lines of text, as it might seem, but rather sets the line height of the text.

This means that the actual visible space between lines will be calculated like this:line-height - font-size= visible spacing between lines of text.

In the following example, the spacing between lines of text would be50px - 20px = 30px:

<p> some long text </p> p { font-size: 20px; line-height: 50px; text-align: justify; width: 400px; }

:

Let the font size for paragraphs be30px. Set the propertyline-heightso that the visible white space between paragraphs is equal to15px.
