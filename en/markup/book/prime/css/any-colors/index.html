Any colors in CSS

In previous lessons you saw how you can set a color with a keyword. There are only a few keywords, however, and they cannot be used to make every possible shade of every color. To do this, there are two more ways to set colors. Throughrgband through the bars#.

See an example of setting color viargb:

p { color: rgb(255, 236, 114); }

See an example of setting color via#:

p { color: #ff0000; }

In order to obtain the desired color, special programs are used. Here is an example of an online color generator:colorscheme.ru, and here is an example of a program that allows you to measure color:colorcop.net.

Using the programs described, generate a color in one and the second format and apply these colors to some page tags.
