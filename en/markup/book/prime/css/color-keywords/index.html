Keywords for color in CSS

Besides the keywordred, which specifies red, other English words for color can be used, e.g.green- green,blue- blue,yellow- yellow,orange- orange,black- black,white- white.

Using appropriate selectors color the headingsh1green, headersh2in blue, headingsh3- in red, and paragraphs - in orange.
