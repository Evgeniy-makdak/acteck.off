Percentages and fr units in CSS grids

Also along withfryou can use the values ​​in%, which also determine which part of the container the column will occupy. In this case, the column size will first be calculated in%, and the remaining free space will be divided into fractions:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> <div>6</div> <div>7</div> <div>8</div> </div> #parent { display: grid; grid-template-columns: 50% 1fr 2fr 30%; border: 2px solid #696989; padding: 10px; width: 600px; } #parent > div { padding: 10px; border: 1px solid #696989; }

:

Let your grid have three columns. Make the first column occupy20%, and the remaining two columns were the same size.

Let your grid have five columns. Make the first column occupy100px, second column20%, and the remaining columns divided the remaining space so that each column was twice as large as the previous one.
