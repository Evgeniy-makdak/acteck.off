Combining rows in CSS grids

Rows and columns in grids form a grid. You can make the element occupy not one grid cell, but several.

In order for an element to occupy more than one row, it should be given the propertygrid-row. In this property, numbers are specified through a slash, indicating the starting and ending positions of the elements in the grid.

In this case, the element will stretch from the first position to the second position (not including it). That is the meaning1 / 2will cause the element to occupy the first one cell, and the value1 / 3will cause the element to occupy the first and second cell (but not including the third).

Example

Let's say we have a grid with three children. Let's place them so that the first element occupies two rows:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> </div> #parent { display: grid; grid-template-columns: 2fr 1fr; height: 300px; width: 400px; padding: 10px; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-row: 1 / 3; /* два ряда */ } #elem2 { grid-row: 1 / 2; } #elem3 { grid-row: 2 / 3; }

Now let's assign three rows to the fourth block:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> <div id="elem4">4</div> </div> #parent { display: grid; grid-template-columns: 2fr 1fr; height: 300px; width: 400px; padding: 10px; border: 2px solid #696989; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem1 { grid-row: 1 / 2; } #elem2 { grid-row: 2 / 3; } #elem3 { grid-row: 3 / 4; } #elem4 { grid-row: 1 / 4; }
