Introduction to CSS Grids

CSS grids are a way to place elements both horizontally and vertically. This is a more advanced method compared to flexes, but more difficult to master.

To make a grid, you need to set the element's propertydisplayin meaninggrid:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; }

After this, the element will turn into a two-dimensional grid, in which elements are arranged vertically (columns) and horizontally (rows). The area where a column and a row intersect is calledcell.
