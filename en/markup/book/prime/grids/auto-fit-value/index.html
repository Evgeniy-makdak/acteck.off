The meaning of auto-fit in CSS grids

Now let's look at the meaningauto-fit, which is also used when specifying columns of the same size in conjunction with the functionrepeat. Its difference from the meaningauto-fillis thatauto-fitadjusts the number of columns to the available width of the container, expanding or contracting them.

Example

Let's look at how value worksauto-fitusing the example of a container with eight elements. Apply together with the valueauto-fitfunction known to you from the previous lessonminmax:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> <div>6</div> <div>7</div> <div>8</div> </div> #parent { display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); border: 2px solid #696989; padding: 10px; width: 600px; } #parent > div { padding: 10px; border: 1px solid #696989; }
