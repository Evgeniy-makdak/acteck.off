Two-column website layout in CSS

Let's make a two-column layout like this:

First, let's create the basic structure of the page:

<div id="wrapper"> <div id="header"> header </div> <div id="container"> <div id="content"> content </div> <div id="sidebar"> sidebar </div> </div> <div id="footer"> footer </div> </div>

Now let's style the wrapper, aligning it to the center of the screen:

#wrapper { width: 1000px; margin: 30px auto; border: 1px solid black; }

Now let’s set the header and footer styles:

#header { height: 200px; border: 1px solid black; } #footer { height: 200px; border: 1px solid black; }

As you can see, we are giving the header and footer a height. We do this like in our schema they don't have any content. If there is content, the height of the blocks should not be set - it will be formed by their content.

Also note that we don't give them a width. The fact is that the header and footer are block elements and their width will automatically be equal to the width of the wrapper.

Let's now ask thempaddingto prevent text from sticking to borders:

#header { height: 200px; padding: 20px; border: 1px solid black; } #footer { height: 200px; padding: 20px; border: 1px solid black; }

Availabilitypaddingbut, however, it will have a bad effect on the size of the blocks - they will become larger than indicated. Let's change the behavior of all blocks so thatpaddingand our blocks did not expand the borders:

* { box-sizing: border-box; }

Now we need to put the content and sidebar in one row. Let's do this using flexes:

#container { display: flex; }

Let's set the content and sidebar to their width. In this case, the sum of their widths should be equal to the width of the wrapper:

#content { width: 800px; } #sidebar { width: 200px; }

Let's say we want to make a space between the sidebar and the content. In this case, the sidebar needs to be setmargin. In this case, we will have to pinch off the width from someone for thismargin. Let's pinch off the content, reducing its width accordingly:

#content { width: 780px; } #sidebar { width: 200px; margin-right: 20px; }

Let's now add height since our elements don't contain content. It is enough to add height to only one element, because the second flex element will adjust in height (why?). Let's set the height of the content:

#content { width: 780px; height: 700px; } #sidebar { width: 200px; margin-right: 20px; }

Let's add the rest of the styles to the content and sidebar:

#content { width: 780px; height: 700px; padding: 20px; border: 1px solid black; } #sidebar { width: 200px; margin-right: 20px; padding: 20px; border: 1px solid black; }

That's it, our layout is ready. Let's put all the code together:

<div id="wrapper"> <div id="header"> header </div> <div id="container"> <div id="sidebar"> sidebar </div> <div id="content"> content </div> </div> <div id="footer"> footer </div> </div> * { box-sizing: border-box; } #wrapper { width: 1000px; margin: 30px auto; border: 1px solid black; } #header { height: 200px; padding: 20px; border: 1px solid black; } #container { display: flex; } #content { width: 780px; height: 700px; padding: 20px; border: 1px solid black; } #sidebar { width: 200px; margin-right: 20px; padding: 20px; border: 1px solid black; } #footer { height: 200px; padding: 20px; border: 1px solid black; }
