Setting background color in CSS

You already know how to change the color of text. Let's now learn how to change the background color. This is done using the propertybackground-color, which takes the same values ​​as the well-known onecolor.

Let's fill the block with an orange background (while making the text color white):

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { width: 300px; height: 100px; color: white; background-color: orange; /* задаем оранжевый цвет фона */ }
