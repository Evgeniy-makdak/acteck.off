Заглавные римские цифры для списков ol в CSS

Чтобы задать заглавные римские цифры нужно в свойстве list-style-type установить значение upper-roman:

<ol> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ol> ol { list-style-type: upper-roman; }
