Shorthand property for border in CSS

Just like for fonts, there is also a shortcut property for bordersborder, which we can use instead of writing3different properties for thickness, color and type of border. In this property, we can simply list the values ​​for the border in any order.

Let's look at an example. Let us have a boundary like this:

#elem { border-width: 1px; border-style: solid; border-color: black; width: 300px; height: 100px; }

Let's rewrite it through the abbreviation property:

#elem { border: 1px solid red; width: 300px; height: 100px; }
