Basics of working with borders in CSS

Now we will learn how to add a border to elements. This is done using three properties: propertyborder-widthsets the thickness of the border,border-color- color, andborder-stylespecifies the border type.

The first two properties work in an obvious way:border-coloraccepts colors in the same format as the propertycolor, and the border thickness is specified in pixels. And here is the propertyborder-styletakes values ​​in the form of keywords. For example, the valuesolidspecifies a solid line.

Let's make, for example, a border thick3pixel, as a solid line, red:

<div id="elem"></div> #elem { border-width: 3px; /* толщина 3px */ border-style: solid; /* в виде линии */ border-color: red; /* красный цвет */ width: 300px; height: 100px; }
