Свойство float для блоков в CSS

Свойство float применимо не только к картинкам, но и к любым другим блокам.

Давайте проверим это.

Пусть у нас есть блок-родитель с классом parent, а внутри него находится текст и блок-потомок с классом child. Обоим блокам задана граница, а потомку еще и ширина с высотой:

<div class="parent"> <div class="child"></div> some long text </div> .parent { border: 1px solid red; text-align: justify; } .child { width: 200px; height: 100px; border: 1px solid green; }

:

Давайте теперь потомку установим float в значение left - текст начнет обтекать наш блок:

<div class="parent"> <div class="child"></div> some long text </div> .parent { border: 1px solid red; text-align: justify; } .child { float: left; width: 200px; height: 100px; border: 1px solid green; }

:
