float property for blocks in CSS

Propertyfloatapplicable not only to pictures, but also to any other blocks.

Let's check it out.

Let us have a parent block with a classparent, and inside it there is text and a descendant block with the classchild. Both blocks are given a border, and the descendant is also given a width and height:

<div class="parent"> <div class="child"></div> some long text </div> .parent { border: 1px solid red; text-align: justify; } .child { width: 200px; height: 100px; border: 1px solid green; }

:

Let's now install for posterityfloatin meaningleft- the text will begin to flow around our block:

<div class="parent"> <div class="child"></div> some long text </div> .parent { border: 1px solid red; text-align: justify; } .child { float: left; width: 200px; height: 100px; border: 1px solid green; }
