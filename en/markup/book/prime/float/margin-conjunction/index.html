Combining float and margin in CSS

Now our text is pressed against the image on the left side. Let's try to move this text a little. To do this, let's set our paragraphs to leftmarginV30px, and the div-parent - a red border.

Suddenly, only the text that is adjacent to the parent div will move to the right, but the text adjacent to the image will not move back:

<div> <img src="img.png" alt=""> <p> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } p { margin-left: 30px; } img { float: left; }

:

To understand why this is so, let's add a green border to paragraphs:

<div> <img src="img.png" alt=""> <p> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } p { margin-left: 30px; border: 1px solid green; } img { float: left; }

:

As we can see, in fact, the paragraphs are moved away from the left edge, but not from the image, but from the parent div. To take a closer look, let's add semi-transparency to the picture through the propertyopacity:

<div> <img src="img.png" alt=""> <p> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } p { margin-left: 30px; border: 1px solid green; } img { float: left; opacity: 0.5; }

:

Now it’s clear that the reaction tomargin-leftyes, only the paragraphs are actually placed under the picture. Let's remove the indentation while leaving the image translucent:

<div> <img src="img.png" alt=""> <p> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } p { border: 1px solid green; } img { float: left; opacity: 0.5; }

:

This is what our paragraphs actually look like - their text is pushed aside by the picture, but physically the paragraphs lie under the picture, this can be seen by the border that starts from the left edge of the parent div.

Let's give it backmarginand put the picture in the first paragraph:

<div> <p> <img src="img.png" alt=""> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } p { margin-left: 30px; border: 1px solid green; } img { float: left; opacity: 0.5; }

:

Now the picture moves along with the paragraphs!
