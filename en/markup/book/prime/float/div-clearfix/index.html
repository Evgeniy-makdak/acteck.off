Separate div with clearfix in CSS

In fact, clearfix is ​​most often used in the form of a separate div, which is placed in front of the div for which you want to unoverlay, like this:

<div class="parent"> <img src="img.png" alt=""> text </div> <div class="clearfix"></div> <div class="parent"> text </div> .parent { border: 1px solid red; } .parent img { float: left; } .clearfix { clear: both; }
