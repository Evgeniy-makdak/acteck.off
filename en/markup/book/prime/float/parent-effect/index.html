Effect of floats on parent in CSS

Let us now have a div containing an image. Let's give the diva a border, but for now we won't give the image a propertyfloat.

Let's see how it will look in the browser:

<div> <img src="img.png" alt=""> </div> div { border: 1px solid red; }

:

Now let's set the image propertyfloatin meaningleft. In this case, an amazing thing will happen - the height of the parent will disappear, its bottom border will start immediately after the top, and the image will crawl out from below its parent:

<div> <img src="img.png" alt=""> </div> div { border: 1px solid red; } img { float: left; }

:

It turns out that the elements that have the property setfloat, do not expand their parent in height.

Let's propertyfloatinstead ofleftlet's write the valueright. The parent's behavior will not change, but the image will start floating on the right:

<div> <img src="img.png" alt=""> </div> div { border: 1px solid red; } img { float: right; }
