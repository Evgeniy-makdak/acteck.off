Penetration of floating elements through tags in CSS

Let us now have the text placed in paragraphs, and the picture, which is given the propertyfloatis located above these paragraphs. In this case, everything will work as it did - the presence of other tags does not interfere with the flow:

<div> <img src="img.png" alt=""> <p> some long text </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } img { float: left; }

:

Now let there be little text in the first paragraph - in this case the picture will also fit into the second paragraph:

<div> <img src="img.png" alt=""> <p> Lorem ipsum dolor sit amet, consectetur adipiscing elitorire. </p> <p> some long text </p> </div> div { width: 400px; border: 1px solid red; text-align: justify; } img { float: left; }
