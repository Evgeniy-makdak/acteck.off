Canceling Wrapping Using the Clear Property in CSS

To solve this problem there is a special propertyclear, which cancels the flow. Meaningleftcancels left wrapping, valueright- on the right, and the valueboth- on both sides. This value is used most often.

So the propertyclearcancels wrapping. In our case, this will allow us to make sure that the floating image from the first div does not overlap the second div.

At the same timeclearshould be given to the element on which floating elements should not be placed, that is, in our case it should be given to the second div.

Let's do this - give the second div besides the classparentalso classclearfixand for this new class we will set the propertyclearin meaningboth- the overlapping image will disappear:

<div class="parent"> <img src="img.png" alt=""> text </div> <div class="parent clearfix"> text </div> .parent { border: 1px solid red; } .parent img { float: left; } .clearfix { clear: both; }

:

Nameclearfixis generally accepted, so use it in future.
