Width and height of an image in HTML

If you set both width and height at the same time, the picture can be distorted. For example, if you set the width100pixels, and height70px, then our smile will become stretched in width:

<img src="smile.png" width="100" height="70">

:

The picture will not necessarily be distorted, because we can simply set both the width and height according to100pixels, in this case our picture will become smaller than it was (and it was130), but its proportions will not be distorted:

<img src="smile.png" width="100" height="100">

:
