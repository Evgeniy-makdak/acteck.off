Tags for paragraphs

One of the main elements of the page isparagraphs. They can be compared to paragraphs in a book - each paragraph begins on a new line and has a so-called red line (this is when the first line of the paragraph text is slightly indented to the right). By default there is no red line, but it is easy to create (more on this later).

A paragraph is created using a tagpThus:

<p> This is a paragraph 1. </p> <p> This is a paragraph 2. </p> <p> This is a paragraph 3. </p>

Create on your page5paragraphs with some text. See how these paragraphs will look in the browser.
