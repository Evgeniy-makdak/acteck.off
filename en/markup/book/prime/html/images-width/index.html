HTML image width

By default, pictures are displayed in their actual size. For example, the picture with a smiley face that we displayed above has a real size130on130pixels and the same size of the picture we see in the browser.

If desired, you can change the size of the image by setting its width. This is done using the attributewidth.

Let's set the width of our image as an example.100pixels. To do this, the attribute valuewidthindicate the number100, like this:

width="100"

What size will the picture on the screen be? You might think the following: width100pixels, as we set, and the real height130, since we didn’t ask it. In fact, this will not be the case.

The width will indeed be100pixels, but the height will be adjusted so that the proportions of the picture are not distorted, that is, it will also become100pixels.

Let's run it and check it out:

<img src="smile.png" width="100">
