Table width and height in HTML

Using attributeswidthAndheightYou can set the width and height of the table. Let's set, for example, our table width to300pixels, and height - in200:

<table border="1" width="300" height="200"> <tr> <td>cell 1</td> <td>cell 2</td> <td>cell 3</td> </tr> <tr> <td>cell 4</td> <td>cell 5</td> <td>cell 6</td> </tr> <tr> <td>cell 7</td> <td>cell 8</td> <td>cell 9</td> </tr> </table>
