Attributes in HTML tags

Tags can also containattributes- special commands that expand the action of the tag. Attributes are placed inside the opening tag in the following format:

<tag attribute1="value1" attribute2="value2">

Quotes can be anything - single or double; it is acceptable not to put them at all if the attribute value consists of one word (but this is not advisable).
