Ordered Lists in HTML

In addition to disordered ones, there are alsoorderedlists. In them, each point is marked with number markers instead of dot markers. Ordered lists are created using the tagol, and the items of such lists are tagsli.

The convenience of ordered lists is that I can insert a new list item anywhere - and the numbering will rearrange itself (that is, you do not have to keep track of it in case of any changes, as would be the case if you arranged it manually).

Let's make an ordered list using tagol:

<ol> <li>text</li> <li>text</li> <li>text</li> <li>text</li> <li>text</li> </ol>
