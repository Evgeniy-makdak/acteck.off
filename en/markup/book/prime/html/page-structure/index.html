Basic HTML Page Structure

A website page is a regular text file with the extension.html. The text of the HTML page along with tags is stored inside this file. This file must have the following tags: taghtml, which should contain the text of the entire site (everything written outside this tag will be ignored by the browser), and inside it there should be two more tags: tagheadfor service page content and tagbody- for the main text, which is visible on the browser screen.

Into the service content, which is located inside the taghead, there are many different things included, but for now we only need two of them. This is a tagtitle, which specifies the title of the page, which will be visible in the browser tab, and the tagmeta, which specifies the page encoding (it is placed in the attributecharsetand usually mattersutf-8).

Also, before the taghtmlthe construction is usually writtendoctype, which indicates the version of HTML in which the site is made. The current version of the language is number five and the doctype for it should look like this:

<!DOCTYPE html>

So let's look at the basic structure of the page:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>This is the title</title> </head> <body> This is the main content of the page. </body> </html>

Create a file with the extension.html. Copy the main page structure into it. Save your file. Then open it in the browser by dragging the file with the mouse or double-clicking on it.
