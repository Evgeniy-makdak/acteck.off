HTML tag styles

HTML tags can be changed in style: colors, sizes, fonts, background, and position on the screen. For this purpose, the CSS language is used, which extends the capabilities of the HTML language.

It turns out that with the help of HTML tags the structure of the site is created, and with the help of CSS styles this structure is decorated and placed in the right places.

The combined use of HTML and CSS is called page layout. This is what we will study in this tutorial.
