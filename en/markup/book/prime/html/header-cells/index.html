Header cells in a table in HTML

Besides tagstdthere are also tagsth, which also create cells. But these will no longer be ordinary cells, butheader cells, which indicate what is in a given column (or row) of the table. By default the text in such cellsthwill be bold and centered.

Let's look at an example. Let us have a table like this with employees:

<table border="1" width="300"> <tr> <td>John</td> <td>Smith</td> <td>200$</td> </tr> <tr> <td>Nick</td> <td>Mayers</td> <td>300$</td> </tr> <tr> <td>Alex</td> <td>Jonson</td> <td>400$</td> </tr> </table>

:

As you can see, the first column stores the employee's first name, the second column stores the last name, and the third stores the salary. Let's make another row at the beginning of the table, in which we will place the column headings in tagsth:

<table border="1" width="300"> <tr> <th>Name</th> <th>Surname</th> <th>Salary</th> </tr> <tr> <td>John</td> <td>Smith</td> <td>200$</td> </tr> <tr> <td>Nick</td> <td>Mayers</td> <td>300$</td> </tr> <tr> <td>Alex</td> <td>Jonson</td> <td>400$</td> </tr> </table>
