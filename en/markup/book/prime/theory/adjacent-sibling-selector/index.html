Adjacent selector in CSS

Adjacent selector+allows you to select an element by its neighbor above.

Example

Let's go through all the tagsp, immediately after the tagsh2, and color them red:

<h2>text</h2> <p> + </p> <p> - </p> <p> - </p> h2 + p { color: red; }

:

Example

Let's go through all the tagsp, immediately after the elements with the class.test, and color them red:

<p class="test"> text </p> <p> + </p> <p> - </p> .test + p { color: red; }

:

Example

Let's access all elements with a class.elem, immediately after the elements with the class.test, and color them red:

<p class="test"> text </p> <p class="elem"> + </p> <p> - </p> .test + .elem { color: red; }

:
