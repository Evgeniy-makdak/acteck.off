Newbie error in descendant selector

Look at the following selector:

p.eee { color: red; }

You should already know that such a selector will select paragraphs with the classeee. For example, these:

<p class="eee"> lorem ipsum </p> <p class="eee"> lorem ipsum </p>

Now look at this selector:

p .eee { color: green; }

This selector differs from the previous one in that it has a space between the tag name and the class name. This space represents the descendant selector. That is, in this case we select all elements with the classeee, located inside paragraphs. For example, this selector will select the following songs:

<p> lorem <span class="eee">ipsum</span> </p> <p> lorem <span class="eee">ipsum</span> </p>
