Element classes in CSS

In previous lessons, we selected page elements by tag name, setting styles, for example, for all paragraphs at the same time. On a page, however, there may be different types of paragraphs to which you need to apply different styles. To solve this problem, you can assign different paragraphs to different CSSclasses.

To assign a tag to a class, you need to write an attribute to this tagclass, and in it - the class name you came up with, consisting of letters, numbers, underscores and hyphens.

Let's look at an example. Let's make paragraphs with two types of classes -eeeAndzzz:

<p class="eee"> paragraph with class eee </p> <p class="eee"> paragraph with class eee </p> <p class="zzz"> paragraph with class zzz </p> <p class="zzz"> paragraph with class zzz </p>

Let's now take a look at the paragraphs with the various classes in CSS and give them some styling. For example, let's color paragraphs with the classzzzin red, and paragraphs with classeee- in green.

To do this, you need to refer to our classes in the CSS file. The call has the following syntax: first comes the dot symbol, and then the class name we invented. That is, to access the classeeeneed to write.eee, and for the classzzz- write.zzz.

So, let's do what is described. Let's add CSS styles to our HTML for the classes we introduced:

.eee { color: green; } .zzz { color: red; }
