Multiple selectors combined in CSS

To select an element that simultaneously falls under several selectors, you need to write these selectors together, without a space.

For example, the following selector will select an element that also has the classbbband classzzz:

.bbb.zzz { }

And the next selector will select the titleh2, which simultaneously has a classbbband classzzz:

h2.bbb.zzz { }

In this way, you can construct arbitrary selector designs. In this case, the order of the selectors used does not matter, except for the rule that tag selectors must be placed at the very beginning.

That is, for example, selectors.bbb.zzzAnd.zzz.bbbcompletely equivalent.

In addition, you can form selectors usingid, for example, these:#elem.eee, orh2#elem.eee, orh2.eee#elemand so on.
