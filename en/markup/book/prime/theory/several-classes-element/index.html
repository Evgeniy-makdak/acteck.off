Element with multiple CSS classes

You can simultaneously assign several CSS classes to a tag. To do this, in the attributeclassthese classes must be listed separated by a space.

Let's look at an example. Let's say we have several CSS classes:

.error { color: red; } .large { font-size: 30px; font-family: Arial; }

Let's give the paragraph both one class and a second one:

<p class="error large"> lorem ipsum </p>

Let's say you have the following classes:

.success { color: green; } .error { color: red; } .large { font-size: 30px; } .small { font-size: 20px; }

May you also have4paragraph. Using the classes provided, make the first paragraph green and large font, the second paragraph red and small font, the third paragraph red and large font, and the fourth paragraph green and small font.

Come up with it and do it3some class. Give some tag all three classes you came up with at the same time.
