Error when using multiple CSS selectors

Let me remind you about a newbie mistake: selector.bbb.zzzand selector.bbb .zzzdo different things. The first one looks for an element that has two classes at the same time, and the second one looks for an element with a classzzz, located inside the classbbb.

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

h2.zzz { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

h2 .zzz { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb.zzz { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb .zzz { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb.zzz.xxx { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb .zzz.xxx { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb.zzz .xxx { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

.bbb .zzz .xxx { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

#elem.bbb { color: red; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

#elem .bbb { color: red; }
