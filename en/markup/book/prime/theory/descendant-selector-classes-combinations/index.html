Complex combinations of descendant selector and classes in CSS

Let's now have the following code:

<div class="block"> <h2 class="header">Title h2</h2> <p> text </p> <h3 class="header">Title h3</h3> <p> text </p> <p> text </p> </div> <div class="block"> <h2 class="header">Title h2</h2> <p> text </p> <h3 class="header">Title h3</h3> <p> text </p> <p> text </p> </div>

Let's find all elements with classheader, located inside an element with a classblock, and give them a fontArial:

.block .header { font-family: Arial; }

As you can see in the HTML code, inside the element with the classblockelements with classheadercan be like headingsh2, and headersh3. Let's write selectors that differentiate these headers and do something with these headers.

Let's select everythingh2with classheader, located inside an element with a classblock:

.block h2.header { font-size: 30px; color: red; }

Now let's select everythingh3with classheader, located inside an element with a classblock:

.block h3.header { font-size: 20px; color: green; }

Let's put all our CSS together:

.block .header { font-family: Arial; } .block h2.header { font-size: 30px; color: red; } .block h3.header { font-size: 20px; color: green; }
