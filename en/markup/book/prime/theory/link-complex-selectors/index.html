Complex selectors with link states in CSS

All the complex selectors that we studied earlier can be used for links.

Let's look at an example. Let our links be in a block with a givenid:

<div id="block"> <a href="#">link</a> <a href="#">link</a> <a href="#">link</a> </div>

Let's set styles for the links from this block:

#block a:link, #block a:visited { color: red; } #block a:hover { text-decoration: none; }

As a rule, conditionslinkAndvisitedare not separated, so the code can be simplified as follows:

#block a { color: red; } #block a:hover { text-decoration: none; }

Tell us what the selector in the code below selects. Then write HTML code that matches this selector. Here is the code with the selector:

#test a { color: red; } #test a:hover { text-decoration: none; }
