Inheriting CSS properties

Suppose we have a paragraph, and there is some text in it in a tagi:

<p> This is some <i>text</i> in paragraph. </p>

Let's color the paragraphs red:

p { color: red; }

As a result, not only the paragraph text, but also the tag text will be coloredi:

The whole point is that the propertycolor inherited. This means that if a parent tag has a certain color, then its descendants will have the same color. Not all CSS properties are inherited, but many are inherited (of those you already know, all are inherited).

If you wish, however, you can override the parent property by specifying a child selector. For example, let's set the paragraph color to red and the italics color to blue:

<p> This is some <i>text</i> in paragraph. </p> p { color: red; } i { color: blue; }

:

The order of the selectors in the CSS code does not matter in this case. If you rearrange the selectoriand selectorp, everything will work the same:

<p> This is some <i>text</i> in paragraph. </p> i { color: blue; } p { color: red; }
