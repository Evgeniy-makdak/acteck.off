Selecting an element by unique id in CSS

As you already know, classes are meant to be assigned to a group of elements. In addition to classes, you can also select elements byid, which is the element's unique identifier. By uniqueness we mean that if we already have an element on the page with the sameid, then another element with the sameidthere shouldn't be.

The unique identifier is specified using the attributeid, in which the name we invented is written. Let's make two blocks, for example. We'll ask the first oneidin meaningblock1, and the second - in the meaningblock2:

<div id="block1"> <p>text</p> <p>text</p> </div> <div id="block2"> <p>text</p> <p>text</p> </div>

To access an element with a givenid, you need to write a symbol#and the name we came up with, like this:

#block1 { color: red; } #block2 { color: green; }

Attributeidused when it is necessary to emphasize the uniqueness of an element. Classes are used when it is assumed that there may be many such elements, even if there is currently only one.

The following code is given:

<div id="elem1"> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div id="elem2"> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div id="elem3"> <h2>Title</h2> <p> text </p> <p> text </p> </div>

Color the contents of the blockelem1in red, blockelem2- in green, and blockelem3- in blue.
