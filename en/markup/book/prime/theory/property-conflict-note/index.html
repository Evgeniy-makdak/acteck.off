A Note on CSS Property Conflict

What was described in the previous lesson only applies to property conflicts. That is, for the case when both one selector and the second have the same properties. If the properties are different, then the properties of both one and the second selector will simply be applied.

Tell us what properties will be applied to headers and what the values ​​of these properties will be:
