Замечание к конфликту свойств CSS

Описанное в предыдущем уроке относится только к конфликту свойств. To есть к случаю, когда и в одном селекторе, и во втором есть одинаковые свойства. Если свойства разные, то просто применятся свойства и одного, и второго селектора.

Расскажите, какие свойства применятся к заголовкам и какие будут значения этих свойств:

<h2>text</h2> h2 { font-size: 20px; color: blue; } h2 { font-style: italic; color: red; }
