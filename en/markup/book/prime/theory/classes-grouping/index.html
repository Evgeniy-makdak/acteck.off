Группировка классов в CSS

Пусть у нас есть несколько классов, которые красят элементы в разные цвета:

.xxx { color: green; } .yyy { color: red; }

Пусть мы хотим сделать так, чтобы эти классы кроме цвета, задавали еще и размер шрифта:

.xxx { color: green; font-size: 30px; } .yyy { color: red; font-size: 30px; }

Так как оба класса задают одинаковый размер шрифта, давайте используем группировку селекторов следующим образом:

.xxx { color: green; } .yyy { color: red; } .xxx, .yyy { font-size: 30px; }

Упростите код, используя группировку селекторов:

.eee { font-size: 20px; line-height: 1.5; font-family: Arial; } .zzz { font-size: 30px; line-height: 1.5; font-family: Arial; }

Упростите код, используя группировку селекторов:

.eee { font-size: 20px; text-align: center; font-family: Arial; } .zzz { font-size: 30px; text-align: center; font-family: Arial; } .ggg { font-size: 35px; text-align: center; font-family: Arial; }
