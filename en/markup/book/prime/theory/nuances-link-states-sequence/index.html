The nuances of link state consistency in CSS

Link pseudo-classes inherit from each other. For example, if I remove the underscore for the state:link, then it will be removed for all states.

Due to inheritance, to work correctly, these pseudo-classes should be placed in exactly the same order as in the example::link, :visited, :hover, :active(unnecessary ones can be omitted). This order is subject to the following mnemonic rule:LoVe HAte.

Make all links able:hoverred and unmarked, able:link- blue, able:visited- green, able:active- black.
