Div tag for grouping tags in HTML

In the previous lesson we looked at the tagspan, which allows you to somehow select a piece of text. There is also a special tagdiv, which allows you to group several tags so that you can then do something at the same time.

Let's use tags as an exampledivwe will group the paragraphs:

<div class="block1"> <p> text </p> <p> text </p> </div> <div class="block2"> <p> text </p> <p> text </p> </div>

Let's assign the color red to the elements from the first group, and green to the elements from the second group:

.block1 { color: red; } .block2 { color: green; }

The following code is given:

<div> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div> <h2>Title</h2> <p> text </p> <p> text </p> <p> text </p> </div> <div> <h2>Title</h2> <p> text </p> <p> text </p> <p> text </p> <p> text </p> </div>

Color the contents of the first div red, the contents of the second div green, and the contents of the third div blue. Give each div its own class for this.

The following code is given:

<div> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div> <h2>Title</h2> <p> text </p> <p> text </p> </div> <div> <h2>Title</h2> <p> text </p> <p> text </p> </div>

Color the contents of the first and third divs red, the contents of the second and fourth divs green. To do this, give the first and third divas one class, and the second and fourth divas another class.
