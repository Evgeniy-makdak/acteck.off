Grouping selectors in CSS

Sometimes the following situation may arise: you need to do the same thing with different tags, for example, color all the headingsh2, h3and paragraphs in red. In this case you will have to write something like this:

h2 { color: red; } h3 { color: red; } p { color: red; }

However, there is a way to make this a little shorter: tag selectors can be combined separated by commas and the CSS code will be applied to all of them at the same time. The above code can be rewritten shorter as follows:

h2, h3, p { color: red; }

Simplify your code by using selector grouping:

h1 { text-align: center; } h2 { text-align: center; } h3 { text-align: center; }
