Selector by id and CSS classes

One element can be givenid, and classes. In this case, some of the element's styles can be set viaid, and some of the styles are done through the class:

<div id="block" class="large"> <p>text</p> <p>text</p> </div> #block { color: red; font-family: Arial; } .large { font-size: 30px; }

Style your blocks according toidand by class:

<div id="block" class="text"> <p>text</p> <p>text</p> <p>text</p> </div> <div class="text"> <p>text</p> <p>text</p> <p>text</p> </div>
