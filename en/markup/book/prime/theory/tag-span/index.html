The span tag for pieces of text in HTML

You already know that in order to make some piece of text, for example, bold, you should use the tagb. However, what if I want to color a piece of text red? For such a case of a tag like tagb, does not exist.

But there is a special tagspan- by itself it does absolutely nothing, but we can apply CSS styles to it, which will allow us to add the desired effect to a piece of text.

Let's look at an example. Let us have some text:

<p> This is a paragraph with text. </p>

Let's say we want to make the word "text" red. To do this, wrap this piece of text in a tagspanand give this tag some class:

<p> This is a paragraph with <span class="red">text</span>. </p>

Well, now let’s color the element with this class red:

.red { color: red; }

The following code is given:

<p> Lorem ipsum <span class="xxx">dolor sit</span> amet. </p> <p> <span class="xxx">Lorem ipsum</span> dolor sit amet. </p>
