Тег span для кусочков текста в HTML

Вы уже знаете, что для того, чтобы сделать некоторый кусочек текста, к примеру, жирным - следует использовать тег b. Однако, что делать, если я хочу покрасить кусочек текста в красный цвет? Для такого случая тега, подобного тегу b, не существует.

Зато существует специальный тег span - сам по себе он абсолютно ничего не делает, но для него можно применить CSS стили, которые позволят нам добавить нужный эффект к кусочку текста.

Давайте посмотрим на примере. Пусть у нас есть какой-то текст:

<p> This is a paragraph with text. </p>

Пусть мы хотим слово "текст" сделать красным. Для этого обернем этот кусочек текста в тег span и этому тегу дадим какой-нибудь класс:

<p> This is a paragraph with <span class="red">text</span>. </p>

Ну а теперь покрасим элемент с таким классом в красный цвет:

.red { color: red; }

Дан следующий код:

<p> Lorem ipsum <span class="xxx">dolor sit</span> amet. </p> <p> <span class="xxx">Lorem ipsum</span> dolor sit amet. </p>

Сделайте элементы с классом xxx размером в 30px.
