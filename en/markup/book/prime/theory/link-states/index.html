Link states in CSS

I think that while visiting various sites on the Internet, you have noticed that links usually respond when you hover the mouse over them. This effect can be achieved by specifying the behavior of links in different states.

For example, like this -a:hover- we will catch the state when the mouse cursor is hovered over the link. At this point, we can, for example, change the color of the link or remove/add an underline to it. Design:hovercalledpseudo-class. Pseudo-classes allow you to catch different states of elements.

Except:hoverthere are also pseudo classes:link, which catch an unvisited link, and:visited, which catch the visited link. Some sites use them to show users where they have been and where they have not been. There is also a pseudo class:active, which catches the following state:the element has been clicked with the mouse, but has not yet been released.

In the following example, for a link in the state:hoverthe underlining is removed, able:linkthe color is set to red, in the state:visited- green, in:active- blue. As a result, it turns out that at the beginning the link will be red, after clicking on it it will be green, if you click on it with the mouse and do not release it, it will be blue, and if you hover it with the mouse, it will become ununderlined:

a:link { color: red; } a:visited { color: green; } a:hover { text-decoration: none; } a:active { color: blue; } <a href="#">link</a>
