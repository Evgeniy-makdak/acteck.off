Common use of link state in CSS

As a rule, states are specified for all types of links at the same time, and then below the behavior of the link when hovered with the mouse is added, like this:

a { color: red; } a:hover { text-decoration: none; } <a href="#">link</a>

:

Make all links in all states blue and not underlined, and in the state:hover- red and underlined.
