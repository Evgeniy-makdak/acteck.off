Descendant selector in CSS

Let us have a listuland listol:

<ul> <li>text</li> <li>text</li> <li>text</li> <li>text</li> <li>text</li> </ul> <ol> <li>text</li> <li>text</li> <li>text</li> <li>text</li> <li>text</li> </ol>

Let's color the tagsliin these lists in red:

li { color: red; }

Now let's say we want to color the tagslilistulin red, and tagslilistol- in green.

In this case, the selector will help usdescendants. It allows you to select tags by their parent. To do this, you need to specify the parent selector, followed by a space - the child selector. In our case, the selectorul liwill select all tagslifrom the listul, and the selectorol li- selects all tagslifrom the listol. Let's paint them in the desired colors:

ul li { color: red; } ol li { color: green; }

A descendant selector does not have to consist of two tag selectors - there can be any number of them, separated by a space. The following code, for example, selects all tagsi, located inside the tagli, which in turn are inside the tagul:

ul li i { color: red; }

The following code is given:

<ul> <li>text <i>italic</i></li> <li>text <i>italic</i></li> <li>text <i>italic</i></li> <li>text <i>italic</i></li> </ul> <p> paragraph text <i>italic</i> </p> <p> paragraph text <i>italic</i> </p>

Color the italics from the tags redul, and in green - italics from tagsp.

The following code is given:

<p> paragraph text <b><i>bold italic</i></b> </p> <p> paragraph text <i>italic</i> </p>

Color the italics inside the tag redb, which in turn is inside the tagp.
