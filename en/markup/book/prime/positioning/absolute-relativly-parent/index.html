Positioning relative to parent in CSS

If the element is givenrelative, and his descendant -absolute, then this child will be positioned relative to its parent, not relative to the browser window.

As a rule, in this case, the parent is specified relative without offsets. In this case, nothing happens to that parent, but all of its children will now be positioned relative to it.

Let's look at examples.

Example

First, let's just make a parent block and a child block without positioning:

<div id="parent"> <div id="child"></div> </div> #parent { width: 500px; height: 300px; margin: 0 auto; border: 1px solid red; } #child { width: 200px; height: 200px; border: 1px solid green; }

:

Example

Let's now absolutely position the green block. Since the parent is not specifiedrelative, then the child will be positioned relative to the browser window:

<div id="parent"> <div id="child"></div> </div> #parent { width: 500px; height: 300px; margin: 0 auto; border: 1px solid red; } #child { position: absolute; top: 20px; left: 20px; width: 200px; height: 200px; border: 1px solid green; }

:

Example

Let us now indicate to the parentrelative. In this case, the child will be positioned relative to its parent:

<div id="parent"> <div id="child"></div> </div> #parent { position: relative; width: 500px; height: 300px; margin: 0 auto; border: 1px solid red; } #child { position: absolute; top: 20px; left: 20px; width: 200px; height: 200px; border: 1px solid green; }
