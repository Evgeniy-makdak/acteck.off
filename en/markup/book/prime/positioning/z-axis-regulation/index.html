Adjusting Z-Axis Overlay in CSS

The stacking order of elements can be adjusted using the propertyz-index, which takes as its value positive or negative integers, or zero. The property only works for elements that have the valuepositiongiven asabsolute, fixedorrelative.

When using this property, the rule for overlapping elements is as follows: the element with the valuez-indexmore.

Let's change the stacking order of the elements from the previous example. To do this, set the first elementz-indexmore than the second one, for example, like this:

#elem1 { z-index: 2; } #elem2 { z-index: 1; }
