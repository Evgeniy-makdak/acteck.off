Fixed positioning in CSS

Now we will deal with fixed positioning. It's similar to absolute, but the difference is in how these types of positioning behave when a scrollbar is present.

As you scroll the page, absolutely positioned elements will scroll along with the page. See example:

<div id="elem"></div> <div id="content"> some long text </div> #elem { position: absolute; top: 30px; right: 30px; width: 100px; height: 100px; border: 1px solid green; } #content { margin: 0 auto; width: 300px; text-align: justify; font: 16px Arial; }

:

You can make it so that when you scroll the page, the positioned element remains in place. To do this, it needs to set a fixed positioning using the propertypositionin the meaning fixed:

<div id="elem"></div> <div id="content"> some long text </div> #elem { position: fixed; /* задаем фиксированное позиционирование */ top: 30px; right: 30px; width: 100px; height: 100px; border: 1px solid green; } #content { margin: 0 auto; width: 300px; text-align: justify; font: 16px Arial; }
