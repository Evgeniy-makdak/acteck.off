Relative positioning of elements in CSS

Relative positioning is specified using the valuerelativefor propertyposition. This positioning allows you to shift elements relative to their current position by a specified amount. In this case, all other elements of the page will think that the element is located where it was originally. That is, with this type of positioning, the element does not fall out of the normal flow.

Offsets from the current position are specified by propertiestop, right, bottomAndleft.

Example

First, let's just make two blocks without positioning:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { width: 100px; height: 100px; border: 1px solid green; } #elem2 { width: 200px; height: 200px; border: 1px solid red; }

:

Example

Let's now add to the first blockrelative. This will not change anything yet, since we did not specify the block offset:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { position: relative; width: 100px; height: 100px; border: 1px solid green; } #elem2 { width: 200px; height: 200px; border: 1px solid red; }

:

Example

Let's now move our block to30pxfrom above, indicating the propertytop. In this case, all other elements will behave as if our block remained in the place where it was originally:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { position: relative; top: 30px; width: 100px; height: 100px; border: 1px solid green; } #elem2 { width: 200px; height: 200px; border: 1px solid red; }

:

Example

Let's also move our block to40pxon the left, specifying the propertyleft:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { position: relative; top: 30px; left: 40px; width: 100px; height: 100px; border: 1px solid green; } #elem2 { width: 200px; height: 200px; border: 1px solid red; }

:

Example

Negative property valuestop, right, bottomAndleftmove in the opposite direction. For example, a positive valuetopshifts down, and negative - moves up.

Let's move our second block to40pxup by giving it a negative property valuetop:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { width: 100px; height: 100px; border: 1px solid green; } #elem2 { position: relative; top: -40px; left: 20px; width: 200px; height: 200px; border: 1px solid red; }
