Absolute positioning of elements in CSS

In this lesson we will look at the absolute positioning of elements. It allows you to position elements at specified page coordinates. For example, you can position the element at position100pxat the top of the page and200pxleft. The element will move there, regardless of the fact that other elements will be located there and will simply appear on top of them. They say that in this case the element falls out ofnormal flowdocument: all other elements will behave as if our element does not exist.

To make an element absolutely positioned, the element must have the propertypositionin meaningabsolute. In addition to this property, two more are needed: one will set the vertical coordinate, and the second - horizontal.

For vertical, you need to set the indent either at the top or bottom. The top margin specifies the propertytop, and the bottom indentation is the propertybottom. For a horizontal line, you need to set the indent either to the left or to the right. The left indent specifies the propertyleft, and the right indentation is the propertyright.

Let's look at examples.

Example

First let's just make two blocks without positioning (Please note that the elements are not pressed against the edge of the window, sincebodyhasmargindefault):

<div id="elem1"></div> <div id="elem2"></div> #elem1 { width: 200px; height: 200px; border: 1px solid red; } #elem2 { width: 100px; height: 100px; border: 1px solid green; }

Example

Let's now give the green block absolute positioning, placing it in the position on top150pxand left100px:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { width: 200px; height: 200px; border: 1px solid red; } #elem2 { position: absolute; top: 150px; left: 100px; width: 100px; height: 100px; border: 1px solid green; }
