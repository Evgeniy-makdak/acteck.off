Two values ​​in the margin property in CSS

If you pass the propertymargintwo values, then the first will set the indentation at the same time from above and below, and the second - at the same time both to the right and to the left.

Let's see an example:

<div id="parent"> <div id="child"></div> </div> #parent { width: 300px; border: 1px solid red; } #child { height: 100px; border: 1px solid green; margin: 40px 20px; }
