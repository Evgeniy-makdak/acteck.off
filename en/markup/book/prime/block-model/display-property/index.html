Working with the display property in CSS

You already know that all tags belong to a specific block model. If desired, however, you can change the model using the propertydisplay. Using the valueblockof this property, the element can be made block-level, and using the valueinline- lowercase.

Let's make the tagsspanbehaved like block elements:

<span>text</span> <span>text</span> <span>text</span> span { display: block; width: 100px; height: 100px; border: 1px solid red; margin-bottom: 20px; }

:

Now let's make the tagsdivbehaved like lowercase:

<div>text</div> <div>text</div> <div>text</div> div { display: inline; width: 100px; height: 100px; border: 1px solid red; }

:

Suppose you have several paragraphs in your HTML code. Make them behave like inline elements.

Suppose you have several links in your HTML code. Make them behave like block elements. Give them a border, width and height.
