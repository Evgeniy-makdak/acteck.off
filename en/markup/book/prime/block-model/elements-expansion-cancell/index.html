Disabling Element Expansion in CSS

As a rule, the extension described in previous lessons is harmful and you would like to get rid of it. There is a CSS property for thisbox-sizing. If you give it a valueborder-box, then neitherpadding, nor the border will extend the block:

<div id="elem1"></div> <div id="elem2"></div> #elem1 { width: 100px; height: 100px; background: #f1f1f1; margin-bottom: 20px; } #elem2 { width: 100px; height: 100px; background: #f1f1f1; padding: 25px; /* задаем отступы */ border: 10px solid green; /* задаем границу */ box-sizing: border-box; /* убираем расширение */ }
