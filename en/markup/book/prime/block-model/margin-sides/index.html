Margins for different sides in CSS

In order to set different indents for different sides, there are separate properties:margin-left- for the left side,margin-right- for the right,margin-top- for the top andmargin-bottom- for the bottom.

Let's look at an example:

<div id="parent"> <div id="child"></div> </div> #parent { width: 300px; border: 1px solid red; } #child { margin-top: 10px; /* сверху */ margin-right: 20px; /* справа */ margin-bottom: 30px; /* снизу */ margin-left: 40px; /* слева */ height: 100px; border: 1px solid green; }
