Отступы padding для разных сторон в CSS

Для внутренних отступов, как и для внешних, существуют свойства для отдельных сторон: padding-left, padding-right, padding-top и padding-bottom. Давайте посмотрим на их работу:

<div id="elem"> some long text </div> #elem { padding-top: 10px; /* сверху */ padding-right: 20px; /* справа */ padding-bottom: 30px; /* снизу */ padding-left: 40px; /* слева */ width: 300px; border: 1px solid red; text-align: justify; }
