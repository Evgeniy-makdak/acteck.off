Working with margins in CSS

Now we will deal with the propertymargin, which specifies the indentation between site elements. Let's see what this looks like in practice.

To begin with, I will give you two blocks nested inside each other withoutmargin(they will stick together and we will see a double border):

<div id="parent"> <div id="child"></div> </div> #parent { width: 300px; border: 1px solid red; } #child { height: 100px; border: 1px solid green; }

:

Now let's askmarginV30pxindoor unit:

<div id="parent"> <div id="child"></div> </div> #parent { width: 300px; border: 1px solid red; } #child { height: 100px; border: 1px solid green; margin: 30px; }
