Four values ​​in the margin property in CSS

If four values ​​are specified in the margin property, then the first value specifies the top margin. The second value determines the indentation to the right, the third - from the bottom, and the fourth - from the left. That is, we start at the top and go clockwise:top-right-bottom-left. See example:

<div id="parent"> <div id="child"></div> </div> #parent { width: 300px; border: 1px solid red; } #child { height: 100px; border: 1px solid green; margin: 10px 20px 30px 40px; }
