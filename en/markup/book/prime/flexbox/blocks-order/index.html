Order of flex blocks in CSS

Suppose we have the following blocks lined up:

<div class="parent"> <div class="child">1</div> <div class="child">2</div> <div class="child">3</div> <div class="child">4</div> <div class="child">5</div> </div> .parent { display: flex; flex-direction: row; justify-content: center; align-items: center; width: 300px; height: 200px; border: 1px solid red; } .child { width: 50px; height: 50px; border: 1px solid green; }

:

Let's change the order they appear on the screen without changing their order in the HTML code.

There is a property for thisorder, which sets the order of specific elements according to the following rule: whoever has a higher value is closer to the end of the axis, and whoever has a smaller value is closer to the beginning.

The value of the property can be a positive or negative number. By default, all elements follow each other, which means thatorderthey have zero.

Let's change the order of our elements. To do this, set the second element to the additional class elem and set the property for this classorderin meaning1:

.elem { order: 1; border: 1px solid red; }

Since all elements will be in order by default0, and our second element will have an order greater than the others, then it will be located later than them:

Let's now set it to a negative value-1:

.elem { order: -1; border: 1px solid red; }

In this case, our block will move to the beginning of the axis:

Try this property yourself for different axis directions.
