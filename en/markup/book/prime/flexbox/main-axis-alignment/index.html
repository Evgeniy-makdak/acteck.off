Aligning flex elements along the main axis in CSS

Propertyjustify-contentAllows you to align elements along the main axis. You have already gone through the meanings beforecenter, space-between, space-around, space-evenly. Let's now explore a few more meanings.

Meaningflex-startpushes the elements to the beginning of the main axis, and the valueflex-end- towards the end. Let's look at examples.

Example

Let's direct the main axis from left to right by setting the propertyflex-directionmeaningrow. Let's press the blocks to the beginning of the axis. For thisjustify-contentset to valueflex-start:

.parent { display: flex; flex-direction: row; /* главная ось слева направо */ justify-content: flex-start; /* блоки к началу главной оси */ }
