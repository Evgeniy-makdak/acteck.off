Aligning flex blocks along the cross axis in CSS

Let's now align the blocks along the transverse axis. The alignment along this axis is specified by the propertyalign-items. Meaningflex-startpresses elements to the beginning of the axis, and the valueflex-end- towards the end.

Let's try it with examples.

Example

Let's direct the main axis from left to right. In this case, the transverse axis will be directed from top to bottom. Let's adjust the placement of our blocks along both the main axis and the transverse axis.

Along the main axis, we press the blocks to its beginning, that is, to the left edge. For thisjustify-contentset to valueflex-start. Along the transverse axis we will also press the blocks to its beginning, that is, to the upper edge. For thisalign-itemsalso set to valueflex-start.

Let's see what we get:

.parent { display: flex; flex-direction: row; /* главная ось направо, поперечная вниз */ justify-content: flex-start; /* блоки к началу главной оси */ align-items: flex-start; /* блоки к началу поперечной оси */ }

Example

Let's now press the blocks to the end of the transverse axis, that is, to the bottom. For thisalign-itemsset to valueflex-end:

.parent { display: flex; flex-direction: row; /* главная ось направо, поперечная вниз */ justify-content: flex-start; /* блоки к началу главной оси */ align-items: flex-end; /* блоки к концу поперечной оси */ }

Example

Let us now direct the main axis from top to bottom. Since the main axis is vertical, the transverse axis will be directed from right to left. Let's push the blocks along both axes to their origin:

.parent { display: flex; flex-direction: column; /* главная ось вниз, поперечная вправо */ justify-content: flex-start; /* блоки к началу главной оси */ align-items: flex-start; /* блоки к началу поперечной оси */ }

Example

Let's change the direction of the main axis - direct it from bottom to top. In this case, the transverse axis will not change its direction, since the main axis is still vertical.

Let's press the blocks to the beginning of both axes:

.parent { display: flex; flex-direction: column-reverse; /* главная ось вверх, поперечная вправо */ justify-content: flex-start; /* блоки к началу главной оси */ align-items: flex-start; /* блоки к началу поперечной оси */ }
