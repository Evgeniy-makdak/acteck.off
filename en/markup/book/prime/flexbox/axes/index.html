Main and cross axis in flex blocks in CSS

We can talk about flexboxes in two terms: the first is a row or column, the second is in terms of axes. We've already dealt with rows and columns, let's now deal with axes. Understanding axes is necessary in order to align elements horizontally or vertically.

When working with flex elements, there is always a main axis and a transverse one. We can align elements bymainaxis and across the main axis. The main axis may have4directions: left to right, right to left, top to bottom and bottom to top.

Directiontransverseaxis depends on the direction of the main one: if the main axis is horizontal, then the transverse axis is directed from top to bottom, if the main axis is vertical, then the transverse axis is directed from left to right. The transverse axis cannot have any other directions.

The direction of the main axis is controlled by the propertyflex-direction. If this property has a valuerow- the main axis is directed from left to right, but if the valuerow-reverse, then from right to left. Meaningcolumndirects the axis from top to bottom, and the valuecolumn-reverse- from bottom to top.
