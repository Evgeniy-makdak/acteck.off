Checking CSS Property Support

There are quite old properties in CSS that are supported in all browsers, and there are new properties that will only be supported in the newest browsers, and not in all.

During layout, you must decide whether this property can already be used. There is a special rule for this:if the property is not supported in browsers used by less than one percent of users, then the property can be used.

However, there are nuances here. After all, there are properties, the lack of support for which will lead to the collapse of the site layout, for example, flexboxes. But there are properties whose performance is not very critical. Let's say broken transfershyphenswill not make the site much worse.

In order to check whether a property is supported by browsers, you need to use the servicecaniuse.com. It shows whether the property is supported by each version of all browsers, as well as the percentage of users who use each version.

With this data, you can make a judgment about whether a feature is already usable. Also, some properties may still be under vendor prefixes in some versions of browsers that users still use. You also need to pay attention to this.
