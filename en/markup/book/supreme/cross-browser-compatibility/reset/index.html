Resetting and Normalizing CSS Styles

By default, browsers add padding to various elements. When designing, we often have to deal with these indentations, canceling them where they are not needed. However, in different browsers the default indentation for the same elements may differ. That is, even if the default indentation suits us, there is no guarantee that it will be the same in another browser.

In the following lessons we will consider approaches to combat this problem.
