Normalizing styles in CSS

It is not necessary to reset all styles to zero. There is an alternative approach callednormalization of styles. In this approach, property values ​​are not reset to zero, but are given specific values ​​so that the default styles are the same and usable across all browsers. There are ready-made files with normalization styles, for example, the popular librarynormalize.css.

Connect the described library to some HTML file. Look at the default page appearance, without your CSS styles.
