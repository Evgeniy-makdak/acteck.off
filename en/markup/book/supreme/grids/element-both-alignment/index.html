Aligning a single element along both axes of a CSS grid

You can align a single element along both the horizontal and vertical axes simultaneously by combining propertiesjustify-selfAndalign-self.

At the beginning of the horizontal and the center of the vertical

Let's set the alignment for our first element to the origin of the horizontal and vertical axes:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: start; align-self: center; }

:

At the center of the horizontal and the beginning of the vertical

Now let's set the alignment of the first element to the center of the horizontal and the beginning of the vertical axes:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: center; align-self: start; }

:

At the end of the horizontal and the center of the vertical

Let's set the alignment of the first element to the end of the horizontal and center of the vertical axes:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: end; align-self: center; }

:

Practical problems

Create a grid consisting of six elements and arrange them in two rows. Align the second element to the origin of the horizontal and the center of the vertical axes.

Now arrange the grid elements in three rows and set the alignment of the third element to the center of the horizontal and the end of the vertical axes.

Modify the previous problem so that the alignment of the fourth element is at the end of the horizontal and the center of the vertical axis.

Now make sure that the alignment of the fifth element occurs at the beginning of the horizontal and the beginning of the vertical axes, and the sixth element - at the end of the horizontal and the center of the vertical axes.
