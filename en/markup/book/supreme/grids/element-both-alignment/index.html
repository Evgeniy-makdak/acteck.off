Выравнивание отдельного элемента по обеим осям CSS грида

Можно выровнять отдельный элемент одновременно и по горизонтальной, и по вертикальной осям путем комбинирования свойств justify-self и align-self.

По началу горизонтальной и центру вертикальной

Давайте установим выравнивание для нашего первого элемента по началу горизонтальной и вертикальной осям:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: start; align-self: center; }

:

По центру горизонтальной и началу вертикальной

Теперь давайте установим выравнивание первого элемента по центру горизонтальной и началу вертикальной осям:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: center; align-self: start; }

:

По концу горизонтальной и центру вертикальной

Давайте установим выравнивание первого элемента по концу горизонтальной и центру вертикальной осям:

<div id="parent"> <div id="elem1">1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; grid-template-columns: 100px 100px; grid-template-rows: repeat(3, 1fr); grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; } #elem1 { justify-self: end; align-self: center; }

:

Практические задачи

Создайте грид, состоящий из шести элементов и расположите их по двум рядам. Выполните выравнивание второго элемента по началу горизонтальной и центру вертикальной осей.

Теперь расположите элементы грида по трем рядам и задайте выравнивание третьего элемента по центру горизонтальной и концу вертикальной осей.

Измените предыдущую задачу, чтобы выравнивание четвертого элемента происходило по концу горизонтальной и центру вертикальной осей.

Теперь сделайте так, чтобы выравнивание пятого элемента происходило по началу горизонтальной и началу вертикальной осей, а шестого элемента - по концу горизонтальной и центру вертикальной осей.
