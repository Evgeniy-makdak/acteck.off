Shorthand for aligning cells on both axes in a CSS grid

Using a shortcut propertyplace-itemsYou can simultaneously align elements within grid cells along both axes. The property accepts two values ​​separated by a space. With the first value we specify the location of the element along the vertical axes, and the second value along the horizontal axes. The property is set on the parent element.

Let's look at how this property works with examples.

At the center of the vertical and the beginning of the horizontal

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> <div>5</div> </div> #parent { display: grid; place-items: center start; grid-template-columns: 100px 100px; grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

Practical problems

Create a grid consisting of six elements and arrange them in two columns. Align the elements to the origin of the horizontal and vertical axes of the grid.

Now arrange the grid elements in three columns and set the alignment of the elements to the center of the horizontal and center of the vertical axes of the grid.

Modify the previous task so that the elements are aligned at the end of the horizontal and the beginning of the vertical axes of the grid.
