Align on both axes inside CSS grid cells

You can simultaneously set the alignment of elements within grid cells along both the horizontal and vertical axes. For this purpose we can combine two propertiesjustify-itemsAndalign-items, which are set in the parent element.

Axes centered

Let's align our elements to the center of the horizontal and vertical axes:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; justify-items: center; align-items: center; grid-template-columns: 100px 100px; grid-template-rows: 100px 100px; grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

At the beginning of the horizontal and the end of the vertical

Let's arrange our elements in cells at the beginning of the horizontal and end of the vertical axes:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; justify-items: start; align-items: end; grid-template-columns: 100px 100px; grid-template-rows: 100px 100px; grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

At the end of the horizontal and the beginning of the vertical

Let's arrange our elements in cells along the end of the horizontal and the beginning of the vertical axes:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> </div> #parent { display: grid; justify-items: end; align-items: start; grid-template-columns: 100px 100px; grid-template-rows: 100px 100px; grid-gap: 10px; padding: 10px; border: 2px solid #696989; height: 200px; width: 400px; } #parent > div { grid-gap: 10px; padding: 10px; box-sizing: border-box; border: 1px solid #696989; }

Practical problems

Create a grid consisting of five elements arranged in three rows. Align the elements in the cells along the beginning of the horizontal and the center of the vertical axes.

Now arrange the grid elements in two rows and set the alignment of the elements inside the cells to the center of the horizontal and the beginning of the vertical axes.

Modify the previous task so that the elements are aligned at the end of the horizontal and the center of the vertical axes.
