box-shadow

Given a block size200on200. Add a red shadow to it, shifted downwards by15px, right to20pxand blur5px.

Given a block size200on200. Add a green shadow to it, shifted upward by15px, right to20pxand without blur.

Given a block size200on200. Add an inner shadow of black color to it, without shifts, but with blur5px.

Given a block size200on200. Give it an inner shadow of black color, shifted downwards by3pxand right to4px, with blur3px.

Given a block size200on200. Add two shadows to it - one on top, one below.

Given a block size200on200. Add four shadows to it so that each is on one side of the block, offset by30pxand without blur. That is, one shadow is strictly above, the other is strictly to the right, and so on.
