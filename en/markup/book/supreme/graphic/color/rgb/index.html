Setting color via rgb in CSS

At the very beginning of the textbook, when we studied colors, I told you that color can be specified either by an English word or byrgb, or through#. The last two formats allow you to get a color of any shade. Let's figure out exactly how these formats work.

To understand these methods, we first need to understand how the desired color is produced on a computer screen.

Actually a separate point on the screen (pixel) cannot glow with all the colors that are needed, since this would be technically very difficult. Each point on the screen can only glow in three colors: red, green and blue. But at the same time and in different proportions.

By combining these colors we can get any color we need, just as artists do with paints: by mixing several colors they get another new one.

To mix colors in CSS you need to specify a keyword as the property valuergb, after which list, separated by commas, specifying in what proportions these three basic colors should be taken. The colors themselves may vary from0to255. Moreover, zero is the absence of color, and255is a pure color (for example, pure red).

The letters themselvesrgbstand forred, green, blue.

Let's look at examples.

Example

Let's mix pure red and pure blue. To do this, put the first value in255, second in0, and the third - in255. We will get a purple color:

<p> текст </p> p { color: rgb(255, 0, 255); }

Example

Let's now set the first value to255, and all the rest - to zero. The result is pure red:

<p> текст </p> p { color: rgb(255, 0, 0); }

Example

This is how we get a pure green color:

<p> текст </p> p { color: rgb(0, 255, 0); }

Example

And if you don’t set it for green255, A100, the result will be partially green:

<p> текст </p> p { color: rgb(0, 100, 0); }

Example

Mix all three colors in arbitrary proportions. Look what happened:

<p> текст </p> p { color: rgb(200, 100, 125); }

Example
