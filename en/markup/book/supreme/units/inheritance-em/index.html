Inheritance and em units in CSS

Let us now have deeper nesting:

<main> <div> <p> text </p> </div> </main>

Let tagmainhas the following font size:

main { font-size: 10px; }

Tegudivset the font size toem:

div { font-size: 2em; /* соответствует 20px */ }

We will also set the font size for the paragraph toem. In this case, it will count its size relative to the div's already calculated font size:

p { font-size: 2em; /* соответствует 40px */ }

Practical problems

Let us have HTML code for which we will solve problems:

<section> <div> <p> text </p> </div> </section>

Determine what font size is inpxwill have divs as a result of running the following code:

section { font-size: 10px; } div { font-size: 2em; }

Determine what font size is inpxwill have paragraphs as a result of running the following code:

section { font-size: 10px; } div { font-size: 2em; } p { font-size: 1.5em; }

Determine what font size is inpxwill have paragraphs as a result of running the following code:

section { font-size: 40px; } div { font-size: 0.5em; } p { font-size: 0.5em; }
