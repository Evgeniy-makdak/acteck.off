vw and vh units in CSS

UnitsvwAndvhallow you to set the size relative to the viewing area of ​​the browser window. Let's look at an example.

Let us have some div:

<div></div>

Let's give it a width of half the viewport width and a height of half the viewport height:

div { width: 50vw; height: 50vh; margin: 30px auto; border: 1px solid red; }

Let's see what we can come up with:
