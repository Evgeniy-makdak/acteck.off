Indentation and em units in CSS

Let's say we have the following tags:

<div> <p> text </p> </div>

Let the paragraph font size be set topx:

p { font-size: 20px; }

Let's set the margin of this paragraph in unitsem. In this case, the indent size will be calculated from the font size of the paragraph, and not from its parent:

p { font-size: 20px; margin: 2em; /* соответствует 40px */ }

Now let the paragraph be given a size ofem:

div { font-size: 20px; } p { font-size: 2em; /* соответствует 40px */ }

Let's ask againmarginto our element. In this case, it will still be calculated from the current font value of the element itself. In this case, the font value itself will be calculated relative to the parent:

div { font-size: 20px; } p { font-size: 2em; /* соответствует 40px */ margin: 2em; /* соответствует 80px */ }
