Introduction to em units in CSS

Unitsemallow you to set the font size as a percentage of the parent size. Let's look at an example.

Let us have these nested tags:

<div> <p> text </p> </div>

Let's set the font size for our div:

div { font-size: 20px; }

And set the paragraph size to2em. This value corresponds to a font twice the parent font size:

p { font-size: 2em; /* соответствует 40px */ }

Now let’s set the paragraph size to0.5em. This value corresponds to half the parent font size:

p { font-size: 0.5em; /* соответствует 10px */ }

Meaning1emwill make the paragraph font the same as the parent div:

p { font-size: 1em; /* соответствует 20px */ }
