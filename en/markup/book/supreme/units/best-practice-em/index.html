Practices for using em in CSS

Let's look at the complete HTML code of the page:

<html> <head> <title></title> </head> <body> <main> <div> <p> text </p> </div> </main> </body> <html>

By default, all page tags have a certain font size. Actually the point is that the taghtmlby default has a certain font size:

html { font-size: 16px; /* значение по умолчанию */ }

Therefore, we can specify page tag sizes inem. At the same time, they will calculate their size relative to their immediate parents.

Let's, for example, specify the font size of the tagmain:

main { font-size: 1.5em; /* соответствует 24px */ }

Now let’s specify the font size for the tagdiv:

div { font-size: 2em; /* соответствует 48px */ }

Let us have HTML code for which we will solve problems:

<main> <h1>header</h1> <div> <p> text </p> <p> text </p> </div> <div> <p> text </p> <p> text </p> </div> </main>

In all problems below, rewrite inemall units specified in pixels:

main { margin: 16px auto; } h1 { font-size: 32px; } p { font-size: 32px; }
main { margin: 32px auto; } h1 { font-size: 32px; } div { font-size: 16px; margin-bottom: 32px; } p { font-size: 20px; }
