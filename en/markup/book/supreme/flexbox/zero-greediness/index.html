Zero greed for flex elements in CSS

Ifflex-growis equal to zero for some element, then this element will not participate in the distribution of free space. By default, greed has a zero value and therefore elements without a givenflex-growdo not share free space among themselves.

Let's do some calculations as an example. Let's say we have three flex elements, each with a width of100px. Let the first element not haveflex-grow(or it is equal0), the second element hasflex-grow, equal2, and the third is equal3.

Let the width of the parent be500px. Then the free space will be equal to200px, and for one unitflex-growwill have to200px / 5 = 40px. It turns out that the width of the first element will remain the same100px, since it does not participate in the distribution, the width of the second one will be100px + 2 * 40px = 180px, and the width of the third is100px + 3 * 40px = 220px.
