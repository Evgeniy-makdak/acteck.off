Tile padding with auto-width blocks in CSS

Let us have a tile with four blocks in a row:

.child { width: 25%; }

Let's add horizontal and vertical percentage padding to our blocks:

.child { margin-right: 1.5%; margin-bottom: 1.5%; } .child:nth-child(4n) { margin-right: 0; }

For now, what we have done will not work correctly, since the total width of the blocks and indents is greater100%. To work correctly, we must bite off a small piece from each block so that there is room for our indents. Let's figure out these pieces.

In our case, it turns out that there are four blocks, and between them there are three indents - one and a half percent each. To calculate the piece taken away, you need to divide the total indentation by the number of blocks:

1.5% * 3 / 4 = 1.125

Then the width of each block will be equal to:

25% - 1.5% * 3 / 4 = 23.875%

You will get the following code:

.child { width: 23.875%; margin-right: 1.5%; margin-bottom: 1.5%; } .child:nth-child(4n) { margin-right: 0; }

You don’t have to calculate this indent yourself, but delegate the calculation work to the functioncalc:

.child { width: calc(25% - 1.5% * 3 / 4); margin-right: 1.5%; margin-bottom: 1.5%; } .child:nth-child(4n) { margin-right: 0; }

Let the function also calculate the width of the blockcalc:

.child { width: calc(100% / 4 - 1.5% * 3 / 4); margin-right: 1.5%; margin-bottom: 1.5%; } .child:nth-child(4n) { margin-right: 0; }
