Greedy flex elements in CSS

Let us now have two flex blocks lined up in a row. These blocks are given a width of100px, and to their parent -300px:

<div class="parent"> <div class="child">1</div> <div class="child">2</div> </div> .parent { display: flex; width: 300px; height: 200px; border: 1px solid red; } .child { width: 100px; height: 100px; border: 1px solid green; }

:

As you can see, the total width of our elements is less than the width of the parent, so there is empty space on the right.

If desired, this empty space can be proportionally divided between our elements. This is done using the propertyflex-grow, given to flex elements. The value of this property is a dimensionless number.

Let's see in practice how this property works.

Example

Now we have two flex blocks with a width of100px. Their total width of elements200px, and the width of the parent is300px. It turns out that there is free space left in100px.

If the elements are not givenflex-grow, then we'll just see this free space. If it is given to them, then the actual width of the elements will be greater than the specified one - they will proportionally divide the free space among themselves and add it to their width.

Let, for example, the first elementflex-growequals1, and the second -3. Let's calculate how much free space each element will receive.

First you need to get the total number of unitsflex-growall our elements. For the first element it is equal1, and in the second -3. This means that in total it is equal to4.

Let's divide now100pxfree space on4and we get that25pxaccounts for one unitflex-grow. It turns out that one unit will be added to the first elementflex-grow, that is25px, and to the second - three units, that is75px.

The width of the first element will be125px, and the second -175px:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> </div> .parent { display: flex; width: 300px; height: 200px; border: 1px solid red; } .child { height: 50px; border: 1px solid green; } .elem1 { width: 100px; flex-grow: 1; } .elem2 { width: 100px; flex-grow: 3; }

:

Example

Let now the width of the parent be400px, width of the first element200px, and the width of the second element is100px. It turns out that the free space is again equal to100px.

Let's set each elementflex-grow, equal1. In total it will be2, that is100pxfree space must be divided into2. It turns out that50pxper unit of greed.

Since all elements have the same meaningflex-grow, then the same value will be added to all elements in50px. This means that the first element will become250px, and the second one will become150px:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> </div> .parent { display: flex; width: 400px; height: 200px; border: 1px solid red; } .child { height: 50px; border: 1px solid green; } .elem1 { width: 200px; flex-grow: 1; } .elem2 { width: 100px; flex-grow: 1; }

Example

Let again the width of the parent be400px, width of the first element200px, and the width of the second element is100px.

Let's now set the first elementflex-growin meaning3, and the second - in the meaning1. It turns out that greed in total is equal to4. Then one unit of greed is equal to100px / 4 = 25px.

It will be added to the first element75pxand he will become275px, and to the second -25px, he will become125px:

<div class="parent"> <div class="child elem1">1</div> <div class="child elem2">2</div> </div> .parent { display: flex; width: 400px; height: 200px; border: 1px solid red; } .child { height: 50px; border: 1px solid green; } .elem1 { width: 200px; flex-grow: 3; } .elem2 { width: 100px; flex-grow: 1; }
