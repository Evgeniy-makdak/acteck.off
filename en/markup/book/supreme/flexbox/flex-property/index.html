The flex shortcut property in CSS

Propertyflexis a shorthand for the properties of greed, compressibility, and size. The order of the abbreviated properties matters and is as follows:flex-grow, flex-shrink, flex-basis. Properties can be omitted from the end, in which case they will take the default value.

Rewrite the following code using a shorthand propertyflex:

.child { flex-basis: 100px; flex-grow: 1; flex-shrink: 2; }

Rewrite the following code using a shorthand propertyflex:

.child { flex-shrink: 0.5; flex-grow: 2; }

Rewrite the following code using a shorthand propertyflex:

.child { flex-shrink: 0.5; flex-grow: 1; }

Rewrite the following code using shorthand properties:

.child { flex: 1 2 200px; }

Rewrite the following code using shorthand properties:

.child { flex: 2 0.7; }

Rewrite the following code using shorthand properties:

.child { flex: 3; }
