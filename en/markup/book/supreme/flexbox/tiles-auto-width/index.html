Tile with auto-width blocks in CSS

In the previous lesson, in our tile it turned out that the width of the elements and their indents should add up to the width of the parent. It's a little less universal. Let's make the elements automatically adjust to the parent's width.

Let us have a parent like this:

.parent { display: flex; flex-wrap: wrap; width: 800px; margin: 50px auto; border: 1px solid red; }

Let the elements of this tile stand at4blocks in a row:

.child { box-sizing: border-box; width: 200px; height: 100px; border: 1px solid green; }

Let's set the width of our blocks so that each of them occupies a quarter of the parent. To do this, let us set their size to25%:

.child { box-sizing: border-box; width: 25%; height: 100px; border: 1px solid green; }
