Introduction to Layout for Different Screens in CSS

In today's world, a website must display well on devices with different screen sizes.

As a rule, devices are divided into four classes: computers (desktop), laptops, tablets (tablet) and telephones (mobile).

Previously, in ancient times, the popular screen sizes of all devices were known, fortunately there were not many of them. And the sites made sure that they were displayed correctly on a well-known list of permissions.

Nowadays there are so many devices that it is impossible to make a list of resolutions for which a website needs to be made. It is now believed that layout should be displayed correctly on a screen of any size.

This is achieved by rearranging the blocks at key points in the layout. Moreover, these key points are selected not for specific devices, but for the content of the site - so that it looks good.

Between key points, the width of site elements should be floated so that it adjusts to the width of the screen. And at key points the blocks are rearranged into other positions.

The described layout is called adaptive orresponsive (responsive). We will study it in the following lessons.
