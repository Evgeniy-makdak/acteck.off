Data object in Vue

The basis of the component is the data that we will manipulate. This data is stored in a special objectdata. This object must be returned as the result of a special method:

export default { data() { return { } } }

Let's store some text in the property of the data object:

data() { return { text1: '111', text2: '222', } }

Stored data can be displayed in a view. This is done in double curly braces, in which the name of the property whose value we want to display is written. Let's print out the values ​​of our properties:

<template> {{ text1 }} {{ text2 }} </template>

Now let’s make each of our messages appear in its own paragraph:

<template> <p>{{ text1 }}</p> <p>{{ text2 }}</p> </template>

Let indatauser's first and last name is stored:

data() { return { name: 'john', surn: 'smit', } }

Output each property in a separate tagdiv.
