Arrays in Vue

Displaying the contents of arrays in Vue is done in the same way as in pure JavaScript. Let's see how it's done. Let indatawe have a propertyarr, which contains some array inside it:

data() { return { arr: [1, 2, 3], } }

Let's print out the elements of this array:

<template> {{ arr[0] }} {{ arr[1] }} {{ arr[2] }} </template>
