Работа с data внутри методов Vue

Внутри методов можно получать доступ к свойствам объекта data. Для этого к ним нужно обращаться через this. Давайте посмотрим на примере. Пусть у нас есть следующее свойство:

data() { return { text: 'str', } }

Давайте внутри метода выведем значение этого свойства:

methods: { show: function() { alert(this.text); } }

Пусть в data хранятся два числа:

data() { return { num1: 1, num2: 2, }, }

Сделайте метод, который при запуске кода выведет сумму наших чисел через alert.
