Working with methods in the Vue framework

In Vue you can create your own functions, which in Vue terms are calledmethods. This is what the property is formethods. It is located as follows:

export default { data() { }, methods: { } }

Further, for brevity, I will write in abbreviated form, like this:

methods: { }

Let's make some test method:

methods: { show: function() { alert('!'); } }

Let's call this method in the view. In this case, it will work immediately upon loading the page:

<template> {{ show() }} </template>
