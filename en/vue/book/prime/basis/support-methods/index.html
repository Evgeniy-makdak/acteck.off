Helper Methods in Vue

You can make auxiliary methods that will be used inside other methods. Such methods must also be accessed viathis. Let's look at an example. Let's say we have the following method:

show: function() { alert(this.text); }

Let's create a helper method that will capitalize the first letter of the passed string:

cape: function(str) { return str[0].toUpperCase() + str.slice(1); }

Let's use an auxiliary method inside the main method:

methods: { show: function() { let text = this.cape(this.text); alert(text); }, cape: function(str) { return str[0].toUpperCase() + str.slice(1); } }
