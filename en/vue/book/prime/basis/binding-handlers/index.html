Hangting Event Handlers in Vue

Let's now run our method on some event. In order to attach an event to any DOM element, you need to use the directivev-on.

In this directive, after the colon you need to indicate the name of the event, and the value is the name of the method that needs to be called upon the occurrence of this event.

Let's try it in practice. Let's say we have the following method:

methods: { show: function() { alert('!'); } }

Let's say we have the following button:

<template> <button>text</button> </template>

Let's make sure that when this button is clicked, the method is calledshow:

<template> <button v-on:click="show">text</button> </template>

Usually everyone uses the shortened versionv-on. It represents a symbol@before the event name:

<template> <button @click="show">text</button> </template>
