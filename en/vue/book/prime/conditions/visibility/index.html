Display by condition in Vue

The next directive we'll study is calledv-if. It can be used to show or hide elements. How this directive works: it takes any property of an object as a parameterdata. If this property has a valuetrue- then the element will be shown, and iffalse- something hidden.

Let's look at an example. Let's say we have a paragraph with an attributev-if. Let the value of this attribute be the propertyvisible, like this:

<template> <p v-if="visible">text</p> </template>

Let's set our property value totrue. In this case the paragraph will be shown:

data() { return { visible: true, } }

But ifvisibleput in valuefalse, then the paragraph will be hidden:

data() { return { visible: false, } }
