v-else directive in Vue

Directivev-ifcan also work in conjunction withv-else. Let's see what this looks like in practice. Let the propertyisAuthcontains whether the user is authorized or not:

data() { return { isAuth: true, // тут или true, или false } }

Let's do it so that, depending on the valueisAuthEither one message or another was displayed:

<template> <p v-if="isAuth">+++</p> <p v-else>---</p> </template>

Property givenisAdmin:

data() { return { isAdmin: true, } }

Depending on the value of the property, output either one text or another.
