Passing data object properties to Vue

You can also pass object properties to child componentsdata. Let's look at an example. Let us have the following properties:

data() { return { name: 'john', surn: 'smit', } }

Let's pass their values ​​to the component:

<template> <User :name="name" :surn="surn" /> </template>

Let the properties of the parent store the name, salary and age of the employee. Pass this data to the componentEmployee.
