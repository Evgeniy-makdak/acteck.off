Передача данных любого типа в Vue

В дочерние компоненты можно передавать не только строки и числа, но и данные любого типа. Для этого нужно байндить атрибут через директиву v-bind или ее сокращенную форму.

Давайте для примера передадим массив:

<template> <User :arg="[1, 2, 3]" /> </template>

А теперь передадим объект:

<template> <User :arg="{a: 1, b: 2, c: 3}" /> </template>

А теперь передадим логическое значение:

<template> <User :arg="true" /> </template>

Передайте в дочерний компонент массив. Выведите элементы этого массива в виде списка ul.
