Iterating through a group of tags in Vue

Sometimes we need to form a group of tags in a loop at once. In this case, we can combine them with a common parent and apply a loop to it:

<template> <div v-for="elem in arr"> <p>{{ elem }}</p> <hr> </div> </template>

Sometimes we don't want to combine tags with a common parent. In this case, they can be combined with the tagtemplateby applying a loop to that tag. This tag combines other tags, but does not end up in the final layout. Let's try:

<template> <template v-for="elem in arr"> <p>{{ elem }}</p> <hr> </template> </template>

Given the following array:

data() { return { items: [1, 2, 3], } }

Using this array, create the following layout:

<div> <p>1</p> <p class="divider"></p> </div> <div> <p>2</p> <p class="divider"></p> </div> <div> <p>3</p> <p class="divider"></p> </div>

Given the following array:

data() { return { items: [1, 2, 3], } }

Using this array, create the following layout:

<ul> <li>1</li> <li class="divider"></li> <li>2</li> <li class="divider"></li> <li>3</li> <li class="divider"></li> </ul>
