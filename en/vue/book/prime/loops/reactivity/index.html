Array reactivity in Vue

Reactivity is triggered even when there are changes to arrays output viav-for. For example, let’s make sure that when you click a button, a new element is reactively added to the array and the changes instantly occur on the screen.

Let's implement what is described. Let us have an array:

data() { return { arr: ['a', 'b', 'c'], } }

Let's display the elements of this array in a loop:

<template> <p v-for="elem in arr"> {{ elem }} </p> </template>

Let's make a button that, when clicked, will add a new element to the array:

<template> <p v-for="elem in arr"> {{ elem }} </p> <button @click="add">add</button> </template>

Let's implement the corresponding method:

methods: { add: function() { this.arr.push('xxx'); } }

Exceptions

Due to JavaScript limitations, Vue is unable to detect the following changes to an array: directly setting an element by index:items[ключ] = новоеЗначениеand explicitly changing the length of the array, for example:items.length = новаяДлина.
