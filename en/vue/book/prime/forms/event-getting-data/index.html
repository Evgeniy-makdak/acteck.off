Retrieving form data by event in Vue

Let us have a button and an input into which a number is entered. Let's click on the button to display the square of the number entered in the input. Let's start implementation.

To solve the problem we need two properties. Propertynumwill change as data is entered into the input, and into the propertyreswe will write the result of squaring:

data() { return { num: 0, res: 0 } }

Let's now make an input and bind a property to itnum, let’s make a button that, when clicked, will execute the methodcalc, and also make a paragraph in which the result will be displayed:

<template> <p>{{ res }}</p> <input v-model="num"> <button v-on:click="calc">work</button> </template>

Let's write the implementation of the methodcalc:

methods: { calc: function() { this.res = this.num ** 2; } }

It turns out that as soon as the method is calledcalc(and this will happen when you click the button), to the propertyresthe result will be recorded and at the same time displayed in our paragraph.

Input given. A button is given. A paragraph is given. A number is entered into the input. Make sure that when you click on the paragraph button, the square root of this number appears.

Two inputs are given. A button is given. A paragraph is given. Numbers are entered into each input. Make sure that when you press a button, the sum of these numbers appears in the paragraph.

Two inputs are given. A button is given. Some text is entered into each input. Make it so that when you click on the button, the text of the first input becomes in the second and vice versa.
