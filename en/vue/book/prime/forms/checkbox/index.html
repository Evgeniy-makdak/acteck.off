Working with checkboxes in Vue

Let's now see how it works withcheckbox. Let's say we have the following switch:

<template> <input type="checkbox"> </template>

Let's make a propertychecked, which will control the operation of thischeckbox:

data() { return { checked: true, } }

Let's bind this property viav-model:

<template> <input type="checkbox" v-model="checked"> </template>

Ifcheckboxmarked - propertycheckedwill mattertrue, and if not marked, thenfalse. To verify this, you can display the property value on the screen, like this:

<template> <input type="checkbox" v-model="checked"> <p>{{ checked }}</p> </template>

Using the ternary operator, you can output something more meaningful:

<template> <input type="checkbox" v-model="checked"> <p>{{ checked ? 'yes' : 'no' }}</p> </template>

Dancheckbox. A paragraph is given. Using a directivev-ifdo the following: ifcheckboxchecked - the paragraph should be shown, and if not marked, it should be hidden.
