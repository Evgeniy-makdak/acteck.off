Working with select in Vue

Working withselectcarried out in the same way - an attribute is hung on the tagv-model, an object property is bound to itdata. This property will contain the text of the selected tagoption.

The following code shows an example of how a dropdown list works:

data() { return { selected: 'value1', // значение по умолчанию } }
<template> <select v-model="selected"> <option>value1</option> <option>value2</option> <option>value3</option> </select> <p>{{ selected }}</p> </template>
