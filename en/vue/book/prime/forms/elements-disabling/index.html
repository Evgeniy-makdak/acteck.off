Blocking elements in Vue

You can reactively block form elements using the attributedisabled. Let's try it. Let's create the corresponding property:

data() { return { isDisabled: true, } }

Let's bind this property to the button:

<template> <button v-bind:disabled="isDisabled">btn</button> </template>

Input given. Make a button, clicking on which will alternately block and unblock our input.

Modify the previous task so that the state of the input is regulated by a checkbox: if the checkbox is checked, then the input is unlocked, and if it is not checked, then the input is blocked.
