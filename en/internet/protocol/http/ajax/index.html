AJAX requests to HTTP

AJAX technology allows you to load part of a page from the server without reloading the entire page.

This technology works through JavaScript. Upon some event, for example, when a button is clicked, the browser sends an HTTP request to the server, requesting part of the page in the form of HTML code, or data in JSON format. After the server sends the response, the browser will display the new data in the specified block.

In the browser debugger you can look at AJAX requests with your own eyes. To do this, select the fetch/xhr filter on the "Network" tab.

Open the browser debugger for a site that uses AJAX technology.

See how new queries appear in the debugger based on the occurrence of events or time.
