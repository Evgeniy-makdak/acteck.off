HTTP messages in the browser debugger

In the browser debugger you can look at HTTP requests and responses with your own eyes. To do this, you need to go to the "Network" tab and then refresh the page. The best way to do this is in the Chrome browser.

In this tab you can view the list of downloaded files, filter by each file type, and also view the HTTP request and response headers.

Open the browser debugger on the "Network" tab for some site on the Internet.

View the list of downloaded files.

Find the column with the statuses of downloaded files.

Deliberately go to a non-existent page on the site and examine the status of the downloaded files. What can you say about them?

Click on each file from the list and examine the HTTP request and response headers.
