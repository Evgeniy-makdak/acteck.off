MIME types in the HTTP protocol

MIME types allow you to specify the type of file you are receiving so that the browser can process it correctly.

For example, the browser must understand that the received file contains CSS or an image and interpret them accordingly.

Of course, most often this information can be obtained from the file extension. For example, it is obvious that in a file with the extension.cssare CSS styles. But it will be easier for the browser if the content type is specified explicitly.

The MIME type in the HTTP protocol is specified in the headerContent-Type. Let's look at the most popular options.

Content - HTML code:

Content-Type: text/html

Content - CSS code:

Content-Type: text/css

Content - JavaScript code:

Content-Type: text/javascript

Content - simple text:

Content-Type: text/plain

Contents - PNG image:

Content-Type: image/png

Contents - JPG image:

Content-Type: image/jpeg

Content - text in JSON format:

Content-Type: application/json

Content - text in XML format:

Content-Type: text/xml
