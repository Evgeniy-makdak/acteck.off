Useful HTTP headers

In this tutorial we'll look at the meanings of some useful headers.

Host header

HeadingHostcontains the name of the domain to which the request is made:

Host: example.com

Along with a start line containing the URI and a titleHostcontaining the domain name, the URL of the requested page is obtained:

GET /test/ HTTP/1.1 Host: example.com

Header Date

HeadingDatecontains the point in time at which the HTTP message was created:

Date: Wed, 21 Oct 2020 07:28:00 GMT

Last-Modified header

HeadingLast-Modifiedin the HTTP response contains the date and time at which, according to the server, the requested resource was changed. It is used as a checker to determine whether a resource has remained unchanged. Example:

Last-Modified: Sat, 16 Jan 2020 21:16:42 GMT

Content-Type header

HeadingContent-Typespecifies the type of content sent and its encoding:

Content-Type: text/html; charset=utf-8

Header Content-Language

HeadingContent-Languagespecifies the language in which the submitted document is written:

Content-Language: ru

Header Content-Length

HeadingContent-Lengthindicates the size of the object body sent to the recipient in bytes:

Content-Length: 10000
