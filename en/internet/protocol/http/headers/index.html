HTTP headers

After the start line, HTTP headers are sent. Headers specify additional parameters of an HTTP message. Technically, each heading should start on a new line. And when all the headers are listed, there should be an empty line and the body of the HTTP message.

Each header consists of a name and a value. They are separated by a colon. Let's look at some header examples:

Content-Type: text/html;charset=utf-8 Content-Language: ru Last-Modified: Sat, 16 Jan 2020 21:16:42 GMT
