Свои HTTP Headers

Можно вводить свои HTTP Headers. Традиционно к именам таких дополнительных заголовков добавляют префикс X- для избежания конфликта имён с возможно существующими.

Одним из популярных заголовков такого типа является X-Powered-By. Он указывает платформу приложений, на которой работает сервер. К примеру, это может быть "PHP" или "Express".

Придумайте идею своего заголовка. Дайте этому заголовку соответствующее имя.
