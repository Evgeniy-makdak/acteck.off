Submitting forms using the POST method in the HTTP protocol

Let's now submit the form using the POST method. Specify the corresponding value for the attributemethod:

<form action="/page/" method="POST"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

After clicking the submit button, the following HTTP request will be executed:

POST /page/ HTTP/1.1

As you can see, when sent using this method, the request URI will not contain any form data. This data will be contained in the request body as a Query String:

test1=value1&test2=2

Make a form that submits data using the POST method and send it to some site.

After submitting the form, examine the contents of the URL and make sure that there are no GET parameters when submitting the form.

Try refreshing the page after submitting the form. What will happen?

In the browser debugger, find the "Payload" tab. See what it will contain.
