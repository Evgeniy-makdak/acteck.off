HTTP request start line

The request line consists of the following parts, separated by a space: the request method, the URI of the requested page, and the HTTP protocol version.

The request method indicates whether we want to receive something from the server or, conversely, send something to the server. Typically, two methods are used - GET to receive and POST to send data.

The requested page URI contains the address of the page that the browser wants to retrieve. A URI differs from a URL in that it does not contain a domain name (it is transmitted separately in the HTTP protocol).

The protocol version is indicated as follows: first the letters HTTP, then a slash, and then the version itself.

Let's write an example of the starting line of an HTTP request to receive a page:

GET /page/test HTTP/1.1

And now an example of a request to send data:

POST /page/test HTTP/1.1
