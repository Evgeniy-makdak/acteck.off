HTTP response status code groups

Response status codes are grouped based on the result. In this case, the group is determined by the first digit of the code number. Let's explore these groups:

Group Description
1xx Codes starting with1, are information messages from the server about the data transfer process.
2xx Codes starting with2, represent success messages.
3xx Codes starting with3, inform the client that in order for the operation to succeed, another request must be made at a different URI. That is, the server uses this response code to force the browser to executeredirectto another address.
4xx Codes starting with4, report that an error occurred on the server due to the fault of the client.
5xx Codes starting with5, report that an error occurred on the server due to the server's fault.
