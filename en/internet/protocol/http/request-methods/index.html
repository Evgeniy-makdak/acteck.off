HTTP request methods

The most common HTTP request methods are GET and POST. Let's look at these methods in more detail, and also explore other, more rare methods. The description of other methods is quite general. The fact is that these methods are widely used and the HTTP protocol indicates only their general purpose. So, here is a description of the methods:

Method Description
GET The GET method is the main method used most often. It is used to easily retrieve pages.
POST The POST method is used to transfer user data. For example, a site visitor wants to leave a comment on an article. In this case, the client must make the request using the POST method. In this case, the user's comment will be sent in the body of the request. And the server will save this comment and in response will return the article page, which will contain all the previously left comments and our new one. Also, using the POST method, files are usually uploaded to the server.
HEAD The HEAD method requests a page in the same way as the GET method, but receives only the headers in response, without the response body.
DELETE The DELETE method explicitly tells the server that the client wants to delete data stored on the server for a given URI.
PATCH The PATCH method explicitly tells the server that the client wants to change the data stored on the server for a given URI.
PUT The PUT method explicitly tells the server that the client wants to upload data to the server for a given URI.
The fundamental difference between the POST and PUT methods is the understanding of the purpose of the specified URIs. The POST method assumes that the specified URI will process the content sent by the client. Using PUT, the client assumes that the content being downloaded matches the resource located at the specified URI.

Real life

In real life, servers only support the GET and POST methods well. Therefore, the GET method is used both to obtain data and to delete and change it. And the POST method is used both for loading data for processing, and for changing and deleting data.
