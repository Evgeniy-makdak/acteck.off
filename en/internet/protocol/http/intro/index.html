Introduction to HTTP Protocol

As you know, sites are located on the Internet and to view the pages of these sites you need a special program - a browser.

Technically, websites are hosted on special computers that are connected to the Internet. Such computers are calledservers. As a rule, in order to place a website on the Internet, the website author buys space on a server from a special organization calledhosting.

The browser can receive website pages from servers located on the Internet. Within the framework of browser-server interaction, the browser is usually calledclient.

There are special rules governing communication between client and server. A set of such rules is calledprotocol. The main protocol for the operation of the site is the HTTP protocol. We will study this protocol as part of this tutorial.
