HTTP redirects

When a client accesses a server at a specific URL, the server can perform a redirect by returning a status code starting with a three.

When the client receives the redirect response, it uses the new URL provided and downloads it immediately. The address to which the transition needs to be made is indicated in the HTTP headerLocation.

Let's look at an example. Let the client make the following request:

GET /page.html HTTP/1.1

Let the server respond with a redirect:

HTTP/1.1 301 OK Location: /test.html
