Glossary

Dictionary of key terms and definitions used in web development and programming.

API

Application Programming Interface — a set of rules and protocols for interaction between different software components.

AJAX

Asynchronous JavaScript and XML — a technique for asynchronous data exchange between browser and server without page reload.

Bundler

A tool that combines multiple files (JavaScript, CSS, images) into one or more bundles for optimized loading.

Virtual DOM

A lightweight representation of the real DOM tree in memory, used by React and other frameworks to optimize UI updates.

Static Site Generator (SSG)

A tool that generates HTML pages at build time, providing fast loading and good SEO optimization.

Generics

The ability to create components, functions, and classes that work with different data types without losing type safety.

Deploy

The process of deploying an application to a server or hosting to provide user access.

Closure

A function that remembers its lexical environment even after the outer function has finished executing.

Interface

In TypeScript — a way to define the structure of an object specifying the types of its properties and methods.

Component

A reusable element of the user interface that encapsulates structure, styles, and logic.

Loader

In Webpack — a module that transforms files of certain types (CSS, images, TypeScript) before adding them to the bundle.

Mixin

A design pattern that allows combining functionality from multiple sources into a single object or class.

Plugin

A software module that extends the functionality of the main application or tool.

Proxy

An intermediary object that intercepts operations on another object and allows implementing custom behavior.

Reactivity

A mechanism for automatically updating the user interface when data changes, used in Vue and other frameworks.

Routing

A navigation mechanism between different pages or views in a single-page application (SPA).

Server-Side Rendering (SSR)

A technology for generating HTML page code on the server before sending it to the client, improving SEO and load speed.

State

Data that determines the current state of a component or application at a specific point in time.

Transpilation

The process of converting source code from one programming language to another (e.g., TypeScript to JavaScript).

Hook

A function in React that allows using state and other React features in functional components.
