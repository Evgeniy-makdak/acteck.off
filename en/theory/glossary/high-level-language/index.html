Язык высокого уровня

Язык высокого уровня представляет собой язык программирования, с помощью которого можно писать наглядный и понятный код, и который не зависит от типа используемой разработчиком ОС.

Главной чертой языка высокого уровня является широкое применение абстракции, т.е. смысловых конструкций для описания сложных понятий (структур данных, операций), которые на машинном коде имеет большую длину и представляются сложными для понимания.

К языкам высокого уровня в настоящее время относятся C++, Java, JavaScript, Python, PHP, C#, Delphi, Fortran, Lisp, Pacal, Ruby, Perl.

  • язык низкого уровня,
    который используется в программировании
  • язык машинный,
    который используется для работы вычислительных машин
  • язык разметки,
    который используется для организации макета и стиля страницы сайта
