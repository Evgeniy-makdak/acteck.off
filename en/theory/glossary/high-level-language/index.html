High level language

High level languageis a programming language with which you can write visual and understandable code, and which does not depend on the type of OS used by the developer.

The main feature of a high-level language is its widespread use of abstraction, i.e. semantic constructs to describe complex concepts (data structures, operations), which in machine code are long and seem difficult to understand.

High-level languages ​​currently include C++, Java,JavaScript, Python, PHP, C#, Delphi, Fortran, Lisp, Pacal, Ruby, Perl.
