PMBOK methodology

MethodologyPMBOK (Project Management Body Of Knowledge, Project Management Body of Knowledge) involves managing a project based on a structured approach.

Working on a project using the PMBOK methodology includes five stages:

  1. Project initiation.
  2. Project planning.
  3. Project implementation.
  4. Project effectiveness.
  5. Closing the project.

The use of the PMBOK methodology is suitable, first of all, for standard projects carried out by small teams, since it does not contain clear instructions, and the distribution of tasks into stages occurs in the work process itself.
