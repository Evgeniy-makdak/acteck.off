Markup language

Markup language- a computer language that is a collection of keywords and tags used to organize the layout and style of a website page.

Using markup language written in a text document, not only the text on the site page is determined, but also the location of all page elements (menu, footer, body text field, etc.).

Markup languages ​​includeHTML(Hypertext Markup Language), XML (Extensible Markup Language) and SGML (Standard Generalized Markup Language).
