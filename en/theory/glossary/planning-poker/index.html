Техника Planning Poker

Техника Planning Poker (Planning Poker) предназначена для оценки объема и сложности задач, которые необходимо реализовать при разработке ПО. Такая оценка проводится широким кругом специалистов различного профиля - разработчиков, тестировщиков, аналитиков, дизайнеров и других участников команды.

Работа по технике Planning Poker включает следующие этапы:

  1. Выбор задачи для оценки.
  2. Раздача все членам команды специальных карт или при помощи онлайн-сервисов (например, Planningpoker.com), если команда работает удаленно. Затем необходимо установить значения для всех карт и выбрать единицу измерения, например, затраты времени для решения задачи можно оценить от 0 до 5.
  3. Перед командой озвучивается задача и дается время на то, чтобы каждый определил какое значение карты на его взгляд будет соответствовать решению. Затем каждому нужно положить выбранную карту рубашкой вверх синхронно с остальными участниками.
  4. Все открывают карты и оценивают полученный результат. Также каждый участник команды объясняет почему он выбрал то или иное значение. Далее идет голосование и сообща принимается решение по поставленной задаче.
