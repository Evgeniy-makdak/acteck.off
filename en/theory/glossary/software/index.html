What is software

BYorsoftwareis a set of programs that ensure the functioning of various applications installed on a computer or other device (gadgets, smartphones, tablets, etc.).

Types of software by purpose:

  • Systemicensures the functioning of the computer system. The most typical example of system software is the Windows, Linux and macOS operating systems. System software also includes firmware for smartphones, smart watches, etc. gadgets.
  • Instrumentalintended for creating other programs. Tool software includes compilers and interpreters of programming languages, programming environments, code editors, frameworks and libraries.
  • Appliedrepresent programs with which the user most often interacts in everyday life, for example, the text editor Notepad. Examples of application software are various applications on a smartphone, browsers, etc.
