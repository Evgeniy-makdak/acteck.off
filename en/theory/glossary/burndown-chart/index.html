Task Burndown Chart

Task Burndown Chartis intended for a graphical representation of the amount of work already done and work to be done.

The burndown chart is very widely used in the agile SCRUM methodology.

There are two types of Burndown Charts:

  1. Work burndown chart for a sprint- displays the number of solved and upcoming tasks in the current sprint.
  2. Burndown chart for project release- represents completed and upcoming tasks remaining before the release or release of the product under development.
