Programming Paradigm

Programming Paradigmis a set of concepts, concepts and rules that define an approach to programming.

The following basic programming paradigms exist:

  • Imperative programming.
  • Declarative programming.
  • Structured programming.
  • Functional programming.
  • Logic programming.
  • Object-oriented programming, which in turn is divided into:
    • Component-Oriented Programming.
    • Prototype-oriented programming.
    • Agent-Oriented Programming.
