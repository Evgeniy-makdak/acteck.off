Programming Paradigm

Programming Paradigm представляет собой набор концепций, понятий и правил, которые определяют подход к программированию.

Существуют следующие основные парадигмы программирования:

  • Императивное Programming.
  • Декларативное Programming.
  • Структурное Programming.
  • Функциональное Programming.
  • Логическое Programming.
  • Объектно-ориентированное Programming, которое в свою очередь подразделяется на:
    • Компонентно-ориентированное Programming.
    • Прототипно-ориентированное Programming.
    • Агентно-ориентированное Programming.

  • схема MVC,

  • техника planning-poker,
