Рекомендации ITIL

Рекомендации ITIL (Information Technology Infrastructure Library, Библиотека инфраструктуры информационных технологий) представляют собой пакет рекомендаций для эффективного сочетания IT-услуг с бизнес-стратегией.

Основные концепции рекомендаций ITIL:

  • Клиентоориентированность при решении каждой задачи.
  • Ограничение избыточной информации для пользователя обо всех трудностях и нюансах организации техподдержки.
  • Предпочтение интегрированным сервисам, а не аппаратным платформам;
  • Следует широко внедрять специальные инструменты для организации совместной работы всех сотрудников технической поддержки.
  • Расширение сферы интересов техподдержки, которая помимо устранения неполадок должна заниматься анализом причин их возникновения.
