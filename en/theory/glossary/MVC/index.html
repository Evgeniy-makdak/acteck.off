MVC schema

SchemeMVC (Model-View-Controller, Model-View-Controller) is an organization of code that is distributed into blocks for each task to be performed.

The MVC code organization diagram includes the following components:

  • Modeldetermines the data and the very structure of the product being developed.
  • Performanceis responsible for the appearance of the product and its interaction with consumers.
  • Controllerprovides interaction between model, view and consumer.
