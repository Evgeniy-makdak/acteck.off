OOP methodology

MethodologyOOPorObject-oriented programmingis intended for writing any computer program or system in the form of a collection of objects, each of which belongs to a specific class and performs specific functions. And all classes are subject to the principles of inheritance.

Basic principles of OOP:

  • Abstractionallows you to highlight important aspects in the model for solving a specific problem, which are formed in the form of a class.
  • Encapsulationorganizes a fast and secure hierarchy of objects within the system. With encapsulation, one simple command is enough to control the process, without unnecessary clarifications.
  • Inheritancedefines parental relationships between objects in a program so that changes made to the parent are not duplicated in the child element.
  • Polymorphismdetermines in which situations processes should be parallelized and which ones should be combined together.
