FIFO data organization method

Method of organizing dataFIFO (First in, first out, First in, first out). assumes that the first data added to memory (or heap) will be the first to be removed from it, i.e. will be at the very top of the stack, while the most recently saved data will be at the bottom.

The following well-known data type works based on FIFO:a bunch, which is designed to work correctly with global variables and dynamically changing objects.
