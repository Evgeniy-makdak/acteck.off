Static files in Express

With Express, you can declare one of your folders to be the root of your site. This folder will be automatically searched for files requested by the browser.

Let's see how it works. Let, for example, the root of the site be the folderpublic. Let there be a file in ittest.html. Then, if you enter the URL into the browser/test.html, then Express will automatically serve the file.

In this case, the folder name will not appear in the URL.public, since the search for the requested file starts inside this folder.

Similarly, CSS files, images, client JS files and other static resources will be provided.

So let's declare our folder as the rootpublic. For this we use the following command:

app.use(express.static(__dirname + '/public/'))

Make a folder to serve static content. Place CSS style files, client JS files, images and a favicon in it. content. Check how these files are sent to the browser.
