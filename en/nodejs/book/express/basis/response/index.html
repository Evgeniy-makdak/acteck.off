Response Object in Express

Let's now understand the capabilities of the response object. As you already know, he has a methodsend, which allows you to send data to the browser:

app.get('/', function(req, res) { res.send('hello world'); });

Response code

By default, Express sends data with a code200. Using the methodstatusyou can change the response code:

app.get('/page/sfdsdf/', function(req, res) { res.status(404).send('hello world'); });

When requesting any page, give the status403.

Redirect

Using the methodredirectyou can redirect:

app.get('/page/sfdsdf/', function(req, res) { res.redirect('/'); });

By default the redirect code will be302 (found). You can change this code with the first parameter of the method:

app.get('/page/sfdsdf/', function(req, res) { res.redirect(301, '/'); });

Write a code that will execute for some address303redirect.

Response type

Using the methodtypeyou can change the response type:

app.get('/', function(req, res) { res.type('text/plain'); res.send('hello world'); });
