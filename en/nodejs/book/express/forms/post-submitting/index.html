Submitting a form via POST in Express

Let's now learn how to submit forms using the POST method. To do this, change the send type in the attributemethod:

<form action="/target/" method="POST"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

We will submit our form when requesting the main page:

app.get('/', function(req, res) { res.render('form'); });

The form itself in our case will be sent to a different address. Let's write a handler for this address. At the same time, since our form is sent using the POST method, we need to write a handler notapp.get, Aapp.post:

app.post('/target/', function(req, res) { res.send('result'); });

The form data submitted using the POST method will be contained in the body of the request. That is, we need to readreq.body:

app.post('/target/', function(req, res) { console.log(req.body); // так просто не будет работать res.send('result'); });

These forms, however, will not easily fall intoreq.body- it will be stupidly empty there. To support the work we need to install middlewarebody-parser:

npm install body-parser

Let's connect the installed software:

import bodyParser from 'body-parser';

We use it to process all forms:

app.use(bodyParser.urlencoded({extended: true}));

And now, after sending toreq.bodywill contain an object with data:

app.post('/target/', function(req, res) { console.log(req.body); // объект с данными res.send('result'); });

Ask the user for their name, city and country. Display the received data on the page.
