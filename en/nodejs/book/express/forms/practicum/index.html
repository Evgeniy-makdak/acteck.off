Practice on forms in Express

Write a script that will convert temperature from Celsius to Fahrenheit. To do this, make an input and a button

Write a script that will calculate the factorial of a number. The number itself is entered into the input and after clicking the button the user should see the result.

An input and a button are given. A number is entered into the input. By clicking the button, display a list of divisors of this number.

Dans2input and button. Numbers are entered into the inputs. By clicking the button, display a list of common divisors of these two numbers.

Write a script that will find the roots of a quadratic equation. To do this, do3input into which the coefficients of the equation will be entered.

Dans3input. Numbers are entered into them. Check that these numbers are Pythagorean triples: the square of the largest number must be equal to the sum of the squares of the other two.

An input and a button are given. The date of birth is entered into this input in the format'01.12.1990'. By clicking the button, display how many days are left until the user’s birthday.

Given a textarea and a button. Text is entered into the textarea. By clicking the button, display the number of words and the number of characters in the text.

Given a textarea and a button. Text is entered into the textarea. By pressing the button you need to calculate the percentage of each character in the text.

Dans3selector and button. The first select is days from1to31, the second select is the months from January to December, and the third is the years from1990to2025. Using these selectors you can select a date. By pressing the button, display the day of the week corresponding to this date.

Make a script horoscope. Inside it is stored an array of horoscopes for several days in advance for each zodiac sign. Upon entering the page, ask the user for his date of birth, determine his zodiac sign and display a prediction for this zodiac sign for the current day.
