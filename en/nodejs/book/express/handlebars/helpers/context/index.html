Helper context in Handlebars

Inside the helperthiswill point to the execution context. Let's look at an example. Let's make a helperupperwhich will take a string from the context and return it in uppercase:

helpers: { upper: function() { return this.toUpperCase(); } }

Let's pass an array of user names to the view:

app.get('/', function(req, res) { res.render('test', { users: ['user1', 'user2', 'user3'] }); });

Let's display our users in the view:

{{#each users}} <p> {{.}} </p> {{/each}}

And now, when outputting, apply our helper to them:

{{#each users}} <p> {{upper}} </p> {{/each}}

As you can see, we are not passing any parameter to the helper. But inthisThe helper will automatically be accessed by each user iterated through, and then eventually output in uppercase.
