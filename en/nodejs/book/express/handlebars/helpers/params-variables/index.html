Variables in helper parameters in Handlebars

Helpers can accept not only numbers and strings as parameters, but also variables. Let's see in practice. Let us pass a variable calledstr:

app.get('/', function(req, res) { res.render('test', { str: 'abcde' }); });

Let's display our variable in the view:

<p> {{str}} </p>

And now, when outputting, we will use the helper we created earlierupper:

<p> {{upper str}} </p>

Pass three date variables to the view. Apply the helper you created earlier to each of these variables to change the date output format.
