Array of objects and helpers in Handlebars

Let us now pass an array of objects to the view:

app.get('/', function(req, res) { res.render('test', { users: [ { name: 'name1', surname: 'surname1' }, { name: 'name1', surname: 'surname1' }, { name: 'name1', surname: 'surname1' }, ]; }); });

Let us have a helper for formatting an object with a user, and it is assumed that the user’s name is taken from the context:

helpers: { format: function() { return this.name + ' ' + this.surname; } }

Let's display our users in the view:

{{#each users}} <p> {{.}} </p> {{/each}}

And now, when outputting, apply our helper to them:

{{#each users}} <p> {{format}} </p> {{/each}}

Let the purchase data be passed to the view:

res.render('test', { purchases: [ { name: 'purch1', cost: 1000, amount: 5 }, { name: 'purch2', cost: 2000, amount: 6 }, { name: 'purch3', cost: 3000, amount: 7 }, ], });

Go through the purchases in a cycle and for each purchase print its price, number of units and the total cost of the purchase (price multiplied by quantity).
