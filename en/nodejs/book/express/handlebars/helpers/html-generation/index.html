Generating HTML code in Handlebars

Using helpers you can also generate HTML code. Let's make, for example, a helper that generates a link:

helpers: { link: function(href, ancor) { return '<a href="'+ href +'">' + ancor + '</a>'; } }

Let's call our helper in the view using triple brackets to avoid tag escaping:

<p> {{{link '/index.html' 'главная'}}} </p>

Make a helper that will create the tagimg. Use this helper to display several pictures on the screen.
