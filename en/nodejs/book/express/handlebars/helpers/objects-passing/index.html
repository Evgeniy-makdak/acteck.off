Passing objects to helpers in Handlebars

Helpers can accept not only primitives as parameters, but also objects or arrays. Let's make a helper as an exampleformat, the parameter will take an object with the user’s first and last name and return them in formatted form:

helpers: { format: function(user) { return user.name + ' ' + user.surname; } }

Let's pass an object with a user to the view:

app.get('/', function(req, res) { res.render('test', { user: { name: 'name', surname: 'surname', } }); });

Let's use our helper in the view:

<p> {{format user}} </p>

Let the purchase data be passed to the view:

app.get('/', function(req, res) { res.render('test', { purchase: { name: 'food', cost: 1000, amount: 5 } }); });

Make a helper that will accept an object with a purchase as a parameter and enter its cost - the amount multiplied by the quantity.
