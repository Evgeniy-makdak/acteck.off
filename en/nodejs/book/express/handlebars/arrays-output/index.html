Displaying Arrays in Handlebars Express

Let us have some pagepage.hbs:

app.get('/page/', function(req, res) { res.render('page'); });

Let an array with some data be passed to this page:

app.get('/page/', function(req, res) { res.render('page', { users: ['user1', 'user2', 'user3'] }); });

Let's say we have a content page:

<div> page </div>

Let's display our array with users on it, writing it in curly brackets:

<div> {{users}} </div>

In this case, the rendering result will be a string representation of the array:

<div> user1,user2,user3 </div>

Let's now output each element of the array in its own tag:

<div> <span>{{user.[0]}}</span> <span>{{user.[1]}}</span> <span>{{user.[2]}}</span> </div>
