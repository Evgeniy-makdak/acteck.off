Session Operations in Express

After installing and connecting the libraryexpress-sessionyou can perform various operations on sessions. This is done using an objectreq.session. You can write data to it and read it from it.

Recording sessions

Let's write something to the session:

app.get('/', function(req, res) { req.session.test = 'abcde'; res.send('hello world'); });

Reading sessions

Let's read from the session:

app.get('/', function(req, res) { let test = req.session.test; console.log(test); res.send('hello world'); });

Removing session variables

To remove a session variable use the commanddelete:

app.get('/', function(req, res) { delete req.session.test; res.send('hello world'); });

Practical problems

Make three routes. Upon entering the first route, record the data in the session. Upon entering the second one, take them out. And when you enter the third one, delete the data.

Make one route. On the first visit to this route, set to record the data in the session, and on the second visit, display this data on the screen.
