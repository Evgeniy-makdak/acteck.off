Working with sessions in Express

Sessionsallow you to save data on the server as the user moves around the site. We can record some information when visiting one page and read it from there when visiting another page.

A library is used to work with sessionsexpress-session. Let's install it:

npm install express-session

Let's connect the installed library:

import expressSession from 'express-session';

A secret key is needed for the library to work, just like for a cookie to work. Let's make it:

let secret = 'qwerty';

We initialize the library by specifying our secret key in the settings:

app.use(expressSession({ secret: secret, }));

Our library requires a library to operate.cookie-parser, since sessions are based on cookies. You can use one secret key for both cookies and sessions:

let secret = 'qwerty'; app.use(cookieParser(secret)); app.use(expressSession({ secret: secret, }));

The complete code will look like this:

import express from 'express'; import cookieParser from 'cookie-parser'; import expressSession from 'express-session'; let app = express(); let secret = 'qwerty'; app.use(cookieParser(secret)); app.use(expressSession({ secret: secret, })); app.get('/', function(req, res) { res.send('hello world'); });
