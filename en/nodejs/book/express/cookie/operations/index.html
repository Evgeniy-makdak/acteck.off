Cookie Operations in Express

After installing and connecting the librarycookie-parseryou can perform various operations on cookies. Let's see how it's done.

Recording cookies

The method used to record cookies iscookieresponse object. The method takes the cookie name as the first parameter, and the value as the second:

app.get('/', function(req, res) { res.cookie('test', 'abcde'); res.send('hello world'); });

Reading cookies

When a client requests a server page, the browser, along with headers, sends the cookies previously set in it for that site. You can read these cookies using the propertycookiesrequest object:

app.get('/', function(req, res) { console.log(req.cookies); // объект с куками res.send('hello world'); });

Deleting cookies

To delete cookies, use the methodclearCookieresponse object. This method takes the name of the cookie to delete as a parameter:

app.get('/', function(req, res) { res.clearCookie('test'); res.send('hello world'); });

Practical problems

Make three routes. When you enter the first route, write the data to a cookie. Upon entering the second one, take them out. And when you enter the third one, delete the data.

Make one route. On the first visit to this route, set a cookie, and on the second visit, display this cookie on the screen.
