Practice cookies in Express

Use the form to ask for the user's name. Write this name into a cookie. The next time you visit the site, greet the user.

Record the moment in time when the user visited the page in a cookie. When you refresh the page, display how much time has passed since you first visited the page.

Create a counter for user visits to the site. Let this counter increase by one each time the page is refreshed.
