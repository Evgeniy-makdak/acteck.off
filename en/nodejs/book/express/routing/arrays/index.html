Routing and arrays in Express

Let us have the following route, the parameter of which is a certain number:

app.get('/test/:num/', function(req, res) { console.log(req.params.num); });

Let us also have an array:

let arr = ['a', 'b', 'c'];

Let's make sure that the array element corresponding to the passed number is sent to the browser:

let arr = ['a', 'b', 'c']; app.get('/test/:num/', function(req, res) { res.send(arr[req.params.num]); });

Given the following array:

let users = ['user1', 'user2', 'user3', 'user4', 'user5'];

Create a route that will send the requested element of this array to the browser.

Modify the previous task so that when requesting a non-existent array element, it returns404page.

Array of objects

Let us now have the following array of objects:

let prods = [ { name: 'prod1', cost: 1000, }, { name: 'prod2', cost: 2000, }, { name: 'prod3', cost: 3000, }, ];

Let's write a route that prints the name and price of the requested product:

app.get('/prod/:id/', function(req, res) { let prod = prods[req.params.id]; res.send(prod.name + ' ' + prod.cost); });

Let's also create a route where you can request a specific product feature (name or cost):

app.get('/prod/:id/:field/', function(req, res) { let prod = prods[req.params.id]; res.send(prod[req.params.field]); });

Given the following array of objects:

let users = [ { name: 'user1', age: 31, }, { name: 'user2', age: 32, }, { name: 'user3', age: 33, }, ];

Make a route that returns one user, and a route that returns a given user characteristic.
