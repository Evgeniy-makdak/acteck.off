Routing and files in Express

Let us have a folderpages, and it contains three files:1.html, 2.html, 3.html. Let's make routes that send the corresponding files to the browser upon request:

app.get('/page/1/', function(req, res) { res.sendFile(__dirname + '/pages/1.html'); }); app.get('/page/2/', function(req, res) { res.sendFile(__dirname + '/pages/2.html'); }); app.get('/page/3/', function(req, res) { res.sendFile(__dirname + '/pages/3.html'); });

Let's merge three routes into one using the parameter:

app.get('/page/:num/', function(req, res) { res.sendFile(__dirname + '/pages/' + req.params.num + '.html'); });

Suppose you have several folders, and files are stored in these folders. Make a route that delivers a given file from a given folder.

Error 404

Let's now make sure that when requesting a non-existent file, a 404 error is returned. To do this we need to dochecking a file for existence. We import a module for thisfsand constants:

import fs from 'fs/promises'; import { constants } from 'fs';

Let's solve the problem with their help:

app.get('/page/:num/', function(req, res) { let path = __dirname + '/pages/' + req.params.num + '.html'; fs.access(path, constants.F_OK).then(() => { res.sendFile(path); }).catch(() => { res.status(404).send('not found'); }); });

You can rewrite the code in a synchronous style:

app.get('/page/:num/', async function(req, res) { let path = __dirname + '/pages/' + req.params.num + '.html'; try { await fs.access(path, constants.F_OK); res.sendFile(path); } catch(err) { res.status(404).send('not found'); } });

Modify the previous task so that a 404 error is returned for a non-existent file.
