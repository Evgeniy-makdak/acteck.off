Number of documents in MongoDB collection

Sometimes we need to know how many documents a collection contains. The method will help us with thiscount. Let's use it to find the number of documents in the collection with users:

let res = await coll.count(); console.log(res);

Now let’s find out the number of users whose age is equal to26years:

let res = await coll.count({age: 26}); console.log(res);
