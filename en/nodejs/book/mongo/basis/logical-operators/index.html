Logical Operators in MongoDB

In this lesson we will look at logical operators. Like conditional statements, they make our queries more flexible.

$and operator

Operator$andtakes an array with conditions and selects documents that satisfy all the conditions from the array:

let cond = {$and: [ {age: {$gte: 25}}, {age: {$lte: 29}}, ]}; let res = await coll.find(cond).toArray(); console.log(res);

Explain what the example condition selects.

Think up and write three examples of using the operator you learned.

$or operator

Operator$ortakes an array with conditions and selects documents that satisfy at least one condition from the array:

let cond = { $or: [ {age: {$lte: 26}}, {salary: 300} ]}; let res = await coll.find(cond).toArray(); console.log(res);

Explain what the example condition selects.

Think up and write three examples of using the operator you learned.

$not operator

Operator$notperforms a negation operation. It selects all documents that do not satisfy the object with conditions. Example:

let cond = {age: {$not: {$gt: 26, $lt: 29}}}; let res = await coll.find(cond).toArray(); console.log(res);

Think up and write three examples of using the operator you learned.
