Array values ​​in MongoDB

The fields of collection documents can have array values. Let's look at an example. Let us have a collection with users, for each user containing his name, an array of favorite colors and an array of favorite numbers:

[ { "name": "user1", "color": ["red"], "numbers": [1, 2, 3, 7] }, { "name": "user2", "color": ["black", "red", "blue"], "numbers": [3, 5, 9] }, { "name": "user3", "color": ["red", "green", "yellow"], "numbers": [1, 8, 9] }, ]

Let's see what operators MongoDB has for working with such fields in the form of arrays.

$all operator

Operator$allselects documents in which the field values ​​are an array that contains all of the specified elements:

let cond = {color: {$all: ['red', 'green']}}; let res = await prods.find(cond).toArray(); console.log(res);

$size operator

Operator$sizeselects documents in which the passed field contains an array of the specified size. Let's, for example, select users who have three favorite colors:

let cond = {color: {$size: 3}}; let res = await prods.find(cond).toArray(); console.log(res);

$elemMatch operator

Operator$elemMatchselects documents where the field contains an array and the array contains at least one element that matches the query.

For example, let's select all users who have at least one favorite number that falls within the given range:

let cond = {numbers: {$elemMatch: {$gt: 3, $lt: 7}}}; let res = await prods.find(cond).toArray(); console.log(res);
