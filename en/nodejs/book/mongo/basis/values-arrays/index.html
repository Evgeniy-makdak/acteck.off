Array values ​​in MongoDB

The fields of collection documents can have array values. Let's look at an example. Let us have a collection of users, for each user containing an array of languages ​​that he knows:

[ { "name": "user1", "langs": ["english", "spanish"], }, { "name": "user2", "langs": ["english", "german"], }, { "name": "user3", "langs": ["german", "spanish"], }, { "name": "user4", "langs": ["spanish", "english"], }, { "name": "user5", "langs": ["german", "english", "spanish"], }, ]

Example

Let's find all documents that have languages ​​in their array'english':

let cond = {langs: 'english'}; let res = await coll.find(cond).toArray(); console.log(res);

Example

Let us now find all documents that have two languages ​​in the array with languages ​​at the same time,'english'And'german':

let cond = {langs: ['english', 'german']}; let res = await coll.find(cond).toArray(); console.log(res);

Example

Now let's display all the documents in which'english'is in first place in the array:

let cond = {'languages.0': "english"}; let res = await coll.find(cond).toArray(); console.log(res);

Practical problems

From the above collection, get all users who know Spanish.

From the above collection, get all users who know German and Spanish.

From the above collection, get all users for whom Spanish is the second element in the array.
