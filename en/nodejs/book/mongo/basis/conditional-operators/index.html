Conditional statements in MongoDB

In this lesson we will look at conditional statements. They help us select documents that are suitable for certain conditions and add flexibility to our requests.

$lt operator

Operator$ltselects all documents in which the specified field is less than the passed value:

let cond = {age: {$lt: 27}}; let res = await coll.find(cond).toArray(); console.log(res);

Get products that have a lower price500.

$gt operator

Operator$gtselects all documents in which a given field is greater than a certain value:

let cond = {age: {$gt: 26}}; let res = await coll.find(cond).toArray(); console.log(res);

Get products that have a higher price500.

$lte operator

The operator selects all documents in which the specified field is less than or equal to the passed value:

let cond = {age: {$lte: 27}}; let res = await coll.find(cond).toArray(); console.log(res);

Get users whose age is less than or equal to44.

Get products whose price is less than or equal to500.

$gte operator

The operator selects all documents in which the specified field is greater than or equal to a certain value:

let cond = {age: {$gte: 26}}; let res = await coll.find(cond).toArray(); console.log(res);

Get products whose price is greater than or equal to500.

$eq operator

Operator$eqselects all documents in which a given field is equal to a specific value:

let cond = {age: {$eq: 25}}; let res = await coll.find(cond).toArray(); console.log(res);

Get products whose price is equal500.

$ne operator

The operator selects all documents in which the specified field is not equal to the passed value:

let cond = {age: {$ne: 25}}; let res = await coll.find(cond).toArray(); console.log(res);

Receive products whose price is not equal500.

$in operator

The operator selects all documents in which the specified field is equal to one of the values ​​of the passed array:

let cond = {age: {$in: [25, 26]}}; let res = await coll.find(cond).toArray(); console.log(res);

Get products whose price is equal500, 600or700.

$nin operator

The operator selects all documents in which the specified field is not equal to the values ​​​​passed in the array, or the field does not exist:

let cond = {age: {$nin: [25, 26]}}; let res = await coll.find(cond).toArray();

Receive products whose price is not equal100or200.

Use with other methods

Conditional statements can also be used with methods for updating a collection or deleting documents:

let cond = {age: {$lte: 27}}; let data = {$set: {salary: 1000}}; await coll.updateMany(cond, data);

Remove all products with a higher price300.

Remove all products whose price is equal100or200.
