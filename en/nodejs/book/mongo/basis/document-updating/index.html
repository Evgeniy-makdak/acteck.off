Updating documents in MongoDB

In this lesson we will look at methods that make it possible to update documents in a collection.

Updating a Single Document

MethodupdateOneupdates the first document that matches the filter specified by the first parameter. As the second parameter, the method takes an object that specifies the fields to update. The key to this object is a special name$set, which specifies an object with fields and their new values.

For example, let's give the first user with age26set the salary at900:

await coll.updateOne({age: 26}, {$set: {salary: 900}});

Find the first product with price300and set a price for it900.

Mass update

MethodupdateManyupdates all documents that fall under the filter. For example, let's give all users with age26Let's set a new salary:

await coll.updateMany({age: 26}, {$set: {salary: 1000}});

Set the price for all products at1000.

Set the price for all products at300and the remainder in10.

Receiving and updating

MethodfindOneAndUpdateupdates the document and returns it with the previous data. Example:

let res = await coll.findOneAndUpdate({age: 25}, {$set: {salary: 333}}); console.log(res);
