Filtering data by condition in MongoDB

You already know how to get all records from a collection:

let res = await coll.find().toArray(); console.log(res);

Let's nowlet's filterdata according to some condition. For example, let's get all users whose salary is equal to500:

let res = await coll.find({salary: 500}).toArray(); console.log(res);

Now let’s get all users whose age is equal to26:

let res = await coll.find({age: 26}).toArray(); console.log(res);

Now we get users for whom both conditions are simultaneously met:

let res = await coll.find({salary: 500, age: 26}).toArray(); console.log(res);
