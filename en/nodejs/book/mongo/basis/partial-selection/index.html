Partial data fetching in MongoDB

MongoDB introduces a number of functions that help you select not all records, but a specific number from a specific location.

Quantity limit

Using the methodlimitYou can limit the number of documents received. For example, let’s select not all users, but only the first three:

let res = await coll.find().limit(3).toArray(); console.log(res);

Sample shift

Using the methodskipYou can select documents not from the beginning of the collection, but from a specific number. For example, let’s select users, starting from the third:

let res = await coll.find().skip(3).toArray(); console.log(res);

Concomitant use

The described methods can be combined:

let res = await coll.find().skip(1).limit(3).toArray(); console.log(res);

You can first perform the sorting, and then apply our methods to its result:

let res = await coll.find().sort({age: 1}).skip(1).limit(3).toArray(); console.log(res);
