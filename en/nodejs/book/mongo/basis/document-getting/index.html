Retrieving a single document in MongoDB

You can get not an array of documents, but one document. For this purpose the method is usedfindOne. Let's use it to get the first user with a salary400:

let res = await coll.findOne({salary: 400}); console.log(res);

If you call the method without a filter, it will return the first document in the collection:

let res = await coll.findOne(); console.log(res);
