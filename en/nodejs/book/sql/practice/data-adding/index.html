Adding a new entry to the database in NodeJS

Let's now create a form to add a new user to our database:

<form action="" method="POST"> name: <input name="name"> age: <input name="age"> salary: <input name="salary"> <input type="submit"> </form>

In fileapp.jslet's write a GET route for rendering the form:

app.get('/add', async function(req, res) { res.render('add'); });

Next, we will create a POST route for processing the form. We will declare variables in itname, ageAndsalary, which will contain data after parsing the corresponding form inputs. Since age and salary will come out of the form as strings, they need to be converted to numbers using the functionNumber. Next, we write a query to the database to insert data, specifying all the necessary variables through interpolation:

app.post('/add', async function(req, res) { let name = req.body.name; let age = Number(req.body.age); let salary = Number(req.body.salary); let query = `INSERT INTO users (name, age, salary) VALUES ('${name}', ${age}, ${salary})`; let [results, fields] = await connection.query(query); res.render('add'); });
