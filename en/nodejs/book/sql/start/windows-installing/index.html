Installing MySQL on Windows OS

There are the following methods for installing MySQL on Windows OS:

  • Boot using MSI (Microsoft Software Installer) - a set of files containing all the necessary information and instructions for installing the program. This installation method is the recommended one.
  • Download and install using a ZIP archive from the official website.
  • If you work with Docker, then MySQL can be installed through an image located on Docker Hub.

On the page withdocumentationchoose the MySQL installation method that is most convenient for you.
