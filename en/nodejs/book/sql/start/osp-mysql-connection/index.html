Соединение с БД MySQL при использовании Open Server Panel

Для разработки приложений на ОС Windows можно (не обязательно) использовать специальное ПО - Open Server Panel (OSP). OSP включает в себя набор необходимого серверного программного обеспечения, включая и базу данных MySQL.

Установку Open Server Panel (OSP) рекомендуется проводить согласно документации. После установки OSP в настройках подключаемых модулей можно выбрать нужную версию MySQL.
