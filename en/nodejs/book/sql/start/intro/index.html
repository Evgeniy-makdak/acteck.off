Introduction to Databases in NodeJS

A database (abbreviated as DB) is the place in which website data is stored. This can be page texts, lists of users with their logins and passwords, product catalogs, and more.

The database consists oftables. You know what a table is from life: it has rows and columns. And at the intersection of rows and columns there are cells.

In databases, columns are often calledfields, and the lines arerecords.

The database itself is a program that stores data and allows this data to be retrieved and modified.

Receiving and changing data occurs usingrequests. Queries are commands written in a special language - SQL.

Working with databases through NodeJS is not particularly difficult. The only problem that may arise is the long initial preparatory stage, which will last for several lessons. But after this, everything will become much easier.

SQL databases run on various engines. The most popular of them are:

  • MySQL
  • Oracle
  • PostgreSQL
  • Microsoft SQL Server

In further lessons, all examples will be based on working with the MySQL engine.
