Practice on organizing databases

There are seas, rivers and their tributaries. Rivers can be tributaries of other rivers or flow directly into the sea. Create a table structure.

User, exchange of messages between users. Create a table structure.

Website with dates of football games. Each game contains the date of the game, the first team and the second team. There are players, each of whom belongs to one team. Create a table structure.

A forum, there are categories in it, topics in them (a topic belongs to only one category), posts in the topics. The topic has an author. Users can exchange private messages. Create a table structure.

A social network, it contains users, their friends, a wall, comments on posts on the wall. Users can exchange private messages. Create a table structure.

Family tree. The user, his grandparents, mother, father, brothers, sisters, children. You can find any relative in any generation (for example, great-great-great-grandfather). It is necessary to preserve and receive family ties. Create a table structure.
