Preparatory manipulations for working with SQL in NodeJS

Let's now learn how to work with databases through NodeJS. To do this, you first need to establish a connection to the database server.

This is done using special modules. The most popular of them ismysql2. Let's install it into our working directory:

npm install mysql2

After installationmysql2you need to register the connection to the database in your working file. It’s most convenient to work through promises, so let’s importmysqlfrom modulemysql2/promise:

import mysql from 'mysql2/promise';

Next, to connect to the database tomysqlthroughawaitapply the methodcreateConnection. In its parameters we set the host name, user, database name and password:

let connection = await mysql.createConnection({ host: 'localhost', // имя хоста database: 'test', // имя базы данных user: 'root', // имя пользователя password: '', // пароль });

As you can see, the result of this method is written to a variable. Using this variable, we will subsequently access the database with which we have established a connection. Let's look at its contents:

console.log(connection);
