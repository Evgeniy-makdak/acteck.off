Fetching maximum value in SQL in NodeJS

To select one maximum value from all records, use the functionMAX.

Example

Let's select the user with the maximumid:

let query = "SELECT MAX(id) FROM users";

Result of the executed code:

[ { 'MAX(id)': 6 } ]

Example

You can simultaneously select the maximum and minimum field value:

let query = "SELECT MAX(id), MIN(id) FROM users";

Result of the executed code:

[ { 'MAX(id)': 6, 'MIN(id)': 1 } ]
