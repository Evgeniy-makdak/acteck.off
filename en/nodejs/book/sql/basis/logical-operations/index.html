Logical operations in SQL query in NodeJS

In the selection condition, you can make more complex combinations using the commandsORAndAND. They work in the same way as their analogues in designif. Let's look at examples.

Example

We will select users with a salary500And age23year:

let query = "SELECT * FROM users WHERE salary=500 AND age=23";

Example

We will select users with a salary500OR age23year:

let query = "SELECT * FROM users WHERE salary=500 OR age=23";

Example

We will select users with a salary from450to900:

let query = "SELECT * FROM users WHERE salary>450 AND salary<900";

Example

We will select users with an age from23to27years inclusive:

let query = "SELECT * FROM users WHERE age>=23 AND age<=27";

Example

Complex command combinationsORAndANDcan be grouped using parentheses to show the precedence of conditions:

let query = "SELECT * FROM users WHERE (age<20 AND age>27) OR (salary>300 AND salary<500)";
