Limiting the number of SQL records in NodeJS

To limit the number of lines output, use the commandLIMIT.

Example

Let's select the first two users:

let query = "SELECT * FROM users LIMIT 2";

Result of the executed code:

[ { id: 1, name: 'user1', age: 23, salary: 400 }, { id: 2, name: 'user2', age: 25, salary: 500 } ]

Example

Let's select all users with a salary500and then usingLIMITLet's take only the first two of those selected:

let query = "SELECT * FROM users WHERE salary=500 LIMIT 2";

Result of the executed code:

[ { id: 2, name: 'user2', age: 25, salary: 500 }, { id: 3, name: 'user3', age: 23, salary: 500 } ]

Example

By usingLIMITYou can select multiple rows from the middle of the result. In the example below we will select from the second line (line numbering from zero),4pieces:

let query = "SELECT * FROM users LIMIT 1,4";

Result of the executed code:

[ { id: 2, name: 'user2', age: 25, salary: 500 }, { id: 3, name: 'user3', age: 23, salary: 500 }, { id: 4, name: 'user4', age: 30, salary: 900 }, { id: 5, name: 'user5', age: 27, salary: 500 } ]
