Renaming a field in NodeJS

There are situations when you need to rename their original fields when displaying records. In such cases, the SQL command is usedAS. It assigns a new name to fields or tables when selecting from the database, while no changes occur in the database itself.

Example

Let's choose from our tableusersnames of all users and give this field a different name:

let query = "SELECT name as user_name FROM users";

Result of the executed code:

[ { user_name: 'user1' }, { user_name: 'user2' }, { user_name: 'user3' }, { user_name: 'user4' }, { user_name: 'user5' }, { user_name: 'user6' } ]

Example

You can only rename one field from a selection:

let query = "SELECT id, age as user_age FROM users";

Result of the executed code:

[ { id: 1, user_age: 23 }, { id: 2, user_age: 25 }, { id: 3, user_age: 23 }, { id: 4, user_age: 30 }, { id: 5, user_age: 27 }, { id: 6, user_age: 28 } ]

Example

To rename a table you need to use the commandASto the right of her name:

let query = "SELECT id, age as user_age FROM users as users_table";

Practical problems

Restore all records in the table from the dump you saved.

Rename the user salary field when outputting it to the console.
