Resource Files in NodeJS

Let's now learn how to give resource files to the browser. Resources include image files, CSS style files, client script files, and so on.

Let's look at working with resources using pictures as an example. Let, for example, at the address/page.htmlwe want to serve an HTML file, and to the address/image.png- a picture.

Let's do this:

http.createServer(async (request, response) => { if (request.url != '/favicon.ico') { let data; let type; if (request.url === '/page.html') { data = await fs.promises.readFile('page.html', 'utf8'); type = 'text/html'; } if (request.url === '/image.png') { data = await fs.promises.readFile('image.png'); type = 'image/png'; // правильно укажем mime-тип } response.writeHead(200, {'Content-Type': type}); response.write(data); response.end(); } }).listen(3000);

Create a filestyles.css. Submit it upon request. Be sure to specify the data type correctly.

Create a filescript.js. Submit it upon request. Be sure to specify the data type correctly.

Place a file with a favicon and name itfavicon.ico. Remove the condition for blocking double requests in your code, and instead serve the correct favicon.

Automatic Browser Requests

Consider the following HTML page:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>my page</title> <link rel="stylesheet" href="styles.css"> <script src="script.js"></script> </head> <body> <p> my first page on NodeJS </b> <img src="img.png"> </body> </html>

As you can see, this page includes a CSS file, a JavaScript file, and an image. This means that when the browser receives and reads the HTML code of our page, it will automatically send more3request to obtain the specified files.

Copy the given HTML page code and place it in a file. Give this file to the browser when prompted. Make sure the browser receives the resource files it requests that our HTML page links to.
