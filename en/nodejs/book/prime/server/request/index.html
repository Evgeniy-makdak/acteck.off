Http server request object in NodeJS

Request objectrequestcontains browser request data to the server. In this object the propertyurlcontains the address of the requested page, propertymethodcontains the HTTP request method, and the propertyheaderscontains an array of sent headers.

Let's display the values ​​of these properties on the screen:

http.createServer((request, response) => { console.log(request.url); console.log(request.method); console.log(request.headers); response.writeHead(200, {'Content-Type': 'text/html'}); response.write('text'); response.end(); }).listen(3000);
