Serving HTML pages from files in NodeJS

Let's now return upon request not just some text, but a correctly formatted HTML page. Let such a page be stored in a filepage.html:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>my page</title> </head> <body> <p> my first page on NodeJS </b> </body> </html>

Let's read this page from a file on request and send it to the browser:

http.createServer(async (request, response) => { if (request.url != '/favicon.ico') { let text = await fs.promises.readFile('page.html', 'utf8'); response.writeHead(200, {'Content-Type': 'text/html'}); response.write(text); response.end(); } }).listen(3000);

As you can see, reading the file is done in an asynchronous manner. In this case, while waiting for the file to be read from disk, the server will be able to process requests from other users to our server.

Given an object with URLs and corresponding HTML page names:

let obj = { '/page1': 'file1.html', '/page2': 'file2.html', '/page3': 'file3.html', }

Make a server based on this object. When requesting an address that exists in an object, provide the corresponding page, and when requesting an address that is missing, provide an error message and status404.
