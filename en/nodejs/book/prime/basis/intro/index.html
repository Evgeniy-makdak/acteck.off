Introduction to NodeJS

This tutorial is about NodeJS - server-side JavaScript. Before studying this tutorial, you already need to: know the JavaScript language, haveinstalledNodeJS, imagineHTTP protocol operation, be able to work with npm, be able to work with the terminal, be able to work with asynchrony, know what CommonJS and ES6 modules in JavaScript are.

NodeJS is regular JavaScript, but does not run in the browser (on the client), and on the server (on the back). NodeJS is an alternative to PHP, Python, and other languages ​​that run on the server. The advantage of NodeJS is that JavaScript will be used both on the client and on the server. This means that you will not need to switch your attention between two languages, and in general - to fully create a website you will only need to know one language.

Running NodeJS

Create a file anywhere with the extensionjs, For example,app.js. Print anything in this file to the console using the commandconsole.log:

console.log('test');

Open the folder with your file in a terminal. Write the following command in it:

node app.js

As a result, NodeJS will execute the code in this file and you will see the console output.

You can, of course, perform any operation that is possible in JavaScript:

let a = 3; let b = 4; console.log(a + b);

In your file, run a loop and use it to print the numbers from1to10.
