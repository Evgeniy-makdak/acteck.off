Name of the folder with the script in NodeJS

If your NodeJS works in the CommonJS style, then the constant will be available in your script files__dirname:

console.log(__dirname);

In ES modules, however, this constant has been removed. However, it is not difficult to obtain it yourself. Let's create a file for this__dirname.js, exporting the path we need to the folder with the script:

import { dirname } from 'path'; import { fileURLToPath } from 'url'; const __dirname = dirname(fileURLToPath(import.meta.url)); export default __dirname;

Now in the required executable file we can import the module we created and get the constant we need:

import __dirname from './__dirname.js'; console.log(__dirname);

Create a file__dirname.js. Connect it to your executable. Print the path to your working folder to the console.
