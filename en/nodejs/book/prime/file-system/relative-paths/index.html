Relative paths in NodeJS

As you already know, you should write the file name in the read or write method parameter. This, however, only works if the file being read is in the same folder in which our script is run.

If the file is located in a different location, then you need to write the path to this file in the method parameter.

Let's look at examples.

Example

Let's say we have the following file structure:

  • index.js
  • /directory/
    • readme.txt

Let's read the contents of the text file. To do this, in addition to the file name, we will also need to indicate the folder in which it is located:

let path = 'directory/readme.txt'; let data = await fs.promises.readFile(path, 'utf8');

Example

Let's say we have the following file structure:

  • /script/
    • index.js
  • readme.txt

In this case, an attempt to read our file, specifying its name as the path, will end with an error:

let path = 'readme.txt'; let data = await fs.promises.readFile(path, 'utf8'); // выдаст ошибку

Why is there an error? The fact is that we wrote the file name in the function parameter. This means that the file being read must be located in the same folder as the file being launched.

However, our readable file is located one level higher, that is, in the folder that contains the folder with the script.

In this case, we must explicitly indicate in the file path that this file should be searched at a higher level. To do this, before the file name you should write../. Let's do this:

let path = '../readme.txt'; let data = await fs.promises.readFile(path, 'utf8');

Example

Let's say we have the following file structure:

  • /script/
    • index.js
  • /directory/
    • readme.txt

In this case, when reading a file, we will first go to the level above, and then specify the path to our file relative to this level:

let path = '../directory/readme.txt'; let data = await fs.promises.readFile(path, 'utf8');

Example

Let's say we have the following file structure:

  • /script/
    • /test/
      • index.js
  • /directory/
    • readme.txt

In this case, we will need to go upstairs two times:

let path = '../../directory/readme.txt'; let data = await fs.promises.readFile(path, 'utf8');
