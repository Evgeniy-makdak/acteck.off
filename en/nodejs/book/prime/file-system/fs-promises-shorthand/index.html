Shortening fs.promises in NodeJS

Let's remember how we connected the modulefs:

import fs from 'fs';

After such a connection, promise variants of the methods of this module will be available to us viafs.promises:

fs.promises.readFile('readme.txt', 'utf8').then(data => { console.log(data); });

However, writing the propertypromisesbefore each necessary method is not very convenient. Let's do something smarter - write it into a variablefsnot the module itself, but its part with promises:

import fs from 'fs/promises';

And now we can omit the annoying property:

fs.readFile('readme.txt', 'utf8').then(data => { console.log(data); });

The following code is given:

import fs from 'fs'; async function func() { let data = await fs.promises.readFile('readme.txt', 'utf8'); console.log(data); } func();

Simplify it according to the theory you have learned.
