fstat method

Methodfstatreturns statistics about a file descriptor as an object. The first parameter of the method specifies the file descriptor we need, the second and third optional parameters indicate an object with options and a callback.

Syntax

fs.fstat(файловый дескриптор, [опции], [коллбэк]);

Object with options

Option Description
bigint Iftrue, specifies the display of numeric values ​​in the formatbigint. Default valuefalse.

Example

Let's asynchronously find out statistical information about the file descriptor through a callback:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); fs.fstat(fileDescriptor, function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Let's synchronously read the statistical information about the file descriptor viafstatSync:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); try { let data = fs.fstatSync(fileDescriptor); console.log(data); } catch (err) { console.log(err); }
