opendir method

Methodopendiropens a folder and returns an objectDir. The first parameter of the method specifies the path to the folder that we want to open. In the second optional parameter we specify an object with options. In the third optional parameter we specify the callback.

Syntax

fs.opendir(путь к папке, [опции], [коллбэк]);

Object with options

Option Description
encoding Data encoding mode. Default value'utf8'.
bufferSize Buffer size (default is32).
recursive Recursive - opens a folder with all contents (boolean by defaultfalse)

Example

Let's say we have the following file structure:

  • /dir/
    • file1.txt
    • file2.txt

Let's open the folder asynchronously viaawait:

<~show~> import fs from 'fs/promises'; try { let data = await fs.opendir('./dir/'); console.log(data); } catch (err) { console.log(err); }

Example

Now let's open the folder asynchronously viathen:

import fs from 'fs/promises'; fs.opendir('./dir/').then(data => { console.log(data); }).catch((err) => { console.log(err); });

Example

Let's open the folder asynchronously via a callback:

import fs from 'fs'; fs.opendir('./dir/', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Now let's synchronize the folder viaopendirSync:

import fs from 'fs'; try { let data = fs.opendirSync('./dir/'); console.log(data); } catch (err) { console.log(err); }
