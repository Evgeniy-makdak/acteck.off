futimes method

Methodfutimeschanges modification and access timestamps through a file descriptor. Timestamps can be specified using a number, string, or objectDate. If the timestamp cannot be converted to the correct number or has a valueNaN, Infinityor-Infinityan error will be thrown. The first parameter of the method specifies the file descriptor we need. In the second parameter we set the new access timestamp that needs to be set, in the third parameter we set the new modification timestamp that needs to be set. In the third optional parameter we set the callback.

Syntax

fs.futimes(файловый дескриптор, время доступа, время модификации, [коллбэк]);

Example

Let's asynchronously change the modification and access timestamps of a file descriptor via a callback. First, we find out statistics about the file descriptor through the methodstatSync, then using the commandsmtimeAndatimeset the file modification and access time. After this, we will enter new values ​​for the access time and file modification time and finally apply the methodfutimes:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt', 'r+'); let stat1 = fs.statSync('file.txt'); console.log('modification time1:', stat1.mtime); console.log('access time1:', stat1.atime); let modeTime = new Date(); let accessTime = new Date(); fs.futimes(fileDescriptor, accessTime, modeTime, () => { let stat2 = fs.statSync('file.txt'); console.log('modification time2:', stat2.mtime); console.log('access time2:', stat2.atime); });

Example

Now let's synchronously change the modification and access timestamps of the file descriptor viafutimesSync:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt', 'r+'); let stat1 = fs.statSync('file.txt'); console.log('modification time1:', stat1.mtime); console.log('access time1:', stat1.atime); let modeTime = new Date(); let accessTime = new Date(); try { fs.futimesSync(fileDescriptor, accessTime, modeTime); let stat2 = fs.statSync('file.txt'); console.log('modification time2:', stat2.mtime); console.log('access time2:', stat2.atime); } catch (err) { console.log(err); }
