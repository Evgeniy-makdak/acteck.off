fchown method

Methodfchownsets the owner of the file. Unlike the methodchownowner change occurs only if the first parameter specifies a file descriptor, not a file name. In the second parameter we set the user identifier, which corresponds to the owner to be set, in the third parameter - the identifier of the group to be set, in the fourth optional parameter you can set a callback.

Syntax

fs.fchown(файловый дескриптор, пользователь, группа, [коллбэк]);

Example

Let's set the file owner asynchronously via a callback withidequal123, Aidlet the groups be equal456:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); fs.fchown(fileDescriptor, 123, 456, err => { if (!err) { console.log('owner is changed'); } else { console.log(err); } });

Example

Now let's get throughfchownSyncsynchronously set the file owner:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); try { fs.fchownSync(fileDescriptor, 123, 456); console.log('owner is changed'); } catch (err) { console.log(err); }
