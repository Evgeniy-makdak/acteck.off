readFile method

MethodreadFilereads the contents of the file. The first parameter of the method specifies the file name, the second and third optional parameters specify the encoding and callback. As a rule, we set the encoding for readingutf8, if we omit the second parameter, then the method will return the buffer value in the form of binary data.

Syntax

fs.readFile(имя файла, [кодировка], [коллбэк]);

Example

Let's read the file asynchronously viaawait:

import fs from 'fs/promises'; try { let data = await fs.readFile('file.txt', 'utf8'); console.log(data); } catch (err) { console.log(err); }

Example

Now let's read the file asynchronously without specifying the encoding:

import fs from 'fs/promises'; try { let data = await fs.readFile('file.txt'); console.log(data); } catch (err) { console.log(err); }

Example

Let's read the file asynchronously viathen:

import fs from 'fs/promises'; fs.readFile('file.txt', 'utf8').then(data => { console.log(data); }).catch(err => { console.log(err); });

Example

Let's read the file asynchronously via a callback:

import fs from 'fs'; fs.readFile('file.txt', 'utf8', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Let's read the file synchronously viareadFileSync:

import fs from 'fs'; try { let text = fs.readFileSync('file.txt', 'utf8'); console.log(text); } catch (err) { console.log(err); }
