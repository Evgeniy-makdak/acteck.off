appendFile method

MethodappendFileadds data to the file. If the file does not exist, it will be created. In the first parameter we pass the path to the file in which we want to add data, in the second parameter - data, in the third optional parameter - an object with options, in the fourth optional parameter - a callback.

Syntax

fs.appendFile(файл, данные, [опции], [коллбэк]);

Object with options

Option Description
flags Set the mode for reading the file. By default its value is'r'- read only.
encoding Data encoding mode. Default valuenull.
mode Access mode. Default value0o666.

Example

Let's append data to a file asynchronously viaawait:

import fs from 'fs/promises'; try { let data = await fs.appendFile('file.txt', 'text'); console.log(data); } catch (err) { console.log(err); }

Example

Let's append data to a file asynchronously viathen:

import fs from 'fs/promises'; fs.appendFile('file.txt', 'text').then(data => { console.log(data); }).catch(err => { console.log(err); });

Example

Let's add data to the file asynchronously via a callback:

import fs from 'fs'; fs.appendFile('file.txt', 'text', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Let's synchronously add data to the file viaappendFileSync:

import fs from 'fs'; try { let text = fs.appendFileSync('file.txt', 'text'); console.log(text); } catch (err) { console.log(err); }
