read method

Methodreadreads data from a file through the file descriptor specified in the first parameter of the method. In the second parameter we set the buffer, in the third optional parameter - options that include the conceptoffset(default is0), length(by default the result of subtracting the buffer length in bytes minusoffset), position(default isnull). In the fourth optional parameter we set the callback.

Syntax

fs.read(файловый дескриптор, буфер, [опции], [коллбэк]);

Example

Let's asynchronously read data from a file via a callback. First, let's create a buffer using the methodallocwhich sets a buffer of a certain size, in our case it would be1024byte:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); const buffer = new Buffer.alloc(1024); fs.read(fileDescriptor, buffer, function(err, bytes, buffer) { let data = buffer.toString('utf8'); if (!err) { console.log(data); } else { console.log(err); } });

Example

Now let's get throughreadSyncsynchronously data from file:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); const buffer = new Buffer.alloc(1024); try { fs.readSync(fileDescriptor, buffer); let data = buffer.toString('utf8'); console.log(data); } catch (err) { console.log(err); }
