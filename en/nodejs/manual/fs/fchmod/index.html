fchmod method

Methodfchmodchanges access rights to a file or folder. Unlike the methodchmodin the first parameter we indicate the file descriptor we need, not the file name. In the second parameter we set the access mode, in the third optional parameter we set the callback.

Syntax

fs.fchmod(файловый дескриптор, режим доступа, [коллбэк]);

Example

Let's asynchronously set up read access for a file for a group of users via a callback:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); fs.fchmod(fileDescriptor, 0o40, err => { if (!err) { console.log('read by group'); } else { console.log(err); } });

Example

Now let's get throughfchmodSyncsynchronously establish access to file recording by a group of users:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); try { fs.fchmodSync(fileDescriptor, 0o20); console.log('write by group'); } catch (err) { console.log(err); }
