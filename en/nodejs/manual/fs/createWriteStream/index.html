createWriteStream method

MethodcreateWriteStreamopens a stream to write data to a file. In the first parameter of the method we set the path to the file that we want to read. The file path can be a string or a buffer. In the second optional parameter we specify an object with options. In the third parameter we specify the callback.

Syntax

fs.createWriteStream(путь к файлу, [опции], коллбэк);

Object with options

Option Description
flags Set the mode for reading the file. By default its value is'r'- read only.
encoding Data encoding mode. Default valuenull.
start The position from which file data reading begins.
end The position at which reading of file data ends.
highWaterMark Determines the amount of data that the stream buffers before it stops requesting more data. Default value64 * 1024.
fd File descriptor. Default valuenull.
mode Access mode. Default value0o666.
autoClose Automatically close a file handle after a thread stops. Default valuetrue.
emitClose Generating stream closure. Default valuetrue.
fs Implements options for opening, reading, and closing a file. Default valuenull.
signal The time after which processing of the data stream will stop. Default valuenull.

Example

Let's open a stream asynchronously and write data to a file via a callback:

import fs from 'fs'; let stream = fs.createWriteStream('file.txt'); stream.write('text', err => { if (!err) { console.log('text written'); } else { console.log(err); } });
