chown method

Methodchownchanges the owner of the file. In the first parameter we indicate the file we need, in the second parameter - the user identifier that corresponds to the owner being set, in the third parameter - the identifier of the group being set, in the fourth optional parameter you can specify a callback.

Syntax

fs.chown(файл, идентификатор пользователя, идентификатор группы, [коллбэк]);

Example

Let's get throughawaitasynchronously set the owner of the file withidequal123, Aidgroups equal456:

import fs from 'fs/promises'; try { await fs.chown('file.txt', 123, 456); console.log('owner is changed'); } catch (err) { console.log(err); }

Example

Let's get throughawaitAsynchronously set the owner of the file from the previous example:

import fs from 'fs/promises'; fs.chown('file.txt', 123, 456).then(() => { console.log('owner is changed'); }).catch((err) => { console.log(err); });

Example

Now let's set the file owner asynchronously via a callback:

import fs from 'fs'; fs.chown('file.txt', 123, 456, err => { if (!err) { console.log('owner is changed'); } else { console.log(err); } });

Example

Now let's get throughchownSyncsynchronously set the file owner:

import fs from 'fs'; try { fs.chownSync('file.txt', 123, 456); console.log('owner is changed'); } catch (err) { console.log(err); }
