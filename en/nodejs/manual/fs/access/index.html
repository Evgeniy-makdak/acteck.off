access method

Methodaccessshows the user's permissions to access a file or folder. In the first parameter we specify the path to the file or folder. In the second and third optional parameters we specify the modifier and callback, respectively.

Syntax

fs.access(файл или папка, [модификатор], [коллбэк]);

Example

Let's asynchronously find out if we have access to the file viaawait:

import fs from 'fs/promises'; try { await fs.access('file.txt'); console.log('+++'); } catch (err) { console.log('---'); }

Example

Now let's asynchronously find out if we have access to the file viathen:

import fs from 'fs/promises'; fs.access('file.txt').then(() => { console.log('+++'); }).catch((err) => { console.log('---'); });

Example

Let's asynchronously find out if we have access to the file through a callback:

import fs from 'fs'; fs.access('file.txt', function(err) { if (!err) { console.log('+++'); } else { console.log('---'); } });

Example

Now let's synchronously find out if we have access to the file viaaccessSync:

import fs from 'fs'; try { fs.accessSync('file.txt'); console.log('+++'); } catch (err) { console.log('---'); }
