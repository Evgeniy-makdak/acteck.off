closeSync method

MethodcloseSyncis used only to synchronously close a file descriptor, thereby clearing the associated file. A file descriptor is a file's I/O stream identifier, which is a non-negative integer.

Syntax

fs.closeSync(файловый дескриптор);

Example

Let's close the file descriptor synchronously. To do this, first use the methodopenSyncLet's define the file descriptor of the file we need:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); try { fs.closeSync(fileDescriptor); console.log('file closed successfully'); } catch (err) { console.log(err); }
