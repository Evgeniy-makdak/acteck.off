readdir method

Methodreaddirreads the contents of a folder and returns an array of folders and files included in it. The first parameter of the method specifies the file name, the second and third optional parameters specify the encoding and callback.

Syntax

fs.readdir(имя папки, [кодировка], [коллбэк]);

Example

Let's read the folder asynchronously viaawait:

import fs from 'fs/promises'; try { let data = await fs.readdir('./dir1/'); console.log(data); } catch (err) { console.log(err); }

Example

Let's read the folder asynchronously viathen:

import fs from 'fs/promises'; fs.readdir('./dir1/').then(data => { console.log(data); }).catch(err => { console.log(err); });

Example

Let's read the folder asynchronously via a callback:

import fs from 'fs'; fs.readdir('./dir1/', function(err, data) { if (!err) { console.log(data); } else { console.log(err); } });

Example

Let's synchronously read the folder viareaddirSync:

import fs from 'fs'; try { let data = fs.readdirSync('./dir1/', 'utf8'); console.log(data); } catch (err) { console.log(err); }
