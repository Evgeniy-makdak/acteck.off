write method

Methodwriteindicates the position in the file from which writing to the buffer begins, and also to indicate which part of the buffer to write to the file. In the first parameter of the method we specify the file descriptor. Methodwritecan be used in two versions: with a buffer and with a string variable. The differences between these options begin with the second parameter.

If we use a method with a buffer, then in the second parameter we indicate the buffer, in the third parameter - the part of the buffer intended for writing. In the fourth optional parameter, we specify the length in bytes that will be written to the file. In the fifth optional parameter we set the position from which the file will be written, and in the sixth optional parameter we set the callback.

In case we need to apply the methodwritewith a string value, in the second parameter we specify the string, in the third optional parameter - the position from which the file will be written. In the fourth optional parameter we set the encoding, the default value of which is'UTF-8'. And in the fifth optional parameter we indicate the callback.

Syntax

When using a buffer:

fs.write(файловый дескриптор, буфер, часть буфера для записи, [длина], [позиция], [коллбэк]);

When using the line:

fs.write(файловый дескриптор, строка, [позиция], [кодировка], [коллбэк]);

Example

Let's write data to a file asynchronously using a buffer. First we will open the file using the methodopen, in the second parameter of which we will add the mode'a', which allows you to simply add data to the file each time, rather than overwriting it with new ones:

import fs from 'fs'; let buffer = new Buffer.from('text'); fs.open('/file.txt', 'a', function(err, fd) { if (err) { console.log(err); } else { fs.write(fd, buffer, 0, buffer.length, null, function(err, writtenbytes) { if (err) { console.log(err); } else { console.log(writtenbytes + ' characters added to file'); } }) } })

Example

Now let's write data to a file asynchronously using a string variable:

import fs from 'fs'; const str = 'text'; fs.open('/file.txt', 'a', function(err, fd) { if (err) { console.log(err); } else { fs.write(fd, str, function(err, bytes) { if (err) { console.log(err); } else { console.log(bytes + ' bytes written'); } }) } })

Example

Let's synchronously write data to a file viawriteSyncusing buffer:

import fs from 'fs'; let buffer = new Buffer.from('text'); try { fs.open('/file.txt', 'a', function(err, fd) { if (err) { console.log(err); } else { fs.write(fd, buffer, 0, buffer.length, null, function(err, writtenbytes) { if (err) { console.log(err); } else { console.log(writtenbytes + ' characters added to file'); } }) } }) } catch (err) { console.log(err); }

Example

Let's synchronously write data to a file viawriteSyncusing a string variable:

import fs from 'fs'; const str = 'text'; try { fs.open('/file.txt', 'a', function(err, fd) { if (err) { console.log(err); } else { fs.write(fd, str, function(err, bytes) { if (err) { console.log(err); } else { console.log(bytes + ' bytes written'); } }) } }) } catch (err) { console.log(err); }
