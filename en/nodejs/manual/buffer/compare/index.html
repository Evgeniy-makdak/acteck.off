compare method

Methodcomparecompares two buffer objects and returns a number indicating their differences. If the method returns0- the two buffers are the same size,1- the first buffer is larger than the second,-1- the first buffer is smaller than the second. In the method parameter we specify two buffers that we want to compare with each other.

Syntax

Buffer.compare(первый буфер, второй буфер);

Example

Let's compare two buffers:

import { Buffer } from 'buffer'; let buffer1 = Buffer.alloc(12); let buffer2 = Buffer.alloc(10); console.log(Buffer.compare(buffer1, buffer2));

Result of the executed code:

1
