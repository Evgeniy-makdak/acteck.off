Buffer.copy property

PropertyBuffer.copycopies data from the buffer to the specified object. In the first parameter of the method we indicate the object where we want to copy the data. In the second optional parameter we set the byte index of the object from which data recording will begin (by default it is equal to0). In the third optional parameter we indicate the index of the buffer bytes from which copying should begin. In the fourth optional parameter, you can specify the index of the buffer bytes from which copying should end.

Syntax

буфер.copy(объект, куда копируем, [начало записи для объекта], [начало копирования для буфера], [конец копирования для буфера]);

Example

Let's copy data from one buffer to another:

import { Buffer } from 'buffer'; let buffer1 = Buffer.from('123'); let buffer2 = Buffer.from('456'); buffer1.copy(buffer2); console.log(buffer2.toString());

Result of the executed code:

'123'
