poolSize property

PropertypoolSizespecifies the size, in bytes, of previously allocated buffer instances that will be used for merging.

Syntax

Buffer.poolSize = значение;

Example

Let's assign a size to the buffer object:

import { Buffer } from 'buffer'; Buffer.poolSize = 10; console.log(Buffer.poolSize);

Result of the executed code:

10

See also

  • methodbyteLength,
    which returns the buffer size
  • methodcopyBytesFrom,
    which copies the base memory to a new buffer
  • propertybyteOffset,
    which returns the byte offset of the given buffer
