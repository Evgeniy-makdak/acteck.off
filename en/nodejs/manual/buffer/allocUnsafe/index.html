allocUnsafe method

MethodallocUnsafecreates a new unfilled buffer object of the specified size. The new buffer may contain information from previously created buffers, so it is considered unsafe. In the method parameter we indicate the buffer size we need in bytes.

Syntax

Buffer.allocUnsafe(размер буфера);

Example

Let's create a new buffer object using the methodallocUnsafe:

import { Buffer } from 'buffer'; let buffer = Buffer.allocUnsafe(10); console.log(buffer);

Result of the executed code:

<Buffer 00 00 00 00 00 00 00 00 00 00>

See also

    which creates a new buffer object
  • methodallocUnsafeSlow,

  • methodbyteLength,
