byteLength method

MethodbyteLengthreturns the size of the buffer in bytes. In the first parameter of the method we indicate the name of the buffer we need, in the second optional parameter - the encoding type (default is'utf8').

Syntax

Buffer.byteLength(размер буфера, [значение для заполнения], [кодировка]);

Example

Let's find out the size of our buffer:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(12); console.log(Buffer.byteLength(buffer));

Result of the executed code:

12
