toJSON method

MethodtoJSONconverts a buffer object to the formatJSON. We do not specify anything in the method parameter.

Syntax

буфер.toJSON();

Example

Let's convert our buffer object to the formatJSON:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer.toJSON());

Result of the executed code:

{ type: 'Buffer', data: [ 97, 98, 99, 100, 101 ] }
