Метод keys

Метод keys возвращает итератор ключей или индексов в объекте буфера. В параметре метода ничего не указываем.

Синтаксис

буфер.keys();

Пример

Давайте найдем индексы в нашем объекте буфера:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); for (let key of buffer.keys()) { console.log(key); }

Результат выполненного кода:

0 1 2 3 4 5

Смотрите также

  • метод entries,
    который возвращает пару индекс-байт из объекта буфера
  • метод values,
    который возвращает итератор байтов в объекте буфера
