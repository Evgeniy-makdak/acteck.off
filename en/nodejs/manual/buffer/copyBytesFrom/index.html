copyBytesFrom method

MethodcopyBytesFromcopies the base memory to a new buffer. In the first parameter of the method we specify the objectTypedArray, which we want to copy. In the second optional parameter, you can specify the starting offset of the copied object (default is0), in the third optional parameter - the number of elements to copy.

Syntax

Buffer.copyBytesFrom(копируемый объект, [смещение], [количество элементов]);

Example

Let's copy the object's memory to a new buffer:

import { Buffer } from 'buffer'; const arr = new Uint16Array([0,2]); const buffer = Buffer.copyBytesFrom(arr); console.log(buffer);

Result of the executed code:

<Buffer 00 00 02 00>
