isBuffer method

MethodisBufferchecks whether an object is a buffer. In the method parameter we indicate the object we need. Method returns valuetrueif the object is a buffer, otherwise -false.

Syntax

Buffer.isBuffer(объект для проверки);

Example

Let's check if our object is a buffer:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(4); console.log(Buffer.isBuffer(buffer));

Result of the executed code:

true

Example

Now let's check if the string is a buffer:

import { Buffer } from 'buffer'; let buffer = 'abcde'; console.log(Buffer.isBuffer(buffer));

Result of the executed code:

false
