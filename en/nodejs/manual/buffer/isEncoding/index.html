isEncoding method

MethodisEncodingchecks whether the given encoding is supported. In the method parameter we specify the encoding. Method returns valuetrue, if the encoding is supported, otherwise -false.

Syntax

Buffer.isEncoding(кодировка);

Example

Let's check using the methodisEncoding, is the encoding supported?'utf8':

import { Buffer } from 'buffer'; console.log(Buffer.isEncoding('utf8'));

Result of the executed code:

true

Example

Now let's check if the encoding is supported'asciivalue':

import { Buffer } from 'buffer'; console.log(Buffer.isEncoding('asciivalue'));

Result of the executed code:

false
