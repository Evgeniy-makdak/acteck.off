Метод values

Метод values возвращает итератор значений или байтов в объекте буфера. В параметре метода ничего не указываем.

Синтаксис

буфер.values();

Пример

Давайте найдем индексы в нашем объекте буфера:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); for (let value of buffer.values()) { console.log(value); }

Результат выполненного кода:

97 98 99 100 101 102

Смотрите также

  • метод keys,
    который возвращает итератор индексов в объекте буфера
  • метод entries,
    который возвращает пару индекс-байт из объекта буфера
