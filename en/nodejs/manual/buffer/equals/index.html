equals method

Methodequalscompares buffer objects and returns a boolean value. If the two buffer objects are the same, then the method returnstrue, otherwise -false. In the method parameter we specify the buffer that we want to compare with the current buffer.

Syntax

буфер.equals(буфер для сравнения);

Example

Let's compare two buffers:

import { Buffer } from 'buffer'; let buffer1 = Buffer.from('abc'); let buffer2 = Buffer.from('def'); console.log(buffer1.equals(buffer2));

Result of the executed code:

false

    which compares two buffer objects
  • propertyBuffer.compare,
    which compares the buffer with a given object
