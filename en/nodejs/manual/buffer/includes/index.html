includes method

Methodincludeschecks whether the buffer object contains the value specified in the first parameter. In the second optional parameter, you can specify the byte index from which the search for the value will begin, in the third optional parameter - the encoding (by default it is equal to'utf8'). Method returnstrue, if the value is in the buffer object, otherwise -false.

Syntax

буфер.includes(значение, [начало поиска], [кодировка]);

Example

Let's look to see if there is a string in our buffer object'abc':

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); console.log(buffer.includes('abc'));

Result of the executed code:

true

See also

  • methodindexOf,
    which looks for the first index of a given value in a buffer object
  • methodlength,
    which returns the number of bytes in the buffer object
