normalize method

Methodnormalizenormalizes the given file path, transforming'..'And'.'into slashes specific to the platform used (Windows or POSIX).

Syntax

path.normalize(путь к файлу);

Example

Let's normalize the file path on the Windows platform:

import path from 'path'; console.log(path.normalize('C:\\folder\\\\dir\\file\\..\\'));

Result of the executed code:

'C:\folder\dir\'

Example

Let's normalize the file path on the Windows platform32bit:

import path from 'path'; console.log(path.win32.normalize('C:////folder\\\\/\\/\\/dir/file'));

Result of the executed code:

'C:\folder\dir\file'

Example

Now let's normalize the file path on the POSIX platform:

import path from 'path'; console.log(path.normalize('/folder/dir1//dir2/file/test/..'));

Result of the executed code:

'/folder/dir1/dir2/file'
