format method

Methodformatreturns the file path as a string value from the object specified in the parameter. The object includes the following keysdir- folder name,root- root path,base- file with extension,name- name of the halyard,ext- extension. When applying the method, you can ignore some keys, but the object must contain information about the folder or file. Depending on the platform, the path syntax in the keys changes. Например, для ключаrootin POSIX the path is written as'/', and for Windows -'C:\\path\\dir''.

Syntax

path.format({папка, корневой путь, файл с расширением, имя фала, расширение});

Example

Let's find out the string path to the file for Windows, while we ignore the root path and indicate the directory path, file name and extension:

import path from 'path'; console.log(path.format({ root: '/ignored', dir: '\\dir', name: 'file', ext: '.txt', }));

Result of the executed code:

'\dir\file.txt'
