Property sep

Propertysepreturns the path segment separator or slash for the current platform as a string value. For Windows the property will return'\', for POSIX -'/'.

Syntax

path.sep;

Example

Let's define a segment separator for our Windows platform:

import path from 'path'; console.log(path.sep);

Result of the executed code:

'\'

Example

Now let's see which delimiter is reserved for POSIX:

import path from 'path'; console.log(path.sep);

Result of the executed code:

'/'
