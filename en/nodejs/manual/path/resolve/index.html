resolve method

Methodresolveconverts a sequence of path segments to an absolute path. The method processes a sequence of paths from right to left, appending each path before creating an absolute path. The returned path is normalized, with slashes removed as necessary. If no path segments are specified as parameters, the method will return the absolute path of the current working directory.

Syntax

path.resolve([сегменты путей]);

Example

Let's convert the segments to an absolute path on the Windows platform:

import path from 'path'; console.log(path.resolve('/dir', './file.txt'));

Result of the executed code:

'C:\dir\file.txt'

Example

Now let’s write more complex path segments:

import path from 'path'; console.log(path.resolve('/folder/dir', '././file.txt'));

Result of the executed code:

'C:\folder\dir\file.txt'

Example

Now let's write the segments so that inside the folderdir1there was a folderdir1with filefile.txt:

import path from 'path'; console.log(path.resolve('/folder/dir1', './dir2/file.txt'));

Result of the executed code:

'C:\folder\dir1\dir2\file.txt'
