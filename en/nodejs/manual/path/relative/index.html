relative method

Methodrelativereturns a relative path from a given path to another path based on the current working directory.

Syntax

path.relative(путь от, путь к);

Example

Let's find the relative path on Windows platform using the methodrelative:

import path from 'path'; console.log(path.relative('C:\\dir1\\file1.txt', 'C:\\dir2\\file2.txt'));

Result of the executed code:

'..\..\dir2\file2.txt'

Example

Now let's find the relative path of the subfolder relative to the main one on the Windows platform:

import path from 'path'; console.log(path.relative('C:\\dir1\\file1.txt', 'C:\\dir1\\dir2\\file2.txt'));

Result of the executed code:

'..\dir2\file2.txt'
