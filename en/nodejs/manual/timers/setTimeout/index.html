setTimeout method

MethodsetTimeoutsets the callback to be executed once after a delay in milliseconds. In the first parameter of the method we indicate the callback, in the second parameter - the delay time for its execution in milliseconds.

Syntax

setTimeout(коллбэк, время задержки);

Example

Let's use the methodsetTimeoutset the callback to be executed with a delay1000milliseconds:

function func() { console.log('text'); } setTimeout(func, 1000);

Result of the executed code:

'text'

Example

Now let's set the callback to be executed with a delay1000milliseconds via promise. There is a difference in the method parameters when executing it through a promise. In the first parameter we indicate the delay time in milliseconds, and in the second parameter we specify the message that we want to display:

import { setTimeout } from 'timers/promises'; let timeout = await setTimeout(1000, 'text'); console.log(timeout);
