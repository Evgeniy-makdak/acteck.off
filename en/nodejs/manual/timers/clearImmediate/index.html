Метод clearImmediate

Метод clearImmediate отменяет немедленное выполнение коллбэка, указанного в параметре метода.

Синтаксис

clearImmediate(название коллбэка);

Пример

Давайте с помощью метода clearImmediate остановим немедленное выполнение коллбэка func2:

setTimeout( function func1() { console.log('setTimeout()'); }, 2000); let interval = setImmediate( function func2() { console.log('setImmediate()'); }); clearImmediate(interval);

Результат выполненного кода:

'setTimeout()'
