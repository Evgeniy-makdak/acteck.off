setImmediate method

MethodsetImmediatesets immediate execution of the callback specified in the first parameter of the method. In the second optional parameter, you can set values ​​for the callback parameters.

Syntax

setImmediate(коллбэк, [значения для параметров коллбэка]);

Example

Let's use the methodsetImmediateset the callback to be executed immediatelyfunc2:

setTimeout( function func1() { console.log('setTimeout()'); }, 2000); setImmediate( function func2() { console.log('setImmediate()'); });

Result of the executed code:

'setImmediate()' 'setTimeout()'

Example

Now let's set the callback to execute immediatelyfunc2via promise:

import { setImmediate } from 'timers/promises'; const immediate = await setImmediate('text'); console.log(immediate);

Result of the executed code:

'text'
