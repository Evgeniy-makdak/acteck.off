ref method

MethodrefclassImmediaterequests that the NodeJS event loop not terminate while activeImmediate. The method returns an objectImmediate.

Syntax

Immediate.ref();

Example

Let's find out if the cycle we set is active:

let Immediate = setImmediate(() => { console.log('text'); }); console.log (Immediate.ref());

Result of the executed code:

Immediate { _idleNext: null, _idlePrev: null, _onImmediate: [Function (anonymous)], _argv: undefined, _destroyed: false, [Symbol(refed)]: true, [Symbol(asyncId)]: 7, [Symbol(triggerId)]: 0 } 'text'
