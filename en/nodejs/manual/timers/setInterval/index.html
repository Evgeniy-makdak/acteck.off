setInterval method

MethodsetIntervalsets the callback execution interval in milliseconds. In the first parameter of the method we indicate the callback, in the second parameter - the delay time for its execution in milliseconds. The third optional parameter can be used to pass values ​​for callback parameters. Method must be appliedclearIntervalso that the callback does not execute indefinitely.

Syntax

setInterval(коллбэк, время задержки, [параметры коллбэка]);

Example

Let's use the methodsetIntervalset the callback to be executed with a delay1000milliseconds:

let i = 0; let timer = setInterval(function() { if (i >= 5) { clearInterval(timer); } else { console.log(++i); } }, 1000);

Example

Now let's set the callback to be executed with a delay1000milliseconds via promise:

let i = 0; for await (let el of setInterval(1000, i)) { console.log(i++); if (i >= 5) break; } console.log(i++);

Example

Let's set up the execution of a callback that has two numbers specified in its parameters2And3. Let the sum of these numbers be displayed only three times:

let i = 0; let timer = setInterval( (num1, num2) => { console.log(`The sum of ${num1} and ${num2} is ${num1 + num2}`); i++; if (i === 3) { clearInterval(timer); } }, 1000, 2, 3);
