clearTimeout method

MethodclearTimeoutcancels the one-time execution of the callback set using the methodsetTimeout. In the method parameter we indicate the name of the callback.

Syntax

clearTimeout(название коллбэка);

Example

Let's use the methodclearTimeoutLet's stop the callback execution:

function func() { console.log('text'); } let timeOut = setTimeout(func, 1000); clearTimeout(timeOut);
