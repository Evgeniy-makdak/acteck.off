unescape method

Methodunescapereturns a string from an encoded URL request that contains the characters'%'. In general, encoding is necessary to correctly transfer the characters of a string into a format that can be transmitted over the Internet. Also some unsafe ASCII characters are replaced with a safe sequence with'%'and two hexadecimal digits.

Syntax

path.unescape(URL);

Example

Let's get a string from the URL:

import querystring from 'querystring'; console.log(querystring.unescape('abc%20def'));

Result of the executed code:

'abc def'
