stringify method

Methodstringify(synonym -encode) returns the URL query string from the object. In the first parameter of the method we indicate the object that will be converted to a URL, in the second optional parameter - a string that determines the value of the keys (by default'='). In the third optional parameter of the method, we specify a substring to delimit key-value pairs (by default its value'&'). In the fourth optional parameter, you can specify an object with options.

Syntax

path.stringify(объект, разделитель ключей, разделитель значений, [опции]);

Object with options

Option Description
stringifyURIComponent Specifies the decoding of percent-encoded characters in the query string. Default valuefalse.

Example

Let's convert the object to a URL:

import querystring from 'querystring'; console.log(querystring.stringify( { 1: 'a', 2: 'b', 3: 'c' } ));

Result of the executed code:

'1=a&2=b&3=c'
