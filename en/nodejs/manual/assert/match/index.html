match method

Methodmatchchecks for a match between a string and a regular expression. The first parameter of the method specifies a string, the second parameter specifies a regular expression, and the third optional parameter specifies an error message. Unlike the methoddoesNotMatchIt is initially assumed that the string and the regular expression match each other, otherwise an error is returned.

Syntax

assert.match(строка, регулярное выражение, [сообщение об ошибке]);

Example

Let's compare the string'abcde'and regular expression/ab/:

import assert from 'assert'; console.log(assert.match('abcde', /ab/));

Result of the executed code:

'undefined'

Example

Now let's compare the line'abcde'and regular expression/fg/:

import assert from 'assert'; console.log(assert.match('abcde', /fg/));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: The input did not match the regular expression /fg/. Input: 'abcde''
