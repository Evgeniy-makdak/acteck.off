doesNotReject method

MethoddoesNotRejectchecks whether the given promise has been rejected. In the first parameter of the method we indicate the asynchronous function or promise that needs to be checked. If we set a promise, then the method waits for its execution; if we specify a function, then it is called immediately, and then we wait for the promise to be returned. In the second optional parameter we indicate the error, which can be a regular expression or a function. In the third optional parameter we set the error message.

Syntax

assert.doesNotReject(асинхронная функция или промис, [ошибка], [сообщение об ошибке]);

Example

Let's check if our promise is rejected:

import assert from 'assert'; await assert.doesNotReject( async () => { await new Promise(resolve => setTimeout(resolve, 1000)); console.log('message'); }, SyntaxError );

Result of the executed code:

'message'
