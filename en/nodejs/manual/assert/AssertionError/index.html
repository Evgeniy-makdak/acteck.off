AssertionError method

MethodAssertionErrorreturns an object that describes the error of the statement we are interested in. In the method parameter we specify an object with options.

Syntax

let message = new assert.AssertionError(опции);

Object with options

Option Description
message Specifies the error message.
actual The actual property of the error that occurred.
expected The expected property of the error that occurred.
operator A property of the operator in the error that occurred.
stackStartFn A function that skips frames before itself when generating a stack trace.

Example

Let's find out if there is an error in strict comparison1And2. For brevity, we will display the error message in line form:

import assert from 'assert'; let { message } = new assert.AssertionError({ actual: 1, expected: 2, operator: 'strictEqual', }); console.log(message);

Result of the executed code:

'Expected values to be strictly equal: 1 !== 2'
