ifError method

MethodifErrorreturns an error for the value specified in the parameter. If the value isundefinedornull, then no error is issued. The method is useful when testing the error argument in callbacks.

Syntax

assert.ifError(значение);

Example

Let's check what error the method will returnifError, if you specify a number in its parameter0:

import assert from 'assert'; console.log(assert.ifError(0));

Result of the executed code:

'AssertionError [ERR_ASSERTION]: ifError got unwanted exception: 0'
