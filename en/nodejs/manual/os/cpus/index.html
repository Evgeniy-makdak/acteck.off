cpus method

Methodcpusreturns an array of objects containing information about each of the processor's logical cores. The returned array of objects includes the following keys:модель, скорость, times.

Syntax

os.cpus();

Example

Let's find out information about the logical cores of the processor we are using:

import os from 'os'; console.log(os.cpus());
