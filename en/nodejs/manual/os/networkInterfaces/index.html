networkInterfaces method

MethodnetworkInterfacesreturns information about the computer's network interfaces as an object. The returned object contains the following keys:address, netmask, family, mac, internal, scopeid, cidr.

Syntax

os.networkInterfaces();

Example

Let's find out information about the network interfaces of our computer:

import os from 'os'; console.log(os.networkInterfaces());
