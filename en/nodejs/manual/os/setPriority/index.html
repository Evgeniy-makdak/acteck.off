setPriority method

MethodsetPrioritysets the scheduling priority of the process specifiedpidand planning priority. In the first optional parameter of the method, you can setpid, which indicates the id of the process whose scheduling priority we are interested in. Its default value is0. In the second parameter we set the priority for the process, the value of which varies from the highest indicator-20to the lowest -19.

Syntax

os.setPriority([pid], приоритет процесса);

Example

Let's set the highest scheduling priority for our process and use the methodgetPriorityLet's check its value:

import os from 'os'; os.setPriority(-20); console.log(os.getPriority());
