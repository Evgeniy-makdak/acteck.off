Свойство EOL

Свойство EOL показывает символ конца строки или маркера, характерного для данной операционной системы. Для Windows свойство os.EOL возвращает '\r\n', для Linux - '\n', для Mac - '\n\r'.

Синтаксис

os.EOL;

Пример

Давайте узнаем символ конца строки используемой нами операционной системы:

import os from 'os'; console.log(os.EOL);
