availableParallelism function

FunctionavailableParallelismreturns the program's default amount of parallelism. The return value is always a positive number.

Syntax

os.availableParallelism();

Example

Let's find out the default amount of parallelism of the program:

import os from 'os'; console.log(os.availableParallelism());
