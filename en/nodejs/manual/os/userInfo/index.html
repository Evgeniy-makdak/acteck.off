userInfo method

MethoduserInfoReturns information about the current active user as an object. For Windows, the returned object contains the following keysuid, gid, username, homedir, shell. In an optional method parameter, you can specify an object with options.

Syntax

os.userInfo([опции]);

Object with options

Option Description
encoding Data encoding mode. Default value'utf8'.

Example

Let's find out information about the current active user:

import os from 'os'; console.log(os.userInfo());
