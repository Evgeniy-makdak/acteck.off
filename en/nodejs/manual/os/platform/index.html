platform method

Methodplatformreturns information about the operating system platform for which the NodeJS file was compiled. After applying the method, a string is returned that can take the following values:'aix', 'darwin', 'freebsd', 'linux', 'openbsd', 'sunos', 'win32'.

Syntax

os.platform();

Example

Let's find out what platform our operating system has:

import os from 'os'; console.log(os.platform());

See also

  • methodrelease,
    which returns information about the operating system version
  • methodtype,
    which returns the operating system name
