warn method

Methodwarnin the moduleconsolereturns warning or error messages to the console. You can specify any values ​​in the method parameter. The method works similar to the methoderror.

Syntax

console.warn(значение);

Example

Let's find out by printing an error message to the console:

let num = 1; if (num > 1) { console.log('number is correct'); } else { console.warn('error'); }

Result of executing the code:

'error'

  • methodinfo,
    which outputs messages to the console
