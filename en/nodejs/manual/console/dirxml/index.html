dirxml method

Methoddirxmlin the moduleconsolereturns the hierarchical structure of the XML/HTML element specified in the method parameter. The resulting data displays the contents and child nodes of the specified element.

Syntax

console.dirxml(элемент XML/HTML);

Example

Let's create a paragraph with child elements called paragraphs:

<div> <p>1</p> <p>2</p> <p>3</p> </div>

Now let’s display the element structure in the console of our browserdiv:

console.dirxml(document.querySelector('div'));

Result of the executed code:

<div> <p>1</p> <p>2</p> <p>3</p> </div>
