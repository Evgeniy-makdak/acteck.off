info method

Methodinfoin the moduleconsoleoutputs messages to the console. You can specify any type of value in a method parameter. The method works similar to the methodlog.

Syntax

console.info(значения);

Example

Let's print the line to the console'abcde':

console.info('abcde');

Result of the executed code:

'abcde';
