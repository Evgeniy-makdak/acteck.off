error method

Methoderrorin the moduleconsoleerror message in the console. You can specify any values ​​in the method parameter.

Syntax

console.error(значение);

Example

Let's find out by printing an error message to the console:

let num = 1; if (num > 1) { console.log('number is correct'); } else { console.error('error'); }

Result of executing the code:

'error'
