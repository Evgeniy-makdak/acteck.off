timeLog method

MethodtimeLogin the moduleconsolereturns the execution time of a code fragment or function. To start the timer, use the methodtime. In the optional parameter, you can specify a label with the name of the code or function that interests us. The return value is measured in milliseconds.

Syntax

console.timeLog([метка]);

Example

Let's find out how long it will take to execute the functiongetSum, but first we need to start the timer using the methodtime:

function getSum(num1, num2){ return num1 + num2; }; console.time(); getSum(1, 2); console.timeLog();

Result of the executed code:

default: 0.101ms

Example

Now let's in the methods parametertimeAndtimeLogLet's indicate the name of the function whose execution time we want to know:

function getSum(num1, num2){ return num1 + num2; }; function getMultiply(num1, num2){ return num1 * num2; }; console.time(getMultiply); getSum(1, 2); console.timeLog(getMultiply);

Result of the executed code:

function getMultiply(num1, num2){ return num1 * num2; }: 0.074ms
