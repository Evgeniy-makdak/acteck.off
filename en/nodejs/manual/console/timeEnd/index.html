timeEnd method

MethodtimeEndin the moduleconsolestops the timer to count the time spent executing a piece of code or function. In the optional parameter, you can specify a label with the name of the code or function that interests us. To start the timer, use the methodtime. The return value is measured in milliseconds.

Syntax

console.timeEnd([метка]);

Example

Let's find out how long it will take to execute the functiongetSum:

function getSum(num1, num2){ return num1 + num2; }; console.time(); getSum(1, 2); console.timeEndEnd();

Result of the executed code:

default: 0.075ms

Example

Now let's in the methods parametertimeAndtimeEndLet's indicate the name of the function whose execution time we want to know:

function getSum(num1, num2){ return num1 + num2; }; function getMultiply(num1, num2){ return num1 * num2; }; console.time(getMultiply); getSum(1, 2); console.timeEnd(getMultiply);

Result of the executed code:

function getMultiply(num1, num2){ return num1 * num2; }: 0.076ms
