getMaxListeners method

MethodgetMaxListenersreturns the current maximum number of listeners for an event. In the method parameter we specify the class objectEventEmitterorEventTarget.

Syntax

getMaxListeners(объект класса EventEmitter или EventTarget);

Example

Let's find out the maximum listener value:

import { getMaxListeners, EventEmitter } from 'events'; let eventEmitter = new EventEmitter(); let listener = () => { console.log('text1'); }; eventEmitter.on('event', listener); console.log(getMaxListeners(eventEmitter, 'event'));

Result of the executed code:

10
