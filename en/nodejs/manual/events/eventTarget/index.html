EventTarget constructor

ConstructorEventTargetcreates events and listeners for them. Nothing is specified in the method parameter.

Syntax

let eventTarget = new EventTarget();

Example

Let's create a new event for the listener:

let eventTarget = new EventTarget(); console.log(eventTarget);

In the resulting code we will see an empty objectEventTarget:

EventTarget {}
