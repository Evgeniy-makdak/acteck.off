Cancellable property

Propertycancelablechecks whether it is possible to cancel this event. If the event can be canceled, the property returns a boolean valuetrue, otherwise -false.

Syntax

event.cancelable;

Example

Let's find out whether it is possible to cancel a click on a button:

<button>Click</button> let button = document.querySelector('button'); button.addEventListener('click', function(event) { console.log(event.cancelable); });

Result of the executed code:

true

  • propertydefaultPrevented,
    which checks whether the event call has been canceled
