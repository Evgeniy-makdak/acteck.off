setMaxListeners method

MethodsetMaxListenerschanges the maximum number of event listeners. The first parameter of the method specifies a positive number, according to which the maximum number of listeners will change. The second optional parameter can be set to a class objectEventEmitteroreventTarget.

Syntax

eventEmitter.setMaxListeners(положительное число, [EventEmitter или eventTarget]);

Example

Let's change the maximum listeners value to number12:

import { setMaxListeners, getMaxListeners, EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); setMaxListeners(12); console.log(getMaxListeners(eventEmitter));

Result of the executed code:

12

See also

  • propertydefaultMaxListeners,
    which returns the default maximum number of listeners
  • methodgetMaxListeners,
    which returns the maximum number of listeners
