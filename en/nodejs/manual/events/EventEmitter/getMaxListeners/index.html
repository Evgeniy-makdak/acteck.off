getMaxListeners method

MethodgetMaxListenersmoduleEventEmitterreturns the current maximum number of listeners, which is set using the methodsetMaxListenersor default methoddefaultMaxListeners.

Syntax

eventEmitter.getMaxListeners();

Example

Let's find out the maximum number of listeners:

import { EventEmitter } from 'node:events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener1() { console.log('text1'); }); eventEmitter.on('event', function listener2() { console.log('text2'); }); console.log(eventEmitter.getMaxListeners());

Result of the executed code:

10

    which returns the number of event listeners
  • methodsetMaxListeners,
