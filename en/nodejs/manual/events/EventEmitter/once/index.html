Once method

MethodoncemoduleEventEmitteradds listeners once for the event specified in the first parameter of the method. In the second parameter of the method we set the listener, which is the callback. After listening, the listener is automatically removed.

Syntax

eventEmitter.once(название события, слушатель);

Example

Let's create a listener:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.once('event', function listener() { console.log('text'); }); console.log(eventEmitter.listenerCount('event'));
