Pulling data from external repositories into Git

It’s a little inconvenient that the data from the external repository is not immediately merged with the desired branch and for this you need to do some more movements. Luckily, with Git, you can pull changes from an external repository by immediately merging them. This is done using the commandpull, followed by the name of the external repository and the name of the branch:

git pull origin master

Pull changes to your local repository with simultaneous merging.
