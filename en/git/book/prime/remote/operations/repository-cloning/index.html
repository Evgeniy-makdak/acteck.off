Cloning an external Git repository

Projects hosted on GitHub can be cloned. This means that you can steal the entire code of this project for yourself. This can be your project or someone else's. After cloning, you will be able to make changes to this project on your local computer.

To clone, you must have a link to the project. May you have this link. In this case, you can clone the project for yourself using the commandclone, followed by a link to the project:

git clone https://github.com/some-project

At the beginning of the cloning process, an empty directory is created with the name of your project, then the repository is initialized in it.git. All files from the source project are copied into it.

If we want the name of our clone to be different from the name of the original folder, then at the end of the command we add a new name:

git clone https://github.com/some-project my-project

Clone your test project hosted on GitHub into a new folder.
