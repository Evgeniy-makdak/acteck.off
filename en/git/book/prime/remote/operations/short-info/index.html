Information о коротких командах в Git

Как вы уже знаете, команды push и pull имеют короткие формы. При необходимости можно посмотерть локальные ветки и соответствующие им внешние. Это делается с помощью известной вам команды:

git remote show origin

В результате мы увидим следующую информацию:

Local branch configured for 'git pull': master merges with remote master Local ref configured for 'git push': master pushes to master

Посмотрите информацию о коротких командах push и pull.
