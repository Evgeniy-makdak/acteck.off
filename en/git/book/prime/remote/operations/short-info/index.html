Information about short commands in Git

As you already know, the teamspushAndpullhave short forms. If necessary, you can look at local branches and their corresponding external branches. This is done using the command you know:

git remote show origin

As a result, we will see the following information:

Local branch configured for 'git pull': master merges with remote master Local ref configured for 'git push': master pushes to master
