Deleting an external repository in Git

You can delete the external repository. More precisely, we can remove the connection between the repository name and its link. The repository itself will still remain on the Internet.

To remove the external repository name, use the commandremote rm, in which the name of the external repository is indicated separated by a space:

git remote rm origin

Let's check if we have a remote repository:

git remote

After executing the command, we will only see an empty line, which means we do not have an external repository, including the monitored branches and all the settings that we made for it.

Remove your external repository name.
