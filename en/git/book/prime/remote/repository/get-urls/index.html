External repository addresses in Git

You can view the read and write addresses of a given external repository. To do this you need to run the commandremotewith a flag-v:

git remote -v

In the message we will see the name that we have given to the external repository and the address for reading the project -fetchand records -push:

origin https://github.com/user/my-project (fetch) origin https://github.com/user/my-project (push)

Review the URL information for your external repository.
