Creating an external repository on GitHub

In the previous lesson, we registered on the GitHub website. Now we need to create our own external repository for tests on it. To do this, go torepository creation page.

When creating a repository, we must specify its name. Let's call it for testing purposesproject. You can also select the repository type:publicorprivate. The public one will be visible to everyone, and the private one will only be visible to you.

Once you have selected the options you want, click on the create button. After that, copy yourself the link to the created repository.

For further presentation I will use my test link. Let my GitHub login beuser, and I called the projectproject. In this case, the link to my repository will look like this:https://github.com/user/project.git.

Of course, your link will differ in the user name and, possibly, the project name. But further for examples I will use this link.
