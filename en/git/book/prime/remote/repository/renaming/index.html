Renaming an external repository in Git

To rename the external repository we use the commandremote rename, after which we indicate the original and new names separated by a space. For example, let's rename the external repository fromoldVnew:

git remote rename old new

Now let's use the commandremoteLet's check whether the new name of the external repository has taken effect:

git remote
