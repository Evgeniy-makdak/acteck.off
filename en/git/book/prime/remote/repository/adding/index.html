Adding external repositories to Git

Now we need to add an external repository to our project. The essence of the addition comes down to the fact that we must come up with a name for the external repository and bind this name to the link to the repository.

Git has a command for this purpose:remote add, after which the repository name and its link are listed, separated by a space. We come up with the name of the repository ourselves. This will be the name by which the repository will be called on our computer.

Let us choose a common name for our external repository:origin. Let's add a repository under this name:

git remote add origin https://github.com/user/project.git

As a result of adding you will have a nameorigin, with which you will access this external repository. However, the repository data will not be downloaded yet. There are other commands for this, which we will study in the following lessons.
