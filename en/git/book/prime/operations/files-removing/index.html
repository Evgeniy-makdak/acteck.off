Удаление файлов в Git

При работе с Git нельзя вручную удалять файлы из папки с проектом. Если мы попытаемся сделать это, то после проверки командой status, он будет отображаться в категории 'Changes not staged for commit' как измененный, но не проиндексированный файл. Чтобы наверняка удалить ненужный нам файл следует использовать команду git rm:

git rm text.txt

Если после выполнения данной команды проверить состояние файлов в репозитории, то будет выведено следующее сообщение:

deleted: text.txt

Корректно удалите файл из папки с вашим проектом.
