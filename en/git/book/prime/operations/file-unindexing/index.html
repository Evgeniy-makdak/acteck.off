Unindexing a file in Git

To unindex a file in Git, use the commandrestore, with flag--staged, followed by the file name. Let's unindex the file:

git restore --staged file.txt

After checking the status, a message will appear in the terminal stating that our file has been modified, but is not indexed:

Changes not staged for commit: modified: file.txt

Index the file in your Git repository. Then unindex it.
