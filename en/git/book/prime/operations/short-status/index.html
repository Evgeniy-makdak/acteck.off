Short form of status in Git

File status messages may seem a bit long-winded. Therefore, it is possible to display the status in a short form. In this form, the status of files will be indicated by the corresponding letter before the file name:

Status Description
no letter unmodified
M modified
T file type changed
A added
D deleted
R renamed
C copied
U updated but unmerged

In order to display the status in a short form, you need to use the following command:

git status -short

Or its short form:

git status -s
