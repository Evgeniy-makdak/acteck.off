Renaming files in Git

To rename files in Git, use the commandmv. We indicate in it the old file name and the new name in the following order:

git mv old.txt new.txt

After checking the status of the files, a message about the file renaming will be displayed in the terminal:

Changes to be committed: renamed: old.txt -> new.txt
