Отмена изменений в файле в Git

В случае, когда нам нужно отменить изменения в файле, и вернуть его в исходный вид, нужно использовать команду restore. Давайте сначала внесем изменения в файл file.txt и отследим его статус:

git status

В сообщении мы увидим, что файл file.txt получил статус modified.

Теперь вернем файл в исходное состояние командой restore:

git restore file.txt

После проверки статуса мы увидим сообщение о том, что коммитить нечего. Если вы проверите содержимое файла, то оно действительно вернется к версии до изменений. Необходимо быть острожными с этой командой, потому что Git заменяет файл его предыдущей версией и все промежуточные изменения стираются.

Внесите изменения в файл в вашем Git репозитории, а затем верните его в исходное состояние.
