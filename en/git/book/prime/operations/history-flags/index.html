Git commit history flags

In order to display the commit history in an abbreviated form, only with the parameters we need, to the commandlogYou can add special flags:

Flag Description
-p Shows the difference in the changes of each commit.
--stat Displays statistics for a commit.
--shortstat Shows the number of changes/insertions/deletions when a command is applied--stat.
--name-only Displays commit information and names of modified files.
--name-status Displays a message about the list of files that have been changed, incl. added and removed.
--relative-date Displays the date changes were applied in a relative format (for example,'1 day ago').
--pretty Changes the commit history output format:=onelinein one line=full- complete and=short- brief conclusion.
--graph Shows the branching of Git branches and their merge history.

Print information about your commits in one line.
