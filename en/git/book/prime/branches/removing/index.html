Removing branches in Git

As mentioned in previous tutorials, Git does not delete branches when merging. If you no longer need a branch, you need to delete it with a special command. To do this you need to use the commandbranchwith a flag-D, after which the name of the branch to be deleted is indicated, separated by a space.

Let's delete our branch as an exampletest:

git branch -D test

After executing the command we will see the following message:

Deleted branch test (was c440a0b).

Delete one of the side branches in your Git repository.
