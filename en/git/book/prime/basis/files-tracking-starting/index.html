Start tracking files in Git

Let's now learn how to enable file tracking in Git. There is a special command for thisadd, after which the file name is indicated, separated by a space:

git add index.html

After executing this command on a specific file, that file will become monitored. Let's check this using the commandstatus:

git status

As a result of executing this command, our file will appear in a different section:

Changes to be committed: new file: index.html

Make every file in your project trackable.
