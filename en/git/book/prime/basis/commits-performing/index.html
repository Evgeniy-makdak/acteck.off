Making commits in Git

Once we have indexed the files, we can commit. A commit saves the current state of the project. At the same time, we must add a comment to each commit about the essence of the changes made.

Commits are made using the commandcommit. Moreover, after the flag-min quotes we must write a comment to the commit. So let's commit our commit:

git commit -m "мой первый коммит"

If desired, if the comment is large, you can type it in a text editor. For this purpose the flag-mnot written and just executed commandcommit. After executing this command, the text editor you linked will open and you can write a comment to the commit in it:

git commit
