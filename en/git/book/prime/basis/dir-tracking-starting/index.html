Start tracking a folder in Git

Adding many files manually is not very convenient. Therefore, you can make all files in a given folder trackable. Let's see in practice. Let our project have the following folderscript:

  • index.html
  • styles.css
  • script/
    • index.js

Let's track its current contents:

git add script

In this case, only those files that were in this folder at the time the command was executed will become monitored, but not new files.

The command itself can be executed many times. In this case, new files will become tracked, but nothing will happen to those that Git is already tracking.

Let your project have a folder with files. Make all files in this folder trackable.

Add a new file to this folder. Check that the new file will be untraceable.
