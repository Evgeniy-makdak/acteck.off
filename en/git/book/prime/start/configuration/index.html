Setting up Git

Once Git is installed, you need to configure it for a specific user. An important feature of Git is that the original settings are retained even after updating its version. At the same time, if necessary, you can change them by running the same commands again.

All settings that you set are saved in a special utilitygitconfig. This utility also determines the features of operations in Git and, in general, the system interface.

Let's find out what Git settings are in our system, as well as in what files they are located. To do this, you need to launch a terminal as an administrator and enter the following command into it:

git config --list --show-origin
