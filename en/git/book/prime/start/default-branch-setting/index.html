Настройка имени главной ветки в Git

В самом начале работы с проектом система Git создает для вас главную ветку с именем master, в которой будет хранится основной репозиторий с проектом.

Начиная с версии 2.28, есть возможность назначить любое имя для главной ветки. Давайте посмотрим, как это делается. Сменим, например, имя главной ветки на main. Для этого нужно ввести следующую команду:

git config --global init.defaultBranch main

Переименуйте вашу основную ветку на main.
