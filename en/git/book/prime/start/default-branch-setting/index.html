Setting the master branch name in Git

When you first start working on a project, Git creates a master branch for you calledmaster, which will store the main repository with the project.

Since version2.28, it is possible to assign any name to the master branch. Let's see how it's done. Let's change, for example, the name of the main branch tomain. To do this, enter the following command:

git config --global init.defaultBranch main

Rename your master branch tomain.
