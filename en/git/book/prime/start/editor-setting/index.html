Selecting a text editor when setting up Git

Once you've set up a username, you need to select a text editor to use for posting to Git.

Let's set up the Notepad++ text editor for Windows as an example. To do this, in the terminal after the commandcore.editoryou need to open double quotes, then specify in single quotes the full path to the file containing your chosen text editor. Adding options-multiInst, --notabbar, -nosessionAnd-noPluginand close the double quotes:

git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin"

A very important nuance is that the selected editor must already be installed on your operating system. If you specify a non-existent editor, then unexpected failures in Git may occur in the future. Therefore, first install a text editor and after that specify it in the settings.
