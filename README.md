## 📚 О проекте

**acteck.off** — это офлайн-образовательная платформа для изучения веб-разработки. Проект представляет собой статический сайт с коллекцией учебных материалов, охватывающих все ключевые области современной frontend и backend разработки.

Материалы структурированы по разделам, каждый из которых содержит учебники (book), справочники (manual), FAQ и практические задания (tasker). Проект доступен на **русском** и **английском** языках.

## 🚀 Как использовать

Проект полностью статический — не требует сервера или сборки. Достаточно открыть любой HTML-файл в браузере:

```bash
# Открыть русскую версию
open ru/index.html

# Открыть английскую версию
open en/index.html
```

### Навигация

- **Главная страница** — карточки разделов с кратким описанием
- **Страница раздела** — список учебников, справочников и материалов
- **Страница урока** — учебный материал с хлебными крошками и навигацией
- **Переключатель языка** — RU/EN в шапке сайта

## 🔄 Конвертация контента

Скрипт `convert.py` предназначен для конвертации учебных материалов из формата [code.mu.off](https://github.com/Evgeniy-makdak/code.mu.off.git) в формат acteck.off. Он:

- Извлекает заголовки и содержимое уроков
- Оборачивает их в единый шаблон acteck.off
- Автоматически исправляет внутренние ссылки
- Создаёт структуру `file/index.html` для корректной работы с протоколом `file://`
