Порядковый номер клика на экран средствами JavaScript

В уроке показывается, как вывести в месте клика по экрану порядковый номер этого клика. Сама по себе задача не очень практическая, но демонстрирует некоторые полезные фишки.

Исходники кода

var i = 0; window.addEventListener('click', function(event) { i++; var elem = document.createElement('div'); elem.innerHTML = i; document.body.appendChild(elem); elem.style.position = 'absolute'; elem.style.left = event.clientX + 'px'; elem.style.top = event.clientY + 'px'; });
