Последний день месяца и високосный год на JavaScript

Как вы знаете, в каждом месяце разное количество дней - в каком-то месяце их 31, в каком-то 30, а в феврале вообще - то 28, то 29. В данном видео я расскажу вам простой способ нахождения последнего дня месяца на JavaScript. И как бонус - мы распространим этот способ на определение того, является ли год високосным или нет.

Исходники кода

var month = 1; var year = 2100; var date = new Date(year, month + 1, 0); console.log(date.getDate());
var year = 2100; var date = new Date(year, 2, 0); if (date.getDate() == 29) { console.log(true); } else { console.log(false); }
