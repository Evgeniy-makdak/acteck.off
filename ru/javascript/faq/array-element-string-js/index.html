Элемент массива в строку JS

В данном уроке мы разберем различные варианты как преобразовать элементы массива в строку на JavaScript. Смотрите варианты, которые могут быть.

Вариант 1

Можно слить элементы массива в строку с помощью метода join:

let arr = ['a', 'b', 'c', 'd', 'e']; let res = arr.join(''); console.log(res); // 'abcde'

Вариант 2

Бывют ситуации, когда у нас есть числовой массив, а мы хотели бы его преобразовать в строку. Давайте посмотрим на примере. Пусть у нас есть следующий массив с числами:

let arr = [1, 2, 3];

Преобразуем его элементы к строкам с помощью специального хитрого приема с методом map:

let res = arr.map(String);

В результате получим массив строк:

console.log(res); // ['1', '2', '3']
