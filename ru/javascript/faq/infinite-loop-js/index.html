Бесконечный цикл JS

Если в теле цикла не будет выполнено условие для выхода из него, цикл может выполняться бесконечно. Посмотрите пример:

let bool = true; while (bool) { console.log('!'); }

А в следующем примере в цикле забыто увеличение счетчика. Такой цикл также будет выполняться вечно:

let i = 1; while (i < 100) { console.log('!'); }
