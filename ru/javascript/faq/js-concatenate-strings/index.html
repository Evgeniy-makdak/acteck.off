JS объединить строки

Объединение строк в JavaScript можно осуществить несколькими способами. Выбор способа зависит от конкретной ситуации и предпочтений разработчика.

Способ 1

Используем оператор '+':

let str1 = 'aaa'; let str2 = 'bbb'; let res = str1 + str2; console.log(res); // 'aaabbb'

Способ 2

Объединим строки, разделив их пробелом:

let str1 = 'aaa'; let str2 = 'bbb'; let res = str1 + ' ' + str2; console.log(res); // 'aaa bbb'

Способ 3

Можно использовать шаблонные строки:

let str1 = 'aaa'; let str2 = 'bbb'; let res = `${str1} ${str1}`; console.log(res); // 'aaa bbb'

Способ 4

Использование метода concat:

let str1 = 'aaa'; let str2 = 'bbb'; let res = ''.concat(str1, str2); console.log(res); // 'aaabbb'

Способ 5

С помощью метода join можно объединить строки из массива с заданным разделителем:

let arr = ['aaa', 'bbb']; let res = arr.join(' '); console.log(res); // 'aaa bbb'
