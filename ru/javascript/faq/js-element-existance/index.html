JS есть ли элемент

Проверка элемента массива

Можно проверить, есть ли элемент с заданным значением в массиве. Это делается с помощью метода includes. Давайте попробуем:

let arr = [1, 2, 3, 4, 5]; let isElem = arr.includes(3); console.log(isElem); // выведет true

Проверка DOM элемента

Можно проверить наличие DOM элемента с помощью метода querySelector. Если элемент найден, то метод вернет его. Если элемента не существует, то метод вернет значение null. Вот как это можно использовать:

let elem = document.querySelector('селектор элемента'); if (elem) { // используем элемент } else { console.error('элемент не найден'); }
