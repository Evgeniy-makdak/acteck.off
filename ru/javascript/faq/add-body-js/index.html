Добавить в body JS

Давайте добавим в body новый тег. Для начала создадим новый тег:

let tag = document.createElement('div');

Запишем ему текст:

tag.textContent = 'text';

Добавим теперь наш новый тег в конец body с помощью метода append

document.body.append(tag);
