JS как сделать ссылку

В данном уроке мы разберем, как средствами JavaScript динамически сделать HTML ссылку и добавить ее в элемент.

Пусть у нас есть следующий блок, в который мы будем добавлять ссылку:

<div id="elem"></div>

Получим этот блок в переменную с помощью метода querySelector:

let elem = document.querySelector('#elem');

Создадим новую HTML ссылку с помощью метода createElement:

let link = document.createElement('a');

Добавим текст ссылке с помощью свойства textContent:

link.textContent = 'text'

Добавим нашу созданную ссылку в конец блока с помощью метода append:

elem.append(link);
