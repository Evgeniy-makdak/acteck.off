JS превратить строку в число

В JavaScript существует несколько способов преобразования строки в число. Давайте их разберем.

Способ 1

Использование функции Number для преобразования любой величины в число:

let num = Number('10'); console.log(num); // 10

Способ 2

Использование оператора + для преобразования любой величины в число:

let num = +'10'; console.log(num); // 10

Способ 3

Использование функции parseInt для преобразования строки в целое число;

let num = parseInt('10'); console.log(num); // 10

Способ 4

Использование функции parseFloat для преобразования строки в число с плавающей точкой:

let num = parseFloat('10.5'); console.log(num); // 10.5
