Как получить элементы массива в JS

С помощью цикла for-of

Чтобы получить и вывести все элементы из массива можно воспользоваться циклом for-of. Посмотрим пример:

let arr = ['a', 'b', 'c', 'd']; for (let elem of arr) { console.log(elem); }

Результат выполненного кода:

'a' 'b' 'c' 'd'

С помощью цикла for

Также для получения всех элементов из массива можно применить цикл for, в котором мы зададим условие перебирать и выводить все элементы столько раз, сколько составляет длина массива. Смотрите пример:

let arr = ['a', 'b', 'c', 'd']; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

Результат выполненного кода:

'a' 'b' 'c' 'd'
