JS цикл с конца

Пример 1

Запустим обычный цикл for с конца:

for (let i = 100; i >= 0; i--) { console.log(i); }

Пример 2

Запустим цикл for по массиву с конца:

let arr = [1, 2, 3, 4, 5]; for (let i = arr.length - 1; i >= 0; i--) { console.log(arr[i]); } console.log(arr);

Пример 3

Используем метод reverse для переворачивания массива, а затем переберем его через цикл for-of:

let arr = [1, 2, 3, 4, 5]; arr.reverse(); for (let elem of arr) { console.log(elem); }
