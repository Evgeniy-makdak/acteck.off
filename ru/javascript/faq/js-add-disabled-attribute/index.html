JS добавить атрибут disabled

Атрибут disabled может использоваться для блокировки элементов формы, например, кнопок или инпутов. В этой статье вы узнаете, как добавить атрибут disabled в элемент с помощью JavaScript.

Пусть у нас есть следующая кнопка:

<button id="btn">btn</button>

Получим ссылку на нашу кнопку с помощью метода querySelector:

let btn = document.querySelector('#btn');

Добавим атрибут disabled к кнопке с помощью следующего кода:

btn.disabled = true;

А теперь отменим атрибут disabled кнопке с помощью следующего кода:

btn.disabled = false;
