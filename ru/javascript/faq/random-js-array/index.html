Рандомный массив JS

В данном уроке мы создадим массив, заполненный рандомными (случайными) целыми числами в заданном диапазоне на JavaScript.

Для этого нам понадобится специальная функция, возвращающая случайное целое число в заданном диапазоне:

function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; }

Используем эту функцию, чтобы создать рандомный массив:

let arr = []; for (let i = 0; i < 10; i++) { arr.push(Math.getRandomInt(1, 100)); } console.log(arr);
