Пробел js строка

В данном уроке мы рассмотрим различные манипуляции с пробелами в JavaScript.

Удаление пробела

Удалить первый пробел из строки можно с помощью метода replace:

let str = 'aaa bbb ccc'; let res = str.replace(' ', ''); console.log(res); // выведет 'aaabbb ccc'

Удаление пробелов

Удалить все пробелы из строки можно с помощью метода replace и регулярных выражений:

let str = 'aaa bbb ccc'; let res = str.replace(/\s/g, ''); console.log(res); // выведет 'aaabbbccc'

Удаление концевых пробелов

Удалить концевые пробелы можно с помощью метода trim:

let str = ' aaa bbb ccc '; let res = str.trim(); console.log(res); // выведет 'aaa bbb ccc'

Разбиение по пробелу

Можно разбить строку по пробелу и получить массив строк с помощью метода split:

let str = 'aaa bbb ccc'; let res = str.split(' '); console.log(res); // выведет ['aaa', 'bbb', 'ccc']
