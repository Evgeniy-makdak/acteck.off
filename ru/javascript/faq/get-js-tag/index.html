Обратиться к тегу JS

В данном уроке мы разберем, как обратиться к HTML тегу на JavaScript. Это делается с помощью специальных методов. Давайте посмотрим на примере. Пусть у нас есть следующий тег:

<div>text</div>

Получим ссылку на этот тег с помощью метода querySelector:

let elem = document.querySelector('#elem');

Теперь в переменной elem хранится ссылка, с помощью которой мы можем обратиться к тегу. К примеру, давайте прочитаем его текст с помощью свойства textContent:

let elem = document.querySelector('#elem'); let text = elem.textContent; console.log(text);
