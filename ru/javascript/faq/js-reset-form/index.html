JS сбросить форму

Форма может быть сброшена с помощью метода reset. Этот метод устанавливает все значения формы в исходное состояние с помощью JavaScript.

Рассмотрим пример. Пусть у нас есть следующая форма:

<form id="form"> <input type="text" name="user" value="john"> <button type="button" id="btn">сброс</button> </form>

Получим ссылки на форму и на кнопку с помощью метода querySelector:

let form = document.querySelector('#form'); let btn = document.querySelector('#btn');

Привяжем к кнопке обработчик клика:

btn.addEventListener('click', function() { });

Внутри обработчика выполним сброс формы:

btn.addEventListener('click', function() { form.reset(); });
