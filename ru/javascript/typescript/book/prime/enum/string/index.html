Строковые перечисления в TypeScript

Ключами могут быть не только числа, но и строки:

enum Season { Winter = 'season 1', Spring = 'season 2', Summer = 'season 3', Autumn = 'season 4' };

Проверим:

let current: Season = Season.Summer; console.log(current); // 'season 3'
