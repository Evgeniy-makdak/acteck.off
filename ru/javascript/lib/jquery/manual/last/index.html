Метод last

Метод last выбирает последний элемент среди уже найденных.

Синтаксис

.last();

Пример

Давайте найдем все абзацы, а затем выберем среди найденных последний и поставим ему текст '!':

<p>text</p> <p>text</p> <p>text</p> $('p').last().text('!');

HTML код станет выглядеть так:

<p>text</p> <p>text</p> <p>!</p>

  • метод first,
    который получает первый элемент
  • метод eq,
    который получает c элемент с определенным номером
