Селектор header

Селектор :header выбирает заголовки h1, h2, h3, h4, h5, h6. Так как :header не относится к CSS спецификации, то для улучшения производительности в современных браузерах лучше в начале отфильтровать элементы с использованием чистого css-селектора, а затем применить .filter(':header').

Синтаксис

Так мы выбираем элементы:

$(':header');

Пример

Давайте поменяем фон всех заголовков на серый:

<h1>Header h1</h1> <p>text</p> <h2>Header h2</h2> <p>text</p> $(':header').css({background: '#ccc'});

Смотрите также

  • метод filter,
    который фильтрует элементы в наборе по заданному селектору
  • метод css,
    который позволяет получать и изменять CSS стили элемента
