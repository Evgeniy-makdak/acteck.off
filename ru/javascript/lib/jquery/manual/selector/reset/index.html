Селектор reset

Селектор :reset выбирает элементы с типом reset, являющиеся кнопками очистки формы. Эквивалентом $(':reset') является $('[type=reset]'). Так как :reset не относится к CSS спецификации, то для улучшения производительности в современных браузерах вместо него лучше применить [type='reset'].

Синтаксис

Так мы выбираем элементы с типом reset:

$(':reset');

Пример

Давайте, выберем все инпуты с типом reset и сделаем им зеленый фон и красную рамку с помощью метода css:

<form> <input type="button" value="button"> <input type="file"> <input type="password"> <button>button</button> <input type="reset"> <input type="radio" name="test"> <input type="radio" name="test"> <input type="checkbox"> <input type="text"> </form> <+javascript+> $('input:reset').css({background: 'green', border: '2px red solid'}); $('form').submit(function(event) { event.preventDefault(); // предотвращает отправку формы }); <-javascript->
