Селектор empty

Селектор :empty выбирает элементы, у которых нет потомков (то есть пустые элементы, в которых нет ни элемента, ни текста).

Синтаксис

Получение пустых элементов:

$(':empty');

Пример

У нас есть табличка table с зеленым фоном, давайте найдем ячейки td, которые не имеют внутри себя ни текста, ни элемента, и окрасим их в серый цвет:

<table> <tr><td>aaa</td><td></td></tr> <tr><td>bbb</td><td></td></tr> <tr><td>ccc</td><td></td></tr> </table> table { border: 1px; } td { width: 40px; background: green; } $('td:empty').css({background: '#ccc'});

Смотрите также

  • селектор parent,
    который выбирает элементы-родители
  • метод css,
    который позволяет получать и изменять CSS стили элемента
