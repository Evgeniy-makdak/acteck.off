Метод wrapInner

Метод wrapInner оборачивает содержимое внутри элемента в заданный тег.

Синтаксис

Параметром метода может служить текст (возможно с тегами), DOM элемент или объект jQuery:

.wrapInner(text);

Вот так выбранные элементы обертываются содержимым, которое будет возвращено в виде html-текста, пользовательской функцией. Функция вызывается отдельно, для каждого из выбранных элементов:

.wrapInner(function(index));

Пример

Давайте обернем абзацы с классом www в тег b:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrapInner('<b></b>');

HTML код станет выглядеть так:

<p class="www"><b>text</b></p> <p class="www"><b>text</b></p> <p class="www"><b>text</b></p> <p>text</p>
