Метод appendTo

Метод appendTo добавляет текст в конец элемента. Также существует метод append, который работает аналогичным образом.

Синтаксис

Вставка текста в конец элемента:

$(текст).appendTo(селектор);

Содержимое может быть не только обычным текстом, но и DOM элементом, селектором, массивом элементов, строкой или объектом jQuery.

Пример

Давайте вставим текст с тегами в конец заданного абзаца:

<p id="test">абзац</p> $('<b>!!!</b>').appendTo('#test');

Результат выполнения кода:

<p id="test">абзац<b>!!!</b></p>

Смотрите также

  • метод append,
    который добавляет текст в конец элемента
  • псевдоэлемент after,
    который добавляет текст в конец элемента на CSS
  • методы prependTo, insertBefore, insertAfter,
    позволяющие добавлять содержимое в определенное место
