Метод addClass

Метод addClass добавляет заданный CSS класс элементу.

Синтаксис

Один CSS класс:

$(селектор).addClass(имя CSS класса);

Несколько CSS классов (строкой):

$(селектор).addClass('класс1 класс2 класс3 ...');

Несколько CSS классов (массивом):

$(селектор).addClass(['класс1', 'класс2', 'класс3', '...']);

Применение функции к каждому элементу в наборе:

$(селектор).addClass(function(номер в наборе, текущие CSS классы элемента));

Пример

У нашего абзаца совсем нет классов. Давайте добавим ему класс www:

<p id="test">text</p> $('#test').addClass('www');

HTML код станет выглядеть так:

<p id="test" class="www">text</p>

Пример

У нашего абзаца уже есть класс. Давайте добавим ему еще и класс www:

<p id="test" class="eee">text</p> $('#test').addClass('www');

HTML код станет выглядеть так:

<p id="test" class="eee www">text</p>

Пример

У нашего абзаца уже есть два класса. Давайте добавим ему еще и класс www:

<p id="test" class="eee ggg">text</p> $('#test').addClass('www');

HTML код станет выглядеть так:

<p id="test" class="eee ggg www">text</p>

Пример

У нашего абзаца уже есть класс. Давайте добавим ему еще два класса - www и ggg:

<p id="test" class="eee">text</p> $('#test').addClass('www ggg');

HTML код станет выглядеть так:

<p id="test" class="eee www ggg">text</p>

  • JavaScript свойство classlist,
    с помощью которого можно добавлять и удалять CSS классы элемента на чистом JavaScript
