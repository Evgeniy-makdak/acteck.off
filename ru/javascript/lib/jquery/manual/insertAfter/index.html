Метод insertAfter

Метод insertAfter добавляет текст после заданного элемента. Также существует метод after, который работает аналогичным образом.

Синтаксис

Вставка текста после элемента:

$(текст).insertAfter(селектор);

Содержимое может быть не только обычным текстом, но и DOM элементом, селектором, массивом элементов, строкой или объектом jQuery.

Пример

Давайте вставим текст с тегами после заданного абзаца:

<p id="test">абзац</p> $('<p>!!!</p>').insertAfter('#test');

HTML код станет выглядеть так:

<p id="test">абзац</p><p>!!!</p>

Смотрите также

  • метод after,
    который добавляет текст после заданного элемента
  • методы insertBefore, appendTo, prependTo,
    позволяющие добавлять содержимое в определенное место на странице
  • метод clone,
    который создает копии выбранных элементов
