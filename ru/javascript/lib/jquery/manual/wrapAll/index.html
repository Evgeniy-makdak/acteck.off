Метод wrapAll

Метод wrapAll оборачивает все элементы набора в один заданный тег. Если эти элементы находятся в разных местах, то jQuery сначала ставит их рядом, а потом оборачивает в нужный тег.

Синтаксис

Параметром метода может служить текст (возможно с тегами), DOM элемент или объект jQuery:

.wrapAll(text);

Также мы можем воспользоваться и функцией для каждого элемента:

.wrapAll(function(index))

Пример

Давайте обернем абзацы с классом www в тег div:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p> $('.www').wrapAll('<div></div>');

HTML код станет выглядеть так:

<div> <p class="www">text</p> <p class="www">text</p> <p class="www">text</p> </div> <p>text</p>
