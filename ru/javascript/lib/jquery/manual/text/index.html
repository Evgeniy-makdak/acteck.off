Метод text

Метод text позволяет получать текст элементов, включая его потомков и изменять его.

Синтаксис

Получение текста:

$(селектор).text();

Изменение текста:

$(селектор).text(новый текст);

Заменяет содержимое выбранных элементов на возвращенное пользовательской функцией значение. Функция вызывается отдельно, для каждого из выбранных элементов. При вызове ей передаются следующие параметры: index — позиция элемента в наборе, value — текущее text-содержимое:

$(селектор).text(function(номер в наборе, текущее содержимое элемента));

Пример

Давайте выведем на экран содержимое нашего абзаца:

<p id="test">text</p> let text = $('#test').text(); alert(text);
