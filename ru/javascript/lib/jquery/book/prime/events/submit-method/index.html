Метод submit в jQuery

Ранее мы использовали такую конструкцию:

$('form').on('submit', function(event) { event.preventDefault(); });

Вместо этого мы можем также в jQuery напрямую воспользоваться методом submit, который привязывает обработчик формы, либо выполняет событие:

$('form').submit(function(event) { event.preventDefault(); });

В следующем примере у нас есть простая форма, к которой мы привяжем обработчик событий при помощи метода submit, с текстовым полем и кнопкой - инпутом с типом submit:

<p>jQuery</p> <form action="/"> <div> <input type="text"> <input type="submit"> </div> </form> <span></span>

Давайте будем вводит текст в текстовое поле, а при нажатии на кнопку проверять этот текст. Если мы введем 'jQuery', то в спане покажем текст 'Good!', с помощью методов text и show, а если что-то другое, то - 'Bad...':

$('form').submit(function(event) { event.preventDefault(); if ($('input').first().val() === 'jQuery') { $('span').text('Good!').show(); return; } $('span').text('Bad...').show(); });
