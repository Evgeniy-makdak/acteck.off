Работа с очередью в jQuery

Для работы с анимацией в jQuery нам может понадобиться такая полезная вещь, как работа с очередью. Очереди позволяют осуществлять последовательность действий с элементом асинхронно, без прерывания выполнения программы. Можно пользоваться fx - стандартной очередью эффектов, что делается довольно часто, а можно использовать свои - пользовательские очереди.
