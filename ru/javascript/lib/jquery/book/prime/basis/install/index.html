Подключение jQuery

Для начала нам нужно скачать jQuery с официального сайта. Затем положить полученный файл в папку с вашим сайтом и подключить его таким образом:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title></title> <script src="путь к файлу с jQuery"></script> </head> <body> </body> </html>

После подключения можно начинать писать код с использованием этой библиотеки.
