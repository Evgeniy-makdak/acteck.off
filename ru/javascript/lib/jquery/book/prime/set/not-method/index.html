Метод not в jQuery

Метод, который мы рассмотрим в этом уроке будет not. Это метод, который делает отрицание.

Получим все абзацы не с классом www:

$('p').not('.www');

При работе с этим методом мы сначала получаем все абзацы, а потом среди них выберем абзацы не с классом www.

В следующем примере у нас есть абзацы:

<p class="www">text</p> <p class="www">text</p> <p>text</p> <p>text</p>

Давайте найдем все абзацы, поставим им в начало текст '!', затем найдем абзацы не с классом www и поставим им в конец текст '?':

$('p').prepend('!').not('.www').append('?');

HTML код станет выглядеть так:

<p class="www">!text</p> <p class="www">!text</p> <p>!text?</p> <p>!text?</p>

Найдите все li, исключив из набора li с классом test.
