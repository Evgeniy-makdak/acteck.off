Поиск всех родителей в jQuery

Метод parent получает непосредственного родителя элемента. Для того, чтобы получить всех родителей элемента - не только непосредственного, но и прародителя, прапрародителя и так далее до корневого элемента используется метод parents.

В следующем примере мы рассмотрим следующий HTML код:

<div> <div class="www"> <div class="www"> <p id="test">text</p> </div> </div> </div>

Затем мы найдем элемент #test и всех его родителей с помощью parents и, если это родители div с классом www, то поставим им в начало текст '!':

$('#test').parents('div.www').prepend('!');

Найдите ВСЕХ родителей тегов b, и те, которые имеют класс test покрасьте в красный цвет.
