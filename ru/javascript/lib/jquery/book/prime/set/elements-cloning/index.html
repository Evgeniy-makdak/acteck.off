Клонирование элементов в jQuery

Метод clone создает глубокие копии выбранных элементов страницы и возвращает их в виде объекта jQuery.

Разберем как работает этот метод на примере абзаца:

<p class="www">text</p>

После абзаца вставим еще один такой же, используя метод insertAfter:

$('.www').clone().insertAfter('.www');

Клонируйте див с классом www, покрасьте его в красный цвет, поставьте ему текст '???' и вставьте его перед этим дивом.
