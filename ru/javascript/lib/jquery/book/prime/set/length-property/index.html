Свойство length в наборе jQuery

Для того, чтобы узнать количество элементов в наборе - используется свойство length.

Например, у нас есть следующий HTML код:

<p>text</p> <p>text</p> <div>text</div> <p>text</p> <p>text</p> <div>text</div>

Давайте узнаем количество абзацев и выведем его в консоль:

let num = $('p').length; console.log(num);
