Фильтрация элементов в jQuery

С помощью метода filter в наборе можно оставить только те элементы, которые удовлетворяют определенному селектору.

Рассмотрим HTML код:

<p>text</p> <p>text</p> <p class="www">text</p> <p class="www">text</p>

Давайте найдем все абзацы, поставим им в начало текст '!' с помощью prepend, затем получим из найденных только абзацы с классом www с помощью метода filter и поставим им в конец текст '?' с помощью append:

$('p').prepend('!').filter('.www').append('?');

Найдите все li в наборе, затем с помощью filter выберите все li с классом www и поставьте им в конец '!'.
