Изменение нескольких свойств атрибутов в jQuery

Аналогично методу attr, мы можем менять значения сразу нескольких свойств, для этого их нужно передать в виде объекта.

Пусть у нас есть следующий чекбокс:

<input type="checkbox" class="aaa" id="test" />

Давайте поменяем одновременно свойства для className и disabled.

$("#test").prop({ className: 'bbb', disabled: 'true' });

HTML код станет выглядеть так:

<input type="checkbox" class="bbb" id="test" disabled>
