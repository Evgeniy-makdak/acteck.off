Метод remove в jQuery

Для удаления элементов в jQuery существуют два метода: remove и detach.

Метод remove удаляет элементы навсегда.

Пусть у нас есть следующий HTML код:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Давайте удалим p с классом www:

$('p').remove('.www');

HTML код станет выглядеть так:

<p>text</p>

А можно сделать и так:

$('p.www').remove();
