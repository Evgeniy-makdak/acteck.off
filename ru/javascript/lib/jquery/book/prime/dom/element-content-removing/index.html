Удаление содержимого элементов в jQuery

С помощью метода empty можно удалить содержимое элементов (сами элементы никуда не денутся). Например, есть следующие абзацы:

<p>text</p> <p>text</p> <p>text</p>

Давайте удалим их содержимое:

$('p').empty();

HTML код станет выглядеть так:

<p></p> <p></p> <p></p>

Сделайте все h2 пустыми.
