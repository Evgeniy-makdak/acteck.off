Метод append в jQuery

Метод append позволяет добавлять текст в конец элементов.

Ниже приведен абзац:

<p>text</p>

Теперь прописываем следующий Javascript код:

$('p').append('<b>!!!</b>');

HTML код станет выглядеть так:

<p>text<b>!!!</b></p>
