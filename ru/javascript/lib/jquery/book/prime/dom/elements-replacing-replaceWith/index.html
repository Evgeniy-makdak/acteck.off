Замена элементов методом replaceWith в jQuery

Можно заменять одни элементы на другие с помощью метода replaceWith.

Рассмотрим пример, пусть у нас есть следующие абзацы:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Давайте заменим все абзацы с классом www на '<div>!!!</div>':

$('.www').replaceWith('<div>!!!</div>');

HTML код станет выглядеть так:

<div>!!!</div> <div>!!!</div> <div>!!!</div> <p>text</p>
