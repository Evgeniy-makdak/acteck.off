Запись значения в атрибут в jQuery

Можно также добавлять атрибут либо записывать значение в атрибут. Пусть у нас есть инпут:

<input type="text" value="!!!" id="test">

Давайте запишем новое значение 'www' в этот же атрибут:

$('#test').attr('value', 'www');

HTML код станет выглядеть так:

<input type="text" value="www" id="test">

Добавьте всем li класс test.

Замените href каждой ссылки на '!'.

Добавьте каждой ссылке атрибут target='_blank'.

Замените содержимое каждой ссылки (анкоры) на их href.

Вставьте вместо содержимого каждой ссылки содержимое href в круглых скобках.

Вставьте в атрибут title каждой ссылки ее анкор.
