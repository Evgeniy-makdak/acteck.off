Метод prepend для вставки элементов в jQuery

Для вставки элементов в jQuery предусмотрено 4 метода: append, prepend, after, before.

Метод prepend позволяет добавлять текст в начало элементов. К примеру, пусть у нас есть абзац:

<p>text</p>

Применяем метод prepend, внутри тегов указывая нужный нам текст:

$('p').prepend('<b>!!!</b>');

HTML код станет выглядеть так:

<p><b>!!!</b>text</p>

Внутри каждого h2 в начале вставьте '!'.
