Удаление только значения атрибута в jQuery

Бывает, что нам необходимо оставить атрибут, но при этом удалить его текущее значение. Давайте рассмотрим следующий пример. Пусть у нас есть инпут:

<input type="text" value="!!!" id="test">

В этом случае мы прописываем атрибут, но добавляем к нему вторым параметром пустое значение в кавычках (без пробела).

$('#test').attr('value', '');

HTML код станет выглядеть так - атрибут value не пропадет из HTML кода:

<input type="text" value id="test">

Измените решение предыдущей задачи так, чтобы удалилось только значение www, но не сам атрибут.
