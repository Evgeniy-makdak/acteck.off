Оборачивание элементов в jQuery

Теги можно оборачивать в другие теги, к примеру, если у нас есть абзацы, то мы их можем дополнительно обернуть в теги div. Для таких вещей используется метод wrap, который позволяет обернуть каждый найденный элемент в указанный тег.

К примеру, пусть у нас есть абзацы с классом www:

<p class="www">text</p> <p class="www">text</p> <p class="www">text</p> <p>text</p>

Давайте их обернем тегом div:

$('.www').wrap('<div>');

HTML код станет выглядеть так:

<div><p class="www">text</p></div> <div><p class="www">text</p></div> <div><p class="www">text</p></div> <p>text</p>

Оберните каждый h3 в div.
