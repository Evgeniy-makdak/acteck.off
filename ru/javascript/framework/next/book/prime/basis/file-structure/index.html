Файловая структура проекта в фреймворке NextJS

После создания проекта в его папке появится определенная файловая структура. Давайте рассмотрим ее самые важные компоненты:

  • /node_modules/
  • /src/
    • /app/
      • globals.css
      • layout.jsx
      • page.jsx
  • next.config.mjs
  • package.json
  • tailwind.config.js

Файлы, которые мы сделаем в процессе работы с проектом будут хранится в папках src и src/app. На данный момент в папке src находится только одна рабочая папка app. А в ней хранятся файлы для работы с настройками CSS (global.css), файл с основной разметкой страницы (layout.jsx) и файл с основным роутом проекта (page.jsx).

В файле next.config.mjs содержатся основные настройки проекта.

В файле tailwind.config.js содержатся основные настройки для стилей Tailwind, применяемых в данном проекте.
