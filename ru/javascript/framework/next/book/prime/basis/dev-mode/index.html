Режим разработки в NextJS

Режим разработки предназначен для того, чтобы сделать процесс написания кода и его тестирования удобным и гибким. В этот режим включены инструменты, позволяющие находить ошибки, тестировать и быстро вносить необходимые правки в код.

Основные особенности режима разработки:

  • Горячая перезагрузка - при внесении любого изменения в код, сервер автоматически перезагружается и страница с кодом обновляется.
  • Вывод ошибок на странице в браузере для того, чтобы было удобнее их увидеть и устранить. Также в браузере показывается местонахождение файлов с ошибками, что облегчает их поиск.
  • Код не оптимизируется для максимальной производительности, поэтому могут быть дополнительные проверки, предупреждения и даже немного медленнее идет загрузка страниц.
  • Детализированный вывод логов в терминале, чтобы было проще отследить происходящее на сервере.

Режим разработки запускается следующей командой:

npm run dev

После чего в терминале появится адрес, перейдя по которому можно увидеть свой проект.

Для выхода из режим разработчика необходимо нажать комбинацию клавиш CTRL+C.
