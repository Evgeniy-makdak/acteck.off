Диалект JavaScript в фреймворке NextJS

Разработку в NextJS можно вести либо на чистом JavaScript, либо на TypeScript. Диалект языка можно будет выбрать при установке NextJS.

В настоящее время более популярна разработка на TypeScript. Для простоты, однако, мы начнем изучение на чистом JavaScript, а затем в отдельном разделе учебника мы поговорим о том, что нужно будет поменять в нашем коде, чтобы перейти на TypeScript.
