Установка фреймворка NextJS

В данном уроке мы установим NextJS. Для установки данного фреймворка вам понадобится NodeJS версии 22 или выше.

Для запуска установки в терминале нужно выполнить следующую команду:

npx create-next-app@latest

После этого запустится процесс установки и вам будет предложено ответить на ряд вопросов, с помощью которых будет сконфигурирован ваш фреймворк.

Вопрос Ответ
What is your project named? Здесь нам нужно задать имя нашему проекту. Можно задать любое. Напишите my-app.
Would you like to use TypeScript? Здесь нам предлагается использовать TypeScript. Мы не будем этого делать, так как пока для простоты мы будем работать на JavaScript.
Выбирайте No.
Would you like to use ESLint? ESLint - это специальная библиотека, проверяющая соответствие вашего кода стандартам оформления.
Выбирайте Yes.
Would you like to use Tailwind CSS? Tailwind - это специальный CSS фреймворк для React.
Выбирайте Yes.
Would you like to use `src/` directory? Здесь спрашивается, хотим ли мы, чтобы наш проект был в папке src.
Выбирайте Yes.
Would you like to use App Router? Включает современный вариант роутинга.
Выбирайте Yes.
Would you like to customize the default import alias (@/*)? Данная настройка позволяет упростить пути к файлам при использовании импортов файлов.
Выбирайте Yes.

После ответов на все предлагаемые вопросы запустится установка фреймворка. Когда проект будет создан, перейдите в его папку:

cd my-app

Создайте проект на NextJS.
