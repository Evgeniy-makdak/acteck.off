Компонент для меню в NextJS

В следующих уроках мы будем изучать специальные возможности NextJS для создания меню сайта. Для этого нам понадобится компонент с меню сайта. В данном уроке мы сделаем заготовку этого компонента, а ссылки будем вставлять в следующих уроках.

Наша заготовка будет выглядеть следующим образом:

export default function Menu() { return <> ... </>; }

Вставим эту заготовку в основной макет сайта:

import Menu from '@/comps/menu/menu.jsx'; export default function RootLayout({children}) { return ( <html lang="en"> <body> <header> <nav> <Menu /> </nav> </header> <main> {children} </main> <footer> footer </footer> </body> </html> ); }

Сделайте компонент для главного меню.
