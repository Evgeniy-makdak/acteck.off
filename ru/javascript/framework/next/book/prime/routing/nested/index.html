Вложенные роуты в NextJS

Пусть у нас есть некоторый URL, состоящий из нескольких сегментов, например, вот такой: /user/show/name.

Чтобы сделать файл, который будет показываться по данному URL, нам нужно будет сделать несколько вложенных папок. В нашем случае вот так:

  • /app/
    • /user/
      • /show/
        • /name/

Давайте сделаем соответствующий файл:

export default function Name() { return <p> user </p>; }

Сделайте файл, показывающий страницу по следующему URL: /catg/prod/name.
