Приватные папки в NextJS

NextJS по умолчанию из всех папок, вложенных в папку app, формирует роуты. Иногда, однако, в папке app нам нужно сделать приватную папку, которая не будет участвовать в роутинге.

Чтобы сделать такую папку, ее имя следует начать с подчеркивания. В этом случае NextJS будет игнорировать ее содержимое.

Сделайте приватную папку. Разместите в ней файл page.jsx. Убедитесь, что вы не можете получить доступ к этому файлу через адресную строку.
