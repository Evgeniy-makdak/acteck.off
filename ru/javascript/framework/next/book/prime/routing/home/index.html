Файл домашней страницы в фреймворке NextJS

У каждого сайта есть домашняя страница. Эта та страница, которая показывается, когда мы просто вбиваем имя домена в адресную строку. To есть это следующий URL: /.

За этот URL отвечает файл src/app/page.jsx, который уже есть по умолчанию после установки фреймворка.

Сделайте так, чтобы при обращении на адрес / в браузер выводилось какое-нибудь сообщение.
