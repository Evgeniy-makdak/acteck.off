Проект Страны-Города в NextJS

Давайте реализуем проект со странами и городами, а также их описаниями.

Описание страны должно включать в себя язык страны, площадь, количество населения.

Для каждой страны нужно хранить массив городов этой страны.

Описание города должно включать в себя площадь, количество населения, год основания.

Сделайте файл с многомерной структурой данных, в которой будут хранится страны и их города.

На главной странице сайта выведите список всех стран в виде ссылок на страницы стран.

На странице страны выведите описание страны, а также список ее городов в виде ссылок на страницы городов.

На странице города выведите описание города, а также ссылку на страницу страны данного города.
