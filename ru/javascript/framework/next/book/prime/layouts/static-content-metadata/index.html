Статическая передача метаданных из контента в NextJS

Давайте теперь научимся задавать свои метаданные для каждого URL. Для этого объект с метаданными нужно будет экспортировать из файла с контентом сайта.

Давайте посмотрим, как это делается. Пусть у нас есть некоторый компонент с контентом сайта:

export default function Test() { return <h1>test component</h1>; }

Зададим метаданные для URL, соответствующего нашему файлу с контентом:

export const metadata = { title: 'My Test page', description: 'Description for test page.', } export default function Test() { return <h1>test component</h1>; }

Сделайте на вашем сайте три страницы: /about, /price, /contacts. Добавьте на этих страницах какой-нибудь текст. Для каждой страницы сделайте свои метаданные.
