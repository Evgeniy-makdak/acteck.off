Передача параметров в метод в Vue

Можно передавать параметры в методы, привязываемые в качестве обработчиков. Давайте посмотрим на практике. Пусть у нас есть следующий метод, параметром принимающий сообщение, которое нужно вывести:

methods: { show: function(str) { alert(str); } }

Привяжем этот метод к разным тегам, передав ему параметром текст для вывода:

<template> <button @click="show('text1')">btn1</button> <button @click="show('text2')">btn2</button> </template>

Сделайте метод, который параметром будет принимать число и через alert выводить квадрат этого числа.

Сделайте две кнопки. По нажатию на первую кнопку выведите квадрат числа 2, а по нажатию на вторую - квадрат числа 3.
