Инлайн объект с CSS классами в Vue

Объект с CSS классами можно писать прямо в качестве значения атрибута class:

<template> <p :class="{active: true, error: false}"> text </p> </template>

Классы, имена которых содержат дефис, необходимо брать в кавычки:

<template> <p :class="{active: true, 'has-error': false}"> text </p> </template>

Дан следующий объект с CSS классами:

{active: true, valid: false}

Примените данный объект к какому-нибудь тегу.
