Директива v-else-if в Vue

Вместо нескольких v-if можно использовать конструкции v-else-if:

<template> <p v-if="num == 1">one</p> <p v-else-if="num == 2">two</p> <p v-else-if="num == 3">three</p> <p v-else>error</p> </template>

Пусть в data хранится следующее:

data() { age: 25, }

Если свойство age меньше 18 - выведите на экран 'подросток', если оно от 19 до 25 - выведите на экран 'молодой человек', а если более 26 - выведите на экран 'мужчина'.
