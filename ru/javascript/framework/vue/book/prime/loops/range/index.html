Цикл в диапазоне в Vue

Можно сделать так, чтобы цикл повторился заданное количество раз. Это делается следующим образом:

<template> <p v-for="num in 10"> {{ num }} </p> </template>

Сделайте 30 абзацев, заполненных целыми числами по порядку.
