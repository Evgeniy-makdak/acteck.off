Формирование select в цикле в Vue

Теги option можно также создавать не в ручную, а с помощью цикла. Смотрите пример:

data() { return { selected: 'value1', options: ['value1', 'value2', 'value3'], } } <template> <select v-model="selected"> <option v-for="option in options">{{ option }}</option> </select> <p>{{ selected }}</p> </template>
