Установка и подготовка React-приложения

В этом уроке мы начнем с установки и подготовки React-приложения, так как в дальнейшем мы будем использовать его в связке с Redux. Есть более быстрые способы создания шаблона приложения React с Redux (что кстати не всегда означает легко, особенно для новичков), которые указаны в официальной документации, но мы пойдем с нуля.

Для начала создадим React-приложение my-app с использованием Vite (это модная замена Webpack). Для этого откроем терминал и введем следующие команды (в предложенных опциях, если таковые имеются, выберем React и затем JavaScript):

npm create vite@latest my-app -- --template react cd my-app npm install

Запустим созданное приложение:

npm run dev

Откроем в браузере предложенный линк http://localhost:5173/ и увидим знакомую заставку. Все работает.

Теперь удалим лишнее. Очистим папку public. В папке src оставим только main.jsx, index.css и App.jsx (в index.css и App.jsx полностью удалим содержимое).

В файле index.html заменим текст тайтла на:

<title>React Redux App</title>

Создайте React-приложение и убедитесь, что оно работает.

Удалите лишние файлы и папки.
