Возврат вложенных тегов JSX функциями

В предыдущем уроке у вас получился следующий код:

function App() { return <div> text </div>; }

Давайте посмотрим, какие особенности имеет возврат вложенных тегов из функции при работе с JSX. Внутри тега, который возвращается через return, может быть сколько угодно вложенных тегов:

function App() { return <div> <p>text1</p> <p>text2</p> </div>; }

У вас есть следующая функция:

function App() { return <ul> </ul>; }

Добавьте внуть тегов ul список из 10 li.
