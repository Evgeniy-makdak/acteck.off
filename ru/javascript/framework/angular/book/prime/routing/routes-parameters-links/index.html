Параметры маршрутов в ссылках в Angular

Если у нас есть маршруты с параметрами, то мы также можем делать ссылки, ведущие на эти маршруты:

<nav> <a routerLink="/aaaa/1" routerLinkActive="active" > Aaaa with parameter </a> <a routerLink="/bbbb" routerLinkActive="active" > Bbbb </a> </nav>
