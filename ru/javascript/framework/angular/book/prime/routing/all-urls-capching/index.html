Захват всех URL в роутинге в Angular

Можно захватить все возможные адреса с помощью команды **. Это используется, чтобы показать ошибку в том случае, если ни один из адресов не подошел. Давайте сделаем это:

const appRoutes: Routes =[ { path: 'aaaa', component: AaaaComponent}, { path: 'bbbb', component: BbbbComponent}, { path: '**', component: NotFoundComponent }, ];

Сделайте компонент, который будет показываться, если ни один из маршрутов не совпадает с URL.
