Переадресация в роутинге в Angular

В Angular можно делать переадресацию маршрутов по другому пути. Это делается с помощью ключа redirectTo, в котором указывается URL, на который нужно перенаправить запрос.

Давайте для примера, если нужный маршрут для запроса не найден, то переадресуем его на главную страницу:

const appRoutes: Routes =[ { path: '', component: HomeComponent}, { path: 'aaaa', component: AaaaComponent}, { path: '**', redirectTo: '/'} ];

Переадресуйте URL /bbbb/ на /aaaa/:

const appRoutes: Routes =[ { path: 'aaaa', component: AaaaComponent}, { path: 'bbbb', component: BbbbComponent}, ];
