Привязка CSS стилей тегу в Angular

Для HTML-элемента мы также можем задать и определенный набор стилей. Делается это с помощью директивы ngStyle. В эту директиву следует передавать объект с CSS свойствами и их значениями.

Давайте попробуем:

<div [ngStyle]="{'color': 'red', 'font-size': '20px'}"> text </div>

Дан объект:

export class AppComponent { public obj = { color: red, fontSize: '30px', fontWeight: 'bold' } }

Стилизуйте какой-нибудь тег с помощью этого объекта.
