Конструкция if в Angular

Условие в Angular также можно сделать с помощью специальной конструкции @if. Эта конструкция позволяет показывать или скрывать блоки кода по условию. Давайте посмотрим, как с ней работать.

Пусть у нас есть следующая переменная:

export class AppComponent { public isAdmin: boolean = true; }

Давайте сделаем див, который будет показан, если эта переменная имеет значение true:

@if (isAdmin) { <div> text </div> }

Сделаем так, чтобы див был показан, если наша переменная имеет значение false:

@if (!isAdmin) { <div> text </div> }

Сделайте свойство isAdult. Покажите текст для взрослых, если в нашей переменной лежит значение true.
