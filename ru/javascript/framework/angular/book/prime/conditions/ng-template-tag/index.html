Тег ng-template в Angular

Бывают ситуации, когда нам нужно обернуть в условие несколько тегов, но не хотелось бы создавать для них общий родительский тег. В этом случае нам поможет специальный тег Angular с именем ng-template. Этот тег группирует, но не появляется в итоговом HTML коде.

Смотрите пример, как им пользоваться:

<ng-template *ngIf="isAdmin"> <div> text1 </div> <div> text2 </div> </ng-template>
