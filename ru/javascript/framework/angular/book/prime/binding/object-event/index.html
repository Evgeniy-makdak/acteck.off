Объект Event в обработчике события в темплейте Angular

В обработчиках событий можно получать объект Event. Для этого нужно в параметр привязанного метода передать специальную переменную $event (с долларом в начале):

<button (click)="show($event)"> button </button>

Давайте получим переданный объект в методе класса:

export class AppComponent { public show(event: MouseEvent): void { console.log(event); } }

По клику получите объект с событием и выведите его в консоль.
