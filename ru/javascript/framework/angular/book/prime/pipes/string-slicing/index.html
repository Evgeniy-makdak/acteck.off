Обрезание строки через канал в Angular

Давайте научимся пользоваться каналом SlicePipe. Этот канал позволяет обрезать строку. При этом он принимает параметры, которые задают начальный и конечный индексы отрезания. Для передачи параметра после имени канала ставят двоеточие и перечисляют параметры.

Давайте просмотрим на примере. В классе компонента зададим свойство name:

export class AppComponent { name: string = 'john'; }

А теперь с помощью канала SlicePipe и обрежем строку, указав для нее начальный и конечный индексы:

<div> {{ name | slice:0:2 }} </div>

В результате получится следующее:

<div> jo </div>

Создайте строку, включающую 11 символов. Выведите на экран отрезок с 2 по 4 индекс.
