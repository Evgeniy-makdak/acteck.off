Каналы pipes в Angular

Каналы (pipes) представляют собой специальные инструменты для форматирования значений, выводимых через интерполяцию.

Давайте посмотрим общий алгоритм использования каналов. Каждый канал имеет имя, например: UpperCasePipe. Этот канал преобразует строку в верхний регистр. Чтобы применить канал, нужно после интересующего нас значения поставить символ | и написать имя, отбросив суффикс 'Pipes'. Например, вот так:

<div>{{ name | uppercase }}</div>

В следующих уроках мы изучим, какие каналы доступны в Angular, а также особенности их применения.
