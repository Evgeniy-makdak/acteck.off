Перевод в верхний регистр через канал в Angular

С помощью канала UpperCasePipe можно перевести строку в нижний регистр. Давайте попробуем на практике. Пусть в классе компонента у нас есть следующая строка:

export class AppComponent { name: string = 'john'; }

Выведем нашу строку в файле темплейта, применив в ней наш канал:

<div> {{ name | uppercase }} </div>

Результат выполнения кода:

<div> JOHN </div>

Пусть у вас есть массив:

export class AppComponent { users: string[] = ['john', 'alex', 'kate', 'mary']; }

Выведите каждый элемент массива в верхнем регистре.
