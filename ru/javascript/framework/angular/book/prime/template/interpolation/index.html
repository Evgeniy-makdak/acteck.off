Интерполяция в Angular

В файле темплейта можно не только выводить текст, но и писать код JavaScript, выполняя различные операции. В терминах Angular это называется интерполяцией.

Для того, чтобы выполнить операцию, ее нужно написать в двойных фигурных скобках. Для примера давайте найдем сумму двух чисел:

<div> {{ 1 + 2 }} </div>

А теперь найдем квадратный корень из числа:

<div> {{ Math.sqrt(2) }} </div>

С помощью интерполяции выведите произведение двух чисел.

С помощью интерполяции округлите дробь до заданного знака в дробной части.
