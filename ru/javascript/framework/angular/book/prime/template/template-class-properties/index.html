Свойства класса компонента в темплейте Angular

Все публичные свойства класса компонента будут доступны внутри темплейта компонента. Давайте посмотрим, как к ним обратится.

Пусть в нашем классе есть следующее публичное свойство:

export class AppComponent { public test: string = 'abcde'; }

Выведем содержимого нашего свойства в верстке. Для этого нужно написать имя свойства внутри двойных фигурных скобок:

<div> {{ test }} </div>

Дана следующая верстка:

<ul> <li>name: john</li> <li>surn: smit</li> </ul>

Сделайте так, чтобы имя и фамилия передавались в темплейт из класса компонента.
