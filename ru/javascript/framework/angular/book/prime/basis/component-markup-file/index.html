Файл с версткой компонента Angular

В файле app.component.html размещается верстка компонента. Этот файл называют темплейтом или представлением компонента. По умолчанию там будет некоторый текст. Его можно смело удалить и написать свой. Давайте для теста напишем там див с текстом:

<div> text </div>

Сделайте тег с текстом в верстке вашего компонента. Проверьте, что изменения появятся в браузере.
