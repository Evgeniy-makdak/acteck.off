Структура проекта Angular

После выполнения команды на создание проекта в его папке появится некоторая файловая структура. В основном там служебные файлы, поддерживающие работу фреймворка. Сейчас нас интересуют только следующая папка и файл:

  • /src/
  • README.md

В папке src вы будете осуществлять всю работу. Подробнее мы разберем ее в следующем уроке.

Файл README.md содержит инструкции по запуску приложения. Обычно его используют, когда получают архив с проектом от коллеги и не понимают, как запустить проект. В этом случае нужные команды терминала можно подсмотреть в этом файле.
