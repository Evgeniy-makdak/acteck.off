Создание проекта Angular

После установки фреймворка Angular мы можем сделать наш первый проект. Для этого создаем папку, в которой будем работать. Открываем ее через терминал и пишем в нем следующую команду ng new, а после нее имя папки с нашим проектом.

Давайте для примера выберем в качестве имени проекта test:

ng new test

Наш проект будет установлен в папку test. Для дальнейших действий нам нужно перейти в терминале в эту папку:

cd test

После того, как установка закончится, мы можем запустить наш проект. Для этого, находясь в папке с сайтом, следует выполнить следующую команду:

ng serve --open

Выполните установку вашего проекта.
