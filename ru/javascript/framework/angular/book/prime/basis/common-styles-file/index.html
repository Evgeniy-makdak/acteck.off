Файл общих стилей проекта в Angular

После создания проекта в папке src будет лежать файл с общими стилями сайта. Этот файл будет иметь имя styles, а вот его расширение может быть разным в зависимости от настроек, выбранных при создании проекта: .css, .less или .sass.

В этом файле нужно размещать стили, относящиеся к структуре сайта, но не к отдельным компонентам. Дело в том, что для каждого компонента Angular его стили будут храниться в папке с этим компонентом.

Напишите код, который расставит по странице блоки вашего сайта: хедер, футер, сайдбары и контент.
