Файл макета в Angular

Макет сайта размещается в файле src/index.html. Он будет выглядеть примерно вот так:

<!doctype html> <html lang="en"> <head> <meta charset="utf-8"> <title>title</title> <base href="/"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="icon" type="image/x-icon" href="favicon.ico"> </head> <body> <app-root></app-root> </body> </html>

В этом файле вам должно быть все понятно, если вы изучали HTML, за исключением тега app-root:

<body> <app-root></app-root> </body>

Данный тег не является частью HTML. Этот тег добавляет сам Angular. В то место, где написан этот тег будет вставлен HTML код основного компонента. Мы еще поговорим подробнее об этом в следующем уроке.

Поменяйте в макете сайта тайтл страницы.

Напишите в теге body какой-нибудь текст. Проверьте, что он появится в браузере.

Добавьте хедер, футер, сайдбары и контент. Пусть тег app-root размещается в контенте сайта.
