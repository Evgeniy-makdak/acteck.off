Удаление дела в чеклисте на JavaScript

Давайте теперь реализуем удаление дел из списка. Для этого на span для удаления навесим событие:

let remove = document.createElement('span'); remove.classList.add('remove'); remove.textContent = 'удалить'; remove.addEventListener('click', function() { // здесь будет код для удаления тега li из списка ul }); li.appendChild(remove);

Допишите недостающую часть кода для решения описанной задачи.
