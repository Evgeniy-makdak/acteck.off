Завершение дела в чеклисте на JavaScript

Навесим аналогичным образом событие на кнопку 'сделано':

let mark = document.createElement('span'); mark.classList.add('mark'); mark.textContent = 'сделано'; mark.addEventListener('click', function() { // здесь будет код для отметки дела сделанным }); li.appendChild(mark);

Для того, чтобы отметить дело сделанным, нужно соответствующему тегу li следует навесить какой-нибудь CSS класс. Назовем, к примеру, этот класс done.

Toгда для этого класса можно написать соответствующий CSS код:

#list .done { text-decoration: line-through; }

Допишите недостающую часть кода для решения описанной задачи.
