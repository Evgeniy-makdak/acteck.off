Сеттеры мнимых свойств в ООП в JavaScript

Кроме геттеров мнимых свойств можно создавать и их сеттеры. В этом случае внутри сеттера мы должны получить данные, разделить их и записать соответствующие части в нужные публичные свойства.

Давайте попробуем. Реализуем сеттер мнимого свойства full:

class User { constructor(name, surn) { this.name = name; this.surn = surn; } get full() { return this.name + ' ' + this.surn; } set full(full) { let [name, surn] = full.split(' '); this.name = name; this.surn = surn; } }

Упростим дестуктуризацию:

class User { constructor(name, surn) { this.name = name; this.surn = surn; } get full() { return this.name + ' ' + this.surn; } set full(full) { [this.name, this.surn] = full.split(' '); } }

Создадим объект класса:

let user = new User('john', 'smit');

Запишем данные в наш мнимый сеттер:

user.full = 'eric jons';

Проверим, что свойства объекта поменялись:

console.log(user.name); // 'eric' console.log(user.surn); // 'jons' console.log(user.full); // 'eric jons'
