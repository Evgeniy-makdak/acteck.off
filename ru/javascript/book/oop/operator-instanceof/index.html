Оператор instanceof в ООП в JavaScript

Оператор instanceof позволяет проверить принадлежит ли объект определенному классу. Давайте посмотрим на примере. Пусть у нас есть следующий класс:

class User { }

Сделаем объект в этого класса:

let user = new User;

Проверим, принадлежит ли объект из переменной нашему классу:

console.log(user instanceof User); // true

Определите, каким будет результат выполнения следующего кода:

class Student { } class Employee { } let employee = new Employee; console.log(employee instanceof Employee); console.log(employee instanceof Student);

Дан следующий код:

class Student { constructor(name) { this.name = name; } } class Employee { constructor(name) { this.name = name; } } let users = [ new Student('user1'), new Employee('user2'), new Student('user3'), new Employee('user4'), new Student('user5'), new Employee('user6'), new Student('user7'), ];

Переберите циклом массив объектов и выведите в консоль только имена работников.
