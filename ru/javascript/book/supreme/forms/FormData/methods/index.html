Методы FormData в JavaScript

FormData получает данные формы в упорядоченном виде. Это как правило нужно, чтобы отправить эти данные на сервер средствами JavaScript. При этом в FormData есть специальные методы, позволяющие получать и модифицировать данные формы.

Эти методы такие же, как и в URLSearchParams: get, set, has, append, delete, getAll

Модифицируйте предыдущую задачу. После получения данных формы добавьте в эти данные еще одну пару ключ-значение и удалите одну из существующих пар.
