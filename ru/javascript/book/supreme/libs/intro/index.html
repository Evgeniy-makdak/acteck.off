Библиотеки в JavaScript

При программировании часто возникают повторяющиеся задачи. Например, при разработке сайта вам может понадобится слайдер картинок. Самостоятельно реализовать слайдер - задача достаточно трудоемкая, а главное - долгая. Как правило программисты, которые сталкиваются с такой задачей, стараются сделать универсальный код, который можно будет применить не только в текущем проекте, но и в других, экономя время.

Такой код программисты оформляют в библиотеки. Библиотека представляет собой JavaScript файл, который можно подключить к HTML файлу и получить набор готовых функций JavaScript, которые можно будет использовать в нашем проекте.

Существует множество библиотек, которые уже созданы программистами и выложены в открытый доступ для всеобщего использования. Поэтому, если перед вами возникает какая-то объемная задача, всегда лучше сначала посмотреть, нету ли готовых библиотек, решающих эту проблему.
