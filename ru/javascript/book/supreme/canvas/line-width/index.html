Смена размера линий в canvas на JavaScript

Toлщина линии задается с помощью свойства lineWidth. Просто присвойте ему толщину, например так: lineWidth = 5 - получится толщина линии в 5px. Посмотрим на примере:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.lineWidth = 5; // толщина 5px ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.stroke();

:

Toлщину также можно менять и для контуров, например нарисованных с помощью rect:

<canvas width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('canvas'); let ctx = canvas.getContext('2d'); ctx.rect(50, 50, 100, 100); ctx.lineWidth = 5; ctx.stroke();
