Динамический множественный импорт в JavaScript

Можно динамически импортировать сразу несколько модулей:

btn.addEventListener('click', function() { Promise.all([ import('./module1.js'), import('./module2.js'), import('./module3.js'), ]) .then(([module1, module2, module3]) => { }); });

Сделайте модуль с массивом названий дней недели и модуль с названием месяцев.

По нажатию на кнопку импортируйте эти модули. Выведите названия дней и месяцев в виде списков ul.
