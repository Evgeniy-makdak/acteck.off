Переименование при импорте в JavaScript

Можно переименовывать функции при импорте. Это делается с помощью команды as. Для примера давайте переименуем функцию root2 в функцию sqrt:

import {root2 as sqrt, root3} from './math.js';

Проверим работу кода:

let res = sqrt(2) + root3(3); console.log(res);

При импорте вашего модуля переименуйте функцию pow2 в square, а функцию pow3 в cube.
