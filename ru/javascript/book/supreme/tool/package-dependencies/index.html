Зависимости в файле package.json в JavaScript

При установке пакетов npm создает специальный файл package.json, в котором он указывает все установленные зависимости и их версии.

Для примера, если у нас установлена библиотека jquery и библиотека lodash, то этот файл может иметь следующий вид:

{ "dependencies": { "jquery": "^3.6.1", "lodash": "^4.17.21" } }

Установите библиотеку jQuery и библиотеку underscorejs. Изучите файл package.json.
