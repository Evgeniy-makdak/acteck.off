Глобальная установка через npm в JavaScript

Некоторые пакеты npm представляют собой не просто библиотеки, а программы, которые после установки добавляют новую команду в терминал.

Такие программы следует устанавливать глобально. Это значит, что они будут установлены не в папку node_modules проекта, а просто на ваш компьютер как программа.

Для запуска глобального режима установки нужно передать флаг -g:

npm install -g somelib

В зависимости от способа установки NodeJS вам может понадобится написать sudo перед командой (обычно это для Linux):

sudo npm install -g somelib

Давайте посмотрим на примере. Возьмем утилиту sloc, которая считает количество строк кода в проекте. Установим ее глобально:

npm install -g sloc

Находясь в папке с любым проектом выполним в терминале команду на подсчет строк, параметром команды передав путь к папке (точка - текущая папка):

sloc .

Установите глобально библиотеку cowsay. После установки в терминале обратитесь к этой библиотеке, заставив корову сказать что-нибудь в терминал.
