Инструмент npx в JavaScript

В настоящее время глобальная установка пакетов считается дурным тоном. Дело, в том, что такая установка захламляет операционную систему, а также не дает одновременно держать несколько версий библиотеки для разных проектов.

Лучше пакеты-программы устанавливать локально в проект, а для их запуска использовать специальный инструмент npx.

Давайте посмотрим, как это делается. Установим локально в проект утилиту sloc (в dev записимости):

npm install -D sloc

Теперь, находясь в папке нашего проекта, мы можем вызвать установленную утилиту:

npx sloc .

Установите локально библиотеку cowsay и затем вызовите ее через npx.
