Короткая форма установки через npm в JavaScript

Все команды npm имеют две формы: длинную и короткую. Длинная форма более понятная, а короткую быстрее писать, чем часто пользуются профессионалы.

Для примера команда install будет иметь короткую форму i. Давайте попробуем короткую форму:

npm i lodash

Перепишите команду на короткую форму:

npm install underscore
