Код HTTP ответа в AJAX запросе в JavaScript

В свойстве status ответа response содержится код HTTP ответа:

button.addEventListener('click', function() { fetch('/ajax.html').then(response => { console.log(response.status); // например, 200 }); });

Выведите текст страницы, если получен статус 200, и ошибку, если получен статус 404.
