Массив объектов с id в JavaScript

Достаточно часто вам придется работать с массивами объектов, у которых каждый элемент будет иметь некоторый случайный id. Например, массив может быть вот такой:

let arr = [ { id: 'GYi9GauC4gBF1e2SixDvu', prop1: 'value11', prop2: 'value12', prop3: 'value13', }, { id: 'IWSpfBPSV3SXgRF87uO74', prop1: 'value21', prop2: 'value22', prop3: 'value23', }, { id: 'JAmjRlfQT8rLTm5tG2m1L', prop1: 'value31', prop2: 'value32', prop3: 'value33', }, ];

В таких массивах объектов обращение к элементам идет по их id. Проблема тут в том, что эти id находятся внутри элементов, а значит для любой работы с элементом его сначала надо будет найти. Приемам работы с такими массивами мы будем учиться в следующих уроках.
