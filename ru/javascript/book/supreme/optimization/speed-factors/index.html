Факторы скорости загрузки сайта в JavaScript

Скорость загрузки и работы страниц сайта очень важна. Ведь чем медленнее грузится ваш сайт, тем меньше пользователей дождутся окончания его загрузки. Из-за этого вы будете терять посетителей и, как следствие, - деньги.

На скорость загрузки сайта влияют множество факторов. В первую очередь, это конечно же, количество данных, передаваемых вами при загрузке сайта: HTML код с текстом, файлы CSS, JavaScript, и, особенно, картинки.

На скорость загрузки сайта также оказывает влияние качество нашего JavaScript кода. Дело в том, что многие при написании кода зачастую допускают ряд ошибок, из-за которых страница будет грузится и работать медленнее, чем могла бы.
