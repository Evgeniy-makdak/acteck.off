Ключи из переменных в многомерных структурах JavaScript

Пусть дан следующий объект:

let obj = { 'sub1': ['11', '12', '13'], 'sub2': ['21', '22', '23'], };

Давайте выведем какой-нибудь элемент из нашего объекта, например, элемент '22':

console.log(obj['sub2'][1]);

Пусть теперь ключи хранятся в переменных:

let key1 = 'sub2'; let key2 = 1;

Выведем с помощью наших переменных элемент многомерной структуры:

console.log(obj[lang][day]);

Дана вот такая структура для хранения списка дел за года, месяцы и дни:

let affairs = { '2018': { 11: { 29: ['name111', 'name112', 'name113'], 30: ['name121', 'name122', 'name123'], }, 12: { 30: ['name211', 'name212', 'name213'], 31: ['name221', 'name222', 'name223'], }, }, '2019': { 12: { 29: ['name311', 'name312', 'name313'], 30: ['name321', 'name322', 'name323'], 31: ['name331', 'name332', 'name333'], } }, }

Пусть даны также три переменные, содержащие год, месяц и день. Выведите дело, соответствующее значениям переменных.

Автор следующего кода хотел вывести элемент со значением '24':

let obj = { key1: { key2: '12', key3: '13', }, key2: { key4: '24', key5: '25', }, } let key1 = 'key2'; let key2 = 'key4'; console.log(obj['key1'][key2]);
