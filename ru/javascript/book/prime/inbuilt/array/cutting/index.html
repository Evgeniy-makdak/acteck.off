Вырезание массивов в JavaScript

splice

Дан следующий массив:

[1, 2, 3, 4, 5]

С помощью метода splice преобразуйте массив в следующий:

[1, 4, 5]

Дан следующий массив:

[1, 2, 3, 4, 5]

С помощью метода splice сделайте из него массив:

[1, 2, 3, 'a', 'b', 'c', 4, 5]

Дан следующий массив:

[1, 2, 3, 4, 5]

С помощью метода splice сделайте из него массив:

[1, 'a', 'b', 2, 3, 4, 'c', 5, 'e']
