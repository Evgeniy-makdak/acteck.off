Активация элементов на JavaScript

Пусть у нас есть некоторая HTML таблица #table с пустыми ячейками.

Давайте сделаем так, чтобы по нажатию на любую ячейку она каким-то образом активировалась, например, становилась красного фона. Для этого активированным ячейкам будем давать какой-нибудь CSS класс:

.active { background: red; }

Реализуем активацию:

let tds = document.querySelectorAll('#table td'); for (let td of tds) { td.addEventListener('click', function() { this.classList.add('active'); }); }

Дана HTML список ul. Сделайте так, чтобы по нажатию на любой пункт списка он активировался красным фоном.

Модифицируйте предыдущую задачу так, чтобы по нажатию на активированный пункт списка активация с него снималась.
