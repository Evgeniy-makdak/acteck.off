Один обработчик ко многим элементам в JavaScript

Одну функцию можно привязать сразу к нескольким элементам. Пусть для примера у нас есть следующая функция:

function func() { console.log('!!!'); }

Есть также две кнопки:

<input id="button1" type="submit" value="button1"> <input id="button2" type="submit" value="button2">

Получим ссылки на эти кнопки в переменные:

let button1 = document.querySelector('#button1'); let button2 = document.querySelector('#button2');

А теперь давайте привяжем нашу функцию func и к первой, и ко второй кнопке:

button1.addEventListener('click', func); button2.addEventListener('click', func);

Даны 5 абзацев:

<p id="elem1">text</p> <p id="elem2">text</p> <p id="elem3">text</p> <p id="elem4">text</p> <p id="elem5">text</p>

Дана следующая функция:

function func() { console.log('message'); }

Привяжите эту функцию ко всем 5-ти абзацам.
