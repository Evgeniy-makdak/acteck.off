Изменение объектов через цикл в JavaScript

Объекты также можно менять в цикле. Пусть, к примеру, у нас есть вот такой объект:

let obj = {a: 1, b: 2, c: 3};

Давайте переберем его циклом for-in и увеличим каждый из его элементов в 2 раза:

for (let key in obj) { obj[key] = obj[key] * 2; }

Проверим результат:

console.log(obj);

Дан объект:

let obj = {x: 1, y: 2, z: 3};

Переберите этот объект циклом и возведите каждый элемент этого объекта в квадрат.

Дан объект:

let obj = {x: 1, y: 2, z: 3};

Переберите этот объект циклом и увеличьте каждый элемент этого объекта на единицу.
