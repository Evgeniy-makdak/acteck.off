Логические операторы без if в функциях JavaScript

Пусть у нас есть функция с ифом. Вот она:

function func(a, b) { if (a > b) { return true; } else { return false; } }

Как вы уже знаете из предыдущих уроков, конструкции if, возвращающие булевы значения, можно переписывать в сокращенной форме. Давайте сделаем это:

function func(a, b) { return a > b; }

Дана следующая функция:

function func(a, b) { if (a == b) { return true; } else { return false; } }

Перепишите ее код в сокращенной форме согласно изученной теории.

Дана следующая функция:

function func(a, b) { if (a != b) { return true; } else { return false; } }

Перепишите ее код в сокращенной форме согласно изученной теории.

Дана следующая функция:

function func(a, b) { if (a + b >= 10) { return true; } else { return false; } }

Перепишите ее код в сокращенной форме согласно изученной теории.

Дана следующая функция:

function func(num) { if (num >= 0) { return true; } else { return false; } }

Перепишите ее код в сокращенной форме согласно изученной теории.
