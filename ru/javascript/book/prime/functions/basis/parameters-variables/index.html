Параметры-переменные функций в JavaScript

В параметры функции можно передавать переменные. Давайте посмотрим на примере. Пусть у нас есть следующая функция:

function func(num) { console.log(num ** 2); }

Пусть у нас есть переменная:

let param = 2;

Вызовем нашу функцию, передав ей параметром число из переменной:

let param = 2; func(param);

Сделайте функцию func, которая параметром будет принимать 3 числа и выводить в консоль их сумму. Пусть даны 3 переменные с числами:

let param1 = 1; let param2 = 2; let param3 = 3;

С помощью созданной вами функции выведите в консоль сумму значений эти переменных.
