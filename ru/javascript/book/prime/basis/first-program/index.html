Первая программа на JavaScript

Давайте напишем нашу первую программу на JavaScript. Вот она:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Моя первая программа</title> <script> alert('Привет, мир!'); </script> </head> <body> моя первая программа </body> </html>

Скопируйте этот код и разместите в HTML файле. Затем откройте этот файл в браузере - и вы увидите диалоговое окошко с текстом.

Сделайте файл с вашей первой программой. Выведите в ней на экран какой-нибудь текст.
