Функция clearTimeout

Функция clearTimeout останавливает таймер, установленный функцией setTimeout. Функция принимает идентификатор того таймера, который нужно остановить. Идентификатор таймера возвращает метод setTimeout.

Синтаксис

clearTimeout(идентификатор);

Пример

Остановим таймер по достижению счетчиком значения 10:

let i = 0; let id = setInterval(function() { i++; if (i == 10) { clearTimeout(id); } else { console.log(i); } }, 1000);
