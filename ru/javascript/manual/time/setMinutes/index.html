Метод setMinutes

Метод setMinutes позволяет изменить количество минут объекту с датой. Принимает первым параметром количество минут, также вторым и третьим необязательными параметрами может принимать секунды и миллисекунды.

Синтаксис

дата.setMinutes(минуты, [секунды], [миллисекунды]);

Пример

Давайте установим минуты для текущего момента времени:

let date = new Date(); date.setDate(59); console.log(date);

Пример

Давайте установим минуты и секунды для текущего момента времени:

let date = new Date(); date.setDate(59, 59); console.log(date);

Смотрите также

  • метод setHours,
    который устанавливает часы
  • метод setSeconds,
    который устанавливает секунды
  • метод setMilliseconds,
    который устанавливает миллисекунды
