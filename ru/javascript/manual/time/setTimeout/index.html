Функция setTimeout

Функция setTimeout задает задержку перед выполнением кода. Первым параметром следует передавать коллбэк, вторым - время в миллисекундах, указывающее через какой промежуток начнет выполнятся код, заданный первым параметром. Функция возвращает уникальный идентификатор, с помощью которого можно остановить таймер. Для этого этот идентификатор следует передать функции clearTimeout.

Синтаксис

setTimeout(функция, время);

Пример

Выведем в консоль какой-нибудь текст с задержкой в 1 секунду:

setTimeout(function() { console.log('text'); }, 1000);
