Метод setMilliseconds

Метод setMilliseconds позволяет изменить количество миллисекунд объекту с датой. Принимает первым параметром количество миллисекунд.

Синтаксис

дата.setMilliseconds(миллисекунды);

Пример

Давайте установим миллисекунды для текущей даты:

let date = new Date(); date.setMilliseconds(999); console.log(date);

Смотрите также

  • метод setHours,
    который устанавливает часы
  • метод setMinutes,
    который устанавливает минуты
  • метод setSeconds,
    который устанавливает секунды
