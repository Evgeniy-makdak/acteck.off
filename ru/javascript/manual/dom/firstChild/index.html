Свойство firstChild

Свойство firstChild хранит первый дочерний узел элемента. Если у элемента нет дочерних элементов - возвращается null.

Синтаксис

элемент.firstChild;

Пример

Давайте получим текст первого узла элемента:

<div id="parent">text<p>parag</p><!--com--></div> let parent = document.querySelector('#parent'); let text = parent.firstChild.textContent; console.log(text);

Результат выполнения кода:

'text'

Смотрите также

  • свойство firstElementChild,
    которое содержит первый элемент
  • свойство lastChild,
    которое содержит последний узел
  • свойство childNodes,
    которое содержит все узлы элемента
