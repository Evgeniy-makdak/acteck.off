Свойство tagName

Свойство tagName содержит имя тега в верхнем регистре (большими буквами).

Синтаксис

элемент.tagName;

Пример

Давайте получим элемент #elem и выведем название его тега:

<div id="elem"></div> let elem = document.getElementById('elem'); console.log(elem.tagName);

Результат выполнения кода:

'DIV'

Пример

Давайте выведем название тега в нижнем регистре. Для этого воспользуемся методом toLowerCase:

<div id="elem"></div> let elem = document.getElementById('elem'); console.log(elem.tagName.toLowerCase());

Результат выполнения кода:

'div'
