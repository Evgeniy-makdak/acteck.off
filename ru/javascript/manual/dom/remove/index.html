Метод remove

Метод remove позволяет удалить элемент. Применяется к тому элементу, который нужно удалить.

Синтаксис

элемент.remove();

Пример

Давайте удалим абзац #elem:

<div id="parent"> <p id="elem">elem 1</p> <p>elem 2</p> </div> let elem = document.querySelector('#elem'); elem.remove();

Результат выполнения кода:

<div id="parent"> <p>elem 2</p> </div>
