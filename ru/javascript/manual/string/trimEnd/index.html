Метод trimEnd

Метод trimEnd удаляет пробелы в конце строки и возвращает новую строку без изменения исходной строки.

Синтаксис

строка.trimEnd();

Пример

Давайте удалим лишние пробелы в строке ' abcde ':

let res = ' abcde '.trimEnd(); console.log(res);

В результате выполнения кода мы увидим, что в конце строки уже нет пробелов:

' abcde'

Смотрите также

  • метод trimStart,
    который удаляет пробелы в начале строки
  • метод trim,
    который удаляет пробелы по краям строки
