Метод padEnd

Метод padEnd дополняет конец текущей строки до достижения длины, заданной в первом параметре. Вторым необязательным параметром указывается строка, которой мы хотим заполнить текущую.

Синтаксис

строка.padEnd(длина строки, [строка для заполнения]);

Пример

Давайте дополним строку, чтобы ее длина составила 10 символов:

let res = 'abcdef'.padEnd(10); console.log(res);

Результат выполнения кода:

'abcdef '

Пример

В случае, если мы в опциональный параметр передаем больше символов, чем требуется, лишние отбросятся и не дополнят строку.

let res = 'abc'.padEnd(5, 'defg'); console.log(res);

Результат выполнения кода:

'abcde'

Пример

Давайте зададим в первом параметре число меньше, чем длина текущей строки:

let res = 'abcde'.padEnd(1); console.log(res);

В результате выполнения кода вернется вся строка:

'abcde'

Смотрите также

  • свойство length,
    которое определяет длину строки
  • метод padStart,
    который дополняет текущую строку с ее начала заданной строкой
  • метод trim,
    который удаляет пробелы по краям строки
  • метод repeat,
    который создает копии строки
