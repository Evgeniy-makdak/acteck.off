Метод startsWith

Метод startsWith проверяет начинается ли строка с указанной в первом параметре подстроки. Если начинается, то возвращает true, а если не начинается, то false. Вторым необязательным параметром метод принимает позицию, с которой начинать проверку (по умолчанию с начала строки).

Синтаксис

строка.startsWith(что ищем, [начало проверки]);

Пример

Проверим, начинается ли строка на заданную подстроку:

let str = 'abcde'; let res = str.startsWith('abc'); console.log(res);

Результат выполнения кода:

true

Пример

Проверим, начинается ли строка на заданную подстроку:

let str = 'abcde'; let res = str.startsWith('xxx'); console.log(res);

Результат выполнения кода:

false

Пример

Начнем проверку с заданной позиции:

let str = 'abcde'; let res = str.startsWith('bc', 1); console.log(res);

Результат выполнения кода:

true
