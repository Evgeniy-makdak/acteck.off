Метод Math.cos

Метод Math.cos возвращает косинус числа (это число должно быть углом в радианах).

Синтаксис

Math.cos(число);

Пример

Давайте выведем косинус числа 34:

console.log(Math.cos(34));

Результат выполнения кода:

-0.8485702747846052

  • метод Math.sin,
    который возвращает синус числа
  • метод Math.tan,
    который возвращает тангенс числа
  • метод Math.asin,
    который возвращает арксинус числа
  • метод Math.acos,
    который возвращает арккосинус числа
  • метод Math.atan,
    который возвращает арктангенс числа
