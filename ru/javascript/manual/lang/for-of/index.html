Конструкция for-of

Конструкция for-of создает цикл для перебора массива.

Синтаксис

for (let переменнаяДляЭлемента of массив) { /* В переменнуюДляЭлемента по очереди будут попадать элементы перебираемого массива. */ };

Пример

Переберем элементы массива и выведем их на экран:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); }

Пример

Сделаем счетчик итераций цикла:

let arr = [1, 2, 3, 4, 5]; let i = 0; for (let elem of arr) { console.log(elem); i++; } console.log(i);
