Конструкция Array

Конструкция Array создает массив из указанных в параметре элементов, которые перечисляются через запятую.

Синтаксис

let массив = new Array(элементы массива через запятую);
let массив = [элементы массива через запятую];

Пример

Давайте создадим массив из чисел:

let arr = new Array(1, 2, 3, 4); console.log(arr);

Результат выполнения кода:

[1, 2, 3, 4]

Пример

А теперь давайте создадим массив из строчных элементов:

let arr = new Array('a', 'b', 'c', 'd'); console.log(arr);

Результат выполнения кода:

['a', 'b', 'c', 'd']
