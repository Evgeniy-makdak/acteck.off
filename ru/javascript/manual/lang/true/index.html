Значение true

Значение true обозначает истину. Его можно использовать в условных выражениях с if и elseif для проверки их на истиннность, а также с помощью true можно определить наличие элемента или свойства.

Синтаксис

true;

Пример

Давайте проверим, есть ли в массиве буква 'a':

let arr = ['a', 'b', 'c', 'd']; console.log(arr.includes('a'));

Результат выполнения кода:

true

Пример

Давайте проверим равно ли значение переменной 1:

let num = 1; console.log(num === 1);

Результат выполнения кода:

true

Пример

Давайте проверим результат выполнения условия:

let test = true; if(test === true) { console.log('+++'); } else { console.log('---'); }

Результат выполнения кода:

'---'

Смотрите также

  • значение false,
    которое обозначает, что значение элемента ложное
  • значение null,
    которое обозначает отсутствие объекта или элемента
  • значение undefined,
    которое обозначает, что значение элемента "не определено"
