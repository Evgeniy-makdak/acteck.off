Свойство offsetHeight

Свойство offsetHeight содержит полную высоту элемента (включает собственно ширину элемента, высоту границ, внутренние отступы, полосы прокрутки):

Синтаксис

элемент.offsetHeight;

Пример

Давайте узнаем полный размер элемента:

<div id="elem"></div> #elem { width: 100px; height: 100px; padding: 15px; border: 10px solid black; } let elem = document.querySelector('#elem'); console.log(elem.offsetHeight);

Результат выполнения кода:

150

Смотрите также

  • свойство clientHeight,
    которое содержит высоту элемента внутри границ
  • свойство offsetWidth,
    которое содержит полную ширину элемента
  • метод getComputedStyle,
    который получает значение CSS свойства элемента
