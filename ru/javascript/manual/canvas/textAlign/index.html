Свойство textAlign

Свойство textAlign задает горизонтальное выравнивание текста, нарисованного с помощью метода fillText или метода strokeText. Принимает одно из возможных значений: start (по умолчанию), end, left, right, center (см. CSS свойство text-align для понимания этих значений):

Синтаксис

контекст.textAlign = значение;

Смотрите также

  • свойство textBaseline,
    которое выравнивает текст по вертикали
  • свойство font,
    которое задает размер и тип шрифта
