Свойство lineWidth

Свойство lineWidth задает толщину линии при рисовании на canvas.

Синтаксис

контекст.lineWidth = целое число;

Пример

Давайте нарисуем линию и установим ей толщину в 5 пикселей:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.lineWidth = 5; // толщина 5px ctx.moveTo(50, 50); ctx.lineTo(150, 50); ctx.stroke();

:

Пример

Toлщину также можно и менять для контуров, например нарисованных с помощью rect:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.rect(50, 50, 100, 100); ctx.lineWidth = 5; ctx.stroke();

:

Смотрите также

  • свойство lineCap,
    которое задает вид конца линии
  • свойство lineJoin,
    которое задает стиль излома линии
