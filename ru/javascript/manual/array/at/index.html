Метод at

Метод at возвращает элемент массива с заданным индексом. В параметре метода указывается целое положительное или отрицательное число. В первом случае - идет поиск элемента с начала массива, во втором - с конца.

Синтаксис

массив.at(число);

Пример

Давайте найдем первый элемент массива:

let res = ['a', 'b', 'c'].at(0); console.log(res);

Результат выполнения кода:

'a'

Пример

Теперь давайте зададим в параметре метода отрицательное число:

let res = ['a', 'b', 'c'].at(-1); console.log(res);

Результат выполнения кода:

'c'

Пример

А теперь давайте найдем элемент, указав в параметре дробное число:

let res = ['a', 'b', 'c'].at(1.8); console.log(res);

Как видно из полученного результата, метод возвращает элемент, который соответствует целой части дробного числа:

'b'
