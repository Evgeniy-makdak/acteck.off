Метод event.preventDefault

Метод event.preventDefault позволяет отменить действия браузера по умолчанию. Например, сделать так, чтобы при клике по ссылке не было перехода на другую страницу или по нажатию на кнопку форма не отправлялась на сервер. Как пользоваться: просто внутри функции, которая привязана к событию, в любом месте следует вызвать event.preventDefault(), где event - это объект Event.

Синтаксис

event.preventDefault();

Пример

Давайте сделаем так, чтобы по клику на ссылку не происходило перехода на другую страницу:

<a href="/" id="elem">ссылка</a> let elem = document.querySelector('#elem'); elem.addEventListener('click', function(event) { event.preventDefault(); alert('вы не можете перейти по этой ссылке!'); });
