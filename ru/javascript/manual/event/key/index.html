Свойство event.key

Свойство event.key позволяет узнать значение нажатой клавиши при вводе текста.

Синтаксис

event.key;

Пример

<input id="elem"> let elem = document.querySelector('#elem'); elem.addEventListener('keydown', function(event) { console.log(event.key); });

Смотрите также

  • свойство event.altKey,
    которое отлавливает нажатие клавиши Alt
  • свойство event.ctrlKey,
    которое отлавливает нажатие клавиши Ctrl
  • свойство event.shiftKey,
    которое отлавливает нажатие клавиши Shift
  • свойство event.metaKey,
    которое отлавливает нажатие клавиши Cmd
  • свойство code,
    которое получает код нажатой клавиши
