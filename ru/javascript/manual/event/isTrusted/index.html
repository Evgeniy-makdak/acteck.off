Свойство event.isTrusted

Свойство event.isTrusted - позволяет проверить реальное ли событие (вызвано действием пользователем) или же сымитировано на JavaScript с помощью метода dispatchEvent. Принимает значение либо true (настоящее), либо false.

Синтаксис

event.isTrusted;

Смотрите также

  • свойство currentTarget,
    которое определяет элемент, на который назначили событие
  • свойство target,
    которое определяет элемент, на который назначили событие
  • свойство type,
    которое показывает тип события
