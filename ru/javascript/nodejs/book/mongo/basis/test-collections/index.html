Тестовые коллекции для следующих уроков

Давайте сделаем тестовую коллекцию users, которую мы будем использовать в примерах следующих уроков:

[ { "name": "user1", "age": 25, "salary": 300 }, { "name": "user2", "age": 25, "salary": 400 }, { "name": "user3", "age": 26, "salary": 400 }, { "name": "user4", "age": 26, "salary": 500 }, { "name": "user5", "age": 26, "salary": 500 }, { "name": "user6", "age": 27, "salary": 500 } ]

Также сделаем тестовую коллекцию prods, для использования в задачах следующих уроков:

[ { "name": "prod1", "cost": 100, "rest": 10 }, { "name": "prod2", "cost": 100, "rest": 10 }, { "name": "prod3", "cost": 200, "rest": 10 }, { "name": "prod4", "cost": 300, "rest": 20 }, { "name": "prod5", "cost": 300, "rest": 30 } ]

С помощью Compass создайте описанные коллекции.
