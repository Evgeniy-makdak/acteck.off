Показ документа из БД

Давайте сделаем маршрут, который будет выводить в браузере определенного юзера. Пусть имя юзера, которого мы хотим вывести, задается параметром маршрута:

app.get('/user/show/:name', function(req, res) { console.log(req.params.name); });

Давайте получим юзера по его имени:

app.get('/user/show/:name', function(req, res) { let name = req.params.name; let user = await coll.findOne({name: name}); console.log(user); });

Отправим данные полученного юзера в браузер:

app.get('/user/show/:name', function(req, res) { let name = req.params.name; let user = await coll.findOne({name: name}); res.send('name: ' + user.name + 'salary: ' + user.salary); });

Сделайте маршрут, который будет показывать в браузере определенный продукт по его имени.
