Вывод одной записи в NodeJS

В переменной results хранится массив полученных записей, даже если она нужна нам одна. В этом случае можно получить эту одну запись как нулевой элемент нашего массива:

let query = "SELECT * FROM users"; let [results, fields] = await connection.query(query); console.log(results[0]);
