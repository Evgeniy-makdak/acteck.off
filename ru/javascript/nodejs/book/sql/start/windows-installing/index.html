Установка MySQL на ОС Windows

Существуют следующие способы установки MySQL на ОС Windows:

  • Загрузка с помощью MSI (Microsoft Software Installer) - набора файлов, содержащего всю необходимую информацию и инструкции по установке программы. Этот способ установки является рекомендуемым.
  • Скачивание и установка с помощью ZIP архива из официального сайта.
  • В случае, если вы работаете с Docker, то MySQL можно установить через образ, находящийся на Docker Hub.

На странице с документацией выберите наиболее удобный для вас способ установки MySQL.
