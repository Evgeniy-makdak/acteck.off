Вывод массивов в Handlebars Express

Пусть у нас есть некоторая страница page.hbs:

app.get('/page/', function(req, res) { res.render('page'); });

Пусть на эту страницу передается массив с некоторыми данными:

app.get('/page/', function(req, res) { res.render('page', { users: ['user1', 'user2', 'user3'] }); });

Пусть у нас есть страница контента:

<div> page </div>

Давайте выведем на ней наш массив с юзерами, написав его в фигурных скобках:

<div> {{users}} </div>

В этом случае результатом рендеринга будет строковое представление массива:

<div> user1,user2,user3 </div>

Давайте теперь выведем каждый элемент массива в своем теге:

<div> <span>{{user.[0]}}</span> <span>{{user.[1]}}</span> <span>{{user.[2]}}</span> </div>

Дан следующий массив:

let products = ['prod1', 'prod2', 'prod3'];

Выведите данные этого массива в представлении.
