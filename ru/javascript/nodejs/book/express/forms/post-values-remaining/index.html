Сохранения значения формы после отправки методом POST в Express

Давайте теперь сделаем так, чтобы после отправки формы в ней остались введенные данные. Для этого в обработчике post выполним рендеринг файла с формой, передав в него данные формы:

app.post('/', function(req, res) { res.render('form', { body: req.body }); });

А в самой форме выведем данные в атрибутах value:

<form action="" method="POST"> <input name="test1" value="{{body.test1}}"> <input name="test2" value="{{body.test2}}"> <input type="submit"> </form>

Спросите у пользователя его логин, пароль и email. Сделайте так, чтобы после отправки формы в ней остались введенные данные.
