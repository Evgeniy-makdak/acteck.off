Необязательные параметры в маршрутах Express

Пусть у нас есть следующий маршрут:

app.get('/test/:num/', function(req, res) { });

Давайте сделаем так, чтобы наличие параметра в запрошенном URL было не обязательным. Для этого поставим после его имени знак вопроса:

app.get('/test/:num?/', function(req, res) { });

Дан маршрут:

app.get('/test/:num1/:num2/', function(req, res) { });

Сделайте так, чтобы второй параметр этого маршрута был не обязательным.
