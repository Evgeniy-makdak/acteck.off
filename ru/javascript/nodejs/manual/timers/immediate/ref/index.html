Метод ref

Метод ref класса Immediate запрашивает, чтобы цикл событий NodeJS не завершался, пока активен Immediate. Метод возвращает объект Immediate.

Синтаксис

Immediate.ref();

Пример

Давайте узнаем активен ли установленный нами цикл:

let Immediate = setImmediate(() => { console.log('text'); }); console.log (Immediate.ref());

Результат выполненного кода:

Immediate { _idleNext: null, _idlePrev: null, _onImmediate: [Function (anonymous)], _argv: undefined, _destroyed: false, [Symbol(refed)]: true, [Symbol(asyncId)]: 7, [Symbol(triggerId)]: 0 } 'text'

  • метод unref,
    который поддерживает цикл событий активным
