Свойство defaultPrevented

Свойство defaultPrevented проверяет отменено ли данное событие. Если событие можно отменено, то свойство возвращает булевое значение true, в противном случае - false.

Синтаксис

event.defaultPrevented;

Пример

Давайте узнаем отменен ли клик на кнопку:

<button>Click</button> let button = document.querySelector('button'); button.addEventListener('click', function(event) { console.log(event.defaultPrevented); });

Результат выполненного кода:

false
