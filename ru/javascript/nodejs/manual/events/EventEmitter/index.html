Конструктор EventEmitter

Конструктор EventEmitter создает событие для слушателей. В параметре метода ничего не указывается.

Синтаксис

let eventEmitter = new EventEmitter();

Пример

Давайте создадим новое событие для слушателя:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener() { console.log('text'); });

Смотрите также

  • метод emit,
    который вызывает слушателя для события
