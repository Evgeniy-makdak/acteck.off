Метод getMaxListeners

Метод getMaxListeners модуля EventEmitter возвращает текущее максимальное количество слушателей, которое устанавливается с помощью метода setMaxListeners или по умолчанию методом defaultMaxListeners.

Синтаксис

eventEmitter.getMaxListeners();

Пример

Давайте узнаем максимальное количество слушателей:

import { EventEmitter } from 'node:events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener1() { console.log('text1'); }); eventEmitter.on('event', function listener2() { console.log('text2'); }); console.log(eventEmitter.getMaxListeners());

Результат выполненного кода:

10

Смотрите также

  • метод listenerCount,
    который возвращает количество слушателей события
  • метод setMaxListeners,
    который изменяет максимальное количество слушателей для события
