Метод escape

Метод escape кодирует любую строку в строку запроса URL, используя символ '%'. Кодирование необходимо для корректной передачи символов строки в формат, который может передаваться через Интернет. Также некоторые небезопасные символы ASCII, заменяются на безопасную последовательность с '%' и двумя шестнадцатеричными цифрами.

Синтаксис

querystring.escape(строка);

Пример

Давайте преобразуем строку в URL-адрес:

import querystring from 'querystring'; console.log(querystring.escape('abc def'));

Результат выполненного кода:

'abc%20def'

Смотрите также

  • метод unescape,
    который возвращает строку из URL-адрес с %
