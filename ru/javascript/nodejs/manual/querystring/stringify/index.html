Метод stringify

Метод stringify (синоним - encode) возвращает строку запроса URL-адреса из объекта. В первом параметре метода указываем объект, который будет преобразован в URL-адрес, во втором необязательном параметре - строку, которая определяет значение ключей (по умолчанию '='). В третьем необязательном параметре метода указываем подстроку для разграничения пар ключ-значение (по умолчанию ее значение '&'). В четвертом необязательном параметре можно задать объект с опциями.

Синтаксис

path.stringify(объект, разделитель ключей, разделитель значений, [опции]);

Объект с опциями

Опция Описание
stringifyURIComponent Задает декодирование символов с процентным кодированием в строке запроса. Значение по умолчанию false.

Пример

Давайте преобразуем объект в URL-адрес:

import querystring from 'querystring'; console.log(querystring.stringify( { 1: 'a', 2: 'b', 3: 'c' } ));

Результат выполненного кода:

'1=a&2=b&3=c'
