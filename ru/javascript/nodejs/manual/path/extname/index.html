Метод extname

Метод extname возвращает расширение файла из указанного в параметре пути. При определении расширения метод ориентируется на значения после '.'. Если в пути нет '.', то вернется пустая строка.

Синтаксис

path.extname(путь к файлу);

Пример

Давайте узнаем расширение из пути к файлу:

import path from 'path'; console.log(path.extname('./dir/file.txt'));

Результат выполненного кода:

'.txt'

Пример

Пусть теперь нет расширения:

import path from 'path'; console.log(path.extname('./dir/file'));

После выполнения кода мы увидим:

''
