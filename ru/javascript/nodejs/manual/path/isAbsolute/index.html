Метод isAbsolute

Метод isAbsolute определяет, является ли путь к файлу абсолютным, и возвращает значения true или false. В параметре метода передаем путь к файлу.

Синтаксис

path.isAbsolute(путь к файлу);

Пример

Давайте узнаем, является ли путь к файлу абсолютным:

import path from 'path'; console.log(path.isAbsolute('./dir/file.txt'));

Результат выполненного кода:

false

Пример

А теперь давайте изменим наш путь и снова проверим абсолютный ли он:

import path from 'path'; console.log(path.isAbsolute('C:/Folder/dir/file.txt'));

Результат выполненного кода:

true
