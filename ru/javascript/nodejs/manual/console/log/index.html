Метод log

Метод log в модуле console выводит сообщения в консоль. В параметре метода можно указывать любые типы значений.

Синтаксис

console.log(значения);

Пример

Давайте выведем в консоль строку 'abcde':

console.log('abcde');

Результат выполненного кода:

'abcde';
