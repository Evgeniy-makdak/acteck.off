Метод clear

Метод clear в модуле console очищает вывод консоли. Работа метода различается в зависимости от используемой операционной системы и типах терминалов. Для операционных систем Linux метод работает так же, как команда очистки оболочки в терминале. В Windows метод очищает только текущее окно просмотра терминала для бинарного файла NodeJs. В параметре метода ничего не указываем.

Синтаксис

console.clear();

Пример

Давайте очистим нашу консоль:

console.clear();
