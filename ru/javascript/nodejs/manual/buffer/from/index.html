Метод from

Метод from создает новый объект буфера из указанного значения. В первом параметре метода задаем значение, из которого мы хотим создать буфер, им могут быть строка, массив байтов, массив буферов, объект, буфер. Во втором необязательном параметре можно задать индекс первого отображаемого байта (умолчанию его значение 0), в третьем необязательном параметре - количество байтов для нового объекта буфера. В четвертом и пятом необязательных параметрах указываем смещение в байтах и кодировку соответственно.

Синтаксис

Buffer.from(значение, [индекс первого байта], [количество], [смещение], [кодировка]);

Пример

Давайте создадим новый объект буфера из строки:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer);

Результат выполненного кода:

<Buffer 61 62 63 64 65>

    который создает новый объект буфера
  • метод concat,
    который объединяет все объекты буфера из массива в один объект
