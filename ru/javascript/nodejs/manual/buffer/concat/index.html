Метод concat

Метод concat объединяет все объекты буфера из заданного массива в один объект. Метод возвращает буфер. В первом параметре метода указываем массив объектов буфера, во втором необязательном параметре - размер объединенного буфера.

Синтаксис

Buffer.concat(массив объектов буфера, [размер объединенного буфера]);

Пример

Давайте объединим объекты буфера из массива:

import { Buffer } from 'buffer'; let buffer1 = Buffer.alloc(4); let buffer2 = Buffer.alloc(6); let arr = [buffer1, buffer2]; let newBuffer = Buffer.concat(arr) console.log(newBuffer);

Результат выполненного кода:

<Buffer 00 00 00 00 00 00 00 00 00 00>

Смотрите также

  • метод isBuffer,
    который проверяет является ли объект буфером
  • метод fill,
    который заполняет объект буфера данными
