Свойство poolSize

Свойство poolSize определяет размер в байтах ранее выделенных экземпляров буфера, которые будут использоваться для объединения.

Синтаксис

Buffer.poolSize = значение;

Пример

Давайте назначим размер объекту буфера:

import { Buffer } from 'buffer'; Buffer.poolSize = 10; console.log(Buffer.poolSize);

Результат выполненного кода:

10

Смотрите также

  • метод byteLength,
    который возвращает размер буфера
  • метод copyBytesFrom,
    который копирует базовую память в новый буфер
  • свойство byteOffset,
    которое возвращает смещение в байтах данного буфера
