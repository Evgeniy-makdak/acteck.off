Метод copyBytesFrom

Метод copyBytesFrom копирует базовую память в новый буфер. В первом параметре метода указываем объект TypedArray, который мы хотим скопировать. Во втором необязательном параметре можно задать начальное смещение копируемого объекта (по умолчанию равно 0), в третьем необязательном параметре - количество элементов для копирования.

Синтаксис

Buffer.copyBytesFrom(копируемый объект, [смещение], [количество элементов]);

Пример

Давайте скопируем память объекта в новый буфер:

import { Buffer } from 'buffer'; const arr = new Uint16Array([0,2]); const buffer = Buffer.copyBytesFrom(arr); console.log(buffer);

Результат выполненного кода:

<Buffer 00 00 02 00>

Смотрите также

  • свойство Buffer.copy,
    которое копирует данные из буфера в заданный объект
  • метод fill,
    который заполняет объект буфера данными
