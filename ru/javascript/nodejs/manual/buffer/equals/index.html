Метод equals

Метод equals сравнивает объекты буфера и возвращает булевое значение. Если два объекта буфера являются одинаковыми, то метод возвращает true, в противном случае - false. В параметре метода указываем буфер, который хотим сравнить с текущим буфером.

Синтаксис

буфер.equals(буфер для сравнения);

Пример

Давайте сравним два буфера:

import { Buffer } from 'buffer'; let buffer1 = Buffer.from('abc'); let buffer2 = Buffer.from('def'); console.log(buffer1.equals(buffer2));

Результат выполненного кода:

false

Смотрите также

  • метод compare,
    который сравнивает два объекта буфера
  • свойство Buffer.compare,
    которое сравнивает буфер с заданным объектом
