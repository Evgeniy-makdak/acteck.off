Метод allocUnsafe

Метод allocUnsafe создает новый незаполненный объект буфера указанного размера. Новый буфер может содержать информацию из ранее созданных буферов, поэтому он считается небезопасным. В параметре метода указываем нужный нам размер буфера в байтах.

Синтаксис

Buffer.allocUnsafe(размер буфера);

Пример

Давайте создадим новый объект буфера с помощью метода allocUnsafe:

import { Buffer } from 'buffer'; let buffer = Buffer.allocUnsafe(10); console.log(buffer);

Результат выполненного кода:

<Buffer 00 00 00 00 00 00 00 00 00 00>
