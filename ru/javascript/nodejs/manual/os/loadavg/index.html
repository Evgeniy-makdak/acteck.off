Метод loadavg

Метод loadavg возвращает средние значения нагрузки системы за 1, 5 и 15 минут. Среднее значение нагрузки рассчитывается операционной системой и выражается в виде дробного числа. Среднее значение нагрузки - это понятие, характерное только для операционной системы Unix. В Windows возвращаемое значение всегда равно [0, 0, 0].

Синтаксис

os.loadavg();

Пример

Давайте узнаем средние значения нагрузки нашей операционной системы:

import os from 'os'; console.log(os.loadavg());
