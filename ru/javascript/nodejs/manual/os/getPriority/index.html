Метод getPriority

Метод getPriority возвращает приоритет планирования для процесса, указанного pid - это необязательный параметр, указывающий идентификатор процесса, чей приоритет планирования должен быть возвращен. Его значение по умолчанию равно 0. Если в параметре метода значение pid не указано или равно 0, то метод getPriority возвращает приоритет текущего процесса.

Синтаксис

os.getPriority([pid]);

Пример

Давайте узнаем приоритет планирования для нашего процесса по умолчанию:

import os from 'os'; console.log(os.getPriority());

Смотрите также

  • метод release,
    который устанавливает приоритет планирования процесса
  • функция availableParallelism,
    которая возвращает количество параллелизма программы по умолчанию
