Свойство devNull

Свойство devNull возвращает путь к файлу нулевого устройства в зависимости от используемой платформы. Нулевое или пустое устройство - это специальный файл в системах класса UNIX, запись в который всегда происходит успешно, независимо от объема записанной информации. Для Windows значение devNull равно '\.\nul', для POSIX - '/dev/null'.

Синтаксис

os.devNull;

Пример

Давайте узнаем путь к файлу нулевого устройства используемой нами платформы:

import os from 'os'; console.log(os.devNull);
