Метод arch

Метод arch возвращает архитектуру процессора операционной системы, для которого скомпилирован текущий NodeJS файл. Возвращаемые значения могут быть следующими: 'x32', 'x64', 'arm', 'arm64', 's390', 's390x', 'mipsel', 'ia32', 'mips', 'ppc', 'ppc64'.

Синтаксис

os.arch();

Пример

Давайте узнаем архитектуру процессора нашей операционной системы:

import os from 'os'; console.log(os.arch());

Смотрите также

  • метод platform,
    который возвращает информацию о платформе операционной системы
  • метод release,
    который возвращает информацию о версии операционной системы
  • метод version,
    который возвращает информацию о версии ядра операционной системы
  • свойство machine,
    которое возвращает тип машины (ПК)
  • метод endianness,
    который возвращает эндианальность процессора
  • метод cpus,
    который возвращает информацию о логических ядрах процессора
