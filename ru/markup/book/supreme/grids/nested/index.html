Вложенные CSS гриды

В системе грид есть возможность добавить в свою же созданную сетку новый вложенный грид. Вложенные гриды широко используются при размещении небольших элементов внутри блоков сайта. Для того, чтобы воспользоваться этой возможностью, нужно в самом дочернем элементе поставить свойство display со значением grid.

Для примера давайте в одной из ячеек грида создадим еще один грид:

<div id="parent"> <div id="elem1">1</div> <div id="elem2">2</div> <div id="elem3">3</div> <div id="elem4"> <div>5</div> <div>6</div> <div>7</div> <div>8</div> </div> </div> #parent { display: grid; grid-template: 1fr 1fr / 1fr 1fr; grid-gap: 5px; border: 2px solid #696989; padding: 10px; width: 600px; height: 200px; } #parent > div { padding: 10px; border: 1px solid #696989; } #elem4 { display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: 1fr 1fr; grid-gap: 5px; border: 2px solid #696989; } #elem4 > div { grid-gap: 5px; padding: 6px; box-sizing: border-box; border: 1px solid #696989; }

:

Пусть у вас есть грид, состоящий из пяти элементов, расположенных по трем столбцам. Создайте во втором элементе вложенный грид, в котором в свою очередь будет три дочерних элемента.

Измените предыдущую задачу так, чтобы во вложенном гриде находилось пять дочерних элементов.

Создайте в первом и третьем элементах по вложенному гриду с тремя дочерними элементами.
