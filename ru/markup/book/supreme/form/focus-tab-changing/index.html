Смена фокуса табом в HTML

tabindex

Даны инпуты:

<input value="3"> <input value="4"> <input value="1"> <input value="2"> <input value="5">

Сделайте так, чтобы по нажатию Tab фокус по инпутам переходил по такому порядку, как указано в атрибутах value.
