Свойство-сокращение flex в CSS

Свойство flex представляет собой сокращение для свойств жадности, сжимаемости и размера. Порядок сокращаемых свойств имеет значение и он следующий: flex-grow, flex-shrink, flex-basis. Свойства можно опускать с конца, в этом случае они примут значение по умолчанию.

Перепишите следующий код через свойство-сокращение flex:

.child { flex-basis: 100px; flex-grow: 1; flex-shrink: 2; }

Перепишите следующий код через свойство-сокращение flex:

.child { flex-shrink: 0.5; flex-grow: 2; }

Перепишите следующий код через свойство-сокращение flex:

.child { flex-shrink: 0.5; flex-grow: 1; }

Перепишите следующий код через сокращаемые свойства:

.child { flex: 1 2 200px; }

Перепишите следующий код через сокращаемые свойства:

.child { flex: 2 0.7; }

Перепишите следующий код через сокращаемые свойства:

.child { flex: 3; }
