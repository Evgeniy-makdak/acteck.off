Влияние жадности элементов на сжатие flex блоков в CSS

Давайте теперь посмотрим влияние жадности. Уберем всем потомкам ширину и поставим им потомкам flex-grow в значение 1, а второму потомку - в значение 3. В результате получится, что второй блок будет в 3 раза больше остальных.

Однако, если сжать окно достаточно сильно, то второй блок уже не будет больше остальных в 3 раза - при сильном сжатии разница стирается. Убедитесь в этом сами, поигравшись с приведенным ниже примером, расширяя и сужая окно браузера:
