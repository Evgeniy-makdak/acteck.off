Объединение через пробел в LESS

Бывают свойства, значения которых нужно объединять не через запятую, а через пробел. Для таких объединений существует специальный оператор +_.

Давайте посмотрим на его работу на примере свойства transform. Сделаем функции .rotate и .skew, задающие соответствующие трансформации:

.rotate(@r) { transform+_: rotate(@r); } .skew(@s) { transform+_: skew(@s); }

Воспользуемся нашими функциями:

div { .rotate(10deg); .skew(10deg); }

В результате компиляции наши трансформации объединятся в одном свойстве transform:

p { transform: rotate(10deg) skew(10deg); }
