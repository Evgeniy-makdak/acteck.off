Ключевые слова CSS и переменные LESS

В CSS встречаются ключевые слова и команды, которые начинаются с собаки, например, @media, @keyframes и другие. LESS умеет корректно их обрабатывать и не воспринимает их как команды. Смотрите пример:

@media (max-width: 600px) { div { width: 300px; } }

Еще пример:

@keyframes anim { from { width: 100px; } to { width: 200px; } }

Напишите несколько медиазапросов. Убедитесь в их корректной работе.
