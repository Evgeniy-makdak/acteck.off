Функция calc в LESS

Давайте сделаем функцию, которая будет оберткой над CSS функцией calc. Пусть наша функция первым параметром принимает имя свойства, а вторым - вычисляемое значение:

.calc(@name, @calc) { @{name}: calc(@calc); }

Воспользуемся нашей функцией:

p { .calc(width, ~'10px + 30%'); }

Результат компиляции:

p { width: calc(10px + 30%); }

Расскажите, каким будет результат компиляции следующего кода:

p { .calc(width, ~'10px + 30%'); .calc(height, ~'10px + 30%'); }
