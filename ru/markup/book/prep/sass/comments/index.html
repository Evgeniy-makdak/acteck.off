Комментарии в SASS

В SASS существуют два типа комментариев: многострочные и однострочные. Давайте изучим их особенности.

Многострочные комментарии будут такие же, как и в CSS:

/* комментарий */ div { font-size: 20px; }

Или вот так:

div { font-size: 20px; /* комментарий */ }

SASS также добавляет однострочные комментарии. Эти комметарии удобнее тем, что их проще писать, а также они не попадают в скомпилированный код. Смотрите пример:

div { font-size: 20px; // комментарий }

Сделайте в вашем коде два типа комментариев. Скомпилируейте код. Посмотрите на результат компиляции.
