Расширение составным селектором в SASS

Применение директивы @extend не ограничивается только селекторами классов. Расширять можно в принципе любой селектор, даже одиночные элементы. Давайте рассмотрим это на примере:

Рассмотрим пример:

.active-link { @extend a:active; } a:active { color: blue; }

В результате компиляции мы увидим, что .active-link наследует свойство color у a:active:

a:active, .active-link { color: blue; }

Расскажите, каким будет результат компиляции следующего кода:

button { @extend .box:hover; } .box:hover { color: #14147e; }

Расскажите, каким будет результат компиляции следующего кода:

button:active { @extend input:focus; } button:blur { @extend input:blur; } input:focus { color: #14147e; border-color: #b5b5d1; } input:blur { color: #040433; background-color: #87878f; }
