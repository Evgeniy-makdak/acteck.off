Заглавные латинские буквы для списков ol в CSS

Если задать в свойстве list-style-type значение upper-alpha, то нумерация будет идти большими латинскими буквами:

<ol> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ol> ol { list-style-type: upper-alpha; }
