Расстояние между рядами в CSS гридах

Можно также задать расстояние между рядами сетки. Это делается с помощью свойства row-gap, которое задается в элементе-родителе.

Давайте зададим расстояние между рядами в гриде:

<div id="parent"> <div>1</div> <div>2</div> <div>3</div> </div> #parent { display: grid; row-gap: 10px; padding: 10px; border: 2px solid #696989; width: 600px; height: 200px; } #parent > div { padding: 10px; border: 1px solid #696989; }

:

Создайте таблицу, в которой расстояние между рядами составит 25%.
