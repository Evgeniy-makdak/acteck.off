Атрибут enctype

Атрибут enctype задает способ кодирования информации HTML формы.

Атрибут применяется к тегу form и не является обязательным.

Значения

Значение Описание
application/x-www-form-urlencoded Вместо пробелов ставится +, русские буквы кодируются их шестнадцатеричными значениями (например, %D0%9F%D1%80%D0%B8%D0%B2%D0%B5%D1%82 вместо "Привет"). Такая операция называется url кодирование. Это значение является значением по умолчанию.
multipart/form-data Данные не кодируются. Это значение следует устанавливать тегу form при отправке файлов. To есть, если в форме есть хотя бы один тег input с атрибутом type в значении file.
text/plain Пробелы заменяются знаком +, буквы и другие символы не кодируются.
