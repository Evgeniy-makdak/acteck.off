Атрибут title

Атрибут title задает всплывающую подсказку для элемента, которая будет появляться по наведению мышкой на элемент (нужно навести и немного подержать курсор, секунду-две).

К сожалению, применить какие-либо CSS стили к этой подсказке невозможно, а выглядит эта подсказка не очень (используйте JavaScript, чтобы сделать стилизованную подсказку).

Пример

Давайте попробуем навести мышкой на текст абзаца из примера - и увидим содержимое атрибута title в виде всплывающей подсказки:

<p title="it's a title">hover me!</p>

:

Смотрите также

  • атрибут placeholder,
    который задает подсказку для поля ввода
  • атрибут alt,
    который задает альтернативный текст картинке
