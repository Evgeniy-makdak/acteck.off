Атрибут alt

Атрибут alt альтернативный текст, который будет показан, если браузер не может найти изображение, указанное в атрибуте src тега img. Является обязательным атрибутом для тега img, при его отсутствии будет ругаться валидатор (программа, которая проверяет корректность HTML и CSS).

Пример

Давайте попробуем поставить неправильный путь к картинке (оставляем его пустым). Вместо картинки мы увидим содержимое атрибута alt (кажется, что это обычный текст - но попробуйте его скопировать - у вас ничего не получится, он будет тянутся как картинка):

<img src="" alt="Картинка веселой обезьянки">
