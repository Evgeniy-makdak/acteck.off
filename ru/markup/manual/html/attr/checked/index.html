Атрибут checked

Атрибут checked делает отмеченным по умолчанию флажок checkbox или переключатель radio.

Является атрибутом без значения.

Пример . Чекбоксы

В следующем примере сделаны два флажка checkbox: первый будет отмечен (так как ему дан атрибут checked), а второй - нет:

<input type="checkbox" checked> <input type="checkbox">

:

Пример . Радио переключатели

Давайте сделаем два радио переключателя - первый будет отмечен (так как ему зададим атрибут checked), а второй - нет:

<input type="radio" name="radio" checked> <input type="radio" name="radio">

:

Смотрите также

  • псевдокласс checked,
    который задает стили для отмеченных элементов
  • атрибут selected,
    который делает отмеченным пункт селекта
