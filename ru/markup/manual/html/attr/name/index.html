Атрибут name

Атрибут name задает имя для поля ввода HTML формы.

Полями формы служат теги input, textarea, select.

Это имя нужно для того, чтобы получить значение поля ввода через язык PHP после отправки формы.

Если задать одинаковые имена для радио переключателей, то они станут единой группой и только один из них сможет быть отмеченным.
