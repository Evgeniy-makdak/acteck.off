Тег i

Тег i задает курсив с помощью HTML.

Пример

Давайте зададим тексту курсив с помощью тега i:

это <i>курсивный</i> текст

:

Смотрите также

  • тег em,
    который создает важный курсивный текст
  • свойство font-style,
    с помощью которого также можно задать курсив
