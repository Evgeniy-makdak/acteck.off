Тег nav

Тег nav предназначен для создания основного меню на HTML странице:

<div id="wrapper"> <header> <nav> <a href="">item 1</a> <a href="">item 2</a> <a href="">item 3</a> </nav> </header> <main> ... </main> <aside> ... </aside> <footer> ... </footer> </div>

Смотрите также

  • тег header,
    который задает верхнюю часть сайта
  • тег footer,
    который задает нижнюю часть сайта
  • тег main,
    который задает основную часть сайта
  • тег aside,
    который задает побочную часть сайта
