Тег doctype

Тег doctype указывает браузеру на версию HTML. Чтобы указать текущую, пятую версию HTML следует написать так:

<!DOCTYPE html>

До выхода HTML5 приходилось писать страшные вещи, которые помнить наизусть было совершенно не реально. Например, такие:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

К счастью, в настоящее время доктайп сократили и такие длинные конструкции больше не нужны. Нет нужды не то что помнить их, но даже разбираться с тем, что они делали.

Тег doctype пишется в самом начале документа, перед тегом html. Тег является обязательным, однако страница будет работать и без него, но в этом случае некоторые вещи будут работать не так, как при наличии doctype. Поэтому всегда пишите его - избежите некоторых проблем.

Пример

Давайте посмотрим на структуру простейшей html страницы:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Это заголовок тайтл</title> </head> <body> Это основное содержимое страницы. </body> </html>

Смотрите также

  • тег html,
    который является главным тегом на странице
  • тег head,
    который содержит служебную информацию
  • тег body,
    который содержит основное содержимое
  • тег title,
    который содержит заголовок страницы
  • тег meta,
    который содержит meta данные
