Тег mark

Тег mark помечает текст как выделенный. Такой текст ничем не отличается от обычного, но его вид может быть изменен с помощью языка CSS.

В отличии от тегов strong и em, которые предназначены для указания важности текста, тег mark необходим для того, чтобы обозначить конкретные части текста, с которыми сейчас работает пользователь и которые важны для него в данный момент.

В браузерах Chrome и Firefox фоновый цвет текста внутри mark выделяется желтым фоном (это можно отменить с помощью CSS свойства background-color).
