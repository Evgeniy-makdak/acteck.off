Тег summary

Тег summary указывает заголовок для тега datalist, который создает спойлер на HTML. С помощью summary можно сворачивать и разворачивать содержимое тега datalist.

Пример

Давайте посмотрим, как работает тег details:

<details> <summary>click me</summary> hidden text </details>
