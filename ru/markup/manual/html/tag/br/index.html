Тег br

Тег br создает разрыв строки (это означает, что текст начнется с новой строки после этого тега). Не требует закрывающего тега.

Распространенная ошибка: использование этого тега для создания отступов между элементами. Для этого следует использовать свойство margin.

Пример

Давайте возьмем какой-то текст. Разобьем его на две строки тегом br:

первая строка <br> вторая строка

:

Смотрите также

  • свойство hyphens,
    которое включает переносы слов по слогам
  • тег wbr,
    который реализует мягкие переносы средствами HTML
  • свойство word-break,
    переносящее буквы длинного слова на новую строку
  • свойство overflow-wrap,
    переносящее буквы длинного слова на новую строку
