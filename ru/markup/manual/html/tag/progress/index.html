Тег progress

Тег progress используется для отображения прогресса завершенности задачи. Изменение значения происходит через JavaScript. Внутри тега указывать текст, который будет отображен в случае, если браузер не понимает этот тег.

Атрибуты

Атрибут Описание
value Текущее значение прогресса.
max Максимальное значение прогресса.

Значением атрибутов служат целые или дробные числа.

Пример

<progress max="100" value="25"> Загружено на 25% </progress>

:

Пример

<progress max="100" value="75"> Загружено на 75% </progress>

:

Пример

<progress max="100" value="100"> Загружено на 100% </progress>

:

Пример

<progress max="100" value="0"> Загружено на 0% </progress>

:

Пример . Дробные значения

<progress max="100" value="10.5"> Загружено на 10.5% </progress>
