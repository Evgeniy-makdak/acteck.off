Тег p

Тег p создает абзац. Абзацы, наряду с заголовками, составляются основную структуру текста в HTML странице.

Пример

Давайте сделаем несколько абзацев:

<p>Это абзац.</p> <p>Это абзац.</p> <p>Это абзац.</p>

:

Смотрите также

  • теги h1, h2, h3, h4, h5, h6,
    которые делают заголовки в тексте
  • тег div,
    который создает блок на странице
