Тег div

Тег div задает блок, который объединяет в себе другие теги. Сам по себе тег ничего не делает, но в сочетании с атрибутами class и id позволяет обратиться через язык CSS к группе объединенных тегов.

Пример

Давайте с помощью тега div сгруппируем несколько абзацев и применим к группе какие-нибудь стили:

<div id="elem"> <p>text</p> <p>text</p> <p>text</p> </div> #elem { color: red; padding: 10px; border: 1px solid black; }

:

Смотрите также

  • тег span,
    который позволяет выделить кусочек текста
  • тег section,
    который также группирует теги
