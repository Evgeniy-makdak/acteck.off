Тег strong

Тег strong задает важный текст, на котором хочет сделать акцент автор. По умолчанию текст, взятый в тег strong будет жирным, однако это поведение можно сменить с помощью CSS свойства font-weight.

Пример

Давайте сделаем жирный текст с помощью тега strong:

this is <strong>bold</strong> text

:

Смотрите также

  • тег b,
    который также создает жирный
  • тег em,
    который также делает важный текст
