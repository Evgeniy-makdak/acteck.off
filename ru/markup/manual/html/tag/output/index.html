Тег output

Тег output используется для вывода результата работы скрипта JavaScript в HTML.

Атрибуты

Атрибут Описание
for Определяет id одного и более элементов для связывания с тегом output.
form Задает имя формы, которой принадлежит область для вывода.
name Задает уникальное имя элемента.
