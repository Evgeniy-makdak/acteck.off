Тег canvas

Тег canvas задает доску для рисования на языке JavaScript. По умолчанию эта доска просто белый прямоугольник, но его можно стилизовать через CSS.

Пример

Давайте создадим доску для рисования и стилизуем ее:

<canvas></canvas> canvas { width: 300px; height: 200px; border: 1px solid black; }

:

    который задает изображение
  • свойство background-image,
    которое задает фоновую картинку
