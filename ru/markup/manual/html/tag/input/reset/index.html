Кнопка сброса полей формы

Тег input с атрибутом type в значении reset создает кнопку сброса полей HTML формы. После нажатия на такую кнопку поля формы приобретут тот вид, который они имели при заходе на страницу (до внесенных пользователем изменений).

Пример

Давайте изменим поля формы, а затем нажмем на кнопку сброса, чтобы вернуть их в изначальное состояние:

<form> <input type="checkbox" checked> <input type="text"> <input type="reset" value="кнопка сброса"> <input type="submit" value="кнопка отправки"> </form>

:

Смотрите также

  • кнопку button,
    также используемую для сброса формы
  • кнопку submit,
    используемую для отправки формы
