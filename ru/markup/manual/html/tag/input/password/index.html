Поле для ввода пароля

Поле для ввода пароля представляет собой input с атрибутом type в значении password. Такое поле отображает текст звездочками, чтобы его не могли подсмотреть через плечо и украсть пароль пользователя.

Пример

Давайте посмотрим, как работает инпут такого типа:

<input type="password" value="123456">
