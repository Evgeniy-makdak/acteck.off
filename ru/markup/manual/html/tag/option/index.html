Тег option

Тег option создает пункт выпадающего списка для тега select в HTML форме.

Атрибуты

Атрибут Описание
value Значение, которое будет передаваться на сервер при выборе данного пункта.
Необязательный атрибут. Если не указан - будет передаваться текст самого тега.
selected Определяет то, что данный пункт списка будет выбран по умолчанию.

Смотрите также

  • тег optgroup,
    который группирует пункты списка
  • тег datalist,
    который создает выпадающий список
