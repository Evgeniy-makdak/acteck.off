Тег rt

Тег rt добавляет аннотацию над текстом. Аннотация в данном случае - расшифровка текста под ней. Тег следует использовать внутри тега ruby (см. его для более полного понимания). Аннотация по умолчанию отображается меньшего размера, чем сам текст. Это можно исправить с помощью CSS свойства font-size.

Пример

Давайте сделаем аннотацию над текстом:

Тут будет просто <ruby>текст<rt>аннотация над текстом</rt></ruby> с аннотацией.
