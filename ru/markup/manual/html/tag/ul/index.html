Тег ul

Тег ul создает так называемый неупорядоченный список с маркерами в виде точек.

Тег ul следует использовать совместно с тегом li, который создает пункты списка (а ul создает сам список, см. пример ниже). Без тега li тег ul не употребляется.

Сменить или убрать совсем маркеры списка вы можете с помощью CSS свойства list-style-type.

Пример

Давайте посмотрим на примере, как выглядит список ul:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul>
