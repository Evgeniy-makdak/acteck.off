Тег ol

Тег ol создает так называемый упорядоченный список с маркерами в виде цифр.

Тег ol следует использовать совместно с тегом li, который создает пункты списка (а ol создает сам список, см. пример ниже). Без тега li тег ol не употребляется.

Преимущество тега ol в том, что он автоматически расставляет нумерацию. Если вы захотите вставить пару пунктов в середину списка - нумерация подстроится под вас.

Сменить или убрать совсем маркеры списка вы можете с помощью CSS свойства list-style-type.

Пример . Список ol

Давайте посмотрим, как выглядит список ol в браузере по умолчанию:

<ol> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ol>

:

Смотрите также

  • тег ul,
    который создает неупорядоченный список
  • свойство counter-increment,
    которое позволяет сделать нумерацию элементов
