Тег blockquote

Тег blockquote предназначен для выделения длинных цитат внутри документа. Тег можно использовать совместно с тегом cite, который нужен для указания ссылки на источник цитаты.

По умолчанию текст, взятый в тег blockquote имеет отступ слева. Его можно убрать с помощью CSS свойства margin.

Атрибуты

Атрибут Описание
cite Ссылка на источник цитаты (при необходимости, если такой источник есть).

Пример

Пример применения тега:

<p> Обычный текст. </p> <blockquote> Длинная длинная цитата. </blockquote> <p> Обычный текст. </p>
