Свойство word-spacing

Свойство word-spacing устанавливает интервал между словами. Значением свойства служат любые единицы для размеров, кроме процентов. Допустимо отрицательное значение. Значение normal позволяет браузеру вычислять интервал между словами самостоятельно (значение по умолчанию).

Если для текста задано выравнивание через text-align в значении justify, то интервал между словами будет установлен автоматически, но не меньше значения, указанного через word-spacing.

Синтаксис

селектор { word-spacing: значение; }

Пример

Давайте поставим расстояние между словами в 30px:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: 30px; text-align: left; }

:

Пример

Установим word-spacing в отрицательное значение - расстояние между словами практически исчезнет:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: -7px; }

:

Пример

Установка данного свойства в ноль не приводит к сливанию слов:

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: 0; }

:

Пример

Установим word-spacing в значение normal (это значение по умолчанию):

<p> Lorem ipsum dolor sit amet, consectetur adipiscing. </p> p { word-spacing: normal; }

:
