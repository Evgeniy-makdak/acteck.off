Свойство padding-inline-end

Свойство padding-inline-end задает конечный внутренний отступ элемента по строчной (горизонтальной) оси.

Синтаксис

селектор { padding-inline-end: размер; }

Пример

Давайте установим конечный внутренний отступ для дочернего элемента по строчной оси на 20px:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-end: 20px; width: 100px; height: 50px; border: 1px solid red; }

:

Пример

А теперь давайте установим конечный внутренний отступ для дочернего элемента по строчной оси на 20%:

<div> <p>text</p> </div> div { width: 400px; height: 200px; border: 1px solid black; } p { padding-inline-end: 20%; width: 100px; height: 50px; border: 1px solid red; }

:

Смотрите также

  • свойство padding-inline-start,
    которое задает начальный внутренний отступ элемента по строчной оси
  • свойство padding-block-end,
    которое задает конечный внутренний отступ элемента по блочной оси
  • свойство padding-block-start,
    которое задает начальный внутренний отступ элемента по блочной оси
  • свойство padding,
    которое задает внутренний отступ
  • свойство margin-bottom,
    которое задает внешний отступ снизу
  • свойство margin-left,
    которое задает внешний отступ слева
  • свойство margin-right,
    которое задает внешний отступ справа
  • свойство margin,
    которое является сокращением для внешнего отступа
