Свойство color

Свойство color устанавливает цвет тексту. Значением свойства служат любые единицы для цвета. Значение по умолчанию как правило черный цвет.

Синтаксис

селектор { color: цвет; }

Пример

Давайте зададим элементу красный цвет:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { color: red; }

:

Пример

Свойство color задает цвет не только тексту, но и маркерам списка:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> li { color: red; }

:

Пример

Свойство color задает также цвет границы (если он не задан в border-color):

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem { color: red; border-style: solid; border-width: 2px; }
