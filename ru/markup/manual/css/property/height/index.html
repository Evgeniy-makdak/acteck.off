Свойство height

Свойство height устанавливает высоту элемента. Значением свойства служат любые единицы для размеров либо ключевое свойство auto, предоставляющее браузеру самостоятельно вычислять размер элемента (поведение по умолчанию).

Синтаксис

селектор { height: значение; }

Пример

Установим блоку высоту в 100px:

<div id="elem"></div> #elem { border: 1px solid black; width: 300px; height: 100px; }

:

Пример

Установим блоку высоту в 200px:

<div id="elem"></div> #elem { border: 1px solid black; width: 300px; height: 200px; }

:

Смотрите также

  • свойство width,
    с помощью которого можно задать ширину элемента
  • атрибут height,
    который устанавливает высоту элементу
