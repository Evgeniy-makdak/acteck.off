Свойство border-top-style

Свойство border-top-style задает стиль верхней границы. Значением свойства служат такие же значения, как и в свойстве border-style.

Синтаксис

селектор { border-top-style: значение; }

Пример

Зададим верхнюю границу типа dashed:

<div id="elem"></div> #elem { border-top-style: dashed; border-top-width: 1px; border-top-color: black; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-top-color,
    которое задает цвет верхней границы
  • свойство border-top-width,
    которое задает толщину верхней границы
  • свойство border-top,
    которое задает границу для верхней стороны
  • свойство border-width,
    которое задает толщину всех сторон границы
