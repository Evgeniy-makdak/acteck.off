Свойство backdrop-filter

Свойство backdrop-filter позволяет применять фильтры (размытие, изменение контраста, цветовые эффекты) к области, находящейся позади элемента. Это создает эффект "стекла" или "морозного стекла".

Синтаксис

селектор { backdrop-filter: none | <фильтр> [<фильтр>]*; }

Значения

Значение Описание
none Фильтры не применяются (значение по умолчанию).
blur() Применяет эффект размытия (например: blur(5px)).
brightness() Изменяет яркость фона (например: brightness(0.5)).
contrast() Изменяет контрастность фона (например: contrast(200%)).
drop-shadow() Добавляет тень к фону.
grayscale() Преобразует фон в оттенки серого (например: grayscale(100%)).
hue-rotate() Изменяет цветовой тон фона (например: hue-rotate(90deg)).
invert() Инвертирует цвета фона (например: invert(100%)).
opacity() Изменяет прозрачность фона (например: opacity(50%)).
sepia() Применяет сепию к фону (например: sepia(100%)).
saturate() Изменяет насыщенность цветов фона (например: saturate(200%)).

Можно комбинировать несколько фильтров, перечисляя их через пробел.

Пример . Простое размытие

Базовый эффект размытия фона:

<div class="blur-example"> <div class="blur-box"> Backdrop blur effect </div> </div> .blur-example { background: linear-gradient(135deg, #667eea, #764ba2); padding: 40px; height: 200px; } .blur-box { backdrop-filter: blur(6px); background-color: rgba(255, 255, 255, 0.2); border-radius: 8px; padding: 20px; width: 80%; margin: 0 auto; text-align: center; color: white; font-size: 22px; }

:

Пример . Теплый светлый фильтр

Эффект теплого освещения с повышенной яркостью:

<div class="warm-light-bg"> <div class="warm-light"> Warm lighting effect </div> </div> .warm-light-bg { background: url('bg.png'); background-size: cover; padding: 50px; height: 250px; } .warm-light { backdrop-filter: brightness(1.2) hue-rotate(20deg); background-color: rgba(255, 235, 205, 0.3); padding: 25px; width: 70%; margin: 20px auto; text-align: center; color: #333; font-size: 24px; }

:

Пример . Размытие с градиентной маской

Сочетание размытия и полупрозрачного градиента:

Поддержка браузерами

Свойство поддерживается в большинстве современных браузеров, но может требовать префикс -webkit- для полной кросс-браузерной совместимости.

Смотрите также

  • свойство filter,
    которое применяет эффекты к самому элементу
  • свойство opacity,
    управляющее прозрачностью элемента
  • свойство background-blend-mode,
    определяющее режим наложения фоновых изображений
