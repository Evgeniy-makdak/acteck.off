Свойство text-orientation

Свойство text-orientation задает ориентацию символов в вертикальном тексте, т.е. к которому задано свойство writing-mode со значениями vertical-lr или vertical-rl. Свойство принимает два значения: mixed (символы поворачиваются на 90° по часовой стрелке) или upright (символы размещаются друг под другом).

Синтаксис

селектор { text-orientation: mixed или upright; }

Пример

Давайте зададим всем символам вертикального текста разворот на 90°:

<p> Text </p> body { width: 200px; height: 100px; } p { text-orientation: mixed; background: #467CBC; color: white; padding: 10px; font-size: 2rem; writing-mode: vertical-lr; }

:

Пример

Давайте разместим все символы вертикального текста друг под другом:

<p> Text </p> body { width: 200px; height: 100px; } p { text-orientation: upright; background: #467CBC; color: white; padding: 10px; font-size: 2rem; writing-mode: vertical-lr; }

:

Смотрите также

  • свойство writing-mode,
    которое задает направление тексту
  • свойство font-kerning,
    которое задает кернинг шрифту
