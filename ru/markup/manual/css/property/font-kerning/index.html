Свойство font-kerning

Свойство font-kerning задает кернинг или расстояние между буквами в слове. Свойство может принимать значения: auto (по умолчанию определяется браузером), normal (применяет кернинг) и none (не применяется кернинг).

Синтаксис

селектор { font-kerning: auto или normal или none }

Пример

Давайте зададим кернинг по умолчанию для нашего текста:

<h1>Header</h1> <div>text text text</div> h1 { font-size: 40px; font-kerning: auto; } div { font-size: 18px; }

:

Пример

А теперь давайте зададим кернинг:

<h1>Header</h1> <div>text text text</div> h1 { font-size: 40px; font-kerning: normal; } div { font-size: 18px; }

:

Пример

Давайте уберем кернинг для нашего текста:

<h1>Header</h1> <div>text text text</div> h1 { font-size: 40px; font-kerning: none; } div { font-size: 18px; }

:

Смотрите также

  • свойство text-orientation,
    которое задает ориентацию символам в вертикальном тексте
  • свойство writing-mode,
    которое задает направление тексту
