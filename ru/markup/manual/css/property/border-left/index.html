Свойство border-left

Свойство border-left задает цвет, тип и толщину для левой границы. Является свойством-сокращением для border-left-width, border-left-style и border-left-color.

Синтаксис

селектор { border-left: толщина тип цвет; }

Порядок свойств значения не имеет.

Пример

Давайте сделаем элементу черную левую границу толщиной в 2px:

<div id="elem"></div> #elem { border-left: 2px solid black; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-right,
    которое задает правую границу
  • свойство border-top,
    которое задает верхнюю границу
  • свойство border-bottom,
    которое задает нижнюю границу
  • свойство border-radius,
    которое скругляет уголки границы
