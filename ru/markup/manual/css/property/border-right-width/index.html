Свойство border-right-width

Свойство border-right-width задает толщину левой границы. Значением свойства служат любые единицы для размеров, кроме процентов, либо ключевые слова thin (граница в 2 пикселя), medium (граница в 4 пикселя) или thick (граница в 6 пикселей). Значение по умолчанию: medium.

Синтаксис

селектор { border-right-width: значение; }

Пример

Зададим правую границу толщиной в один пиксель:

<div id="elem"></div> #elem { border-right-width: 1px; border-right-style: solid; border-right-color: black; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-right-style,
    которое задает тип правой границы
  • свойство border-right-color,
    которое задает цвет правой границы
  • свойство border-right,
    которое задает границу для правой стороны
  • свойство border-width,
    которое задает толщину всех сторон границы
  • свойство border-left,
    которое задает левую границу
  • свойство border-left-color,
    которое задает цвет левой границы
  • свойство border-left-width,
    которое задает толщину левой границы
  • свойство border-left-style,
    которое задает стиль левой границы
