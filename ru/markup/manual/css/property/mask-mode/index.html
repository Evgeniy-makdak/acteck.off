Свойство mask-mode

Свойство mask-mode устанавливает способ взаимодействия маски с фоном элемента. Оно определяет, будет ли маска использовать альфа-канал (прозрачность) или значения яркости (luminance) изображения-маски.

Синтаксис

селектор { mask-mode: alpha | luminance | match-source; }

Значения

Значение Описание
alpha Маска использует только альфа-канал (прозрачность) изображения.
luminance Маска использует значения яркости изображения (белый = видимый, черный = прозрачный).
match-source Для изображений использует альфа-канал, для SVG - luminance (значение по умолчанию).

Пример . Режим alpha

Применение SVG-маски в режиме alpha:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask: url("heart.svg") center/150px no-repeat; mask-mode: alpha; }

:

Пример . Режим luminance

Применение SVG-маски в режиме luminance:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask: url("heart.svg") center/150px no-repeat; mask-mode: luminance; }

:

Пример . SVG маска с режимом match-source

Использование значения по умолчанию (match-source) для SVG:

<img id="image" src="image.jpg"> <svg width="0" height="0"> <mask id="svg-mask"> <path d="M150 15L183 111L285 111L204 171L237 267L150 216L63 267L96 171L15 111L117 111Z" fill="white"/> </mask> </svg> #image { width: 500px; height: 281px; mask: url(#svg-mask); mask-mode: match-source; }

:

Пример . Композиция с разными режимами

Комбинирование масок с разными режимами наложения:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; mask: url("heart.svg") 100px 50px / 150px no-repeat alpha, url("arrow.svg") 200px 30px / 150px no-repeat luminance; mask-composite: add; }

:

Смотрите также

  • свойство mask-image,
    задает изображение для маски
  • свойство mask-composite,
    определяет взаимодействие нескольких масок
  • свойство mask,
    сокращение для всех свойств маскирования
