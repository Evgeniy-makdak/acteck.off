Свойство border-left-color

Свойство border-left-color задает цвет левой границы. Значением свойства служат любые единицы для цвета. Значение по умолчанию: такой же цвет, как и у текста блока.

Синтаксис

селектор { border-left-color: цвет; }

Пример

Зададим левую границу красного цвета:

<div id="elem"></div> #elem { border-left-color: red; border-left-width: 1px; border-left-style: solid; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-left-style,
    которое задает тип левой границы
  • свойство border-right-style,
    которое задает стиль правой границы
  • свойство border-right-color,
    которое задает цвет правой границы
  • свойство border-left-width,
    которое задает толщину левой границы
  • свойство border-left,
    которое задает границу для левой стороны
  • свойство border-width,
    которое задает толщину всех сторон границы
