Свойство width

Свойство width устанавливает ширину элемента. Значением свойства служат любые единицы для размеров либо ключевое свойство auto, предоставляющее браузеру самостоятельно вычислять размер элемента (поведение по умолчанию).

Синтаксис

селектор { width: значение; }

Пример

Установим ширину элемента в 300px:

<div id="elem"></div> #elem { width: 300px; height: 100px; border: 1px solid black; }

:

Пример

Давайте сделаем так, чтобы элемент занял 70% от ширины своего родителя (в нашем случае это будет окно браузера):

<div id="elem"></div> #elem { width: 70%; height: 100px; border: 1px solid black; }

:

Смотрите также

  • свойство height,
    с помощью которого можно задать высоту элемента
  • атрибут width,
    с помощью которого можно задать ширину элемента
