Свойство list-style

Свойство list-style позволяет одновременно установить тип маркера list-style-type, картинку list-style-image и его положение list-style-position для списков ul или ol.

Порядок свойств значения не имеет.

Пример

Давайте установим для нашего списка квадратные маркеры:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> ul { list-style: square outside; }

:

Пример

А теперь давайте установим в свойстве list-style изображение для маркера и его расположение внутри текстового поля:

<ul> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> <li>list item</li> </ul> ul { list-style: url("li.gif") inside; }
