Свойство counter-reset

Свойство counter-reset сбрасывает значение счетчика в ноль. Данное свойство обязательно нужно использовать при работе с counter-increment, при этом оно устанавливается для родителя.

Синтаксис

селектор { counter-reset: название [начальное значение] | none; }

Значения

Значение Описание
название Обычное слово (как название класса или id). Можно задавать несколько названий, разделяя их пробелом. В этом случае одновременно будут меняться несколько счетчиков.
начальное значение Целое положительное или отрицательное число. Необязательный параметр.
none Запрещает увеличение счетчика для текущего селектора. Является значением по умолчанию.
