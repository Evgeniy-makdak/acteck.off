Свойство outline-width

Свойство outline-width задает толщину внешней рамки - границы, которая не занимает места. Значением свойства служат любые единицы для размеров, кроме процентов, либо ключевые слова thin (граница в 2 пикселя), medium (граница в 4 пикселя) или thick (граница в 6 пикселей). Значение по умолчанию: medium.

Синтаксис

селектор { outline-width: значение; }

Пример

Давайте зададим рамку элементу:

<div id="elem"></div> #elem { outline-width: 1px; outline-style: solid; outline-color: black; width: 300px; height: 100px; }

:

Смотрите также

  • свойство outline-color,
    которое задает цвет рамки
  • свойство outline-style,
    которое задает стиль рамки
  • свойство outline,
    которое является свойством сокращением для рамок
  • свойство outline-offset,
    которое задает смещение рамки
