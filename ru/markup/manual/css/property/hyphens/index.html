Свойство hyphens

Свойство hyphens включает переносы слов в браузере с помощью знака переноса "-". Для поддержки русского языка необходимо добавить атрибут lang="ru" к тегу, к которому применяются переносы (или к родителю этого тега). Можно просто добавить этот атрибут к тегу html. Без этого переносы работать не будут.

Синтаксис

селектор { hyphens: auto | manual | none; }

Значения

Значение Описание
auto Расставляет знаки переноса там, где это необходимо.
manual Добавляет переносы только в тех местах, где добавлен wbr тег или спецсимвол &shy; и только при необходимости. Такие переносы называются мягкими.
none Отменяет расстановку переносов (слова не будут переносится, даже если есть мягкие переносы).

Значение по умолчанию: manual.
