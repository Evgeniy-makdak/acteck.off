Свойство text-fill-color

Свойство text-fill-color property задает цвет заливки символов текста или их прозрачность с помощью значения transparent.

Синтаксис

селектор { text-fill-color: цвет или transparent; }

Пример

Давайте зададим нашему тексту градиентную заливку с прозрачностью:

<p> This is a text. </p> body { width: 200px; height: 100px; } p { -webkit-text-fill-color: transparent; text-fill-color: transparent; font-size: 26px; background: linear-gradient(#c42222, #ffef5e); -webkit-background-clip: text; background-clip: text; }

:

Смотрите также

  • свойство text-stroke,
    которое задает обводку текста
  • свойство text-stroke-color,
    которое задает цвет обводке текста
  • свойство text-stroke-width,
    которое задает толщину обводки текста
