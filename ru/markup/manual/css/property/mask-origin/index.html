Свойство mask-origin

Свойство mask-origin устанавливает точку отсчёта для позиционирования маски. Оно определяет, относительно какой части элемента (границы, отступы или содержимое) будет размещаться маска.

Синтаксис

селектор { mask-origin: content-box | padding-box | border-box; }

Значения

Значение Описание
border-box Маска позиционируется относительно границы элемента (включая border).
padding-box Маска позиционируется относительно padding области (по умолчанию).
content-box Маска позиционируется относительно содержимого элемента.

Пример . border-box

Маска позиционируется относительно границы элемента:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: border-box; }

:

Пример . padding-box

Маска позиционируется относительно области отступов:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: padding-box; }

:

Пример . content-box

Маска позиционируется относительно содержимого:

<img id="image" src="image.jpg"> #image { width: 500px; height: 281px; padding: 20px; border: 30px dashed rgba(0,0,0,0.3); mask-image: url("heart.svg"); mask-position: center; mask-size: 200px; mask-repeat: no-repeat; mask-origin: content-box; }

:

Смотрите также

  • свойство mask-clip,
    определяет область отсечения маски
  • свойство mask-position,
    устанавливает положение маски
  • свойство mask,
    сокращение для всех свойств маскирования
  • свойство background-origin,
    аналогичное свойство для фона элемента
