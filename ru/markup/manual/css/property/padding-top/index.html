Свойство padding-top

Свойство padding-top устанавливает внутренний верхний отступ элемента. Значением свойства служат любые единицы для размеров.

Синтаксис

селектор { padding-top: значение; }

Пример

Давайте сделаем внутренний верхний отступ в 30px:

<div id="elem"> some long text </div> #elem { padding-top: 30px; width: 300px; border: 1px solid black; text-align: justify; }

:

Смотрите также

  • свойство padding-bottom,
    которое задает внутренний отступ снизу
  • свойство padding-left,
    которое задает внутренний отступ слева
  • свойство padding-right,
    которое задает внутренний отступ справа
  • свойство padding,
    которое является сокращением для внутреннего отступа
  • свойство margin,
    которое задает внешний отступ
