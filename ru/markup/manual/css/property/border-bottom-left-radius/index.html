Свойство border-bottom-left-radius

Свойство border-bottom-left-radius создает скругление для нижнего левого угла границы и фона. Значением свойства служат любые единицы для размеров. Значение по умолчанию: 0.

Синтаксис

селектор { border-bottom-left-radius: значение; }

Пример

Установим скругление 10px для нижнего левого угла:

<div id="elem"></div> #elem { border-bottom-left-radius: 10px; border: 1px solid black; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-top-left-radius,
    которое задает скругление для верхнего левого угла
  • свойство border-top-right-radius,
    которое задает скругление для верхнего правого угла
  • свойство border-bottom-right-radius,
    которое задает скругление для нижнего правого угла
  • свойство border-radius,
    которое задает скругление одновременно для всех углов
  • свойство border,
    которое является свойством-сокращением для границы
