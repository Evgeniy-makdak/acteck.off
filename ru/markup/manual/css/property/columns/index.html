Свойство columns

Свойство columns задает одновременно ширину и количество колонок в многоколоночном тексте. Является свойством-сокращением для column-count и column-width. Значение по умолчанию: auto auto (?).

Синтаксис

селектор { columns: ширина количество; }

Порядок свойств значения не имеет. Одно из них может быть опущено.

Смотрите также

  • свойство column-gap,
    которое задает промежуток между колонками
  • свойство column-rule,
    которое задает границу между колонками
  • свойство column-span,
    которое задает количество столбцов, на которые должен растягиваться элемент
