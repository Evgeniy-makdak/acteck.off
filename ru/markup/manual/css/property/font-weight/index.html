Свойство font-weight

Свойство font-weight устанавливает либо отменяет жирный текст.

Синтаксис

селектор { font-weight: bold | normal; }

Значения

Значение Описание
bold Делает жирный текст.
normal Отменяет жирный. Обычно необходимо в случае, если вам необходимо сделать нежирным тег, который по умолчанию жирный (например, заголовки h1-h6 или th).

Значение по умолчанию: для каждого элемента свое - абзацы по умолчанию нежирные, а вот заголовки - жирные.

Пример

Зададим жирный шрифт для абзаца:

<p> Lorem ipsum dolor sit amet. </p> p { font-weight: bold; }

:

Пример

Отменим жирный шрифт для заголовка (у него он по умолчанию жирный):

<h2>Lorem ipsum</h2> h2 { font-weight: normal; }

:
