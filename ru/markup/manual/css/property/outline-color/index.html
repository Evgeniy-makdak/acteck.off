Свойство outline-color

Свойство outline-color задает цвет внешней рамки - границы, которая не занимает места. Значением свойства служат любые единицы для цвета. Значение по умолчанию: такой же цвет, как и у текста блока.

Синтаксис

селектор { outline-color: цвет; }

Пример

Давайте зададим рамку элементу:

<div id="elem"></div> #elem { outline-width: 1px; outline-style: solid; outline-color: red; width: 300px; height: 100px; }

:

Смотрите также

  • свойство outline-width,
    которое задает толщину рамки
  • свойство outline-style,
    которое задает стиль рамки
  • свойство outline,
    которое является свойством сокращением для рамок
  • свойство outline-offset,
    которое задает смещение рамки
