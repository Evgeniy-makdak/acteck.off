Свойство border-top

Свойство border-top задает цвет, тип и толщину для верхней границы. Является свойством-сокращением для border-top-width, border-top-style и border-top-color.

Синтаксис

селектор { border-top: толщина тип цвет; }

Порядок свойств значения не имеет.

Пример

Давайте сделаем элементу черную верхнюю границу толщиной в 2px:

<div id="elem"></div> #elem { border-top: 2px solid black; background: #f9f9f9; width: 300px; height: 100px; }
