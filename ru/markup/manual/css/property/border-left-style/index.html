Свойство border-left-style

Свойство border-left-style задает стиль левой границы. Значения свойства такие как в border-style.

Синтаксис

селектор { border-left-style: значение; }

Пример

Зададим левую границу типа dashed:

<div id="elem"></div> #elem { border-left-style: dashed; border-left-width: 1px; border-left-color: black; background: #f9f9f9; width: 300px; height: 100px; }

:

Смотрите также

  • свойство border-right-style,
    которое задает стиль правой границы
  • свойство border-right-color,
    которое задает цвет правой границы
  • свойство border-left-color,
    которое задает цвет левой границы
  • свойство border-right-width,
    которое задает толщину правой границы
  • свойство border-left-width,
    которое задает толщину левой границы
  • свойство border-left,
    которое задает границу для левой стороны
  • свойство border-right,
    которое задает правую границу
  • свойство border-width,
    которое задает толщину всех сторон границы
