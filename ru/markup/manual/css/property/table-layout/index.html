Свойство table-layout

В некоторых случаях браузер может уменьшать ячейки даже если им задана ширина. Свойство table-layout позволяет указать браузеру то, что ячейки таблицы всегда должны заданной ширины.

Синтаксис

селектор { table-layout: fixed | auto | inherit; }

Значения

Значение Описание
auto Браузер может изменять размеры ячеек.
fixed Браузер не будет менять размеры ячеек.
inherit Наследует значение родителя.

Значение по умолчанию: auto.
