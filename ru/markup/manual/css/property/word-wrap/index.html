Свойство word-wrap

Свойство word-wrap (устаревшее название), оно же overflow-wrap (новое название) позволяет перенести буквы длинного слова на новую строку, если это слово не влазит в ширину контейнера. Смотрите свойство overflow-wrap для получения нужной информации.
