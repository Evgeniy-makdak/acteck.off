Свойство shape-margin

Свойство shape-margin задаёт отступ вокруг формы, определённой свойством shape-outside. Это позволяет увеличить расстояние между формой и обтекающим её текстом.

Синтаксис

селектор { shape-margin: значение; }

Пример

Круг с отступом в 20px:

<div class="shape"></div> <p> some long text </p> .shape { width: 150px; height: 150px; float: left; shape-outside: circle(50%); shape-margin: 10px; clip-path: circle(50%); background: #3498db; } p { width: 500px; text-align: justify; }

:

Пример

Треугольник с отступом в 1em:

<div class="shape"></div> <p> some long text </p> .shape { width: 150px; height: 150px; float: left; shape-outside: polygon(0% 0%, 100% 50%, 0% 100%); shape-margin: 1em; clip-path: polygon(0% 0%, 100% 50%, 0% 100%); background: #e74c3c; } p { width: 500px; text-align: justify; }

:

Смотрите также

  • свойство shape-outside,
    которое определяет форму для обтекания
  • свойство clip-path,
    которое создаёт видимую форму элемента
  • свойство float,
    необходимое для работы shape-margin
