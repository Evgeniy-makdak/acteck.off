Псевдокласс target

Псевдокласс target задает то, как будет выглядеть целевой элемент. Целевой элемент - это тот, на id которого мы ссылаемся, указывая в конце урла его id после решетки # (пример: /css/target.html#my-id).

Синтаксис

селектор:target { }
