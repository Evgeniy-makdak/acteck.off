Псевдокласс enabled

Псевдокласс enabled задает стили для активного инпута. Под активным имеется ввиду инпут, которому не задан атрибут disabled.

Синтаксис

input:enabled { }

Пример

Применим стили для активного инпута:

<input> <input disabled> input:enabled { border: 1px solid green; box-shadow: 0 0 3px green; }

:

Смотрите также

  • псевдокласс disabled,
    который задает стили для неактивного инпута
