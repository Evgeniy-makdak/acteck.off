Псевдокласс hover

Псевдокласс hover задает стили для ссылки, на которую мы навели мышкой. Обычно этот псевдокласс применяется совместно с псевдоклассами link, visited и active.

Синтаксис

a:hover { }

Пример

Непосещенная ссылка будет красного цвета, посещенная - голубого. Если вы наведете на нее мышкой - она станет зеленого цвета, а если нажмете левую кнопку мыши и будете ее удерживать - станет черной (состояние active):

<a href="#">ссылка</a> a:link { color: red; } a:visited { color: blue; } a:hover { color: green; } a:active { color: black; }

:

Смотрите также

  • псевдокласс link,
    который задает стиль непосещенной ссылке
  • псевдокласс visited,
    который задает стиль посещенной ссылке
  • псевдокласс active,
    который задает стиль ссылке по активации
  • псевдокласс focus,
    который задает стиль ссылке в фокусе
