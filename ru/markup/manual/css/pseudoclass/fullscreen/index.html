Псевдокласс fullscreen

Псевдокласс fullscreen задает стиль элементам для полноэкранного режима браузера. Для того, чтобы перейти из стандартного режима отображения странички в полноэкранный и обратно нужно нажать клавишу F11 на клавиатуре.

Синтаксис

селектор:fullscreen { }

Пример

Давайте по клику развернем элемент с текстом в полноэкранном режиме и перекрасим ему фон:

<div id="txt"> Full screen </div> <p> <button onclick="fullScreen()">Click</button> </p> body { margin: 0; } #txt { padding: 10px; background: #467CBC; color: white; width: 200px; } #txt:-webkit-full-screen { background: #E37D76; } #txt:-moz-full-screen { background: #E37D76; } #txt:-ms-fullscreen { background: #E37D76; } function fullScreen() { let elem = document.getElementById('txt'); // Получаем элемент if (elem.webkitRequestFullscreen) elem.webkitRequestFullscreen(); // Chrome, Opera, Safari else if (elem.mozRequestFullScreen) elem.mozRequestFullScreen(); // Firefox else if (elem.msRequestFullscreen) elem.msRequestFullscreen(); // Internet Explorer, Edge else if (elem.requestFullscreen) elem.requestFullscreen(); }
