Псевдоэлемент first-letter

Псевдоэлемент first-letter позволяет задать стили первой букве текста в элементе.

Синтаксис

селектор::first-letter { }

Пример

Давайте применим стили к первым буквам каждого абзаца:

<p> Lorem ipsum dolor sit amet. </p> <p> Lorem ipsum dolor sit amet. </p> <p> Lorem ipsum dolor sit amet. </p> p::first-letter { color: red; font-size: 27px; }
