Псевдоэлемент placeholder

Псевдоэлемент placeholder задает стили для подсказки placeholder в тегах input или textarea.

Синтаксис

селектор::placeholder { color: red; }

Пример

Давайте покрасим подсказку в красный цвет:

<input type="text" placeholder="подсказка"> input::placeholder { color: red; }

:

Пример . Подсказка в textarea

Покрасим подсказку в красный цвет в теге textarea:

<textarea placeholder="подсказка"></textarea> textarea::placeholder { color: red; }

:

Смотрите также

  • атрибут placeholder,
    который задает подсказку для поля ввода
  • псевдокласс placeholder-shown,
    который задает задает стили полю ввода с подсказкой
