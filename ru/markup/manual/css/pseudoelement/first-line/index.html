Псевдоэлемент first-line

Псевдоэлемент first-line позволяет задать стили первой линии текста.

Синтаксис

селектор::first-line { }

Пример

Покрасим первую линию текста каждого абзаца в красный цвет:

<p> some long text </p> <p> some long text </p> p::first-line { color: red; } p { width: 300px; text-align: justify; }

:

Пример

Поставим ширину абзацев в процентах. Запустите приведенный код и поизменяйте ширину окна браузера:

<p> some long text </p> <p> some long text </p> p::first-line { color: red; } p { width: 50%; margin: 0 auto; text-align: justify; }

:
