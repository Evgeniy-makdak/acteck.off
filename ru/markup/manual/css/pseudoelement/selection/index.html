Псевдоэлемент selection

Псевдоэлемент selection позволяет задать цвет и фон выделенному тексту.

Синтаксис

селектор::selection { }

Пример

Выделите текст и вы увидите, что он будет красный на черном фоне:

<div id="elem"> Lorem ipsum dolor sit amet. </div> #elem::selection { color: red; background-color: black; }

:

Смотрите также

  • свойство user-select,
    которое запрещает выделение текста
  • свойство pointer-events,
    которое убирает реакцию на мышь
