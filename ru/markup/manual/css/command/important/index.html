Команда !important

Команда !important задает приоритет стилю или любому CSS-свойству, для которого она указана.

Синтаксис

селектор { свойство: значение !important; }

Пример

Давайте сделаем синий цвет шрифта абзаца приоритетным по отношению к зеленому цвету:

<div>text</div> div{ color: blue !important; width: 600px; height: 100px; font-size: 20px; } div { color: green; }

:

Смотрите также

  • команда @import,
    которая импортирует CSS-файл
  • команда @media,
    которая задает стиль для типа носителя
  • команда @supports,
    которая задает стиль элементам, возможности которых поддерживаются браузером
