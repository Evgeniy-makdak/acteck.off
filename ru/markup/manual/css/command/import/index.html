Команда import

Команда @import предназначена для импорта CSS-файла в текущую стилевую таблицу. В данной команде можно указывать для каких типов носителей предназначен импорт: all (все типы, используется по умолчанию), print (устройства для печати), screen (экран монитора).

Синтаксис

import url('имя файла') [типы носителей]; @import 'имя файла' [типы носителей];

Смотрите также

  • команда @media,
    которая задает стиль для типа носителя
  • команда @supports,
    которая задает стиль элементам, возможности которых поддерживаются браузером
  • команда !important,
    которая задает приоритет стиля
