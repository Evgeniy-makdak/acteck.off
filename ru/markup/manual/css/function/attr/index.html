Функция attr

Функция attr позволяет получить в CSS значение атрибута HTML тега.

Синтаксис

селектор { свойство: attr(имя, тип, значение); }

Параметры

Параметр Описание
имя Имя атрибута элемента передаваемое в CSS.
тип Тип значения свойства CSS. Допустимы следующие ключевые слова: string (строки), color (цвет), url (веб-адрес), integer (целое число), number (вещественное число), angle (угол), time (время), deg (градус), а также единицы CSS вроде em, px и др.
значение Значение, которое будет использоваться в том случае, если атрибута нет или браузер не может его получить.

Пример

Давайте после текста ссылки выведем значение ее атрибута href в круглых скобках:

<a href="http://google.com">ссылка</a> a::after { content: ' (' attr(href) ')'; }

:

Смотрите также

  • свойство content,
    которое позволяет добавлять текст в элементы
  • псевдоэлемент before,
    который делает вставку после текста тега
  • псевдоэлемент after,
    который делает вставку перед текстом тега
