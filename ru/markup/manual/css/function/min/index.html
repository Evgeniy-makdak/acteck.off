Функция min

Функция min позволяет выбрать минимальное из переданных значений. Функция в качестве параметров принимает два и более значений, разделенных запятыми, и возвращает минимальное из них.

Синтаксис

селектор { свойство: min(значение1, значение2); }

Пример

Сейчас ширина блока будет иметь значение или 300px или 50% от родителя, в зависимости от того, какое из значений меньше:

<div id="parent"> <div id="child"></div> </div> #child { width: min(300px, 50%); height: 100px; margin: 0 auto; border: 1px solid red; } #parent { border: 1px solid green; }
