Функция clamp

Функция clamp принимает три параметра: минимальное значение, предпочтительное значение и максимальное значение. Она возвращает:

  • минимальное значение, если предпочтительное меньше минимального
  • максимальное значение, если предпочтительное больше максимального
  • предпочтительное значение, если оно находится между минимальным и максимальным

Синтаксис

селектор { свойство: clamp(min, pref, max); }

Пример . Ограничение размера блока

В следующем примере предпочтительный размер блока будет 30% от ширины экрана. Но меньше 100px и больше 300px он стать не сможет:

<div id="con1"></div> <div id="targ"></div> <div id="con2"></div> #targ { width: clamp(100px, 30%, 300px); height: 100px; border: 1px solid red; margin: 30px auto; } #con1 { width: 100px; height: 100px; border: 1px solid black; margin: 30px auto; } #con2 { width: 300px; height: 100px; border: 1px solid black; margin: 30px auto; }

:

Пример . Адаптивный размер шрифта

В следующем примере предпочтительный размер шрифта будет 4vw. Но меньше 16px и больше 32px он стать не сможет:

<p id="con1"> responsive text example </p> <p id="elem"> responsive text example </p> <p id="con2"> responsive text example </p> #elem { font-size: clamp(16px, 4vw, 32px); color: red; } #con1 { font-size: 16px; } #con2 { font-size: 32px; }

:

Пример . Адаптивный радиус скругления

В следующем примере предпочтительный радиус скругления будет 3vw. Но меньше 5px и больше 20px он стать не сможет:

<div id="con1"></div> <div id="elem"></div> <div id="con2"></div> div { width: 100px; height: 100px; margin: 20px auto; background: lightblue; } #elem { border-radius: clamp(5px, 3vw, 20px); background: #efafc6; } #con1 { border-radius: 5px; } #con2 { border-radius: 20px; }

:
