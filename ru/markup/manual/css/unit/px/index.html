Единица px

Абсолютная единица измерения px представляет собой ячейку на матрице электронного устройства. К примеру, если ширина экрана смартфона равна 480px, это значит, что на экране расположено 480 ячеек, каждая из которых равна 1px.

Пример

Давайте зададим прямоугольнику ширину в 200px и высоту 100px:

<div> </div> div { border: 1px solid black; width: 200px; height: 100px; }

:

Пример

Давайте зададим тексту абзаца шрифт в 30 пикселей:

<p> text </p> p { font-size: 30px; }

:
