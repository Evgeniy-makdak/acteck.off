Единица em

Относительная единица измерения em устанавливает размер шрифта дочернему элементу, ориентируясь на размер его родителя. К примеру, шрифт, равный 2em означает, что он больше своего родителя в два раза.

Пример

Давайте зададим тексту элемента <p> шрифт в полтора раза больше, чем шрифт его родителя <div>:

<div> text1 <p> text2 </p> </div> div { font-size: 14px; } p { font-size: 1.5em; }

:
