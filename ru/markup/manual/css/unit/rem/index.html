Единица rem

Относительная единица измерения rem устанавливает размер шрифта элементу в зависимости от шрифта корневого элемента - <html>. К примеру, шрифт, равный 2rem означает, что он больше корневого элемента в два раза.

Пример

Давайте зададим тексту заголовка шрифт в полтора раза больше, чем шрифт корневого элемента, а абзацу - 1rem:

<h1>Text1</h1> <p>text2</p> body{ font-size: 14px; } h1 { font-size: 1.5rem; } p { font-size: 1em; }
