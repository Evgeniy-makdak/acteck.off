Встроенные стили в NextJS

В NextJS доступна стилизация через атрибут style. Такие стили называются встроенными. Работа с ними осуществляется также, как и работа со встроенными стилями в React.

Для примера покрасим абзац в красный цвет:

export default function Test() { return <p style=> text text text </p>; }

Примените к какому-нибудь тегу встроенный стиль.
