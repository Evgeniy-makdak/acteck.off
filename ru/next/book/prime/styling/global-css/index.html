Глобальный CSS в NextJS

Внутри папки app по умолчанию есть файл globals.css с глобальным CSS стилями нашего проекта.

Глобальные стили можно импортировать в любой макет, страницу или компонент. Без импорта они не применятся.

Давайте импортируем файл с глобальными стилями в основной макет сайта:

import '@/app/globals.css';

Поменяйте фон вашей страницы.

В файле globals.css задайте какой-нибудь CSS класс. Примените его в каком-нибудь компоненте.
