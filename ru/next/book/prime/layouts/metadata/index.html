Метаданные в NextJS

Давайте теперь посмотрим, как добавить на страницу тайтл и метаописание. В NextJS работа с ними осуществляется совместно и поэтому их называют одним общим словом метаданные.

Метаданные задаются с помощью специального объекта, содержащего тайтл и метаописание:

{ title: '...', description: '...', };

При этом NextJS сам выполнит вставку метаданных в соответствующие теги. Нам не нужно заботиться об этом. Поэтому в основном макете сайта отсутствует тег head, так он будет сгенерирован автоматически:

export default function RootLayout({children}) { return ( <html lang="en"> <body> {children} </body> </html> ); }

Существуют различные способы задать метаданные. Их мы будем изучать в следующих уроках.
