Макет по умолчанию в NextJS

В предыдущем уроке мы выяснили, что у нас должен быть общий макет сайта, в который вставляется контент, тайтл и мета-описание страницы. В NextJS по умолчанию уже есть файл с общим макетом сайта. Он находится в файле app/layout.jsx.

После установки в файле с макетом уже есть некоторый код. Этот код, однако, содежит много лишнего. Давайте сделаем минимальный макет сайта, в котором легко будет разобраться. Для этого откройте файл с макетом, удалите из него все и вставьте следующее:

export const metadata = { title: 'Test site', description: 'This is my page description.', }; export default function RootLayout({children}) { return ( <html lang="en"> <body> {children} </body> </html> ); }

Исправьте содержимое по умолчанию в основном макете сайта.
