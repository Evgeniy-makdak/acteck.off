Блокировка элементов в Vue

Можно реактивно блокировать элементы формы с помощью атрибута disabled. Давайте попробуем. Сделаем соответствующее свойство:

data() { return { isDisabled: true, } }

Привяжем это свойство к кнопке:

<template> <button v-bind:disabled="isDisabled">btn</button> </template>

Дан инпут. Сделайте кнопку, нажатие на которую будет по очереди заблокировать и отблокировать наш инпут.

Модифицируйте предыдущую задачу так, чтобы состояние инпута регулировалось чекбоксом: если чекбокс отмечен, то инпут отблокирован, а если не отмечен - то инпут заблокирован.
