Испускание события в атрибуте в Vue

Можно испустить событие прямо в атрибуте обработчика:

<template> <button @click="$emit('show')"> btn </button> </template>

Можно при этом передать параметры:

<template> <button @click="$emit('show', 'xxx', 'yyy')"> btn </button> </template>

Сделайте кнопку, по клику на которую будет испускаться некоторое событие.
