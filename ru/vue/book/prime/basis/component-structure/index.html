Структура компонента Vue

Каждый компонент должен состоять из двух следующих тегов:

<script> </script> <template> </template>

В теге template следует писать верстку, то есть представление нашего компонента:

<template> <div> Hello World </div> </template>

А в теге script следует писать JavaScript код нашего компонента. Этот код должен представлять собой объект, который будет экспортироваться из нашего файла компонента:

<script> export default { } </script>

В экспортируемом объекте указываются различные настройки и функции компонента. Одной из настроек является имя этого компонента:

export default { name: 'App', }

В теге template вашего компонента выведите какой-нибудь текст с тегами.
