Работа с методами в фреймворке Vue

Во Vue можно создавать свои функции, которые в терминах Vue называются методами. Для этого предназначено свойство methods. Оно располагается следующим образом:

export default { data() { }, methods: { } }

Далее для краткости я буду писать в сокращенном виде, вот так:

methods: { }

Давайте сделаем какой-нибудь тестовый метод:

methods: { show: function() { alert('!'); } }

Вызовем этот метод в представлении. В этом случае он сработает сразу по загрузки страницы:

<template> {{ show() }} </template>
