Выполнения кода в представлениях Vue

В фигурных скобках можно выполнять различный JavaScript код. Давайте рассмотрим это на примере. Пусть у нас есть два свойства с числами - var1 и var2:

data() { return { var1: 1, var2: 2, } }

Давайте, например, сложим значения наших свойств:

<template> {{ var1 + var2 }} </template>

Результатом этого кода будет следующий HTML:

<div id="app"> 3 </div>

Пусть в data хранится следующее:

data() { return { num: 5, } }

Выведите на экран квадрат свойства num.

Пусть в data хранится следующее:

data() { return { num1: 1, num2: 2, num3: 3, } }

Выведите на экран сумму свойств num1, num2 и num3.
