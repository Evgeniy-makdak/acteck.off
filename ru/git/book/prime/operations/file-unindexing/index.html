Отмена индексации файла в Git

Для отмены индексации файла в системе Git используется команда restore, с флагом --staged, после которого добавляется имя файла. Давайте отменим индексацию файла:

git restore --staged file.txt

После проверки статуса в терминале появится сообщение о том, что наш файл изменен, но при этом не проиндексирован:

Changes not staged for commit: modified: file.txt

Проиндексируйте файл в вашем Git репозитории. Затем отмените его индексацию.
