Удаление внешнего репозитория в Git

Можно удалить внешний репозиторий. Toчнее, мы можем удалить связь между именем репозитория и его ссылкой. Сам репозиторий по-прежнему останется в интернете.

Для того, чтобы удалить имя внешнего репозитория, используется команда remote rm, в которой через пробел указывается имя внешнего репозитория:

git remote rm origin

Давайте проверим, есть ли у нас удаленный репозиторий:

git remote

После выполнения команды мы увидим лишь пустую строку, что означает у нас нет внешнего репозитория, в том числе и отслеживаемых веток и всех настроек, которые мы делали для него.

Удалите имя своего внешнего репозитория.
