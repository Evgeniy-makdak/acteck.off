Имена точек входа в Webpack

Toчкам входа можно давать имена. Для этого настройку entry сделать в виде объекта. Ключи объекта и будут именами точек входа. Именованные точки входа будут собираться в отдельные файлы сборки.

Давайте попробуем. В следующем примере в результате после сборки в папке dist у нас появятся два файла: file1.js и file2.js.

export default { entry: { file1: './src/test1.js', file2: './src/test2.js' }, };

Сделайте три точки входа. Соберите их в отдельные файлы.
