Toчка выхода в Webpack

Toчка выхода регулируется настройкой output. Эта настройка представляет собой объект. В этой настройке можно указать имя файла сборки с помощью ключа filename:

export default { entry: './src/index.js', output: { filename: 'build.js', } };

В результате сборка по-прежнему будет размещаться в папке dist, но уже в указанном нами файле.

Сделайте так, чтобы ваша сборка собиралась в файл bundle.js.
