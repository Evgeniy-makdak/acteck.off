Путь к папке сборки в Webpack

Можно задавать абсолютный путь к папке сборки. Как правило это делается для того, чтобы обеспечить правильную работу продвинутым возможностям Webpack. Давайте зададим этот путь.

Для начала импортируем модуль path:

import path from 'path';

Теперь укажем путь к папке сборке с помощью настройки path:

export default { entry: [ './src/test1.js', './src/test2.js' ], output: { filename: 'build.js', path: path.resolve('dist'), } };

Укажите путь к вашей папке сборки.

Поменяйте папку сборки с dist на другую.
