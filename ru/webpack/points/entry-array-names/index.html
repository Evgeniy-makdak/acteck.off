Именованные точки-массивы в Webpack

Именованные точки входа могут содержать массивы точек входа. Массив точек входа будет слит в один файл сборки. В следующем примере получится два файла: file1.js и file2.js:

export default { entry: { file1: [ './src/test1.js', './src/test2.js', ], file2: './src/test3.js' }, };

Расскажите, как будут работать следующие настройки и что получится в результате:

export default { entry: { bund1: [ './src/test1.js', './src/test2.js', ], bund2: [ './src/test3.js', './src/test4.js', ], bund3: [ './src/test5.js', './src/test6.js', ], }, };
