Несколько точек входа в Webpack

Можно сделать несколько точек входа. Для этого настройку entry следует сделать массивом:

export default { entry: [ './src/test1.js', './src/test2.js' ], };

Все точки входа будут собраны в один общий файл сборки. При необходимости можно задать имя этому файлу:

export default { entry: [ './src/test1.js', './src/test2.js' ], output: { filename: 'build.js', } };

Сделайте три точки входа в вашем проекте. Пусть к каждой точке подключаются несколько модулей из других файлов.

Соберите проект в файл bundle.js.
