Подключение CSS библиотек в Webpack

Можно устанавливать через npm CSS библиотеки и подключать их к CSS файлам. Давайте посмотрим на примере библиотеки normalize.css. Установим ее:

npm install normalize.css

Импортируем библиотеку прямо в CSS файл. При этом в начале пути следует поставить тильду ~. Она скажет Webpack, что файлы необходимо искать в node_modules. Выполним импорт:

@import '~normalize.css'

Установите и импортируйте библиотеку modern-css-reset.

Соберите проект. Посмотрите, куда попадает установленная библиотека при сборке.
