Настройка импорта CSS стилей в Webpack

После установки лоадеров мы можем воспользоваться ими в файле конфигурации. Сделаем это:

export default { entry: './src/index.js', module: { rules: [ { test: /\.css$/i, use: ['style-loader', 'css-loader'], }, ], }, };

Выполните соответствующие настройки.
