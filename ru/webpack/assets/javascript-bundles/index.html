Ассеты в JavaScript бандлах в Webpack

Давайте настроим ассеты для .png файлов:

export default { entry: './src/index.js', output: { filename: 'bundle.js', path: path.resolve('dist') }, module: { rules: [ { test: /\.png$/, type: 'asset/resource' } ] }, plugins: [ new HtmlWebpackPlugin(), ], };

Подключим .png файл к точке входа:

import img from './images/main.png';

В переменную img попадет новый путь к файлу картинки:

console.log(img);

Настройте ассеты для .jpg файлов.

Подключите к точке входа несколько картинок.

Выведите в файле точки входа новые пути к картинкам.
