Абсолютные пути в Webpack

Для работы Webpack нужен встроенный в NodeJS модуль path. Давайте импортируем его:

import path from 'path';

От этого модуля нам нужен метод resolve. Этот метод параметрами принимает относительный путь и делает из него абсолютный от корня операционной системы

let test = path.resolve('src'); console.log(test); // абсолютный-путь/src

Можно передавать несколько путей через запятую. В этом случае они будут объединены через слеш:

let test = path.resolve('src', 'images'); console.log(test); // абсолютный-путь/src/images

Посмотрите, что будет результатом выполения данного кода:

let test = path.resolve('dist'); console.log(test);

Посмотрите, что будет результатом выполения данного кода:

let test = path.resolve('dist', 'assets'); console.log(test);
