Плагины в Webpack

Webpack выполняет узкий спектр задач. Для расширения возможностей нужно устанавливать и подключать дополнительные плагины.

Давайте рассмотрим общий алгоритм установки и подключения плагинов. Пусть для этого у нас есть некий воображаемый плагин test-webpack-plugin. Давайте рассмотрим работу с плагинами на его примере (код ниже - просто схема, так как реально такого плагина нет).

Итак, для начала плагин необходимо установить через npm:

npm install test-webpack-plugin --save-dev

После этого плагин нужно импортировать к файлу конфигурации:

import TestWebpackPlugin from 'test-webpack-plugin';

Затем этот плагин нужно вызвать в настройке plugins:

export default { entry: './src/index.js', plugins: [new TestWebpackPlugin()], };

В настройке plugins может быть не один плагин, а несколько:

export default { entry: './src/index.js', plugins: [ new TestWebpackPlugin1(), new TestWebpackPlugin2() ], };
