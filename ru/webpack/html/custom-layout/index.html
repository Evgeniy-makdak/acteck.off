Свой файл макета в Webpack

На практике обычно у нас уже есть свой HTML макет сайта. В этом случае нам нужно, чтобы Webpack брал наш макет, а не генерировал свой. Давайте посмотрим, как это делается.

Пусть у нас есть следующий макет:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>My layout</title> </head> <body> </body> </html>

Заставим Webpack взять наш макет. Для этого пропишем его в следующей настройке:

export default { entry: './src/index.js', plugins: [ new HtmlWebpackPlugin({ template: './src/index.html', }), ], };

В результате после сборки в папке dist будет наш макет и в нем автоматически будут подключены все бандлы:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>My layout</title> <script defer src="main.js"></script> </head> <body> </body> </html>

Сделайте свой макет. Настройте Webpack так, чтобы он брал ваш макет.
