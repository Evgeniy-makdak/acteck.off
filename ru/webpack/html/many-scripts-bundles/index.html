Несколько бандлов скриптов в макете в Webpack

Если в результате сборки у нас получится несколько бандлов, то Webpack подключит каждый из них. Давайте проверим. Пусть у нас есть следующие настройки:

export default { context: path.resolve( 'src'), entry: { test1: './test1.js', test2: './test2.js' }, output: { filename: '[name].build.js', path: path.resolve('dist'), }, plugins: [ new HtmlWebpackPlugin(), ], };

После сборки к файлу макета будут подключены оба наших бандла:

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Webpack App</title> <script defer src="test1.build.js"></script> <script defer src="test2.build.js"></script> </head> <body> </body> </html>

Сделайте три точки входа. Выполните сборку. Изучите файл макета и убедитесь, что там подключены все ваши бандлы.
