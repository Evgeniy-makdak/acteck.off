Смена режима сборки в файле конфигурации в Webpack

Режим сборки регулируется в файле конфигурации с помощью настройки mode. Давайте укажем режим разработки:

export default { mode: 'development', entry: './src/index.js', };

А теперь укажем продакшн режим:

export default { mode: 'production', entry: './src/index.js', };

Соберите проект в режиме разработки. Откройте в редакторе файл бандла. Изучите его. Запомните размер этого файла.

Соберите проект в продакшн режиме. Откройте в редакторе файл бандла. Изучите его. Сравните размер файла с размером сборки в режиме разработки.
