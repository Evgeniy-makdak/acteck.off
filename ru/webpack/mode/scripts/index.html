Режимы сборки через scripts в Webpack

Можно задавать режим сборки не файле конфигурации, а через настройку scripts файла package.json. Сейчас в этом файле у нас расположена команда build, которая собирает проект в том режиме, который указан в файле конфигурации:

"scripts": { "build": "webpack" }

Давайте добавим команду, собирающую проект в режиме разработки:

"dev": "webpack --mode development"

Давайте также добавим команду, собирающую проект в продакшн режиме:

"prod": "webpack --mode production"

В итоге наш файл package.json станет выглядеть следующим образом:

"scripts": { "build": "webpack", "dev": "webpack --mode development", "prod": "webpack --mode production", }

Теперь в терминале мы сможем различными командами вызывать сборку в различных режимах. Для сборки в режиме разработки мы напишем следующее:

npm run dev

А для сборки в продакшн режиме - следующее:

npm run prod

Внесите в ваш package.json соответствующие настройки.
