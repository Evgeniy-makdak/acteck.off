Глоссарий

Словарь основных терминов и определений, используемых в веб-разработке и программировании.

API

Application Programming Interface — набор правил и протоколов для взаимодействия между различными программными компонентами.

AJAX

Asynchronous JavaScript and XML — технология асинхронного обмена данными между браузером и сервером без перезагрузки страницы.

Бандлер (Bundler)

Инструмент, объединяющий множество файлов (JavaScript, CSS, изображения) в один или несколько бандлов для оптимизации загрузки.

Виртуальный DOM

Легковесное представление реального DOM-дерева в памяти, используемое React и другими фреймворками для оптимизации обновлений интерфейса.

Генератор статических сайтов (SSG)

Инструмент, генерирующий HTML-страницы на этапе сборки, что обеспечивает быструю загрузку и хорошую SEO-оптимизацию.

Дженерики (Generics)

Возможность создавать компоненты, функции и классы, работающие с различными типами данных без потери типизации.

Деплой (Deploy)

Процесс развертывания приложения на сервере или хостинге для обеспечения доступа пользователей.

Замыкание (Closure)

Функция, которая запоминает свое лексическое окружение даже после того, как внешняя функция завершила свое выполнение.

Интерфейс (Interface)

В TypeScript — способ определения структуры объекта с указанием типов его свойств и методов.

Компонент (Component)

Переиспользуемый элемент пользовательского интерфейса, инкапсулирующий структуру, стили и логику.

Лоадер (Loader)

В Webpack — модуль, преобразующий файлы определенных типов (CSS, изображения, TypeScript) перед добавлением в бандл.

Миксин (Mixin)

Шаблон проектирования, позволяющий комбинировать функциональность из нескольких источников в одном объекте или классе.

Плагин (Plugin)

Программный модуль, расширяющий функциональность основного приложения или инструмента.

Прокси (Proxy)

Объект-посредник, перехватывающий операции над другим объектом и позволяющий реализовать пользовательское поведение.

Реактивность (Reactivity)

Механизм автоматического обновления пользовательского интерфейса при изменении данных, используемый в Vue и других фреймворках.

Роутинг (Routing)

Механизм навигации между различными страницами или представлениями в одностраничном приложении (SPA).

Серверный рендеринг (SSR)

Технология генерации HTML-кода страницы на сервере перед отправкой клиенту, улучшающая SEO и скорость загрузки.

Состояние (State)

Данные, определяющие текущее состояние компонента или приложения в определенный момент времени.

Транспиляция (Transpilation)

Процесс преобразования исходного кода из одного языка программирования в другой (например, TypeScript в JavaScript).

Хук (Hook)

Функция в React, позволяющая использовать состояние и другие возможности React в функциональных компонентах.
