Серверная сторона

Серверная сторона - все программы и операции, выполняемые на стороне сервера, который отсылает данные по запросу клиента или пользователя.

Для программирования приложений на серверной стороне есть специально предназначенные языки такие как PHP, Python, NodeJS.

Наиболее популярными веб-серверами в последнее время являются:

  • Apache представляет собой программное обеспечение, работающее разных операционных системах. Архитектура Apache состоит из ядра и модулей, представляющих собой серверные функции, которые выполняются по каждому запросу. Также модули можно дополнительно сконфигурировать для управления динамическим контентом или обработки HTTP-запросов. Ранее Apache являлся самым популярным веб-сервером, но из-за увеличения уровня трафика работа с настройкой данного веб-сервера значительно усложнилась.
  • Nginx появился позже Apache в связи с потребностью устранить основные недостатки в работе последнего. Каждый рабочий процесс в Nginx обеспечивает одновременную обработку тысяч HTTP-соединений. В связи с чем данный веб-сервер обеспечивает стабильную работу при больших и флуктуирующих нагрузках на данные.
