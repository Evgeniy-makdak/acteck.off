Система Git

Система Git представляет собой систему управления версиями рабочего проекта. Git имеет распределенную архитектуру, что подразумевает размещение файлов и репозиториев проекта в различных ветках, при этом основной стабильный код находится в главной ветке проекта, а работа над новыми изменениями и их тестированием идет в побочных, т.н. тематических ветках. При каждом сохранении изменений в файлах, система Git делает своего рода их снимок, который хранится во временной папке.
