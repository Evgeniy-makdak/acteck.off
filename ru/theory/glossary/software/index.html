Что такое ПО

ПО или программное обеспечение представляет собой совокупность программ, обеспечивающих функционирование различных приложений, установленных на компьютере или другом устройстве (гаджеты, смартфоны, планшеты и прочее).

Виды ПО по назначению:

  • Системное обеспечивает функционирование компьютерной системы. Наиболее характерный пример системного ПО - операционные системы Windows, Linux и macOS. Также к системным ПО относят прошивки смартфонов, умных часов и т.п. гаджетов.
  • Инструментальное предназначено для создания других программ. К инструментальным ПО относят компиляторы и интерпретаторы языков программирования, среды программирования, редакторы кода, фреймворки и библиотеки.
  • Прикладное представляют собой программы, с которыми пользователь наиболее часто взаимодействует в обычной жизни, например, текстовый редактор блокнот. Примерами прикладного ПО являются различные приложения на смартфоне, браузеры и прочее.
