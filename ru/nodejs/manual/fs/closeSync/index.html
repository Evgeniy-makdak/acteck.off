Метод closeSync

Метод closeSync используется только для синхронного закрытия файлового дескриптора, тем самым очищая связанный с ним файл. Файловый дескриптор - это идентификатор потока ввода-вывода файла, который представляет собой неотрицательное целое число.

Синтаксис

fs.closeSync(файловый дескриптор);

Пример

Давайте синхронно закроем файловый дескриптор. Для этого сначала с помощью метода openSync определим файловый дескриптор нужного нам файла:

import fs from 'fs'; let fileDescriptor = fs.openSync('file.txt'); try { fs.closeSync(fileDescriptor); console.log('file closed successfully'); } catch (err) { console.log(err); }
