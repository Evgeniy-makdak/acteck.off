Свойство byteOffset

Свойство byteOffset возвращает смещение в байтах данного буфера.

Синтаксис

буфер.byteOffset;

Пример

Давайте узнаем смещение в байтах нашего объекта буфера:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer.byteOffset);

Результат выполненного кода:

16

Смотрите также

  • свойство poolSize,
    которое определяет размер выделенных экземпляров буфера
  • метод byteLength,
    который возвращает размер буфера
