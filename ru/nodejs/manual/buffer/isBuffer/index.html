Метод isBuffer

Метод isBuffer проверяет является ли объект буфером. В параметре метода указываем нужный нам объект. Метод возвращает значение true, если объект является буфером, в противном случае - false.

Синтаксис

Buffer.isBuffer(объект для проверки);

Пример

Давайте проверим является ли наш объект буфером:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(4); console.log(Buffer.isBuffer(buffer));

Результат выполненного кода:

true

Пример

А теперь давайте проверим является ли строка буфером:

import { Buffer } from 'buffer'; let buffer = 'abcde'; console.log(Buffer.isBuffer(buffer));

Результат выполненного кода:

false
