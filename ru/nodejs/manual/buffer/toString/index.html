Метод toString

Метод toString преобразует объект буфера в строчное значение. В первом необязательном параметре метода можно указать кодировку (по умолчанию она равна 'utf8'). Во втором необязательном параметре можно задать индекс, с которого будет начинаться преобразование (по умолчанию он равен 0). Во третьем необязательном параметре можно указать индекс, на котором будет заканчиваться преобразование (по умолчанию он равен длине объекта буфера).

Синтаксис

буфер.toString();

Пример

Давайте преобразуем наш объект буфера в строку:

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcde'); console.log(buffer.toString());

Результат выполненного кода:

'abcde'
