Метод includes

Метод includes проверяет есть ли в объекте буфера значение, указанное в первом параметре. Во втором необязательном параметре можно задать индекс байтов, с которого начнется поиск значения, в третьем необязательном параметре - кодировку (по умолчанию она равна 'utf8'). Метод возвращает true, если значение есть в объекте буфера, в противном случае - false.

Синтаксис

буфер.includes(значение, [начало поиска], [кодировка]);

Пример

Давайте поищем есть ли в нашем объекте буфера строка 'abc':

import { Buffer } from 'buffer'; let buffer = Buffer.from('abcdef'); console.log(buffer.includes('abc'));

Результат выполненного кода:

true

Смотрите также

  • метод indexOf,
    который ищет первый индекс заданного значения в объекте буфера
  • метод length,
    который возвращает количество байтов в объекте буфера
