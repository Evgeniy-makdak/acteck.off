Метод isEncoding

Метод isEncoding проверяет, поддерживается ли данная кодировка. В параметре метода указываем кодировку. Метод возвращает значение true, если кодировка поддерживается, в противном случае - false.

Синтаксис

Buffer.isEncoding(кодировка);

Пример

Давайте проверим с помощью метода isEncoding, поддерживается ли кодировка 'utf8':

import { Buffer } from 'buffer'; console.log(Buffer.isEncoding('utf8'));

Результат выполненного кода:

true

Пример

А теперь давайте проверим, поддерживается ли кодировка 'asciivalue':

import { Buffer } from 'buffer'; console.log(Buffer.isEncoding('asciivalue'));

Результат выполненного кода:

false
