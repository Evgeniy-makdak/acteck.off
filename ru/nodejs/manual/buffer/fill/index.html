Метод fill

Метод fill заполняет объект буфера данными, указанными в первом параметре метода. Во втором необязательном параметре можно задать индекс байтов, с которого начнется заполнение объекта буфера, в третьем необязательном параметре - индекс, на котором закончится заполнение. В четвертом необязательном параметре можно задать кодировку (по умолчанию она равна 'utf8').

Синтаксис

буфер.fill(данные, [начало заполнения], [конец заполнения], [кодировка]);

Пример

Давайте заполним буфер данными:

import { Buffer } from 'buffer'; let buffer = Buffer.alloc(4); buffer.fill('a'); console.log(buffer.toString());

Результат выполненного кода:

'aaaa'

    который записывает строку в объект буфера
  • метод includes,
    который проверяет есть ли в объекте буфера данное значение
