Метод dirname

Метод dirname возвращает имя папки из указанного в параметре пути.

Синтаксис

path.dirname(путь);

Пример

Давайте узнаем имя папки из пути:

import path from 'path'; console.log(path.dirname('./dir/file.txt'));

Результат выполненного кода:

'./dir'

Смотрите также

  • метод basename,
    который возвращает имя файла из пути
