Метод parse

Метод parse преобразует путь из строчного значения в объект со следующими ключами: dir - имя папки, root - корневой путь, base - файл с расширением, name - имя фала, ext - расширение. В параметр метода передаем путь, синтаксис которого отличается в зависимости от используемой платформы. Например, для POSIX путь записывается как '/dir/file.txt', а для Windows - '\\dir\\file.txt'.

Синтаксис

path.parse(путь к файлу);

Пример

Давайте преобразуем путь из строчного значения в объект на платформе Windows:

import path from 'path'; console.log(path.parse('./dir/file.txt'));

Результат выполненного кода:

{ root: '\\', dir: '\\dir', base: 'file.txt', ext: '.txt', name: 'file' }
