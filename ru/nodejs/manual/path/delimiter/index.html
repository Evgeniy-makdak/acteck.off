Свойство delimiter

Свойство delimiter возвращает разделитель пути для текущей платформы в виде строкового значения. Для for Windows свойство вернет ';', для POSIX - ':'.

Синтаксис

path.delimiter;

Пример

Давайте определим разделитель пути для нашей платформы:

import path from 'path'; console.log(path.delimiter);

Смотрите также

  • свойство sep,
    которое возвращает разделитель сегментов пути
