Свойство sep

Свойство sep возвращает разделитель сегментов пути или слеш для текущей платформы в виде строкового значения. Для Windows свойство вернет '\', для POSIX - '/'.

Синтаксис

path.sep;

Пример

Давайте определим разделитель сегментов для нашей платформы Windows:

import path from 'path'; console.log(path.sep);

Результат выполненного кода:

'\'

Пример

А теперь давайте посмотрим какой разделитель зарезервирован для POSIX:

import path from 'path'; console.log(path.sep);

Результат выполненного кода:

'/'
