Метод join

Метод join объединяет все сегменты пути, используя разделитель, характерный для конкретной платформы. После объединения нормализует полученный путь. Все сегменты, указываемые в параметре метода должны иметь строковые значения.

Синтаксис

path.join(сегменты пути);

Пример

Давайте объединим сегменты в один общий путь:

import path from 'path'; console.log(path.join('/dir', 'file.txt'));

Результат выполненного кода:

'\dir\file.txt'

Смотрите также

  • метод normalize,
    который нормализует путь
  • метод resolve,
    который возвращает абсолютный путь из сегментов
