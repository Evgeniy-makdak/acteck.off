Метод networkInterfaces

Метод networkInterfaces возвращает информацию о сетевых интерфейсах компьютера в виде объекта. Возвращаемый объект содержит следующие ключи: address, netmask, family, mac, internal, scopeid, cidr.

Синтаксис

os.networkInterfaces();

Пример

Давайте узнаем информацию о сетевых интерфейсах нашего компьютера:

import os from 'os'; console.log(os.networkInterfaces());

    который возвращает информацию о версии операционной системы
