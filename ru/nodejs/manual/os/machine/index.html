Свойство machine

Свойство machine возвращает в виде строчного значения тип машины или ПК пользователя. Возможные значения включают 'arm', 'arm64', 'aarch64', 'mips', 'ppc64', 'ppc64le', 's390', 's390x', 'i386', 'i686', 'x86_64'.

Синтаксис

os.machine();

Пример

Давайте узнаем тип используемой нами машины:

import os from 'os'; console.log(os.machine());
