Метод setPriority

Метод setPriority устанавливает приоритет планирования процесса, указанного pid и приоритетом планирования. В первом необязательном параметре метода можно задать pid, который, указывает идентификатор процесса, чей приоритет планирования нас интересует. Его значение по умолчанию равно 0. Во втором параметре задаем приоритет для процесса, значение которого варьируется от самого высокого показателя -20 до самого низкого - 19.

Синтаксис

os.setPriority([pid], приоритет процесса);

Пример

Давайте установим самый высокий приоритет планирования для нашего процесса и с помощью метода getPriority проверим его значение:

import os from 'os'; os.setPriority(-20); console.log(os.getPriority());
