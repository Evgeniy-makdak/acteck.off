Функция availableParallelism

Функция availableParallelism возвращает количество параллелизма программы по умолчанию. Возвращаемое значение всегда является положительным числом.

Синтаксис

os.availableParallelism();

Пример

Давайте узнаем количество параллелизма программы по умолчанию:

import os from 'os'; console.log(os.availableParallelism());
