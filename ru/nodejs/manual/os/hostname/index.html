Метод hostname

Метод hostname возвращает имя хоста операционной системы в виде строки.

Синтаксис

os.hostname();

Пример

Давайте узнаем имя хоста нашей операционной системы:

import os from 'os'; console.log(os.hostname());

Смотрите также

  • метод networkInterfaces,
    который возвращает информацию о сетевых интерфейсах компьютера
  • свойство machine,
    которое возвращает тип машины (ПК)
  • метод type,
    который возвращает имя операционной системы
