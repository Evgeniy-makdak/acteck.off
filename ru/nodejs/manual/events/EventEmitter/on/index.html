Метод on

Метод on модуля EventEmitter добавляет слушателей для события, указанного в первом параметре метода. Во втором параметре метода задаем слушателя, который является коллбэком. Новый слушатель добавляется в конец массива слушателей.

Синтаксис

eventEmitter.on(название события, слушатель);

Пример

Давайте создадим слушателя:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener() { console.log('text'); }); console.log(eventEmitter.listenerCount('event'));

Смотрите также

  • метод addListener,
    который добавляет слушателя для события
  • метод prependListener,
    который добавляет слушателя для события в начало массива слушателей
  • метод once,
    который однократно добавляет слушателя события в конец массива
  • метод prependOnceListener,
    который однократно добавляет слушателя события в начало массива
  • метод removeAllListeners,
    который удаляет всех слушателей для события
  • метод removeListener,
    который удаляет одного слушателя для события
