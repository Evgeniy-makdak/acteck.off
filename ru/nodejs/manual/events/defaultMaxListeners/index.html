Свойство defaultMaxListeners

Свойство defaultMaxListeners возвращает текущее максимальное количество слушателей по умолчанию.

Синтаксис

events.defaultMaxListeners;

Пример

Давайте узнаем максимальное количество слушателей для объекта EventEmitter по умолчанию:

import { EventEmitter } from 'events'; console.log(EventEmitter.defaultMaxListeners);

Результат выполненного кода:

10

Смотрите также

  • метод getMaxListeners,
    который возвращает максимальное количество слушателей
  • метод setMaxListeners,
    который изменяет максимальное количество слушателей события
