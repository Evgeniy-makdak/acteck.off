Свойство errorMonitor

Свойство errorMonitor отслеживает ошибки слушателей и возвращает Symbol.

Синтаксис

events.errorMonitor;

Пример

Давайте узнаем есть ли у нас ошибки в коде:

import { EventEmitter } from 'events'; const eventEmitter = new EventEmitter(); eventEmitter.on('event', function listener() { console.log('text'); });

Результат выполненного кода:

Symbol(events.errorMonitor)
