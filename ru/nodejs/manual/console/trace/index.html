Метод trace

Метод trace в модуле console возвращает трассировку стека выполняемого кода в консоль. В первом необязательном параметре указываем сообщение, во втором необязательном параметре - значения, которые можно подставить в сообщение из первого параметра.

Синтаксис

console.trace([сообщение], [значение для сообщения]);

Пример

Давайте узнаем трассировку стека при выполнении кода:

function func1(){ function func2() { console.trace(); }; func2(); }; func1();

Результат выполненного кода:

'Trace at func2 (file://путь к функции, включая номер строки редактора кода) at func1 (file://путь к функции, включая номер строки редактора кода) at file://путь к файлу, в котором выполняется код at ModuleJob.run (node:internal/modules/esm/module_job:192:25)'
