Метод groupCollapsed

Метод groupCollapsed в модуле console сворачивает группы сообщений в консоли, а также запускает их. В необязательном параметре метода можно указать метку для группировки сообщений. Работа метода аналогична методу group.

Синтаксис

console.groupCollapsed([метка]);

Пример

Давайте свернем группы сообщений в консоли:

console.log('ab'); console.log('cd'); console.groupCollapsed(); console.log('cd');

Результат выполнения кода:

'ab' 'cd' 'cd'

Смотрите также

  • метод group,
    который группирует сообщения на консоли
  • метод groupEnd,
    который завершает группировку сообщений на консоли
