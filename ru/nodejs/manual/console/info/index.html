Метод info

Метод info в модуле console выводит сообщения в консоль. В параметре метода можно указывать любые типы значений. Работа метода аналогична методу log.

Синтаксис

console.info(значения);

Пример

Давайте выведем в консоль строку 'abcde':

console.info('abcde');

Результат выполненного кода:

'abcde';
