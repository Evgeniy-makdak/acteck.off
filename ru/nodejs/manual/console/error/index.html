Метод error

Метод error в модуле console сообщение об ошибке в консоли. В параметре метода можно указывать любые значения.

Синтаксис

console.error(значение);

Пример

Давайте узнаем выведем сообщение об ошибке в консоль:

let num = 1; if (num > 1) { console.log('number is correct'); } else { console.error('error'); }

Результат выполнения кода:

'error'
