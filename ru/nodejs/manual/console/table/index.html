Метод table

Метод table в модуле console выводит таблицу с заданными значениями в консоль. В первом параметре метода указываются нужные нам данные, во втором необязательном параметре - массив названий столбцов в таблице.

Синтаксис

console.table(значения, [названия столбцов]);

Пример

Давайте выведем в консоль таблицу с заданными в виде массива значениями:

console.table(['ab', 'cd', 'ef']);

Результат выполненного кода:

┌─────────┬────────┐ │ (index) │ Values │ ├─────────┼────────┤ │ 0 │ 'ab' │ │ 1 │ 'cd' │ │ 2 │ 'ef' │ └─────────┴────────┘
