Метод warn

Метод warn в модуле console возвращает предупреждающие сообщения или сообщения об ошибке в консоли. В параметре метода можно указывать любые значения. Работа метода аналогична методу error.

Синтаксис

console.warn(значение);

Пример

Давайте узнаем выведем сообщение об ошибке в консоль:

let num = 1; if (num > 1) { console.log('number is correct'); } else { console.warn('error'); }

Результат выполнения кода:

'error'
