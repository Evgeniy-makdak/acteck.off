Метод timeLog

Метод timeLog в модуле console возвращает время выполнения фрагментом кода или функции. Для запуска таймера используется метод time. В необязательном параметре можно указать метку с именем кода или функции, которая нас интересует. Возвращаемое значение измеряется в миллисекундах.

Синтаксис

console.timeLog([метка]);

Пример

Давайте узнаем сколько времени займет выполнение функции getSum, но вначале нам нужно запустить таймер с помощью метода time:

function getSum(num1, num2){ return num1 + num2; }; console.time(); getSum(1, 2); console.timeLog();

Результат выполненного кода:

default: 0.101ms

Пример

А теперь давайте в параметре методов time и timeLog укажем имя функции, время выполнения которой, мы хотим узнать:

function getSum(num1, num2){ return num1 + num2; }; function getMultiply(num1, num2){ return num1 * num2; }; console.time(getMultiply); getSum(1, 2); console.timeLog(getMultiply);

Результат выполненного кода:

function getMultiply(num1, num2){ return num1 * num2; }: 0.074ms
