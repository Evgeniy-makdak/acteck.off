Метод dirxml

Метод dirxml в модуле console возвращает иерархическую структуру элемента XML/HTML, указанного в параметре метода. В полученных данных отображается содержимое и дочерних узлов заданного элемента.

Синтаксис

console.dirxml(элемент XML/HTML);

Пример

Давайте создадим абзац с дочерними элементами - параграфами:

<div> <p>1</p> <p>2</p> <p>3</p> </div>

Теперь выведем в консоль нашего браузера структуру элемента div:

console.dirxml(document.querySelector('div'));

Результат выполненного кода:

<div> <p>1</p> <p>2</p> <p>3</p> </div>
