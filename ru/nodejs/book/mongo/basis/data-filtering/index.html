Фильтрация данных по условию в MongoDB

Вы уже умеете получать все записи из коллекции:

let res = await coll.find().toArray(); console.log(res);

Давайте теперь отфильтруем данные по некоторому условию. Например, получим всех юзеров, у которых зарплата равна 500:

let res = await coll.find({salary: 500}).toArray(); console.log(res);

А теперь получим всех юзеров, у которых возраст равен 26:

let res = await coll.find({age: 26}).toArray(); console.log(res);

А теперь получим юзеров, для которых одновременно выполняются оба условия:

let res = await coll.find({salary: 500, age: 26}).toArray(); console.log(res);

Из тестовой таблицы с продуктами получите все продукты, у которых остаток равен 10.

Из тестовой таблицы с продуктами получите все продукты, у которых цена равна 100, а остаток равен 10.
