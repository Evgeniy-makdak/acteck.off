Получение одного документа в MongoDB

Можно получить не массив документов, а один документ. Для этого используется метод findOne. Давайте с его помощью получим первого юзера с зарплатой 400:

let res = await coll.findOne({salary: 400}); console.log(res);

Если вызвать метод без фильтра, то он вернет первый документ в коллекции:

let res = await coll.findOne(); console.log(res);
