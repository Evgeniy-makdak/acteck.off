Оформление всех документов из БД

Давайте теперь сделаем маршрут, который выведет всех юзеров из коллекции:

app.get('/users', function(req, res) { });

Получим юзеров из базы и отправим их в представление:

app.get('/users', function(req, res) { let users = await coll.find().toArray(); res.render('users', {users: users}); });

В представлении выведем наших юзеров, например, в виде таблицы:

<table> {{#each users}} <tr> <td>{{name}}</td> <td>{{age}}</td> <td>{{salary}}</td> </tr> {{/each}} </table>

Дана следующая верстка:

<div> <h2>prod1</h2> <p> <b>100$</b>, rest: 10 </p> </div> <div> <h2>prod2</h2> <p> <b>100$</b>, rest: 20 </p> </div> <div> <h2>prod3</h2> <p> <b>300$</b>, rest: 30 </p> </div>

Выведите записи из коллекции с продуктами, оформив их в приведенную верстку.
