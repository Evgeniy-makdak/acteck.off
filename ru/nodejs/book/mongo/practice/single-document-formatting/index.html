Оформление одного документа из БД

Давайте теперь будем отправлять в браузер данные не просто в виде строки, а оформив их в нужные нам теги. Для этого отрендерим соответствующий hbs файл, передав в него данные юзера:

app.get('/user/show/:name', function(req, res) { let name = req.params.name; let user = await coll.findOne({name: name}); res.render('user', {user: user}); });

В файле представления оформим данные юзера удобным нам образом:

<h1>{{ user.name }}</h1> <div> salary: {{ user.salary }} <br> age: {{ user.age }} </div>

Выведите данные продукта, оформив их в какую-нибудь верстку.

Сделайте так, чтобы, если запрошенный продукт не найден, то рендерился файл с 404 ошибкой.
