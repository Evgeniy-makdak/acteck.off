Элемент radio в Express

Давайте теперь научимся работать с radio. Сделаем несколько переключателей в нашей форме:

<form action="" method="POST"> <input type="radio" name="radio" value="1"> <input type="radio" name="radio" value="2"> <input type="radio" name="radio" value="3"> <input type="submit"> </form>

После отправки формы в req.body переключателя будет содержаться значение атрибута value отмеченного переключателя, либо undefined, если ничего не было отмечено:

app.post('/', function(req, res) { console.log(req.body.radio); // '1', '2', '3' или undefined });

С помощью двух переключателей спросите у пользователя его пол. Выведите результат на экран.
