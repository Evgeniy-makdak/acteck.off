Отправка формы методом POST на текущую страницу в Express

Давайте теперь будем отправлять нашу форму методом POST на текущую страницу:

<form method="POST"> <input name="test1"> <input name="test2"> <input type="submit"> </form>

В данном случае, в отличие от метода GET, нам придется написать два обработчика. Первый обработчик будет методом get и он будет срабатывать по заходу на страницу с формой:

app.get('/', function(req, res) { res.render('form'); });

А второй обработчик будет методом POST и он будет срабатывать после отправки формы:

app.post('/', function(req, res) { res.send('result'); });

Дана форма, отправляемая методом POST. Сделайте так, чтобы форма и ее результат были на одном URL.
