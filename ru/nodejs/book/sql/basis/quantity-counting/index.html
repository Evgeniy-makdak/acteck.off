Подсчет количества через SQL запрос в NodeJS

С помощью команды COUNT можно подсчитать количество строк в выборке. Символ звездочки, указанный в параметре команды, означает, что выбираются все записи из таблицы.

Пример

Давайте, например, подсчитаем всех юзеров в таблице:

let query = "SELECT COUNT(*) FROM users";

Результат выполненного кода:

[ { 'COUNT(*)': 6 } ]

Пример

А теперь подсчитаем всех, у кого зарплата равна 900:

let query = "SELECT COUNT(*) FROM users WHERE salary=900";

Результат выполненного кода:

[ { 'COUNT(*)': 2 } ]

Пример

Также можно задать в параметре команды COUNT имя поля. И команда подсчитает все записи, у которых заданное поле не равно NULL:

let query = "SELECT COUNT(salary) FROM users";

Результат выполненного кода:

[ { 'COUNT(salary)': 6 } ]

Практические задачи

Подсчитайте всех юзеров, чей возраст менее 30.

Подсчитайте всех юзеров с зарплатой, равной 500 и возрастом более 23.
