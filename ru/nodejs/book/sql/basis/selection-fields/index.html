Поля выборки при SQL запросе в NodeJS

В предыдущих уроках при выборке из БД в результат попадали все поля (столбцы) таблицы. Это на самом деле не обязательно - можно указать, какие конкретно поля нам нужны.

Для этого вместо * после команды SELECT через запятую перечисляются имена нужных полей.

Посмотрим на примере. Давайте при выборке из нашей таблицы users достанем только имя и возраст работника:

let query = "SELECT name, age FROM users WHERE id >= 3";

После выполнения кода в консоль выведутся следующие данные:

[ { name: 'user3', age: 23 }, { name: 'user4', age: 30 }, { name: 'user5', age: 27 }, { name: 'user6', age: 28 } ]

Выберите из таблицы users имя, возраст и зарплату для каждого работника.

Выберите id, name и age юзеров, зарплата которых больше 400, но меньше 900.
