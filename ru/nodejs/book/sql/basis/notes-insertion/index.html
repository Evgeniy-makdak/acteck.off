Вставка записей через SQL запрос в NodeJS

Давайте теперь научимся добавлять новые записи в таблицу. Это делается с помощью команды INSERT INTO. Она имеет следующий синтаксис:

let query = "INSERT INTO таблица (поле1, поле2...) VALUES (значение1, значение2...)";

Давайте в нашу таблицу users добавим нового юзера. Обратите также внимание на то, что при вставке не указывается столбец id и его значение. И это правильно, так как значение этого столбца проставится базой автоматически:

let query = "INSERT INTO users (name, age, salary) VALUES ('user', 30, 1000)";

После успешного выполнения кода выведется следующая информация:

ResultSetHeader { fieldCount: 0, affectedRows: 1, insertId: 7, info: '', serverStatus: 2, warningStatus: 0, changedRows: 0 }
