Создание объектов ООП классов в TypeScript

Пусть у нас есть следующий класс User:

class User { }

Давайте создадим объект этого класса:

let user = new User;

Типом переменной, содержащей объект класса, должно быть имя этого класса. Доработаем наш код:

let user: User = new User;
