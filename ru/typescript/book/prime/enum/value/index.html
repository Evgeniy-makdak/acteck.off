Ключ перечисления по значению в TypeScript

В перечислениях можно по названию элемента получить его ключ:

let current: number = Season.Winter; console.log(current); // 0

Расскажите, каким будет результат выполнения следующего кода:

enum Fruits { Apples, Bananas, Oranges, Grapes }; let myFruit: number = Fruits.Apples; console.log(myFruit);
