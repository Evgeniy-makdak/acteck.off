Полезные значения из файла php.ini на хостинге

Давайте рассмотрим полезные настройки из файла php.ini. Эти настройки как правило доступны для редактирования на виртуальном хостинге:

Настройка Описание
display_errors Вывод ошибок в браузер. Значение on включает вывод, а значение off отключает.
memory_limit Ограничение на количество оперативной памяти, выделяемой на один скрипт. Измеряется в мегабайтах.
max_execution_time Ограничение на время выполнения скрипта Измеряется в секундах.
post_max_size Ограничение на размер данных, отправленных методом POST. Измеряется в мегабайтах.
upload_max_filesize Ограничение на размер файлов, отправляемых через инпут с типом file. Измеряется в мегабайтах.
