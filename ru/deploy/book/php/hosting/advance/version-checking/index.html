Проверка версии PHP на хостинге

Версии PHP, установленные на вашем компьютере и на хостинге, могут отличаться. Это может привести к тому, что скрипт, работающий локально, не будет работать на хостинге из-за различия в версиях.

Для того, чтобы узнать какая версия PHP установлена, нужно сделать PHP файл со следующим содержимым:

<?php phpinfo(); ?>

Затем нужно обратиться к этому файлу через адресную строку браузера. В самом верху страницы будет написана текущая версия PHP.
