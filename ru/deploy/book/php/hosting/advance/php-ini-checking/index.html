Проверка настроек php.ini для VH

Настройки php.ini можно проверить с помощью функции phpinfo:

<?php phpinfo(); ?>

Эта функция выводит всю конфигурацию PHP, в том числе и php.ini. Поиском по странице можно найти значение нужной настройки, например, memory_limit.

В результате, который выводит нам phpinfo, для каждой настройки присутствует два значения: Master Value - это то, что действует на весь сайт, и Local value - то, что действует на данный конкретный файл, внутри которого запущена функция phpinfo.
