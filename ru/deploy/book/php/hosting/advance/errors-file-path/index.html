Изменение пути к файлу с PHP ошибками на VS

Давайте сделаем так, чтобы файл с ошибками лежал в папке logs. Для этого нужно предварительно создать эту папку через FTP и убедится, что в нее разрешена запись.

Теперь в настройке error_log укажем имя файла с папкой:

<?php ini_set('display_errors', 'off'); ini_set('log_errors', 'on'); ini_set('error_log', 'logs/errors.log'); echo $test; ?>

Переместите ваш файл с ошибками в отдельную папку.
