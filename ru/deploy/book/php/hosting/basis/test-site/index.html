Тестовый сайт с PHP на хостинге

Итак, у вас уже есть домен и виртуальный хостинг с PHP, а также FTP-доступы к вашему сайту.

Давайте для начала сделаем тестовый скрипт с каким-нибудь простым PHP кодом, чтобы проверить работу PHP на хостинге. Для примера можно сделать вот такое:

<?php echo 'test'; ?>

Подключитесь к вашему сайту через FTP, выложите тестовый скрипт с PHP и проверьте его работу, обратившись к сайту через браузер.
