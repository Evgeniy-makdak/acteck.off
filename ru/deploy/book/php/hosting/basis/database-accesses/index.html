Доступы к базе данных на хостинге

После изучения учебника по работе с SQL на виртуальном хостинге вы должны уже знать, что для работы с базой данных на виртуальном хостинге вам нужно создать базу и юзера. Давайте создадим их для дальнейшей работы с базой данных через PHP. Для этого решите следующие задачи:

Найдите в панели управления хостингом работу с базами данных.

Создайте новую базу данных.

Создайте нового юзера для вашей базы данных.

Зайдите в вашу базу данных через PMA и создайте в ней тестовую таблицу с тестовыми данными.
