Тестирование mysqli подключения к БД на хостинге

Давайте теперь потестируем подключение к базе данных через расширение mysqli с помощью следующего тестового кода:

<?php $host = ''; // имя хоста $name = ''; // имя базы данных $user = ''; // имя пользователя $pass = ''; // пароль $link = mysqli_connect($host, $user, $pass, $name); $query = 'SELECT * FROM users'; $res = mysqli_query($link, $query) or die(mysqli_error($link)); var_dump($res); ?>

Подставьте ваши доступы в тестовый код, затем залейте этот код на ваш сайт и потистируйте соединение с базой данных.
