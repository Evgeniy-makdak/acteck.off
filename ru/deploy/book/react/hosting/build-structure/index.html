Структура React билда

Давайте рассмотрим основную структуру билда проекта. Если вы заглянете в папку build, то увидите там файл index.html и папку static. Давайте изучим их назначение.

Файл index.html представляет собой основной файл сайта, который будет запускаться в браузере. Он содержит начальную верстку проекта и подключение файлов скриптов и стилей.

Папка static содержит скрипты (папка js), стили (папка css) и медиа-файлы (картинки и т.п., папка media)

Изучите структуру билда вашего проекта.
