Сборка React для хостинга

Пусть ваш React проект достиг такой точки, когда вы уже готовы выложить его на хостинг. Для этого вам нужно подготовить билд проекта.

Для создания билда, находясь в папке с проектом выполните следующую команду:

npm run build

После этого в корне вашего проекта появится папка build, содержащая сборку сайта. Содержимое этой папки мы и будем выкладывать на хостинг в следующих уроках.

Сделайте билд вашего проекта.
