Проверка React билда

Итак, мы получили билд своего React проекта. Перед выкладкой на хостинг желательно проверить работу билда на локальном компьютере.

Однако, просто так открыть файл index.html в браузере нельзя - это не будет работать. Проблема здесь в путях к файлам статики, которые начинаются на / и тем самым являются абсолютными.

При желании можно вручную исправить абсолютные пути на относительные, но это неудобный способ. Удобнее запустить билд проекта на локальном сервере.

Существуют различные типы локальных серверов. В следующих уроках мы изучим несколько вариантов.

Попробуйте открыть файл index.html из билда напрямую в браузере. Что из этого получится?
