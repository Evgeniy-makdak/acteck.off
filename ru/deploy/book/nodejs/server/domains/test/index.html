Тестовые домены для VPS сервера на NodeJS

Для начала давайте сделаем подготовительные манипуляции. Купите у регистратора два домена для тестов, либо сделайте на уже купленном домене два поддомена. Далее в учебнике я буду называть их test1.com и test2.com, хотя ваши доменные имена, конечно же, будут другими.

На сайте регистратора в панели управления доменами обоим тестовым доменам установите для A-записи IP вашего VPS сервера.
