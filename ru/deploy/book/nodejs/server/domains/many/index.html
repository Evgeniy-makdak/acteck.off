Несколько доменов на VPS сервере для NodeJS

Для того, чтобы у нас на одном VPS сервере было несколько сайтов со своими доменами, нам нужен обратный прокси сервер. Эта такая программа, которая будет слушать 80 порт и разруливать запросы. В качестве такой программы мы будем использовать сервер Nginx.

Как это будет работать. Пусть у нас есть два домена. Пусть на нашем VPS скрипт для первого домена запущен на 3001 порту, а скрипт для второго домена - на 3002 порту. Обратный прокси сервер Nginx будет слушать 80 порт, смотреть, какой домен был запрошен, и будет перенаправлять запрос на соответствующий порт.

В следующих уроках мы разберемся, как установить и настроить Nginx в режиме обратного прокси сервера.
