Привязка одного домена к VPS серверу на NodeJS

Давайте попробуем самый простой способ привязать домен к серверу. Этот способ сработает, если на вашем сервере будет только один домен.

Для этого способа нужно запустить скрипт тестового сайта на 80 порту:

app.listen(80, function() { console.log('running'); });

Сайт, запущенный на таком порту будет доступен только по IP адресу, без указания порта:

http://12.345.678.90

Далее вам нужно иметь домен или поддомен, купленный у регистратора. В панели регистратора доменов заходим в раздел Домены, там ищем Управление зоной и там ищем наш домен (или поддомен) и меняем для него A-запись на IP нашего сервера.

Ждем несколько минут (не более 10) и после этого пробуем зайти на доменное имя. Если все сделано правильно, мы должны увидеть результат работы тестового скрипта.

Привяжите домен к вашему тестовому сайту.
