Тестовый скрипт для проверки NodeJS хостинга

Давайте сделаем простой тестовый скрипт, который мы загрузим на хостинг, чтобы проверить его работу.

У нас будет два файла. Файл с приложением:

import express from 'express'; let app = express(); app.get('/', function(req, res) { res.send('it works'); }); app.listen(3000, function() { console.log('running'); });

И файл package.json:

{ "type": "module", "dependencies": { "express": "^4.18.2" } }

Сделайте файлы с тестовым скриптом. Потестируйте его работу на локалке.
