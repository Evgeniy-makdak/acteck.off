Установка NVM на VPS сервер

Давайте теперь установим NVM. Для начала узнаем последнюю версию NVM, посмотрев ее по следующей ссылке. Я вижу, что сейчас эта версия v0.39.3 (у вас может быть больше).

Теперь скачаем и установим NVM, указав в URL нужную нам версию:

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

После того, как NVM будет установлен, перезапустите терминал. Затем проверьте, что NVM установился:

nvm -v

Установите NVM и проверьте, что он установился.
