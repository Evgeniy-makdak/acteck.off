Способ установки NodeJS на VPS сервер

Теперь пришло время установить NodeJS на наш VPS сервер. Для этого к серверу следует подключиться через SSH и выполнять необходимые команды через терминал.

Существует несколько способов установить NodeJS. Мы воспользуемся менеджером версий NVM (Node Version Manager), так как он позволяет выбрать версию NodeJS для установки. Хотя этот способ чуть сложнее.
