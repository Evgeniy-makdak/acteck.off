Установка NodeJS через NVM на VPS сервер

Посмотрим список доступных для установки версий NodeJS:

nvm ls-remote

Установим последнюю LTS версию (номер вашей версии может быть больше):

nvm install 18.13.0

Посмотрим список установленных версий:

nvm list

Укажем менеджеру, какую версию следует использовать:

nvm use 18.13.0

Проверим, какая версия NodeJS используется:

node -v

Установите и активируйте нужную версию NodeJS.
