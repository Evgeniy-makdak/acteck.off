Оперативная память VPS для NodeJS хостинга

Оперативная память, или RAM, является самым важным и дорогостоящим показателем.

Для учебных целей нам хватит 1 Гб. Меньше брать не стоит, так как в этом случае сервер будет периодически ложиться от нехватки памяти.

Для боевых ненагруженных проектов будет достаточно 1-2 Гб. Рекомендую изначально покупать 1 Гб, затем, если памяти будет не хватать, ее просто можно будет докупить. Для этого нужно будет написать в поддержку хостинга и сказать, что вам нужно еще оперативной памяти для указанного сервера.

Поиграйтесь с размером оперативной памяти и посмотрите как меняется стоимость сервера.
