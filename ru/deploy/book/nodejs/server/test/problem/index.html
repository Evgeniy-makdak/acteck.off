Проблема тестового сайта на NodeJS на VPS сервере

Итак, мы запустили наш тестовый сайт онлайн. Есть, однако, проблема - если закрыть терминал, то сайт не будет работать.

Нам, конечно же, нужно, чтобы сайт работал постоянно, без нашего коннекта к серверу. Это можно сделать, но для этого нужно установить специальную программу, которая будет держать скрипт запущенным даже после отключения терминала от сервера.

В NodeJS есть несколько библиотек для этого. Это forever и pm2. Мы будем использовать PM2, так как он поддерживает скрипт запущенным даже после перезагрузки сервера. В следущем разделе учебника мы займемся изучением этой библиотеки.

Убедитесь, что отключение терминала от сервера останавливает сайт.
