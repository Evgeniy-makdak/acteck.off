Загрузка тестового NodeJS сайта на VPS сервер

После установки NodeJS у нас все готово для того, чтобы развернуть и запустить первый тестовый сайт. Этим мы сейчас и займемся.

Для начала перейдем в папку с сайтами:

cd /var/www

Теперь создадим папку нашего тестового сайта. Я назвал его test.com. Но лучше назвать эту папку так, как будет называться ваш домен. Это не обязательно, но поможет избежать путаницы.

Итак, создадим папку для сайта:

mkdir test.com

Создайте папку для тестового сайта.

Залейте туда файлы тестового сайта через SFTP (без папки node_modules).
