Типы доменных зон

Доменные зоны можно разделить на несколько типов. Давайте разберемся, что это за типы.

Национальные зоны

Каждая страна имеет свою доменную зону. Например, .ru для России, .uk для Великобритании, .us для США.

Доменные зоны есть и у стран, уже прекративших свое существование. Например, зона .su для Советского Союза.

Региональные зоны

Есть доменные зоны для некоторых регионов. Например, .eu для Евросоюза, .asia для Азии, .moscow для Москвы.

Международные зоны

Изначально были введены общие, международные зоны для сайтов определенной тематики. Например, .com для коммерческих организаций, .info для информационных сайтов, .net для сетевых услуг.

В настоящее время разделение на категории не соблюдается и в этих зонах размещаются любые сайты.

Тематические зоны

Не так давно появилось целое множество доменов для различных тематик. Например, .art, .shop, .club.
