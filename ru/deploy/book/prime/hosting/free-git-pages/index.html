Бесплатный хостинг Git Pages

Существуют особые бесплатные хостинги, более доверенные. Одним из них является хостинг Git Pages. Он позволяет запускать сайты прямо из вашего GitHub репозитория. Для этого автоматически создается поддомен, работающий через https://.

К плюсам такого хостинга следует отнести удобство выкладки и то, что он не блокируется месседжерами и социальными сетями. К минусам такого хостинга следует отнести то, что ваш домен будет не второго уровня. Кроме того, этот хостинг поддерживает только статику.

В целом этот хостинг - хороший вариант для статичных сайтов для портфолио верстальщика или фронтенда.

Если вы еще не умеете работать с Git, для начала вам нужно изучить учебник по работе с Git.

Выложите сайт из своего GitHub репозитория.
