# Вопросы с собеседований — Interview Questions

## 📚 Описание

Раздел с 121 вопросом и ответами для подготовки к собеседованиям на JavaScript, Angular, React и Frontend разработчика.

## 📁 Структура файлов

```
acteck.off/
├── ru/
│   └── interview/
│       ├── index.html          # Русская версия страницы вопросов
├── en/
│   └── interview/
│       ├── index.html          # English version of questions page
├── assets/
│   ├── css/
│   │   └── interview.css       # Стили для раздела интервью
│   └── js/
│       ├── interview-data.js   # Данные вопросов (121 вопрос в 5 категориях)
│       └── interview.js        # JavaScript функционал (категории, поиск, навигация)
└── вопросы с собеседований.docx # Исходный документ с вопросами
```

## 📊 Структура вопросов

Вопросы разделены на 5 категорий:

1. **Основные Вопросы с Ответами для Senior JavaScript Разработчика** (24 вопроса)
2. **Дополнительные Вопросы с Ответами для Senior JavaScript Разработчика** (27 вопросов)
3. **20 Вопросов с Ответами для Собеседования Angular Разработчика** (20 вопросов)
4. **25 Вопросов с Ответами на Собеседовании для React Разработчика** (25 вопросов)
5. **25 Вопросов с Ответами для Собеседования Frontend Разработчика** (25 вопросов)

## 🚀 Использование

### Открыть русскую версию:
```
ru/interview/index.html
```

### Открыть английскую версию:
```
en/interview/index.html
```

### Через главный сайт:
1. Откройте `ru/index.html` или `en/index.html`
2. Кликните на карточку "Вопросы с собеседований" / "Interview Questions"
3. Выберите категорию вопросов
4. Кликните на вопрос для просмотра ответа

## 📊 Статистика

- **Всего вопросов:** 121
- **Категорий:** 5
- **Языки:** Русский и Английский

## 🛠 Техническая реализация

- Данные хранятся в `assets/js/interview-data.js` (встроены для работы без сервера)
- Категории отображаются карточками с эмодзи
- При клике на категорию открывается список вопросов
- При клике на вопрос открывается подробный ответ с кодом
- Поиск работает по вопросам и ответам
- Код форматируется с подсветкой синтаксиса
